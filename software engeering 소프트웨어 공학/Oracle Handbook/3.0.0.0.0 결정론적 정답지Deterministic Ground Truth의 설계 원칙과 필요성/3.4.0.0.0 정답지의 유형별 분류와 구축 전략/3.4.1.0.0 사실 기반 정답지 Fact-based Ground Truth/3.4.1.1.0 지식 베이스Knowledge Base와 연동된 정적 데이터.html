3.4.1.1 지식 베이스(Knowledge Base)와 연동된 정적 데이터

가장 전통적이고 단단한 형태의 결정론적 잣대다. 주로 변동성이 적은 규정집 매뉴얼이나 수학적 상수, 자산 정보 등을 테스트할 때 활용된다.

  • 구축 방식: 별도의 데이터베이스 테이블이나 JSON 파일에 고정된 상식 묶음(Fact-set)을 매핑한다.
  • 예시 설계:
  • 질문: “태양계에서 가장 큰 행성은 무엇인가?”
  • 정답지(Ground Truth): ["목성", "Jupiter"]
  • 오라클 함수: if any(fact in LLM_Response for fact in Ground_Truth_List)

이 정답 시스템 구조는 완벽한 신뢰성을 제공하지만, 한편으로는 세상의 정보(대통령 당선, 환율 등)가 바뀔 때마다 사람이 일일이 JSON 파일을 열어서 내용을 최신화해 주어야 한다는 정적(Static) 패러다임의 한계를 동시에 지니고 있다.
