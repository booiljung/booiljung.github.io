1. 키워드와 임베딩의 AND 연산 오라클

이 결점을 봉합하기 위해 의미론적 유사성에 하드코딩된 필수 키워드 집합을 결합(AND Operation)한 강력한 방어 정답지를 구성한다.

  • [기준 정답지]: “취소를 원하시면 1544-0000으로 전화 주시기 바랍니다. 수수료가 부과될 수 있습니다.” (유사도 > 0.80 요구)
  • [결정론 강제 키워드]: 배열 ["1544-0000", "수수료"] 포함 여부 필수
