3.1.1.3 ’그럴듯함(Plausibility)’과 ’진실(Truth)’의 괴리 분석

엔터프라이즈 임원진들이 AI 테스팅 단계에서 시연 영상을 볼 때 가장 빈번하게 빠지는 치명적인 착각이 있다. 모델이 내놓은 답변의 문장이 유려하고 논리적인 뼈대를 잘 갖추고 있으니, 이 답변의 팩트(Fact) 또한 완벽하게 진실일 것이라는 인지적 편향이다.

1. 환각의 미학 (Aesthetics of Hallucination)

LLM은 진실성(Truth) 을 학습한 적이 단 한 번도 없다. 그들은 오직 무수한 인터넷 데이터를 학습하며 “어떻게 말해야 가장 인간이 보기에 그럴듯하게(Plausible) 믿을까?” 라는 발화 패턴만을 익혔을 뿐이다.

  • 그럴듯함 (Plausibility): “2024년 애플의 영업이익은 3,450억 달러에 달하며, 이는 전년 대비 약 5.3% 상승한 수치로 아이폰 15 Pro Max 모델의 판매 호조가 극도로 크게 기여했습니다.”
  • 진실 (Truth): 회계 장부상 팩트체크 결과 영업이익은 완전한 허구였다.

하지만 인간은 소수점까지 들어간 이 유려한 문법 체계를 짚어보는 순간, ’그럴듯함’에 매몰되어 그것이 백과사전적 진실(Truth)이라고 속아버린다.

결정론적 정답지(Deterministic Ground Truth)는 이러한 LLM의 화려한 언변, 즉 ’그럴듯함’을 가차 없이 산산조각 내버리는 역할을 한다.
“네 문장이 얼마나 자연스러운지는 관심 없고, 네가 내뱉은 숫자가 데이터베이스의 3번째 컬럼 profit 값과 100% 매칭하는가?” 라는 잔혹하리만치 차가운 칼날을 들이대는 것. 이것이 AI의 능수능란한 감언이설을 찢고 진실의 가죽만 남기도록 강제하는 품질 보증(QA)의 핵심 척도다.
