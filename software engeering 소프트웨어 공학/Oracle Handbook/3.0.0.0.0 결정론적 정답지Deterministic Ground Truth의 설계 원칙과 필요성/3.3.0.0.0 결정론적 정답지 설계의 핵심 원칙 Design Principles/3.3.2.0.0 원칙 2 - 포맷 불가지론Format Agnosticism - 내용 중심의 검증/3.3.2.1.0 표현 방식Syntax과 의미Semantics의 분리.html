3.3.2.1 표현 방식(Syntax)과 의미(Semantics)의 분리

우리가 결정론적으로 붙잡고 싶은 것은 팩트의 ‘의미(Semantics)’ 이지, 그 팩트를 출력하는 ’표현의 방식(Syntax)’이 아니다.

예컨대 “우리의 주력 상품의 무게는 얼마인가?” 에 대해, AI 모델 1은 “1.5 kg” 이라고 대답할 수 있고 AI 모델 2는 “1500g“이라고 답변할 수 있다.

  • 만약 정답지(Ground Truth)가 하드코딩된 문자열 "1.5 kg"만을 찾도록 구속되었다면, 두 번째 모델은 의미적으로 완벽한 정답을 내놓았음에도 불구하고 오라클의 비교기(Comparator)를 통과하지 못한다.

이를 방어하기 위해서 정답지는 단위 환산 계수(Unit Converter)를 거치는 등 비교를 수행하기 전 입력과 출력의 표준 정규화(Canonical Normalization) 과정을 설계에 기본 포함시켜야 한다.
