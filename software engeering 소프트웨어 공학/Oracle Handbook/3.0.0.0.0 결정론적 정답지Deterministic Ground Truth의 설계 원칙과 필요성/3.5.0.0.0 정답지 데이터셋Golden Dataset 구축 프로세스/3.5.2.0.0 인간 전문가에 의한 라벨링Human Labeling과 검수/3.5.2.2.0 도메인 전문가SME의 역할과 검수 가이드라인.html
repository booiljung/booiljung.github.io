3.5.2.2 도메인 전문가(SME)의 역할과 검수 가이드라인

데이터의 라벨링 작업은 흔히 크라우드소싱(Crowdsourcing) 아르바이트생에게 푼돈을 주고 넘기는 작업으로 치부되지만, 기업 환경의 ’정답지 구축’에서는 패망의 지름길이다.

  • SME (Subject Matter Expert)의 필수성: 세무 상담 AI의 정답지가 올바른지 판단하기 위해서는 일반 대학생이 아닌, 세무사 자격증을 지닌 회계 전문가(SME)가 참여해야만 환각을 식별할 수 있다.
  • 검수 가이드라인 (Guideline): SME가 작업하더라도 ’무엇을 기준으로 채점할 것인가’에 대한 회사의 마스터플랜 가이드라인이 존재해야 한다.
  • “만약 법 조항이 올해 바뀌었다면 작년 데이터를 어떻게 처리할 것인가?”
  • “사용자가 욕설과 세무 질문을 동시에 던졌다면, 거절할 것인가 아니면 욕설을 무시하고 답변할 것인가?”

이러한 엣지 케이스 처리 지침서(Guideline Book)가 라벨러들의 모니터 옆에 상시 부착되어 있을 때, 비로소 인간이 만든 정답지가 AI보다 낫다는 품질의 권위를 가지게 된다.
