3.2.4.1 설명 가능한 AI(XAI)를 위한 근거 데이터 확보

현대의 AI 규제 법안들은 “컴퓨터가 그렇게 말했기 때문입니다“라는 블랙박스식 변명을 엄격히 금지하고 있다. 만약 AI 챗봇이 고객에게 불리한 계약 조건을 안내했다면, 회사는 도대체 ‘왜’ 그런 문장을 생성했는지 원인을 역추적하여 법원에 소명해야만 한다. 이를 뒷받침하는 것이 설명 가능한 AI(Explainable AI, XAI) 패러다임이다.

1. 블랙박스(Blackbox)를 해체하는 쐐기

결정론적 정답지 데이터셋(Golden Dataset)은 텍스트 정답만 담고 있지 않다. 그 구성 요소 내부에는 ’이 정답이 도출되기 위해 참조해야 했던 법령 문서의 특정 조항(Citation)’과 ’로직 트리(Logic Tree)’가 메타데이터로 단단히 결속되어 있다.

  • 사용자 질문이 AI 생성을 거쳐 최종 출력될 때, 보수적 오라클은 이 출력을 단순히 비교하는 것을 넘어 “출력 문장 내에 정답지가 규정한 근거 조항 ID(Citation ID)가 누락되었는가?“를 강제 스캐닝한다.
  • 누락되었다면 가차 없이 Fail 처리됨으로써, 시스템 밖으로 나가는 모든 AI의 발화(Utterance)가 법적 설명 가능성, 즉 ’근거(Ground)’라는 꼬리표를 달고 배포되도록 강제하는 역할을 수행한다.
