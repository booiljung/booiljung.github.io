2.4 AI 및 LLM(거대 언어 모델) 도입으로 인한 테스트 패러다임의 붕괴

지금까지 우리가 살펴본 소프트웨어 테스팅과 오라클의 역사적 분류 체계는 “기계는 인간이 시킨 대로만 정확하게 연산한다“는 결정론적 신화(Deterministic Myth) 위에 쌓아 올린 바벨탑이었다. 그러나 파라미터(Parameter)의 크기가 수천억 개에 달하는 대규모 언어 모델(LLM)이 엔터프라이즈 아키텍처의 한가운데로 영입되면서 이 바벨탑은 산산조각이 나고 말았다.

본 절에서는 AI의 도입이 어째서 기존 소프트웨어 테스트 패러다임을 뿌리째 붕괴시키는지 그 원인을 해부한다. 결정론의 상실부터 시작하여, 정답이 무한대로 뻗어 나가는 다중 정답의 늪, 악명 높은 환각(Hallucination) 현상, 그리고 문맥적 동등성을 판별하지 못하는 구형 오라클의 한계까지, LLM 시대에 불어닥친 전례 없는 QA(Quality Assurance)의 위기를 낱낱이 파헤친다.
