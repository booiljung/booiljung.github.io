2.11.1 예측 및 분류 모델(Predictive)에서의 정답(T/F) 도출과 혼동 행렬(Confusion Matrix)

2010년대 중후반을 지배했던 전통적 기계 학습, 즉 예측형(Predictive) AI는 본질적으로 ‘초정밀 분류기(Classifier)’ 였다. 엑스레이(X-ray) 사진을 보고 암세포가 있는지(True), 없는지(False)를 판별하거나, 이 이메일이 스팸(Spam)인지 아닌지를 갈라내는 역할을 수행했다.

1. 확정된 정답과 혼동 행렬의 평화

이 시대의 오라클은 매우 평화롭고 명확했다. 입력값에 대한 ’정답 클래스(Label)’가 사전에 수학적으로 확정되어 있었기 때문이다.

  • 모델의 출력값은 결국 0 아니면 1, 혹은 제한된 n개의 카테고리(Category) 중 하나로 강제 수렴된다.
  • 테스트 스크립트는 모델의 예측값(Predicted)과 실제 인간이 매겨둔 정답(Actual)을 1:1로 비교한다.
  • 이들은 그 유명한 혼동 행렬(Confusion Matrix) 이라는 강력한 오라클 매트릭스 안에 안착한다.
