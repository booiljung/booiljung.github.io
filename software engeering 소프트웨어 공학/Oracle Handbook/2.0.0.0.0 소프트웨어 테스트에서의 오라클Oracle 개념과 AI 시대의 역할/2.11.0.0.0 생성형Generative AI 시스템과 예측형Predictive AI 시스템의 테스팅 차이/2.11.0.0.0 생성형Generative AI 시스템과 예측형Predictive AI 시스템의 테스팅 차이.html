2.11 생성형(Generative) AI 시스템과 예측형(Predictive) AI 시스템의 테스팅 차이

세간의 많은 튜토리얼들이 AI 테스팅을 논할 때 범하는 치명적인 오류는, 전통적인 ‘예측형 AI(Predictive AI)’ 시절의 지표를 최신 ’생성형 AI(Generative AI)’에 그대로 대입하려는 나태함이다. 이 두 시스템의 본질적인 출력 구조는 완전히 다르며, 따라서 테스트 파이프라인과 오라클의 설계 사상의 근본 뿌리부터 절단되어야 한다.

이 섹션에서는 예측형 모델에서 사용되던 확고한 이진 판별의 세계와, 생성형 모델이 열어젖힌 무한한 출력 공간 사이의 구조적 단층을 비교 분석한다. 이를 통해 “왜 기존의 머신러닝 테스트 지표가 LLM 앞에서는 쓰레기가 되는가?“에 대한 근본적인 해답을 증명한다.
