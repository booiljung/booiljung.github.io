12.12.2 테이블/컬럼 매핑 사전을 활용한 스키마 기반 논리적 동등성 평가 로직

테이블 조인(Join)의 경로는 다양하다. A JOIN B 와 B JOIN A의 동등성을 AST 파서가 알아채려면 DB 스키마 구조(Schema Topology) 매핑 사전이 오라클 메모리에 로드되어 있어야 한다.

  • 오라클은 스키마 메타데이터 딕셔너리를 활용해, 대상 쿼리들이 결국 같은 PK-FK 연결망(Edge)을 타고 있는지를 그래프 이론에 기반하여 유추할 수 있다.
  • 복잡하지만, 이 단계까지 도달하면 실행기(Executor) 없이도 90%에 육박하는 의미론적 검증 능력을 발휘한다.
