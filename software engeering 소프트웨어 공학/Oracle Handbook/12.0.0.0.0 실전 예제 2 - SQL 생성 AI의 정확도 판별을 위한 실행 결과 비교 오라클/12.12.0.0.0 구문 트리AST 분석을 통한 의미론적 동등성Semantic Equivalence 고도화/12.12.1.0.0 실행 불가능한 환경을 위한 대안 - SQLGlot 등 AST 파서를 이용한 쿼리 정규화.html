12.12.1 실행 불가능한 환경을 위한 대안: SQLGlot 등 AST 파서를 이용한 쿼리 정규화

파이썬 기반 파서인 sqlglot 등을 이용해 SQL 텍스트를 추상 구문 트리(Abstract Syntax Tree) 객체 단위로 분해한다.

  • SELECT a, b 나 SELECT b, a 는 트리의 projection 노드에서 리스트(List)가 아닌 집합(Set)으로 교차 비교하면 동등성 인식이 가능하다.
  • 불필요한 공백, 대소문자, 생략 가능한 AS 별칭, INNER JOIN 을 JOIN 으로 치환하는 구문 평탄화(Canonicalization)를 거친 후 양쪽 트리의 깊이(Depth)와 리프(Leaf) 노드들을 비교(Tree Diffing)하여 무실행 환경에서의 방어선을 구축한다.
