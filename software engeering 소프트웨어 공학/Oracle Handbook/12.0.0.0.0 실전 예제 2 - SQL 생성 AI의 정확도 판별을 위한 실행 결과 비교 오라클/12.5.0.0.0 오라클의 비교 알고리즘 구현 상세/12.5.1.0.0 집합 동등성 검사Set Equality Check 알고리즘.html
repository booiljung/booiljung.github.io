12.5.1 집합 동등성 검사(Set Equality Check) 알고리즘

단순히 길이(A) == 길이(B) 만 체크하는 것은 속임수에 당할 수 있다.

  • 파이썬 기반 방어 로직: 
    def is_set_equal(golden_res, ai_res):
    # 1. 길이가 다르면 즉시 탈락
    if len(golden_res) != len(ai_res): return False
    
    # 2. 정규화된 Row 해시들의 카운팅(Bag Semantics) 교집합 비교
    golden_counter = Counter([hash_row(r) for r in golden_res])
    ai_counter = Counter([hash_row(r) for r in ai_res])
    
    return golden_counter == ai_counter

이처럼 파이썬 표준 라이브러리의 `collections.Counter`를 활용하면 집합론(Set Theory) 기반의 견고한 비교를 수행할 수 있다.
