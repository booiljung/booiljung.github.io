12.4.4 부동 소수점(Floating Point) 비교를 위한 허용 오차(Tolerance) 설정

AI가 반올림 함수(ROUND())를 임의로 쓰거나 DB 엔진 내부의 실수 연산 오차로 인해 10.33333 과 10.33334가 부딪히는 경우다.

  • 방어 기제: 오라클 내부의 스칼라(Scalar) 비교 연산자는 == 가 되어서는 안 된다. math.isclose(a, b, rel_tol=1e-5) 와 같은 상대 오차 허용 함수로 실수(Float) 비교를 래핑(Wrapping)해야 데이터 분석용 SQL 검증이 무너지지 않는다.
