12.8.2 Reference SQL과 Generated SQL의 병렬 실행 클래스 구현

Python의 asyncio 혹은 ThreadPoolExecutor를 활용해 평가 속도를 끌어올린다.

import pandas as pd
from sqlalchemy import create_engine

engine = create_engine("sqlite:///:memory:")
# (여기서 초기 골든 데이터 insert 로직 수행)

def fetch_result(query: str):
    try:
        return pd.read_sql(query, engine)
    except Exception as e:
        return str(e) # 에러 시 에러 문자열 로깅

이 뼈대 위에서 골든 DB의 결과(fetch_result(golden))와 AI의 결과(fetch_result(ai))가 동시(Concurrency)에 뻗어나가는 체계를 짜는 것이 실행기 모듈의 기본이다.
