12.8.3 Pandas DataFrame을 활용한 결과 비교 로직 구현 코드

데이터가 DataFrame에 적재되면 판다스가 자랑하는 equals 기능을 쓸 수 있지만, 먼저 순서 무관성(Order Agnosticism)을 맞춰야 한다.

def check_equivalence(df_golden, df_ai):
    # 에러 문자열이 껴 있으면 무조건 Fail
    if isinstance(df_golden, str) or isinstance(df_ai, str):
        return False
        
    # 행 길이 불일치 
    if len(df_golden) != len(df_ai): return False
    
    # 1. 컬럼명을 버리고 값 배열만 추출 (차원 축소)
    gold_vals = df_golden.values.tolist()
    ai_vals = df_ai.values.tolist()
    
    # 2. 내부 데이터 정렬로 난수성 제거 후 비교
    return sorted(gold_vals) == sorted(ai_vals)

이 로직이야말로 “컬럼명은 무시하고, 행의 순서도 무시하되, 담겨있는 알맹이의 동치성은 보장하라“는 오라클 룰의 파이썬식 구현체다.
