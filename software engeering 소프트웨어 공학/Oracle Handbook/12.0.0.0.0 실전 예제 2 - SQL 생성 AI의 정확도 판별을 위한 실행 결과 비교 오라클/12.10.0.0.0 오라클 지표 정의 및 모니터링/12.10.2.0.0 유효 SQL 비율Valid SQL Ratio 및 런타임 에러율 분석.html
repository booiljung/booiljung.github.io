12.10.2 유효 SQL 비율(Valid SQL Ratio) 및 런타임 에러율 분석

EX Score가 낮더라도, 틀린 이유가 “원하는 데이터가 안 나와서“인지 “문법을 박살 내서 DB가 에러를 뿜었는지“는 천지 차이다.

  • VSR (Valid SQL Ratio): 성공적으로 실행되어 에러 없이 ResultSet을 리턴한 쿼리(오답 포함) / 전체 쿼리
  • VSR 값이 낮다면 AI 프롬프트가 해당 데이터베이스 방언(Syntax)을 아예 이해하지 못하고 있다는 징후다. 주로 환각 컬럼(Hallucinated Columns) 호출이나 JOIN 문법 비틀림이 주원인이므로 이를 우선적으로 타격(Tuning)해야 한다.
