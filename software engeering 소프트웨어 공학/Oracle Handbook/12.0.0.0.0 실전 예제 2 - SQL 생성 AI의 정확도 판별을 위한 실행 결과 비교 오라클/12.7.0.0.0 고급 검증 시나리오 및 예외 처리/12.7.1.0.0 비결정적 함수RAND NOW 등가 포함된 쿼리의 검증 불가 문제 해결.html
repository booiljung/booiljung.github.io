12.7.1 비결정적 함수(RAND, NOW 등)가 포함된 쿼리의 검증 불가 문제 해결

“오늘 가입한 회원을 보여줘” -> AI가 SELECT * FROM user WHERE date = CURRENT_DATE() 구조를 짤 수 있다.

  • 문제는 CI 환경이 돌아가는 시점마다 CURRENT_DATE() 결괏값이 달라서 멱등성이 깨진다는 것이다.
  • AST 변조기(AST Mutator): 쿼리가 실행기(Executor)로 넘어가기 전 조작(Mangle) 단계를 거친다. SQLGlot 같은 파서를 이용해 쿼리 내의 CURRENT_DATE(), NOW(), RAND() 노드를 찾아내고 이를 고정된 샌드박스의 상수 문자열(예: '2023-11-01', 0.5)로 전부 치환(Override)한 후 실행하여 비교의 객관성을 확보한다.
