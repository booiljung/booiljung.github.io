11.1.3 비즈니스 크리티컬(Business-Critical) 시스템에서의 허용 오차(Tolerance) 제로 원칙

영화 추천 챗봇이나 요리 레시피 챗봇에서 토마토 1개를 2개로 잘못 말하는 것은 사용자 경험(UX) 저하에 그친다. 반면, B2B나 금융, 의료(Business-Critical) 도메인에서 약관 위반 결괏값을 내보내는 것은 곧바로 수십억 원의 소송(Lawsuit)과 직결된다.

  • 원칙: 수학적 산출과 법적 조건(Rule)이 걸린 질의응답에서는 AI 모델 자체의 지능(능력)을 1%도 신뢰해서는 안 된다. 허용 오차(Tolerance)는 0(Zero)이어야 한다.
