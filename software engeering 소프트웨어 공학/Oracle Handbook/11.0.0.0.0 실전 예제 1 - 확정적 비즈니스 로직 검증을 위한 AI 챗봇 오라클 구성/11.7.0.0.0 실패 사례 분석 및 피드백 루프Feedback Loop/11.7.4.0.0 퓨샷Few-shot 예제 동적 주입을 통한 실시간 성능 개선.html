11.7.4 퓨샷(Few-shot) 예제 동적 주입을 통한 실시간 성능 개선

파인튜닝(PEFT 등)은 시간이 오래 걸린다. 가장 빠르게 모델을 교정하는 방법은 프롬프트를 고치는 것이다.

  • 빈번하게 오라클 에러를 내는 특정 유형(예: 나이 계산)의 에러 사례를 3~4개 발췌하여, 메인 모델의 시스템 프롬프트 상단 [Few-Shot Examples] 블록에 하드코딩한다.
  • “유저가 ‘올해 스물’ 이라고 하면 age=20 으로 매핑하라. 절대 문자를 넣지 마라.”
  • 이 조치만으로도 오라클 매핑 정확도가 70%에서 99%로 수직 상승한다.
