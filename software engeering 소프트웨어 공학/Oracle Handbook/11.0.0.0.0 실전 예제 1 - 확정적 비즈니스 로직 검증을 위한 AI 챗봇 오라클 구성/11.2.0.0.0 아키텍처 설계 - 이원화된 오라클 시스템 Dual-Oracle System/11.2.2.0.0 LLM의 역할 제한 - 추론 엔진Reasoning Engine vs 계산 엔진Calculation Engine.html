11.2.2 LLM의 역할 제한: 추론 엔진(Reasoning Engine) vs 계산 엔진(Calculation Engine)

  • 추론 엔진(LLM): “19살“이라는 텍스트에서 age=19 라는 변수를 추출해 내는 작업, 그리고 딱딱한 API 에러 메시지를 “죄송합니다 고객님“이라는 유려한 문장으로 포장하는 것까지만 허용한다.
  • 계산 엔진(Oracle): 나이가 20세 이상인지 판별하고, 5만 원에 20%를 수학적으로 더해 6만 원을 도출하는 100% 결정론적 연산은 무조건 사내 서버 로직(Oracle C API 또는 Java Spring Bean)이 전담한다.
