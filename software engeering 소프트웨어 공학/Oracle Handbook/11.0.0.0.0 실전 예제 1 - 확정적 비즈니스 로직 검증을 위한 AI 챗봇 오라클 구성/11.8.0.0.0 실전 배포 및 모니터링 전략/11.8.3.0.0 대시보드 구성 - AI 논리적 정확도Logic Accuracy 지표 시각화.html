11.8.3 대시보드 구성: AI 논리적 정확도(Logic Accuracy) 지표 시각화

팀이 추적해야 할 차트는 단순 응답 시간(Latency)이나 서버 CPU 점유율이 아니다.

  • 핵심 KPI 대시보드 (Grafana 등):
  • Oracle Match Rate: AI가 호출한 파라미터가 오라클의 기대치와 얼마나 일치하는가 (목표 99.9%).
  • Parameter Extraction Fail Rate: 유저의 입력을 JSON 변수로 파싱하는 데 실패한 비율.
  • Tone Tolerance Alert: 보조 AI Judge가 경고를 띄운 거친 대화의 빈도수.
    개발팀은 매일 이 논리적 정확도 지표를 보며 프롬프트를 튜닝해야 한다.
