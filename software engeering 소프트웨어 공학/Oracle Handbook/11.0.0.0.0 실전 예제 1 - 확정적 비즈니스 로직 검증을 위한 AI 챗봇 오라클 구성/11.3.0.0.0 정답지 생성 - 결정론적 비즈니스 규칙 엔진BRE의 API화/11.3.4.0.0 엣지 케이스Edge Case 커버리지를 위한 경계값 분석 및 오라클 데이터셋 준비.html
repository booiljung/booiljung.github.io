오라클 모듈을 구현했다면, 이제 이 엔진에 불을 지피기 위한 가혹한 테스트용 연료(Payload) 쌍(Pair)들을 생성해야 한다. 경계값 부근에선 AI도 헷갈리고 레거시 로직도 터지기 쉽다.

  • age: 19, age: 20, age: 21 의 3연속 찌르기(Off-by-one 테스트).
  • age: -5, age: 250 (논리적 비정상 입력 거부에 대한 일관성 확인).
    위 입력들을 로컬 오라클 API에 던져 정답 세트(expected_outputs)를 벌크 생성하여, 이 챕터 앞단에서 논의한 “골든 데이터셋” 저장소에 장전시킨다.
