11.5.4 대화 톤앤매너(Tone & Manner) 검증을 위한 보조 LLM-as-a-Judge 설정

숫자와 로직이 완벽히 맞아도, 봇이 사용자에게 반말을 하거나 공격적인 어조(Aggressive Tone)를 사용한다면 프로덕션 배포는 실패다.

  • 메인 계산이 끝난 후, 보조 AI 판사(Auxiliary Judge)에게 최종 스크립트를 던져 회사의 브랜드 레퍼런스(Brand Identity) 가이드라인(존댓말 사용, 지나치게 짧은 단답 금지 등)을 준수했는지 1~5점으로 스코어링(Scoring)하게 한다.
  • 이 점수가 4점 이하면 계산이 완벽했더라도 전체 테스트 케이스를 Warning 처리한다.
