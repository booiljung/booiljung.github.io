7.5.3 자기 선호 편향(Self-Preference Bias): 자사 모델 생성물을 더 높게 평가하는 문제

세상의 모든 LLM은 자신이 속한 모델 패밀리(Family)의 말투와 코딩 스타일을 내재적으로 선호한다.
예를 들어 GPT-4를 심판으로 고용하면, Claude의 답변보다 GPT-3.5의 구사 방식에 본능적으로 친숙함을 느끼고 더 높은 기준(Score)을 책정하게 된다.

  • 해결책: 평가 시스템을 구성할 때는 가급적 독립적인 제3의 모델 진영을 선택하거나, “이 문장은 인간/타사 AI가 썼다“고 암시하는 블라인드 프롬프트를 씌우는 것이 이상적이다. 또는 오픈소스 오라클 모델(예: Llama의 Judge 버전)을 별도로 튜닝하여 평가 전용으로만 종속시키는 것이 가장 안전한 해독제다.
