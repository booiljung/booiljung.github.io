7.5.2 서술적 편향(Verbosity Bias): 길고 장황한 답변을 선호하는 경향 억제

대부분의 LLM 모델(GPT 계열 특히 강함)은 짧고 명료한 진실보다, 틀린 말이라도 길고 미사여구가 화려하게 붙어 있는 답변에 높은 점수를 남발한다. 이를 서술적 편향(Verbosity Bias)이라 부른다.

  • 해결책 (Length Punishment):
    평가 프롬프트 루브릭(Rubric)에 "답변의 길이는 품질에 아무런 가산점이 되지 않으며, 장황하게 핵심을 흐릴 경우 오히려 강력한 감점 요인으로 작용한다" 라는 제약 조항을 최우선으로 투입하여 장문 맹신을 부숴야 한다.
  • 길이 강제 전처리: 프롬프트를 오라클에 올리기 전에 양쪽 답변을 특정 글자 수로 요약(Summarize)하거나 동일한 분량으로 잘라내는(Truncate) 파이프라인 처리를 고려하라.
