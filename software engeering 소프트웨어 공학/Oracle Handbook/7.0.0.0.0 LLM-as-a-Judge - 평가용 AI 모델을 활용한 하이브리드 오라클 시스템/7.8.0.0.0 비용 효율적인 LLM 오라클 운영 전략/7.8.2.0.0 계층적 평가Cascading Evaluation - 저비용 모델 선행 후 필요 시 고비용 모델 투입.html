7.8.2 계층적 평가(Cascading Evaluation): 저비용 모델 선행 후 필요 시 고비용 모델 투입

전수 검증과 비용의 타협안으로 자주 쓰이는 멀티-티어(Multi-tier) 라우팅 오라클이다.

  • 1차 관문 (경량 모델): gpt-4o-mini 나 로컬 Llama 모델이 1차 평가를 단행한다. 이 때 점수가 1~2점으로 확실하게 망했거나, 확실하게 성공한 5점이라면 1차 모델의 평가를 그대로 최종 저장하고 닫는다 (전체 트래픽의 80% 소화).
  • 2차 관문 (헤비 모델): 1차 평가 결과, 점수가 3점이거나 confidence가 기준치 이하로 떨어진 “애매한” 20%의 응답들만 상위로 쏘아 올려 GPT-4o 나 Claude 3.5 Sonnet 에게 심층 판결(Supreme Court)을 요구한다.
