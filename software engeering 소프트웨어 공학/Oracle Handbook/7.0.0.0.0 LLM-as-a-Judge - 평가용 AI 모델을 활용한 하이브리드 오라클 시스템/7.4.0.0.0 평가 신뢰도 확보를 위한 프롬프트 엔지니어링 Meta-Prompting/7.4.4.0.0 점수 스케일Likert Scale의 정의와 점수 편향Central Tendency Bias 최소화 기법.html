7.4.4 점수 스케일(Likert Scale)의 정의와 점수 편향(Central Tendency Bias) 최소화 기법

1점에서 5점까지의 리커트 척도(Likert Scale)를 설계할 때, “1점: 나쁨, 5점: 좋음” 같은 빈약한 스케일 기준은 치명적인 응답 편향을 유발한다. AI는 귀찮거나 모를 때 3점을 주려는 ’중심화 경향(Central Tendency Bias)’이 짙다.

편향을 부수는 스케일 가이드 원칙:

  • 1점: 사용자 질문과 완전히 무관하며, 사실과 다른 치명적 환각 포함.
  • 2점: 부분적으로 답변했으나 논점 이탈 심각.
  • 3점: 질문에 응답했으나, 논문 참조나 명확한 수치 근거 제시 누락.
  • 4점: 정확하고 유용하나, 더 나은 최적화 여지가 존재함.
  • 5점: 완벽한 정답이며, 부작용과 예외 케이스까지 포괄하여 설명함.

각 단위(Tick)마다 행위 기반의 명확한 통과 조건(Condition)을 매핑하지 않으면, 70% 이상의 응답이 무가치한 3~4점으로 뭉치게 된다.
