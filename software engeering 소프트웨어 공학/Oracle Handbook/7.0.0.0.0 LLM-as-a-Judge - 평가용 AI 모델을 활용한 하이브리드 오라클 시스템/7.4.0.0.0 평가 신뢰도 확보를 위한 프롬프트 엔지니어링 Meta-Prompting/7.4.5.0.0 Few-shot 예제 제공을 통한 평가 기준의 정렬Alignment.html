7.4.5 Few-shot 예제 제공을 통한 평가 기준의 정렬(Alignment)

평가 프롬프트 내에 인간이 매긴 모범 채점 사례(Golden Examples)를 2~3개 넣어주는 것(Few-shot Prompting)은 기준선의 정렬(Alignment)을 극명하게 개선한다.

[평가 예시 1]
입력 문장: ...
정답 해설: 이 문장은 ~한 이유로 감점 요인이다.
최종 점수: 2

[평가 예시 2]
입력 문장: ...
정답 해설: 모든 항목을 충족했다.
최종 점수: 5

이 퓨샷 예제는 단순한 지시문을 넘어 심판관 LLM의 파라미터를 암묵적(Implicit)으로 일시 조정하는 효과를 발휘하며, 까다로운 비즈니스 특화 도메인일수록 평가 정확도를 수직 상승시킨다.
