7.7.3 3단계: JSON Schema Validation을 통과한 데이터에 대한 의미론적 심층 평가

1단계와 2단계의 혹독한 기계적 채점을 뚫고 올라온 소수의 데이터(Cleaned Data)만이 맞이하는 최종 관문(Layer 3)이 바로 AI 관문이다.
