7.3.3 유해성(Toxicity) 및 편향성(Bias) 탐지를 위한 안전 오라클 기준

성능 평가를 넘어선, B2C 서비스의 존폐를 가르는 최고 보안 등급의 오라클이다.

LLM 생태계의 대표적인 안전망인 Llama Guard나 OpenAI의 Moderation API의 핵심 철학을 도입한다.

  • 혐오 발언(Hate speech), 자해 유도(Self-harm), 불법 행위 촉구(Illegal acts)의 3가지 핵심 카테고리를 프롬프트 상단에 상수(Constant)로 정의한다.
  • 판별된 유해성이 어느 카테고리에 속하는지 Enum 형태로 강제 반환하게 하고, 트리거 시 즉시 System Exit 또는 Fallback Response로 연결하는 하드 와이어드(Hard-wired) 아키텍처를 구축하라.
