7.3.1 모호함을 제거하는 평가 가이드라인(Guidelines) 작성 원칙

가이드라인은 길고 장황한 산문(Prose)이 아니라, 즉시 실행 가능한 조건문(If-Then Statement)의 집합이어야 한다.

  • Bad: “답변이 유용한지 평가해 주세요.”
  • Good: “사용자의 질문에 포함된 모든 쟁점에 대해 1회 이상 답변이 포함되었는가? 답변 내에 실행 가능한 URL이나 코드 스니펫이 제공되었는가? 이 두 가지를 모두 만족하면 5점, 하나만 만족하면 3점, 모두 없으면 1점을 부여하라.”

인간이 보기에 숨 막힐 정도로 촘촘하게 제약 조건을 나열해야 한다. 평가 기준이 뾰족할수록(Pungent) GPT-4와 Claude 3.5 간의 평가 일치도(Kappa)는 기하급수적으로 상승한다.
