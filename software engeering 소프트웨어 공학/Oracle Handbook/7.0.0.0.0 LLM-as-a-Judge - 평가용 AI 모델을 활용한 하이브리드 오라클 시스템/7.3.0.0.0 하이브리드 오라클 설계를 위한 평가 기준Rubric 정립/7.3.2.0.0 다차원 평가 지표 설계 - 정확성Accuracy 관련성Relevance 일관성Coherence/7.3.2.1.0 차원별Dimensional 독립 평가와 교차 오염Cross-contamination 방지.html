7.3.2.1 차원별(Dimensional) 독립 평가와 교차 오염(Cross-contamination) 방지

정확성과 유창함(Fluency)은 별개의 지표다. 하지만 LLM 심판에게 이를 한 번에 물어보면, 글이 너무 유창하게 잘 쓰였다는 이유로 명백한 사실적 오류(Inaccuracy)마저 높은 점수를 주어버리는 후광 효과(Halo Effect) 가 발생한다. 이를 평가의 교차 오염(Cross-contamination)이라 부른다.

방어 전략:

  • API를 호출할 때 “정확성만을 평가하는 프롬프트“와 “관련성만을 평가하는 프롬프트“를 별도의 세션(Session)으로 완전히 분리 분할(Decoupling)하여 호출한다. 혹은 JSON 파라미터 구조를 통해 각 지표 사이에 단절벽(Barrier)을 강제로 형성시킨 후 점수를 취합해야 한다.
