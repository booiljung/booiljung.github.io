7.2.3 Pairwise Comparison(쌍별 비교) 방식의 승자 판별 오라클

단일 답변을 주고 1점에서 5점까지 점수를 매기는 절대 평가는 사실 LLM이 굉장히 어려워하는 작업이다. 반면, “두 개의 답 중 어느 것이 더 낫느냐“를 고르는 상대 평가(Pairwise Comparison) 능력은 모델 파라미터 규모와 상관없이 매우 정확하다.

A/B 테스트 환경의 필수 아키텍처:

  • 프롬프트 V1의 답변과 V2의 답변을 동시에 오라클에 올린다.
  • 심판 LLM은 A가 이겼는지, B가 이겼는지, 혹은 무승부(Tie)인지 출력한다.
  • 대표 평가 프레임워크인 챗봇 아레나(Chatbot Arena)의 ELO 레이팅 기반 평가가 바로 이 쌍별 비교 오라클을 수만 번 반복시켜 구축한 랭킹 시스템이다.
