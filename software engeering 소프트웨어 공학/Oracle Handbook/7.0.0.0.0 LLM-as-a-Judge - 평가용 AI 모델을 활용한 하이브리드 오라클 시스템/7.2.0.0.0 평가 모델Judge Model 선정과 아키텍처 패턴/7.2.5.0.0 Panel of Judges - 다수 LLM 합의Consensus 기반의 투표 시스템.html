7.2.5 Panel of Judges: 다수 LLM 합의(Consensus) 기반의 투표 시스템

단일 심판관(Judge)의 환각이나 근시안적 판별력을 극복하기 위해 제안된 최고 등급의 평가 아키텍처다.

  • GPT-4, Claude 3.5, Gemini 1.5 Pro 세 개의 최고 티어 모델을 동시에 심판석에 앉힌다.
  • 동일한 평가 프롬프트를 전송한 후, 세 모델 중 두 개 이상이 Fail을 던지면 이를 최종적으로 모델 불합격 판정으로 간주한다.
  • 각 모델마다 특유의 편향성(예: Claude는 도덕성에 민감하고 코딩에 관대함)이 상쇄되므로 평가의 신뢰도 곡선이 완벽에 가깝게 선형 수렴하게 된다. 다만, API 호출 비용 또한 3배 이상 폭증하므로 핵심적인 골든 데이터셋(Golden Dataset) 구축 시에만 한정하여 사용하라.
