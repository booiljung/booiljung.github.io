7.9.3 챗봇 시나리오 테스트에서의 의도 파악(Intent Recognition) 정확도 평가

사용자는 결코 정갈하게 말하지 않는다. “비밀번호 까먹었음 어떡함”, “비번 분실” 등 수십 가지 파편화된 언어를 동일한 의도(Intent)로 인식하는지를 테스트해야 한다.

  • 오라클 구축: {"query": "비번 까먹음", "expected_intent": "RESET_PASSWORD"} 형태의 골든 데이터셋 1,000개를 구성한다.
  • 챗봇 모델이 내뱉은 응답이나 분류 해시를 가로채어, LLM 심판에게 “이 응답이 expected_intent를 성공적으로 해소(Resolve)하는 방향으로 작성되었는가? (Pass/Fail)” 만을 엄격하게 묻는다.
    이는 기존의 무미건조한 키워드(Regex) 매칭 테스트와 차원을 달리하는, 인간의 뉘앙스를 심리적으로 읽어내는 고차원의 단위 테스트(Unit Test)다.
