7.9.4 다국어 번역 품질 평가를 위한 교차 언어적 의미 보존 확인

기계 번역(MT)의 품질을 측정할 때 과거에는 단순 단어 매칭 척도인 BLEU를 썼으나, 의미론의 훼손을 전혀 반영하지 못하는 한계가 짙었다.

  • LLM 오라클 도입: 프롬프트 안에 <SOURCE>(일본어 원문)와 <TRANSLATION>(한국어 번역본)을 통으로 집어넣는다.
  • “너는 완벽한 이중 언어 구사자(Bilingual)다. 목적어 누락, 뉘앙스의 상실, 존댓말/반말 표기의 어색함 세 가지 축에서 번역의 완전함을 10점 만점으로 평가해라.”
    BLEU 점수로는 완벽했던 오역(Literal Translation)들이, 시맨틱 오라클을 통과하자마자 3점을 받고 반려되는 정화 현상이 조직 내 번역 파이프라인 전반에 일어났다.
