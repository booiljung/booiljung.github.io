7.6.2 Cohen’s Kappa 및 Krippendorff’s Alpha를 이용한 평가 일치도 분석

점수의 상관관계뿐만 아니라 “Pass/Fail” 같은 범주형(Categorical) 오라클을 사용할 때의 평가 지표다.

  • Cohen’s Kappa (코헨의 카파): 우연에 의해 두 평가자(인간과 AI)의 의견이 일치할 확률을 제외한 순수 일치도. 0.6 이상이면 양호, 0.8 이상이면 완벽한 합의로 간주한다.
  • Krippendorff’s Alpha: 다수의 인간 평가자와 여러 개의 LLM (예: GPT-4, Claude 3)이 동시에 패널(Panel)로 심사할 때 사용하는 통합 신뢰 지수.
    엔터프라이즈 오라클 담당자는 매주 파이프라인 성능 리포트에 이 카파 지수를 모니터링하여, 심판 모델 버전 업데이트가 오라클 성능을 퇴보(Regression)시키지 않았는지 증명해야 한다.
