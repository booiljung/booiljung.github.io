7.1.4 Reference-based 평가와 Reference-free 평가의 차이점 및 적용 시나리오

LLM 심판이 채점을 진행할 때, 인간이 만들어둔 ’모범 답안(Reference)’의 존재 여부에 따라 아키텍처가 갈라진다.

  • Reference-based (참조 기반 평가)

  • 인간이 작성한 완벽한 요약문이나 번역문이 존재할 때 쓴다.

  • 프롬프트: “다음은 모범 답안이다. 평가 대상 문장이 모범 답안의 사실과 얼마나 일치하는지 1~5점으로 평가하라.”

  • 적용처: 번역, 기계 리딩 독해(MRC), 챗봇 답변 품질 평가. 확고하고 변동 없는 잣대가 마련되어 오탐률이 극히 적다.

  • Reference-free (참조 없는 평가)

  • 정답이 정해지지 않은 창의적 작문이 대상일 때 쓴다.

  • 프롬프트: “다음 제출된 에세이가 문법적으로 올바르며, 논조가 전문적인지 자체적인 기준에 의해 평가하라.”

  • 적용처: 코드 생성 검토, 이메일 초안 작성. 심판 모델의 백그라운드 지식(World Knowledge)에 기대는 빙식이므로, 반드시 파라미터가 거대한 모델(GPT-4 등)을 기용해야 한다.
