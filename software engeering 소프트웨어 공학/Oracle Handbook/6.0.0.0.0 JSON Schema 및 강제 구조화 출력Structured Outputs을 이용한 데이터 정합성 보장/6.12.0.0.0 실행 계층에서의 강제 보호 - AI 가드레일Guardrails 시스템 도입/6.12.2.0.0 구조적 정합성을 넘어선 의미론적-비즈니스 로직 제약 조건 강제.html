6.12.2 구조적 정합성을 넘어선 의미론적-비즈니스 로직 제약 조건 강제

오라클의 난이도가 가장 높은 영역은 타입은 맞지만 말이 안 되는 의미론적(Semantic) 모순이다.

  • 상황: 고객이 “이 상품 환불해주세요“라고 했는데 AI가 응답 JSON의 intent 필드는 "REFUND"(구조적 정답)로 내뱉었으면서도, 정작 생성한 텍스트 메시지에는 “환불이 불가능합니다”(논리적 모순)라고 적어 보내는 환각.
  • 가드레일 제약: 이 때 가드레일 엔진에 의미 상충 방지(Contradiction Check) 파이프라인(예: NLI 모델 기반 Entailment 검사)을 배치한다.
    가드레일은 이 두 필드 간의 내용적 이율배반을 감지하여 런타임에서 발송을 원천 차단(Blocking)함으로써 치명적인 CS 대응 사고를 막는다.
