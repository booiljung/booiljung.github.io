6.3.3 Fine-tuning을 이용한 특정 포맷 출력 학습

매번 긴 프롬프트를 태우는 것이 비효율적이라면, 아예 모델의 가중치(Weights) 자체를 JSON을 뱉어내도록 미세조정(Fine-Tuning)할 수 있다. 오로지 입력 텍스트와 완벽히 구조화된 JSON 응답 쌍만으로 구성된 수만 건의 데이터셋을 학습시킨다.

  • 장점: 프롬프트를 쓸 필요도 없이 자연스럽게 구조화 데이터를 내뱉는다. 속도가 빠르며 토큰 비용이 크게 절감된다.
  • 단점: 스키마(Schema) 구조가 변경될 때마다 모델을 처음부터 다시 파인튜닝해야 한다. 유지보수(Maintenance) 비용이 천문학적으로 높아지며 실시간 비즈니스 요구사항 변동에 대처할 수 없다.
