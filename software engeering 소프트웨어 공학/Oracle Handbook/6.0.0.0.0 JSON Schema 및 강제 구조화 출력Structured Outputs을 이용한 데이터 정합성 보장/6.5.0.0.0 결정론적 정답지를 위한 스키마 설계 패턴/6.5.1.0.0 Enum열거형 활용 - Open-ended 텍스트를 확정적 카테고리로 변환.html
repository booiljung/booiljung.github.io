6.5.1 Enum(열거형) 활용: Open-ended 텍스트를 확정적 카테고리로 변환

범주화(Categorization) 오라클을 구축할 때 가장 중요한 것은 응답의 경우의 수를 고정시키는 것이다.
영화 리뷰를 분석하라고 시키면 LLM은 “매우 긍정적”, “조금 부정이 섞임”, “그저 그런 편” 등 수백 가지 어휘를 창조해 낸다.

이를 제압하기 위해 Enum을 사용하라.

from enum import Enum
class Sentiment(str, Enum):
    POSITIVE = "POSITIVE"
    NEGATIVE = "NEGATIVE"
    NEUTRAL = "NEUTRAL"

이렇게 정의한 필드는 JSON Schema의 enum: ["POSITIVE", "NEGATIVE", "NEUTRAL"] 속성으로 매핑된다. AI가 “훌륭하지만 아쉬운” 같은 답변을 하려 들지라도 시스템은 이 3가지 텍스트 토큰 중 하나로 선택을 강제 라우팅(Routing)시킨다.
