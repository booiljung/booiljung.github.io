6.5.3 자기 비평(Self-Correction) 필드 추가를 통한 데이터 정합성 검증

LLM의 단방향 생성 특성(Auto-regressive)을 역으로 이용하는 방법이다. 모델이 생성 중인 데이터의 정합성을 스스로 의심(Doubt)하도록 스키마를 유도한다.

{
  "extracted_email": "user@example.com",
  "is_valid_email_syntax": true,
  "confidence_explanation": "문서 상단 연락처 항목에서 정확히 일치하는 패턴을 찾았습니다."
}

추출된 extracted_email이 진짜 이메일 형식이 맞는지 불리언(Boolean) 값인 is_valid_email_syntax에 응답하게 강제한다. 이렇게 하면 모델은 앞서 자기가 뱉은 단어를 되돌아보는(Reflection) 효과적인 Attention Computation을 재수행하게 되며, 이 불리언 값이 false이거나 검증 설명이 불충분할 때 오라클 필터가 이를 즉결 처분(Reject)할 수 있다.
