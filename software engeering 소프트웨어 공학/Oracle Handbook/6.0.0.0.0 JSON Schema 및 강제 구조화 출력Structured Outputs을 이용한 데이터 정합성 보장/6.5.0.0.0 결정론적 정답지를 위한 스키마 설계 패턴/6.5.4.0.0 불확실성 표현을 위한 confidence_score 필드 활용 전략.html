6.5.4 불확실성 표현을 위한 confidence_score 필드 활용 전략

환각(Hallucination) 방지의 핵심은 모델에게 “모른다고 말할 권리“와 “얼마나 확신하는지 계량할 수 있는 수단“을 쥐여주는 것이다.

  • 필드: confidence_score (type: number, minimum: 0.0, maximum: 100.0)
  • 프롬프트 지시: “텍스트에서 금액이 명시적으로 보이면 90점 이상, 문맥으로 유추해야 한다면 50점, 완전히 흐릿하다면 0점을 주어라.”

이 스코어(Score)는 인간 개입 파이프라인(Human-in-the-loop, HITL)으로 넘기는 훌륭한 라우터가 된다. 오라클 시스템은 confidence_score < 80인 응답 체인은 무조건 실패(Fail)시킴으로써, 쓰레기 데이터가 운영 DB에 적재되는 대참사를 막는다.
