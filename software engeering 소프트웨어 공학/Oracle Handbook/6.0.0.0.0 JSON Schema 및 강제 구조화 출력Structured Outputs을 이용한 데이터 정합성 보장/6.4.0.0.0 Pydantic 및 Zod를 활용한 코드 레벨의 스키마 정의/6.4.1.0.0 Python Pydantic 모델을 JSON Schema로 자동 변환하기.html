6.4.1 Python Pydantic 모델을 JSON Schema로 자동 변환하기

Pydantic은 파이썬 생태계의 데이터 유효성 검증 표준이다. 클래스 형태로 스키마를 선언하면, model_json_schema() 메서드를 통해 완벽하게 호환되는 JSON Schema 규격을 동적으로 만들어낸다.

from pydantic import BaseModel, Field

class UserProfile(BaseModel):
    name: str = Field(description="사용자의 실명")
    age: int = Field(ge=0, le=150, description="0에서 150 사이의 나이")

# 이 한 줄이 OpenAI의 response_format으로 직접 들어간다.
schema_dict = UserProfile.model_json_schema()

개발자는 더 이상 깊은 딕셔너리와 씨름할 필요 없이, 파이썬 객체의 프로퍼티(Property)로 오라클의 기준선을 확립한다.
