6.2.2 데이터 타입(Types) 정의와 엄격함의 정도

LLM을 길들일 때 타입(type)은 가장 기초적인 오라클이다.

{
  "age": {
    "type": "integer",
    "minimum": 0,
    "maximum": 150
  }
}

여기서 LLM은 "30"(String)을 반환할 수 없다. 오직 숫자로만 구성된 30 포맷을 내야 한다. 최근의 강제 디코딩(Structured Output) 엔진들은 이 타입 선언에 따라 아예 토큰 생성기의 허가된 샘플러(Sampler) 목록을 제한해버린다. 타입의 엄격함이 올라갈수록 오라클의 디버깅 시간은 단축된다.
