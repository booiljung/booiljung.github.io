6.6.2 Outlines 라이브러리를 활용한 정규표현식 및 JSON 스키마 가이드 생성

HuggingFace나 vLLM 위에서 구조화 출력이 필요하다면 Outlines 라이브러리가 정답이다.
파이썬 언어로 된 이 라이브러리는 백그라운드 엔진의 로짓(Logits) 마스킹 계층(Layer)을 직접 건드린다.

import outlines

model = outlines.models.transformers("meta-llama/Llama-2-7b-chat-hf")
generator = outlines.generate.json(model, UserProfile) # Pydantic 모델 투입
result = generator("User John is 25 years old.")

Outlines는 투입된 Pydantic 모델을 해석하여 로컬 모델이 추론하는 매 틱(Tick)마다 “다음 글자로 { 가 올 확률 100%, 나머지 토큰은 0%“로 FSM(유한 상태 기계)을 즉각 렌더링한다. 수만 건의 로컬 회귀 테스트가 필요한 오라클 검증 시, JSONDecodeError를 0으로 수렴하게 만드는 원천 기술이다.
