6.8.5 JSON Diff를 이용한 모델 버전 간의 미세한 응답 변화 감지

결정론적 구조의 가장 아름다운 점은 기계적인 비교(Diff)가 가능하다는 것이다.
DeepDiff 같은 라이브러리를 통해 프로덕션 모델(GPT-4o)의 완벽한 골든 샘플(Golden JSON)과 저비용 모델(GPT-4o-mini)의 응답을 트리 순회하며 비교한다.

from deepdiff import DeepDiff
# 차이점 탐지
diff = DeepDiff(golden_record, gpt_mini_response, ignore_order=True)
if 'type_changes' in diff or 'missing_dict_keys' in diff:
    print("구조적 결함 발생, 오라클 테스트 실패")

문장의 어투가 바뀌는 것은 무시하면서도, 배열 내 아이템 개수가 빠졌거나 타입이 변경된 핵심적인 결함만을 정확히 타격하여 찾아낼 수 있다.
