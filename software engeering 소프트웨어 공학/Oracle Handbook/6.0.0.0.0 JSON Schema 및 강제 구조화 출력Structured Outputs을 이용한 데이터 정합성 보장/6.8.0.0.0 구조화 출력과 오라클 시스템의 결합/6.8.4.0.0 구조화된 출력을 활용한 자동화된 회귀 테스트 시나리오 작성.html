6.8.4 구조화된 출력을 활용한 자동화된 회귀 테스트 시나리오 작성

프롬프트를 V1에서 V2로 수정하거나 기본 모델을 교체했을 때 구버전과 동일한 퀄리티를 내는지 검증하는 회귀 테스트(Regression Test)는 필수다.
비정형 텍스트 출력이라면 두 버전을 비교하기 위해 또 다른 LLM을 동원하는 LLM-as-a-Judge 방식(비용과 시간이 많이 듦)을 써야 한다.

하지만 출력물이 정형화된 JSON 배열이라면 테스트가 극도로 단순해진다.

  1. V1 출력의 items 배열 길이가 V2의 items 배열 길이와 동일한지(추출률 비교).
  2. 필수 파라미터 내역이 동일한지.
    구조화 출력은 테스트의 채점 비용을 0에 가깝게 낮추어 자동화 파이프라인의 극단적 스케일링을 가능케 한다.
