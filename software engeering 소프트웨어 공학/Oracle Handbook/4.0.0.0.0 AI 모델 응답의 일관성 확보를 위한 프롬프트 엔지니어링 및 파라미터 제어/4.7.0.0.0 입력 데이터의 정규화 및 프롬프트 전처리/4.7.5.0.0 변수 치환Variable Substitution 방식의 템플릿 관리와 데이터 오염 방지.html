4.7.5 변수 치환(Variable Substitution) 방식의 템플릿 관리와 데이터 오염 방지

결정론적 소프트웨어 환경에서 프롬프트를 함수(Function)처럼 사용하려면, 파이썬이나 자바스크립트의 f-string 기반 문자열 보간법(String Interpolation)을 엄격하게 관리해야 한다.

  • 프롬프트 템플릿을 코드 내부에 하드코딩하지 말고, 텍스트 파일(예: oracle_prompt.j2 즉 Jinja2 포맷 등)로 분리 관리하라.
  • {{ user_input }} 같은 플레이스홀더(Placeholder) 위치에는 정규화를 마친, Escape 처리된 텍스트만 최종적으로 주입해야 한다. 특히 JSON 구조를 다룰 때 따옴표(") 처리를 잘못하면 파서 전체가 무너지므로 주의하라.

이러한 기계적인 분리 구조는 추후 4.12절에서 설명할 DSPy 프레임워크와 결합할 때, 시스템이 스스로 최적의 템플릿을 찾아 자동 컴파일(Compile) 하게 만드는 기초 토대가 된다.
