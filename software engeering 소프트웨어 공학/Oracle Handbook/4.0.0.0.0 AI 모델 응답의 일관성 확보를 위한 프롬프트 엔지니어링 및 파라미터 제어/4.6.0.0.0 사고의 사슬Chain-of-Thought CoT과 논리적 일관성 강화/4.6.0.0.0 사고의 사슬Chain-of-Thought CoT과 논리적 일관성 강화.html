4.6 사고의 사슬(Chain-of-Thought, CoT)과 논리적 일관성 강화

LLM은 본질적으로 직관주의자(Intuitionist)다. 아주 복잡한 수학 문제를 주더라도 중간 계산 과정 없이 정답을 한 번에 찍어서 맞히려고 덤벼드는 경향이 있다. 이 ’직관적 도약’은 결정론적 환경 구축의 가장 큰 원수다. 도약하는 거리만큼 할루시네이션(Hallucination)이 침투할 공간이 생기기 때문이다.

이 절에서는 “어떻게 그 결과에 도달했는지“를 먼저 설명하도록 강제함으로써, 모델이 스스로의 출력 토큰을 징검다리 삼아 논리적 비약을 방지하고 결과의 일관성을 비약적으로 높이는 기법인 Chain-of-Thought (CoT) 구조를 오라클 설계에 적용하는 방법을 논한다.
