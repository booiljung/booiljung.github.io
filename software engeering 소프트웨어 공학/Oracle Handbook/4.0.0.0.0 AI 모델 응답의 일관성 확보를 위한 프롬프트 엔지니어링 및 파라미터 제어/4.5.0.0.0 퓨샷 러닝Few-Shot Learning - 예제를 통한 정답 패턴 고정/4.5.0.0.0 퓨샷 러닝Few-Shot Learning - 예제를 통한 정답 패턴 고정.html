4.5 퓨샷 러닝(Few-Shot Learning): 예제를 통한 정답 패턴 고정

아무리 정교한 지시문(System Prompt)이라도, 그 지시를 행동으로 번역하는 해석의 과정에서 오차가 발생할 수 있다. “항상 짧게 대답해“라는 추상적 지시보다 더 강력하고 물리적인 제어 방식은 “이러한 입력이 들어왔을 때는 이렇게 대답하라“는 예제 세트(Input-Output Pairs)를 직접 주입하는 것이다.

이 절에서는 In-Context Learning의 핵심 무기인 퓨샷(Few-shot) 예제가 어떻게 모델 내부의 활성화 가중치(Weights)를 조작하여 파서(Parser) 친화적인 기계적 형식을 강제로 유지하게 만드는지, 그 원리와 구성 전략을 파헤친다.
