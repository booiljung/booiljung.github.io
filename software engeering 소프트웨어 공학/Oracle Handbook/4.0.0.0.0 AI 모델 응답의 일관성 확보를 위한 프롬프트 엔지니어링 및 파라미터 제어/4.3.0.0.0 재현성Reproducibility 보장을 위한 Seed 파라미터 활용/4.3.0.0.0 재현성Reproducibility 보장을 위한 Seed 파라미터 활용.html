4.3 재현성(Reproducibility) 보장을 위한 Seed 파라미터 활용

소프트웨어 테스팅, 특히 회귀 테스트(Regression Testing)를 수행할 때 동일한 입력에 대해 어제와 오늘의 출력이 다르다면 테스트 파이프라인 전체가 무의미해진다. AI 엔지니어들은 이 근본적인 가변성을 통제하기 위해 컴퓨터 과학의 오래된 기술인 난수 시드(Random Seed)를 LLM(Large Language Model) 호출 계층에 통합하기 시작했다.

이 절에서는 LLM이 생성하는 무작위성의 시작점인 시드(Seed)를 통제하여, Temperature > 0 인 상황에서도 기계적인 재현성을 확보하려는 API 수준의 노력과 그 한계를 다룬다.
