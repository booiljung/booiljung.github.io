4.4.2 오라클 역할을 수행하기 위한 엄격한 페르소나(Persona) 정의 기법

우리는 AI가 친절하게 인사하거나 부가 설명을 늘어놓길 원치 않는다. AI가 텍스트를 읽고 기계적으로 “PASS” 혹은 “FAIL“을 판독하는 센서(Sensor)처럼 동작하게 하려면, 그에 맞는 자아(Persona)를 부여해야 한다.

이 절에서는 오라클의 평가 신뢰도를 획기적으로 높일 수 있는 ‘전문 검사관(Expert Auditor)’ 페르소나와 감정을 배제하는 지시문 작성 기법을 세분화하여 설계한다.
