9.4.1 프로젝트 전용 코딩 컨벤션(Style Guide) 학습과 강제화

AI가 만들어낸 코드가 구글 스타일 가이드(Google Style Guide)를 따르더라도 우리 팀의 가이드(.eslintrc, .pylintrc)에 어긋나면 그것은 오답이다.

  • 컨텍스트 주입: 코드 생성을 요청하는 프롬프트 최상단에 사내 컨벤션 요약본이나 린터 설정 파일의 내용을 시스템 프롬프트(System Prompt)로 강제 주입한다.
  • 오라클의 철퇴: 생성된 코드가 오라클 도마에 오르면, 컨테이너는 오직 프로덕션(Main) 브랜치에 있는 .eslintrc 매니페스트만을 기준으로 린팅을 수행한다. 탭(Tab)을 썼는데 규칙이 스페이스(Space)라면, AI는 “나는 완벽한 로직을 짰다“고 항변하더라도 0점 처리된다.
