9.8.3 인지 복잡도(Cognitive Complexity)를 통한 이해하기 어려운 코드 필터링

순환 복잡도의 맹점을 보완하는 것이 소나큐브(SonarQube)가 창시한 인지 복잡도다. 논리적 분기 수가 적어도, 비트 연산자나 삼항 연산자가 기괴하게 꺾여있으면 인간의 인지에 과부하를 준다.

  • 오라클은 SonarScanner 알고리즘을 런타임에 차용하여, AI가 연속적인 && 와 || 로 뒤갈린 조건 논리를 짰을 때 Cognitive Complexity 점수를 폭발시킨다.
  • 점수가 상한(예: 15)을 넘어가면 봇의 코드를 기각하고, “중간 변수(Intermediate Variable)를 사용하여 조건문의 의도를 명확하게 쪼개라“고 코칭한다.
