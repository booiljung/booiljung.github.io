9.6.3 사용 금지된 라이브러리(Blacklisted Libraries) 및 라이선스 위반 모듈 차단

AI는 라이선스라는 귀찮은 법적 제약을 전혀 고려하지 않고 코드를 생성한다. GPL v3 라이브러리를 무심코 기업용 상용 소스에 임포트하는 행위는 회사 전체 코드를 강제 공개시켜야 하는 최악의 런타임 외적 참사다.

  • 블랙리스트 체커: 오라클은 패키지를 확인한 즉시 의존성-라이선스-데이터베이스를 조회한다.
  • GPL, AGPL 같은 감염형(Copyleft) 라이선스가 식별되면, 기능 작동 여부와 무관하게 해당 생성 코드를 “Compliance 위반 에러“로 낙인찍고, MIT나 Apache 2.0 라이선스의 대안 라이브러리(Alternative)를 사용하도록 강제하는 프롬프트 피드백을 전달한다.
