9.6.4 더 이상 지원되지 않는(Deprecated) API 호출 감지

LLM의 지식 가중치는 과거 튜닝 시점에 고정되어 있다(예: 2021년). 따라서 이미 2년 전에 Deprecated 처리되고 어제 모듈에서 삭제된 API를 태연하게 호출할 수 있다.

  • 최신 정적 린터 시스템(예: ESLint의 eslint-plugin-deprecation 과 TypeScript AST)을 파이프라인 전면에 세운다.
  • 이 오라클은 함수 시그니처 위에 @deprecated 데코레이터나 주석이 달린 API를 AI 코드가 참조하고 있는 경우, 이를 즉각 에러로 격상(Treat warnings as errors)시켜 과거의 잔재물(Legacy API)이 신규 코드베이스에 침투하는 것을 막는다.
