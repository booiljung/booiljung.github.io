9.6.1 AI가 생성한 가짜 패키지명 및 버전(Package Hallucination) 탐지 기법

AI 코더가 가장 취약한 부분이 바로 패키지 버전 관리다. requests==99.9.9 처럼 아예 나올 수 없는 버전을 명시하거나 가짜 패키지를 끌어들일 때 오라클이 작동해야 한다.

  • 오라클은 파이썬 AST 파싱을 통해 Import와 ImportFrom 구문을 모두 긁어모은다.
  • 추출된 이름 배열(예: ['pandas', 'numpy', 'my_hallucinated_lib'])을 사내 로컬 환경에 캐싱된 화이트리스트(Whitelist) DB 테이블과 O(1) 해시 탐색으로 일일이 대조한다.
  • 단 하나라도 리스트에 없는 패키지명 식별 시, 즉시 Dependency Error를 던지고 빌드를 셧다운한다.
