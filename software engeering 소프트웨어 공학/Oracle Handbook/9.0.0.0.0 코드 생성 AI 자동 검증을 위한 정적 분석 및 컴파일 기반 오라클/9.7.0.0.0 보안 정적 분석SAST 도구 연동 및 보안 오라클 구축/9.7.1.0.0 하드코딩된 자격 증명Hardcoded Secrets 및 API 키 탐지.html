9.7.1 하드코딩된 자격 증명(Hardcoded Secrets) 및 API 키 탐지

AI는 종종 설명의 편의를 위해(혹은 환각에 의해) const AWS_KEY = 'AKIA...' 식으로 실제 비밀키의 포맷을 문자열 그대로 코드에 박아(Hard-coding) 버린다.

  • TruffleHog / GitLeaks 오라클: 생성된 코드가 임시 파일 메모리에 적재되는 순간, 정규표현식과 엔트로피 연산(Shannon Entropy)을 기반으로 한 자격 증명 스캐너가 백그라운드에서 돌아간다.
  • “높은 엔트로피를 가진 40자리 영문 숫자의 문자열 선언” 또는 특정 API 프로바이더 식별자(sk-로 시작하는 토큰)가 발견되면, 오라클 계기판에는 적색경보가 울리며 이 코드는 버전 관리 시스템(Git)에 절대 닿지 못하고 메모리에서 소각된다.
