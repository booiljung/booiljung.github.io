9.7.5 AI 생성 코드의 보안 점수 산정 및 배포 승인 기준(Gatekeeper) 설정

보안 취약점 방어선을 여러 겹으로 쌓았다면 이를 수치화해야 한다. 오라클 게이트키퍼(Oracle Gatekeeper)의 심판 기준표를 세운다.

  • Critical(심각 위반): 하드코딩 비밀번호, SQL 인젝션 패턴. 탐지 즉시 파이프라인 킬(Kill).
  • Minor(경미 위반): 구버전 라이브러리 경고 등. 합계가 일정 치(Threshold)를 넘지 않으면 통합되지만, AI에게 “경고 3회 누적, 리팩토링 필수” 태그를 발급한다.
  • 이 점수판은 오라클 대시보드에서 인간 리뷰어(Reviewer)에게 투명하게 공유되어, AI 개발 에이전트가 만든 소스가 인간의 신뢰를 유지하며 브랜치에 녹아들게 하는 강력한 가시성(Visibility)을 제공한다.
