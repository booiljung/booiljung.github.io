9.2.2 Python의 ast 모듈과 Java의 JavaParser를 활용한 구문 분석 실습

언어별 내장 모듈을 통해 무거운 컴파일 시스템 없이도 가볍게 오라클 모듈을 구현할 수 있다.

  • Python AST 오라클 예시:
import ast

def oracle_check_syntax(ai_generated_code: str) -> bool:
    try:
        # 이 구문 하나만으로 들여쓰기, 괄호 매칭 에러가 즉시 검출된다.
        tree = ast.parse(ai_generated_code)
        return True
    except SyntaxError as e:
        return False # Fail-fast 반환
  • JavaParser 활용: Java의 경우 AI가 내뱉은 코드가 클래스 껍데기를 가지고 있는지, 패키지 선언이 파일 경로와 일치하는지를 AST 기반으로 엄격하게 따진다. 오라클 시스템은 여기서 CompilationUnit 객체를 파싱하고 트리를 워크(Walk)하며 규칙 준수 여부를 채점한다.
