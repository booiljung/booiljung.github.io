9.1.3 정적 분석 및 컴파일 오라클의 정의와 범위 설정

정적 분석 오라클은 “소스 코드를 실행하지 않고, 그 코드의 텍스트와 구조만으로 참/거짓 또는 합격/불합격을 수학적으로 도출하는 결정론적 판별 시스템” 이다.

이 오라클의 검증 범위(Scope) 는 다음과 같다.

  1. 구문(Syntax) 오라클: 오타, 괄호 누락 방어.
  2. 타입(Type) 오라클: String을 인자로 받는 곳에 Integer를 집어넣는 에러 방어.
  3. 구조(Structural) 오라클: 구현하라고 지시당한 함수나 클래스(Interface)가 스펙에 맞게 정확히 명명되어 코딩되었는지 확인.
  4. 안전성(Security) 오라클: 평문으로 적힌 패스워드나 위험한 함수(eval)의 사용 감지.
