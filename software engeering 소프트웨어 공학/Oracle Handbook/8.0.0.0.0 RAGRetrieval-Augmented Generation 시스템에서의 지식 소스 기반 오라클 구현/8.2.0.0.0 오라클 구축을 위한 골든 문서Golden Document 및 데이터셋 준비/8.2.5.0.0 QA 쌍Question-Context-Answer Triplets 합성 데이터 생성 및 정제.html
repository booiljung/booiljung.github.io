8.2.5 QA 쌍(Question-Context-Answer Triplets) 합성 데이터 생성 및 정제

인간이 직접 오라클 테스트용 질문 10,000개를 만들 수는 없다. 가장 우수한 오라클 파이프라인은 정답지를 LLM에 역산시켜 얻어낸다.

  1. 정제된 Context Chunk를 강력한 모델(GPT-4o)에게 주입한다.
  2. 프롬프트: “이 텍스트 패시지만을 읽고 답할 수 있는 고난도 질문(Question) 3개와, 이에 대한 완벽한 정답(Answer)을 추출하여 JSON으로 반환하라.”
  3. 생성된 결과를 다시 또 다른 LLM(가령 Claude 3.5)에게 먹여 “이 응답 세트가 정말로 저 문맥에 100% 매핑되는지” 교차 확인(Cross-checking) 시킨다.
    이렇게 자동 합성된 (Question, Context, Answer) 조합이 바로 오라클 평가를 위한 골든 트라이앵글(Golden Triplet) 이다.
