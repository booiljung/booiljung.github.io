8.2.6 오라클 검증용 ‘정답 없음(Unanswerable)’ 데이터셋의 중요성

평가 데이터셋을 대답 가능한 질문으로만 구성하는 것은 치명적 편향이다. RAG 시스템 오라클의 최고난도 임무는 모델의 용감함을 꺾고 “제가 모르는 내용입니다” 를 출력하도록 징벌하는 것이다.

  • 의도적 혼란 투입: Context에 사과에 대한 이야기만 적어 두고, Question으로 바나나의 효능을 묻는(Unanswerable) 테스트 셋을 전체의 20% 분량으로 배정하라.
  • 이 테스트에서 모델이 환각을 일으켜 대답을 지어낸다면, 오라클은 Faithfulness에서 0점을 퍼붓고 테스트 빌드를 즉각 세워야 한다. 진정한 지능은 모르는 것을 침묵하는 데에서 기인한다.
