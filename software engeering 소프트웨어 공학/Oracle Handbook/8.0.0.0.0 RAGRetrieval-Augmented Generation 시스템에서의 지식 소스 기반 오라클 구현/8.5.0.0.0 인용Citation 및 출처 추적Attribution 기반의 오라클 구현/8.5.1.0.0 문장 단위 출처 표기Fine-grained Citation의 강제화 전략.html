8.5.1 문장 단위 출처 표기(Fine-grained Citation)의 강제화 전략

단락 끝에 [참조: 문서 A, B, C] 라고 뭉뚱그려 적는(Coarse-grained) 방식은 오라클이 검증하기 가장 싫어하는 최악의 패턴이다.

오라클 친화적 아키텍처에서는 프롬프트를 통해 문장 단위 인용(Fine-grained Citation) 을 강제해야 한다.

  • “모든 마침표 앞에는 반드시 단 1개의 참조 문서 ID를 표기하라. 예: 이순신은 장군이다 [Doc_01]. 그리고 거북선을 만들었다 [Doc_03].”
  • 이렇게 제약하면, 오라클 시스템은 텍스트를 파싱(Parsing) 하자마자 n번째 문장과 n번째 인용 청크를 1:1로 매핑하여 고속 병렬 채점(Parallel Matrix Evaluation)을 수행할 수 있게 된다.
