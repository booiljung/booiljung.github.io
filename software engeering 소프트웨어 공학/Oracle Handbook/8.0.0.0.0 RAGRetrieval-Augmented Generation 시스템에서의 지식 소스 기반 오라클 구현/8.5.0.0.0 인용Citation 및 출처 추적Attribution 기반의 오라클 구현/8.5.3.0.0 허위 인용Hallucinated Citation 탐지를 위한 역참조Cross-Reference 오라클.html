8.5.3 허위 인용(Hallucinated Citation) 탐지를 위한 역참조(Cross-Reference) 오라클

가장 교활한 RAG 환각은 존재하지도 않는 문서를 인용하는 행위다. 봇이 [대법원 판례 2024-다-001]이라고 당당하게 답변했지만, 검색기 로그를 까보면 그런 청크는 전달된 적도 없는 경우다.

  • 역참조 룩업 테이블(Look-up Table): 오라클은 생성 봇이 반환한 텍스트를 파싱하여, 대괄호 [] 안에 들어있는 모든 인덱스를 세트(Set) 자료형으로 추출한다.
  • 이 세트가 검색 파이프라인(Retriever)이 건네준 Context UUID Set에 속하지 않는 Index(Out-of-Bound)를 포함하고 있다면 역참조 위반 에러를 발생시킨다. 이는 LLM 호출 없이 파이썬 코드 단 3줄로 구현할 수 있는 가장 가성비 높은 오라클 모듈이다.
