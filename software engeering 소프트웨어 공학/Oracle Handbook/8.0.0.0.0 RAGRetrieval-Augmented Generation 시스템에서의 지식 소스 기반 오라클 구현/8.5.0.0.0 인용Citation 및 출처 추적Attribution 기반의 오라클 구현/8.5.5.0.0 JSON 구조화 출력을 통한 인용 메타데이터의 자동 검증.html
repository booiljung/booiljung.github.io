8.5.5 JSON 구조화 출력을 통한 인용 메타데이터의 자동 검증

텍스트 중간에 대괄호를 파싱하는 것은 이스케이프(Escape) 문자나 오타에 취약하다. 실전 엔터프라이즈 RAG 오라클은 답변 포맷 자체를 JSON의 배열(Array) 구조로 완전히 결빙(Freezing)시킨다.

{
  "sentences": [
    {"text": "서울은 한국의 수도이다.", "citations": ["doc_city_01"]},
    {"text": "인구는 약 천만 명이다.", "citations": ["doc_pop_99"]}
  ]
}

이런 형태로 출력을 강제하면 오라클 시스템은 자연어 처리기조차 필요 없이, Pydantic 모델의 validator나 schema check만으로도 인용구의 결측치, 타입 에러, 범위를 밀리초 단위로 완벽하게 쳐낼 수 있다. (6장 구조화 출력의 실전 응용이다.)
