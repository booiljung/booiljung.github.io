8.9.2 의료(Medical) 도메인에서의 가이드라인 준수 및 금기 사항 필터링

의료 Q&A 봇의 오라클은 답변의 창의성을 철저하게 거세(Castration)하는 데 목적이 있다. 의학 가이드라인은 생명과 직결되므로 ’추론’이 개입할 여지가 없다.

  • 금기(Contraindication) 매트릭스 필터: 오라클 앞단에, 절대 처방하거나 권고해서는 안 되는 금기 약물 리스트와 증상 매트릭스(Hard-coded Rules)를 배치한다.
  • 봇이 “타이레놀을 드셔보세요“라고 생성하더라도, 오라클이 환자의 이전 쿼리 맥락에서 ’간 질환’이라는 키워드를 탐지했다면, LLM 채점기까지 갈 것도 없이 Rule-based 차단기가 작동하여 해당 세션을 즉시 “의료 전문가와 상담하십시오“로 덮어씌운다.
