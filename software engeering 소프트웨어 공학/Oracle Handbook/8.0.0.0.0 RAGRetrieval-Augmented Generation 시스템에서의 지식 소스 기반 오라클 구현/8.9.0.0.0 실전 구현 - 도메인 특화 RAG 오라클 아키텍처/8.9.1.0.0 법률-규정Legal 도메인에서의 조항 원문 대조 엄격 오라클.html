8.9.1 법률/규정(Legal) 도메인에서의 조항 원문 대조 엄격 오라클

법률 문서에서는 단어 하나의 뉘앙스가 승패를 가른다. “할 수 있다(May)“를 “해야 한다(Must)“로 요약하는 순간 RAG 시스템은 치명적 위증을 한 셈이다.

  • 조항(Article) 고정 오라클: 생성된 텍스트가 법률적 결론을 내릴 때, 오라클은 NLI나 의미론적 관용(Semantic Tolerance)을 0으로 둔다.
  • 오라클은 답변 내에 등장하는 핵심 법률술어를 Context(조항 원문)과 형태소 단위로 Exact Match가 이루어지는지 강제 대조한다. 만약 LLM이 유저를 돕는답시고 판례의 어려운 말을 ’쉬운 일상어’로 의역(Paraphrasing)했다면, 이 친절함마저 무자비하게 징벌하여 원래의 보수적 워딩으로 롤백시켜야 한다.
