8.6.1 RAGAS(Retrieval Augmented Generation Assessment) 프레임워크의 오라클 적용

업계 표준으로 자리 잡은 RAGAS 프레임워크는 RAG 평가를 직관적인 점수 지표로 계량하는 철학을 제공한다.

  • 핵심 원리: 막연히 “이 RAG 시스템이 좋은가?“를 묻지 않고 환각의 원인을 Generation(생성)과 Retrieval(검색)의 두 축으로 분할격리한다.
  • 실시간 파이프라인 통합: RAG 생성물이 유저에게 도달하기 직전 비동기(Async) 워커가 이 프레임워크의 메트릭(문맥 정밀도, 문맥 재현율, 충실성, 답변 관련성)을 병렬로 계산한다. 이 스코어 카드의 평균값이 지정된 임계값(예: 0.8)을 하회하면 생성된 응답을 숨기고(Fallback) 기본 응답을 던지는 파이프라인이 RAG 오라클의 골자다.
