8.6.2 문맥 관련성(Context Relevance) 평가 프롬프트 설계

검색기가 엄청나게 많은 문서를 가져왔을 때, 과연 그것들이 사용자 쿼리와 직접 매핑되는지 검증(Audit)하는 심판 프롬프트다.

  • 지시문 패턴: “다음 질문(Question)과 검색된 문맥(Context)을 대조하라. 문맥 내에서 질문에 대답하는 데 필요한 결정적 문장들의 개수를 카운트하라. 총 문장 수 대비 유효 문장 수의 비율을 1~5점 척도로 계산하여 숫자로 반환하라.”
  • ఈ 프롬프트는 노이즈가 과다한 5점짜리 긴 문서가 오히려 RAG 모델의 어텐션(Attention)을 망치는 것을 방지하기 위해, “지나치게 쓸데없는 내용이 많으면 감점하라“는 패널티 로직을 반드시 포함해야 한다.
