8.6.4 충실성(Faithfulness) 점수 산출 및 자동 반려(Rejection) 시스템 구축

RAG 파이프라인의 최고 뇌관인 환각(Fact-making)을 적발하는 자동 사형 집행기다.

  1. Assertion Extraction: 답변을 명제(Proposition) 단위의 배열로 나눈다. (예: ["A는 10원이다", "B는 파란색이다"])
  2. Fact Check Loop: 심판관 LLM에게 각 명제와 문맥(Context)을 함께 주고, {"명제": "A는 10원이다", "Verdict": 1} 형식의 바이너리 판별을 돌린다.
  3. Rejection Threshold: 만약 Verdict: 0 (문맥에 근거 없음) 단 1개라도 발생 시, 전체 통과율(Faithfulness Score)이 1.0 미만으로 떨어진다. 엔터프라이즈 환경에서는 0.99도 용납하지 않고 해당 생성을 전면 폐기하고 재시도(Retry)한다.
