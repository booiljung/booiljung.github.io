8.8.4 인간 피드백(RLHF) 데이터를 활용한 오라클 임계값 미세 조정

오라클의 채점 기준이 너무 깐깐하면 쓸만한 응답도 모조리 거부되어 사용성(Usability)이 박살나고, 기준이 헐렁하면 환각이 새어나간다.

  • 이 로그와, 당시 RAG 오라클이 배정했던 0.0~1.0 사이의 채점 점수를 상관 분석(Correlation)한다. 만약 유저들이 좋아요를 누른 답변들의 오라클 충실성 점수가 0.7 부근에 집중되어 있다면, 과도하게 경직되어 있던 시스템 임계값(Threshold)을 기존 0.9에서 0.75로 낮추는 근거가 된다. 이 피드백 루프야말로 살아 숨 쉬는 MLOps의 정점이다.
