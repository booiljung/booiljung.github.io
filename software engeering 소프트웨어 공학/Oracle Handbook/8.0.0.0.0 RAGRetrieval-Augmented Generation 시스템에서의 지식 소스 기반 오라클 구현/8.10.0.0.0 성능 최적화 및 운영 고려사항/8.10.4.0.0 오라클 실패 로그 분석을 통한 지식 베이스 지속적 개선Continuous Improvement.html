8.10.4 오라클 실패 로그 분석을 통한 지식 베이스 지속적 개선(Continuous Improvement)

오라클이 빨간 불점을 켰다는 것(Fail)은 단순히 봇이 실수했다는 것만을 의미하지 않는다. 많은 경우, 지식 베이스(데이터 원단) 자체가 썩어 있다는 거대한 경고장이다.

  • 오라클이 패배시킨 [Context Relevance = 0] 로그들을 한 곳에 모아(ELK 스택 등) 클러스터링 해라.
