8.1.5 결정론적 RAG 오라클 구축을 위한 3대 요소: 관련성(Relevance), 충실성(Faithfulness), 정확성(Accuracy)

RAG의 성능을 계량하기 위해서는 오라클 시스템을 다음 세 가지 스코어링 보드로 삼분(Trisection)해야 한다. (이는 RAGAS나 TruLens 같은 프레임워크의 본질적 철학이다.)

  1. Context Relevance (문맥 관련성): 검색된 텍스트 청크가 질문(Query)에 직결되어 있는 핀포인트 정보인가, 아니면 노이즈가 가득한 쓰레기 데이터인가?
  2. Faithfulness (충실성): 생성된 텍스트가 검색된 Context에서 100% 논리적으로 연역(Entailed)될 수 있는가?
  3. Answer Relevance (답변 관련성): 생성된 텍스트가 엉뚱한 동문서답이 아니고, 처음에 던져진 사용자 쿼리(Query)를 직접 해소(Resolved) 했는가?
    이 세 개의 톱니바퀴 점수가 모두 임계점(Threshold)을 넘었을 때만, 오라클은 비로소 초록불(Pass)을 점등한다.
