8.3.1 검색 품질 평가 지표의 오라클 활용: Recall@K, MRR(Mean Reciprocal Rank)

오라클 시스템이 평가를 수행할 때 기준선(Baseline) 성능을 잡는 고전적인 정보 검색(IR) 정량 지표다.

  • Recall@K (리콜): 검색기가 반환한 상위 K개의 문서 중에, 정답을 포함한 문서를 ‘1개라도’ 건져냈는가? (생성 모델은 하나만 맞아도 말을 지어낼 수 있기 때문에, Precision보다 Recall이 RAG 환경에서는 압도적으로 구원자 역할을 한다).
  • MRR (Mean Reciprocal Rank): 첫 번째로 찾은 정답 문서가 몇 번째 랭크(순위)에 위치해 있는가를 역수로 변환한 지표다. 1번째에 나오면 1.0, 3번째에 나오면 0.33 점이다. 오라클 대시보드에서 MRR < 0.2로 나오면 사용자가 대답을 얻기 위해 과도한 토큰과 대기 시간이 낭비되고 있다는 뜻이므로, 즉시 검색 알고리즘 튜닝을 해야 한다.
