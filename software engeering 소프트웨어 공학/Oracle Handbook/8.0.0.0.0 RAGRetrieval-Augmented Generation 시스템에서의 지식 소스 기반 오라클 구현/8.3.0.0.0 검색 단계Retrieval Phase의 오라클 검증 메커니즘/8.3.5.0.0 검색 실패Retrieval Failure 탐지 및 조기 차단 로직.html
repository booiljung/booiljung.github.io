8.3.5 검색 실패(Retrieval Failure) 탐지 및 조기 차단 로직

가장 바보 같은 시스템은, 검색기에서 “일치하는 문서 없음” 혹은 불필요한 문서를 던져줬는데도 모델에게 억지로 대답하라고 윽박지르는 것이다.

  • 조기 차단(Early Exit): 검색 과정에서 Top-K 문서들의 Reranker 평균 점수가 임계값을 미달하면, LLM 엔진 호출(Generation API) 프로세스를 원천 통제한다.
  • Fail-safe 응답 렌더링: 곧바로 사용자에게 “해당 내용을 사내 지식 베이스에서 찾을 수 없습니다.“라는 표준 상수 문자열로 대체하여 통제된 상태를 선언한다. 이 기능은 가장 강력하면서도 저렴한 1차 오라클이다.
