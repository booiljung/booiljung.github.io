8.3.2 쿼리 의도와 검색 결과의 의미론적 일치도(Semantic Similarity) 측정

일반적인 텍스트 유사도는 키워드가 몇 개 겹쳤는지(BM25)나, 임베딩 공간의 점이 얼마나 가까운지(Cosine Similarity)만을 계량한다. 이는 똑똑한 오라클이 아니다.

오라클 엔진은 사용자의 질문이 “어떻게 해야 하는가?(How)“인지, “누구인가?(Who)“인지, “왜 문제인가?(Why)“인지를 파악하고, 검색 트래픽이 던져주는 문서가 그 의도(Intent)에 구문론적으로 들어맞는지 측정해야 한다. 이를 위해서는 NLI(Natural Language Inference) 기반의 크로스 인코더(Cross-encoder) 모델을 평가용 게이트웨이로 사용하여 쌍별 일치(Pairwise Matching)를 수행하라.
