8.3.4 리랭커(Reranker) 점수 기반의 임계값(Threshold) 설정 및 필터링 오라클

거친 망으로 건져올린 수십 개의 문서를 최종적으로 정밀하게 줄 세우는 도구가 리랭커(Reranker, 예: Cohere Rerank) 모델이다. 리랭커는 0.0~1.0 사이의 일치도를 도출한다.

  • 필터링 임계값 로직: 리랭커 점수가 0.3 이하인 문서는 그 어떤 일이 있어도 생성형 LLM 시스템 내부 시스템 프롬프트(System Prompt)로 넘어가면 안 된다.
  • 오라클은 이 0.3 이라는 워터마크를 동적으로 조율한다. 보안이 생명인 금융 챗봇에서는 임계값을 코어 비즈니스 룰에 의해 0.7 이상으로 강제하여, 쓸모없는 토큰 투입을 막고 오판을 철저하게 분리 격리해야 한다.
