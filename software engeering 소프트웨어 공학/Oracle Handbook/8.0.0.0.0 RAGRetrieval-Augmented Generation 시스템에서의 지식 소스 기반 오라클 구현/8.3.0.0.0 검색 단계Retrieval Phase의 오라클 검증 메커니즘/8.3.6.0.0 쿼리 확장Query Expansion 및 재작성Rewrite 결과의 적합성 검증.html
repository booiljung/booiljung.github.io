8.3.6 쿼리 확장(Query Expansion) 및 재작성(Rewrite) 결과의 적합성 검증

사용자의 모호한 단어 “그것 환불 어떻게 해?“를 시스템이 “갤럭시 S24 모델 환불 규정“으로 알아듣고 자가 번역하는 기술을 쿼리 재작성(Query Rewrite)이라 부른다.

  • 여기서 발생하는 오작동: AI가 쿼리를 지나치게 확장하여 “아이폰 환불 규정까지 전부 다 검색해라“고 오기를 부릴 수 있다.
  • Rewriter Oracle: 이를 제어하기 위해 쿼리를 수정한 모델의 결과값을, 역으로 초소형 분류 모델이나 NLI 모델에 통과시켜 “새로 작성된 쿼리가 원본 쿼리의 원래 의미(Intent Grounding)를 훼손하지 않았는가?“를 Pass/Fail 로직으로 심판해야 한다. 탈락 시 원본 파싱 문자열로 롤백하라.
