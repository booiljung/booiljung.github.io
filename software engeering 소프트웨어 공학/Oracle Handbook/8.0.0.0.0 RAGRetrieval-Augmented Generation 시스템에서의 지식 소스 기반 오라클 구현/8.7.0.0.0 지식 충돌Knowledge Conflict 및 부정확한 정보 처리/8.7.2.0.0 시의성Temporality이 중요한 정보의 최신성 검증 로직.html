8.7.2 시의성(Temporality)이 중요한 정보의 최신성 검증 로직

“우리 회사 CEO가 누구지?“라는 질문에 작년 임원 명단이 불려오면 RAG는 쓸모가 없다. 오라클은 정보의 유통 기한을 관리해야 한다.

  • 청크 메타데이터에 기입된 [last_updated_date] 속성을 추출한다.
  • 오라클 시스템 프롬프트(System Prompt)에 현재 서버의 CURRENT_DATE를 시스템 변수로 주입한다.
  • 지시문: “제공된 청크의 작성일자와 서버의 현재 시간을 대조하라. 만약 질문이 최신 정보를 요구하는 ’상태성 쿼리(Stateful Query)’임에도 검색된 문서가 1년 이상 지체된 정보라면, 답변을 생성하지 말고 ’정보 만료 에러’를 반환하라.”
