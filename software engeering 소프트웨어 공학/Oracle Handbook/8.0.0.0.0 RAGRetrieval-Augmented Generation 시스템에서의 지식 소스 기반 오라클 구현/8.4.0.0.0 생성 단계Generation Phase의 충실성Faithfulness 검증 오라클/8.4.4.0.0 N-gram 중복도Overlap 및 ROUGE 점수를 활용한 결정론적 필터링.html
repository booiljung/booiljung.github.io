8.4.4 N-gram 중복도(Overlap) 및 ROUGE 점수를 활용한 결정론적 필터링

LLM의 연산조차 사치스러울 만큼 트래픽이 몰리는 단계에서는, 구시대적이지만 압도적으로 연산이 가벼운 통계적 NLP 지표가 구원투수가 된다.

  • 원문 Context와 출력된 Answer 간의 ROUGE-L(최장 부분 일치) 점수를 계산한다.
  • RAG의 본질은 “주어진 것을 정리해서 대답하는 것“이므로, 원본 문장에 쓰인 고유 명사나 핵심 구문(N-gram)의 교집합 비중(Overlap)이 극단적으로 낮다면(예: < 15%), 모델이 외부의 뇌피셜(World Knowledge)을 끌어오고 있다는 치명적인 징후다. 이를 1차 필터링 플래그로 활용하라.
