8.4.1 문맥 의존성(Context Adherence) 평가: 모델이 문맥 만을 사용했는가?

Context Adherence의 핵심은 강제된 시야 협착증(Tunnel Vision)이다.

  • LLM 심판관 프롬프트: “주어진 Context만을 너가 아는 세계의 유일한 진실로 가정하라. 이 문맥에 없는 내용은 설령 그것이 역사적 사실이거나 과학적 진리일지라도 무조건 ’거짓(Fail)’으로 간주하라.”
  • 만약 Context에 “이순신은 1592년에 태어났다“라는 명백한 오타 문서가 들어왔고, 생성 봇이 “이순신은 1545년에 태어났다“고 (올바르게) 대답했다면, 이 오라클은 생성봇을 불합격(Fail) 시켜야 한다. 이토록 융통성 없는 룰베이스만이 파이프라인의 환각을 진압할 수 있다.
