8.4.6 주장-증거(Claim-Evidence) 매핑을 통한 논리적 비약 검사

이 검사는 모델이 그럴싸한 일반론을 펼치는 것을 철저히 억누른다.

  • 오라클 지시문: “생성된 텍스트를 문장 단위로 분해하고, 각 문장(주장, Claim)마다 원문에서 정확히 어느 부분(증거, Evidence)을 근거로 삼았는지 1:1 매핑 표명(Mapping JSON)을 작성하라.”
  • 만약 특정 주장에 대응하는 근거를 Context 내에서 찾지 못해 [EVIDENCE_NOT_FOUND] 라고 표기하게 된다면, 그 즉시 논리적 비약(Logical Leap)으로 간주하여 감점한다.
