8.4.2.1 자기 모순(Self-Contradiction)과 외부 지식 모순의 구분

답변이 틀린 데에는 두 가지 양상이 있으며, 오라클은 이를 명확히 분리해서 디버깅 로그를 남겨야 한다.

  1. 자기 모순(Self-Contradiction): 앞 문장에서는 “금리가 인상되었다“고 해 놓고 뒷 문장에서는 “금리가 인하되어 시장이 안정화되었다“고 말하는 케이스. 이 경우 오라클은 외부 문맥(Context)을 대조할 필요도 없이 답변 자체의 논리적 모순성만으로 즉시 기각한다.
  2. 외부 지식 모순(External Knowledge Contradiction): 답변 자체는 매우 논리적이나, 그 내용이 Context의 명제와 대치되는 경우. 이 경우 오라클은 “어느 청크의 어떤 명제를 위반했는지” 포인터를 반환해야 한다.
