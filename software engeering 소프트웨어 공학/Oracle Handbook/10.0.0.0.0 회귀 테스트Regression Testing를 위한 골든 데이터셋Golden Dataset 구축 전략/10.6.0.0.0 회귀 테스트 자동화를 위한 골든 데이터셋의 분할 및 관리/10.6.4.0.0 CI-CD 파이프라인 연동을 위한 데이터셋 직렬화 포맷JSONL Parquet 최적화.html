10.6.4 CI/CD 파이프라인 연동을 위한 데이터셋 직렬화 포맷(JSONL, Parquet) 최적화

데이터가 100MB를 넘어가면 Github Repo에 올려두고 파이썬 루프(for line in lines...)를 돌리는 원시적 방식은 디스크 I/O 병목의 지옥을 이끈다.

  • JSONL 포맷: 테스트 드라이버가 대용량 메모리를 점유하지 않고, 제너레이터(Generator) 방식으로 한 줄씩 읽어 비동기로 쏘아 올리기 최적화된 형식이다.
  • Parquet 변환: 메타데이터 분석용 데이터웨어하우스 연동이나 병렬 노드(PySpark)에서 분산 채점을 수행할 때, 컬럼 지향 스토리지 구조를 통해 I/O 스트리밍 속도를 극대화한다. 이 인프라 설계가 결국 CI 워크플로우를 “분” 단위에서 “초” 단위로 압축해 낸다.
