10.2.3 구조화된 출력(Structured Output) 데이터셋: JSON/XML 스키마 적합성 검증용

Function Calling이나 도구 주도(Tool-use) 에이전트를 검증할 때 쓰이는 핵심 데이터셋이다.

  • 적용 대상: 영수증 파싱 AI, 예약 시스템 연동 AI.
  • 데이터 구조: 기대 결과값(expected_output)에 비정형 텍스트 대신 완벽한 JSON Schema나 JSON Object 형태를 집어넣는다.
  • 오라클 매핑: 파이썬의 Pydantic 체커나 JSON Schema Validatior 오라클을 연결한다. 의미적 모호성 없이, 오라클은 모델의 출력값에서 required field가 누락되거나 타입(Int 자리에 String)이 틀어진 오류를 기계적으로 차단한다.
