10.5.2 골든 데이터셋의 ‘정답’ 유효기간 관리 및 정기적 재검토 프로세스

세상의 지식은 유통기한이 있다. 2022년에 구축한 “현재 미국 대통령은?” 이라는 골든 데이터(정답: 바이든)는 2025년의 오라클 환경 시스템에서는 오답(Bug)이 된다.

  • TTL(Time To Live) 메타데이터: 시간성에 민감한 정책 질문(예: 금리, 환불 규정, 법률) 데이터에는 수명(TTL: 6개월) 태그를 달아둔다.
  • 배치가 돌며 만료된 데이터 리스트를 인간 검증자(HITL) 대시보드로 올려보낸다. 검증자는 ‘Extend(연장)’, ‘Modify(수정)’, ‘Deprecate(폐기)’ 중 하나를 수동으로 결단해 데이터셋의 시의성(Freshness) 부패를 막는다.
