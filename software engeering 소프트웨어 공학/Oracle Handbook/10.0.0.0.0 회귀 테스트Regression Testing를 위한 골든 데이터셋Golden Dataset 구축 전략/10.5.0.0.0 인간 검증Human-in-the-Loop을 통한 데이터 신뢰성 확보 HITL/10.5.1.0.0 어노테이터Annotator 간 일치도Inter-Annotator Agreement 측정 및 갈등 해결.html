10.5.1 어노테이터(Annotator) 간 일치도(Inter-Annotator Agreement) 측정 및 갈등 해결

사람의 생각도 저마다 다르다. 동일한 AI의 답변을 보고 A 전문가는 “합격“을, B 전문가는 “불합격“을 선언할 수 있다.

  • Fleiss’ Kappa / Cohen’s Kappa 지표: 어노테이터 3명 이상이 독립적으로 작업한 결과의 일치도를 통계적으로 측정한다.
  • 일치도(Kappa)가 0.7 이하로 떨어지는 주제(Domain)는 인간들 사이에서도 가이드라인이 모호하다는 증거다. 이 경우 해당 데이터들을 전량 파기하거나, 리드(Lead) 어노테이터 주도하에 ’갈등 해결(Conflict Resolution) 회의’를 열어 평가 가이드북의 허점을 수정한 뒤에만 골든 데이터베이스에 편입시킨다.
