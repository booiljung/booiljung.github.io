10.1.2 회귀 테스트에서의 ’정답(Ground Truth)’의 의미 변화: 단순 텍스트 일치에서 의도 일치로

소프트웨어 1.0 체제에서 회귀 테스트(Regression Test)는 expected == actual 이라는 단순 문자열/바이트 단위의 일치를 의미했다.

  • AI 시대의 Ground Truth: AI 소프트웨어에서 ’정답’이란 “토시 하나 틀리지 않음“을 의미하지 않는다. 어제는 “환불 불가입니다“라고 대답했던 챗봇이 오늘 “규정상 환불을 처리할 수 없습니다“라고 대답했다면, 문자열 비교(Exact Match) 로직은 실패(Fail)를 반환하겠지만, 의미론적(Semantic) 관점에서는 완벽한 통과(Pass)다.
  • 따라서 골든 데이터셋의 ‘기대 출력(Expected Output)’ 필드는 단순한 텍스트 덩어리가 아니라, 1) 절대 포함되어야 할 앵커 단어, 2) 포함되지 말아야 할 개념, 3) 합의된 비즈니스 의도(Intent Payload)의 집합체로 재정의(Redefinition)되어야 한다.
