10.3.1 프로덕션 로그 기반의 실제 사용자 데이터(Real-world Data) 마이닝 및 익명화

엔지니어가 상상한 테스트 케이스와 실제 유저가 뱉어내는 엉망진창의 입력은 차원이 다르다. 가장 훌륭한 골든 데이터는 이미 서비스 중인 프로덕션 로그(Production Log)에 묻혀 있다.

  • 파이프라인 레이아웃: 서비스되고 있는 서버의 ElasticSearch나 Datadog 로그에서 User Input 과 Final AI Output 쌍을 추출한다.
  • 클렌징 및 탈식별화(Anonymization): 수집된 원시 데이터에는 주민번호, 카드번호 등 PII(개인식별정보)가 가득하다. Presidio 나 AWS Macie 같은 NLP 스캐너 도구를 파이프라인에 배치하여, “홍길동의 카드번호는 1234…“를 “[NAME]의 카드번호는 [CARD_NUM]…” 으로 정교하게 마스킹(Masking)한 뒤 비로소 골든 저장소에 입고시켜야 한다.
