10.3.3 합성 데이터(Synthetic Data) 생성 전략: LLM을 활용한 데이터 증강 및 다양성 확보

인간과 실제 로그 기반 수집으로는 ‘1만 건’ 수준의 대규모 골든셋을 구축하기에 단위 비용(Cost)이 너무 압도적이다. GPT-4o 같은 강력한 추론 모델을 교사(Teacher)로 세워 학생 프롬프트용 훈련 데이터를 다량으로 주조(Forging)해 내야 한다.

  • Seed Expansion(시드 배양): 인간 전문가가 작성한 고품질 데이터(Seed) 50개를 교사 모델의 컨텍스트(Context)에 밀어 넣는다.
  • 프롬프트 엔지니어링: “방금 본 인간의 입력-출력 패턴의 어조와 도메인을 유지하면서, 페르소나(화난 고객, 말이 빠른 고객, 오타가 많은 고객)를 10가지로 변형하여 총 500개의 합성(Synthetic) 질문-정답 쌍 쌍을 생성하라.”
  • 이를 통해 낮은 비용으로 거대한 회귀 테스트 방어벽의 볼륨을 부풀릴 수 있다.
