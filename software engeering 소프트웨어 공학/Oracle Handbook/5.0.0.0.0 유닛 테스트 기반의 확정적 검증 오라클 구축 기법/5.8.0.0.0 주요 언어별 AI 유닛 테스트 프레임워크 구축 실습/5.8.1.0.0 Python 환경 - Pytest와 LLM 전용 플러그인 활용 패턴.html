5.8.1 Python 환경: Pytest와 LLM 전용 플러그인 활용 패턴

Python은 거대한 AI 생태계를 보유하고 있다. 유닛 테스트 역시 압도적 표준인 pytest 위에서 구성된다.

  • pytest-asyncio: LangChain 등 비동기(Async) LLM 호출을 테스트할 때 필수적이다.
  • pytest-vcr: 앞서 5.4절에서 언급한 VCR 패턴을 Pytest에 데코레이터(@pytest.mark.vcr) 한 줄로 통합해 준다. HTTP 요청을 가로채 카세트(Cassette)에 기록하므로, pytest 속도를 수백 배 끌어올린다.

실전 패턴

import pytest
from my_ai_oracle import check_sql_injection_prompt

@pytest.mark.vcr() # 응답을 저장/재생
def test_sql_injection_detection_oracle():
    malicious_input = "DROP TABLE users;"
    response = check_sql_injection_prompt(malicious_input)
    assert response.status == "FAIL"
    assert "보안정책 위반" in response.reason

단 3줄의 코드만으로, 평생 변하지 않는 확정적 보안 검증 오라클이 탄생한다.
