5.6 논리적 추론 검증을 위한 중간 과정 검사(Intermediate Verification)

아무리 튼튼하게 설계된 AI 오라클이라도 결괏값 도출 과정이 완전한 블랙박스라면 신뢰할 수 없다. 특히 복잡 다단한 비즈니스 룰을 태운 Chain-of-Thought (CoT) 파이프라인이라면, 최종 답변(PASS, FAIL)이 도출되기 직전의 그 짙은 안개 속을 밝혀 단계별 상태를 검증하는 화이트박스 식 관찰탑이 필요하다.

본 절에서는 AI의 뇌 속 연산 과정을 쪼개어 가로채고, 외부 코드 실행기 등을 동원해 결괏값이 아닌 ’과정(Process)’을 디버깅하는 오름차순 검수 기법에 대해 살펴본다.
