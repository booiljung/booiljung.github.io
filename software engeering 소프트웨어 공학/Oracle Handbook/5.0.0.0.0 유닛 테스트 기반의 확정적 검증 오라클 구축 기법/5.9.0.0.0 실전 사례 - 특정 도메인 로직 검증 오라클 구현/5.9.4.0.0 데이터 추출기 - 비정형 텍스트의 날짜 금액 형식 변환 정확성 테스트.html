“내일 모레 점심 쯤에 2만 오천원 입금할게“라는 복잡한 자연어를 {"date": "202X-MM-DD 12:00:00", "amount": 25000} 으로 정규화해 뽑아내는 NER(명명개체인식) 봇이다.

오라클 테스트 로직 (타임 머신 / 스키마 검증):

  1. “내일 모레“라는 상대적 시간을 확정시키기 위해 프롬프트 템플릿에 {{ System_Date = "2024-01-01" }}를 하드코딩해서 주입한다.
  2. AI의 응답 JSON을 로드하여 amount가 정확한 정수 25000 타입을 가지는지 타입 검증을 수행한다.
  3. date 값이 “2024-01-03 12:00:00“이라는 정적 정규표현식 포맷(^\d{4}-\d{2}-\d{2} \d{2}:\d{2}:\d{2}$)과 완벽히 일치하는지 Regex 검수를 통과시킨다.
