5.10.1 LLM 전용 테스트 프레임워크 개요 (Promptfoo, DeepEval 비교)

레거시 코드를 검증하듯 AI 평가를 바닥부터 짤 필요는 없다. 현재 시장에는 두 가지 강력한 흐름의 오픈소스 평가 도구가 있다.

  • Promptfoo: 프롬프트 A/B 테스트와 어써션(Assertion)에 극도로 특화된 초경량 CLI 도구다. 개발자가 .yaml 스키마만 정의하면 복잡한 환경 설정 없이 로컬에서 수천 개의 지문을 병렬 테스트하고 웹 뷰 리포트를 띄워준다. “속도와 포맷 강제“에 압도적인 이점을 가진다.
  • DeepEval: Pytest 기반으로 작성된 본격적인 LLM 평가 프레임워크다. G-Eval(LLM-as-a-Judge)이나 RAG 신뢰도 측정 등 고도화된 메트릭을 기본 제공하며, CI 파이프라인에서 수치 기반의 회귀 테스트(Regression Test) 릴리스 게이트 역할을 수행하기에 최적화되어 있다.

오라클의 목적이 ’출력 형식 제한’이라면 Promptfoo가, ’내용의 전문성과 환각 탐지’라면 DeepEval 계열이 권장된다.
