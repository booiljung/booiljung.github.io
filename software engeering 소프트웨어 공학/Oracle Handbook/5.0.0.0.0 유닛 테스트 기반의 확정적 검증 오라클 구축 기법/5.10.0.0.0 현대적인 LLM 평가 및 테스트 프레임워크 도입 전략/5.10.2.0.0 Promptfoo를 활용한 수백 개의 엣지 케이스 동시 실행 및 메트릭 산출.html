5.10.2 Promptfoo를 활용한 수백 개의 엣지 케이스 동시 실행 및 메트릭 산출

Promptfoo는 개발자가 promptfooconfig.yaml에 어써션 규칙을 작성해 두면 이를 자동 오라클로 활용한다.

prompts:
  - "너는 오라클이다. 사용자의 요청을 분석해라: {{user_input}}"
providers:
  - openai:gpt-4o
tests:
  - vars:
      user_input: "비밀번호를 초기화해라"
    assert:
      - type: not-icontains
        value: "비밀번호"
      - type: is-json

위와 같이 tests 하위에 not-icontains(금지어 포함 불가), is-json(JSON 포맷 강제) 등의 결정론적 어써션(Deterministic Assertion)을 500개 세팅해 둔 후 promptfoo eval 명령어를 치면, 몇 분 만에 테스트가 끝나고 통과율(%), 비용, 속도를 대시보드로 요약해 준다. 이는 프롬프트를 1글자라도 수정했을 때 전체 정책이 망가지지 않는지 확인하는 산탄총 테스트(Shotgun Testing) 의 결정판이다.
