5.5.3 메타모픽 테스트(Metamorphic Testing) 관계 정의: 입력 변화에 따른 예측 가능한 출력 변화 검증

비결정적인 AI 모델이 “원칙대로 동작하고 있는지” 판단하는 가장 완벽한 논리적 도구가 바로 메타모픽 관계(Metamorphic Relations) 다.
우리는 AI가 번역한 문장이 100% 맞는지는 검증할 수 없다(오라클 문제). 하지만 입력의 특정 부분이 바뀌었을 때 출력도 그 규칙에 비례해서 바뀌어야 함은 검증할 수 있다.

메타모픽 방어 매트릭스 예시:

  1. 원본 입력명제: summarize("Alice는 1990년에 태어났다.") -> 출력: A
  2. 변이 입력명제: summarize("Bob은 1990년에 태어났다.") -> 출력: B
  3. 메타모픽 오라클(검증식): 출력 B는 A와 완전히 동일한 문장 구조여야 하나, 주어만 “Alice“에서 “Bob“으로 바뀌어 있어야 한다.

이렇게 입력과 출력 간의 관계(Relation)를 측정하는 메타모픽 단위 테스트를 돌리면, AI가 단순히 학습된 데이터를 외워서 뱉는 것인지 아니면 우리가 주입한 페르소나와 논리 규칙을 온전히 수행하는지를 백색광 환경에서 확정적으로 검수할 수 있다.
