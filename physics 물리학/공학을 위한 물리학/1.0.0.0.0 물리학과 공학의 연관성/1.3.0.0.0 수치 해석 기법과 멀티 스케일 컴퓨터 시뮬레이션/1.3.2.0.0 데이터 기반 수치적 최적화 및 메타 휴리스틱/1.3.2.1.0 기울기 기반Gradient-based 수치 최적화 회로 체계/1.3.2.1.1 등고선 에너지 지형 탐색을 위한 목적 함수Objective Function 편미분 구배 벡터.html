1.3.2.1.1 등고선 에너지 지형 탐색을 위한 목적 함수(Objective Function) 편미분 구배 벡터

물리적 최적화 설계의 본질은 주어진 시스템이 만족해야만 하는 조건을 지키면서, 에너지, 비용 혹은 오류의 척도를 최소화하는 해를 도출하는 것이다. 복잡한 다변수 함수(Multivariable Function) 환경에서 각 독립 변수(Independent Variable)의 조합은 다차원의 에너지 초평면(Energy Hyperplane) 혹은 등고선 지형(Contour Topography)을 형성하며, 이 거대한 곡면 어딘가에 가장 깊고 안정된 골짜기인 전역 최소점(Global Minimum)이 존재한다. 이러한 최적해 구간의 방향성을 지시하는 나침반 역할을 수행하기 위하여, 목적 함수(Objective Function)에 대한 각 변수별 편미분(Partial Derivative) 성분을 집약한 구배 벡터(Gradient Vector)의 수학적 유도와 공학적 응용을 정립하라.

1. 목적 함수(Objective Function)와 다차원 에너지 지형

공학 최적화에서 설계 변수 벡터를 \mathbf{x} = [x_1, x_2, \dots, x_n]^T \in \mathbb{R}^n 라 정의할 때, 최적화의 표적이 되는 목적 함수 f(\mathbf{x}) 는 n차원의 공간 좌표를 단일한 스칼라(Scalar) 실수 값으로 변환하는 매핑 f: \mathbb{R}^n \rightarrow \mathbb{R} 이다.
예를 들어, 탄성 구조 부재의 경량화 설계라면 f(\mathbf{x}) 는 구조물의 총 체적(Volume)이 될 수 있으며, 머신러닝 시스템에서는 예측 모델과 정답 사이의 평균 제곱 오차(Mean Squared Error, MSE)인 손실 함수(Loss Function) \mathcal{L}(\theta) 가 이 목적 함수의 역할을 수행한다.

이러한 n개의 변수로 축조된 목적 함수는 기하학적으로 n+1 차원 공간에서의 거대한 초곡면(Hypersurface)을 만든다. 이 지형을 특정 스칼라 상수 c 로 절단했을 때 나타나는 n 차원의 폐곡선 f(\mathbf{x}) = c 가 바로 등고선(Contour Line) 혹은 등위면(Level Set)이다. 목적 함수를 최소화한다는 것은 이 등고선들의 고도가 가장 낮아지는 골짜기의 중심 좌표 \mathbf{x}^* 를 찾아 이주하는 공학적 여정이다.

2. 편미분(Partial Derivatives)과 구배 벡터(Gradient Vector)의 형성

등고선 지형을 등반 혹은 하강하기 위해서는 현재 좌표 \mathbf{x} 에서 한 걸음을 옮길 때 고도가 얼마나 급격하게 변하는 지에 대한 국부적 정보가 필수적이다. 변수 벡터의 특정 방향인 x_i 에 대해서만 극한의 미소 전진을 했을 때의 목적 함수의 변화율을 해당 성분에 대한 편미분(Partial Derivative) \frac{\partial f}{\partial x_i} 라 한다.

\frac{\partial f(\mathbf{x})}{\partial x_i} = \lim_{\Delta x \to 0} \frac{f(x_1, \dots, x_i + \Delta x, \dots, x_n) - f(x_1, \dots, x_i, \dots, x_n)}{\Delta x}

각 방향별 편미분 계수들로 구성된 n 차원의 열 벡터(Column Vector)가 바로 스칼라장 f 영역의 공간적 미분인 구배 벡터(Gradient Vector) \nabla f(\mathbf{x}) 이다.

\nabla f(\mathbf{x}) = \left[ \frac{\partial f}{\partial x_1}, \frac{\partial f}{\partial x_2}, \dots, \frac{\partial f}{\partial x_n} \right]^T

이 구배 벡터는 미적분학의 본질적 특성상, 목적 함수의 표면 등고선에 항상 수직(Orthogonal)으로 뻗어나가는 방향타를 갖는다. 즉 \nabla f(\mathbf{x}) 는 현재 지점에서 고도가 가장 최대 속도로 치솟는 가장 가파른 상승(Steepest Ascent) 궤도를 수학적으로 강제 지시한다.

3. 방향 미분(Directional Derivative)과 최소 강하 경로의 역학

구배 벡터가 과연 가장 가파른 방향과 동치인지 증명하기 위해서는, 임의의 단위 방향 벡터 \mathbf{d} (||\mathbf{d}|| = 1)로 이동할 때의 함수 변화율인 방향 미분(Directional Derivative) D_{\mathbf{d}} f(\mathbf{x}) 를 전개하여 고찰하라.
다차원 테일러의 급수 전개에 의해 1차 방향 미분은 구배 벡터와 투영 방향 벡터의 내적(Dot Product)으로 정의된다.

D_{\mathbf{d}} f(\mathbf{x}) = \nabla f(\mathbf{x}) \cdot \mathbf{d} = ||\nabla f(\mathbf{x})|| ||\mathbf{d}|| \cos \theta

위 수식에서 함수 값의 변화율인 D_{\mathbf{d}} f(\mathbf{x}) 가 최대치가 되기 위해서는 두 벡터 사이의 각도 \theta = 0^\circ 이어야 하므로, 상승 방향 \mathbf{d} 가 결국 구배 \nabla f(\mathbf{x}) 와 평행해야 한다는 삼각함수론적 진리에 도달한다.
반대로 목적 함수를 최소화하기 위해 에너지를 깎아내려야 하는 엔지니어의 입장에서는 변화율이 가장 음수로 극대화되는 \theta = 180^\circ 방향이 절실하다. 따라서 이 음의 구배인 역벡터 -\nabla f(\mathbf{x}) 가 가장 가파른 하강(Steepest Descent)을 보장하는 탐색 경로의 근원적 이정표가 됨을 확고히 명심해야 한다.

4. 국소 최적점(Local Minimum) 검증을 위한 일차성 극값 필요조건

초평면의 험난한 협곡을 따라 지속적으로 역-구배 방향으로 이동하다 보면, 지형의 오르내림이 평평해져 더 이상 굴러떨어질 사면이 남지 않은 바닥에 도달하게 된다. 극소점 혹은 최소점이 될 수 있는 지형적 특이점에서는 모든 방향 스칼라에 대한 기울기가 0으로 수렴해야 한다.
따라서 다변수 목적 함수의 비제약 공간 국부 최소값(Local Minimum) 최적해 \mathbf{x}^* 에서 함수가 정류점(Stationary Point)이 되기 위해서는 다음의 일차 필요 조건(First-order Necessary Condition)인 페르마 정리(Fermat’s theorem for stationary points)를 절대적으로 만족해야 한다.

\nabla f(\mathbf{x}^*) = \mathbf{0}

이는 각각의 편미분 성분이 모두 0 이라는 의미이다. (\frac{\partial f}{\partial x_i} = 0, \forall i)
하지만 기울기 벡터가 0 이 된 지점이라 하여 반드시 그곳이 진정한 최소점이라고 확신해서는 안 된다. 지형이 최대 극대점(Local Maximum)이거나, 한쪽으로는 상승하고 한쪽으로는 하강하는 능선인 안장점(Saddle Point)일 때도 기울기는 동일하게 \mathbf{0} 벡터가 되기 때문이다. 수렴한 정류점 \mathbf{x}^* 가 안전한 수치적 최소점인지 판별하기 위해서는, 이를 미분 기하학적으로 이중 검증하는 양의 정부호(Positive Definite) 특성의 헤시안(Hessian Matrix) 2차 체계를 결사적으로 동반하여 평가해야 한다. 이에 관한 엄밀한 수학적 안정성 분석은 문헌 “Convex Optimization” 및 2차 구배 분석 파트의 논거를 철저히 이수하라.
