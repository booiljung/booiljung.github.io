1.3.2.1.7 위상 최적화(Topology Optimization) SIMP 기법 기반의 재료 밀도 최적화 및 강성 극대화 설계

유한 요소법(FEM)을 기반으로 한 구조 최적화의 발전 과정 중 가장 드라마틱하고 혁신적인 수학적 형상 도출 기법은 단연 위상 최적화(Topology Optimization)이다. 치수나 경계면을 소극적으로 변경하는 형상 최적화(Shape Optimization)와 달리, 위상 최적화는 주어진 설계 영역(Design Domain) 내부에서 하중을 지탱하지 못하는 불필요한 재료를 완전히 제거하고 새로운 구멍(Hole)을 창출해 내어 궁극의 유기적인 골조를 자생적으로 획득한다. 이 과정에서 이산적(Discrete)인 재료 물질의 유무(0 또는 1) 상태를 미분 가능한 연속 변수로 완화(Relaxation)시켜 강력한 기울기 기반(Gradient-based) 최적화 회로에 종속시키는 SIMP(Solid Isotropic Material with Penalization) 기법의 역학적 근간과 최적 수렴 체계를 확립하라.

1. 재료 유무(0-1) 문제의 연속 밀도장(Density Field) 완화

구조 위상 최적화의 본질적인 목적은 제한된 재료 부피 분율(Volume Fraction) 체적 내에서 대상 구조물의 유연도(Compliance), 즉 변형 에너지(Strain Energy)를 전역적으로 최소화하여 전체 시스템의 강성(Stiffness)을 극대화하는 것이다. 설계 대상 공간을 구성하는 각 유한 요소 e 마다, 재료가 비어 있음(Void, \rho_e = 0) 또는 완전히 차 있음(Solid, \rho_e = 1)으로 변수를 할당하는 이진(Binary) 정수 계획법은 2^N 이라는 폭발적인 조합의 경우의 수를 낳아 연산이 사실상 불가능해진다.

이를 해결하기 위해, 각 요소의 밀도 \rho_e 를 0과 1 사이의 실수 값을 갖는 연속 밀도장 변수(Continuous Density Variable)로 변환(Relaxation)하라.
0 < \rho_{\min} \le \rho_e \le 1

이 수학적 우회를 거치면, 불연속적인 설계 공간이 미분 가능한 매끄러운 에너지 지형으로 탈바꿈하여, 목적 함수에 대한 밀도의 편미분 \frac{\partial c}{\partial \rho_e} 을 즉각적으로 도출하는 기울기 기반 연속 최적화 알고리즘의 궤도에 안착시킬 수 있다. (수치 특이점 방지를 위해 \rho_{\min} = 10^{-3} 과 같은 미소 하한계를 도입함을 내포한다.)

2. SIMP (Solid Isotropic Material with Penalization) 기법과 강성 페널티 수식

만약 연속된 밀도 \rho_e = 0.5 로 수렴하는 요소들이 최적 형상에 무수히 잔류한다면, 공학 모델링으로서는 해석되나 현실 제조(가공, 3D 프린팅)가 불가능한 회색 영역(Gray Area) 즉 모호한 중간 밀도 재료 투성이의 폼(Foam) 구조가 도출되는 치명적인 결함이 발생한다.

이러한 중간 밀도 요소의 잔류를 제거하고 결과물이 명확하게 0(빈 공간) 또는 1(단단한 고체)로 흑백화되도록 유도하기 위해 물리적 페널티(Penalization)를 부여하는 기법이 SIMP 모델이다. SIMP 모델은 밀도 상수 \rho_e 가 요소 강성 행렬(Element Stiffness Matrix) \mathbf{k}_e 의 영률(Young’s Modulus)에 비례관계를 맺도록 강제하되, 이 밀도를 p 제곱하여 비선형적으로 감쇠시킨다.

E(\rho_e) = \rho_e^p E_0
\mathbf{k}_e(\rho_e) = \rho_e^p \mathbf{k}_0

여기서 E_0 와 \mathbf{k}_0 는 속이 꽉 찬 순수 고체 재료의 탄성 계수와 고유 강성 행렬이며, 페널티 인자 p (일반적으로 p \ge 3)는 임계 제어 계수이다. 만일 \rho_e = 0.5 인 경우, p=3 을 취하면 강성은 원래의 0.125 배로 급락하지만 제조 과정에서 소비되는 재료의 부피(비용) 비중은 0.5 로 그대로 산정된다. 즉, 최적화 시스템은 가성비가 지극히 떨어지는 중간 밀도의 배치를 극도로 혐오하게 되어 재료를 0이나 1의 극단으로 스스로 밀어내도록 압박당하게 된다. “A 99 line topology optimization code written in Matlab“과 같은 상징적 알고리즘 논거의 구심점은 이 페널티 매핑 함수에 빚지고 있다.

3. 목적 함수 편미분 설계 민감도 해석 (Design Sensitivity Analysis)

최소화하고자 하는 주된 목적 함수는 외부 하중 벡터 \mathbf{F} 와 이에 따른 전체 전역 변위 벡터 \mathbf{U} 에 의해 결정되는 전체 시스템 유연도(Compliance, c)이다. 각 N개의 유한 요소들의 하위 집합 관점에서는 밀도와 강성 행렬의 함수로 조립된다.

\text{Minimize} \quad c(\boldsymbol{\rho}) = \mathbf{F}^T \mathbf{U} = \sum_{e=1}^N \mathbf{u}_e^T \mathbf{k}_e(\rho_e) \mathbf{u}_e
\text{Subject to} \quad \mathbf{K}(\boldsymbol{\rho}) \mathbf{U} = \mathbf{F}
\text{and} \quad V(\boldsymbol{\rho}) = \sum_{e=1}^N \rho_e v_e \le V^*

전체 체적이 제한 한계 V^* 내에 속해야 한다는 제약 하에서, 1.3.2.1 파트에서 정립한 최적화 회로(라그랑지안 체계 등)에 이 문제를 태우기 위해서는 밀도 \rho_e 가 유연도 스칼라 값 c 에 미치는 수학적 기울기 미분값(Sensitivity)이 요구된다.
외력 하중 \mathbf{F} 가 밀도 분율에 대해 상수로 고정되어 있다는 조건 하원, 목적 함수의 역-기울기 민감도는 연쇄 법칙 부가에 따라 다소 직관적인 형태로 수축 유도된다.

\frac{\partial c}{\partial \rho_e} = -p \rho_e^{p-1} \mathbf{u}_e^T \mathbf{k}_0 \mathbf{u}_e

이 미분값이 의미하는 철학은 심원하다. 즉, 특정 요소의 변형 에너지 한계치(\mathbf{u}_e^T \mathbf{k}_0 \mathbf{u}_e)가 거대한 요소에 밀도 \rho_e 를 아주 미세하게 보충(증가)시켰을 때 전체 유연도 c 가 가장 극적으로 하강(마이너스 부호)한다는 기하학적 의미망을 내포한다. 최적화 솔버(Optimality Criteria Method 등)는 이 기울기가 예리하게 치솟은 지역에 체적 자원을 집중적으로 배급하여 국부 강성을 키운다.

4. 메쉬 의존성과 체스보드 패턴 제어 아키텍처

위 민감도 공식을 통해 기계적으로 밀도를 갱신하다 보면, 공간상 흑백 타일이 얼룩덜룩하게 교차하며 칠해져 요소의 정합을 깨뜨리는 체스보드(Checkerboard) 수치 불안정성 현상과, 유한 요소 메쉬를 잘게 나눌수록 거시적인 구멍 하나 대신 미세한 그물망 구멍이 무한히 생성되는 메쉬 의존성(Mesh Dependency) 오류에 부딪힌다.

이를 물리치기 위해 특정 요소 주변의 일정 반경 r_{\min} 이내에 속한 이웃 요소들의 민감도 구배들을 상호 가중 평균하여 거시적인 경계를 뭉뚱그려(Smoothing) 필터링하는 파장(Filter) 제어 기법을 연계 작동시켜라.

SIMP 기반의 최적 적분 체제는 항공우주 기체의 날개 공력 하중 뼈대나 고가속 모터 브라켓 설계 등의 공학 현장에서 설계자의 물리적 한계를 부수고 자연 진화적 유기 곡선 골격을 단련해 내는 강력한 결정론적 제너레이티브 디자인(Generative Design)의 알파이자 오메가임을 공표하라.
