1.3.2.1.6 딥러닝 다층 신경망 연쇄 법칙(Chain Rule) 역전파(Backpropagation) 에포크(Epoch) 역학

생물학적 뉴런의 시냅스 결합 구조를 수학적으로 모사한 인공 신경망(Artificial Neural Network)은 다수의 은닉층(Hidden Layer)을 통해 고도의 비선형적(Non-linear) 특징 형상을 추출해 내는 탁월한 지능적 메타 모델이다. 입력 데이터 \mathbf{X} 가 여러 층의 선형 가중치(Weight) 곱과 비선형 활성화 함수(Activation Function)를 차례로 통과하여 최종 예측값 \mathbf{\hat{Y}} 를 출력하는 순전파(Forward Propagation) 과정은 직관적이다. 그러나 거대한 모델을 “학습“시킨다는 것은 출력단에서 발생한 손실 오차(Loss Error)를 최소화하는 정밀한 가중치 행렬 세트를 찾아내는 거대한 스케일의 기울기 기반(Gradient-based) 최적화 과정이다. 수십에서 수백 개의 레이어를 거슬러 올라가며 수백만 개의 가중치 매개변수에 대한 극소량의 편미분 벡터를 오차 없이 단번에 통제하기 위해서는 연쇄 법칙(Chain Rule)에 기반한 역전파(Backpropagation) 알고리즘 체계를 도입해야만 한다.

1. 퍼셉트론(Perceptron) 계층 내 순전파 역학과 손실 함수

다층 퍼셉트론(Multilayer Perceptron, MLP)의 임의의 l 번째 레이어에 위치한 뉴런들의 수학적 거동을 정식화하라. 이전 레이어 l-1 에서 전달된 출력 활성화 벡터 \mathbf{a}^{[l-1]} 에 대해 가중치 행렬 \mathbf{W}^{[l]} 을 곱하고 편향(Bias) 벡터 \mathbf{b}^{[l]} 를 더한 행렬 연산식을 선형 가설 스칼라 벡터 \mathbf{z}^{[l]} 라고 정의한다.

\mathbf{z}^{[l]} = \mathbf{W}^{[l]} \mathbf{a}^{[l-1]} + \mathbf{b}^{[l]}

이후 이 선형 결합 결과는 시그모이드(Sigmoid)나 ReLU(\max(0, z))와 같은 비선형 활성화 함수 \sigma 를 통과하여 다음 계층으로 넘겨질 l 층의 최종 출력 \mathbf{a}^{[l]} 로 변환된다.

\mathbf{a}^{[l]} = \sigma^{[l]}(\mathbf{z}^{[l]})

마지막 레이어 L에서 최종 도출된 예측값 \mathbf{a}^{[L]} = \mathbf{\hat{Y}} 와 실제 도메인 정답 \mathbf{Y} 간의 괴리를 산술적으로 평가하기 위해 평균 제곱 오차(MSE)나 교차 엔트로피(Cross-Entropy) 형태의 비용 함수(Cost Function) \mathcal{J} 를 확전한다.
궁극적인 과제는 이 스칼라 비용 함수 \mathcal{J} 를 최소화하기 위해 네트워크 곳곳에 숨겨진 모든 층별 가중치 \mathbf{W}^{[l]} 와 편향 \mathbf{b}^{[l]} 에 대한 개별적인 편미분 기울기 즉 \frac{\partial \mathcal{J}}{\partial \mathbf{W}^{[l]}}, \frac{\partial \mathcal{J}}{\partial \mathbf{b}^{[l]}} 를 발췌하는 것이다.

2. 미적분학의 연쇄 법칙(Chain Rule) 파동 편미분

합성 함수의 미분을 각 단계 단일 함수의 미분들의 곱으로 치환하여 분해하는 미적분학의 기초 “연쇄 법칙” 적용은 다층망에서 기적과도 같은 연산 단축을 불러온다. l 번째 레이어 파라미터가 전체 손실 \mathcal{J} 에 미치는 영향도는 곧 그 사이를 가로막고 있는 수많은 후속 레이어들의 미분 변동량의 연속된 곱으로 직렬 연결된다.

우선, l 번째 계층의 선형 선형 결합 변수 \mathbf{z}^{[l]} 의 미세 변동이 최종 손실 \mathcal{J} 에 미치는 직접적인 민감도(오차 항)를 \boldsymbol{\delta}^{[l]} 로 정의하고 다음 연쇄 법칙으로 전개하라.

\boldsymbol{\delta}^{[l]} = \frac{\partial \mathcal{J}}{\partial \mathbf{z}^{[l]}}

출력층(Output Layer)인 L 에서의 최초 오차 민감도는 예측값과 정답에 대한 비용 함수의 도함수와 활성화 함수의 도함수의 아다마르 곱(Hadamard Product, \odot 요소별 곱)으로 명쾌하게 유도된다.

\boldsymbol{\delta}^{[L]} = \nabla_{\mathbf{a}^{[L]}} \mathcal{J} \odot \sigma^{[L]\prime}(\mathbf{z}^{[L]})

3. 오차 역전파(Backpropagation)의 행렬 점화식

가장 말단에서 계산된 오차 벡터 \boldsymbol{\delta}^{[L]} 가 도출되면, 이를 마치 메아리처럼 네트워크를 거꾸로 관통시킴(Backward)으로써 점진적으로 이전 층 l 의 오차 \boldsymbol{\delta}^{[l]} 를 연쇄 점화식 형태로 발굴해내는 것이 오차 역전파 기술의 백미이다.
이러한 역방향 정보의 파동은 바로 뒤편 층의 가중치 행렬의 전치 행렬(Transpose)을 통해서 수학적인 굴절을 겪으며 아래와 같이 계승된다. (관련 알고리즘의 위대한 성과는 논문 “Learning representations by back-propagating errors” 에 의해 독립적인 전산 과학 체계로 자리 잡았다.)

\boldsymbol{\delta}^{[l]} = \left( (\mathbf{W}^{[l+1]})^T \boldsymbol{\delta}^{[l+1]} \right) \odot \sigma^{[l]\prime}(\mathbf{z}^{[l]})

이제 은닉층 l 에 배당된 목적 함수의 최종 편미분, 즉 가중치와 편향의 방향 구배가 다음과 같은 완벽하게 정제된 역학 도출식으로 확립된다.

\frac{\partial \mathcal{J}}{\partial \mathbf{W}^{[l]}} = \boldsymbol{\delta}^{[l]} (\mathbf{a}^{[l-1]})^T
\frac{\partial \mathcal{J}}{\partial \mathbf{b}^{[l]}} = \boldsymbol{\delta}^{[l]}

각 층마다 독립적으로 수치 방식을 사용해 복잡한 미분을 구할 상황(Gradient Checking)을 원천 배제한 채, 이 점화식 기반 단일 행렬 곱만 수행하면 심층 망 내부 수십억 개의 파라미터 기울기가 우아하게 일괄 추출된다.

4. 에포크(Epoch) 갱신 주기와 최적화 여정

전체 방대한 공학 학습 데이터를 신경망에 단 한 번 통과시켜 역전파를 통해 1회의 가중치 업데이팅을 완주하는 주기를 1 에포크(Epoch)라 호칭한다. 신경망 모델의 비선형 가중치 최적화 모델은 다음의 Flowchart 아키텍처에 근거해 반복적인 진화(Epoch Iterator)를 수반한다.

flowchart TD
    A[신경망 아키텍처 W, b 무작위 정규화 난수 초기화 설정] --> B[단일 미니배치 혹은 전체 데이터세트 순전파 Forward Pass 개시]
    B --> C[네트워크 출력을 통한 활성화 벡터 a 및 선형 z 임시 스토리지 캐싱]
    C --> D[출력 오차 J 와 정답 Y 대비 말단 층의 초기 델타 𝛿^L 연산]
    D --> E[레이어 역순 후진: 𝛿^l = W^T * 𝛿^l+1 ⊙ 𝜎' 연쇄법칙으로 역전파 Backward Pass 행진]
    E --> F[계층별 델타 정보와 순전파 캐시 a를 결합하여 편미분 구배 ∇W, ∇b를 확정]
    F --> G[경사 하강법 혹은 Adam 등 적응형 관성 솔버를 이용한 가중치 실시간 갱신: W_new = W_old - η*∇W]
    G --> H{모든 데이터를 소비하여 1 Epoch 가 완료되었는가?}
    H -->|아니오: 잔여 데이터 존재| B
    H -->|예: 1 Epoch 완수| I{검증 무결성 조기 종료 조건 만족 혹은 목표 Epoch M회 도달?}
    I -->|아니오| B
    I -->|예: 수렴 달성| J[최종 초정밀 수렴 가중치 텐서 네트워크 구축 및 배포]

에포크 횟수가 늘어날수록 신경망은 훈련 데이터의 국부적인 비선형 특징 기저에 수렴(통계적 착취)하지만, 너무 많은 에포크는 훈련 데이터의 잡음 노이즈마저 암기해버리는 과적합(Overfitting) 사태를 야기한다. 따라서 드롭아웃(Dropout) 결손 규제나 조기 종료(Early Stopping)와 같은 공학적 예방 메커니즘을 이 기울기 강하 역전파 전선 상에 유가적으로 병렬 매설하여 일반화 성능(Generalization) 모델링에 주력하라.
