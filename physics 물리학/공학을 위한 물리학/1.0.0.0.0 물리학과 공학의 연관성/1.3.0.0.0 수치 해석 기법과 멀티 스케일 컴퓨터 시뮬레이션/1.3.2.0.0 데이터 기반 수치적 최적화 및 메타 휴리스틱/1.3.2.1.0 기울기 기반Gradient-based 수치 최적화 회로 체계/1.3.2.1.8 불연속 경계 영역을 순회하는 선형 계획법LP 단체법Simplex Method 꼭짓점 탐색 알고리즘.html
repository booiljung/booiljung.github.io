1.3.2.1.8 불연속 경계 영역을 순회하는 선형 계획법(LP) 단체법(Simplex Method) 꼭짓점 탐색 알고리즘

공학 최적화에서 목적 함수와 제약 조건이 모두 선형 방정식(Linear Equation)이나 선형 부등식(Linear Inequality)으로만 구성된 체계를 선형 계획법(Linear Programming, LP)이라 정의한다. 유체 수송망의 최단 거리 배관 설계, 전력 그리드의 부하 분배, 제조 공정의 원자재 혼합 비율 할당 등 방대한 이산 및 연속 제약 네트워크 시스템이 이 선형 계획법으로 환원된다. 선형 등식과 부등식들이 교차하면서 형성하는 설계 영역은 다차원 공간 상에서 반드시 평평한 면과 뾰족한 끝점을 지닌 다면체(Polyhedron)를 이룬다. 이러한 기하학적 뼈대의 가장자리(Edge)를 따라 꼭짓점(Vertex)들만을 순회하며 수학적으로 전역 최적해(Global Optimum)를 정복하는 위대한 해법인 단체법(Simplex Method)의 역학을 탐구하라.

1. 선형 계획법의 표준형(Standard Form) 변환과 행렬 대수

선형 계획법을 알고리즘적으로 분해하기 위해서는 부등식이 난무하는 혼합 제약 연립방정식들을 등식으로만 이루어진 엄밀한 표준형(Standard Form)으로 재단해야 한다.
설계 변수 벡터 \mathbf{x} = [x_1, \dots, x_n]^T 에 대하여 모델을 다음과 같이 구성하라.

목적 함수 극대화(혹은 극소화):
\text{Maximize} \quad Z = \mathbf{c}^T \mathbf{x}

제약 조건:
\mathbf{A} \mathbf{x} \le \mathbf{b} \quad \text{and} \quad \mathbf{x} \ge 0

(\mathbf{A} \in \mathbb{R}^{m \times n}, \mathbf{b} \in \mathbb{R}^m)
부등식 제약을 등식으로 강제 치환하기 위해 부족한 공간만큼의 음이 아닌 여유 변수(Slack Variable) 벡터 \mathbf{s} = [s_1, \dots, s_m]^T \ge 0 를 도입하여 제약 공백을 매워라.

\mathbf{A} \mathbf{x} + \mathbf{s} = \mathbf{b} \quad \Rightarrow \quad [\mathbf{A} \quad \mathbf{I}] \begin{bmatrix} \mathbf{x} \\ \mathbf{s} \end{bmatrix} = \mathbf{b}

이를 통합된 새로운 계수 행렬 \mathbf{\hat{A}} \in \mathbb{R}^{m \times (n+m)} 과 가상 공간이 추가된 변수 벡터 \mathbf{\hat{x}} \ge 0 로 확장하면, 표준 등식 제약 연립방정식 \mathbf{\hat{A}} \mathbf{\hat{x}} = \mathbf{b} 구조가 정립된다. 선형 방정식의 특성상 내부의 곡률이 전혀 존재하지 않으므로, 이 다면체의 극소점(Minimum)이나 극대점(Maximum)은 어중간한 내부 평면이 아닌 반드시 두 개 이상의 제약 조건 경계가 엇갈리는 초다면체 극단점(Extreme point, 꼭짓점) 중 하나에서 결정됨을 증명할 수 있다.

2. 기저 실행 가능해(Basic Feasible Solution, BFS)의 수학적 점유

다면체의 수많은 꼭짓점 중 하나에 위치해 있다는 수학적 사실은, \mathbf{\hat{x}} 성분 중 n 개(즉 전체 변수 n+m 개 중 행렬의 여분 차원수만큼)의 변수가 0으로 결착된 상태임을 대변한다. 0이 아닌 나머지 m 개의 독립된 변수들을 기저 변수(Basic Variables, \mathbf{x}_B), 강제로 0으로 추락한 변수들을 비기저 변수(Non-basic Variables, \mathbf{x}_N)라 선언하라.
제약 행렬을 기저 \mathbf{B}와 비기저 \mathbf{N}으로 분할하여 \mathbf{\hat{A}} = [\mathbf{B} \quad \mathbf{N}] 로 정렬하면 아래와 같은 강제 분해식이 도출된다.

\mathbf{B} \mathbf{x}_B + \mathbf{N} \mathbf{x}_N = \mathbf{b}

\mathbf{x}_N = \mathbf{0} 이므로, 기저 변수의 해는 \mathbf{x}_B = \mathbf{B}^{-1} \mathbf{b} 로 단순화된다. 이때 모든 기저 변수가 \mathbf{x}_B \ge 0 인 양의 제약을 만족할 때, 이 점을 기저 실행 가능해(Basic Feasible Solution, BFS)라 명명하며 기하학적으로 이는 완벽한 실행 영역(Feasible Region)의 꼭짓점좌표와 1:1 대응한다. 단체법(Simplex Method)은 이 BFS의 행렬 분할을 이웃한 다른 꼭짓점의 분할 상태로 연속해서 갈아끼우는 행렬 피벗(Pivot) 변환의 진수이다.

2. 심플렉스(Simplex) 이동 역학과 판별 검정

단체법의 순회 최적화는 특정 BFS에서 출발하여 목적 함수 Z 를 가장 많이 증가시킬 수 있는 0 상태의 비기저 변수 즉 진입 변수(Entering Variable) 하나를 선발하여 값을 0 초과로 올려주는 행위로 시작된다. 역으로 이 진입 변수가 증가함에 따라 제약 조건의 한계에 부딪혀 기존의 기저 변수 중 누군가는 가장 먼저 0으로 곤두박질치게 되는데 이를 탈락 변수(Leaving Variable)로 지목하라. 수학자 조지 단치히(George Dantzig)가 고안한 “The simplex method of linear programming” 알고리즘은 아래의 표 해법(Tableau Pivot) 루티닝을 전개한다.

flowchart TD

    B --> C[목적 함수 행에 위치한 비기저 변수들의 감소 이익 환산 계수 Reduced Cost 집중 분석]
    C --> D{모든 환산 계수가 0보다 작거나 같은가? 즉, 최적인가?}
    D -->|예: 전역 수렴| E[현재 꼭짓점이 글로벌 최적해 해단. 최적 변수 x_opt 반환]
    D -->|아니오: 증가 여지 있음| F[가장 큰 양수의 환산 계수를 지닌 비기저 변수를 진입 변수 Entering Var 로 선출]
    F --> G[우변 상수항 b를 진입 열의 양수 계수들로 나누어 최소 비율 검사 Minimum Ratio Test 수행]
    G --> H[비율이 가장 작은 행의 기존 기저 변수를 탈락 변수 Leaving Var 로 강제 방출]
    H --> I[피벗 Pivot 요소를 1로, 나머지 열 성분을 0으로 만드는 가우스-조던 소거법 전개]
    I --> J[신규 꼭짓점 BFS 이동 완료, 목적 변수 Z 상승. 단계 C로 회귀]

3. 제약 공간 다면체 극단 탐색의 물리적 지배력

심플렉스 알고리즘의 위대한 점은 비록 전체 꼭짓점의 조합의 수가 {n+m \choose m} 에 달하는 가공할 규모일지라도, 모든 점을 무식하게 전수 조사(Exhaustive Search)하지 않는다는 것이다. 오로지 인접한 가장자리(Edge)만을 타고 함수 값이 단조 증가(Monotonic Increase)하는 안전한 오르막길 계통만 밟기 때문에, 거대한 팩토리얼(Factorial) 시간 복잡도를 피해 통상 제약 조건 개수 m 의 2~3배 이내의 극도로 짧은 선형 보폭만으로 종착지(최고 고위도 꼭짓점)에 도착하는 마술과 같은 전산 효율을 뿜어낸다.

심플렉스 해법에서 탈락 변수를 고를 때 수행하는 최소 비율 검사(Minimum Ratio Test)는 기하학적 의미에서 시스템 포인트 파티클이 경계 밖 즉 허용 불가능한 비물리적 영역(Unfeasible Domain)으로 탈선하려는 것을 필사적으로 틀어쥐는 안전 제동 장치(Brake) 역할을 수행한다. 결과적으로 경사 하강법과 같은 미분 곡선 강하가 무력해지는 극단적인 선형 다차원 벽면 지형에서, 단체법은 각 날선 면의 기저 교환(Basis Exchange)을 통해 유일하고 정합적인 공학 배분 해(Allocation Solution)를 쟁취하는 초정밀 기하학적 등반의 완성형 패러다임임을 통찰하라.
