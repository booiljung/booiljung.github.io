1.3.2.1.3 이차 비제약 최적화의 빠른 수렴을 돕는 켤레 기울기법(Conjugate Gradient) 행렬 지향 탐색

단순한 경사 하강법(Gradient Descent)이 겪는 지그재그(Zig-zag)형 진동과 이로 인한 느린 수렴 속도를 극복하기 위해, 헤시안(Hessian) 행렬 전체를 계산하고 저장해야 하는 뉴턴법(Newton’s Method)의 방대한 메모리 의존성을 회피하는 중간자적 해법이 바로 켤레 기울기법(Conjugate Gradient Method, CG)이다. 이 행렬 지향 탐색법은 선형 대수 방정식 시스템 \mathbf{Ax} = \mathbf{b} 에 대한 반복 솔버이자, 동시에 f(\mathbf{x}) = \frac{1}{2} \mathbf{x}^T \mathbf{A} \mathbf{x} - \mathbf{b}^T \mathbf{x} + c 와 같은 이차(Quadratic) 비제약 목적 함수 공간에서 n 차원 궤적을 단 n 번의 이동만으로 완전히 제압하는 기하학적 정밀 타격 알고리즘이다.

1. 켤레 방향(Conjugate Direction)과 \mathbf{A}-직교성(Orthogonality)

경사 하강법의 근본적인 문제는 한 번 최적화했던 탐색 방향의 정보를 다음 스텝에서 완전히 소각해버린다는 데 있다. 이로 인해 과거에 이미 오차를 감소시켰던 성분에 대하여 불필요한 직교 진동이 재발하게 된다. 이를 보완하기 위해 수학자 헤스텐스(Hestenes)와 스티펠(Stiefel)은 논문 “Methods of conjugate gradients for solving linear systems“에서 ’켤레 방향(Conjugate Direction)’이라는 획기적인 선형 대수학적 개념을 체계화하였다.

양의 정부호(Positive Definite)이고 대칭(Symmetric)인 시스템 행렬 \mathbf{A}에 대하여, 영벡터가 아닌 임의의 두 탐색 방향 벡터 \mathbf{d}_i와 \mathbf{d}_j가 다음의 \mathbf{A}-직교(A-orthogonal) 조건을 만족할 때, 두 벡터는 서로 켤레(Conjugate) 관계에 있다고 정의한다.

\mathbf{d}_i^T \mathbf{A} \mathbf{d}_j = 0 \quad (\text{단}, i \neq j)

물리적으로 이는 i 번째 스텝에서 함수를 \mathbf{d}_i 방향으로 최소화시켜 \mathbf{x}_{i+1} 에 도달한 후, 그 다음 스텝에서 수직 방향 \mathbf{d}_j 를 따라 탐색하더라도 기존 공간 상에서의 최적화 상태가 전혀 침해당하거나 간섭받지 않는 독립적인 이동 축이 형성됨을 의미한다.

2. 켤레 기울기법의 행렬 연산 알고리즘

n 차원 변수 공간에서 위 조건을 만족하는 서로 독립인 n 개의 켤레 방향 벡터 집합 \{\mathbf{d}_0, \mathbf{d}_1, \dots, \mathbf{d}_{n-1}\} 이 마련된다면, n 번의 단일 방향 최적화(Line Search) 과정만으로 전체 전역 최적해(Global Optimum) \mathbf{x}^* 에 오차 없이 수렴하는 절대적 결정성(Determinism)을 확보할 수 있다.

CG 알고리즘의 매 회귀 스텝 k에서의 설계 변수 \mathbf{x} 는 다음과 같이 업데이트된다.

\mathbf{x}_{k+1} = \mathbf{x}_k + \alpha_k \mathbf{d}_k

이차 에너지 곡면 상에서 잔차(Residual) 혹은 역-구배 \mathbf{r}_k = \mathbf{b} - \mathbf{A}\mathbf{x}_k = -\nabla f(\mathbf{x}_k) 에 의해 스텝 이동 보폭 \alpha_k 는 가장 완벽한 1차원 최소 지점을 보장할 수 있도록 다음과 같이 엄밀하게 닫힌 해(Closed-form solution)로 유도된다.

\alpha_k = \frac{\mathbf{r}_k^T \mathbf{r}_k}{\mathbf{d}_k^T \mathbf{A} \mathbf{d}_k}

켤레 구배 벡터의 파생: CG의 가장 빛나는 전산학적 우월성은 모든 켤레 기저 벡터를 미리 연산하여 저장할 필요 없이, 가장 최신의 구배 정보와 바로 직전의 탐색 방향성인 \mathbf{d}_{k-1} 오직 이 두 가지 벡터의 선형 결합(Linear Combination)만으로 다음 켤레 방향 \mathbf{d}_k 를 귀납적으로 창출해낸다는 데 있다. 직교 보정 계수 \beta_k 에 대응하는 식은 다음과 같다.

\beta_k = \frac{\mathbf{r}_{k+1}^T \mathbf{r}_{k+1}}{\mathbf{r}_k^T \mathbf{r}_k}
\mathbf{d}_{k+1} = \mathbf{r}_{k+1} + \beta_k \mathbf{d}_k

계산 과정 내내 N \times N 차원의 조밀한 헤시안 역행렬 연산이 불필요하며 오직 행렬-벡터 곱셈(\mathbf{A}\mathbf{d}_k) 한 단계만이 메모리를 점유하므로, 유한 요소법(FEM) 등 수백만 자유도 규모의 구조 해석에서 최고의 속도적 위상(High-Performance Status)을 선점한다.

3. 비선형 공학 함수로의 확장 체계 (Non-linear CG)

만일 목표하는 물리 에너지 계통이 단순한 2차 함수의 곡면을 넘어 일반적인 고차 비선형(Non-linear) 토폴로지를 갖는 경우라면, 상수 행렬 \mathbf{A} 를 직접적으로 전제할 수 없다.
이를 타개하기 위하여 플레처-리브스(Fletcher-Reeves, FR) 또는 폴락-리비에르(Polak-Ribière, PR) 형태의 일반화된 비선형 켤레 기울기 회로(Non-linear Conjugate Gradient)를 도입하라. 이들은 스칼라 수축 계수 \beta_k 를 구배의 2-Norm(유클리드 노름) 비율로 근사 환형화시켜 다음과 같은 제어 루틴을 확립한다.

flowchart TD
    A[문제 정의, 초기 변수 x_0 추정] --> B[초기 역구배 설정: r_0 = -∇f(x_0)]
    B --> C[초기 탐색 방향 설정: d_0 = r_0]
    C --> D[루프 시작 반복 k]
    D --> E[함수 f(x)에 대한 1차원 선 탐색 Line Search 수행하여 α_k 산출]
    E --> F[설계 변수 현시점 갱신: x_k+1 = x_k + α_k * d_k]
    F --> G[진행된 위치에서 신규 타겟 역구배 관측: r_k+1 = -∇f(x_k+1)]
    G --> H{수렴 허용 오차 만족 ||r_k+1|| < Tol}
    H -->|아니오| I[Fletcher-Reeves 공식 등에 의해 직교 편향 스칼라 β_k 연산]
    I --> J[새로운 켤레 탐색 방향 조합: d_k+1 = r_k+1 + β_k * d_k]
    J --> K[k 계수 증가 후 다음 단계 D 회귀]
    H -->|예| L[로컬 최적성 획득 연산 종료 및 해 도출]

비선형 PR 기법에 의한 수치 안정도 등은 문헌 “Note sur la convergence de méthodes de directions conjuguées” 에 의해 그 견고함이 이론적으로 정립되어 있다.
결과적으로, 켤레 기울기법의 행렬 연산 알고리즘 네트워크는 1차 미분(Gradient)의 가벼운 메모리 부하와 2차 곡률(Hessian) 지향의 빠른 수렴 속력이라는 상반된 두 매력 요소를 수학적 직교성이라는 브릿지(Bridge)를 통하여 가장 세련되게 절충한 수치 최적화의 금자탑 회로 체계임을 통찰해야만 한다.
