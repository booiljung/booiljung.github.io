1.3.2.1.2 경사 하강법(Gradient Descent)의 로컬 미니멈(Local Minimum) 탈출과 잔여 학습률 제어

목적 함수(Objective Function)의 기울기에 의존하여 하강하는 경사 하강법(Gradient Descent)은 고차원 공학 최적화에서 가장 강력한 나침반이지만, 현실의 물리적 에너지 지형은 수많은 골짜기와 굴곡을 품은 고도의 비볼록(Non-convex) 함수인 경우가 다반사이다. 가장 깊은 계곡인 전역 최소점(Global Minimum)에 도달하기 전에 시야가 가로막힌 얕은 구덩이에 빠져 기울기가 0이 되어버리는 로컬 미니멈(Local Minimum) 덫은 기울기 기반 최적화의 최대 난형이다. 또한 이 여정을 무사히 완주하기 위해서는 탐색의 보폭을 결정하는 학습률(Learning Rate)의 전략적 제어가 해의 진동과 발산을 제어하는 절대적인 운전대로 작용한다. 이러한 수치적 난관들을 수학적으로 회피하고 종착지에 안착하기 위한 돌파 전략과 적응형 제어 메커니즘을 상세히 구축하라.

1. 비볼록(Non-convex) 지형에서의 로컬 미니멈 고착 현상

설계 변수 \mathbf{x} 에 대한 목적 함수 f(\mathbf{x}) 가 전역 최소점 \mathbf{x}^* 에서 진정한 우주의 바닥 상태 에너지를 가질 때, 주변 국부 영역 N(\mathbf{x}_{local}) 에만 한정되어 f(\mathbf{x}_{local}) \le f(\mathbf{x}) 를 만족하는 지점을 국소 최적점(Local Minimum)이라 한다.

완전한 볼록(Convex) 형태인 밥그릇 모양의 함수에서는 단 하나의 로컬 미니멈이 곧 글로벌 미니멈이므로, 어느 점에서 공을 굴리든 순수 경사 하강 수식 단 한 줄로 해가 도출된다.

\mathbf{x}_{k+1} = \mathbf{x}_k - \eta \nabla f(\mathbf{x}_k)

그러나 다층 신경망이나 트러스 위상 최적화와 같은 고차원 비볼록(Non-convex) 지형에서는 무수히 많은 로컬 미니멈들이 지뢰처럼 매설되어 있다. 일반적인 배치 경사 하강법(Batch Gradient Descent)은 현재 위치에서 \nabla f = 0 이 관측되는 순간 매개변수 \mathbf{x} 의 갱신량을 0으로 계산하여 해당 얕은 구덩이에 영원히 스스로 연금시킨다. 목적 함수가 치명적으로 복잡할 경우, 단순한 기울기 하강 알고리즘만으로는 수학적으로 이 덫에서 탈출할 어떠한 추진력도 만들어낼 수 없다.

2. 확률적 경사 하강법(SGD)과 노이즈(Noise)를 통한 탈출 역학

로컬 미니멈을 타파하기 위한 가장 실용적인 공학적 타개책은 전체 데이터세트를 사용하는 대신 무작위로 추출된 미니 배치(Mini-batch) 혹은 단일 샘플만을 이용하여 기울기를 근사 평가하는 확률적 경사 하강법(Stochastic Gradient Descent, SGD)을 활용하는 것이다.

전체 목적 함수가 N 개의 부분 함수 f_i 의 평균으로 정의된다고 할 때 (f(\mathbf{x}) = \frac{1}{N} \sum f_i(\mathbf{x})), 무작위 샘플 i 에 대해 추출된 편향된 기울기 \nabla f_i(\mathbf{x}_k) 를 이용하면 탐색 경로는 참된 역-구배의 방향에서 불규칙하게 이탈하며 강력한 임의 보행(Random Walk) 노이즈를 수반하게 된다.

\mathbf{x}_{k+1} = \mathbf{x}_k - \eta_k \nabla f_i(\mathbf{x}_k)

이 수치적인 흔들림(Fluctuation)은 알고리즘이 협소한 로컬 미니멈 내부의 경박한 극소점에 접근했을 때, 잘못 추정된 경사 벡터의 튕김 현상을 유발하여 구덩이 밖으로 탈출할 돌파의 운동 에너지를 제공한다. 확률적 노이즈가 비볼록 공간의 에너지 장벽(Energy Barrier)을 넘게 하는 통계 역학 원리는 물리화학의 어닐링(Annealing) 과정과 흡사한 안정성을 보장하며, 이에 관련된 SGD의 탈출 성과는 논문 “Stochastic gradient descent escapes saddle points efficiently“를 통해 엄밀히 증명된다.

2. 모멘텀을 이용한 곡면 협곡에서의 관성 추진

얕은 로컬 미니멈 이외에도 비볼록 공간에서 기울기 0으로 탐색을 정지시키는 지독한 지형은 바로 곡률 구조가 엇갈린 안장점(Saddle Point)이다. 특정한 방향으로는 아래로 기울어 있지만 그와 직교하는 방향으로는 위로 솟아 있는 이 말안장 지점에서 안장점을 탈출하고 깊은 협곡(Ravine)의 바닥을 신속하게 횡단하기 위해서는 모멘텀(Momentum) 항을 융합하라. 관성 벡터 \mathbf{v} 를 통한 가속 평가는 다음과 같다.

\mathbf{v}_{k+1} = \gamma \mathbf{v}_k + \eta \nabla f(\mathbf{x}_k)
\mathbf{x}_{k+1} = \mathbf{x}_k - \mathbf{v}_{k+1}

모멘텀 계수 \gamma 에 의해 지형의 진동 방향인 계곡 벽면 방향으로는 음양의 가속도가 누적 상쇄되어 톱니바퀴형 진동이 억제되고, 골짜기 바닥을 향하는 지속 방향으로는 속도가 배가(Aggregation)되어 험준한 지형을 박차고 나가는 도주 속도(Escape Velocity)가 실리게 된다.

3. 학습률(Learning rate) 감쇠와 적응형 제어 스케줄링

기울기에 곱해지는 비례 상수인 학습률 \eta 를 상수로 고정하는 것은 최적화 말기에 파국을 초래한다. 목표점 근방에서 학습률이 지나치게 거대하면 최적 해를 건너뛰어 발산(Overshooting)하며 영구히 핑퐁 진동하고, 반대로 너무 극소하면 평원에 멈춰 선 채 수억 번을 돌려도 수렴점까지 도달하지 못해 컴퓨터 자원을 무의미하게 소모시킨다.
따라서 해석 스텝(Epoch)이 진행됨에 따라 보폭을 점진적으로 소멸시키는 스텝 감쇠(Step Decay)나 코사인 담금질(Cosine Annealing) 스케줄링(Learning Rate Scheduling)을 강제 적용하라.

\eta_k = \eta_0 \cdot \beta^{\lfloor \frac{k}{s} \rfloor}

여기서 \eta_0 는 초기 학습률, \beta 는 감쇠율, s는 감쇠 주기이다. 최근에는 여기에 그치지 않고 각 독립 변수별로 과거 기울기의 자승(Squared Gradients) 누적값을 바탕으로 보폭을 각 차원마다 개별 제어하는 적응형 최적화(Adaptive Optimization)가 표준적인 동역학 장치로 지배하고 있다.

대표적인 Adam(Adaptive Moment Estimation) 솔버의 경우, 자주 갱신되는 매개변수는 보폭을 세밀하게 타격하고, 희귀하게 나타나는 매개변수는 거대한 보폭을 띠게 하여 급변하는 안장점과 골짜기 계단 지형에서의 갇힘 현상을 극한으로 배제한다. 이토록 고도화된 속도-학습률 이중 제어기(Dual Controller)야말로, 가장 험준한 비볼록 물리 시스템 수학 공간을 무사히 통과하기 위해 기계 공학자가 반드시 통제해야 할 조종간임을 망각해서는 안 된다.
