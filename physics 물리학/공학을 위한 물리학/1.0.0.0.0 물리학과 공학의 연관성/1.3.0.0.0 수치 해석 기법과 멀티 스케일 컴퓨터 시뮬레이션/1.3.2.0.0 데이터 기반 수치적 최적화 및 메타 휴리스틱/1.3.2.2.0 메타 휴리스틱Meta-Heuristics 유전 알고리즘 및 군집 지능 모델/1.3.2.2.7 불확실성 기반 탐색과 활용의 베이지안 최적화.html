1.3.2.2.7 불확실성 기반 탐색과 활용의 베이지안 최적화

공학 설계에서 전산 유체 역학(CFD)이나 거대 분자 동역학(MD) 시뮬레이션과 같이 단 1회의 목적 함수 평가에 수 시간에서 수일의 막대한 연산 비용이 소모되는 블랙박스(Black-box) 최적화 딜레마를 마주할 때가 있다. 이러한 극한의 환경에서는 평가 횟수 자체가 치명적인 자원이 되므로, 수만 세대를 반복하는 유전 알고리즘이나 미분법은 적용이 불가능하다. 이에 대한 궁극의 수학적 해결책으로, 지금까지 관측된 소수의 데이터 점(Data point)들을 기반으로 목적 함수의 형태를 사전 확률(Prior Probability)과 사후 확률(Posterior Probability)의 통계적 추론으로 재구성하여 다음 탐색 지점을 가장 지능적으로 지정하는 베이지안 최적화(Bayesian Optimization)가 대두되었다. 오차의 불확실성을 모델 안에 정량적으로 내재화시킨 가우시안 프로세스(Gaussian Process)와 획득 함수(Acquisition Function)의 상호 역학을 논증하라.

1. 가우시안 프로세스(Gaussian Process)와 대리 모델링 역학

베이지안 최적화의 첫 번째 축은 값비싼 진짜 목적 함수 f(\mathbf{x}) 를 흉내 내는 값싼 통계적 대리 모델(Surrogate Model)을 세우는 것이다. 이를 위해 어떠한 비선형 곡선 변동이라도 수학적으로 안아낼 수 있는 비모수적(Non-parametric) 확률 모델인 가우시안 프로세스(Gaussian Process, GP)를 도입하라.

관측된 설계 변수 \mathbf{x} 와 이에 대응하는 함수 평가값의 집합 \mathcal{D} = \{(\mathbf{x}_i, f(\mathbf{x}_i))\}_{i=1}^n 이 주어졌을 때, 아직 가보지 않은 미지의 공간 \mathbf{x}^* 에서의 함수 값 예측은 단일 스칼라 값이 아닌, 정규 분포(Gaussian Distribution)의 형태를 띤다.

f(\mathbf{x}^*) \mid \mathcal{D} \sim \mathcal{N}(\mu(\mathbf{x}^*), \sigma^2(\mathbf{x}^*))

여기서 기댓값(표준 추정치) 평균 함수 \mu(\mathbf{x}^*) 와 불확실성(예측 분산)을 대변하는 공분산 함수 \sigma^2(\mathbf{x}^*) 는 커널 함수(Kernel Function) k(\mathbf{x}_i, \mathbf{x}_j) 행렬 \mathbf{K} 연산을 거쳐 획득된다.
평균 함수 \mu(\mathbf{x}) 는 현재까지 관측된 점들을 매끄럽게 잇는 가장 신빙성 높은 곡선의 능선을 제시하며, 분산 함수 \sigma(\mathbf{x}) 는 관측점에서 멀어질수록 예측을 신뢰할 수 없는 오차의 너울(Uncertainty Band)을 팽창시킨다. 이처럼 데이터 공백 지점에서 에베레스트처럼 치솟는 \sigma 의 불확실성 정보야말로 베이지안 최적화가 전역 최적해를 향해 이륙하는 가장 결정적인 비행 계기판이 된다.

2. 획득 함수(Acquisition Function)를 통한 탐색과 활용의 지휘

GP 통계망이 재구성한 공간에서, 단 1회의 귀중한 추가 함수 평가 기회가 주어졌을 때 과연 어느 좌표 \mathbf{x}_{\text{next}} 에 유한 요소 시뮬레이터를 회전시킬 것인가? 이를 절대적으로 평가하고 선출하는 수학적 판사가 바로 획득 함수(Acquisition Function, a(\mathbf{x}))이다.

  • 활용 (Exploitation): GP 모델의 평균 예측치 \mu(\mathbf{x}) 가 가장 좋은(최적값일 확률이 높은) 지점을 파고든다.
  • 탐색 (Exploration): GP 모델의 신뢰 구간, 즉 불확실성 오차 \sigma(\mathbf{x}) 가 가장 커서 미개척 상태로 남겨진 미지의 심연을 찌른다.

획득 함수는 이 두 가지 상충하는 본능의 합의점을 수학적으로 산출한다. 가장 널리 쓰이는 지표인 기대 향상도(Expected Improvement, EI) 함수의 경우 논문 “A tutorial on Bayesian optimization of expensive cost functions” 등에 의해 다음과 같이 정량적으로 통제된다.

\text{EI}(\mathbf{x}) = \mathbb{E}\left[ \max(0, f(\mathbf{x}^+) - f(\mathbf{x})) \right]

여기서 f(\mathbf{x}^+) 는 지금까지 발견된 가장 우수한 최적 도출값이다. 가우시안 확률 분포 하에서 EI의 해석적 적분은 평균과 분산의 우아한 대수식으로 치환된다.

a_{\text{EI}}(\mathbf{x}) = (\mu(\mathbf{x}) - f(\mathbf{x}^+)) \Phi(Z) + \sigma(\mathbf{x}) \phi(Z)
Z = \frac{\mu(\mathbf{x}) - f(\mathbf{x}^+)}{\sigma(\mathbf{x})}

첫 번째 항은 기댓값 파이를 향한 강렬한 활용 압력을 지시하며, 두 번째 항은 오차 편차를 뚫고 들어가는 과감한 탐색 열풍을 지시한다. 다음 샘플링 포인트는 이 보조 함수가 그리는 전역 공간 내에서 a_{\text{EI}}(\mathbf{x}) 를 최대로 증폭시키는(Maximize) 좌표 \mathbf{x}_{\text{next}} 로 낙점된다.

2. 베이지안 능동 추론 루프의 연쇄

이렇게 블랙박스 함수와 추론 대리자, 그리고 지휘관인 획득 함수의 연동은 설계자의 수동적 개입 없이 최단 연산 사이클로 미지의 해를 스스로 좁혀 나가는 베이지안 업데이트 연쇄(Bayesian Updating Ring)를 형성한다.

flowchart TD
    A[공학 설계 변수 범위 설정 및 소수의 라틴 하이퍼큐브 Initial Sampling D 완료] --> B[관측 데이터 집합 D 요소를 통한 비용 함수 Black-box 시뮬레이션 단회 수행]
    B --> C[확보된 D 군집을 바탕으로 가우시안 프로세스 GP 대리 모델 피팅 및 하이퍼파라미터 학습]
    C --> D[GP가 예측한 전체 설계 공간의 평균 곡면 μ 와 불확실성 편차 텐서 σ 도출]
    D --> E[μ 와 σ 를 통합 연산하는 획득 함수 Acquisition Function a_EI 표면 방정식 전개]
    E --> F[탐색 Exploration 과 활용 Exploitation 의 트레이드오프가 최정점에 달한 x_next 를 수학적 최대값으로 식별]
    F --> G[저렴한 대리 모델 상에서의 식별 종료. 실제 고비용 공학 시뮬레이터에 x_next 물리적 투입]
    G --> H[실제 함수 값 f_x_next 확보 완료. 신규 데이터 쌍을 관측 집합 D 에 병합 누적 D = D ∪ x_next, f]
    H --> I{비용 예산 Budget 최대 평가 횟수 초과 혹은 모델 오차 수렴 관찰?}
    I -->|아니오| C
    I -->|예: 종단 완료| J[가장 높은 확률적 확신으로 획득된 공학 시스템 글로벌 전역 파라미터 최적 x_opt 반환]

미분 계수를 추출할 수 없는 비행기 날개의 와류 소음 계수 최적화, 혹은 평가 한 번에 수억 원의 재료 소실이 발생하는 신소재 배합 비율 도출의 사지에서 이 베이지안 최적화의 우위는 절대적이다. 이 확률적 접근법은 공학 최적화 모델이 단순한 수식 전개 단계를 넘어, 불확실성이라는 맹수 자체를 통계적 무기로 길들여 공간을 지배하는 가장 지능적인 메타 휴리스틱 설계론의 종착역임을 명심하라.
