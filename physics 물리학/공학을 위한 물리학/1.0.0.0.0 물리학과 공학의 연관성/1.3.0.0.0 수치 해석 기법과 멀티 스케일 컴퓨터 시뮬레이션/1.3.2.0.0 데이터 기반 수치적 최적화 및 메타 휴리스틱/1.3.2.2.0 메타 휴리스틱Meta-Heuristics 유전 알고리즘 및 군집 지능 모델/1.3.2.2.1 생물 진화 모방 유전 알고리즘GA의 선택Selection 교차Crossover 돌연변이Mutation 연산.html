1.3.2.2.1 생물 진화 모방 유전 알고리즘(GA)의 선택(Selection), 교차(Crossover), 돌연변이(Mutation) 연산

자연계의 진화 관성을 묘사하는 찰스 다윈(Charles Darwin)의 적자생존 원리와 멘델(Mendel)의 유전 법칙을 차용하여, 수학적 해공간(Solution Space)에서 미분 연산 없이 최적해를 탐색하는 독보적인 휴리스틱 기법이 유전 알고리즘(Genetic Algorithm, GA)이다. 이 알고리즘은 단일 해석 개체를 다루는 기존 역학 방정식과 달리, 다수의 이산적/연속적 공학 변수들을 일련의 염색체(Chromosome) 유전자 코드로 변환(Encoding)하여 세대(Generation) 집단을 형성한다. 이후 철저히 확률 통계에 의존하는 선택, 교차, 돌연변이라는 3대 유전 연산자를 거쳐 열성 인자를 도태시키고 우성 인자를 결합하는 계통적 진화 메커니즘을 상세히 확립하라.

1. 개체 인코딩(Encoding)과 적합도(Fitness) 함수 공간 설계

유전 알고리즘의 최우선 국면은 공학적 실린더의 두께, 밸브의 압력, 재료의 밀도 등 설계 변수 \mathbf{x} = [x_1, x_2, \dots, x_n]^T 를 가상의 유전자 염기 서열로 인코딩(Encoding)하는 것이다. 전통적인 방식은 이를 0과 1로 구성된 비트 스트링(Binary String)으로 매핑하는 것이며, 부동 소수점(Floating-point) 변수가 주를 이루는 최근의 복잡한 기계 공학 최적화에서는 변수 자체를 실수형 유전자(Real-coded Gene) 배열로 직결시킨다.

집단 스웜(Swarm) 내에 존재하는 임의의 개체 i 에 대하여 목적 함수의 가치 평가를 내리기 위해 적합도 함수(Fitness Function) F(\mathbf{x}_i) 를 설계하라. 목적이 유체 저항 최소화(f_{\text{drag}})라면, 적합도 값은 그에 반비례하도록 F_i = 1 / (1 + f_{\text{drag}}(\mathbf{x}_i)) 와 같이 취하여 오차가 작을수록 생존 스코어가 높아지는 역단조 함수로 조립된다.

해석이 진행될수록 집행되는 군집 내에서의 적합도 분표 평가식은 아래와 같다.
P_i = \frac{F_i}{\sum_{j=1}^{N} F_j}

여기서 N은 전체 군집의 개체 수이며, P_i 는 해당 개체가 다음 세대로 자신의 생물학적 유전 정보를 전달할 수 있는 생존 확률 밀도이다.

2. 진화의 착취 기전: 선택(Selection) 연산

선택(Selection) 단계는 군중 속에서 자성(Magnetic)이 강한 고효율 개체를 포착하여 부모 세대 집단(Mating Pool)으로 복제 승급시키는 **활용(Exploitation)**의 절대적 장치이다. 가장 극단적인 우수자원을 뽑아냄으로써 알고리즘의 수렴 폭발력을 통제한다. 대표적인 선택 압력(Selection Pressure) 할당 기법으로는 룰렛 휠 선택(Roulette Wheel Selection)과 토너먼트 선택(Tournament Selection)이 가동된다.

  1. 룰렛 휠 선택 (Roulette Wheel Selection): 다트판의 면적을 개체의 선형적 평가 점수 P_i 에 비례하도록 분해 분할하고, 난수 r \in [0, 1) 를 회전시켜 걸리는 원형 호 안의 개체를 부모로 스카우트한다. 누적 확률 분포(CDF) C_i = \sum_{j=1}^i P_j 에 대하여 C_{i-1} < r \le C_i 를 만족하는 개체 i 가 발탁된다.
  2. 토너먼트 선택 (Tournament Selection): 확률적 잡음에 의한 유독 우수한 하나의 수퍼 개체가 매번 룰렛을 독식(Premature Convergence)하는 사태를 방어하기 위한 공학적 페널티 장치이다. 군중에서 임의로 소수의 k 개체를 리그전으로 차출한 뒤, 이 k 그룹 내에서 가장 적합도가 높은 개체만을 승자로 채택하는 이산적 절충안이다.

이러한 선택의 여과를 통과하면 다음 세대를 생성할 자원 교배 풀(Mating Pool)이 풍성한 우점종 유전자로 채워지게 된다.

3. 우성 형질의 폭발적 융합: 교차(Crossover) 연산

유전 알고리즘이 다른 무작위 무차별 대입법(Random Search)과 궤를 달리하며 차원 우위를 갖는 핵심 기둥은 단연 교차(Crossover) 연산에 있다.
교배 풀에 들어간 부모 개체 두 쌍이 난수 확률(통상 P_c = 0.6 \sim 0.9)에 따라 자신의 유전자 블록 열을 끊어 서로 교환(Swap)함으로써 새로운 자손(Offspring) 두 기체를 인쇄해 낸다. 이는 논문 “Adaptation in natural and artificial systems“에서 주창된 스키마 무결성(Schema Theorem)의 근간으로서, “짧고 우수한 유전자 블록들은 교환을 통해 더 우수한 거대 블록으로 폭발적으로 조직화된다“는 수학적 진리를 동력으로 삼는다.

1점 교차(One-point Crossover) 기하학적 파편화:
부모 염색체의 임의의 교차점 위치 k (1 \le k < n) 를 단일 난수로 확정한다.
\text{Parent 1} = [x_1, x_2, \dots, | , x_k, \dots, x_n]
\text{Parent 2} = [y_1, y_2, \dots, | , y_k, \dots, y_n]
절단선 이후의 서열을 상호 교환하여 자손을 합성한다.
\text{Offspring 1} = [x_1, x_2, \dots, y_k, \dots, y_n]

실수 전용 인코딩에서는 유전자 값 자체의 평균과 분산을 결합하는 산술 교차(Arithmetic Crossover) 연산을 통해 차원 공간 상 가상의 내분점 벡터 위치로 자손 입자를 산란시킨다.

\mathbf{x}_{\text{offspring}} = \alpha \mathbf{x}_{\text{parent1}} + (1 - \alpha) \mathbf{x}_{\text{parent2}} \quad (\alpha \in [0,1])

4. 미지의 설계 계곡 탐험: 돌연변이(Mutation) 연산

교배만 반복하게 되면 결국 모집단의 유전자 풀이 초기 군단이 보유했던 특정 대립 유전자로 획일화되어, 수백만 번 세대를 강제 증폭시켜도 로컬 미니멈(Local Minimum) 이외의 신규 지형으로 탈출하지 못하는 조기 수렴(Premature Convergence) 현상이 발발한다. 이를 봉쇄하기 위한 유일한 탈출구가 바로 초-낮은 확률 통제(통상 P_m = 0.01 \sim 0.05) 하에 구동되는 돌연변이(Mutation) 필터이다.

돌연변이는 알고리즘의 탐색(Exploration) 기능을 담당하며 개별 유전 인자를 임의의 뜬금없는 제3의 파라미터로 무작위 변경시켜 개체를 미지의 허공 좌표로 점프하게 만든다. 실수형 체계의 가우시안 돌연변이(Gaussian Mutation)는 다음과 같이 정규 분포를 차용해 변이를 충격한다.

x_i^{\text{new}} = x_i + \Delta x_i, \quad \Delta x_i \sim \mathcal{N}(0, \sigma^2)

선택, 교차, 돌연변이 변조의 연동을 단일 시퀀스로 조망하면 다음과 같은 거대한 세대 윤회의 사이클을 확보한다.

유전 알고리즘 설계에서 위 세 가지 연산에 부여되는 계수 제어(Parameter Tuning)는 통일장 이론급의 난이도이다. 교차율(P_c)이 너무 높으면 기 확보한 우수 설계가 끊임없이 토막나 파괴되며, 돌연변이율(P_m)이 높으면 설계가 전역 수렴을 망각하고 영원히 무작위 도보(Random Walk)만 전전하게 되는 파국이 발생하므로 세밀한 확률 동역학 텐서의 균형이 공학 최적화의 성공 곡선을 판가름한다.
