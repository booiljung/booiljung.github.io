1.3.2.2.5 다목적 최적화 상충 관계 지표를 분리하는 파레토 전선

단일한 스칼라(Scalar) 목적 함수 하나만을 추종하는 단일 목적 최적화(Single-objective Optimization)와 달리, 현실의 공학 시스템은 언제나 복수의 제어 목표가 치열하게 충돌하는 전장에 놓여 있다. 항공기의 주익 설계에서는 항력(Drag)을 최소화하면서 동시에 양력(Lift)을 극대화해야 하며, 전기 자동차 모터 시스템에서는 가속 토크를 높이면서도 배터리 전력 소모를 최소화해야 한다. 하나를 얻으면 반드시 다른 하나를 희생해야 하는 이러한 상호 배타적 갈등 구조를 트레이드오프(Trade-off) 상충 관계라 칭한다. 여러 목적 함수들을 단일한 점수로 강제 병합하지 않고, 각각의 목적 공간 차원을 완벽하게 분리하여 승자와 패자를 가리는 다목적 최적화(Multi-objective Optimization, MOO)의 핵심 진리인 파레토 전선(Pareto Front)의 수학적 분해 역학을 명확히 규명하라.

1. 다목적 최적화 함수 공간과 지배성(Dominance)의 수학적 논거

임의의 설계 변수 벡터 \mathbf{x} 에 대하여 극소화해야 하는 객관적 다중 함수 체계 \mathbf{F}(\mathbf{x}) = [f_1(\mathbf{x}), f_2(\mathbf{x}), \dots, f_m(\mathbf{x})]^T 를 정의하라.
스칼라 최적화에서는 f(\mathbf{x}) < f(\mathbf{y}) 의 명확한 서열이 존재하지만, 벡터 최적화에서는 어떤 해가 다른 해보다 우월하다는 기준을 결정하기 위해 지배성(Dominance)이라는 복합 관계 논리가 요구된다.

설계 해 \mathbf{x}_1 이 \mathbf{x}_2 를 **엄격하게 지배(Dominate, \mathbf{x}_1 \prec \mathbf{x}_2)**한다고 공표하기 위해서는, 다음 두 가지 절대 조건이 동시 충족되어야 한다.

  1. 모든 목적 함수 i \in \{1, 2, \dots, m\} 에 대하여, \mathbf{x}_1 은 결코 \mathbf{x}_2 보다 나쁘지 않아야 한다. 즉, f_i(\mathbf{x}_1) \le f_i(\mathbf{x}_2) 이다.
  2. 적어도 하나의 목적 함수 j \in \{1, 2, \dots, m\} 에서는 \mathbf{x}_1 이 \mathbf{x}_2 보다 확연히 우수해야 한다. 즉, f_j(\mathbf{x}_1) < f_j(\mathbf{x}_2) 이다.

이 수학적 지배 조건을 만족한다면, 설계자 입장에서 \mathbf{x}_2 는 \mathbf{x}_1 에 비해 어느 한 요소도 특출나지 못한 완벽한 하위 호환 매물에 불과하므로 최적화 도면에서 즉시 삭제 폐기되어야 함을 시사한다.

2. 비지배 해(Non-dominated Solutions) 집합과 파레토 전선의 전개

진화 알고리즘이 탐색한 수많은 모집단(Population) 스웜 내에서 지배 관계를 상호 전수 비교해 보면, 누군가에게 지배당하는 개체(Dominated)와 그 누구에게도 지배당하지 않는 훌륭한 개체(Non-dominated)로 분열된다. 어떤 \mathbf{x}^* \in \text{Feasible Region} 에 대하여 집합 내에 \mathbf{x}^* 를 지배할 수 있는 \mathbf{x} 가 단 하나도 존재하지 않을 때, 이 \mathbf{x}^* 를 **파레토 최적 해(Pareto Optimal Solution)**라 격상시킨다.

이탈리아의 경제학자 빌프레도 파레토(Vilfredo Pareto)의 이름에서 따온 이 명칭은 최적 최전방이라는 유기적 곡면으로 구상된다. 설계 공간 내에서 이러한 속성을 띤 파레토 최적 해들의 전체 집합을 역투영하여 m 차원 목적 함수 공간 상에 플로팅(Plotting)한 경계 곡면 전체를 **파레토 전선(Pareto Front)**이라 정의한다. 파레토 최적성의 수학적 원리는 공학 문헌 “Multiobjective optimization using evolutionary algorithms“를 파이프라인 삼아 전산 툴에 융합되었다. 파레토 전선에 위치한 해들은 각각이 “서로 다른 성향의 궁극적 스위트 스팟“을 의미한다. (어떤 해는 f_1 에만 극단적으로 몰두하고, 또 다른 해는 f_1, f_2 를 균형 있게 취한다.) 그 어떠한 알고리즘도 이 파레토 전선의 안쪽(이상적인 공백 영역)으로의 침투는 수학적으로 불가능하다.

3. 유전 알고리즘 기반 파레토 전선 분리 메커니즘

단일 가중치(Weighted Sum)로 F = w_1 f_1 + w_2 f_2 처럼 여러 목적 함수를 단일식으로 후퇴하게 통합해 미분 최적화 경로를 밟으면 파레토 전선의 볼록(Convex) 부위 일부 해만을 파편적으로 찾거나 비볼록 전선일 경우 수렴조차 불가능해진다. 다중 변수를 개별 축으로 남긴 채 전면적인 다목적 탐색을 강제하는 유전 알고리즘 스웜(MOGA)은 다음과 같은 분리 지표 기반 서열화 흐름을 취한다.

4. 파레토 전선의 공학적 의사 결정(Decision Making) 위상

다목적 최적화 알고리즘의 결론은 단 하나의 스칼라 해 \mathbf{x}_{\text{opt}} 이 아니라, 한 무리의 우수한 선택지들이 선을 이루고 있는 파레토 프론티어(Frontier) 집합 전체를 공학 설계자에게 헌납하는 행위이다.

최적 솔버는 “성능이 가장 뛰어나고 가장 가벼우면서 가격을 0으로 만드는 마법“을 주지 못한다. 대신 파레토 전선을 구축함으로써 “만일 안전 계수를 f_1=2.0 으로 양보한다면, 제조 단가 f_2 를 최대 $5,000 까지 감싸 내릴 수 있는 최저의 트레이드오프 기하 형상은 \mathbf{x}_{\text{pareto}}^{(i)} 이다“라는 가장 합리적이고 논리적인 역학 한계선을 들이민다. 이렇듯 파레토 경계면 탐색은 상충하는 다차원의 물리/경제 스칼라 충돌을 화해 불가능한 벡터 텐서 그대로 유지한 채, 인간의 고등 사유를 개입시켜 타협점을 지정하게 하는 공학 모델링 최고 단계의 조력 메커니즘이다.
