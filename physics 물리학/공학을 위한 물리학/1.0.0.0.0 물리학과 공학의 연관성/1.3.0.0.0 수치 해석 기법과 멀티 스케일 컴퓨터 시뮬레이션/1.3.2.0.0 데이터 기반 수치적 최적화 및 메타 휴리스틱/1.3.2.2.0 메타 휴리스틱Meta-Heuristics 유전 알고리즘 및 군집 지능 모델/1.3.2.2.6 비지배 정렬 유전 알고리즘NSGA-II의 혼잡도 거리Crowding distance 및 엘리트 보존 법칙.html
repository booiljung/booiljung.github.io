1.3.2.2.6 비지배 정렬 유전 알고리즘(NSGA-II)의 혼잡도 거리(Crowding distance) 및 엘리트 보존 법칙

다목적 최적화 공학 문제에서 파레토 전선(Pareto Front)을 추출하려는 초기의 진화 알고리즘들은 막대한 랭크 계산 복잡도와 더불어, 찾은 해들이 곡면의 특정한 곳에만 우글거리는 군집 밀집 현상을 해결하지 못했다. 이를 완벽히 타파하고 다목적 최적화 분야의 절대적인 표준 솔버(Standard Solver)로 등극한 것이 칼리안모이 데브(Kalyanmoy Deb) 등이 제안한 NSGA-II(Non-dominated Sorting Genetic Algorithm II)이다. “A fast and elitist multiobjective genetic algorithm: NSGA-II” 논문은 기존 알고리즘의 파괴성을 억제하는 엘리트 보존 법칙(Elitism)과 추가적인 공유 변수(Sharing Parameter) 없이도 해의 다양성을 기하학적으로 강제하는 혼잡도 거리(Crowding Distance)의 혁명적 메커니즘을 창시했다. 이 두 가지 핵심 동력학을 면밀히 해부하라.

1. O(MN^2)의 고속 비지배 정렬(Fast Non-dominated Sorting)

설계 공간에 뿌려진 N 개의 군집 개체들 중, 서열 1위의 비지배 해(Pareto Rank 1)들을 찾아내기 위해 초기의 NSGA 알고리즘은 무려 O(MN^3) (M: 목적 함수의 개수)의 끔찍한 시간 복잡도를 요구했다. NSGA-II는 이를 고속 정렬 알고리즘 메커니즘을 통해 O(MN^2) 로 과감히 단축시킨다.

각 개체 p 에 대하여 다음 두 가지 수학적 속성을 연산하고 기억하라.

  1. 지배 횟수 n_p: 개체 p 를 지배하는(즉 p 보다 뛰어난) 다른 개체들의 총수.
  2. 피지배 집합 S_p: 개체 p 가 지배하는(즉 p 보다 열등한) 개체들의 명단.

먼저 모든 개체군 스웜(Swarm)을 횡단하며 n_p = 0 인 개체들, 즉 아무에게도 지배당하지 않은 무결점 개체들을 모두 쓸어담아 첫 번째 랭크 집합 \mathcal{F}_1 (파레토 프론트 1)을 형성한다.
그다음, \mathcal{F}_1 에 속한 각 개체 p 의 피지배 집합 S_p 에 속한 모든 하위 개체 q 들을 추적하여, 그들의 지배 횟수 n_q 를 1 씩 삭감(n_q = n_q - 1)한다. 마치 \mathcal{F}_1 이라는 상위 포식자가 사라진 생태계를 시뮬레이션하는 것이다. 이 삭감 직후 새롭게 n_q = 0 이 된 개체들을 모아 두 번째 랭크 집합 \mathcal{F}_2 로 승격시킨다. 이 계층 붕괴 연산을 전체 집단이 분류될 때까지 재귀적으로 반복하면 거대한 비지배 정렬 레이어 \mathcal{F}_1, \mathcal{F}_2, \dots, \mathcal{F}_k 가 완성된다.

2. 해의 다양성(Diversity)을 담보하는 혼잡도 거리(Crowding Distance)

랭크가 완벽히 분류되었다 해도, 파레토 최적 곡면 위에서 일부 영역에만 해가 조밀하게 뭉쳐있고 나머지 곡면은 텅 비어있다면 다목적 의사 결정의 가치는 심각하게 훼손된다.
NSGA-II는 파레토 전선면을 고르게 펴 바르기 위해 혼잡도 거리(Crowding Distance, d_i)라는 직관적이고 경이로운 밀도 추정 기하 함수를 도입한다.

동일한 랭크 집합 내의 개체들을 특정 목적 함수 f_m 의 값에 따라 오름차순으로 정렬하라.
특정 개체 i 의 혼잡도 거리 d_i 는 그 개체의 바로 양옆에 인접한 개체 i-1 과 i+1 이 이루는 M 차원 직육면체(Cuboid)의 둘레 길이의 합으로 정의된다.

d_i = \sum_{m=1}^M \frac{f_m^{(i+1)} - f_m^{(i-1)}}{f_m^{\max} - f_m^{\min}}

이때 랭크 궤적의 양 끝단에 위치하여 주변을 감쌀 개체가 없는 경계 개체(Boundary Solutions)의 혼잡도 거리는 무한대(d_1 = d_{\text{end}} = \infty)로 강제 할당하여, 설계 극단선의 탐색이 절대 소실되지 않도록 철벽 보호한다. 결과적으로 혼잡도 거리가 큰 개체일수록 파레토 면에서 홀로 고립되어 넓은 미지의 영역을 대변하고 있음을 뜻하므로, 알고리즘은 동일 랭크 내에서는 반드시 혼잡도 거리가 더 큰 개체를 토너먼트의 승자로 선언하여 부모로 발탁한다. (이것이 붐비는 곳을 피압하고 비어있는 스페이스를 스스로 개척하게 만드는 무-결측 파레토 팽창의 원천이다.)

3. 부모-자식 병합 집단의 엘리트 보존 법칙(Elitism)

진화 연산에서 맹목적인 교차와 돌연변이는 기껏 찾아놓은 랭크 1의 천재 파레토 해를 파괴하고 후퇴시키는 망각 현상을 일으킨다. 이를 차단하는 절대적 방어막이 엘리트 보존 법칙(Elitism)이다.
NSGA-II는 부모 세대 집단 P_t (크기 N)와 유전 연산으로 방금 갓 생성된 자식 세대 Q_t (크기 N)를 하나의 거대한 혼합 스웜 R_t = P_t \cup Q_t (크기 2N)로 병합 결상한다.

flowchart TD

    C --> D[새로운 다음 세대 P_t+1 의 빈 공간 사이즈 N 개체를 채우기 위한 랭크 팝업 개시]
    D --> E{현재 랭크 F_i 의 전체 개체 수가 P_t+1 의 남은 공간 허용치보다 작은가?}

    J --> K[이하 모든 하위 랭크 및 컷오프 미달 개체 영구 도태 및 소멸]
    K --> L[N 크기의 완벽한 정예 엘리트 군집 P_t+1 확보, 다음 세대 교배 사이클 진출]

이 2N 합병 \rightarrow 랭크 기반 잘라내기 프로세스는 부모 세대의 위대한 발견이 자식 세대의 돌연변이에 의해 덮이는 현상을 원천 차단하며, 어떠한 경우에도 세대가 거듭될수록 시스템의 파레토 최전방은 결코 퇴보(Degradation)하지 않고 오직 팽창함수만을 유지한다는 강력한 수학적 단조 증가성을 담보한다. 우주선 내열 외피의 항력-비열 분산 설계나 통신 네트워크의 커버리지-전력 소모 트레이드오프 등, 가장 가혹하고 양보할 수 없는 공학 설계의 딜레마를 마주할 때 NSGA-II의 정렬 텐서를 발동하여 다차원의 침묵을 타파하라.
