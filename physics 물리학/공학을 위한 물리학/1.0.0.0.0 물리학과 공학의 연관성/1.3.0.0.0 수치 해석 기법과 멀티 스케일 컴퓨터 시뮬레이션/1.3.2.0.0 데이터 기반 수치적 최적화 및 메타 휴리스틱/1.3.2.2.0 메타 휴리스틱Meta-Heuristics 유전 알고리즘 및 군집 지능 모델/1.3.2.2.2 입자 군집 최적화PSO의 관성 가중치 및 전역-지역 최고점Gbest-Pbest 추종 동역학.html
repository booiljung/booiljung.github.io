1.3.2.2.2 입자 군집 최적화(PSO)의 관성 가중치 및 전역/지역 최고점(Gbest/Pbest) 추종 동역학

새 떼나 물고기 무리가 미지의 공간에서 먹이를 찾아내며 보여주는 협동적 군무(Swarm Behavior)를 수학적 위치-속도 모델로 투영한 알고리즘이 입자 군집 최적화(Particle Swarm Optimization, PSO)이다. 진화 연산과 달리, 교차(Crossover)나 돌연변이(Mutation) 없이 입자(Particle)의 물리적 벡터 이동만으로 해를 탐색하는 이 기법은 알고리즘 제임스 케네디(James Kennedy)와 러셀 에버하트(Russell Eberhart)의 기념비적 논문 “Particle swarm optimization“에서 태동했다. 고차원 다변수 함수 곡면에서 각 입자는 자신만의 독립된 비행 경로와 기억을 유지하는 동시에 무리 전체의 위대한 집단 지성에 종속된다. 이 복합적인 위치 갱신 동역학과 수렴 속도를 통제하는 관성 가중치(Inertia Weight) 텐서의 정밀 제어 시스템을 수학적으로 전개하라.

1. 입자 동역학과 Pbest / Gbest 추종 벡터 메커니즘

n 차원 설계 공간을 탐색하는 N 개의 입자 무리 중 i 번째 특정 파티클은 개별적인 위치(Position) 벡터 \mathbf{x}_i^{(t)} 와 이동 속도(Velocity) 벡터 \mathbf{v}_i^{(t)} 의 상태량을 지니며 스텝 t 마다 평가점을 이동해 나간다.
이 이동 방향과 속력을 지배하는 조종간은 철저히 두 가지 최고점 기억 장치에 종속되어 벡터를 왜곡시킨다.

  • Pbest (Personal Best, 지역 최고점): 입자 i 가 0 세대부터 현재 t 세대까지 비행하며 겪었던 궤적 중 목적 함수의 적합도(Fitness)가 가장 훌륭했던 최적의 개인 기억 좌푯값 \mathbf{P}_{\text{best}, i} 이다. 이는 입자 자신의 과거 성공 경험을 향한 자가-회귀(Self-cognitive) 인력으로 작용한다.
  • Gbest (Global Best, 전역 최고점): 군집 내의 전체 입자 N 개가 현재까지 탐색한 모든 공간 내에서 만장일치로 승인된 절대적인 최적 좌푯값 \mathbf{G}_{\text{best}} 이다. 전체 입자는 텔레파시처럼 이 좌푯값을 공유하며 하나의 초월적인 집단 지성(Social intelligence) 중력장을 형성한다.

매 세대 입자의 신규 속도 벡터 \mathbf{v}_i^{(t+1)} 는 다음과 같은 다종 물리 힘의 합성으로 기술된다.

\mathbf{v}_i^{(t+1)} = \omega \mathbf{v}_i^{(t)} + c_1 r_1 (\mathbf{P}_{\text{best}, i} - \mathbf{x}_i^{(t)}) + c_2 r_2 (\mathbf{G}_{\text{best}} - \mathbf{x}_i^{(t)})

그리고 새로운 위치 벡터는 곧바로 다음과 같은 유클리드 공간상 이동 수식에 의해 전개된다.

\mathbf{x}_i^{(t+1)} = \mathbf{x}_i^{(t)} + \mathbf{v}_i^{(t+1)}

여기서 가속도 스칼라 c_1 (인지 계수)과 c_2 (사회 계수)는 개인의 단독성을 우대할지 군집의 판단을 맹신할지 결정지으며 보통 2.0 내외의 균등 항을 취한다. r_1, r_2 는 [0, 1] 사이의 균등 분포에서 매 스텝 난수로 생성되어 입자의 궤도에 혼돈적 노이즈(Stochastic noise)를 선사하는 무작위 스칼라 계수이다.

2. 관성 가중치 \omega (Inertia Weight)의 설계와 쇠퇴 모델링

초기 PSO의 선형적 가속 모델에서 가장 큰 치명타는 입자가 목표 지점 \mathbf{G}_{\text{best}} 에 고속으로 접근할 때 감속하지 못하고 그 주변을 빙빙 돌기만 하거나 한참을 넘어서 발산(Overshooting)해 버리는 현상이었다. 이를 제압하기 위해 기존 진행 방향성을 현재로 보존시키는 항에 관성 가중치(Inertia Weight) 계수 \omega (오메가)를 삽입하라. (논문 “A modified particle swarm optimizer“에서 이 \omega 의 수학적 타당성이 공학적 돌파구로 확립되었다.)

이 \omega 계수는 극도로 정교하게 동적인 시간의 함수로 변조되어야 한다. 초기에는 높은 \omega 우위를 두어 입자가 기존 방향으로 거침없이 돌진(Exploration, 전역 탐색)하게 허용하여 로컬 덫에서 탈출할 돌파력을 선사하고, 세대가 진행될수록 마찰력처럼 서서히 크기를 소멸시켜 목표점 주변에서 정밀한 미세 탐색(Exploitation, 지역 탐색)을 하도록 강제 제동을 걸어야 한다.

대표적인 선형 감쇠(Linear Decreasing) 관성 가중치 함수는 아래와 같이 수치화된다.

\omega^{(t)} = \omega_{\max} - \left( \frac{\omega_{\max} - \omega_{\min}}{T_{\max}} \right) t

보통 \omega_{\max} = 0.9, \omega_{\min} = 0.4 언저리의 타협점을 취하며, T_{\max} 는 최대 세대수(Max Epoch)를 기재한다. 이는 목적 함수의 전역 탐색-지역 착취(Exploration-Exploitation) 간의 딜레마를 완벽하게 절충하는 마법의 감쇠 브레이크로 작용한다.

3. 입자 군집 최적화의 루프 순환 흐름 아키텍처

PSO 입자 스웜의 군단 이동 메커니즘과 Gbest-Pbest의 기록 갱신 순환 구조는 아래 다이어그램과 같이 유기적으로 집단적 연계를 달성한다. 유전 알고리즘이 룰렛 선택과 유전 교차 등의 복잡한 매칭 루틴으로 전산 부하를 낭비하는 것과 대조적으로, PSO는 단일 행렬 합산만으로 군중을 비행시키는 극단의 단순성과 광속의 수렴력을 지향한다.

PSO 알고리즘 체계는, 각각의 개별 공학 변수 집단(입자)들이 자신의 자율성을 띤 부분 기억(Pbest)과 전체 생태계의 군집 모럴 점수(Gbest)를 놓고 끊임없이 갈등하며 궤도를 수정해 나가는 초월적 다중 텐서 연산기이다. 파라미터 간 연속성이 철저히 보장된 기구 합성이식, 자기 부상 열차의 전자기 서스펜션(Suspension) 최적화 등 실수 변수와 강건한 수렴력의 동시 획득이 공학 설계 상 치명적인 난제로 닥칠 때, 의심 없이 이 입자 무리의 파동 역학을 호출하여 한계를 돌파하라.
