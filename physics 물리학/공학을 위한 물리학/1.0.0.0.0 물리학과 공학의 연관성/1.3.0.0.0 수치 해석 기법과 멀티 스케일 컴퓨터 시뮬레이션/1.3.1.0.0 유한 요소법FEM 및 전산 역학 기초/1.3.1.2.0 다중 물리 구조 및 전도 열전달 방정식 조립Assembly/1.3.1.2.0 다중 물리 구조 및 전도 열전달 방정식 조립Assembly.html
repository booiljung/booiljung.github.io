1.3.1.2 다중 물리 구조 및 전도 열전달 방정식 조립(Assembly)

모든 개별 유한 요소(Finite Element)가 각자의 고립된 우주 속에서 요소 강성 매트릭스(\mathbf{k}^{(e)})와 하중 벡터(\mathbf{f}^{(e)})를 토해냈다면, 이제 이 파편화된 조각들을 물리적 현실의 거대한 시스템으로 결속시킬 차례다. 유한 요소법(FEM)의 극적인 하이라이트는 이 수백만 개의 국부 행렬(Local Matrix)들을 전역 좌표계의 캔버스 위에 하나의 초거대 연립 방정식으로 꿰매는 공간 조립(Assembly) 과정에 있다.

이 조립의 철학은 놀랍게도 단일 물리학(Single-physics)의 경계를 가볍게 뛰어넘는다. 고체 역학에서의 ’변위(Displacement)’와 ’힘(Force)’의 관계를 엮던 구조 강성 방정식은, 온도장(Temperature Field)을 해석하는 열역학에서는 ’온도(Temperature)’와 ’열 유속(Heat flux)’의 관계를 엮는 전도 열전달 방정식으로 그 기호와 의미만 바뀔 뿐, 수학적 적통의 뼈대는 완전히 동일한 대수 매트릭스의 결합으로 취급된다. 즉, 컴퓨터에게 있어 철판이 휘어지는 문제나 철판이 뜨겁게 달아오르는 문제는 동일한 라플라스(Laplace) 및 푸아송(Poisson) 편미분 방정식의 조립(Assembly) 연산일 뿐이다.

본 장에서는 고립된 요소 단위의 지배 방정식을 구조망 전체의 전역 매트릭스(Global Matrix, \mathbf{K}) 로 압축 확장하는 병합 메커니즘을 규명하고, 변위장과 온도장이라는 두 가지 다른 물리계가 메쉬(Mesh)라는 단일 전산망 위에서 어떻게 교차 융합(Multiphysics Coupling)되는지 그 근본적인 조립 역학을 조망한다.

1. 지배 방정식의 쌍대성: 기계적 저항과 열적 저항의 대칭

요소를 지배하는 갤러킨(Galerkin) 약형식(Weak Form)은 물리적 도메인이 바뀌어도 그 텐서의 형태를 유지한다. 단일 요소 내에서 성립하는 행렬 방정식은 다음과 같은 철저한 쌍대성(Duality)을 띤다.

1.1 ) 구조 탄성 역학의 지배 방정식

\mathbf{k}^{(e)} \mathbf{d}^{(e)} = \mathbf{f}^{(e)}

  • \mathbf{k}^{(e)} : 요소 구조 강성 행렬 (단위 변위를 일으키는 데 필요한 물리적 저항력 텐서)
  • \mathbf{d}^{(e)} : 요소 절점 변위 벡터 (구조가 움츠러들거나 늘어나는 DOF)
  • \mathbf{f}^{(e)} : 요소 절점 외력 벡터 (외부에서 가해지는 물리적 하중)

2) 전도 열전달(Heat Conduction)의 지배 방정식

\mathbf{k}_T^{(e)} \mathbf{T}^{(e)} = \mathbf{q}^{(e)}

  • \mathbf{k}_T^{(e)} : 요소 열전도 행렬 (단위 온도차를 발생시키는 열 저항/전도도 텐서)
  • \mathbf{T}^{(e)} : 요소 절점 온도 벡터 (열역학적 스칼라 DOF)
  • \mathbf{q}^{(e)} : 요소 절점 열 유속/열원 벡터 (외부에서 주입되는 열에너지)

구조 역학의 변위가 벡터(Vector)인 반면 온도는 스칼라(Scalar)라는 차이가 존재하지만, 공간 적분망(가우스 구적법)을 거쳐 매트릭스를 도출해 내는 산술적 메인 루틴은 치명적으로 동일하다. 이러한 수학적 대칭성 덕분에, 현대의 해석 솔버들은 구조 역학 코드를 약간만 변형하여 무한한 열전달 해석모듈로 치환전개하는 전산학적 우위를 점할 수 있다.

2. 전역 행렬(Global Matrix)의 조립: 토폴로지 맵핑(Topology Mapping)

이 고립된 \mathbf{k}^{(e)} 매트릭스들은 서로 인접한 요소들과 절점(Node)을 공유하며 맞닿아 있다. 조립(Assembly)이란 어떤 임의의 전역 절점(Global Node) I 에 연결된 모든 요소들이 내뿜는 저항(강성) 텐서를 그 절점 위치의 주소에 차곡차곡 단순 합산으로 누적(Superposition)시키는 대수학적 덧셈 공정이다.

\mathbf{K} = \sum_{e=1}^{N_e} \mathbf{k}^{(e)}, \quad \mathbf{F} = \sum_{e=1}^{N_e} \mathbf{f}^{(e)}

이때 1번 요소의 3번 로컬(Local) 절점이 전역 메쉬 시스템에서는 1532번 절점에 해당한다면, 1번 요소행렬의 (3,3) 성분은 전역 형렬 \mathbf{K} 의 (1532, 1532) 주소지에 타격 병합되어야 한다. 이 주소록을 매칭해 주는 배열을 연결성 행렬(Connectivity Matrix), 혹은 스티어링 벡터(Steering Vector)라 부르며, 수천만 개의 텐서가 정확한 배열의 제자리를 찾아 들어가도록 교통정리를 수행한다.

3. 다중 물리(Multiphysics)의 연성 조립(Coupled Assembly)

현대 공학은 단일 물리에 만족하지 않는다. 위성이 태양열을 받아 한쪽 면이 극도로 팽창하며 안테나 구조를 뒤틀어버리는 열탄성(Thermoelasticity) 문제, 혹은 전기 차량의 배터리가 발열하여 내부 압력을 폭발시키는 현상들은 구조와 열이 하나로 얽힌 다중 물리 공간이다.

이러한 연성(Coupled) 해석 시, 솔버의 조립망은 두 개의 지배 방정식을 하나로 묶어 거대한 비대칭 초대형 행렬(Monolithic System)을 직조하거나, 순차적으로 해를 주고받는 스태거링(Staggered) 기법으로 진화한다.

이렇듯 Assembly 공정은 조그만 모래알 같은 유한 요소들을 강력한 물리적 사슬로 묶어 현실의 형체를 부여하는 전산 역학의 연금술이다. 이후 완성된 전역 강성 행렬 \mathbf{K} 는 동적, 정적, 혹은 비선형 특성에 따라 분해(Decomposition) 솔버의 도마 위로 올라가 극한의 역행렬 해체 과정을 맞이하게 될 것이다.
