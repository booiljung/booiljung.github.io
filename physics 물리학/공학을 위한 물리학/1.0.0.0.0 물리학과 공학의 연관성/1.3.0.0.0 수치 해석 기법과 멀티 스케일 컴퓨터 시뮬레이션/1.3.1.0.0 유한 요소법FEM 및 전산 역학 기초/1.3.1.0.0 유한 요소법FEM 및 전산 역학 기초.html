1.3.1 유한 요소법(FEM) 및 전산 역학 기초

항공기의 기체 프레임, 교량의 미세한 균열, 충돌하는 자동차의 찌그러진 프레임 등 현실 세계의 기하학적 형상은 미적분학의 해석적 해(Analytical Solution)가 접근조차 불가능할 정도로 극도로 기괴하고 복잡하다. 공학자들은 이 거대하고 난해한 연속체(Continuum)를 한 번에 해석하려는 오만을 버리고, 시스템을 수학적으로 해석 가능한 무수히 많은 작고 단순한 조각—요소(Element)—들로 찢어발겨 분석한 뒤 다시 하나로 꿰어 맞추는 우회 전술을 창안했다. 이것이 현대 고체 역학과 구조 해석의 가장 강력한 지배 도구인 **유한 요소법(FEM: Finite Element Method)**이다. 미분 방정식을 대수 방정식 행렬(\mathbf{K}\mathbf{u} = \mathbf{f})로 강제 치환하는 이 위대한 전산 역학(Computational Mechanics)의 기본 철학과 수학적 붕괴 메커니즘을 아래와 같이 선포한다.

1. 연속체의 절단: 요소(Element)와 절점(Node)의 그리드망

자연계의 물질은 무한대의 자유도(Degrees of Freedom, DOF)를 가진다. FEM은 이 무한의 우주를 유한한 시스템으로 강제 압축하기 위해 구조물을 삼각형, 사각형, 혹은 3차원 사면체(Tetrahedron)나 육면체(Hexahedron) 같은 **유한 요소(Finite Element)**들로 토막 낸다. 이 과정이 바로 기하학적 메쉬(Mesh) 생성이다.

이 잘게 쪼개진 각각의 요소들은 서로 허공에 떠 있는 것이 아니라, 모서리의 꼭짓점인 **절점(Node)**이라는 핵심 링크를 통해 서로 단단히 결속된다. 수치 해석기는 구조물 전체의 무한한 변위(Displacement) 분포를 계산하지 않는다. 오직 이 한정된 절점들의 변위 움직임만을 추적하며, 요소 내부의 빈 공간은 인접한 절점값들을 가중 평균 내어 추정하는 형상 함수(Shape Function) 혹은 보간 함수(Interpolation Function)라는 수학적 거미줄로 채워버린다.

2. 형상 함수와 가상 일의 원리(Principle of Virtual Work)

물리적 미분 방정식(탄성 방정식 등)을 하나의 요소에 적용하기 위해 갤러킨법(Galerkin Method)이나 가상 일의 원리 등 텐서 변분법(Calculus of Variations)의 칼날을 정조준한다.

어떤 단일 요소(Element e) 내부에서, 변위 벡터 \mathbf{u}^{(e)} 와 작용하는 내/외부 에너지의 총합이 물리적 평형을 이루어야 한다는 에너지 최소화 원칙을 투하하라.
\int_{V^{(e)}} \delta \mathbf{\epsilon}^T \mathbf{\sigma} dV = \int_{V^{(e)}} \delta \mathbf{u}^T \mathbf{b} dV + \int_{S^{(e)}} \delta \mathbf{u}^T \mathbf{t} dS
(여기서 \delta \mathbf{u}는 가상 변위, \mathbf{\epsilon}은 변형률 텐서, \mathbf{\sigma}는 응력 텐서, \mathbf{b}, \mathbf{t}는 각 체적력과 표면력이다.)

여기에 요소 내부의 변위(\mathbf{u})를 절점 변위(\mathbf{d})와 형상 함수 행렬(\mathbf{N})의 조합인 \mathbf{u} = \mathbf{N}\mathbf{d} 로 박아넣고, 변형률 \mathbf{\epsilon} = \mathbf{B}\mathbf{d} (기하 행렬 \mathbf{B} 사용)와 훅의 법칙 \mathbf{\sigma} = \mathbf{E}\mathbf{\epsilon} (물성 행렬 \mathbf{E} 사용)을 모조리 대입하여 식을 짓이기면 적분식은 기하학적 상수를 담은 거대 행렬로 응결된다.

\left[ \int_{V^{(e)}} \mathbf{B}^T \mathbf{E} \mathbf{B} dV \right] \mathbf{d}^{(e)} = \mathbf{f}^{(e)}

이 수식의 좌측 공간을 지배하는 극악한 적분 텐서 덩어리가 바로 해당 단일 요소가 보유한 고유의 뼈대, 요소 강성 행렬(Element Stiffness Matrix) \mathbf{k}^{(e)} 이다.

3. 우주 어셈블리(Assembly)와 전역 강성 행렬(Global Stiffness Matrix) 결착

이제 분해되었던 수백만 개의 요소 파편들을 다시 하나의 우주로 결합해야 한다. 전체 시스템의 지배 방정식은 개별 \mathbf{k}^{(e)} 들을 전체 시스템의 자유도 넘버링 트리에 맞추어 거대한 캔버스 위로 덧셈 누적(Superposition)시키는 어셈블리(Assembly) 연산을 거쳐 탄생한다.

\mathbf{K} = \sum_{e=1}^{N_{elem}} \mathbf{k}^{(e)}, \quad \mathbf{F} = \sum_{e=1}^{N_{elem}} \mathbf{f}^{(e)}

이 결합의 결과물로 컴퓨터 메모리에는 엄청난 크기의 연립 1차 대수 방정식 행렬이 폭발적으로 압착된다.
\mathbf{K} \mathbf{U} = \mathbf{F}

  • \mathbf{K}: 전역 강성 행렬 (Global Stiffness Matrix). 전체 구조물의 저항력을 담은 초거대 스파스(Sparse) 대칭 텐서.
  • \mathbf{U}: 우리가 알아내야 할 전역 미지 절점 변위 벡터 (Global Nodal Displacement Vector).
  • \mathbf{F}: 시스템에 인가된 전역 하중 벡터 (Global Force Vector).

4. 특이점의 파괴와 해의 복원

최초로 조립된 전역 강성 행렬 \mathbf{K} 는 우주 공간에 둥둥 떠 있는 물체와 같아 \det(\mathbf{K}) = 0 인 특이 행렬(Singular Matrix) 상태에 머물러 역행렬 연산이 불가능하다. 여기에 지지점이나 볼트 축 등의 **경계 조건(Boundary Conditions)**을 억지로 쑤셔 넣어 행렬망의 불안정성을 강제로 고정시켜야 비로소 행렬 \mathbf{K} 는 양의 정부호성(Positive Definite)을 쟁취하며 해석 솔버의 타격을 받을 준비를 끝마친다.

슈퍼컴퓨터 알고리즘이 \mathbf{U} = \mathbf{K}^{-1} \mathbf{F} 라는 거대한 역산을 관통해 수백만 개의 절점 변위(Displacement) 해를 토해내면, 공학자는 다시 보간망의 거미줄(B행렬과 E행렬)을 역주행하여 절점이 아닌 구조물 전 영역에 걸친 치명적인 응력장(Stress Field) 텐서를 컴퓨터 화면 위 극채색의 등고선 맵으로 화려하게 복원해 낸다. 방정식의 패배를 기하학의 분할력으로 극복한 이것이 바로 유한 요소법의 기적이다.
