1.3.1.1 공학적 유한 요소(FEM) 메쉬 생성과 형상(Shape) 함수 전개

유한 요소법(FEM)의 절대적인 정밀도와 수치적 성패는 지배 방정식을 풀기도 전에, 기하학적 연속체를 도대체 어떻게 찢어발겼는가 하는 **메쉬(Mesh)**의 품질과 요소 내부를 지배하는 보간망인 **형상 함수(Shape Function)**의 기하학적 차수에 따라 90% 이상 결정된다. 공학자는 설계 도면의 연속적인 공간 속에 칼을 들이대어 1D 선(Lines), 2D 형판(Plates/Shells), 그리고 3D 고체(Solid) 덩어리를 각각의 특성에 맞는 유한 요소들로 분할하는 해체 쇼를 벌인다. 본 절에서는 연속체를 무자비하게 이산화(Discretization)하는 요소망 생성 기법의 본질과, 이렇게 찢어진 조각들 사이의 텅 빈 공간을 수학적 거미줄로 직조하여 물리적 텐서를 연속적으로 이어 붙이는 형상 함수의 전개 공식을 해부한다.

1. 차원 강하와 기하학적 해체: 유한 요소망(Mesh) 생성 체계

세상의 모든 프레임이나 구조물이 3D 육면체로 이루어진 것은 아니다. 연산 자원은 극도로 제한되어 있기에, 공학자는 기하학적 형상의 두께나 길이 비율을 판독하여 불필요한 차원 공간을 강제로 짓눌러버리는(Dimension Reduction) 차원 강하 전술을 감행한다.

  1. 1D 선 요소 (Truss / Beam Elements): 철탑이나 교량의 뼈대처럼, 단면적 공간에 비해 길이가 압도적으로 긴 부재들은 3차원의 스페이스를 모조리 버리고 오직 1차원 선(Line)으로 붕괴된다. 축 방향 인장과 압축만을 견디는 트러스(Truss)와 굽힘 모멘트(Bending Moment)까지 수용하는 보(Beam) 요소가 이에 해당한다.
  2. 2D 평면 요소 (Plane Stress / Plane Strain / Shell): 압력 용기의 표면이나 얇은 항공기 외피처럼, 한쪽 방향의 두께가 극도로 얇거나(평면 응력, Plane Stress) 무한히 긴 댐과 같이 횡방향 변형이 통제된(평면 변형률, Plane Strain) 경우 2차원 공간으로 압착된다. 이들은 주로 3개의 절점을 가진 삼각형(CST, LST) 요소나 4개 절점의 사각형(Quadrilateral) 요소로 분할된다.
  3. 3D 입체 요소 (Solid Elements): 엔진 블록이나 구형 베어링처럼 어떤 차원으로도 찌그러뜨릴 수 없는 거대한 덩어리는, 가장 무거운 전산 부하를 요구하는 3D 요소망을 짜야 한다. 4절점 사면체(Tetrahedron)나 8절점 육면체(Hexahedron/Brick) 요소가 동원되어 기하학적 체적(Volume) 공간을 빼곡히 채워 넣는다.

메쉬를 생성할 때 응력(Stress)이 극심하게 집중되는 구멍 주변이나 날카로운 모서리 구석에는 요소망을 극도로 촘촘하게 박아 넣는 세밀화(Refinement) 작업이 필수적이며, 이를 무시하면 응집된 파괴 텐서망을 영원히 찾아낼 수 없다.

2. 빈 공간의 지배자: 형상 함수(Shape Function)의 가중 보간망

요소망이 격자 절점(Node)들로 공간을 찢어놓았지만, 유한 요소법은 컴퓨터가 오직 ’절점’에서만의 변위(\mathbf{u}_i)만을 계산하는 극단적 이산화 시스템이다. 그렇다면 절점과 절점 사이의 텅 빈 공간 한가운데 박혀있는 입자의 변형률이나 응력은 도대체 어떻게 도출하는가? 공학자는 이 미지의 공간에 형상 함수(Shape Function, \mathbf{N}) 라는 이름의 마법의 보간 융단(Interpolation Matrix)을 깔아버린다.

어떤 단일 요소 내부 임의의 좌표 (x, y) 에서의 실제 변위 \mathbf{u}(x, y) 는, 그 요소를 둘러싼 각 절점들의 변위 벡터 배열 \mathbf{d}^e = [u_1, u_2, \dots , u_n]^T 와 각각의 절점에 할당된 형상 함수 N_i(x,y) 들의 가중 선형 조합(Weighted Linear Combination)으로 완벽하게 조립된다.

\mathbf{u}(x, y) = \sum_{i=1}^{n} N_i(x, y) \mathbf{u}_i = \mathbf{N} \mathbf{d}^e

형상 함수 N_i 는 철저하고도 절대적인 수학적 율법을 따른다.

  • 크로네커 델타 법칙 (Kronecker Delta Property): 함수 N_i 는 자신의 고향인 i 번째 절점 좌표 위에서는 그 값이 정확히 1 이 되어 변위를 100\% 지배(투영)하지만, 자신과 상관없는 다른 모든 j 절점 좌표 위에서는 그 값이 가차 없이 0 으로 소멸한다. (N_i(x_j, y_j) = \delta_{ij})
  • 강체 이동 보전 (Rigid Body Mode): 요소 내 어떤 위치에서든 모든 형상 함수들의 합은 완벽하게 1 이 되어야 한다. (\sum N_i = 1). 이로써 요소 전체가 한꺼번에 이동할 때 내부 응력이 0 이 되는 강체 거동을 파괴하지 않고 보존해 낸다.

3. 선형(Linear)과 고차(Higher-Order) 다항식의 전쟁

형상 함수의 차수(Polynomial Degree)는 모델의 유연성을 결정짓는 무기다.
예를 들어 2D 삼각형의 각 꼭짓점(절점 3개)만을 이용해 변위를 예측하는 **1차 선형 요소(Linear Element)**는 상수 성분의 변형률만을 표현할 수 있다. 이는 계산이 비약적으로 빠르나, 구조물이 구부러질 때 응력의 그라데이션을 표현하지 못해 시스템이 실제보다 더 가공할 정도로 뻣뻣해지는 ’전단 잠김(Shear Locking)’이라는 저주에 빠지기 일쑤다.

이에 맞서 공학자는 테두리 한가운데 중간 절점(Mid-node)들을 추가로 박아 넣고 2차(Quadratic) 다항식의 포물선 곡면 망을 펼쳐내는 고차 요소(Higher-Order Element) 전술을 구사한다. 전산 부하량이 제곱으로 폭증하지만, 응력의 정교한 굽힘 곡률을 포착해 내고 요소의 개수를 무리하게 늘리지 않고도 해석적 정해(Exact Solution)에 기하급수적으로 수렴하는 절대적 해법이 된다.
