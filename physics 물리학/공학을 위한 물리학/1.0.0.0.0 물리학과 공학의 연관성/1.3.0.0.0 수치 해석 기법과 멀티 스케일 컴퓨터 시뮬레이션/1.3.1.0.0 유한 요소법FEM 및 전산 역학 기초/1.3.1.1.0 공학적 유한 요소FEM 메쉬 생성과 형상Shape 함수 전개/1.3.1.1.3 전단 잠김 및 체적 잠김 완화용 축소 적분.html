1.3.1.1.3 전단 잠김 및 체적 잠김 완화용 축소 적분

컴퓨터의 무한한 정밀도를 맹신하며 가우스 구적법(Gauss Quadrature)의 적분점 갯수를 무비판적으로 늘려 ’완전 적분(Full Integration)’을 수행하는 것은, 때때로 기계 부품 시뮬레이션을 파멸로 이끈다. 특히 1차 선형(Linear) 다항식 베이스의 저차 요소들이 얇은 판재의 굽힘(Bending)이나 고무와 같은 비압축성(Incompressible) 물성을 마주할 때, 유한 요소망은 수학적 모순에 빠져 실제 물리적 강성보다 수백 배 이상 비정상적으로 뻣뻣해지는 끔찍한 이상 고착 현상을 일으킨다. 이를 유한 요소법의 가장 치명적인 병리 현상인 **잠김(Locking)**이라고 부른다. 본 절에서는 기하학적 파탄을 야기하는 **전단 잠김(Shear Locking)**과 물성적 파탄인 **체적 잠김(Volumetric Locking)**의 메커니즘을 낱낱이 해부하고, 수학적 정확도를 일부러 훼손시킴으로써 공학적 진실을 구원해 내는 마법과도 같은 축소 적분(Reduced Integration) 텐서 타격법을 선포한다.

1. 전단 잠김(Shear Locking): 형상 차수의 결핍과 기성 전단 변형률의 저주

티모셴코 보(Timoshenko Beam)나 민들린-라이스너 평판(Mindlin-Reissner Plate) 요소처럼, 두께 단면이 변형 후에도 직각을 유지할 필요 없이 독립적인 전단(Shear) 변형을 허용하는 요소들을 1차 선형 보간(Bilinear Interpolation)으로 엮었다고 가정하라.

판재가 외부의 굽힘 모멘트를 받아 둥글게 휘어지려 할 때, 1차 선형 요소의 직선 테두리는 결코 부드러운 곡률을 표현할 수 없다. 굽힘 변형을 모사하기 위해 변위 구배를 억지로 꺾는 순간, 선형 다항식은 요소를 순수하게 구부리지 못하고 평행사변형으로 비틀어버리는 ’기성 전단 변형률(Parasitic Shear Strain)’을 가짜로 발생시킨다.
결말은 참혹하다. 굽힘 에너지로 소비되어야 할 하중이 존재하지도 않는 허구의 전단 에너지를 극복하는 데 모조리 소모되면서, 컴퓨터는 이 철판이 마치 티타늄 통뼈처럼 어마어마하게 뻣뻣하다고 착각하게 된다. 수학적 완전 적분(Full Integration, 2 \times 2 가우스 점)을 거친 그 결점 없는 미적분적 완벽함이, 역설적으로 요소를 돌덩이처럼 묶어버리는 전단 잠김(Shear Locking) 현상을 폭발시킨 것이다.

2. 체적 잠김(Volumetric Locking): 비압축성의 감옥과 체적 팽창 행렬의 폭주

전단 잠김이 기하학적 굽힘의 파탄이라면, **체적 잠김(Volumetric Locking)**은 포아송 비(Poisson’s Ratio) \nu \to 0.5 로 향하는 고분자 고무나 소성(Plasticity) 유동 상태의 비압축성 초탄성체(Hyperelastic Material)에서 발현되는 물성적 파탄이다.

비압축성 재료는 힘을 받을 때 모양은 찌그러지더라도 내부 적분 체적, 즉 \nabla \cdot \mathbf{u} = \epsilon_x + \epsilon_y + \epsilon_z = 0 이라는 절대적인 체적 불변의 계율을 지킨다. 이를 유한 요소 강성 행렬식에 주입하면 체적 탄성 계수(Bulk Modulus) K = \frac{E}{3(1-2\nu)} 가 분모의 0 수렴으로 인해 무한대(\infty)로 폭주해 버린다.
이 끔찍한 구속 조건식을 품은 채 선형 솔리드 요소망을 완전 적분으로 짓누르게 되면, 체력을 아끼며 유연하게 모양을 바꿔야 할 요소망 전체의 절점들이 체적 불변의 법칙에 갇혀 한 치의 찌그러짐조차 허용하지 못하고 완전히 굳어버린다. 물성이 요소를 감옥에 가둬버리는 이 극단적 체적 잠김 사태는 막대한 수리적 비선형 발산을 일으킨다.

3. 독을 독으로 제압하라: 축소 적분(Reduced Integration)과 아워글라스(Hourglass) 파괴

놀랍게도 선진 공학자들은 이 수리적 파탄을 해결하기 위해, 다항식의 차수를 높이는 무식하고 값비싼 연산 투입을 버리고 오히려 “적분점의 갯수를 일부러 깎아내려 수학적 정확도를 박살 내는” 역발상을 창안했다. 이것이 바로 **축소 적분(Reduced Integration)**이다.

2D 사각형 요소의 완전 적분이 2 \times 2 (4개의 가우스 점)라면, 공학자는 이를 정중앙 1개의 가우스 점(1 Point Quadrature)으로 강제 격하해 버린다.
놀라운 속임수가 발동한다! 정중앙 1개의 포인트만 타격할 경우, 기생적으로 발생하던 가짜 잔여 전단 변형률(Parasitic Shear)과 폭주하던 체적 변형률의 왜곡된 파형이 적분점 위치에서 정확히 0 으로 교차 소멸해 버린다. 즉, 시스템에서 쓸데없이 뻣뻣하게 저항하던 가짜 강성 행렬이 컴퓨터의 스캐닝 라인에서 완벽하게 은폐 삭제되며, 구조물은 원래 지녔어야 할 본연의 유연함을 되찾게 된다.

그러나 수학을 훼손한 대가는 가혹하게 뒤따른다. 적분점을 헐겁게 잡은 탓에 요소가 변형을 일으키고 있음에도 정중앙의 가우스 점에서는 변형률 에너지가 0 으로 감지되는 가짜 불능 상태(Zero-energy Mode)가 탄생한다. 그 결과 변형 메쉬 전체가 지그재그 모양으로 무너져 내리는 모래시계(Hourglass) 현상이 구조망을 찢어발기게 된다.

자연의 완전한 비선형을 불완전한 1차 컴퓨터 다항식으로 가둔 대가가 ’Locking’이라면, 수학적 완결성을 고의로 망가뜨려 물리적 정해를 낚아채는 ’축소 적분’은 고체 역학과 전산 대수학이 타협을 통해 빚어낸 궁극의 실전형 페널티 제어 텐서이다.
