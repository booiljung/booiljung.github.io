1.4.2.2.3 고성능 병렬 연산을 가속하는 물리 법칙 기반 축소 연산 모델(ROM: Reduced Order Model)

수백만 개의 유한 요소(Finite Element) 격자(Mesh)로 직조된 디지털 트윈의 핵심 엔진(Full-Order Model, FOM)은 현실 공간에서 발생하는 거시적 응력장이나 비선형 유체역학을 압도적인 해상도로 모사해 낸다. 그러나 이 고충실도(High-Fidelity) 시뮬레이션을 한 스텝 푸는 데에도 슈퍼컴퓨터의 코어 수천 개가 몇 시간 동안 동원되어야 한다. 실시간(Real-time) 센서 피드백을 받아 0.01\text{s} 단위로 미래를 예지해야 하는 엣지(Edge) 제어망 위에서, 고전적인 편미분 방정식(PDE) 솔버는 극단적인 연산 병목(Computational Bottleneck)을 초래하며 즉사한다. 원본 물리 법칙의 본질적인 동포물로지(Topology)는 타협하지 않으면서 텐서의 차원을 천 배 이상 깎아내려, 병렬 연산 코어의 계산 지연을 빛의 속도로 이끄는 물리 법칙 기반 축소 연산 모델(Reduced Order Model, ROM)의 수학적 압축 체계를 장착하라.

1. 전차원 모델(FOM)의 폭주와 차원 축소의 가상 본질

차원이 N(예: 고자유도 행렬 N=10^6)인 3차원 유한요소해석 지배 방정식은 질량 행렬 \mathbf{M}, 감쇠 행렬 \mathbf{C}, 강성 행렬 \mathbf{K} 로 이뤄진 비선형 결합 텐서이다.

\mathbf{M} \mathbf{\ddot{u}}(t) + \mathbf{C} \mathbf{\dot{u}}(t) + \mathbf{K}(\mathbf{u}) \mathbf{u}(t) = \mathbf{F}(t)

CPS가 감당해야 할 현실적인 한계 시간 내에 이 10^6 \times 10^6 거대 희소 행렬(Sparse Matrix)의 역행렬을 도출하는 것은 불가능에 가깝다.
그러나 다행히도, “Model Order Reduction: Theory, Research Aspects and Applications” 서적 등 수많은 응용수학 문헌이 증명하듯, 기계 장치가 요동치는 실제 거시적 궤적 공간은 전체 N 차원 중 극히 극소수의 r 차원(r \ll N) 매니폴드(Manifold) 표면 위로만 끌려가는 현상(Attraction)을 보인다. 즉, 수백만 개의 격자 중 독립적으로 요동치는 자유도는 사실 몇십 개에 불과하며, 나머지는 이 핵심 자유도에 선형적으로 끌려다니는 거대한 관성 종속체에 지나지 않는다.

2. 적절 직교 분해(Proper Orthogonal Decomposition, POD): 데이터 기반의 기저 투영

물리 모델 차원 축소의 제1 법칙은 적절 직교 분해(Proper Orthogonal Decomposition, POD) 혹은 카루넨-루브 전개(Karhunen-Loève Expansion)이다. 이는 주성분 분석(PCA)을 함수 공간과 갤러킨 투영법(Galerkin Projection) 레벨로 확장한 연속계 텐서 분해 모듈이다.

전차원 모델(FOM) 시뮬레이션을 미리 몇 번 돌려(Offline Stage), 시간 단위로 떨어지는 거대한 결과 백터들을 기둥(Column)으로 삼는 스냅샷 행렬(Snapshot Matrix, \mathbf{X}) 공간을 획득한다.

\mathbf{X} = [\mathbf{u}_1, \mathbf{u}_2, \dots, \mathbf{u}_T] \in \mathbb{R}^{N \times T}

이 \mathbf{X} 에 대하여 특이값 분해(Singular Value Decomposition, SVD)를 강제 타격한다(\mathbf{X} = \mathbf{U} \mathbf{\Sigma} \mathbf{V}^T). 계산된 좌측 특이 벡터(Left Singular Vector) \mathbf{U} 에서 가장 거대한 에너지를 머금은 상위 r 개의 직교 기저(Orthogonal Basis) 벡터만을 적출하여 투영 행렬 \mathbf{\Phi} \in \mathbb{R}^{N \times r} 를 주조한다. 물리 세계의 원래 상태 \mathbf{u}(t) 는 저차원 공간의 궤적 \mathbf{a}(t) 와 갤러킨 매트릭스의 선형 곱으로 극도로 압축(\mathbf{u}(t) \approx \mathbf{\Phi} \mathbf{a}(t))된다.

2. 갤러킨 투영(Galerkin Projection)에 의한 미분 방정식의 축소

\mathbf{u}(t) \approx \mathbf{\Phi} \mathbf{a}(t) 를 원래의 10^6 차원 뉴턴-코시 비선형 모델 방정식에 강제로 우겨넣은 후, 전면에 다시 투영 행렬의 전치 \mathbf{\Phi}^T 를 곱하는(Left-multiplying) 연산을 통해 차원을 강제 탈피시킨다.

\mathbf{\Phi}^T \mathbf{M} \mathbf{\Phi} \mathbf{\ddot{a}}(t) + \mathbf{\Phi}^T \mathbf{C} \mathbf{\Phi} \mathbf{\dot{a}}(t) + \mathbf{\Phi}^T \mathbf{K} \mathbf{\Phi} \mathbf{a}(t) = \mathbf{\Phi}^T \mathbf{F}(t)

\mathbf{M}_r \mathbf{\ddot{a}}(t) + \mathbf{C}_r \mathbf{\dot{a}}(t) + \mathbf{K}_r \mathbf{a}(t) = \mathbf{F}_r(t)

이 순간 거대했던 10^6 \times 10^6 의 지배 방정식 텐서망은 마법처럼 r \times r (예: 20 \times 20) 의 극소형 축소 모델(Reduced Order Model, \mathbf{M}_r, \mathbf{C}_r, \mathbf{K}_r)로 치환된다. 물리적 신뢰도 오차는 1\% 미만을 보장하면서도, 연산 속도는 10,000 배 이상 폭주하는 절대적인 전산 역학의 권능이다.
