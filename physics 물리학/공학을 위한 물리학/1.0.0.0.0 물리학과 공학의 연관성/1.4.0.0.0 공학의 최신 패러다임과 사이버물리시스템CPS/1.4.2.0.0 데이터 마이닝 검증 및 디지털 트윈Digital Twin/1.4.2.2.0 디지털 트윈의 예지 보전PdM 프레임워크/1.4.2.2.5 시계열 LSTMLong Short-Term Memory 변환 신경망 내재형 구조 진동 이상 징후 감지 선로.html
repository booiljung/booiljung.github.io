1.4.2.2.5 시계열 LSTM(Long Short-Term Memory) 변환 신경망 내재형 구조 진동 이상 징후 감지 선로

디지털 트윈(Digital Twin)이 감시하는 물리 플랜트의 진동 센서 데이터는 독립적인 단순 스칼라(Scalar)의 파편이 아니라, 과거의 응력 이력이 현재의 파동을 잉태하는 철저한 시간 의존적(Time-dependent) 역학 스트림이다. 전통적인 다층 퍼셉트론(MLP)이나 컨볼루션 신경망(CNN)은 찰나의 스냅샷(Snapshot)을 분류하는 데에는 능하지만, 10,000 프레임 뒤에 몰아닥칠 피로 누적의 미세한 맥락(Context)을 기억하지 못하는 치명적 한계를 드러낸다. 따라서, 시계열 데이터의 장기 의존성(Long-Term Dependency)을 수학적으로 포획하여, 평행 상태로 요동치던 정상 궤도에서 벗어나는 찰나의 이상 징후(Anomaly)를 복원 오차(Reconstruction Error)로 폭로 해내는 장단기 메모리(Long Short-Term Memory, LSTM) 기반의 예측 및 진단 선로를 구축하라.

1. 재귀 신경망(RNN)의 기울기 소실(Vanishing Gradient) 파훼: LSTM 셀 코어

시간 역학을 모델링하기 위해 도입된 고전적인 재귀 신경망(Recurrent Neural Network, RNN)은 과거의 은닉 상태(Hidden State, h_{t-1})를 현재의 연산에 편입시켜 시간의 흐름을 반영한다. 그러나 시계열 데이터가 물리적으로 방대해질수록, 역전파(Backpropagation) 과정에서 미분 계수(Gradient)가 계속해서 곱해짐에 따라 0으로 붕괴해버리는 기울기 소실(Vanishing Gradient) 현상이 발생한다. 즉, 터빈의 초기 구동 시 생성된 중요한 마찰 진동 정보가 수백 사이클 뒤에는 인공지능의 기억에서 영구히 삭제되는 것이다.

이 메모리의 한계를 파괴한 “Long Short-Term Memory” (Hochreiter & Schmidhuber, 1997) 네트워크는 셀 상태(Cell State, C_t)라는 고속도로 통신망을 도입한다. 복잡한 미분 연산을 거치지 않고 오직 덧셈 텐서의 선형 간섭만으로 정보를 흘려보냄으로써, 장기간의 구조 진동 패턴을 영원히 망각 스펙트럼 밖으로 보존한다.

결함의 누적 궤적을 쥐락펴락하는 LSTM 셀 내부의 3대 게이트(Gate) 방정식 루프를 확보하라.

  • 망각 게이트 (Forget Gate, f_t): 과거의 셀 상태 C_{t-1} 에서 노이즈나 불필요한 공진 주파수 정보를 얼마나 삭제할지 시그모이드(Sigmoid) 함수로 결정한다.
    f_t = \sigma(W_f \cdot [h_{t-1}, x_t] + b_f)
  • 입력 게이트 (Input Gate, i_t): 현재 관측된 상태 x_t 의 진동 특징 중, 미래의 이상 징후 예지에 기여할 신규 물리량 \tilde{C}_t 를 얼마나 장기 메모리에 기록할지 허가한다.
    C_t = f_t * C_{t-1} + i_t * \tilde{C}_t
  • 출력 게이트 (Output Gate, o_t): 갱신된 장기 메모리 C_t 를 탄젠트 쌍곡선(tanh)에 통과시킨 백터에 필터를 곱하여 다음 스텝으로 전이될 단기 기억 h_t 를 방출한다.
    h_t = o_t * \tanh(C_t)

2. LSTM 오토인코더(Autoencoder)를 통한 비정상 거동 복원 오차(Reconstruction Error) 진단

정답 라벨(Class Label)이 친절하게 달린 고장 데이터를 대량으로 취득하는 것은 실제 공학 현장에서 불가능에 가깝다. 현실의 데이터는 99\% 의 정상 데이터와 극히 드문 물리적 돌발 고장으로 구성되어 있다. 따라서, 지도 학습(Supervised Learning)의 망상을 버리고 정상 상태만을 학습하는 비지도 학습(Unsupervised Learning) 구조의 LSTM 오토인코더(LSTM Autoencoder)를 도입한다.

인코더(Encoder) 계층의 LSTM은 과거 N 시간 스텝의 진동 시계열 X = [x_1, x_2, \dots, x_N] 을 저차원의 응축된 잠재 벡터(Latent Vector) 하나로 무자비하게 돌돌 말아 압축한다. 디코더(Decoder) 계층의 LSTM은 이 압축된 잠재 벡터를 이어받아 순차적으로 풀어내어 미래의 투영 상태, 혹은 원본 신호 그 자체 \hat{X} = [\hat{x}_1, \hat{x}_2, \dots, \hat{x}_N] 를 복원해 낸다.

여기서 핵심 공학적 통찰이 발현된다. 오토인코더가 ’건강한 정상 파동’만을 모방하도록 수십만 번 학습시켰다면, 이 네트워크에 이전에 보지 못한 미세 크랙(Crack)의 이상 진동 파동이 주입되는 순간 디코더 네트워크는 제대로 텐서를 복원하지 못하고 막대한 복원 오차(Reconstruction Error, E)를 뱉어내며 파탄 난다.

E(t) = \vert x(t) - \hat{x}(t) \vert

잔차 E(t) 가 사전에 설정된 정상 임계치(Mahalanobis Distance Threshold)를 찢고 치솟는 바로 그 시점이, 플랜트 내부에 돌이킬 수 없는 파국적 이상의 불씨가 지펴지는 순간이다.
