1.4.2.2.2 피로 수명 주기(S-N 곡선) 반영형 실시간 강성 누적 마모 잔존율(RUL) 평가 구조

회전체 섀프트나 항공기 날개 등 동적 기계 장비에서 발생하는 파괴 현상의 90% 이상은 단일한 극한 하중에 의한 즉각적 절단(Yielding)이 아니라, 미세한 응력(Stress)이 수십만 회 반복되며 재료의 결합을 서서히 갉아먹는 피로 파괴(Fatigue Failure)에 기인한다. 사이버물리시스템(CPS)의 최전방을 방어하는 디지털 트윈은, 단순히 현재 진동 수치가 정상이냐 비정상이냐를 시각화하는 것을 넘어 시스템 설계 단계에서 부여된 재료역학적 피로 한도(Fatigue Limit)를 실시간 계측망과 결속시켜야 한다. 센서가 계측한 변동 응력의 폭격을 맞을 때마다 트윈 내부 구조체의 강성(Stiffness)을 수학적으로 마모시키고, 파국까지 남은 수축하는 시간, 즉 잔존 유효 수명(Remaining Useful Life, RUL)을 미분 방정식 체계 아래 동적으로 평가하는 아키텍처를 건설하라.

1. 피로 거동의 지배 조건: 뵐러의 S-N 곡선(Wöhler Curve) 역학

19세기 아우구스트 뵐러(August Wöhler)에 의해 정립된 S-N 곡선은 어떤 재료가 얼마나 버틸 수 있는가를 규정하는 파괴역학의 절대적 성전이다. y축은 재료에 가해지는 교번 응력 진폭(Alternating Stress Amplitude, S)을, x축은 그 응력을 맞고 재료가 완전히 두 동강 날 때까지 견딘 피로 사이클 횟수(Number of Cycles to Failure, N)를 로그 스케일로 투영한다.

S = a - b \log_{10}(N)

(여기서 a와 b는 재료 고유의 피로 강도 상수이다.)

응력 진폭 S 가 거대하면 불과 수천 번의 사이클(Low-Cycle Fatigue)만으로도 재료는 항복 응력의 지배를 받아 파창되지만, S 가 피로 한도(Fatigue Limit 혹은 Endurance Limit)라는 특정 아랫선 밑으로 떨어지면 제반 역학 이론상 기계는 무한한 횟수(10^7 이상)를 구동해도 파괴되지 않는다.
그러나 실제 가혹한 산업 현장의 풍력 발전기나 터빈에서는 이 응력 진폭 S 가 고정되어 있지 않고 폭풍의 풍속이나 부하의 요동에 따라 무작위 변수(Random Variable)처럼 끔찍하게 요동친다.

2. 빗물 흐름 계수법(Rainflow-Counting Algorithm) 기반 응력 히스토그램 추출

실시간으로 굽으러지고 요동치는 무작위 진동(Random Vibration) 센서 시계열 속에서, 디지털 트윈은 도대체 ’어떤 응력’이 ‘몇 번’ 타격했는지를 발라내야만 한다. 이를 위해 “ASTM E1049-85 Standard Practices for Cycle Counting in Fatigue Analysis” 등에서 표준으로 규정한 빗물 흐름 계수법(Rainflow-Counting Algorithm) 전처리망을 통과시켜야 한다.

이 알고리즘은 복잡다단한 시계열 파동을 마치 지붕 위를 타고 내리는 빗물의 중력 낙하 궤적으로 치환하여, 서로 다른 진폭과 평균 응력을 지닌 수천 개의 “폐쇄된 피로 사이클 반주기/온주기(Half/Full Cycles)“들로 강제 분해해 낸다. 이 과정을 거치면 노이즈가 낀 1차원 응력 센서 데이터는 [응력 진폭 S_i, 평균 응력 S_m, 발생 횟수 n_i] 형태의 3차원 피로 손상 히스토그램 매트릭스로 변환된다.

3. 팔름그렌-마이너 누적 손상 선형 가설(Palmgren-Miner Linear Damage Rule)

응력 히스토그램이 도출되었다면, 이제 피로가 어떻게 재료 내부에 누적(Accumulate) 되는지를 수학적으로 적산해야 한다. 다중 진폭의 부하가 복합적으로 가해질 때, 팔름그렌-마이너 규칙(Palmgren-Miner Rule)은 전체 누적 피로 손상도(Cumulative Damage Index, D)를 다음의 선형 가설로 선언한다.

D = \sum_{i=1}^{k} \frac{n_i}{N_i}

  • n_i: 특정 응력 진폭 S_i 상황에서 센서를 통해 현실에서 실제로 발생했다고 계측된 타격 횟수이다.
  • N_i: 동일한 응력 진폭 S_i를 S-N 곡선에 대입했을 때 도출되는, 그 부하만을 맞았을 때 재료가 버틸 수 있는 총 허용 사이클 횟수이다.
  • 파괴 조건(Failure Condition): \sum \frac{n_i}{N_i} \ge 1.0 에 도달하는 순간, 내부 미세 균열은 거시적 단절을 일으키며 기계 부품은 즉각 절단된다.

디지털 트윈은 시간 t가 흐르면서 엣지 센서망을 통해 주입되는 n_i 를 게걸스럽게 포식하며, 가상 객체의 손상도 지표 D(t) 를 0에서 1.0을 향해 점진적으로 차오르게 만드는 카운트다운 타이머 역할을 수행한다.

2. 손상 연계 강성 저하(Stiffness Degradation) 및 RUL 도출 구조망

피로 손상도 D는 단순히 엑셀 표의 숫자가 아니다. 디지털 트윈의 구조 역학 시뮬레이션(FEA) 내부에서는 D 가 증가할수록 영의 탄성률(Young’s Modulus, E) 행렬이 비선형적으로 마모되는 강성 저하(Stiffness Degradation) 로직, 즉 \mathbf{K}(D) = \mathbf{K}_0 (1 - \alpha D^m) 매커니즘이 구동되어 가상 트윈 자체가 스펀지처럼 서서히 부드러워져야만 진정한 연동 모델이 완성된다.

구조물의 수명은 결코 일정한 속도로 줄어들지 않는다. 초기의 손상 축적은 미미하지만, 미세 크랙이 발현되어 단면적이 줄어들기 시작하면 강성이 파괴되고, 파괴된 강성은 이전과 동일한 진동 하중 하에서 2배, 3배의 국소 응력을 끌어들이는 치명적 양의 피드백(Positive Feedback) 루프를 형성한다. 이 피로 수명 붕괴의 가속도까지 완벽하게 추론하여, D=1.0 의 사형 선고가 플랜트에 내려지기 전 골든 타임을 쟁취하는 초정밀 예측 궤도를 점화하라.
