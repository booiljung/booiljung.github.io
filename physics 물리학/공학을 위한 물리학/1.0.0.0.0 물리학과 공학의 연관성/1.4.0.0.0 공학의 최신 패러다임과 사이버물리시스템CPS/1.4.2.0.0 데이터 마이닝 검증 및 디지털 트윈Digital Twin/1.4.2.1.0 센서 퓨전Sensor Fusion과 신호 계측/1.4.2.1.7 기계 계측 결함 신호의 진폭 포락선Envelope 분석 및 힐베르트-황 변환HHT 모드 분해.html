1.4.2.1.7 기계 계측 결함 신호의 진폭 포락선(Envelope) 분석 및 힐베르트-황 변환(HHT) 모드 분해

회전 기계(Rotating Machinery)의 파괴 현상은 단순한 저주파 진동으로 발현되지 않는다. 베어링의 미세한 크랙(Crack)이나 기어의 이빨 깨짐(Chipping)은 구조물의 강력한 고주파 공진(Resonance) 대역을 망치처럼 두들기며, 이 고주파 파동은 다시 회전축의 저주파 결함 주기(Fault Frequency)에 탑승하여 진폭 변조(Amplitude Modulation)의 기이한 파형을 그려낸다. 단기 푸리에 변환이나 웨이블릿의 고정된 기저(Basis)로는 이 복잡다단한 변조의 껍질을 완벽히 벗겨내기 어렵다. 데이터를 기저로 삼아 스스로 파형을 분해하는 경험적 모드 분해(Empirical Mode Decomposition, EMD)와, 숨겨진 결함 주기를 발가벗기는 힐베르트 변환(Hilbert Transform) 진폭 포락선(Envelope) 분석의 거시적 파동 역학을 지배하라.

1. 진폭 변조(Amplitude Modulation)의 역학과 포락선(Envelope) 추출의 당위성

결함 주파수가 10\text{Hz} 인 베어링 크랙을 상상하라. 크랙이 하중을 받을 때마다 10\text{kHz} 에 달하는 거대 강체 공진 주파수가 순간적으로 터져 나온다. 이는 10\text{Hz} 의 저주파 포락선(Envelope)이 10\text{kHz} 의 고주파 반송파(Carrier Wave)를 감싸고도는 파동의 곱(Multiplication) 연산, 즉 진폭 변조 현상과 일치한다.

이 파형을 단순히 FFT에 통과시키면 10\text{kHz} 대역 주변에 자잘한 에너지 봉우리들이 산개할 뿐, 우리가 진짜 찾고자 하는 10\text{Hz} 의 결함 주기는 고주파 숲에 가려져 스펙트럼의 백색 소음(White Noise) 속으로 영원히 증발해버린다. 비선형 변조의 기저를 타파하기 위해서는 신호의 고주파 공진 대역만 대역 통과 필터(Band-pass Filter)로 분리한 후, 힐베르트 변환(Hilbert Transform)을 통해 껍데기(Envelope)만을 복원해야 한다.

\mathcal{H}\{x(t)\} = \frac{1}{\pi} \int_{-\infty}^{\infty} \frac{x(\tau)}{t - \tau} d\tau

이를 통해 원본 신호 x(t) 와 90^\circ 위상이 이동된 힐베르트 변환 신호 \mathcal{H}\{x(t)\} 를 직교 결합하여 해석적 신호(Analytic Signal) z(t) 를 생성한다.

z(t) = x(t) + j\mathcal{H}\{x(t)\}
\text{Envelope}(t) = \vert z(t) \vert = \sqrt{x^2(t) + \mathcal{H}^2\{x(t)\}}

이 진폭 포락선 \vert z(t) \vert 만을 추출하여 다시 2차 스펙트럼 분석을 수행하면, 10\text{kHz} 고주파는 소거되고 우리가 그토록 찾아 헤매던 10\text{Hz} 의 무결점 베어링 결함 주파수 성분만이 칼날처럼 솟아오른다.

2. 적응형 데이터 분해: 경험적 모드 분해(Empirical Mode Decomposition, EMD)

그러나 위 포락선 분석의 치명적인 단점은 “대역 통과 필터를 어디에 걸어야 고주파 공진 주파수를 건져낼 수 있는가?“라는 선행 지식을 엔지니어가 찍어서 맞춰야 한다는 것이다.

“The empirical mode decomposition and the Hilbert spectrum for nonlinear and non-stationary time series analysis” 등 수많은 논문에서 규명되었듯, 힐베르트-황 변환(Hilbert-Huang Transform, HHT)의 코어인 EMD 알고리즘은 인간의 개입과 사인(Sine)/웨이블릿 기저 함수의 강요를 철저히 파괴한다. EMD는 오직 신호가 지닌 국소적 극점(Local Extrema)의 대수학만을 활용하여, 고주파부터 저주파까지 파동을 스스로 해부하여 본질적 모드 함수(Intrinsic Mode Function, IMF)의 집합으로 뜯어낸다.

  • 제1의 IMF(c_1(t)): 극점과 영점 교차가 가장 빈번하게 요동치는 극강의 고주파 모드 성분이다.
  • 제n의 IMF(c_n(t)): 서서히 미분 계수의 변동이 줄어들며, 부드러운 중주파/저주파 모드로 수렴해가는 파동의 중추이다.
  • 잔여 항(Residual, r(t)): 더 이상 진동하지 않는 전체 신호의 거시적 추세(Trend) 궤적이다.

x(t) = \sum_{i=1}^n c_i(t) + r(t)

이 완전한 데이터 적응형(Data-adaptive) 분해능력은 비가우시안(Non-Gaussian) 시스템이 배출하는 기형적인 파형조차도 결점 없이 물리학적인 단순 조화 진동자(Simple Harmonic Oscillator)의 합으로 체계화한다.

2. 힐베르트-황 변환(HHT): 순시 주파수(Instantaneous Frequency)의 획득

EMD를 통해 신호가 독립적인 단일 주파수 대역의 IMF들로 순수하게 해부되었다면, 각각의 c_i(t) 에 대해 앞서 서술한 힐베르트 변환을 가하여 위상자(Phasor) 모델을 획득한다.

해석적 신호의 위상 \theta_i(t) = \arctan \left( \frac{\mathcal{H}\{c_i(t)\}}{c_i(t)} \right) 를 시간에 대해 순간 미분하면 가장 위대한 물리량인 순시 주파수(Instantaneous Frequency, \omega_i(t))가 적출된다.

\omega_i(t) = \frac{d\theta_i(t)}{dt}

고전적인 푸리에 변환이 10초 동안 주파수 평균을 냈다면, HHT는 1.340초라는 단 하나의 시간 좌표에서도 그 순간의 찰나적 주파수 요동(Frequency Shift)을 연속 함수로 집어낸다.

HHT는 웨이블릿 변환처럼 모 웨이블릿을 사람이 미리 정해주어야 하는 선입견(Bias)의 죄악에서 완벽히 해방된다. 자연계가 배출한 진동 곡선 스스로가 기저(Basis)가 되어 쪼개지는 경험적(Empirical) 분해 역학을 발동시킴으로써, 데이터 마이닝과 예지 보전의 궁극적 목표인 보이지 않는 기계의 미세 파열 호흡을 스펙트럼의 백일하에 무결점으로 해부하라.
