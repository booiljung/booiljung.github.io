1.4.2.1.8 주성분 분석(PCA) 및 선형 판별 분석(LDA)을 도입한 다차원 결함 특징 표상(Feature Representation)

사이버물리시스템(CPS)의 거대 다중 모달 계측망이 쏟아내는 빅데이터는 날것(Raw Data) 그대로일 때 철저한 차원의 저주(Curse of Dimensionality)에 시달린다. FFT, CWT, HHT 등 신호 처리 기법들을 통해 추출해 낸 수백 개의 특징 변수(Feature Textures)들은 대다수 서로 강하게 연관되어 있거나(Correlated) 단순한 백색 소음에 불과한 잉여 정보를 다크매터처럼 포함한다. 컴퓨팅 리소스를 폭식하는 거대한 텐서를 분류기(Classifier)로 밀어 넣기 전에 가장 순수한 물리적 결함 정보만을 추출하여 저차원 유클리드 공간으로 압축 투영하는 선형 대수학 모델, 주성분 분석(Principal Component Analysis, PCA)과 선형 판별 분석(Linear Discriminant Analysis, LDA)의 다차원 전이 역학을 확립하라.

1. 차원의 저주와 차원 축소(Dimensionality Reduction)의 수학적 당위성

“진폭”, “위상 변이”, “RMS”, “첨도(Kurtosis)”, “웨이블릿 에너지 계수” 등 시계열 데이터로부터 뽑아낸 통계적 특징 인자들을 전부 하나의 벡터 \mathbf{x} = [x_1, x_2, \dots, x_D]^T 로 병합하면 차원 D 가 수백에서 수천으로 폭증한다.

차원이 증가할수록 고장(Fault)과 정상(Normal) 상태를 구분하는 수학적 경계(Decision Boundary)를 확정하기 위해 필요한 학습 데이터의 양이 지수 함수적으로 폭증한다. 또한 측정 변수들끼리의 선형 종속성(Linear Dependence), 예컨대 진동의 RMS 값과 절대 피크(Peak) 값이 동시에 증가하는 다중공선성(Multicollinearity)이 발생하여 머신 러닝 알고리즘의 판별 매트릭스를 특이 행렬(Singular Matrix) 직전까지 붕괴시킨다.
따라서, 이 혼탁하고 방대한 특징 텐서를 d \ll D 인 핵심 저차원 공간 \mathbf{y} \in \mathbb{R}^d 로 선형 변환(\mathbf{y} = \mathbf{W}^T \mathbf{x})시켜 지배적인 인자(Dominant Component)만을 살아남게 하는 직교 투영(Orthogonal Projection) 기법을 장착해야만 한다.

2. 주성분 분석(PCA): 데이터 분산(Variance)의 최대화와 직교 고유벡터

기계 장치의 진동 데이터가 고장을 향해 치닫을 때, 정상 상태의 군집에서 가장 넓고 맹렬하게 대각 방향으로 퍼져나가는 분산(Variance)의 축(Axis)이야말로 고장의 성질을 가장 잘 머금고 있는 지표이다. 주성분 분석(PCA)은 데이터의 라벨(Label) 정보 없이 오버랩된 데이터 구름 그 자체의 기하학적 팽창 방향을 찾아내는 비지도적(Unsupervised) 선형 대수학이다.

원시 특징 벡터 집합 \mathbf{X} 에 대하여, 공분산 행렬(Covariance Matrix) \mathbf{\Sigma} 를 도출한다.

\mathbf{\Sigma} = \frac{1}{N-1} \sum_{i=1}^N (\mathbf{x}_i - \bar{\mathbf{x}})(\mathbf{x}_i - \bar{\mathbf{x}})^T

이 대칭 행렬 \mathbf{\Sigma} 의 고유값(Eigenvalue, \lambda) 분해를 수행한다.

\mathbf{\Sigma} \mathbf{v}_k = \lambda_k \mathbf{v}_k

고유값 \lambda_k 가 큰 순서대로 정렬된 고유벡터(Eigenvector) \mathbf{v}_1, \mathbf{v}_2, \dots, \mathbf{v}_d 가 바로 새로운 공간의 직교 기저(Orthogonal Basis), 즉 주성분(Principal Component)이 된다. 제1 주성분(PC1)은 데이터의 분산이 가장 극대화되는 궤적 축이며, 불필요한 노이즈가 섞인 잡동사니 차원들은 고유값이 0에 수렴하게 되어 자연스럽게 잘려(Truncate) 나간다.

3. 선형 판별 분석(LDA): 클래스(Class) 간 분리능의 극대화

PCA가 데이터의 포괄적인 형태에 집중했다면, 선형 판별 분석(LDA)은 “정상 마모”, “베어링 파손”, “모터 권선 단락“과 같은 시스템의 ‘정답 라벨(Class)’ 정보를 이미 알고 있을 때 가동되는 지도적(Supervised) 차원 축소망이다.
데이터를 저차원 위상 공간에 억지로 우겨넣더라도, 정상 데이터 구름과 고장 데이터 구름이 한데 엉켜서(Overlap) 겹쳐지면 아무런 의미가 없다. LDA는 클래스 간의 이격 거리는 폭발시키고, 동일 클래스 내부의 분산은 극도로 수축시키는 최적의 투영 행렬 \mathbf{W} 를 색출해 낸다.

  • 클래스 내 산포 행렬 (Within-class Scatter Matrix, \mathbf{S}_W): 같은 종류의 결함 군집끼리 뭉치는 밀집도의 방증이다.
  • 클래스 간 산포 행렬 (Between-class Scatter Matrix, \mathbf{S}_B): 서로 다른 결함 군집들의 평균이 멀찍이 떨어져 있는 이격도를 대변한다.

LDA는 다음의 레일리 몫(Rayleigh Quotient) 최적화 공식을 통해 분별능 지표 J(\mathbf{W}) 를 극대화한다.

J(\mathbf{W}) = \frac{\vert \mathbf{W}^T \mathbf{S}_B \mathbf{W} \vert}{\vert \mathbf{W}^T \mathbf{S}_W \mathbf{W} \vert}

이 공식의 최대화 해는 일반화된 고유값 문제(Generalized Eigenvalue Problem) \mathbf{S}_W^{-1} \mathbf{S}_B \mathbf{w} = \lambda \mathbf{w} 로 귀결되며, 고속 매트릭스 역행렬을 타격해 결함 종류들을 산산이 격리(Separation)하는 선형 평면의 절대각을 확보한다.
