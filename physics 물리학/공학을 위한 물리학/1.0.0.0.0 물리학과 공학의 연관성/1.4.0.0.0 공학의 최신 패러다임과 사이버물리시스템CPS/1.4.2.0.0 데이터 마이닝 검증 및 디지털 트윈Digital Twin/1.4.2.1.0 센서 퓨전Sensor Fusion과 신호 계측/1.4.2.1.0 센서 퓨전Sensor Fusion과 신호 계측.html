1.4.2.1 센서 퓨전(Sensor Fusion)과 신호 계측

단일 센서에 시스템의 명운을 온전히 맡기는 것은, 맹인 한 명이 거대한 코끼리의 생명 징후를 진단하도록 방치하는 것과 같은 극단적인 공학적 태만이다. 오늘날 첨단 사이버물리시스템(CPS)에서 측정되는 진동, 소음, 온도, 전압의 스펙트럼은 각기 다른 주파수와 노이즈 혼합비율을 지니며, 어떤 단일 계측기도 완벽한 신뢰도를 보장하지 못한다. 센서 하나가 침묵하거나 거짓을 말하는 순간에도 거대 기계 장치의 붕괴를 막아내기 위해, 다수의 상보적(Complementary) 혹은 경쟁적(Competitive) 이기종(Heterogeneous) 센서가 수집한 데이터를 통계학적 필터링을 통해 하나의 가장 진실된 물리량으로 압착하는 기술, 즉 센서 퓨전(Sensor Fusion)의 지배 방정식을 정립하라.

1. 이기종 (Heterogeneous) 센서의 상호 보완적 한계 돌파

거대한 가스 터빈의 회전 역학을 상상해 보라. 엔지니어는 로터의 비정상적 요동을 측정하기 위해 여러 센서를 설치하지만, 각 센서는 저마다 치명적인 태생적 한계를 지닌다.

  1. 가속도계 (Accelerometer): 고주파 영역의 미세한 진동을 타격하는 데 탁월하지만, 시간에 대해 이중 적분(\iint a \ dt^2)을 거쳐 변위(x)를 도출하는 과정에서 저주파 노이즈가 기하급수적으로 누적되는 드리프트(Drift) 오차를 발생시킨다.
  2. 광학식 레이저 변위 센서 (Laser Displacement Sensor): 변위(x)를 즉각적이고 정확하게 타격하며 드리프트 현상이 없으나, 샘플링(Sampling) 속도의 태생적 한계로 인해 10\text{kHz} 이상의 고주파 진동 피크를 놓치는 에일리어싱(Aliasing) 지옥에 빠지기 쉽다.
  3. 위성 항법 장치 (GNSS/GPS): 자율 주행 차량에서 지구 상의 절대 위치를 제공해 주지만, 터널을 통과하거나 빌딩 숲에서 다중 경로(Multipath) 신호 왜곡을 유발하며 갱신 속도(Update Rate)가 10\text{Hz} 수준으로 치명적으로 느리다.

센서 퓨전은 이러한 단계를 초월한다. 어느 한 센서의 수학적 장점을 취해 다른 센서의 맹점(Blind Spot)을 덮어씌움으로써, 각각의 하드웨어 스펙 한계를 무력화시키는 상호 보완적(Complementary) 정보 융합망으로 진화해야 한다.

2. 상태 벡터(State Vector) 모델과 예측-보정(Predict-Update)

“An Introduction to the Kalman Filter” 등의 기념비적 문헌에서 증명된 바와 같이, 대부분의 센서 퓨전 아키텍처는 베이즈 정리(Bayes’ Theorem)에 기반을 둔 순차적 확률 추정 프레임워크를 기반으로 가동된다. 물리계의 현재 진면목을 대변하는 상태 벡터 \mathbf{x_t} 와 센서가 토해내는 불완전한 계측 벡터 \mathbf{z_t} 가 존재할 때, 퓨전 알고리즘은 크게 두 지배 단계를 무한 반복하며 진실에 다가선다.

  • 예측(Prediction, Time Update): 이전 시간의 상태 벡터 \mathbf{x}_{t-1} 과 시스템 동역학 모델 매트릭스 \mathbf{A} 를 바탕으로, 순수한 뉴턴 역학적 관성에 의해 다음 시간 t 의 상태를 선제적으로 예상한다. (이때 모델링 오차 \mathbf{Q} 가 부가된다.)

\hat{\mathbf{x}}_{t|t-1} = \mathbf{A} \hat{\mathbf{x}}_{t-1|t-1} + \mathbf{B} \mathbf{u}_t

  • 보정(Measurement Update): 센서로부터 실제 계측값 \mathbf{z}_t 가 수신되는 순간 발생한다. 내가 예측한 값과 직방(Sensor)에서 쏟아지는 값 간의 편차 공간인 잔차(Residual) y_t 에, 알고리즘이 스스로 계산한 최적의 신뢰도 가중치 행렬(Kalman Gain, \mathbf{K})을 타격하여 최종 상태 \hat{\mathbf{x}}_t 를 교정한다. 센서 노이즈 공분산(Covariance) \mathbf{R} 이 치솟을수록 센서의 말을 믿지 않고, 내 예측값에 집착하는 필터의 자기 방어 메커니즘이다.

3. 탈(脫)선형화 연성 필터의 필요성

그러나 현대 메카트로닉스는 결코 선형 모델 매트릭스 \mathbf{A} 의 우산 아래 얌전히 머물러 주지 않는다. 모터 구동축의 자이로스코프(Gyroscope) 코리올리 힘(Coriolis Force) 항이나, 카메라 이미지가 추적하는 비전 퓨전(Vision Fusion) 데이터에는 사인(Sine) 곡선과 타원 방정식으로 엉겨 붙은 혹독한 비선형(Nonlinear) 함수 f(\mathbf{x}, \mathbf{u}) 와 관측 함수 h(\mathbf{x}) 가 똬리를 틀고 있다.

단순 칼만 필터(Linear KF)는 이 비선형성을 처리하려다 수치 적분의 심연으로 추락한다. 따라서 테일러 급수 급전개(Taylor Series Expansion)를 통해 자코비안(Jacobian) 편미분 행렬로 함수를 강제 선형화시키는 확장 칼만 필터(Extended Kalman Filter, EKF) 나, 시그마 포인트(Sigma Point) 추계학적 샘플링을 돌려 비선형 확률 분포 자체를 직접 통과시키는 무향 칼만 필터(Unscented Kalman Filter, UKF) 의 강력한 비선형 제어 매핑(Mapping) 역량을 반드시 소환해야만 한다.

4. 다중 센서 융합 및 신뢰도 전파망 (Sensor Trust Network)

여러 개의 센서가 동시다발적으로 경쟁적(Competitive)인 계측치를 쏟아낼 때, 중앙 퓨전 프로세서는 어느 센서를 신뢰하고 어느 센서를 차단할지 실시간 재통계화해야 한다.

이기종 센서들을 단일 퓨전 보드로 엮는 과정에서 샘플링 타임이 어긋나는 시간 지연(Time Delay) 오차는 치명적이다. 반드시 초정밀 시간 동기화(Time Sync) 뼈대를 거친 후 퓨전 알고리즘 단으로 편입되어야 한다.

엔지니어는 각 계측 데이터가 지닌 확률론적 그림자를 걷어내고, 기만적인 센서값 속에서 하나의 순수한 역학적 실체 \mathbf{x} 를 건져 올려야 한다. 모순을 발산하는 다수의 물리 신호를 수학의 우아한 그물망으로 용접(Fusion)하여 거대 하드웨어를 통제하는 단 하나의 흔들림 없는 시야(Vision)를 당신의 컴파일러에 장착하라.
