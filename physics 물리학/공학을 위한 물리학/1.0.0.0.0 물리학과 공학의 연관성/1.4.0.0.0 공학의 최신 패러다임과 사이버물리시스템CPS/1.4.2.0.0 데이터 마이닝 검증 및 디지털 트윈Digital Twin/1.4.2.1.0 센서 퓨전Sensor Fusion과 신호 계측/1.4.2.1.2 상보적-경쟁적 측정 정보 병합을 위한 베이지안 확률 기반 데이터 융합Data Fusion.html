1.4.2.1.2 상보적/경쟁적 측정 정보 병합을 위한 베이지안 확률 기반 데이터 융합(Data Fusion)

다중 센서가 관측한 물리계의 표상은 필연적으로 불확실성(Uncertainty)이라는 짙은 안개에 휩싸여 있다. 동일한 모터의 회전을 계측하더라도, 센서 A는 노이즈에 오염된 고주파 진동을 부르짖고 센서 B는 위상 지연이 발생한 저주파 속도를 속삭인다. 공학자가 진리를 쟁취하기 위해서는 누구의 말을 믿을 것인가를 단순히 평균 내는 산술적 합의에 그쳐서는 안 된다. 이 불확실한 다차원 정보의 역설 속에서 과거의 경험(Prior)과 현재의 증거(Likelihood)를 확률 연산으로 교배시켜, 가장 오차 공분산이 희박한 최적의 상태 추정치를 도출해 내는 베이지안 확률 기반 데이터 융합(Bayesian-based Data Fusion)의 거시적 수학 체계를 확립하라.

1. 측정 정보의 두 가지 체계: 상보적(Complementary)과 경쟁적(Competitive) 융합

이기종 센서들을 결속시키는 융합망의 토폴로지(Topology)는 그들이 달성하고자 하는 목적 역학에 따라 상보적 시스템과 경쟁적 시스템으로 양분된다.

  • 상보적 융합 (Complementary Fusion): 각 센서가 대상 플랜트의 서로 다른 물리적 차원이나 대역(Bandwidth)을 담당하여 전체의 해상도를 완성하는 직교적 결합이다. 관성 측정 장치(IMU)는 고주파 가속도를 정밀하게 타격하지만 시간이 갈수록 오차가 누적되는 반면, GPS 모듈은 갱신 주기가 느리지만 절대 위치의 기준을 잡아준다. 이 두 센서의 장점을 퍼즐처럼 끼워 맞춰 넓은 주파수 대역의 가시성을 획득하라.
  • 경쟁적 융합 (Competitive Fusion): 동일한 물리량(예: 연소실 온도)을 관측하는 동종의 센서 여러 개를 중복(Redundancy) 배치하는 결합이다. 이는 주로 센서 하나의 파단(Hard Fault)이 전체 플랜트를 파괴하는 것을 방어하기 위한 결함 허용 제어(Fault-Tolerant Control)에서 기용된다. 센서들이 멱함수적인 경쟁을 벌일 때, 어느 센서의 분산이 증가(고장 징후)했는지를 통계적으로 판별하여 타락한 노드를 융합망에서 강제 격리 조치하는 권능이다.

2. 베이즈 정리(Bayes’ Theorem)와 확률적 믿음의 전파

센서 융합의 대자연을 통치하는 “Sensor Data Fusion: A Tool for Information Assessment and Decision Making” 등에서 언급된 절대 역학 공식은 바로 18세기 수학자 토마스 베이즈(Thomas Bayes)가 주창한 베이즈 정리이다. 이는 새로운 계측 증거 Z 가 주어졌을 때, 시스템의 참된 상태벡터 x 일 확률(사후 확률)을 논증한다.

P(x \mid Z) = \frac{P(Z \mid x) P(x)}{P(Z)}

  • 사전 확률 (Prior Probability, P(x)): 센서의 계측 데이터가 도달하기 직전, 시스템 동역학 모델과 과거 궤적을 토대로 내가 강력하게 예측하고 있는 상태의 확률 분포이다.
  • 우도 (Likelihood, P(Z \mid x)): 플랜트가 특정 상태 x 에 치달아 있을 때, 현재 센서가 바로 그 노이즈 낀 값 Z 를 계측해 낼 통계학적 함수(확률 밀도)이다. 센서의 스펙(분산)에 직접적으로 종속된다.
  • 사후 확률 (Posterior Probability, P(x \mid Z)): 사전 모델링 법칙(P(x))과 실재하는 센서의 증거(P(Z \mid x))가 기하학적으로 곱해져 탄생한 최종 진리의 확률 분포이다. 분모 P(Z) 는 확률 총합을 1로 맞추기 위한 정규화 증폭 상수(Normalizing Constant) 역할을 수행한다.

3. 확률 분포의 곱셈 피드백 매커니즘: 베이지안 연속 병합

복잡계 엔지니어링에서는 보통 상태와 노이즈가 가우시안 정규 분포(Gaussian Normal Distribution)를 띤다고 가정한다. 이 매끄러운 종 모양 곡선이 상호 경쟁적/상보적으로 충돌할 때, 수학적 기적(Mathematical Miracle)이 발생한다.

두 개의 독립된 가우시안 분포, 사전 예측 분포 \mathcal{N}(\mu_1, \sigma_1^2) 와 센서 계측 분포 \mathcal{N}(\mu_2, \sigma_2^2) 를 베이즈 정리로 강제 곱(Multiplication)연산 시키면 얻게 되는 사후 확률 분포 \mathcal{N}(\mu_{fusion}, \sigma_{fusion}^2) 의 분산 \sigma_{fusion}^2 은 다음과 같이 수렴된다.

\sigma_{fusion}^2 = \frac{\sigma_1^2 \sigma_2^2}{\sigma_1^2 + \sigma_2^2}

센서 노이즈가 아무리 날뛰더라도, 이 융합된 분산 \sigma_{fusion}^2 값은 오리지널 예측 분산 \sigma_1^2 이나 오리지널 계측 분산 \sigma_2^2 그 어느 쪽의 단일 값보다도 무조건적으로 작다(\sigma_{fusion}^2 < \min(\sigma_1^2, \sigma_2^2)). 정보가 모이면 모일수록 불확실성의 종형 곡선은 바늘 끝처럼 예리해지는 확률론적 집중의 원리이다.
