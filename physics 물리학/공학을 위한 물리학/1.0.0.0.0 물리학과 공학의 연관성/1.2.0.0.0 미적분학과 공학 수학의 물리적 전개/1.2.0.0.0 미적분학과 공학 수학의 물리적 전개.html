1.2 미적분학과 공학 수학의 물리적 전개

자연계의 근원적 물리 법칙은 정지된 스냅샷(Snapshot)의 덧입힘으로 온전히 설명될 수 없으며, 시간과 공간상에서 끊임없이 요동치고 상호 작용하는 유체적인 상태 변화의 역학으로 귀결된다. 이를 수학적인 엄밀성을 통해 모델링하고 투영(Projection)하기 위해, 공학과 물리학의 세계관은 필연적으로 미적분학(Calculus) 및 고등 공학 수학(Advanced Engineering Mathematics)이라는 언어 체계를 차용한다. 뉴턴(Newton)과 라이프니츠(Leibniz)에 의해 태동한 미적분학은 거시적인 물리량의 순간적인 변화율과, 미소 요소들의 누적을 통한 전체 에너지의 합산을 추적하는 궁극의 수리 도구이다. 본 장에서는 단순한 수식의 연산을 철저히 배제하고, 이러한 수학적 기저 원리들이 공학 시스템의 지배 방정식(Governing Equation)과 설계 공간 속에서 어떠한 물리학적 숨결로 전개되는지를 엄밀히 해부한다.

1. 공간과 시간의 동역학적 관측 도메인

공학 시스템에서 발생하는 변위, 속도, 가속도, 온도 구배, 전자기장의 플럭스(Flux) 등 모든 상태 변수(State Variable)는 시간 차원(t)과 공간 차원(x, y, z)을 도메인으로 삼는 연속 함수(Continuous Function)로 거동한다. 이 차원 위에서 물리적 직관을 연산 가능한 텐서(Tensor) 스페이스로 환원하기 위해서는 두 가지 상호 보완적인 미적분학의 창날이 투입된다.

  • 미분(Differentiation) 역학: 거대 구조물의 국부적 응력 집중, 고속 유동장의 난류 와류(Eddy), 전자기파의 전파와 같은 찰나의 국소적 변화를 정밀하게 분해한다. 이는 연속적인 스칼라나 벡터 장에서 극미의 시공간 변화 \Delta t, \Delta x 가 0으로 수렴할 때 발생하는 극한의 변화율, 즉 기울기(Gradient)의 물리량을 도출해 낸다. 미분 역학은 시스템 내부에 어떤 강제적인 원인(힘, 열원, 전하)이 작용하고 있는지를 역추적하는 미시적 진단 체계라 할 수 있다.
  • 적분(Integration) 역학: 반대로 흩뿌려진 미시적 요소(Differential Elements)들을 전체 도메인 공간에 대하여 결합(Summation)함으로써 거시적인 총량(Total Amount)을 회수해 내는 합성 프로세스이다. 비균일 매질을 통과하는 열의 누적 전도량, 시간의 흐름에 따른 질점의 최종 변위, अथवा 유체 볼륨이 지닌 총 질량과 운동 에너지 역학계는 모두 적분 공간 내의 스칼라 내적 산물이다.

2. 공학 수학 행렬의 물상적 융합

자연계의 1차원적인 변화를 넘어, 다중 입자계(Multi-body System)의 상호 충돌, 열역학계에서의 복합적 혼합, 3차원 트러스 구조물이 받는 응력 변형 등의 복잡계로 나아가면 미적분학은 단일 스칼라 연산을 넘어 고차원 수학 체계인 선형대수학(Linear Algebra) 및 편미분 방정식(Partial Differential Equation, PDE)과 필연적으로 융합하게 된다.

이 고등 수학의 개입으로 말미암아, 엔지니어는 비로소 자연의 방대한 불확실성을 유한하고 명확한 대수 역학계 행렬(Matrix)로 축약(Reduction)시킬 수 있게 된다. 각 부재들의 고유 진동수를 규명하는 고유치 진단(Eigenvalue Problem), 제어 시스템의 응답 안정성을 점치는 특성 방정식(Characteristic Equation), 변위를 공간 주파수 자원계로 치환하는 변환(Transform) 수학은 모두 이러한 이성적 추상화의 극치를 달린다.
이어지는 1.2장의 하위 절에서는 미분 방정식의 응용과 모달 공간 해석을 주축으로 하여, 이러한 순수 수학의 이론들이 물리적 하드웨어—자동차의 서스펜션, 고층 빌딩의 방진 구조, 반도체의 열 배출—에서 실제로 어떤 역학적 지배력을 행사하며 작동하는지에 대한 응용 과정을 총망라할 것이다.
