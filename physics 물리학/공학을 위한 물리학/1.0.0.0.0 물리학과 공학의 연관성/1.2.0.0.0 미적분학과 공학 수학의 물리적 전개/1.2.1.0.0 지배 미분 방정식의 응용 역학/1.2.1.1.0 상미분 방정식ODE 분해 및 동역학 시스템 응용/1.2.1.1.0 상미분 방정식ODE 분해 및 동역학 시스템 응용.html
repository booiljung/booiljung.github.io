1.2.1.1 상미분 방정식(ODE) 분해 및 동역학 시스템 응용

공간적 얽힘이 배제되고 오직 시간(t)이라는 단일 독립 변수계를 축으로 삼아 진화하는 공학 시스템을 모사할 때, 인류가 휘두르는 가장 날카로운 수학적 검증 도구는 상미분 방정식(Ordinary Differential Equation, ODE)이다. 차의 서스펜션을 이루는 진동 역학, 로봇 매니퓰레이터의 모터 관성, 혹은 밀폐 용기 내 압력 팽창의 시간적 추적 등은 모두 편미분이 배제된 이 순수 ODE 도메인 위에 건설된다. 본 절에서는, 이러한 시간 역학계의 근간을 해부하여 시스템을 규정짓는 미분 방정식 체계를 수학적으로 분해(Decomposition)하고, 자연계의 동역학 시스템(Dynamic System)을 역설계(Reverse Engineering)하는 공통적 응용 메커니즘의 정수를 파헤친다.

1. 상미분 방정식의 계수(Order)와 선형성(Linearity) 확립 관측

물리계의 지배적인 시간 방정식을 마주했을 때 엔지니어가 최우선적으로 단행해야 하는 임무는 해당 텐서 방정식의 수학적 등급(Class)을 판독하는 차원 해석이다.

  1. 계수(Order)의 한계: 차 방정식에 포함된 최고차 도함수의 횟수가 곧 시스템의 역학적 계수를 결정한다.
  • 1계(1st Order) 시스템: 속도와 저항만으로 매핑되는 시스템이다. RC 전기 회로나 방사성 붕괴, 뉴턴의 냉각 법칙 등이 이에 속하며, 관성 텐서(질량)가 부재하므로 결코 스스로 왕복 진동 파형을 형성할 수 없는 단방향 소멸계이다.
  • 2계(2nd Order) 시스템: 가속도(d^2x/dt^2) 치수가 결합된 시스템이다. 위치 에너지와 운동 에너지를 상호 교환할 수 있는 관성과 탄성이 얽혀 있어 비로소 조화 진동(Harmonic Oscillation) 역학이 태동하는 본격적인 공학 동역학의 근원계가 된다.
  1. 선형성(Linearity)의 구속: 시스템 방정식을 구성하는 종속 변수 y와 그 도함수 y', y'' 들이 오직 합과 스칼라 상수 곱의 1차원으로 연합되어 있다면, 이 선형(Linear) 모델은 **중첩의 원리(Principle of Superposition)**라는 막강한 해석적 특권을 얻어, 복잡한 입력을 분해 연산한 후 결과를 합산하는 궁극의 해석적 명료함을 담보한다.

2. 동차(Homogeneous)와 비동차(Non-homogeneous)의 인과 역학

미분 방정식의 우항에 배치된 독립 변수 함수 f(t) 의 유무는 철저히 자연의 ‘외부 강제 개입’ 역학과 결부된다.

a_n \frac{d^ny}{dt^n} + \dots + a_1 \frac{dy}{dt} + a_0 y = f(t)

  • 동차 방정식(Homogeneous, f(t)=0): 외부에서 가해지는 힘이나 전압 소스가 완전히 단절(0)된 닫힌계를 상징한다. 이 방정식의 솔루션 기저인 고유 해(Complementary Function, y_c)는 오로지 시스템이 초기에 머금고 있던 위치 에너지(초기 조건)가 질량, 감쇠기, 스프링이라는 내부 인자 간의 싸움을 통해 얼마나 급격히 혹은 천천히 방전(소산)되는가를 지배하는 과도 응답(Transient Response) 거동을 지시한다.
  • 비동차 방정식(Non-homogeneous, f(t) \neq 0): 모터의 회전력이나 지진파와 같이 외부에서 지속적으로 쏟아지는 강제 함수(Forcing Function) f(t) 에 의해 시스템이 항구적인 구속을 받게 되는 역학이다. 이 강제 항은 모델에 특수 해(Particular Solution, y_p)를 발생시키며, 시간이 무한대로 흘러 내부 에너지가 모두 소멸한 이후에도 시스템이 기계적으로 따를 수밖에 없는 최종적인 정상 상태 응답(Steady-state Response) 윤곽을 지배한다.

엔지니어는 먼저 시스템의 전원을 완전히 차단(f(t)=0)한 채 내부 뼈대인 특성 방정식(Characteristic Equation)의 근을 발췌(Root finding)하여 시스템이 조화 진동할지, 곧장 추락할지의 속성을 예단해야 한다(y_c). 그 후 시스템 내부에 가동되는 외력 f(t) 에 강제로 주파수 동기화된 특수 해 체계(y_p)를 도출하여, 이 둘의 상태 벡터를 합산한다. 이로써 외부 자극과 내부 반성이 거칠게 충돌하는 t=0의 시작 격동기(과도 상태)를 지나, 궁극의 평형점(정상 상태)으로 안착하게 되는 경이로운 전 역학계의 시간 축 파이프라인을 온전히 조망하게 된다.
