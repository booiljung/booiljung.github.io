1.2.1.1.5 상태 변수 벡터와 전이 행렬 매트릭스를 통한 다자유도(MDOF) 연립 방정식 모드 분리 연산

현대 공학 구조물—초고층 마천루, 다축 제어 우주 발사체, 혹은 수백 개의 부품이 연쇄된 차량 파워트레인—은 결코 단일 질량점(Single Mass) 역학으로 해석될 수 없다. 이들은 서로 질량, 스프링, 댐퍼들로 얽히고설켜 수십, 수십만 개의 좌표 공간이 동시에 진동하는 다자유도(Multiple Degrees of Freedom, MDOF) 복합계이다. 특정 좌표계(가령 3층의 지진 변위)의 거동 방정식에는 필연적으로 2층과 4층의 변수들이 오염되어(Coupled) 침투해 있으므로, 이를 개별적으로 역산하는 행위는 본질적으로 단절되어 있다. 본 절에서는, 이 거대한 상호 간섭 연립 미분 방정식계를 단일한 상태 공간(State Space)의 특권적 벡터-행렬 텐서계로 압축하고, 고유치(Eigenvalue) 공간 분해 역학을 이용해 각각의 오염된 좌표 파동을 독립적인 모드(Modal) 주파수 진동으로 완벽히 분리해버리는 대수학의 물리적 마법을 관통한다.

1. 운동 방정식의 매트릭스화 및 시스템 간섭(Coupling) 인식

N 개의 질량 텐서가 위치 x_1, x_2, \dots, x_N 에 존재하며 상호 스프링과 댐퍼로 체인형 연쇄되어 있는 MDOF 모델을 상정하라. 각 층별로 뉴턴 제2법칙을 전개하면 개별 식들은 독립되지 못하고 거대한 행렬 역학 형태로 응축된다.

\mathbf{M} \ddot{\mathbf{x}}(t) + \mathbf{C} \dot{\mathbf{x}}(t) + \mathbf{K} \mathbf{x}(t) = \mathbf{F}(t)

  • 질량 행렬(Mass Matrix, \mathbf{M}): 대개 대각 행렬(Diagonal Matrix) 특성을 띄나 자이로스코프 관성계나 회전 조향 연동에서는 비대각 성분(Off-diagonal)이 나타나는 관성 텐서군이다.
  • 강성 행렬(Stiffness Matrix, \mathbf{K}) 및 감쇠 행렬(\mathbf{C}): 시스템의 기하 역학 간섭을 주도한다. 비대각 성분에 위치한 k_{ij} 텐서들은 지점 j 의 찰나적 변동이 지점 i 에 물리적 탄력으로 직격되는 상호 연성(Coupling) 오염도의 실체이다.

이러한 상호 지배 연립 미분 공간 속에서는 어떠한 단방향 솔루션 궤적도 도출될 수 없으므로, 엔지니어는 모달 행렬 공간(Modal Matrix Space)이라는 4차원 대수 통로로 이 시스템을 강제 전송시킨다.

2. 상태 변수 배열과 상태 공간 방정식(State-Space Equation) 텐서화

차원이 크고 2계 미분이 포함된 MDOF 연립 체계를 컴퓨팅 시뮬레이션에 무리 없이 이식하기 위하여, 제어 공학의 코어 역학인 상태 공간 방정식(State-Space Model) 변환법이 개입된다.
기존의 N 차원 변위 벡터 \mathbf{x} 와 그 1차 도함수인 속도 벡터 \dot{\mathbf{x}} 를 통합하여, 하나의 거대한 2N 차원 상태 벡터(State Vector) 치수 \mathbf{z}(t) = \begin{bmatrix} \mathbf{x}(t) \\ \dot{\mathbf{x}}(t) \end{bmatrix} 로 재조립하라.

이후 시스템 매트릭스는 거대한 시스템 전이 행렬(System Transition Matrix) \mathbf{A} 를 수반하는 1계(1st Order) 상태 방정식망으로 강하된다.

\dot{\mathbf{z}}(t) = \mathbf{A} \mathbf{z}(t) + \mathbf{B} \mathbf{u}(t)

\mathbf{A} = \begin{bmatrix} \mathbf{0} & \mathbf{I} \\ -\mathbf{M}^{-1}\mathbf{K} & -\mathbf{M}^{-1}\mathbf{C} \end{bmatrix}

무질서하게 간섭하던 2계 다항식 다발들이, 제어 신호 \mathbf{u}(t) 와 단일 매트릭스 \mathbf{A} 가 교차 통제하는 고도로 세련되고 압축된 하나의 1계 선형 미분 방정식 체계로 제압되는 순간이다. 공학자는 이제 오직 저 \mathbf{A} 행렬 텐서망 내부에 고립된 극점(Eigenvalues) 정보만을 해체함으로써 전체 시스템 수만 개의 다이내믹 궤적 성향을 확정짓는다.

3. 고유치 모드 분리 연산과 전이 궤도 디커플링(Decoupling)

매트릭스 텐서계 내에 잔존하는 좌표 간섭(Coupling)을 영구적으로 박살내기 위해 모달 해석(Modal Analysis)이 도래한다. 행렬 \mathbf{M}^{-1}\mathbf{K} 가 보유한 고유벡터(Eigenvector) 매트릭스 \mathbf{\Phi} 를 축출하라. 이 축 행렬 \mathbf{\Phi} 는, 무질서한 층간 변위 좌표 다발 \mathbf{x}(t) 를 각자 서로 물리적으로 독립된 진동 패턴을 지닌 직교(Orthogonal) 모달 좌표계 \mathbf{q}(t) 로 치환하는 다이아몬드 차원 관문이다. (\mathbf{x} = \mathbf{\Phi} \mathbf{q})

변환 행렬을 앞뒤 선형 대수로 곱하여 축을 뒤틀어버리면(예: \mathbf{\Phi}^T \mathbf{M} \mathbf{\Phi}), 이전의 복잡한 강성 및 질량 행렬들은 오직 대각선(Diagonal) 요소들만 살아남는 완벽한 대각 행렬(Diagonal Matrix) 차원이 강제된다.
즉 수천 스케일로 연립되어 엔지니어를 조롱하던 방정식 텐서망이 공간 좌표축을 ’물리적 위치’에서 ’주파수 모드 형상(Mode Shape)’으로 회전시키자마자, 완전히 뿔뿔이 흩어진 독립적인 N 개의 단일 질량-스프링(1-DOF) 미분 방정식들로 깔끔하고 우아하게 토막 나 분해(Decoupled)된다. 이 상태 변수 대각화 역학이야말로 다자유도 시스템을 정복하는 선형 대수학-미분 역학의 가장 위대하고 아름다운 결합 예술이라 칭송받는다.
