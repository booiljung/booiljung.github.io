1.2.1.1.4 룬게-쿠타(Runge-Kutta) 4차법을 활용한 로렌츠(Lorenz) 카오스 시스템 수치 적분 메커니즘

강제 외력이 투입된 2계 선형 진동 방정식처럼, 인간의 지성인 적분학(Analytical Integration)이 그 명시적(Explicit) 궤적 솔루션을 토해낼 수 있는 미분 방정식은 대자연에서 극히 일부의 파편에 불과하다. 마찰 계수가 속도의 제곱에 비례하는 비선형 공기 저항 역학, 혹은 에드워드 로렌츠(Edward Lorenz)가 기상 모델링 중 발견한 연립 비선형 카오스(Chaos) 시스템 등은 절대 대수적 적분기로 붕괴되지 않는다. 이 난공불락의 비가역 시스템에 맞서 공학자는 해석학적 자존심을 꺾고 이산화(Discretization)된 컴퓨팅 연산 권력을 동원하는 수치 해법(Numerical Method)으로 접근해야만 한다. 본 절에서는, 수치 적분계의 전설적 워크호스(Workhorse)인 룬게-쿠타 4차법(Runge-Kutta 4th Order Method, RK4)의 미시적 기울기 합산 메커니즘을 규명하고, 그것이 그 유명한 로렌츠 어트랙터(Lorenz Attractor)의 초기 조건 민감성을 어떻게 화면에 직조해 내는지를 살핀다.

1. 룬게-쿠타 4차법(RK4)의 기하학적 텐서 융합 메커니즘

어떤 상태 변수 y(t) 의 변화율이 t 와 y 에 모두 종속되는 1계 비선형 상미분 방정식 \frac{dy}{dt} = f(t, y) 을 관측하라. 가장 원시적인 오일러 공식(Euler Method)은 오직 현재 시간 t_n 에서의 단일 기울기 f(t_n, y_n) 망을 맹신하여 다음 궤적을 억측하기에, 한 스텝 h 가 전진할 때마다 기하학적 곡률의 왜곡 오차(Truncation Error)가 걷잡을 수 없이 증폭된다.

독일 수학자 카를 룬게(Carl Runge)와 마르틴 쿠타(Martin Kutta)는 이 절명적 오차 곡선을 억제하기 위해, 한 스텝 \Delta t = h 공간 내부 속을 4개의 중간 지점으로 분할 타격하고 각각의 미시적 기울기(K-tensor)를 가중 평균하는 치밀한 작전을 전개한다.

k_1 = f(t_n, y_n)
k_2 = f\left(t_n + \frac{h}{2}, y_n + \frac{h}{2}k_1\right)
k_3 = f\left(t_n + \frac{h}{2}, y_n + \frac{h}{2}k_2\right)
k_4 = f(t_n + h, y_n + h k_3)

이후, 최종 스텝 진화 점 y_{n+1} 은 이 4개의 K-텐서를 1:2:2:1 의 심프슨 법칙(Simpson’s Rule) 비율로 융합하여 결정된다.

y_{n+1} = y_n + \frac{h}{6} (k_1 + 2k_2 + 2k_3 + k_4)

k_1 이 현재의 찰나적 맹검 기울기라면, k_2 와 k_3 는 \frac{1}{2} 스텝 전진한 구간 내부에서의 중간 예측 기울기이고, k_4 는 스텝 끝단의 기울기이다. 이 중간 지점들의 교차 검증 덕분에 RK4는 단순히 스텝 h 의 1제곱이 아닌, 무려 h^4 스케일에 비례하여 맹신의 오차를 소거해버리는 압도적 정확도를 자랑하며 모든 과학 연산 라이브러리(ODE Solver)의 디폴트 심장부로 군림하게 되었다.

2. 로렌츠 시스템(Lorenz System): 카오스의 방정식

대형 컴퓨터조차 쩔쩔맬 정도로 복잡한 비선형 연립 방정식만이 기괴한 궤적을 만들 것이란 인류의 통념을 로렌츠가 1963년 단 세 줄의 방정식으로 깨부수었다. 대기 대류(Atmospheric Convection) 롤(Roll) 현상의 역학 공간을 극도로 축약한 로렌츠 모델은 오직 x, y, z 상태 공간 축을 배회한다.

\frac{dx}{dt} = \sigma (y - x)
\frac{dy}{dt} = x (\rho - z) - y
\frac{dz}{dt} = x y - \beta z

프란틀 수(Prandtl Number, \sigma), 레일리 수(Rayleigh Number, \rho), 기하학 비율(\beta)로 구성된 이 모델의 심연은 x z 와 x y 라는 단 두 개의 미약한 곱빼기 비선형항이다. 그러나 이 작은 교란 텐서는 시스템 내부에 나비 효과(Butterfly Effect)—초기 조건의 우주적 민감도(Sensitive Dependence on Initial Conditions)—라는 카오스의 악마를 잉태시킨다.

비선형 역학의 극한을 체감하는 이 전개도는, 해석적 불가능성의 절망을 RK4라는 이산 수학 수치 텐서망으로 강행 돌파하여, 대자연에 내재된 숨 막히는 카오스 철학형 형이상학의 실체를 모니터 공간에 그려낸 공학 연산의 금자탑이라 하겠다.
