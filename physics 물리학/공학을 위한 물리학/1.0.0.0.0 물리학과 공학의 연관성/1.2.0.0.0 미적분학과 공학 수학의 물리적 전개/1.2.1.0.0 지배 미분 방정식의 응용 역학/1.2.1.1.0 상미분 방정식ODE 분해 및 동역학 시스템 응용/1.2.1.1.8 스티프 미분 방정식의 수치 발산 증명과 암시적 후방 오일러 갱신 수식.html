1.2.1.1.8 스티프 미분 방정식의 수치 발산 증명과 암시적 후방 오일러 갱신 수식

공학의 역학 시스템 중에는 극단적으로 이질적인 시간 척도(Time Scale)들이 하나의 방정식 안에 잔혹하게 공존하는 현상들이 무수히 존재한다. 찰나의 순간에 폭발적으로 연소하는 화학 기체의 압력 상승과, 몇 시간에 걸쳐 서서히 방열되는 엔진 블록의 온도 하강이 동시에 연립되어 있는 시스템이 그 예이다. 이처럼 어떤 성분은 파괴적으로 빠르게 붕괴하고, 다른 성분은 지루하게 천천히 변화하는 연립 미분 방정식계를 **스티프 미분 방정식(Stiff Differential Equation)**이라 부른다. 룬게-쿠타 4차(RK4)를 포함한 대부분의 명시적(Explicit) 적분기들은 이 ’스티프(Stiff)’한 강성을 마주하면 해가 완전히 붕괴해버리는 수치 발산(Numerical Divergence)의 치명상에 직면한다. 본 절에서는 명시적 오일러(Explicit Euler) 모델의 발산성을 수학적으로 증명하고, 이를 극복할 유일무이한 구원책인 암시적 후방 오일러(Implicit Backward Euler)의 절대 안정성(Absolute Stability) 메커니즘을 규명한다.

1. 명시적 오일러 진격의 수치계 발산 증명

가장 극한의 스티프 모델 테스트베드인 1계 선형 감쇠 방정식 y' = \lambda y (단, \lambda \ll 0)를 상정하라. 여기서 \lambda 가 매우 큰 음수라는 것은 시스템이 t=0 이 시작되자마자 원점 y=0 을 향해 수직 강하에 가깝게 추락하는 강한 소산 역학을 뜻한다.
이 방정식의 진해(Exact Solution)는 y(t) = y(0) e^{\lambda t} 이며, 시간 t \to \infty 에 따라 완벽하게 0으로 소멸한다. 이제 현재의 점 y_n 에서 접선의 기울기로 미래 y_{n+1} 을 타격하는 ‘명시적 오버추어’ 행위인 명시적 전방 오일러(Explicit Forward Euler)법을 전개하라.

y_{n+1} = y_n + h f(t_n, y_n) = y_n + h \lambda y_n = (1 + h \lambda) y_n

여기서 수치 적분이 한 스텝씩 진행될 때마다 y_n 에 곱해지는 증폭 계수, 즉 증폭 인자(Amplification Factor)는 |1 + h\lambda| 로 확정된다.
수치 솔루션이 진해처럼 원점 0 으로 무사히 감쇠 수렴하기 위해서는 이 증폭 인자의 절대값이 반드시 1보다 작아야 한다는 **절대 안정성 영역(Region of Absolute Stability)**이 선언된다.

|1 + h\lambda| \le 1 \implies -2 \le h\lambda \le 0 \implies h \le \frac{2}{|\lambda|}

이러한 수식이 폭로하는 진실은 잔혹하다. |\lambda| 의 스케일이 10^6 정도로 거대한 전기-화학 반응망(Stiff System)이라면, 엔지니어가 수치 폭주를 막기 위해 설정해야 할 허용 스텝 h 는 2 \times 10^{-6} 이하로 산산조각 난다. 단 1초를 연산하기 위해 수백만 번의 스텝을 낭비해야 하는, 소위 시간 단위당 컴퓨팅 자원의 출혈 감염이 발발하는 것이다. 만일 스텝 h 가 이 임계 스케일을 조금이라도 초과한다면, (1+h\lambda) 는 -1을 뚫고 끝없는 양/음의 부호 교대를 일으키며(오버슛 발산), 컴퓨터의 메모리를 오버플로우(Overflow)로 침몰시킨다.

2. 암시적 후방 오일러(Implicit Backward Euler)의 구원 텐서

명시적법의 패배는 시스템이 ’현재 기울기(f_n)’를 맹신하고 미래 공간으로 무리하게 발을 내딛기 때문에 기인한다. 이를 타파하고자 후방 오일러 갱신(Backward Euler Update) 기법은, 미지에 감싸져 아직 당도하지 않은 ‘미래의 궤적(f_{n+1})’ 지점에서 측정한 기울기를 거꾸로 투사하여 방금 전의 족적 결착을 완성하는 기이한 암시적(Implicit) 타임머신 루프를 채택한다.

y_{n+1} = y_n + h f(t_{n+1}, y_{n+1})

동일한 스티프 테스트 방정식 f(t, y) = \lambda y 에 위 암시적식을 체결하라.

y_{n+1} = y_n + h \lambda y_{n+1}

이 수식은 대수 연산을 통해 다음과 같이 미래 상태 y_{n+1} 만을 좌항으로 고립시키는 역산(Inversion) 매커니즘으로 환원된다.

y_{n+1} = \frac{1}{1 - h\lambda} y_n

3. 무조건적 절대 안정성(A-Stability)의 수복

암시적 궤도가 품고 있는 증폭 계수는 우항의 \left| \frac{1}{1 - h\lambda} \right| 이다. 놀라운 역학이 발생한다. \lambda \ll 0 인 한, h 가 그 어떤 거대한 10, 1000, 10000 의 스텝 스케일을 취하더라도 분모인 (1 - h\lambda) 는 항상 1보다 커지며, 증폭 인자는 영원히 1보다 작다.

이 무조건적 진정력을 **A-안정성(A-Stability)**이라 명명한다. 스티프 방정식 공학 모델에서 후방 오일러나 암시적 룬게-쿠타, BDF(Backward Differentiation Formula) 등의 암시적 방법은 컴퓨팅 연산의 구원자이다. 비록 양함수(y_{n+1} = \dots) 꼴을 역산하기 위해 매 스텝 뉴턴 랩슨(Newton-Raphson)과 같은 역행렬(Jacobian Inverse) 해법을 거쳐야만 하는 무거운 계산 부하가 발생하지만, 명시적 기법이 수만 번 떨어야 하는 살인적 스텝 분할망을 굵직하고 안전한 거대 단일 스텝들로 단박에 돌파해 내기 때문이다. 공학 시뮬레이션의 근간에서 다중 척도 시간계가 엮일 때, 오직 이 암시적 대수 모델만이 대변동의 쓰나미 속에서도 물리적 우주의 침착한 소산성을 컴퓨터 화면 위에 이식할 수 있다.
