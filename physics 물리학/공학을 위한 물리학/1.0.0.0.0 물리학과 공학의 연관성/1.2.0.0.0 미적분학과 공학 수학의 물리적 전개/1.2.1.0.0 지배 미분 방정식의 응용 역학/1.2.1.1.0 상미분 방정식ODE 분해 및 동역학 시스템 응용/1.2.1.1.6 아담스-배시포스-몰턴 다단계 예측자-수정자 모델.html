1.2.1.1.6 아담스-배시포스-몰턴 다단계 예측자-수정자 모델

앞장의 룬게-쿠타(Runge-Kutta) 4차법이 단 한 번의 시간 스텝을 전진하기 위해 방정식 내부를 4번이나 강제 평가(Evaluation)하는 압도적 무력을 과시했다면, 거대 규모의 유체 동역학이나 N-체(N-body) 우주 궤도 역학 시뮬레이션에서는 이 매번의 연산 비용이 치명적인 병목(Bottleneck)으로 변모한다. 이에 맞서 수치 해석학자들은 단일 스텝(Single-step)의 폐쇄성에서 벗어나, 시스템이 걸어온 과거의 궤적(History) 텐서들을 기억하고 조합하여 미래를 투시해 내는 다단계 방법(Multi-step Method)을 창조하였다. 그 정점에 서 있는 것이 바로 개방형(Explicit) 외삽인 아담스-배시포스(Adams-Bashforth) 예측 역학과, 폐쇄형(Implicit) 내삽인 아담스-몰턴(Adams-Moulton) 수정 역학을 직렬 결합한 예측자-수정자(Predictor-Corrector) 복합 알고리즘이다.

1. 아담스-배시포스(Adams-Bashforth)의 외삽 예측계 (Predictor)

미분 방정식 y' = f(t, y) 를 적분 구간 t_n 에서 t_{n+1} 로 전진시키기 위해 과거의 좌표들을 이끌어 내라. 4단계 아담스-배시포스 방법(AB4)은 현대 스텝 단 하나가 아닌, 이미 계산되어 저장된 과거 4개의 스텝(t_n, t_{n-1}, t_{n-2}, t_{n-3})에서의 기울기 f 값들을 소환한다.
이 4개의 과거 기울기 점을 역학적 앵커로 삼아 3차 라그랑주 보간 다항식(Lagrange Interpolating Polynomial)을 펼쳐 미래의 스텝 구간으로 외삽(Extrapolation)을 강행한다.

y_{n+1}^{(P)} = y_n + \frac{h}{24} \left[ 55 f_n - 59 f_{n-1} + 37 f_{n-2} - 9 f_{n-3} \right]

이 예측값 y_{n+1}^{(P)} 의 압도적인 강점은 룬게-쿠타처럼 방정식을 4번 푸는 것이 아니라, 과거의 값은 메모리에 캐싱(Caching)해 두고 오직 단 한 번만의 곱셈 합산으로 미래의 위치를 단숨에 억측(Predict)해낸다는 연산 효율성에 있다.

2. 아담스-몰턴(Adams-Moulton)의 내삽 수정계 (Corrector)

그러나 오직 과거의 경험만을 토대로 미래로 던져진 예측값 y_{n+1}^{(P)} 는 발산의 위험성을 품는다. 다단계 알고리즘은 이 억측된 값을 진값으로 인정하지 않고 즉각 검증대 위로 올린다. 이 예측값을 지배 방정식 f 에 투입하여, 아직 오지 않은 미래 위치에서의 임시 종점 기울기 f(t_{n+1}, y_{n+1}^{(P)}) 를 새롭게 개산(Evaluate)한다.
여기에 폐쇄형 4차 아담스-몰턴 방법(AM4) 역학이 개입한다. 가장 오래된 과거 데이터 하나(f_{n-3})를 폐기하고, 방금 확보한 미래의 임시 기울기(f_{n+1}^{(P)})를 편입시켜 구간 내를 강하게 구속하는 내삽(Interpolation) 다항식을 다시 짜부라뜨려 최종 종착점을 묵시적으로 수정(Correct)한다.

y_{n+1}^{(C)} = y_n + \frac{h}{24} \left[ 9 f(t_{n+1}, y_{n+1}^{(P)}) + 19 f_n - 5 f_{n-1} + f_{n-2} \right]

미래의 기울기 정보까지 선반영된 이 폐쇄형 수정식은 수치계의 절대 안정성(Stability)을 부여하는 물리적 댐퍼 역할을 완벽히 수행한다.

2. PECE(Predict-Evaluate-Correct-Evaluate) 알고리즘 병합 루프

예측자(Predictor)와 수정자(Corrector)를 직렬로 엮어 기어링(Gearing)하는 이 엔진의 1사이클 거동은 PECE 관문으로 구획된다.

만약 수정된 값과 예측된 값의 기하학적 난단 오차(Truncation Error)가 허용치를 초과한다면, 시스템은 수정된 y_{n+1}^{(C)} 를 임시 예측값의 자리에 다시 끼워 넣고 수정자(Corrector) 수식을 반복 연산하여 무결점으로 수렴해 나가는 강력한 통제력을 획득한다.
단, 이 다단계 모델은 치명적인 시스템적 태생 한계를 수반한다. 최초 t=0 시점에는 과거의 궤적(4개의 텐서)이 전혀 존재하지 않기 때문에 원천적으로 부팅이 불가능하다. 따라서 이 거대한 예측자-수정자 크루즈 엔진이 자가 동력을 얻을 때까지, 최초의 3~4스텝은 반드시 단일 스텝의 제왕인 룬게-쿠타 4차법(RK4)을 보조 모터로 점화시켜 마중물 데이터를 채운 후, ABM 궤도로 이륙시키는 이중 하이브리드(Hybrid) 연산 설계가 필수적으로 요구된다.
