1.2.1.1.3 비선형 단진자(Pendulum) 대변위 운동 방정식의 위상 평면(Phase Plane) 특이점 궤적 역학

자연계의 수많은 공학 시스템을 모델링할 때, 엔지니어들은 종종 계산의 편의를 향해 타협한다. 예를 들어 단진동(Simple Harmonic Motion)을 묘사하는 단진자(Simple Pendulum) 계에서 발생 각도 \theta 가 매우 미세하다고 억측하여 \sin(\theta) \approx \theta 로 취환하는 선형화(Linearization) 기법이 대표적이다. 그러나 놀이기구의 바이킹, 굴착기의 매달린 크레인 붐, 혹은 시계 추의 과도 탈선 등 대변위(Large Displacement)를 수반하는 극한 거동 상황에서는 이 선형화 기만술이 처참히 격파당하고, 예측 궤적은 실제 궤도에서 돌이킬 수 없이 이탈해 버린다. 본 절에서는, 비가역적인 비선형(Nonlinear) 삼각함수 텐서를 그대로 존치시킨 채, 기하학적인 위상 공간(Phase Space) 도메인을 빌려 특이점(Singularities)과 세퍼라트릭스(Separatrix, 궤적 분리선)가 연출하는 비선형 단진자의 날 것 그대로의 역동적 만유 궤도를 낱낱이 파헤친다.

1. 대변위 단진자의 비선형 지배 방정식 도출

질량 m 인 강체가 끈의 길이 L 로 매달려 중력 g 의 지배를 받으며, 회전의 마찰 감쇠 계수가 c 인 단진자 역학 시스템을 관측하라. 뉴턴의 회전 운동 제2법칙(\Sigma M = I \alpha) 또는 라그랑주 역학계에 의해 각변위 \theta(t) 가 따라야 할 시간 지배 운동 방정식은 다음과 같이 선포된다.

m L^2 \frac{d^2\theta}{dt^2} + c \frac{d\theta}{dt} + m g L \sin(\theta) = 0

시간 미분을 뉴턴 역학의 닷(Dot) 표기법으로 환산하여 각 가속도 텐서 \ddot{\theta} 만을 분리 추출하면 다음과 같다.

\ddot{\theta} = - \frac{c}{m L^2} \dot{\theta} - \frac{g}{L} \sin(\theta)

이 수식의 심장부에 똬리를 튼 $ \sin(\theta) $ 항이야말로 선형 대수학적 해법을 모조리 좌초시키고 시스템을 비동차·비선형의 영겁으로 몰아넣는 블랙홀이다. 위 수식은 해석적인 명시적 솔루션(Analytical Explicit Solution) $ \theta(t) = \dots $ 형태로 결코 종결지어지지 않으며 타원 적분(Elliptic Integrals)이라는 특수 기하 공간으로의 우회를 강제한다.

2. 상태 공간(State-Space) 환원과 위상 평면(Phase Plane) 매핑

이 해석 불가능한 2계 비선형 상미분 방정식을 시각적 기하 역학으로 굴복시키기 위해, 프랑스 수학자 앙리 푸앵카레(Henri Poincaré)가 창시한 위상 평면(Phase Plane) 벡터 장 분해 기법을 소환한다. 2계 미분 방정식을 두 개의 연립된 1계 미분 방정식으로 찢어 상태 변수(State Variable) 텐서로 재배열하라. 상태 변수를 각도 x_1 = \theta 와 각속도 x_2 = \dot{\theta} 로 투사하면 지배 방정식은 다음처럼 하강 분리된다.

\dot{x}_1 = x_2
\dot{x}_2 = - \frac{g}{L} \sin(x_1) - \frac{c}{m L^2} x_2

엔지니어는 이제 시간 t 축을 완전히 폐기 처분하고, 가로축을 변위(x_1=\theta), 세로축을 속도(x_2=\dot{\theta})로 삼는 2차원 평면, 즉 위상 평면 위에 \left[ \dot{x}_1, \dot{x}_2 \right]^T 의 속도 벡터 텐서들을 거대한 바람장(Vector Field)처럼 빼곡히 그린다. 시스템의 동적 거동은 이 벡터장을 따라 흘러가는 유역(Flow)의 기하학적 띠(Trajectory)로 화려하게 현형한다.

3. 특이점(Singular Point)과 세퍼라트릭스(Separatrix) 역학 분기점

시스템의 모든 미분 변화율이 완전히 0이 되어 정지하게 되는 평형점, 즉 특이점(임계점, Singularities)은 (\dot{x}_1=0, \dot{x}_2=0) 을 충족하는 좌표이다. 위성 평면상에서 이 좌표계의 해는 x_2 = 0 이고 \sin(x_1) = 0 인 지점, 즉 \theta = 0, \pm\pi, \pm2\pi, \dots 의 궤적들에서 창발한다.

  • 중력 하강 중심점(Center / Stable Focus): 진자가 정확히 바닥을 향해 수직으로 늘어진 \theta = 0 기점이다. 에너지가 주어지면 속도 벡터들은 이 중심을 시계 방향 타원 궤적으로 공전(진동)한다. 감쇠 c>0 이 작동하면 점차 에너지를 상실하며 소용돌이(Spiral)를 그리며 원점으로 소멸 수렴하는 절대적 안정성의 닻이다.
  • 중력 역전 안장점(Saddle Point): 진자가 연직 위를 향해 불안하게 역으로 서 있는 \theta = \pm\pi 특이점 좌표다. 이 지점에서는 파라미터가 미세한 섭동(Perturbation)만 받아도 한쪽 방향은 빨려 들어가고 반대 방향은 극단적으로 우주로 밀어내는 치명적인 안장점(Saddle) 텐서계가 형성된다.
  • 세퍼라트릭스(Separatrix): 이 안장점들을 절묘하게 양끝으로 연결하며 이어지는 아치 형태의 절대 경계선이다. 만일 이 경계선 외부로 이탈할 수준의 과도한 운동 에너지가 \dot{\theta} 로 주입된다면? 진자는 더 이상 왔다 갔다 하는 ’진동’이 아니라, 중심축을 회전 모터마냥 360도 끊임없이 팽글팽글 맴도는 **연속 회전(Whirling Mode)**의 파동계로 영구 탈주해버린다.

비선형성을 배제 없이 응시하는 이 위상 평면 시각화 기법은, 미분 방정식 안에 감춰진 카오스적 분기점(Bifurcation)과 진동-회전의 절대 경계 영토망을 평면 좌표 역학의 해부도로서 엔지니어의 망막에 완벽히 출력해 내는 현대 동역학의 등대이다.
