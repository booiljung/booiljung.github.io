1.2.1.1.7 경계값 문제(BVP) 해결을 위한 사격법(Shooting Method) 알고리즘 및 뉴턴 반복법 결합

지금껏 다루어 온 진동 궤적 추적이나 카오스 시뮬레이션은 모두 t=0 시점에 변위와 속도가 동시에 주어지고 그 초기점으로부터 맹렬히 진격해 나가는 초기값 문제(Initial Value Problem, IVP) 영역에 소속된다. 반면 공학에는 축의 양단이 지지대에 묶인 보(Beam)의 휨 변형이나 두 방열판 사이의 정상 상태 열전도 분포처럼, 독립 변수의 ’시작점(x=a)’과 ‘끝점(x=b)’ 각 양끝에서 상태가 결박되어 지배받는 공간적 역학이 존재한다. 이처럼 시종이 갇힌 차원을 **경계값 문제(Boundary Value Problem, BVP)**라 칭한다. 초기 기울기가 미궁에 빠진 BVP를 풀기 위해 미분 방정식 역산기는 탄도학에서 차용한 우회 타격술을 가동하는데, 그것이 바로 대포의 앙각을 조정하여 표적에 명중시키는 원리를 빼닮은 사격법(Shooting Method)과 뉴턴 반발 알고리즘의 맹렬한 융합이다.

1. 초기값 문제(IVP)와 경계값 문제(BVP)의 단절 역학

2계 지배 상미분 방정식 y'' = f(x, y, y') 에 걸린 구속 조건의 태세를 해체하라.

  • IVP 공간: x=0 에서 위치 y(0) = \alpha 와 시작 기울기 y'(0) = v_0 가 한곳에서 동시에 투척된다. 룬게-쿠타(RK4)를 동원해 그저 전방으로 뻗어 나가면 궤적은 일방향성솔루션으로 해골이 완성된다.
  • BVP 공간: 시작점 x=a 의 위치 y(a) = \alpha 는 주어지나, 치명적이게도 초기 기울기 y'(a) 는 단절(Unknown)되어 있다. 대신 수만 킬로미터(혹은 유한 미터) 떨어진 종착점 x=b에서의 목표 고도 y(b) = \beta 에 도달해야만 한다는 결과적 짐을 물리적으로 짊어져 버린다. 기계적 수치 적분기들은 0초 시점의 속도(기울기)를 입력받지 못하면 단 한 걸음의 전진 연산조차 수행할 수 없는 맹점에 치명타를 입는다.

2. 사격법(Shooting Method)의 탄도 최적화 궤도

사격법의 철학은 결핍된 BVP를 강제로 IVP로 위장시켜 수치 연산기를 기만하는 데서 발아한다. 대포를 쏠 때 표적(\beta)에 맞추기 위해 임의의 앙각(가상의 초기 기울기 z)을 설정하고 일단 포탄을 쏴보는 역학 메커니즘이다.

  1. 가상의 기울기 사격: y'(a) = z_1 이라는 임의의 초기 기울기 탄도를 억측(Guess)해 낸다.
  2. IVP 강행 적분: 이제 y(a)=\alpha, y'(a)=z_1 이 확보되었으므로 RK4 등을 엔진 삼아 표적 공간 x=b 까지 맹목적 전진 포격을 수행한다.
  3. 오차(Error) 추적: 도착점에서의 솔루션 y(b, z_1) 스칼라를 실제 원했던 진값 타겟 \beta 와 충돌시켜 탄도의 잔차(Residual) 역학 E(z_1) = y(b, z_1) - \beta 를 발췌한다.

이 잔차 함수 E(z) = 0 이 되도록 만드는 단 하나의 마법 같은 앙각 z^* 를 색출하는 과정이 사격법을 완전한 역학 솔루션으로 승화시킨다. 선형(Linear) 방정식 조준 하에서는 단 두 번의 시험 사격(z_1, z_2)만으로 보간 유도선을 이끌어내어 진값을 단번에 타격하지만, 비선형 BVP 공간에서는 뉴턴-랩슨(Newton-Raphson) 텐서 반복 수복법이 전면 개입한다.

3. 뉴턴-랩슨(Newton-Raphson) 알고리즘 병합 통제

임의의 앙각 함수 잔차를 \mathbf{0} 좌표로 때려 부수기 위해, 사격 엔진 표적지 오차 함수 E(z) 의 기울기 변화율 \frac{\partial E}{\partial z} 야코비안(Jacobian) 텐서를 역산하는 루프(Loop)를 시전한다.

미분 계수 \frac{\partial E}{\partial z} 는 인접한 다른 가상 탄도(z + \Delta z)를 쏴본 편차 데이터로 근사하며, 시스템은 z_{k+1} = z_k - \frac{E(z_k)}{E'(z_k)} 공식을 따라 스스로 포신의 앙각을 조준 수정한다. 서너 번의 가상 사격(Iterations)이 공기를 가르고 나면, 뉴턴법의 자비 없는 2차 수렴 속도로 인해 포탄은 결국 경계점 \beta 를 정확히 관통(Shooting)하게 된다. 결국 BVP의 사격법은 공학자가 제어할 수 없는 경계의 불일치성(Mismatch) 굴레를, IVP 수치 적분기와 구근 탐색기(Root-Finding)의 결합이라는 경이로운 메타 연산 알고리즘으로 분쇄 통과해버리는 창의적 승리의 전형을 증명한다.
