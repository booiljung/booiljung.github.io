1.2.1.2.8 극초음속 충격파 해석을 위한 특성선법(Method of Characteristics) 기반 비선형 쌍곡선 장 이론

마하 수(Mach Number) 5를 초월하여 대기권을 돌진하는 우주 왕복선이나 극초음속 미사일의 표면에서는, 유동의 파동 전파 속도보다 비행체의 속도가 압도적으로 빨라져 버려 공기가 앞으로 회피할 틈조차 주어지지 않는다. 이 극단의 관성 충돌 구역에서는 완만하고 연속적인 미적분학의 전제가 스스로 찢겨지며, 밀도, 압력, 온도가 불과 수 밀리미터 단위 안에서 불연속적으로 폭발 상승하는 초자연적 파단면, 즉 **충격파(Shock Wave)**가 벼락처럼 잉태된다. 수치 적분기들이 매끄러운 다항식으로 이 단층 파쇄면을 해석하려 들다간 즉각적인 발산과 초물리적 깁스 현상(Gibbs Phenomenon)에 궤멸당하고 만다. 이 충격파와 팽창파 구역의 불연속장을 정확히 절제하여 수술하기 위해 공학은 비선형 쌍곡선 방정식(Nonlinear Hyperbolic Equation)의 본질인 파동의 생명선, **특성선법(Method of Characteristics, MOC)**을 강제 조준한다.

1. 비선형 쌍곡선 편미분 텐서의 파동 적층 역학

공기 역학의 압축성 유동(Compressible Flow)에서 음속 장벽(Sonic Wall)을 돌파한 공기의 연속체 운동량 방정식은 비선형 쌍곡선 PDE 모델인 \frac{\partial u}{\partial t} + u \frac{\partial u}{\partial x} = 0 (비점성 버거스 방정식, Inviscid Burgers’ Equation) 의 체계로 환원된다.

일반적인 파동 방정식이 고정된 선형 파동 속도 c 를 지닌 반면, 비선형 쌍곡선 장의 치명적인 역학 구도는 전파 속도 자체가 현재의 유동 상태 변수 u 에 의해 지배받는다는 점이다. 즉, 파동의 산 꼭대기(높은 속도/가속 영역) 부분이 골짜기 바닥(느린 영역)보다 더 맹렬히 전방 공간을 파먹으며 진격한다. 이 미학적 비선형성은 시간이 지날수록 완만하던 파형을 극단적으로 비대칭화 시켜 전면이 깎아지른 수직 절벽으로 일어서게 하고, 마침내 하나의 공간 좌표에 여러 개의 다중 물리 상태들이 겹치게 되는 물리적 붕괴, 이른바 파단(Breaking)으로 비화시킨다. 이 일어선 깎아지른 수직 구배가 바로 수학적 도메인 위에 벼락처럼 꽂혀 내리는 기하학적 **충격파(Shock Wave)**의 탄생 메커니즘이다.

2. 특성선법(Method of Characteristics): 편미분의 상미분 강등 텐서망

강력한 불연속 충격파와 프란틀-마이어 팽창파(Prandtl-Meyer Expansion Wave)가 공존하는 극초음속 유동장은 고정된 격자망 유한 차분법(FDM)으로는 파벽을 뚫지 못한다. 가혹한 비선형 구배에 직면한 편미분 해석자들은 공간의 그리드를 강제하는 대신, 정보(에너지 상태)가 전파되어 나가는 ‘파동의 고유한 경로’ 자체에 올라타는 좌표 변환을 시도하는데 이것이 바로 특성선법(MOC) 이다.

특성선(Characteristic Line) \frac{dx}{dt} = u(x,t) 곡선을 따라 이동하며 유동 관찰 카메라를 밀착시킨 체인 룰(Chain Rule) 전미분(Total Derivative) 텐서를 펼쳐라.

\frac{du}{dt} = \frac{\partial u}{\partial t} + \frac{dx}{dt} \frac{\partial u}{\partial x} = \frac{\partial u}{\partial t} + u \frac{\partial u}{\partial x} = 0

압도적이다. x, t 에 무수히 얽혀 있던 복잡한 편미분 방정식(PDE) \frac{\partial u}{\partial t} + u \frac{\partial u}{\partial x} = 0 이, 특정 속도 경로 \frac{dx}{dt} = u 를 따라가기만 하면 단 한 줄의 상미분 방정식(ODE) 분쇄 선언 \frac{du}{dt} = 0 로 환골탈태한다. 즉, 이 특성선 위에서 입자의 속성 u 는 시간의 흐름을 초월하여 영구히 보존(Constant)된다는 불변의 기하학적 궤도가 입증된 것이다.

3. 마하선(Mach Line)과 리만 불변량(Riemann Invariants)망의 교차 파괴

마하 M > 1 의 초음속 유체 궤도에서는 노즐 벽면이나 물체 전연부의 교란이 오직 하류의 제한된 원뿔 곡면, 즉 마하 콘(Mach Cone) 에테르망 안으로만 침투한다. 이 콘을 형성하는 파동의 등고선을 **마하선(Mach Line)**이라 부른다.

  • 마하 각(Mach Angle): \mu = \sin^{-1}\left(\frac{1}{M}\right)

2차원 초음속 정상 유동에서는 이 마하선을 따라 나아가는 좌향(Left-running)과 우향(Right-running) 특성선 C_+, C_- 장이 격전지처럼 교차한다. 특성선을 타면 물리량이 불변한다는 앞선 해석에 따라, 수학자 리만(Riemann)은 각 마하선을 따라 절대 불변하는 보존 맵인 리만 불변량(Riemann Invariants) J_+, J_- 속성 텐서를 매핑해 냈다.

엔지니어는 우주선 노즈콘 주변을 고정된 사각형 그리드가 아니라, 이 C_+ 와 C_- 특성선들이 서로 거미줄처럼 얽혀 파고드는 다이아몬드 교차망으로 덮어버린다. 두 특성선이 펼쳐지며 압력이 낮아지면 팽창파를 조각해 내고, 두 특성선이 앞서가던 파동을 추월하며 치명적으로 꼬여(Intersect) 접히는 좌표계에 즉시 랭킨-휴고니오(Rankine-Hugoniot) 불연속 도약 조건을 이식시켜 벼락같은 충격파면을 산출해결한다. 특성선법은 비선형 편미분 방정식의 카오스를 상미군의 순수 보존 역학망으로 전환해 낸 초음속 공기 역학자들의 최강 수학 병기이다.
