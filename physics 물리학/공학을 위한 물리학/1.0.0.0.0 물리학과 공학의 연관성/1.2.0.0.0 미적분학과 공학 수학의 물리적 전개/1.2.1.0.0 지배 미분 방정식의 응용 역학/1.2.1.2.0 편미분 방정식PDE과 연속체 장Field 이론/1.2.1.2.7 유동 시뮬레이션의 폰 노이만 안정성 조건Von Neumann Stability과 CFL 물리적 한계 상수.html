1.2.1.2.7 유동 시뮬레이션의 폰 노이만 안정성 조건(Von Neumann Stability)과 CFL 물리적 한계 상수

유한 차분법(FDM)이나 유한 체적법(FVM)을 동원하여 편미분 방정식(PDE)을 대수계로 강제 붕괴시킬 때, 이산화된 숫자의 파편들은 언제나 오차(Error)를 수반한다. 컴퓨터 메모리 속에서 시간 스텝 \Delta t 를 밟고 미래로 전진할 때마다, 이 미세한 오차가 격자망을 타고 어떻게 번져나가는지를 통제하지 못한다면 시뮬레이션은 순식간에 수치적 발산(Numerical Divergence)의 대폭발을 일으킨다. 존 폰 노이만(John von Neumann)은 이 이산화된 오차의 전파를 조화 파동의 중첩으로 분해하여 감쇠 여부를 판별하는 궁극의 해석 기법을 세웠다. 더 나아가 이 수학적 안정성은 쿠랑-프리드릭스-루이(Curant-Friedrichs-Lewy)가 제창한 CFL 조건과 충돌하며 시간 전진 스텝 \Delta t 가 결코 파동의 전파 속도를 뛰어넘을 수 없다는 물리학적 절대 구속(Physical Constraint)으로 승화된다.

1. 폰 노이만 안정성 해석(Von Neumann Stability Analysis) 메커니즘

어떤 수치 스킴이 시간에 따라 발산할 것인지, 수렴할 것인지를 판독하기 위해 수치 공간의 오차 파동을 투시하라. 현재 격자 x_j 와 시간 t_n 에서 생성된 미세한 오차 성분 \epsilon_j^n 이 오직 시뮬레이션의 차분 방정식 자체만을 타고 파급된다고 상정한다. 폰 노이만은 이 임의의 오차 폭풍을 공간상의 독립된 푸리에 급수(Fourier Series) 조화 함수 다발로 분리해 낸다.

\epsilon_j^n = G^n e^{i k m \Delta x}

여기서 i 는 허수 단위, k_m 은 공간 파수(Wavenumber), G 는 한 번의 시간 스텝 \Delta t 가 지날 때마다 오차 파동에 곱해지는 폭발적 **증폭 계수(Amplification Factor)**이다.
만일 유동 미분 방정식을 이산화한 대수식에 이 오차 함수를 고스란히 체결했을 때, 도출되는 증폭 계수의 크기가 |G| \le 1 이라는 임계 조건을 방어해 낸다면, 오차는 스텝이 지날수록 무한히 팽창하지 않고 소산(Dissipation)되거나 유지된다. 이것이 바로 선형 차분 방정식에서 필연적으로 통과해야 할 절대적 방어선, **폰 노이만 안정성(Von Neumann Stability)**이다.

2. 대류 파동과 코랑-프리드릭스-루이(CFL) 물리적 한계 상수

폰 노이만의 수학적 안정성 기준을 1차원 쌍곡선형 대류 방정식(Convection Equation) \frac{\partial u}{\partial t} + c \frac{\partial u}{\partial x} = 0 의 전방-시간 후방-공간(FTBS) 차분망에 결속시켜 전개하면, 증폭 계수 제어망은 놀랍도록 직관적인 하나의 무차원 상수군으로 귀결된다.

C = \frac{c \Delta t}{\Delta x} \le 1

여기서 도출된 변수 C 를 쿠랑 수(Courant Number) 또는 **CFL 수(CFL Number)**라 명명한다. c 는 유체나 파동의 실제 공간 전파 속도이다. 이 부등식은 수학의 껍질을 찢고 강력한 동역학적, 물리적 인과율(Causality)을 선포한다.

  • 한 스텝 \Delta t 시간 동안 유체 입자나 충격파가 실제로 이동할 수 있는 물리적 거리는 c \Delta t 이다.
  • 컴퓨터가 짜놓은 격자망의 크기는 \Delta x 이다.
  • CFL 조건 \frac{c \Delta t}{\Delta x} \le 1 은, 스텝 당 **“물리적 파동의 전파 거리가 결코 전산 모델의 한 격자 사이즈를 초과해서는 안 된다”**는 냉혹한 선언이다.

만일 시간 족쇄 \Delta t 를 무심코 크게 잡아 C > 1 이 되는 순간, 정보(파동 에너지)는 컴퓨터가 연산하기도 전에 인접 격자 노드를 건너뛰어 물리적 인과율을 파괴해 버리며 증폭 계수 |G| > 1 에 진입하여 화면 가득 무한대의 에러를 토해낸다.

2. 이산화 영향 영역(Domain of Dependence)의 기하 포획 메커니즘

이러한 CFL 조건의 절대적 지위를 기하학적 인과율로 형상화하라. 편미분 방정식의 역학적 공간에서 특정 위치와 시간 (x_0, t_0) 의 상태를 결정짓는, 과거 시간축으로부터 뻗어 나오는 빛의 원뿔과도 같은 기하망을 ’물리적 의존 영역(Physical Domain of Dependence)’이라 부른다.

이 시각 구도로 파악하면, 명시적(Explicit) 수치 스킴에서는 컴퓨터의 수치적 의존 영역 삼각형의 면적이, 자연계의 파스칼 역학이 요구하는 물리적 쌍곡선 의존 영역을 완전히 집어삼켜 감싸지 못하면 해가 절대적으로 붕괴될 수밖에 없음이 확정된다. 수십 마하를 돌파하는 극초음속 비행 유동 시뮬레이션에서는 \Delta t 가 10^{-8} 초 단위로 극단적으로 강하하는데, 엔지니어는 결국 এই 엄혹한 폰 노이만과 CFL의 저주를 뚫고 계산 부하를 감당하기 위해 앞서 다루었던 암시적(Implicit) 수치 행렬 대수(무조건적 안정망) 해법을 거대 클러스터 컴퓨터에 병렬 연합시킬 수밖에 없는 운명에 도달한다.
