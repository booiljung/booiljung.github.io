1.2.1.2.2 변수 분리법(Separation of Variables)을 이용한 2차원 정상 상태 경계값 문제(BVP)의 해석 해

2차원 얇은 금속판의 온도 구배, 혹은 무관성 유체의 압력 포텐셜 등 전 공간적 평형 상태를 조명하는 라플라스 방정식(Laplace’s Equation, \nabla^2 u = 0)은 \frac{\partial^2 u}{\partial x^2} + \frac{\partial^2 u}{\partial y^2} = 0 이라는 지독하게 결집된 다변수 편미분 텐서 다발이다. 이 얽매인 2차원 xy 구속계를 부수기 위해 인류는 수많은 수치 공학적 파쇄망을 창안했지만, 해석학(Analytical)적으로 가장 우아하면서도 무결한 수학적 절단기는 다름 아닌 **변수 분리법(Separation of Variables)**이다. 이 장에서는 절대적으로 상호 연성된 2차원 해 u(x, y) 공간을 두 개의 1차원 독립 텐서 스칼라 곱으로 이격 시킨 뒤, 경계값 문제(Boundary Value Problem, BVP)를 강제 조립하여 해석적인 고유 함수망(Eigenfunctions)을 복원해 내는 궁극의 대수적 분할 역학을 강행한다.

1. 다변수 텐서의 고립 치환과 분리 상수(Separation Constant)의 발현

해석 해 공간 u(x, y) 의 형상이 x 에만 전적으로 종속된 좌표 함수 X(x) 와 y 에만 지배받는 좌표 함수 Y(y) 의 선형 독립적인 곱으로 구성될 것이라 맹렬히 억측(Assumption)하라.

u(x, y) = X(x) \cdot Y(y)

이 대담한 가설을 라플라스 다항식 u_{xx} + u_{yy} = 0 에 강제로 대입하고 전개한다.

X''(x)Y(y) + X(x)Y''(y) = 0

이 식을 다시 u(x, y) = X(x)Y(y) 스칼라로 나누어 텐서를 이항하면 역학적 경이로움이 발생한다.

\frac{X''(x)}{X(x)} = - \frac{Y''(y)}{Y(y)}

좌항은 철저히 x 만의 독자적 우주이고 우항은 철저히 y 만의 독자적 우주이다. 양 끝 차원이 어떻게 움직이든 항상 등식이 성립하려면, 좌항과 우항 모두 \pm \lambda^2 와 같은 절대적인 상수로 결박되어 있어야만 한다는 논리 전개가 불을 뿜는다.
이 상수를 **분리 상수(Separation Constant)**라 칭하며, 거대한 1개의 2계 편미분 방정식은 다음과 같이 두 개의 초라하고 다루기 쉬운 2계 상미분 방정식(ODE)계로 강제 격리된다.

X''(x) - \lambda^2 X(x) = 0
Y''(y) + \lambda^2 Y(y) = 0

이제 공학자는 무한 대류의 2차원 편미분을 완전히 배제하고, 앞서 익숙하게 다루었던 질량-스프링 진동계 1차원 상미분 시스템 두 개만을 독립적으로 상대하는 전초 기지를 확보하였다.

2. BVP 투입과 고유치(Eigenvalues)/고유 함수(Eigenfunctions) 역산

분리 상수 \pm\lambda^2 의 부호 조합은 X 쪽이 파동(사인/코사인) 궤적을 치면 Y 쪽은 폭발/감쇠 지수 함수(익스포넨셜) 궤적으로 보상 상쇄하여 합계 \nabla^2 = 0 을 만들어 내는 양극단 시스템 스펙트럼이다.
어디를 조화 파동 공간으로 삼고 어디를 감쇠 포텐셜로 삼을 지는, 시스템 외벽을 감싸고 있는 기하학적 **경계 조건(Boundary Conditions)**에 의해 전면 통제된다.

가로 길이 L, 세로 폭 H 의 타원형 플레이트에서, 하단/좌/우측 3면의 온도가 모두 고정 0 도로 묶여 있고 상단(y=H) 측면에서만 알 수 없는 분포 f(x) 로 가열되는 경계값 문제(BVP)를 관측하라.
좌우측 벽면 조화 텐서 X(0) = 0 및 X(L) = 0 에 미분 해를 연동하면 이산 고유치(Discrete Eigenvalues) \lambda_n 의 집합만이 생존한다.

\lambda_n = \frac{n\pi}{L} \quad (n = 1, 2, 3, \dots)

그리고 이 고유치에 얽매인 조화 진동(Harmonic) 형상은 단독 **고유 함수(Eigenfunctions)**를 직조한다.

X_n(x) = \sin\left(\frac{n\pi}{L} x\right)

결국 분리되어 도출된 각 ODE의 특수 해를 다시 곱의 조합으로 조립하면, n 번째의 모드 솔루션 u_n(x, y) = X_n(x)Y_n(y) 가 복원 생성된다.

3. 선형 중첩(Superposition) 통제와 푸리에 급수 계수 결착

단 하나의 u_n 해망으로는 상단 면에 가해진 임의의 열 분포 f(x) 라는 극악의 비선형 곡선을 절대로 모사할 수 없다. 선형 라플라스 방정식의 강력한 특권인 **중첩의 원리(Principle of Superposition)**를 극단으로 발휘하라. 모든 분리 모드 n 개의 해들을 c_n 이라는 미정 계수 장벽과 함께 무한대로 합산(Summation)하는 최후의 솔루션 형태를 조립한다.

u(x, y) = \sum_{n=1}^{\infty} c_n \sin\left(\frac{n\pi}{L} x\right) \sinh\left(\frac{n\pi}{L} y\right)

이 무한 급수의 모달 계수 c_n 들을 확정 짓기 위해 상단 경계 구속인 u(x, H) = f(x) 를 투척한다. 이 순간 변수 분리법의 긴 역학 구도망은 직교(Orthogonal) 기저 위에서 임의 함수를 분해하는 푸리에 급수(Fourier Series) 역산이라는 거대한 공학 수학적 지형 구조로 완벽히 흡수 결합되어, 최후의 c_n 미지수들을 도출해 냄으로써 모든 거시적 BVP 해설에 완벽한 마침표를 찍게 된다.
