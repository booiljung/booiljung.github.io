1.2.1.2.6 유한 차분법(FDM) 기반 전방/후방/중앙 차분망 테일러 급수 절단 오차(Truncation Error)

수치 공학자가 공간의 무장된 구배 스칼라인 미분(Derivative) \frac{du}{dx} 텐서를 컴퓨터 메모리가 인식할 수 있는 대수적 이산화(Discretization) 조각으로 격하시키는 시초적 발명은 유한 차분법(Finite Difference Method, FDM)이다. 무한히 좁혀 들어가는 극한(\lim_{\Delta x \to 0})의 개념을 제거하고, 그 대신 고정된 그물망 간격 \Delta x 만큼을 둔탁하게 잘라내어 기울기를 강제 근사(Approximation)시키는 이 역학은 본질적으로 수학적 원죄를 잉태한다. 실제 연속성 미분 곡선과 이 이산화된 격자 직선 사이에는 피할 수 없는 괴리 영역이 생기며, 지배 방정식 해석 알고리즘은 이 오차를 맹목적으로 누적하며 발치해간다. 본 절에서는 미분 이산화의 모태인 테일러 급수(Taylor Series)를 전개하여, 전방/후방/중앙 차분망(Difference Scheme)이 각각 수반하는 절위, 즉 절단 오차(Truncation Error)의 역학적 등급 차이를 수학적으로 맹렬히 파헤친다.

1. 테일러 급수(Taylor Series) 전개를 통한 미분 지평의 투시

임의의 노드 위치 x_i 에 존재하는 지배 함수 u(x) 를 기점으로, \Delta x 만큼 전진한 미래 공간 노드 x_{i+1} (혹은 후퇴한 노드 x_{i-1}) 에서의 함수 값을 추정하기 위해 무한 미분항 미사일을 사출하는 테일러 급수를 전개하라.

전방 공간으로 투사된 테일러 급수 (미래 공간 노드 x_{i+1}):
u(x_{i+1}) = u(x_i) + \Delta x \left. \frac{du}{dx} \right|_{i} + \frac{\Delta x^2}{2!} \left. \frac{d^2u}{dx^2} \right|_{i} + \frac{\Delta x^3}{3!} \left. \frac{d^3u}{dx^3} \right|_{i} + \mathcal{O}(\Delta x^4)

후방 공간으로 투사된 테일러 급수 (과거 공간 노드 x_{i-1}):
u(x_{i-1}) = u(x_i) - \Delta x \left. \frac{du}{dx} \right|_{i} + \frac{\Delta x^2}{2!} \left. \frac{d^2u}{dx^2} \right|_{i} - \frac{\Delta x^3}{3!} \left. \frac{d^3u}{dx^3} \right|_{i} + \mathcal{O}(\Delta x^4)

FDM의 기본 사상은 이 무한한 파동의 시리즈 중 1계 미분 조각 \frac{du}{dx} 만을 좌항으로 고립시키고, 나머지 고차 미분 파편들을 가차 없이 쓰레기통에 내버려 **절단(Truncation)**해 버리는 것이다. 버려진 텐서(Tensor) 파편 집합체, 이것이 곧 절단 오차(Truncation Error)의 실체이다.

2. 전방/후방 차분(Forward/Backward Difference)의 1차 정확도 한계망

전방 궤도 수식에서 1계 기울기 \left. \frac{du}{dx} \right|_{i} 만을 우격다짐으로 역추출하라.

\left. \frac{du}{dx} \right|_{i} = \frac{u(x_{i+1}) - u(x_i)}{\Delta x} \quad - \left[ \frac{\Delta x}{2} \frac{d^2u}{dx^2} + \frac{\Delta x^2}{6} \frac{d^3u}{dx^3} + \dots \right]

  • 수식의 앞부분 \frac{u_{i+1} - u_i}{\Delta x} 는 우리가 컴퓨터에 코딩할 전방 차분(Forward Difference) 공식 텐서이다.
  • 우측의 거대한 대괄호 마이너스 항이 바로 이산화 수치 계산에 동반되는 오차다. 이 오차의 가장 큰 척도를 결정짓는 선도항(Leading Term)은 \Delta x^1 에 비례한다. 이를 기호로 표기하면 \mathcal{O}(\Delta x) 가 되며, 우리는 이 차분망을 1차 정확도(First-Order Accuracy) 스킴이라 격하시킨다.

동일하게 후방 궤도를 재배열하면, 방향만 과거를 향할 뿐 구조적으로 등가인 후방 차분(Backward Difference) 공식을 토출해 낸다.
\left. \frac{du}{dx} \right|_{i} = \frac{u(x_i) - u(x_{i-1})}{\Delta x} + \mathcal{O}(\Delta x)
비대칭적으로 물리계의 한쪽(미래 혹은 과거) 정보만을 편향하여 채택하는 1차 시스템은 강한 수치적 둔화를 지닌다. 특히 \Delta x^1 의 오차 등급은 격자망 강도를 2배 오밀조밀하게 조여도(\Delta x/2) 오차가 고작 절반으로 떨어지는 느린 수렴의 지옥도(Hellscape)를 수반한다.

3. 기하 대칭의 충돌과 중앙 차분(Central Difference)의 압도적 2차 정확도

이 비대칭의 1차 오류 굴레를 박살 내기 위해, 수치 해석자는 전방 노드식과 후방 테일러 수식을 전면 충돌시켜 단순 ’뺄셈’을 강행하는 역학 기동을 수행한다. u(x_{i+1}) - u(x_{i-1}) 연산을 수행하면, 부호가 동일했던 짝수 차수 항(\Delta x^2 곡률 미분항 등)이 극적으로 상쇄(Cancellation)되어 완전히 증발하는 기적이 일어난다.

u(x_{i+1}) - u(x_{i-1}) = 2 \Delta x \left. \frac{du}{dx} \right|_{i} + \frac{2 \Delta x^3}{3!} \left. \frac{d^3u}{dx^3} \right|_{i} + \mathcal{O}(\Delta x^5)

이 수식에서 다시 \frac{du}{dx} 를 역산하여 고립시켜라.

\left. \frac{du}{dx} \right|_{i} = \frac{u(x_{i+1}) - u(x_{i-1})}{2\Delta x} \quad - \left[ \frac{\Delta x^2}{6} \frac{d^3u}{dx^3} + \dots \right]

결과 물리식 \frac{u_{i+1} - u_{i-1}}{2\Delta x} 은 양옆 노드 간격 속에서 대칭적으로 균형을 잡는 중앙 차분(Central Difference) 다항식이다.

이 중앙 차분 구조에 잔존하는 선도 오차망은 \Delta x^1 이 아닌 무려 \mathcal{O}(\Delta x^2) 의 2차 정확도(Second-Order Accuracy) 로 두 계단 격상되었다. 즉 격자 간격을 단 절반만 줄여도, 수치 오차는 1/2 이 아니라 대수 법칙에 의해 1/4 로 급격히 사멸(Vanish)한다. 단지 대칭(Symmetry)이라는 점 하나를 공간에 부여한 것만으로, 컴퓨터에 부과되는 연산 비용의 장벽을 획기적으로 무력화시킨 수치 차분 텐서학의 근원적 승리이다.
