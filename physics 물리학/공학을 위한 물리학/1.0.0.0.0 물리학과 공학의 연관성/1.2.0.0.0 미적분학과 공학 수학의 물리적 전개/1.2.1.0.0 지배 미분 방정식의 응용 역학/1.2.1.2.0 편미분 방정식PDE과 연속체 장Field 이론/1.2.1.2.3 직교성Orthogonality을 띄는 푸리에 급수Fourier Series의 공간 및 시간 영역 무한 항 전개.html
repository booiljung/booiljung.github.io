1.2.1.2.3 직교성(Orthogonality)을 띄는 푸리에 급수(Fourier Series)의 공간 및 시간 영역 무한 항 전개

변수 분리법 등 공학의 해석학적 수단들이 궁극의 종착역으로 낙점한 곳은 항상 조화 함수(Harmonic Function)인 사인(Sine)과 코사인(Cosine)의 조합이다. 그러나 자연계나 산업 설비에서 시스템을 때리는 충격은 매끄러운 삼각 함수가 아니다. 모터의 구동 토크는 각진 사각파(Square Wave)를 띄며, 심장 박동 모니터의 시그널은 날카로운 톱니파(Sawtooth Wave)처럼 불연속적이다. 이러한 난해한 비선형 구속 조건들을 선형 미분 방정식의 진해(Exact Solution) 시스템에 합체시키기 위해, 장 바티스트 조제프 푸리에(Jean-Baptiste Joseph Fourier)가 고안한 직교 함수 전개(Orthogonal Function Expansion)의 미학이 동원된다. 본 절에서는, 어떠한 기괴한 외력 함수조차도 무한한 수의 삼각 파동 다발로 갈기갈기 찢어 재조립해 내는 푸리에 급수(Fourier Series)의 공학수학적 본질을 직교성(Orthogonality) 텐서의 시각에서 철저히 해부한다.

1. 조화 함수 공간의 내적(Inner Product)과 직교성 텐서의 발현

기하학적 공간에서 두 벡터가 수직(90^\circ)을 이룰 때 내적(Dot Product)이 0이 되듯, 함수 공간(Function Space)에서도 두 함수를 곱하여 특정 구간에서 적분한 값이 0으로 소각된다면 두 함수는 서로 **직교(Orthogonal)**한다고 선언한다. 사인과 코사인 함수계는 이 직교성의 궁극적 화신이다.

기준 구간 [-L, L] 에 대하여, 서로 다른 주파수 정수 m, n (m \neq n)을 지닌 조화 함수들을 상호 적분하라.

\int_{-L}^{L} \cos\left(\frac{m\pi x}{L}\right) \cos\left(\frac{n\pi x}{L}\right) dx = 0
\int_{-L}^{L} \sin\left(\frac{m\pi x}{L}\right) \sin\left(\frac{n\pi x}{L}\right) dx = 0
\int_{-L}^{L} \sin\left(\frac{m\pi x}{L}\right) \cos\left(\frac{n\pi x}{L}\right) dx = 0 \quad (\text{모든 } m, n \text{에 대해 성립})

오직 같은 주파수(m = n)일 때만 적분값이 L 이라는 공간 텐서로 생존해 남는다. 이 성질은 삼각함수들이 각각 서로의 영토를 철저히 보장하고 침범하지 않는, 선형 대수학에서의 직교 기저 벡터(Orthogonal Basis Vectors) 역할을 온전히 수행함을 수학적으로 증명한다.

2. 임의 함수의 무한 조화 급수 붕괴 및 푸리에 계수 도출

직교성이라는 절대적 기저망이 구축되었으므로, 이제 \pm L 주기(Period)를 가지는 그 어떤 불연속적, 비선형적 형상의 함수 f(x) 라도 무한히 많은 소리굽쇠(조화 함수)들의 합으로 낱낱이 붕괴시킬 수 있다.

f(x) = \frac{a_0}{2} + \sum_{n=1}^{\infty} \left[ a_n \cos\left(\frac{n\pi x}{L}\right) + b_n \sin\left(\frac{n\pi x}{L}\right) \right]

이 웅장한 선언문 속에서 미지의 모달 계수(Modal Coefficients) a_n 과 b_n 을 빼내기 위해서는 양변에 타겟 주파수 함수를 곱한 뒤 구간 적분(내적)을 강행하면 된다. 이 순간, 앞서 증명한 직교성에 의해 다른 모든 주파수 항들은 0으로 연쇄 파괴되어 증발하고, 오직 타겟 주파수의 텐서만이 남아 계수를 역산해 낸다.

a_n = \frac{1}{L} \int_{-L}^{L} f(x) \cos\left(\frac{n\pi x}{L}\right) dx
b_n = \frac{1}{L} \int_{-L}^{L} f(x) \sin\left(\frac{n\pi x}{L}\right) dx

이 푸리에 상수식은 원래의 파형 f(x) 안에 n 번째 고조파(Harmonic) 주파수 에너지가 얼마나 함유되어 있는지를 가리키는 지독히도 명징한 스펙트럼 인디케이터(Spectrum Indicator)이다. 직류 성분인 평균값 \frac{a_0}{2} 부터 시작해, 1차 정현파(기본파), 2차 극세 정현파 등으로 합성이 진행될수록 뭉툭했던 곡선은 점점 더 사각파의 날카로운 모서리를 따라 완벽한 기하학적 수렴(Convergence)을 이륙시킨다.
