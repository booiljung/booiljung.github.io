1.2.2.1.5 특이값 분해(SVD: Singular Value Decomposition) 공간 기저 매핑 및 유사 역행렬 맵핑 연산

전통적인 고유값 분해(Eigen-Decomposition)는 대수학의 눈부신 성취임에 분명하나, 오직 ’정방 행렬(Square Matrix)’이라는 완벽히 대칭된 거울의 우주에서만 뼈대를 추출할 수 있다는 치명적인 족쇄를 지닌다. 현실 물리학과 공학의 데이터는 센서의 개수와 취득된 데이터의 개수가 일치하지 않는 비정방, 즉 직사각형(Rectangular) 형태의 불균형 행렬 \mathbf{A}_{m \times n} 로 쏟아진다. 이 일그러진 차원의 비대칭 텐서를 다루기 위해 공학은 정방 행렬의 특권을 버리고, 세상의 모든 행렬을 세 개의 극한 거울망으로 쪼개버리는 궁극의 선형 분해 병기, **특이값 분해(SVD: Singular Value Decomposition)**를 발동한다. 본 절에서는 SVD가 어째서 공간의 회전과 직교 맵핑의 제왕인지 수식적으로 투시하고, 정칙(Non-singular)되지 않은 시스템에서조차 오차를 최소화하여 최적해를 강제색출하는 유사 역행렬(Pseudo-inverse) 연산 체계로의 진화를 갈파한다.

1. 비대칭 텐서의 3단 해부: SVD 직교 기저 매핑 역학

임의의 직사각형 행렬 \mathbf{A} \in \mathbb{R}^{m \times n} 은, 어떠한 비선형적 찌그러짐을 지니고 있더라도 다음과 같은 세 개의 파편 \mathbf{U}, \mathbf{\Sigma}, \mathbf{V}^T 의 완벽한 행렬곱으로 무조건 분해될 수 있다는 것이 SVD의 가장 숭고한 보존 법칙이다.

\mathbf{A} = \mathbf{U} \mathbf{\Sigma} \mathbf{V}^T

  1. \mathbf{V}^T \in \mathbb{R}^{n \times n} (입력 공간의 직교 회전 연산자):
    입력된 n 차원 공간의 벡터를 뒤틀림 없이 오직 뼈대 자체의 방향만 직교(Orthogonal) 회전시키는 우-특이 벡터(Right Singular Vectors)망이다. 이들은 \mathbf{A}^T\mathbf{A} 의 고유벡터들로 구성된다.
  2. \mathbf{\Sigma} \in \mathbb{R}^{m \times n} (특이값 스케일링 텐서):
    대각선 성분을 제외한 모든 것이 0인 뼈대이다. \mathbf{V}^T 로 방향이 틀어진 벡터들의 길이를 각각 \sigma_1, \sigma_2, \dots 만큼 증폭하거나 짓누르는(Scaling) 일차원 스칼라 확장/수축 기폭제이다. 이 \sigma 스칼라들을 **특이값(Singular Values)**이라 명명하며, 언제나 \sigma_1 \ge \sigma_2 \ge \dots \ge 0 의 철저한 크기 파워 서열로 내림차순 정렬된다.
  3. \mathbf{U} \in \mathbb{R}^{m \times m} (출력 공간의 직교 회전 연산자):
    스케일링이 끝난 변형 벡터를 최종 목적지인 m 차원 출력 공간에 맞추어 다시 한 번 직교 회전시켜 안착시키는 좌-특이 벡터(Left Singular Vectors)망이다. 이 축들은 \mathbf{A}\mathbf{A}^T 의 고유벡터들로 구성된다.

결국 임의의 찌그러진 행렬 \mathbf{A} 도 본질을 통찰하면 [1. 회전] \rightarrow [2. 길이 증폭/압축] \rightarrow [3. 재회전] 이라는 극도로 조화로운 세 박자의 직교 기저 매핑(Basis Mapping) 시퀀스에 불과했던 것이다.

2. 최소 자승법의 제왕: 무어-펜로즈 유사 역행렬(Moore-Penrose Pseudo-inverse)

실험 측정 방정식 \mathbf{A}\mathbf{x} = \mathbf{b} 에서, 방정식이 너무 많아 과결정(Overdetermined) 되었거나, 아예 겹쳐서 기하학적 체적이 0 이 되어버린 특이 상태(Singular Matrix)라면 고전적 역행렬 \mathbf{A}^{-1} 은 비명을 지르며 연산을 거부한다. SVD는 이 절망의 공간에서 유사 역행렬(Pseudo-inverse) \mathbf{A}^+ 를 강제로 직조해 내어, 거성처럼 공학적 최적해를 투하한다.

\mathbf{A} = \mathbf{U} \mathbf{\Sigma} \mathbf{V}^T \implies \mathbf{A}^+ = \mathbf{V} \mathbf{\Sigma}^+ \mathbf{U}^T

여기서 \mathbf{\Sigma}^+ 는 기적의 텐서이다. 원래 특이값 대각 행렬 \mathbf{\Sigma} 의 원소 \sigma_i 들을 역수 \frac{1}{\sigma_i} 로 뒤집어 주되, 만일 \sigma_i = 0 이라 나눗셈 폭발이 일어나는 성분이라면 이를 과감히 0 으로 강제 치환해 버려 알고리즘의 발산을 막는다.
이 역추적 맵핑 연산 \mathbf{x}^+ = \mathbf{A}^+ \mathbf{b} 는 단순한 타협이 아니다. 이 해 \mathbf{x}^+ 는 남겨진 수많은 오차의 제곱합 ||\mathbf{A}\mathbf{x} - \mathbf{b}||^2 을 공간 기하학적으로 완벽히 ’최소화(Least-Squares)’시킬 뿐만 아니라, 그 도출된 해 자체의 체적 크기 ||\mathbf{x}||^2 조차 가장 작도록 방어해내는 극한의 공학적 **최적 정규해(Optimal Minimum-norm Solution)**를 보장한다.

특이값 \sigma_1, \sigma_2, \sigma_3 는 시스템이 머금고 있는 에너지(분산량)의 압도적인 서열을 직설적으로 폭로한다. 센서 노이즈나 무의미한 실험 오차가 섞여 있다면, 그 성분들은 아주 미미한 값의 꼬리 특이값(\sigma_{n})들로 강등되어 하위 차원에 유배된다. 공학자는 이러한 찌꺼기 부분 공간을 단칼에 잘라내 버리고 단 몇 개의 거대한 상위 특이값 \sigma_1, \sigma_2 로만 행렬을 역조립(Truncated SVD)함으로써, 가장 오염되지 않은 기저 시그널만을 발치해 내는 궁극의 데이터 여과 장치로 SVD를 승격시킨다. SVD는 단순한 행렬 분해 테크닉이 아니라, 데이터 자체가 품고 있는 유전적 뼈대와 노이즈 기하 공간을 메스로 갈라내는 우주적 해부학이다.
