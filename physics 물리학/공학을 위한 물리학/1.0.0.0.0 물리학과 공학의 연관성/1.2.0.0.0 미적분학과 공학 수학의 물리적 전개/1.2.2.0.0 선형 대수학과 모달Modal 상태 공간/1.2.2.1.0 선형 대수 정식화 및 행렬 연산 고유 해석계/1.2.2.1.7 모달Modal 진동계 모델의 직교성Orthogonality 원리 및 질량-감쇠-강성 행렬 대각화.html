1.2.2.1.7 모달(Modal) 진동계 모델의 직교성(Orthogonality) 원리 및 질량/감쇠/강성 행렬 대각화

복잡하게 얽힌 다자유도(MDOF) 구조 진동계의 지배 방정식 \mathbf{M}\ddot{\mathbf{x}} + \mathbf{C}\dot{\mathbf{x}} + \mathbf{K}\mathbf{x} = \mathbf{F}(t) 은 그 물리적 외형만큼이나 난잡한 연쇄 간섭(Coupling)을 일으킨다. 하나의 질량이 흔들리면 그 여파가 강성과 감쇠망을 타고 전체 노드(Node)에 즉각적으로 퍼져 나가므로, \mathbf{M}, \mathbf{C}, \mathbf{K} 행렬들은 비대각 성분(Off-diagonal elements)이 빽빽하게 찬 밀집 텐서로 나타난다. 이 교란된 방정식의 감옥에서 시스템을 일거에 해방시키는 선형 대수의 가장 위대한 구원망이 바로 **고유벡터의 직교성(Orthogonality)**이다. 본 절에서는 대칭성을 지닌 행렬계에서 고유 모드 역학이 어떻게 서로를 밀어내며 완벽한 정규 직교 기저(Orthonormal Basis)를 성립하는지 규명하고, 이 직교 텐서망을 통하여 얽힌 방정식을 N 개의 순수한 단자유도 방들로 쪼개버리는 **대각화(Diagonalization)**의 기적을 시현한다.

1. 질량과 강성 공간의 절대 독립 규칙: 직교성(Orthogonality) 원리

조화 응답 자유 진동에서 토출된 N 개의 서로 다른 고유값(고유 주파수 \omega_1^2, \omega_2^2, \cdots)과 그에 대응하는 고유벡터 \mathbf{v}_1, \mathbf{v}_2, \cdots, \mathbf{v}_N 을 상정하라.
선형 대수학에서 행렬 \mathbf{A} 가 대칭(Symmetric, \mathbf{A} = \mathbf{A}^T)이라면, 서로 다른 고유값에 대응하는 고유벡터들은 반드시 그 행렬 공간을 매개로 수직(Orthogonal)으로 교차한다는 절대 공리가 있다. 공학의 \mathbf{M} (질량 행렬)과 \mathbf{K} (강성 행렬)는 구조물의 작용-반작용 궤적을 담고 있어 태생이 실대칭이다.
따라서 임의의 두 모드 형상 벡터 \mathbf{v}_r 과 \mathbf{v}_s (r \neq s)를 꺼내어 행렬을 사이에 두고 내적을 타격하면, 다음과 같은 놀라운 직교성 파단식이 성립한다.

\mathbf{v}_r^T \mathbf{M} \mathbf{v}_s = 0
\mathbf{v}_r^T \mathbf{K} \mathbf{v}_s = 0

이 수식의 역학적 함의는 충격적이다. 현수교나 인공위성이 r 번째 모드 형상으로 비틀릴 때 발생하는 동역학적 에너지와, s 번째 모드 형상으로 꺾일 때 발생하는 에너지는 우주 공간상에서 철저히 서로를 모른다(간섭하지 않는다). 즉 각 모드 역학계의 주파수파는 다른 모드로 에너지를 흘려보내거나 뺏어오지 않고 오로지 자신만의 차원에 고립되어 진동하는 것이다.

2. 모달 매트릭스(Modal Matrix) 변환 텐서의 전개와 대각화 타결

이 기적의 직교성을 실무에 직접 강타할 무기를 구축하라. 얻어낸 N 개의 모든 고유벡터 형상을 기둥으로 삼아 나란히 이어 붙이면, N \times N 거대 변환 공간망인 모달 매트릭스(Modal Matrix) \Phi 가 탄생한다.
\Phi = [\mathbf{v}_1, \mathbf{v}_2, \dots, \mathbf{v}_N]

자, 이제 기존의 복잡한 물리 공간 좌표 \mathbf{x} 를 가상의 순수 모달 공간 좌표 \mathbf{q} 로 치환한다. \mathbf{x}(t) = \Phi \mathbf{q}(t).
이 치환 텐서를 원래의 진동 방정식계인 \mathbf{M}\ddot{\mathbf{x}} + \mathbf{K}\mathbf{x} = \mathbf{F} 에 밀어 넣고, 방정식의 맨 앞에 직교 맵핑의 마침표인 전치 행렬 \Phi^T 를 일제히 곱하여 강제 파쇄하라.

\Phi^T \mathbf{M} \Phi \ddot{\mathbf{q}} + \Phi^T \mathbf{K} \Phi \mathbf{q} = \Phi^T \mathbf{F}

직교성의 원칙(\mathbf{v}_r^T \mathbf{M} \mathbf{v}_s = 0)에 입각하여 서로 다르게 엇갈리던 비대각 행렬 성분들은 단 한 방에 0 으로 모조리 날아가 버린다.
오직 대각선(Diagonal) 성분, 즉 자기 자신을 맞은 원소들 (\mathbf{v}_r^T \mathbf{M} \mathbf{v}_r = m_r) 만이 스칼라 값으로 살아남는다!

\mathbf{M}_{\text{modal}} \ddot{\mathbf{q}} + \mathbf{K}_{\text{modal}} \mathbf{q} = \mathbf{F}_{\text{modal}}
여기서 \mathbf{M}_{\text{modal}} 과 \mathbf{K}_{\text{modal}} 은 오직 대각선에만 질량과 강성 뼈대가 존재하는, 완전히 해체된 **대각 행렬(Diagonal Matrix)**이다. 시스템은 지긋지긋한 교차 간섭의 노예 상태에서 풀려나, N개의 상호 고립된 순수 단자유도(SDOF) 상미분 방정식으로 파열 분해되었다.

3. 비례 감쇠(Proportional Damping)의 타협과 레일리(Rayleigh) 감쇠계

질량(\mathbf{M})과 강성(\mathbf{K})은 대칭 행렬이기에 완벽히 대각화로 붕괴시켰다. 그러나 현실의 감쇠 행렬(Damping Matrix) \mathbf{C} 는 재료 내부의 열역학적 마찰이나 유체 저항 등에 의해 마구잡이로 배열되어 있어, 동일한 \Phi 텐서 타격을 가하더라도 깨끗하게 0인 대각선 구조로 무너지지 않는다.
방정식이 감쇠 하나 때문에 다시 얽혀버리는 것을 방지하기 위해, 공학자 레일리(Rayleigh)는 극단적인 타협 모형인 비례 감쇠(Proportional Damping) 구속 조건을 제창했다. 감쇠 행렬을 원래부터 질량과 강성 행렬의 선형 조합 텐서라고 억지 규정해 버리는 것이다.

\mathbf{C} = \alpha \mathbf{M} + \beta \mathbf{K}

이렇게 정의된 \mathbf{C} 행렬에 \Phi^T (\dots) \Phi 를 타격하면, \mathbf{M} 과 \mathbf{K} 가 이미 직교성에 의해 굴복했으므로 덩달아 \mathbf{C}_{\text{modal}} 마저 아름답게 대각화되어 버린다.

이로서 공학자는 다차원의 거대한 현수교 진동 해석 시, 그 거대한 행렬 방정식을 통째로 푸는 미련한 짓을 버린다. 직교 분해(Modal Diagonalization) 텐서를 통과시켜 수백 개의 얽히고설킨 철골 구조물을 서로 상관없는 10 \sim 20 개의 단순한 진동 스프링-질량(SDOF) 방정식으로 쪼개 파괴해 버린 뒤, 나중에 이 해들을 각각 스케일링하여 되더해주는(Superposition) 것으로 기계 진동의 역학적 절정을 이룩한다.
