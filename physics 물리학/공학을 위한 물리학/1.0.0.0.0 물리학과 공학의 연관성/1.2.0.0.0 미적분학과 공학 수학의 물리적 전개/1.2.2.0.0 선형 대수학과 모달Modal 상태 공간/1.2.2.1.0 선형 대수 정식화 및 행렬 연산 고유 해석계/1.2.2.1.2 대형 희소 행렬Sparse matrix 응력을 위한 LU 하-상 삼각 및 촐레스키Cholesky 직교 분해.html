1.2.2.1.2 대형 희소 행렬(Sparse matrix) 응력을 위한 LU 하/상 삼각 및 촐레스키(Cholesky) 직교 분해

유한 구조 요소망(FEM)이나 나비에 스토크스 전산 유동(CFD)이 산출해 내는 시스템 행렬은 그 크기가 수만, 수백만을 넘나드는 매머드급이다. 그러나 이 거대한 시스템 방정식의 99% 영역은 노드들의 국부적 연결성만을 대변하느라 0이라는 공허한 숫자들로 가득 찬, 일명 희소 행렬(Sparse Matrix) 상태원형이다. 이런 텅 빈 대형 희소 행렬에 고전적인 가우스 소거법을 매번 내리친다면, O(N^3) 의 연산 폭동으로 인하여 슈퍼컴퓨터의 코어가 녹아내릴 것이다. 더불어 하중 입력(\mathbf{b} 벡터)이 계속해서 바뀌는 동역학 제어 시뮬레이션에서는, 가우스 전방 소거를 그 때마다 처음부터 다시 반복해야하는 극도의 비효율을 수반한다.
공학은 이 막대한 메모리와 연산 누수를 타개하기 위해, 밀집 소거 계통을 버리고, 원래의 행렬 \mathbf{A} 자체를 두 개의 유전자 블록, 즉 하삼각(Lower Triangular, L) 행렬과 상삼각(Upper Triangular, U) 행렬의 결합체로 영구 분리 보존해 두는 LU 분해(LU Decomposition) 기법을 동원한다.

1. 텐서 블록의 해체: LU 분해(LU Factorization) 역학 아키텍처

초기 계수 행렬 단위 \mathbf{A} 를 아무런 우항 입력 벡터 \mathbf{b} 없이 철저하게 독자적인 두 개의 계단 행렬 인자(Factor) 곱으로만 해체(Factorize)하라.

\mathbf{A} = \mathbf{L} \mathbf{U}

  • \mathbf{L} 행렬 (Lower Triangular Matrix): 대각선 위쪽이 모두 0으로 비어있는 하삼각 행렬이다. 가우스 소거 과정에서 하위 원소들을 박살내기 위해 사용되었던 ‘승수(Multiplier, m_{ij})’ 궤적 기록들이 이 뼈대에 고스란히 봉인 저장된다.
  • \mathbf{U} 행렬 (Upper Triangular Matrix): 대각선 아래쪽이 모두 0인 상삼각 행렬이다. 파괴적인 전방 소거작업이 최종 완료되고 남은 앙상한 행렬 결과물 본체이다.

이제 \mathbf{A}\mathbf{x} = \mathbf{b} 구조에 LU 텐서 결함을 체결하면, \mathbf{L} \mathbf{U} \mathbf{x} = \mathbf{b} 가 된다. 이 연립 다항식은 직관을 초월하는 두 단계의 미끄러지는 추적 알고리즘(Two-step Solver Network)으로 이형된다.

  1. 전방 통행망 (Forward Substitution):
    가상의 промежуточный(매개) 변수 벡터 공간 \mathbf{y} 를 선언하고 \mathbf{U}\mathbf{x} = \mathbf{y} 로 포박하라. 그러면 \mathbf{L} \mathbf{y} = \mathbf{b} 역학 폼이 발현된다. \mathbf{L} 은 밑으로 갈수록 넓어지는 하삼각 구조이므로, 수학은 가장 상단, y_1 이 독단으로 떨어지는 위에서부터 폭포 연산으로 밑바닥(y_N)까지 일거에 쓸어내린다.

  2. 후방 투입망 (Backward Substitution):
    찾아낸 사냥감 벡터 \mathbf{y} 를 가지고 다시 봉인되었던 \mathbf{U}\mathbf{x} = \mathbf{y} 를 풀어낸다. 이번엔 \mathbf{U} 가 거꾸로 뒤집힌 피라미드 형태이므로, 가장 밑바닥 방정식 x_N 방정식부터 풀어서 위쪽 꼭대기(x_1)로 거슬러 올라가 미지수 텐서를 완벽 장악한다.

이 LU 분해의 역학적 경이로움은, 입력원 \mathbf{b} 벡터가 수송 하중, 진동, 풍압 등에 의해 수백수천 번 바뀌더라도(\mathbf{b}_1, \mathbf{b}_2, \dots), 가장 시간과 돈이 많이 드는 분해 \mathbf{A}=\mathbf{L}\mathbf{U} 작업은 시뮬레이션 최초 1회만 단행하면 영구 재사용이 가능하다는 컴퓨터 구조학적 승리 선언이다. \mathbf{b} 가 바뀔 때마다 \mathbf{L} 과 \mathbf{U} 에 가볍게 대입(Substitution, O(N^2))만 반복하면 초고속으로 실시간 해를 뿜어낸다.

2. 대칭 강성 파동 응답을 위한 촐레스키(Cholesky) 직교 제곱 분해 계통

위의 우아한 LU 분해조차도, 대상 행렬 \mathbf{A} 가 대칭(Symmetric)이라는 특별한 파동 성찰 계통을 타고났을 때에는 더 지독하고 반으로 접힌 거울(Mirror) 텐서로 극한의 공간 절약을 시도할 수 있다.
질량 보존계와 연속 유체 역학 시스템 내의 강성 행렬(Stiffness Matrix) 파동은 거의 예외 없이 고유의 거울 대칭성(\mathbf{A} = \mathbf{A}^T)과 정부호(Positive Definite, \mathbf{x}^T \mathbf{A} \mathbf{x} > 0)라는 무결점 역학 방벽을 유지한다. 이런 청정 텐서 \mathbf{A} 장에 한해서, 우린 \mathbf{U} 를 폐기하고 오직 매개체 \mathbf{L} 의 거울 투영인 **전치 행렬(Transpose, \mathbf{L}^T)**만으로 전체 행렬계를 짜맞추는 촐레스키 직교 분해(Cholesky Decomposition) 망을 결격한다.

\mathbf{A} = \mathbf{L} \mathbf{L}^T

촐레스키 행렬의 대각 성분들은 역추적 알고리즘 전개 시 l_{jj} = \sqrt{a_{jj} - \sum (l_{jk})^2} 라는 제곱근 연산 지형을 강제 통과해야 한다. 앞서 행렬이 정부호(Positive Definite)라는 물리적 검증을 통과했다면 루트 안의 값이 언제나 양동 수치를 유지하므로 무결점으로 도출된다.
촐레스키 분해의 위용은 메모리와 속도에 있다. \mathbf{A} = \mathbf{L} \mathbf{L}^T 는 서로 대칭되는 수치들을 곱셈하므로 단 하나의 유전자 블록 \mathbf{L} 행렬 정보만 RAM에 기입해 놓으면 복제가 끝나 컴퓨터 저장 체적을 정확히 절반 반토막 내버린다. 연산 횟수(Flops) 또한 일반 \mathbf{L}\mathbf{U} 분해에 비해 소거 궤적 속도가 2배로 수직 상승한다. 더욱이 가우스 연산에서 겪었던 주축 0 파단(Pivot) 붕괴 오류의 저주성에서 선천적으로 면제되어 있는, 거대 구조 역학 해석 시스템의 가장 안전하고 고귀한 분할 연산 무기 체계이다.
