1.2.2.1.3 고유값(Eigenvalue) 및 고유벡터(Eigenvector) 벡터장을 활용한 역학 공진 모드 분석

공학의 모든 기계계와 우주의 모든 분자 격자들은 외부의 진동이 없더라도 내면에 자신들만이 가진 고유한 호흡, 즉 자연 주파수(Natural Frequency)를 품고 있다. 다자유도(MDOF) 구조물이나 항공기를 타격할 때 구조 전체가 파도처럼 꿈틀거리는 동역학적 현상은 복잡해 보이지만, 본질적으로는 선형 대수 행렬이 내포한 몇 가지 독립적인 진동 뼈대들의 중첩일 뿐이다. 이 뼈대를 찾아내어 기하학적으로 박제하는 도안이 바로 **고유값(Eigenvalue)**과 **고유벡터(Eigenvector)**의 추출망이다. 본 절에서는 거대한 질량-강성 텐서가 뿜어내는 벡터 공간 내에서 고유 시스템이 어떻게 진동 모드(Vibration Mode)로 발현되며, 그 웅장한 공진 시스템을 주파수 대역과 형상 좌표 장으로 해부하는 모달 역학을 전개한다.

1. 다자유도 역학계의 고유 텐서 방정식(Eigenvalue Problem) 정식화

자유 진동(Free Vibration) 상태의 다자유도 공학계 운동 방정식 \mathbf{M}\ddot{\mathbf{x}} + \mathbf{K}\mathbf{x} = 0 망에 접근하라.
시스템 전체가 동일한 단일 각주파수 \omega 로 영구 진동하며 그 형태는 변치 않는다는 조화 응답(Harmonic Response) 해, \mathbf{x}(t) = \mathbf{V} e^{i\omega t} 텐서를 꽂아 넣어라. \ddot{\mathbf{x}}(t) = -\omega^2 \mathbf{V} e^{i\omega t} 가 되어 미분항이 대수식으로 추락하며, 최종적으로 질량과 강성의 처절한 벡터 공간 충돌식인 고유치 문제가 포효한다.

(\mathbf{K} - \omega^2 \mathbf{M}) \mathbf{V} = 0

  • 스칼라 텐션 \lambda = \omega^2 (Eigenvalue): 괄호 안의 행렬이 특이 상태(Singular)로 무너져 내리는 임계 상수이다. 이 고유값 \lambda 는 구조물이 스스로 진동하려 하는 물리적 각주파수의 제곱(\omega^2)으로 강제 치환되며, 외부의 타격 주파수와 일치하는 순간 폭발적인 공진(Resonance)을 야기하는 기계적 척추뼈의 수치이다.
  • 스캐너 형상 \mathbf{V} (Eigenvector): 공간 변환 연산자 \mathbf{K} - \lambda\mathbf{M} 가 0으로 짓눌려 버리는 영공간(Null Space)의 기저 벡터이다. 특정 주파수 \omega_i 에서 건축물이나 항공기 날개가 어떤 기하학적 형상으로 굽어지며 춤을 추는지를 보여주는 불변의 고유 모드 형상(Mode Shape) 벡터이다.

2. 특성 다항식(Characteristic Polynomial) 궤적과 스펙트럼(Spectrum) 해부

명백한 비형상 해 \mathbf{V} \neq 0 를 건져내기 위한 유일한 생존 조건은 앞쪽의 물리적 스캐너 괄호 망의 체적이 0으로 납작해지는 특성 방정식(Characteristic Equation) 선언뿐이다.

\det(\mathbf{K} - \omega^2 \mathbf{M}) = 0

시스템이 N 차원이면, 위 행렬식을 풀어헤쳤을 때 \lambda (\omega^2) 에 대한 N 차 다항식 파동이 토출된다. 이 방정식의 N 개 근(Roots)들이 바로 시스템에 내재된 서로 다른 N 개의 순수 공진 주파수들이다. 공학은 가장 낮은 주파수근 \omega_1 을 **1차 기본 모드(Fundamental Mode)**라 부르며, 구조 해석의 절대적 감시 대상으로 지정한다. 주파수가 낮을수록 에너지를 흡수하기 쉬워 외부 지진이나 바람의 거시적 요동에 의해 파멸적인 1차 편향(Bending) 공진으로 찢어질 확률이 가장 높기 때문이다. \mathbf{A} 행렬의 고유값들을 크기순으로 나열한 이 군집 대역을 선형 대수는 행렬의 **스펙트럼(Spectrum)**이라 칭한다.

2. 고유벡터장(Eigenvector Field) 직교성(Orthogonality)과 모달 공간 결착

각각의 고유값 \omega_i^2 를 원래의 고유 방정식에 대입하여 얻어낸 N 개의 고유벡터 \mathbf{V}_1, \mathbf{V}_2, \dots, \mathbf{V}_N 들은 그냥 독립적인 선들이 아니다. 대칭 행렬인 \mathbf{M} 과 \mathbf{K} 가 산출해 낸 이 고유 형상 벡터들은 시스템의 질량 텐서와 강성 텐서 위에서 서로를 절대적으로 외면하는 무결점 직교(Orthogonality) 역학망을 성립한다.

\mathbf{V}_i^T \mathbf{M} \mathbf{V}_j = 0 \quad (\text{if } i \neq j)
\mathbf{V}_i^T \mathbf{K} \mathbf{V}_j = 0 \quad (\text{if } i \neq j)

이 직교성은 공학의 짐을 극한으로 해방시킨다. 비대각 간섭으로 엉켜있던 수만 개의 복잡한 미분 역학 교점들이 형상 벡터들(모달 매트릭스)을 좌표축으로 삼아 재정렬되는 순간, 행렬식은 아름다운 대각선(Diagonal) 행렬로 폭삭 뼈대를 드러낸다. 우리는 이를 통해 바람에 흩날리는 현수교의 거대한 진동을 ‘첫 번째 활처럼 휘는 모드 에너지 50\% + 두 번째 꽈배기처럼 비틀리는 모드 에너지 20\% \dots’ 와 같이 서로 철저히 수직(직교)인 성분들의 단순 합(Superposition)으로 붕괴시켜 파악하는 모달 가설의 궁극적 권력을 거머쥔다.
