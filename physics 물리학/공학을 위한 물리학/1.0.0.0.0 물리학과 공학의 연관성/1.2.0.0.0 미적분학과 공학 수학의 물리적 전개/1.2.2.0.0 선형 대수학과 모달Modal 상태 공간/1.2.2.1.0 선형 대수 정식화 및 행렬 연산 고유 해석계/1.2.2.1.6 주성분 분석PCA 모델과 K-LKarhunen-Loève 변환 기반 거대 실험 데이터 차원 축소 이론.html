1.2.2.1.6 주성분 분석(PCA) 모델과 K-L(Karhunen-Loève) 변환 기반 거대 실험 데이터 차원 축소 이론

유체역학의 난류 분석, 항공 우주 구조물의 수백만 채널 스트레인 게이지 데이터, 그리고 방대한 열역학적 계측망에 이르기까지, 현대 공학이 빨아들이는 거대 실험 데이터(Massive Experimental Data)는 그 자체로 수백, 수천 차원의 광활한 수학적 고립계이다. 이 무지막지한 고차원 데이터 속성을 그대로 해석 엔진에 집어넣는다면 컴퓨터는 O(N^3) 의 파멸적 ’차원의 저주(Curse of Dimensionality)’에 삼켜지고 만다. 이 방대한 정보의 바다 속에서 진정으로 본질적인 물리적 변화량만을 축출하여 수천 차원의 데이터를 단 몇 개의 축으로 무자비하게 짓눌러버리는(Compression) 선형 대수의 마법이 존재한다. 이것이 통계학에서 제창된 **주성분 분석(PCA: Principal Component Analysis)**이며, 공학과 신호 처리 역학의 언어로는 **카루넨-뢰베 변환(Karhunen-Loève Transform, K-L Transform)**이라는 이름으로 군림하는 절대적 차원 축소(Dimensionality Reduction) 이론이다.

1. 분산폭의 파동과 공분산 행렬(Covariance Matrix) 텐서 구축

차원 축소의 첫 번째 역학적 동역은 ’가장 크게 요동치는 축이 곧 가장 폭발적인 정보(에너지)를 담고 있다’는 물리학적 전제에서 시발한다. 특정 방향으로의 분포(분산, Variance)가 크다는 것은 그 축을 따라 시스템의 물리적 변동 특성이 가장 선명하게 표출됨을 의미한다. 반대로 분산이 0에 수렴하는 축은 구조에 동역학적 기여를 하지 못하는 죽은 노이즈 차원이다.

관측된 m 개의 변수를 지닌 n 번의 계측 데이터 행렬 \mathbf{X} (평균이 0으로 정규화되었다고 가정)에 대하여, 데이터가 어느 방향으로 가장 강압적으로 퍼져 나가는지를 보여주는 스캐너 계기판, 공분산 행렬(Covariance Matrix) \mathbf{C} 를 조립하라.

\mathbf{C} = \frac{1}{n-1} \mathbf{X} \mathbf{X}^T

이 \mathbf{C} 텐서는 m \times m 의 대칭 행렬(Symmetric Matrix) 구조를 지니므로 필연적으로 양단의 모든 직교 고유벡터들을 포획할 수 있는 선형 대수학적 축복을 부여받는다. 대각선 항은 각 단일 센서 데이터의 자가 분산(에너지)을, 비대각 항은 서로 다른 두 변수 간의 물리적 간섭(상관관계) 스펙트럼을 폭로한다.

2. K-L 변환과 공간의 직교 회전: 주성분(Principal Component)의 발치

PCA/K-L 변환의 궁극적 목표는 이 서로 상관성(Correlation)이 얽기섥기 꼬여있는 기존의 기하 축들을 통째로 비틀어 직교 회전시킴으로써, 모든 데이터 요소들이 서로 완벽히 독립적이고(Uncorrelated) 상호 간섭이 무효화된 완전히 새로운 직교 기저(Orthogonal Basis) 우주로 매핑하는 것이다.

이 새로운 기하학적 축들을 색출하기 위해 공분산 행렬 \mathbf{C} 의 고유값 분해(Eigen-Decomposition)를 타격하라.
\mathbf{C} \mathbf{v}_i = \lambda_i \mathbf{v}_i

산출된 m 개의 고유값 \lambda_1 \ge \lambda_2 \ge \dots \ge \lambda_m 들이 바로 데이터가 분출하는 ’에너지 폭(분산의 절대 크기)’이며, 결부된 고유벡터 \mathbf{v}_i 들은 그 분산이 맹렬하게 뻗어 나가는 새로운 기하학적 직교 축, 즉 **주성분(Principal Component)**이다. 놀랍게도 이 결과는 앞선 SVD 절에서 데이터 행렬 \mathbf{X} 에 직접 특이값 분해를 가하여 얻어낸 뼈대와 정확하게 동일한 유전학적 등가 공간에 놓여있다. (\lambda_i = \sigma_i^2 / (n-1))

2. 에너지 절단(Truncation)을 통한 텐서 데이터의 무손실 차원 축소

이제 차원 축소의 마법이 전개된다. 수학자들은 가장 거대한 \lambda_1 스칼라를 품은 제1 주성분(\mathbf{v}_1)이 시스템 데이터 분산 에너지의 대부분을 짊어지고 있으며, 하위로 곤두박질치는 꼬리 쪽의 \lambda 들은 물리적으로 쓸모없는 센서 잡음(Noise)에 불과하다고 파단한다.

m 차원의 거대 공간 축들 중 오직 상위 K 개 (K \ll m) 의 주성분 형상 벡터만 뗴어내어 절단 변환 행렬 \mathbf{W}_K = [\mathbf{v}_1, \mathbf{v}_2, \dots, \mathbf{v}_K] 를 구축한다.
장엄한 크기의 원본 데이터 \mathbf{X} 를 새로운 직교망 \mathbf{W}_K^T 에 정면 충돌시켜 투영(Projection) 시키면, \mathbf{Y} = \mathbf{W}_K^T \mathbf{X} 라는 극단적으로 크기가 찌부러진, 그러나 우주적 관상(본질적 정보의 99%)을 고스란히 간직한 저차원 정보 덩어리로 변태한다.
PCA 역학은 이처럼 방대한 실험계의 카오스 속에서 시스템을 구동하는 핵심적인 몇 가닥의 거시적 ’배후 매커니즘’만을 정확히 저격하여 건져올림으로써, 복잡성계의 차원 저주를 분쇄하는 가장 지배적인 실험 통계 역학의 사령관이다.
