1.2.2.2 현대 제어 이론의 상태 공간 표기법과 동적 시스템 판별

라플라스 변환(Laplace Transform)을 무기로 삼아 단일 입력과 단일 출력(SISO)의 입출력 관계망에만 집착했던 고전 제어 이론(Classical Control Theory)은 복잡하고 다원화된 현대 시스템 앞에서 참담한 한계를 노출했다. 수십 개의 추진기가 얽혀있는 로켓의 자세 제어나 기계, 전기, 유체 역학이 동시에 난무하는 사이버 물리 시스템(CPS)의 거동을 고전적인 블랙박스(전달 함수) 접근법으로 통제하려는 시도는 실패를 예약하는 것과 다름없었다. 공학자들은 입력과 출력으로만 시스템을 보던 근시안적 시야를 버리고, 시스템 **‘내부’**에서 고동치는 모든 미분 상태들의 실시간 궤적을 끄집어내어 행렬 공간에 박치기해 버리는 혁명적 전환점, 이른바 **상태 공간 표기법(State-Space Representation)**의 텐서 도메인으로 도약했다.

이 절에서는 모든 고계 미분 방정식을 1차 미분 방정식들의 거대한 행렬 식 묶음으로 파괴하고 재조립하는 상태 방정식(State Equation)과 출력 방정식(Output Equation)의 정식화(Formulation) 기법을 장악한다. 나아가, 내부 궤적을 예측해내는 행렬 지수 함수(Matrix Exponential)의 파동적 수렴과 동역학계 제일의 철칙, 기하학적 차원에서의 가제어성(Controllability) 및 가관측성(Observability) 판별(Determination) 이론의 핵심을 전개할 것이다. 입력의 힘이 내부의 모든 상태를 굴복시킬 수 있는지, 그리고 출력의 창살 틈으로 숨겨진 시스템의 뼈대를 관측해 낼 수 있는지를 검문하는 위대한 기하학적 수사의 전모가 바로 이곳, 현대 제어 대수학에서 펼쳐진다.
