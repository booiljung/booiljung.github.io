1.2.2.2.7 칼만 필터(Kalman Filter) 행렬의 예측-교정(Predict-Update) 알고리즘 및 오차 공분산 추정 모델

이론 속의 수학적 상태 공간 모델(\mathbf{A, B, C, D})은 실험실의 무균 상태에서나 성립하는 순수한 환상에 불과하다. 거친 우주 공간을 항해하는 인공위성이나 자율주행 차량이 직면하는 현실의 측정 데이터(\mathbf{y})와 물리적 동역학 \mathbf{\dot{x}} 는 언제나 난기류, 기계적 떨림, 그리고 전자 소자에서 끓어오르는 열잡음(Thermal Noise)에 의해 지독하게 오염되어 있다. 1960년 루돌프 칼만(Rudolf Kalman)은 이 험악한 노이즈의 바다 속에서 센서의 거짓말과 수학 모델의 불완전성을 동시에 타파하고, 단면적인 관측 데이터로부터 가장 순수한 진실의 상태 벡터 \mathbf{x} 궤적을 실시간으로 역추산해내는 불멸의 추정기(Estimator) 편미분 알고리즘을 발표했다. 이것이 아폴로 달 착륙선 궤도 제어의 일등 공신이자, 확률론적 제어계의 최고 금자탑인 **칼만 필터(Kalman Filter)**이다. 본 절에서는 가우시안 확률장(Gaussian Probability Field) 내에서 오차 공분산 연쇄를 추적하며 진실을 강제 맵핑해내는 칼만 필터의 2단 루프, 즉 **예측(Predict)**과 **교정(Update)**의 텐서 역학을 낱낱이 파헤친다.

1. 확률적 침투와 두 가지 에러 공분산(Covariance) 텐서의 상정

칼만 필터를 기동하기 위해선 먼저 확정적(Deterministic)이었던 기존의 이산 상태 공간 방정식을 확률론적(Stochastic) 요동계로 오염시켜 정식화해야 한다.

\mathbf{x}_k = \mathbf{A} \mathbf{x}_{k-1} + \mathbf{B} \mathbf{u}_k + \mathbf{w}_k
\mathbf{y}_k = \mathbf{C} \mathbf{x}_k + \mathbf{v}_k

이 수식계에는 두 개의 거대한 불확실성 마귀가 개입한다.

  1. \mathbf{w}_k: 시스템/프로세스 노이즈 (Process Noise). 모터의 떨림이나 모델링되지 않은 외란 등 내부에서 파생되는 에러의 파동.
  2. \mathbf{v}_k: 측정 노이즈 (Measurement Noise). 센서 자체가 품고 있는 본질적인 분해능 한계와 외부 전자기 잡음.

칼만 필터의 천재성은 이 알 수 없는 노이즈의 순간값 자체를 찾아내려 하지 않고, 이들이 평균이 0 인 정규 분포(Gaussian Distribution)를 극히 우아하게 따른다는 통계 법칙을 역이용하는 데 있다. 공학자는 대신 이 노이즈들의 에너지 규모를 상징하는 공분산 행렬(Covariance Matrix) \mathbf{Q} (프로세스 노이즈 공분산)와 \mathbf{R} (측정 노이즈 공분산)을 설정해 줌으로써 시스템의 불확실성을 통제 가능한 체적망으로 규격화한다.

우리가 궁극적으로 추적해야 할 것은 추정된 가상의 상태 \hat{\mathbf{x}} 와 진짜 보이지 않는 현실 상태 \mathbf{x} 간의 차이, 즉 추정 오차(Estimation Error) \mathbf{e}_k = \mathbf{x}_k - \hat{\mathbf{x}}_k 가 품고 있는 분산의 크기, 즉 오차 공분산 행렬(Error Covariance Matrix) \mathbf{P}_k 를 실시간으로 모니터링하며 이를 수학적으로 가장 작게(최소화) 뭉그러뜨리는 것이다.

2. 시간의 흐름과 진실의 붕괴: 1단계 사전 예측(Predict) 연산

새로운 센서 데이터 \mathbf{y}_k 가 아직 도착하기 직전, 시스템은 오로지 자신이 알고 있는 내부 수학 모델 \mathbf{A, B} 와 과거의 기억만을 바탕으로 현재 시간에 자기가 어디 있을지를 눈을 감고 대략 짐작(Time Update)해 보아야 한다.

  • 사전 상태 예측 (A Priori State Estimate):
    \hat{\mathbf{x}}_{k|k-1} = \mathbf{A} \hat{\mathbf{x}}_{k-1|k-1} + \mathbf{B} \mathbf{u}_k
    (과거의 출력 \hat{\mathbf{x}}_{k-1} 을 시스템 동역학 \mathbf{A} 를 통과시켜 단순히 전진시킨다.)
  • 사전 오차 공분산 예측 (A Priori Error Covariance):
    \mathbf{P}_{k|k-1} = \mathbf{A} \mathbf{P}_{k-1|k-1} \mathbf{A}^T + \mathbf{Q}
    (이 대수식의 물리적 의미는 섬뜩하다. 시간이 흐름에 따라 과거의 오차 \mathbf{P}_{k-1} 가 \mathbf{A} 매트릭스를 타고 부풀어 오르는 동시에, 외부의 공정 노이즈 \mathbf{Q} 가 무자비하게 추가 유입되며 우주의 엔트로피처럼 우리의 ’불확실성(P)’을 폭증시킨다.)

3. 센서의 빛과 진실의 융합: 2단계 사후 교정(Update) 대수 역학

시간이 미분적으로 흐르며 불확실성 \mathbf{P} 가 끝없이 팽창하려는 찰나, 외부 세계를 관측하는 센서로부터 한 줄기의 오염된 빛, 계측 데이터 \mathbf{y}_k 가 입력된다! 칼만 필터는 자신의 눈을 감고 때려 맞춘 가상의 출력(\mathbf{C}\hat{\mathbf{x}}_{k|k-1})과 현실의 실제 출력(\mathbf{y}_k) 간의 잔차(Residual/Innovation)를 타격하여 궤적을 강제 교정(Measurement Update)한다.

이 교정의 핵심 비율을 조절하는 관문이 바로 위대한 칼만 이득(Kalman Gain) \mathbf{K}_k 이다.
\mathbf{K}_k = \mathbf{P}_{k|k-1} \mathbf{C}^T ( \mathbf{C} \mathbf{P}_{k|k-1} \mathbf{C}^T + \mathbf{R} )^{-1}
식이 의미하는 황금 비율을 갈파하라. 센서가 쓰레기 노이즈(\mathbf{R})를 많이 품고 있다면 행렬 역수가 쪼그라들어 칼만 이득 \mathbf{K} 는 작아진다(센서를 믿지 말고 원래 예측 모델을 고수하라). 반대로 내 수학 모델의 불확실성(\mathbf{P})이 폭주했다면 \mathbf{K} 는 커져버린다(내 머리를 믿지 말고 들어오는 센서값 \mathbf{y} 에 궤적을 즉각 의탁하라).

강력하게 산출된 \mathbf{K}_k 텐서를 바탕으로 시공간의 두 번째 루프, 사후 교정을 결착한다.

  • 사후 상태 교정 (A Posteriori State Estimate):
    \hat{\mathbf{x}}_{k|k} = \hat{\mathbf{x}}_{k|k-1} + \mathbf{K}_k ( \mathbf{y}_k - \mathbf{C} \hat{\mathbf{x}}_{k|k-1} )
    (예측 궤적에, 실제와 예측 간의 오차를 \mathbf{K} 스케일만큼 섞어 넣어 진짜 궤적의 멱살을 잡고 끌어온다.)
  • 사후 오차 공분산 교정 (A Posteriori Error Covariance):
    \mathbf{P}_{k|k} = ( \mathbf{I} - \mathbf{K}_k \mathbf{C} ) \mathbf{P}_{k|k-1}
    (센서 정보를 성공적으로 빨아들이는 순간, 팽창을 거듭하던 불확실성 체적 \mathbf{P} 가 \mathbf{K} 타격에 의해 즉각 짓눌려(단축) 쪼그라들게 됨을 선형 기하학이 완벽하게 증명한다.)

이 경이로운 Predict-Update 2단 파이프라인은, 모델링의 불확실성에 의한 확산과 센서 계측에 의한 수축이라는 역학적 호흡을 무한히 반복(Recursive)한다. 칼만 필터는 엄청난 계산 분량의 과거 데이터를 메모리에 쌓아둘 필요 없이 전 시간의 추정치(\hat{\mathbf{x}}_{k-1})와 불확실성 텐서(\mathbf{P}_{k-1}) 한 조각만을 기억한 채 미래를 개척하는 메모리리스(Memoryless) 확률 행렬 연산의 극치다. 실시간으로 변동하는 동역학 텐서망 복판에서 가장 오차가 적은 절대 정규 위치를 짚어내는 제어 스펙트럼의 궁극적 눈(Observer)으로 군림한다.
