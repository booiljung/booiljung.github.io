1.2.2.2.6 선형 이차 제어기(LQR) 설계 시의 리카티(Riccati) 대수 방정식 및 비용 페널티 함수 최소화 역학

공학자가 아커만 공식으로 강제적인 극점(Pole) 배치를 이룩했다 한들, 시스템의 기계적 한계를 무시하고 무자비하게 빠른 응답만을 강요한다면 구동 모터는 과전류로 타버리거나 물리적 액추에이터는 파괴될 것이다. 진정한 제어의 미학은 ’궤적의 목표 달성’과 ’최소한의 에너지 소모’라는 두 가지의 상충되는 절대 가치 사이에서 기하학적 황금비를 이끌어내는 데 있다. 이 궁극의 타협점을 선형 대수학의 최적화 방정식으로 승화시킨 이론이 바로 **선형 이차 제어기(LQR: Linear Quadratic Regulator)**이다. 본 절에서는 상태 오차 극복과 제어 에너지 페널티를 동시에 짓누르는 이차 비용 함수(Quadratic Cost Function)를 정식화하고, 이를 해결하여 우주에서 가장 완벽한 피드백 이득(Feedback Gain) \mathbf{K} 를 추출해 내는 비선형 텐서 붕괴식, **대수적 리카티 방정식(ARE: Algebraic Riccati Equation)**의 심장부로 진입한다.

1. 최적의 형벌망: 이차 비용 페널티 함수(Quadratic Cost Penalty Function)의 조립

안정적인 평형점 \mathbf{x}=0 으로 시스템을 몰아가기 위해 가해지는 시간의 흐름 속에서, 시스템의 동작 상태 불량률과 에너지 소모량을 동시에 누적 산정하는 가혹한 심판의 스칼라 적분, 비용 함수(Cost Function) J 를 정의하라. LQR 이론은 다음과 같은 무한 구간(Infinite Horizon)의 적분 텐서를 선포한다.

J = \int_{0}^{\infty} \left( \mathbf{x}^T(t) \mathbf{Q} \mathbf{x}(t) + \mathbf{u}^T(t) \mathbf{R} \mathbf{u}(t) \right) dt

이 수식은 두 개의 극명한 형벌(Penalty) 텐서로 양분되어 시스템을 압박한다.

  1. 상태 페널티 \mathbf{x}^T \mathbf{Q} \mathbf{x}: 여기서 행렬 \mathbf{Q} (반양정부호 대칭 행렬, \mathbf{Q} \ge 0)는 상태 벡터 \mathbf{x} 가 원점(목표점)에서 조금이라도 벗어나 있을 때 부과되는 징벌적 가중치다. \mathbf{Q} 를 거대하게 키울수록 시스템은 오차를 용납하지 못해 극도로 신경질적이고 날카롭게 반응하여 궤적을 억지로 끌고 온다.
  2. 제어 에너지 페널티 \mathbf{u}^T \mathbf{R} \mathbf{u}: 행렬 \mathbf{R} (양정부호 대칭 행렬, \mathbf{R} > 0)는 조작 입력 \mathbf{u} 를 뿜어낼 때 소모되는 막대한 물리적 에너지(전력, 연료 등)에 대한 과세 스케일러다. \mathbf{R} 을 증가시킬수록 제어기는 자원을 아끼려 구두쇠처럼 행동하며, 매우 부드럽고 둔감하게(느리게) 반응한다.

LQR 제어기의 궁극적 설계 목적은, 저 두 가지 항목이 치열하게 충돌하는 적분망 J 의 체적을 우주적 관점에서 최소화(Minimize)하는 궁극의 최적 제어 입력 \mathbf{u}(t) = -\mathbf{K} \mathbf{x}(t) 를 색출하는 것이다.

2. 최적화의 심장: 대수적 리카티 방정식(ARE)의 비선형 텐서 결착

이 거대한 무한 적분 함수의 최소해를 변분법(Calculus of Variations)의 칼날로 도려내면, 해답은 놀랍게도 또 한 번 가상의 에너지 그릇인 리아푸노프(Lyapunov) 역학망과 맞닿아 있음을 알게 된다. 최적의 입력 텐서를 찾기 위해서는 시스템 행렬 \mathbf{A}, 입력 행렬 \mathbf{B}, 그리고 사용자가 하달한 고문 행렬 \mathbf{Q, R} 가 한데 엉켜 극한의 균형을 이루는 다음의 2차 비선형 행렬 공식을 만족하는 미지의 양의 정부호 행렬 \mathbf{P} 를 찾아내야만 한다.

\mathbf{A}^T \mathbf{P} + \mathbf{P} \mathbf{A} - \mathbf{P} \mathbf{B} \mathbf{R}^{-1} \mathbf{B}^T \mathbf{P} + \mathbf{Q} = \mathbf{0}

이 극악무도한 자태를 뽐내는 비선형 대수식이 바로 **대수적 리카티 방정식(ARE: Algebraic Riccati Equation)**이다. 이 식은 본래 시간 편미분을 포함하는 미분 리카티 방정식에서, 시간이 무한대로 흘러 평형 상태(t \to \infty)에 도달하여 시간 변동 편미분항이 증발해버려 남게 된 시불변 시스템(LTI) 제어의 정수다. 현대 제어 컴퓨터 알고리즘은 해밀토니안(Hamiltonian) 행렬 분해나 슈어(Schur) 분해법을 극한으로 회전시켜 이 \mathbf{P} 의 유일한 해를 발치해 낸다.

2. 궁극의 타협점: 최적 피드백 이득 행렬 \mathbf{K} 정식화

리카티 방정식의 화염 속에서 궁극의 텐서 \mathbf{P} 가 무사히 역산되어 토출되었다면, 공학자의 임무는 사실상 종료된다. 최적의 성능을 보장하는 신성한 제어 이득 행렬 \mathbf{K} 는 단순히 \mathbf{P} 의 렌즈에 입력비 가중치와 입력 텐서를 투과시키는 찰나의 행렬 곱셈만으로 강제 유도되기 때문이다.

\mathbf{K} = \mathbf{R}^{-1} \mathbf{B}^T \mathbf{P}

이 LQR 이론 체계는 기존 극점 배치(Pole Placement)에서 “도대체 어느 위치가 올바른 극점인가?“라는 공학자의 직관적 딜레마를, 수학적으로 한 치의 오차가 없는 최적화(Optimization)의 논리로 파괴했다. 공학자가 물리적 성능 제한(Q, R)의 한계선만 설정해 주면, 리카티 방정식 스캐너는 다차원 시스템 내부의 모든 얽힌 간섭망을 스캔하여, 가장 빠르고 부드러우면서도 비용이 들지 않는 극점의 안식처를 스스로 찾아내어 이득 행렬 \mathbf{K} 로 응결시킨다. 선형 이차 제어는 제어 공학의 주도권을 인간의 경험으로부터 냉혹한 대수학적 최적 궤적 계산의 텐서 도메인으로 인계한 역사적 금자탑이다.
