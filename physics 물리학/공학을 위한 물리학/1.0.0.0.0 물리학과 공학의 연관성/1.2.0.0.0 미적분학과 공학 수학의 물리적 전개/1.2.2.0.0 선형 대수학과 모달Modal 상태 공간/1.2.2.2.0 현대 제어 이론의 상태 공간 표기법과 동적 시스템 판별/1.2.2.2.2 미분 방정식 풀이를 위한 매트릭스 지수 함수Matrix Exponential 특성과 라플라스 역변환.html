1.2.2.2.2 미분 방정식 풀이를 위한 매트릭스 지수 함수(Matrix Exponential) 특성과 라플라스 역변환

상태 공간 표기법으로 압축된 \mathbf{\dot{x}}(t) = \mathbf{A}\mathbf{x}(t) + \mathbf{B}\mathbf{u}(t) 식은, 고전적인 스칼라 1차 미분 방정식 \dot{x} = ax + bu 와 놀라울 정도로 쌍둥이 구조를 띤다. 스칼라 방정식의 본연적 해방구가 자연 대수 기저 e^{at} 에 기초한 지수 함수적 팽창/수축이었다면, 텐서 도메인의 행렬 공간에서도 이 법칙은 소름 끼치도록 정확하게 평행 이동된다. 공학자는 기하 공간에서 e^{\mathbf{A}t} 라는 기괴한 다차원 행렬 진동 함수, 즉 **상태 천이 행렬(State Transition Matrix)**을 끌어내어 무한 차원의 미분 연립계를 일거에 녹여버린다. 본 절에서는 행렬 위에서 정의되는 지수 함수(Matrix Exponential)의 대수적 전개와 기하학적 파동 특성을 해부하고, 이를 복소 s-평면의 라플라스 역변환(Inverse Laplace Transform)과 융합하여 입력 동역학(Convolution)을 물리적 시간 공간 영역(Time Domain) 위로 완벽히 복원해 내는 제어 대수학의 최정점을 점화한다.

1. 1차 선형 상태 미분 방정식의 호모지니어스(Homogeneous) 자유 거동해

외부에서 인가되는 조작 힘이 완벽히 소거된(\mathbf{u}(t) = 0), 순수한 자발적 시스템 궤적 \mathbf{\dot{x}}(t) = \mathbf{A}\mathbf{x}(t) 를 관측하라.
테일러 급수(Taylor Series) 전개에 의해 어떤 스칼라 지수 함수가 e^{at} = 1 + at + \frac{(at)^2}{2!} + \dots 로 해체되듯, 행렬 공간 \mathbf{A} 의 지수 함수, 이른바 상태 천이 행렬 \Phi(t) 역시 무한한 행렬 거듭제곱의 합으로 정의된다.

\Phi(t) \equiv e^{\mathbf{A}t} = \mathbf{I} + \mathbf{A}t + \frac{1}{2!} \mathbf{A}^2 t^2 + \frac{1}{3!} \mathbf{A}^3 t^3 + \dots

이 경이로운 무한 차원 행렬식은, 시스템에 어떠한 힘을 가하지 않았을 때 초기 상태 \mathbf{x}(0) 가 자가 동역학 행렬 \mathbf{A} 의 성정에 의해 시간이 흐름에 따라 어떤 파동을 그리며 허물어질 것인가에 대한 해답 그 자체이다.
\mathbf{x}(t) = e^{\mathbf{A}t} \mathbf{x}(0)

매트릭스 지수 함수 e^{\mathbf{A}t} 는 일반적인 스칼라 지수와 매우 유사한 역학적 공리들을 만족하지만, 행렬 곱셈의 비가환성(Non-commutativity, \mathbf{A}\mathbf{B} \neq \mathbf{B}\mathbf{A})이라는 함정이 숨어있다. 따라서 e^{\mathbf{A}+\mathbf{B}} = e^\mathbf{A} e^\mathbf{B} 는 오직 \mathbf{A} 와 \mathbf{B} 가 서로 교환 가능(\mathbf{A}\mathbf{B} = \mathbf{B}\mathbf{A})할 때만 극적으로 타결된다는 날카로운 제약을 지닌다. 하지만 \Phi(0) = \mathbf{I} 이고 역행렬인 \Phi^{-1}(t) = \Phi(-t) 가 성립한다는 시간 역행 보존 법칙은 변함없이 유효하여 시스템 궤적을 과거로 거슬러 역추적하는 것을 가능케 한다.

2. 외부 조작 입력(Forced Input)과 콘볼루션(Convolution) 적분 해망

이제 본격적으로 외부 제어 힘 \mathbf{u}(t) 가 개입하는 완전한 형태(Non-homogeneous)의 \mathbf{\dot{x}}(t) = \mathbf{A}\mathbf{x}(t) + \mathbf{B}\mathbf{u}(t) 식을 공략한다.
상태 천이 행렬을 적분 인자(Integrating Factor)로 삼아 양변에 강제 곱셈을 가하여 수식을 붕괴시키면, 물리계의 궤적은 초기 조건의 소멸파와 입력 인가에 따른 강제 응답(Forced Response)의 맹렬한 합성 파동으로 재조립된다.

\mathbf{x}(t) = e^{\mathbf{A}t} \mathbf{x}(0) + \int_{0}^{t} e^{\mathbf{A}(t-\tau)} \mathbf{B} \mathbf{u}(\tau) d\tau

우측 공간을 장악하고 있는 적분항이 바로 합성곱(Convolution Integral) 텐서망이다. 시간 \tau 에서 가해진 충격파 \mathbf{B}\mathbf{u}(\tau) 가, 남은 시간 (t-\tau) 동안 시스템의 자가 뼈대인 e^{\mathbf{A}(t-\tau)} 를 타고 어떻게 메아리처럼 공간을 뒤흔들며 현재 시간 t 까지 도달하는가를 철저하게 누적 합산하는 인과율(Causality) 역학의 완벽한 묘사도이다.

3. 라플라스 도메인(Laplace Domain) 치환과 레졸번트 행렬(Resolvent Matrix) 맵핑

시간 영역에서 끝없는 무한 급수로 펼쳐지는 e^{\mathbf{A}t} 를 컴퓨터 없이 손으로 계산하는 것은 파멸적이다. 여기서 고등 공학자들은 미분 방정식을 대수 공간으로 폭파시키는 전술, **라플라스 변환(Laplace Transform)**을 상태 공간 행렬 위에 직투하한다.

상태 식 \mathbf{\dot{x}}(t) = \mathbf{A}\mathbf{x}(t) + \mathbf{B}\mathbf{u}(t) 양변에 라플라스 연산을 강타하여 s-평면으로 찢어발겨 보아라.
s\mathbf{X}(s) - \mathbf{x}(0) = \mathbf{A}\mathbf{X}(s) + \mathbf{B}\mathbf{U}(s)
(s\mathbf{I} - \mathbf{A}) \mathbf{X}(s) = \mathbf{x}(0) + \mathbf{B}\mathbf{U}(s)

방정식이 순수한 대수적 연립식으로 떨어졌다. 이 궤적 변수를 건져내기 위한 핵심 코어, 즉 시스템의 심장부를 가르는 역전 텐서가 바로 **레졸번트 행렬(Resolvent Matrix)**인 (s\mathbf{I} - \mathbf{A})^{-1} 이다.

\mathbf{X}(s) = (s\mathbf{I} - \mathbf{A})^{-1} \mathbf{x}(0) + (s\mathbf{I} - \mathbf{A})^{-1} \mathbf{B}\mathbf{U}(s)

시간 도메인에서 도출했던 자연 진동해 \mathbf{x}(t) = e^{\mathbf{A}t} \mathbf{x}(0) 의 궤적과 라플라스 역변환(Inverse Laplace Transform)을 강제로 맞부딪히게 하면, 현대 제어의 기념비적 등가식이 탄생한다.
e^{\mathbf{A}t} = \mathcal{L}^{-1} \{ (s\mathbf{I} - \mathbf{A})^{-1} \}

특성 방정식 \det(s\mathbf{I} - \mathbf{A}) 가 레졸번트 행렬의 분모 스칼라로 직결됨을 주목하라. 즉 \det(s\mathbf{I} - \mathbf{A}) = 0 이 되는 지점(s 근)이 곧 시스템 텐서가 품고 있는 \mathbf{A} 의 고유값(Eigenvalue)이자, 제어 시스템 전체의 안정성을 파괴하거나 수렴시키는 절대적 극점(Pole) 스펙트럼이다. 라플라스 변환은 e^{\mathbf{A}t} 의 수렴 한계를 가장 완벽하고 직관적으로 타결하는 최고의 역전파 무기 체계인 것이다.
