1.2.2.2.5 에너지 감소 함수에 기초한 리아푸노프(Lyapunov) 방정식과 비선형 대역적 점근 안정성

모든 동역학 시스템은 필연적으로 불안정성(Instability)이라는 악마의 중력과 사투를 벌인다. 선형 시스템(Linear Systems)의 안정성은 상태 행렬 \mathbf{A} 의 고유값(Eigenvalue)이 품고 있는 복소 평면 좌반면(Left-Half Plane) 지향성만으로 손쉽게 판독되지만, 시스템이 비선형(Nonlinear) 시공간 \mathbf{\dot{x}} = f(\mathbf{x}) 으로 몸살을 앓기 시작하면 고유값 스캐닝이라는 장비는 무력화된다. 러시아의 수학자 알렉산드르 리아푸노프(Aleksandr Lyapunov)는 이 혼돈의 미분 방정식들에 직접 들이댈 수 있는 궁극의 물리적 직관, ’에너지(Energy) 감소 보존칙’을 수학적 차원으로 격상시켰다. 본 절에서는 가상의 역학적 에너지를 규정하는 리아푸노프 함수(Lyapunov Function)를 직조하고, 그 에너지가 시간의 흐름에 따라 지속적으로 출혈(감소)함을 증명함으로써 비선형 공간조차 평형점(Equilibrium Point)으로 굴복시키는 대역적 점근 안정성(Global Asymptotic Stability) 판별의 대수학적 권능을 해부한다.

1. 가상 에너지 우물: 리아푸노프 함수(Lyapunov Function)의 양의 정부호성

자연계의 진자나 스프링 댐퍼 시스템이 흔들림을 멈추고 중앙으로 정지하는 이유는 마찰과 감쇠로 인해 시스템이 보유한 총에너지가 점점 깎여나가기 때문이다. 리아푸노프는 기하학적 상태 공간(State-Space) 내에 존재하는 추상적인 수학 시스템에 대해서도 이 역학적 에너지 모델을 강제로 덧씌웠다.

시스템의 원점 \mathbf{x} = 0 이 기계적 평형점(Equilibrium Point)이라 할 때, 시스템의 상태 벡터 \mathbf{x} 에 전적으로 종속되는 가상의 스칼라 에너지 함수 V(\mathbf{x}) 를 정의하라.
안정성 판별의 척도가 되기 위해, 이 V(\mathbf{x}) 는 반드시 에너지 우물의 형태인 **양의 정부호성(Positive Definite)**을 띠어야 한다.

  1. 원점에서는 에너지가 완전히 고갈되어야 한다: V(0) = 0
  2. 원점을 제외한 모든 상태 공간에서는 에너지가 무조건 양수여야 한다: V(\mathbf{x}) > 0 (for all \mathbf{x} \neq 0)

가장 대표적이고 다루기 쉬운 V(\mathbf{x}) 의 형태는, \mathbf{P} 라는 대칭 양의 정부호(Symmetric Positive Definite) 텐서 행렬 행렬을 중심에 박아넣은 이차 형식(Quadratic Form) 텐서다.
V(\mathbf{x}) = \mathbf{x}^T \mathbf{P} \mathbf{x} > 0

이 에너지 함수망은 등고선을 그렸을 때 완벽한 타원형 구를 형성하며, 상태 벡터가 원점에서 멀어질수록 급격하게 높은 에너지 장벽을 형성하는 철옹성을 세운다.

2. 미분 출혈 메커니즘과 점근 안정성(Asymptotic Stability) 사냥

에너지 우물이 완성되었다면, 이제 이 시스템이 우물 밖으로 탈출할 것인지, 우물 바닥(원점)으로 처박힐 것인지 운명을 결정해야 한다. 시간에 따른 이 에너지 함수의 변화율 \dot{V}(\mathbf{x}) 를 미분 타격하여 그 부호를 철저히 감시하라.
연쇄 법칙(Chain Rule)에 의해 미분은 상태 미분 텐서와 교합한다.

\dot{V}(\mathbf{x}) = \frac{\partial V}{\partial \mathbf{x}} \mathbf{\dot{x}} = \frac{\partial V}{\partial \mathbf{x}} f(\mathbf{x})

이 \dot{V}(\mathbf{x}) 가 궤적 평면 위에서 음의 정부호(Negative Definite), 즉 \mathbf{x} \neq 0 인 모든 곳에서 \dot{V}(\mathbf{x}) < 0 조건을 만족한다고 선언해보라.
이는 동역학계가 현재 어떤 기이한 춤사위를 추며 비선형적으로 움직이더라도, 시간이 1초라도 흐르면 시스템 내부의 에너지가 무조건 허공으로 찢겨나감(출혈)을 기하학적으로 완벽히 보장하는 것이다.
시스템의 에너지가 무한히 계속해서 줄어들기만 한다면, 이 궤적은 종국에 에너지가 0인 가장 깊은 구렁텅이, 즉 기계적 평형점(\mathbf{x}=0)으로 돌이킬 수 없이 빨려 들어갈 수밖에 없다! 리아푸노프 이론은 이를 **점근 안정성(Asymptotic Stability)**이라 명명하며 칭송한다. 더 나아가, 만약 상태 궤적이 원점으로부터 무한대(||\mathbf{x}|| \to \infty)로 떨어진 광대한 공간 어디에서 출발하더라도 이 에너지 보존칙(\dot{V}(\mathbf{x}) < 0 및 V(\mathbf{x}) \to \infty)이 절대적으로 유지된다면, 이를 **대역적 점근 안정성(Global Asymptotic Stability)**이라는 궁극의 우주적 평형 상태로 확정 짓는다.

2. 선형화된 파괴 도구: 리아푸노프 방정식(Lyapunov Equation)

비선형계의 절대 진리인 리아푸노프 정리를 역으로 LTI 선형 시스템 \mathbf{\dot{x}} = \mathbf{A}\mathbf{x} 에 투하해보자.
V(\mathbf{x}) = \mathbf{x}^T \mathbf{P} \mathbf{x} 라 두고 양변을 시간 미분 파면기에 집어넣어라.
\dot{V}(\mathbf{x}) = \mathbf{\dot{x}}^T \mathbf{P} \mathbf{x} + \mathbf{x}^T \mathbf{P} \mathbf{\dot{x}}
여기에 \mathbf{\dot{x}} = \mathbf{A}\mathbf{x} 를 꽂아 넣어 동역학을 교배한다.
\dot{V}(\mathbf{x}) = \mathbf{x}^T \mathbf{A}^T \mathbf{P} \mathbf{x} + \mathbf{x}^T \mathbf{P} \mathbf{A} \mathbf{x} = \mathbf{x}^T (\mathbf{A}^T \mathbf{P} + \mathbf{P} \mathbf{A}) \mathbf{x}

\dot{V}(\mathbf{x}) 가 ’음의 정부호’가 되게 하려면, 거대 행렬 덩어리 (\mathbf{A}^T \mathbf{P} + \mathbf{P} \mathbf{A}) 전체가 -\mathbf{Q} (여기서 \mathbf{Q} 는 임의의 양의 정부호 행렬) 라는 음수 스케일러로 변압되어야만 한다.
따라서 현대 제어 시스템 해석 최강의 대수 연립식, **리아푸노프 방정식(Lyapunov Equation)**이 폭발하며 등장한다!

\mathbf{A}^T \mathbf{P} + \mathbf{P} \mathbf{A} = -\mathbf{Q}

대수학의 기적은 여기서 빛을 발한다: 어떤 시스템 행렬 \mathbf{A} 가 본래 안정하다면, 공학자가 아무렇게나 긍정적인 에너지 타겟 행렬 \mathbf{Q} (보통 \mathbf{Q}=\mathbf{I})를 던져 주어도, 컴퓨터는 저 리아푸노프 식을 풀어내어 한 치의 오차도 없이 완벽하게 유효한 에너지 그릇 \mathbf{P} 행렬을 역산해 낸다. 미분 방정식의 해(e^{\mathbf{A}t})를 단 한 번도 풀지 않고, 오직 행렬 \mathbf{A, P, Q} 의 곱셈 대수학만으로 시스템의 대역적 파멸과 안정을 단칼에 감별해 내는 것, 이것이 리아푸노프 에너지 역학이 확보한 절대 권력이다.
