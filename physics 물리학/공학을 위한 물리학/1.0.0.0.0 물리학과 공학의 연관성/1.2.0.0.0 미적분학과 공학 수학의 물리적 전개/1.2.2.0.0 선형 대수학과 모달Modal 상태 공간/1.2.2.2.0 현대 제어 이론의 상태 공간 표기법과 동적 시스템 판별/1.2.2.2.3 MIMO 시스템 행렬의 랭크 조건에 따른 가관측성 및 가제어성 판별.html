1.2.2.2.3 MIMO 시스템 행렬의 랭크 조건에 따른 가관측성 및 가제어성 판별

현대 제어 이론이 창안한 \mathbf{A, B, C, D} 상태 공간 텐서망은 완벽해 보이지만, 그 기하학적 심장부에는 실무 공학자를 덮치는 두 가지 치명적인 질문이 도사리고 있다. 첫째, 우리가 가하는 조향 입력 \mathbf{u}(t) 의 힘이 이 거대한 행렬의 모든 내부 상태 \mathbf{x}(t) 구석구석까지 살아서 도달하여 시스템을 통제할 수 있는가? 둘째, 우리가 바깥에서 모니터링하는 센서 출력 \mathbf{y}(t) 의 창살을 통해, 보이지 않는 시스템 내부의 모든 \mathbf{x}(t) 궤적의 춤사위를 역산하여 추정해 낼 수 있는가? 미국 수학자 루돌프 칼만(Rudolf Kalman)은 1960년대에 제어계의 이 두 가지 본질적인 능력을 각각 **가제어성(Controllability)**과 **가관측성(Observability)**이라 명명하고, 선형 대수 행렬의 **계수(Rank)**라는 강력한 수학적 판별자로 시스템의 능력을 무자비하게 해부해 버리는 역사적인 랭크 결정 정리(Rank Condition)를 타결했다.

1. 지배 역학망의 확인: 가제어성 행렬(Controllability Matrix)과 랭크 조건

어떤 다중 입력 다중 출력(MIMO) 시스템이 유한한 시간 t_f 내에 무작위의 초기 상태 \mathbf{x}(0) 에서 사용자가 원하는 어떠한 최종 타겟 상태 \mathbf{x}(t_f) 로든 강제 멱살을 쥐고 끌고 갈 수 있다면, 이 시스템은 **완전 가제어(Completely Controllable)**하다고 선언된다.

입력 벡터 \mathbf{u} 가 시스템 내부로 파고들 때, \mathbf{B} 행렬을 통해 직접 투사되는 힘, 시스템의 자체 진동망 \mathbf{A} 를 한 번 튕겨서 들어가는 힘 \mathbf{A}\mathbf{B}, 두 번 튕기는 \mathbf{A}^2\mathbf{B} \dots 식으로 힘의 파동이 내부 차원으로 퍼져나간다. 칼만은 시스템이 가진 n 차원의 공간 전체를 이 입력 파동들이 전부 덮을 수 있는지를 묻는다. 이를 수학적으로 결착하기 위해 차분 공간의 체적을 측정하는 n \times (n \times m) 크기의 거대한 가제어성 행렬(Controllability Matrix) \mathcal{C} 를 조립하라.

\mathcal{C} = [ \mathbf{B} \quad \mathbf{A}\mathbf{B} \quad \mathbf{A}^2\mathbf{B} \quad \dots \quad \mathbf{A}^{n-1}\mathbf{B} ]

시스템이 컨트롤 가능하다는 물리적 의미는, 저 \mathcal{C} 행렬을 구성하는 기둥 벡터(Column Vectors)들이 서로 평행하지 않고(\text{Linear Independent}), n 차원의 벡터 우주 전체를 그물처럼 팽팽하게 펼쳐낼 수 있다는 뜻이다.
즉, 선형 대수학의 칼을 들이대어 **\mathcal{C} 행렬의 랭크(Rank)가 정확히 시스템의 고유 차원수 n 과 일치함(Full Rank)**을 증명해 내면 된다.
\text{Rank}(\mathcal{C}) = n
만약 랭크 검문에서 \text{Rank} < n 이 산출된다면, 입력의 힘이 닿지 않는 ’데드 존(Uncontrollable Subspace)’이 시스템 내부에 응어리져 있음을 폭로하는 것이다. 아무리 거대한 제어 전압을 쏟아부어도, 그 숨겨진 구석의 상태는 통제력을 벗어난 채 혼자만의 진동으로 발산해 버릴 수 있다.

2. 투시경 역학의 확인: 가관측성 행렬(Observability Matrix)과 랭크 조건

입력의 주도권과는 반대로, 시스템 내부 상태 좌표 \mathbf{x}(t) 자체는 센서 장비의 한계로 직접 측정할 수 없으나 오로지 제한된 계기판인 출력 방정식 \mathbf{y}(t) = \mathbf{C}\mathbf{x}(t) 의 바늘 움직임표만을 보고 내부의 진실을 완벽히 복원해 낼 수 있다면, 이 시스템은 **완전 가관측(Completely Observable)**하다고 간주된다.

이번에는 출력 센서 관측망 \mathbf{C} 를 거점으로 시작하여, 진폭이 자가 성장하는 시스템 내부 궤적 \mathbf{A} 로부터 얼마나 많은 정보를 역산할 수 있는지를 조사한다. 관측의 파동 \mathbf{C}, 한 번의 미분 변화량을 역추적하는 \mathbf{C}\mathbf{A}, 두 번의 역추적 \mathbf{C}\mathbf{A}^2 \dots 등 정보를 층층이 쌓아 내려간 (p \times n) \times n 차원의 가관측성 행렬(Observability Matrix) \mathcal{O} 텐서를 하방 결착 조립하라.

\mathcal{O} = \begin{bmatrix} \mathbf{C} \\ \mathbf{C}\mathbf{A} \\ \mathbf{C}\mathbf{A}^2 \\ \vdots \\ \mathbf{C}\mathbf{A}^{n-1} \end{bmatrix}

가관측성에 대한 기하학적 척추 법칙 역시 앞선 가제어성과 완전히 대칭(Duality)을 이룬다. \mathcal{O} 행렬의 행렬식망이 모든 미지수 \mathbf{x} 공간에 대한 정보를 제공하려면, 이 행렬의 열 기둥들이 선형 독립적이어야 한다.
\text{Rank}(\mathcal{O}) = n
만일 랭크가 n 차원에 미치지 못한다면(Rank Deficient), 여러 개의 다른 내부 상태들이 센서 화면상에서는 완전히 똑같은 하나의 출력값 \mathbf{y} 로 중첩(Aliasing)되어 나타난다는 뜻이다. 즉, 시스템 내부에 출력 센서로는 절대 투시 불가능한 ’블라인드 존(Unobservable Subspace)’이 형성되어 있으며, 제어 공학의 내부 피드백 로직과 칼만 필터 예측망의 눈을 파괴해 버린다.

이 쌍대성 원리(Duality Principle)는 자연계 시스템 해석망에서 가장 우아한 대칭적 균형을 폭로한다. 시스템 (\mathbf{A}, \mathbf{B}) 가 제어 가능하다는 것은, 똑같은 시스템을 뒤집은 상태 행렬 \mathbf{A}^T 와 가상의 출력 행렬 \mathbf{C}_{\text{virtual}} = \mathbf{B}^T 를 가진 시스템이 완벽히 투명하게 관측 가능해진다는 것과 완벽한 수학적 등가를 이룬다. 제어 입력 파동이 내부를 장악하는 기하학과 관측 렌즈가 내부를 투과하는 역학은 동전의 양면처럼 랭크 행렬 공간에 얽혀있는 궁극의 진리이다.
제어 공학자는 시스템의 피드백 설계(Pole Placement)나 옵저버(Observer) 알고리즘을 구동하기 전, 반드시 이 두 거대한 행렬들의 Rank를 스캐닝하여 시스템 설계의 태생적 물리 한계를 파단해야만 한다.
