1.2.2.2.4 극점 배치(Pole Placement) 기법 및 아커만(Ackermann) 공식 기반 폐루프 피드백 이득 행렬 설계

앞선 랭크 스캐닝을 통해 시스템이 ’완전 가제어(Completely Controllable)’하다는 가혹한 관문을 통과했다면, 공학자는 드디어 이 물리계의 영혼을 완벽하게 재설계할 수 있는 신의 권능을 획득한 것이다. 자발적 진동계 \mathbf{\dot{x}} = \mathbf{A}\mathbf{x} 가 가진 태생적인 동역학극점(Pole)들이 불안정하여 발산으로 치닫거나 너무 느리게 응답한다면, 우리는 강제적인 제어 입력 \mathbf{u}(t) 를 시스템의 실시간 상태 궤적과 교배하여 새로운 인공 텐서망을 직조해야 한다. 이것이 상태 피드백 제어의 정수, 극점 배치(Pole Placement) 이론이다. 본 절에서는 센서로 읽어 들인 상태 벡터에 음의 피드백 이득(Gain)을 타격하여 단일 시스템 행렬을 개조하는 역학을 도출하고, 설계자가 원하는 어떤 극점이든 자유자재로 꽂아 넣을 수 있게 해주는 마법의 대수식, **아커만 공식(Ackermann’s Formula)**의 수학적 결착법을 해부한다.

1. 상태 궤적의 덫: 풀-상태 피드백(Full-State Feedback) 제어 법칙

오직 외부에서 무비판적으로 신호를 쏘아 보내는 개루프(Open-Loop) 제어는 외란이나 모델링 오차에 파멸적으로 취약하다. 제어 공학자는 시스템의 현재 혈류 상태를 모두 파악하여 약물을 실시간으로 조절하는 폐루프(Closed-Loop) 순환 제어계를 지향한다. 이를 위해 입력 \mathbf{u}(t) 를 설계할 때, 외부 기준 신호 \mathbf{r}(t) 뿐만 아니라 현재의 상태 벡터 \mathbf{x}(t) 전부를 센싱하여 되먹임(Feedback)하는 강력한 구속 조건을 씌운다.

\mathbf{u}(t) = -\mathbf{K} \mathbf{x}(t) + \mathbf{r}(t)

이 수식은 극도로 파괴적이다. 여기서 이득 행렬(Gain Matrix) \mathbf{K} [m \times n] 는 상태 변수들의 각 진폭에 맞서 얼마나 강한 반작용(Negative Feedback) 제동을 걸 것인가를 결정하는 비례 스케일러다. 이 궤적 통제법을 본래의 선형 상태 방정식 \mathbf{\dot{x}} = \mathbf{A}\mathbf{x} + \mathbf{B}\mathbf{u} 파이프라인 정중앙에 강제로 복붙하여 융합해보라.

\mathbf{\dot{x}}(t) = \mathbf{A}\mathbf{x}(t) + \mathbf{B} (-\mathbf{K}\mathbf{x}(t) + \mathbf{r}(t))
\mathbf{\dot{x}}(t) = (\mathbf{A} - \mathbf{B}\mathbf{K}) \mathbf{x}(t) + \mathbf{B} \mathbf{r}(t)

기적적인 사태가 발생했다. 태생적으로 시스템이 물려받았던 자가 동역학 행렬 \mathbf{A} 가, \mathbf{B}\mathbf{K} 라는 인공적인 피드백 침투 텐서에 의해 폐루프 시스템 행렬(Closed-Loop System Matrix) \mathbf{A}_{cl} = (\mathbf{A} - \mathbf{B}\mathbf{K}) 로 완전히 개조(Modification)되었다! 이제 시스템의 붕괴나 수렴을 관장하는 새로운 운명의 극점(Pole)들은 오로지 \det(s\mathbf{I} - (\mathbf{A} - \mathbf{B}\mathbf{K})) = 0 이라는 새로운 특성 방정식의 뿌리에 의존하게 된다.

2. 제어의 연금술: 극점 배치(Pole Placement) 역학

가제어성(Controllability) 랭크가 n 으로 꽉 차 있다면, 현대 제어 수학은 그야말로 연금술에 가까운 절대 법칙을 증명해 낸다: “공학자가 이득 행렬 \mathbf{K} 의 성분들을 적절히 튜닝하기만 하면, 폐루프 행렬 (\mathbf{A} - \mathbf{B}\mathbf{K}) 의 고유값(극점) n 개를 복소 평면(Complex Plane) 상의 어디든지 원하는 위치에 100% 강제로 꽂아 넣을 수 있다.”

만약 시스템이 너무 흔들린다면 감쇠율을 높이도록 왼쪽 s-복소 평면 깊숙이 극점을 유배상륙시키고, 응답이 지나치게 느리다면 실수축(Real Axis)에서 멀리 떨어진 곳으로 자리를 발치해 옮기면 된다.

3. 대수학의 저격수: 아커만 공식(Ackermann’s Formula) 도출

그러나 복잡한 고차원(예: n=5 이상) 텐서 시스템에서 원하는 극점을 맞추기 위해 \mathbf{K} = [k_1, k_2, \dots, k_n] 의 연립 대수 방정식을 무식하게 생으로 푸는 짓은 극도로 난해하며 비선형적 늪에 빠질 수 있다. 이를 해소하기 위해 위르겐 아커만(Jürgen Ackermann)은 가제어성 행렬과 케일리-해밀턴 정리(Cayley-Hamilton Theorem)를 결탁시킨 궁극의 자동 \mathbf{K} 탐색 해법, **아커만 공식(Ackermann’s Formula)**을 유도해 냈다. (주로 SISO 시스템 즉, m=1 환경에서 가장 직관적 수식을 뽐낸다)

목표로 하는 꿈의 폐루프 극점들 \mu_1, \mu_2, \dots, \mu_n 을 조립하여 당신만이 원하는 이상적인 특성 다항식 \Delta_d(s) 를 구축하라.
\Delta_d(s) = (s - \mu_1)(s - \mu_2)\dots(s - \mu_n) = s^n + \alpha_{1}s^{n-1} + \dots + \alpha_{n}

이 스칼라 변수 s 자리에 원시 자가 행렬 \mathbf{A} 텐서를 그대로 때려 박아넣어 엄청난 크기의 형상 행렬 \Delta_d(\mathbf{A}) 로 변압시킨다.
\Delta_d(\mathbf{A}) = \mathbf{A}^n + \alpha_{1}\mathbf{A}^{n-1} + \dots + \alpha_{n}\mathbf{I}

아커만의 해법은 이제 단 한 큐의 명료한 기하 행렬 연산으로 피드백 이득 격발 스칼라망 \mathbf{K} 를 추출해 낸다.

\mathbf{K} = \begin{bmatrix} 0 & 0 & \dots & 0 & 1 \end{bmatrix} \mathcal{C}^{-1} \Delta_d(\mathbf{A})
(단, \mathcal{C} = [ \mathbf{B} \quad \mathbf{A}\mathbf{B} \quad \dots \quad \mathbf{A}^{n-1}\mathbf{B} ] 는 앞서 증명한 선형 파괴 역학의 가제어성 행렬)

아커만 공식 수식 맨 앞에 배치된 마스크 벡터 \begin{bmatrix} 0 & 0 & \dots & 0 & 1 \end{bmatrix} 의 존재는, 시스템의 꼬리 끝단까지 파고드는 역추어(Inverse trace)의 가장 핵심 코어를 뽑아내겠다는 선형 대수학의 저격총이나 다름없다.
결과적으로 극점 배치 기법과 아커만의 텐서 연산학은 공학자를 불완전한 외란에 끌려다니는 노예가 아닌, 시스템의 운명(\mathbf{A}-\mathbf{B}\mathbf{K})을 자신이 원하는 응답 진폭계로 정확히 세공해 내는 마스터 창조자로 군림하게 한다.
