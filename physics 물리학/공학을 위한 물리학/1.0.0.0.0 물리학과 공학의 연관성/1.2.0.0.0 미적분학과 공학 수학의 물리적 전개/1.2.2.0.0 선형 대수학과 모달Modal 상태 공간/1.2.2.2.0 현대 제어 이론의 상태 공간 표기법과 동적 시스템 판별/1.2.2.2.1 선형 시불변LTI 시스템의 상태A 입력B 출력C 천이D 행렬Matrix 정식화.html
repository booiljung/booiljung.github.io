1.2.2.2.1 선형 시불변(LTI) 시스템의 상태(A), 입력(B), 출력(C), 천이(D) 행렬(Matrix) 정식화

자연계의 수많은 공학 시스템은 외부에서 가해지는 조작된 입력(Input)에 따라 시스템 내부의 에너지가 어떻게 요동치고, 궁극적으로 어떠한 출력(Output)을 바깥으로 토해내는가에 대한 미분 인과율(Causality)로 해석된다. 고전 제어의 전달 함수(Transfer Function)가 이 복잡한 변환 과정을 단일한 ’블랙박스’로 뭉뚱그려 라플라스 변환(Laplace Transform)이라는 s-도메인으로 도피했다면, 현대 제어 이론(Modern Control Theory)은 시스템 내부의 심장부로 직접 쳐들어가 미분 방정식의 날것 그대로를 기하학적 텐서 공간에 직조해 넣는다. 가장 다루기 쉬우면서도 본질적인 역학을 내포한 선형 시불변(LTI: Linear Time-Invariant) 시스템을 대상으로, 고계 미분 방정식을 1차 미분 행렬 다발로 강제 분해해 버리는 상태 공간(State-Space) 표기법의 4대 핵심 텐서인 \mathbf{A}, \mathbf{B}, \mathbf{C}, \mathbf{D} 행렬의 정식화(Formulation) 메커니즘을 선언한다.

1. 고계 미분 방정식의 파괴와 1차 상태 벡터(State Vector)의 도출

어떤 물리 시스템을 구속하는 지배 방정식이 n 계 선형 상미분 방정식으로 주어졌다고 상정하라.
y^{(n)} + a_{n-1}y^{(n-1)} + \dots + a_1 \dot{y} + a_0 y = u(t)
여기서 u(t) 는 단일 입력, y(t) 는 단일 출력이다. n 차 미분의 포물선적 요동을 그대로 다루는 것은 제어 알고리즘 설계에 치명적인 복잡성을 야기한다. 따라서 공학자는 시스템의 과거 역사와 현재의 물리적 에너지를 완벽에 가깝게 기록해 두는 가장 작은 수의 변수 집합, 즉 상태 변수(State Variable) x_1, x_2, \dots, x_n 을 다음과 같이 새롭게 정의하여 연쇄적으로 물고 물리는 1차 미분 사슬을 강제 편입한다.

x_1 = y
x_2 = \dot{y} = \dot{x}_1
x_3 = \ddot{y} = \dot{x}_2
\vdots
x_n = y^{(n-1)} = \dot{x}_{n-1}

마지막 미분항 계통식은 본래의 n 계 미분 방정식으로부터 추출되어 다음과 같이 전개된다.
\dot{x}_n = y^{(n)} = -a_0 x_1 - a_1 x_2 - \dots - a_{n-1} x_n + u(t)

이로써 골치 아프던 단 하나의 n 계 미분 방정식은, 서로가 서로를 미분으로 엮고 있는 투명하고 다루기 쉬운 n 개의 1차 상미분 방정식들로 완벽하게 해체되었다. 이 압착된 변수들의 기둥 \mathbf{x} = [x_1, x_2, \dots, x_n]^T 가 바로 시스템의 영혼(Memory)을 담는 **상태 벡터(State Vector)**이다.

2. 상태 공간 방정식(State-Space Equations)의 4대 텐서 매트릭스 결착

위에서 도출된 1차 다항식 사슬망을 다중 입력과 다중 출력을 허용하는 MIMO(Multiple-Input Multiple-Output) 도메인으로 확장하여 거대한 행렬 스펙트럼으로 덮어씌워라. 현대 제어 이론의 가장 위대한 두 줄의 역학식이 선포된다.

1. 상태 방정식 (State Equation): 시스템의 속도가 내부 에너지와 외부 조작에 의해 어떻게 요동치는지를 지배한다.
\mathbf{\dot{x}}(t) = \mathbf{A} \mathbf{x}(t) + \mathbf{B} \mathbf{u}(t)

2. 출력 방정식 (Output Equation): 은밀히 고동치는 내부 상태 변수들 중 무엇이 센서 관측을 통해 외부 세계로 발현되는지를 결정짓는다.
\mathbf{y}(t) = \mathbf{C} \mathbf{x}(t) + \mathbf{D} \mathbf{u}(t)

이 수식들을 규제하는 네 개의 행렬은 다음과 같은 기하 기계적 의미를 지니고 시스템 궤적을 쥐락펴락한다.

  • \mathbf{A} 행렬 (System Matrix / State Matrix) [n \times n]: 시스템 그 자체의 뼈대이자 자가 동역학 텐서이다. 외부 입력이 완전히 차단된 상태(\mathbf{u}=0)에서도, 시스템이 가진 강성과 감쇠, 마찰 등의 역학 기전만으로 어떻게 자가 진동하고 붕괴(안정성)하는지 그 천성의 위상을 오롯이 간직하고 있다. \mathbf{A} 의 고유값(Eigenvalue)이 바로 시스템의 극점(Pole)이다.
  • \mathbf{B} 행렬 (Input Matrix / Control Matrix) [n \times m]: 조종 장치나 액추에이터에서 발화된 폭발적인 제어 입력 \mathbf{u} 가 n 차원의 상태 공간 내부로 침투할 때, 어떤 변수를 중점적으로 타격하여 강제 거동시킬지 배분하는 조향 및 비율 배분 매개 텐서이다.
  • \mathbf{C} 행렬 (Output Matrix / Observation Matrix) [p \times n]: 내부 상태 \mathbf{x} 들이 센서를 통해 바깥의 모니터(\mathbf{y})로 투사될 때, 렌즈 역할을 하는 관측 기저망이다.
  • \mathbf{D} 행렬 (Feedthrough Matrix / Transition Matrix) [p \times m]: 외부의 강제 입력 \mathbf{u} 가 시스템의 무거운 관성(상태 \mathbf{x})을 거치지도 않고, 빛과 같은 속도로 센서(출력 \mathbf{y})에 곧바로 꽂혀버리는 직접 패스(Direct bypass)의 비율을 통제한다. 일반적인 물리적 기계 시스템이나 서보 제어기에서는 대부분 영행렬(\mathbf{0})로 폐기된다.
