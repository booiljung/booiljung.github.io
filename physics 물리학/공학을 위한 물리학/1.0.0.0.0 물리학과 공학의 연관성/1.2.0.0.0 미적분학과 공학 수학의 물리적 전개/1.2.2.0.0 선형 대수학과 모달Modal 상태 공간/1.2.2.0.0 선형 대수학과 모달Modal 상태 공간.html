1.2.2 선형 대수학과 모달(Modal) 상태 공간

앞서 공학의 지배 방정식 다루었던 미분 다항식의 세계는 연속된 시공간을 가로지르는 우아한 곡선이자 파동의 학문이었다. 미적분학(Calculus)이 물리계의 극미한 순간 변화율과 누적 궤적을 쥐락펴락했다면, 선형 대수학(Linear Algebra)은 그러한 궤적들을 수천, 수만 개의 연립 방정식이라는 차가운 대수적 철창(Matrix) 속에 가두고 지배하는 조직적 기하학이다. 현대 공학에서는 거대한 철교의 각 트러스(Truss) 부재가 받는 응력 분포망, 유체 해석 격자 수백만 개에서 오가는 유량 정보, 그리고 다중 인공위성 편대(Formation)의 무수한 제어 신호가 개별적인 방정식이 아니라 하나의 거대한 행렬군(Matrix System)으로 묶박된다.
이 장에서는 이러한 거시적 연립 시스템을 행렬 구조로 정식화(Formulation)한 직후, 시스템이 내포한 본연의 진동 성향과 공명 특성을 추적하여 기하 공간 전체를 고유한 축으로 강제 재정렬시켜버리는 거대한 차원 변환 역학, 즉 모달(Modal) 분석과 상태 공간(State-Space) 텐서 이론의 절대적 지배력을 수립한다.

1. 개별 방정식의 덩어리화와 행렬 공간망(Matrix Space)의 역학적 선언

스프링 k_1, k_2, k_3 과 질량체 m_1, m_2, m_3 이 일렬로 연결되어 각자의 변위 x_1, x_2, x_3 로 진동하는 3자유도(3-DOF) 시스템 역학을 상정하라.
뉴턴 제2법칙에 의해 이 진동계는 서로 다른 입자들의 변위가 엉켜있는 3개의 연립 2계 상미분 방정식으로 파생되며, 공학자는 이 상호 간섭(Coupling)된 수식의 난잡함 속에서 길을 잃게 된다. 그러나 선형 대수는 이 얽힌 상미분 방정식 다발의 스칼라 관성과 강성 계수들을 거대한 블록으로 뜯어내어 3 \times 3 정방 행렬(Square Matrix)이라는 단일 스펙트럼 안에 질서 정연하게 묶어 세운다.

행렬 텐서를 동원한 단 하나의 지배 역학 선언:
\mathbf{M} \ddot{\mathbf{x}} + \mathbf{C} \dot{\mathbf{x}} + \mathbf{K} \mathbf{x} = \mathbf{F}

여기에 나타난 질량 행렬 \mathbf{M}, 감쇠 행렬 \mathbf{C}, 강성 행렬 \mathbf{K} 는 더 이상 단순한 숫자의 배열이 아니다. 이 행렬들은 입력 벡터 \mathbf{x} 를 받아 방향을 비틀고, 크기를 증폭/축소하여 완전히 새로운 차원의 벡터 공간망으로 이격시켜(Transform) 버리는 강력한 공간 변환(Spatial Transformation) 선형 연산자(Linear Operator)로서 군림한다.

2. 간섭(Coupling) 역학의 붕괴와 고유치/고유벡터(Eigenvalues and Eigenvectors) 진단

행렬 방정식 안쪽, 질량 행렬이나 강성 행렬의 비대각(Off-diagonal) 성분이 살아있는 한 행렬식 내의 x_1, x_2, x_3 는 서로의 진폭에 치명적인 응력 간섭(Coupled Dynamics)을 가한다. 이 응력의 사슬을 끊어버릴 수 있는 선형 대수학의 유일한 구원 검이 바로 **고유치(Eigenvalue)**와 고유벡터(Eigenvector) 발치 역학이다.

특정 행렬 \mathbf{A} 공간 스캐너가 어떤 벡터 \mathbf{v} 를 덮쳤음에도 불구하고, 그 벡터의 방향성 기하 축은 전혀 뒤틀리지 않고 오직 스칼라 값 \lambda 만큼 스케일만 증폭/축소되는 경이로운 불변 궤적이 존재한다.
\mathbf{A} \mathbf{v} = \lambda \mathbf{v}
여기서 \lambda 요소를 고유치, 방향 불변의 기둥 벡터 \mathbf{v} 를 고유벡터라 명명한다. 공학 진동계에서 복원해 낸 고유치 \lambda (혹은 \omega^2)는 그 건축물이나 기계 부품이 외부에서 두들겨 맞았을 때 스스로 공명하며 무너져 내릴 수 있는 선천적 고유 주파수(Natural Frequency) 역학 특성이다. 그리고 이 물리적 고유 주파수 텐서에 맞춰 구조물의 층간 뼈대가 실제로 어떻게 꺾여 춤을 출지에 대한 영구 불변의 형상(Shape) 스냅샷이 고유벡터(Eigenvector / 모드 형상 벡터)로 발현되는 것이다.

2. 모달(Modal) 상태 공간 텐서로의 직교 분해(Orthogonal Decoupling)

수만 자유도의 복잡계 행렬식 \mathbf{K}\mathbf{x} 의 교차 얽힘은 공학적으로 해석이 난해하다. 이제 선형 대수학의 꽃, 형태적 오염 제거술인 모드 대각화(Modal Diagonalization) 루프를 가동하라.
시스템 역학은 각각의 N 개 모달 형상(고유벡터)들을 기둥으로 이어 붙여 하나의 거대한 모달 전이 행렬 매트릭스 \Phi = [\mathbf{v}_1, \mathbf{v}_2, \dots, \mathbf{v}_N] 를 포획 생성한다. 이 \Phi 매트릭스를 좌표계 변환 텐서의 핵심 모선으로 세우고 기존 방정식계 앞뒤로 강제 내적 결합시키는 변환, 이른바 상태 공간 좌표 변환(Coordinate Transformation) 기동 x = \Phi q 을 타결한다.

모달 변환 텐서를 견뎌낸 새로운 운동 방정식은 대각화(Diagonalized)라는 대수적 폭격에 의하여, 복잡하게 이리저리 충돌하던 비대각 간섭 항들이 모조리 0으로 섬멸되는 기적을 토해낸다. 남은 것은 거대한 행렬의 사슬에서 풀려나, 각자의 차원에서 외롭게 진동하는 상호 독립적인 N개의 순수 단자유도(1-DOF) 방정식들뿐이다.
공학자는 이 고립된 단일 모드 좌표계(q 공간) 안에서 기존의 해법망을 거침없이 전개한 뒤, 다시 원래의 \Phi 좌표계로 회귀하여 만물의 동력학적 궤장을 눈앞에 완벽 복원해 낸다. 선형 대수와 모달 해석학은 복잡한 다자유도 공학 미분 방정식을 개별 질점 역학으로 해부해 내는 강력한 차원 지배의 절단도(刀)이다.
