## 0.1 스케일 효과(Scale effect) 왜곡 보정을 위한 모형/원형의 부분적 상사성 스케일 조율 기법

역학적 모델링 과정에서 상사성(Similitude) 분석은 실물 크기의 원형(Prototype) 시스템을 예측하기 위해 축소 모형(Model)을 설계할 때 필수적인 방법론이다. 완벽한 역학적 상사성을 확보하기 위해서는 기하학적 상사(Geometric Similarity), 운동학적 상사(Kinematic Similarity), 그리고 동역학적 상사(Dynamic Similarity)의 세 가지 조건을 모두 동시에 만족해야 한다. 그러나 실제 공학적 실험 환경에서는 축소 비율이 커질수록 서로 다른 유체역학적 혹은 열역학적 무차원 수들을 동시에 일치시키는 것이 물리적으로 불가능해지는 현상에 직면한다. 이러한 상사성 붕괴로 인해 모형 실험 결과와 원형 시스템의 실제 거동 사이에 오차가 발생하는 현상을 스케일 효과 (Scale Effect) 라 정의한다.

따라서 현대 공학에서는 모든 상사 조건을 완벽히 일치시키는 완전 상사성(Complete Similarity)을 포기하고, 시스템 지배 역학에 가장 큰 영향을 미치는 주요 무차원 수만을 선별적으로 일치시키고 나머지 팩터로 인한 오차를 수학적으로 보정하는 부분적 상사성 (Partial Similitude) 기법을 채택하여 적용한다.

스케일 효과의 발생 원인과 한계 분석
선박의 저항 성능이나 댐 수문의 월류량을 평가하기 위한 수조 유동 실험을 가정하자. 모델 선박 설계 시 동역학적 상사를 위해 거시적 중력과 관성력의 비율인 프루드 수(Froude Number, Fr)를 일치시켜야 하며, 동시에 미시적 유동에 영향을 미치는 점성력과 관성력의 비인 레이놀즈 수(Reynolds Number, Re) 또한 일치시켜야 한다.

  • 프루드 수(Froude Number): Fr = \frac{V}{\sqrt{gL}}
  • 레이놀즈 수(Reynolds Number): Re = \frac{\rho V L}{\mu}

원형의 길이 스케일 L_p 와 모형의 스케일 L_m 의 비율이 \lambda = L_p / L_m 이라 할 때, 프루드 수를 일치(Fr_m = Fr_p)시키기 위해 모형의 유속 V_m 은 V_m = V_p / \sqrt{\lambda} 가 되어야 한다.
그러나 이 속력을 레이놀즈 수 공식에 대입하면, 동점성계수(\nu = \mu/\rho)가 동일한 유체(물)를 사용할 경우, 모형의 레이놀즈 수 Re_m 은 원형 레이놀즈 수 Re_p 의 \lambda^{-3/2} 배로 급격히 감소하게 된다. 즉 동일한 작동 유체를 사용하는 한, 프루드 수와 레이놀즈 수를 동시에 맞추는 것은 물리적으로 불가능하며, 이로 인해 압력 저항과 마찰 저항의 스케일 비율 간에 왜곡이 발생하는 것이 스케일 효과의 본질이다.

부분적 상사성의 조율 및 적용 기법
이러한 모순을 해결하고 오차를 보정하기 위해 공학자들은 스케일 조율 기법을 다음과 같이 적용한다.

  1. 지배 무차원 수의 분리 및 선택적 일치 원리
    시스템 거동을 지배하는 가장 핵심적인 무력(Force)을 분석하여, 지배적인 무차원 수만을 강제로 일치시킨다. 예를 들어 선박의 조파 저항(수면 파도 생성 저항)이 지배적인 경우 프루드 수(Fr) 일치를 최우선 조건으로 설정하여 모델을 설계하고, 풍동 실험과 같이 공기 저항 및 후류 와류가 지배적인 항공기 동체 모델의 경우 레이놀즈 수(Re) 일치를 최우선으로 구속한다.

  2. 외삽법(Extrapolation) 및 오차 스케일 계수의 적용
    조율되지 않은 나머지 무차원 수의 편차로 인해 발생하는 2차적 오차 변수는 수학적 외삽 곡선 모델을 통해 보상한다.
    예를 들어 프루드 수를 일치시킨 선박 모형 실험에서 측정된 총 저항력 계수(C_{Tm})에서, ITTC(International Towing Tank Conference) 상관 결선 모델에 의해 이론적으로 계산된 모형 스케일의 마찰 저항 계수(C_{Fm})를 차감하여 순수 잉여 저항 계수(C_{Rm})를 추출한다. 모형과 원형 간에 잉여 저항 계수는 프루드 수 일치로 인해 동일하다(C_{Rm} = C_{Rp})는 상사성 가정을 적용한 뒤, 다시 원형의 거대 스케일에 맞게 산출된 원형 마찰 저항 계수(C_{Fp})를 더하여 최종적으로 원형 시스템의 절대 총 저항 계수(C_{Tp})를 역산해낸다.
    C_{Tp} = C_{Rp} + C_{Fp} \quad (\text{단, } C_{Rp} = C_{Tm} - C_{Fm})

  3. 조작된 모델 (Distorted Model) 설계 기법
    하천이나 해안선의 유동 시뮬레이션 모델에서 실제 수평 길이 수 킬로미터 비율에 맞춰 수직 깊이 스케일을 동일하게 축소하면, 모형의 깊이가 밀리미터 단위로 얇아져 표면 장력과 점성이 비정상적으로 지배하는 치명적인 스케일 왜곡이 발생한다. 이를 방지하기 위해 유도된 것이 왜곡 모형(Distorted Model) 기법이다. 수평 방향 축척(\lambda_x)과 수직 방향 깊이 축척(\lambda_y)을 서로 다르게 조율 적용(\lambda_x \neq \lambda_y)하여 기하학적 상사성을 의도적으로 파괴함으로써, 난류 및 중력 유동 특성의 동역학적 상사를 확보하고 점성 스케일 효과를 억제하는 고도의 조율 보상 기법이다.

이와 같은 부분적 상사성 모델 보정 설계는 실험실의 한정된 스케일 데이터로부터 지구적 거시 규모의 실제 구조물, 열교환기, 항공 우주 시스템의 물리적 메커니즘을 경제적이면서도 수학적으로 무결하게 역산 예측할 수 있는 강력한 공학적 판단 기준을 제공한다.
