1.1.1.1.6 버킹엄(Buckingham) 파이(Pi) 정리의 독립 무차원군(Pi-terms) 매트릭스 방정식

레일리 멱급수법(Rayleigh’s Method of Indices)이 지닌 대수적 수효성(Algebraic Cardinality)의 한계, 즉 미지수의 개수가 방정식의 개수보다 많아지는 복잡계 거동을 해석하기 위해 1914년 에드가 버킹엄(Edgar Buckingham)은 차원 해석법의 정수를 담은 파이(Pi, \Pi) 정리를 발표하였다. 버킹엄 파이 정리는 물리 시스템의 거동을 서술하는 다수의 유도 변수(Derived Variables) 집합을 차원 동차성(Dimensional Homogeneity)에 입각하여 최소 단위의 필수 무차원군(Dimensionless Independent Pi-terms) 매트릭스로 압축하는 가장 체계적이고 보편적인 공학 수학 알고리즘이다.

1. 파이(Pi) 정리의 수학적 선언과 차원 축소성

버킹엄 파이 정리의 핵심 골자는 다음과 같이 요약된다.
“어떤 물리적 현상이 n개의 상호 연관된 물리 변수(Physical Variables: q_1, q_2, \dots, q_n)에 의해 지배되며, 이 변수들을 구작(Compose)하는 기본 차원(Base Dimensions: M, L, T, \Theta 등)의 개수가 m개라면, 해당 물리 방정식은 원래의 n개 변수 대신 서로 독립적인 (n - m)개의 무차원군 \Pi_1, \Pi_2, \dots, \Pi_{n-m}의 함수 관계로 완전히 재구성될 수 있다.”

  • 원형 함수 (Original Function): f(q_1, q_2, \dots, q_n) = 0
  • 축소된 무차원 함수 (Reduced Dimensionless Function): \Phi(\Pi_1, \Pi_2, \dots, \Pi_{n-m}) = 0 또는 \Pi_1 = \Psi(\Pi_2, \dots, \Pi_{n-m})

여기서 (n - m)의 수치는 현상을 온전히 해석하기 위해 필요한 최소한의 실험 횟수나 무차원 선도(Dimensionless Chart)의 축(Axis) 개수가 됨을 의미하며, 이는 공학적 매개변수 연구(Parametric Study)의 복잡도를 기하급수적으로 축소하는 위력을 지닌다. “On Physically Similar Systems: Illustrations of the Use of Dimensional Equations“라는 버킹엄의 원전은 이러한 차원 축소가 갖는 스케일링의 지배력을 현대 공학에 각인시킨 교본이다.

2. 반복 변수(Repeating Variables)의 선정과 매트릭스 도출 알고리즘

파이 변수들을 체계적으로 추출하기 위해서는 원리를 실체화하는 단계별 선형 대수학(Linear Algebra)적 매트릭스 설정이 요구된다.

  1. 지배 변수의 총수 판별 (n):
    현상에 관여하는 모든 독립 및 종속 유도 물리량을 빠짐없이 나열한다.
  2. 기본 차원의 수 결정 (m):
    나열된 유도 물리량들을 구성하는 M, L, T (열전달 포함 시 \Theta) 등 독립 기저 차원의 개수 m을 확인한다. (종종 변수 텐서의 계수(Rank)가 m보다 작을 경우 예외적으로 m을 감소시켜야 한다.)
  3. 반복 변수(Repeating Variables) m개 선택:
    전체 n개의 변수 중에서 시스템의 핵심 특성을 대변하는 m개의 반복 변수를 선정한다. 이들은 각 \Pi 항에 공통으로 나타나며, 이들 자체만으로는 무차원군을 형성할 수 없어야 한다. (일반적으로 유체의 물성(밀도 \rho), 기하학적 척도(길이 D), 운동학적 특성(속도 v)을 한 개씩 선별하는 것이 정석이다.)
  4. 대수적 선형 연립 시스템을 통한 지수(Exponent) 풀이:
    남은 (n-m)개의 비반복 종속 변수를 순차적으로 가져와 반복 변수들과 멱함수 형태로 결합한 후, 차원 동차성 원리(\Pi = M^0 L^0 T^0 = 1)를 만족시키는 지수 매트릭스를 푼다.

예를 들어 \Pi_1을 구성할 때, 반복 변수(\rho, v, D)와 비반복 변수(점성계수 \mu)의 조합은 다음과 같이 기술된다.
\Pi_1 = \mu \cdot \rho^a \cdot v^b \cdot D^c

이를 M, L, T 차원으로 환원하면:
(M^0 L^0 T^0) = (M^1 L^{-1} T^{-1}) \cdot (M^1 L^{-3})^a \cdot (L^1 T^{-1})^b \cdot (L^1)^c

각 기본 차원의 지수 합을 0으로 두어 선형 방정식을 수립한다:

  • M: 1 + a = 0 \rightarrow a = -1
  • T: -1 - b = 0 \rightarrow b = -1
  • L: -1 - 3a + b + c = 0 \rightarrow c = -1

결과적으로 지수를 본 식에 대입하면 \Pi_1 = \frac{\mu}{\rho v D} 가 도출되며, 이 역수는 레이놀즈 수(Re)로 환원된다.

3. 차원 매트릭스(Dimensional Matrix)와 영공간(Null Space)의 동치성

현대 전산 공학(Computational Engineering) 환경에서는 파이 정리를 알고리즘화하기 위해 이를 선형 매트릭스로 취급한다. 각 변수(q_1 \dots q_n)를 열(Column)로, 기본 차원(M, L, T, \dots)을 행(Row)으로 하는 지수 행렬(Exponent Matrix) \mathbf{A}를 구성하라.

이때, 새로운 무차원 \Pi 군을 찾는 행위는 벡터 \mathbf{x} = [x_1, x_2, \dots, x_n]^T 를 미지수로 갖는 동차 선형 방정식(Homogeneous Linear System)을 푸는 것과 완벽히 수학적 궤를 같이 한다.
\mathbf{A} \mathbf{x} = \mathbf{0}

선형대수학의 랭크-널리티 정리(Rank-Nullity Theorem)에 따라 널 공간(Null Space)의 기저(Basis) 벡터들의 개수가 바로 (n - m)개의 독립된 \Pi 군에 해당한다. 관성력, 점성력, 중력, 표면장력 등 수많은 요소가 복잡하게 얽힌 다상 유동(Multiphase Flow) 시스템 시뮬레이터를 코딩할 때, 이 매트릭스 기반의 파이 정리 추출 루틴 자동화는 방대한 실험 변수를 최소 차원의 상사 매니폴드(Similarity Manifold)로 사상(Mapping)시켜주는 불멸의 수학적 엔진이다.
