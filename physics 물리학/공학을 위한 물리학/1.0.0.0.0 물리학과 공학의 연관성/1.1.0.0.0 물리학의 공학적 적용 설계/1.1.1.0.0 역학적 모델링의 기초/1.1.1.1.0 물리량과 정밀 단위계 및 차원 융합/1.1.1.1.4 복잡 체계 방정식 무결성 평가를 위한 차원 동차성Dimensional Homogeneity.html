1.1.1.1.4 복잡 체계 방정식 무결성 평가를 위한 차원 동차성(Dimensional Homogeneity)

공학적 해석을 위해 수립된 모델링 방정식이 물리적 타당성(Physical Validity)을 확보하기 위해서는 차원 동차성(Dimensional Homogeneity)의 원리를 반드시 만족해야 한다. 프랑스의 수학자 푸리에(Jean-Baptiste Joseph Fourier)에 의해 확립된 이 원리는, 자연 법칙을 기술하는 모든 유효한 대수적, 미분, 적분 방정식의 식을 구성하는 각 항(Term)들이 서로 동일한 물리적 차원(Dimension)을 지녀야만 함을 천명한다. 이는 복잡 체계 방정식의 무결성을 1차적으로 평가하는 가장 직관적이고 강력한 검증 도구이다.

1. 차원 동차성의 수학적 정의와 공학적 당위성

임의의 물리적 시스템을 서술하는 지배 방정식이 다음과 같은 n개의 항산(Additive Terms)으로 구성되어 있다고 가정하자.

A + B = C + D

차원 동차성의 원리에 따르면, 기저 차원(Base Dimensions) 집합 \{M, L, T, \Theta, \dots\}에 대해 각 항의 차원 연산자 [\cdot] 기댓값은 논리적으로 완전히 동치여야 한다.

[A] = [B] = [C] = [D]

만약 길이가 1미터인 철근과 온도가 20켈빈인 구역의 값을 단순 덧셈(1 \text{ m} + 20 \text{ K}) 연산한다면 이는 수학적으로는 계산 가능하나 물리적으로는 아무런 의미가 없는 결과(Nonsense)를 도출한다. 따라서 동차성 원리는 미지의 공학 방정식이나 난해한 경험식(Empirical Formula)을 유도하고 검증할 때, 그 식이 기초적인 물리적 오류를 범하고 있지 않은지를 판별하는 필수적인 필터링 채널로 기능한다.

2. 미적분학 연산자 내에서의 차원 이행 규칙

공학에서 널리 쓰이는 미분 방정식과 적분 방정식 내에서도 차원 동차성은 엄격한 수학적 이행 규칙(Transition Rules)을 따른다.
종속 변수 y를 독립 변수 x에 대해 미분하거나 적분할 때, 해당 미적분 연산자 자체가 내포한 차원 변화율을 정확히 추적해야 한다.

  • 미분 연산의 차원(Dimension of Differentiation):
    \frac{dy}{dx} 의 차원은 y의 차원을 x의 차원으로 나눈 몫과 같다. \left[\frac{dy}{dx}\right] = \frac{[y]}{[x]}
    (예: 속도 v = \frac{dx}{dt} 일 때, [v] = \frac{L}{T} = L\cdot T^{-1})

  • 고계 미분 연산의 차원(Dimension of Higher-Order Differentiation):
    \frac{d^n y}{dx^n} 의 차원은 \left[\frac{d^n y}{dx^n}\right] = \frac{[y]}{[x]^n} 로 산출된다.

  • 적분 연산의 차원(Dimension of Integration):
    \int y \, dx 의 차원은 y의 차원과 x의 미소 변화량 차원을 곱한 승수와 같다. \left[\int y \, dx\right] = [y] \cdot [x]
    (예: 일 W = \int F \, dx 일 때, [W] = [F] \cdot L = M\cdot L^2 \cdot T^{-2})

3. 초월 함수(Transcendental Functions)의 무차원 제약 조건

지수 함수(e^x), 로그 함수(\ln x, \log x), 그리고 삼각 함수(\sin x, \cos x 등)를 통칭하는 초월 함수는 공학 계산에서 파동의 감쇠(Wave Attenuation)나 시스템의 위상(Phase) 응답을 서술할 때 빈번히 등장한다.
차원 동차성을 훼손하지 않기 위한 가장 극명한 제약 조건은, 이러한 초월 함수의 인자(Argument, 즉 입력값 x)는 반드시 무차원(Dimensionless)이어야 한다는 사실이다.

예를 들어 감쇠 진동계의 응답식 x(t) = X_0 e^{-\zeta \omega_n t} \cos(\omega_d t - \phi) 에서,
지수 함수의 항인 (-\zeta \omega_n t)를 살펴보면:

  • 감쇠비 \zeta (무차원, [-])
  • 고유 진동수 \omega_n (시간의 역수, [T^{-1}])
  • 시간 t (시간, [T])
    따라서 [\zeta \omega_n t] = [-] \cdot [T^{-1}] \cdot [T] = 1 (무차원)이 됨을 알 수 있다.
    이 규칙이 위배되는 순간 테일러 급수 전개(Taylor Series Expansion)에서 서로 다른 차원의 무한 급수 항들을 덧셈하는 모순이 발생하기 때문에, 이는 방정식의 무결성을 깨뜨리는 가장 전형적인 징후로 간주된다.

4. 경험식(Empirical Equations)과 숨겨진 차원 상수

순수 이론적 해석론이 미치지 못하는 열전달이나 유체 역학 분야에서는, 실험 데이터를 커브 피팅(Curve Fitting)하여 도출한 경험식을 널리 활용한다.

예를 들어, 난류 유동에서의 마찰 손실 수두(Friction Loss Head)를 구하는 헤이즌-윌리엄스 경험식(Hazen-Williams Equation) V = 0.849 C R_h^{0.63} S^{0.54} 와 같은 식은 언뜻 보기에 좌변의 속도 [L \cdot T^{-1}] 와 우변의 차원 조합이 불일치하는 것처럼 보인다.
이러한 경우의 방정식은 ’차원적 비동차 방정식(Dimensionally Nonhomogeneous Equation)’이라 불리며, 특정 지정된 단위계(예: 미터, 초) 내부에서만 유효하게 작동하는 상수 0.849 나 계수 C 속에 복잡하게 변환된 숨겨진 차원(Hidden Dimensions)이 내포되어 있음을 시사한다.

따라서 경험식을 다룰 때는 “A Review of Dimensional Analysis in Engineering“과 같은 문헌에서 권고하듯이, 해당 경험식이 도입된 원시 논문에서 어떤 단위를 전제로 숨겨진 계수를 도출했는지 반드시 확인하고, 일반화된 무차원 형태(Dimensionless Form)로 변환하여 사용하는 습관을 들이는 것이 치명적 설계 오류를 방지하는 최선의 방책이다.
