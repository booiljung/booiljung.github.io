## 0.1 크로네커 델타(Kronecker delta)와 레비-치비타(Levi-Civita) 혼합 기호를 활용한 벡터 대수

초기 고전 역학의 직관적인 기하학적 벡터 연산은 계가 3차원 다중 축을 넘어 추상화될수록 그 수식 전개 과정이 기하급수적으로 복잡해진다. 이를 타개하기 위해 현대 물리 공학과 유체 역학, 텐서 해석학에서는 성분 지표(Index) 방식과 아인슈타인 합 규약을 도입하여 복잡한 다중 벡터 곱 연산을 단순한 대수적 기호 조작으로 압축한다. 그 핵심 연산자가 바로 크로네커 델타(Kronecker delta) 와 레비-치비타 혼합 기호(Levi-Civita Symbol) 이다.

1. 아이덴티티 텐서 연산자: 크로네커 델타 (Kronecker Delta, \delta_{ij})
크로네커 델타는 수학적으로 행렬의 주대각 성분만을 추출하는 단위 행렬(Identity Matrix) 특성을 지닌 2계(Second-rank) 대칭 텐서 기호이다.
단 두 가지의 정수 스칼라 값 특성을 내포한다.
\delta_{ij} = \begin{cases} 1 & \text{if } i = j \\ 0 & \text{if } i \neq j \end{cases}

  • 필터링 및 인덱스 치환 특성 (Substitution Property): 임의의 벡터 A_j 에 크로네커 델타를 곱하여 합산하면, 오로지 i=j 인 성분만 살아남아 기호를 치환하는 필터링 효과가 발생한다. ($ \delta_{ij} A_j = A_i $)
  • 벡터 내적(Dot Product)의 대수화: 서로 직교하는 단위 직교 기저 벡터 \mathbf{\hat{e}}_i, \mathbf{\hat{e}}_j 간의 내적은 크로네커 델타로 완벽히 치환 투영된다($ \mathbf{\hat{e}}_i \cdot \mathbf{\hat{e}}j = \delta{ij} $). 이에 따라 임의의 두 벡터 망 \mathbf{A}, \mathbf{B} 의 내적은 다음과 같이 간결한 스칼라 텐서식으로 축소된다.
    \mathbf{A} \cdot \mathbf{B} = (A_i \mathbf{\hat{e}}_i) \cdot (B_j \mathbf{\hat{e}}_j) = A_i B_j (\mathbf{\hat{e}}_i \cdot \mathbf{\hat{e}}_j) = A_i B_j \delta_{ij} = A_i B_i

2. 순열 텐서 연산자: 레비-치비타 기호 (Levi-Civita Symbol, \epsilon_{ijk})
벡터의 외적(Cross Product) 및 회전(Curl) 연산 모델링을 대수학적으로 완벽히 렌더링하기 위해 도입된 3단위 지표 기호가 레비-치비타 심볼 혹은 완전 반대칭 텐서(Totally Antisymmetric Tensor)이다. 인덱스 i, j, k 의 배열 순서가 우종전(Cyclic)인지 좌종전인지에 따라 부호가 역전된다.
\epsilon_{ijk} = \begin{cases} +1 & \text{if } (i,j,k) \text{ is an even permutation (123, 231, 312)} \\ -1 & \text{if } (i,j,k) \text{ is an odd permutation (213, 132, 321)} \\ 0 & \text{if any two indices are identical (e.g., 112, 331)} \end{cases}

  • 벡터 외적(Cross Product)의 대수화: 단위 벡터 간 외적 구조 \mathbf{\hat{e}}_i \times \mathbf{\hat{e}}_j = \epsilon_{ijk} \mathbf{\hat{e}}_k 원리에 따라, 벡터 외적 \mathbf{C} = \mathbf{A} \times \mathbf{B} 의 k 번째 성분은 다음과 같이 조립된다.
    C_k = \epsilon_{ijk} A_i B_j

3. 기호 간 혼합 연산 기법 (\epsilon-\delta 항등식)
연속체 역학의 난류 와동(Turbulent Eddy)이나 맥스웰 전자기학 쌍대 방정식 내의 다중 외적(예: \mathbf{A} \times (\mathbf{B} \times \mathbf{C}))을 렌더링 풀이할 때, 다음과 같은 두 레비-치비타 기호 간의 스칼라 내적 치환 법칙(Identity)인 \epsilon-\delta 항등식이 핵심 역학 지표로 사용된다.
\epsilon_{ijk} \epsilon_{ilm} = \delta_{jl} \delta_{km} - \delta_{jm} \delta_{kl}

이러한 고차원 선형 텐서 기호들의 융합 대수학 전개 방식은 아무리 복잡하게 꼬인 3차원 공학 수식이라 할지라도 평면 상의 1차원적인 스칼라 지표 조작만으로 기계 및 유동 전산 유도식을 오류 없이 기계적으로 직조하고 프로그래밍 언어로 최적화 변환할 수 있는 철학적 수학 기반을 완성시킨다.
