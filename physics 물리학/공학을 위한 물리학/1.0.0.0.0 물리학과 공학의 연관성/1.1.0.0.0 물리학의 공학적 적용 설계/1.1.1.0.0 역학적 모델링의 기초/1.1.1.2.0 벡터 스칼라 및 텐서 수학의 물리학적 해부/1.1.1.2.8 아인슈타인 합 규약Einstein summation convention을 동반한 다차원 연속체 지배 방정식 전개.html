## 0.1 아인슈타인 합 규약(Einstein summation convention)을 동반한 다차원 연속체 지배 방정식 전개

고전 다변수 미적분학과 3차원 유역 공간의 유체 탄성 복합 방정식을 다룰 때, 나비에-스토크스 방정식(Navier-Stokes Equations)이나 일반화된 훅의 법칙 등은 지표(Index)와 합산 기호(\Sigma)가 무수히 나열되어 수식의 가독성과 지배 역학적 통찰의 포착을 가로막는 병목 현상을 유발한다. 이 한계 장벽을 붕괴시킨 패러다임이 바로 알베르트 아인슈타인(Albert Einstein)에 의해 제안 및 보편화된 수학 표기 체계인 아인슈타인 합 규약 (Einstein Summation Convention) 이다.

1. 합 규약의 논리적 메커니즘과 수학적 압축성
이 합 규약의 핵심 본질은 극도로 명징하다. “단일 수식 내에서 동일한 지표 인덱스(예: i 나 j)가 두 번 이상 중복되어 출현할 경우, 해당 축 지표에 대해서는 공간 차원이 허락하는 범위(일반적으로 1, 2, 3 차원)를 모두 순차 대입하여 더한다는 합산 기호 \Sigma 를 의도적으로 생략한다.” 라는 규칙이다.

  • 예시 1: 두 벡터망 \mathbf{A}, \mathbf{B} 의 내적 곱셈 스칼라 연산
    고전적 표기 체계: \sum_{i=1}^3 A_i B_i = A_1 B_1 + A_2 B_2 + A_3 B_3
    아인슈타인 표기: \mathbf{A \cdot B} = A_i B_i (인덱스 i 가 중복되므로 \Sigma 생략)

  • 예시 2: 텐서와 벡터의 행렬 곱핑 연산 (작용 응력에 따른 기계 표면 하중력 추출)
    고전적 표기 체계: T_i = \sum_{j=1}^3 \sigma_{ij} n_j
    아인슈타인 표기: T_i = \sigma_{ij} n_j (인덱스 j 가 중복 가합 지표, i 는 자유 독립 지표)

이러한 규약 아래에서 중복된 더미 인덱스(Dummy Index)는 임의의 다른 문자로 자유 변경 매핑해도 수식 논리에 무리가 없으며, 오직 홀로 남은 독립 지표(Free Index, 위 식에서 i)만이 결과 산출물의 텐서 차원(1개면 벡터, 2개면 2계 텐서)을 엄격히 규격 관장한다.

2. 다차원 연속체 지배 방정식(Governing Equations)의 우아한 전개
이 합 규약의 도입은 연속체 모델(Continuum Model)의 미분 방정식을 렌더링할 때 범접할 수 없는 미적 단순성과 논리적 파괴력을 발휘한다.
가령 3차원 코시 운동량 방정식(Cauchy Momentum Equation)은 복합적인 가속도 이송 도함수 요소와 텐서 구배 역학을 동시에 포함하는 괴물 같은 규모이지만, 이를 지표 표기법 합 규약 수렴 형태로 압축하면 다음과 같은 마스터 단일 코드 방정식으로 경량화된다.
\rho \left( \frac{\partial v_i}{\partial t} + v_j \frac{\partial v_i}{\partial x_j} \right) = \frac{\partial \sigma_{ji}}{\partial x_j} + \rho g_i
위의 수식 아키텍처는 다음의 다물리 역학을 한 줄의 시(Poem)처럼 담아낸다.

  • \rho: 거시적 매질 밀도 (스칼라 0계 텐서)
  • \frac{\partial v_i}{\partial t}: 위치 정체점 국소 가속도 배열
  • v_j \frac{\partial v_i}{\partial x_j}: 중복된 j 지표에 따른 공간 이류(Advective) 대류 가속도의 3축 합산 전개항
  • \frac{\partial \sigma_{ji}}{\partial x_j}: 응력 텐서 장의 j 축 좌표계 발산(Divergence) 즉 마찰 점성/압력 추진력
  • \rho g_i: 거시적 중력 체적 하중 축 벡터 포스

아인슈타인 합 규약은 단순히 수식의 길이를 줄이는 단축키(Shortcut)에 머물지 않는다. 이 지표 법칙은 현대 공학자들이 전산 유체 역학(CFD)이나 유한 요소 해석(FEA) 컴퓨팅 코어를 구축할 때 배열 포인터(Pointer)와 다중 for 루프(Loop) 로직을 수치 행렬식과 일대일로 절대 오류 없이 사상(Mapping)시킬 수 있도록 돕는 가장 위대한 학술적 코딩 알고리즘의 척추 역할을 수행하고 있다.
