## 0.1 위치, 속도, 가속도 벡터의 시간 도함수적 공간 투영과 궤적 곡선학

고전 역학 및 동역학 시스템에서 입자나 강체의 3차원 공간 내 거동을 서술하는 가장 근본적인 수학적 도구는 시간 변수 t 에 대해 매개 변수화된 이동 궤적 곡선의 벡터 함수이다. 본 절에서는 입자의 위치를 정의하는 벡터망이 미적분학의 시간 도함수(Time Derivative) 연산을 통해 어떻게 공학적 속도와 가속도 텐서로 투영 및 변환되는지 역학적 메커니즘을 상세히 규명한다.

1. 위치 벡터 (Position Vector)와 궤적 방정식
관찰자의 절대 데카르트 직교 좌표계(Cartesian coordinate system)의 원점(Origin, O)에서 출발하여 공간 상의 특정 입자 위치 P(x, y, z) 를 직접 지목하는 거시적 모멘텀 선분이 바로 위치 벡터 \mathbf{r}(t) 이다.
입자가 시간에 따라 이동하면 좌표계의 각 독립 성분은 시간 t 의 스칼라 함수로 정의된다.
\mathbf{r}(t) = x(t)\mathbf{\hat{i}} + y(t)\mathbf{\hat{j}} + z(t)\mathbf{\hat{k}}
여기서 \mathbf{\hat{i}}, \mathbf{\hat{j}}, \mathbf{\hat{k}} 는 각각 x, y, z 축 방향의 단위 직교 기본 벡터(Unit Basis Vector)이다.

2. 속도 벡터 (Velocity Vector) - 궤적 곡선의 1차 미분 체계
입자의 속도 벡터 \mathbf{v}(t) 는 위치 벡터 \mathbf{r}(t) 의 시간에 대한 1차 도함수(First Derivative)로 엄격히 정의된다. 이는 미소 시간 dt 동안 발생하는 입자의 미세 변위 d\mathbf{r} 의 극한 비율 메커니즘을 의미한다.
\mathbf{v}(t) = \lim_{\Delta t \to 0} \frac{\mathbf{r}(t + \Delta t) - \mathbf{r}(t)}{\Delta t} = \frac{d\mathbf{r}}{dt} = \dot{x}\mathbf{\hat{i}} + \dot{y}\mathbf{\hat{j}} + \dot{z}\mathbf{\hat{k}}

  • 기하학적 투영 해석: 입자가 그리는 3차원 곡선 궤적(Trajectory Curve) 상의 임의의 점 P 에서 구한 속도 벡터 \mathbf{v} 는 수학적으로 언제나 해당 지점 곡선의 접선(Tangent) 방향과 완벽히 일치하는 기하학적 렌더링을 띠게 된다. 즉, 속도는 입자의 진행 순간 방향과 스칼라 크기(속력, Speed)를 동시에 구명한다.

3. 가속도 벡터 (Acceleration Vector) - 속도의 시간 변화율과 곡면 투영
가속도 벡터 \mathbf{a}(t) 는 속도 벡터 \mathbf{v}(t) 의 시간에 대한 1차 도함수이자, 위치 벡터 \mathbf{r}(t) 의 2차 도함수(Second Derivative)로 산출된다.
\mathbf{a}(t) = \frac{d\mathbf{v}}{dt} = \frac{d^2\mathbf{r}}{dt^2} = \ddot{x}\mathbf{\hat{i}} + \ddot{y}\mathbf{\hat{j}} + \ddot{z}\mathbf{\hat{k}}
가속도는 속도의 ’크기 변화’뿐만 아니라 ‘방향의 변화’ 비율까지 포괄하여 측정하는 고차원적 역학 지표이다. 곡선 운동을 하는 입자의 가속도 벡터는 공학 해석의 편의성을 위해 기하학적으로 접선 성분(Tangential Component) 과 법선 성분(Normal Component) 인 두 가지 직교 시스템 팩터로 분해(Decomposition)하여 투영 분석하는 것이 일반적이다.

가속도의 자연 좌표계(Natural Coordinate System) 분해
입자가 곡률 반경(Radius of Curvature) \rho 를 가지는 궤적을 속력 v 로 선회 이동할 때, 가속도는 궤적 접선 방향 단위 벡터 \mathbf{\hat{e}}_t 와 구심을 향하는 법선 방향 단위 벡터 \mathbf{\hat{e}}_n 으로 다음과 같이 분할된다.
\mathbf{a}(t) = a_t \mathbf{\hat{e}}_t + a_n \mathbf{\hat{e}}_n = \frac{dv}{dt} \mathbf{\hat{e}}_t + \frac{v^2}{\rho} \mathbf{\hat{e}}_n

  • 접선 가속도 (a_t = dv/dt): 오직 입자의 주행 속력(Speed)의 증감 폭만을 제어하는 모멘텀이며, 곡면 궤적의 진행 저항 혹은 추진력에 기인한다.
  • 법선/구심 가속도 (a_n = v^2/\rho): 입자 이동의 진행 방향 궤적을 억지로 굴절시키는 조향(Steering) 모멘텀을 담당한다. 이 팩터는 곡선 경로 이탈을 방지하고 회전 원운동을 유지시키는 기하학적 구속력이자 원심력의 역학적 근원이다.
