1.1.2.3.7 유한요소/전산해석(FEM/CFD) 모델의 입출력 변수 불확실성 정량화(UQ) 파이프라인

현대 공학 설계의 패러다임이 고비용의 물리적 프로토타입(Prototype) 지향에서 가상의 디지털 트윈(Digital Twin) 및 컴퓨터 응용 공학(CAE) 시뮬레이션으로 전환됨에 따라, 유한요소법(Finite Element Method, FEM) 및 전산유체역학(Computational Fluid Dynamics, CFD)의 역할은 절대적이다. 하지만 아무리 정교한 지배 방정식(Governing Equation)과 이산화(Discretization) 격자망을 구축했다 할지라도, 모델에 주입되는 경계 조건(Boundary Condition), 재료 물성치(Material Properties), 형상 공차(Geometric Tolerance) 등의 입력 매개 변수는 현실계의 본질적인 불확실성(Uncertainty)을 내포하고 있다. 이러한 입력 스페이스의 무작위 변동성이 전산 모사 해(Solution) 공간으로 파급되어 예측 결과의 신뢰도를 저하시키는 현상을 추적하고 통제하는 학문적 기술이 바로 불확실성 정량화(Uncertainty Quantification, UQ)이다. 본 절에서는 FEM/CFD 모델의 입력 불확실성이 해석 결과로 연동되는 관통 파이프라인(Pipeline)과 역학 체계를 정립한다.

1. 전산 모델 내재 불확실성의 근원적 분류 체계

컴퓨터 해석에서의 UQ 파이프라인을 구축하기 전, 시스템에 도입되는 불확실성을 물리학적 및 정보 수학적 역학 관점에서 두 가지 거대 분파로 구획하는 것이 선결 과제이다.

  • 우연적 불확실성 (Aleatory Uncertainty): 재료 파단 강도의 산포, 자연계 풍속의 난류 변동성, 또는 생산 공정에서 발생하는 피할 수 없는 임의적 형상 오차 등 시스템과 자연 자체에 필연적으로 각인된 무작위적(Random) 통계 변동성을 지칭한다. 이는 지식을 아무리 축적하더라도 근원적으로 소거할 수 없는 물리적 변동성이다.
  • 인식적 불확실성 (Epistemic Uncertainty): 해석 대상의 물리적 구동 메커니즘을 엔지니어가 불완전하게 지각하고 있거나 계측 데이터가 심각하게 결핍되어 발생하는 ’지식의 부재(Lack of knowledge)’로 인한 불확실성이다. 단순화된 마찰 모델 적용, 제한된 물성치 데이터로 인한 PDF 추정 오류, 또는 CFD 난류 모델(Turbulence Model)의 수학적 가정에서 파생되는 오류들이 이에 속하며, 이는 정보를 보강하고 모델을 고도화함으로써 경감(Reduction) 가능한 변혁적 오차이다.

2. UQ 파이프라인의 역학적 프레임워크

복잡계 전산 모사에서 불확실성 정량화 파이프라인은 크게 입력 변수의 확률론적 모델링, 전파 역학 계산, 그리고 출력 반응의 구조적 통계 해석이라는 3단 체계 행렬로 거동한다.

2.1 입력 변수의 확률 공간 (Probability Space) 매핑

가장 선행되어야 할 작업은 단일한 스칼라(Scalar) 대푯값으로 입력되던 초기치, 경계 조건 변수들(예: 영률 E, 대류 열전달 계수 h, 입력 압력 P 등)을 확률 밀도 함수(Probability Density Function, PDF) 모델로 치환하는 것이다. 가우스 분포, 베타 분포, 로그 정규 분포 등 각 변수의 물리 화학적 성질에 조응하는 최적 PDF를 투영 설계하라. 더 나아가 공간적으로 분포가 변하는 불균일 재료 특성의 경우 랜덤 장(Random Field) 이론이나 카루넨-루에브 전개(Karhunen-Loève Expansion)를 적용하여 상관성을 띤 국부적 불확실성을 역학 격자에 입히는 수학적 전처리가 요구된다.

2.2 반응면 기법(Response Surface Methodology, RSM) 기반 비간섭형 전파 침투

FEM과 CFD와 같은 거대 해석 시스템은 통상적으로 한 번의 단일 해석 과정에도 방대한 수치 미분 적분 자원과 컴퓨팅 시간(CPU time)을 포식한다. 따라서 몬테카를로 시뮬레이션(MCM)을 나이브(Naive)하게 직접 접목하는 방식은 현실적으로 구동 불가능하다. 이를 회피하기 위한 가장 진보적 대응책이 해석 솔버(Solver) 내부 코드를 수정하지 않는 비간섭형(Non-intrusive) 대리 모델(Surrogate Model) 혹은 반응면(Response Surface) 기법의 도입이다.

Y(\mathbf{X}) \approx \tilde{Y}(\mathbf{X}) = \sum_{j=0}^{P} \alpha_j \Psi_j(\mathbf{X})

여기서 진해석 함수 Y(\mathbf{X}) 는 메타 모델 \tilde{Y} 로 치환된다. 이 과정에서 다항식 혼돈 전개(Polynomial Chaos Expansion, PCE) 혹은 가우시안 프로세스 회귀(Gaussian Process Regression, Kriging)가 주력 전파 엔진으로 등판한다. 몇 번의 신중하게 조준된 해석 점(Collocation Points)만을 추출 연산한 뒤, 입력 불확실성 공간과 출력 응답 공간을 잇는 압축된 다차원 지형도(형상면)를 수학적으로 축조해 내는 것이 핵심이다.

가동된 대리 모델은 연산 소요 비용이 사실상 제로(Zero)에 수렴하므로, 이 압축된 메타 공간 위에서 수백만 번의 몬테카를로 난수 대입을 전개하여 최종 관심 대상(응력 집중부의 최대 등가 응력, 양력 계수 강하량 등)의 확률 밀도 역학 곡선을 즉각적으로 회수할 수 있다.
결론적으로 UQ 파이프라인의 완성은 무미건조하게 결정론적인(Deterministic) 안전계수(Safety Factor)를 맹신하던 과거의 설계를 타파하고, “시스템이 99.7% (3\sigma) 의 확률로 허용 응력을 넘지 않을 것이다” 라는 강건 설계(Robust Design) 기반의 통계적 인증 메커니즘을 부여하는 현대 공학의 최고봉에 등극하게 된다.
