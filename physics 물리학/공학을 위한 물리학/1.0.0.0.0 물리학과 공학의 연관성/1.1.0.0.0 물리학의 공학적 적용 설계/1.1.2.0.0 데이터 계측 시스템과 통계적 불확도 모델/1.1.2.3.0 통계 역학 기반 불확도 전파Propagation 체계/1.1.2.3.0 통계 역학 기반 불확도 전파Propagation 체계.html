1.1.2.3 통계 역학 기반 불확도 전파(Propagation) 체계

현대 공학 계측 및 물리 시스템 관측에 있어서 어떤 주어진 측정 대상의 대상 계측량(Measurand)은 결코 완전하고 오차 없는 단일한 진값(True value)으로 확정 및 측정될 수 없다. 모든 계측과 관찰 과정에는 필연적으로 물리적, 통계역학적 불확실성이 내포되며, 이를 통계 수학적 및 열역학적 확률 기법을 투영하여 정량적이고 객관적인 스케일로 규명하는 것이 바로 불확도(Uncertainty)의 기본 개념이다. 본 절에서는 측정 장비, 센서 결함 및 관측 환경에 의해 다발적으로 발생한 기초 독립 불확도 인자들이 다변수 수학적 방정식을 거쳐 최종 공학 출력 결과물에 어떠한 비선형적인 양태로 그 통계적 분산(Variance) 기여도를 파급 및 증폭시키는지 규명하는 불확도 전파(Uncertainty Propagation) 체계에 대해 상세히 전개한다.

1. 측정 결과의 통계 역학적 확률 분포 본질

계측기를 통해 수집된 각 개별 입력 변수 데이터 X_i는 완벽하고 고정된 실수가 아니라 특정한 확률 밀도 함수(Probability Density Function, PDF)를 연속적으로 갖는 확률 변수(Random Variable)로 모형화되어야 역학적 타당성을 거머쥔다. 이 확률적 분포의 통계적 중심 성향을 대변하는 추정값을 기대값(Expected Value, x_i)이라 칭하며, 이 기대 추정값 주변의 분산 정도를 광범위하게 나타내는 물리적 거리 지표인 통계 2차 모멘트 표준 편차(Standard Deviation)가 곧 당해 변수 데이터에 내재된 표준 불확도(Standard Uncertainty, u(x_i))로 정의된다.

1.1 계측 불확도 유입 모델의 분류 체계 구획 (Type A vs Type B)

측정학에 관한 국제 기구 지침에 따라서 기초적인 시스템 불확도 유입 요인은 아래와 같이 분류된다.

  • A형 불확도 (Type A Evaluation of Uncertainty): 실험자가 수행한 일련의 독립적인 반복 관측 군집 데이터에 기초하여, 선형 통계적 분석—즉 무작위 산술 평균의 계산과 단일 관측 혹은 평균값의 실험적 표준 편차를 활용해 직접적인 가우시안(Gaussian) 확률 분포를 빈도주의적으로 추정하는 방법론이다. 반복 측정이 불확도 분해능을 통제할 수 있을 때 주로 도입하라.
  • B형 불확도 (Type B Evaluation of Uncertainty): 실험적 연속 통계 관측 군집 외에 이미 실험실 내에 확보된 선험적 물리학 지식, 계측 장비의 외부 교정 증명서(Calibration Certificate), 센서 제조사에서 배포한 공식 환경 규격서(Specification), 이전 유사 환경에서의 기준 측정 데이터 경험 분포 등을 기반으로, 직사각형(Rectangular), 혹은 삼각형(Triangular) 형태의 확률적 한계 분포 구간을 사전적으로 역산출하여 엮어내는 학술 분석이다.

2. 불확도 전파 편미분 법칙 (Law of Propagation of Uncertainty, LPU)

공학적 최종 출력 목표량 Y가 복수 개의 분리된 독립 입력 변수 벡터 집합 \mathbf{X} = (X_1, X_2, \dots, X_N)에 의해 비선형적인 다변수 함수 관계 Y = f(X_1, X_2, \dots, X_N)로 결합되어 구속된다고 가정하라.
물리적 계측을 통해 각 입력 채널 변수의 계측 추정값 x_i와 도출된 표준 불확도 스칼라 역량 u(x_i)가 주어졌을 때, 최종 시뮬레이션 및 데이터 출력 결과 y의 합성 표준 불확도(Combined Standard Uncertainty, u_c(y))를 판별해 내는 것이 불확도 전파 체계의 핵심이다.

이를 수학적으로 정식화하기 위하여 다변수 테일러 급수 급항 전개(Taylor Series Expansion)를 기반으로 1차 국소 선형 근사(First-Order Local Linear Approximation)를 기립 수행하면, 다발적 파라미터로 인한 출력 변수의 미소 스케일 변동률은 다음의 전미분(Total Differential) 연속 방정식으로 무한소 환원된다.

dy \approx \sum_{i=1}^{N} \frac{\partial f}{\partial x_i} dx_i

이 미시적 수학 공식에서 편미분항인 \frac{\partial f}{\partial x_i}를 개별 매개변수의 변동 에너지가 최종 결괏값 도출 공간에 미치는 전송 영향력을 기울기로 정미하는 감도 계수(Sensitivity Coefficient, c_i)라고 엄격하게 칭한다.
목표 출력량 분포의 정규화 분산(Variance)인 u_c^2(y)를 전 방위적으로 도출해 내기 위해 시스템 차원의 분산-공분산 행렬 연산(Variance-Covariance Matrix Operation)을 접목 기속하면, 결과적으로 다음과 같은 완전한 형태를 띠는 2차 다항 불확도 전파 방정식을 얻게 된다.

u_c^2(y) = \sum_{i=1}^N \left( \frac{\partial f}{\partial x_i} \right)^2 u^2(x_i) + 2 \sum_{i=1}^{N-1} \sum_{j=i+1}^N \frac{\partial f}{\partial x_i} \frac{\partial f}{\partial x_j} u(x_i, x_j)

여기 우변의 잔여항 u(x_i, x_j)는 두 다중 계측 채널 변수 스펙트럼 간의 통계 역학적 교차 결합 상관 상태를 비례적으로 나타내는 공통 분산, 즉 공분산(Covariance) 요소이다. 서로 통계적으로 결합된 센서 환경의 간섭 오차(Interference error) 역률을 평가할 때 필수적으로 삽입하라.

2.1 상호 독립적 입출력계의 불확도 가중 전파 직교성

일반적인 통상 수준의 공학 거시적 설계 환경에서 관여 변수들이 서로 물리적 의존성 없는 상호 독립(Mutually Independent) 상태일 때, 피어슨 상관 계수(Correlation Coefficient) 역학이 수치상 영산(Zero)이 되므로 미분 교차항 텐서인 공분산이 소거된다. 이 원리로 인해 위 공식은 아래와 같이 더욱 직관적이고 다채로운 직교 합성(Vector Orthogonal Synthesis) 수식으로 극명히 간소화된다.

u_c(y) = \sqrt{ \sum_{i=1}^N \left( c_i \cdot u(x_i) \right)^2 }

이 공식의 물리학적 공간 의의는 극도로 자명하다. 각 입력 센서 변수로 인해 태동하는 개별 불확도 팩터(Factor)들이 가상의 다차원 공간계 내에서 마치 상호 직교하여 간섭하지 않는 스패닝 기저 벡터(Spanning Basis Vectors)들처럼 묘사되며, 이것의 총체적 역학 에너지 총량이 스칼라 유클리디안 노름(Euclidean Norm) 연산 법칙과 동일한 제곱합의 제곱근(Root Sum Square, RSS) 비선형 투영 형태로 최종 오차 공간에 합산된다는 점이다.

3. 요인별 불확도 인자 민감도 기여 및 구조 시각적 파이프라인 정립

다중 복합 계측 데이터 병합 시스템을 총괄 설계하는 엔지니어는 어떤 단일 스펙 측정 센서 채널의 특정 잡음이나 비선형 분해능 불량이 전체 시스템 성능의 발목을 가장 극단적으로 잡고 있는지 신속하게 판별할 의무가 있다.
이를 위해 각 불확도 발원 인자가 결과에 미치는 상대 분산의 지배 투여율, 즉 불확도 예산(Uncertainty Budget)을 수치적으로 개별 병합 해부해야 한다. 특정 변수 x_i에서 기인한 불확도 전단 기여율 P_i는 백분위로 계산되며 다음과 같은 율법으로 표현된다.

P_i = \frac{ \left( c_i \cdot u(x_i) \right)^2 }{ u_c^2(y) } \times 100 (\%)

이 시각적 파이프라인 체계도를 바탕으로 데이터 전문 실험자는 오차 지분이 확률적으로 가장 지배적인 입력 채널(예컨대 환경적 영향을 크게 받는 아날로그 온도 보상 프로브나 고주파 대역 증폭단 등)에 대해 계측 보강 비용이나 공차 관리를 재분배하고 계측 정밀도 이득과 비용 편익(Cost-Benefit)을 계량적으로 역추적하여 최적화할 수 있다.
결과적으로 불확도 전파 메커니즘 모델의 엄격한 과학적 정립은 수천 개의 변수가 요동치는 최신 컴퓨터 복잡계 물리학 시뮬레이션 시스템의 구동이나, 빅데이터 정밀 회귀 적합 모델 검증 기법을 현장에 투입할 때 그 신뢰도를 선제적으로 확증하는 필수적 물리 모델링 전제 조건으로 강력히 작용하게 된다.
