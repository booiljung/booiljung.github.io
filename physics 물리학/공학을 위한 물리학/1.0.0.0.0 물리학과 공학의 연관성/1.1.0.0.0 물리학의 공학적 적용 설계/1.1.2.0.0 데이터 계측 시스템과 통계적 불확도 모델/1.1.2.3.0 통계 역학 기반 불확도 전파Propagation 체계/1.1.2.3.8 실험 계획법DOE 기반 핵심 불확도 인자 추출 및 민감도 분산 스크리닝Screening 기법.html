1.1.2.3.8 실험 계획법(DOE) 기반 핵심 불확도 인자 추출 및 민감도 분산 스크리닝(Screening) 기법

다변수 복합 계측 체계나 거대 컴퓨터 시뮬레이션 모델에 내재된 입력 불확도 변수의 개수가 수십 내지 수백 개로 팽창하게 되면, 차원의 저주(Curse of Dimensionality)로 인해 해석의 연산량은 우주적 규모로 기하급수적 발산을 일으킨다. 시스템의 최종 응답 공간에 관여하는 수많은 인자들 중, 실제로 결괏값의 통계적 분산(Variance)에 지배적인 타격을 입히는 핵심 변수는 극소수인 파레토 법칙(Pareto Principle)의 역학적 형태를 띠게 마련이다. 따라서 막대한 자원의 낭비를 방지하고 불확도 모델링의 효율을 극대화하기 위해, 전체 예비 파라미터 집단에서 시스템을 관통하는 치명적인 핵심 인자(Vital Few)만을 통계 역학적으로 추출해 내는 실험 계획법(Design of Experiments, DOE) 기반 민감도 스크리닝(Screening) 메커니즘을 적용하라.

1. 실험 계획법(DOE) 기반 다차원 공간계 탐색

스크리닝의 초기 목표는 최소한의 실험(혹은 전산 해석) 횟수만으로 변수들 사이의 비선형적인 주효과(Main Effect)와 다변수 간섭에 의한 상호 작용 효과(Interaction Effect)를 포착하는 공간 축조이다. 직교 배열(Orthogonal Array) 혹은 라틴 하이퍼큐브 샘플링(Latin Hypercube Sampling, LHS)은 입력 설계 변수 공간 \mathbf{X} 내에서 각 파라미터를 빈틈없이 찌르도록(Probing) 실험점들을 등방적으로 안배하는 고효율의 샘플링 텐서 조합론이다.

1.1 직교성을 활용한 분산 분해(ANOVA)

설계된 DOE 매트릭스에 입각하여 시스템을 구동시킨 뒤 결괏값을 회수하면, 분산 분석(Analysis of Variance, ANOVA) 수학을 접목하여 전체 시스템의 응답 분류 에너지를 독립적인 성분들의 합으로 직교 분해(Orthogonal Decomposition)한다.

SSTO = \sum_{i=1}^N SS_i + \sum_{i<j} SS_{ij} + SS_{Error}

여기서 SSTO는 계측된 전체 변동의 총 제곱합을 지칭하고, SS_i는 단일 변수 X_i에 의해 자체적으로 유발된 주효과 제곱합, SS_{ij}는 X_i와 X_j의 충돌로 융합된 2차 상호작용 변동 역학을 의미한다. ANOVA를 통한 이 분해 선언은 전체 오차 에너지 공간에서 각 입력 파라미터가 장악한 지분을 정량적으로 확증하는 분수령이 된다.

2. 민감도 분산 지표: 소볼 지수(Sobol’ Index)와 역학 스크리닝

선형 국소 영역의 단순한 편미분 계수(\partial f / \partial x_i)에 의존하는 원시적인 민감도 체계를 넘어서, 입력 변수의 전역적인 PDF와 완전히 비선형적인 곡면계를 전면 포용하여 상대적 기여도를 평가하는 전역 민감도 해석(Global Sensitivity Analysis, GSA)의 정수가 바로 소볼 지수(Sobol’ Index) 역학이다.

어떤 입력 파라미터 X_i의 주효과가 최종 출력 분산 V(Y)에 직접적으로 타격하는 단일 기여율 비율인 1차 소볼 지수(First-Order Sobol’ Index, S_i) 의 방정식은 조건부 분산(Conditional Variance)을 바탕으로 아래와 같이 정의된다.

S_i = \frac{V_{X_i} ( E_{\mathbf{X}_{\sim i}}[Y | X_i] )}{V(Y)}

1.2 총효과 지수(Total Effect Index)의 지배력 확인

더욱 치명적인 역학 인덱스는 해당 변수 고유의 작용뿐 아니라 다른 모든 잔재 변수들과 얽힌 교차 결합(Coupling) 상호작용 지분까지 모조리 병합하여 평가하는 총효과 소볼 지수(Total Effect Sobol’ Index, S_{Ti}) 이다.

S_{Ti} = 1 - \frac{V_{\mathbf{X}_{\sim i}} ( E_{X_i}[Y | \mathbf{X}_{\sim i}] )}{V(Y)}

해당 행렬값이 0에 가깝다면 그 변수 X_i는 전체 계측 시스템의 불확실성에 어떤 형태와 결합으로도 파장을 미치지 못하는 정적 노이즈(Trivial Factor)에 불과함이 역학적으로 증명된다. 이런 둔감 인자 무리는 스크리닝 체계를 통해 후속 정밀 불확도 전파 체계망에서 완전히 소거(Screen-out)하여 상수화 시킴으로써 모델의 차원 축소(Dimensionality Reduction)를 달성하라.
