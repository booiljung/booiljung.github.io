1.1.2.3.2 A형/B형 불확도 평가 방법론과 가우시안/직사각형 확률 밀도 함수(PDF) 분포 역학

물리량의 계측 결과는 본질적으로 고정된 스칼라 절대값이 아니라 통계 역학적으로 규정된 확률의 안개 속에 존재한다. GUM(Guide to the Expression of Uncertainty in Measurement)에 의거하여, 엔지니어는 시스템 불일치를 유발하는 독립 기초 불확도 인자들을 획득 기법과 정보의 원천에 따라 철저히 Type A와 Type B 부류로 양분하여 평가해야만 한다. 본 절에서는 두 평가 방법론의 본질과, 해당 불확도를 표준 스케일로 정규화하기 위해 투영하는 가우시안(Gaussian) 및 직사각형(Rectangular) 등 확률 밀도 함수(Probability Density Function, PDF)의 공간 연속 분포 역학 모델을 상세히 수립한다.

1. A형(Type A) 불확도 평가와 빈도주의 통계학

Type A 평가 방법론은 오직 실험실 벤치에서 동일한 조건 하에 물리량을 독립적으로 거듭 관측하여 추출해 낸 유의미한 표본 데이터 군집의 빈도주의적 통계(Frequentist Statistics)에 전적으로 기대는 불확도 산출 패러다임이다.

1.1 평균과 실험 표준 편차 도출 메커니즘

어떤 측정 환경에서 특정 물리적 신호를 n번 반복하여 샘플링 취득했을 때 계측 변수 벡터 x_k (k=1, 2, \dots, n)가 도출되었다고 가정하라. 이 표본 데이터의 최적 기댓값 추정치는 통계적 산술 평균(Arithmetic Mean, \bar{x})이 된다.
분산의 산포도를 측정하기 위해 개별 표본 데이터 군의 실험 표준 편차(Experimental Standard Deviation, s(x)) 행렬을 도출한다.

s(x) = \sqrt{ \frac{1}{n-1} \sum_{k=1}^n (x_k - \bar{x})^2 }

1.2 중심 극한 정리와 가우시안(Gaussian) PDF 분산의 정규화

단일 측정이 아닌 ‘평균값(\bar{x})’ 자체를 그 계측 모델의 대푯값 변수로 수합할 때, A형 표준 불확도 u_A(x)는 이 평균의 실험 표준 편차(Experimental Standard Deviation of the Mean) 로 정량화되어야 한다.

u_A(x) = \frac{s(x)}{\sqrt{n}}

이 연산의 수학적 근원에는 확률 역학의 토대인 중심 극한 정리(Central Limit Theorem)가 존재한다. 복수의 무작위 잡음 및 변동 요인이 합산될 경우, 그것의 확률 밀도 분포 공간 형태는 표본 수 n이 증가함에 따라 본연의 분포 형상과 관계없이 자연계의 중심적 대칭 종 모양 곡선인 가우시안 정규 분포(Gaussian Normal Distribution)로 급격히 수렴한다. 따라서 Type A 불확도 분포는 명확히 가우시안 PDF 형태임이 공학적으로 승인된다.

2. B형(Type B) 불확도 평가와 베이지안(Bayesian) 선험적 신뢰

반면 단일한 회수만 계측했거나, 아예 통계적 실험 추출 자체가 물리적으로 원천 봉쇄된 파라미터(예: 해외에서 구입한 계측 센서의 공식 스펙 한계치, 물성표에 기재된 영률 오차, 문헌의 과거 캘리브레이션 성적서 값 등)에 대해서는 Type B 평가 방법론을 전격 도입하라. 이 기법은 통계 빈도학적 실험이 아닌 외부 선험 정보(Apriori Knowledge)와 베이지안적 주관 확률 신뢰도에 근거하여 확률 분포 공간의 한계를 규정한다.

2.1 직사각형 분포(Rectangular Distribution) 역학

B형 평가에서 가장 널리 통용되는 보수적 접근 모델은 직사각형 혹은 균등 확률 분포(Uniform Distribution) PDF 모델이다. 제조사에서 제공한 장비 규격서에 계측 오차 한계(Tolerance Limits)가 특정 구간 \pm a 로 제공되었고, 이 범위 내에서 어떤 값이 더 발생할 확률 성향이 높은지 편중 정보(Bias Information)가 완전히 부재한다고 단정될 때 기용한다.
이 구간 내 모든 연속 공간상의 확률 밀도는 1/(2a)의 동일 평탄 상수(Constant)로 유지된다.
역학적 모멘트 적분을 수행하여 해당 직사각형 PDF 면적 공간계 내에서 분산을 구하고 그 제곱근인 B형 표준 불확도 u_B(x)를 도출하는 공식은 아래와 같다.

u_B(x) = \frac{a}{\sqrt{3}}

이 계수 \sqrt{3}은 직사각형 스케일 한계 경계를 가우시안 통계 1\sigma에 대응되는 표준 편차 차원으로 균등화(Equalization) 및 정규화하기 위한 통계 기하학적 인자이다.

2.2 삼각형 및 분산 특정 확률 밀도(PDF) 모델

한계 범위 \pm a 내에 있으나 관측 특성상 중심 기댓값 부근에 데이터가 역학적으로 더 조밀하게 집중될 것이라 합리적 추론이 가능한 시스템 기전(예: 두 직사각형 분포가 중첩되어 합성된 공차 모델 등)에서는 도함수 및 적분에 의해 삼각형(Triangular) PDF 모델을 설계하라. 이 경우 꼬리 구간의 확률 밀도가 0에 선형 수렴하므로, B형 표준 불확도는 다소 완화된 분모 인자를 갖는다.

u_B(x) = \frac{a}{\sqrt{6}}

만약 예전의 공식 교정 성적 증명서(Calibration Certificate)에서 이전에 다른 계측 기관이 남긴 확장 불확도 U 와 포함 인자 k (예: 신뢰도 95%, k=2) 가 선험적으로 기재되어 있다면, 이 PDF 분포는 가우시안으로 간주되어 단순히 역연산을 가한다.

u_B(x) = \frac{U}{k}

3. 평가 체계의 병합적 역학 통합 정규화

계측 데이터 처리 과정에서는 계측기의 내부적 체계 오차(B형: 제조사 규격, 환경 변수 등)와 외부 노이즈에 의한 무작위 오차 변동(A형: 반복 취득 데이터)이 동시에 복합적으로 존재한다. 하지만 GUM 표준에 따르면 궁극적으로 이 두 가지 기원의 분산 인자들은 PDF 스케일을 일목요연하게 1\sigma급의 분산 표준 편차로 매핑시킨 다음, 독립된 직교 텐서계로서 모두 구분 없이 동등한 벡터 자격으로 불확도 전파 방정식(RSS 합성 공간) 내에 차원 투입되어 합성 표준 불확도 u_c를 재구축하는 통섭과 융합의 핵심 역학 재구성을 거치게 된다.
