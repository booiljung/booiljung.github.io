1.1.2.3.5 몬테카를로 시뮬레이션(Monte Carlo Simulation)을 이용한 비선형 물리계 불확도 수치 추정

전통적인 GUM(Guide to the Expression of Uncertainty in Measurement) 프레임워크는 테일러 급수(Taylor Series) 기반의 1차 선형 근사(First-Order Linear Approximation)를 중심으로 작동한다. 그러나 복잡한 비선형 역학계(Nonlinear Dynamics)나 다상 유체 체계, 고도의 전자기장 교차 커플링 등 수학적 비선형성(Nonlinearity)이 극한에 달하는 공학 계측 모델에서는 1차 근사가 심각한 수학적 절단 오차(Truncation Error)를 촉발한다. 더욱이 입력 변수의 확률 밀도 함수(Probability Density Function, PDF)가 가우시안(Gaussian) 분포에 수렴하지 않고 극단적인 비대칭(Asymmetric) 양태를 보일 때, 해석적 전파 방정식은 참값 구간 추정에 실패한다. 본 절에서는 이러한 해석적 한계를 극복하고 컴퓨터 난수 거동을 통해 확률 분포를 직접적으로 매핑(Mapping)하는 몬테카를로 보충 지침(GUM Supplement 1, Monte Carlo Method)의 비선형 불확도 수치 추정 역학을 해부한다.

1. 몬테카를로 시뮬레이션(MCM)의 통계 역학적 원리

몬테카를로 시뮬레이션(Monte Carlo Simulation)은 복잡하고 해석 불가능한 다변수 방정식을 풀이하기 위해, 통계 역학적인 의사 난수 생성기(Pseudo-Random Number Generator, PRNG)를 활용하여 대규모 수치 표본을 폭격하듯 생성하고, 시스템의 거동을 강제적으로 현출시켜 관찰하는 완전한 수치 해석(Numerical Analysis)적 돌파 기법이다.
불확도 전파의 관점에서 MCM은 각각의 입력 파라미터 X_i에 대해 사전에 할당된 PDF 공간(예컨대 가우시안, 직사각형, 삼각형 등)으로부터 M개(보통 10^5 \sim 10^6 개)의 임의의 표본을 추출(Sampling)하는 것에서 출발한다.

x_i^{(k)} \sim \text{PDF}(X_i) \quad \text{for } k = 1, 2, \dots, M

1.1 역학 모델 계의 개별 투영과 분포 수합

생성된 무작위 입력 벡터군 \mathbf{x}^{(k)} = (x_1^{(k)}, x_2^{(k)}, \dots, x_N^{(k)}) 은 평가하고자 하는 대상 비선형 물리 계측 방정식 Y = f(\mathbf{X}) 에 각각 독립적으로 M번 대입 연산된다.

y^{(k)} = f(\mathbf{x}^{(k)})

이 연산을 통해 얻어진 거대한 출력 표본 y^{(k)} 군집은, 수학적 근사화나 어떠한 변형 없이 모델 내부의 비선형적 굴곡과 교차 공분산(Covariance) 텐서를 온전히 통과한 순수한 결과물이다.

2. 결괏값의 경험적 확률 밀도 함수(Empirical PDF) 및 불확도 추출

수십만 번 시행된 y^{(k)} 의 결괏값 풀(Pool)을 빈도주의적 히스토그램(Histogram) 공간계로 분류하면, 최종 출력량의 경험적 확률 밀도 함수(Empirical PDF)가 가동된다. 여기서 비대칭 꼬리(Heavy tail)의 발생이나 비정규적 편향 분포 공간 구조를 통계 시각적으로 명백히 관측할 수 있다.

  • 기댓값의 최적 추정(Estimate of Measurand): 시스템의 기댓값 y는 도출된 경험적 집단의 산술 평균으로 회귀하여 규명된다.
    y \approx \frac{1}{M} \sum_{k=1}^M y^{(k)}
  • 표준 불확도 도출(Standard Uncertainty): 표준 불확도 u(y)는 이 출력 집단의 실험 표준 편차(Experimental Standard Deviation) 역학으로 직관적으로 산출된다.
    u^2(y) \approx \frac{1}{M-1} \sum_{k=1}^M \left( y^{(k)} - y \right)^2

2.1 짧은 최단 신뢰 구간(Shortest Coverage Interval) 산출

가우시안 분포에 얽매인 GUM 체계는 대칭적인 신뢰 구간 y \pm U 만을 고집한다. 그러나 MCM에서 드러난 비대칭 분포에서는 좌우 확장에 따른 포함 인자(Coverage Factor)를 일괄 적용할 수 없으므로, 해당 데이터 배열을 오름차순으로 정렬(Sorting)한 뒤, 목표 확률 p(예: 95%)를 담보하면서도 공간적 길이가 가장 짧은 수치 범위를 색인 탐색하여 비대칭 신뢰 구간 [y_{low}, y_{high}]을 명확히 지정해 내는 기술적 진보를 이룩한다.

3.1 해석적 GUM 모델과의 공학적 유효성 검증 체계

몬테카를로 시뮬레이션 모델 수립은 GUM 1차 선형화 프레임워크의 수학적 건전성 제고를 검증(Validation)하는 보조 및 최상위 심판자 역할을 겸비한다. 만약 전통적인 편미분 전파 방식에 의해 도출된 확장 불확도 U 구간이 MCM이 전산 계산해 낸 최단 포용 한계 구간 범주에 오차 허용비 이내로 수렴한다면, 해당 공학 모델의 1차 근사화 성향이 물리적으로 온당함을 공증하는 파라미터로 사용할 수 있다. 반대로 두 간극이 임계치를 벗어난다면 당해 시스템 설계자는 선형 근사 수식을 전면 폐기하고 MCM 기반의 확률론적 수치 전산망을 전격적으로 통치 기반으로 삼아야 한다. 이는 고난도 파동 분산(Wave Dispersion), 방사선량 교정, 극소전력 반도체 노이즈 측정에서 필수불가결하게 적용된다.
