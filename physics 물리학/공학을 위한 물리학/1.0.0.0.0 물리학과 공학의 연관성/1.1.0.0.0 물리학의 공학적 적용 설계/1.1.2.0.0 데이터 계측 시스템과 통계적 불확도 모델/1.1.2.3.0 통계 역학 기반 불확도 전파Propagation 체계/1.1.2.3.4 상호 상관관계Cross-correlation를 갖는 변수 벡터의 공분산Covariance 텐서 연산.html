1.1.2.3.4 상호 상관관계(Cross-correlation)를 갖는 변수 벡터의 공분산(Covariance) 텐서 연산

공학 시스템의 설계와 계측에 있어 직면하는 파라미터 환경 분석의 가장 도전적인 과제 중 하나는 획득된 입력 변수 풀(Pool)이 통계 역학적으로 완벽히 독립적(Independent)으로 거동하지 않는다는 물리적 한계 상황이다. 즉 온도와 압력처럼 물리 화학 방정식 상태 변화에 의해 상호 교차로 얽혀 있거나, 동일한 기준 센서를 기점으로 차동 계측되어 발생하는 등, 변수 간 상호 상관관계(Cross-correlation)를 본연적으로 내포한 시스템 역학 메커니즘이 존재한다. 이때 개별 불확도 스칼라만을 단순 제곱합으로 합성하는 직교 모델은 중대한 수학적 왜곡을 낳게 된다. 본 절에서는 이러한 종속성이 불확합의 전파 분산 스펙트럼에서 어떻게 발산 또는 감쇄되는지, 공분산(Covariance) 텐서(Tensor) 연산 및 상관 계수(Correlation Coefficient) 행렬 역학을 통해 고도화하여 서술한다.

1. 확률 변수 공간의 교차 간섭: 공분산 모델링

두 확률 파라미터 X_i 와 X_j가 상호 수학적 종속 구조 얽힘(Entanglement)을 가질 때, 이 두 변수 군집이 동조하여 분산(Variance) 기복을 일으키는 척도인 확률 교차 역학 벡터항을 공분산(Covariance, u(x_i, x_j))이라 칭한다.
변수의 수집이 빈도주의 통계를 통해 n쌍의 개별 관측 벡터(x_{i,k}, x_{j,k})로 추출되었다면, 실험적 공분산의 수학 모델은 두 변수의 평정된 기댓값(평균) 이탈분을 상호 적분하여 도출된다.

u(x_i, x_j) = \frac{1}{n-1} \sum_{k=1}^n (x_{i,k} - \bar{x}_i)(x_{j,k} - \bar{x}_j)

이 공분산이 양(+)의 값을 띄면 두 파라미터는 동기식으로 비례하여 진폭 증대 경향을 보이고, 음(-)의 값을 무장하면 서로 역위상으로 상쇄 진동하는 역학 관계를 맺고 지배한다.

1.1 피어슨 상관 계수(Pearson Correlation Coefficient)의 매핑

두 매개 변수 사이의 교차 영향을 보다 보편적인 무차원 단위 무관 역학으로 계량하기 위해 공분산을 각 표준 편차(표준 불확도)의 스칼라 에너지 곱으로 정규화(Normalization)하여 상관 계수(Correlation Coefficient, r(x_i, x_j))라는 강력한 지표로 재정의하라.

r(x_i, x_j) = \frac{u(x_i, x_j)}{u(x_i) \cdot u(x_j)}

이 역학 계수 성분은 항시 -1 \le r(x_i, x_j) \le +1 의 철저한 속박 범위를 지닌다. 식을 환원 전개하면, 공차 연산을 위한 공분산 벡터항을 편의상 다음과 같이 각 표준 불확도의 승산으로 교체 환원할 수 있다.

u(x_i, x_j) = r(x_i, x_j) \cdot u(x_i) \cdot u(x_j)

2. 공분산 텐서를 포괄한 전미분 불확도 전파 방정식

앞서 규명한 미세 테일러 급수 선형 근사로 도출한 국소 기울기(민감도 계수) 환경에 공분산 텐서들을 접합하게 되면, 그 미분 체계가 최종 결과물 공간계 분산(Variance) u_c^2(y) 에 전파되는 양태는 뚜렷이 아래와 같은 거시적 이차 동차 다항 행렬 체계로 귀결된다.

u_c^2(y) = \sum_{i=1}^N \left( \frac{\partial f}{\partial x_i} \right)^2 u^2(x_i) + 2 \sum_{i=1}^{N-1} \sum_{j=i+1}^N \left( \frac{\partial f}{\partial x_i} \right) \left( \frac{\partial f}{\partial j} \right) u(x_i, x_j)

이 공식의 우변 첫 번째 항 군집은 각 변수들이 자신의 편차 파장을 홀로 직교 공간에 파종하는 독립 자승항이며, 두 번째 항 집군에 자리 잡은 이중 적분 구조가 다름 아닌 상관성을 기반으로 한 상호 간섭 공분산 텐서항이다.

1.1 행렬 도메인에서의 텐서적 고차 전개

이 역학 공식을 제어 공학의 행렬 역학(Matrix Mechanics) 스칼라 투영 모델로 차원 격상하면, 감도 계수 벡터 C = [\partial f/\partial x_1, \dots, \partial f/\partial x_N]^T 와 공분산 행렬(Covariance matrix) \Sigma 사이의 앞뒤 쌍차 행렬 연립으로 우아하게 구축된다.

u_c^2(y) = C^T \Sigma C

여기서 대각 항등 위치 선상에 배치된 성분들은 단일 분산 u^2(x_i) 들이고, 비대각원 공간에 상호 작용한 요소들이 바로 교차 공분산 u(x_i, x_j) 들이다.
