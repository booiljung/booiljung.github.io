1.1.2.3.3 다변수 시스템의 테일러 급수 1차 선형 근사 및 민감도 계수(Sensitivity coefficient) 전파 방정식

공학계를 관통하는 대부분의 물리적 계측 시스템은 단일한 입력 변수만으로 최종 측정량(Measurand)을 결정짓는 경우가 극히 드물다. 일반적인 역학 모델에서 출력 변수 Y는 다수의 상호 독립적 혹은 종속적 입력 변수들 X_1, X_2, \dots, X_N 에 의한 비선형 다변수 함수 Y = f(X_1, X_2, \dots, X_N) 로 규정된다. 이 섹션에서는 개별 입력 변수가 내포한 불확도(Uncertainty)가 비선형 통로를 거쳐 최종 출력의 불확도로 어떻게 전파(Propagation)되는지 규명하기 위해, 해석학적 도구인 다변수 테일러 급수(Taylor Series) 1차 선형 근사 기법과 민감도 계수(Sensitivity Coefficient)의 역학적 원리를 엄밀히 분석한다.

1. 다변수 함수의 테일러 급수 (Taylor Series) 전개 역학

물리량 예측 방정식 Y = f(X_1, X_2, \dots, X_N) 이 주어졌을 때, 각 입력 변수의 최적 추정값(기댓값)을 x_1, x_2, \dots, x_N 이라 하고 이들이 기댓값으로부터 미세하게 변동하여 오차를 야기하는 상태를 x_i + \delta x_i 라 정의하라. 이 미소 변위 \delta x_i 가 출력 \delta y 에 미치는 영향을 해석하기 위해 해당 함수를 최적 추정값 주변에서 전개하면 다변수 테일러 급수를 얻을 수 있다.

y + \delta y = f(x_1 + \delta x_1, x_2 + \delta x_2, \dots, x_N + \delta x_N)

이를 수식적으로 급수 전개(Expansion)하면 다음과 같은 무한 다항식 매트릭스로 거동한다.

y + \delta y = f(x_1, \dots, x_N) + \sum_{i=1}^N \frac{\partial f}{\partial x_i} \delta x_i + \frac{1}{2!} \sum_{i=1}^N \sum_{j=1}^N \frac{\partial^2 f}{\partial x_i \partial x_j} \delta x_i \delta x_j + \dots

여기서 1차 항들은 변수의 기울기를, 2차 이상의 고차항(Higher-Order Terms)들은 공간의 곡률 연산과 교차 다항식을 대변한다.

1.1 선형화를 위한 1차 근사(First-Order Linear Approximation)의 물리학적 타당성

대부분의 정밀 계측 모델에서 각 입력 변수의 불확도 편차 \delta x_i는 그 기준값 x_i에 비해 극도로 미미한 미소량(Infinitesimal) 스케일 비율을 지닌다. 따라서 \delta x_i \delta x_j 와 같은 2차 이상의 역학적 다항 항들은 수학적으로 0에 한없이 수렴하는 무시 가능한 고위 오차가 된다. 고로 시스템의 전미분 공간을 구성하기 위해 1차 선형 근사(1st-Order Linear Approximation)만을 취하는 것이 공학적으로 정당화된다.

\delta y \approx \sum_{i=1}^N \frac{\partial f}{\partial x_i} \delta x_i

이 방정식은 비선형 거동을 하는 다차원 시스템을, 기댓값 공간계라는 매우 좁은 국소 영역(Local region) 내에서 다차원 평면(Hyperplane) 공간으로 선형화하여 투영하는 수학적 변환 메커니즘이다.

2. 민감도 계수(Sensitivity Coefficient)의 도출과 물리적 의미

위의 1차 선형 근사 방정식에서 도출된 다변수 함수의 각 1차 편미분(Partial Derivative) 항을 측정학에서는 민감도 계수(Sensitivity Coefficient, c_i) 라고 엄격하게 규정한다.

c_i = \frac{\partial f}{\partial x_i}

이 민감도 계수 텐서 성분은 특정 입력 파라미터 x_i의 1 단위 변동 에너지가 최종 결괏값 도출 공간인 y의 스케일에 어느 정도의 기울기로 영향을 미치는가, 즉 불확도의 증폭비(Amplification Ratio) 혹은 감쇠비(Attenuation Ratio)를 지시하는 절대적 공간 지표이다.
각 계열의 변수가 이질적인 물리량(예: 길이, 전압, 온도 등) 단위를 지니고 있을지라도, 이 계수 c_i가 개입함으로써 모든 파라미터는 y 와 동일한 출력 단위의 벡터 도메인으로 규격화된다.

2. 선형화된 근사에 기반한 표준 불확도 전파 방정식

앞서 유도한 선형 편차 변동 방정식의 양변에 통계 역학적 기댓값(Expectation) 연산자 혹은 분산(Variance) 연산자를 취하게 되면 개별 오차 분포 역학들이 하나로 융합된다. 모든 입력 변수들이 확률론적으로 완벽하게 상호 독립적(Mutually Independent)이라 조건 지어질 때(즉, 공분산이 소거된 직교 투영 공간), 전체 시스템 y의 합성 표준 불확도 u_c(y)는 계수 c_i로 전치 증폭된 개별 불확도들의 제곱합의 제곱근(Root Sum Square, RSS)으로 정립된다.

u_c^2(y) = \sum_{i=1}^N \left( \frac{\partial f}{\partial x_i} \right)^2 u^2(x_i) = \sum_{i=1}^N (c_i \cdot u(x_i))^2

본 불확도 전파 모델의 수학적 정립은 수천 개의 제어 변수가 요동치는 최신 제어 시스템 컴퓨터 시뮬레이션 환경에서, 각 파라미터를 역으로 스크리닝(Screening) 하여 물리적 체계상 치명적인 약점 노드가 어디인지 검증해내는 가장 근원적이고 필수적인 물리학 모델링 지표로 군림한다.
