## 0.1 1차 계측 시스템 내 램프(Ramp)-스텝(Step) 강제 입력의 물리적 과도 응답 지연 함수

데이터 계측 센서는 이상적인 거울이 아니라 내부에 고유한 관성과 저항 역학을 품고 있는 물리 계통이다. 따라서 외부의 변화 신호가 강제 입력(Forcing Input)될 때, 즉각적으로 반응하지 못하고 특유의 지연 궤적을 그리며 쫓아가는 과도 응답(Transient Response) 미분 함수를 전개하게 된다. 열전대 온도계나 RC 전기 회로망 필터로 대표되는 1차 계측 시스템(First-order Measurement System)에서 가장 대중적인 두 가지 스탠다드 입력 타격—스텝(Step)과 램프(Ramp)—에 대한 수리 역학적 응답 방식을 분석한다.

1. 스텝 지배 입력 (Step Input)과 지수형 도달 궤적
시스템이 평형을 유지하던 시간 t=0 축을 기점으로 입력 물리량이 순식간에 A 라는 스칼라 상수 크기만큼 수직 점프 폭발하는 타격을 스텝 입력 함수라 정의한다. (x_i(t) = A u(t))
이 때 계측 시스템 방출 텐서가 스크린에 토해내는 출력 데이터 방정식 x_o(t) 는 1차 선형 상미분 방정식을 통과하여 다음과 같은 지수 함수적 도달 궤적을 렌더링한다.
x_o(t) = K A \left[ 1 - \exp\left(-\frac{t}{\tau}\right) \right]

  • 여기서 K 는 정적 감도(Static Gain)이고, \tau 는 시정수(Time Constant)이다.
  • 출력 그래프는 처음에는 수직에 가깝게 폭발적으로 상승하다가 점진적으로 기세가 꺾이며 KA 라는 최종 이상 도달치에 수렴해 들어간다. 시간이 t=\tau 초만큼 흘렀을 때, 출력은 최종 타겟의 63.2\dots\%, t=3\tau 에는 약 95\%, t=5\tau 에 도달하면 99.3\% 를 통과하며 기계적 안정화(Settling) 국면에 접어드는 통계적 특성을 증명한다.

2. 램프 지배 입력 (Ramp Input)과 지속적 추종 오차
미사일이 발사되어 고도를 지속적으로 서서히 높이거나 기체의 압력이 일정 기울기로 팽창할 때, 시간에 비례하여 선형적으로 기울기를 파생시키며 증대하는 강제 신호를 램프 입력 함수로 명명한다. (x_i(t) = R \cdot t, R 은 변화율 상수)
이 램프 환경망에서 계측 센서가 뱉어내는 과도 지배 출력 응답은 다음과 같은 수리 전개식으로 렌더링된다.
x_o(t) = K R \left[ t - \tau + \tau \exp\left(-\frac{t}{\tau}\right) \right]

  • 충분한 시간 단면(t \gg \tau)이 지나 지수항 \exp(-t/\tau) 이 0 으로 소멸 증발하고 나면, 출력 곡선은 입력 곡선과 완벽히 동일한 기울기(평행)를 쟁취하며 쫓아가기 시작한다.
  • 추종 오차 (Following Error) 한계: 그러나 두 곡선이 평행을 이룬다 할지라도, 실제 센서의 눈금 스케일과 물리적 참값 사이에는 영원히 좁혀지지 않는 수평 시간 차이 \tau(Time Lag)와 수직 수치 오차 K R \tau (동적 오프셋)가 영구적으로 평행 궤도를 그리며 잔류하게 된다.

공학적 결론으로서, 1차 계측 시스템은 서서히 변화하는 램프 유동장에서는 시정수 팩터에 비례하는 맹목적 추종 오차(Lag error)를 유발하므로, 제어 공학의 피드백 연산 로직(PID 제어)을 통해 센서 역학의 이 거시적 느림보 습성을 동반 삭제해야만 유도 무기 체계 등의 즉각적 방위 태세가 성립될 수 있다.
