## 0.1 화이트 노이즈(White noise) 백그라운드 모델과 파워 스펙트럼 밀도(PSD) 평가

계측 시스템이 아무리 외부와 물리적으로 격리되어 평형 상태를 유지한다 하더라도, 전자 회로 소자 내부를 구성하는 전자의 열적 무작위 요동(Thermal Agitation)이나 주변 환경의 난수 파동으로 인해, 베이스라인 신호 스크린에는 끊임없이 요동치는 미세 진동 파형이 렌더링된다. 공학에서는 어떠한 대역 필터로도 걸러낼 수 없는 이 근본적인 통계적 무작위 신호의 바다를 화이트 노이즈(White Noise) 백그라운드 모델로 규정하여 해석한다.

1. 화이트 노이즈(White Noise)의 물리 통계적 역학
화이트 노이즈는 모든 주파수 대역에 걸쳐 동일한 파워 에너지를 확률적으로 고르게 분배하고 있는 백색광의 스펙트럼 특성과 동치 된다는 의미에서 명명되었다.

  • 이 신호 모델의 역학적 특징은 시간 영역(Time Domain)에서의 데이터 간 상관성(Correlation)이 완전히 붕괴되어 있다는 점이다. 즉, 현재 타임 축(t)에서 노이즈가 폭증했더라도, 아주 미세한 순간 폭(\Delta t) 직후에 발생할 다음 노이즈의 진폭이나 방향을 수학적으로 추론하는 것은 불가능하다(마르코프 연쇄의 난수 극한). 이는 곧 순수하게 독립된 다발적 에너지가 무질서하게 시스템을 타격하고 있음을 증명한다.

2. 주파수 스펙트럼의 에너지 해골, 파워 스펙트럼 밀도 (PSD, Power Spectral Density)
시간 축에서 혼돈 그 자체인 노이즈 파형을 주파수 대역 공간(Frequency Domain)으로 투영하여, ’어느 진동수 영역에 파괴적인 에너지가 가장 밀집되어 있는가’를 통계적으로 도출해 내는 수학적 평가지표가 바로 파워 스펙트럼 밀도(PSD)이다.

  • 시간 함수적 난파 신호 x(t) 를 푸리에 변환(Fourier Transform)하여 획득한 주파수 복소수 함수 X(f) 에 대하여, 그 진폭의 제곱 구조를 취하여 단위 주파수당 에너지의 밀집도를 역산해 낸다(단위 예: V^2/\text{Hz} 또는 g^2/\text{Hz}).
    S_{xx}(f) = \lim_{T \to \infty} \frac{1}{T} |X(f)|^2

3. PSD 기반 공학적 신호 대 잡음비 무결성 평가
순정 화이트 노이즈의 PSD 선도는 모든 주파수 대역에서 평탄한 상수(S_{xx}(f) = \text{Constant})를 그린다. 만약 로켓 엔진의 연소실 압력을 계측한 데이터의 PSD 곡선을 도출했을 때, 평온한 백그라운드 상수 라인을 뚫고 특정 주파수(예: 300Hz)에서 뾰족하게 솟아오른 델타 피크(Peak)가 스캔되었다면, 이는 열전자의 무작위 요동이 아니라 엔진 챔버 구조물 자체가 300Hz로 미친 듯이 공명 진동(Resonance)하며 찢어지고 있음을 알리는 기계 물리적 적색경보다.

이처럼 계측 회로망에서 노이즈의 질량을 PSD로 계량화하는 작업은, 무가치해 보이는 잡음 에너지 속에서 기계 시스템의 파멸적 진동수를 색출해 내어 참된 ’신호(Signal)’를 필터링 구출해 내는 핵심 전산 역학의 시발점이다.
