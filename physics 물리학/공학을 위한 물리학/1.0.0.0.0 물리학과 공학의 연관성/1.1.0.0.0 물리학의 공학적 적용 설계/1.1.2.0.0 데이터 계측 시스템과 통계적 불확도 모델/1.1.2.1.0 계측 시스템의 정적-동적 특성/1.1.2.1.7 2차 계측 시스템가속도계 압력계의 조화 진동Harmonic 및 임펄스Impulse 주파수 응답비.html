## 0.1 2차 계측 시스템(가속도계, 압력계)의 조화 진동(Harmonic) 및 임펄스(Impulse) 주파수 응답비

데이터 계측 공학에서, 내부에 명확한 질량(Mass, m), 점성 댐핑(Damping, c), 그리고 탄성 복원력(Stiffness, k)의 세 가지 관성-에너지 소산 메커니즘을 모두 지니고 있는 계기를 2차 계측 시스템(Second-order Measurement System)이라 정의한다. 압력계의 벨로우즈(Bellows)나 우주 발사체에 장착되는 기계적 가속도계(Accelerometer)가 물리적으로 이에 귀속되며, 이들은 외부에서 조화 진동이나 임펄스 타격이 가해졌을 때 1차 시스템(온도계)과는 차원이 다른 복합적인 공진(Resonance) 및 감쇠 궤적을 렌더링한다.

1. 조화 진동 (Harmonic Oscillation) 주파수 응답과 진폭비 (Amplitude Ratio)
엔진의 회전축 등에서 지속적으로 발생하는 사인파(Sine wave) 형태의 조화 진동 신호 x_i(t) = A \sin(\omega t) 가 센서에 입력될 때, 2차 시스템은 강제 진동의 지배를 받는다.

  • 시스템이 토해내는 출력 진폭의 배율은, 입력 주파수 \omega 와 센서 고유의 자연 주파수(Natural Frequency) \omega_n = \sqrt{k/m} 의 비율(\omega/\omega_n), 그리고 댐핑 비(Damping Ratio) \zeta = \frac{c}{2\sqrt{km}} 의 상호 작용 역학으로 산출된다.
  • 주파수 응답 역학: 입력 주파수 \omega 가 센서의 고유 고유 진동수 \omega_n 에 근접할수록(즉, \omega/\omega_n \approx 1), 진폭비는 파멸적인 공진(Resonance)을 일으키며 수십 배로 폭주한다. 따라서 댐핑 비 \zeta 가 빈약한 가속도계를 사용할 경우, 스크린에는 실제 진동보다 아찔하게 왜곡 증폭된 거짓 데이터가 표출된다. 공학자들은 이 진폭 왜곡을 억제하고 주파수 응답 라인을 전 대역폭(Bandwidth)에 걸쳐 평탄화(Flat)하기 위해 실리콘 오일 등을 주입하여 댐핑 비를 최적화 수치인 \zeta = 0.6 \sim 0.7 부근에 튜닝하는 설계 규범을 적용한다.

2. 임펄스 지배 입력 (Impulse Input)과 과도 진동 링잉 (Ringing)
망치 장비로 외벽을 급격하게 타격하거나 폭발 파편이 충돌하는 등, 시간 폭이 극한으로 제로(\Delta t \to 0)에 수렴하면서 거대한 충격량 에너지를 주입하는 디랙 델타 함수(Dirac Delta Function) 형태의 타격을 임펄스 입력이라 칭한다.

  • 이 극한의 에너지가 2차 계측 시스템을 관통할 때, 센서 픽업 소자는 즉시 정지 상태로 복귀하지 못하고 에너지를 방전하며 출렁이는 과도 감쇠 진동 모드(Transient Underdamped Oscillation)에 빠져든다.
  • 이를 ’링잉(Ringing) 현상’이라 부르며, 댐핑 팩터가 부족한(\zeta < 1) 트랜스듀서는 타격 이후에도 수 밀리초(ms) 이상 노이즈 궤적을 그리며 춤을 추게 된다. 정밀 임펄스 캡처를 위해서는 \zeta 를 고의적으로 1.0(임계 감쇠, Critical Damping) 이상으로 비틀어 진동의 잔상을 강제 소멸시켜야 한다.
