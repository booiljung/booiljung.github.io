## 0.1 계통 오차(Systematic error) 보정(Calibration) 스케일 소급성 및 우연 오차 가우시안 정량화

아무리 수억 원의 자본을 투입한 초정밀 계측 네트워크 시스템이라 할지라도, 계측기가 토해내는 출력 데이터 수치의 거시적 통계망을 해부해 보면, 데이터의 타겟 적중률을 방해하는 두 가지 배타적인 물리 역학적 오차군—계통 오차(Systematic Error)와 우연 오차(Random Error)—이 철저하게 혼입되어 있다. 본 절에서는 이 두 오차의 발생 체계를 역학적으로 붕괴시키고, 물리량 참값을 도출해 내는 측정 표준망의 통계 규범을 구명한다.

1. 계통 오차 (Systematic Error)와 스케일 소급성 (Traceability)
계통 오차는 계측 데이터가 참값으로부터 일정한 편향성(Bias)을 가지고 특정 단일 방향축(전체적으로 높게 측정되거나, 기형적으로 낮게 측정됨)으로 쏠려버리는 결정론적(Deterministic) 왜곡 오차를 뜻한다.

  • 물리적 원인: 센서 다이어프램의 영구적 노후 변형, 전원부 베이스 전압의 오프셋 붕괴, 혹은 애초에 계측기의 눈금이 잘못 각인된 설계 스펙 등, 기계 구조와 운영 환경 자체의 ’구조적 결함’에서 기인한다.
  • 캘리브레이션과 소급성(Traceability): 계통 오차는 반복 측정을 천만 번 수행하며 통계 평균을 낸다 한들 절대 그 오차망이 소멸되지 않고 편향 상태를 유지한다. 이를 타파하는 유일한 역학적 해법은 국제도량형국(BIPM)에서 공인한 국가 측정 표준 참값(National Standard)과 내 장비를 1:1로 비교 대조하는 보정(Calibration) 이다. 내 센서를 상위 표준 연구 장비와 맞대어 오차 상수 역산을 조율하고 그 혈통적 근거를 증명해 내는 절차망을 공학 권율로 ’측정의 소급성(Traceability)’이라 칭한다.

2. 우연 오차 (Random Error)와 가우시안 정규 분포 정량화
계통 오차를 수학적 캘리브레이션 상수로 완벽히 차감 제거하고 나면, 데이터는 비로소 참값 타겟 주변을 맴돌게 되지만, 여전히 열잡음(Thermal noise), 공통 모드 간섭, 사용자(Human)의 무의식적 타격 등의 무작위 난수 변수에 의해 계측값들이 미시적으로 흩뿌려지는 난류를 겪게 된다. 이를 우연 오차라 명시한다.

  • 정량적 정규 분포 분해: 우연 오차는 우주 물리계의 엔트로피 섭리에 따라 타겟 참값을 정점으로 삼고 양방향으로 완벽한 대칭 종 모양을 렌더링하는 가우스 정규 분포(Gaussian Normal Distribution) 를 철저히 복종 추종한다.
    f(x) = \frac{1}{\sigma \sqrt{2\pi}} \exp \left( -\frac{(x - \mu)^2}{2\sigma^2} \right)
  • 위 확률 밀도 역학 함수에서, 표본의 통계적 평균(Mean, \mu)은 시스템이 궁극적으로 도달하고자 하는 무결한 물리적 참값을 지시하며, 표준 편차(Standard Deviation, \sigma)는 계측기 센싱 성능의 미시적 정밀도(Precision) 즉 오차 파편들의 흩어짐 반경 스케일을 정량화하는 절대적 성능 보증 지표가 된다. 결과적으로 68.27\% 의 계측 데이터 팩터가 \mu \pm 1\sigma 대역 내벽에 갇히게 됨을 수학적으로 증명할 수 있다.

이렇듯 계측 공학도는 계통 오차를 하드웨어 역학으로 배제(Calibration)시킨 후, 통제 불가능한 우연 오차의 확률망을 소프트웨어 중심의 극한 반복 데이터 평균화(Averaging)로 짓눌러 억제함으로써, 불확도가 철저히 정량화된 사이버물리시스템에 순결한 신호를 공급할 수 있다.
