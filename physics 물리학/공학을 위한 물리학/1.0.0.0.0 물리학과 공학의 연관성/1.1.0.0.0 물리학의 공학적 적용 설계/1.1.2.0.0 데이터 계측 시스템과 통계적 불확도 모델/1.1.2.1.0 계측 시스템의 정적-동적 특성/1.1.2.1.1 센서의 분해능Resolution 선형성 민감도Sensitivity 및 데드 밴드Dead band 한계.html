## 0.1 센서의 분해능(Resolution), 선형성, 민감도(Sensitivity) 및 데드 밴드(Dead band) 한계

계측 장비를 공학적 시스템에 적용하기 위해서는 대상 센서가 외부 물리량의 미세한 변화를 얼마나 정밀하고 규칙적으로 감지해 낼 수 있는지 규명하는 스펙 척도가 필수적이다. 본 절에서는 센서의 정적 품질을 결정짓는 핵심 성능 파라미터인 분해능, 선형성, 민감도, 그리고 응답 한계인 데드 밴드의 물리적 메커니즘을 정의한다.

1. 분해능 (Resolution)의 절대 스케일
분해능은 계측 시스템이 감지하고 유의미한 수치로 표출할 수 있는 입력 물리량의 가장 작은 변화폭(Minimum detectable change) 을 의미한다.

  • 기계 부품 가공의 정밀도를 결정하는 캘리퍼스의 최소 눈금이나, 아날로그 신호를 디지털로 전환하는 A/D 컨버터(Analog-to-Digital Converter)의 비트(Bit) 수가 이에 해당한다. 예를 들어 10비트 A/D 변환기는 5V의 전체 전압 측정 대역(Full-scale)을 1024개의 계단으로 쪼개므로, 5\text{V} / 1024 \approx 4.88\text{mV} 보다 미세한 전압 변화는 센서가 아예 인지하지 못하고 동일한 값으로 뭉뚱그려 처리하게 된다. 즉, 분해능 스펙트럼보다 미시적인 변화는 관측 불가능한 정보의 암흑 영역에 속하게 된다.

2. 선형성 (Linearity)과 오차 역산의 한계
이상적인 센서는 입력된 물리량의 텐서 크기와 출력되는 전기적 신호의 관계가 완벽한 1차 방정식의 직선 그래프를 묘사해야 한다.

  • 공학에서 선형성이란 실제 센서가 토해내는 캘리브레이션 궤적 곡선이 이상적인 평형 1차 직선(Ideal Straight Line)에서 얼마나 벗어나 일탈해 있는지를 퍼센트 편차로 계량화한 제원이다. 입력 변수 대비 출력이 비선형적(Non-linear)으로 요동칠 경우, 산출된 전압값만으로 원래의 물리적 압력이나 온도를 수학적으로 역산(Reverse Calculation)하기 위한 보정 알고리즘의 컴퓨팅 부하가 극심해지는 구조적 난관이 발생한다.

3. 민감도 (Sensitivity) 혹은 정적 게인 (Static Gain)
민감도는 계측 시스템의 입력 타겟 신호가 한 단위 변동할 때, 이에 대응하여 출력이 얼마나 역동적으로 솟아오르는지(증폭되는지)를 결정짓는 다이내믹 팩터이다. 수학적으로는 캘리브레이션 입력-출력 관계 선도의 국소적 기울기(Gradient) 도함수로 정의된다.
K = \frac{\Delta (\text{Output Signal})}{\Delta (\text{Input Physical Quantity})}

  • 민감도가 높은 열전대(Thermocouple) 센서는 고작 1도의 온도 상승만으로도 거대한 아날로그 전압 플럭스를 쏟아내어 고정밀 온도 감시가 가능하게 하지만, 동시에 백그라운드의 미세한 진동 노이즈까지 과대 증폭시켜 버리는 신호 왜곡 역효과를 초래하므로 트레이드오프(Trade-off) 튜닝 설계가 필수적이다.

4. 물리적 데드 밴드 (Dead Band / Dead Zone) 한계 영역
데드 밴드는 시스템에 외부 물리량이 명백히 입력되고 그 수치가 가변 중임에도 불구하고, 기계 역학적 마찰력이나 톱니바퀴의 백래시(Backlash) 한계 특성 탓에 결과 센서의 출력 계기판 수치가 단 1스케일도 미동조차 하지 않고 침묵하는 무응답 마비 구간을 뜻한다.

  • 기어 박스가 결합된 각 변위 센서가 역방향으로 회전 방향을 전환할 때, 톱니 간의 이격된 미세 빈 공간을 채우기 위해 헛도는 각도만큼 출력이 먹통이 되는 현상이 대표적인 데드 밴드 한계이다.
