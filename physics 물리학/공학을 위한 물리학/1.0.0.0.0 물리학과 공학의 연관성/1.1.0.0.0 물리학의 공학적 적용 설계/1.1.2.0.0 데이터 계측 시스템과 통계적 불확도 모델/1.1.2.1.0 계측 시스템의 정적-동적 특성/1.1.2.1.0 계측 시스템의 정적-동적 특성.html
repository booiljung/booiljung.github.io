## 0.1 계측 시스템의 정적/동적 특성

물리적 현상을 스캔하여 정량화된 전기적, 역학적 신호로 변환하는 센서(Sensor) 및 트랜스듀서(Transducer) 네트워크는 기계적 관성과 전기 회로적 한계를 동시에 가진다. 특정 물리량(입력 변수)이 주입될 때 계측 장비가 토해내는 산출 데이터(출력 변수)의 성능 응답 역학을 규명하기 위해서는, 시간에 지배받지 않는 정상 상태의 정적 특성(Static Characteristics) 과 시간에 따라 급변하는 과도 상태의 응답을 분석하는 동적 특성(Dynamic Characteristics) 으로 시스템을 분리하여 해부해야 한다.

0.1.1 계측 시스템의 정적 특성 (Static Characteristics)

지정된 입력 물리량이 시간적 변동 없이 안정적으로 지속해서 유지될 때, 계측 센서가 출력하는 응집된 기계 구조적 성능 지표이다.

  • 정확도(Accuracy)와 정밀도(Precision):
  • 정확도: 계측된 평균 지시값이 소급성이 검증된 ’절대 참값(True Value)’에 얼마나 근접하게 적중했는지를 평가하는 거시적 편향성(Bias) 오류 지표이다.
  • 정밀도: 동일한 물리량 상태를 극미한 시차를 두고 반복 계측했을 때, 그 결과값들이 서로 얼마나 조밀하게 응집되어 일치하는가를 나타내는 미시적 분산성(Variance) 지표이다. 정밀도가 높다고 정확도가 담보되는 것은 아니며 둘은 철저히 독립적이다.
  • 민감도 (Sensitivity): 센싱 타겟인 입력 물리량의 아주 희미한 단위 미세 변화량(\Delta x)에 대하여, 계측 장비가 반응하여 토해내는 출력 신호의 변화량폭(\Delta y) 비율 스케일이다. 민감도가 큰 소자일수록 아주 미약한 신호도 증폭 감지해 낼 수 있다 (K = \Delta y / \Delta x).
  • 분해능 (Resolution): 센서가 감지해 내어 유의미한 계측값 변화로 표출할 수 있는 입력 물리 환경의 ’가장 최소 단위의 미세 분절 양’을 의미한다.
  • 히스테리시스 (Hysteresis) 및 선형성 (Linearity): 물리량을 0에서 점진 증폭시키며 측정할 때의 궤적과, 역으로 최대치에서 점진 감쇠 감소시키며 측정할 때 계측 출력 궤적이 일치하지 않고 벌어지는 이력 왜곡 현상을 히스테리시스라 칭한다. 이상적인 센서는 입력과 출력 사이의 비례 매핑 선도가 완벽한 직선형(선형성)을 렌더링해야 오차 역산 보정이 용이해진다.

0.1.2 계측 시스템의 동적 특성 (Dynamic Characteristics)

고속 엔진 축의 폭발 압력 파동이나 난류 유동의 와동(Eddy)처럼 시간 도함수 단위로 격변하는 물리 신호(Transient signal)가 센서에 직격되었을 때, 계측 시스템이 입력의 다이내믹 포텐셜을 추종하여 반응해 내는 과도 응답(Transient Response) 성능 역학이다.

계측기 내부 구조물의 고유 질량, 기계적 감쇠 인자, 축전/인덕턴스 구조 팩터로 인해 센서는 외부 주입 신호를 즉시 그대로 출력하지 못하고 1차 지연계(First-order system, 예: 열전대 온도계) 혹은 2차 진동 지연계(Second-order system, 예: 피에조 압력 센서) 미분 방정식 양태로 끌려가며 지연 응답하게 된다.

  • 동적 반응 시간 지표:
  • 상승 시간(Rise Time): 스텝 형태의 파격적인 입력이 들어왔을 때, 센서 출력이 최종 목표 도달치의 일정 구간(일반적으로 10%에서 90% 구간)까지 치솟는 데 스며드는 시간 한계치이다.
  • 정착 시간(Settling Time): 센서의 기계적 진동이 억제되고 최종 정상 상태 타겟 출력의 허용 오차 밴드(예: \pm 2\%) 이내로 완벽히 수렴 진입하는 데 걸리는 보정 시간이다.
  • 대역폭(Bandwidth)과 주파수 응답성: 계측 장비가 진폭 감소 현상비 없이 무결하게 감지 추종할 수 있는 입력 신호의 강제 주파수 허용 범위 스펙트럼이다. 센서의 고유 떨림 주파수(Natural Frequency)를 초과하는 고속의 입력 주파수가 지속될 경우, 동적 위상 지연(Phase Shift) 현상과 주파수 응답 파괴가 발생한다.
