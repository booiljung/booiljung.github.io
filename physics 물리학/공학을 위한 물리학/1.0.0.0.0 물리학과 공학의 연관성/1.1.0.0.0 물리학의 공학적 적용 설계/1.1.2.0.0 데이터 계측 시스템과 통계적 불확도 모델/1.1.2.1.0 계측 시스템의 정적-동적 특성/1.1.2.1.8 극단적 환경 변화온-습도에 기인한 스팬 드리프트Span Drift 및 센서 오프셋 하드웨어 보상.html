## 0.1 극단적 환경 변화(온/습도)에 기인한 스팬 드리프트(Span Drift) 및 센서 오프셋 하드웨어 보상

물리적 계측 통제망이 적용되는 현장은 항온 항습이 유지되는 이상적인 통제실(Control Room)에 국한되지 않는다. 제트 엔진 내부의 초고온(Hyper-thermal) 폭염 구간이나, 우주 발사체 외벽의 극성 결빙(Freezing) 환경, 100% 상대 습도로 짓누르는 심해 잠수정 외판 등 악의적인 자연 외란(Disturbance) 기조에 무방비로 노출된다. 본 절에서는 이러한 거시적 환경 팩터가 센서의 정적 특성에 가하는 파괴적 역학인 ‘제로 오프셋’ 및 ‘스팬 드리프트’ 왜곡 메커니즘을 규명하고 트랜스듀서 하드웨어 단에서의 방어 렌더링을 탐구한다.

1. 거시적 환경 팽창에 의한 스팬 드리프트(Span Drift)
앞서 다룬 제로 편향(Zero-shift)이 데이터 출력 궤적의 ‘y절편’ 자체를 통째로 위아래로 밀어버리는 오차라면, 스팬 드리프트는 환경 외란이 타격하여 센서 감도 기울기(Sensitivity, Gain) 팩터 자체를 가변시켜 버리는 보다 치명적인 왜곡 메커니즘이다.

  • 압력계 내부의 금속 스프링과 다이어프램은 주변 대기의 온도가 수 백도로 요동치면, 열역학 제1법칙과 재료 강도의 섭리에 따라 영 씨 강성 모듈러스(Young’s Modulus)가 붕괴되며 기계적인 탄력을 상실한다.
  • 그 결과, 예전에는 100kPa 압력에 1mm만큼 수축하던 질량체가, 펄펄 끓는 온도 환경 하에서는 모듈러스 약화로 인해 1.5mm씩 흐물거리며 초과 변위하게 된다. 이는 센서의 정적 감도 K 자체가 증폭되어 버렸음을 의미하며, 센서의 특성 곡선 기울기가 점점 가파르게(또는 둔하게) 비틀리는 선형성 왜곡을 스팬 드리프트라 명시한다.

2. 센서 오프셋의 전자-기계 하드웨어 보상 (Hardware Compensation) 렌더링
이렇게 환경적 외란이 주입되어 초래된 오프셋 및 스팬 이동 오차는, 소프트웨어 통계 평균치 필터링(Random Noise 억제) 따위로는 절대 극복해 낼 수 없다. 따라서 공학자는 반드시 계측기 하드웨어 보드 내부에 열역학적 역보상(Inverse-compensation) 회로망을 이식하여 물리적 붕괴를 하드웨어로 요격해야 한다.

  • 온도 보상 더미 게이지(Dummy Gauge) 렌더링: 기계 변형률을 계기하는 스트레인 게이지(Strain Gauge) 회로망 구축 시, 타겟 변위를 직접 당겨서 감지하는 ‘액티브 게이지(Active Gauge)’ 옆에, 기계적 하중은 전혀 받지 않지만 오로지 주변 온도 변화에 의한 열 팽창 저항치 변동만을 흡수하는 ’더미 게이지(Dummy Gauge)’를 휘트스톤 브릿지(Wheatstone Bridge)의 인접 암(Arm)에 병렬로 장착한다. 이렇게 회로망을 직조하면, 환경 열기 탓에 동시에 치솟는 두 게이지의 잉여 전압 오프셋 파동이 브릿지 역학에 의해 정확히 마이너스 상쇄(Cancellation)되어 순수 변위 신호만 스크린으로 추출된다.
