## 0.1 데이터 계측 시스템과 통계적 불확도 모델

물리학의 순수 이론과 수학적 지배 방정식이 공학적 현실(Engineering Reality) 계층으로 넘어와 실제 하드웨어 시스템으로 구현될 때, 이론과 현실을 이어주는 유일한 검증 교량은 정확한 물리량의 계측(Measurement)이다. 아무리 정교하게 설계된 사이버물리시스템(CPS)의 시뮬레이터와 고급 다중 물리학 해석 모델이라 할지라도, 그 입력 변수나 경계 조건으로 투입되는 데이터가 부정확하다면 이는 치명적인 엔지니어링 실패로 귀결될 수밖에 없다. 따라서, 본 절에서는 외부 환경의 아날로그 물리적 현상을 측정 가능 지표로 디지털화하여 포획하는 ’데이터 계측 시스템(Data Acquisition System)’의 공학적 메커니즘을 규명한다.

더 나아가, 계측 행위 자체에는 필연적으로 물리적, 기계적, 환경적 요인으로 인한 미세한 노이즈와 오차가 수반된다. 자연계 단위의 절대적 ’정답’을 무오류로 도출하는 것은 열역학적, 양자역학적으로 불가능하기 때문이다. 이에 따라 실험 데이터를 단순한 숫자의 나열로 받아들이는 맹신을 배제하고, 해당 결과값이 지니는 신뢰 구조와 오차 한계를 철저한 통계 수학적 모델링으로 역산해 내는 통계적 불확도(Statistical Uncertainty) 평가망 체계를 구축하는 과정을 서술한다. 이를 통해 공학도는 복잡계 시스템의 출력 데이터를 학술적으로 검증하고 보수적으로 신뢰성을 평가할 수 있는 논리적 도구를 획득하게 된다.
