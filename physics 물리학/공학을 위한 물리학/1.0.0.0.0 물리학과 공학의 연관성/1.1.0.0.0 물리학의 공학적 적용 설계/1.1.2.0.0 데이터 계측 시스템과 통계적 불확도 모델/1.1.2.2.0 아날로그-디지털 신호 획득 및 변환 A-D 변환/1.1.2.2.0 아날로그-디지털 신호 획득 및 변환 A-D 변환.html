## 0.1 아날로그-디지털 신호 획득 및 변환 (A/D 변환)

연속적인 시간과 공간 속에서 무한한 해상도로 전개되는 뉴턴 역학적 물리량(압력, 온도, 속도 등)은 본질적으로 아날로그(Analog)의 띠를 띠고 있다. 그러나 현대 공학의 두뇌인 마이크로프로세서(Microprocessor)와 사이버물리시스템(CPS)은 오직 ’0’과 ’1’이라는 이산적(Discrete) 양자화 데이터만을 취급할 수 있는 논리적 족쇄를 차고 있다. 이 연속된 우주의 물리적 실재를 유한한 비트(Bit)의 디지털 우주로 번역해 내는 공학적 관문이 바로 아날로그-디지털 변환(A/D Conversion) 체계이다.

1. 신호 변환의 물리적 3단계 메커니즘
자연계의 연속적인 전기 신호 x(t) 가 디지털 메모리에 최종적인 이진 데이터 체인으로 안착하기 위해서는, 본질적으로 다음 세 가지의 치명적인 역학적 절단과 재조합의 지배를 받아야만 한다.

  1. 표본화 (Sampling): 연속적으로 흐르는 시간(Time)의 강물을 일정한 간격(\Delta t)으로 싹둑싹둑 썰어내어, 띄엄띄엄 존재하는 점(Point) 데이터들의 연속체로 추락시키는 작업이다. 이로 인해 신호는 시간축에 대해 이산적 배열 x[n] 으로 붕괴된다.
  2. 양자화 (Quantization): 연속적인 진폭(Amplitude) 수치의 스케일을 유한한 층계참(Step)으로 강제 반올림하여 근사(Approximation) 표기하는 절차다. 이 과정에서 아날로그 진솟값과 디지털 양자 계단 간의 물리적 오차인 양자화 노이즈(Quantization Noise)가 필연적으로 파생하여 원본 데이터의 순결성을 오염시킨다.
  3. 부호화 (Encoding): 양자화된 층의 숫자들을 마이크로컨트롤러가 흡수할 수 있도록 이진 코드(Binary Code, 0010, 0011 등)의 매트릭스로 맵핑하는 마지막 암호화 렌더링 단계이다.

2. 변환 대역폭과 정밀도의 공학적 딜레마
A/D 변환 시스템의 물리적 가치는 ’얼마나 자주 신호를 엿보는가(Sampling Rate)’와 ’얼마나 미세하게 눈금을 쪼개는가(Resolution, Bit)’에 의해 결정된다.

  • 1초에 수십억 번 데이터를 쓸어 담는 기가헤르츠(GHz)급 오실로스코프 지배망은 고주파 파동의 진동을 포착할 수 있으나 메모리 부하와 대역폭 타격을 기하급수적으로 유발한다. 반면 분해능을 24비트(Bit)까지 끌어올린 시스템은 나노볼트(nV) 수준의 심연까지 관측할 수 있으나 변환 속도가 소라게처럼 느려져 동적 맹점에 빠진다.
