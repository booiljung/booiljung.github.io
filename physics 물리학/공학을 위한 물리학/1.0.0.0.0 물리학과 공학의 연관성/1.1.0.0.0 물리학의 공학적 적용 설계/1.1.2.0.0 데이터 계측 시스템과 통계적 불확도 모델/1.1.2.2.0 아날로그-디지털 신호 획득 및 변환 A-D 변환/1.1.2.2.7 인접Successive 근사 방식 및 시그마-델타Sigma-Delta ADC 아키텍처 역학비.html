## 0.1 인접(Successive) 근사 방식 및 시그마-델타(Sigma-Delta) ADC 아키텍처 역학비

동일한 아날로그 파동을 소화해야 함에도, 레이더 신호처럼 속도가 최우선인 강박적 환경 수렴망과, 지진계나 저울처럼 수 나노볼트(nV)의 오차도 불허하는 극정밀 해상도 추종망은 요구하는 A/D 변환 엔진의 기계적 혈통이 완전히 다르다. 이 양 극단의 렌더링 설계를 공학적으로 타협 없이 양분하고 있는 두 가지 핵심 반도체 아키텍처가 바로 인접 근사(Successive Approximation Register, SAR) 방식과 시그마-델타(\Sigma-\Delta) 오버샘플링 방식이다.

1. 인접 근사 (SAR) 아키텍처: 이진 탐색(Binary Search)의 맹속 스케일링
자동차 전자 제어 유닛(ECU)과 산업용 다채널 데이터 로거(Data Logger) 시장의 지배자인 SAR ADC는, ‘업앤다운’ 숫자 맞추기 게임 논리로 알려진 ’이진 탐색 트리 알고리즘’을 아날로그 비교기 하드웨어에 이식한 구조망이다.

  • N 비트 해상도의 디지털 값을 도출해야 할 경우, SAR 로직 내부는 최상위 비트(MSB, 딱 절반을 가리키는 진폭)부터 시작하여 센서 입력값과 맹렬하게 이진 비교(Comparator)를 수행한다. 크면 1, 작으면 0 을 렌더링하고 바로 다음 하위 비트로 내려가 남은 범위를 다시 반토막 내며 역학적 저울질 과정을 반복한다.
  • 동역학적 이점: N 비트 변환을 완료하는 데 시계추(Clock) 틱이 정확히 N 번만 소비되므로, 변환 응답 시간이 절대적으로 일정(Deterministic)하고 지연 시간(Latency)이 거의 없는 제로화 응답성을 선사한다. 주로 10비트에서 18비트 해상도와 메가헤르츠(MHz)급 준고속 샘플링 밴드를 모두 균형감 있게 충족해야 하는 시스템의 뼈대로 채택된다.

2. 시그마-델타 (\Sigma-\Delta) 아키텍처: 나이퀴스트 섭리의 시간-공간 치환 한계 돌파
반면 초정밀 전자 저울, 의료용 뇌파(EEG) 스캐닝 장비 등 24비트(Bit) 이상의 극한 분해능을 갈구하는 영역에선 SAR의 물리적 저항망 한계가 박살 난다. 이를 시간-도메인의 과표본화(Oversampling) 논리로 전복시켜 돌파해 내는 극상의 기법이 시그마-델타 변환기이다.

  • \Sigma-\Delta ADC는 나이퀴스트 한계 주파수를 수백 배 찢어넘기는 파멸적인 초고속 샘플링 클럭을 기반으로, 현재 신호와 이전 신호의 ’미세한 변동 차이값(Delta)’만을 극저분해능(고작 1비트) 양자화기로 쉴 새 없이 타격 렌더링한다.
  • 이후 거대한 적분기(Sigma)와 디지털 필터 스웜을 거치면서 1비트의 데이터 난사가 무려 24비트급의 극정밀 데이터로 환골탈태 압착 데시메이션(Decimation) 계산된다.
  • 노이즈 셰이핑(Noise Shaping): 특히 양자화 에러 노이즈 덩어리를 억지로 고주파수 영역으로 쓸어 담아 내다 버리는 ‘잡음 성형 메커니즘’ 덕분에 SNR은 폭주적으로 극대화되나, 이 거대한 연산 렌더링을 처리하느라 최종 디지털 도출 속도가 극도로 굼떠지는(수십 샘플/초) 치명적 딜레마를 지닌다.

궁극적으로 계측 전자 공학의 구조는 속도와 저지연을 얻어 실시간 타격망(SAR)을 건설할 것인가, 응답성을 제물로 바쳐 극강의 노이즈 면역 해상도(\Sigma-\Delta)를 구득할 것인가의 비정한 트레이드오프 전장으로 직조되어 있다.
