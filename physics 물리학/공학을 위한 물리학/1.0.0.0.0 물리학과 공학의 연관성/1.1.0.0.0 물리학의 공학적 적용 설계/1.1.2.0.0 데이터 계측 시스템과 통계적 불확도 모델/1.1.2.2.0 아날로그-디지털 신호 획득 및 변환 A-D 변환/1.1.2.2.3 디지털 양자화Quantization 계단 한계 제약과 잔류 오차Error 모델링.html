## 0.1 디지털 양자화(Quantization) 계단 한계 제약과 잔류 오차(Error) 모델링

시간의 강산을 표본화(Sampling)로 베어낸 후 남겨진 잔해물들, 즉 각각의 이산 시간 노드(t_n)에서의 진폭 전압치들은 여전히 무한한 소수점 아래 궤적을 뿜어내는 아날로그적 속성을 띠고 있다. 마이크로프로세서가 이를 소화하기 위해선 무한소 크기의 전압 곡선을 계단 형태의 한정된 디지털 이산 층위로 구겨 넣거나 내리찍어 반올림하는 압착 역학이 필요한데, 이 무자비한 매핑 작업을 양자화(Quantization) 라 칭한다.

1. 양자화 분해능 (Bit Resolution) 한계와 스텝 층위(q)
A/D 변환기의 성능은 가용한 비트 수(n-bit) 하드웨어에 의해 양자화 층위의 갯수가 결정된다. 예를 들어, n=8 비트 시스템은 2^8 = 256 개, 12 비트는 4096 개의 이산 계단(Step)을 형성한다.

  • 0부터 5V까지의 전압 도메인(Full-Scale Range, FSR)을 계측하는 센서일 때, 양자화 계단의 한 칸 높이 q (혹은 Least Significant Bit, LSB 해상도)는 다음 수리식에 의해 강제 할당된다.
    q = \frac{V_{max} - V_{min}}{2^n}
  • 즉 8비트 구동 시, 계단 한 칸은 약 19.5\text{mV} 의 틈새 간격을 갖게 된다. 여기서 공학적 한계 제약이 폭발하는데, 실제 5.011V, 5.015V 등 미세하게 진동하는 저주파 아날로그 신호라도 그 차이가 19.5\text{mV} 를 돌파하지 못하면, 컨버터 렌더링 텐서는 이 모든 수치를 동등한 계단(예를 들어 전부 이진수 11110001 매핑)으로 뭉뚱그려 반올림해 버리는 정보 말살을 자행한다.

2. 잔류 양자화 오차 (Quantization Error)의 통계적 노이즈 모델링
이 강제적인 반올림(또는 절사) 과정에서, 원본 아날로그 수치 x(t) 와 디지털로 인코딩된 이진 계단 수치 x_q(t) 사이에는 영원히 극복할 수 없는 수학적 틈새 차이가 파생되며, 공학계는 이를 잔류 양자화 오차(Quantization Error, e_q)라 명명한다. (e_q = x_q(t) - x(t))
이 오차는 필연적으로 물리 계측망 속에서 원본 신호를 괴롭히는 거대한 백색 소음(White Noise)과 동일한 역학적 해악을 끼친다.

  • 통계적 분포 규범: 양자화 오차 e_q 는 통계 역학적으로 -\frac{q}{2} 에서 +\frac{q}{2} 까지의 마이크로 스케일 구간 내에서 완벽히 무작위로 요동치는 균등 분포(Uniform Distribution)의 확률 밀도 모델을 렌더링한다.
  • 분산 한계치: 이 에러 모델망의 수학적 분산(Variance, 노이즈 파워의 척도) \sigma_q^2 는 수식 \frac{q^2}{12} 로 전개된다.

이 섭리는 잔혹한 물리적 결론을 투영한다. 비트 분해능을 선형적으로 증대시켜 양자화 스텝 q 의 간격을 극한으로 오밀조밀하게 쪼개지 않는 이상(n \to \infty), 이 잔류 양자화 에러망이 빚어내는 통계 분포 노이즈망은 센싱 데이터를 영구적으로 마모시키며 계측 시스템의 신호 대 잡음비(Signal-to-Noise Ratio) 상한선을 치명적으로 틀어막아 버리게 된다.
