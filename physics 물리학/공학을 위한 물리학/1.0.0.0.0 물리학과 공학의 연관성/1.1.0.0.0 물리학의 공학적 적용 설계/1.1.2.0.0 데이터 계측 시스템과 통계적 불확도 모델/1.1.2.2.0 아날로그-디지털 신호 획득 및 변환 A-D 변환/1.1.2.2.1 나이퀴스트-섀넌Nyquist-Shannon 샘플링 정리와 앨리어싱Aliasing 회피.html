## 0.1 나이퀴스트-섀넌(Nyquist-Shannon) 샘플링 정리와 앨리어싱(Aliasing) 회피

연속된 아날로그 시간 함수를 단속적인 점들의 배열로 난도질하는 표본화(Sampling) 작업을 수행할 때, 가장 치명적인 화두는 “점들의 나열만으로 원래 파도의 곡선을 잃어버리지 않고 100% 무결하게 부활(Reconstruction)시킬 수 있는가” 이다. 이 파문학적 난제에 명확한 물리-수학적 한계 규범을 선포한 위대한 섭리가 바로 나이퀴스트-섀넌 샘플링 정리(Nyquist-Shannon Sampling Theorem)이다.

1. 나이퀴스트 한계 주파수 역학
정보 역학자 클로드 섀넌과 해리 나이퀴스트는 아날로그 파동의 훼손 없는 디지털 전환을 위해서는, 샘플링 카메라가 터지는 속도(샘플링 주파수, f_s)가 계측하고자 하는 대상 신호가 품고 있는 ’가장 격동적인 진동수(최고 주파수, f_{max})’보다 역학적으로 최소 2배 이상 신속해야만 한다고 수식화했다.
f_s \ge 2 f_{max}

  • 즉, 100Hz로 요동치는 엔진의 진동 파형의 극점과 골짜기를 놓치지 않고 포획하려면, 초당 최소 200번(200Hz) 이상의 셔터 스피드로 데이터를 썰어 담아야(Sampling) 원본 아날로그 파동의 위상 렌더링에 필요한 최소한의 마디(Node)를 확보할 수 있다는 뜻이다.
  • 이 최소 한계 임계점인 2 f_{max} 를 나이퀴스트 주파수(Nyquist Frequency) 라 칭한다.

2. 주파수 영토의 기만, 앨리어싱(Aliasing) 왜곡망
만약 공학자의 오판이나 하드웨어 한계 탓에 나이퀴스트 정리를 위반하고 대상 주파수보다 게으르게(f_s < 2 f_{max}) 센싱 데이터를 획득하면 어떤 역학적 참사가 발생하는가?
센서의 느린 셔터 속도가 원본 고주파 파동의 오르막과 내리막을 엉뚱한 리듬으로 건너뛰며 스캔해버린 탓에, 컴퓨터 스크린 상에는 원본에 존재하지도 않던 전혀 다른 거짓된 ‘저주파 파동(Low-frequency ghost)’ 곡선이 환영처럼 렌더링된다.

  • 고주파 파동이 저주파로 교묘하게 위장하여 시스템을 속이는 이 파괴적인 스펙트럼 겹침 현상을 앨리어싱(Aliasing) 이라 명시한다.
  • 영화 속에서 고속으로 맹렬하게 전진 회전하는 자동차 타이어 휠이 셔터 프레임과의 비동기화 탓에 오히려 역방향으로 천천히 역회전하는 것처럼 시각적으로 보이는 현상이, 바로 시각적 앨리어싱의 가장 노골적인 증명이다. 앨리어싱이 한 번 데이터에 깊게 혼입되어 버리면, 디지털 후처리 알고리즘 그 어떤 것으로도 진짜와 가짜 주파수를 수리적으로 분리해 낼 방법이 영원히 소멸된다.

따라서 계측 엔지니어는 앨리어싱이라는 비가역적 파멸을 방어하기 위해 아날로그 신호를 획득하는 가장 첫 번째 관문에서 나이퀴스트 기준을 초과하는 고속 ADC를 기용하고, 불필요한 고주파 노이즈를 사전에 도륙해 버리는 ‘앤티 앨리어싱(Anti-aliasing)’ 아날로그 필터를 센서 전단에 물리적으로 설치하여 잠재적 왜곡의 싹을 잘라내야 한다.
