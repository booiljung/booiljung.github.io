1.1.2.4.4 다항식 곡선 피팅(Curve fitting) 파서 및 런지(Runge) 현상에 따른 고차 진동 억제식

공학 데이터 해석에서 가장 고전적이고 직관적인 공간 보간(Interpolation) 접근법은 이산적인 데이터 점들을 포괄하는 하나의 거대한 기저 함수(Basis Function) 공간, 즉 다항식(Polynomial) 급수를 전개하는 것이다. 주어진 N+1개의 관측 점이 있을 때, 이를 오차 없이 완벽하게 통과하는 유일한 역학적 곡선은 반드시 N차 다항식으로 규명된다는 대수학적 정리(위어스트라스의 근사 정리 등)에 근거하여 곡선 피팅(Curve Fitting)이 시도된다. 그러나 무차별적인 고차 다항식의 전개는 필연적으로 기하학적 파동 폭주를 야기한다. 본 절에서는 다항식 보간의 매트릭스 파서(Parser) 메커니즘을 파헤치고, 다항식의 급수가 낳는 재앙인 런지(Runge) 현상의 발생 기전과 이를 물성적으로 통제하는 체비쇼프(Chebyshev) 억제식 역학을 규명한다.

1. 다항식 보간의 대수학적 파서(Parser) 메커니즘

N+1개의 좌표군 (x_0, y_0), (x_1, y_1), \dots, (x_N, y_N) 에 대하여, 이 모든 점을 정확히 궤적 상에 포함하는 곡선 P_N(x)를 아래와 같은 N차 거듭제곱 급수로 가정하라.

P_N(x) = a_0 + a_1 x + a_2 x^2 + \dots + a_N x^N

이때 미지의 역학 계수 벡터 \mathbf{A} = [a_0, a_1, \dots, a_N]^T 를 추출해 내기 위해, 각 독립 변수 x_k들의 거듭제곱으로 구성된 반데르몬드 행렬(Vandermonde Matrix) \mathbf{V} 를 구축하여 선형 시스템 \mathbf{V} \mathbf{A} = \mathbf{Y} 를 조립한다.

1.1 라그랑주(Lagrange) 및 뉴턴(Newton) 다항식 기저 형상

반데르몬드 행렬의 직접 역산은 조건수(Condition Number)가 급격히 불량해지는 약점이 있으므로, 공학 파서(Parser)들은 기저 함수(Basis function)의 형태를 변형하여 해를 즉각적으로 조합한다. 대표적으로 라그랑주 기저 다항식(Lagrange Basis Polynomial)은 각 노드 x_k 에서만 물리적으로 1이 되고 나머지 모든 노드에서는 0이 되는 거름망(Filter) 형태의 가중치 기저 L_k(x) 를 투영한다.

P_N(x) = \sum_{k=0}^N y_k L_k(x), \quad L_k(x) = \prod_{j=0, j \neq k}^N \frac{x - x_j}{x_k - x_j}

이로써 어떠한 수치적 행렬 역산 없이 데이터 점을 관통하는 거시 다항식이 그 즉시(On-the-fly) 해석적으로 결정된다.

2. 고차 다항식의 재앙: 런지 현상(Runge’s Phenomenon)

N개 점을 잇기 위해 N-1차 다항식을 구성하는 방식에는 치명적인 물리학적 결함이 존재한다. 점의 개수가 많아질수록 궤도를 더욱 유체적으로 정밀하게 묘사할 것이라는 직관은 완전히 빗나간다.

Carl David Tolmé Runge 가 규명한 이 현상은, 등간격(Equidistant)으로 촘촘하게 배열된 노드망 위에서 차수 N이 증가할수록, 보간 다항식의 양 끝단(경계 부근)에서 궤적이 통제 불능의 상태로 상하 진동(Oscillation)하며 무한대로 발산(Divergence)해 버리는 기막힌 수학적 파탄계이다.

이 고차 진동의 근원은 고주파수(High-frequency) 성분을 강제로 주입받은 극도로 높은 등급의 다항식이, 중앙의 점들을 모두 맞추기 위한 급격한 관성 변화를 경계 부근 공간에 강제 방출(Release) 하면서 발생하는 구조 응력(Structural Stress)의 해소 현상과 유사하다.

3. 체비쇼프(Chebyshev) 노드 배치를 통한 진동 공간 억제

런지 현상의 진동 재앙을 물리적으로 봉쇄하기 위해, 고차 다항식 피팅을 고수해야만 하는 특수 상황에서 공학자는 더 이상 계측 노드를 “등간격“이라는 단순 무식한 1차원 선형망으로 쪼개어 배치해서는 안 된다. 이 발산 역학을 압살하는 해법은 데이터의 관측점, 즉 노드(Node)의 위치 배열 자체를 체비쇼프 다항식(Chebyshev Polynomials)의 근(Roots)에 조응하는 비선형 밀도계 공간으로 재조정하는 것이다.

구간 [-1, 1]상에서 n개의 체비쇼프 노드 위치 x_k 는 상수각을 기반으로 한 코사인(Cosine) 사영으로 계산된다.

x_k = \cos \left( \frac{2k - 1}{2n} \pi \right), \quad k = 1, \dots, n

3.1 억제식의 공간 역학적 의의

이 체비쇼프 노드 배치망은 기하학적으로 원의 둘레를 등간격 분할한 뒤, 그것을 x 차원 직선으로 프로젝션(Projection)시킨 점군 분포이다. 이 역학적 조작으로 인해, 점들의 밀도는 중앙부에서는 희소(Sparse)해지고, 진동이 폭주하려던 양극단 경계 부근을 향할수록 극도로 조밀(Dense)하게 집중되는 공간 응집을 보인다.
경계부에 밀집된 노드망은 다항식이 상하로 튀어 오르려는 고차 도함수(Higher-order derivative) 팽창 에너지를 핀(Pin)처럼 강하게 속박하고, 무질서한 진동을 중앙 영역으로 분산시켜 다항식 보간의 전체 공간 궤도를 매끄럽고 평온한 융합 곡선으로 안정화하는 물리학적 매직(Magic)을 완성한다.
