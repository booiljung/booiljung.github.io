1.1.2.4.5 다차원 센서 기하 계측용 B-스플라인(B-Spline) 및 큐빅 곡면의 매끄러움(Smoothness) 미분

앞선 단원에서 규명되었듯, 단일한 전역 고차 다항식(Global Polynomial)으로 수많은 센서 데이터를 무리하게 꿰뚫으려는 시도는 런지(Runge) 현상이라는 역학적 진동 폭주로 좌초하게 된다. 자동차 외장 형상 설계(Aero-dynamic surface)나 항공기 날개의 공력 곡면, 다차원 로봇 팔의 제어 궤적 도출 등에서는 국부적인 데이터 변화가 전체 곡면에 원치 않는 파형(Ripple)을 전파하는 것을 결단코 용납할 수 없다. 이를 극복하기 위해 등장한 현대 기하 모델링(CAGD; Computer-Aided Geometric Design)의 궁극적 무기가 바로, 전체 구간을 유한한 국소 조각(Piecewise)들로 분할한 뒤, 그 접합점 부위에서 고계도(High-order) 미분 기저의 매끄러움(Smoothness) 특성을 강제하는 큐빅 스플라인(Cubic Spline)과 B-스플라인(Basis-Spline) 텐서 역학계이다.

1. 큐빅 스플라인(Cubic Spline)의 국소 곡선 접합 궤도

N+1개의 이산 센서 점 (x_0, y_0) 부터 (x_N, y_N)이 주어졌을 때, 큐빅 스플라인의 물리적 철학은 이 전체 구역을 하나의 곡선으로 강제하는 대신, 점과 점 사이의 개별적인 국소 소구간 [x_{i}, x_{i+1}] 마다 고온에서 휘어지는 탄성 강철 자(Spline)처럼 각각 독립된 3차 다항식 조각(Piece) S_i(x) 를 투입하는 것이다.

S_i(x) = a_i + b_i(x - x_i) + c_i(x - x_i)^2 + d_i(x - x_i)^3 \quad \text{for } i = 0, \dots, N-1

1.1 매끄러움(Smoothness)과 곡률 통제 미분 역학

수많은 파편 곡선 S_i(x)들이 단순히 꼭짓점 매듭(Knot, x_i)에서 서로 맞닿는 것(위치 연속성, C^0)만으로는 파단이나 뾰족한 교차 첨점(Cusp)을 막을 수 없다. 물리적으로 유려하게 이어지는 공간을 창출하기 위해 엔지니어는 매듭 지점에서 두 국소 곡선의 미분 역학계가 정확히 일치하도록 공간에 압력을 가한다.

  1. 접선(Tangent) 연속성 (C^1): 1차 도함수가 일치해야 한다.
    S'_{i-1}(x_i) = S'_i(x_i)
  2. 곡률(Curvature) 연속성 (C^2): 2차 도함수(가속도 역학계)까지 완전히 일치해야 한다.
    S''_{i-1}(x_i) = S''_i(x_i)

이 C^2 미분 연속성 구속 조건(Constraint)들을 행렬 공간에 정렬하면 거대한 삼중 대각 행렬(Tridiagonal Matrix) 방정식망이 축조된다. 이 삼중 대각 연립계는 토마스 알고리즘(Thomas Algorithm)에 의해 계산 부하 없이 그 즉시 국소 계수 텐서들을 최적화해 내며, 이로써 인간의 시야에 곡률의 단절이 전혀 감지되지 않는 궁극적인 1차원 매끄러움이 정교하게 달성된다.

2. B-스플라인(B-Spline)의 전역-국소(Global-Local) 분리 통제 역학

큐빅 스플라인조차 데이터 점들을 “반드시 직접 관통(Interpolation)“해야 한다는 강박 탓에, 특정 센서가 심각한 오차로 튀어버리면 그 주변 구간들의 형태가 국소적으로 거세게 뒤틀리는 한계를 내포한다. 점을 통과할 의무를 벗어던지고, 점들을 오직 형상을 잡아당기는 자석(Control Points, 제어점)으로만 활용하여 매끄러운 곡선을 빚어내는 통제 기법이 기저 스플라인, B-스플라인(Basis-Spline)이다.

곡선 위의 임의의 형상 점 P(t)는 n개의 제어점 \mathbf{P}_i와 차수 p의 블렌딩 함수(Blending Function) N_{i,p}(t) 들의 공간 가중치 텐서망으로 설계된다.

P(t) = \sum_{i=0}^n \mathbf{P}_i N_{i,p}(t)

1.1 칵스-드 보어(Cox-de Boor) 점화식 역학

이 B-스플라인의 블렌딩 함수 N_{i,p}(t) 는 매듭 벡터(Knot Vector) 스페이스 위에서 칵스-드 보어 재귀 공식에 의해 계층적으로 직조된다.
이 수식의 기하학적 파괴력은 “국소 통제(Local Control)“라는 독점적 역학에 있다. 특정 제어점 \mathbf{P}_k 하나를 공간상에서 강제로 이동시킨다 할지라도, 블렌딩 함수의 수학적 종관 범위(Support)가 제한되어 있기 때문에, 곡선 전체가 뒤틀리지 않고 그 점과 인접한 단지 p+1개의 매듭 구간 곡률 텐서만 유체적으로 변화하게 된다. 이는 항공기 날개 단면을 설계할 때 꼬리 날개 형상을 수정하더라도 주익 중심의 형상 방정식이 전혀 붕괴되지 않는 극강의 공학적 모델링 안정성을 담보한다.

2. 다차원 표면 장력: 텐서 적(Tensor Product) 기반 큐빅 곡면

이 모델계에서 3차원 좌표계상의 표면 S(u, v)는 두 매개 변수 스페이스(u, v)에 축조된 스플라인 기저 함수들을 수학적으로 교차 승산(Cross-multiplication)시켜 직교(Orthogonal) 그물망을 펼친다.
결론적으로 국소 분할(Piecewise)과 고차 미분 연속성(Smoothness Constraint), 편적분 텐서라는 이 연합된 B-스플라인 역학계만이, 불연속적이고 거칠게 계측된 무수한 센서 스캐닝 데이터 파편들을 유체 공기 역학이나 빛의 반사마저 매끄럽게 흐르는 우아한 공학적 솔리드(Solid) 곡면 지배 방정식으로 완벽하게 봉합해 내는 현대 스캐닝/형상 계측계의 원천 기술이 된다.
