1.1.2.4.8 공학 역문제(Inverse problem) 시스템 파라미터 계수 식별 프로세스 및 희소행렬 공간 차원 복구

고전 역학에서 지배 방정식(Governing Equation)과 재료 물성, 경계 조건(Boundary Condition) 등의 투입 변수들이 모두 온전히 주어졌을 때, 시스템의 미래 응답(Response)을 도출하는 궤도를 순방향 문제(Forward Problem)라 칭한다. 반면 실제 공학 현장—가령 비파괴 검사, 의료 영상(CT/MRI), 석유 탐사, 구조물 건전성 모니터링—에서는 시스템 내부를 직접 들여다보거나 해부할 수 없는 경우가 절대다수이다. 이때 외부에서 한정적으로 계측된 불완전한 결과 응답 데이터들만을 취합하여, 시스템 내부 깊숙이 은닉되어 있는 원인, 즉 미지의 재료 파라미터 계수나 기하학적 형상을 역추적하여 발굴해 내는 일련의 학문적 해부학 체계를 역문제(Inverse Problem)라 명명한다. 본 절에서는 이러한 불량 조건(Ill-posed)의 역역학 시스템 파라미터 식별 프로세스와, 불충분한 센서 데이터의 한계를 극복하는 압축 센싱(Compressed Sensing) 기반 희소행렬(Sparse Matrix) 차원 복구의 진면목을 조명한다.

1. 역문제의 물리학적 본질과 적격성 결여(Ill-posedness)

역문제의 철학은 본질적으로 열역학 제2법칙에 위배되려는 시도와 맥을 같이 한다. 수많은 원인(균열, 열원, 밀도 분포 등)들이 모여 동일하거나 흡사한 거시적 결과를 빚어낼 수 있기 때문이다. 자끄 아다마르(Jacques Hadamard)가 정립한 바에 따르면 역문제는 보통 다음의 물리적 적격성(Well-posedness) 요건 중 하나 이상을 필연적으로 상실한 불량 조건(Ill-posed) 상태에 노출된다.

  1. 존재성(Existence) 결여: 주어진 잡음 계측 데이터를 완벽하게 생성할 수 있는 물리적 계수가 아예 존재하지 않을 수 있다.
  2. 유일성(Uniqueness) 붕괴: 계측 값이 너무 적어서 무한히 많은 파라미터 조합이 똑같은 응답을 뿜어낸다.
  3. 안정성(Stability) 붕괴: 계측 데이터에 미세한 노이즈만 가해져도 역산된 내부 파라미터 행렬이 우주적 스케일로 발산(Blow-up)해 버린다.

이 역학적 한계를 돌파하기 위해서는, 순수 대수학적 차원의 행렬 역산(Matrix Inversion)을 시도할 것이 아니라, 물리적 사전 지식(Prior Knowledge)을 구속 조건으로 주입하여 해를 강제로 유도하는 정규화(Regularization) 역학관으로 반드시 진입해야 한다.

2. 시스템 파라미터(System Parameter) 계수 식별 프로세스

공학 시스템 내부의 열전도 계수 k(x) 나 탄성 계수 E(x) 의 공간적 분포 역학을 외부 센서 데이터(온도장, 변위)로부터 복원해 내는 과정은 복잡한 연속 최적화(Continuous Optimization) 파이프라인을 관통한다. 이는 목적 함수 J(k) 를 오차 공간 상에서 최소화하는 알고리즘으로 귀결된다.

J(k) = \frac{1}{2} \vert\vert u_{sim}(k) - u_{meas} \vert\vert^2 + \alpha R(k)

여기서 u_{sim}은 연산된 모사 응답, u_{meas}는 계측 응답이며, R(k)는 극단적인 진동을 막기 위한 티호노프 정규화(Tikhonov Regularization) 억제식, \alpha는 그 억제 장력의 튜닝 스칼라이다.

2.1 수반 변수법(Adjoint State Method) 기반 초고속 기울기 역산

거대 유한요소(FEM) 망에서 미지의 파라미터가 수십만 개에 달할 때, 유한 차분(Finite Difference)으로 파라미터의 민감도(기울기 행렬, Gradient)를 일일이 구하는 것은 물리적으로 영겁의 시간을 소요한다. 이를 타파하는 궁극의 시스템 식별 역학이 바로 수반 방정식(Adjoint Equation) 체계이다.

순방향 상태 변수(State Variable) u 를 해석한 후, 센서 계측 오차를 마치 가상의 하중(Virtual Load)처럼 역으로 인가하여 시스템을 시간 또는 공간 축에서 거꾸로 구동시키는 수반 변수 \lambda 공간장을 한 번 해석한다. 이 단 두 번의 해석(Forward + Adjoint) 결과 텐서들을 내적(Inner product) 융합하는 것만으로 수십만 개 파라미터에 대한 전역 민감도(Gradient \nabla J)를 그 즉시 쏟아내는 역추적 역학의 백미를 발휘한다.

3. 압축 센싱(Compressed Sensing)과 희소행렬(Sparse Matrix) 차원 복구

의료 검측이나 지진파 탐사 시, 계측 가능한 센서 채널 수(M)는 복원해야 할 파라미터 공간의 차원(N)보다 턱없이 부족하다 (M \ll N). 이는 선형 대수학적으로 M개의 방정식으로 N개의 미지수를 푸는 절대 불가능의 부족 결정(Under-determined) 시스템이다. 이 파멸적 차원 결핍을 우회하는 현대 통계 물리학의 기적이 압축 센싱(Compressed Sensing)이다.
