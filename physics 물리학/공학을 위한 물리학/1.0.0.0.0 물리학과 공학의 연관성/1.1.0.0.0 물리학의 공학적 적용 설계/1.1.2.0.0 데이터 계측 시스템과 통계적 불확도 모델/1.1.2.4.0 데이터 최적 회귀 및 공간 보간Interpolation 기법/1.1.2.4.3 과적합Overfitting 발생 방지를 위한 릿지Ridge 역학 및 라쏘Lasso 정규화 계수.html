1.1.2.4.3 과적합(Overfitting) 발생 방지를 위한 릿지(Ridge) 역학 및 라쏘(Lasso) 정규화 계수

공학 공간에서 계측 데이터의 내재적 경향성을 추출하기 위해 회귀 모델 구축을 감행할 때, 엔지니어는 종위의 딜레마에 부딪힌다. 다변수 시스템일수록, 혹은 고차 다항식(High-degree Polynomial) 필터를 도입할수록 회귀 방정식의 자유도는 극단적으로 해방되어 현재 확보해둔 학습 데이터(Training Data)를 완벽에 가깝게 통과해 낸다 (R^2 \approx 1). 허나 이는 모델이 데이터 군계에 은닉된 진실규명적 본질을 학습한 것이 아니라, 그 표본들에 일시적으로 함유된 백색 소음(White Noise)과 계측 잔재 오류까지 모두 무비판적으로 암기해버린 재앙, 즉 과적합(Overfitting) 진동 체계에 빠져들었음을 시사한다. 새로운 미지의 미래 데이터 점군(Test Data)이 입력되었을 때 이러한 과적합 파이프라인은 공간적으로 무섭게 요동치며 치명적인 예측 발산을 초래한다. 본 절에서는 이러한 모델의 수학적 비대화 증후군을 억제하고 거시적 통계 관성을 부여하는 릿지(Ridge)와 라쏘(Lasso) 등 정규화(Regularization) 텐서 역학을 상술한다.

1. 계수 폭발 현상과 패널티 역학의 도입

과적합 증후군이 발병한 수리 체계의 내부를 해부해 보면 공통적인 병리 현상이 관측된다. 복잡하게 얽히고 요동치는 궤도를 억지로 현출하기 위해 회귀 모델 차원의 파라미터 계수(Coefficient, \beta) 텐서 값들이 서로 엄청난 스케일 위아래로 폭발(Blow-up)하여 서로 상쇄/융합을 다툰다는 점이다. 즉, 작은 입력 \delta x 동요에 결괏값이 급격히 발산해 버리는 고감도(High Sensitivity) 비선형 상태가 되어 버린다.

이 고삐 풀린 계수들의 팽창을 억누르기 위해, 최소자승법의 근원적 오차 공간 SSE 뒤에 계수 크기에 비례하는 강력한 스칼라 감쇠 마찰력(Penalty Term)을 접합하는 것이 역학적 정규화(Regularization)의 기본 제동 프레임워크이다.

\text{목적 함수(Objective Function)} = SSE + \lambda \times (\text{Penalty Tensor})

여기서 \lambda(람다)는 정규화 계수(Regularization Parameter)로, 원본 데이터 매핑 충실성(SSE 최소화)과 모델의 구조적 단순성 확보(마찰력 최대 가동)라는 두 물리학적 축 사이를 시소처럼 조율하는 제어 튜닝 노브(Tuning Knob) 역할을 수행한다.

2. 릿지(Ridge) 역학: L2 정규화와 연속적 수축망

릿지 회귀(Ridge Regression, 혹은 티호노프 정규화 Tikhonov Regularization) 기법은 계수 벡터의 스칼라 제곱합 즉, L2-Norm 에너지를 패널티항 공간으로 편입시킨다.

J_{Ridge} = \sum_{k=1}^N (y_k - \sum_{j=1}^m x_{kj} \beta_j )^2 + \lambda \sum_{j=1}^m \beta_j^2

1.1 분산의 억제와 다중공선성(Multicollinearity) 타파

이 릿지 공간에 편입된 제곱 패널티 함수 원리는 계수 \beta_j가 커지면 커질수록 그 폭주 에너지를 2차 함수적으로 가파르게 처벌한다. 따라서 시스템을 구성하는 모든 연립 변수들의 계수들이 공간의 영점(0)을 향해 전체적으로 잔잔히 구속, 수축(Shrinkage)되는 억제력이 전방위로 발생하게 된다.
아울러 공학 시스템에서 여러 계측 변수가 상호 극도로 밀접한 상관관계(예: 부피와 무게 등 다중공선성)를 가져 일반 최소자승 역행렬의 행렬식이 0에 수렴하려는 고환성(Singularity) 위기에 처하더라도, 대각 행렬계에 지속적으로 스칼라 상수 \lambda를 투여하여 가역 행렬 계를 강제할 수 있도록 수학적 퇴화 단층을 안정화시키는 지대한 우위를 확보한다. 단, 릿지 역학은 계수들을 0에 가깝게 누를 뿐 정확히 0으로 소멸시키지는 않아 무의미한 독립 변수를 완전히 제거해 낼 수는 없다.

2. 라쏘(Lasso) 역학: L1 정규화와 차원 축소 압박

라쏘 회귀(Lasso Regression, Least Absolute Shrinkage and Selection Operator) 기법은 모델 계수의 텐서 절댓값 합, 즉 L1-Norm 척도를 제어 역학 패널티로 소환한다.

J_{Lasso} = \sum_{k=1}^N (y_k - \sum_{j=1}^m x_{kj} \beta_j )^2 + \lambda \sum_{j=1}^m \vert \beta_j \vert

3.1 모서리 해(Corner Solution)와 변수 선택능(Feature Selection)

이 L1 정규화 공간계 기하학은 L2 기반인 릿지와 본질적으로 상이한 스펙트럼 궤도를 지닌다. 릿지의 패널티 영역이 매끄러운 원(Circle)의 형태를 띠어 회귀 타원이 경계선과 매끄럽게 접선으로 교차한다면, 라쏘의 L1 패널티 역학 영역은 뾰족한 다이아몬드(Diamond) 모서리 축을 형성한다.
등오차 곡선(Contour)이 팽창하다가 라쏘의 다이아몬드 구역에 맞닿게 되면 가장 뾰족하게 돌출된 좌표축 위 모서리에서 직접 충돌하여 접착되는 현상이 기하학적 확률 속으로 발생하게 된다.
