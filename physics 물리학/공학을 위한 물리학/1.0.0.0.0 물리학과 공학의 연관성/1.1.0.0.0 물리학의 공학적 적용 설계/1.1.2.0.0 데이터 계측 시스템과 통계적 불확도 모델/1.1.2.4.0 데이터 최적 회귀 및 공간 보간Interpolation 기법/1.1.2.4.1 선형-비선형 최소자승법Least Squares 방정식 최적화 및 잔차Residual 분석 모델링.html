1.1.2.4.1 선형/비선형 최소자승법(Least Squares) 방정식 최적화 및 잔차(Residual) 분석 모델링

공학 시스템의 거시적인 역학 거동은 종종 데이터 점들을 모두 오차 없이 관통하는 구불구불한 다항식 곡선보다는, 단일한 본질적 지배 방정식(Governing Equation) 모델—가령 훅의 법칙(선형)이나 뉴턴의 냉각 법칙(지수 함수)—로 더 완벽하게 요약된다. 센서 기기로 획득한 방대한 개수의 관측 데이터를 이러한 저차원의 기저 방정식 역학 궤도로 매핑할 때, 내재된 가우시안 노이즈(Gaussian Noise)를 제압하고 모델의 파라미터를 확정적으로 수렴시키는 대수학적 황금률이 바로 최소자승법(Method of Least Squares)이다. 본 절에서는 전체 오차 에너지를 스칼라 공간계에서 최하점으로 끌어내리는 이 최적화 방법론을 선형과 비선형 스텍트럼으로 나누어 구축하고 잔차 분석(Residual Analysis)의 공학적 지표를 선언한다.

1. 선형 최소자승법(Linear Least Squares)의 대수 행렬 체계

N 쌍의 계측된 독립 변수 및 종속 변수 데이터 군집 (x_k, y_k) 가 존재하고, 모델 설계자가 이 변수 군이 선형 방정식 모델 y = a x + b (또는 다중 선형 변수상에서 y = a_1 x_1 + \dots + a_m x_m) 의 매커니즘을 추종한다고 물리학적으로 확정 지었다고 가정하라.

1.1 오차의 무차원 스칼라 텐서화 및 최소화 역학

각 특정 지점 x_k 에서 실제 데이터 y_k와 이상적 모델이 짚어낸 예측값 f(x_k) = ax_k + b 사이에는 필연적인 세로 방향의 갭이 발생하며 이를 잔차(Residual) e_k 로 칭한다.

e_k = y_k - (ax_k + b)

부분적인 양(+)의 편차와 음(-)의 편차가 허무하게 상쇄되어 본질적인 오차 증폭을 눈가림하는 것을 방지하기 위해, 오차 텐서를 비선형 에너지 단위인 제곱으로 승산하여 총합 스칼라 연산 공간을 구축한다. 이 전체 오차 제곱합(Sum of Squared Errors, SSE) 행렬이 바로 최소자승법의 관측 지표이다.

SSE = \sum_{k=1}^N e_k^2 = \sum_{k=1}^N (y_k - a x_k - b)^2

이 SSE 에너지 공간이 다차원 밥그릇(Convex) 형태의 포물면 공간을 형성한다는 공학적 진리를 빌려 최적의 계수 텐서 a, b를 탐색하라. 벡터 미적분학에 의거하여 SSE 를 모델 변수 a 와 b 에 대해 각각 편미분(Partial Derivative)하고, 그 거동 기울기가 0 이 되는 무중력 상태(극솟값 정점)로 연립 방정식을 봉함시킴으로써 최적 계수를 도출해 낸다.

\frac{\partial (SSE)}{\partial a} = -2 \sum_{k=1}^N x_k (y_k - a x_k - b) = 0
\frac{\partial (SSE)}{\partial b} = -2 \sum_{k=1}^N (y_k - a x_k - b) = 0

이를 선형대수학적 행렬 연산 구도로 각성시키면 가우스-조던 소거법(Gauss-Jordan Elimination) 등으로 단 한 번의 직접 풀이(Direct Solver)만으로 최적화 시스템 정규 방정식(Normal Equations)을 돌파하게 된다.

2. 비선형 최소자승법(Non-linear Least Squares) 최적화 회귀

그러나 구조 진동의 감쇠조화진동(Damped Harmonic Oscillation) 모델 y = A e^{-\gamma t} \sin(\omega t) 이나 유체역학 파이프의 달시 위스바흐(Darcy-Weisbach) 비선형 마찰계수 모델 등은 a, b와 같은 예측 타깃 변수들이 연산 공간 안에서 삼각 함수나 지수 위치에 얽혀(Entangled) 들어가므로 상기의 선형 편미분 대수 연산으로 결코 평정할 수 없다.

2.1 가우스-뉴턴(Gauss-Newton) 및 레벤버그-마쿼트(Levenberg-Marquardt) 최적화 알고리즘

비선형 시스템의 자승 모델은 다차원 공간 상에서 울퉁불퉁한 수많은 지엽적 함정(Local Minima)을 내포한 기하학적 난제로 진화한다. 엔지니어는 선형화와 재귀적 접근이 결합된 수치 해석 기반의 반복 연산 점화식(Iterative Scheme)을 채택해야만 한다.

초기 추측 계수 벡터 \mathbf{\beta}_0를 시드(Seed)로 꽂아 넣은 후, 현재 편차의 야코비안 행렬(Jacobian Matrix) J—모델 편미분의 변화량 텐서—를 추출하여 한 걸음씩 국소적으로 스칼라 내리막(Gradient Descent)을 밟아가며 오차 계곡의 바닥(Global Minima)으로 수렴 점착하는 가우스-뉴턴(Gauss-Newton) 알고리즘이 발동된다.
만일 역행렬의 발산 징후나 악조건(Ill-conditioned) 수렴 정체가 판명될 시에는 이 알고리즘 계의 황제격인 레벤버그-마쿼트(Levenberg-Marquardt) 제어 역학을 소환하여 감쇠 계수 \lambda 로 야코비안을 안정화시키면서 신속한 역수렴 방정식을 달성하라.

3. 잔차(Residual) 분석 지표와 모델 건전성 검증

거대 연산으로 선형/비선형 방정식의 회귀 계수를 찾았다고 통계적 모델링의 역학 평가를 종결지어서는 안 된다. 획득한 수리 모델이 시스템의 이면을 얼마나 온전히 지배하고 있는지 확증하려면 반드시 잔차 체계의 사후 부검 분석(Post-mortem Analysis)이 수반되어야 한다.

  • 결정계수(R-squared, \mathbf{R^2}): 도출된 회귀 곡선이 원천 데이터 군의 전체 분산 산포도 중 과연 몇 %를 수학적으로 설명(비례 투영)해 냈는가를 알리는 절대 지표이다. R^2가 1에 수렴할수록 오차 요인(SSE)이 증발하고 모델의 기저 지배력이 총체적임을 입증한다.
  • 잔차의 패턴 진단(Residual Pattern Diagnosis): 잔차 e_k 점들을 독립 변수 도메인 축을 따라 플롯(Plot) 했을 때, 그 점들이 0을 중심으로 상하 무작위 백색 소음(White Noise) 파동과 같이 일정하고 무작위적인 띠(Band)를 유지해야만 진정한 모델을 구축한 것이다. 만약 잔차들이 2차원 부메랑 형상의 체계적인 패턴이 궤도에 돌출된다면, 그것은 엔지니어가 선형 모델로 재단해서는 안 될 비선형 공학 체계를 강제로 구겨 넣은 심각한 언더피팅(Underfitting) 범과 임을 즉시 시인해야 한다.
