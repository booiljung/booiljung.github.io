1.1.2.4.7 푸리에 주파수 변환을 통한 대역 제한(Band-limited) 계측 신호 분포 이산 데이터 복원

시간이나 공간을 따라 연속적인 흐름을 지닌 아날로그(Analog) 신호를 디지털 데이터 계측 시스템(DAQ) 시스템으로 포획할 때, 물리량은 연속 축 위에서 일정한 간격의 파편화된 이산(Discrete) 샘플(Sample) 군망으로 전락하게 된다. 이렇게 조각난 점들을 이어 본래의 유려한 연속 역학 곡선으로 거듭 수복해 내는 과정 역시 거시적인 보간(Interpolation)의 범주에 정초한다. 그러나 진동, 음향, 다이내믹 충격파와 같은 주파수 의존적인 조화(Harmonic) 특성을 지닌 공학 신호들은 단순한 다항식이나 기하학적 커브가 보간할 수 없는 초월적 파동 철학을 수반한다. 본 절에서는, 계측된 신호의 에너지가 일정한 주파수 한계 내에 갇혀 있다는 대역 제한(Band-limited)의 물리학적 특수성을 기화로, 푸리에 공간 교환 법칙을 통한 이산 점군의 완벽한 보간, 즉 나이퀴스트-섀넌(Nyquist-Shannon) 원리 기반의 아날로그 진값 복원 파이프라인을 규명한다.

1. 나이퀴스트-섀넌(Nyquist-Shannon) 샘플링 정리와 한계 대역성

시간에 따라 변화하는 동적 물리 신호 x(t) 역학 시스템이 갖는 가장 중대한 사전 전제는 바로 ’대역 제한(Band-limited)’이라는 성질이다. 즉, 푸리에 변환(Fourier Transform)을 통해 시간 도메인 t를 주파수 스펙트럼 도메인 f 로 편환(Mapping)시켰을 때, 이 신호의 에너지 파동 성분이 임계 최대 주파수 f_{max} 이상에서는 완전히 소멸하여 통과하지 못하는(차단된) 상태라 가정하라.

이러한 물성적 구속 조건이 충족될 때, 섀넌의 샘플링 역학 정리는 이 신호를 시간 간격 T_s 로 샘플링할 경우의 임계 방어선을 선언한다.
데이터 수집 표본화 주파수 f_s = 1/T_s 는 계측 대상인 원천 신호의 최고 진동 주파수 f_{max} 보다 반드시 2배 아득히 빠른 속도로 채집되어야 한다는 나이퀴스트 비율(Nyquist Rate) 준칙이다.

f_s > 2 f_{max}

만일 이 방어선이 붕괴되어 조밀하게 포착되지 못하면 고주파 파동 영역이 저주파로 위장 침투하여 스펙트럼이 겹쳐버리는 알리아싱(Aliasing)이라는 결코 되돌릴 수 없는 파괴적 재앙이 계측기 내부에 발발한다.

2. 대역 제한 체계의 수리적 보간: 싱크(Sinc) 필터링 함수 역학

나이퀴스트 제약을 위배하지 않고 이산 데이터 행렬 x[n] = x(n T_s) 가 온전히 포획되었다고 파악될 시, 그 어떤 정보의 손실조차 개입하지 않은 채 원래의 아날로그 진함수 x(t) 로 100% 무결점 복원하는 궁극의 보간 방정식이 휘태커-섀넌 보간 공식(Whittaker-Shannon Interpolation Formula)이다.
이 과정에는 n번째 시간 노드의 좌표상에 거대한 진폭으로 솟아올랐다 점차 소멸하는 특수 직교 기저기인 이상적 저대역 필터(Ideal Low-pass Filter), 즉 싱크(Sinc) 함수 텐서가 대입된다.

x(t) = \sum_{n=-\infty}^{\infty} x[n] \cdot \text{sinc} \left( \frac{t - n T_s}{T_s} \right)
\text{sinc}(x) = \frac{\sin(\pi x)}{\pi x}

2.1 시간 및 주파수 쌍대성의 공간 관통 메커니즘

이 공식의 물리학적 의미는 경이롭다. 각각의 파편화된 측정 점 x[n] 들의 진폭에다 고주파가 차단된 둥글고 잔잔한 물결파(\text{sinc} 곡선 파동)들의 장막을 입혀 이 무한 융합 곡선을 스칼라 합산하는 것이다.
다항식 적합이 점들의 기하학적 근접성에 기댔다면, 이 보간은 신호 역계 내에 고주파 급격 변동(충격 진동)이 존재하지 않음을 이미 보증받은 대수 법칙상에서 결착된다. 데이터 점과 점 사이의 비어있는 허공은 푸리에 영역의 저대역 스펙트럼 분포 특성에 조작 역산되어 물결파의 상호 간섭(Interference)으로 물리적 틈새를 오차 없이 스스로 메우게 된다.

3. 공학 적용계: 이산 신호 재샘플링(Resampling)의 정수

현대의 진동 동역학 주파수 계측이나 전기 음향 시스템 등에서 관측된 신호를 가공할 때, 다항식 기반의 보간(Linear or Cubic) 선을 사용하여 임의의 시간 사이 값을 추정하는 행위는 주파수 영역에서 필연적으로 가짜 주파수(Spurious Frequency Harmonic) 노이즈를 창조해버리는 비과학적 왜곡 처사이다.
오직 이상적인 싱크(Sinc) 텐서를 투입하는 대역 제한 복원망—실용적인 디지털 신호 처리에서는 무한 급수를 유한 윈도우(Windowed Sinc)로 대체하거나 이산 푸리에 변환(FFT) 상에서 영-패딩(Zero-Padding)을 밀어 넣어 시간 파형으로 역변환(IFFT)하는 기가 막힌 차원 가공술—만으로 신호의 변질(Distortion)을 원천 차단한 상태에서의 정밀 시간 해상도 증폭, 즉 무오차 공간 재샘플링(Resampling)의 대위업을 달성토록 해야 한다.
