2.4.2 시스템 동특성 파악을 위한 모달 해석

공학 구조물은 눈에 보이지 않지만 특유의 주파수 자국, 즉 ’고유 진동수(Natural Frequency)’를 뼈에 새기고 탄생한다. 바람에 강하게 흔들리다 무너져 내린 타코마 나로스 브릿지(Tacoma Narrows Bridge) 사건이나, 터빈 주파수가 일치하는 순간 산산조각이 나는 공장 로터의 붕괴는 모두 한 가지 물리적 결함을 놓쳤기 때문이다. 외부에서 시스템을 두드리는 힘(Excitation)의 주파수가 시스템이 태초부터 갈망하던 고유 진동수와 맞아떨어지는 순간, 공명(Resonance)이라는 무한대 파괴의 춤판이 벌어진다는 사실이다.

이러한 공명 파탄을 예지하고 회피하기 위해 공학자들은 외력이 가해지기 전, 구조물 본연의 진동 설계도를 엑스레이처럼 투시해 낸다. 구조물이 지닌 뼈대의 부드러움(강성 행렬)과 무거움(질량 행렬)을 오일러 공역 공간 위에서 찢고 분해하여, 물체가 허공에서 어떤 주파수로 어떠한 뒤틀림 모양(Mode Shape)을 만들어 낼지 추적하는 궁극의 동역학 해석 기법, 모달 해석(Modal Analysis) 의 지배 방정식 속으로 진입한다.
