2.4.2.2 주파수 응답 함수(FRF)와 진동 감쇠 공진 회피(Resonance Avoidance) 평가

구조물의 고유 주파수(\omega_n)를 찾아내는 것은 진동 해석의 절반에 불과하다. 뼈대의 취약한 주파수 혈를 알았다고 해서 기계의 파괴를 영원히 막을 수는 없다. 현실의 모터, 엔진, 타이어, 바람(Wind Load)은 시시각각 변하는 복잡한 광대역 주파수를 내뿜으며 이 쇳덩어리를 향해 무자비한 정상 분력 타격(Excitation)을 퍼붓기 때문이다. 외부에서 들어오는 힘의 주파수 파동(\omega)과 기계의 내부 고유치(\omega_n)가 충돌하는 순간, 구조물이 얼마나 미친 듯이 증폭되어 요동칠지를 판별하는 수학적 투시경이 바로 주파수 응답 함수(Frequency Response Function, FRF) 다.

이 장에서는 입력된 1뉴턴(\text{N})의 힘이 어떻게 수백 밀리미터(\text{mm})의 공진 진폭(Resonance)으로 둔갑하는지, 그 극단적인 비율 편차를 \text{Hz} 스펙트럼 위에 펼쳐본다. 나아가 이 파멸적인 공진계의 에너지를 억누르고 빨아들이기 위해 댐퍼(Damping)를 쑤셔 넣고 거대한 쇳덩이(TMD)를 달아 진동을 회피하는 물리적 타협의 세계를 심층 전개한다.
