2.4.2.2.8 구조물 진동 민감 스펙트럼 수명 연장을 위한 형상 위상 최적화(Topology Optimization) 진동계 제어

마찰 댐퍼나 쇳덩어리(TMD) 추를 다는 행위는 이미 뼈대에 문제가 터진 후 처방하는 사후 약방문이다. 애초에 처음부터 쇳물을 부어 브래킷이나 엔진 마운트 하우징을 찍어낼 때, 그 기하학적 형태 안에 “진동 주파수 혐오(Resonance Aversion)” 코드를 DNA로 심어둘 수는 없을까? 엔지니어가 깎는 게 아니라, 미적분과 유전 알고리즘 기반 컴퓨터가 스스로 철 덩어리를 갉아먹으며 진동의 폭압을 피하는 뼈대를 진화 창조하는 위대한 생성 설계론, 그 정점인 형상 위상 최적화(Topology Optimization) 진동 통제계의 세계를 펼친다.

1. 컴플라이언스(Compliance) 최소화 및 주파수 회피 목적 함수(Objective Function) 세팅

모터가 윙윙거리는 특정 공간에서 부품이 살아남기 위해 컴퓨터는 꽉 찬 철제 사각형 블록 덩어리를 던져 놓고 룰(목적 함수)을 지정한다.
“질량(비용)은 원래의 30\% 만 남기고 모조리 깎아 내라(Mass Constraint). 단, 첫 번째 고유 주파수 \omega_1 의 마지노선을 절대 150\text{Hz} 밑으로 떨어뜨리지 말라!”

이 극단적인 최적화 문제의 타겟이 되는 수식은 대체로 동적 컴플라이언스(Dynamic Compliance, C = \mathbf{F}^T \mathbf{X}, 뼈대의 낭창거림을 뜻하는 텐서)를 극소화하는 것이다.

  • 컴퓨터(SIMP 유한 분요소 해석기 등)는 사각형 블록 안의 수만 개 메쉬(Mesh) 픽셀 하나하나에 0\text{(완전 빔 허공)} 부터 1\text{(단단한 쇳덩어리)} 사이의 가짜 밀도 텐서 x_e 부여한다.
  • 그리고 편미분 기울기 유전 도함수(Sensitivity Analysis)를 추적하여, 진동 방어(강성 기여도)에 아무런 쓸모가 없는 빈둥거리는 공간(응력 0)부터 차례차례 픽셀을 0 스칼라로 갉아내(삭제) 버린다.

2. 뼈대 진화 렌더링 무기: 바이오닉 구조 텐서 결속

이 피 말리는 미적분 반복문(Iteration)이 100바퀴쯤 돌고 나면, 반듯했던 사각형 쇳덩어리는 가장 기괴하고 외계 생명체(Bionic) 같은 거미줄이나 나무뿌리, 혹은 사슴 뼈다귀 모양의 유기적 뼈대로 둔갑한다.

이 파괴적 유기 형상의 본질을 들여다보라. 컴퓨터가 남긴 뿌리 조직 뼈대들은 정확하게 외부 모터가 후려치는 진동 타격(Excitation)이 흘러나가는 ‘힘의 계곡 선(Load Path)’ 공간만을 한 치의 오차 없이 물어뜯고 단단한 I-Beam 모양으로 뭉쳐져 있다.

  • 모터 결합부 부근은 살을 두껍게 집중 배치하여 고주파(150\text{Hz} 이상 침투 방어) 진벽을 짜고,
  • 부품 가장자리 끝단은 질량을 모조리 파내버려(관성 하중 극소화 감소) 공차의 피로파괴(Fatigue Life 파열) 곡선 자체를 소멸시켰다.

단순히 덜덜 떨리는 기구에 두꺼운 쇠를 더 붙이는 야만적 수치 대입의 시대는 종말했다. 형상 위상 최적화 모델 위에서 부품은 시스템의 파동을 이길 수 있는 방향의 스트레스 라인으로만 ‘뼈를 잇고’, 쓸모없는 살은 스스로 증발시키며 주파수를 가지고 논다(\text{Shifting Frequency Boundaries}). 우주 탐사선 부품이 벌집 모양으로 뻥뻥 뚫려 있으면서도 발사 폭격을 견디는 이유는 진동 해석 수학자가 \mathbf{K} - \omega^2\mathbf{M} 편미분 공간 속에서 쓸모없는 \mathbf{M} 질량 블록을 수학적 페널티 킬(Penalty Kill)로 모조리 도려냈기 때문이다.
