2.4.2.2.2 공진 주파수(Resonance) 구역 진폭 증폭비율(Q-Factor)의 구조적 감쇠 의존 임계 방정식

모터가 30\text{Hz} 로 덜덜덜 돌고 있는데 옆에 있는 철판이 마침 30\text{Hz} 의 1차 고유 주파수(\omega_n)를 가졌다면, 기계는 공진(Resonance)의 무한 구렁텅이로 빠질 운명이다. 미분 방정식 미적분학에 따르면 외부 입력 주파수 \omega 와 시스템 고유 \omega_n 이 정확히 맞아떨어져(\omega = \omega_n) 진동 주파수비율(r=\omega/\omega_n)이 딱 1.0 에 수렴하는 순간, 바닥에서 밀어주는 변위 y_{in} 대비 구조물이 허공으로 튀어오르는 응답 진폭 x_{out} 의 확대 비율(M_a = x_{out}/y_{in})에서 질량(관성, \omega^2 M)과 스프링(탄성, K)의 방어선이 서로를 0 으로 소멸시키며 완벽하게 붕괴해 버리기 때문이다.

이제 이 절망적인 공진의 피크 스파이크 높이를 결정하는 최후의 생명줄은 구조체가 품고 있는 끈적이는 마찰 에너지, 감쇠비(Damping Ratio, \zeta) 단 하나뿐이다. r=1 일 때 구조물이 얼마나 요동칠지 가늠하는 Q-팩터(품질 계수)와 대수 감소율(Logarithmic Decrement)의 감쇠 평가 스칼라 텐서를 해부한다.

1. 정적 처짐 대비 동적 진폭 확대 계수(Magnification Factor)의 수학적 점프

스프링 K 에 하중 F_0 를 그냥 얌전히 얹어놓으면 철판은 기하학적으로 X_0 = F_0 / K 만큼 찌그러진다(정적 처짐).
반면 이 F_0 를 주파수 \omega 로 흔들어(동하중) 댈 때 진폭 치솟음 계수 모델(Magnification Factor M_a)은 무자비한 분수 함수로 돌변한다.

M_a = \frac{1}{\sqrt{(1 - r^2)^2 + (2 \zeta r)^2}} \quad \text{(단, } r = \frac{\omega}{\omega_n}\text{)}

  1. 저주파 r \ll 1: 모터가 아주 느리게 도는 구간. (1-r^2) \approx 1 에 가까워 동적 진폭 M_a 은 1 즉, 정적 처짐과 똑같이 얌전하게 움직인다.
  2. 공진 도달의 무저갱 r = 1: 질량 관성과 스프링 반력이 서로 상쇄되며 (1-1) = 0 으로 분모의 첫 괄호가 영원히 타버려 증발한다.
    그러면 진폭 계수는 오직 허수부 감쇠항에 멱살이 잡힌 채 M_{a, \text{max}} = \frac{1}{2\zeta} 라는 극한의 미분 피크로 치솟아 오른다.
  • 만약 댐퍼가 전혀 없는 이론적 우주 기계(\zeta = 0)라면 진폭은 1/0 = \infty (무한대). 즉 공진 타격 한 방에 다리가 가루가 되어 폭발한다.
  • 강철 기계 부품이 서로 용접된 일반적 구조물의 감쇠비는 기껏해야 \zeta \approx 0.02 \sim 0.05(2 \sim 5\%).
    분모 스칼라 2 \times 0.02 = 0.04 를 깔면, 1/0.04 = \mathbf{25\text{배}}.
    즉 정적으로 1\text{mm} 겨우 찌그러지던 쇠파이프가 공진 주파수를 맞는 순간 무려 2.5\text{cm} 씩 미친 듯이 허공을 찢으며 출렁이는 파열(Resonance amplification) 상태로 터지는 것이다.

2. Q-Factor(품질 계수)와 점성/구조 감쇠 계수의 Half-Power 밴드 폭

공진 피크가 25 배만큼 뾰족하게 치솟아오른 텐서 수치를 무선 통신학, 전기/기계 진동학에서는 대역폭 필터 성능을 나타내는 척도인 Q-팩터(Quality Factor, Q = 1/(2\zeta) = M_a) 라 칭한다.
공학자가 구조물의 감쇠비(Damping Ratio \zeta)를 거꾸로 알아내고 싶다면 센서로 이 피크를 타격하고, FFT 그래프 봉우리를 그린 뒤 이 공명 피크의 양쪽 어깨 가파른 폭(Bandwidth, 반전력 폭 Half-Power Bandwidth) 을 절삭해 텐서를 역산한다. 봉우리가 수직으로 얇고 날카롭게 설수록(Q-factor 오버 슈팅), 이 시스템의 감쇠는 0 에 수렴하는 극단적 불안정 쇳덩어리임을 뜻한다.

우주선 추진체의 초음파 센서 마운트는 발사 순간 토해내는 무작위 엔진 폭발 진동(Random Excitation)에 휩싸인다. 이 폭발적인 연산 속에서 가속도 파손을 막으려면 공학자는 공진점을 아예 벗어나도록(r \ll 1 혹은 r \gg 1) 설계하거나 그게 불가능하다면 값비싼 점탄성 코팅을 발라 무식하게 시스템의 \zeta 를 \text{10\%} 단위로 끌어올리는 피크(Q factor) 참수 기구학을 집행해야 한다. 고체 철판이 요동치며 소산시키는 모든 에너지는 이 임계 방정식 렌더링에 영구히 귀속된다.
