2.4.2.2.5 다중 입력/다중 출력(MIMO) 시스템의 부분 적수 코히어런스(Coherence) 분석망 신호 평가

실험실 벤치 위에 조그만 쇳조각 하나 올려놓고 망치 하나로 때려보는 초보적인(SISO, Single Input Single Output) 세계를 넘어보자. 자동차, 선박, 여객기 본체 같은 거대한 모달(Modal) 덩어리를 해석하려면 차체 사방 5군데에 제각기 다른 광대역 모터 진동기(Shaker)를 동시에 달아 비대칭으로 쑤셔 흔드는 지옥도, 이른바 다중 입력 다중 출력 파동망(Multi-Input Multi-Output, MIMO) 시스템의 주파수 대수응답 텐서를 풀어야 한다.

문제는 내가 차체 지붕에 뽑아낸 응답 가속도 y_1 이 도대체, 저 멀리 1번 셰이커(앞바퀴)가 친 힘에 요동치는 건지, 3번 셰이커(머플러) 파동에 진동하는 건지, 아니면 옆 공장 트럭이 쿵 지나가면서 발생시킨 더러운 노이즈 잡음이나 비선형 마찰(Rattling) 때문에 튀는 값인지 오리무중에 빠진다는 점이다. 이 신호의 ‘순결도’ 혹은 ‘오리지널 주범을 찾는 색출망’, 그것이 스펙트럼 밀도(PSD) 파이프 연산이 쏘아 올린 모달 신뢰성 스칼라, 기여도 함수 코히어런스(Coherence, \gamma^2) 다.

1. 십자 전력 스펙트럼(Cross-Power Spectrum)과 다원 혼합비율

여러 대의 셰이커가 각각 힘 궤적 집합 무리 파동 \mathbf{f}(t) = [f_1(t), f_2(t), \dots]^T 를 때리고 응답 \mathbf{x}(t) 배열을 분출할 때, 공학자는 단순한 푸리에 변환 찌꺼기를 곱하는 게 아니라 상호 스펙트럼 밀도 함수 행렬(Cross-Spectra Density Matrix, \mathbf{S}_{ff}, \mathbf{S}_{xf}) 을 곱하여 각 힘과 진폭 사이의 인과 관계(Causality Ratio) 비율 끈적임 지도를 도출한다.

  • 단일 계통 간섭 결속(Ordinary Coherence, \gamma^2_{xy}):
    \gamma^2(\omega) = \frac{|S_{xf}(\omega)|^2}{S_{xx}(\omega) S_{ff}(\omega)}
  • 이 \gamma^2 수식은 정확하게 0 부터 1.0 (0\% \sim 100\%) 사이에서만 진동하는 비율(Ratio) 파동 텐서다.
  • 만약 출력 가속도 x 의 30\text{Hz} 피크를 봤는데, 그 피크 자리를 내리긋는 코히어런스 차트 값이 0.99 (99\%) 였다? 그것은 “지금 당신이 보고 있는 출력 진동의 99\% 는, 오직 당신이 때린 해머(입력 f) 폭죽의 에너지로 만들어진 순도 100% 진품 선형 데이터다“라는 면죄부 발급이다.

2. 결함 신호(Drop)의 폭락 구배와 다중 부분 코히어런스(Partial)의 필터링 진압

하지만 만약 30\text{Hz} 에서 거대한 FRF 진폭 피크가 떴는데, 그 아래 코히어런스 텐서 선을 확인했더니 \gamma^2 = 0.2 (고작 20\%)로 푹 파여서(Drop) 곤두박질치고 있다면?

  1. 그 무섭게 솟은 피크 진동은, 내가 조종하는 입력 셰이커가 친 게 아니라 외부에서 몰래 섞여 들어온 기생 잡음(Noise) 노이즈 일 확률이다. (에어컨 실외기 소음 진동 혼용).
  2. 구조물 내부의 볼트가 풀려 구조가 선형의 아름다운 후크 법칙(Hooke’s Law)을 무참히 부수고 덜컹덜컹 부딪히는 굉음 비선형성 타격(Non-linearity) 파동에 오염되었을 확률.
  3. 내 센서는 여기 1번 출력에 있는데, 저 멀리 2번 3번 셰이커(Other Inputs)의 주파수파가 뼈대를 타고 흘러들어와 센서를 친(Leakage/Secondary coupling) 확률이다.

이 3번 에러를 지워버리기 위해 MIMO 공학자들은 각 셰이커의 영향을 수학적 백색 잡음으로 덜어내는 부분 코히어런스(Partial Coherence) 혹은 다중 코히어런스(Multiple Coherence) 초월 대수망을 건다.

수많은 주파수 요동(H-Matrix) 속살에서, 단순한 진폭 곡선만 바라보는 건 초보 동역학 분석가다. 마스터 진동 공학자는 FRF 창을 띄우자마자, 항상 화면 그 아래 칸에 스칼라 0\sim1 사이를 벌벌 떨며 평행선을 그리는 코히어런스(\gamma^2) 함수 텐서를 곁눈질로 매섭게 관찰한다. 이 코히어런스 등고선이 계곡처럼 무너지는 그 지점은, 내가 만든 역학 시뮬레이션 모델 수식과 시스템 인과관계가 현실 파괴에 의해 동맥처럼 끊어지는 선형 종말의 자리이기 때문이다.
