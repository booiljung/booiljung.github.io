2.4.2.1.6 베셀 함수(Bessel Function)로 해석하는 원형 막(Membrane) 및 실린더의 축대칭 모달 차동

자동차 직사각형 철판이나 I-빔처럼 직교 좌표계(x, y)로 곧게 자를 수 있는 구조물은 그나마 사인-코사인(Sine/Cosine) 1차원 파동 함수로 해석이 맞아떨어진다.
하지만 비행기 제트 엔진의 실린더 구조통이나 드럼 악기의 원형 막(Circular Membrane) 은 어떠한가? 타격이 가해질 때 이 둥그런 표면을 타고 번져나가는 호리병 모양의 파동 굴곡이나, 동심원(Concentric Circle)처럼 밀려 올라가거나 부풀어 오르는 진동 모드를 x-y 직각 눈금자로 풀려 들었다간 미적분식이 통째로 파열해 영구 에러 무한루프에 빠지고 만다.
공학자들은 극좌표계(r, \theta) 특유의 휘어진 가장자리 파동 텐서를 유폐시키기 위해, 행성 궤도의 요동을 풀던 수학 극의, 베셀 함수(Bessel Function의 제1종 및 제2종 함수계) 를 꺼내 들어 원형/원통형 기하 구조의 축대칭(Axisymmetric) 진동 모드 지도를 완성한다.

1. 극좌표계 파동 방정식과 분수 차수의 제1종 베셀 함수(J_n(kr))

북(Drum)의 가죽인 둥근 막(반지름 a) 가장자리를 단단히 꿰매 고정해 놓고 중앙을 망치로 꽝 쳤다고 가정하자.
막 표면의 출렁거림 변위 w(r, \theta, t) 를 지배하는 연속 파동 방정식 \nabla^2 w = \frac{1}{c^2} \frac{\partial^2 w}{\partial t^2} 에 원통형 극좌표 연산자 텐서를 풀어 넣으면, 반지름 r 에 대한 지독한 상미분 방정식 형태(Bessel’s Equation)가 똬리를 틀게 된다.

r^2 \frac{d^2 R}{dr^2} + r \frac{dR}{dr} + (\lambda^2 r^2 - n^2) R = 0

  • 일반 1차원 사인/코사인이 원점에서 멀어질수록 영원히 똑같은 진폭으로 넘실대는 반면, 둥근 막 중앙점(r=0)에서 터진 진동 에너지는 원 테두리(r=a)로 사방팔방 퍼져나가며 면적이 넓어지므로 에너지가 흩뿌려지고 깎여나간다(Decay).
  • 이 “멀어질수록 진폭이 줄어들며 파도치는” 기묘한 감쇠 파도를 가장 완벽하게 그려내는 초월 함수가 바로 제1종 베셀 함수 J_n(kr) 이다.

2. 노드 원(Nodal Circle)과 노드 지름(Nodal Diameter) 모드 형상의 예술

가장자리가 고정(r=a 에서 w=0)된 둥근 테두리 경계 조건을 베셀 함수 J_n(\lambda a) = 0 이라는 뿌리(Root)에 밀어 넣고 컴퓨터 솔버(Solver)를 돌리면, 우리가 여태껏 단 한 번도 보지 못한 환상적인 방사/축대칭 분수 고유 주파수와 n, m 차 위성 모달 궤도 모양이 렌더링된다.

  • 기본 축대칭 모드(0,1 모드): 둥근 막 전체가 위쪽으로 볼록하게 하나로 팽창했다 수축하는, 호흡(Breathing) 진동. 중앙부가 가장 높이 솟구치고 가장자리로 갈수록 베셀 함수 곡률선에 맞춰 부드럽게 가라앉는다.
  • 노드 라인 분할 모드(1,1 / 2,1 모드): n \ge 1 이 활성화되면 J_n(kr) \cos(n\theta) 곱의 각도 \theta 성분이 깨어나며 막 표면을 피자 조각 자르듯 노드 지름(Nodal Diameter) 으로 갈라친다. 막의 왼쪽 절반이 위로 볼록(+) 솟을 때, 오른쪽 절반은 정확히 아래로 깊게 오목(-)하게 파이는 대칭 파동이 요동친다.
  • 노드 서클 수축 팽창 모드(0,2 / 0,3 모드): 베셀 함수의 두 번째, 세 번째 파도 뿌리가 0 에 닿으면서 중앙은 위로(+) 치솟는데 그 테두리는 도넛 링 모양으로 푹 꺼져(-) 서로 반대로 오르내리는 파문(Ripple)의 극치 노드 원(Nodal Circle) 파동이 나타난다.
