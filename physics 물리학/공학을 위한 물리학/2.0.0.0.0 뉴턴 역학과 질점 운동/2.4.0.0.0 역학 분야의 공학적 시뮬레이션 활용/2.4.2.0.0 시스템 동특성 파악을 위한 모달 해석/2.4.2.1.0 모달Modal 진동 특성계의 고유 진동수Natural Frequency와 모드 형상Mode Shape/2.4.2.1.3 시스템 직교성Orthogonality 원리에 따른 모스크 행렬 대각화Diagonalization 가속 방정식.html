2.4.2.1.3 시스템 직교성(Orthogonality) 원리에 따른 매트릭스 대각화(Diagonalization) 가속 방정식

거대한 10만 개짜리 유한 요소(FEM) 질점 텐서 방정식에 수많은 고유 벡터 진동 모드(Mode Shape, \mathbf{\phi}_1, \mathbf{\phi}_2 \dots)들이 쏟아져 나왔다. 수학자들이 도출해 낸 이 고유 벡터들 사이에는, 마치 x 축과 y 축이 90^\circ 직각으로 긋고 지나가 서로 절대 간섭하지 않는 것처럼 기괴한 대칭 행렬망의 역학적 직교성(Orthogonality Principle) 이 성립한다.

이 놀라운 직교성 원리는 “1번 모달 진동(굽힘)과 2번 모달 진동(비틀림)은 각자 소모하는 질량 관성 에너지와 탄성 스프링 에너지를 절대 훔쳐 먹거나 나누지 않고 철저히 독립적으로 우주를 떠다닌다“라는 거대한 열역학적 완전 결별을 증명한다.

1. 질량 직교성(Mass Orthogonality)과 강성 직교성(Stiffness Orthogonality)의 내적 제로(Zero)

서로 다른 주파수를 띠는 1번 펄럭임 모드 \mathbf{\phi}_1 과 3번 요동 모드 \mathbf{\phi}_3 를 가져다가, 가운데 구조물의 질량 행렬 \mathbf{M} 이나 강성 행렬 \mathbf{K} 를 끼워 넣고 양옆으로 곱(내적)을 전개해 보라. 이 기묘한 행렬곱 스칼라는 무조건 0(Zero) 으로 파괴된다.

  • 질량 행렬 내적: \mathbf{\phi}_1^T \mathbf{M} \mathbf{\phi}_3 = 0
  • 강성 행렬 내적: \mathbf{\phi}_1^T \mathbf{K} \mathbf{\phi}_3 = 0

반대로 자기 자신(오직 1번 모드는 1번 모드끼리)을 집어넣어 내적을 구하면 0 이 아닌 특정한 스칼라, 즉 모달 질량(Modal Mass, m_1) 혹은 모달 강성(k_1) 진폭 에너지양이 살아남는다.

  • \mathbf{\phi}_1^T \mathbf{M} \mathbf{\phi}_1 = m_1 \ne 0
  • \mathbf{\phi}_1^T \mathbf{K} \mathbf{\phi}_1 = k_1 = \omega_1^2 m_1

이 아름다운 0 과 1(\text{스칼라} 진폭)의 흑백 논리는 곧 거대한 연립 미분 폭풍(Coupling System)의 숨통을 영원히 끊을 수 있는 단칼의 절단 검(Decoupling Sword), 즉 매트릭스 대각화(Matrix Diagonalization) 의 재료가 된다.

2. 모달 변환(Modal Transformation)과 완전 독립 1자유도(SDOF) 연립 분리 타격

우리는 전체 변위 위치 다항 벡터 \mathbf{x}(t) 를 “내가 추출한 주 모드 형상 벡터 거판(\mathbf{\Phi})들의 각각의 가중치(\mathbf{q}(t)) 곱의 합산(Superposition)“이라는 새로운 가상 세계 무대(Modal Coordinates)로 모핑(Morphing)하여 치환할 수 있다.
즉 \mathbf{x} = \mathbf{\Phi} \mathbf{q} 로 두고, 이 식을 원본 뉴턴 방정식 \mathbf{M}\ddot{\mathbf{x}} + \mathbf{K}\mathbf{x} = \mathbf{F} 에 우겨 넣은 뒤 앞에 형상 텐서 전치 행렬 \mathbf{\Phi}^T 를 전체 곱해 버린다.

그러면 모든 비대각의 교차 연성항(Off-diagonal, 구조체 부품 간 얽히고설킨 멱살잡이)이 위에서 증명한 ‘직교성 내적 0’ 버프로 완전히 날아가 버린다! 남는 것은, 거대한 \mathbf{10만 \times 10만} 행렬 다자유도계가, 단순하고 초라한 \text{1자유도(SDOF)} 독립식 N 개로 찢어져 나열된다.

m_k \ddot{q}_k(t) + c_k \dot{q}_k(t) + k_k q_k(t) = P_k(t) \quad (\text{단 } k=1, 2, \dots N)

엔지니어는 এই 모드 중첩(Mode Superposition) 의 직교 스위치 덕에, 10만 개 변위 좌표계의 그물망에 구속되지 않고 가장 강력하게 요동치는 1\text{차} \sim 10\text{차} 주파수 질량 가중치(Truncation) q_1, \dots, q_{10} 모달 미분 방정식 10개만 풀어서 쇳덩어리의 최종 타격 궤적을 거의 무결점 텐서로 계산(FRF Synthesis)해 낼 수 있게 되었다. 직교성이란, 물리와 재료가 지닌 근본적 우주의 진동 파동 에너지 분할선, 그 불가침 무결 지대다.
