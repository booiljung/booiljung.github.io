2.4.2.1.7 실험 모달 해석(EMA) 타격 해머와 진동 센서 위상 차(Phase Difference) 비교 곡선 추출

컴퓨터의 가상 공간 속에서 수천만 개의 유한 요소(FEM) 메쉬를 그려 고유 주파수를 아무리 정밀하게 뽑아냈다고 한들, 실제 세상의 끈적거리는 철판이나 용접 부위의 감쇠나 비선형 접촉 상태 앞에서는 수식과 현실이 무참히 틀어지기 일쑤다. 시뮬레이션의 환각을 진짜 물리 공간에서 검증하고 때로는 수정하기 위해, 엔지니어들은 철제 부품을 나일론 끈으로 매단 뒤 직접 손으로 두들겨 패는 아날로그와 디지털의 결합, 실험 모달 해석(Experimental Modal Analysis, EMA) 에 돌입한다.

1. 임팩트 해머(Impact Hammer)와 역학 주파수 응답 파동

해머 끝에는 타격하는 순간 부품에 얼마나 묵직하게 두들겨 맞았는지(F_{in}(t)) 정밀 기록하는 로드셀(Load Cell) 역각 센서가 박혀 있다.

  • 이 타격을 맞고 부품이 부르르 떨 때 수백 개의 지점(Node)에 매미처럼 달라붙은 극소형 가속도계(Accelerometer)들이 살벌하게 미세한 쇳덩이의 가상 요동 진동(A_{out}(t)) 가속도를 전기 신호 배열로 뽑아낸다.
  • 이 시간축 타격(Time Domain Pulse)과 가속도 응답파를 재빨리 랩뷰(LabVIEW)나 오실로스코프를 거쳐 고속 푸리에 변환(FFT) 해버리면, 망치의 광대역 주파수 범위(0 \sim 1만\text{Hz})에 걸쳐 타격을 주었을 때 시스템이 ’어느 주파수’에서 미친 듯이 민감하게 펄쩍 뛰어오르는가(응답 증폭이 터지는가) 비율 지도(전달 함수: H(\omega) = \text{Output}(\omega) / \text{Input}(\omega)) 인 주파수 응답 함수(Frequency Response Function, FRF) 가 스크린에 폭발한다.

2. 위상(Phase) 곡률계의 180^\circ 절벽: 공진 폭발 지상 증명

FFT 로 뽑힌 주파수 함수 차트에는 단순히 30\text{Hz} 에서 바짝 솟아오른 뾰족한 진폭(Magnitude) 텐서 꼭대기만 있는 게 아니다. 공학자가 진정한 고유 주파수(Natural Frequency, \omega_n)를 찾아 판별하는 가장 절대적인 진리의 눈금자는 진폭 피크 값 아래에 숨어있는 위상각 보데 선도(Phase Angle Bode Plot) 의 치명적 역전 절벽(Phase Shift)이다.

  • 저주파로 칠 때 부품은 망치가 밀면 밀리는 대로 망치와 똑같은 위상(0^\circ)으로 그냥 얌전히 따라 밀려간다.
  • 그러다 망치질을 초당 더 빨리, 더 빠르게 때리는 주파수 에너지 \omega 가 서서히 물체의 진짜 고유 진동수 \omega_n 심장 근방에 바짝 도달하는 순간,
  • 위상의 90^\circ 지연: 입력 힘과 응답 속도가 드디어 물리적 한계를 뚫고 일치(동위상)하며 시스템이 힘의 모든 텐서 에너지를 끝도 없이 블랙홀처럼 빨아들여 미친 듯이 솟구친다(공진 증폭의 꼭대기 피크).
  • 위상의 180^\circ 절벽 폭락: 하지만 진동 주파수가 그 공진 폭발 에너지를 1\text{Hz} 라도 단 1밀리미터라도 뚫고 초월(Exceed)하는 그 즉시, 위상 곡선은 망치가 미는 방향과 정반대 편으로 -180^\circ 로 수직 낙하하며 꼬꾸라진다. 질량의 무거운 관성이 이제 망치의 스피드를 도저히 쫓아가지 못하고 박자가 역전(Out-of-Phase)되어 버린 것이다.
