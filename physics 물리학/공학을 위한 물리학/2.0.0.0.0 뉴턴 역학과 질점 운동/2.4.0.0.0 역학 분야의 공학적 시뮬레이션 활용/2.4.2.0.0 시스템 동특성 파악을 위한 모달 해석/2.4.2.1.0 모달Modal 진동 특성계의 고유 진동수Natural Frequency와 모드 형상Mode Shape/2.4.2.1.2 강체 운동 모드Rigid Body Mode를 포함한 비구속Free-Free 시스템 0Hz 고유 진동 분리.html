2.4.2.1.2 강체 운동 모드(Rigid Body Mode)를 포함한 비구속(Free-Free) 시스템 0Hz 고유 진동 분리

인공위성이나 하늘을 나는 비행기, 심지어 고무줄에 달려서 허공에 대롱대롱 매달려 있는 자동차 브레이크 디스크를 떠올려보자. 이들은 우주 공간이나 허공에 떠 있기 때문에 어떤 나사나 볼트로도 땅에 단단하게 고정(Fixed)되어 있지 않다. 이런 기하학적 비구속(Unconstrained) 혹은 자유-자유(Free-Free) 경계 조건을 가진 시스템의 모달 해석(Modal Analysis)을 돌리면 엄청나게 당황스러운 수학적 마비 상태가 첫 모드부터 연출된다.

컴퓨터가 가장 첫 번째로 뱉어내는 1번, 2번, 3번 고유 진동수가 \omega = 0\text{Hz}, 무려 0 헤르츠 로 계산되어 나오기 때문이다. 1 초에 0 번 진동한다는 것은 도대체 물리학적으로 무엇을 의미하는가? 바로 부품이 구부러지거나 찌그러지는 ’탄성 변위’가 전혀 없이, 쇳덩어리 모양이 일정한 형태 그대로 굳은 채 우주 공간을 둥둥 떠가는 강체 운동 모드(Rigid Body Mode) 의 출현을 선고하는 것이다.

1. 특이 강성 행렬(Singular Stiffness Matrix) 붕괴와 0\text{Hz} 모달 분할

땅에 핀(Pin) 하나라도 구속되어 있는 일반 다자유도계의 강성 행렬 \mathbf{K} 는 당당하게 역행렬 \mathbf{K}^{-1} 을 갖는 정칙(Non-singular) 텐서(양의 정부호)다.
하지만 허공에 떠 있는 Free-Free 비구속 구조물은 땅으로부터 잡아당기는 힘이 없기 때문에 스프링 매트릭스 \mathbf{K} 의 행렬식(Determinant)이 무참하게 0 으로 붕괴해 버린다(특이 행렬 \text{Singular}).

이 \mathbf{K} 의 특이성에 의해 \mathbf{K}\mathbf{\phi} - \omega^2\mathbf{M}\mathbf{\phi} = 0 고유치 방정식을 푸는 순간, 좌우/상하/전후 병진(3개)과 회전(3개), 총 6개의 독립적인 0\text{Hz} 주파수 해(\omega_{1\sim6} = 0)가 쏟아져 나온다.
이 첫 6개의 모드 형상 벡터 \mathbf{\phi} 를 애니메이션으로 틀어보면 날개가 펄럭이거나 몸체가 휘어지는 진동은 일절 관찰되지 않는다. 기계 덩어리가 통째로 뻣뻣하게 위로 붕 떠오르거나(Heave), 수평으로 날아가거나(Surge), 혼자 팽이처럼 빙글 도는(Yaw) 완벽한 거시 강체 관성 이동 궤적만을 그려낸다.

2. 유연 모달(Flexible Modal)과의 직교 분리 및 강체-탄성 연성계(Flex-Rigid) 독립 통제

우주에 떠 있는 위성의 강성 행렬이 0\text{Hz} 로 특이 공간을 점령했다고 컴퓨터 소프트웨어(ANSYS 등)가 튕기거나 크러시(Crash) 되지 않도록, 현대 고유치 솔버(Solver)인 Block Lanczos 등의 알고리즘은 이 0\text{Hz} 주파수를 허공(Shift)으로 띄워서 걸러내는 지수 이동 제어(Shifted Eigenvalue Extraction) 마법을 부린다.

  • 스펙트럼 분해의 미학: 알고리즘이 \omega_{1\sim6}=0 강체 모드를 성공적으로 벗겨낸 직후, 마침내 7번째 주파수(\omega_7, 예: 15\text{Hz})부터 쇳덩어리가 진짜로 찌그러지고 구부러지는 첫 번째 유연체 탄성 고유 진동(Flexible Bending Mode) 이 태동하기 시작한다.
  • 이 0\text{Hz} 강체 공간과 +1\text{Hz} 이상의 탄성 공간은, 직교성(Orthogonality)이라는 질량 행렬망 속에서 서로 완벽하게 수직 배방(Orthogonal)되어 섞이지 않는다. 즉 로켓이 앞으로 날아가는 가속(강체 0\text{Hz})과 로켓 표면이 떨리는 것(탄성 80\text{Hz})은 철저하게 분리된 에너지 계통이라는 점을 증명한다.

망치로 타이어나 드럼통을 공중에 끈으로 매달아놓고 때리는 ‘자유-자유 패키지’ 충격 모달 실험(Hammer Test)에서, 우리가 매단 저 끈의 고무줄 강성이 너무 뻣뻣하면 0\text{Hz} 로 나와야 할 강체 모드가 5\text{Hz} 나 10\text{Hz} 구역으로 오염 상승하여 밀고 들어오게 된다. (이것이 서스펜션(스프링) 서브 시스템의 탄생이다.)
순수한 비구속 0\text{Hz} 모달 평형이 지켜져야만 우리가 궁금해하는 자동차 뼈대의 순수 \text{1st Bending Mode}(30\text{Hz}) 를 외부 바닥 계통의 방해 없이 완벽히 관찰해 낼 수 있는, 자유 공중 분리(Isolation)의 해석적 극점이다.
