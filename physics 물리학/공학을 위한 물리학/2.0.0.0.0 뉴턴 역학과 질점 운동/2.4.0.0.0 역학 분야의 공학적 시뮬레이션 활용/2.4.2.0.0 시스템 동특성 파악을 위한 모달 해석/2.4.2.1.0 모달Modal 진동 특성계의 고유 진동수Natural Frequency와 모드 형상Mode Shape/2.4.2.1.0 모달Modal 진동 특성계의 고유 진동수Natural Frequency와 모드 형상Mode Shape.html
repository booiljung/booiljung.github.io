2.4.2.1 모달(Modal) 진동 특성계의 고유 진동수(Natural Frequency)와 모드 형상(Mode Shape)

아무리 기하학적으로 복잡하게 얽힌 수백만 개의 유한 요소(FEM) 망이나 자동차 샤시(Chassis) 트러스 뼈대라 할지라도, 그 안을 흐르는 모든 복잡한 진동 궤적은 결국 시스템이 지닌 몇 가지 정해진 ’비밀스러운 패턴’들의 가중치 단순 합(Superposition) 조합에 불과하다. 이 구조물의 가장 깊은 무의식적 떨림, 그 근간에 자리 잡은 DNA 와도 같은 두 가지의 변수가 바로 고유 진동수(Natural Frequency, \omega) 와 그 주파수에서 구조물이 어떠한 모양으로 찌그러지고 뒤틀리는지를 나타내는 모드 형상(Mode Shape, \phi) 이다.

이 장에서는 어떠한 외부 타격이나 구동 모터(Input Force)도 없는 무의 공간, 자유 진동 방정식(\mathbf{M}\ddot{\mathbf{x}} + \mathbf{K}\mathbf{x} = 0)의 정적막을 찢고 나오는 고유치 행렬 해설의 톱니바퀴를 돌려본다. 거대 연성계 매트릭스가 직교성 기법에 의해 어떻게 순수한 하나의 독립 모드로 낱낱이 유리(Decoupling)되는지 수치적 카타르시스를 맛본다.
