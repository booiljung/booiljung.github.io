2.4.2.1.8 주 모드(Principal Mode) 에너지 분포 비율에 기반한 유효 질량(Effective Mass) 참여 계수 판정

건물에 100만 개의 질점(10^6 DOF, 자유도)이 있다면, 이 건물 안에는 100만 개의 고유 진동 주파수와 100만 가지의 뒤틀리는 모드 형상이 뻗어나간다.
그렇다면 지진이 꽝 하고 건물을 때릴 때(P(t)), 이 100만 개의 고유 모드들이 모두 다 똑같이 미친 듯이 활성화(Excited)되어 춤을 출까? 절대 그렇지 않다. 건물을 부수는 진짜 위험한 밴드 주파수 대역 모드는 기껏해야 상위 1\text{차} \sim 10\text{차} 이내의 몇 개뿐, 나머지 99만 9천 개의 자질구레하고 딱딱한 고주파 형상 거동들은 지진 에너지를 흡수하지도 뱉지도 못한 채 유령처럼 버려진다(Truncation).
누가 어떤 모드가 진짜 건물 전체를 뒤흔드는 주인공(Principal Mode) 인가? 그 공헌도를 퍼센트(\%)로 투명하게 잘라 보여주는 오차 검증 스칼라, 유효 질량 참여율(Effective Mass Participation Factor) 텐서를 도출한다.

1. 지진 하중 여기(Excitation) 벡터와 모달 참여 텐서 결속식

바닥이 수평축(X 방향)으로 흔들리는 지진 가속도 \ddot{x}_g(t) 가 들어오면, 건물의 모든 질량 노드(Node)들은 관성력 -m_i \ddot{x}_g(t) 에 의해 모조리 옆으로 후려쳐진다.
방향 벡터 \mathbf{R} (여기서 X 방향이면 \mathbf{R} = [1, 0, 0, 1, 0, 0 \dots]^T)과 전체 질량 행렬 \mathbf{M} 을 곱한 외부 타격 에너지 덩달이 \mathbf{M R} 이, 내가 구한 i 번째 고유 모드 형상 \mathbf{\phi}_i 와 얼마나 박자가 맞는지를 내적(Inner Product) 하여 뽑아낸 상관 스칼라 값을 모달 참여 계수(Modal Participation Factor, \Gamma_i) 라고 한다.

\Gamma_i = \frac{\mathbf{\phi}_i^T \mathbf{M} \mathbf{R}}{\mathbf{\phi}_i^T \mathbf{M} \mathbf{\phi}_i}

이 값(\Gamma_i)이 0 이라면, 그 모드가 아무리 공진 주파수(\omega)를 갖고 있어도 이 X 방향 지진 하중 앞에서는 털끝만큼도 춤을 추지 않는다(비활성. 예: 위아래로만 요동치는 Z 축 모드는 수평 X 지진파에서 \Gamma=0 으로 완벽히 멸종).

2. 모달 유효 질량(Effective Mass m_{\text{eff}, i}) 90% 법도

단순히 모달 참여 계수에 그치지 않고, 그 i 번째 모드가 동원시켜 잡아 흔드는 ‘건물 전체 질량 대비 실효 질량’ 이 몇 \text{kg} 에 달하는지를 산출할 수 있다.

m_{\text{eff}, i} = \frac{(\mathbf{\phi}_i^T \mathbf{M} \mathbf{R})^2}{\mathbf{\phi}_i^T \mathbf{M} \mathbf{\phi}_i} \quad (\text{단, 질량 정규화시 분모=1 이므로 } m_{\text{eff}, i} = \Gamma_i^2)

1번, 2번, 3번 \dots 모드까지의 유효 질량을 다 합쳤을 때 이 값이 건물 전체 진짜 무게(Total Mass)의 약 80\% \sim 90\% 이상을 훌쩍 넘어가면(Mass Participation \ge 0.9), 공학자는 미련 없이 해석 스텝을 멈춘다(Truncation).
