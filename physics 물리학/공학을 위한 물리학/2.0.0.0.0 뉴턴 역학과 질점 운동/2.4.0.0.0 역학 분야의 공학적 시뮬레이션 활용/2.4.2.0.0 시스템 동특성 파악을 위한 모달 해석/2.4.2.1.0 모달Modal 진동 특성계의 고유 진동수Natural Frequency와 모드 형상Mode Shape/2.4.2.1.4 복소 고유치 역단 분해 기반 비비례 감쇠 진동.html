2.4.2.1.4 복소 고유치 역단 분해 기반 비비례 감쇠 진동

구조물의 질량과 강성만이 아름답게 교차하는 무감쇠(Undamped) 혹은 정갈한 비례 감쇠(Rayleigh Damping 등, \mathbf{C} = \alpha\mathbf{M} + \beta\mathbf{K})를 거느린 모달 역학계에서는, 이전 장에서 보았듯 고유 벡터 \mathbf{\phi} 가 완벽한 실수(Real) 값으로 추출된다. 실수 모드 형상(Real Mode Shape)은 “기계 뼈대의 모든 절점(Node)이 시간축 상에서 정확히 동시에 꼭대기(Peak)에 도달하고 동시에 0 을 지나치는, 위상이 일자로 단단히 결속된 1차원 거대한 파동 줄넘기(\text{Standing Wave})“라는 질서정연함을 상징한다.

하지만 자동차 엔진 마운트에 특수한 고경도 유체 댐퍼나 스크랩핑 마찰재, 고자기장 능동 베어링이 억지로 비대칭적으로 장착된 파탄 시스템이라면 어떨까? 즉 댐퍼(감쇠) 행렬 \mathbf{C} 가 더 이상 \mathbf{M} 과 \mathbf{K} 에 비례하지 않고 혼자서 대각선 비례를 깨트리는 ‘비비례 감쇠(Non-proportional Damping)’ 상태. 이제 실수 행렬 미학은 붕괴하고, 기계 구조체는 기괴하고 소름 돋는 복소 모달(Complex Modal) 세계이자 점착 파도(Traveling Wave) 영역으로 곤두박질친다.

1. 2계 미분 방정식을 1계 상태 공간(State-Space) 행렬로의 찢김(State Vector)

이 비례성이 깨진 흉측한 \mathbf{M}\ddot{\mathbf{x}} + \mathbf{C}\dot{\mathbf{x}} + \mathbf{K}\mathbf{x} = 0 다자유도 시스템에, 앞 장의 아름다운 실수 모드 형상 변환 \mathbf{\Phi} 를 때려봐야 \mathbf{\Phi}^T \mathbf{C} \mathbf{\Phi} 의 비대각선 원소는 직교하지 못하고(Non-zero) 찌꺼기를 남겨 연립 방정식을 절대로 분리 단절(Decoupling) 시키지 못하고 목을 조른다.

공학자들은 이 난관을 부수기 위해, 가속도 \ddot{\mathbf{x}} 변수를 아예 지워버리고 전체 N 자유도 시스템을 2N 자유도의 \dot{\mathbf{y}} = \mathbf{A} \mathbf{y} 라는 1차 상태 공간(State Space) 선형 스칼라 매트릭스로 억지로 찢어발겼다(\mathbf{y} = [\mathbf{x}^T, \dot{\mathbf{x}}^T]^T).

이 거대해진 비대칭 \mathbf{2N \times 2N} 시스템 행렬 \mathbf{A} 의 고유치 문제 \det(\mathbf{A} - \lambda \mathbf{I}) = 0 를 풀면, 모달 주파수는 더 이상 허수 축 하나(\pm i\omega)에 박히지 못하고, 무거운 감쇠 상수 실수부(\sigma)와 진폭 허수부(\pm i\omega_d)로 쌍을 이룬 켤레 복소수 고유치(Complex Conjugate Eigenvalues, \lambda_{1,2} = -\sigma \pm i \omega_d) 로 추출된다.

2. 복소 형상 벡터(Complex Mode Shape)와 진행파(Traveling Wave) 요동의 환각

진짜 끔찍한 변화는 고유 주파수 스칼라가 아니라, 구조물의 뒤틀림을 표시하는 쇳덩이 고유 형상 벡터 \mathbf{\psi} 자체가 복소수(\mathbf{\psi} = \mathbf{Re} + i\mathbf{Im}) 공간 위상 편차를 장착했다는 것이다.

  • 실수 모드(Standing Wave): 모든 점이 각자 높이는 달라도, 멈칫! 하고 정지하는 최고점 도달 시기(Phase Angle 0^\circ 혹은 180^\circ)가 똑같은 합창 단원 같았다.
  • 복소 모드(Traveling Wave 진행파 환각): 비비례 감쇠가 덕지덕지 붙은 헬기 날개 복소 모드 \mathbf{\psi} 를 애니메이션으로 돌려보라.
    1번 노드가 공중 최고점을 찍고 내려오는 와중에, 2번 노드는 이제야 뒤늦게 최고점(위상 지연, Phase Lag 각도 존재)으로 우뚝 치고 올라오는 “꿈틀대는 파도 치기(Wave Propagation)” 기괴한 모션 시퀀스를 그린다. 기계 덩어리에 어딘가 지독한 댐퍼(외란 감쇠 고점)가 붙어 있어, 진동파 에너지가 시스템 전체로 똑같이 돌지 못하고 강제 소산되는 특정 파이프 지점으로 파동이 스멀스멀 빨려 들어가는 흡입 환각(Traveling Node)이 물리력으로 발현한 것이다.

엔지니어가 자동차 브레이크를 아주 살짝 밟을 때(t=0) 브레이크 패드가 “끼이익” 하고 진동 파탄 소음(Squeal Noise)을 내는 이유는, 접촉 마찰 계수 변화율이 시스템 행렬 깊숙이 비대칭 행렬 감쇠(Negative Damping)를 쏴버리기 때문이다. 복소 고유치 \lambda = -\sigma + i\omega 연립 탐색기에 브레이크 시스템 모델을 던져 넣었을 때, 실수부 -\sigma 가 0 을 뚫고 +\sigma 의 양수로 튀어 오르는 순간. 브레이크 진동은 1초도 안 되어 멈추지 않고 스스로 무한대 증폭 폭파되는 찢어지는 비명(Squeal) 곡선으로 수렴한다. 이것이 시스템 불안정성(Instability) 선고 판결문으로서의 복소 모달 해석이다.
