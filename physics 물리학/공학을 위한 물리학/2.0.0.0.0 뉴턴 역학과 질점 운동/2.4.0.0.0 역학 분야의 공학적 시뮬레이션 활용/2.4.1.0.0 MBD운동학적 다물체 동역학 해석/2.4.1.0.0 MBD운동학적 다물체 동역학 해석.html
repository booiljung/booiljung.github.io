2.4.1.0.0 MBD(운동학적 다물체 동역학) 해석

자동차 한 대에는 약 3만 개의 부품이 얽혀 있다. 기어가 치합되어 맞물리기 시작하면 기계를 구성하는 수백 개의 막대, 크랭크, 피스톤은 서로에게 살인적인 간섭 힘(구속력)을 뿌리며 제약된 궤적(Constrained Motion)을 타고 연동한다. 여기서 오직 강체 동역학의 기하학적 링키지와 그 사이에 걸리는 조인트(Joint) 역학만으로 복잡한 시스템의 궤도를 100% 추적해 내는 기계 시뮬레이션의 황제, 다물체 동역학(Multibody Dynamics, MBD) 영역으로 진입한다.

MBD는 기계공학이 구축한 뉴턴-오일러 방정식(Newton-Euler Equation)의 가장 순수하고 잔혹한 연산의 장이다. 부품 하나가 병진/회전하기 위해서는 필연적으로 나머지 모든 부품의 수식 매트릭스가 0.001초마다 비선형 라그랑지안 미분을 거치며 연립(Coupling) 되어야만 한다. 우리는 이 장에서 아무런 저항도, 에너지 손실도 없는 순수 기구학적 위상(Topology) 공간을 탐험하며 메커니즘이 태동하는 구조 원리를 부순다.
