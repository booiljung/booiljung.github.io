2.4.1.1.4 슬라이더-크랭크 구조의 변위계, 속도계 역학 분해 및 가속도 벡터 코리올리 정리

내연기관 엔진의 근본 심장부인 피스톤-크랭크 축, 혹은 산업용 공기 압축기의 작동 뼈대는 오직 하나의 완전한 루프 기하학으로 수렴한다. 회전하는 크랭크(r), 뻗어 나가는 커넥팅 로드(l), 그리고 실린더 튜브를 따라 1차원 직선 왕복만 허락된 피스톤(슬라이더), 바로 슬라이더-크랭크(Slider-Crank) 메커니즘이다.

단 하나의 불이 폭발하며 피스톤을 눌렀을 뿐인데, 기계를 따라 내려간 힘은 크랭크 축을 빙글빙글 돌려 자동차 바퀴를 돌리게 만든다. 이 거대한 직선(x(t))과 회전(\theta(t)) 운동의 무결점 변환기(Converter) 속에 숨겨진 공간 벡터 기하 미적분의 연산 탑, 변위, 속도, 가속도 코리올리 텐서망을 적어내려 본다.

1. 지독한 삼각함수 비선형 얽힘: 피스톤 위치 방정식(x)

피스톤 핀 위치 x 에서 크랭크 중심(모터 축, 원점)까지의 직선거리를 삼각함수 제2 코사인 법칙과 투영(Projection) 텐서로 썰어내면 가공할 비선형 방정식(Non-linear Equation)이 강림한다. 모터 축이 \theta 로 일정하게 돌 때, 실린더 속 피스톤은 완벽한 사인파(Sine Wave)인 단순 조화 진동(SHM)을 절대로 하지 못한다.

x(\theta) = r \cos\theta + l \sqrt{1 - \left(\frac{r}{l} \sin\theta\right)^2}

이 루트 \sqrt{} 안에 기생하는 비율 (r/l)^2 덕분에, 피스톤은 위로 올라갈 때(TDC 부근)는 미친 듯이 느려졌다가 멈추고, 밑으로 내려갈 때(BDC)는 훨씬 짧은 시간 안에 맹렬히 쳐서 바닥을 찍고 도망치는 형상적 비대칭 가속을 뿜어낸다. 만약 커넥팅 로드 l 이 무한히 길다면 모를까, 실제 엔진처럼 좁은 공간(r/l \approx 1/3)에서 이 비선형성은 피스톤과 엔진 블록이 서로를 찢어발기려 드는 거대한 관성 진동 2차 스펙트럼(Secondary Vibration)을 난사해 버린다.

2. 상대 속도 점프와 미분 가속도장의 연쇄 십자 포화 (코리올리 교차 진입)

위치 함수를 미분하여 피스톤 속도 \dot{x} 와 각 뼈대의 속도 벡터 폴리곤(Velocity Polygon)을 그리면 극단적인 직교가 발생한다.
V_{\text{피스톤}} = V_{\text{크랭크핀}} + V_{\text{상대_로드}}. 여기서 모터 각속도 \omega 가 일정한 상황에서 한 번 더 미분해 가속도 다각형(Acceleration Polygon)을 소환하라. 크랭크핀 속도만 부드럽게 돌아갈 뿐, 커넥팅 로드(Connecting Rod)가 휘저어지면서 뿜어내는 가속도 텐서에는 4가지의 무자비한 법선-접선-코리올리 연쇄 파동 벡터가 쏟아져 내린다.

  1. 원심 구심 가속도(Normal/Centripetal Acceleration): 각 관절 끝에서 핀을 파고들며 뜯어내려는 V^2/R 텐션 파괴력.
  2. 접선 가속도(Tangential Acceleration): 로드(Rod)의 회전 각가속도 \alpha 에 의해 피스톤을 실린더 윗벽/아랫벽으로 패대기치려는 측면 압력(Side Thrust)의 주원인.
  3. 이 관절들이 움직이는 좌표계 자체 위로 슬라이더(미끄러짐)가 허용되는 기구들(\text{예: 유압 실린더, 퀵-리턴 메커니즘})에서는, 코리올리 가속도(Coriolis Acceleration, 2 \mathbf{\omega} \times \mathbf{v}_{slip}) 가 벡터의 배를 가르며 기괴한 \text{90}^\circ 비틀림 붕괴 직교 모멘트를 발생시킨다.

수치 시뮬레이션(MBD)으로 \mathbf{\Phi}_q \ddot{\mathbf{q}} = \mathbf{\gamma} 라는 전역 행렬 가속도 연방정식을 단칼에 베어낼 때, 저 변위 비선형 속 루트 항 덩어리가 결국 우변 공간 벡터 \mathbf{\gamma}, 즉 속도 및 자이로(코리올리)들의 교차항인 관성 비선형 외란 벡터 족쇄로 조립된다. MBD는 이 삼각함수 파동과 미적분의 아귀다툼을 사람이 풀 필요 없이, 오직 질점 구속 조건 좌표(\mathbf{q})만으로 슬라이더의 측압(Side force)과 마찰 효율 한계를 몽땅 기하 행렬식으로 환원시켜 모니터에 뿌려준다.
