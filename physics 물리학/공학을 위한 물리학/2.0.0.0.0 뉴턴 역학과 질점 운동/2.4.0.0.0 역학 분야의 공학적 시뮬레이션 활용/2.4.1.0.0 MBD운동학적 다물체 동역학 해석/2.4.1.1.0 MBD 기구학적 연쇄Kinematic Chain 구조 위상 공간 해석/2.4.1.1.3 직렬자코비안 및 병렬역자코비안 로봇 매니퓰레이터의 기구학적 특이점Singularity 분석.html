2.4.1.1.3 직렬(자코비안) 및 병렬(역자코비안) 로봇 매니퓰레이터의 기구학적 특이점(Singularity) 분석

다관절 로봇 팔(Serial Manipulator)의 끝단(End-Effector)이 공간의 목표 좌표 \mathbf{x} 에 도달하게 하려면, 베이스부터 어깨, 팔꿈치, 손목에 달린 모터 각각의 회전 각도 \mathbf{\theta}(자유도)를 어떻게 꺾어야 하는가? 이것이 바로 역기구학(Inverse Kinematics)의 존재 이유다. 문제는 এই 각도 해답 변환의 심장부에 자코비안 행렬(Jacobian Matrix, \mathbf{J}) 이 도사리고 있는데, 특정한 자세에 도달하면 이 자코비안이 블랙홀처럼 모든 역학 기능을 빨아들여 마비시키는 치명적인 특이점(Singularity) 을 낳는다는 것이다.

위상학적으로 줄줄이 이어진 직렬 뼈대(Serial Robot)와, 여러 개의 다리가 하나의 발판을 부여잡는 거미형 병렬 뼈대(Parallel Robot)는 이 기하학적 특이점을 마주하는 텐서 함수망 방식이 정확히 정반대로, 데칼코마니처럼 역전된다.

1. 직렬구조(Serial)의 역행렬 파괴: 작업 공간의 경계선(Boundary)과 팔 뻗음

직렬 로봇에서 손끝의 속도 \dot{\mathbf{x}} 는 자코비안 \mathbf{J} 와 모터 각속도 \dot{\mathbf{\theta}} 의 곱(\dot{\mathbf{x}} = \mathbf{J}(\mathbf{\theta}) \dot{\mathbf{\theta}})으로 직진 전개된다.
당신이 손끝을 \dot{\mathbf{x}} 만큼 이동시키고 싶다면 역행렬 \dot{\mathbf{\theta}} = \mathbf{J}^{-1} \dot{\mathbf{x}} 을 미친 듯이 컴퓨터에 던질 것이다.

하지만 로봇이 팔을 바깥으로 ‘끝까지 쫙 펴버리는’ 순간(오버 익스텐션, Boundary Singularity)이나 손목 조인트가 \mathbf{0} 도가 되어 어깨 축과 일자로 나란히 정렬되는 순간(Inner Singularity), 기하학적 종속성이 태동하여 자코비안 \mathbf{J} 의 행렬식(Determinant, \det(\mathbf{J}))이 무참하게 0(Zero) 으로 꽂힌다. \mathbf{J}^{-1} 연산은 분모가 0 이 되어 터지고, 시뮬레이터 속 로봇 모터는 단 1\text{mm} 전진을 위해 ‘무한대(\infty \text{ RPM})’ 속도로 회전하라는 미친 제어 명령값을 받게 되며 에러를 뿜어낸다.
결국 직렬 구조의 특이점은 더 이상 로봇 팔이 뻗거나 특정 방향으로 움직일 수 없는 ’작업 불가 구속(Losing DOF)’의 무덤이다.

2. 병렬구조(Parallel) 포어워드 싱귤래리티: 외력 증폭과 흔들림 요동

스튜어트 플랫폼(Stewart Platform)이나 델타 로봇(Delta Robot) 등 여러 개의 다리가 끝판 하나를 잡고 있는 병렬 기구의 속도 방정식은 \mathbf{J}_x \dot{\mathbf{x}} = \mathbf{J}_q \dot{\mathbf{q}} 라는 기괴한 쌍둥이 자코비안 암호화 포맷을 갖는다. 여기서 손끝 속도 \dot{\mathbf{x}} 를 뽑아내려면 \dot{\mathbf{x}} = \mathbf{J}_x^{-1} \mathbf{J}_q \dot{\mathbf{q}} 이라는 이중 역행렬 구조를 깨야 한다.

이 병렬 로봇 세계에서는 모터 조인트의 \mathbf{J}_q 가 0 이 되는 것보다, 다리들이 기묘하게 평행해지거나 교차해버려서 역자코비안 공간 행렬 \det(\mathbf{J}_x) 이 0 이 되는 포어워드 특이점(Forward Singularity) 이 가장 두려운 악재다.
이 지점에 다다르면 직렬 로봇과 정반대의 재앙이 터진다. 모터(입력단 \dot{\mathbf{q}})를 꽉 틀어막고 완전히 정지시켜 놨는데도, \mathbf{J}_x 역행렬이 터지면서 결과단인 로봇 끝판 플랫폼 \mathbf{x} 이 혼자서 덜덜덜덜 저절로 요동치며 돌아가는 통제 불능의 추가 자유도 생성(Gaining DOF) 파탄이 작렬하는 것이다.

수학적으로 자코비안은 기구 변수들 사이의 다차원 기울기 벡터(Gradient Vector) 덩어리다. 특이점이란 이 경사면이 완전히 평평해져서, 컴퓨터가 기울기를 타고 어느 쪽으로 내려가야 정답 역기구학 해(IK Solution)가 나오는지 방향타를 영구적으로 상실한 구름 속 늪지대 평원으로 치환됨을 의미한다. MBD 위상 역학의 핵심은 기하 자체를 비틀어 로봇 설계 초기 단계에서부터 이 지옥의 늪지대를 로봇 작업 영역(Workspace) 바깥으로 내동댕이치는 역설계다.
