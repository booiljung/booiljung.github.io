2.4.1.1.0 MBD 기구학적 연쇄(Kinematic Chain) 구조 위상 공간 해석

시계 장인의 정밀한 톱니바퀴부터 굴삭기의 포크레인 암(Arm)까지, 모든 기계 장치는 서로 뻗어 나가며 연결된 뼈대들의 그루터기(Tree) 또는 무한 루프(Closed-loop Chain)로 귀결된다. 이를 다물체 역학계에서는 기구학적 연쇄(Kinematic Chain) 단위라 정의한다.

이 장에서는 수십 개의 쇳덩어리가 어떻게 조인트(Joint) 핀과 만나는지, 그리고 이 기하학적 연결망이 형성하는 거시적 위상 구조(Topology)의 본질을 분해한다. 속도와 가속도 이전에, 과연 이 기계 뭉치가 모터 하나로 내가 원하는 목적지까지 “움직일 수 있느냐“를 증명하는 순수한 그라쇼프 곡선과 그뤼블러 다자유도 검증 방정식의 기하 텐서 세계에 진입하라.
