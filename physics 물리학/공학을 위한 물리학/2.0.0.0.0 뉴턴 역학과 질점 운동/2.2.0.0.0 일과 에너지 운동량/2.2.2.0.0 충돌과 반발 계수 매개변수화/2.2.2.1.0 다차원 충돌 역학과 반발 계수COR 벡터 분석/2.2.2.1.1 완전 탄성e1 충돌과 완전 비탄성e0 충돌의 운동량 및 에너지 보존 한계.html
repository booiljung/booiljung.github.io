2.2.2.1.1 완전 탄성(e=1) 충돌과 완전 비탄성(e=0) 충돌의 운동량 및 에너지 보존 한계

고립계(Isolated System)에서 상호 작용하는 질량 m_1, m_2 의 두 물체 간 충돌은 어떠한 경우에도 전체 운동량(Total Momentum) 보존 법칙(\Sigma \mathbf{p}_{\text{initial}} = \Sigma \mathbf{p}_{\text{final}})을 파괴할 수 없다. 외부 매질 공기 저항이나 충돌 이외 공간 외력이 개입하지 않는 찰나(\Delta t \rightarrow 0) 동안, 시스템 내력인 충격력만이 교환되기 때문이다.

그러나 총 운동량이 변하지 않는다고 해서 초기 가지고 있던 운동 에너지(K)마저 보존되는 것은 아니다. 역학적 운동 에너지가 충돌 후에도 소산됨 없이 유지될 것인가 혹은 형상 파괴나 열에너지로 소실될 것인가를 판가름하는 매개 변수가 뉴턴의 반발 계수(Coefficient of Restitution, e) 다.
반발 계수 e 는 두 물체의 상대 접근 속도 대비 상대 분리 속도의 수학적 비율 텐서를 의미한다.

e = \frac{\Delta v_{\text{separation}}}{\Delta v_{\text{approach}}} = \frac{v_{2}' - v_{1}'}{v_1 - v_2}

여기서 분모 기호 변수 v 는 충돌 전 속도, 상단 프라임 기호 v' 는 충돌 후 속도다. 항상 양의 계수값(0 \le e \le 1)을 가지며 충돌 축(법선 방향) 선상 궤적에 걸친 변위 미적분이다.

1. 완전 탄성 충돌(Perfectly Elastic Collision, e = 1)

스프링이 압축되었다가 한 치의 에너지 오차 손실 없이 100\% 반발 텐서로 되돌려놓는 극한의 물리계 우주 모델이다. 당구공이나 단단한 강철구의 충돌, 혹은 원자나 아원자 입자 간의 전자기적 산란(Scattering) 수준만이 이 극한의 e=1 거시 근사치를 보여준다.

이 시스템은 운동량 방정식군(식 1)에 더하여 시스템의 운동 에너지 총량 \frac{1}{2}mv^2 이 파괴되지 않는다는 절대 무손실 보존 방정식(식 2)이 하나 더 성립하는 우월한 등식계를 갖는다.

  1. 운동량 보존: m_1 v_1 + m_2 v_2 = m_1 v_{1}' + m_2 v_{2}'
  2. 운동 에너지 보존: \frac{1}{2}m_1 v_1^2 + \frac{1}{2}m_2 v_2^2 = \frac{1}{2}m_1 v_{1}'^2 + \frac{1}{2}m_2 v_{2}'^2

이 연립방정식쌍을 타개하여 충돌 직후의 분리 속력 v_{1}', v_{2}' 를 도출하면 다음과 같은 간결한 선형 조합의 결과를 분사한다. 질량 m_1 = m_2 이고 만약 v_2 = 0 이었다면, v_{1}'=0, v_{2}'=v_1 으로 정확히 정지 상태와 속도 벡터가 뒤바뀌는 운동량 트레이딩(Trading)이 성립한다.

2. 완전 비탄성 충돌(Perfectly Inelastic Collision, e = 0)

점토 덩어리 두 개가 부딪치거나, 달리는 기차의 잭 커플러가 물려 들어갈 때 일어나는 카타스트로피 현상, 바로 충돌 후 두 물체의 상대 분리 속도가 영(0)이 되어 하나의 거대한 단일 질량 덩어리로 합착 교행하는 상태(e=0)이다.

v_{1}' = v_{2}' = v_f

이 경우 충돌 이후 속력 v_f 는 오직 운동량 보존 단일 대수 방정식 구조 하나로 완전히 장악된다.

v_f = \frac{m_1 v_1 + m_2 v_2}{m_1 + m_2}

하지만 주의하라. 겉보기에 가장 단순해 보이지만, 이 시스템은 우주에서 기계적 동역학 충돌 에너지(Kinetic Energy) 손실률 편차가 물리적 극댓값(Maximum) 에 오르는 가장 잔혹한 소산 효율 매커니즘이다. 초기 운동 에너지가 충격-파괴 현상을 통해 물체의 형상을 영구적으로 부수거나 변형시키고 찌그러뜨리는 데 기계적 일(Work)로 전부 소모(-W_{nc})되어 버리거나 막대한 열로 비가역 승화 흡수되어 버린다.
