2.2.2.1.5 충돌 직후 마찰 임펄스(Frictional Impulse) 작용 선 및 슬립(Slip) 정지 조건

어떠한 이상적인 교과서도, 공이 바닥에 부딪힐 때 접선 궤적으로 발생하는 쿨롱 마찰력(Coulomb Friction)을 완벽히 무시할 수 없다. 스핀을 듬뿍 먹인 당구공이나 회전하는 테니스공이 바닥을 강타할 때, 충돌 접촉면에서는 극도로 짧은 찰나(Microsecond) 동안 미끄러짐(Sliding)에 저항하는 거대한 마찰 충격량, 즉 마찰 임펄스(Frictional Impulse) 가 뿜어져 나온다. 이 마찰 임펄스는 질점의 단순한 병진 속도(Tangential Velocity)를 깎아먹을 뿐만 아니라, 역으로 물체의 중심을 비틀며 회전(Spin) 각속도마저 뒤흔드는 연성(Coupled) 파동을 창조한다.

1. 쿨롱 마찰 충격량의 비율 텐서

충돌은 매우 짧은 시간(\Delta t)에 폭발하는 현상이므로, 힘 F 가 아닌 그것의 적분치인 충격량(Impulse, P = \int F dt) 계통으로 해석망을 스위칭해야 한다.
법선 축(n-축)을 내리찍는 수직 충격량을 P_n 이라 하고, 이와 맞물려 접선 축(t-축) 방향으로 표면을 쓸고 지나가는 마찰 충격량을 P_t 라 정의하라. 쿨롱 마찰의 기본 원리는 시스템이 극단적인 충격량 영역으로 올라서도 그 비율 상수를 유지한다.

P_t = \pm \mu_k P_n

방향 보정 부호(\pm)는 무조건 상대 미끄럼 속도(v_{rel}) 입사 벡터의 정반대(저항)를 향한다.
법선 충격량 P_n 을 도출하고 나면 동마찰 계수 \mu_k 에 비례하여 접촉면을 긁어내는(Scraping) 마찰 임펄스 P_t 의 크기가 확정된다.

2. 슬립 정지(Slip Halt) 여부와 각운동량 변환 동역학

문제는 찰나의 충격 시간(\Delta t) 도중에 일어나는 ’상태의 변이’다. 마찰 임펄스 P_t 가 물체의 접선 속도(v_t)를 급격히 감속시키고, 동시에 강력한 마찰 토크(P_t \times r)를 유발하여 각속도(\omega)를 급가속시킨다. 두 속도가 서로 타협점을 찾아 완벽한 구름 속도 조건(v_t = r\omega) এ 도달하면, 접촉점의 상대 미끄럼 속도는 충돌 도중에 즉각 0 이 되어버린다.

엔지니어는 충돌 과정을 적분하며 이 슬립 보존 대 슬립 정지 조건을 판독해야 한다.

  1. 지속적 슬립(Continuous Sliding):
    수직 충격량(P_n)에 비해 마찰 계수(\mu_k)가 너무 작거나, 입사 접선 속도가 너무 빠를 때 발현된다. 전체 충돌 시간 내내 공은 미끄러지며(Slip) 바닥을 긁고, 반사각을 예측할 때는 P_t = \mu_k P_n 한계치를 모조리 접선 속도 감쇄 부하로 인가해야 한다.
  2. 슬립 정지 및 접지(Slip Halt):
    강력한 고무공이나 높은 마찰면에서 나타난다. 충돌 찰나의 중간 구역(Intermediate Time)에서 이미 미끄럼이 정지(v_c = 0)한다. 이 시점 이후 남은 충격 시간 동안 마찰력은 \mu_k P_n 을 발휘하지 못하고 정지 마찰 한계 미만으로 뚝 떨어져 시스템은 ‘자연스러운 순수 구름(Pure Rolling)’ 궤적으로 이탈한다.

엔지니어가 무인 수직 이착륙기(VTOL)의 랜딩 기어가 강풍에 밀려 아스팔트 바닥에 비스듬히 스치며 충돌할 때, 지지대가 버텨야 할 휨 모멘트(Bending Moment)를 도출하고 싶다면 반발 계수 e 법선 방정식 하나로는 턱없이 부족하다. 슬립이 계속될지 도중에 정지할지의 한계점을 분리하고 표면을 찢어대는 마찰 임펄스 P_t 의 소산 텐서를 장악하라.
