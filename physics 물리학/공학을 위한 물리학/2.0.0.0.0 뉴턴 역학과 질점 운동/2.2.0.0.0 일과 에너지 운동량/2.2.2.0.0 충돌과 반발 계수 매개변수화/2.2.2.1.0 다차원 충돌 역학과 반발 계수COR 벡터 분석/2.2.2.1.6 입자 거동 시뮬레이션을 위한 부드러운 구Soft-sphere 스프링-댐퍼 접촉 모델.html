2.2.2.1.6 입자 거동 시뮬레이션을 위한 부드러운 구(Soft-sphere) 스프링-댐퍼 접촉 모델

다량의 모래알(Granular Material)이 호퍼에서 쏟아질 때나 지진 카오스 시 콘크리트 잔해 블록 수천 개가 엉켜 무너질 때, 엔지니어는 고전적인 강체(Rigid Body) 반발 계수(e)와 운동량 보존 결합 행렬을 연립하여 풀 수 없다. 충돌이 동시다발적 체인 반응으로 끝없이 지속되는 이산 요소법(Discrete Element Method, DEM) 시뮬레이션에서는, 각 1 마이크로초 단위의 통합 시간에 입자 간의 접촉 역학 지배 지침, 곧 부드러운 구(Soft-sphere) 접촉 모델을 각 입자마다 투여하여 충돌을 미세하게 스캐닝한다.

강체 가정을 박살 내고, 두 입자가 부딪힐 때 가상의 ’찌그러짐 체적(\delta)’을 측정하여 이로부터 선형 스프링의 복원력과 점성 댐퍼의 감쇠력을 미분방정식으로 합성(Kelvin-Voigt 거시 모델)하는 것이 이 공학적 속임수의 정수다.

1. 법선(Normal) 방향 선형 스프링-대시팟 모델망

반경 R_i 와 R_j 인 두 입자가 거리 d 에 포진해 있고 속도 v_i, v_j 로 정면 충돌한다 가정하라. 두 입자 간의 가상 중첩된(찌그러진) 변위 깊이, 즉 침투량(Penetration, \delta)은 \delta = R_i + R_j - d 이다.
이 \delta 를 축으로 물체를 밀어내는 충돌 상호 작용 수직력(F_n) 텐서 방정식은 다음과 같다.

F_n = - k_n \delta - c_n \dot{\delta}

  • 탄성 스프링항(k_n \delta): 입자가 찌그러질수록 비례하여 물체를 강하게 튕겨내는 헤르츠(Hertz) 연성 또는 선형 강성계 반발력이다. 탄성 에너지를 비축했다가 100% 되돌려주며 입성을 강제로 분리한다.
  • 점성 댐핑항(c_n \dot{\delta}): 찌그러지는 속도(상대 접근 속도 \dot{\delta})에 저항하는 비가역 에너지 소산계다. 자동차 서스펜션처럼 충격 에너지를 오일의 마찰 열(Dissipation)로 바꿔 없앤다. 이 임계 감쇠비(c_n)를 통해 우리는 거시적 반발 계수 e 를 역산하여 세팅할 수 있다.

이 대수 방정식 \Sigma F_n = m \ddot{x} 의 미분 적분이 뉴턴의 운동량 교체(e-적용)를 물리 엔진(Physics Engine)에서 완벽히 대체 모방한다.

2. 접선(Tangential) 마찰 슬라이딩 댐퍼와 쿨롱 스핀계

DEM 엔진은 오직 찌그러짐(\delta)에만 집착하지 않는다. 사선 충돌(Oblique Impact) 시 법선과 90^\circ 로 직교하는 접선 방향 미끄럼, 즉 마찰 충돌 궤적을 억제해야 시스템 스핀(Spin)이 전개된다.
접선 방향의 누적 표면 미끄럼 변위 \delta_t 를 추적하여 접선 반력 F_t 를 조립하라.

F_{t, \text{trial}} = - k_t \delta_t - c_t \dot{\delta_t}

그러나 여기서 쿨롱 마찰 법칙(Coulomb’s Law) 한계 파티셔닝이 작동한다. 접선 탄성력이 버틸 수 있는 최대치는 F_{t, max} = \mu F_n (마찰 계수 \times 수직항력) 이다.
접선 힘이 이를 초과하면 입자는 표면을 맹렬히 긁으며 슬라이딩(Sliding) 파단 조건, 즉 |F_t| = \mu F_n 으로 강제 절단 폐색(Truncated)되고 접선 댐퍼를 통한 운동 에너지 산일이 격발된다. 이것이 수십만 개의 흙먼지와 모래 입자들이 지그재그 산비탈 모래시계형태(Angle of Repose)로 쌓이게 만드는 궁극적 시뮬레이터 텐서 통제법이다.

수 마이크로초(\mu s) 크기의 시간 타임 스텝을 가지며 공간상의 파티클 100만 개를 6축 동역학으로 동시에 렌더링하는 광물 분쇄기 설계망(Crusher Simulation), 제약사 분말 정제 믹싱 시뮬레이션 등에 사용되는 최첨단 공학 물리 코어 엔진(LIGGGHTS, EDEM 등)은, 바로 이 입자 1쌍의 Soft-sphere 부드러운 충격 스프링 매트릭스를 무한대로 병렬 연산 확장시킨 결과물인 것이다.
