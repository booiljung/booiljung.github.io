2.2.2.1.3 다물체(N-body) 동시 충돌 시스템의 뉴턴 반발 계수 및 포아송(Poisson) 반발 계수

당구공 두 개가 만나는 2체 문제(Two-body Problem)는 운동량 역학의 가장 단순한 근사치다. 세 개 혹은 수십 개의 구슬(Newton’s Cradle)이나 모래알 같은 강체 군집 입자들이 거의 동일한 찰나의 순간(dt)에 동시 다발 연쇄적(Simultaneous) 충돌을 일으키는 다물체(N-body) 시스템을 직면해 보라.
어느 점을 기준으로 충돌의 복원과 파괴를 판정할 것인가? 운동 역학자들은 이 난해한 연성 텐서망을 통제하기 위해, 충돌의 소산(Dissipation)을 정의하는 반발 계수를 두 가지 상이한 관점(속도 기반 vs 충격량 기반)으로 분해하여 재정립했다.

1. 운동 속도 결합망: 뉴턴 반발 계수(Newton’s COR)

뉴턴(Newton)이 제창한 고전 반발 계수는, 두 물체가 접촉하는 한 지점의 기하학적 국부 상대 속도 궤적에 근거한 운동학적(Kinematic) 척도다.

e_N = \frac{v_{2}' - v_{1}'}{v_1 - v_2}

  • 접근 시 상대 속력 분모와 튕겨 나갈 때의 분리 상대 속력 비율을 일대일로 대조한다.
  • N-body 충돌처럼 세 체 이상이 줄줄이 맞닿은 체인(Chain) 형태에서, 각 접촉면마다 개별 e_N 방정식을 할당(e_{12}, e_{23}, e_{34} \dots)하여 다차원 거대 연립 미분 방정식 행렬로 편입시킬 수 있다.
  • 한계: 그러나 다물체나 편심 스핀이 심하게 걸린 마찰 동반 비대칭 시스템에서는, 이 속도 비율 e_N 이 물리적 극댓값 1 을 초과하거나 음수(역방향 침투)로 발산하여 위배되는 에너지 창조의 모순(Energy Creation Paradox) 버그를 이따금 토해낸다.

2. 임펄스(Impulse) 적분 텐서망: 포아송 반발 계수(Poisson’s COR)

포아송(Poisson)은 아예 속도(기구학)를 버리고, 충돌기에 실제 접촉면에서 발생하여 오가는 충격량(Impulse, \int F dt)의 적분비를 해부용 메스로 사용했다. 이 텐서적 정의는 충돌을 ‘압축(Compression)’ 기간과 ‘복원(Restitution)’ 기간이라는 시간 축으로 절개하고 분해한다.

e_P = \frac{\int_{t_c}^{t_f} F_r \, dt}{\int_{t_i}^{t_c} F_c \, dt} = \frac{\text{복원 충격량(Restitution Impulse)}}{\text{압축 충격량(Compression Impulse)}}

  • 압축기(t_i \rightarrow t_c): 두 물체가 부딪혀 국부적 속도가 완전히 같아지는 순간까지, 물체가 찌그러지며 서로가 밀어댄 총 충격량 적분 파동.
  • 복원기(t_c \rightarrow t_f): 탄성 에너지 복원력으로 인해 두 물체가 서로 밀어내고 분리에 이르는 시간대의 튕김 충격량 파동.

포아송 계수 e_P 는 에너지 기반 텐서의 정직한 대용품(Surrogate)이다.
물체가 마찰을 수반하며 엉뚱한 방향으로 요동치며 비틀리더라도, 면에 수직인 방향성에 기인한 이 충격량의 비율은 결코 1.0(보존)을 넘어 에너지를 창조해 내지 않는 수학적으로 완벽히 안전한(Safe) 한계를 제공한다.

충동 시뮬레이터를 개발할 때(예: DEM 파티클 이산 요소 렌더링), 오직 직선 중심 충돌에 머무는 단순한 모델은 뉴턴의 속도 식을 사용하지만, 마찰과 소성이 얼룩진 N-body 카오스 궤적 렌더링 물리 엔진(Unreal Engine, Havok 등)은 시스템 에너지 발산을 막기 위해 포아송 베이스 임펄스 알고리즘을 코어에 밀어 넣어 시스템 붕괴를 사전에 통제한다.
