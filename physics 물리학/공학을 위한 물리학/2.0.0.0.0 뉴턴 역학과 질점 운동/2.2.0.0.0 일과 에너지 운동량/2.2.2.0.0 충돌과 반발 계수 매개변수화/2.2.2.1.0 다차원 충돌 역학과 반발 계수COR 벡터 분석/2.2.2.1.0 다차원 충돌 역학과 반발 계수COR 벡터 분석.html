2.2.2.1.0 다차원 충돌 역학과 반발 계수(COR) 벡터 분석

물체와 물체가 충돌하는 공간은 1차원의 단순한 직선이 아니다. 타구봉이 당구공을 칠 때도, 인공위성이 우주 궤도 공간 쓰레기(Debris)와 부딪힐 때도, 질량의 질주 궤적은 3차원의 복잡한 병진(Translation) 운동 벡터와 회전(Rotation) 각속도 벡터가 사방으로 뒤엉켜 있다. 이렇게 충돌을 3차원의 기하학적 난잡함 속으로 던져놓았을 때, 가장 먼저 분리해 내야 할 스칼라 및 벡터 기준선은 무엇인가? 그것은 바로 두 물체의 표면이 만나서 이룩하는 공통 접평면(Tangent Plane)과, 그 면에 직교 방향으로 미끄럼 없이 내리꽂히는 충돌 법선(Line of Impact) 망이다.

충돌은 매우 짧은 순간(\Delta t \rightarrow 0) 동안 접촉면에 걸쳐 엄청난 충격력(Impulse, \int \mathbf{F} dt)을 유발한다. 이 충격 텐서는 두 물체가 접근하며 박살 나는 압축기(Deformation Period)와, 탄성 복원력을 통해 튕겨 나가는 복원기(Restitution Period)로 쪼개어 해석된다. 두 주기 간의 충격력 벡터 크기의 비이자, 분리 속도와 접근 속도 벡터 간의 환산 스칼라인 반발 계수(COR, Coefficient of Restitution) e 를 축으로 하여, 에너지 소산과 기구학적 운동량 전파 계통을 분리 제어하는 고등 충돌 방정식을 유도하라.
