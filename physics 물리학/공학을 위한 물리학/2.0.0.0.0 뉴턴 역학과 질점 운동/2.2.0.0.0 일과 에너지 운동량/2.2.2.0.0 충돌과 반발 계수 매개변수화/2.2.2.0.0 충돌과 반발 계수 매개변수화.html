2.2.2 충돌과 반발 계수 매개변수화

두 물체가 극히 짧은 시간 내에 격돌하며 상호작용하는 현상, 즉 충돌(Impact)은 기계계의 국부적(Local) 영역에 막대한 충격 하중(Impulsive Force)을 발산시키는 복잡한 비선형 기구학적 동역학 과정이다. 이 찰나의 순간 동안 시스템에 외부 힘(외력)이 관여하지 않는다면, 물체들의 총 운동량(Total Momentum)은 철저히 보존되지만, 총 운동 에너지(Kinetic Energy)의 보존 여부는 두 물체 표면의 탄성적/소성적 변형 궤적 및 재질 특성에 완전히 종속된다. 이러한 에너지 소산(Dissipation) 역학 메커니즘을 단일한 무차원 거시 파라미터로 압축하여 결합한 것이 바로 반발 계수(Coefficient of Restitution, COR) 이다.

엔지니어는 충돌 현상에서 발생하는 난해한 미시적 탄소성(Elastoplastic) 변형 에너지 장부들을 하나하나 미분하여 계산하는 대신, 이 반발 계수 e 를 매개변수화(Parameterization)함으로써 시스템의 충돌 후 이탈 속도 및 잔존 운동 에너지를 대수적으로 단번에 예측해 낸다. 반발 계수는 제로(0, 완전 비탄성)에서 일(1, 완전 탄성)까지의 스펙트럼 수치를 가지며, 시스템 설계 시 자동차 크럼플 존(Crumple Zone)의 충격 흡수 한계, 다물체 접촉계의 튕김 반발 제동력, 그리고 구름 베어링 내부 미립자 거동의 소산 동역학망을 조율하는 핵심 지표로 군림한다. 본 장에서는 다물체 충돌계 내부를 관통하는 운동량 보존 법칙과 파괴 및 반발 계수의 다차원 벡터 해석망을 수학적으로 정립한다.
