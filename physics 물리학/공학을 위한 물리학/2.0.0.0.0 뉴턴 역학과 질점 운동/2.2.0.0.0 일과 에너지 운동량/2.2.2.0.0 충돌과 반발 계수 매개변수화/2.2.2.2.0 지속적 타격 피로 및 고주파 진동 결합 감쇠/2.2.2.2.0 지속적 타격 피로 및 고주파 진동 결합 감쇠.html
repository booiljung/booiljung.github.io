2.2.2.2.0 지속적 타격 피로 및 고주파 진동 결합 감쇠

앞선 챕터들이 단 한 번의 파괴적이고 강렬한 충돌(Single Impact)에서 일어나는 모멘텀의 환전과 에너지 소산(반발 계수, HIC)을 추적했다면, 본 장에서는 시스템의 수명을 서서히, 그러나 확실하게 갉아먹는 지속적이고 반복적인 마이크로 충격(Repeated Impacts) 과 그로 인한 고주파 진동(High-Frequency Vibration) 의 암울한 공존 매커니즘을 다룬다.

거대한 굴착기의 브레이커(Breaker)가 암석을 수만 번 내리치거나, 노후화된 기어 박스 내부에서 백래시(Backlash) 여유 틈새를 오가며 기어 이빨들이 초당 수백 번씩 육박전을 벌일 때, 이 구조물들은 단순한 정적 변압 응력 한계(Yield Strength)를 초과하여 파괴되지 않는다. 대신 각각의 햄머링 압력 파동이 누적되며 미세한 균열(Micro-crack)을 번식시키고, 물체 자체에 고유하게 내재된 공진(Resonance) 고주파수 대역을 끊임없이 자극하여 피로 파괴(Fatigue Failure) 공간으로 시스템을 강제로 밀어 넣는다. 본 장에서는 찰나의 임펄스가 푸리에 공간에서 어떻게 무한대의 주파수 에너지로 변환되며 고체를 피로역으로 몰아가는지에 대한 비가역 동역학망을 펼쳐 보인다.
