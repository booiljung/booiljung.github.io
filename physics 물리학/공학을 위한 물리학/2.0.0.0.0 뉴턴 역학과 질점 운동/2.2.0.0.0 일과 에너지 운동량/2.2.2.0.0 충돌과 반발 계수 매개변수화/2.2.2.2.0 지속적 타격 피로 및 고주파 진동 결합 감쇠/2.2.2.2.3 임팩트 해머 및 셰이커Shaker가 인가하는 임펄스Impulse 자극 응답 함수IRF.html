2.2.2.2.3 임팩트 해머 및 셰이커(Shaker)가 인가하는 임펄스(Impulse) 자극 응답 함수(IRF)

항공기 동체나 초고층 빌딩, 혹은 미세한 반도체 웨이퍼의 동적인 취약점(고유 진동수와 감쇠비)을 생포하는 진동 모드 해석(Modal Analysis)의 왕도는, 언제나 시스템이 외부에서 들어오는 힘(자극)에 어떻게 진동(응답)하는가를 기록하는 충격 응답 함수(IRF, Impulse Response Function) 에 닿아 있다. 그리고 이 자극의 트리거를 당기는 무기가 바로 순간 타격 임펄스 발생기인 ’임팩트 해머(Impact Hammer)’와, 연속적 파동으로 구조물을 울려버리는 ‘가진기(Shaker)’ 기구학이다.

1. 찰나의 임펄스와 임팩트 해머(Impact Hammer)

임팩트 해머는 망치 대가리에 압전(Piezoelectric) 힘 센서(Force Transducer)가 내장된 충격 무기다.
이 장비가 허용하는 유일한 역학 지배식은 찰나의 델타 함수 F(t) \approx P \cdot \delta(t) 이다. 망치로 물체를 타격하는 1 밀리초(ms) 안팎의 가장 짧은 순간에, 강렬하게 솟구쳤다 꺼지는 힘 파동이 계측기를 통해 기록된다. 동시에 물체 표면에 붙은 가속도계(Accelerometer)가 망치를 맞고 흔들리며 잔향이 사그라지는 x(t) 스위프 감쇠 자유 진동 곡선을 녹음한다.

엔지니어는 충격력(입력)과 가속도(출력)를 푸리에 공간으로 쌍끌이 이송시킨 후 둘을 나눈 주파수 응답 함수(FRF, Frequency Response Function, H(\omega)) 모델을 조립한다.

H(\omega) = \frac{X(\omega)}{F(\omega)}

하지만 망치는 사람이 손으로 휘두르는 단발성 충격이므로 필연적으로 다음과 같은 두 가지 난망을 동반한다.

  • 충격 에너지 자체(P)가 제한적이어서 댐핑(Damping)이 엄청나게 큰 고무 블록이나 거대 교량 구조물을 끝까지 울리기는 턱없이 부족하다.
  • 이중 타격(Double Hit), 즉 망치가 한 번의 스윙에서 용수철처럼 퉁퉁 두 번 치게 되는 오염된 임펄스가 섞이면, \delta(t) 의 평탄한 주파수 스펙트럼이 짓이겨지며 FRF 해석망이 완전히 박살 나 버린다.

2. 무한 파동과 시스템 마인드 컨트롤: 가진기(Shaker) 역학

해머가 부족한 모든 것을 가진기(Shaker, 전자기 또는 기계식 진동 발생 모터)가 지배한다. 셰이커 볼트는 시험체의 중심을 단단히 물고 끝도 없이 진동 전력을 쏟아부어 구조물을 영원히 흔들어버린다.

셰이커는 망치처럼 델타 임펄스(\delta(t)) 하나로 허리케인을 만들지 않는다. 대신 조종사가 스로틀을 부드럽게 올리듯, 낮은 주파수부터 초고주파까지 계단식으로 사인 곡선의 진동수를 훑으며 올라가는 사인 스윕(Sine Sweep) 파동이나 모든 주파수를 섞어 무작위로 타격하는 랜덤 노이즈(Random Noise) 를 인가한다.

  1. 지속적 제어력: 막강한 파워를 쏟으며 아무리 거대하고 둔감한 비행기 날개라도 강제로 최대 공진 지점(Resonant Peak)까지 부풀려(Blow Up) 변형시킬 수 있다.
  2. 비선형성(Non-linearity) 발각: 에너지를 단계적으로 미분하며 올리다 보면, 처음엔 잘 맞던 선형 스프링-질량 모델이 갑자기 특정 강한 출력 주파수에서 펑 터지며 주저앉거나 엇나가는 거대 비선형성 결함(접합부 느슨함, 큰 균열)을 생포하는 진단 모니터로 변신한다.
