2.2.2.2.4 회전체 로터(Rotor)의 공진 회피를 위한 마찰-구조 접합부 쿨롱 감쇠기(Coulomb Damper)

제트 엔진의 엄청난 회전체, 터빈 블레이드(Turbine Blade)들이 \text{RPM} 만 단위를 돌파하며 회전할 때, 블레이드는 기체(Air) 파동의 끊임없는 공기 역학적 다중 타격(Multi-impact)과 후류를 맞는다. 가녀린 날개가 스스로의 고유 진동수 모드(Resonance Component)와 타격 주기가 겹치는 그 지옥과 같은 임계 회전수(Critical Speed)를 통과할 때, 진폭 한계를 찢어발기려 하는 블레이드의 터짐을 막아주는 유일한 구명줄이 원반 디스크와 블레이드 뿌리의 미세한 결합 공간 사이에 쑤셔 넣은 쿨롱 마찰 감쇠기(Coulomb Damper) 다.

이 기계적 접합부는 두 강철이 맞닿아 서로 비벼지는 가장 원시적이면서도 극한의 건조 마찰력(Dry Friction) 매커니즘을 동원하여 회전체의 진동 에너지를 어마어마한 마찰열 덩어리로 변환하고 대기 중으로 기화시켜 버린다.

1. 접합부 파팅 라인의 미세 슬립(Micro-slip) 역학

점성 댐퍼(Viscous Damper) 유체를 바를 수 없는 수백 도 이상의 고온 블레이드 틈새에는, 작은 강철 금속판이나 쐐기(Wedge Damper, Under-platform Damper)를 끼워 넣는다. 블레이드가 회전 원심력(Centrifugal Force)으로 바깥으로 강하게 쏠릴 때 원반 디스크와 블레이드 틈새 사이에서 이 댐퍼는 꽉 끼어 접촉면 수직 항력(F_n) 텐서를 구축한다.

자 이제, 블레이드가 진동 모드(Bending Mode)에 돌입하며 좌우로 치명적으로 흔들리기 시작한다.

  • 댐퍼 부품과 블레이드 사이의 틈새 접촉면에서 아주 미세한 마이크로 슬립(Micro-slip) 전이 및 쿨롱 마찰이 강제로 발현된다.
  • 변위 진폭 주기가 한 바퀴 돌 때마다, 접촉면은 서로 역방향으로 긁히면서 마찰력 F_f = \mu F_n 에 진동 변위 \Delta x 를 곱한 직사각형 면적 구조의 비가역적 에너지 소산 궤적(Hysteresis Loop) 을 그려낸다.

2. 쿨롱 감쇠의 임계 수직 항력(Optimal Normal Force) 튜닝점

쿨롱 감쇠는 속도 비례 점성 감쇠처럼 무조건 댐퍼 계수를 키운다고 좋아지는 선형계통이 아니다. 쿨롱 마찰을 유발하는 수직 항력 F_n 을 조절하는 역학 설계점은 극단적인 비선형성을 지닌 채 한 가지 뾰족한 구세주 포인트를 겨냥해야만 한다.

  • 수직 항력이 너무 약할 때(F_n \approx 0): 마차 바퀴처럼 심하게 헛돌며 긁히지만(슬립 변위 거대), 마찰 계면 응력 F_f 가 너무 작아 파괴력 소모 일 W = F_f \times \Delta x 값이 보잘것없는 수치로 증발해 버린다. 블레이드는 공진을 맞아 터진다.
  • 수직 항력이 무식하게 높을 때(F_n \rightarrow \infty): 댐퍼가 블레이드와 디스크 사이를 본드처럼 꽉 물고 완전 마찰 융착 상태(Sticking)에 돌입해 버린다! 두 부품이 슬립(Slip) 없이 하나의 덩어리로 굳어 움직이므로, 변위 \Delta x = 0 이 되어 이 역시 소산 에너지 W = 0 이다. 블레이드는 구속되어 더 뻣뻣해진 채 또 터진다.
  • 최적의 공학 한계 조율점(Optimal Slipping Force): 이 양극단을 피한 중간의 적절한 체결력 구역—마찰력(F_f)은 높게 유지하되 어떻게든 아슬아슬하게 미끄러짐(\Delta x \neq 0)을 유발해 주어 마찰력 \times 미끄럼 변위(W) 직사각형 소산 면적을 최고조 극댓값(Maximum Hysteresis Area) 으로 부풀리는 바로 그 스위트 스폿(Sweet Spot) 설계점이 쿨롱 공진 회피(Coulomb Resonance Escape) 다이어그램의 정점이다.

수천만 원짜리 단결정 터빈 블레이드 설계의 마지막 방어선은 정밀한 곡선의 초공기역학이 아니라, 날개 뿌리에 끼워진 값싼 웨지형 강철 틈새 장치의 삐걱거리는 마찰 끽끽거림(Micro-Stick-Slip Hysteresis)—이 지극히 아날로그적인 파괴 열에너지 환전 방식에 전폭적으로 기대고 있는 기계 공학의 고전적 역설(Paradox)을 보여준다.
