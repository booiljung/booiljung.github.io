2.2.2.2.8 충격 흡수 장치 내 비선형성(Non-linear) 점성 오일 댐퍼의 오리피스 유속 강하 방정식

자동차 바퀴가 포트홀(Pothole)이나 과속 방지턱을 사정없이 들이받을 때 시스템의 피로 누적과 탑승객의 멀미를 막아내는 영웅은, 타이어의 고무도, 용수철의 탄성력도 아닌 거대한 서스펜션(Suspension) 기둥 속에 갇힌 시커먼 유압 오일(Viscous Fluid)이다. 단순 교과서에서 F = c\dot{x} 라며 감쇠력을 단순히 속도에 비례하는 선형 상수로 대충 얼버무렸지만, 시속 100 km/h 로 방지턱을 강타하는 피스톤 실린더는 오일을 좁은 바늘구멍(오리피스, Orifice)으로 멱살 잡듯 쥐어짜 내며 유체 동역학의 지옥 같은 비선형성(Non-linear) 매트릭스를 난도질해 폭발시킨다.

진정한 유압 충격 댐퍼의 제어력은 선형 \dot{x} 에 있지 않고 난류(Turbulent Flow) 와 베르누이 차압(\Delta P) 에 갇힌 속도의 제곱항(\dot{x}^2)에 무겁게 묶여 있다.

1. 베르누이와 오리피스 관통 속도의 비선형 역학

피스톤(면적 A_p)이 충격을 받아 실린더를 속도 v = \dot{x} 로 무자비하게 압축한다고 상상하라.
안에 가득 차 있던 오일은 실린더가 부서지지 않으려면 피스톤 헤드에 뚫려 있는 아주 미세한 유로 구멍, 즉 오리피스(단면적 A_o)를 광속을 방불케 하는 유속 v_o 로 미친 듯이 비집고 탈출해야 한다.
유체의 비압축성 체적 유량 보존 법칙(Q = A_p \cdot v = A_o \cdot v_o)에 의해, 구멍을 통과하는 오일의 속도는 피스톤 압축 속도를 면적비(A_p / A_o)만큼 극단적으로 증폭시킨 고속 제트류(Jet Flow)가 된다.

이 엄청난 고속 난류 제트는 베르누이 방정식 에너지 차압(Pressure Drop) 지배하에 다음과 같은 반작용 압력차(\Delta P)를 실린더 헤드 상하에 무식하게 찍어 누른다.
\Delta P = \frac{1}{2} \rho v_o^2 = \frac{1}{2} \rho \left( \frac{A_p}{A_o} \right)^2 \dot{x}^2

2. 댐퍼의 전단 저항 복원력: 변이(Variation)를 용납하지 않는 이차식(v^2) 감쇠 폭주

이 차압 \Delta P 가 면적 A_p 인 피스톤 머리를 짓누르면, 엔지니어가 서스펜션을 조율할 때 목도하는 가장 끔찍하면서도 매력적인 비선형 댐핑 방어력 방정식이 입증된다.

F_{damping} = \Delta P \cdot A_p = \underbrace{ \left[ \frac{1}{2} \rho \frac{A_p^3}{A_o^2} C_d \right] }_{C_{\text{non-linear}}} \dot{x}^2

  • 속도의 제곱 비례(v^2): 타이어가 아주 천천히 흔들릴 때는 \dot{x}^2 가 사실상 0이 되어 오일은 오리피스를 쑥쑥 통과해 버리고 승차감을 물렁(Soft)하게 유지해 준다.
  • 충돌 타격 시 급속 잠김(Hydraulic Lock): 하지만 자동차가 웅덩이에 빠져 바퀴가 빛의 속도로 처박힐라치면(\dot{x} \uparrow), 댐퍼 방어력은 미친 듯이 폭주(\dot{x}^2)하여 마치 바위처럼 실린더가 단단하게 굳어버리며 차체가 주저앉는 것을 목숨 걸고 막아낸다.
