2.2.1.1.8 전하와 전자기장 모델에 유추 가능한 일반화된(Generalized) 역학계 퍼텐셜 보존 상수

고전 기계 역학자들은 오직 mgh 의 중력과 \frac{1}{2}kx^2 의 스프링만이 보존 에너지장의 지배를 받는 특수한 현상이라 착각해 왔다. 그러나 제임스 클러크 맥스웰(James Clerk Maxwell)의 손을 거친 전기 전자공학계는 기계 역학 방정식의 텐서 구조를 완벽하게 모방하는 쌍둥이 형제, 즉 쌍대성(Duality)을 지닌 전자기장 방정식을 토해냈다.
기계 역학의 질량(m)은 전자기학의 전하량(q)으로, 변위(x)는 전하의 궤적으로, 기계적 힘(F)은 전기력(F_e)으로 일대일(1:1) 치환 대응시켜라. 그러면 거대한 스칼라 산맥인 전위(Electric Potential), 즉 전압(Voltage) 의 실체가 기계 역학과 완벽히 동기화되어 강림한다.

1. 쿨롱의 법칙(Coulomb’s Law)과 보존 중심력장의 평행 우주

두 점전하 q_1, q_2 가 거리 r 만큼 떨어져 있을 때 발생하는 전기력은 만유인력과 정확히 같은 1/r^2 방사 대칭 중심력이다(다만 전하의 종류에 따라 인력과 척력이 갈릴 뿐이다).

\mathbf{F}_e = k_e \frac{q_1 q_2}{r^2} \mathbf{e}_r

이 쿨롱 벡터장 중심력 \mathbf{F}_e 역시 \infty (무한대)를 기점으로 적분하면, 기계적 보존 에너지 우물과 하나도 다르지 않은 정전기 퍼텐셜 에너지 U_e 스칼라 방정식을 도출한다.

U_e(r) = - \int_{\infty}^{r} \mathbf{F}_e \cdot d\mathbf{r} = k_e \frac{q_1 q_2}{r}

전기장(\mathbf{E} = \mathbf{F}_e/q)을 적분한 단위 전하당 퍼텐셜 에너지 스칼라 값, V = U_e/q 가 바로 전기 공학 전체를 지배하는 전압(전위) 다.
역학적 시스템에서 높이가 같으면 물이 흐르지 않듯, 전자기계에서 전위(전압) 차이가 0(\Delta V = 0)이면 전류라는 입자의 흐름(속도)은 만들어지지 않는다.

2. 라그랑지안(Lagrangian) 및 해밀토니안(Hamiltonian) 일반화 퍼텐셜

단순한 위치 좌표(q_i)에만 종속된 보존력(V(x))을 넘어서라.
자기장(Magnetic Field, \mathbf{B}) 공간에서 운동하는 전하의 로런츠 힘(Lorentz Force, \mathbf{F} = q(\mathbf{E} + \mathbf{v} \times \mathbf{B}))은 속도 \mathbf{v} 직교 성분에 의존하며, 더 이상 기계적 보존력 행렬식 \nabla \times \mathbf{F} = \mathbf{0} 에 종속되지 않는다.

하지만 비선형 동역학계 창시자 라그랑주(Lagrange)는 이 기계-전기 복합 시스템조차도 일반화된 퍼텐셜(Generalized Potential, U(q_i, \dot{q_i})) 의 범주로 흡수해 버림으로써 에너지 보존의 교의를 지켜냈다.

라그랑지안 방정식의 연산 텐서 안에 일반 퍼텐셜 U 를 다음과 같이 박아 넣는다.

F_i = -\frac{\partial U}{\partial q_i} + \frac{d}{dt} \left( \frac{\partial U}{\partial \dot{q}_i} \right)

일반화 퍼텐셜 U 는 단순 스칼라 에너지뿐만 아니라 벡터 퍼텐셜(Vector Potential, \mathbf{A}) 속성까지 통합된 광의의 식이다 (U = qV - q\mathbf{v}\cdot\mathbf{A}).

기계 엔지니어가 모터나 발전기 설계를 수학적으로 뚫어내야 할 때, 로터(Rotor)의 기계 다이나믹스(J\ddot{\theta} + b\dot{\theta}) 행렬과 코일(Coil)의 전자기 상태 방정식(L\dot{i} + Ri = V) 행렬을 따로 떼어놓고 시뮬레이션하는 것은 무지다.
두 방정식은 동일한 쌍대성(Duality)을 지닌 에너지 교환 스칼라 함수의 한 몸통에서 뿜어져 나오는 라그랑지안 미분방정식 형제들이기 때문이다!
이 초월적(Generalized) 에너지 변환 체계를 시스템 역학(System Dynamics) 행렬망에 하나로 올려놓지 못한다면 현대 전기 자동차 파워트레인의 최적화 제어에 결코 도달할 수 없다.
