2.2.1.1.4 3차원 공간 내 물체의 등퍼텐셜 면(Equipotential Surface)과 힘 벡터의 직교성 방정식

눈에 보이지 않는 거대한 스칼라 우물, 퍼텐셜 에너지장 V(x,y,z) 의 수학적 구조를 기하학의 도화지 위로 끌어올린다면 도대체 어떤 모형이 나타날 것인가?

어떤 3차원 공간 내에 퍼텐셜 에너지 스칼라 값이 모조리 일정하게 같은 상수(Constant, C)를 갖는 점들의 자취를 모아 연결하면, 그것은 매끄러운 2차원의 곡면(Surface) 계층들로 공간을 빼곡히 채우게 된다. 양파의 껍질처럼 공간을 감싸는 이 곡률 에너지층이 바로 등퍼텐셜 면(Equipotential Surface) 이다. 즉, V(x,y,z) = C_1, C_2, C_3 \dots 를 만족하는 무한개의 이 면적들 사이에서, 이 층과 층 사이를 아우르는 보수적인 힘 벡터 \mathbf{F} 는 기하학적 수직 타격을 행하는 사신이다.

1. 퍼텐셜 공간의 전미분(dV) 소멸과 등퍼텐셜 슬라이딩 운동

한 줄기 등퍼텐셜 곡면 V(x,y,z) = C 위를 어떤 입자가 임의의 변위(Displacement) 벡터 d\mathbf{r} = dx\,\mathbf{i} + dy\,\mathbf{j} + dz\,\mathbf{k} 를 그리며 미끄러지듯 유영하는 상태를 가정하라.
표면 위를 따라가는 동안, 에너지 스칼라는 전혀 높아지지도 낮아지지도 않고 우주적으로 일정하므로 이 퍼텐셜의 전미분(Total Differential) 스칼라 변화율은 수학적으로 정확히 영(0)이 되어야 한다.

dV = \frac{\partial V}{\partial x} dx + \frac{\partial V}{\partial y} dy + \frac{\partial V}{\partial z} dz = 0

그래디언트 연산 \nabla V 의 편의를 빌려 이 식을 재정립하면,

dV = (\nabla V) \cdot (d\mathbf{r}) = 0

이 된다. 여기서 보존력의 정의방정식 \mathbf{F} = -\nabla V 를 내리치면 결론은 다음과 같다.

-\mathbf{F} \cdot d\mathbf{r} = 0

등퍼텐셜 표면상에서 움직인 어떤 변위 궤적(d\mathbf{r})과도 그 보존력 벡터 \mathbf{F} 는 정확히 내적(Dot Product) 값이 영(0) 이다.
내적이 0이라는 것은 두 벡터가 기하학적으로 오차 없는 90^\circ 직교(Orthogonality) 상태에 있음을, 즉 어긋남 없는 공간 교차망임을 뜻한다.

따라서 보존력 벡터 \mathbf{F} 는 어떠한 굽은 퍼텐셜 면 표면일지라도 예외 없이 그 표면에 완벽하게 꽂히는 법선(Normal Vector) 방향으로 직각의 에너지를 폭발시키며 인력망/원심 역장을 뿜어낸다.

  • 지형학(Tomography): 등고선 지도의 선(등퍼텐셜) 위를 걷는 인간은 고상한 에너지를 쓰지 않는다. 경사가 제일 가파르게 내리꽂는 법선 방향인 언덕을 내려갈 때(기울기), 거대한 중력 가속 \mathbf{F}_g 역학이 최대로 방출된다.
  • 전자기 대전 도체(Electrostatics): 정지한 평형 도체 표면상의 모든 금속 원자는 동일한 자유 전자 등퍼텐셜 전압 상태에 있다. 그러므로 금속 도체 외부로 뿜어져 나오는 유도 전기력선(Electric Field Lines) \mathbf{E} 는 어떠한 굽은 도체 형상(뾰족한 바늘이나 둥근 구면)일지라도 그 표면을 수직으로 뚫고 90^\circ 로 오차 없이 오만한 각도로 치솟아야만 한다.

어설픈 수치해석으로 점전하 근처의 전계 메시(Mesh)를 그릴 때 곡면과 각도가 90^\circ 가 아닌 굴절된 힘 벡터가 산출되었다면, 그 적분 경계가 완전히 파탄 난 것이다. 이 직교 방정식망을 반드시 추적 필터 센서로 감시하라.
