2.2.1.1 경로 독립성 보존력(Conservative Force)과 퍼텐셜 에너지장

역학적으로 완벽한 물레방아가 있다 하자. 한 번 물이 떨어져 임펠러를 돌리면 그 물이 다시 꼭대기로 역류하여 재순환할 수 있는가? 불가능하다. 자연계의 대다수 흐름은 비가역적이며 에너지를 잃는다.

하지만 물리학의 절대 방정식 세계에서, 중력(mg)이나 탄성력(-kx), 정전기력(kq_1q_2/r^2) 같은 특권적인 힘들만이 특별한 역학적 장부 결제를 허락받았다. 이 숭고한 백터장 내에서 움직이는 질점에 한 일(Work)은 궤적이 얼마나 복잡하게 비틀어져 춤을 추었는가, 즉 경로(Path)에 전혀 구애받지 않으며 오직 처음 출발점과 최후 종착점의 좌표만으로 적분 값이 칼로 무 베듯 결정된다.

이것이 바로 상태 함수로서의 보존력(Conservative Force) 이다.
경로 독립성(Path-independence)이라는 이 엄청난 우대 조건은, 보존력이 행한 모든 일이 단 1줄의 소실 없이 보이지 않는 가상의 위치 창고, 즉 퍼텐셜 에너지(Potential Energy, V) 의 우물 속에 환전되어 저장됨을 의미한다. 만약 물체가 다시 원래 위치로 귀환하는 폐곡선 환상 궤도를 순환해 돌아왔다면(Round Trip), 그동안 보존력이 한 일의 총합은 완벽한 영(0)이 되어야 한다. 이 장에서는 3차원 공간장 위에 펼쳐지는 보존력의 벡터적 회전(Curl) 소멸 물리와 극소-극대점 곡면의 퍼텐셜 웅덩이를 통해 무차원적 동역학 안정성을 가려내는 기준을 도출한다.
