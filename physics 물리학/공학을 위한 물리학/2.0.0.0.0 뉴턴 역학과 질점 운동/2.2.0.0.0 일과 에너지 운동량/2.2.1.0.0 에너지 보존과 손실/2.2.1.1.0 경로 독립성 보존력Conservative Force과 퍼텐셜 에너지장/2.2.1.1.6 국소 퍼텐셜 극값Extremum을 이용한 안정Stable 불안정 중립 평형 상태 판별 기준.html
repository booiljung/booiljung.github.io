2.2.1.1.6 국소 퍼텐셜 극값(Extremum)을 이용한 안정(Stable), 불안정, 중립 평형 상태 판별 기준

역학적 시스템이 멈추어 섰다. 시스템의 가속도 \ddot{x} 도 영(0)이고, 속도 \dot{x} 도 영(0)이다. 엔지니어는 이 고요한 상태를 평형점(Equilibrium Point) 이라 부른다. 그러나 평형이라고 해서 다 같은 평형이 아니다. 거대한 댐 위에 올려진 1톤짜리 돌덩이도 멈춰있고, 깊은 계곡 바닥에 가라앉은 바윗돌도 멈춰있다. 미세한 지진파(Perturbation)가 그들을 때렸을 때 댐 위의 바위는 수만 명의 목숨을 앗아가는 파멸(Catastrophe)로 추락하고, 계곡 바닥의 바위는 귀찮은 듯 살짝 흔들리다 다시 원래 위치로 안착한다.

그렇다면 복잡한 고차원 연립 미분 방정식을 처음부터 다시 풀지 않고, 오직 에너지 스칼라 우물 표면의 기울기 파라미터만으로 이 평형의 본질적 운명을 어떻게 진단(Diagnostic)할 수 있는가? 바로 일계 미분(기울기)과 이계 미분(곡률)의 기하학적 심판, 즉 퍼텐셜 에너지 극값(Extremum) 해석법이 그 해답이다.

1. 퍼텐셜 평형의 필요 충분 수학망: 구배 dV/dx = 0

1차원 공간상에서 퍼텐셜 에너지 스칼라 함수를 V(x) 라 할 때, 이 위치에 있는 입자가 보존력 \mathbf{F} = -dV/dx 로부터 받는 힘이 정확히 0 이 되려면, 미분 접선의 기울기가 수평(Level)이 되어야만 한다.

F(x_{eq}) = -\left( \frac{dV}{dx} \right)_{x=x_{eq}} = 0

시스템이 멈춰있다는 것은 V(x) 함수 지형도에서 그 위치(x_{eq})가 산의 가장 날카로운 꼭대기이거나, 계곡의 가장 깊은 바닥이거나, 아니면 한없이 평평한 대지의 한가운데에 서 있다는 것을 뜻한다. 이것이 뉴턴 제1법칙 정역학 평형(Statics)의 대수학적 필요조건이다.

2. 2차 미분 텐서의 심판: 안정성의 3가지 부호 판별법

기울기(1차 미분)가 0 이 된 상태에서, 그 점이 어떤 기하학적 곡률(Curvature) 형태를 띠고 있는지 판독하기 위해 이계 미분 연산자(d^2V/dx^2) 를 x_{eq} 에 발동시켜라. 이 곡률의 부호(Sign)가 시스템의 영원한 안정성 운명을 칼날처럼 찍어낸다.

K_{\text{eff}} = \left( \frac{d^2 V}{dx^2} \right)_{X_{eq}}

이 성분은 해당 극소 평형점 인근에서의 시스템 구조적 유효 강성(Effective Stiffness, K_{\text{eff}}) 스프링 상수와 완벽히 수학적으로 동치된다.

  1. 안정 평형(Stable Equilibrium): d^2V/dx^2 > 0
    미분 곡률이 양수(+), 즉 아래로 볼록한 U자형 계곡 바닥이다. 입자를 오른쪽으로 미세하게 밀면(\Delta x > 0) 1차 미분 기울기가 양수가 되어 힘 \mathbf{F} = -dV/dx 는 다시 왼쪽(-)으로 미는 복원력(Restoring Force)을 발휘한다. 시스템 유효 강성이 양수(+)이므로 외란을 맞으면 원래 자리로 되돌아가고자 안정 한계 고리(Limit Cycle) 주위를 맴돌며 영원한 조화 진동(Harmonic Oscillation)을 행한다. (예: 진동하는 추, 컵 속의 구슬)

  2. 불안정 평형(Unstable Equilibrium): d^2V/dx^2 < 0
    미분 곡률이 음수(-), 즉 위로 볼록한 뒤집힌 밥그릇의 꼭대기(산봉우리)다. 나비가 앉는 아주 작은 진동 오차(dx)만 가해져도, 입자가 내려가는 쪽의 기울기가 더욱 음수로 곤두박질치며 시스템은 \mathbf{F} 라는 거대한 가속 폭주의 파멸 에너지(+쪽으로 밀림) 강제 이탈을 겪는다. 유효 강성이 음수(-)인 음강성(Negative Stiffness) 궤적을 뿜어낸다. (예: 손가락 위에 거꾸로 세워진 연필, 임계 좌굴 하중이 넘어선 기둥)

  3. 중립 평형(Neutral Equilibrium): d^2V/dx^2 = 0
    미분 곡률 자체가 0, 즉 끝없이 평평한 유리판 위(수평대)의 질점이다. 밀면 밀린 그 자리에서 다시 힘이 0 이 되어 정지해버린다. 복원력도, 파괴력도 존재하지 않는 우주의 망각 상태다. (예: 당구대 위의 정지한 공)

우주 궤도의 역학이나 대형 잠수함의 수중 부력 밸런싱 설계망에서 평형 위치의 해(Root)를 찾는 것은 빙산의 일각이다. 엔지니어는 시스템 상태 공간의 전체 퍼텐셜 헤시안(Hessian Matrix) 2차 미분 행렬을 스윕(Sweep)하여, 행렬의 고유치(Eigenvalues)가 양의 정부호(Positive Definite)인지 철저하게 심판해야만 21세기 기계 문명의 카오틱 붕괴를 예방할 수 있다.
