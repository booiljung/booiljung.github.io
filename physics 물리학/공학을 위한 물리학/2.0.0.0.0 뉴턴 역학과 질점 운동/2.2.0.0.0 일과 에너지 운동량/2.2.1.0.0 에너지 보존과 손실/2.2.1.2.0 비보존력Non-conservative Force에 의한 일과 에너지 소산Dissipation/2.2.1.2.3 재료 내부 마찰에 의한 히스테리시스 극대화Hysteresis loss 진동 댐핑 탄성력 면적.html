2.2.1.2.3 재료 내부 마찰에 의한 히스테리시스 극대화(Hysteresis loss) 진동 댐핑 탄성력 면적

교량, 고층 철골 빌딩, 엔진 블록과 같이 단단한 고체 덩어리들은 겉보기에 유체 점성이나 접촉면 쿨롱 마찰과 무관해 보인다. 그러나 구조물에 엄청난 진폭의 공진 주기(Resonance)가 주입되었을 때 가속의 폭주를 막고 시스템을 살려내는 최후의 보루는 놀랍게도 재료 자체의 미시적 원자 격자 틈새에 숨어있는 내부 마찰(Internal Friction) 이다.

금속이나 고분자 폴리머와 같은 고체 물질을 압축했다가 원래 길이로 다시 늘려놓을 때(하중-제하 사이클, Loading-Unloading Cycle), 압축하며 밀어 넣었던 거시적 역학적 일(Work)은 늘어날 때 100% 반환되지 못한다. 철강 내부의 금속 결정 입계(Grain Boundary)들이 서로 미끄러지고 부딪히며 소산되는 이 내부 열역학의 정수를 히스테리시스 댐핑(Hysteresis Damping) 혹은 구조 감쇠(Structural Damping)라 칭한다.

1. 이력 곡선(Hysteresis Loop)과 폐곡선 적분 내부의 면적 손실

시간 t 구간 1주기(\tau = 2\pi/\omega) 진동 동안, 임의의 재료 막대에 가해진 복원력 F(x) 에 의한 에너지를 추적하라. 변위 x 를 가로축, 복원력 F 를 세로축으로 플로팅(Plotting)했을 때, 인장할 때의 경로 그래프 곡선과 압축될 때 궤적을 돌아오는 복귀 곡선이 결코 겹치지 않고 기괴하게 벌어지며 하나의 타원형 풍선, 닫힌 낭(Loop)을 형성하게 된다.

이 현상은 변형률(Strain)이 응력(Stress)의 변화 속도를 따라가지 못하고 미세하게 뒤처지는 위상 지연(Phase Lag)의 사생아다.

\Delta W_{\text{dissipate}} = \oint F_{hysteresis} \, dx

이 1주기 폐곡선 타원 \oint 의 텅 빈 면적(Area) 크기가 바로 마찰 점성 때문에 1회 왕복 진동 동안 소멸하여 열로 타버린(소산된) 에너지의 정직한 결재 영수증이다.
그리고 놀랍게도 이 히스테리시스 타원 소실 면적 \Delta W 는 외부 강제 주파수 속도(\omega)에는 거의 무관하고, 오로지 이 오븐을 얼마나 강력하게 구부렸는지에 대한 진폭의 제곱(X^2) 에만 극단적으로 종속된다(\Delta W \approx \pi h X^2).

2. 복합 재료 설계망의 히스테리시스 극대화(Damping Maximization)

강철(Steel)이나 알루미늄이나 티타늄의 히스테리시스 면적, 즉 내부 소산 역량 손실 계수(\eta)는 고작 0.001 \sim 0.005 에 불과하다. 엄청난 탄성으로 복원 보존은 탁월하지만, 공진 에너지를 흡수하고 태워버릴 마찰 점성 능력은 처참하다. 비행기가 난기류를 만날 경우 강철만으로는 한계 고리 진동을 멈추고 제동을 걸 수 없다.

수많은 우주 발사체나 지진 대비 빌딩 구조망에는 철골 구조물의 접합부(Joint) 사이에 이 이력 타원 면적 텐서가 기형적으로 거대한 뚱뚱한 특수 고무, 즉 고감쇠 고무 베어링(High Damping Rubber Bearing, HDRB) 이나 점탄성 감쇠기(Viscoelastic Damper)를 끼워 넣는다.
엔지니어의 목적은 결코 구조물을 부서지지 않게 ‘견고히’ 만드는 데서 그치지 않는다. 지진 에너지가 전달된 순간, 이 폴리머 구조의 분자 사슬체계가 서로 맹렬히 얽히고 비벼대어(히스테리시스 마찰), 빌딩이 가속의 한계 \ddot{x} 에 달해 부러지기 전에 기계적 에너지를 열로 신속하게 태워 밖으로 대량 폐기(Dissipate)하는 소산(Loss) 필터망 자체를 기획하는 것이다.
