2.2.1.2.5 초음속 스킨 프릭션(Skin friction)에 의한 항공 우주선의 공력 가열(Aerodynamic heating)

유성(Meteor)이 대기권으로 돌진할 때 빚어내는 무려 수천 도 단위의 폭압적 불꽃(Fireball)이나 극초음속 스크램제트(Scramjet) 비행체가 마하 5(M \ge 5)를 뚫을 때, 기체는 더 이상 압력 형상에 의한 뒤쪽 부압 견인력 항력(C_d)에만 시달리지 않는다. 마하의 벽 너머에서 항공 우주 시스템을 짓뭉개는 사신은, 유체의 점성 분자층이 기체 표면에 직격으로 마찰(Rubbing)하면서 막대한 운동 에너지를 순박한 열 에너지로 100% 강제 환전시키는 스킨 프릭션(Skin Friction, 표면 점성 마찰) 이다.

차디찬 성층권 대기(섭씨 영하 50도)라 할지라도 기체가 엄청난 극도로 빠른 상대 속도(V_{\infty})를 가지면, 물체 최전선의 정체점(Stagnation Point)에서 돌진해 오던 공기 입자들의 이동 속도는 물체 표면에 부딪히며 순식간에 영(0)으로 구속(No-slip Condition)된다. 이 \Delta V 역학 에너지 폭수 손실 분량 전체가 시스템의 경계층(Boundary Layer) 내부의 절대 온도 상승으로 꽉 차오르는 현상, 바로 공력 가열(Aerodynamic Heating) 의 메커니즘이다.

1. 정체 온도(Stagnation Temperature) 에너지 방정식 변환

비점성, 비전도 압축성 유동 모델인 완전 기체(Ideal Gas) 상태에서 엔탈피(h)와 운동 에너지 합의 스칼라 보존식을 상기하라.
멀리 떨어져 오고 있는 자유 흐름 공기(T_{\infty}, V_{\infty})가 기체 최전선에 부딪혀 운동 에너지가 정지(속도 V=0)되면, 이 손실된 에너지가 몽땅 정적분압 열에너지로 흡인된 피크(Peak) 온도인 정체 온도(T_0) 텐서식을 빚어낸다.

h_0 = h_{\infty} + \frac{V_{\infty}^2}{2}
C_p T_0 = C_p T_{\infty} + \frac{V_{\infty}^2}{2}

양변을 열용량(C_p)으로 나누고 음속(a)과 마하수(M = V_{\infty}/a)를 밀어 넣어라. 이상 기체 속에서 표면에서 버티고 서서 견뎌내야 하는 정체 한계 온도는 마하의 ‘제곱’(M^2) 승수로 비례 폭증한다.

T_0 = T_{\infty} \left( 1 + \frac{\gamma - 1}{2} M^2 \right)

차디찬 대기권 영하 50^\circ C(약 223K) 지점을 속도 마하 1(M=1, 정체 온도 267K), 마하 3(M=3, 정체 온도 624K)으로 뚫는 비행체의 코앞은 무려 섭씨 350^\circ C 의 아궁이에 휩싸인다. 마하 7(M=7) 궤도를 넘어가면 정체 온도는 2400K(약 2100^\circ C)를 돌파해 티타늄조차 엿가락처럼 녹아내리는 절망적인 가열 파괴 임계점(Thermal Catastrophe)을 선언한다.

2. 공학적 에너지 소산 절연(Insulation) 스킨 프로텍션 방어 가이드

기계적 에너지가 보존 없이 열 에너지(U_{\text{int}})로 쏟아져 내린다는 것은 시스템의 열역학적 완전한 죽음을 의미한다.
엔지니어는 이렇게 수직 상승한 텐서 에너지가 우주선 탑승자와 페이로드의 피부 안쪽으로 침투하는 열 흐름 전도(\dot{q} = -k \nabla T)를 차단할 극소 열 저항 장벽 재료를 뒤집어씌워야 한다.

Space Shuttle이 지구로 궤도 재진입(Re-entry)할 때 마하 25의 에너지를 받아내는 하단부는 실리카 세라믹 계열의 가벼우면서도, 기화(Sublimation)하며 열을 대량으로 버리는 어블레이터(Ablator, 소모성 삭마재) 방열 타일이 덕지덕지 수천 개씩 코팅되어 있다.
역학 에너지가 소산(Dissipation)될 때 발생하는 이 무자비한 고열로부터, 어떻게 기체의 골격 구조(Structural Integerity)를 지킬 것인가는, 우주 비행 열역학의 최전방 절대 명제다. 뾰족한 로켓 코(Nose Cone)는 항력은 적지만 이 거대한 열 소산량을 뾰족한 맨 앞 점 한 곳(정체점)에 미치도록 응축시켜 녹여버리기에 뭉툭한 기하학 단면 체적 열 분산망(Blunt Body) 설계로 회귀해야만 했다.
