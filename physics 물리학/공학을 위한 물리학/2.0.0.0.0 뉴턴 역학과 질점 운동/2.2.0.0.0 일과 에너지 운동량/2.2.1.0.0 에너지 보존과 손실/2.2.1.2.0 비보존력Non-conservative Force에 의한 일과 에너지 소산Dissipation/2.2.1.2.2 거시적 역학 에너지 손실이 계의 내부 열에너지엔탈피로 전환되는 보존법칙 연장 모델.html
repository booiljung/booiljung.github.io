2.2.1.2.2 거시적 역학 에너지 손실이 계의 내부 열에너지(엔탈피)로 전환되는 보존법칙 연장 모델

기계공학의 좁은 시야에서는 쿨롱 마찰과 점성 저항성 소산이 에너지를 ‘도둑맞은’ 파괴적인 손실(Loss)로 인식된다. 즉 초깃값 점 A에서 운동 에너지(K_1)와 퍼텐셜 에너지(V_1)의 합이 다음 도착점 B에서 보존되지 못하고 작아지는 현상을 다음과 같이 서술한다.

K_1 + V_1 + W_{\text{nc}} = K_2 + V_2

여기서 W_{\text{nc}} = \int \mathbf{f}_k \cdot d\mathbf{r} 는 비보존력(Non-conservative Force)이 수행한 음(-)의 일이며, 기계적 에너지(E = K+V) 총량의 강제적인 감가를 뜻한다 (E_1 > E_2).
하지만 물리학의 그랜드 유니파이드(Grand Unified) 거시계에서는 창조되거나 소멸하는 에너지는 없다. 도둑맞은 기계적 에너지는 어디로 탈출한 것인가? 바로 시스템 내부를 구성하는 분자들의 무질서한 미시적 열 운동 에너지(Internal Thermal Energy) 로 숨어든 것이다.

1. 열역학 제1법칙의 편입: 기계 에너지와 내부 에너지(U)의 융합

거시적 뉴턴 역학이 열역학의 우주로 확장하는 순간, 기계적 방정식의 잔차(Residual)는 정확히 열과 일이라는 에너지 교환 텐서의 정산표로 귀결된다.
비보존력 마찰(f_k) 시스템을 통과하며 거시계에서 삭제된 기계적 에너지는 분자 간의 무자비한 충돌과 접촉 경계면의 전단 소성 파괴 마찰을 연료로 삼아, 물체와 바닥 면의 절대 온도(T)를 폭증시키는 계의 내부 에너지 증가율(\Delta U_{\text{int}})로 100\% 완벽하게 환전된다.

\Delta K + \Delta V - W_{\text{other}} = Q - \Delta U_{\text{int}}

또는, 시스템 외부와의 열 교환이 완벽히 차단된 단열(Adiabatic, Q=0) 마찰 블록계를 정의한다면, 마찰이 깎아먹은 기계적 일(-W_{\text{nc}})은 양의 내부 에너지 상승(+\Delta U_{\text{int}})으로 완벽하게 보상 치환된다.

E_{\text{mechanical, initial}} = E_{\text{mechanical, final}} + \Delta U_{\text{int}}

3. 공학적 융합 설계: 에너지 소산 한계의 능동 회수(Active Recovery)

자동차 브레이크 패드가 1500 kg 의 금속 덩어리를 정지시킬 때, \frac{1}{2}mv^2 이라는 메가줄(Megajoule) 단위의 폭력적인 기계적 에너지는 단순히 허공으로 증발하는 것이 아니다. 두 조각의 카본 세라믹 디스크 브레이크의 내부 에너지(U_{\text{int}})로 집중 투하되어 섭씨 800 도 이상의 플라스마 붕괴에 가까운 적열 상태 온도로 치솟아 오른다(Thermal Fading 현상 유발).

만약 현대의 기계공학이 이 비보존 시스템의 파멸을 그저 ’음(-)의 일’이라는 방정식의 구멍으로 방치했다면 전기 자동차의 미래는 없었을 것이다. 하이브리드 전기 제동 망은 브레이크 패드의 쿨롱 마찰을 배제하고 파라데이(Faraday) 전자기 유도를 엑추에이터 위상에 걸어 역기전력 발전기(Regenerative Braking)로 스위칭한다!

즉 운동 에너지(\Delta K)를 열(\Delta U_{\text{int}})로 태워버리는 소모성 마찰 궤적을 차단하고, 배터리의 전기 화학적 퍼텐셜 공간(Chemical Potential \Delta V_c) 속으로 구겨 넣는(Storing) 초월적 우회 보존망을 덮어씌웠다. 텐서를 열로 누수시키는 것이 아니라 계의 또 다른 보존 창고로 돌려막아 회수하는 연장 모델こそが, 비선형 동역학과 열역학을 엮어내는 21세기 메카트로닉스 역학의 궁극이다.
