2.2.1.2.0 비보존력(Non-conservative Force)에 의한 일과 에너지 소산(Dissipation)

영구 기관(Perpetual Motion Machine)을 꿈꾸는 자들은 보존력의 완벽한 탄성 환전에 매료된다. 그러나 현실의 공학 시스템은 그 거대한 에너지 궤환 고리를 파고드는 우주의 징수관, 바로 비보존력(Non-conservative Force) 에 의해 점진적으로 피를 흘린다.

수식을 위해 이상화된 마찰 없는 책상이나 진공 속의 진자는 존재하지 않는다. 모든 롤러 베어링 구슬은 쿨롱 마찰(Coulomb Friction)에 의해 깎이고, 모든 압축 유압 실린더는 오일의 점성(Viscous Damping)과 와류에 휩싸여 에너지를 강제로 탈취당한다. 이 비보존력들은 경로의 앞뒤를 구별하며, 어느 길을 거치든 그들이 한 일의 총량은 언제나 시스템의 기계적 에너지를 갉아먹는 음의 일(Negative Work, W_{nc} < 0) 로 수렴한다.

결국 시스템에서 사라진 속도 에너지와 높이의 스칼라 에너지는 파괴되어 사라진 것이 아니다. 우주의 열역학 제1법칙이라는 절대 장부 속으로 편입되어, 결코 되돌릴 수 없는(Irreversible) 거시적 비가역 형태인 ‘열(Heat)’ 에너지, 즉 소산(Dissipation) 구역으로 밀려난 것일 뿐이다.
본 챕터에서는 기계 공학 설계의 핵심, 비보존력들에 의한 에너지 소실 매커니즘을 적분망으로 묶어 시스템 붕괴의 진행 속도를 엄밀히 계측해 낸다.
