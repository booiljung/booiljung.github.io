2.2.1.2.1 경로에 종속적인(Path-dependent) 쿨롱 평면 마찰과 점성 저항의 음의 일(Negative Work)

우주의 보존 법칙 \nabla \times \mathbf{F} = \mathbf{0} 제약을 받지 않는 비보존력들은 그야말로 무법자다. 똑같이 점 A에서 점 B로 박스를 끌어 옮기는 작업을 하더라도, 똑바로 직선으로 끌고 최단 거리로 간 경우와 넓은 운동장을 지그재그로 빙빙 돌다 도착한 경우의 에너지 환산 영수증 청구액은 하늘과 땅 차이다. 이것이 마찰력이나 공기 저항 같은 경로 종속적 제동력(Path-dependent Retarding Force) 의 진정한 본질이다.

1. 쿨롱 마찰(Coulomb Friction): 접촉 궤적의 적분 폭발

엔지니어가 무거운 철판을 바닥(\mu_k)에서 밀어 옮길 때 작용하는 동마찰력(Kinetic Friction)은 단순하다. 항상 철판이 미끄러지는 속도 벡터 단위 방향(\mathbf{u}_t)의 정반대(-) 방향을 향하는 크기 f_k = \mu_k N 의 역-텐서력이다.
마찰력이 시스템에 뿜어낸 일(Work)을 수학적으로 엄밀하게 평가하라.

W_f = \int_{Path} \mathbf{f}_k \cdot d\mathbf{r} = \int_{Path} (-\mu_k N \mathbf{u}_t) \cdot (ds\, \mathbf{u}_t) = - \int_{s_A}^{s_B} \mu_k N \, ds

선행하던 벡터 방향 내적은 그 즉시 -1 로 사라지고, 오직 남은 것은 시스템이 미끄러지고 부벼댄 기하학적 전체 곡면 이동 거리, 윤곽선 총 길이(Total Arc Length, \Delta s)와의 곱뿐이다.
직선 최단 거리라면 W_f = -\mu_k N (\Delta x) 이지만, 만약 \Delta x 좌표에 이미 안착해있는 시스템을 엔지니어가 변덕을 부려 원래 위치로 끌고 돌아왔다면(폐곡선 궤적)?
에너지는 환급되지 않는다! 복귀 시에도 마찰력은 방향을 또다시 등 뒤 역방향으로 틀기 때문에 일의 누적 총합은 0 이 되지 않고 무려 두 배(-2\mu_k N \Delta x)의 파괴적 에너지 유출을 달성한다. 이처럼 마찰 적분은 결코 부호를 되돌릴 수 없는 영원한 음의 일(Negative Work, W_f < 0) 로 수렴한다.

2. 점성 저항(Viscous Drag): 속도 파라미터가 직조하는 비선형 소산

공기 역학이나 유압 실린더 베어링 내부를 통과할 때, 마찰력은 수직 항력(N)과 별 상관이 구속력을 끊어버리고, 다가오는 속도 그 자체에 비례하는 괴물로 변모한다(\mathbf{F}_v = -c\mathbf{v}).
이 점성 저항력이 거대한 질량계로부터 뜯어내는 에너지 양은 단순한 거리가 아닌 시간-속도 미분 지배의 지배를 받는 경로의 최단 거리 질주 방정식으로 환산된다.

W_v = \int \mathbf{F}_v \cdot d\mathbf{r} = \int (-c\mathbf{v}) \cdot (\mathbf{v} dt) = -c \int v^2 dt

이 일(Work) 역시 무조건적인 음수(-c \int v^2 dt < 0)이다. 유체 속을 헤엄쳐 경로 1로 느리게 가든, 경로 2로 포탄처럼 질주하든 이 저항에 소진된 에너지는 속도 프로파일의 제곱 비율에 따라 무자비하게 달라진다.
