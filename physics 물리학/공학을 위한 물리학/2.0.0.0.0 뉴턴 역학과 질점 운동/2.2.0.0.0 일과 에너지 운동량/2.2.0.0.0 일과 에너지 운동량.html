2.2 일과 에너지, 운동량

지금까지 우리는 힘(Force)과 가속도(Acceleration)라는 벡터 방정식을 통해 공간을 누비는 질점의 순간적인 움직임을 집요하게 추적해 왔다. 뉴턴의 운동 방정식 \sum F = ma 는 입자의 현재 우주적 궤적을 서술하는 가장 직관적인 도구이다. 그러나 공학의 시스템 스케일이 거대해지고 수만 개의 파라미터가 얽히는 복잡계로 진입할 때, 엔지니어는 매 순간 변하는 벡터(Vector)의 늪에 빠져 수치 해석적 재앙을 맞이하게 된다.

이러한 벡터 추적의 한계를 타파하고 시스템의 시작과 끝, 원인과 결과만을 관통하는 스칼라(Scalar) 역학 체계로의 도약이 필요하다.
이로써 우리는 복잡한 경로를 묻지 않고 오직 상태(State) 간의 변환만을 추적하는 일(Work)과 에너지(Energy) 의 교의를 확립한다. 또한 질량과 속도의 기하학적 결합체이자 강력한 충돌 역학을 지배하는 운동량(Momentum) 보존의 절대 규칙을 제정한다.

이 챕터는 에너지가 보존력에 의해 잠재적으로 축적되고 비보존력에 의해 무자비하게 소산되는 모든 역학적 교환 비율을 명확히 규명한다. 시스템 내에서 발생하는 충돌의 반발 계수와 에너지의 이합집산을 방정식화하여, 엔지니어로 하여금 거시적 관점에서 파괴와 수렴을 지배하도록 유도할 것이다.
