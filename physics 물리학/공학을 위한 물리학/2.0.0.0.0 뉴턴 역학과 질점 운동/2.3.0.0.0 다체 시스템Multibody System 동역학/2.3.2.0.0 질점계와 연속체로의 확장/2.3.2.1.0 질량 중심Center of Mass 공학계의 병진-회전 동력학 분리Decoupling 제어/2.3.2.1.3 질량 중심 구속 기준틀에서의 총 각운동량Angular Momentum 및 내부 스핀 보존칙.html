2.3.2.1.3 질량 중심 구속 기준틀에서의 총 각운동량(Angular Momentum) 및 내부 스핀 보존칙

다체 시스템(Multibody System)이 공간을 병진(Translate)하는 동안, 시스템의 파편들이 제각각 중심점에 얽혀 회전하는 복잡한 “스핀(Spin)” 상태를 분석해 보자. 수많은 팽이가 우주에서 한 덩어리로 묶여 날아가고 있을 때, 제멋대로 돌고 있는 개별 입자들의 각운동량(Angular Momentum)은 어떻게 단 하나의 거시 방정식으로 통합될 수 있는가?
해답은 시스템을 ‘고정된 절대 우주 원점(O)’ 에서 바라보는 시선과, ‘날아가는 질량 중심(CM)’ 위에서 파편들을 관측하는 시선, 이 두 텐서 레이어를 완벽하게 직교 분해(Orthogonal Decomposition)하는 데서 쏟아진다.

1. 우주 좌표계(Inertial Frame) 총 각운동량의 순수 분리: \mathbf{H}_{total} = \mathbf{H}_{궤도} + \mathbf{H}_{스핀}

절대 원점 O에서 뻗어 나간 입자의 위치 벡터를 \mathbf{r}_i 라 할 때, 계 전체의 총 각운동량 \mathbf{H} 는 모든 입자의 (\mathbf{r}_i \times m_i \mathbf{v}_i) 의 시그마 연산이다. 여기에 \mathbf{r}_i = \mathbf{r}_{cm} + \mathbf{r}'_i 와 \mathbf{v}_i = \mathbf{v}_{cm} + \mathbf{v}'_i 라는 상대 변위(CM 기준틀 시점치환) 방정식을 밀어 넣고 식을 전개하라.
놀랍게도 교차 적립되는 상대 운동량 항(\sum m_i \mathbf{r}'_i 등)은 이 전 챕터에서 증명했듯 모두 0으로 폭파되어 완전히 날아간다.

그 결과, 징그럽게 얽혀있던 다체 시스템의 수만 개 각운동량 벡터(\mathbf{H})들은 단 두 개의 거시 텐서 블록으로 영구 쪼개진다.

\mathbf{H}_{O} = (\mathbf{r}_{cm} \times M \mathbf{v}_{cm}) + \sum_{i=1}^{N} (\mathbf{r}'_i \times m_i \mathbf{v}'_i)

  • 궤도 각운동량(Orbital Angular Momentum, \mathbf{H}_{cm}): 첫 번째 항 (\mathbf{r}_{cm} \times M \mathbf{v}_{cm}).
    마치 전체 질량 M 이 질량 중심점 하나에 응집되어 우주 원점을 중심으로 타원 궤도를 돌 때 생기는 ’거대한 공전 스핀’이다. 지구 전체가 태양을 도는 공전 궤도 각운동량이다.
  • 상대 스핀 각운동량(Absolute Spin Angular Momentum, \mathbf{H}_{spin}): 두 번째 항 \sum (\mathbf{r}'_i \times m_i \mathbf{v}'_i).
    이것은 오직 내가 질량 중심(CM) 위에서 가부좌를 틀고 시스템을 바라볼 때 관측되는, 계 내부 입자들의 지들끼리의 ’맴돌이(회전) 텐서’다. 지구가 자신의 핵을 돌고 있는 자전(Spin) 역학이다.

2. CM 기준틀의 가장 위대한 특권: 외력과 스핀 토크(Torque) 방정식의 절대 단절

이제 모멘트 텐서 진폭, 즉 우주 외부에서 이 다체계를 비틀어버리는 총 토크(Sum of Torques) 방정식 가동망(\sum \mathbf{M} = d\mathbf{H}/dt)을 돌려보자. \mathbf{H}_O 가 궤도(공전)와 스핀(자전)으로 쪼개졌듯이, 이 거시 비틀림 방정식 역시 정확하게 병진 궤도 록(Lock)과 자전 록으로 쌍방 분할(Decoupling)된다.

  • 병진 궤도 모멘트 연산: \sum (\mathbf{r}_i \times \mathbf{F}_{ext}) \rightarrow 폭탄의 폭발이나 화학 부품 간 충돌 같은 내부 마찰력 벡터는 역시 토크에서도 완벽하게 서로 상쇄(Cross-product 상쇄)된다. 파편들이 안에서 아무리 돌고 회전 반발을 해도, 시스템의 거시적인 궤도 각운동량 곡선은 외부 토크 역학(\sum \mathbf{M}_{ext}) 외에는 절대로 뒤틀리지 않는다.
  • 자전 스핀(\mathbf{H}')과 토크(\mathbf{M}_{cm}): CM을 중심으로 각 파편에 작용하는 외력의 토크 총합, \sum \mathbf{M}_{cm} = \frac{d\mathbf{H}'_{cm}}{dt}

인공위성이 우주 궤도를 비행하며 내부의 자세 제어 엔진 휠(Reaction Wheel)을 돌릴 때 이 직교 분해는 극대화된다. 휠을 아무리 비틀어 돌려대도 위성의 거시적인 지구 공전 궤도(\mathbf{H}_{궤도})는 머리카락 한 톨만큼도 변하지 않는다(내력 토크 상쇄).
하지만 위성의 본체는 휠 역회전 분산 스핀(\mathbf{H}_{스핀} 보존을 맞추기 위한 각운동량 교환)에 의해 즉각 90도 회전(자세 제어)한다. 오직 각운동량이 공전과 자전으로 분리됨을 증명한 다체 동역학의 기둥 덕분에 21세기 무중력 자세 제어 시뮬레이터가 완성된 것이다.
