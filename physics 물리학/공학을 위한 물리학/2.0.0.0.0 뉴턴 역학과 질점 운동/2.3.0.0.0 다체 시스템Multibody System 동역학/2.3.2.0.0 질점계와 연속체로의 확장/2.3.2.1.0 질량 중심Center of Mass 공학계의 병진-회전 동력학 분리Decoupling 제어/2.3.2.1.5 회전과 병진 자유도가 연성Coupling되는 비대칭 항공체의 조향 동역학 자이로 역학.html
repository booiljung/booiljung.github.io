2.3.2.1.5 회전과 병진 자유도가 연성(Coupling)되는 비대칭 항공체의 조향 동역학 자이로 역학

질량 중심(CM)을 이용해 병진(Translational) 외력 궤도와 회전(Rotational) 스핀 모멘텀을 직교 분리(\mathbf{P} 와 \mathbf{H} 의 디커플링)하는 고전 역학의 축복은, 대칭적인 구슬이나 매끈한 다트가 허공을 날아갈 때나 성립하는 순수한 교과서적 이야기다.
현실의 전투기(F-35)나 비대칭 페이로드를 단 인공위성, 혹은 공기 저항이 작용하는 나선형 로켓은 질량 분포 측면에서 기괴한 비대칭성 트러스트(Asymmetric Trust/Mass)를 지닌다. 이들은 회전하려 할 때마다 질량이 쏠리는 공간으로 강제로 밀려가며, 병진하려 할 때마다 공기역학적 및 관성적 엇갈림 때문에 자신의 의지와 상관없이 기수가 틀어져 빙글빙글 도는, 병진과 회전이 지저분하게 얽힌 연성(Coupling) 시스템의 재앙적 역학 구배를 마주하게 된다.

1. 비로컬(Non-CM) 기준 관측점: 질량 중심-오프셋의 족쇄

“질량 중심점 모멘트 기준틀을 잡아라!”
그러나 전투기 센서(IMU)는 기수 통신 라돔(조종석 앞)에 장착되어 있지, 전투기 뱃속 한가운데 허공 질량 중심에 떠 있지 않다. 센서 기준 좌표계 \mathbf{A}(임의의 점)에서 회전 각운동량 \mathbf{H}_A 를 렌더링하려면 이 우아한 분리 식은 다시 파괴된다. 질량 중심이 센서에서 \mathbf{\bar{r}}_C 만큼 떨어져 있다면 병진 속도 벡터 \mathbf{v}_A 항이 각운동량 토크 미분 속에 무자비하게 침입(Coupling)한다.

\sum \mathbf{M}_A = \dot{\mathbf{H}}_A + \mathbf{v}_A \times (M \mathbf{v}_C)

센서(혹은 제어 원점) \mathbf{A} 가 가속하며 움직일 때, 가속하는 무게 추의 위치 편향 \mathbf{\bar{r}}_C 때문에, 전투기는 그냥 ’앞으로 가속’만 했을 뿐인데 병진 운동량(M\mathbf{v}_c) 벡터가 \mathbf{v}_a 에 외적(Cross Product)되면서 조종사도 모르는 무지막지한 강제 회전 모멘트가 기수에 가해진다. 오프셋 위치가 유발하는 거시적 커플링의 살인적인 비행 오류다.

2. 질량 관성 행렬의 비대각 텐서와 자이로스코픽 코리올리 토션(Torsion)

비대칭 항공기 내부에 들어있는 수많은 펌프, 회전축 로터, 연료 탱크의 슬로싱(Sloshing) 등은 회전 관성 행렬 텐서 \mathbf{I} 를 처참하게 깨진 대칭계로 만든다.
오일러의 비선형 리지드 바디(Rigid Body) 회전 각가속도 \dot{\omega} 방정식은 관성 모멘트 행렬과 각속도의 외적으로 자이로스코픽 결합을 잉태한다.

\mathbf{M}_{cm} = \mathbf{I} \dot{\mathbf{\omega}} + \mathbf{\omega} \times (\mathbf{I} \mathbf{\omega})

항공체가 급격히 기수(Pitch)를 들 때는, 그 비가역적 \mathbf{\omega} 교차 외적이 날개를 돌려버리는 가공할 롤링(Rolling) 모멘트와 요잉(Yawing) 모멘트를 측면으로 방출한다(\omega \times I\omega 항 가속화).
즉, 앞으로 똑바로 가려는 추진 기관(\mathbf{v})은 비대칭 무게 중심에 의해 롤/요 회전을 짓이기고, 그 회전(\mathbf{\omega})은 다시 비대각 관성 텐서 위에서 크로스 스핀 모멘트(\mathbf{\omega} \times \mathbf{H})를 만들어, 결국 기체를 나선 궤도로 곤두박질치게 만든다.

우주 발사체 위성을 자세 제어하려 추력기(Thruster)를 켜면 질량 중심 오프셋에 의해 필연적으로 위성이 뒤로 밀려버린다(Translation Coupling). 위성의 스핀 휠을 돌려 자세를 잡으려 하면 비대칭 관성 덕분에 궤도가 미세하게 출렁인다. 이것을 잡아 돌려 뉴턴-오일러 연립 비선형 매트릭스로 모조리 역계산해 내는 작업이 항공 우주 제어 공학의 역동역학(Coupled Inverse Dynamics) 마스터클래스다.
