2.3.2.1.1 N개 강체 입자 질량 중심 연속 방정식과 운동량 중심 직교 분해(Decomposition) 역학 공간

N개의 입자로 구성된 거대한 구름 덩어리, 혹은 볼트와 너트로 뭉쳐진 자동차의 형상을 어떻게 단 하나의 수학적 좌표로 대표할 것인가. 고전 물리학계는 시스템 내 모든 입자 위치 \mathbf{r}_i 에 각각의 중량 가중치 m_i 를 곱해서(가중 평균) 공간상의 단좌표점 반경 벡터 \mathbf{r}_{cm} (질량 중심 벡터) 으로 통합해 낸다.

\mathbf{r}_{cm} = \frac{1}{M} \sum_{i=1}^{N} m_i \mathbf{r}_i \quad \left( M = \sum m_i \right)

  • 이산점계(Discrete Particles): 볼트(m_1)와 너트(m_2)처럼 개별 입자의 시그마 합(\Sigma)으로 렌더링.
  • 연속체 강체 공간(Continuum): 형상을 지닌 강체의 경우 질량이 체적 요소안에 무한대로 촘촘히 흩뿌려져 있으므로(dm = \rho \, dV), 위 시그마 식은 공간 밀도 적분 방정식 $ \mathbf{r}_{cm} = \frac{1}{M} \int_V \mathbf{r} \rho dV $ 로 부드럽게 연속 치환된다.

1. 운동량 다발의 결집체, 총 선운동량 방정식 정립

우주에서 절대 좌표계(Inertial Frame) 위에 서 있는 관찰자 시점에서, 파편 수십만 개로 쪼개져 튕겨 나가는 입자들의 위치 식 양변을 시간 dt 로 나누어 미분해 보라. ($ \sum m_i \dot{\mathbf{r}}i = M \dot{\mathbf{r}}{cm} $ )

\mathbf{P}_{total} = \sum_{i=1}^{n} m_i \mathbf{v}_i = M \mathbf{v}_{cm}

이 단순한 미분 한 줄은 다물체 동역학의 기둥이 된다. 무수히 많은 파편의 난해한 속도와 찌그러짐을 다 더한 시스템 전체의 “거시적인 거대 총 선운동량 \mathbf{P}_{total}” 은, 그저 전체 덩어리의 중량(M)이 질량 중심점 하나가 가지는 궤적 속도(\mathbf{v}_{cm})로 날아가고 있는 단일 입자(Particle)의 관성 에너지와 수학적으로 완벽하게 동일하게 등가(Equivalent) 취급될 수 있음을 고발하는 것이다.

1. 관찰자 시점 분할: 질량 중심-직교 분해망(Orthogonal Decomposition)

자, 이제 절대 좌표계에 서 있던 공학자는 시선을 돌려, 빛의 속도로 날아가고 있는 바로 그 ’질량 중심 \mathbf{r}_{cm}’의 좌표계 (이른바 중심 기준틀 Center-of-Mass Frame) 위에 통째로 올라탄다.
내가 질량 중심점과 똑같은 속도로 시뮬레이션 세계를 날아가면서 주위 시스템 파편들을 바라보면 어떤 현상이 관측될까? 파편들이 날아다니든, 터져서 멀어지든, 그 변위 벡터합은 내 시점에서는 언제나 완벽하게 ‘0(Zero)’ 이다.

  • 상대 변위: \mathbf{r}'_i = \mathbf{r}_i - \mathbf{r}_{cm}
  • 상대 속도: \mathbf{v}'_i = \mathbf{v}_i - \mathbf{v}_{cm}
  • CM 병진 프레임(Translated CM Frame) 내 상대 운동량의 대수 법칙:
    \sum m_i \mathbf{r}'_i = \mathbf{0}, \quad \sum m_i \mathbf{v}'_i = \mathbf{0}

드론이 허공에서 돌팔매질 요동을 치며 추락할 때 비행 제어기(FC)는 질량계를 바라보는 2개의 두뇌가 나뉜다. 하나는 전체 중량(M) 덩어리 중심점이 바닥으로 포물선을 그리며 пада는 절대 가속 낙하 역학이고, 두 번째 센서는 오로지 \mathbf{r}_{cm} 위에서 자가 팽이 돌듯 회전하며 발악하는 회전(\dot{\theta}) 스핀 역학이다. 이 \mathbf{v}_{cm} 과 가상 기준틀 \mathbf{v}'_i 의 병진/회전 선형 분할. 뉴턴과 오일러 다물체계를 이룩해 낸 최초의 공간 직교 절단 분해(Decomposition)다.
