2.3.2.1.7 차량 군집 주행(Platooning) 등 가상 연결 질량 중심 시스템의 동시 다발 병진-제동 제어

수십 대의 화물 트럭 트레인(Truck Platooning)이 시속 100km로 아스팔트 위를 달릴 때, 각 트럭은 무선 통신(V2V) 망을 통해 눈에 보이지 않는 전자 스프링(Virtual Spring-Damper)으로 묶여 있다. 금속 기어로 연결되지 않고 오로지 데이터(Data) 텐서로 얽힌 이 다물체 시스템은 어떻게 물리적 충돌 없이 안전하게 교차로를 돌파하거나 돌발 정지할 수 있을까?

자율주행 제어 공학자들은 이 수십 대의 트럭 무리를 N개의 흩어진 강체 파편군으로 보지 않고, 단일 거대 관성 플랫폼인 ‘군집 전체의 통합 질량 중심(Global CM)’ 시스템으로 매핑(Mapping) 시켜 제어한다. 병진 역학과 상대 위치(\mathbf{r}'_i) 제어를 직교 분해하여 통제하는 질점계 고전 물리학의 부활이다.

1. 가상 질량 중심선(Virtual CM Trajectory) 설계와 전진 구속

N대의 군집 주행 트럭이 있을 때, 통신 제어기는 개별 트럭마다 독립적인 목표 위치를 뿌려주지 않는다. 대신 제일 앞 선두 차량(리더)부터 뒤쪽까지 트럭의 질량을 가중 평균한 거대한 가상 무게 중심(\mathbf{R}_{cm})의 병진 궤적(Translational Trajectory) 한 가닥을 고속도로에 선명하게 그려 넣는다.

\mathbf{R}_{cm}(t) = \frac{1}{M_{total}} \sum_{i=1}^{N} m_i \mathbf{r}_i(t)

이제 컴퓨터는 수많은 가속 페달과 브레이크(개별 구동력 \mathbf{F}_i)들의 총합 \sum \mathbf{F}_i 가, 오직 이 무겁고 거대한 가상의 \mathbf{R}_{cm}(t) 한 점 덩어리를 고속도로 차선 중앙으로 부드럽게 가속 및 이동시키는 단 한 줄의 거시 병진 제어 명령 \mathbf{F}_{total} = M_{total} \ddot{\mathbf{R}}_{cm} 로 환원되도록 지휘한다.
어느 한 트럭이 언덕을 오르느라 속도가 떨어져도, 시스템은 1번 트럭을 쏘아댈 것이 아니라 전체 구동력 합을 늘려 시스템의 우주 질량 중심점 궤도 이탈을 막는 데 매진한다.

2. CM 기준 국부(Local) 상대 변위망 제어와 다발 제동(Simultaneous Braking)

전체 무리가 한 덩어리로 잘 전진하고 있다면, 그다음 임무는 내부 질점들 간의 충돌, 즉 스핀(상대 궤도 \mathbf{r}'_i)을 억제하는 내부 텐서 제어다.

  • 각 트럭은 자신의 위치를 허공의 거대 질량 중심 \mathbf{R}_{cm} 에 대비하여, “나는 CM으로부터 +10m 앞에 있어야 하는 점(\mathbf{r}'_{+10})”, “나는 CM에서 -20m 뒤떨어져 있는 꼬리칸(\mathbf{r}'_{-20})“이라는 상대 좌표 타겟만을 부여받는다.
  • 도로에 맹수가 뛰어들어 선두 트럭이 급브레이크 파동을 터뜨리면, 그 관성 펄스 충격파는 거대 질량 중심 \mathbf{R}_{cm} 의 급가속도 축소(\ddot{\mathbf{R}}_{cm} < 0)를 즉각 선포한다.
  • 그 순간 뒤따라오는 차량들은 앞차를 보고 속도를 줄이는 것이 아니라, 갑자기 통신으로 뒤로 쭉 물러나는 ’가상 질량 CM 상대 영점’의 감속 궤도 자체를 감지하고, 자신의 상대 좌표 \mathbf{r}'_i 를 사수하기 위해 브레이크 액추에이터를 100분의 1초 만에 동시에 밟아버린다(Simultaneous Actuation).
