2.3.2.1.0 질량 중심(Center of Mass) 공학계의 병진-회전 동력학 분리(Decoupling) 제어

미사일이 폭발하여 파편 수백 개가 공중으로 비산할 때, 개별 파편들의 궤적은 공기 저항과 무작위적인 내부 폭발력 탓에 난해한 카오스(Chaos) 역학계를 그린다. 하지만 그 모든 파편 조각들의 궤적을 평균 내어 합산해 보면, 오직 거대한 포물선(Parabola) 궤도만을 우직하게 묵묵히 날아가는 단 하나의 절대적인 기하 중심점이 존재함을 깨닫게 된다.
이 신비로운 질량 가중치 가상 좌표점, 역학계 전체 시스템 질량의 ’영혼’이 모여 있는 곳을 칭하여 우리는 질량 중심(Center of Mass, CM) 이라 일컫는다.

질점계를 다루는 공학의 가장 위대한 테크닉은, 관찰자의 시선(좌표계 기준틀)을 우주 허공에 고정된 절대(Absolute) 원점에 두지 않고 이 떠내려가는 ‘질량 중심 점’ 위로 통째로 옮겨 심는 것이다. 그 순간 시스템 전체의 거칠고 복잡했던 모든 벡터 모멘텀 파도들은 거시적인 전체 병진(Translational) 표류와 자기 자신을 깎아 먹으며 도는 내부 회전(Internal Spin) 이라는 두 개의 완전히 독립된(Decoupled) 방정식으로 무결하게 직교 분해(Orthogonal Decomposition)된다.
