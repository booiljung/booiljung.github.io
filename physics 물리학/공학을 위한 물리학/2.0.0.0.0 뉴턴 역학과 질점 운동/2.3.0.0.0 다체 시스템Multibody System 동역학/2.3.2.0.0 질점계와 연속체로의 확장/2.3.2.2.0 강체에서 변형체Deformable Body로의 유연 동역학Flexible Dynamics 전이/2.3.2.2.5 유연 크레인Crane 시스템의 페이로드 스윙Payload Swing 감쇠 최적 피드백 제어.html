2.3.2.2.5 유연 크레인(Crane) 시스템의 페이로드 스윙(Payload Swing) 감쇠 최적 피드백 제어

항만 터미널이나 고층 빌딩 건설 현장에 우뚝 솟은 거대한 타워 크레인(Tower Crane)을 보라. 조종사가 레버를 당겨 크레인의 지브(Jib, 수평 구조물)를 회전(Slewing) 시키고 트롤리(Trolley)를 전진시킬 때, 100미터 길이의 강철 케이블 끝에 매달린 수십 톤의 화물(Payload)은 필연적으로 진자(Pendulum) 요동을 일으키며 흉포하게 허공을 맴돈다(Swing). 화물의 파괴적 흔들림을 육안으로 보고 멈출 때까지 기다리거나 역으로 조종간을 당겨 감쇠시키는 것은 인간의 감에 의존한 반쪽짜리 하등 제어다.

더군다나 타워 크레인의 기둥 뼈대 탑과 길쭉한 지브(Jib) 자체는 완벽한 강체(Rigid Body)가 아니라, 하중과 관성에 의해 휩쓸리며 끊임없이 굽힘 진동(Flexible Bending)을 일으키는 거대한 유연 동역학(Flexible Dynamics) 텐서망의 응집체다. 강체 운동(대차 이동), 유연체 굽힘 진동(크레인 뼈대), 그리고 페이로드의 비선형 스윙 진동을 단 하나의 방정식으로 묶어 제압하는 최적 피드백 라그랑지안(LQR/MPC 제어계) 의 도달 경로를 묘사한다.

1. 강체-유연-진자 연성(Coupling) 오일러 미분 방정식 모델링

크레인 시스템의 라그랑지안(L = T - V)을 렌더링하기 위해서는 세 가지 독립된 역학 진동계를 미분 계수로 융합해야 한다.

  • q_r: 대차(Trolley)의 1차원 직선 강체 구동 변위.
  • q_f: 크레인 암(Jib)의 분포 유연 진동 모드(Modal Coordinates, 부동 기준틀 내 휨 변수).
  • \theta: 강철 와이어에 매달린 화물 덩어리(Payload)의 국부 스윙(Swing) 각도 좌표.

\mathbf{M}(\mathbf{q}) \begin{bmatrix} \ddot{q}_r \\ \ddot{q}_f \\ \ddot{\theta} \end{bmatrix} + \mathbf{K} \begin{bmatrix} q_r \\ q_f \\ \theta \end{bmatrix} + \mathbf{C}_{coriolis}(\mathbf{q}, \dot{\mathbf{q}}) \dot{\mathbf{q}} = \begin{bmatrix} u_{ctrl} \\ 0 \\ 0 \end{bmatrix}

무서운 사실은 우리의 제어 모터 입력 변수(Input, u_{ctrl})는 대차를 움직이는 동력(\ddot{q}_r) 단 한 곳에만 들어간다는 점이다. 즉 뼈대의 휨(\ddot{q}_f)이나 화물의 스윙(\ddot{\theta})은 모터가 직접 붙잡고 브레이크를 걸 수 없는 상태, 과소 구동계(Underactuated System) 에 놓여 있다.
대차(Rigid)가 앞으로 가속(\ddot{q}_r)하면 저 거대한 결합 질량 매트릭스 \mathbf{M}(\mathbf{q}) 의 극악한 연성(Coupling) 텐서를 타고 휨 진동과 진자 스윙 각가속도로 에너지가 강제 전가된다.

2. LQR 방산 제어 역학을 통한 화물 스윙(Swing) 제로화 최적 피드백

목표 지점(q_{target})까지 가장 빨리 대차를 이동시키면서도, 부착된 부재의 진동(q_f)과 화물 요동(\theta)을 도착과 동시에 완벽하게 0 으로 소멸시켜 버리려면(Anti-swing), 모터는 기계 역학 텐서를 뒤집는 자가 치유 피드백 연산을 수용해야 한다.

  1. 상태 공간 선형화(State Space Linearization): \theta 가 작다고 가정(\sin\theta \approx \theta)하여 비선형 매트릭스를 \dot{\mathbf{x}} = \mathbf{A}\mathbf{x} + \mathbf{B}u 의 1차 연립 미분 방정식으로 강제 찢어발긴다. (\mathbf{x} = [q_r, \dot{q}_r, \theta, \dot{\theta}, q_f, \dot{q}_f]^T)
  2. 비용 함수(Cost Function) 최적화 텐서 주입: \mathbf{J} = \int (\mathbf{x}^T \mathbf{Q} \mathbf{x} + u^T R u) dt 이라는 에너지 페널티(Penalty) 함수를 건다. Q 행렬의 가중치를 통해 “화물 \theta 의 흔들림에 지독하게 무거운 벌점“을 내린다.
  3. LQR(Linear Quadratic Regulator) 최적 피드백 게인 \mathbf{K}_{gain} 추출:
    리카티 미분 대수 방정식(Riccati Equation)을 풀어내면 모터의 출력 토크는 더 이상 인간 조종사의 레버에 의존하지 않게 된다.
    $ u_{ctrl} = - \mathbf{K}_{gain} \mathbf{x} = - ( k_1 q_r + k_2 \dot{q}_r + k_3 \theta + k_4 \dot{\theta} + k_5 q_f ) $

이 제어 시스템이 장착된 차세대 무인 크레인 로봇은, 페이로드를 목적지 위로 날려 보낼 때 화물이 앞으로 흔들리려는 관성 에너지를 모터의 미세한 가속-감속 ‘지그재그 펄스’ 위상으로 빼앗아 먹으며(Virtual Damping), 화물이 도착 지점에 닿는 순간 물병 하나 흘리지 않고 공중에 못 박힌 듯 정지시킨다. 강체 운동과 유연/진동(Swing) 변수망이 오일러 라그랑지안 안에서 철저히 묶여(Coupled) 있음을 역으로 찌른 피드백 공학의 대마술이다.
