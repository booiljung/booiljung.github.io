2.3.2.2.2 연속체 유연 구조물의 대변위(Large Displacement) 비선형 유한 요소(FEA) 동력학

부동 기준틀 연성 공식(FFRF)이 로봇 팔이나 헬기 날개처럼 ‘크게 움직이되 변형은 작은(Large rigid motion, Small deformation)’ 얌전한 시스템을 정복했다면, 긴 낚싯대가 활처럼 휘어지거나 자동차 와이퍼가 유리를 쓸며 크게 출렁일 때는 속수무책으로 무너진다.
유연 구조물이 자기 몸길이의 10\% 를 넘어서게 엄청나게 구부러지는 순간(Geometrically Non-linear), 기존의 단순한 선형 스프링 강성 행렬(\mathbf{K}\mathbf{u})은 그 휘어짐의 곡률(Curvature) 기하를 전혀 담아내지 못한 채 무용지물이 된다. 강체와 변형의 경계를 완전히 허물고, 모든 것을 하나하나의 작은 블록(유한 요소)으로 쪼개 전체 절대 기준틀에서 다시 조립해 버리는 절대 노달 좌표 공식(Absolute Nodal Coordinate Formulation, ANCF) 과 대변위 비선형 FEA 역학이 그 낚싯대를 집어삼킨다.

1. 기하학적 비선형성(Geometric Nonlinearity)과 강성 행렬의 타락

선형 탄성학은 물체가 변형되어도 “그 모양이 처음과 거의 같을 것“이라고 순진하게 기만한다(\sin\theta \approx \theta).
하지만 길다란 다리미 줄이나 안테나가 90도로 꺾여버리면, 구조물이 하중을 받는 ‘방향’ 자체가 실시간으로 비틀려 돌아간다. 이를 기하학적 비선형성(Geometric Nonlinearity) 이라 부른다.

단순 빔 역학에서는 변형률 텐서 \varepsilon 를 오직 선형(\partial u / \partial x)으로만 잡았지만, 대변위 영역에서는 폰 카르만(von Karman)이나 그린-라그랑주(Green-Lagrange) 변형률 텐서를 소환해야 한다.

\varepsilon = \frac{\partial u}{\partial x} + \frac{1}{2} \left( \left(\frac{\partial u}{\partial x}\right)^2 + \left(\frac{\partial v}{\partial x}\right)^2 \right)

끝에 붙은 저 이차항 기하학적 파편 \frac{1}{2}(\dots)^2 이, 직선이 곡선으로 변하면서 추가로 찢어지는 텐션 비선형 스칼라의 정체다. 이 이차항 덕분에 우리의 FEA 강성 행렬 \mathbf{K} 는 상수(Constant Matrix)의 지위를 박탈당하고, 변위 \mathbf{u} 에 위치와 방향이 묶인 기괴한 접선 강성 행렬(Tangent Stiffness Matrix, \mathbf{K}_T(\mathbf{u})) 로 흉폭하게 돌변한다.

2. 절대 노달 좌표망(ANCF): 강체-유연체 역학계의 극단적 스칼라 융합

거시적 강체 회전(예: 헬기 로터 회전)과 부품 자체가 엿가락처럼 휘어지는 휨(대변위)을 동시에 다루기 위해 ANCF 방법론은 오일러 회전각 \theta 따위를 아예 폐기해 버린다.
대신 요소(Element) 끝단 노드에 오직 절대 위치(\mathbf{r})와, 그 위치에서의 절대 투영 기울기/변화율 백터(Position Vector Gradient, \partial \mathbf{r}/\partial \mathbf{x}) 만을 쑤셔 넣는다.

  • 강체 회전각 \theta 나 국부 좌표 변위를 따로 분리(FFRF)하지 않는다.
  • 변위 파라미터 벡터 \mathbf{e} = [\mathbf{r}^T, (\partial \mathbf{r}/\partial x)^T, \dots]^T 단 하나로 강체 회전과 유연한 굽힘 곡률(Curvature)을 동시에 모터 시뮬레이션으로 밀어버린다.

이 ANCF 비선형 FEA 모델을 쓰면, 질량 행렬 \mathbf{M} 이 우연히도 일정한 상수(\text{Constant}) 매트릭스로 굳어진다. 대신 엄청난 비선형 곡률의 찌그러짐은 오직 탄성력 벡터 단 하나로 몰빵된다. 시뮬레이터 수치 적분기가 비틀리는 타이어나 고무 벨트 드라이브, 외과 수술 로봇의 카테터(Catheter) 줄을 0.001초 단위로 부드럽게 감아 돌릴 수 있게 된 이유, 이 절대 노달 텐서망이 뉴턴-랩슨(Newton-Raphson) 적분법 위에서 맹렬하게 구배를 헤집기 때문이다.
