2.3.2.2.6 다물체 접촉(Contact) 영역 부근의 국부적 헤르츠(Hertz) 역학 변형 및 응력 파동

지금까지의 강체 구속식(Constraint Equation)은 두 물체가 맞물릴 때 떨어지지도, 파고들지도 않는다는 기하학적 등식 \Phi = 0 으로 표면 입자를 영구 결합시켰다. 이 조인트 역학은 베어링처럼 이상적으로 설계된 링키지(Linkage)에는 맞지만, 톱니바퀴 여러 개가 충돌하며 물려 돌아가거나 쇠공이 강철판 위를 구르는 ‘접촉(Contact)’ 현상 앞에서는 완전히 허구적인 가설로 돌변한다.

금속 대 금속이 맞부딪히는 진짜 접촉면은 결코 점(Point)이나 이상적 선(Line)이 아니다. 눌리는 압력에 의해 금속의 원자 격자가 탄성적으로 찌그러지며 국부적으로 둥그스름한 타원형 면적(Elliptical Patch) 을 만들어 파고드는 침투(Penetration, \delta) 동역학계를 폭발시킨다. 두 개의 반짝이는 구슬이 부딪치는 찰나, 그 미세한 접촉 계면 안에서 어떤 괴기한 탄성 응력 폭풍이 소용돌이치며 튕겨 내는 힘을 창조하는가? 하인리히 헤르츠(Heinrich Hertz)가 밝혀낸 다체 충돌-접촉 반력의 심연을 서술한다.

1. 헤르츠 접촉 강성(Contact Stiffness)과 비선형 침투력 방정식

헤르츠 접촉 역학은 두 구형(또는 실린더 곡면) 유연체가 미세하게 겹쳐 파고드는 깊이, 침투량(Penetration, \delta) 을 핵심 변수로 삼는다.
강체 역학 시뮬레이터(MBD)가 두 물체의 표면 거리가 0 보다 작아졌다고 판단하는 순간, 즉시 그 침투 깊이 \delta 에 비례하는 수직 반력(Normal Contact Force, \mathbf{F}_n)을 양 물체의 질량 중심 계통 텐서로 쏘아 올린다.

F_n = K \cdot \delta^n

  • 선형 스프링(F = kx)과 달리, 곡면 대 곡면이 만날 때는 찌그러질수록 접촉 면적이 기하급수적으로 부풀어 오르기 때문에 지수 n 은 1.5(점 접촉) 혹은 10/9(선 접촉) 에 달하는 흉악한 비선형 탄성(Nonlinear Elasticity) 반발력을 그린다.
  • 강성 계수 K 는 서로 다른 두 금속의 영률(Young’s Modulus, E)과 포아송 비(\nu), 그리고 곡률 반경(Radius, R)의 극도로 기하학적 혼합비(Equivalent Stiffness)로 구축된다. 단단할수록 점근선은 무한대에 가깝게 치솟고, 시뮬레이터의 수치 텐서망을 단층 모서리로 썰어버린다.

2. 응력 포물선 그물망 체계와 전단 항복(Shear Yielding) 파괴 기원

시뮬레이터가 표면을 튕겨내는 거시 힘 벡터 \mathbf{F}_n 을 뽑아내는 동안, 현미경 아래의 공학자들은 접촉 계면 표면이 아니라 살짝 파고들어 간 지하(Sub-surface) 영역을 두려워한다.
헤르츠 역학에 따르면, 표면에서 타원형의 반구(Semi-ellipsoid) 형상으로 분포하는 압박 응력(Compressive Pressure)이 가장 높은 중심점보다 훨씬 더 치명적인 파동은, 표면을 뚫고 내부로 조금 기어 들어간 국부 심층 지하에서 터진다.

두 금속이 피 터지게 부딪히면, 표면의 압축은 찢어지지 않고 버티지만 재질 내부 격자를 어긋나 밀어버리는 최대 전단 응력(Maximum Shear Stress, \tau_{max}) 의 파동 코어가 접촉 반경 a 의 약 0.48a 깊이 아래, 이른바 심부에 꽈리를 튼다.

베어링 볼이 1년 넘게 레일을 구르다가 표면이 비늘(Flake)처럼 우수수 떨어져 나가는 스폴링(Spalling) 피로 파괴 현상은, 표면의 마찰 때문이 아니라 바로 이 둥그스름한 헤르츠 접촉 힘이 표면 아래 심부에 무한정 쏟아부은 \tau_{max} 전단 파동 사이클의 누적 결과다. MBD 시뮬레이터는 이제 단순히 물체를 튕겨내는 기하 충돌 맵을 넘어, 헤르츠 공식을 타고 내부에 터져 오를 진단 응력의 지하 혈을 예측해 내는 마스터 솔버 도구로 등극했다.
