2.3.2.2.0 강체에서 변형체(Deformable Body)로의 유연 동역학(Flexible Dynamics) 전이

해석 역학과 기구학의 눈부신 성취 뒤에는 한 가지 지독한 수학적 타협이 도사리고 있었다. 바로 시스템 내의 모든 부품(로봇 팔, 위성 안테나, 헬리콥터 로터)이 ’절대로 휘어지거나 찌그러지지 않는 완벽한 쇳덩어리(Rigid Body)’라는 교란된 전제다.
부품 위 두 점 사이의 거리가 영원히 변하지 않는다는 이 r_{ij} = \text{const} 강체(Rigid) 고정 방정식은 매트릭스를 획기적으로 가볍게 만들었지만, 우주 정거장의 길쭉한 태양 전지판이 회전할 때 채찍처럼 출렁거리며 찢어져 버리거나 로봇 관절이 고속 기동 시 미세하게 휘청이는 잔류 진동(Residual Vibration) 타격 현상 앞에서는 처참하게 붕괴했다.

본 장에서는 고전 동역학이 구축한 철옹성의 외벽을 부수고, 파트 자체가 힘과 관성에 의해 스스로 휘어지고 파동 치는 유연 동역학(Flexible Multibody Dynamics) 의 차원으로 도약한다. 공간 상의 ‘거시적 이동(Rigid Motion)’ 텐서와, 그 부품 내부에서 격렬하게 떨리는 ‘미시적 탄성 변형(Elastic Deformation)’ 스칼라장이 어떻게 하나의 오일러 행렬 안에서 연립(Coupling)하며 소통하는지 파헤쳐라. 단단함의 환상을 버리는 순간, 진정한 현대 공학 시뮬레이션의 문이 열린다.
