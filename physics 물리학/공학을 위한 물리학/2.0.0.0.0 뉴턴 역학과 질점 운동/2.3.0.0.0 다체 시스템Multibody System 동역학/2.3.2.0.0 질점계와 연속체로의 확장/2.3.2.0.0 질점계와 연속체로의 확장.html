2.3.2 질점계와 연속체로의 확장

지금까지 고전 역학의 스포트라이트는 오직 이상적으로 점압축된 단일 질점(Particle)들이나 이들을 조인트로 강제 결합한 강체 구조망(Rigid Body System)의 궤적에만 머물렀다. 부피와 모양을 무시하고 모든 질량을 좌표 위의 한 점으로 은유했던 이 순수한 미니멀리즘은, 크기와 형태를 가진 현실의 우주선, 유체, 그리고 변형 가능한 유연체(Flexible Body) 앞에서는 필연적으로 무너져내린다.

본 장에서는 개별 입자들의 단순한 총합을 뛰어넘어 수십억 개의 미립자가 상호 구속되며 거시적인 계(System)를 이루는 질점계(System of Particles) 단위의 거동 확장법을 다룬다. ’질량 중심(Center of Mass)’이라는 가상의 관성 균형점을 창조해 내어 복잡한 다체계의 병진 진동과 내부 스핀을 직교 분리(Decoupling)시켜 버리는 통제 역학의 묘수를 서술할 것이다. 더 나아가, 강체의 벽을 허물고 형상 치수 자체가 시간과 힘에 종속되어 출렁이는 연속체 유연 동역학(Continuum Flexible Dynamics) 공간으로 진입하는 해석적 교두보를 마련한다.
