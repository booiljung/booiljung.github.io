2.3.1 구속 조건과 라그랑주 역학

고전 역학에서 뉴턴 미분 방정식(\mathbf{F} = m\mathbf{a})은 개별 질점에 작용하는 모든 벡터 힘을 직교 좌표계(Cartesian Coordinates) 위에서 일일이 영사하여 추적해야 한다는 치명적인 설계 결함을 안고 있다. 단 하나의 질점이 아닌, 수십 개의 로봇 팔 관절이나 서스펜션 링크가 서로 맞물려 돌아가는 다체 시스템(Multibody System)을 마주하게 되면, 부품 간을 체결하고 있는 조인트(Joint)나 경첩이 뿜어내는 정체불명의 구속력(Constraint Force) 수백 개를 모조리 방정식에 그려 넣어야 하는 수리적 절망 상태에 빠진다.

해석 역학(Analytical Mechanics)의 거장, 라그랑주(Joseph-Louis Lagrange)는 이 벡터의 지옥에서 탈출하기 위해 뉴턴의 힘 중심 역학계를 에너지 스칼라 역학계로 완전 히 환골탈태시켰다. 본 장에서는 시스템의 군더더기를 걷어내고 순수한 운동의 자유도(DOF, Degree of Freedom)만을 뽑아내는 일반화 좌표계의 마법과, 복잡한 기계 구속 계통을 한 줄의 편미분 방정식으로 평정해 버리는 오일러-라그랑주(Euler-Lagrange) 미분 텐서망의 물리적 전개와 위상을 서술한다.
