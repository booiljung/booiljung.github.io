2.3.1.1.5 가상 일의 원리(Principle of Virtual Work)와 평형 시스템의 능동력 대수합 제로화

뉴턴 역학(\sum \mathbf{F} = \mathbf{0})으로 복잡한 로봇 링키지의 정적 평형 상태를 풀려면, 각 관절(Joint)에서 작용하는 알 수 없는 수십 개의 구속력(Constraint Force, \mathbf{f}_c)과 외부에서 작용하는 능동력(Active Force, \mathbf{F}_a)을 모조리 분리(Free Body Diagram)해서 풀어야 한다. 그러나 요하네스 베르누이(Johann Bernoulli)가 창안하고 달랑베르(d’Alembert)가 심화시킨 가상 일의 원리(Principle of Virtual Work) 는, 이 성가신 ’구속력’들을 계산에서 단 한 방에 영구적으로 소멸시키는 마법을 부린다.

그 마법의 핵심은, 이상적인 구속력(마찰 없는 핀, 매끄러운 표면, 팽팽한 길이 고정 끈)은 시스템이 구속 조건을 위반하지 않고 움직이는 ‘가상 변위(\delta\mathbf{r})’ 방향에 대해 언제나 수직(직교, Orthogonal)이라는 역설에서 출발한다.

1. 이상적 구속력의 가상 일(Virtual Work) 제로 법칙

정적 평형 상태에 있는 다체 시스템의 i 번째 입자에 작용하는 총 힘 \mathbf{F}_i 는, 모터나 중력처럼 외부에서 주입되는 능동력 \mathbf{F}_i^{(a)} 와, 부품들이 뜯어지지 않게 구조적으로 버티고 있는 구속력 \mathbf{f}_i^{(c)} 의 벡터 합이다. 평형 상태이므로 당연히 합력은 0이다 (\mathbf{F}_i^{(a)} + \mathbf{f}_i^{(c)} = \mathbf{0}).

이제 시간을 얼어붙게 만들어 놓고(\delta t = 0), 구속 장벽을 깨지 않으면서(Geometric Compatibility) 시스템의 각 점을 미세하게 비틀어 보는 가상 변위 \delta\mathbf{r}_i 를 인가하고 입자 전체의 일(Work)을 모두 합산해 보자.

\delta W = \sum_{i=1}^{N} \mathbf{F}_i \cdot \delta\mathbf{r}_i = \sum_{i=1}^{N} \left( \mathbf{F}_i^{(a)} + \mathbf{f}_i^{(c)} \right) \cdot \delta\mathbf{r}_i = 0

놀랍게도 기어, 핀셋, 강체의 길이 고정을 유지하는 이상적인 구속력 벡터(\mathbf{f}^{(c)})는 항상 가상 변위(\delta\mathbf{r}) 방향과 직교(내적 = 0)하거나 서로 크기가 같고 방향이 역상쇄되어(뉴턴 3법칙 작용-반작용), 전체 시스템에 구속력이 투입한 가상 일의 총합은 영원히 0으로 사라진다. ($ \sum \mathbf{f}_i^{(c)} \cdot \delta\mathbf{r}_i = 0 $)

2. 가상 일의 원리(Principle of Virtual Work)의 완성

머리 아픈 구속력이 0으로 연기처럼 증발했으므로, 오직 외부에서 부품을 끄집어당기는 능동력(외력, 모터 토크, 중력) \mathbf{F}_i^{(a)} 들만이 시스템이 움직인 가상 변위 \delta\mathbf{r}_i 방향과 결합하여 다음과 같은 단 하나의 강력한 원리 방정식으로 남는다.

\delta W = \sum_{i=1}^{N} \mathbf{F}_i^{(a)} \cdot \delta\mathbf{r}_i = 0
