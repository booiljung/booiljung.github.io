2.3.1.1.6 구속력(Constraint Force) 추출을 돕는 라그랑주 미정계수법(Lagrange Multipliers) 물리

가상 일의 원리나 오일러-라그랑주 거시 미분 방정식은 구속력(Constraint Force) 자체를 ‘일(Work)을 하지 않는 기생충’ 취급하며 계산에서 모조리 소거해 버렸다. 이것은 시스템의 궤적을 찾거나 시뮬레이션을 돌리기 위해 좌표 q_k 를 찾아내는 데는 완벽하지만, 아주 중대한 공학적 약점을 내포하고 있다.
로봇 팔을 묶고 있는 힌지 핀(Hinge Pin)이 파괴될지, 서스펜션 볼 조인트가 견뎌낼 수 있는지 설계하려면 바로 그 ’사라져버린 조인트 구속력의 한계 수치’를 측정해야 하기 때문이다! 변수를 아예 버리지 않고 구속 방식을 유지한 채, 각 구속 방정식의 팽팽한 ’긴장도’를 역으로 계산해 내기 위한 천재적인 수리 공학 스케일링, 그것이 라그랑주 미정계수법(Method of Lagrange Multipliers) 이다.

1. 구속 조건 기하 행렬(Jacobian)과 미정계수 벡터(\lambda)의 결합

자유도(DOF)를 소거하지 않은 상태에서 구속 조건 \Phi_j(\mathbf{q}) = 0 이 그대로 남아있다고 가정하라.
이 시스템은 라그랑지안(L = T - V) 운동 편미분 방정식에서 완전히 자유롭게 풀릴 수 없다. 여기서 각 구속 기하 방정식의 자코비안 구배(Gradient) \nabla \Phi_j = \partial \Phi_j / \partial q_k 벡터 앞에, 아직 결정되지 않은 스칼라 미정계수 \lambda_j (Lagrange Multiplier)를 곱명분으로 주입해 운동 방정식의 우변(구속 작용 방향)으로 강제 소환해버린다.

\frac{d}{dt} \left( \frac{\partial L}{\partial \dot{q}_k} \right) - \frac{\partial L}{\partial q_k} = \sum_{j=1}^{m} \lambda_j \frac{\partial \Phi_j}{\partial q_k}

  • m: 유지된 구속 조건 방정식의 수.
  • k: 소거되지 않은 n 차원의 일반화 좌표 변수 인덱스.
  • 이로써 시스템 변수는 원래의 좌표 벡터 \mathbf{q} (n 개)에 미정계수 벡터 \mathbf{\lambda} (m 개)가 추가된, (n + m) 연립 방정식 매트릭스로 팽창(Expansion)된다.

2. 미정계수(\lambda)의 물리적 실체: 단면 구속력(Constraint Force Magnitude)

수학자들에게 \lambda 는 그저 최적화 문제를 풀기 위해 구속 곡면상에서 헤시안 행렬을 스위칭시키는 스칼라 매개변수에 불과하지만, 기계 공학자에게 방정식의 우변 \mathbf{F}_c = \mathbf{\Phi_q}^T \mathbf{\lambda} 는 너무나도 명백한 “구속력/구속 토크(Constraint Force/Torque)” 그 자체의 투영 벡터다.

  • \frac{\partial \Phi_j}{\partial q_k} (자코비안 행렬): 구속력이 작용하는 기하학적 뱡향(Direction) 과 힘이 행사되는 위치, 즉 법선(Normal) 구배 벡터다.
  • \lambda_j (라그랑주 미정계수): 그 기하학적 방향으로 정확히 얼마나 강하게 핀이 당겨지고 있는지를 나타내는 구속 방향의 스칼라 힘의 크기(Magnitude/Tension) 다.
