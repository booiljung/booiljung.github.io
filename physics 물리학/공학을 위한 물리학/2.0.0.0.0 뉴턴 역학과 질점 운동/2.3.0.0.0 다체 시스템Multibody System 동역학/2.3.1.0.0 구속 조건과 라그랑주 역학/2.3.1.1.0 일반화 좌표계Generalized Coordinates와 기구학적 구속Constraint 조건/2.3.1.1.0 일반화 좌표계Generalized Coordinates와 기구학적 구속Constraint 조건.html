2.3.1.1.0 일반화 좌표계(Generalized Coordinates)와 기구학적 구속(Constraint) 조건

단순한 이중 진자(Double Pendulum)를 데카르트 직교 좌표계 위로 올려 보라. 첫 번째 질량의 위치 (x_1, y_1) 와 두 번째 질량의 위치 (x_2, y_2), 즉 무려 4개의 독립 변수가 등장한다. 하지만 두 막대의 길이가 L_1, L_2 로 굳건히 고정되어 있다는 기구학적 구속(Constraint)이 발동하는 순간, 4개의 변수 중 2개는 나머지 2개에 의해 완벽하게 묶여 노예처럼 끌려다니는 잉여(Redundant) 데이터로 전락한다.

해석 역학의 최우선 목적은 시스템을 기술하는 변수의 수를 이 쓸모없는 잉여들을 모두 잘라내고, 계의 진정한 상태를 가장 적은 수량으로, 오차 없이 독립적으로 정의할 수 있는 일반화 좌표계(Generalized Coordinates) 스칼라 벡터 세트로 축소(Reduction)시키는 것이다. 이 축소된 미니멀리즘의 차원 수가 바로 그 시스템의 절대 자유도(DOF, Degree of Freedom)와 일치한다.
