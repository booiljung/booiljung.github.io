2.3.1.1.7 동적 구속 파열(Constraint Violation) 방지를 위한 바움가르테(Baumgarte) 안정화 제어

다물체 동역학(MBD) 시스템을 모델링하고 시뮬레이터를 돌려보면 엔지니어들은 종종 끔찍한 기하학적 파국에 직면한다. 미분 대수 방정식(DAE)을 뉴마크-\beta 법이나 Runge-Kutta 적분기 같은 수치 알고리즘으로 시간 전진(Time Marching) 시킬 때, 허공에 매달아 놓은 로봇 팔이나 차량의 서스펜션이 서서히 분해되며 뼈마디가 떨어져 나가는(Drift) 현상이 나타나기 때문이다.

이론상 구속 방정식(\Phi=0)을 두 번 미분한 가속도 제약식(\mathbf{\Phi_q} \ddot{\mathbf{q}} = \gamma)만을 만족시키면, 컴퓨터 속 물체는 영원히 연결된 채 우주를 유영해야 한다. 하지만 유한한 시간 간격(\Delta t)으로 수치 적분을 수행하는 한 찰나의 절단 오차가 티끌처럼 누적되어(Drifting), 마침내 좌표들이 제멋대로 요동치며 조인트 핀이 기가 막히게 뽑혀버린다. 이른바 구속 파열(Constraint Violation) 오류다.

1. 뼈마디 이탈을 치유하는 바움가르테(Baumgarte) 스프링-댐퍼 서보(Servo) 제어

1970년대, 바움가르테(J. Baumgarte)라는 시뮬레이션 역학의 천재는 이 드리프트 현상을 막기 위해 순수 수학적 가속도 방정식에 피드백 제어 공학의 혼을 불어넣었다. 그는 단순히 \ddot{\Phi} = 0 이라는 식을 풀게 강제하지 않고, 현재 오차만큼 스프링과 댐퍼로 당겨서 구속 곡면으로 돌아오게 만드는 인공적인 역학 페널티를 만들어냈다.

\ddot{\Phi} + 2\alpha \dot{\Phi} + \beta^2 \Phi = 0

(\alpha, \beta > 0: 인위적 페널티 댐핑 계수 및 고유 진동수 상수)

이것이 그 유명한 바움가르테(Baumgarte) 안정화 수식이다.

2. 복원 제어 매트릭스(\gamma^*) 기반 드리프트 영점 회귀 진동 억제망

원래의 가속도 제약식 \ddot{\Phi} = \mathbf{\Phi_q}\ddot{\mathbf{q}} + \frac{d}{dt}(\mathbf{\Phi_q})\dot{\mathbf{q}} = 0 이었던 우변 항에 바움가르테 수식을 밀어 넣으면, DAE 솔버가 풀어야 할 진짜 목표 \gamma 가 \gamma^* 로 교정(Correction)된다.

  • 시스템이 완벽하게 연결 상태를 잃지 않았다면 (\Phi=0, \dot{\Phi}=0), 이 수식은 고전적 미립자 역학과 완전히 동일하게 작동한다.
  • 하지만 수치 적분기가 미세한 오차로 링크 끝을 1mm 떨어뜨려 버린 순간(\Phi \neq 0), 그 탈선은 2계 선형 상미분 방정식의 감쇠 거동(\alpha, \beta)에 의해 제어 이론의 PID 제어기처럼 즉각적으로 영점(0)을 향해 필사적으로 복귀한다.

바움가르테 기법은 구속 위반 시스템을 수학적 구배(Gradient) 계곡의 가장 밑바닥, 즉 \Phi=0 이라는 안정점 표면 계곡 한가운데로 블랙홀처럼 잡아당기는 유령 스프링-댐퍼 시스템이다. 이 오차 복원력 공학 덕분에 1개월 내내 돌리는 나사(NASA)의 다물체 수치 우주 궤도 시뮬레이션도, 아무리 강한 충격을 묘사하는 자동차 충돌 시뮬레이터도 차체 프레임이 공간 내에서 해체되지 않고 질긴 기구학적 이음새(Joint Array)를 절대적으로 온전하게 보존할 수 있다.
