2.3.1.1.3 적분 경로가 특정되는 비홀로노믹(Non-holonomic) 미분 구속과 마찰 없는 구름(Rolling) 한계

홀로노믹 기하 대수 방정식(f(q,t)=0)으로 변수를 묶고 자유도(DOF)를 소거해 버리는 쾌감은, 오토바이나 외발 자전거(Unicycle)가 땅을 구르는 무미끄럼 구름(Rolling without Slipping) 현상을 만나는 순간 산산이 부서진다.
바퀴가 돌아갈 때 속도 변수들 사이에는 기하학적 미분 관계, 즉 v_x = R\dot{\theta}\cos\phi, v_y = R\dot{\theta}\sin\phi 와 같은 미분 구속 텐서망이 단단하게 얽혀 있다.

수학자는 이 미분 방정식군(\sum A_i dq_i + A_0 dt = 0)을 시간 t 에 대해 적분하여 깔끔한 홀로노믹 위치 대수식(f(q)=0)으로 돌려놓고 변수를 소거하려 시도할 것이다. 그러나 아무리 적분하려 발버둥 쳐도, 바퀴의 궤적이 지나간 경로(Path) 자체에 철저히 종속되어 결코 닫힌 해(Closed-form Solution)로 통합되지 않는 악질적인 구속. 속도(미분)는 구속하지만 위치는 속박하지 못해 자유도 숫자를 1개도 줄이지 못하는 족쇄, 이것이 바로 비홀로노믹 구속 조건(Non-holonomic Constraint) 이다.

1. 적분 불가능(Non-integrable) 미분 구속과 잉여 좌표의 역설

탁자 위의 구형(Sphere) 공을 상상해 보라. 이 공은 구르면서(미니멀리즘 2차원 구름 속도 제약) 탁자 위 어떤 (x, y) 지점이든 도달할 수 있고, 도달한 그 지점에서 모든 방향(\theta, \phi, \psi)으로 방향을 틀어 정지해 있을 수 있다. 이 시스템의 위치를 규명하려면 5개의 변수(x, y, \theta, \phi, \psi)가 모조리 필요하다. 동역학적 자유도는 구름 속도 제한 때문에 단 3개뿐인데, 시스템의 기하 묶임을 표현하려면 무려 5개의 일반화 좌표 스칼라를 써야 하는 기괴한 차원 불일치 버그가 터져 나온다.

이처럼 비홀로노믹 라그랑주 역학은 위치 좌표의 수(n) > 운동의 순수 자유도(n-m) 라는 치명적 모순을 내포한다.
방정식을 대수적으로 소거(x = f(\dots))할 수 없기 때문에, 우리는 쓸모없는 과잉 잉여 좌표(q_k)들을 그대로 매다 놓은 채로 복잡한 편미분 운동 방정식을 돌려야만 한다.

2. 미정계수법(Lagrange Multipliers) 개입의 맹점

비홀로노믹 구속을 무시하고 일반화 좌표 방정식을 쓸 수는 없다. 바퀴가 미끄러지지 않게 붙잡는 미지의 마찰 구속력(Constraint Force) 벡터를 시뮬레이터에 구겨 넣기 위해, 역학에서는 \sum A_i dq_i = 0 이라는 미분(속도) 구속식을 이용해 인위적으로 일(Work)에 결합시키는 라그랑주 미정계수(\lambda) 를 동원한다.

\frac{d}{dt} \left( \frac{\partial L}{\partial \dot{q}_k} \right) - \frac{\partial L}{\partial q_k} = Q_k + \sum_{j=1}^{m} \lambda_j a_{jk}

수식 끝에 매달린 \lambda_j a_{jk} 가 바로 시스템이 “미끄러지지 않고 굴러라!“라고 소리치며 바닥에서 뿜어내는 비홀로노믹 구속력(마찰력)의 실체다.

자동차 주차를 생각해 보라. 옆구리 방향 진입 속도가 0이라는 비홀로노믹 제약(\dot{y} = 0)이 있지만, 지그재그 전후진 기동(적분 경로 우회)을 수없이 반복하면 차는 결국 평행 주차 공간 깊숙이(y 변이 획득) 안착할 수 있다. 속도 제약에 갇혔지만, 시간 경로만 틀어내면 위치 제약은 사라지는 가장 경이로우며 시뮬레이터를 뻗게 만드는 공학적 카오스 족쇄. 그것이 바로 바퀴, 스케이트, 트레일러 등 모든 조향 구동 시스템을 지배하는 비홀로노믹 미분 공간이다.
