2.3.1.1.8 레오노믹(Rheonomic) 시간 종속적 구속계에서의 라그랑지안(Lagrangian) 역학 비보존 에너지파

대부분의 교과서에서 만나는 단순한 진자나 미끄러지는 블록(Block) 시스템들은 모두 공간 기하학의 벽면이 시간에 따라 변하지 않는 ‘스클레로노믹(Scleronomic, 고정 구속)’ 시스템이다. 이 정적인 구속 조건하에서는 라그랑지안(Lagrangian, L = T - V) 역학계가 보장하는 절대 명제, 즉 닫힌 고립계 내부의 역학적 보존 에너지(Mechanical Energy, E = T + V = \text{const}) 보존 법칙이 아름답게 성립한다.

그러나 구속 곡면 자체가 지진처럼 요동치거나 팽창하는 순간, 이 보존의 신화는 무참히 깨진다. 이를테면, 진자의 매달린 피벗(Pivot) 점 자체를 누군가가 오실레이터 모터에 매달아 위아래로 미친 듯이(y_0 = A \sin \omega t) 강제로 흔들어대는 상황을 상상해 보라. 이 시스템의 구속 조건은 레오노믹(Rheonomic, 시간 종속 구속) 이 되며, 외부 모터가 진자에게 시시각각 에너지를 무지막지하게 때려 박거나 뽑아내게 된다.

1. 운동 에너지(T)의 시간 종속성과 동차 함수(Homogeneous Function) 파괴

스클레로노믹 구속에서 운동 에너지 T 는 오직 일반화 속도 \dot{q} 에 대한 순수한 이차 동차 함수(Quadratic Form) 였다(T = \frac{1}{2} \sum M_{ij} \dot{q}_i \dot{q}_j).
하지만 기차 트랙 자체가 시간에 따라 회전하는 코리올리 계나 피벗이 움직이는 레오노믹 시스템에서는 변환 방정식 \mathbf{r} = \mathbf{r}(q_1, \dots, q_n, t) 에서 오인된, 속도 수식 끝에 불순물처럼 들러붙는 무시무시한 편미분 시간 종속항(\partial \mathbf{r} / \partial t)이 튀어나온다.

이 시간 꼬리뼈 때문에 운동 에너지 T 의 무결한 이차식 형태가 산산조각 나며, 0차항(T_0)과 1차항(T_1, 자이로-코리올리 교차항)이 불연속적으로 섞여 들어가는 타락을 겪는다.

T(\mathbf{q}, \dot{\mathbf{q}}, t) = T_2 + T_1 + T_0

이 시점부터 우리가 숭앙해 온 라그랑지안 운동 에너지 T 와 해밀토니안(Hamiltonian, H = \sum \dot{q}_k p_k - L) 상수 보존계는 철저히 파탄 난다.

2. 라그랑지안 전미분(Total Derivative) 에너지 시간 팽창파(\partial L / \partial t)

더 이상 총에너지 E 는 상수가 아니다. 시스템 에너지가 시간 t 에 대해 어떻게 파동치는지를 나타내는 해밀토니안의 전미분 정리는 다음과 같이 에너지가 끊임없이 출렁이는 변화율(Power Flow)을 고발한다.

\frac{dH}{dt} = - \frac{\partial L}{\partial t}

  • \partial L / \partial t = 0: 스클레로노믹계. 구속이 가만히 있으므로 에너지가 외부로 새 나가지 않고 보존된다(dH/dt = 0).
  • \partial L / \partial t \neq 0: 레오노믹계. 외부에서 구속 조건(레일, 피벗)을 쥐고 흔드는 찰나마다, 시스템의 해밀토니안(마치 에너지와 같은 양)은 증가하기도 하고 소멸하기도 하는 비보존의 폭풍 속으로 내던져진다.

놀이공원의 자이로드롭 의자에 앉아 있는 사람이나, 자동차의 차체(바디)와 연결된 현가장치의 움직임을 서술할 때 지면 자체가 노면을 따라 강제 진동(Base Excitation)을 인가하면 그 시스템은 완벽한 레오노믹 구속이다. 모터나 지진이 끊임없이 좌표의 벽을 강제로 축소-팽창시키며 “일(Work)“을 쑤셔 넣고 있기 때문에, 이 방정식의 라그랑지안에서 \partial L / \partial t \neq 0 은 곧 폭발하는 외부 에너지 펌핑 엔진(Pumping Engine)의 심장 박동과도 같다.
