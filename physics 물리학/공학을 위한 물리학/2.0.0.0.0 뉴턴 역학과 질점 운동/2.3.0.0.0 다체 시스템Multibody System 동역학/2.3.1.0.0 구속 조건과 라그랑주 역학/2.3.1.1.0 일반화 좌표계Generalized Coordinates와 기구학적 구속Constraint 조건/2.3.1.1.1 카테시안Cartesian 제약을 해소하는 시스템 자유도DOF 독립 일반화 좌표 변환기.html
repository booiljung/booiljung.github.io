2.3.1.1.1 카테시안(Cartesian) 제약을 해소하는 시스템 자유도(DOF) 독립 일반화 좌표 변환기

가장 기초적인 데카르트 좌표계(Cartesian Coordinates)는 3 차원 공간을 기술하는 데 x, y, z 라는 세 개의 직교 축을 사용한다. N개의 입자(Particle)로 구성된 복잡한 다체 시스템(Multibody System)을 이 좌표계에 올리면 총 3N 개의 독립적인 위치 변수와 모멘텀을 추적해야 한다.
하지만 기계 부품들은 뿔뿔이 흩어져 날아다니는 기체 분자가 아니다! 이 부품들은 딱딱한 관절(Hinge), 볼 조인트(Ball-and-Socket), 슬라이더(Slider)라는 ’구속(Constraint)’이라는 족쇄로 서로 단단히 묶여 있다. 구속이 늘어날수록 시스템이 마음대로 뻗어나갈 수 있는 ’자유의 방향’은 무참히 잘려 나간다.

1. 그리블러-구츠바흐(Grübler-Kutzbach) 자유도 방정식과 자유 차원의 축소

구속이 없는 3차원 공간의 3N 개의 변수에서, 이 모든 부품의 구속이 가하는 제약 방정식의 개수(m)를 모조리 빼버린 순수한 나머지 독립 변수의 개수. 이것이 바로 기계 구조의 진짜 미니멀리즘 차원, 자유도(DOF, Degree of Freedom) 다.

\text{DOF} = 3N - m

(2차원 평면 기구의 경우는 \text{DOF} = 3(N-1) - 2j_1 - j_2 로, 강체의 기본 3자유도에서 조인트 구속을 감산한다.)

만약 엔지니어가 3N 개의 무식한 카테시안 방정식을 행렬에 올리면, 그중 m 개의 식은 아무 쓸모없는 0(=0) 이거나 행렬을 불능(Singular)으로 만들어 시뮬레이션을 터뜨리는 폭탄이 된다. 그러므로 시스템 방정식을 설계할 때는 x, y, z 따위의 종속적인 종석 변수를 완전히 파기하고, 딱 DOF 개수(n)와 정확히 일치하는, 어떤 것으로도 엉켜있지 않은 완벽한 독립 스칼라들을 창조해 내야 한다. 이것이 바로 일반화 좌표(Generalized Coordinates) 벡터 \mathbf{q} = [q_1, q_2, \dots, q_n]^T 의 강림이다.

2. 변환(Transformation) 행렬과 매핑(Mapping)

일반화 좌표 q_k 는 길이 L 일 수도, 각도 \theta 일 수도, 혹은 물리적으로 보이지 않는 기하학적 투영 면적일 수도 있다. 오직 중요한 것은 그것들이 시스템 내부에서 일체의 상호 구속 간섭을 일으키지 않는 순수한 직교 차원이라는 사실이다.
기존의 유물인 3N 개의 위치 변수 \mathbf{r}_i 들은, 이 새롭고 가벼운 일반화 좌표 q_j (단 n 개)와 시간 t 의 결합 매핑 함수로 전부 완벽하게 복원(역변환)될 수 있다.

\mathbf{r}_i = \mathbf{r}_i (q_1, q_2, \dots , q_n, t) \quad (i = 1, 2, \dots , N)
