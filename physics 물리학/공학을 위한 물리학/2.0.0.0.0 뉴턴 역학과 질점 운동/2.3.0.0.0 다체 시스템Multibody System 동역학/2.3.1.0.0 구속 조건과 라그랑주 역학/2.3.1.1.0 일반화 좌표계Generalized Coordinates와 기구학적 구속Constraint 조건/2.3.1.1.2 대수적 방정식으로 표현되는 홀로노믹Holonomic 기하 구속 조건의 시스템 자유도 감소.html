2.3.1.1.2 대수적 방정식으로 표현되는 홀로노믹(Holonomic) 기하 구속 조건의 시스템 자유도 감소

자유도(DOF)를 결정짓고 속박하는 이 구속(Constraint)이라는 속성은 기하 공간에서 과연 어떤 수학적 형태를 취하는가? 궤도 열차가 움직이는 레일이나, 정해진 길이 안에서만 움직이도록 고정된 로봇 링크처럼 속도장(\dot{q})이나 미분항 따위가 모조리 제거된 채, 오직 시스템의 위치 좌표군(q_k)과 시간(t) 만으로 그 족쇄를 서술할 수 있는 구속을 홀로노믹 구속 조건(Holonomic Constraint) 이라 통칭한다.

f(q_1, q_2, \dots, q_n, t) = 0

단순 진자(Simple Pendulum)를 살펴보라. 추가 움직이는 궤적은 x^2 + y^2 - L^2 = 0 이라는, 미분이 전혀 없는 강박적 대수(Algebraic) 방정식, 순수한 기하학 방정식 하나로 완벽히 속박되어 있다.

1. 홀로노믹 구속의 강력한 차원 절단(Reduction) 효과

홀로노믹 구속의 가장 큰 공학적 가치는, 이것이 뉴턴의 미분 방정식에서 미지의 ‘구속력(Constraint Force)’ 항을 완전히 없애버릴 수 있게 해주는 마법의 대수(Algebraic) 스위칭 도구라는 데 있다.
식 f(q_k, t)=0 형태로 시스템이 한 겹 묶여 있다면, 우리는 곧바로 방정식에서 종속 변수 x 를 \sqrt{L^2 - y^2} 으로 치환하여 변수 하나를 영구적으로 소멸시킬 수 있다.

  • 만약 시스템에 N 개의 구속 방정식이 홀로노믹하다면,
  • 수학적으로 N 개의 종속 좌표계가 완벽하게 탈락한다.
  • 즉, 시스템의 전체 자유도(DOF)가 그 구속 방정식의 다중도만큼 정확하게 정수 단위로 칼같이 절단된다.

2. 시간 종속성에 따른 레오노믹(Rheonomic)과 스클레로노믹(Scleronomic) 분기점

비록 홀로노믹 대수식은 속도 \dot{q} 항이 없지만, 시간 t 가 변수로 개입하느냐에 따라 시스템의 역학 진폭은 극명하게 갈린다.

  • 스클레로노믹(Scleronomic) 구속:
    위의 식에서 t 항마저 없는, 시간이 흘러도 공간 기하 장벽이 끄떡없이 멈춰있는 고정 구속식이다 (f(q_1, \dots, q_n) = 0). 외부에서 이 시스템에 강제로 에너지를 퍼붓는 기계 장치가 없다. 오직 보존계 라그랑주 역학만이 파동친다.
  • 레오노믹(Rheonomic) 구속:
    식에 \dots + t) = 0 처럼 외부의 모터 등에 의해 시간에 따라 구속 궤도가 인위적으로 틀어지고 요동치는 비정상 궤도계다. 예컨대 기차가 달리는 레일 자체가 시간에 따라 상하로 요동치는 지진파를 맞고 있다면 그 레일의 방정식은 레오노믹 구속이 된다. 이 시스템 안의 롤러코스터는 가만히 있어도 레일의 시간 편미분에 의해 에너지를 흡수-상실하므로 에너지 보존을 강제로 파괴시킨다.

산업 현장에서 공학자가 마주치는 유니버설 조인트 구속, 로봇 암의 핀 조인트 모델 등 다물체 동역학계(MBD) 공간을 지배하는 제약의 99\% 는 방정식 f=0 으로 치환될 수 있는, 통제하기 쉽고 자유도를 칼같이 도려내는 ‘착한 구속’, 즉 홀로노믹 구속이다. 일반화 좌표 공간 \mathbf{q} 로의 도시는 오직 이 등식들의 성립을 전제로만 작동한다.
