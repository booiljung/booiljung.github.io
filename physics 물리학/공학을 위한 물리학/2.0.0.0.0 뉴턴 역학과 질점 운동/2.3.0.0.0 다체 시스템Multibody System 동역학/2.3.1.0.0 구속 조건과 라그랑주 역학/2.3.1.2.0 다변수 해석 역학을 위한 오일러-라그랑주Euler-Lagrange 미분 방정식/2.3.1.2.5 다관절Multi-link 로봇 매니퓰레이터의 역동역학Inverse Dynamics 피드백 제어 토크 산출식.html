2.3.1.2.5 다관절(Multi-link) 로봇 매니퓰레이터의 역동역학(Inverse Dynamics) 피드백 제어 토크 산출식

산업 공단에서 6자유도 직렬 편전 다관절 로봇(Serial Manipulator)이 자동차 문짝을 쥐고 \pm 0.1 \text{mm} 의 오차 없이 허공을 가로지르는 기하학적 궤적을 그리려면, 모터(Actuator)는 그 무거운 철덩어리가 휘두를 때 뿜어져 나오는 관성력(Inertia), 지구 중력(Gravity), 그리고 관절끼리 회전하며 요동치는 원심력과 코리올리(Coriolis) 요동을 매 0.001 초마다 계산해서 정확히 상쇄(Cancel-out)하는 반발 토크 \tau(t) 를 뱉어내야 한다.
로봇 공학에서 원하는 “궤적(q, \dot{q}, \ddot{q})이 이미 주어졌을 때”, 그것을 실현하기 위한 “모터의 토크 타겟(\tau)을 역으로 계산“하는 극악무도한 미적분 퍼즐, 이것이 바로 역동역학(Inverse Dynamics) 편미분이다.

로보틱스 1분과, 우리는 오일러-라그랑주 진영이 선사한 완벽한 텐서 방정식(행렬 연립 시스템)을 지배함으로써 이 역동역학 피드백 제어기의 심장을 렌더링한다.

1. 다관절 로보틱스 오일러-라그랑주 행렬계(Dynamic Matrix Equation)

수많은 링크 체인이 결합된 n-자유도 운동 방정식의 구조는 L = T - V 라는 단 하나의 스칼라 파동에서 터져 나와, 결국 아래와 같은 고전 역학계 순수 매트릭스 \mathbf{M-C-g} 블록으로 강제 팽창 분해된다.

\mathbf{\tau} = \mathbf{M}(\mathbf{q}) \ddot{\mathbf{q}} + \mathbf{C}(\mathbf{q}, \dot{\mathbf{q}}) \dot{\mathbf{q}} + \mathbf{g}(\mathbf{q}) + \mathbf{J}^T(\mathbf{q}) \mathbf{F}_{ext}

  1. 질량-관성 행렬 \mathbf{M}(\mathbf{q}):
    각 관절이 얼마나 무겁게 느껴지는지를 정의하는 n \times n 대칭(Symmetric) 및 양의 정부호(Positive Definite) 행렬이다. 팔을 쭉 뻗으면 관성이 커져 회전하기 힘들어지듯, 이 행렬은 오직 로봇의 ’자세(\mathbf{q})’에 전적으로 종속되어 시시각각 비선형적으로 변동한다.
  2. 원심력 및 코리올리 행렬 \mathbf{C}(\mathbf{q}, \dot{\mathbf{q}}):
    속도 \dot{\mathbf{q}} 의 제곱항 텐서(\Gamma_{ijk} 크리스토펠 기호 기반)들의 무덤이다. 로봇 허리가 빠르게 돌 때 어깨가 밖으로 꺾여 날아가려는(v^2/R) 원심력이 뿜어져 나오고, 두 관절이 동시에 회전할 때 \dot{q}_1 \dot{q}_2 로 역회오리치는 코리올리 힘(Coriolis Force)이 토크 모터를 사정없이 뒤틀어버린다.
  3. 중력 구배 벡터 \mathbf{g}(\mathbf{q}):
    포텐셜 에너지 V 에서 유도되어 지구 중력이 관절을 아래로 짓누르는 힘이다. 자세(\mathbf{q})에 따라 모터가 이겨내야 하는 모멘트 암(Moment Arm)이 달라진다.
  4. 엔드 이펙터(End-effector) 외력 투영 \mathbf{J}^T \mathbf{F}:
    로봇 끝단에 닿은 용접건이 벽을 미는 외부 힘(\mathbf{F}_{ext})은, 기구학적 자코비안 행렬의 전치(\mathbf{J}^T)를 타고 각 관절의 모터 토크(\mathbf{\tau})로 고스란히 치환 환산된다.

2. 역동역학 피드백 제어(Feedback Linearization / Computed Torque Control)

이 악마 같은 비선형 방정식(Non-linear M, C, G) 탓에, 단순한 공업용 PID 제어 모터를 그대로 달면 로봇 팔이 고속으로 움직일 때 궤도를 이탈하여 허공을 휘젓고 파괴된다.
역동역학 제어의 궁극, 이른바 계산 토크 제어(Computed Torque Control) 는 로봇의 두뇌 컴퓨터 모터 드라이버 안에 이 오일러-라그랑주 비선형 행렬을 통째로 이식(Copy)하는 데서 출발한다.

드라이버가 로봇에게 주는 명령 토크 \tau 를 무식하게 전압만 높이는 것이 아니라, 아래처럼 인공지능식으로 오일러 방정식 역산 수식을 덮어씌워 출력한다.

\tau_{control} = \underbrace{ \mathbf{\hat{M}}(\mathbf{q}) \left[ \ddot{\mathbf{q}}_d + \mathbf{K}_v \dot{\mathbf{e}} + \mathbf{K}_p \mathbf{e} \right] }_{ \text{관성-가속도 PID 선형화} } + \underbrace{ \mathbf{\hat{C}}(\mathbf{q}, \dot{\mathbf{q}})\dot{\mathbf{q}} + \mathbf{\hat{g}}(\mathbf{q}) }_{ \text{비선형 외란 상쇄 텐서} }

  • \mathbf{e} = \mathbf{q}_d - \mathbf{q}: 목표 궤도 궤적(d)과 실제 센서(q) 위치 오차.

저항을 없애겠다며 마찰력 공식을 역으로 거꾸로 더해주면 저항이 0이 되는 원리다. 컴퓨터가 \hat{M}, \hat{C}, \hat{g} 라는 완벽한 수학적 로봇 쌍둥이(Digital Twin 역학 모델)를 내장한 채, 시시각각 변하는 비선형의 찌그러짐을 ‘더해서’ 지워버린다. 그러면 저 미친 비선형 로봇 시스템은 그 즉시 \ddot{\mathbf{e}} + \mathbf{K}_v \dot{\mathbf{e}} + \mathbf{K}_p \mathbf{e} = 0 이라는, 학부 2학년도 눈을 감고 제어할 수 있는 아름다운 선형(Linear) 독립 2계 스프링-댐퍼 상미분 방정식으로 파괴적 환골탈태를 마친다. 오일러-라그랑주는 이렇게 쇳덩어리 기계의 영혼을 선형 대수학의 빛으로 해독해 냈다.
