2.3.1.2.6 스틱-슬립(Stick-Slip) 등 불연속 마찰이 존재하는 다체 계열 미분 대수 방정식(DAE) 헤센

오일러-라그랑주 시스템 역학은 모든 물체의 움직임이 연속적이고 부드럽게 미분 가능하다(C_2 이상)는 아름다운 플라톤 철학 공간 위에 성립되어 있다. 하지만 공학자들이 시뮬레이터를 돌릴 때 부딪히는 지옥 문은, 브레이크 패드가 긁히거나 모터 베어링이 녹이 슬었을 때 발생하는 강력한 불연속 마찰(Discontinuous Friction) 이 개입하는 직후다.

쿨롱(Coulomb) 드라이 마찰은 ’속도의 방향’에 따라서 거대한 덩어리 상수값(\mu F_N)만큼의 스텝(Step) 함수로 점프하며(\text{sgn}(\dot{q})), 정지한 부품은 외부 힘이 최대 정지 마찰 한계치를 뚫을 때까지 절대 한 치도 움직이지 않는 ‘점착(Stick)’ 상태와 ‘미끄러지는(Slip)’ 상태 사이를 무한대로 변덕스럽게 오간다. 시뮬레이터 수치 적분기 입장에서는 해밀토니안의 속도 구배(Jacobian, Hessian)가 ’0’이나 무한대로 요동치는(Singularity) 연속성의 학살 현장이다.

1. 쿨롱 스틱-슬립(Stick-Slip) 변환의 스위칭 구속력 함수 매트릭스화

운동 방정식 행렬 \mathbf{M}(\mathbf{q}) \ddot{\mathbf{q}} = \mathbf{Q}_{active} + \mathbf{F}_{friction} 내부에 쿨롱 마찰 스펙트럼 블록 모델링을 삽입하기 위해, 마찰력은 속도 \dot{q} 의 늪에서 아래와 같이 두 가지의 기계(Machine) 상태 텐서로 쪼개진다.

  • 슬립(Slip) 상태: \dot{q} \neq 0
    마찰 방향은 \dot{q} 와 반대이며 크기는 일정하다.
    F_f = -\mu_k F_n \cdot \text{sgn}(\dot{q})
  • 스틱(Stick) 상태: \dot{q} = 0
    이때부터 마찰력은 \mu_s F_n 이 아니라, \mathbf{Q}_{active} 가 얼마로 밀든 기계를 정지 유지시키기 위한 “미지의 피드백 부반력 계수” 즉 추가적인 신종 라그랑주 미정계수(\lambda_{stick}) 벡터망으로 스위칭된다.
    정지 마찰 상한계수(-\mu_s F_n \le \lambda_{stick} \le \mu_s F_n)를 초과하기 전까지, \mathbf{\ddot{q}} 식 자체를 통째로 0 으로 락다운(Lock-down) 시킨 후 미정계수 크기를 탐색한다. 이 과정에서 헤센(Hessian) 역행렬은 궤도를 멈춘다.

2. 불연속 이탈의 평활화 방어: 카르노-루그리(Karnopp-Lugre) 모델 곡률 근사망

진짜 \dot{q} = 0 을 판별하는 조건문(If-Else) 스위칭 루프 스펙트럼을 만들면, MBD(다물체 역학계) Runge-Kutta 적분 엔진 속도 스텝이 10^{-6} 의 오차로 무한 진동하며 시간(Stiff) 행렬의 한계를 못 이겨 챗터링(Chattering / Zeno behaviour) 에러로 폭파되어 버린다.

  1. 카르노(Karnopp) 평활 역학 모델: \dot{q} 가 정확히 0이 아니라, 아주 미세한 |\dot{q}| < \epsilon 구간 폭 텐서 영역 안에 들어오면 인위적 점착(Stick) 마찰 상한선으로 강제 병합하여 연속성 스위치를 보존시킨다.
  2. 루그리(Lugre) 모델 (동적 브리슬 상태): 표면 사이 마이크로 단위의 미세 털(Bristle)들이 구부러지는 미분 스프링-댐퍼 동적 상태 변수(z) 융합 루프를 투입한다.
    \dot{z} = v - \frac{\sigma_0 |v|}{g(v)} z, \quad F_{friction} = \sigma_0 z + \sigma_1 \dot{z} + \sigma_2 v
    단일 변수 대신 비선형 미분 변수 z 를 동역학 매트릭스에 한 줄 더 얹어서, 마찰의 역전(Break-away) 충격과 Stribeck 효과, 프레팅 궤도(Fretting)까지 매끄러운 2계 미분 곡선 도함수망 안으로 전부 복원, 녹여버린다.
