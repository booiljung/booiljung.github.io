2.3.1.2.8 극좌표 및 구면 좌표계 등 직교하지 않는 텐서(Tensor) 체계에서의 크리스토펠 기호 기반 역학

우주 왕복선이 지구를 공전할 때, 데카르트 좌표계(x,y,z)로 그 궤적의 운동 에너지(T = \frac{1}{2}m(\dot{x}^2 + \dot{y}^2 + \dot{z}^2))를 적고 뉴턴 방정식을 푸는 것은 공간의 구형(Spherical) 대칭성을 무참히 짓밟는 행위다. 그래서 엔지니어는 극좌표계(r, \theta)나 구면 좌표계(r, \theta, \phi)라는 영구 곡선 좌표계를 소환한다.

하지만 이때 운동 에너지는 T = \frac{1}{2}m(\dot{r}^2 + r^2\dot{\theta}^2) 와 같이 반경 r 이라는 좌표 자체의 찌그러짐 함수(Metric Constant)를 품게 된다. 운동 에너지가 선형적(M\dot{q}^2)이지 않고 변수에 매달린 곡률 텐서(g_{ij}(q))로 돌변해버리기 때문에, 오일러-라그랑주 분쇄기 속으로 들어갔다 나오면 단순 평행 이동에는 결코 나타나지 않았던 기괴한 비선형 원심 가속도와 코리올리 가속도 구배들이 무더기로 쏟아져 나온다.

이 곡률 항의 폭격을 수식적으로 완벽히 통제하고 정의하는 미분 기하학의 연산 연장(Tool), 그것이 바로 계량 텐서(Metric Tensor)와 자코비 미적분의 조합으로 탄생한 크리스토펠 기호(Christoffel Symbols, \Gamma_{ijk} ) 이다.

1. 계량 텐서(g_{ij})와 운동 에너지의 비유클리드화(Non-Euclidean Form)

일반화 좌표공간 \mathbf{q} = [q_1, \dots, q_n]^T 를 무한소 전진(ds^2) 시킬 때, 직교 좌표의 단순 dx^2+dy^2 처럼 일정하지 않고, 각 변수의 위치(q_m)에 따라 스케일이 늘어나거나 축소되는 비례-결합 상수를 계량 텐서(Metric Tensor) g_{ij}(\mathbf{q}) 라 부른다.

이 좌표 변환에 따르면, 운동 에너지는 다음과 같은 곡선 공간 내적 행렬로 압축된다.
T = \frac{1}{2} \sum_{i=1}^{n} \sum_{j=1}^{n} g_{ij}(\mathbf{q}) \dot{q}_i \dot{q}_j

  • 구면 좌표계(q_1=r, q_2=\theta, q_3=\phi)의 경우:
    g_{11} = 1, \quad g_{22} = r^2, \quad g_{33} = r^2 \sin^2\theta, 나머지 교차항 g_{i \neq j} = 0.

2. 코리올리-향심력 분기 편미분: 크리스토펠 기호 텐서 합(\Gamma_{ijk})

앞서 구한 이 징그러운 비선형 운동 에너지 T 를 오일러-라그랑주 식 \frac{d}{dt} \left( \frac{\partial T}{\partial \dot{q}_k} \right) - \frac{\partial T}{\partial q_k} 에 밀어 넣고 미분에 미분을 거듭(체인 룰 폭격)하면, \ddot{q} 항(순수 관성가속도) 외에도 속도들이 제곱으로 곱해진 비선형 잡음 텐서 \dot{q}_i \dot{q}_j 들이 떨어져 내린다.

이 비선형 잡음 텐서 배열 계수에 미분 기하의 곡선 미분학인 크리스토펠 기호 제1종 \Gamma_{kij} 을 씌운다.

\sum_{j} g_{kj}(\mathbf{q}) \ddot{q}_j + \sum_{i} \sum_{j} \underbrace{ \frac{1}{2}\left( \frac{\partial g_{ki}}{\partial q_j} + \frac{\partial g_{kj}}{\partial q_i} - \frac{\partial g_{ij}}{\partial q_k} \right) }_{ \Gamma_{kij} } \dot{q}_i \dot{q}_j = Q_k

  • \Gamma_{k i i} \dot{q}_i^2: 속도의 단순 제곱. 원운동 반경을 탈출하려는 원심력(Centrifugal Force, m r \dot{\theta}^2) 을 토해낸다.
  • \Gamma_{k i j} \dot{q}_i \dot{q}_j (i \neq j): 두 개의 다른 속도 축이 교차(예: 원 반경 연장 \dot{r} + 회전 \dot{\theta})할 때 발생하는 코리올리 힘(Coriolis Force, 2m \dot{r} \dot{\theta}) 이 쏟아져 나온다.

크리스토펠 텐서는 뉴턴을 끝없이 괴롭혔던 ’비관성계의 가속도 벡터 혼돈’을 그저 g_{ij} 의 미분 연산 몇 줄로 해결해 버린 기하학적 학살 파동이다. 로봇의 3개 축이 뒤스윙하며 복잡한 우주 궤도선을 그릴 때, 로봇 제어기는 이 크리스토펠 기호 배열을 MBD 코리올리 매트릭스 \mathbf{C}(\mathbf{q}, \dot{\mathbf{q}}) 에 박아놓음으로써 수치 절단 오차 제로에 가까운 천체의 운동을 재현해 낸다.
