2.3.1.2.1 달랑베르(d’Alembert) 원리에서 다체 시스템 라그랑지안(L = T - V) 함수로의 뉴턴 역학 전이

가상 일의 원리 \sum \mathbf{F}_i^{(a)} \cdot \delta\mathbf{r}_i = 0 은 오직 정지한(Static) 평형 시스템에서만 통용되던 폐쇄적 수식이었다. 장 르 롱 달랑베르(Jean le Rond d’Alembert)는 움직이는 동역학 시스템의 가속도 \mathbf{a}_i 를 뉴턴의 제2법칙 \mathbf{F}_i = m_i \mathbf{a}_i 에서 뒤집어, **“관성력(-m_i \mathbf{a}_i)을 마치 또 하나의 능동 외력(Active Vector)처럼 취급해버리자”**는 발칙한 논리를 세웠다.

즉 \mathbf{F}_i - m_i \mathbf{a}_i = \mathbf{0} 이라는 식을 통해, 우주를 날아가는 로켓이나 진동하는 엔진의 미친 듯한 가속도 공간을 마치 관성력이 시스템을 정지시키고 있는 정적 평형(Virtual Equilibrium) 공간으로 강제 환원시켜 버린 것이다.

1. 달랑베르-라그랑주 거시 미분 적분 전개

달랑베르의 가상 일의 원리 확장에 따라, 구속력을 완전히 무시한 동력학적 가상 일의 대수합 수식은 이렇게 진화한다.
\sum_{i=1}^{N} (\mathbf{F}_i^{(a)} - m_i \ddot{\mathbf{r}}_i) \cdot \delta \mathbf{r}_i = 0

이제 해석 역학계는 이 무자비한 뉴턴 벡터 방정식 \mathbf{r}_i 들을, n 개의 티끌 없는 독립 일반화 좌표 q_j 의 세계로 탈바꿈시킨다. 체인 룰(역변환 매핑)과 편미분의 기나긴 조립, 특히 관성력 텐서 -m \ddot{\mathbf{r}} \cdot \delta \mathbf{r} 를 좌표계 속도 \dot{q} 와 위치 q 에 대한 시간에 대한 전미분 형태로 끌어올리는 극도로 복잡한 치환 적분을 거치게 된다.

2. 뉴턴의 벡터 족쇄를 끊은 순수 에너지 스칼라 역학의 도래

그 기나긴 적분의 늪을 빠져나오는 순간, 공간을 지배하던 수백 개의 벡터 기호들은 \sum 이라는 묶음 아래로 전부 빨려 들어가고, 오직 스칼라 에너지 함수인 운동 에너지(T) 와 보존력이 행사하는 포텐셜 위치 에너지(V) 두 항만이 잿더미 속에서 떠오른다.

라그랑주는 이 두 스칼라 기능 덩어리를 합쳐, 자신만의 절대 함수 라그랑지안(Lagrangian, L) 을 창조한다.

L(\mathbf{q}, \dot{\mathbf{q}}, t) = T - V

이 오일러-라그랑주 거시 미분 방정식은 기계 공학을 르네상스로 이끈 결정타였다. 조인트마다 걸린 구속력 벡터를 구하기 위해 머리를 싸매던 뉴턴식 도해법은 폐기되었다. 다체 시스템이 아무리 복잡하게 얽혀 있어도, “그 시스템 전체의 운동 에너지 T 는 몇 줄의 수식인가?”, “스프링과 중력이 만든 위치 에너지 V 는 수식으로 무엇인가?” 이 두 가지만 L = T - V 에 밀어 넣으면, 그 자체로 시스템이 굴러갈 우주의 결정론적 궤적이자 오일러 방정식이 완벽하게 토해져 나온다.
