2.3.1.2.4 마찰 및 점성 저항 등 비보존 일반화 힘 적용 레일리 방산 함수

라그랑지안(L = T - V) 시스템은 위치 좌표(q)의 제곱이나 속도(\dot{q})의 제곱에 절대적으로 복속되는 아름다운 대칭 우주라 여겨지지만, 현실의 쇠망치와 구리스 점성 오일 댐퍼가 가동하는 그 순간 에너지는 열경계로 무자비하게 흩어지며 비가역적으로 붕괴한다. 포텐셜 함수 V 속에 숨길 수 없는 이런 힘, 즉 공간의 방향이 아니라 경로의 속도에 종속되어 에너지를 잡아먹는 비보존력(Non-conservative Force)을 수식에 쑤셔 넣으려면 어떻게 해야 할까?

속도에 비례(F = c\dot{x})하여 감쇠 저항으로 작동하는 이상적인 점성 댐퍼(Viscous Damper)나 유체 역학의 항결(Drag) 저항은, 단순히 외력 항(Q_k)에 벡터를 넣는 대신 존 윌리엄 스트럿 제3대 레일리 남작(John William Strutt, 3rd Baron Rayleigh)이 창안한 레일리 소산 함수(Rayleigh Dissipation Function, D) 라는 무차원-절대 감쇠 스칼라 함수로 우아하게 통합 편입된다.

1. 레일리 에너지 소산 함수(D)의 동적 매트릭스 성형

레일리는 다체 시스템에 거미줄처럼 엉켜있는 수많은 점성 대시팟(Dashpot, 감쇠 계수 c_{ij}) 시스템에 대해, 운동 에너지(T)가 속도의 2차식으로 구조화된 것과 정확히 병렬적인 대칭 구조로, 속력 소산의 이차 동형 함수인 제3의 스칼라, 소산 함수 D 를 정의했다.

D = \frac{1}{2} \sum_{i=1}^{n} \sum_{j=1}^{n} c_{ij} \dot{q}_i \dot{q}_j

  • 전파되는 에너지 증발량(Power Dissipation)은 이 레일리 함수 D 의 정확히 2배다 (P_{loss} = 2D = \sum c \dot{q}^2).
  • এই 레일리 소산 함수는 각 일반화 속도 \dot{q}_k 로 편미분될 때, 바로 그 축을 잡아채는 비보존 저항-감쇠 일반화 힘 Q_{k, damping} 을 토해낸다.

Q_{k, damping} = - \frac{\partial D}{\partial \dot{q}_k}

2. 감쇠를 포괄한 완전 팽창 오일러-라그랑주 미분 역학

비보존 소산계의 정수 레일리 함수가 동역학 편미분 행렬로 스위칭되면, 에너지 보존을 증명했던 고전 라그랑지안 미분식은 다음과 같이 완전히 타락(혹은 현실화)한다.

\frac{d}{dt} \left( \frac{\partial L}{\partial \dot{q}_k} \right) - \frac{\partial L}{\partial q_k} + \frac{\partial D}{\partial \dot{q}_k} = Q_k^{nc}

(단, Q_k^{nc} 는 여전히 V 나 D 로 포섭할 수 없는, 외부 모터 토크나 바람 등의 쌩짜 비보존 능동 외력(Non-conservative Active Force))
