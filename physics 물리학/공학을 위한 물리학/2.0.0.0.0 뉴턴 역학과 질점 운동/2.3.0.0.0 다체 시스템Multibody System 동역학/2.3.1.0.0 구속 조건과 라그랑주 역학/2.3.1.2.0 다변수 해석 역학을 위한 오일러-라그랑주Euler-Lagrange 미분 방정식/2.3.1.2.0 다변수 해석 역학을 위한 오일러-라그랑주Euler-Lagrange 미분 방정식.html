2.3.1.2.0 다변수 해석 역학을 위한 오일러-라그랑주(Euler-Lagrange) 미분 방정식

가상 일의 원리(Principle of Virtual Work)가 정역학(Statics)의 구속력을 증발시키는 완벽한 해법이었다면, 움직이는 미분, 즉 가속도가 존재하는 동역학(Dynamics)에서는 이 원리를 어떻게 확장해야 할까? 뉴턴의 제2법칙(\mathbf{F} - m\mathbf{a} = 0) 가속도를 그저 반대 방향을 향해 뻗치는 역관성 힘(Inertial Force, 달랑베르의 역력)으로 둔갑시켜 시스템 정적 평형(Virtual Work = 0)에 욱여넣는 순간, 고전 역학의 마지막 관문이 열린다.

이 챕터에서는 벡터(방향)에 얽매여 있던 공간의 투박한 \mathbf{F} = m\mathbf{a} 방정식이, 어떻게 순수한 에너지 스칼라(운동 에너지 T, 위치 에너지 V)의 편미분 텐서 공간으로 우아하게 이착륙하는지, 그리고 그 궁극의 미니멀리즘 산물인 변분법(Calculus of Variations)의 정수, 오일러-라그랑주 방정식(Euler-Lagrange Equation)의 형태 구축과 활용을 탐구한다.
