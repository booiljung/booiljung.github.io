2.3.1.2.2 일반화 속도(Generalized Velocity) 및 좌표 편미분 기반 2계 상미분 선형/비선형 공식 유도

달랑베르의 원리와 해밀턴의 최소 작용의 원리(Principle of Least Action, \delta \int L dt = 0)라는 두 거대한 산맥이 만나는 정점에서, 고전 역학을 지배하는 단 하나의 절대 방정식이 탄생한다.
시스템의 총 에너지 파동 함수인 라그랑지안(L = T - V)을 일반화 좌표 q_k 와 그것의 시간 미분인 일반화 속도 \dot{q}_k 로 찢어발기는(편미분) 이 2계 상미분 방정식은, 벡터와 관성력의 복잡성을 한 줄의 거시적 편미분 텐서로 압축해 낸다.

\frac{d}{dt} \left( \frac{\partial L}{\partial \dot{q}_k} \right) - \frac{\partial L}{\partial q_k} = 0 \quad (k = 1, 2, \dots, n)

1. 켤레 운동량(Conjugate Momentum)과 일반화 힘(Generalized Force)의 투영

오일러-라그랑주 방정식의 두 거대한 항은 각기 다른 역학적 실체를 품고 있다.

  1. \partial L / \partial \dot{q}_k (켤레 운동량, p_k):
    라그랑지안을 속도 방향으로 편미분한 값이다. 만약 q_k 가 직선 위치 좌표(x)라면 이 값은 뉴턴의 선형 운동량(mv)이 되고, q_k 가 각도(\theta)라면 이 값은 각운동량(I\omega)이 된다. 이 일반화 모멘텀을 다시 시간 dt 로 쪼갠 것(d p_k / dt)이 바로 방정식의 첫 번째 항(관성-가속력 텐서)이다.
  2. \partial L / \partial q_k (퍼텐셜 복원력 및 구심/코리올리 힘):
    라그랑지안을 위치 변수 그 자체로 편미분한 값이다. 주로 위치 에너지 V 가 미분되며 튀어나오는 탄성력이나 중력 복원력, 그리고 위치 변화에 따라 비선형적으로 출몰하는 원심력(Centrifugal Force) 벡터를 토해낸다.

2. 궤적 최적화 행렬: M(\mathbf{q})\ddot{\mathbf{q}} + \mathbf{C}(\mathbf{q}, \dot{\mathbf{q}})\dot{\mathbf{q}} + \mathbf{g}(\mathbf{q}) = \mathbf{0} 구조로의 치환

1980년대 컴퓨터 시뮬레이터(Multibody Dynamics)가 발전하면서, 이 우아한 오일러 편미분 기호 덩어리는 기계 제어 공학자들이 써먹을 수 있는 강력한 비선형 로봇 동역학(Rigid Body Dynamics) 텐서 행렬로 완전히 분해(Decomposition)된다.

  • \ddot{q} 항(가속도)들만 묶어 뽑아낸 \mathbf{M}(\mathbf{q}) 행렬은 진동하는 관성의 덩어리(Mass/Inertia Matrix)다.
  • \dot{q}_i \dot{q}_j 들이 서로 곱해져 생겨나는 잔여항 \mathbf{C}(\mathbf{q}, \dot{\mathbf{q}}) 는 곧 회전으로 인해 발생하는 비선형 코리올리 가속도와 원심력 행렬의 총체다.
  • 속도와 무관하게 \partial V / \partial q 로만 구성된 \mathbf{g}(\mathbf{q}) 는 중력(Gravity) 구배 벡터를 담담하게 가리킨다.
