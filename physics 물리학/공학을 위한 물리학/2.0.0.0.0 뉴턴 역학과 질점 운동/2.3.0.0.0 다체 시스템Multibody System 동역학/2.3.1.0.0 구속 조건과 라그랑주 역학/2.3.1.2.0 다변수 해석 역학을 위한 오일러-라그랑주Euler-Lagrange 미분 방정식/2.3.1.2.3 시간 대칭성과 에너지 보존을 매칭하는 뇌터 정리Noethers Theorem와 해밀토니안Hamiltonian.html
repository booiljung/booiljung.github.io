2.3.1.2.3 시간 대칭성과 에너지 보존을 매칭하는 뇌터 정리(Noether’s Theorem)와 해밀토니안(Hamiltonian)

오일러-라그랑주 미분 역학의 진정한 철학적 깊이는, 단순히 기계의 속도 연립 항을 계산해 내는 수학적 편리를 넘어 1915년 에미 뇌터(Emmy Noether)에 의해 밝혀진 물리학의 절대 진리, 대칭성(Symmetry)과 보존 법칙(Conservation Law)의 결합 정리에 있다.

특정 일반화 좌표 q_k 가 라그랑지안 L(\mathbf{q}, \dot{\mathbf{q}}, t) 식 안에 명시적으로 보이지 않는다면(즉, \partial L / \partial q_k = 0), 이 변수를 순환 좌표(Cyclic Coordinate) 혹은 무시가능 좌표(Ignorable Coordinate) 라고 부른다. 오일러 방정식 \frac{d}{dt} \left( \frac{\partial L}{\partial \dot{q}_k} \right) - 0 = 0 이 발동하며, 그 변수에 대응하는 켤레 운동량(Conjugate Momentum)은 영원히 상수로 보존된다 (p_k = \text{const}).

뇌터 정리는 여기서 더 나아가, 공간과 시간의 ‘대칭 이동’ 그 자체가 우주를 지배하는 스칼라 에너지와 모멘텀을 락(Lock) 걸어 묶어 치워버리는 불멸의 보존망을 형성함을 입증했다.

1. 공간 대칭성과 선형/각 운동량의 불멸

  • 공간 이동 대칭(Spatial Translation Symmetry):
    시스템이 일정한 방향(x 축)으로 한 칸 평행 이동하건 두 칸 이동하건 그 시스템 내부의 에너지가 똑같다면, 라그랑지안에는 x 변수가 없다(\partial L / \partial x = 0). 이 공간 대칭은 즉각 뉴턴의 선형 운동량 보존(p_x = const) 으로 발화한다.

  • 공간 회전 대칭(Spatial Rotation Symmetry):
    어떤 장치(예: 원통형 위성) 중심축 \theta 를 빙빙 돌려보아도 위치 에너지가 달라지지 않는다면(\partial L / \partial \theta = 0), 이 회전 대칭성은 각운동량 보존(Angular Momentum = const) 이라는 거대한 스핀의 방어벽을 축조한다.

2. 시간 이동 대칭(Time Translation Symmetry)과 해밀토니안(H) 에너지 보존

그렇다면 공간 이동이 선형 운동량, 공간 회전이 각운동량 보전을 낳는다면, 시간(Time, t) 공간을 축으로 평행 이동시켜 본다면 대칭성은 무엇을 남기는가?
라그랑지안 L(q, \dot{q}, t) 에 시간 변수 t 가 노골적으로 노출되어 있지 않은(외부에서 강제로 시스템을 흔들지 않는 스클레로노믹계) 상황(\partial L / \partial t = 0)이라면. 우리는 라그랑지안을 시간으로 전미분하는 체인 룰을 통해 새로운 보존 텐서 덩어리인, 해밀토니안(Hamiltonian, H) 을 길어 올리게 된다.

H = \sum_{k=1}^{n} \dot{q}_k \frac{\partial L}{\partial \dot{q}_k} - L = \sum_{k=1}^{n} \dot{q}_k p_k - L
\frac{dH}{dt} = - \frac{\partial L}{\partial t} = 0 \quad \Rightarrow \quad H = \text{Constant}

  • 시간 t 이동에 대한 라그랑지안 파동의 절대 대칭성(불변성)은,
  • 운동 에너지(T)가 속도의 완전 이차 동차 함수일 때, 해밀토니안 H 가 시스템의 진정한 총 역학 에너지(E = T + V)로 각성하게 만든다.
  • 즉, 시간에 대한 불변성/대칭성이 곧 역학적 시스템 “에너지(Energy)의 보존“을 증명하는 물리적 심판자가 되는 것이다.

수소 원자 궤도를 도는 전자부터 고온 원심 분리기, 그리고 궤도 역학 시뮬레이터(Orbit mechanics) 엔진에 이르기까지, 제어 코딩 엔지니어가 \partial L / \partial \theta = 0 혹은 \partial L / \partial t = 0 인 것을 찾아내면 그 즉시 미분 방정식 시스템은 한 차원 축소되어 적분할 필요가 없는 완벽한 ‘상수(Constant) 배열’ 블록으로 굳어진다. 이 대칭성의 예술은 동역학 오차를 0으로 맞추는 위대한 공학적 치트 키(Cheat Key)이며, 현대 이론 물리학이 기대고 있는 궁극의 미학이다.
