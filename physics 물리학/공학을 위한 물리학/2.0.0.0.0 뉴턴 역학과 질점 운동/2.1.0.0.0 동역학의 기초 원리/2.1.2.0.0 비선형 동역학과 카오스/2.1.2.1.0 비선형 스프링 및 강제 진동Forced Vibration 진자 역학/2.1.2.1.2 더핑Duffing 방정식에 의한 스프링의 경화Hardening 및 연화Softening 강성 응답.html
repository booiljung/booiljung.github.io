2.1.2.1.2 더핑(Duffing) 방정식에 의한 스프링의 경화(Hardening) 및 연화(Softening) 강성 응답

강철 스프링은 후크의 법칙(F=kx)을 맹목적으로 추종하는 선형적인 신화에 불과하다. 철탑의 거대한 케이블이나 나사형 코일은 한없이 늘이지거나 과도하게 압착될 때 더 이상 입력 변위에 정비례하는 반발력을 내뿜지 못한다. 강성이 스스로 단단해져 스프링 상수가 폭발적인 곡률을 그리며 치솟는 경화(Hardening), 또는 구조가 파탄나며 탄성 한계를 상실하여 상수가 꺾여버리는 연화(Softening) 현상을 수학적으로 통제하지 못한다면, 그 어떤 댐퍼로도 시스템 붕괴를 막을 수 없다.

이 비선형적인 탄성 복원력 모델의 정점, 조화 진동자(Harmonic Oscillator)의 심장부에 3차 다항식(Cubic Polynomial)의 혼돈을 처넣은 식이 바로 더핑(Duffing) 방정식이다.

1. 3차 비선형 모델과 복원력 텐서의 파편화

기본적인 선형 복원력 k\cdot x 에, 변위의 세제곱인 \alpha\cdot x^3 항을 기습적으로 삽입하여 만든 비선형 방정식(Non-linear Equation)이다. 독일 엔지니어 게오르크 더핑(Georg Duffing)은 기계계의 극한 한계 응력망을 통제하기 위해 이 방정식을 주조해 냈다. 비감쇠, 자유 진동 상태의 기본 더핑 방정식은 다음과 같이 선언된다.

\ddot{x} + \omega_0^2 x + \alpha x^3 = 0

또는

m\ddot{x} + k x + \alpha x^3 = 0

단순한 오차항 추가가 아니다. 이 3차 \alpha 항의 부호 하나가 시스템의 전체 동역학 운명(Dynamics Fate)을 극단적으로 둘로 갈라놓는다.

2. 알파 파라미터(\alpha)의 갈림길: 경화(Hardening) 시스템

비선형 계수 \alpha > 0 이라면, 이것은 밀면 밀수록 강하게 저항하는 무자비한 매커니즘이다.

스프링이 변위 x 를 얻어 팽창하거나 수축할 때, x^3 항은 항상 x 와 동일한 방향(부호)을 공유한다. 즉, 원래의 복원력(kx)을 부추기고 맹렬하게 증폭시킨다. 늘어날수록 스프링 자체가 강철 밧줄처럼 뻣뻣해지며, 겉보기 강성(Apparent Stiffness, K_{eq} = k + \alpha x^2) 구간이 곡률을 그리며 수직으로 치솟는다.

진폭이 크면 클수록 실질 강성이 상승하므로, 오버슈트(Overshoot) 시 고유 주파수(\omega_n)마저도 진폭에 종속되어 기하급수적으로 빨라지는 주파수-진폭 의존성(Amplitude-Frequency Dependency)을 토해낸다.

3. 부서지는 탄성 구조: 연화(Softening) 시스템

비선형 계수 \alpha < 0 이라면, 시스템은 파국의 끝을 향해 달린다.

x^3 항은 원래의 선형 복원력(kx)을 깎아먹고 빼앗아 가는 약탈자 항으로 변모한다. 약간 변형되었을 때는 버티지만 큰 변위가 주어지면 겉보기 강성(K_{eq} = k - |\alpha| x^2)이 꺾이며 부드러워진다. 진자가 한계점(항복점)을 넘어 중력 복원력을 잃어버리는 비선형 진자(\sin\theta) 거동이나, 너무 세게 당겨져 느슨해진 철사에 완벽하게 비례하는 역학적 상황이다. 진폭이 커질수록 시스템은 나른해지고 둔탁해지며, 고유 주파수는 오히려 처참하게 무너져 내린다.

4. 더핑 방정식의 페이즈 포트레이트(Phase Portrait) 동역학

상대적인 속도(Velocity) 축과 변위(Displacement) 축이 만드는 2차원 위상 평면을 훑어보면 궤도의 곡률 망이 왜곡된다. 무감쇠(Undamped) 조건에서의 위상 궤적망을 설계 도식으로 쳐다보라.

외부 가진(Forcing) 없이 스스로 붕괴하는 자유 진동계도 충분히 치명적이지만, \alpha 시스템에 외부에서 하모닉 외력(Harmonic Force, F_0 \cos(\omega t))을 때려 넣으면 더핑 시스템은 전 우주적으로 가장 통제하기 어려운 다중 해(Multiple Solutions)와 카오스(Chaos) 폭풍을 일으킨다. 스프링은 직선의 감옥에서 해방되었지만, 엔지니어 스스로를 해석 불능의 지옥도 안으로 몰아넣고 만 것이다.
솔버(Solver)는 k x + \alpha x^3 가 초래하는 야코비안 행렬(Jacobian Matrix)의 비대칭성을 절대 선형 근사로 무시해서는 안 된다.
