2.1.2.1.3 비선형 강제 진동계의 다중 해(Multiple solutions)와 점프(Jump) 공진 형상 궤적

선형 진동계에서 강제 가진 주파수(Forcing Frequency, \omega)를 조절하며 시스템의 진폭 응답(Amplitude Response) 곡선을 그리면, 그 곡선은 고유 주파수(\omega_n) 근방에서 완벽한 산맥(Bode Plot)을 이루며 유일무이한 단방향 함수성을 유지한다. 특정 주파수 \omega 마다 오직 단 한 개의 진폭 X 만이 부여되는 결정론적 1:1 매핑이다.

그러나 3차원 비선형 복원력 \alpha x^3 가 개입된 더핑(Duffing) 미분 방정식(\ddot{x} + c\dot{x} + \omega_0^2 x + \alpha x^3 = F_0 \cos\omega t)에서는 수학의 이 가장 고고한 원칙마저 산산조각 난다. 진동수 백본 곡선(Backbone Curve) 자체가 진폭을 따라 기괴하게 일그러지며 오른쪽(경화) 또는 왼쪽(연화)으로 맹렬하게 넘어간다. 이제 같은 가진 주파수 \omega 하에서 시스템이 2~3개의 전혀 다른 진폭 크랙을 지닌 다중 해(Multiple Solutions) 영토에 포획된다.

1. 뼈대 곡선(Backbone Curve)의 비틀림 변형망

비선형 계수 부호에 따라 응답 진폭 X 와 가진 주파수 \omega 파라미터 간의 공진 최고점의 궤적(Backbone)이 축을 이탈한다.

  • 선형 구조 (\alpha = 0): 공진 주파수는 진폭 크기와 무관하게 정직한 수직 기둥(\omega \approx \omega_n) 형태를 유지한다.
  • 경화성 스프링 시스템 (\alpha > 0): 진폭이 커질수록 시스템 강성(Stiffness)이 굳어지며 유효 주파수(\omega_{eq})가 팽창한다. 공진 산맥은 주파수가 증가하는 우측(+방향) 대역을 향해 맹렬하게 대가리를 들이밀며 왜곡된다.
  • 연화성 스프링 시스템 (\alpha < 0): 진폭 팽창 시 강성이 허물어지며 유효 주파수가 감쇠한다. 공진 산맥은 주파수 축의 좌측(-방향)으로 부서지듯 꺾인다.

이 응답 곡률의 비틀림(Bending) 현상으로 인해 \omega 축 위의 특정 대역 구간(\omega_1 < \omega < \omega_2)에서는 한 개의 가진 주파수에 대해 3개의 교차 진폭 해(Roots)가 거미줄처럼 얽혀 파생된다.

2. 공진 응답 궤적의 3원 다중 해 (Multiple Solutions) 영역

우측으로 구부러져 일그러진 경화성 시스템 주파수 대역의 절단면 수직축 위에서, 수식은 정확하게 세 개의 정상 상태(Steady-state) 궤도 해집합(X_1, X_2, X_3)을 산출한다.

  • X_1 (상단 곡선): 거대한 진폭으로 고속으로 떨리는 정상 상태 응답(Stable).
  • X_2 (중단 곡선 곡률 역전점): 이론적으로 수식 역학망에 도출되지만, 물리적 실현이 완전히 불가능한 불안정 해(Unstable Node). 주변의 아주 미세한 진동 섭동(Perturbation)만 기동해도 시스템이 스스로 붕괴하여 X_1 이나 X_3 로 내동댕이쳐진다.
  • X_3 (하단 곡선): 극히 미미한 진폭으로 조용하게 떨리는 얌전한 응답계(Stable).

동일한 선풍기 모터의 강제 주파수를 때려 넣었는데, 어떤 날은 선풍기가 조용하고(진폭 X_3) 어떤 날은 통제 불능으로 진동(진폭 X_1)하는 미스테리한 현상이 바로 다중 해 역학의 발현이다.

3. 주파수 일주에 따른 무자비한 점프(Jump) 현상

동일한 가진 주파수 \omega 대역에서 시스템이 도대체 어느 진폭(상단 X_1, 또는 하단 X_3)을 택할 것인가는, 현재 어떤 주파수 방향으로 이 영역에 파고들고 있느냐 하는 주파수 경로 파괴(Sweep History)에 의해 극단적인 히스테리시스(Hysteresis)를 동반하며 결정된다.

주파수를 가속(Sweep-Up)할 때와 감속(Sweep-Down)할 때, 공진 진폭 궤적이 서로 다른 길을 밟고 갑자기 절벽에서 떨어지거나 폭발하는 기형적 전이 현상을 점프 현상(Jump Phenomenon) 이라 규정한다.

4. 제어 기법의 재구성 및 엔지니어링 위상 확보

엔지니어는 항공기 엔진의 회전 속도를 올릴 때(Sweep-up), 예측하지 못한 강제 주파수 점프(Jump-Up) 궤도에 얻어맞고 터빈 날개가 로터리 외벽을 긁어버리는(Rubbing) 대참사를 무조건 예방해야 한다.

  • 선형 시스템의 지오메트리 위상 마진율 \pm 10\% 확보라는 고루한 방어벽은 당장 찢어버려라.
  • 더핑 시스템의 비선형 계수 마진폭(Margin)을 계산하여 공진 대역 \omega_1 \dots \omega_2 가 운전 영역 범위로 침범하지 않도록 능동 조화 제어력(Active Harmonic Control)으로 곡선을 밀어내야 한다.
  • 카오틱 발산(Chaotic Divergence)이나 주기 배가 분기(Period-Doubling Bifurcation)로 넘어가기 직전의, 오직 X_1 과 X_3 를 오버랩하는 다중 해 마진율까지 시뮬레이션 위상평면으로 강제 포획하라.
