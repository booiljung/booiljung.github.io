2.1.2.1.1 대변위 자유 진동을 지배하는 타원 적분(Elliptic Integral) 진자 주기의 비선형성

단순 진자(Simple Pendulum)가 미세하게 떨릴 때, 우리는 \sin(\theta) \approx \theta 라는 치명적인 선형 타협을 통해 주기(Period, T)가 오직 끈의 길이(l)와 중력(g)에만 묶여 있는 고고한 상수(T = 2\pi\sqrt{l/g})라고 가르친다. 이른바 등시성(Isochronism)이다. 그러나 진자의 출발 각도가 10^\circ, 20^\circ 를 돌파하여 대변위(Large Amplitude) 스윙으로 번져나갈 때, 복원력의 진짜 지배자인 \sin(\theta) 가 비선형의 칼을 뽑아 든다. 이제 주기는 결코 상수로 머물지 않으며, 진폭(Amplitude)의 요동에 철저하게 종속되어 버린다. 그 진정한 족쇄를 풀기 위해 타원 적분(Elliptic Integral)의 수문으로 진입하라.

1. 대변위 진동의 비선형 지배 미분 방정식(Nonlinear ODE)

질량 m이 길이 l인 강성 줄에 매달려 각도 \theta 로 흔들리는 진자 시스템에 뉴턴의 회전 운동 방정식(Rotational Equation of Motion) \Sigma M_O = I \alpha 를 적용한다. 중력 복원 토크는 -mgl\sin\theta 이며, 질량 관성 모멘트는 I = ml^2 이다.

ml^2 \frac{d^2\theta}{dt^2} = -mgl \sin\theta

양변을 소거하고 고유 각주파수(Natural Angular Frequency)를 \omega_n^2 = g/l 로 치환하면, 선형화(Linearization)의 거짓을 벗어던진 순수한 비선형 2계 상미분 방정식이 위용을 드러낸다.

\frac{d^2\theta}{dt^2} + \omega_n^2 \sin\theta = 0

해석적인 닫힌 해(Analytical Closed-form Solution)가 존재하지 않는 이 야만적인 \sin\theta 항등식은 대수적인 조작만으로는 절대 주기 T 를 토해내지 않는다.

2. 에너지 보존 모델에 기반한 위상 공간 적분

이 혼돈의 궤력을 뚫어내려면 미분 대신 에너지 보존법칙(Conservation of Energy)의 적분 텐서로 우회해야 한다.
초기 최대 진폭(Maximum Amplitude)을 \theta_0 라 할 때, 진자가 가장 높은 곳(속도 0) 에서 가진 최저점 대비 최대 위치 에너지(Potential Energy)는 전체 계통 에너지와 동일하다.

E_{total} = mgl(1 - \cos\theta_0) = \frac{1}{2}ml^2\dot{\theta}^2 + mgl(1 - \cos\theta)

이 수식에서 순간 각속도(Angular Velocity, \omega = d\theta/dt)를 \theta에 관한 함수로 고립시키고, 삼각함수의 반각 공식(Half-angle Formula: 1-\cos\theta = 2\sin^2(\theta/2))을 격발하라.

\frac{d\theta}{dt} = 2\omega_n \sqrt{\sin^2(\frac{\theta_0}{2}) - \sin^2(\frac{\theta}{2})}

진자가 최고점 \theta_0 에서 최하점 0 까지 스윙하는 데 걸리는 시간은 1/4 주기(T/4)다. 이 변수를 적분형으로 묶어라.

T = \frac{2}{\omega_n} \int_{0}^{\theta_0} \frac{d\theta}{\sqrt{\sin^2(\frac{\theta_0}{2}) - \sin^2(\frac{\theta}{2})}}

3. 제1종 완전 타원 적분(Complete Elliptic Integral of the First Kind) 도출

적분 기호를 풀기 위해 가짜 각도 변수 치환(Substitution) 적분을 집행한다.
k = \sin(\frac{\theta_0}{2}) 라고 두고, \sin(\frac{\theta}{2}) = k \sin\phi 형태의 더미 변수 \phi 를 도입하여, \theta 가 0 에서부터 \theta_0 까지 증가할 때 새로운 각 \phi 가 0 에서부터 \pi/2 까지 팽창하도록 매핑(Mapping)한다.

미분소 d\theta 를 d\phi 로 치환하고 극한을 전환하면, 진동 주파수의 본질이 18세기 프랑스 수학자 아드리앵 마리 르장드르(A. M. Legendre)가 수립한 제1종 완전 타원 적분 기호 K(k) 내부로 완벽하게 수렴한다.

T = \frac{4}{\omega_n} \int_{0}^{\pi/2} \frac{d\phi}{\sqrt{1 - k^2 \sin^2\phi}} = \frac{4}{\omega_n} K(k)

  • 진폭 파라미터 k: k = \sin(\theta_0 / 2) 로, 최초 진동 폭이 커질수록 1에 수렴한다.
  • 비선형 주기 T: 진폭 \theta_0 가 커질수록(즉 k가 증가할수록) 적분 분모가 1보다 작아지며 전체 K(k) 영역 매그니튜드(Magnitude)가 급격하게 부풀어 오른다. 따라서 진폭이 클수록 주기는 무한히 늘어나 느려진다. 등시성은 철저하게 붕괴되었다.

3. 주기 오차율의 무한 급수 확장 스펙트럼

컴퓨터가 타원 적분을 루프(Loop)로 처리하는 대신, 테일러 급수(Taylor Series) 전개를 통해 오차의 임원 구역을 시각적으로 전개하라.

T = 2\pi\sqrt{\frac{l}{g}} \left[ 1 + \left(\frac{1}{2}\right)^2 \sin^2\frac{\theta_0}{2} + \left(\frac{1\cdot 3}{2\cdot 4}\right)^2 \sin^4\frac{\theta_0}{2} + \cdots \right]
