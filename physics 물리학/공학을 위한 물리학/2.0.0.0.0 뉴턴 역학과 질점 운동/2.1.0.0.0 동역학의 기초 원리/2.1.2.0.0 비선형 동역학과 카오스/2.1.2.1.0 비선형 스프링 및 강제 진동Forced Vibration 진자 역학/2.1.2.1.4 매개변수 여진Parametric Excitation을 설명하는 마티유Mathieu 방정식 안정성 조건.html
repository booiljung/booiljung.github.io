2.1.2.1.4 매개변수 여진(Parametric Excitation)을 설명하는 마티유(Mathieu) 방정식 안정성 조건

시스템을 진동시키는 고전적인 하모닉 강제력망(Harmonic Forcing)은 미분 방정식 우변에 명시적인 외력 함수(F_0 \cos\omega t)로 도열한다. 그러나 공학의 파멸적 재앙 중 하나인 매개변수 여진(Parametric Excitation)은 우변에서 힘을 가하지 않는다. 대신 지배 미분 방정식 좌변에 기생하여 시스템의 내부 강성(Stiffness) 자산이나 관성(Inertia) 파라미터 자체를 시간에 따라 주기적으로 흔들어버린다. 이 은밀한 내부 붕괴 스펙트럼을 기술하는 궁극의 수학적 텐서가 바로 마티유 방정식(Mathieu Equation)이다.

1. 시스템 내부 파라미터의 요동과 마티유 방정식

질량 m 의 진자 끈 길이 l 이 시간에 따라 주기적으로 늘어났다 줄어든다(l(t) = l_0 + \Delta l \cos\omega t)고 상상하라. 혹은 수직 기둥의 상단에 얹어진 축하중(Axial Load)이 헬리콥터 블레이드처럼 주기적으로 맥동(Pulsation)한다고 가정하라. 이 경우 스프링의 본연적 강성 k 는 상수가 아니라, k(t) = k_0 (1 + \epsilon \cos\omega t) 형태로 시간 종속적(Time-dependent)인 맥동 파라미터로 전락한다.

이 파라미터의 떨림을 무감쇠 1계 조화 진동 미분 방정식 좌변에 산입하여 전개한 것이 마티유 방정식이다. 표준화된 수학적 폼은 다음과 같다.

\frac{d^2x}{d\tau^2} + (\delta + \epsilon \cos\tau) x = 0

  • \tau = \omega t: 무차원 시간 계수.
  • \delta = (\omega_n / \omega)^2: 매개변수 맥동 주파수(\omega) 대비 고유 주파수(\omega_n)의 비율.
  • \epsilon: 파라미터 맥동(Parametric Fluctuation)의 진폭 조절 섭동(Perturbation) 계수.

이 방정식의 해악은 우변에 아무런 명시적 외력이 없음에도( 0 임에도 불구하고), 좌변 계수들의 엇박자 결합에 의해 응답 변위 x 의 진폭이 지수 함수적으로 폭주 발전(Parametric Resonance)할 수 있다는 점이다. 그네를 타는 아이가 누군가 밀어주지 않더라도 자신의 다리를 주기적으로 접었다 펴는(매개변수 \epsilon 조정) 것만으로 그네의 진폭을 맹렬하게 증폭시키는 역학이 바로 이 방정식의 현현이다.

2. 스트럿(Strut) 가이드 안정성 차트: 마티유-힐스(Mathieu-Hills) 분기망

마티유 방정식은 파라미터 \delta 와 \epsilon 의 기묘한 배합 비율에 따라, 진동이 소멸되며 안정을 찾는 지대(Stable Region)와 진폭이 무한대로 폭발하는 카오스의 발산 지대(Unstable Region)로 극단적으로 절개된다.

해석학자들은 플로케 이론(Floquet Theory)을 동원하여 불안정 영역의 점근적 거미줄을 분리해 냈다. \epsilon 이 매우 작을 때(\epsilon \ll 1), 가장 파괴적인 첫 번째 주기적 주 공진(Principal Parametric Resonance)은 \delta \approx 1/4 에서 발현된다.
즉, 시스템을 흔드는 파라미터의 요동 주파수 \omega 가 고유 주파수 \omega_n 의 정확히 2배(\omega \approx 2\omega_n) 일 때 시스템은 가장 맹렬하고 폭력적으로 부서진다. 외부 강제 진동(Forced Vibration)이 1:1 (\omega = \omega_n)에서 공진하는 것과 본질적으로 차원이 다른 수학적 변절이다.

4. 구조 설계 시 회피적 프로시저

교량이나 선박의 롤링(Rolling) 동역학에서 매개변수 여진은 극도의 은밀도 때문에 파악하기 힘들다. 선체로 밀려오는 파도 주파수가 선체 고유 롤링 주파수의 2배에 달할 때(\omega \approx 2\omega_n), 선박은 파도로부터 맞은 적이 없음에도 무시무시한 각도로 전복될 수 있다.

엔지니어는 이 \omega \approx 2\omega_n (또는 \delta \approx 1/4, 1, 9/4 \dots 의 분기 다항 타겟점)를 차트에서 완전히 밀어내기 위해 기구학 설계 단계에서 시스템 강성 위상 변수를 대폭 개조하거나 유체 활성 감쇠기(Active Viscous Damper)를 추가하여 에너지 인풋율을 마티유-스트럿(Ince-Strutt) 차트의 안정 피막 안으로 구속해야만 한다.
