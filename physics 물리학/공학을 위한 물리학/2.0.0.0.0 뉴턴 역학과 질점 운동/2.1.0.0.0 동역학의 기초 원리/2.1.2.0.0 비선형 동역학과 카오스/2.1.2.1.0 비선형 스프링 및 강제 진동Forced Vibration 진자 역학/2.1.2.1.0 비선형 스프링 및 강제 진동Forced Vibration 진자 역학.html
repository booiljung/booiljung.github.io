2.1.2.1 비선형 스프링 및 강제 진동(Forced Vibration) 진자 역학

대칭적이고 선형적인 후크의 법칙(Hooke’s Law)인 F=kx 와, sin(\theta) \approx \theta 라는 공학적 타협에 기반한 미소 진동(Small Oscillation) 모델은 매우 실용적인 수학적 도구다. 그러나 이 달콤한 근사(Approximation)는 변위가 일정 임계(Threshold)를 초과하는 순간 가차 없이 붕괴한다. 재료가 한계 이상으로 늘어나며 격자 구조가 경화(Hardening)되거나, 반대로 항복점 이상으로 구부러지며 연화(Softening)될 때 탄성 복원력의 텐서는 치명적으로 비틀린다.

또한 진자의 스윙 폭이 커져서 중력 복원 토크가 선형 각도를 이탈하는 대변위 자유 진동계(Large Amplitude Free Vibration)와, 외부에서 끊임없이 비선형 가진(Excitation)이 쏟아지는 강제 진동(Forced Vibration) 시스템은 예측을 조롱하듯 무수한 다중 해(Multiple Solutions)와 기이한 점프(Jump) 현상을 속출시킨다.

본 장에서는, 오직 한 개의 유일한 평형점과 주파수에 수렴하지 않고 매 순간 에너지를 내뱉고 빨아들이며 스스로 주기를 변화시키는 대변위 비선형 진자(Non-linear Pendulum) 와 경화/연화 스프링 코일(Duffing Oscillator) 의 동역학 망을 해체한다. 이 비선형 자유/강제 진동 시스템을 수치적으로 제어하고 예측하는 위상 공간 분석법과 매개변수 안정성 조건들을 낱낱이 파헤칠 것이다.
