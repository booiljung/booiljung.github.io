2.1.2.1.7 푸앵카레 단면(Poincaré Section) 투영을 통한 준주기(Quasi-periodic) 궤도 패턴 분류

우주 행성들의 N-체 문제(N-body Problem)나 거대한 해양 시추선(Offshore Platform)이 태풍에 노출될 때, 시스템을 두들기는 주파수는 결코 단 한 개의 정현파(Sine Wave)로 이루어지지 않는다. 두 개 이상의 독립적인 주파수(\omega_1, \omega_2)가 비선형 시스템을 동시에 강타할 때, 위상 평면(Phase Plane) 위를 질주하는 상태 궤적(State Trajectory)은 두 번 다시 자신이 지나왔던 과거의 점 위로 돌아오지 않는 끝없는 털실 뭉치 같은 궤도를 직조해 낸다. 이를 준주기(Quasi-periodic) 운동이라 한다.

어지럽게 꼬인 3차원 상태 공간 속에서 진동의 규칙성과 카오스의 징후를 분리해 내기 위해 프랑스의 앙리 푸앵카레(Henri Poincaré)가 고안한 절단 마술, 바로 푸앵카레 단면(Poincaré Section) 투영 기법을 집행한다.

1. 준주기 궤도의 비공약수(Incommensurable) 주파수 역학망

비선형 시스템을 찢고 들어오는 두 조화 강제력의 주파수 \omega_1 과 \omega_2 가 있다고 가정하라. 이 두 주파수의 비율(\omega_1 / \omega_2)이, 2/3 이나 5/7 처럼 정수 비(Rational Number)로 약분되어 떨어진다면(공약수 비, Commensurable), 두 파동은 언젠가 최소 공배수 시간에 도달하여 완벽히 합쳐지며 주기를 완성한다(주기 운동, Periodic).

그러나 \omega_1 / \omega_2 가 \sqrt{2} 나 원주율 \pi 처럼 끝없이 나아가는 무리수(Irrational Number) 비례로 맞물릴 경우, 이 시스템의 시간 축은 그 어떤 공배수도 찾아내지 못한다. 위상 공간을 떠도는 입자는 뫼비우스의 띠 위를 방황하듯 끝없이 궤면을 맴돌며 도넛 형태의 형상체, 즉 원환면(Torus) 표면 전체를 거미줄처럼 영원히 빈틈없이 감싸버린다. 이 궤도는 분명히 일정한 체적 안에 결박되어 안정적이지만(Bounded), 두 번 다시 똑같은 지류를 밟지 않는 극한의 준주기 궤도 다.

2. 3차원 원환면(Torus) 절단과 푸앵카레 단면(Poincaré Section) 도출

연속적인 시간 t 흐름 속에서 얽히고설킨 3차원 또는 고차원 공간(x, \dot{x}, t)의 선형 궤적 실타래(Trajectory Flow)를 쳐다보는 것은 맹목적이고 무의미한 시각적 공해일 뿐이다.

푸앵카레는 이 시간을 강제로 토막 내는 칼날 평면층을 고안했다. 바로 가진력의 주기 단위( T = 2\pi/\omega ) 등, 정확하게 일정한 시간 간격 t = nT (n=1,2,3...) 마다 한 번씩 스트로보스코프(Stroboscope) 카메라 불빛을 터뜨려 변위 x 와 속도 \dot{x} 좌표면 위에 점(Point) 하나를 찍는 이산 동역학 매핑 기법(Discrete Mapping)이다.

  • 주기성 1(Period-1 조화 응답): 굳건한 1차 주기 운동은 카메라가 터질 때마다 똑같은 자리로 돌아와 있다. 단면 위에는 오직 유일한 점 하나(A Single Point) 만이 덩그러니 각인된다.
  • 주기성 n(Period-n 분기 응답): 주기가 n 배로 늘어난(Period-Doubling) 서브하모닉 진동은 단면 위에 n 개의 고립된 점들 이 무리 지어져 번갈아 찍힌다.
  • 준주기 성향(Quasi-periodic): 주파수 비례가 무리수(Irrational)로 깨져 있을 때, 점들은 결코 같은 궤도를 밟지 않는다. 수만 번의 점을 찍어보면 점들은 흩날리지 않고 일련의 완벽하게 닫힌 고리 혹은 도넛 단면의 절단면 곡선(Closed Curve 관)을 형성한다.
  • 카오스(Chaos): 모든 주기 질서의 완전한 붕괴. 점들은 단면 위에 소용돌이치듯 산발적이고 복잡한 프랙탈 구름 배열(Fractal Strange Attractor) 형태로 자욱이 번진다.

4. 진동 제어 분석 모델링의 이산 지배(Difference Equation)화

푸앵카레 맵(Poincaré Map)은 단순한 구경거리가 아니다. 미분 방정식(연속 공간)을 차분 방정식(이산 공간, Difference Equation) x_{n+1} = f(x_n) 으로 강제 다운 그레이딩 시키는 막강한 연산 텐서다. 이 단면에 찍힌 점들의 수렴(Stable)이나 발산(Unstable) 벡터를 계산하면, 로터 터빈의 블레이드가 두 개의 다른 모드(Mode) 주파수에 의해 맥동하며 피로 파괴로 직행하는 준주기 균열장(Quasi-periodic Crack-propogation) 을 미연에 시각적으로 격리하고 파쇄할 수 있다.
