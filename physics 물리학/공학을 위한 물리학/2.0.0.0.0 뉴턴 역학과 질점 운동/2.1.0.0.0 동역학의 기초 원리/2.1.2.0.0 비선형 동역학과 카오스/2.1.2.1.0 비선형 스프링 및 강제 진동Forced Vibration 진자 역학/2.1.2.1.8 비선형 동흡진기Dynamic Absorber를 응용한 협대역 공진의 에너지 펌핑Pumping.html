2.1.2.1.8 비선형 동흡진기(Dynamic Absorber)를 응용한 협대역 공진의 에너지 펌핑(Pumping)

강제 진동(Forced Vibration)이 주 구조물(Primary System)을 무자비하게 강타하여 파괴적인 공진(Resonance) 상태로 몰아넣을 때, 엔지니어는 시스템의 질량이나 강성을 뜯어고치는 설계의 근본적인 퇴행 대신 보조 시스템을 체결한다. 주 구조물 위에 매달린 작은 질량과 스프링 블록, 이른바 동흡진기(Dynamic Vibration Absorber, DVA) 가 바로 그것이다.

선형(Linear) 구조의 단일 주파수 맞춤 동흡진기는 해당 가진 주파수 영역에서 주 구조물의 진동을 정확히 0 으로 억압하는 완벽한 역학 방파제(Anti-resonance)를 구축할 수 있다. 하지만 가진 모터의 회전 속도가 설계된 목표 주파수를 조금이라도 빗나가는 순간, 흡진기는 통제를 잃고 2개의 새로운 협대역 폭주 공진점(Resonant Peaks)을 부화시킨다. 이 좁고 답답한 대역폭(Bandwidth)의 감옥을 파괴하기 위해 기계 공학자는 동흡진기의 스프링에 비선형성(Non-linear Stiffness) 을 주입한다. 바로 비선형 동흡진기(Non-linear DVA) 체계와 ’에너지 펌핑(Energy Pumping)’의 마법이다.

1. 선형 튠(Tuned) 흡진기의 한계와 비선형(Non-linear) 대역폭의 확장

질량 M, 강성 K 의 주 진동계에 조그만 흡진기(질량 m, 선형 강성 k)를 체결할 때, 흡진기의 고유 주파수(\omega_a = \sqrt{k/m})를 시스템을 타격하는 불량 가진 주파수 선상(\omega)에 완벽히 동기화(Tuning)시킨다.
조화 가진력 F_0 \sin\omega t 아래에서, 미분 방정식 행렬계는 기적같이 주 질량의 변폭 X=0 이라는 정상 상태 해를 도출한다. 모든 진동 에너지가 주 구조물에서 작은 흡진기로 완벽히 전이(Transfer)되며, 주 구조물은 정지해 있고 오직 매달린 꼬마 덩어리(m)만이 맹렬하게 요동치는 기구학적 역병(Anti-resonance) 상태다.

그러나 선형 시스템의 주파수 응답 함수 곡선(Frequency Response Function)은 너무 협소하다. 모터 결함으로 외력 주파수(\omega)가 10\% 라도 빗나가면 X 는 즉각 폭발적으로 발산(Divergence)한다. 비선형 스프링(경화 스프링, \alpha x^3)을 투입하면 그 어떤 불량 주파수가 들어와도, 이 곡선은 오른쪽으로 거대하게 기울어지거나 비틀리면서(Backbone Curve Bending) 공진 0점 회피 대역폭(Effective Absorber Bandwidth) 자체가 광범위하게 확장되는 견고한 제어의 성곽을 구축한다.

2. 타겟 에너지 전이 (Targeted Energy Transfer, TET)와 에너지 펌핑

완벽하게 선형적인 복원 강성 항(kx) 자체를 제거해버리고 오직 순수한 3차 비선형 스프링 매커니즘(\alpha x^3) 만으로 흡진기를 무장시킨, 이른바 비선형 국부 부착계(Non-linear Energy Sink, NES)라 부르는 궁수들을 펄스 지진(Seismic Shock)이 때리는 건물이나 항공기 주익에 설치한다.

이 매커니즘은 선형 흡진기처럼 특정 단일 주파수 대역의 에너지만 기다리지 않는다. 주 구조물이 어떠한 주파수로 미친 듯이 떨리더라도, NES의 비선형 흡진기는 광대역 스펙트럼(Broadband) 에 걸쳐 발생한 충격 에너지를 오로지 일방향(One-way)으로만 지체 없이 자신의 덩어리 내부로 빨아들여(Targeted Energy Transfer) 모조리 댐핑 열 에너지로 태워버린다.

이를 공학적 에너지 펌핑(Energy Pumping) 현상이라 규정한다. 에너지의 역류가 차단되며 오직 폭포수처럼 비선형 댐퍼 포텐셜 곡면을 타고 흘러내려가 소멸하는, 역학 평면에서 카오스와 분기 폭주를 막는 가장 창조적인 위상 동기화(Phase Synchronization) 제어이다.

3. 에너지 임계 임펄스(Energy Threshold) 조건

TET 에너지 펌핑 매커니즘의 수학적 성립은 절대 아무 때나 일어나는 것이 아니다.

비선형 NES 흡진기가 주 진동계의 에너지를 맹렬하게 펌핑해 내어 자가-공진(1:1 Resonance Capture) 상태에 돌입하려면, 주 구조물이 머금은 순간 초기 타격 에너지(Initial Energy)가 반드시 특정 수학적 안장점 한계(Saddle-node Bifurcation Threshold)를 임계 초과해야 한다. 충격이 작을 때는 NES 시스템이 무반응 상태로 게으르게 진동하지만, 대규모 충돌(Impact) 에너지가 위상 감옥을 때리고 들어오면 비선형성의 \alpha x^3 항이 잠에서 깨어나 발작적으로 주 구조물의 에너지를 삼켜버리는 일종의 역학적 다이오드(Diode) 혹은 퓨즈(Fuse) 방식을 채택하고 있다.
