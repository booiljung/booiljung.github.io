2.1.2 비선형 동역학과 카오스

17세기 아이작 뉴턴(Isaac Newton)의 이래로, 고전 고학은 자연을 선형(Linear)이라는 수학적 환상의 틀 안에 가둬놓고 해석해 왔다. F=kx 와 같이 입력과 출력이 완벽하게 비례하고, 부분의 합이 전체의 합과 같으며(중첩의 원리, Superposition Principle), 초기 조건의 미세한 오차가 결과에도 미미한 오차만을 낳는 선형 시스템은 예측 가능하고 통제하기 쉬운 인공적인 낙원이었다.

그러나 우주와 기계 공학의 실제 파편들은 이 이상적인 비례식을 비웃듯 산산조각 낸다. 구조물이 휘어지며 강성이 변비례적으로 치솟는 대변위(Large Displacement) 거동, 두꺼운 윤활유의 비선형 점성 한계, 유체의 난류(Turbulence) 소용돌이는 선형 미분 방정식의 해를 갈기갈기 찢어버린다. 더 나아가, 결정론적(Deterministic) 방정식으로 완전히 통제된 시스템이라 하더라도, 초기 조건의 터무니없이 작은 미세한 변화가 나비 효과(Butterfly Effect)를 일으켜 시스템 전체를 예측 불가능한 혼돈의 폭풍우 속으로 몰아넣을 수 있다.

이 단원에서는 선형화(Linearization)라는 공학의 오랜 타협안을 버리고 자연의 진짜 민낯인 비선형 동역학(Nonlinear Dynamics) 과 카오스 이론(Chaos Theory) 의 심연을 들여다본다. 엔지니어는 선형성의 거짓된 위로에서 벗어나, 여러 개의 평형점이 공존하는 다중 해(Multiple Solutions)와 점프 현상(Jump Phenomenon), 그리고 위상 공간(Phase Space)을 뒤틀어버리는 프랙탈 차원의 기하학적 혼돈 시스템을 역학적으로 통제할 수치적 무기를 장착해야만 한다.
