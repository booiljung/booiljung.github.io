2.1.2.2.7 공력 탄성학적 플러터(Flutter) 발산 진동계의 카오스 영역 진입 사전 억제 설계

항공기 날개, 거대한 현수교의 상판, 압축기 터빈 블레이드는 유체와 구조물이 융합된 공력 탄성학(Aeroelasticity)의 최전선이다. 구조물의 탄성(Elastic Force), 고유 질량의 관성(Inertial Force), 그리고 다가오는 유체의 공기 역학적 양력/항력(Aerodynamic Force) 이 세 가지 텐서가 공력 탄성학적 삼각형(Collar’s Triangle)을 구성한다.

유속(Velocity, U)이 임계 한계 유속(Flutter Speed, U_f)을 돌파하는 순간, 공기가 구조물로 주입하는 에너지가 자체 점성 및 구조 감쇠비가 소산시키는 댐핑 역량을 압도해버린다. 비선형 유동 감쇠 파라미터가 ’음수(Negative)’로 역전되며, 날개의 플런징(비틀림 없이 위아래로 진동, Plunging)과 피칭(비틀림 진동, Pitching) 두 모드가 결합(Coupling)하여 무자비하게 스스로 에너지를 뿜어내는 자려 진동, 즉 플러터(Flutter) 로 진화하고 기체를 완전히 찢어버린다.

1. 선형 임계 유속의 호프 분기(Hopf Bifurcation) 한계 타파

전통적인 공기 역학 엔지니어는 선형 고유치 분석법(Theodorsen’s Theory)을 이용해 플러터 주파수가 발생하는 임계 비행 속도(U_f)를 찾고자 노력했다. 속도 U 를 파라미터 \mu 로 취급했을 때, 선형 연립 시스템 행렬의 켤레 복소수 고유치의 실수부가 정확히 0 이 되며 허수축(Imaginary Axis)을 교차하는 순간이 플러터 속도다(정지 상태 분기).

그러나 선형 이론은 플러터 발현점만 짚어줄 뿐, 익형 주변 유동의 와류 박리, 날개 내부 구조의 비선형 스프링 자유 간극, 후단 조종면의 마찰 한계 등이 엮인 카오틱 영역에서는 속수무책이다. 구조물이 호프 분기(Hopf Bifurcation)를 거쳐 한계 고리(Limit Cycle) 궤도에 안착한 뒤 파이겐바움 주기-배가(Period-Doubling Cascade) 분기 경로로 카오스로 다이빙하는 비선형 동력학의 전 과정을 파악하지 않고서는, 현대의 초음속 전진익(Forward-Swept Wing) 설계망의 재앙을 막을 수 없다.

2. 플러터 카오스 모델 예측 지배망: 갈러킨 투영법 및 상평면 해석

2자유도(피치-플런지) 비선형 익형 모델 방정식은 스트로할 수(Strouhal Number)를 포함한 유동 시간 지연 계수, 3차 다항식 복원력(더핑 방정식 형태) 그리고 쿠타-주코프스키(Kutta-Joukowski) 유동 함수가 난잡하게 믹스된 시스템이다.

  • 구조와 유동 편미분 방정식을 시간-공간 좌표로 이산화시키는 갈러킨(Galerkin) 투영 기반 상미분 방정식 도출.
  • 압축 스피드 증가 시, 주기-1 에서 주기-2 로, 다시 주기-4 로 진동이 소용돌이치다 결국 로렌츠 스펙트럼과 유사한 이상 끌개(Strange Attractor) 분기망으로 추락.

3. 플러터의 예방과 비선형 공력 강건 설계(Robust Design) 시스템

비단 항공기 주익만이 아니다. 고속 터빈 팬의 블레이드가 실속 플러터(Stall Flutter)의 비선형 카오스 진동에 말려 들어가 엔진 샤프트를 폭파시키는 지옥도는 오직 비선형 모델을 포괄한 능동/수동 억제 기법 을 수식화할 때만 막아낼 수 있다.

  1. 비선형 질량 밸런싱(Mass Balancing): 날개 앞전(Leading Edge)에 수동으로 무거운 납 덩어리 웨이트(Weight)를 달아 질량 중심(Center of Mass)을 탄성 중심(Elastic Axis)보다 전방, 혹은 완벽히 일치하도록 이동시켜 피치-플런지 연성 커플링 모드 발생 매트릭스를 물리적으로 차단한다.
  2. 스마트 전자기 액추에이터 및 ATEF 탑재 제어: 날개에 내장된 압전 합금(Piezoelectric) 센서나 활성 엣지 플랩(Trailing Edge Flap)을 통해 외부의 바람이 고유 댐핑비에 음수(-) 인자를 넘겨주지 못하도록 위상을 조작하는 능동 공력 탄성학 제어기(Active Aeroelastic Control) 루프를 구축한다.

이 고차원 카오스 매핑 방정식의 파라미터 경계를 지키지 못한다면 비싼 티타늄 합금을 두르더라도 플러터에 의한 파괴 반경 폭주를 영원히 멈출 순 없을 것이다.
