2.1.2.2 카오스(Chaos) 시스템 이론과 공학적 동적 안정성(Stability) 제어

공학자들은 오랫동안 미분 방정식의 결정론적 우주를 믿어왔다. 뉴턴의 운동 법칙에 완벽한 초기 조건과 경계 조건을 대입하면, 기계 시스템의 수백 시간 뒤 궤적을 소수점 아래 10자리까지 정확히 추적할 수 있다고 착각했다. 그러나 비선형 미분 방정식이 직조해내는 실제 자연의 거동은 선형 근사(Linearization)의 맹신을 비웃듯, 가장 사소하고 관측조차 불가능한 미세한 오차를 눈사태처럼 증폭시켜 시스템 전체를 무작위성의 폭풍 속으로 던져버린다.

이러한 초기 조건에 대한 극단적 민감성(Sensitive Dependence on Initial Conditions) 과 장기 예측의 절대적 불가능성을 지배하는 비선형 동역학적 질서, 바로 카오스(Chaos) 다.

1. 노이즈(Noise)와 카오스(Chaos)의 본질적 패러다임 분리

관측 데이터가 요동치고 예측 경로를 이탈할 때, 과거의 공학자들은 이를 외부 환경의 무작위 노이즈(Random Noise)라 치부하며 필터링의 대상으로 삼았다.

그러나 카오스 이론은 그 요동이 외부의 잡음이 아니라, 완벽하게 결정론적이고 어떠한 확률론적 함수도 개입되지 않은 순수한 시스템 내부의 수학적 지배 에 의해 자발적으로 발현됨을 증명했다. 시스템은 나비의 날갯짓 하나만큼의 섭동(나비 효과, Butterfly Effect)만으로도 전혀 다른 우주를 잉태하며 궤도를 틀어버린다.

2. 공학의 위협이자 새로운 기회, 카오틱 다이나믹스

전력 전송 네트워크의 전압 붕괴, 항공기 주익의 플러터(Flutter), 펌프 실린더 블록의 파괴적인 채터링 진동, 심지어 내연 기관 연소 행정의 불꽃 소산(Miss-fire) 현상까지, 엔지니어를 패배시켜 온 수많은 비정상 역학의 근저에는 카오틱 분기(Bifurcation)의 붕괴 선이 예복선처럼 깔려 있다.

본 챕터에서는 카오스의 발현을 판별하는 리아푸노프 지수(Lyapunov Exponent)부터 이상 끌개(Strange Attractor)의 프랙탈적 차원 분석, 카오스의 발산을 제압하고 특정 궤도로 안내하는 OGY(Ott-Grebogi-Yorke) 제어 기법까지 철저하게 파헤친다. 단순히 예측을 포기하는 것이 아니라, 무작위의 이면에 숨겨진 비선형 스케일링 법칙을 통제하여 파국적 붕괴 직전의 에너지를 공학적 한계 내로 안전하게 연착륙시키는 동적 안정성(Stability) 제어망을 구축하라.
