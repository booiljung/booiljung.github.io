2.1.2.2.2 파라미터 변화에 따른 위상 분기점(Bifurcation Point) 추적 및 호프(Hopf) 분기 판별망

동력학 시스템(Dynamical System)의 안정성은 시스템 파라미터(질량, 강성, 제어 이득, 혹은 레이놀즈 수)의 미세한 다이얼 조작에 따라 완전히 다른 위상 영역으로 급전 직하한다.

파라미터 변수 \mu 의 크기를 서서히 증가시켰을 때, \mu 가 어느 임계점(Critical Value, \mu_c)을 통과하는 순간 평온하던 고정점(Fixed Point)이 흔들리며 완전히 다른 성질의 궤도를 내뱉는다. 시스템의 정성적 형상(Topology)이 박살 나고 분열하는 이 구조적 진화와 파멸의 변곡점 수리망이 바로 위상 분기점(Bifurcation Point) 이다.

1. 정류 상태 분기망: 안장 노드(Saddle-Node) 및 피치포크(Pitchfork) 진화

1차원 혹은 평면 차원의 미분 방정식 지배계 \dot{x} = f(x, \mu) 에서 관찰되는 기본적인 평형점 분열의 종류는 파라미터를 따라 극단적으로 다르다.

  • 안장-노드 분기(Saddle-Node Bifurcation): 파라미터가 증가함에 따라 서로 다른 두 우주의 평형점(안정 노드와 불안정 안장점)이 서로 다가와 정면충돌(Collision)한 후 허공으로 동귀어진(Annihilation)하여 소거되는 블루 스카이 분기다.
  • 초임계 피치포크 분기(Supercritical Pitchfork Bifurcation): 구조물의 기하학적 대칭 조건에서 발생한다. 압축 하중 한계를 초과하는 기둥의 좌굴(Buckling) 동역학과 마찬가지로, 하나의 안정된 고정점이 임계점 \mu_c 에서 파탄 나며 2개의 새로운 안정 점(좌/우 굽힘)으로 갈라져 포크 모양으로 분기한다. 직선으로 달리고자 하나 두 갈래 길 중 하나로 무조건 이탈해야만 하는 대칭성의 자발적 붕괴(Spontaneous Symmetry Breaking)다.

2. 호프 분기(Hopf Bifurcation): 고정점에서 리미트 사이클(Limit Cycle)로의 변태

1차원 선형 공간을 벗어나 다차원 계에서 마주해야 할 엔지니어 최대의 사신파는 호프 분기다. 이것은 유체의 난류 전이나 전투기 외판의 공력 탄성학적 플러터(Flutter, 자려 진동) 발현을 설명하는 절대 수학 방정식이다.

비선형 미분 방정식의 평형점(Equilibrium Point) 주위에서 야코비안(Jacobian) 행렬 선형화를 수행했을 때, 파라미터 \mu 가 증가하면서 복소 공액 고유치 세트 \lambda_{1,2} = \alpha(\mu) \pm i \beta(\mu) 의 실수부인 \alpha(\mu) 가 음수(-)의 심연을 뜷고 임계점 \mu_c 에서 정확히 0 이 된 뒤 양수(+) 영토로 탈출할 때 벌어지는 마법이다.

이 임계의 순간 \mu_c 를 호프 분기점이라 한다. 즉, 중심점으로 빨려 들어가 소멸하던 에너지가 파라미터 임계선을 돌파하자마자 스스로 나선형으로 진폭을 뿜어내며 바깥쪽을 향해 팽창(Unstable Spiral)한다. 그러나 시스템은 폭파하지 않고 비선형 댐핑(예: 반 델 폴 진동자)에 가로막혀 하나의 유일한 닫힌 고리인 한계 고리(Limit Cycle) 를 직조해낸다.

  • 점(Point)이 선(Closed Curve)으로 변태하는, 자려 진동이 최초로 폭발하는 순간의 역학 지표다.

3. 주기 배가(Period-Doubling) 분기와 피겐바움(Feigenbaum) 카오스 진입망

물리학자 미첼 파이겐바움(Mitchell Feigenbaum)이 제시한 궁극의 카오스 루트 중 수많은 기계 공학자를 절망시킨 경로는, 시스템의 파라미터가 상승하며 기존의 1번 돌아오던 주기궤도(Period-1)가 갑자기 속도를 주체 못 하고 2번 궤도를 도는 형상(Period-2)으로 분기하는 주기 배가 분기(Period-Doubling) 루트 점이다.
