2.1.2.2.5 능동 피드백 제어기를 적용한 자기 부상(Maglev) 열차의 끌개 불안정성 극복 궤적

강철 레일의 굴레를 집어던진 자기 부상(Magnetic Levitation, Maglev) 기차는 마찰 영(0)이라는 축복을 받았지만, 역학적 대가로 지상에서 가장 통제하기 어려운 극도의 개방 루프 불안정계(Open-Loop Unstable System)라는 비선형의 사신을 짊어지게 되었다.

이 시스템의 심장은 강자성체 레일과 전자석(Electromagnet) 사이의 거리에 반비례하고 전류의 제곱에 비례하는 비선형 흡입 인력망(Attractive Force, F_m \propto i^2/z^2)이다. 자석이 조금이라도 레일에 다가가 간극(Air Gap, z)이 줄어들면, 자력이 빛의 속도로 치솟으며 기차를 레일 바닥으로 무자비하게 추락-격돌(Touch-down)시킨다. 반대로 간극이 멀어지면 부력이 순식간에 소멸하며 열차는 아래로 곤두박질친다. 이른바 1차원 위상 평면 위 안장점(Saddle Point)에서 발현되는 가장 파멸적인 끌개 불안정성(Attractor Instability) 이다.

1. 전자기-로터 역학 방정식 시뮬레이션: 비선형성 극복

1축 자기 부상계를 모델링하는 운동 방정식과 전기 회로 방정식(전압 e, 전류 i, 변위 z) 모델망은 다음과 같다.

  1. 운동 방정식: 질량 m 에 중력(mg)과 전자석 흡입력(F_m)이 반대 텐서로 작용한다.
    m\ddot{z} = mg - C \frac{i^2}{z^2}
  2. 전자기 전기-회로 방정식: 직렬 저항(R)과 인덕턴스(L(z)) 코일에서의 역기전력 편미분망.
    e = R i + \frac{d}{dt}(L(z) i)

이 지배 방정식에 의거, 기차는 오직 단 하나의 불안정한 평형 고정점(z_0, i_0)만을 가진다. 이 점의 야코비안 선형화 행렬을 도출해 고유치(Eigenvalues)를 관측하면, 반드시 최소한 1개 이상의 양(+)의 실수 고유치가 출현하여 궤적을 심연으로 발산시킨다. 제어기(Controller) 없이는 단 1초도 부상 궤도 점근을 유지할 수 없는 본질적 발산 시스템이다.

2. 상태 관측기(Observer) 피드백 루프 능동 제어(Active Control)

극도로 휘어지는 비선형 공간을 국부 선형화 구역(Operating Point)에 안착시키려면, 위상 지상 연산망 보조기가 필요하다. 열차가 z_0 간극 십자가 위에서 떨어지려 할 때 전류 i 를 폭발적으로 증폭 소산시키는 PID, 또는 상태 공간 제어(State-Space Control)를 기반으로 한 능동 자기 베어링(AMB, Active Magnetic Bearing) 제어 피드백을 가동해야 한다.

  • 위상 앞섬 보상기(Phase-Lead Compensator): 불안정 극점(Pole)을 좌반면(LHP)으로 끌고 오기 위해 속도항(\dot{z})에 해당하는 막대한 가상 댐핑(미분 제어기 K_d)을 투입하여 센서계 위상 지연을 상쇄.
  • 옵저버(Observer) 텐서 결합: 간극 변위 z 측정 센서 신호 하나만으로 직접 계측이 불가능한 속도 속성 \dot{z} 와 외란 전류 요동치를 동시점 시뮬레이터(루엔버거 관측기)로 역산 추정하여, 마이크로초 단위 딜레이를 예측 타격 보상.
