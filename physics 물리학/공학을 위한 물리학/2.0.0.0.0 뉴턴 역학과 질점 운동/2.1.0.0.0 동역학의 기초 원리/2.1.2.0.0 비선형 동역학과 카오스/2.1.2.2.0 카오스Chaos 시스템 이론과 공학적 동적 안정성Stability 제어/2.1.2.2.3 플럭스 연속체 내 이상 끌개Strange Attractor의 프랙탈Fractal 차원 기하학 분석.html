2.1.2.2.3 플럭스 연속체 내 이상 끌개(Strange Attractor)의 프랙탈(Fractal) 차원 기하학 분석

물리학자와 엔지니어가 예측 가능한 세계에 작별을 고해야 하는 궁극의 성소는 이상 끌개(Strange Attractor) 안이다. 점(Fixed Point), 선(Limit Cycle), 그리고 원환면(Torus)이라는 정수(Integer) 형태의 유클리드 기하학 차원에 머물던 선형-균형 동역학은 파라미터가 카오스 영토로 넘어가면서 종말을 맞는다.

대신 그 자리에 나타나는 것은 끝없이 구부러져 접히고(Folding), 극도로 늘어나 찢어지며(Stretching), 무한정 세분화되는 프랙탈(Fractal) 조직망 구조를 가진 위상 공간의 괴물, 이상 끌개다. 기상학자 에드워드 로렌츠(Edward Lorenz)가 기상 예측 3원 연립 대류 방정식에서 발견한 궤도는, 언제나 하나의 독특한 유한 영역(끌개 세트)에 빨려 들어가지만, 그 내부에서는 수백만 번 겹치면서도 결코 두 번 다시 같은 점을 지나지 않는 무한 비주기성(Non-periodic) 나비 날개 궤적을 직조해냈다.

1. 접힘(Folding)과 늘어남(Stretching)이 빚어내는 무한 위상 밀도

이상 끌개의 자기 조직화(Self-organization)는 리아푸노프 지수(Lyapunov Exponents) 스펙트럼의 충돌에서 태어난다.
최고 리아푸노프 지수 \lambda_1 > 0 은 우주가 지수 함수적으로 팽창(Stretching)함을 означа한다. 위상 공간 내의 두 이웃한 입자는 광속으로 서로 멀어져 시스템을 전방위로 폭주시킨다. 그러나 이 시스템이 에너지 보존 혹은 전체 소산계(Dissipative System)의 지배 아래 있다면, \sum \lambda_i < 0 규칙에 의해 위상 체적은 필연적으로 쪼그라들어야 한다.

즉, 늘어나 시스템 밖으로 이탈하려는 궤적을 3차원 위상 평면 체적의 수축성(\lambda_3 < 0)이 억눌러 강박적으로 좁은 구역 안으로 강제로 구부려 접어 넣는(Folding) 모순 적 물리 현상이다. 이 무서운 인력과 척력이 무한하게 반복되면 파이(Pie) 생지 반죽처럼 수억 번 늘어났다 겹겹이 쌓이는 나노 스케일의 층(Layers) 형성 거동이 나타난다.

2. 프랙탈 기하학 기전과 카푸츠(Kaplan-Yorke) 용량 차원(Dimension)

이상 끌개의 형태를 돋보기로 보면 무한대의 미세 구조(Micro-structure)가 끊임없이 반복되는 자기 유사성(Self-similarity)을 품고 있다.
이 궤적을 3차원의 실체라고 부를 수 없고, 2차원의 면적이라고 칭할 수도 없다. 이들은 정수(Integer) 차원이 아닌 소수(Fractional) 차원, 즉 프랙탈 차원(Fractal Dimension) 을 지닌다.

카오스 제어 공학자는 박스 카운팅 차원(Box-counting Dimension, D_0) 텐서를 가져와 유도한다.
이 위상 끌개를 덮기 위해 필요한 한 변의 길이가 \epsilon 인 n 차원 큐브(Box)의 개수를 N(\epsilon) 이라 할 때,

D_0 = \lim_{\epsilon \to 0} \frac{\ln N(\epsilon)}{\ln(1/\epsilon)}

또한, 리아푸노프 지수 스펙트럼(\lambda_1 \ge \lambda_2 \dots \ge \lambda_n)에 기반하여 계산하는 카플란-요크(Kaplan-Yorke) 추측 정보 차원도 중요한 카오스 스펙트럼을 제공한다. 로렌츠 끌개의 프랙탈 차원은 대략 D_0 \approx 2.06 차원으로, 2차원의 선형적인 표면에 기생하지만 미세하게 공간으로 들떠 3차원적 복잡성을 소유하고 있는 경계 초월적인 입체라는 것을 증명한다.
