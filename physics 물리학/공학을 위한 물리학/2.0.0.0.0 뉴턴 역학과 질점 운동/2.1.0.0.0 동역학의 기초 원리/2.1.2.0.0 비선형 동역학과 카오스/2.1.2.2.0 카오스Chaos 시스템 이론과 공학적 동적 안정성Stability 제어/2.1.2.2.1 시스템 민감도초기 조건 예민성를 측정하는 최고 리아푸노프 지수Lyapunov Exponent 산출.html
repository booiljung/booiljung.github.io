2.1.2.2.1 시스템 민감도(초기 조건 예민성)를 측정하는 최고 리아푸노프 지수(Lyapunov Exponent) 산출

공학적으로 완벽히 설계된, 동일한 부품 편차와 동일한 강성 행렬을 지닌 두 대의 기계 시스템이 있다고 가정하라. 한 시스템이 x_0 의 초기 위치에서 진동을 시작했고, 다른 시스템은 그보다 단 10^{-6} mm (나노미터 급의 미세한 공차)만큼 떨어진 x_0 + \delta x_0 지점에서 기동(Start-up)했다.

선형 동역학계라면 1시간 후 두 기계의 궤적 오차 \delta x(t) 는 기껏해야 상수로 유지되거나 서서히 진동할 뿐, 예측 불가능해지지 않는다. 변위는 언제나 예측 가능성의 끈으로 부드럽게 묶여 있다.

그러나 이 기계가 극도의 비선형 응력을 방사하는 시스템 한계 대역을 넘어선 순간(카오스), 이 극미한 10^{-6} mm 의 틈새 오차는 시간에 따라 지수 함수적(Exponential) 으로 폭발적으로 부풀어 오르며, 1분 후에는 한 기계는 멀쩡히 회전하는데 다른 기계는 스스로 붕괴하여 파열하는 완전히 다른 운명의 궤도를 질주하게 된다. 우리는 이것을 초기 조건에 대한 극단적 민감성(Sensitive Dependence on Initial Conditions) 이라 부르며, 이 괴물 같은 오차 발산율을 측정하는 궁극적 척도가 바로 최고 리아푸노프 지수(Largest Lyapunov Exponent, LLE, \lambda_1) 다.

1. 리아푸노프 지수의 지수적 발산 함수망 도출

위상 공간(Phase Space) 상에서 지극히 가까운 곳에서 함께 구르기 시작한 두 궤적, 즉 x(t) 와 x(t) + \delta x(t) 의 거리차가 시간 t 에 미분적 영향을 받으며 어떻게 성장하는지 수식적으로 포획하라.

|\delta x(t)| \approx e^{\lambda t} |\delta x_0|

시간이 흐를수록 오차의 크기 |\delta x(t)| 가 무한대 한계를 향해 확장될 때, 그 성장 계수가 \lambda 가 된다.
이 \lambda 를 수학적으로 완벽하게 끄집어내기 위한 미분 극한 방정식 텐서는 다음과 같이 요약된다.

\lambda = \lim_{t \to \infty} \lim_{\delta x_0 \to 0} \frac{1}{t} \ln \left| \frac{\delta x(t)}{\delta x_0} \right|

카오스 공학자는 위상 벡터장(Vector Field) 전체를 구성하는 시스템 차원(예: 3차원)만큼(\lambda_1, \lambda_2, \lambda_3)의 리아푸노프 지수 스펙트럼(Lyapunov Spectrum)을 한데 묶어 뽑아낸다. 그중에서도 시간 방향으로 가장 격렬하게 폭주하는, 제일 큰 값을 가진 최고 리아푸노프 지수(\lambda_1, LLE) 의 부호(Sign)가 시스템의 운명을 판가름 짓는 절대적 사신이다.

2. 최고 리아푸노프 지수의 부호(Sign) 추적 판별망

  • \lambda_1 < 0 (음수):
    두 시스템이 제아무리 멀리 떨어져 출발했어도 궤적 오차 폭 e^{\lambda t} 가 지수적으로 감쇠하며 소멸해버린다(수렴). 시스템 에너지가 파열하지 않고 매끄러운 고정점(Fixed Point)이나 안착된 인력계에 빨려들어가 평온한 질서를 유지하는 안전 제어 대역이다.
  • \lambda_1 = 0 (영점):
    오차가 유지되며 일정한 주기로 반복되는 안정 한계 고리(Limit Cycle) 주기 궤도. 이들은 시계추처럼 영원히 함께 동행한다.
  • \lambda_1 > 0 (양수): 카오스(Chaos) 확정 선언
    두 궤적이 시간 평면에 지수적 팽창 에너지를 뿜어내며 맹렬히 찢어진다. 서로의 궤도가 예측하지 가능한 범위를 완전히 이탈해버리는 나비 효과가 현실화된 상태다. 이 지수를 목격한 역학자는 시스템이 카오틱 붕괴로 돌입했음을 인정해야만 한다.

3. 계측과 시계열 데이터 상 리아푸노프 예지망

실제 터빈 블레이드나 선박 스태빌라이저에서 수집되는 센서 진동 빅데이터(Time Series Data)는 모든 변수를 측정할 수 없으나, 타켄스(Takens)의 위상 궤적 지연 좌표(Delay-coordinate) 매립(Embedding) 정리를 이용하면 과거 센서 값 하나만으로도 리아푸노프 텐서를 복원할 수 있다.

우주 궤도 역학 설계나 심해 잠수정 안정성 평가 설계망에서 리아푸노프 지수 양수 영역(\lambda_1 > 0)은 완전한 접근 불가 지대, 셧다운 회피 대역이다. 날카로운 초기 조건의 창끝만이 시스템의 붕괴를 경고할 수 있지만, 리아푸노프 지수를 해독하지 못하는 엔지니어는 1분 뒤에 자신이 마주할 재앙의 이유조차 짐작하지 못할 것이다.
