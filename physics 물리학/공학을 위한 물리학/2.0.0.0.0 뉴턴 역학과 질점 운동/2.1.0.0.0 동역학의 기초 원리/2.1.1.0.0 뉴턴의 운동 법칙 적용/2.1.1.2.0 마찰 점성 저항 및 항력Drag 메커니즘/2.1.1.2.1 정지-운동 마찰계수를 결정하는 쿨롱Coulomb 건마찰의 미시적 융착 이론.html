2.1.1.2.1 정지/운동 마찰계수를 결정하는 쿨롱(Coulomb) 건마찰의 미시적 융착 이론

엔지니어가 마주하는 가장 직관적이면서도 심오한 비보존력(Non-conservative Force)은, 오일이나 윤활유의 개입 없이 거칠게 맞물린 두 고체 표면 사이에서 발생하는 건마찰(Coulomb Friction)이다. 육안으로 매끄러워 보이는 금속 표면이라도 미시적(Microscopic) 수준으로 파고들면 무수한 톱니와 봉우리(Asperity)들로 가득한 산맥이다. 이 돌기들이 압착될 때 발생하는 미세 응착(Adhesion)과 절삭 파괴 역학이 마찰 저항의 진정한 본질이다.

1. 미시적 응착 이론(Adhesion Theory)과 진접촉 면적

건마찰의 힘은 물체의 기하학적인 겉보기 접촉 면적(Apparent Area)과는 철저하게 무관하다. 블록을 넓게 눕히든, 좁게 세워 놓든 마찰력의 크기는 동일하다. 이에 대한 비밀을 쿨롱의 마찰 법칙은 형태적으로 단순하게 서술하지만, 그 본질은 보우든(Bowden)과 테이버(Tabor)의 융착(Junction) 이론으로 규명된다.

실제로 두 고체가 접촉할 때, 오직 극히 제한된 면적의 돌기 최상단들만이 서로 충돌하여 소성 변형(Plastic Deformation) 구역을 형성한다. 이 진접촉 면적(Real Contact Area, A_r)은 표면에 가해지는 강한 조여짐과 압박, 즉 수직 항력(Normal Force, N)에 놀랍도록 선형적으로 비례하며 확장된다.

A_r \propto N

이 작은 접합부들은 마치 용접 덩어리처럼 콜드 웰딩(Cold Welding)을 일으켜 찰흙처럼 달라붙는다. 물체를 측면으로 전진시키기 위해서는 이 융착점들을 횡방향 탄성 한계 너머로 찢어버려야 한다. 전단 강도(Shear Strength)를 \tau 라 할 때, 총 마찰 저항력 F_f 는 다음과 같이 유도된다.

F_f = A_r \cdot \tau \propto N

최종적으로 비례 상수를 무차원 스칼라 \mu 로 치환함으로써 압도적으로 단순하고 아름다운 마찰 텐서 방정식이 탄생한다.

F_f = \mu N

2. 정지 마찰(Static Friction)의 가변 저항 한계 텐서

밀림이 시작되기 직전까지, 두 물체의 접착 봉우리 지점들은 스프링처럼 질기게 버티며 외력 P 와 정확하게 동등한 반발 강도를 내뿜는다. 이 구간에서 정지 마찰력 F_s 는 고정된 스칼라 상수가 아니다. 그것은 0 에서부터 미끄러지기 직전 극대화점 사이를 자유자재로 요동치는 가변 응답(Variable Response) 텐서다.

0 \le F_s \le \mu_s N

이 부등식 체계는 융착이 견딜 수 있는 궁극적 파괴 하중, 즉 최대 정지 마찰력(Maximum Static Friction) 한계점을 규정한다.

(F_s)_{max} = \mu_s N

이 한계점은 오직 표면의 화학적 불순물과 물리적 거칠기 정도만을 내포한 물성 통계량인 정지 마찰계수 \mu_s 와, 계면을 수직으로 찌르는 억압 조절력인 수직 항력 N 에 의해서만 지배된다.

3. 미끄럼 파괴와 운동 마찰(Kinetic Friction)의 돌입 계단

외력이 최대 정지 마찰력을 돌파(P > \mu_s N)하는 찰나, 얽혀있던 산맥의 결합 부위들이 물리적으로 파단(Fracture)되며 찢겨 나간다. 일단 한 번 미세 용접이 부서지고 질점이 상대 속도 v > 0 로 파편 위를 미끄러지기 시작하면, 표면 요철들이 새롭게 융착될 충분한 틈새 시간(Relaxation Time)이 주어지지 않는다.

따라서 질점이 역학적으로 이동할 때 대항하는 파동 저항인 운동 마찰력 F_k 는 융착 시간 부족으로 인해 항상 최대 정지 마찰력보다 억제된다.

F_k = \mu_k N \quad (\mu_k < \mu_s)

이 수식의 불연속적인 단절은, 무거운 가구를 밀 때 처음에 엄청난 힘이 들다가 막상 밀리기 시작하면 급격하게 저항 텐서가 풀리며 폭주하는 일상적 동역학 현상을 완벽하게 규명한다. 속도가 극단적으로 빠르지 않은 통상적인 기계 공학 설계에서 \mu_k 는 상수(Constant)로 강제 간주된다.
