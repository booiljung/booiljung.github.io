2.1.1.2.3 스토크스(Stokes) 법칙에 기반한 레이놀즈 수(Re)가 낮은 층류의 선형 점성 저항

거시 세계의 고속 운동체들은 공기를 찢고 난류(Turbulence) 소용돌이를 뒤로 흘리며 날아가지만, 미시 영역이나 극도로 끈적거리는 매질 내부에서는 전혀 다른 유체 역학적 독재자가 군림한다. 혈액 위를 떠다니는 적혈구 나노 입자, 두꺼운 모터 오일(Motor Oil) 속으로 침강하는 쇠구슬의 거동은 레이놀즈 수(Reynolds Number, Re)가 1을 맴도는 절대적인 층류(Laminar Flow)의 감옥에 갇힌다. 이 영역을 지배하는 동력학은 형태학적으로 가장 단순하면서도 엄밀한 스토크스 법칙(Stokes’ Law)의 선형 선언이다.

1. 관성력에 대한 점성력의 승리: 낮은 레이놀즈 수의 세계

유체 속을 헤엄치는 물체의 거동을 결정하는 가장 중대한 무차원(Dimensionless) 지표는 레이놀즈 수(Re)다. 이것은 유동을 가속하고 흩뿌리려는 관성력(Inertial Force)과, 유동을 뭉치게 하고 질서를 부여하려는 점성력(Viscous Force) 간의 역학적 줄다리기 비율이다.

Re = \frac{\rho v L}{\mu_{viscosity}}

여기서 \rho 는 유체의 밀도, \mu_{viscosity} 는 동점성 계수, v 는 물체의 상대 속도, L 은 물체의 특성 길이 스케일이다.
물체가 극도로 작거나(L \rightarrow 0), 이동 속도가 굼뜨거나(v \rightarrow 0), 혹은 글리세린처럼 유체의 점도(\mu_{viscosity})가 비정상적으로 높을 때, 시스템은 Re \ll 1 의 구역으로 추락한다.

이때 물리계는 스토크스 흐름(Stokes Flow) 상태에 진입하며 대혼돈의 난류 파동이나 후류(Wake)는 완전히 말소된다. 관성 항성분이 붕괴하므로, 물체가 유체를 통과하는 역학은 철저하게 매끄럽고 층층이 미끄러지는 점성 전단(Viscous Shear) 저항에 의해서만 결정된다.

2. 구형 입자의 스토크스 항력(Drag Force) 미분 유도

스토크스(George Gabriel Stokes)는 1851년, 나비에-스토크스 방정식(Navier-Stokes Equations)에서 비선형 대류항(Convective Acceleration Term)을 과감하게 소거하고 편미분 압력 구배와 점성 텐서만을 남겼다. 이 스토크스 근사법(Stokes Approximation)을 완벽한 반경 r의 구형 입자에 대해 적분하면, 입자의 운동을 억누르는 저항력 F_D 가 속도에 정확히 선형 비례하여 도출된다.

F_D = 6 \pi \mu_{viscosity} r v

  • F_D: 스토크스 항력 (선형 점성 저항력)
  • \mu_{viscosity}: 절대 점도(Absolute Viscosity, N\cdot s/m^2 또는 Pa\cdot s)
  • r: 구의 반경 (Characteristic dimensions)
  • v: 물체와 유체 간의 상대 미끄럼 속도

속도에 선형적으로 비례(F_D \propto v) 한다는 사실이 이 방정식의 알파요 오메가다. 고성능 제트기와 자동차가 겪는 저항력은 속도의 제곱(v^2)에 비례하여 기하급수적으로 폭주하지만, 스토크스 유동 내의 입자는 오직 자신이 내딛는 속도에 정비례하는 정직한 저항만을 받는다.

2. 선형 점성 감쇠 모델 (Linear Viscous Damping Model)

스토크스의 선형 관계망은 모든 기계 진동 및 1차원 동역학계 시스템의 감쇠기(Dashpot) 모델링으로 곧장 직각 투영된다.

F_{damping} = -c \cdot v = -c \cdot \dot{x}

물리가 아무리 복잡한 형상을 띠더라도 레이놀즈 수가 낮다면 댐핑 텐서의 비례 상수 c 로 모든 것을 단순화할 수 있다. 구형 입자의 경우 이 저항 계수는 정확히 c = 6\pi\mu_{viscosity} r 이 된다. 이 선형 함수는 진동 방정식( m\ddot{x} + c\dot{x} + kx = 0 )을 수학적으로 우아하게 닫아버리는 해석적 선형 해(Analytical Solution)의 핵심 열쇠로 작동한다.
