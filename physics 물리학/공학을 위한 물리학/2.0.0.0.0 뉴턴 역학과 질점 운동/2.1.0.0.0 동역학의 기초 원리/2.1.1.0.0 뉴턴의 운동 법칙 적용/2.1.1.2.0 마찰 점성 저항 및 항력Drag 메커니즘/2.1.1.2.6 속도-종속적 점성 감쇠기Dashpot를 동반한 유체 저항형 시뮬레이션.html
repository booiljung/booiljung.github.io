2.1.1.2.6 속도-종속적 점성 감쇠기(Dashpot)를 동반한 유체 저항형 시뮬레이션

기계 시스템에서 스프링(Spring)은 위치 에너지(Potential Energy)를 저장했다가 고스란히 뱉어내는 완벽한 보존 텐서(Conservative Tensor)지만, 이것만으로는 진동을 결코 멈출 수 없다. 자동차의 서스펜션이 요철을 넘은 후 통통 튀는 상태로 영원히 진동하지 않으려면, 에너지를 강제로 흡수하여 열로 태워버리는 소산 매커니즘(Dissipation Mechanism)이 축을 제어해야 한다. 오일로 가득 찬 실린더와 구멍 뚫린 피스톤으로 구성된 대시팟(Dashpot)은 유체 점성 저항의 속도-종속적(Velocity-dependent) 특성을 물리적으로 구현한 궁극의 댐핑(Damping) 디바이스다.

1. 선형 감쇠기(Linear Dashpot) 모형과 감쇠력 벡터망

대시팟의 피스톤 속도 v = \dot{x} (시간에 대한 미분)로 실린더 내부의 점성 오일을 밀어낼 때, 유체는 좁은 오리피스(Orifice)를 통과하며 극심한 층류 전단(Laminar Shear) 마찰을 겪는다. 이때 발생하는 저항력은 스토크스 법칙(Stokes’ Law)에 근거하여 피스톤의 순간적인 진입 속도에 정확하게 선형 비례한다.

F_{damping} = -c \cdot v = -c \cdot \dot{x}

음수기호(-)는 감쇠력이 언제나 물체의 진행 방향(속도 방향)과 정반대로 맞서며 시스템의 에너지를 깎아낸다는 비보존적 렌츠 법칙(Lenz’s Law)의 상징이다. 여기서 비례 상수 c 는 감쇠 계수(Damping Coefficient)로서 단위는 N\cdot s/m 이며, 유체의 동점도(Kinematic Viscosity)와 오리피스의 단면적 형상에 의해 결정되는 공학적 고정 설계 변수다.

2. 1자유도(1-DOF) 감쇠 진동계(Damped Oscillator) 조립망

질량(m), 스프링(k), 점성 감쇠기(c)가 결합된 1자유도 동역학 블록 시스템의 자유 물체도(FBD)를 그려 뉴턴 제2법칙으로 통합하라.

\Sigma F = -k x - c \dot{x} = m \ddot{x}
m \ddot{x} + c \dot{x} + k x = 0

변위(x), 속도(\dot{x}), 가속도(\ddot{x})가 계수들과 결합된 완전한 2계 제차 선형 상미분 방정식(2nd-order Homogeneous Linear ODE)이 도출된다. 공학자들은 이 수식을 정규화하기 위해, 감쇠비(Damping Ratio, \zeta)와 고유 진동수(Natural Frequency, \omega_n)라는 무차원 파라미터 텐서를 주입한다.

\ddot{x} + 2\zeta\omega_n \dot{x} + \omega_n^2 x = 0
\left( \omega_n = \sqrt{\frac{k}{m}}, \quad \zeta = \frac{c}{2\sqrt{km}} = \frac{c}{c_c} \right)

감쇠비 \zeta 는 실제 시스템의 감쇠 계수(c)를, 진동을 완벽하게 억제하는 임계 감쇠 계수(c_c = 2\sqrt{km})로 나눈 치명적 지표다. 이 한 개의 스칼라 비율 오차에 따라 기계의 운명은 요동치는 진동의 지옥에 빠질 수도, 둔탁한 감쇠의 늪에 갇힐 수도 있다.

3. 감쇠비(\zeta) 스펙트럼에 따른 3대 점성 감쇠 궤적 곡률

미분 방정식의 특성 방정식(Characteristic Equation) 뿌리(Roots) 판별식을 통해, 시뮬레이터 곡선은 3가지 상태 거동(Behavior Mode)으로 운명을 달리한다.

  • 과감쇠 (Overdamped, \zeta > 1): 감쇠기(c)의 저항이 스프링(k)의 복원력을 압도한다. 시스템은 진동(Oscillation)할 틈조차 없이, 극도로 무겁고 느릿하게 평형점(0)을 향해 지수 함수적으로(Exponentially) 서서히 기어 들어간다. 중장비 포신이나 육중한 도어 클로저(Door Closer)에 적용된다.
  • 임계 감쇠 (Critically Damped, \zeta = 1): \zeta 가 정확히 1인 상태로, 진동을 일으키지 않으면서도 시스템이 가장 신속하고 예리하게 잔상 없이 평형 0점으로 복귀하는 경계점이다. 자동차 현가장치(Suspension)나 정밀 위치 제어 계측 시스템이 미친 듯이 영점 조준하는 이상적 타겟 스칼라값이다.
  • 미흡 감쇠 (Underdamped, 0 < \zeta < 1): 스프링이 대시팟의 제어력을 벗어난 상태. 시스템은 오버슈트(Overshoot)를 유발하며 중심선을 수없이 교차하는 조화 진동 곡선을 그린다. 다만 대시팟이 에너지를 갉아먹으므로 진폭은 e^{-\zeta\omega_n t} 형태의 포락선(Envelope)을 따라 매 싸이클마다 붕괴(Decay)한다. 일반적인 충격 흡수 장치의 과도 응답 곡선이다.

대시팟은 단순히 충격을 분쇄하는 스펀지가 아니다. 그것의 본질은 변위가 아닌 속도(v) 에 얽매여 작동한다. 고속으로 강하게 타격할수록 댐퍼는 바위처럼 굳어 저항(c\cdot v)을 폭증시키며(Bump), 천천히 누르면 물처럼 부드럽게 텐션을 해제(Rebound)한다. 이 속도 추종적 강성(Velocity-tracking Stiffness)이 다이내믹 시뮬레이션 해석에 미분기를 추가하여 시스템을 길들이는 핵심 이유다.
