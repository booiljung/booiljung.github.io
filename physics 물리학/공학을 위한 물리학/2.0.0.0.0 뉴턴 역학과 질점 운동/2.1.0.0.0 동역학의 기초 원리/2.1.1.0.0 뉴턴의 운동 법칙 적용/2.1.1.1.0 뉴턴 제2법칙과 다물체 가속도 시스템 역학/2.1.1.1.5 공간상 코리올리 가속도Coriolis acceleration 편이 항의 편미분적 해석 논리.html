2.1.1.1.5 공간상 코리올리 가속도(Coriolis acceleration) 편이 항의 편미분적 해석 논리

자전하는 지구 표면을 스쳐 지나가는 대륙간 탄도 미사일(ICBM)의 궤적이나, 회전하는 매니퓰레이터(Manipulator) 암(Arm) 위를 미끄러지는 슬라이더(Slider)의 미세한 곡률 비틀림은 단순한 원심력(Centrifugal Force)의 발현이 아니다. 이것은 회전하는 비관성계(Non-inertial Reference Frame) 내부에서 운동 벡터가 지니는 상대 속도와, 프레임 자체의 각속도가 얽히며 창출해 내는 극단적인 기학학적 착시이자 동시에 실재하는 물리적 파괴력인 코리올리 가속도(Coriolis Acceleration)의 산물이다. 회전체 상부에서 움직이는 모든 입자의 운동 편이(Deflection) 현상을 2계 편미분 벡터의 외적(Cross Product)망으로 해체하라.

1. 상대 속도 벡터의 회전 미분과 코리올리 항의 탄생

코리올리 가속도는 절대 정지한 질점이나 회전계에 완전히 못 박혀 고정된 질점에서는 결코 발생하지 않는다. 이것은 오직 회전하는 좌표계 내부에서 관성 모델의 상대 속도(\mathbf{v}') 가 존재할 때만 발현되는 횡단 동역학이다.

비관성계의 각속도를 \boldsymbol{\omega}, 비관성계 내의 관찰자가 측정한 질점의 상대 위치를 \mathbf{r}' = x'\mathbf{i} + y'\mathbf{j} + z'\mathbf{k} 라 하자.
절대 관성계에서의 속도 벡터(\mathbf{v}_{abs})는 체인 룰(Chain Rule)에 의해 다음과 같이 두 텐서의 합산으로 직조된다.

\mathbf{v}_{abs} = \left( \frac{d\mathbf{r}'}{dt} \right)_{xyz} + \boldsymbol{\omega} \times \mathbf{r}' = \mathbf{v}' + \boldsymbol{\omega} \times \mathbf{r}'

이를 다시 시간(t)에 대해 편미분하여 도함수를 추출할 때, 코리올리 가속도의 쌍둥이 모멘텀이 벼락처럼 튀어나온다.

\mathbf{a}_{abs} = \frac{d}{dt}(\mathbf{v}')_{xyz} + \frac{d}{dt}(\boldsymbol{\omega} \times \mathbf{r}')
\mathbf{a}_{abs} = \mathbf{a}' + \boldsymbol{\omega} \times \mathbf{v}' + \dot{\boldsymbol{\omega}} \times \mathbf{r}' + \boldsymbol{\omega} \times (\mathbf{v}' + \boldsymbol{\omega} \times \mathbf{r}')

이 중 \boldsymbol{\omega} \times \mathbf{v}' 항이 두 번 중첩되어 기구학적 방정식에 2\boldsymbol{\omega} \times \mathbf{v}' 라는 특이 항을 강제로 삽입한다. 이 항이 바로 회전 좌표계와 병진 운동 좌표계의 연성(Coupling)을 규명하는 코리올리 가속도(\mathbf{a}_{coriolis}) 이다.

\mathbf{a}_{coriolis} = 2\boldsymbol{\omega} \times \mathbf{v}'

가상력(Fictitious Force)의 관점에서는 반작용 계수인 -m 이 곱해져 코리올리 힘(Coriolis Force, \mathbf{F}_{coriolis} = -2m(\boldsymbol{\omega} \times \mathbf{v}')) 으로 전개된다.

2. 벡터 외적(Cross Product)의 방향성 역학 규칙

코리올리 가속도의 방향은 오직 오른나사 법칙(Right-hand Rule)을 거치는 수학적 외적망에 의해 결정된다.

  • \boldsymbol{\omega} 벡터: 회전 좌표계의 팽창 회전축 (반시계 방향 회전 시 평면에서 수직으로 솟아오르는 z 벡터)
  • \mathbf{v}' 벡터: 회전 평면 위를 이동하는 질점의 진행 방향 벡터
  • 결과 벡터 \mathbf{a}_{coriolis}: 질점의 진행 방향(\mathbf{v}')에 대해 수직하게(Orthogonal) 작용하여 물체의 측면 궤적을 굽어지게(Curvature) 만드는 편향력.

코리올리 힘은 질점의 속도 방향을 가로막거나 밀어주며 에너지를 흡수/방출하는 일(Work)을 일절 수행하지 않는다(\mathbf{F}_{coriolis} \cdot \mathbf{v}' = 0). 오직 궤적의 횡단 곡률만을 맹렬하게 비틀어버리는 무자비한 방향타 역할을 수행한다.

3. 회전 원판 위의 방사형 운동 시뮬레이션

반시계 방향 각속도 \boldsymbol{\omega} = \omega \mathbf{k} 로 도는 회전 원판의 중심에서, 반경 외곽으로 일정한 상대 속도 \mathbf{v}' = v \mathbf{i} 로 미끄러지는 질점을 상상하라.

\mathbf{F}_{coriolis} = -2m(\omega \mathbf{k} \times v \mathbf{i}) = -2m\omega v \mathbf{j}

외곽으로 향하려는 질점의 위치(r)가 커질수록, 해당 지점의 원주 접선 속도(\omega r)는 증가해야 한다. 그러나 질점 자체가 지닌 각운동량의 일관성으로 인해 질점은 늘어난 접선 속도를 기구학적으로 따라가지 못하고 원판 회전의 역방향(-\mathbf{j})으로 뒤처지며 휜 궤적을 그리게 된다. 이것이 바로 북반구에서 북쪽/남쪽으로 포탄을 발사할 때 포탄이 우측으로 편향(Deflection) 되는 근본적인 역학적 원리다.

4. 다체 동역학의 코리올리 제어 블록도

공학자는 코리올리 가속도가 시스템 자코비안(Jacobian) 행렬에 미치는 비선형 커플링 현상을 억제하거나 철저하게 계산에 산입해야 한다.
