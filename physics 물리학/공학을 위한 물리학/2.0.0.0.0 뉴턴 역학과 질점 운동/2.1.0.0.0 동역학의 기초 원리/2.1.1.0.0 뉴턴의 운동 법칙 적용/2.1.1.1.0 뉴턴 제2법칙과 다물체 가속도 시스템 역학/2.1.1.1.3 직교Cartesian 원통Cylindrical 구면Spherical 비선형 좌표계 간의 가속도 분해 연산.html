2.1.1.1.3 직교(Cartesian), 원통(Cylindrical), 구면(Spherical) 비선형 좌표계 간의 가속도 분해 연산

다물체 시스템의 동역학적 궤적을 제어하기 위해 엔지니어는 시스템의 구속 조건(Constraint Condition)과 기하학적 대칭성(Symmetry)에 최적화된 기준틀(Reference Frame)을 도입해야 한다. 직교 좌표계(Cartesian Coordinate System)는 공간을 고정된 그물망으로 분할하지만, 관절 로봇이나 인공위성 등 곡면을 유영하는 질점의 역학은 기저 벡터(Basis Vector) 자체가 시간에 따라 회전하는 원통 좌표계(Cylindrical Coordinate System)와 구면 좌표계(Spherical Coordinate System)로의 다변수 비선형 변환(Non-linear Transformation)을 강제한다.

1. 직교 좌표계(Cartesian Coordinates)의 절대 고정 기반 기구학

가장 원초적인 데카르트 좌표계 x, y, z의 단위 기저 벡터 \mathbf{i}, \mathbf{j}, \mathbf{k}는 공간상에서 방향이 결코 변하지 않는 절대적 불변량(d\mathbf{i}/dt = 0)이다. 시스템의 위치 벡터 \mathbf{r}을 시간에 대해 두 번 미분하면 순수한 2계 상미분(Ordinary Differential Equation)만으로 가속도 텐서가 분해된다.

\mathbf{a} = \ddot{x}\mathbf{i} + \ddot{y}\mathbf{j} + \ddot{z}\mathbf{k}

이 직교계는 선형적인 미끄럼 운동(Translational Motion)을 기술하는 데 탁월하지만, 원운동을 포함하는 회전 시스템에서는 각속도와 구심 가속도의 물리적 의미를 직관적으로 파악하기 어렵게 만드는 수학적 병목점(Bottleneck)으로 작용한다.

2. 원통 좌표계(Cylindrical Coordinates)의 회전 이동 텐서 결합

레이더 안테나의 방위각 추적이나 매니퓰레이터의 스윙 궤적 등, 단일 축을 중심으로 한 스핀 운동 이력이 포함되는 경우 원통 좌표계 (r, \theta, z)로 전환하라.

위치 벡터 \mathbf{r} = r \mathbf{e}_r + z \mathbf{e}_z를 미분할 때, 방사 반경을 가리키는 단위 벡터 \mathbf{e}_r 과 접선 방향의 단위 벡터 \mathbf{e}_\theta 는 고정되지 않고 \theta의 변화에 따라 회전한다(방향의 미분: d\mathbf{e}_r/dt = \dot{\theta}\mathbf{e}_\theta, d\mathbf{e}_\theta/dt = -\dot{\theta}\mathbf{e}_r). 이를 적용한 2차 시간 미분 가속도 연산은 다음과 같은 비선형 역학 계수를 토해낸다.

\mathbf{a} = (\ddot{r} - r\dot{\theta}^2)\mathbf{e}_r + (r\ddot{\theta} + 2\dot{r}\dot{\theta})\mathbf{e}_\theta + \ddot{z}\mathbf{e}_z

  • 방사 반경 방향(Radial, \mathbf{e}_r):
  • \ddot{r}: 순수 반경 팽창 가속도.
  • -r\dot{\theta}^2: 회전에 의해 강제 유발되는 시스템의 구심 가속도(Centripetal Acceleration).
  • 접선 횡방향(Transverse, \mathbf{e}_\theta):
  • r\ddot{\theta}: 순수 각가속도에 의한 접선 가속도항.
  • 2\dot{r}\dot{\theta}: 반경의 팽창 속도와 회전 각속도가 교차(Coupling)하여 파생되는 코리올리 가속도(Coriolis Acceleration).

원통 좌표계는 원심력과 코리올리 섭동력(Perturbation Force)을 명시적으로 노출시켜, 내재된 회전 관성의 불안정성을 예측하는 강력한 도구를 제공한다.

2. 구면 좌표계(Spherical Coordinates)의 다중 교차 가속도 해체망

우주 비행체의 궤도 역학이나 3차원 공간을 자유롭게 탐색하는 완전 다관절 조향 시스템은 반경(R), 방위각(Azimuth Angle, \theta), 고각(Elevation Angle, \phi)으로 구성된 구면 좌표계 (R, \theta, \phi)의 극단적 비선형성에 직면한다.

기저 벡터 집합 \mathbf{e}_R, \mathbf{e}_\theta, \mathbf{e}_\phi 가 모두 3차원 공간에서 독립적으로 회전하므로, 가속도 \mathbf{a}는 3차원 코리올리 복합 텐서 행렬망으로 파편화된다.

a_R = \ddot{R} - R\dot{\theta}^2\sin^2\phi - R\dot{\phi}^2
a_\theta = R\ddot{\theta}\sin\phi + 2\dot{R}\dot{\theta}\sin\phi + 2R\dot{\theta}\dot{\phi}\cos\phi
a_\phi = R\ddot{\phi} + 2\dot{R}\dot{\phi} - R\dot{\theta}^2\sin\phi\cos\phi

이 가공할 수준의 편차 방정식(Differential Equation)은, 방위각과 고각의 미세한 각속도 요동이 전체 반경 방향 추력(Thrust)에 얼마나 거대한 역방향 반항력을 초래하는지 적나라하게 입증한다. 짐벌락(Gimbal Lock) 회피 또는 자이로스코프(Gyroscope) 역학 설계 시 이 연립 항들을 필수적으로 보정하라.

4. 좌표계 간 벡터 변환(Coordinate Transformation) 프로시저

서로 다른 센서와 액추에이터는 각기 편향된 기준틀을 지닌다. 유도 무기 체계 통합을 위해 직교, 원통, 구면 간의 야코비안(Jacobian) 투영 행렬을 수립하여 국소 가속도 벡터망을 자유자재로 연산하라.
