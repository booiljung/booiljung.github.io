2.1.1.1.8 비선형 가스 압력 하중 등 시간에 따라 급변하는 시불변 외력의 수치적 전진 시간 적분

중력(mg)이나 이상적인 스프링 복원력(-kx)과 같이 변위나 상수에 깔끔하게 종속되는 정적(Static) 하중 구배는 공학적 실전에서 극히 드물다. 예를 들어 내연기관 챔버 내부를 팽창시키는 폭발 가스의 비선형 압력, 혹은 탄도학(Ballistics)에서 발사체를 타격하는 추진 장약의 급변하는 압력 하중은 해석적인 닫힌 형태(Closed-form Solution)의 적분 상수를 거부한다. 오직 컴퓨터의 극단적인 연산 능력을 빌린 수치 적분(Numerical Integration)만이 시간의 흐름(Time-marching)에 따라 요동치는 미분 방정식의 파면을 추적할 수 있다.

1. 시간 종속적 비선형 하중 함수와 동역학 모델링

피스톤 챔버 내부에서 연소 가스가 폭발하며 발생시키는 압력 파동 P(t) 를 가정하라. 이 압력이 단면적 A의 피스톤 헤드를 타격하여 질량 m을 밀어낼 때, 배압(Back pressure)과 마찰 저항, 그리고 위치 반발력을 뚫고 나아가는 동역학 지배 방정식(Governing Equation)은 다음과 같이 직조된다.

m \frac{d^2x}{dt^2} = P(t) A - c \frac{dx}{dt} - k x - F_{friction}

여기서 가스 압력 P(t) 는 어떤 순간순간 급격하게 치솟았다가 지수함수적으로 붕괴하는 임펄스(Impulse)의 스펙트럼 곡선이다. 이 비선형 상미분 방정식(ODE)을 풀어내기 위해, 가속도 \ddot{x} 항을 고립시키고 속도 계형 변수 v 로 치환하는 상태 공간(State-space) 1계 미분 방정식 행렬로 분할하라.

\frac{dx}{dt} = v
\frac{dv}{dt} = \frac{1}{m} \left( P(t) A - c v - k x - F_{friction} \right) = f(t, x, v)

2. 수치적 전진 시간 적분: 오일러 방법론(Euler Method)의 한계

가장 원초적인 이산화(Discretization) 모델인 전진 오일러 방법(Forward Euler Method)은, 특정 시간 t 에서의 기울기(도함수)가 다음 미소 시간 스텝 t + \Delta t 까지 변함없이 유지된다고 맹신하는 선형 근사법이다.

x_{i+1} = x_i + v_i \Delta t
v_{i+1} = v_i + a_i \Delta t = v_i + f(t_i, x_i, v_i) \Delta t

밀리초(ms) 단위로 절벽처럼 꺾이는 가스 압력 파동 앞에서, 오일러 방법의 오차는 스텝마다 누적되어 시스템 에너지가 인위적으로 증폭되는 수치적 발산(Numerical Instability)이라는 재앙을 초래한다.

3. 고해상도 차분망: 룬게-쿠타(Runge-Kutta) 4차(RK4) 알고리즘

폭발적인 비선형 하중에서 해석의 수렴성(Convergence)과 신뢰성을 보장하기 위해, 공학계는 룬게-쿠타 4차법(RK4)을 표준 시간 전진 적분기(Time-stepping Integrator)로 강제 채택한다. RK4는 \Delta t 구역 내부의 4개 중간점 기울기를 가중 평균하여, 곡선 궤도에 가장 완벽하게 밀착하는 예측 벡터를 도출한다.

각 스텝마다 가속도 거동 f(t, x, v) 에 대해 다음 4개의 기울기 계수( k_1, k_2, k_3, k_4 )를 포획하라.

  • k_1: 스텝 시작점(t_i)에서의 명목 상 기울기
  • k_2, k_3: 구간 중앙(t_i + \Delta t / 2)에서 예측된 중간 기울기들
  • k_4: 구간 끝(t_i + \Delta t)에서의 종단 기울기

최종적으로 다음 프레임에서의 속도와 위치 텐서(Tensor)는 아래의 선형 결합 수식으로 전개된다.

v_{i+1} = v_i + \frac{\Delta t}{6} (k_1 + 2k_2 + 2k_3 + k_4)
x_{i+1} = x_i + v_i \Delta t + \text{보정항}

이 정밀한 포락선 알고리즘은 오차율을 O(\Delta t^4) 수준으로 억눌러, 극강의 비선형 가스 압축-폭발 충격파 내에서도 질점의 궤적을 잃지 않고 정확히 조준한다.
