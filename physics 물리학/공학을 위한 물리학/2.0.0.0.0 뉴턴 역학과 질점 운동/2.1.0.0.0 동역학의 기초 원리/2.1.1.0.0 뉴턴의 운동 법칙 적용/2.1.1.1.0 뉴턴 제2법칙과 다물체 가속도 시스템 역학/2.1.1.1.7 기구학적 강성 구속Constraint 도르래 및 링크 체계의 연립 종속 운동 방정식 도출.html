2.1.1.1.7 기구학적 강성 구속(Constraint) 도르래 및 링크 체계의 연립 종속 운동 방정식 도출

기계 공학에서 질점들은 결코 우주 공간에 홀로 부유하지 않는다. 그들은 늘어나지 않는 강인한 케이블(Cable), 강체 링크(Rigid Link), 그리고 도르래(Pulley)들의 결합망에 의해 서로의 운명이 잔인하게 속박된 종속 운동(Dependent Motion)을 수행한다. 이 강성 구속(Constraint) 체계를 수학적인 등식으로 규명하지 못한다면, 개별 물체에 뉴턴의 제2법칙을 아무리 적용해 보아야 미지수의 늪에 빠져 해답을 도출할 수 없다. 기하학적 구속 방정식을 통해 다물체의 변위-속도-가속도를 하나의 연립 방정식 텐서로 꿰뚫어라.

1. 전역 기하 학적 기준 좌표설정과 총 케이블 길이 불변성

종속 운동의 기준선은 가장 움직이지 않는 견고한 벽단면, 혹은 고정 도르래 장치의 회전 중심을 전대역 데이텀(Global Datum)으로 설정하는 것에서 출발한다. 케이블이 감기는 도르래 시스템에서 각 부하 블록으로 뻗어 나가는 위치 변위를 투영하라.

늘어나거나 끊어지지 않는 이상적 케이블의 총 길이를 L_{Total} 이라 하면, 이 길이는 위치 벡터 y_A, y_B, y_C 들의 대수적 총합과 상수 기하학적 아크(Arc) 단면들의 결합으로 기술된다.

L_{Total} = \Sigma(\text{가변 선분}) + \Sigma(\text{고정 궤적 아크 길이})

가변 선분은 블록 A, B의 위치 y_A, y_B 에 종속적이다. 예를 들어 평범한 1단 이동 도르래의 경우 총 길이는 다음과 같이 모델링된다.

L_{Total} = y_A + 2y_B + \text{상수항(Constant)}

2. 시간 편미분을 통한 구속 속도 및 가속도의 종속 연립

총 길이 L_{Total} 은 시간 \Delta t 가 흘러도 팽팽한 장력(Tension) 내에서 변하지 않는 상수로 남는다. 이 위치 구속 방정식을 시간 t 에 대해 차례로 편미분(Derivative)하여 속도와 가속도의 종속성을 강제로 해체하라.

  • 1계 미분 (속도 구속망):
    \frac{d}{dt}(L_{Total}) = \dot{y}_A + 2\dot{y}_B + 0 = 0
    v_A + 2v_B = 0 \quad \Rightarrow \quad v_A = -2v_B
  • 2계 미분 (가속도 구속망):
    \ddot{y}_A + 2\ddot{y}_B = 0
    a_A + 2a_B = 0 \quad \Rightarrow \quad a_A = -2a_B

이 미분 결과는 블록 B가 단위 가속도로 지면을 향해 떨어질 때, 블록 A는 반드시 그 두 배의 가속도로 위로 튕겨 올라가야만 하는 도르래의 가혹한 운동 속박 메커니즘을 증명한다. 부호(-)는 이동 방향의 기하학적 역전을 뜻한다.

3. 링크 체계(Linkage)의 코사인 텐서 비선형 구속

도르래가 아닌 일정 길이(L)의 강체 막대기(Rigid Link)로 슬라이더(Slider)들이 직교 벽면에 묶여 미끄러지는 경우, 제약 조건은 피타고라스의 직각 역학에 종속되어 더욱 비선형적으로 비틀린다. 슬라이더 A가 x축을 따라, B가 y축을 따라 이동할 때 링크에 의한 구속 방정식은 아래와 같다.

x^2 + y^2 = L^2

이 궤적을 시간으로 1계 편미분하여 음함수 미분법을 동원하면 속도의 연성 텐서가 튀어나온다.

2x \frac{dx}{dt} + 2y \frac{dy}{dt} = 0 \quad \Rightarrow \quad x v_A + y v_B = 0
v_B = -\left(\frac{x}{y}\right) v_A = -\tan\theta \cdot v_A

이를 한 번 더 시간으로 타격해 가속도 관계식을 도출하라.

(v_A^2 + x a_A) + (v_B^2 + y a_B) = 0
a_B = -\frac{x}{y}a_A - \frac{v_A^2 + v_B^2}{y}

막대로 이어진 체계는 단순한 상속 가속비를 낳지 않는다. 속도의 제곱(v^2) 항이 포함되면서 A가 일정한 속도로 움직여도(a_A=0) B의 가속도는 결코 0이 되지 않는 기구학적 코리올리 혼조 현상이 발생한다.

4. 다체 자유 물체도(FBD) 연립과 시스템 행렬 조립

운동학적 가속도 매트릭스 제약이 완성되면, 마지막으로 각 질량 질점에 뉴턴의 자유 물체도(FBD) 작용/반작용 장력망을 결합하여 가속도 시스템 해(Simulation Solver)를 추출하라.

구속된 종속 운동계에서는 뉴턴 동역학 미분 방정식의 개수(예: 2개) 보다 풀어야 할 미지수의 체적 개수(예: a_A, a_B, T 로 총 3개) 가 구조적으로 많을 수밖에 없다. 이때 기구학적 구속(Constraint)을 미분하여 얻어낸 가속도 종속 텐서(a_A + 2a_B = 0)는 이 행렬식 결핍을 채워 완벽한 유일해를 구토하게 하는 열쇠로 맹활약한다.
