2.1.1.1.4 비관성 회전 병진 좌표계의 동역학적 관성력(가상력) 및 오일러(Euler) 가속도 행렬

로보틱스, 우주 발사체, 그리고 공학적 다물체 시스템(Multibody System)은 지면과 같은 완벽한 정지 관성계(Inertial Reference Frame)에 머무르지 않는다. 스스로 가속하고 맹렬하게 회전하는 자동차나 전투기의 기체에 부착된 비관성 좌표계(Non-inertial Reference Frame) 내부에서 물리 현상을 통제하기 위해서는, 뉴턴의 제2법칙에 유령처럼 개입하여 궤적을 비틀어 놓는 가상력(Fictitious Force) 메커니즘을 수학적으로 완벽하게 포획해야 한다.

1. 비관성계의 기구학(Kinematics) 벡터 미분 방정식

절대 관성계 X-Y-Z 내부에, 임의의 궤적을 따라 병진 가속도 \mathbf{a}_0를 가지며 동시에 각속도 \boldsymbol{\omega} 및 각가속도 \boldsymbol{\alpha}로 회전하는 상대 비관성 좌표계 x-y-z 가 존재한다고 가정하라. 이 회전 격자망 내부의 관찰자가 측정한 어떤 입자의 상대 위치 벡터를 \mathbf{r}' 이라 하고, 상대 속도를 \mathbf{v}', 상대 가속도를 \mathbf{a}' 이라 정의한다.

회전하는 기저 벡터(Basis Vector)를 시간에 대해 미분하는 포아송(Poisson) 운동학 공식을 적용하여, 절대 관성계에서 바라본 입자의 절대 공간 가속도 \mathbf{a}_{abs} 를 편미분 분해하면 아래의 거대한 5항 벡터 가속도 매트릭스가 전개된다.

\mathbf{a}_{abs} = \mathbf{a}_0 + \mathbf{a}' + \boldsymbol{\alpha} \times \mathbf{r}' + \boldsymbol{\omega} \times (\boldsymbol{\omega} \times \mathbf{r}') + 2\boldsymbol{\omega} \times \mathbf{v}'

이 수식은 관측자의 공간적 시야가 회전함에 따라 발생하는 유클리드 공간의 기하학적 비틀림을 나타내는 절대적 기준선이다.

2. 뉴턴 제2법칙의 재조립과 동역학적 4대 관성력(가상력) 행렬

비관성계 내의 관찰자는 위 수식 우변의 가속도 항들이 자신이 딛고 있는 바닥 좌표계의 운동 때문이라는 것을 인지하지 못한다. 따라서 관찰자는 \mathbf{a}' 만을 자신이 관측하는 시스템 가속도로 인식하며, 뉴턴 방정식을 m\mathbf{a}' = \Sigma\mathbf{F}_{real} + \Sigma\mathbf{F}_{fictitious} 형태로 강제 개조하게 된다.

시스템의 질량 m을 곱하여 좌변으로 넘겼을 때 파생되는 각각의 항은, 존재하지 않으나 질점에 물리적인 타격을 가하는 관성력(Inertial Force) 변위 텐서로 작용한다.

\Sigma\mathbf{F}_{fictitious} = -m\mathbf{a}_0 - m(\boldsymbol{\alpha} \times \mathbf{r}') - m[\boldsymbol{\omega} \times (\boldsymbol{\omega} \times \mathbf{r}')] - 2m(\boldsymbol{\omega} \times \mathbf{v}')

  • 병진 관성력 (Translational Inertial Force, -m\mathbf{a}_0): 기준계 자체가 직선 방향으로 가속(\mathbf{a}_0)할 때 발생하는 반작용 렌츠(Lenz) 저항이다. 브레이크를 밟는 차량 내부의 승객이 앞으로 튕겨 나가는 현상의 단일 근원이다.
  • 원심력 (Centrifugal Force, -m[\boldsymbol{\omega} \times (\boldsymbol{\omega} \times \mathbf{r}')]): 회전 중심으로부터 반경 \mathbf{r}' 외곽으로 물질을 강제 이탈시키려는 방사 가속 벡터다. 터빈 블레이드나 원심 분리기의 고속 회전 시 파단 한계를 위협하는 주범이다.

2. 오일러 가속도(Euler Acceleration)와 횡방향 타격력

가상력 방정식의 두 번째 항인 오일러 힘(Euler Force, -m(\boldsymbol{\alpha} \times \mathbf{r}')) 의 본질적 역학을 제어하라.

이 항은 좌표계의 각가속도(\boldsymbol{\alpha} = d\boldsymbol{\omega}/dt)가 존재할 때, 즉 각속도가 균일하지 않고 변동할 때만 발현되는 횡단 가속도 성분이다. 오일러 각속도 변위는 회전 반경과 직교 방향으로 질량을 타격하며 불균일한 비틀림 변형 에너지(Torsional Strain Energy)를 낳는다.
예를 들어, 세탁기 모터가 급정거하거나 드릴 척(Chuck)이 기동할 때, 좌표계 내에 구속된 질점은 코어 회전 방향의 역방향으로 격렬한 오일러 섭동력(Perturbation)을 얻어맞게 된다. 회전체 동역학계(Rotor Dynamics) 설계 시 이 오일러 횡단 응력을 구조 해석망 텐서에 즉각 삽입하라.

관성력은 실재하는 상호작용 힘(Real Force)이 아니지만, 비관성계라는 장부 위에서는 실존하는 물리적 파괴력으로 거동한다. 오일러 행렬의 공간 편미분 특성을 완벽히 조작하여 짐벌 락(Gimbal Lock)과 비선형 회전 혼돈을 역학 시스템에서 제거하라.
