3.1.2.1.3 직교 방향 코사인 행렬(DCM, Direction Cosine Matrix)의 유니터리(Unitary) 성질과 직교 회전비

우주선이 마구잡이로 돌아가도, 그 뼈대가 고무줄처럼 늘어나거나 찌그러지지 않는 한(강체, Rigid Body), 수학 세계의 신은 이 기체를 옭아매는 가장 완벽한 3 \times 3 기하학적 매트릭스, 방향 코사인 행렬(Direction Cosine Matrix, DCM, \mathbf{C}) 의 권능을 허락한다.
이 9마리의 코사인-사인(Cosine-Sine) 조합물 덩어리는 단순히 좌표계를 번역기처럼 \text{Body} 에서 \text{Earth} 로 옮겨줄 뿐만 아니라, 텐서 행렬계에서 가장 고결하고 파괴 불가능한 다이아몬드, 직교(Orthogonal) 행렬이라는 특권을 부여받고 우주 공간의 각도 보존 법칙을 호위한다.

1. 방향 코사인(Direction Cosine) 텐서의 투영 벡터 탄생

3차원 우주 고정 좌표계 X, Y, Z 의 각 축(단위 벡터 \mathbf{I, J, K})과, 전투기가 미친 듯이 돌아서 꺾여있는 기체 로컬 축 x, y, z (단위 벡터 \mathbf{i, j, k})를 허공에 겹쳐 놓고 보자.

  • 전투기의 x 축 꼬챙이(\mathbf{i})를 우주의 X 축 계곡으로 플래시를 비춰 투영(Projection)하면 고전 삼각비에 의해 \cos\theta_{xX} 가 떨어져 나온다.
  • 동일하게 x 방향 꼬챙이를 Y 축 벽, Z 축 벽에 빔을 쏴서 생기는 3 마리의 코사인 그림자들이 바로 3 \times 3 매트릭스의 첫 번째 줄(Row) \begin{bmatrix} C_{11} & C_{12} & C_{13} \end{bmatrix} 을 구성한다.
  • 이 아홉 번의 무차별 일대일 각도 내적(\mathbf{i} \cdot \mathbf{I}, \mathbf{i} \cdot \mathbf{J} 등) 잔해들을 모은 것이 바로 \mathbf{C} 행렬, 즉 방향 벡터투영 코사인 텐서다. 이 아홉 마리의 기하 투영파는 오직 강체의 회전 ‘비틀기만’ 담당하며 점의 이동(Translation)에는 철저히 묵비권을 행사한다.

2. 유니터리(Unitary) 성질: 직교 텐서의 \text{Transpose(전치)} = \text{Inverse(역행렬)} 기적

비행기 좌표계(V_{\text{Body}})를 넣으면 지구 좌표(V_{\text{Earth}})로 치환해 주는 공식이 \mathbf{V}_{\text{Earth}} = \mathbf{C} \cdot \mathbf{V}_{\text{Body}} 다.
그럼 반대로, 지상의 관제탑 탑재 컴퓨터가 우주의 좌표를 비행기 로컬 좌표망으로 바꿔 돌려주고 싶을 땐 어떻게 할까? 정상적인 선형 대수학(Linear Algebra)이라면 저 토 나오는 3 \times 3 복합 삼각 행렬 \mathbf{C} 의 역행렬(Inverse, \mathbf{C}^{-1})을 구해야 하는 끔찍한 수학적 행렬식(Determinant 분수식) 나눗셈 지옥이 열린다.

하지만 강체 병진 역학은 여기서 기하학적 기적을 토해낸다!

  • 강체는 회전하면서 축과 축 사이의 각도 90^\circ (직교성) 수평선이 늘어나거나 줄어들지 않고(보존, Preservation of Length and Angle), 자기들끼리 서로를 내적하면 \mathbf{i} \cdot \mathbf{j} = 0, \mathbf{i} \cdot \mathbf{i} = 1 로 완벽히 상쇄 파멸되는 오소노멀(Orthonormal) 속성을 띠고 있다.
  • 이 지독한 3 \times 3 직교 보존(Orthogonality) 속성 때문에, 이 행렬은 미친 짓을 안 해도 자신의 역행렬(\mathbf{C}^{-1})이, 그냥 행렬의 가로세로 줄을 확 뒤집어버리는 자기 자신의 전치 행렬(\mathbf{C}^T, Transpose)과 100\% 일치하는 경악스러운 대수 치환(Orthogonal Matrix Identity) 능력을 갖는다.

\mathbf{C}^{-1} = \mathbf{C}^T \quad (\text{Hence, } \mathbf{C}^T \cdot \mathbf{C} = \mathbf{I}_{\text{Identity } 3 \times 3})

\mathbf{C} 텐서의 또 하나 충격적인 사실은 행렬식(Determinant \det(\mathbf{C})) 값이 무조건 +1 로 박힌다는 것이다. 행렬식은 부피의 팽창 비율을 의미한다. 행렬식이 +1 이라는 소리는 “전투기가 3\text{D} 공간에서 아무리 난리 치면서 돌아도, 그 기하학적 형태의 찌그러짐(스케일, 넓이)은 절대 파괴되지 않고 거울상 반사(역전: \det = -1)도 되지 않은 그 순수 회전 상태를 보존하고 있음“을 가장 우아하게 증명하는 수학 판결문이다. 방향 코사인 행렬은 그 자체로 회전 우주의 파수꾼이다.
