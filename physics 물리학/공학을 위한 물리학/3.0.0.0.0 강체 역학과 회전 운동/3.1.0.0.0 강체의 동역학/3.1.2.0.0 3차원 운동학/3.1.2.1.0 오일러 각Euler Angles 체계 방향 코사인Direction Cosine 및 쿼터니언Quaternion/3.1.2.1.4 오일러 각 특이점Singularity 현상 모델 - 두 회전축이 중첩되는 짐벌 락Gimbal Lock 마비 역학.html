3.1.2.1.4 오일러 각 특이점(Singularity) 현상 모델: 두 회전축이 중첩되는 짐벌 락(Gimbal Lock) 마비 역학

오일러 각(Euler Angles) 체계는 직관적이고 인간이 이해하기 가장 아름답다. 비행기가 위로 20^\circ 솟고(\text{Pitch}), 옆으로 15^\circ 누웠다(\text{Roll}). 완벽하다.
하지만 1969년, 인류를 달에 보낸 아폴로 11호 비행 관제 컴퓨터에서부터 현대의 최신 드론, 수백억짜리 \text{3D} 게임 엔진 애니메이션에 이르기까지 모든 오일러 각 기반 코더들의 피를 말려버리는 저주받은 수학적 종말 단계가 존재한다.
비행기 기수가 완벽하게 수직, 즉 하늘 꼭대기 정점(Pitch 90^\circ, Zenith)이나 바닥(Nadir)을 찌르는 그 찰나의 순간! 비행기가 돌아가던 \mathbf{Z} 축과 마지막 \mathbf{X} 축이 우주 궤도에서 물리적으로 하나로 겹쳐버리고(\text{Coincide}), 컴퓨터는 비행기가 롤(Roll)을 했는지 요(Yaw)를 했는지 3차원 분간 능력을 영원히 상실하며 통제 불능 스핀에 빠지는 대수학적 공황 장애, 이것이 공포의 짐벌 락(Gimbal Lock) 이다.

1. 피치(Pitch) \mathbf{90^\circ} 의 저주: 독립 변수의 파멸

항공기 \text{Z-Y-X} 오일러 변환 \mathbf{R} = R_z(\psi) R_y(\theta) R_x(\phi) 에서, 조종사가 조이스틱을 당겨 기체를 수직(\text{Pitch } \theta = 90^\circ 또는 \pi/2)으로 꽂아보자.
3 \times 3 오일러 삼각 행렬 공식 전체에 \theta = 90^\circ 를 쑤셔 넣으면, \cos(90^\circ) = 0 이므로 수식의 뼈대가 되는 행렬 아홉 칸 중 절반이 처참하게 0\text{(Zero)} 이 되어 지워진다!
남은 쪼가리들을 합치면 거미줄 행렬은 이렇게 붕괴된다.

\mathbf{R} \xrightarrow{\theta=90^\circ} \begin{bmatrix} 0 & 0 & -1 \\ \sin(\phi - \psi) & \cos(\phi - \psi) & 0 \\ \cos(\phi - \psi) & -\sin(\phi - \psi) & 0 \end{bmatrix}

  • 이 텐서를 파고들어라. 기호 \phi\text{(Roll)} 와 \psi\text{(Yaw)} 가 독립적인 자유도(Degree of Freedom)를 박탈당하고, 통째로 파열하며 엉겨 붙어 오직 둘의 뺄셈 차이, (\phi - \psi) 혹은 덧셈 묶음으로만 존재(하나의 변수화) 해 버린다!
  • 즉 컴퓨터 입장에서는 수직으로 솟아가는 로켓이 왼쪽으로 돌 때 이게 수직 상승인 채로 몸을 비틀어 돈 건지(Roll 등정), 아니면 방향타를 차서 돈 건지(Yaw 선회) 를 수학적으로 ’결코 분간(Decouple)’할 방법이 소멸한다!
  • 3\text{D} 좌표계가 자유도 하나를 잃어버리고(3\text{DOF} \rightarrow 2\text{DOF}) 평면 2차원으로 찌그러지는 공간 함몰, 이것이 짐벌(회전 링) 두 개가 물리적으로 일자로 포개져 꽉 잠겨버리는 \text{Gimbal Lock (특이점, Singularity)} 텐서의 파괴 기전이다.

2. 야코비(Jacobian)의 역행렬 0 붕괴와 제어 불가(Uncontrollable) 상태

이게 왜 비행기와 인공위성을 떨어뜨리느냐?
기계 공학 비행 컴퓨터는 지금 내 짐벌(자이로스코프) 센서 각속도 무전파 \vec{\omega} 를 받아서 역산(Inverse Matrix)하여 비행기 각도 \dot{\Phi} 로 스티어링 치환해야 한다(\dot{\Phi} = \mathbf{J}^{-1} \cdot \vec{\omega}).

  • 이 변환 텐서(Euler Kinematic Equation Jacobian) 역시 분모에 항상 \cos(\text{Pitch}) 를 달고 다닌다 (\frac{1}{\cos\theta}).
  • 비행기가 90^\circ (\theta \rightarrow 90^\circ) 하늘을 찌르면 \cos(90^\circ) = 0 이 되므로! 제어 미분기 야코비 안의 분수식이 아찔한 1 / 0 = \text{무한대}(\infty) 통제 불능(Singularity) 오류 쇳덩어리를 내뿜는다.
  • 오토파일럿 컴퓨터 CPU 는 \text{NaN(Not a Number)} 혹은 램 폭주를 띄우며 모터 출력을 극단으로 쏘아버리고, 로켓은 공중에서 허우적대며(Tumble) 산산조각 폭발하는 파멸을 맞이한다.

\theta = 90^\circ 에서 발생한 이 우주적 오일러 축 버그(Bug)는 절대 수학적 꼼수 미분 따위로 치유되지 않는다. 구(Sphere)라는 위상수학(Topology) 공간을 달랑 원통 좌표축 3개의 숫자(\phi, \theta, \psi)로 커버(매핑)하려는 시도 자체가 품은 근본적 지도학적 한계, 지구본 평면(Mercator) 왜곡처럼 어쩔 수 없이 북극/남극에서 터지는 특이점(Singularity) 종말 텐서다.
이 특이점의 무저갱을 영원히 피하고 우주를 완벽하게 360^\circ 자유 통제하기 위해, 수학자들은 복소수 4차원의 결계, 쿼터니언(Quaternion, 사원수) 이라는 궁극의 행렬 체제를 3\text{D} 렌더링에 이식하게 된다.
