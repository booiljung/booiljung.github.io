3.1.2.1 오일러 각(Euler Angles) 체계 방향 코사인(Direction Cosine) 및 쿼터니언(Quaternion)

광활한 우주 공간에서 유영하는 인공위성이나 허공을 도는 전투기를 생각해 보자. 이 기체들이 지금 하늘(천정)을 향해 코를 들고 있는지, 아니면 땅바닥을 향해 거꾸로 추락하고 있는지(Attitude, 자세) 도대체 수학적으로 어떻게 묘사할 수 있을까?
컴퓨터 모니터 안의 3D 공간을 설계하는 프로그래머와 기계 동역학 공학자는 물체의 회전을 표현하기 위해 단순히 X, Y, Z 각도를 3개 부르는 원시적 수준을 넘어, 수학의 극한에 닿은 정밀한 기하학 툴을 사용한다.
본 장에서는 18세기에 고안된 인간에게 가장 직관적인 회전 좌표계인 오일러 각(Euler Angles) 의 미학부터, 3 \times 3 기하 매트릭스로 엮어내는 방향 코사인 행렬(Direction Cosine Matrix, DCM), 그리고 짐벌 락(Gimbal Lock)이라는 공간 붕괴의 공포를 피해 아인슈타인과 게임 엔진 프로그래머들이 선택한 궁극의 4차원 허수 단위 쿼터니언(Quaternion) 대수학까지, 우주 회전 변환망의 진보를 탐구한다.
