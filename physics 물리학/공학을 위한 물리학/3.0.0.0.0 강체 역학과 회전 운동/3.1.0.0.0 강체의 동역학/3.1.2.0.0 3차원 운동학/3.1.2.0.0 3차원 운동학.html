3.1.2 3차원 운동학

질점이 단순히 선을 따라 움직이는 선형 운동학의 시대를 지나, 거대한 부피를 지닌 강체(Rigid Body) 객체가 우주 공간 3차원 축(x, y, z) 전체를 무대로 빙글빙글 회전하며 나아가는 3차원 운동학(\text{3D Kinematics}) 의 영토에 진입한다.
평면 위의 동역학이 단순히 속도 벡터의 덧셈과 뺄셈이었다면, 3차원 회전 운동학은 오일러 각(Euler Angles), 방향 코사인 행렬(Direction Cosine Matrix), 그리고 4차원 복소수 쿼터니언(Quaternion)과 같은 비선형 대수 행렬망이 지배하는 공간 회전(Rotation)의 미적분학이다. 이 장에서는 항공기, 로봇 팔, 인공위성이 허공에서 자세를 뒤틀 때 발생하는 기하학적 텐서의 본질과 각속도의 변환 렌더링을 추적 해부한다.
