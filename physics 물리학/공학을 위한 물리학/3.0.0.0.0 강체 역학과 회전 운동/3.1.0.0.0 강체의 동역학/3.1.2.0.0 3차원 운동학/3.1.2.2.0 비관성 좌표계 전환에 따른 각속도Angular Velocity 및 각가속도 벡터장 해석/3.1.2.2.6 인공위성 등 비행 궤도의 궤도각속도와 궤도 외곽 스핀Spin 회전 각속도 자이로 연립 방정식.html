3.1.2.2.6 인공위성 등 비행 궤도의 궤도각속도와 궤도 외곽 스핀(Spin) 회전 각속도 자이로 연립 방정식

허구한 날 달나라로 쏘아 올린 인공위성(\text{Satellite})이 도대체 3차원 우주 안에서 어떻게 돌아가고 있는지, 그 각속도(\vec{\omega})의 정체를 해체해 보자. 정지 궤도가 아닌 한, 인공위성은 모행성(지구)을 중심으로 거친 극좌표 원운동을 하며 하루에 십수 번씩 지구를 맴돈다(\text{Orbital Rate}, 궤도 공전 각속도). 동시에 위성은 자기가 태양 전지판을 펴거나 안테나를 쏠려고 자기 몸뚱어리의 중심축에 모터를 처박고 360^\circ 미친 듯이 자전(\text{Spin Rate}, 회전 각속도)을 먹인다.
즉, 지구를 빙빙 도는 그 거대 궤도 공전 회전판 좌표계 안에서, 위성 본체의 또 다른 로컬 스핀 텐서 자이로가 터져 나오는 2 중 다중 회전 병합(Dual Rotation Coupling)의 연립 방정식장, 그것이 인공위성을 절대 통제 불능(\text{Tumbling}) 늪으로 빠지기 직전 구해내는 운동학 뼈대다.

1. 궤도 각속도(\vec{\omega}_0)와 기체 각속도(\vec{\omega}_{spin})의 텐서 적층

인공위성의 우주 절대 각속도 방향 코사인 행렬 \mathbf{C} 를 미분하여 얻어야 하는 텐서는, 지구 고정 좌표(\text{ECI, Earth-Centered Inertial})에서 바라본 인공위성의 최종 절대 회전 벡터 \vec{\omega}_{abs} 다.

\vec{\omega}_{abs} = \vec{\omega}_{0} (\text{궤도각속도 Orbital Rotation}) + \vec{\omega}_{spin} (\text{자전각속도 Body Spin Rate})

  • 1\text{차 회전 } \vec{\omega}_0: 만약 인공위성이 아무 모터도 안 틀고(\vec{\omega}_{spin} = 0) 그냥 가만히 돌덩어리처럼 \text{LEO}(저궤도)를 둥그렇게 원 모양으로 돌고 있다고 치자.
    위성 앞머리가 항상 진행 곡면 접선(Tangent)을 향하고 바닥(센서)이 계속 지구 중심을 내려다보며 돌려면, 기체 전체는 한 바퀴 우주 궤도를 비행할 때 딱 1 번, 지구를 내려다보기 위해 스스로 공전 궤도 중심축(\text{Pitch Axis})으로 360^\circ 뒤집어져야(1회전) 하는 궤도 각속도(보통 \vec{\omega}_0 \approx -0.001 \text{ rad/s} 음수 피치)가 탑재된 상태다. 이 기괴한 각속도가 베이스라인으로 먼저 깔린다.
  • 2\text{차 회전 } \vec{\omega}_{spin}: 이제 여기에 자이로 휠(\text{Reaction Wheel}) 모터가 개입해 위성을 Roll 이나 Yaw 로 까딱거리며 비틀게 만드는 내부 각속도 통제 변위를 때려 박아 얹어 더한다(Vector Addition). 이 각속도 텐서 \vec{\omega}_{spin} 가 \vec{\omega}_{abs} 와 \vec{\omega}_0 사이의 오차를 보정하기 오토파일럿 보상(Euler Equation 제어 입력) 엔진이다.

2. 오일러-자이로 연립 방정식망, 그리고 중력 구배(Gravity Gradient)의 역습

미사일 컴퓨터(제어기)는 \mathbf{I}\vec{\alpha}_{abs} + \vec{\omega}_{abs} \times (\mathbf{I}\vec{\omega}_{abs}) = \vec{\tau}_{ext} (오일러 강체 동역학) 미분방정식 안에, 방금 겹겹이 쌓아 올린 (\vec{\omega}_0 + \vec{\omega}_{spin}) 텐서를 쑤셔 넣고 끓인다. 문제는 이 기체가 궤도를 돌 때, 조용히 놔두면 우주 진공이 외부 토크 \tau_{ext} = 0 이라 영원히 멈출 줄 알았건만, 지구의 질량이 뿜어내는 가히 악마적인 미세 결박, 중력 구배 토크(Gravity Gradient Torque) 가 덤벼든다는 사실이다.

\vec{\tau}_{\text{gravity}} = 3 \omega_0^2 (\vec{r}_c \times (\mathbf{I} \cdot \vec{r}_c))

  • 인공위성이 길쭉한 통나무 모양이라고 치자. 지구 밑바닥(땅)과 좀 더 가까운 통나무의 대가리는 더 강한 지구 중력에 당겨 떨어지려 하고, 꼬리 쪽은 아주 몇 \text{m} 멀다는 이유로 중력이 미세하게 약해서 덜 당겨진다.
  • 이 대가리와 꼬리의 중력차(Gradient) 텐서가, 결국엔 길쭉한 우주선을 아예 땅을 향해 일직선 수직으로 서버리게(Stable 정렬) 만드는 자비 없는 거대 토크 \vec{\tau} 미분 회전력으로 돌변하여 덮친다.
