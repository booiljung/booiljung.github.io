3.1.2.2.2 각속도 벡터의 포아송 적분 속도(Poisson Kinematic) 방정식 및 회전 행렬 시간에 대한 편미분파

수학자 시메옹 드니 포아송(Poisson)은 “물체가 회전하며 자세 행렬(\mathbf{C})을 계속 틀어버릴 때, 컴퓨터가 이 각도를 미분해서 자이로스코프 센서가 뱉어내는 3차원의 진짜 물리적 \text{각속도}(\vec{\omega}) 를 어떻게 쥐어짜 낼 수 있는가?“라는 숙제를 풀었다.
방향 코사인 행렬(\text{DCM})은 9차원의 거대 숫자 배열이고, 각속도 \vec{\omega} = [\omega_x, \omega_y, \omega_z] 는 달랑 3개의 1차원 스칼라 좌표 꼬챙이다.
3개의 물리적 \mathbf{RPM} 이 어떻게 9차원 텐서를 비틀어가며 시공간 운동학(Kinematics) 방정식을 엮어내는가? 이 장은 회전하는 모든 사물에 대한 미적분의 핵심, 포아송 운동학 비대칭 편미분 방정식(Poisson Kinematic Equation) 에 바쳐진다.

1. 텐서의 시간에 대한 미분: \dot{\mathbf{C}} = \mathbf{C} \cdot ?

어떤 뼈대(좌표축 매트릭스 \mathbf{C})가 회전할 때, 그 축들의 변형 속도(미분, \dot{\mathbf{C}})는 무조건 현재 지금의 자세 각도 뼈대(\mathbf{C})에다가 무언가(각속도 연립 텐서)를 곱한 것으로 나온다.
포아송은 이 식을 다음과 같이 기하학적 파멸 선고로 압축했다.

\dot{\mathbf{C}} = -\tilde{\omega} \mathbf{C}
(혹은 곱셈 참조 좌표계 기준에 따라 \dot{\mathbf{C}} = \mathbf{C} \tilde{\omega} 구배)

  • 여기서 \mathbf{C} 는 현재 기계가 위치한 3 \times 3 기하학 자세(\text{Attitude}) 매트릭스.
  • 저 정체불명의 괴상한 변수 기호 **\tilde{\omega} (오메가 틸드)**가 바로 이 우주 공간 비틀림을 지휘하는 폭군, 자이로스코프 각속도 파편 3개(\omega_x, \omega_y, \omega_z)를 엮어 만든 3 \times 3 교차대칭(Skew-Symmetric) 치환 행렬이다!

2. 교차 대칭 행렬(Skew-Symmetric Matrix, \tilde{\omega})의 발현

포아송은 벡터 십자가 외적(Cross Product, \vec{\omega} \times \vec{r}) 연산 방식을 컴퓨터가 가장 좋아하는 선형 대수 매트릭스 1단계 껍질 곱셈 따위로 둔갑시켜 버리기 위해, 스칼라 벡터 숫자 3개를 행렬 9칸 지뢰밭에 기괴하게 줄 세워서 0 으로 구멍 내 박아 넣었다.

\tilde{\omega} = \left[ \vec{\omega} \times \right] = \begin{bmatrix} 0 & -\omega_z & \omega_y \\ \omega_z & 0 & -\omega_x \\ -\omega_y & \omega_x & 0 \end{bmatrix}

포아송 포메이션의 섬뜩한 대칭 규칙:

  1. 대각선 가운데 뼈대는 전부 무조건 0 (Zero).
  2. 대각선을 찔러 넘어가면 양옆의 숫자가 절대치 크기는 같은데 부호가 무조건 반대(-, +) 로 켤레 대치(Antisymmetric) 되며 박힌다.
  3. 내 매트릭스를 Transpose(전치 행렬로 대각 뒤집기)시키면, 기가 막히게 원래 행렬에 통째로 마이너스 (-) 부호를 곱한 자기 자신 폭파 수치와 동일해진다. (\tilde{\omega}^T = -\tilde{\omega})

포아송 운동학 편미분 방정식인 \dot{\mathbf{C}} = -\tilde{\omega} \mathbf{C} 는 기하학적 각도(\text{Angle Matrix})와 3차원 물리적 회전 속력(\text{RPM Rate Vector})이라는 전혀 다른 차원의 두 괴물을 가장 우아하게 연결하는 스큐 대칭(Skew-Symmetric)의 교량이다. 회전 기체가 돌며 뒤엉킨 \text{Euler Angle} 좌표로 분해하지 말고, 9마리의 코사인 \text{DCM} 이라는 무지성 매트릭스 행렬 그 자체에 편미분을 때려 박아서, 오차 한계 없이 실시간으로 컴퓨터 메모리에 적분 텍스처를 칠해버리는 것이 포아송의 승리 역학이다.
