3.1.2.2.1 베이스 전역(Global) 고정 좌표계와 강체 부착 로컬(Local) 회전 좌표계 간 운동학적 속도 매핑

지구라는 멈춰있는 땅바닥(Global Frame, \text{X-Y-Z}) 위를 헬리콥터가 날아가고 있다. 조종석 안에는 파리 한 마리가 이리저리 무작위로 윙윙거리며 날아다닌다. 땅바닥에 서 있는 사람은 헬리콥터 투명 유리창 너머로 날아다니는 파리를 보며 “파리가 무지하게 빨리 날아간다!“라고 소리치고, 헬리콥터 안에 앉은 조종사(Local Frame, x-y-z)는 “파리가 겨우 시속 1\text{km/h} 로 맴도네“라고 착각한다.
이 두 개의 프레임 사이, 즉 “멈춰있는 절대 우주축(Inertial Base)” 과 “자체가 미친 듯이 빙글빙글 돌고 있는 이동 관측자의 방(Rotating Local Rigid Body)” 사이에서 속도를 정밀하게 번역해 주는 좌표 매핑 수식은 모든 모바일 로봇과 우주 제어 공학의 최우선 뼈대다.

1. 관성 벡터(\vec{R}) 분해: 병진(Translation)과 회전(Rotation)의 텐서 해부

땅에 서 있는 사람이 본 파리의 진짜 절대 위치 벡터 \vec{R} 은,

  1. 원점에서 헬기 중심까지의 거리(\vec{R}_0)와
  2. 헬기 의자에서 파리까지의 거리(\vec{r}, 조종사가 본 파리의 위치)의 합(\vec{R} = \vec{R}_0 + \vec{r})이다.
    여기서 아주 조심해야 할 텐서의 악마가 숨어있다. 조종사가 파리를 쳐다본 꼬챙이 벡터 \vec{r} 은 r_x \mathbf{i} + r_y \mathbf{j} + r_z \mathbf{k} 로 쪼개지는데, 여기서 \mathbf{i, j, k} (헬리콥터의 방향 꼬챙이)는 우주 공간에서 가만히 있지 않고 계속 빙글빙글 각속도 \vec{\omega} 를 머금고 돈다는 점이다!

\vec{V}_{\text{Global}} = \frac{d\vec{R}}{dt} = \frac{d\vec{R}_0}{dt} + \frac{d}{dt}(r_x \mathbf{i} + r_y \mathbf{j} + r_z \mathbf{k})

  • d\vec{R}_0 / d t: 헬리콥터가 땅바닥 대비 앞으로 튀어나가는 순수 병진 속도(\vec{V}_{\text{Heli}}).
  • d/dt (r_x\mathbf{i} + \dots) 체인 룰: 수학 미분법 연쇄 법칙에 의해 이 항은 폭발한다.
    1. \mathbf{i, j, k} 를 상수로 두고 거리 r 숫자들만 미분한 파트(조종사가 헬기 안에서 본 파리의 순수 비행 속도 \vec{v}_{\text{rel}})
    2. 내 파리 거리 r 들은 가만히 냅둔 채, 헬기 자체가 틀어지며 변해버리는 축 좌표 꼬챙이 모양의 미분 \frac{d\mathbf{i}}{dt} 회전 기어 파트.

2. 각속도 크로스 프로덕트(\vec{\omega} \times \vec{r}) 교차 항의 발현

고체역학의 신 오일러는 방금 살펴본 “헬기가 틀어지면서 생기는 축의 변화율 \frac{d\mathbf{i}}{dt}“를 기괴할 정도로 완벽하고 심플한 벡터 외적(Cross Product) 단 한 줄로 분쇄해 버렸다.

\frac{d\mathbf{i}}{dt} = \vec{\omega} \times \mathbf{i}

헬리콥터가 각속도 \vec{\omega} 로 선회하며 우주 공간을 비틀 때, 그 내부의 모든 부품(\mathbf{i}, \mathbf{j}, \mathbf{k} 나사가 꽂힌 방향)의 물리적 시간 꺾임 미분은 오직 \mathbf{\vec{\omega} \times (\text{위치 벡터})} 라는 원심 접선 궤적으로 매핑되어 튀어나온다.

최종 속도 매핑 방정식 (Kinematics Translation Equation):
\vec{V}_{\text{Global}} = \vec{V}_{\text{Heli}} + \vec{v}_{\text{rel}} + (\vec{\omega} \times \vec{r})
