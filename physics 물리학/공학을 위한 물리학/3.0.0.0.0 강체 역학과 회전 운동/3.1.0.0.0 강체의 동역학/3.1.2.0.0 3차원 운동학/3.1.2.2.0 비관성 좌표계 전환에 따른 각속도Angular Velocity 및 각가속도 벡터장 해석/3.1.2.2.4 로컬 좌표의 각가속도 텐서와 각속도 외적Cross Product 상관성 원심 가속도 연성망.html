3.1.2.2.4 로컬 좌표의 각가속도 텐서와 각속도 외적(Cross Product) 상관성 원심 가속도 연성망

각속도(\vec{\omega})가 그저 물약 병에 채워진 푸른 피라면, 미분 냄비에서 한 번 더 푹 고아 끓여낸 각가속도(Angular Acceleration, \vec{\alpha} = \frac{d\vec{\omega}}{dt}) 는 우주 기계의 살점을 뜯어내는 붉은 화염이다.
단순히 속도가 빨라지거나 느려지는 1차원 가속도 스칼라 체계를 치워버려라. 3차원 허공에서 물체가 거대한 회전을 물고 또 돌면서 가속(\vec{\alpha})을 때려 넣으면, \vec{\alpha} 라는 화살표 자체도 돌아간 축의 변위(\vec{\omega} \times) 위에서 춤을 춘다! 이는 기계 진동의 핵폭탄인 공진을 부르고, 관성 항법 센서(\text{IMU}) 가속도계를 미치게 만드는 비선형 연성(Coupling) 렌더링 미분이다.

1. 각가속도의 미분 본질: 회전 속도는 누가 꼬았나?

조종사가 로컬 방안(x, y, z)에 앉아 계기판을 보며 “아, 우리 비행기가 \vec{\alpha}_{\text{local}} 만큼 각가속도를 먹었군!” 하고 적어 놓았다.
그러나 우주 바깥, 절대 고정 지구 좌표계(\text{Global Inertial Frame})에서 이 비행기를 바라보는 신(God)의 눈길에 잡히는 진짜 물리적 각가속도(\vec{\alpha}_{\text{global}}) 수치는 조종사의 것과 100\% 다르게 폭발한다.

\text{절대 각속도 벡터 } \vec{\omega}_{\text{abs}} 를 우주의 시간(t)에 대해 미분하자.

\vec{\alpha}_{global} = \frac{d}{dt} \vec{\omega}_{abs} = \frac{d}{dt} (\vec{\omega}_{heli} + \vec{\omega}_{local})

  • \mathbf{1항 (\dot{\vec{\omega}}_{heli})}: 헬리콥터 뼈대 자체가 미친 듯이 돌면서 내뿜게 된 모터 각가속도 화살표.
  • \mathbf{2항 (\dot{\vec{\omega}}_{local})}: 조종사가 자기 방 안에서 팔을 젓어 돌리는 순수 각가속도 텐서.
  • 문제의 핵 \mathbf{3항}: 조종사(Local)의 축을 물고 있는 x, y, z 화살표 뼈대 자체가 1 항의 \vec{\omega}_{heli} 때문에 허공에서 같이 비틀리며 넘어가고 있었다! 즉, 여기서 다시 포아송(Poisson)의 악령 교차 벡터 회전 미분 항(\vec{\omega} \times) 타격이 부활한다!

2. 연성 회전망, 최후의 절대 각가속도 결속 방정식 (Kinematic Acceleration Equation)

결국 우주 바깥에서 관측된 기계 부품(조종사 팔)의 최종 유효 각가속도 폭발망은 다음과 같이 3 개의 항으로 분해되어 우주 공간을 덮친다.

\vec{\alpha}_{global} = \vec{\alpha}_{heli} + \vec{\alpha}_{local} + \mathbf{\vec{\omega}_{heli} \times \vec{\omega}_{local}}

  • 가장 악랄한 항(\vec{\omega}_{heli} \times \vec{\omega}_{local}): 자이로스코프 효과 토크라 불리기도 하는 এই 교차 연성항(Cross-coupled Term)을 직관해 보라.
    헬리콥터가 아무런 회전 각가속도 없이(\vec{\alpha}_{heli} = 0) 아주 평온하게 일정 속도(\vec{\omega}_{heli} = 100\text{RPM}) 로만 스핀(Spin) 치고 있고, 조종사 팔도 일정 속도(\vec{\alpha}_{local} = 0, \vec{\omega}_{local} = 50\text{RPM}) 로만 조이스틱을 돌린다고 쳐도,
    엄청난 물리적인 타격 가속도(\vec{\alpha}_{global} 망치) 가 \vec{\omega} \times \vec{\omega} 내적 공간 위에서 펑 터져 나와버린다!

\vec{\omega}_1 \times \vec{\omega}_2 의 교차 텐서 외적은 3 차원 로봇 관절의 미분 속살 그 자체다. 오직 스칼라 1차원의 바보 같은 숫자 덧셈에선 일정한 속도면 무조건 가속도가 0 이었지만, 화살표(방향, Direction) 가 지배하는 회전의 세계에선 속도 크기(Magnitude)가 영원히 상수 \text{Constant} 라도 방향만 뒤틀리면 무조건 가속도를 파열시킨다! 이 파열음(가속도)이 곧 힘(\mathbf{F=ma, \tau = I\alpha})으로 돌변하여 쇳물 베어링을 깨버리는 3\text{D} 공간의 거친 공학 십자가다.
