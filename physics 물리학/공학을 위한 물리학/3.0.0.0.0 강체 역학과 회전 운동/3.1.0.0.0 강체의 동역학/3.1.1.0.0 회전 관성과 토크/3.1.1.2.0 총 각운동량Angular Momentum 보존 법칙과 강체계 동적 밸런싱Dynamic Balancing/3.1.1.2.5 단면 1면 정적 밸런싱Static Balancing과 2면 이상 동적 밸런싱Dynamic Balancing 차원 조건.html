3.1.1.2.5 단면 1면 정적 밸런싱(Static Balancing)과 2면 이상 동적 밸런싱(Dynamic Balancing) 차원 조건

자동차 정비소 타이어 가게에 가서 바퀴를 교체해 본 엔지니어라면, 정비공이 커다란 타이어를 휠 밸런스 기계(Balancing Machine)에 꽂아 윙윙 돌린 뒤 안쪽 림(Rim)과 바깥쪽 림에 납 테이프 조각을 하나씩 각각 턱턱 붙여주는 모습을 목격했을 것이다. 왜 납덩이를 타이어 한가운데에 한 번만 붙이지 않고 굳이 왼쪽 볼, 오른쪽 볼 ’두 군데(2-Plane)’에 갈라서 붙일까?
그것이 바로, 정지해 있을 때는 축이 아무리 평온해도(정적 평형) 회전하는 순간 \vec{\omega} \times \vec{H} 로 발작하는 3차원 비틀림 텐서 불평형(Couple Unbalance)을 진압하는, 강체 기구학 동적 밸런싱(Dynamic Balancing) 의 신성한 다중 단면(Multi-Plane) 결속 조건이다.

1. 정적(Static) 밸런싱: 1면(Single-Plane) 무게 오차 잡기 (단순 원심력 상쇄)

엄청나게 얇은 날을 지닌 파이 톱니바퀴(원판 Disk, 두께가 거의 0)가 회전축에 꽂혀 있다고 치자. ఈ 디스크가 돌기 시작할 때 \text{RPM} 이 흔들리는 이유는, 오직 원판 무언가 한쪽에 질량(m_e)이 몰려 있어서 그쪽으로 쏠린 원심력 \vec{F}_c = m e \omega^2 이 밖으로 튀어나가려고 요동치기 때문이다.

  • 해결책 (1-Plane Balancing): 그냥 공학자는 수학도 미분도 잡고, 가장 단순하게 \mathbf{\sum \vec{F} = 0} 이라는 힘의 평형 방정식만 전개한다. 반대편 180^\circ 지점을 찾아, 딱 질량 오차만큼 똑같은 보정 질량(m_c r_c) 납덩이를 딱 1번만 하나 붙이면 끝이다.
  • 이것이 두께가 얇은 기어(Gear), 자동차 클러치, 얇은 풀리(Pulley) 등에 쓰이는 가장 원시적인 1면 정적 밸런싱(Static Balancing)이다. 돌리지 않고 그냥 칼날에 얹어만 놔도 무거운 쪽이 툭 떨어져서 어디가 편심(\text{Heavy spot})인지 직관적으로 튀어나온다.

2. 동적(Dynamic) 밸런싱: 2면(Two-Plane) 모멘트 오차 잡기 (비틀림 텐서 상쇄)

하지만 터보차저(Turbocharger), 커다란 자동차 실린더 크랭크축이나 대포알같이 ‘길쭉한 통나무통(Cylinder)’ 로 길어지는 순간. 이야기는 완전히 달라진다.
통나무는 정지해 있을 땐 균형을 잡고 칼 축 위에 아주 다소곳이(정적으로) 잘 서 있을 수 있다! “왼쪽 맨 위(+Z 축)” 모서리에 파리 100마리 무게가 몰려있고, “오른쪽 맨 아래(-Z 축)” 모서리에 파리 100마리 무게가 정확히 반대로 몰려 있으면 \mathbf{\sum \vec{F} = 0} 은 정지계에서 완벽히 일치 성립하기 때문이다. (정적 밸런스 완료 상태.)

  • 그러나 자비 없는 스위치로 모터를 윙 돌리기 시작하면?
  • 왼쪽 위(+Z)는 위로 올라가려고 원심력을 뿜고, 오른쪽 아래(-Z)는 아래로 내려가려고 강력한 원심력을 수직으로 뿜어버린다! 힘의 합력은 0 일지 언정, 두 힘이 서로 반대 허공을 찟어 발기면서 거대한 회전 비틀림 우력, 커플 텐서 모멘트(\sum \vec{M}_{couple} \ne 0) 를 창조해 샤프트의 중심 허리를 두 동강 낼 기세로 비틀어버린다. 동적 진동 붕괴(Wobbling) 발작이다.

자동차 크랭크(Crank)축에 달려있는 미친 듯한 쇳덩어리 덩이들 카운터 웨이트(Counter Weight)들은 절대 똑같은 자리에, 같은 곳을 바라보도록 몰려 있지 않다. 하나는 위를 보면 다른 놈은 비틀어진 90^\circ 에 달려서 동적 비틀림(Couple Unbalance \sum \vec{M} = 0) 을 방어하는 제2방어선, 제3방어선 다중면(Multi-plane) 역학 무기로 서로를 수십 수백 RPM 의 늪 속에서 견제한다. 힘이 평형을 이루어도 모멘트가 붕괴하면 기계는 피로 파절(Fatigue Fracture)로 분해된다.
