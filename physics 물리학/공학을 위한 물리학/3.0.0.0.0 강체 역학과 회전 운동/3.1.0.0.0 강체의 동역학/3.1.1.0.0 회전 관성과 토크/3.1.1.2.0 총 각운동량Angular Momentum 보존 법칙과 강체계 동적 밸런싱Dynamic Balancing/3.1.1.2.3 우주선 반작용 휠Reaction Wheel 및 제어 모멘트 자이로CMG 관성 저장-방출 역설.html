3.1.1.2.3 우주선 반작용 휠(Reaction Wheel) 및 제어 모멘트 자이로(CMG) 관성 저장/방출 역설

허블 우주 망원경(Hubble Telescope)이 먼 퀘이사 별을 찍으려고 아주 미세하게 고개를 0.001^\circ 돌려야 한다 치자. 로켓 추진 가스를 칙칙 뿜는 스러스터(Thruster) 방식은 너무 거칠고, 한 번 발사한 연료는 영원히 소모되어 망원경의 수명을 깎아 먹는다.
기계 공학자와 로봇 제어 공학자는 이 무중력 진공 공간에서 우주선 본체의 방향을, 외부 가스 분출 없이 오직 뱃속에 품고 있는 쇳덩어리 바퀴(Rotator)를 이리저리 굴리는 것 하나만으로 유령처럼 돌려버리는 궁극의 각운동량 텐서 저장/방출 무기, 반작용 휠(Reaction Wheel) 과 제어 모멘트 자이로(Control Moment Gyro, CMG) 를 탑재한다. 이것은 뉴턴 역학 3법칙 작용-반작용과 각운동량 보존 \vec{H}=C 이 엮어낸 다물체(Multi-Body) 회전 역학의 최고 걸작이다.

1. 반응 바퀴(Reaction Wheel): I_{rotor} \cdot \omega_{rotor} 와 본체의 반동 교환식

우주선 본체(Body) 안에 무거운 팽이 로터(Wheel) 하나를 모터와 연결해 달아놓았다 치자. 처음에 둘 다 가만히 멈춰 있으면 우주선의 총 각운동량은 절대 진리의 0\text{(Zero)} 이다.
이제 망원경이 오른쪽(+x)으로 사진을 찍기 위해 머리를 돌리고 싶다. 그럼 미친 듯이 로켓을 쏠 필요 없이, 그냥 몸속의 모터에 전기를 먹여 내부의 무거운 쇳덩어리 팽이(Wheel)를 왼쪽(-x)으로 팽팽 돌리기 시작한다.

\vec{H}_{total} = \vec{H}_{body} + \vec{H}_{wheel} = 0
\vec{H}_{body} = -\vec{H}_{wheel} \quad \rightarrow \quad I_{body} \omega_{body} = - (I_{wheel} \omega_{wheel})

  • 팽이가 왼쪽으로 각운동량 100을 꾸역꾸역 먹으며 돌기 시작하면(\vec{H}_{wheel} = -100), 지독한 \vec{H}_{total} = 0 보존 철칙에 의해 우주선 몸뚱어리는 어쩔 수 없이 자신을 강제로 오른쪽 각운동량 100으로 밀어붙여야 한다(\vec{H}_{body} = +100).
  • 즉, 배 속의 바퀴를 돌리는 행위만으로 몸체 전체가 반대 방향으로 반동 회전을 개시한다! 우주선이 원하는 위치에 도달하면 바퀴를 그냥 브레이크 밟아 멈추면 몸체도 덩달아 멈춘다.
  • 단점은, 우주선이 계속 한 방향으로만 돌아야 하면 바퀴의 \text{RPM} 이 터질 듯이 무한대로 끝없이 올라가다 모터가 깨져버리는 ‘모멘텀 포화(Momentum Saturation)’ 한계에 박힌다는 늪이다.

2. 제어 모멘트 자이로(CMG): 토크 증폭기 기계식 무한 레버리지 기구

단순 반응 바퀴가 “내 팽이 속도를 올려서 몸체를 돌린다“라면, 거대 우주 정거장(ISS) 급에 달리는 CMG는 차원이 다른 역학 지렛대를 쓴다. CMG 안의 팽이는 모터 속도 \text{RPM} 을 바꾸지 않고 항상 \mathbf{10,000 \text{RPM}} 의 무자비한 고정 속도로 웽웽 돌고 있다. 대신, 이 팽이의 꽂혀 있는 축 지지대(Gimbal) 통째의 방향(각도)을 슬쩍 비틀어 축 방향 화살표를 꺾어버린다!

\vec{\tau}_{body} = - \vec{\omega}_{gimbal} \times \vec{H}_{wheel}

  • 이미 10,000 \text{RPM} 으로 팽배하게 회전 에너지가 가득 찬 팽이 축 \vec{H}_{wheel} 무기를, 짐벌 모터가 아주 쥐꼬리만 한 힘으로 살짝 축을 밀어서 \vec{\omega}_{gimbal} 각속도로 틀어(Precession) 제껴버리면!
  • 이 두 개의 거대 각 텐서 벡터가 직진(\vec{A} \times \vec{B} 스매시)으로 충돌하면서 본체를 후려치는 반작용 출력 토크(\vec{\tau}_{body})는 상상을 초월하는 증폭 텐서 폭파를 일으킨다.
  • 짐벌 모터가 고작 1 \text{N}\cdot\text{m} 의 살짝 힘으로 축을 틀어도, 본체를 잡아 돌리는 토크는 100 \text{N}\cdot\text{m} 급인 수백 배의 물리적 토크(Torque Amplifier) 로 뻥튀기되어 튀어나온다.

\mathbf{RW}(Reaction Wheel) 가 단순한 에너지 질량 보존의 은행(Bank)과 톱니바퀴의 1차원적 연산이라면, \mathbf{CMG}(제어 모멘트 자이로)는 이미 거대하게 굴러가고 있는 각운동량 괴물 벡터 화살표 무리를 살짝 비틀어, 우주 전체 공간계를 거대한 각속도 크로스 프로덕트(\vec{\omega} \times \vec{H}) 지렛대 망치로 때리는 3차원 기구학 텐서의 파괴자다. H=I\omega 의 공학적 착취, 이 두 개의 자이로가 없었다면 달의 뒷면을 조준하는 우주 시대는 존재하지 못했다.
