3.1.1.2.1 회전 중심계 외부 토크 평형(Zero net torque) 상태에서의 강체계 스핀 각운동량 보존

고전 뉴턴 역학의 가장 완벽한 성역은 “외부에서 힘(Force)이 쳐들어오지 않으면 쇳덩어리는 영원히 일정한 속도로 직진한다(선운동량 보존, 관성의 법칙)“는 p = mv 명제다.
하지만 우주정거장이나 지구같이 팽이처럼 빙글빙글 자전하는 거대 회전 덩어리(Rotating Body)들의 세계로 넘어가면, 이 보존의 룰은 질량과 속도라는 1차원 선분을 버리고 관성 모멘트 행렬(I) 과 각속도 벡터(\omega)가 곱해진 ‘각운동량(Angular Momentum, \vec{H})’ 텐서 공간으로 진화한다.
허공에서 도는 팽이에 아무도 손을 대지 않고 어떠한 모터 회전 축비틀림(Torque) 마찰도 존재하지 않는다면, 이 기계가 품고 있는 회전 에너지는 영원히 단 1의 파열도 없이 보존되는, 기계 공학 설계의 절대 방어 진지 방정식 \sum \vec{\tau}_{ext} = 0 \rightarrow \frac{d\vec{H}}{dt} = 0 을 전개한다.

1. 외부 토크 제로(Zero Net Torque)와 각운동량 시간 미분 파괴

우선 회전하는 강체계의 모든 동력학은 오일러(Euler)의 회전 운동 방정식에서 출발한다. 물체의 각운동량 \vec{H} 가 시간에 따라 커지거나 줄어드는(즉 바퀴가 더 빨리 돌거나 브레이크가 잡히는 미분량) 원인 제공자는 오직 외부에서 몽키스패너로 축을 비틀거나 공기 저항이 거는 마찰 회전력, 외부 총 토크(\sum \vec{\tau}_{ext}) 뿐이다.

\sum \vec{\tau}_{ext} = \frac{d\vec{H}}{dt}

  • 만약 물체가 우주 무중력의 진공 공간에 떠 있어서 아무런 바람 저항도 받지 않고, 모터의 전원도 다 꺼져 연결축이 끊겨 있다면 \sum \vec{\tau}_{ext} = 0(Zero Net Torque) 이라는 완벽한 고립(Isolated)의 평형 상태가 성립한다.
  • 그 순간 위의 미분 방정식 좌변이 0 이 되므로, \frac{d\vec{H}}{dt} = 0 이 달성된다.
  • 수학적 선고: 이것을 시간에 대해 적분하면 \vec{H}(t) = \text{Constant}(영구 불변 상수) 가 된다. 기체가 빅뱅 우주부터 멸망까지 회전하며 쥐고 있는 각운동량 총량(I \cdot \omega)은 절대로, 단 한 방울도 줄어들거나 증가하지 않고 화석처럼 봉인 보존(Conservation)되는 것이다.

2. I_i \omega_i = I_f \omega_f 체제: 초기와 나중 상태의 텐서 불변성

이 \vec{H}_{initial} = \vec{H}_{final} 이라는 아름다운 절대 결속식 속에서 우리는 \vec{H} = I \vec{\omega} 공식을 풀어넣을 수 있다.
즉 외부에서 손가락 하나 대지 않아도, 물체가 돌아가는 도중에 로봇 팔을 접거나 펼쳐서 시스템 자체의 구조 골격, 즉 회전 관성(I) 몸집을 스스로 내부에서 부풀리거나 줄인다면?

I_{initial} \cdot \omega_{initial} = I_{final} \cdot \omega_{final} = \text{Constant}

  • 아무런 자극(Torque)을 주지 않았음에도 불구하고, 오로지 살짝 팔을 접어 관성 I 가 줄어들었다(I_{final} \downarrow)는 그 내부 기하학적 요동만으로!
  • 저 영원부동한 곱셈 공식 상수(\text{Constant})를 사수하기 위해, 남은 속성인 각속도 \omega_{final} 가 맹목적으로 미친 듯이 치솟아 올라 기계의 회전 \text{RPM} 이 벼락같이 폭발한다(\omega_{final} \uparrow).

이 외부 토크 제로(\tau_{\text{net}} = 0)의 평형계 속에서 기계 동역학 설계자는 엄청난 공학적 이득(Leverage)을 얻어낸다. 만약 돌아가는 인공위성 내부의 우주인이 커피잔을 떨어트리든 내부에서 팔벌려뛰기를 하든(Internal Forces and Torques), 그 어떤 내부 폭파 수치도 시스템 겉을 감싸고 있는 전체 총 각운동량 \vec{H} 텐서를 터뜨릴 수 없다. (내부 토크는 작용-반작용으로 \sum \tau_{\text{int}} = 0 이 되므로). 이 절대 불변의 결속 방정식은 고전 역학의 아름다운 족쇄이자 기계 제어의 나침반이다.
