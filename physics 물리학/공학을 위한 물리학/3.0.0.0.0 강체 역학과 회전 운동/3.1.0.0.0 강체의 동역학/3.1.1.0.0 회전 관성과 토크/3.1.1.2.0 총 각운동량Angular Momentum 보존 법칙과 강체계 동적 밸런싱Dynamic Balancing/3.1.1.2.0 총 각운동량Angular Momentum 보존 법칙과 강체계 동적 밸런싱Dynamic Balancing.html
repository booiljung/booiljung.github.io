3.1.1.2 총 각운동량(Angular Momentum) 보존 법칙과 강체계 동적 밸런싱(Dynamic Balancing)

자유 낙하하는 물체가 직선으로 바닥에 꽂히는 병진 운동에서 선운동량(p=mv)이 지배자라면, 팽이나 우주 정거장처럼 허공에서 스핀(Spin)을 도는 강체(Rigid Body)의 운명은 철저하게 각운동량(Angular Momentum, \vec{H}) 이라는 회전 관성(I)과 각속도(\omega)의 융합 텐서에 묶여 있다.

우주 공간과 같이 마찰력과 같은 어떠한 외부 토크(\tau_{ext}) 개입도 없는 완벽한 고립계에서, 기계 시스템이 품은 각운동량의 총합은 결코 창조되거나 파괴되지 않는다(보존 법칙). 공학자들은 이 냉혹한 불변의 수식을 이용해 자세를 뒤집는 인공위성의 제어 모멘트(CMG)를 만들고 피겨 스케이팅 선수의 회전 속력을 폭발시킨다.
반면 축을 중심으로 도는 로터(Rotor)의 미세한 편심 질량은, 모터가 도는 매 찰나마다 원심력을 흩뿌리며 기계 뼈대를 부수려는 동적 불평형(Dynamic Unbalance) 파동을 터뜨린다. 우리는 이 각운동량의 보존 역설과, 회전 불평형 진폭을 잡아 억누르는 밸런싱 기구학의 지배 방정식 속으로 진입한다.
