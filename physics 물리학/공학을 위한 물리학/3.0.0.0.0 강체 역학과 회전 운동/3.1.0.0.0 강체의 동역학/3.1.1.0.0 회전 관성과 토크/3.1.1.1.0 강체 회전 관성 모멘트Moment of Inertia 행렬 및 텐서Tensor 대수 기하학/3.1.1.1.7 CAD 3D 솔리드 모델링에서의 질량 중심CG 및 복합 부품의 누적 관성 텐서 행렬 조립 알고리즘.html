3.1.1.1.7 CAD 3D 솔리드 모델링에서의 질량 중심(CG) 및 복합 부품의 누적 관성 텐서 행렬 조립 알고리즘

수백억 개의 미적분 조각 \int x \, dm 을 현대 공학자가 펜으로 직접 푸는 짓은 더 이상 존재하지 않는다. 기계 공학과 항공 우주 설계에 투입되는 모든 강체는 그저 카티아(CATIA), 솔리드웍스(Solidworks), NX 와 같은 \text{3D} 솔리드 컴파일러(CAD) 그래픽 화면 속에 폴리곤(Polygon) 껍데기와 수학적 파라메트릭(Parametric) 방정식으로 떠 있을 뿐이다.
하지만 CAD 의 ‘물성치 평가(Mass Properties Evaluation)’ 버튼 하나를 틱 클릭하는 순간 이 10만 조각짜리 V8 엔진 블록은 단 0.1초 만에 절대 오차율 0에 가까운 센터 오브 그래비티(CG, 무게 중심) x_c, y_c, z_c 와, 그 중심을 빙글빙글 도는 3 \times 3 관성 텐서(Inertia Matrix) 무결을 스크린에 토해낸다. 컴퓨터는 도대체 그 지독한 \mathbf{I = \int r^2 dm} 연산을, 어떻게 그 짧은 지연 시간 속에 찢어발기고 렌더링(Rendering)한 것일까?

1. 이산화(Discretization)와 복셀(Voxel)/다면체 적분(Polyhedral Integration)

CAD 속 컴퓨터 내부에 복잡하게 우글쭈글 찌그러진 부품의 3D 부피(Volume) 방정식 표면 전체(Boundary Representation, B-Rep)를 우겨 넣고 해석학 미분을 풀지 않는다. 그래픽 카드는 재빨리 이 덩어리를 수백만 개의 큐브(Voxel) 덩어리나 가장 다루기 쉬운 사면체(Tetrahedron) 조각으로 산산조각 썰어버린다.

  • 각 조각 스칼라 볼륨 V_i 과 재질 밀도 \rho_i 가 곱해져 파편 질량 m_i 가 생겨나고, 각 조각들의 작은 무게 중심축 (x_i, y_i, z_i) 이 절대 원점 기준 벡터로 맵핑(Mapping)된다.
  • 거시 무게 중심(Global CG): x_c = \frac{\sum m_i x_i}{\sum m_i}. 이것은 그냥 평범한 다중 시그마 가중 평균(Weighted Average) 수식이다. 메쉬 파편 10^5 개를 더블 플롯(Double float)으로 누적 합산하는 행위는 현대 CPU 아키텍처에게 1밀리초도 걸리지 않는 원시 덧셈에 불과하다.

2. 병렬화(Parallelization)된 평행축 정리(Parallel-Axis Theorem) 행렬 스매싱

더 잔인한 텐서, 조립체의 거대한 3 \times 3 관성 텐서를 단번에 쏘아 올리는 행위의 알파와 오메가는 저 위대한 슈타이너 텐서, 평행축 정리(I = I_{cg} + m d^2) 가 장악한다.

  1. 로컬 관성 추출: 쪼개진 수만 개의 뾰족한 사면체마다, 놈들 자기 자신 고유의 I_{xx, local} 이나 관성 곱 I_{xy, local} 텐서(수학 테이블에 이미 고정 상수로 박혀 있음)를 즉시 불러온다.
  2. 글로벌 CG 편차 거리 곱(m d^2): 해당 사면체 조각이 방금 찾아낸 저 엔진 전체의 ‘거대 무게 중심(CG)’ 원점으로부터 x 방향, y 축 거리 방향으로 얼마나(Distance Offset \Delta x, \Delta y) 떨어져 있는지 거리를 제곱하여 \mathbf{[m \Delta y^2, \dots]} 질량 페널티 행렬을 강제 결속시킨다.
  3. 이 두 개 행렬식 조각(자체 관성 + 거리 편차 벌금)을 백만 번, N 번 \sum \mathbf{I}_{i\_global} 미친 듯이 병렬로 더해버리면! 그 모든 스칼라 찌꺼기 덧셈의 잔해가 뭉쳐져 CAD 창 팝업에 “엔진 모멘트: I_{xx} = 2.45 \text{ kg}\cdot\text{m}^2” 라는 무적의 역학 텐서를 선고한다.

수학(Calculus)이 18세기에 오일러(Euler)와 라그랑주(Lagrange)의 머릿속에서 \int 인테그랄 기호의 추상 기하학으로 그려낸 저 우주 관성의 영적 진리를, 이 시대 실리콘 칩 컴퓨터 공학자들은 철저하게 스티븐 슈타이너의 대수식(md^2) ’단순 무식 더하기 알고리즘 루프’로 찢고 뭉개어 하드 코딩해 버렸다. 우리가 CAD 조립 창에서 구멍(Hole)을 하나 뻥 뚫을 때 질량이 스윽 날아가듯(-m), 그 구멍 체적만큼의 음수 모멘트(-I_z)가 전체 행렬망에서 즉시 절삭되어 동역학 방정식이 파열되는 가상 모의 공간, 이것이 현대 CAD-CAE 다물체 시스템 렌더링 텐서의 심장이다.
