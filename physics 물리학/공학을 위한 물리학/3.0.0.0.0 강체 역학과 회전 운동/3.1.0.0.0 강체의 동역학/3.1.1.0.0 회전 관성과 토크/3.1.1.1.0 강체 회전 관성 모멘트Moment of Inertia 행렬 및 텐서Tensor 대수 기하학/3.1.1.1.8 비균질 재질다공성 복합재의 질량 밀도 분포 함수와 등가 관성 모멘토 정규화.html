3.1.1.1.8 비균질 재질(다공성, 복합재)의 질량 밀도 분포 함수와 등가 관성 모멘토 정규화

엔지니어가 무식하게 쇳물 하나(Steel \rho_0 = 7850\text{kg/m}^3)로만 붓고 깎아 만든 옛날의 캠축, 크랭크 로드(Rod)는 중학생도 관성 모멘트(I)를 뽑아낸다. \rho_0 가 상수(Constant)이니 거대한 3중 적분 기호 \int \rho (y^2+z^2)dV 바깥으로 시원하게 튕겨 나가며 오직 순수한 모양(Geometry)의 부피 적분만 남기 때문이다.
하지만 우주선의 초경량 구동 팔(Arm)이나 방탄 스포츠카의 카본 파이버 휠 껍데기, 심지어 내부에 구멍이 잔뜩 난 다공성 폼(Foam) 알루미늄을 주조해서 쓸 때는 어떠한가? 위치 x, y, z 에 따라 쇠가 꽉 차 있기도 하고 카본 실타래가 들어차 있거나 텅 빈 공기 구멍이 존재하는 이종 재질 혼합 결속망(Composite / Heterogeneous Function) 에 진입하면 밀도 \rho 는 더 이상 상수가 아니라 공간의 사악한 함수식 \rho(x,y,z) 로 돌변해 적분 기호 안을 미친 듯이 휘저어 놓는다. 이 텐서를 수학적으로 압축 정규화(Normalization)하는 비균질 밀도 분포 적분 모델을 해부한다.

1. 공간 가변 밀도 함수(\rho(x, y, z))와 부피 텐서 속 미분 덫

위성체 터빈 디스크 구조물은 회전축 근처 중심부는 버틸 힘이 필요해 무거운 티타늄 합금 파편(\rho_{\text{high}})을 깔고, 바깥쪽 가장자리 림(Rim) 쪽은 구심력을 버티기 위해 초경량 카본 복합재 섬유망(\rho_{\text{low}})으로 짠다.
이런 반지름 r 방향 밀도 변화(Gradient)를 도면에 단순히 \rho(r) = \rho_0 e^{-\alpha r} 식의 지수 감소 스칼라 함수로 때려 박았을 때 관성 모멘트식에 떨어지는 폭탄은 자명하다.

I_z = \iiint_{V} \left( \rho_0 e^{-\alpha r} \right) \cdot r^2 \cdot (r dr d\theta dz) = \rho_0 \int e^{-\alpha r} r^3 dr

  • r^2 관성 반경 함수에 가변 밀도의 초월 함수 파동 밀도가 얽혀(Coupled) 해석적 적분식 전개를 거침없이 뒤틀어버린다.
  • 무게 중심(\text{CG}) 좌표계 자체도 더 이상 “물체의 한가운데 기하학적 중심“을 가리키지 않고, 엄청나게 무거운 티타늄 금속 덩이 쪽으로 심하게 치우쳐(Shift) 위치 텐서가 완전히 파괴 망각된다.

2. 등가 관성계 렌더링과 질량 중심 편차 다중 모달(Modal) 회피

공학자는 이런 복잡한 비균질 다공성 복합 적층(Laminate Composite) 판을 회전 파이프로 말아서 돌릴 때 3D CAD나 시뮬레이터(ANSYS) 상에서 재질을 10개씩 레이어로 쪼개어 적분하지 않는 요령(Equivalence)을 쓴다.

  1. 등가(균질화) 텐서 추출: 전체 부품을 구성하는 티타늄 부피, 카본 부피, 에폭시 본딩 체적들을 모조리 합쳐 전체 파츠 무게 M_{\text{total}} 의 무식한 스칼라를 재고 전체 기하 체적 V_{\text{total}} 로 나눠 “이 파츠 전체는 마치 가상의 평균 균질 젤리 덩어리와 같다(\rho_{\text{eq}} = \frac{M}{V})“라는 등가 밀도(Equivalent Density) 숫자로 우선 정규화해 버린다.
  2. 기하 공간에 이 \rho_{\text{eq}} 하나만 뿌려서 \bar{I}_{approx} (가짜 기준 관성)을 빛의 속도로 뽑아놓고.
  3. 실제 카본 복합재 모델의 각 층(Layer 1, Layer 2..)에서 진짜 발생할 각각의 재질 관성들을 적층이론(Laminate Stack-up) 매트릭스로 개별로 쪼개어 A, B, D 강성 텐서파들로 수작업 업데이트하는 매크로 스케일 \leftrightarrow 마이크로 픽셀 복원 동력 연립망을 가동한다.

20\text{kg} 의 자동차 주철 휠 껍데기가 우레탄(초고분자 중합체)과 경량 마그네슘, 내부 빈 공간 허니콤(Honeycomb) 구멍으로 버무려진 미친 복합재 구조로 바뀌면, 똑같은 모양이지만 휠의 속이 빈 테두리와 꽉 찬 살대 중심(Spoke)의 밀도 비율 역전 텐서는 휠 스핀의 초기 가속 반응(I\ddot{\theta})을 극한으로 파열시킨다.
관성 모멘트는 단순히 “형상 크기(r^2)“의 문제가 아니라, “무거운 놈(Density \rho_{\max})이 어디에 숨어 있느냐“를 찾는 재료 과학과 기하 미적분의 X-RAY 영사기 다. 공간 밀도 함수는 곧 구조체의 자전 가속 운명을 암호화한 공학 맵핑이다.
