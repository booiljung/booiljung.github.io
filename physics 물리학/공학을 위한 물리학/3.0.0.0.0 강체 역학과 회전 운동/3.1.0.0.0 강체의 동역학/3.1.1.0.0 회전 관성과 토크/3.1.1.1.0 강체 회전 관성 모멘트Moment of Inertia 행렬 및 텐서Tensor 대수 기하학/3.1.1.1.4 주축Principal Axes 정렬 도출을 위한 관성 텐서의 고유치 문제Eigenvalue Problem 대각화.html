3.1.1.1.4 주축(Principal Axes) 정렬 도출을 위한 관성 텐서의 고유치 문제(Eigenvalue Problem) 대각화

비대칭 강체를 아무 축(x, y, z)으로나 막상 돌려보니, 옆구리로 치고 들어오는 관성 곱(I_{xy}, I_{yz}, I_{zx}) 텐서들 때문에 각가속도 연립 방정식이 풀리지 않는 진흙탕에 빠졌다. 수학자와 공학자들은 이 난잡한 3 \times 3 행렬에서 비대각(Off-diagonal) 원소들을 모조리 0 으로 날려버리고, 오직 대각선에만 깔끔하게 숫자가 남는 대각 행렬(Diagonal Matrix) 공간으로 물체의 좌표축 자체를 억지로 뒤틀어버리는 ’상대성 회전 공간 마법’을 기획했다.

이 행렬식의 세 대각 원소를 주 관성 모멘트(Principal Moments of Inertia, I_1, I_2, I_3) 라 부르며, 이들이 향하는 직교 파동선 3개를 주축(Principal Axes) 이라 명명한다. 이 좌표계 회전 스핀(Spin)을 찾기 위해 발동하는 절대 방정식, 고유치 문제(Eigenvalue Problem)의 수학적 환상망을 작동시킨다.

1. 각속도(\vec{\omega})와 각운동량(\vec{H})의 동조 정렬 타겟: 대각화 전개

우리가 원하는 우주 최고의 이상적인 회전 상태는, 자동차 타이어처럼 내가 돌리려는 축(\vec{\omega}) 방향과 그 쇳덩어리가 뿜어내는 거시적 운동량 에너지축(\vec{H}) 화살표가 비틀어지지 않고 완벽히 일자로 포개지는(동일선상) 상황이다.
이 수식적 동기화는 \vec{H} = I \vec{\omega} 방정식에서 행렬 텐서 \mathbf{I} 를 스칼라 상수 고유치 \lambda 로 강제로 짓눌러버리는 형태(\mathbf{I}\vec{\omega} = \lambda \vec{\omega} \rightarrow (\mathbf{I} - \lambda \mathbf{E})\vec{\omega} = 0)로 치환된다.

  • 이때 의미 있는 회전이 발생하려면 행렬식(Determinant)이 파괴되어야 하므로 극강의 다항 방정식 \det(\mathbf{I} - \lambda \mathbf{E}) = 0 이 터져 나온다.
  • 이 방정식을 풀면 나오는 3 개의 람다 스칼라값(\lambda_1, \lambda_2, \lambda_3)이 바로, 비대각 찌꺼기를 날려버린 물체 고유의 순수 저항력인 주 관성 모멘트(I_1, I_2, I_3) 다.
  • 물체는 비대칭일지언정, I_1 축을 중심으로 돌려버리면 베어링을 옆으로 구타하는 관성 곱 텐서 요동파(I_{xy}=I_{yz}=I_{zx}=0)가 완전히 0 (Zero)으로 소멸하는 평온 상태로 돌입한다.

2. 관성 주축(Principal Axes) 벡터의 직교성과 형태학적 렌더링

이제 도출된 I_1, I_2, I_3 를 원래 행렬식 빈칸에 하나씩 집어넣어, 대체 부품을 공간상에서 어느 각도로 비틀었을 때 그 마법의 축이 펼쳐지는지 고유 벡터(Eigenvector) 3 개(\vec{n}_1, \vec{n}_2, \vec{n}_3)를 도출하면 된다.

  • 놀랍게도 이 3 개의 고유 벡터 축은 반드시 서로 90^\circ 도 직각으로 교차(Orthogonal) 하는 완벽한 삼차원 데카르트(Cartesian) 주축망을 새로 렌더링한다(대칭 행렬 실수 고유치 정리).
  • 어떤 물체의 주축 중 하나(예: 휴대폰의 가장 긴 세로축 I_1)는 물체가 지닌 기하 형태 중 관성 저항이 가장 쥐똥만큼 작아 돌리기 쉬운 최소 축이 되고, 중간 축(I_2), 그리고 관성이 극한으로 커서 미친 듯이 돌리기 힘든 최대 축(I_3) 세 극점으로 완벽 무결하게 재정렬된다.

하지만 중간 크기의 관성을 지닌 좁은 면 직교축(I_2)으로 던져 올리면, 테니스 라켓은 비행 중간에 갑자기 미친 듯이 뒤척이며 축을 180^\circ 뒤바꿔버리는 그 유명한 드자니베코프 현상(Dzhanibekov Effect 혹은 텐니스 라켓 정리) 의 공중 파단(Instability) 무저갱에 빠져 허우적댄다.
그 중간 크기 I_2 고유치 축 자리는 미분학적으로 극단적 ‘안장점(Saddle Point 불안정)’ 이기 때문이다. 이처럼 단순한 곱셈 행렬 다항식 하나가 하늘을 나는 위성과 라켓의 3\text{D} 안정 궤적을 갈라치는 우주적 스핀 좌표계 판결문이다.
