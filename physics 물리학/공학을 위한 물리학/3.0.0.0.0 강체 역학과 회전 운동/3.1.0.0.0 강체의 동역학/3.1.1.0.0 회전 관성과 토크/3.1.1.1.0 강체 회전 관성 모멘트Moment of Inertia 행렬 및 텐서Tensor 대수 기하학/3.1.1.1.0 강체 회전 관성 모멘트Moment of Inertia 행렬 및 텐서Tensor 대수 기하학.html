3.1.1.1 강체 회전 관성 모멘트(Moment of Inertia) 행렬 및 텐서(Tensor) 대수 기하학

점 덩어리(Particle) 하나가 우주를 날아갈 때 유일하게 필요한 저항 관성은 그 스칼라 질량(m) 하나다. 그러나 크기와 부피를 지닌 강체(Rigid Body) 가 허공에서 빙글빙글 회전하기 시작하면, 질량이라는 방어구는 3\text{D} 공간상에 흩뿌려진 복잡한 기하학적 텐서망으로 찢겨 나간다.

축구공과 긴 막대기가 같은 무게(1\text{kg})를 가졌어도 축구공은 쉽게 팽이처럼 도는데 막대기는 돌리기 어려운 이유는, 똑같은 질량이라도 그 쇳물 입자들이 회전축(Axis of Rotation)에서 얼마나 멀리 떨어져 포진해 있느냐가 우주적 회전 관성(Resistance to angular acceleration)을 근본적으로 지배하기 때문이다. 우리는 이 회전을 거부하는 거대한 질량-거리의 분산 기하학을 스칼라 숫자가 아닌, 3 \times 3 매트릭스로 엮인 공간 방정식, 회전 관성 모멘트 텐서(Moment of Inertia Tensor) 로 해부한다. 이 장에서는 미적분을 이용한 고전적 관성 추출부터 텐서 대각화 행렬식 수치해석 세계로의 진입을 전개한다.
