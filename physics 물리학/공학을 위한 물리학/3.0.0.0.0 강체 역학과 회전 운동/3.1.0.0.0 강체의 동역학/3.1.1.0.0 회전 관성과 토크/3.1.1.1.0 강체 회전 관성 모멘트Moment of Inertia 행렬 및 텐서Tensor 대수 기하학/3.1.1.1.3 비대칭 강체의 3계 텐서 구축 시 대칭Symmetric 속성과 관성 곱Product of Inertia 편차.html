3.1.1.1.3 비대칭 강체의 3계 텐서 구축 시 대칭(Symmetric) 속성과 관성 곱(Product of Inertia) 편차

우리는 둥근 원판이나 육면체 같은 완벽히 거울 대칭인(축대칭, x 축으로 돌리나 -x 축으로 돌리나 똑같은) 기하 형상에만 익숙하다. 이런 아름다운 부품들은 x 축, y 축, z 축으로 굴릴 때 튀어나오는 회전 저항(I_{xx}, I_{yy}, I_{zz})만이 독립적으로 존재한다. 하지만 세상 기계의 절반 이상, 예컨대 비행기 프로펠러가 충돌로 한쪽이 부러져 나간 상태나, 삐딱하게 깎인 캠(Cam) 톱니바퀴 같은 ’비대칭(Asymmetric) 덩어리’를 회전시키려고 하면 우주 공간은 끔찍한 오일러 행렬의 늪으로 변모한다.

물체를 x 축으로 돌렸는데 물체가 기괴한 형태로 찌그러져 있어서, 갑자기 y 축과 z 축 방향으로도 요동치며 떨어져 나가려는 ’교차(Coupling) 반항력’이 파도처럼 밀려온다. 이 현상을 렌더링하기 위해 단 3개의 숫자만으로 묘사되던 관성을 3 \times 3 짜리 행렬(Matrix), 관성 텐서(Inertia Tensor) 로 진화시키고 그 저주스러운 비대각 편차인 관성 곱(Product of Inertia) 의 수학망을 해부한다.

1. 뼈대의 삐딱함, 관성 곱(I_{xy}, I_{yz}, I_{zx}) 교차 미분

관성 テン서 \mathbf{I} 는 대각선 방향으로 구석구석 박혀 있는 우리가 아는 주 관성 요소 I_{xx}=\int (y^2+z^2)dm 세 개를 가진다.
하지만 비대칭 강체가 진짜로 숨기고 있는 무기는 바로 대각선 밖의 영역, I_{xy}, I_{yz}, I_{zx} 로 엮인 비대각(Off-diagonal) 원소, 즉 관성 곱(Product of Inertia) 이다.

  • I_{xy} = \int xy \, dm
  • (주의: 기존 회전 관성과 달리, 부호가 양수(+)도 될 수 있고 음수(-)도 터져 나오며 심지어 대칭이면 0\text{(Zero)} 이 된다).
  • 이 I_{xy} 행렬 요소가 0 이 아니라 살아있다는 것은, 물체 형상의 질량이 1사분면(+x, +y)과 3사분면(-x, -y) 쪽으로 편향되어 삐딱하게 기형적으로 몰려 있음을 고발하는 적분 지표다.
  • 기하학의 기적으로, I_{xy} 와 I_{yx} 는 축의 순서를 바꿔봐도 적분식 속 x,y 의 순서만 바뀔 뿐 그 값은 완전히 동일한 I_{xy} = I_{yx} 상수 비율로 남는다. 고로 이 3 \times 3 텐서는 거울 보듯 정확히 맞물리는 대칭 행렬(Symmetric Matrix) 이 된다.

2. 각속도(\vec{\omega})와 각운동량(\vec{H})의 기괴한 파단 렌더링

고등학교 물리에서는 “물체가 회전하면 \vec{H} = I \vec{\omega} 에 의해 각운동량과 각속도 회전축이 같은 방향 화살표로 쭉 나란히($ \uparrow \uparrow $) 뻗는다“고 맹목적으로 외운다.
하지만 비대칭 강체(관성 곱 I_{xy} \ne 0) 앞에서는 이 공식은 휴짓조각이 된다. 쇳덩어리를 오직 x 축으로만(모터 축) 강제로 꽂아 \vec{\omega} = [\omega_x, 0, 0]^T 로 돌려보자.
각운동량 텐서 행렬곱 \vec{H} = \mathbf{I} \cdot \vec{\omega} 를 전개하면

H_x = I_{xx} \omega_x
H_y = I_{yx} \omega_x \ne 0
H_z = I_{zx} \omega_x \ne 0

라는 경악스러운 궤적이 튀어나온다. 명백히 모터는 x 방향 직진으로만 돌고 있는데, 물체 내부에 잠재된 기형적 살점 편차(I_{yx}, I_{zx}) 탓에 물체는 하늘(y)과 옆면(z) 쪽으로도 마구잡이로 비틀려 날아가려는 극단적 각운동량 에너지를 토해낸다!

이것이 모터 축을 박살 내는 주범, 바로 동적 불평형(Dynamic Unbalance) 이다. 각속도 벡터 화살표와 각운동량 벡터 화살표가 서로 평행하지 않고 비틀린 각도로 허공을 돈다. 베어링은 그 비틀어진 H 축을 억지로 수평으로 붙잡아 두어야 하니, 모터가 1바퀴 돌 때마다 베어링 껍데기는 지진이 난 것처럼 위아래로 후려쳐 맞게 된다. 대칭 행렬과 관성 곱 텐서는 강체 기구학이 겪는 파괴적 횡풍(Cross-coupling) 역학 수식의 첫 장이다.
