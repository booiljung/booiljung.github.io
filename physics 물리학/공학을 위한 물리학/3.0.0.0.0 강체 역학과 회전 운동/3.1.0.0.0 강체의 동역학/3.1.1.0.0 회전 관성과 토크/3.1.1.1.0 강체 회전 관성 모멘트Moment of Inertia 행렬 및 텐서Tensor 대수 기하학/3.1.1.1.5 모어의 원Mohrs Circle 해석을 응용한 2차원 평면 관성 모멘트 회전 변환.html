3.1.1.1.5 모어의 원(Mohr’s Circle) 해석을 응용한 2차원 평면 관성 모멘트 회전 변환

건축용 H-빔 철골 하나를 바닥에 내려놓았다고 하자. 보통 엔지니어는 기둥을 수직(y 축)이나 수평(x 축)이라는 네모 반듯한 도면으로만 힘을 계산한다(I_x, I_y, 관성 곱 I_{xy}). 하지만 강풍이 불어서 이 기둥 전체가 지면에서 서서히 15^\circ, 30^\circ (\theta) 비틀어진 각도로 파동을 맞아 눕게 된다면, 원래 우리가 알고 있던 I_x, I_y 텐서 값들은 새로운 축(x', y') 위에서는 도대체 얼마의 스칼라 수치로 변이될까?

삼각함수 미분과 \cos(2\theta), \sin(2\theta) 의 그 복잡하고 짜증 나는 변환 역학 다항식을 통째로 집어 던지고, 공학자 크리스티안 모어(Christian Mohr)가 창안해 낸 위대한 기하학적 나침반, ‘모어의 원(Mohr’s Circle)’ 을 통해 회전 텐서를 투시한다. 응력(Stress)뿐만 아니라 면적/질량 관성(Inertia)에도 이 완벽한 원의 대수 대칭 방정식이 소름 돋게 똑같이 강림한다.

1. 좌표 공간의 2\theta 투영 변환과 원 궤적의 렌더링

기존의 x, y 직교 좌표계를 지닌 부품을 공간에서 \theta 각도만큼 부드럽게 옆으로 틀어(x', y') 기울였다면, 새로운 축의 관성 모멘트 I_{x'} 값은 아래의 비선형 텐서 공식 곡선을 탄다.

I_{x'} = \frac{I_x + I_y}{2} + \frac{I_x - I_y}{2} \cos(2\theta) - I_{xy} \sin(2\theta)

이 수식의 본질을 가만히 뚫어지게 보라. \sin^2 + \cos^2 = 1 의 피타고라스 등가식을 강제로 씌우면, 이 방정식은 (x-a)^2 + y^2 = R^2 이라는 가장 순결하고 아름다운 2차원 원(Circle)의 매개변수 방정식 치환 뼈대로 완벽하게 포개어 돌변한다.

  • 원의 중심점 좌표 (C): \frac{I_x + I_y}{2} (물체를 어느 각도로 돌리든 영구적으로 변하지 않는 두 축의 불변 중심 평균 스칼라)
  • 원의 반지름 (R): \sqrt{\left(\frac{I_x - I_y}{2}\right)^2 + I_{xy}^2} (이 원둘레의 크기를 결정하는 극값 진폭)

2. 모어의 원 기하학을 통한 극값 주축(Principal Axis) 정렬 직관

좌표축과 삼각함수 테이블표는 덮어도 좋다. 그냥 공책에 X 축에는 스칼라 텐서 I_x 나 I_y 를, Y 축에는 교차하는 더러운 편차 텐서인 관성 곱 I_{xy} 를 찍어놓고 중심 C 에서 반지름 R 을 돌려 커다란 원 하나를 컴퍼스로 그려 버려라. 그러면 이 뼈대가 뒤틀릴 때 맞이할 운명이 한 장의 지도처럼 투시되어 렌더링된다.

  • 주 관성 모멘트 극점(Principal Moments of Inertia): 원이 가로축(X축) 한가운데를 끊고(Intersect) 통과해 지나가는 양극단 두 점! 그 자리가 파이프 단면이 세상에서 가장 빳빳하게 굽힘을 버티는 최고 강성(I_{max} = C + R)과 가장 낭창하게 꺾여버리는 최저 강성(I_{min} = C - R) 극지대다.
  • 주축의 회전각도 텐서(\theta_p): 이때 내 손에 들린 H-빔 철골을 도대체 몇 도(\theta) 삐딱하게 더 틀어 제껴야 저 궁극의 주축(관성 곱 I_{xy} 가 0 이 되어 사라지는 무결의 X 축 평면선)에 닿을까?
