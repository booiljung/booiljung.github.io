3.1.1.1.2 평행축 정리(Parallel-Axis) 및 직교축 정리(Perpendicular-Axis)를 융합한 복합 형상 관성 변환

엔지니어가 우주선 모터를 설계할 때마다 부품의 모양이 바뀔 때마다 그 괴랄하고 처참한 \int r^2 dm 3차원 부피 미적분 노가다를 매번 새로 풀어야 한다면 인류는 아직도 증기 기관에 머물러 있을 것이다.
실제 기계 도면에서 파이프, 원판, 육면체처럼 정형화된 모델(Primitive Element)들의 질량 중심(Center of Gravity, CG) 고유 관성값, \bar{I} 들은 이미 공학용 수학 테이블표에 완벽하게 래퍼런스로 찍혀있다.

진짜 설계 최적화 지옥은, 로봇 팔 관절 다발처럼 중심이 한참 벗어난 편심 질량 어셈블리가 회전축에 거대하게 매달릴 때 시작된다.
중심축(\text{CG})을 통과하지 않고 저기 옆으로 3\text{m} 빗겨나서 빙글빙글 공전(Revolution) 궤도를 도는 쇳덩어리들. 우리는 이 부품들의 ’자전’과 ‘공전’ 저항을 더티한 미적분 없이 기괴하게 압축해 내는 평행축 정리(Parallel-Axis Theorem) 와 직교축 정리(Perpendicular-Axis Theorem) 텐서로 단번에 묶어버린다.

1. 중심으로부터 찢겨 나간 공전 오프셋 텐서의 파멸적 상승: 평행축 정리 (Parallel-Axis Theorem)

어떤 부품 단품이 자기 자신의 무게 중심(\text{CG}) 축(z')을 중심으로 팽이처럼 자전할 때 가지는 최소 회전 저항 관성 텐서를 \bar{I}_{z'} 이라고 하자.
이제 이 부품을 다른 거대한 터빈 뼈대 중심축(z)에 매달고 조립 반경(거리 d)만큼 멀리 외곽으로 밀어내 묶어버리면(오프셋, Offset), 이 부품 전체를 통째로 굴려 돌릴 때 느끼는 새로운 회전 관성 I_z 는 아래의 단 한 줄의 대수적 기적(Theorem of Steiner)으로 완성된다.

I_z = \bar{I}_{z'} + m d^2

이 공식을 보면 기계 공학자가 왜 항상 가공 단가와 불평형(Unbalance)의 늪 사이에서 피눈물을 짜내는지 알게 된다.

  • 원래 부품이 자전할 때의 1 차원 관성(\bar{I}_{z'})은 얼마 안 된다.
  • 그러나 거대한 시스템으로 편심 이동(평행축 이동, d \ne 0)되는 순간, 뒤에 붙은 m d^2 텐서(점 질량이 중심에서 멀어진 공전 파단 관성) 가 괴물처럼 피어오르며 전체 마스터 관성 I_z 를 기하급수적으로 폭발적으로 잡아 늘린다. 거리가 2배 멀어지면 돌리는 데 필요한 토크 저항은 무려 4배 상승한다(d^2 지배). 시스템 조립 역학에서 각 부품의 위치 레이아웃(Placement Layout) 자체가 회전 파워를 수천 배 절약하거나 허공으로 날려버릴 수 있는 공학 텐서의 정점이다.

2. 평면 얇은 판을 지배하는 수직 반발축 모멘트: 직교축 정리 (Perpendicular-Axis Theorem)

만약 쇳물 덩어리 구조물이 엔진 팬(Fan) 블레이드나 거대한 피자판처럼 매우 얇고 납작한 2차원 두께 판상 구조(z 축 단면 두께가 무시할 만함, 면 질량 dm = \sigma dA)라면, x 축으로 기계판을 굴릴지, y 축으로 판을 넘길지에 대해 각각의 관성 I_x, I_y 를 따로 다 구했더라도 z 축 관성은 굳이 적분을 새로 깔 필요가 없다.

I_z = I_x + I_y \quad (\text{단, 얇은 $x-y$ 평면 객체 전용})

  • 즉 두께가 얇은 막대기나 판(Sheet)을 위에서 펜으로 콕 찌른 뒤 돌려 팽이로 쓰려고 할 때의 저항(I_z)은, 그냥 판을 옆으로 뒤집을 때(I_x)와 앞뒤로 넘길 때(I_y)의 저항 관성 스칼라를 두 개 그냥 숫자 더하기(Sum) 하면 끝이라는 것이다.
