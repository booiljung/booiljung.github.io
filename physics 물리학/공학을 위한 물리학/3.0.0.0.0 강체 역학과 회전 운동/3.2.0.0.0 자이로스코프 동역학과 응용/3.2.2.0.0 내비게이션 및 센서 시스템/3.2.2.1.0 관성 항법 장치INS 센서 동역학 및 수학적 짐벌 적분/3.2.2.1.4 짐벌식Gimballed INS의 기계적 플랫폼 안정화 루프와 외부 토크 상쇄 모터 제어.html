3.2.2.1.4 짐벌식(Gimballed) INS의 기계적 플랫폼 안정화 루프와 외부 토크 상쇄 모터 제어

스트랩다운(Strapdown) 방식이 나오기 전 아폴로 우주선과 초기 대륙간 탄도 미사일(\text{ICBM}), 잠수함에 쓰인 항법 장치의 근본은, 물리적인 쇠 테두리 반지를 이중 삼중으로 감싸 센서를 보호하는 짐벌 플랫폼(\text{Gimballed Platform}) 방식을 사용했다.
비행기가 360^\circ 로 미친 듯이 뒤집히고 흔들려도, 이 짐벌 속에 있는 ‘핵심 센서 블록(이너 짐벌 플랫폼, Platform)’ 은 태초에 세팅된 우주의 북쪽이나 지구 중심 중력 방향을 소름 돋게 똑바로 쳐다본 채 절대 0.1 도(degree) 도 흔들리지 않고(Stabilized) 고요하게 안착해 있어야 한다. 이 괴물 같은 역학적 단열(Isolation) 방어막을 구축하기 위한 하드웨어 모터의 오일러 피드백 무력화 루프(\text{Stabilization Loop})를 해부한다.

1. 흔들림 감지와 토크 역분사(Counter Torque) 제어막

짐벌식 \text{INS} 는 기어와 베어링으로 맞물린 3개의 둥근 반지(\text{Roll, Pitch, Yaw} 짐벌 링) 안에 제일 중요한 ‘가속도계(Accelerometer)’ 판때기를 띄워놓고 있다.
비행기가 바람을 처맞고 몸뚱어리가 확 흔들렸다고 치자. 베어링 마찰 때문에 이 이너 플랫폼 판때기도 비행기를 따라 야금야금 한쪽으로 기우뚱(Tilt) 딸려 돌아가려 할 것이다.
이때, 판때기 바로 옆에 박혀 있던 ‘제어용 자이로(\text{Gyroscopes})’ 3개가 이 미세한 각속도 찌그러짐(\text{Error } \Delta\theta) 파동을 빛의 속도로 측정해 낸다!

  • 서보 모터(Servo Motor)의 역습: 자이로가 “야! 판이 우측으로 0.001도 딸려 돌아갔어!” 라고 외치면, 바깥쪽 짐벌 관절(Joint)에 붙어있던 무지막지한 힘의 토크 모터(Torque Motor)가 컴퓨터 전기를 받아 즉시 발작한다.
  • \tau_{\text{motor}} = -K_p \cdot \Delta\theta - K_d \cdot \dot{\theta} (액티브 피드백 제어식): 모터가 기체 흔들림과 완벽히 정확하게 똑같은 힘(1:1) 으로 역방향 토크(\text{Counter Torque})를 우아앙 뿜어내어 플랫폼이 딸려가려는 걸 강제로 쥐어 뜯어 제자리 각도(Null Position)로 원상복구 패대기쳐버린다.

2. 짐벌식의 지상 절대성: 3차원 투영 적분 행렬 면제

플랫폼이 이렇게 외부 몸살 진동과 완벽히 격리(Isolation)되어 “1년 내내 언제나 위쪽은 절대 하늘(Zenith), 앞쪽은 트루 노스(True North 북극성)“를 가리키고 있으면 어떤 축복이 일어나는가?

당장 이전 장에서 피를 토하며 계산했던 무지막지한 시간 미분 방향 코사인 행렬(DCM, C_b^n) 연산이 필요가 아예 없어져 버린다(\mathbf{C_b^n = E} 단위 행렬, 식 증발)!
가속도계가 우주/지구 좌표축과 완벽히 일치된 짐벌 방 안에 고정돼 있으므로, 그냥 가속도 선이 찍어내는 숫자를 곧이곧대로 지구 중력만 빼주고(-g), 속도(V=\int A dt)와 거리(X=\int V dt)로 무지성 단순 1차원 적분만 빵빵 쳐나가도 비행기 위치가 모니터에 똑바로 찍힌다. 컴퓨터 칩셋의 성능이 쓰레기였던 1960년대 달 착륙 항법을 유일하게 해낸 역학적 하드웨어 짱박기 전술의 위엄이다.

\mathbf{Gimballed~INS} 는 비행기의 역학(Body Dynamics)과 항법 센서 공간 역학(Platform Dynamics)을 완전히 분리(\text{Decoupling})시키는 신의 방벽이었다.
다만 쇳물과 기어로 깎아 만든 이상 덩어리가 어마어마하게 크고, 움직일수록 생겨나는 베어링 마모(질량 비대칭 드리프트), 그리고 우주 미아 \text{Gimbal~Lock} 위험 때문에 21세기 반도체 마이크로 센서 시대(\text{Strapdown})에 자리를 내주고 거대 핵잠수함이나 \text{ICBM} 한가운데서만 그 무쇠의 영속성을 뿜어내고 있다. 하지만, 컴퓨터 칩셋 따위가 없던 시절 “오직 역학 제어 장치(모터) 하나로 우주 좌표를 쥐어짠” 이 하드웨어 솔루션은 고전 제어 피드백(Feedback) \text{Null} 보존 역학의 승리탑이다.
