3.2.2.1 관성 항법 장치(INS) 센서 동역학 및 수학적 짐벌 적분

관성 항법 장치(Inertial Navigation System, INS)는 지구자 자기장이나 외부 전리층 반사파의 도움 없이 오직 내부에 품고 있는 자이로스코프(Gyroscope)와 가속도계(Accelerometer)가 뱉어내는 순수 역학 미분값만을 모조리 쓸어모아, 시간에 대해 연속적으로 적분(Integration)하여 비행체의 절대 위치(X, Y, Z)와 고각(자세)을 그려내는 폐쇄 자급적 항법 계산의 결정체다.
수십 센티미터의 거대 기계식 짐벌 플랫폼에서부터 눈에 보이지 않는 실리콘 칩 속의 초정밀 진동 덩어리에 이르기까지, 물리적 관성을 전압과 광학으로 변환시키는 마법과도 같은 센서 기술과, 이 센서들의 치명적인 오차 누적을 방어하기 위한 수학적 적분 동역학망 모델들을 하나씩 정복한다.
