3.2.2.1.2 광섬유 자이로스코프(FOG)와 사냑 효과(Sagnac Effect)를 이용한 초정밀 위상차 계측

오일러의 우주는 단단한 쇳덩어리(강체)와 팽이가 주도했지만, 아인슈타인의 특수 상대성 이론을 접목한 현대 공학자들은 전혀 다른 무기인 빛(광자, Light) 을 돌리기 시작했다. 물리적인 베어링의 마찰과 마모, 짐벌 락(Lock) 따위의 재앙을 겪지 않기 위해 원통형 유리 섬유 타래(디지털 빔 코일, 광섬유 5\text{km} 둘레) 안에 두 가닥의 레이저 광선(Laser Beam)을 양방향 엇갈리게 둥글게 쏘아 보내 우주를 읽어내는 정밀도 10^{-6} \, \text{deg/hr} 초극각의 항해자, 광섬유 자이로스코프(Fiber Optic Gyroscope, FOG) 와 링 레이저 자이로(Ring Laser Gyro, RLG) 다. 이 시스템 속 우주 광속 상수(c)가 회전 센서로 거듭나는 양자 간섭의 근본, 사냑 효과(Sagnac Effect) 구조를 들여다본다.

1. 사냑 효과(Sagnac Effect): 양방향 빛의 도착 시간 편차(Time Difference)

동그랗게 말려있는 광섬유(\text{Fiber Coil}, 반경 R, 면적 A) 한가운데서 레이저 광원을 쪼개어 빛 A는 시계 방향(\text{CW})으로, 빛 B는 반시계 방향(\text{CCW})으로 똑같이 다이빙시켜 한 바퀴를 돌고 출발지점에서 다시 만나도록 했다 쳐보자.
만약 이 코일(비행기)이 멈춰 있다면, 빛 A 와 빛 B 는 똑같은 광속(c)으로 움직이며 정확히 둘 다 도착 라인을 동시에 밟고 들어와 결승 테이프를 끊는다.

  • 허나 전투기가 시계방향(\text{CW})각속도 \Omega 로 휙 돌아버린다면 어떻게 될까?
  • 빛 A (시계 방향 빔)는 달리는 기차 파도에 역풍을 맞은 듯 결승 테이프가 빛을 피해 윙 도망가 버리니 도착 거리가 \Delta L 만큼 길어져 빛이 코일을 빠져나오는 데 애를 먹고(=도착이 \textbf{늦어짐}).
  • 반대로 빛 B (반시계 방향 빔) 입장에선 결승 테이프가 빛을 마중 나오듯 확 다가오니 엄청나게 도착 거리가 짧아져 금방 뛰어 들어온다(=도착 \textbf{빨라짐}).

\Delta t = \frac{4A}{c^2} \Omega
\Delta \Phi (\text{위상차 Phase Shift}) = \frac{8\pi A N}{\lambda c} \Omega

이 사냑 타임 워프 오차 공식에서, 광속 \mathbf{c^2} 가 분모에 박혀 빛의 속도로 스케일(10^{16})을 찍어 누르는 바람에 그 위상차(\Delta \Phi)가 먼지보다 작지만, 코일을 파마머리마냥 수천 번(N 가닥) 칭칭 감아서 면적 합(NA)을 뻥튀기하면 기계는 이 미세한 레이저 파장들의 “엇박자 물결 간섭 무늬(Interference Fringe Area)” 를 간섭계로 찢어 읽어내어 비행기의 정확한 \text{\Omega (각속도)} 값을 도출해 낸다!

2. 정적 마찰(Stiction)이 없는 무결점(Solid-State) 신뢰성과 록인(Lock-in) 공황

FOG 와 RLG 에는 돌아가는 모터(\vec{\omega}_{spin}) 쇠구슬 팽이나 짐벌이 지구상에 아예 없다(이를 관성 항법 장치계에서 \text{Strapdown (스트랩다운)} 방식이라 칭함). 오직 멈춰있는 유리 렌즈와 센서가 비행기 몸체에 나사로 꽉 박힌 채, 레이저 빔만 거울을 때리며 날아다닌다.
따라서 베어링 오일 마찰 따위의 드리프트 오차(\tau_f \rightarrow 0)가 발동할 수가 없어 기존 기계 팽이와는 차원이 다른 이론적 무오류의 정점으로 불린다.

  • 록인 효과(Lock-in Effect, 광학적 정지 마찰): 그러나 링 레이저 자이로(RLG)의 저주가 있다. 비행기가 \Omega = 0.0001^\circ/\text{hr} 처럼 각속도를 아주 미세하게 틀며 움직일 때, 빛 입자 파동 두 가닥이 레이저 반사 거울 면에서 스캐터링(Scattering 빛 산란) 되어 서로에게 기생충처럼 간섭 동기화(Frequency Pulling)하는 발작을 일으켜버린다!
  • 두 빛의 주파수가 서로 엉켜 붙어 찰칵!(Locking) 하나로 통일돼버리면서, 사냑 위상차 \Delta \Phi = 0 를 찍고 자기가 돌아가는 각속도(\Omega) 정보를 통째로 사멸시켜 센서가 먹통장님이 되어버린다(“어? 비행기 멈춰 있는 건가? 나 각속도 0인데?”).
  • 이 빛의 기계적 오작동 파장을 없애려고 엔지니어들은 기계적 부르르 떠는 모터 진동 모터(Dither Motor)를 링에 달아 코일을 하루 종일 진동시켜 이 록인 지대를 빛이 정신 못 차리게 흔들어 강제로 돌파해 빠져나오는 미치광이 액티브 통제법을 발랐다.
