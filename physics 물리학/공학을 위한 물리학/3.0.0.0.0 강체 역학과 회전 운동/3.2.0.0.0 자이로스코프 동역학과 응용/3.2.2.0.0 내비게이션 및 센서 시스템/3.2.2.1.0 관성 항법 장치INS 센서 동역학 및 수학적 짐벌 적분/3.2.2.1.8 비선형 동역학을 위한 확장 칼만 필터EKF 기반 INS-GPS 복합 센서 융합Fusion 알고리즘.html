3.2.2.1.8 비선형 동역학을 위한 확장 칼만 필터(EKF) 기반 INS-GPS 복합 센서 융합(Fusion) 알고리즘

지금까지 증명했듯, 기계든 광섬유든 세상 어떠한 궁극의 관성 센서(INS)도 결국 ’적분의 저주(t^2 누적 오차)’를 피해 갈 순 없다. 즉 INS 는 수십 초 동안은 이 세상에서 가장 민감하고 매끄럽게 우주선의 진동을 따라가는 완벽한 단거리 눈알(High Frequency)이지만, 눈을 감고 미분값만 더하다 보니 1 시간이 지나면 모니터 상의 내 비행기는 안드로메다로 날아가 있다(Low Frequency 드리프트).
반대로 \text{GPS}(위성 항법 장치) 는 우주 밖에 박힌 위성에서 전파로 삼각측량을 때려버리니 10년이 지나도 오차가 10미터 밖을 안 벗어나는 완벽한 장기 절대 위치(Absolute) 눈알이지만, 전파가 1초에 한 번(1Hz) 뚝뚝 끊겨서 들어오므로 빠른 텀블링이나 진동을 잡을 수가 없다.
공학자들은 이 극단적으로 상반된 성질을 가진 두 눈알, 곧 눈이 빠르게 도는 불량 근시(INS) 와 느리고 깜빡거리지만 원시인(GPS) 을 수술 코드로 신경을 꿰매어 완벽한 신의 눈(2000Hz 초정밀/무오차 궤적)을 창조했다. 이것이 센서 융합(Sensor Fusion), 즉 확장 칼만 필터(Extended Kalman Filter, EKF) 수학의 맹렬한 역학이다.

1. 융합의 핵심: 비선형 상태 공간(State Space) 모델과 테일러 직진화

칼만 필터는 엄청난 숫자의 배열(행렬)로 굴러간다. 비행기의 오차 상태 변수(Error State Vector, \delta x)를 15개짜리 덩어리로 만든다(3개 위치 오차, 3개 속도 오차, 3개 자세 오차, 자이로 바이어스 3개, 가속도계 바이어스 3개).

  • 문제는 자이로 역학과 방향 코사인 변환 이 몽땅 다 삼각함수 파동과 제곱 승수로 꼬여있는 고도 비선형(Nonlinear) 식 이란 것이다. 기본 칼만 필터(Linear KF) 는 선형 직선 예측밖에 못해서 저 삼각함수 폭발을 따라가다 수식이 뻗어버린다.
  • 여기서 자코비안(Jacobian) 편미분 행렬을 우겨 넣어, 비선형 곡선을 현재 점 위치마다 잘게 썰어 직진(Taylor Series 1차 선형화 근사) 시키는 수술법이 바로 확장 칼만 필터(EKF) 의 뼈대다. 비행기가 복잡하게 비틀릴 때마다 컴퓨터가 그 곡선을 잘게 썰어서 매초 선형(Linear_{slice}) 상태 공간 모델 행렬을 새롭게 구워내어 칼만 엔진 속으로 끊임없이 연료를 밀어 넣는다.

2. 예측(Predict)과 수정(Update): 두 종교 간의 타협점 행렬 K

  1. 시간 예측 (Time Update, INS 독주): \text{GPS} 신호가 안 들어오고 끊긴 그 1초 길디 긴 시간 동안, INS 가 매초 1000 번씩 미분 적분 계산을 독점해 궤적을 예쁘게 그린다. 그동안 오차 덩어리 폭풍 행렬(공분산 P)은 t^2 법칙에 따라 뱃속에서 스멀스멀 미친 듯이 풍선처럼 부풀어 오른다(나 오차 이빠이 생겼어, 불안해!).
  2. 측정 조율 (Measurement Update, GPS 린치): 딱 1초 뒤, 지붕에서 극적으로 외부 기준점 \text{GPS} 의 진실의 문이 부왕- 열리며 좌표 방망이를 때리고 들어온다.
    이때 오일러 수학은 두 녀석의 의견을 맹목적으로 믿지 않고 저울(칼만 게인 K, Kalman Gain)에 올린다. “너 INS, 아까까지 1 초 동안 적분 에러 폭증률 공분산 얼마였어? 너 \text{GPS}, 오늘 대기 전리층 안 좋아서 전파 잡음(Noise R) 심하다며 얼만데?”
    가장 불안정이 적은, 즉 분산(Variance)이 더 좁은 쪽으로 K 매트릭스 수식을 치우치게 곱셈 가중치를 먹인다!
  3. 칼만 이득 교정: 만약 \text{GPS} 잡음이 적으면 맹렬하게 \text{GPS} 가 제시한 위치 쪽으로 INS 가 적분 쳐놓은 가상 위치를 홱 당겨서(Correction) 좌표를 깎아 치유해 버리고, 동시에 INS 센서 속에 묻어있던 공장 결함 바이어스 에러까지 역추적(Estimation)해 쪼아내 고쳐버린다.

수억 불짜리 \text{F-35} 전투기에 들어간 항법 알고리즘은 오일러의 \text{Coupling} 방정식을 그냥 푸는 게 아니다. 매 초 단위로 뻗어나가는 기계 센서 미분 지옥의 곡선(불안과 오차발산의 팽창 텐서 \text{Covariance $P$})을, 저 우주 밖 \text{GPS} 신호라는 확률적 망치(\text{Matrix } K)로 쉼 없이 두드려 깨부수어(Update) 다시 원점 \text{Zero} 바이어스로 리셋 시키는 영원불멸의 “가중치 타협 피드백 루프” 다.
순수 역학(Dynamics)으로만은 이룰 수 없는 항해 궤도의 신뢰도를 통계 오차 수학(Stochastic Estimator)으로 강제 병목 체결해 버린 공학 역사의 최고 성배가 바로 \text{EKF Sensor Fusion} 방정식이다.
