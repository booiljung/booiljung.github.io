3.2.2.1.5 스트랩다운(Strapdown) INS 자이로/가속도계 수치 적분(Integration) 및 오차 누적률(Drift) 물리

거대하고 기름이 새는 무쇠 짐벌(Gimbal) 모터를 떼다 버리고, 조그마한 센서 칩들을 우주선 바닥(Body) 쇳덩어리에 “단단히 나사로 박아 묶어버린(Strap-down)” 최신 전자식 항법 장치가 스트랩다운 관성 항법 시스템(\text{SDINS})이다.
짐벌이 없으니 비행기가 몸을 돌리면 센서도 똑같이 우당탕 같이 뒤집힌다! 기계적 역학 격리(Isolation) 혜택이 사라진 대신, 마이크로프로세서 컴퓨터가 초당 만 번씩 3차원 행렬(DCM, 쿼터니언) 미분방정식을 피똥 싸게 풀어내 우주 좌표를 “전자 두뇌 보드 속에서 가상으로” 렌더링 세팅하는 컴퓨팅 하드 코어 적분(Integration)망이다. 하지만 이 수학 연산의 밑단, 즉 적분이라는 속성은 아주 티끌만 한 오차 하나를 우주적 괴물로 뻥튀기하는 무서운 누적 독(Drift)을 품고 있다.

1. 두 번의 미분-적분 층계: 각속도(\omega) \rightarrow 가속도(a) \rightarrow 거리 위치(X)

스트랩다운 방식의 항로 계산 칩(Navigation Computer) 알고리즘은 센서 데이터를 먹고 2계층의 폭포수 적분으로 몸을 던진다.

  1. 1단계 적분: 자세(Attitude)의 각성:
    우선 \text{Gyro} 센서가 비행기가 얼마나 틀어졌는지 각속도 바늘 값 \omega = [p, q, r] 을 계속 토해낸다. 컴퓨터는 이를 적분하여 오일러 각 변환(DCM 9개 숫자나 \text{Slerp Quaternion}) 행렬을 가공한다. (비행기 바디 좌표계 \rightarrow \text{Nav(지구 절대)} 좌표계 변환 행렬 완성).
  2. 2단계 폭포: 속도(v) 와 거리(x) 창조:
    그러면, 이번엔 가속도계(\text{Accelerometer})에서 읽어온 기체 액셀 파워 \vec{f}^b(비중력 가속도, 힘력) 에 방금 만든 저 요술 변환 행렬 \mathbf{C_b^n} 을 딱 곱해준다! \rightarrow \vec{f}^n = \mathbf{C_b^n} \cdot \vec{f}^b
    방향이 우주 북동쪽으로 예쁘게 정렬된 가속도 벡터 파편 \vec{f}^n 이 뜨면, 거기서 지구 중력 상수 g 를 퉁 쳐서 잡아 뽑아버린다.
    순수하게 엔진만의 알짜배기 가속도가 나왔으니 이걸 한 번 적분(\int)하면 기차의 현재 비행 속도(\vec{v}), 또 한 번 더 누적 시간 적분(\int\int)을 갈기면 기체가 북쪽으로 날아간 거리 절대 위치(\vec{P}) 가 모니터에 드디어 뜬다.

2. 미분 잡음 드리프트(Drift): “시간 곱하기 시간(t^2)” 의 사형 선고

이 아름다운 수학 수치 적분(수학 더하기 누적망)의 뱃속에는 암 덩어리가 박혀 있다. 초저가 스마트폰 자이로와 가속도계 부품 불량, 혹은 온도 상승으로 잡음(Noise) 에러가 조금씩 끼어들면 미분 기호 한가운데 오차 바이어스(Bias Error) \delta 가 묻어 들어간다.

  • 수학적 발작 폭발 이중 적분:
    가속도계 센서 자체의 영점 오차(0 점이 안 맞아서 늘 0.001 m/s^2 를 오버해서 측정 중인 공장 결함, \Delta a_{bias})를 컴퓨터가 죽어라고 적분하면 어떻게 될까.
    거리 에러 \Delta S = \int \int \Delta a_{bias} dt \, dt = \mathbf{\frac{1}{2} \Delta a_{bias} \, t^2}
    즉, 센서의 쥐똥만 한 \Delta a 오차가 시간에 대해서 제곱배(\mathbf{t^2}) 속도로 미친 듯이 늘어난다!
  • 복합 자이로 에러 투과:
    이게 끝이 아니다. 1단계에서 자이로스코프(\text{Gyro}) 센서 칩 자체의 오차 찌꺼기 편향(\Delta\omega_{bias})가 들어가면 기체 코 각도가 틀어지게 렌더링되는데(Angle Error \propto t), 틀어진 각도 탓에 가속도계가 우주 중력 g 를 비스듬하게 잘못 받아 처먹고 투영해 버린다(g \cdot \Delta\theta). 이 가짜 가속도 각이 다시 2번이나 적분망에 들어가 거리 오차로 변신하면, 자이로 오차는 거리에 대해 t^3 (시간의 3제곱, 큐빅 발산) 속도로 우주 미아 에러 지뢰폭발을 터트린다!

수식은 거짓말을 하지 않는다. 미적분의 적분 기호 \int 는 그 안에 들어온 가장 추악하고 미세한 노이즈(\Delta) 먼지조차도 시간에 관해 영원히 쓰레기통 속에 저장하고 증식시킨 채 부피를 불려나가는(Cummulative Memory) 지독한 괴물이다.
과거 짐벌식 장치 시절엔 쇠판 베어링 마모로 기계가 스스로 물리적으로 썩어갔다면, 반도체 \text{Strapdown} 시대엔 부품은 아무 이승의 때가 묻지 않건만 \text{CPU} 연산 소프트웨어 텐서 미분 자체가 숫자를 폭주시키며 가상으로 항법의 목을 조르는 ‘결정론적 역학 표류(Deterministic Drift Bias)’ 시대로 변모한 것이다. 이 끔찍한 시간 곱셈 발산망(t^2, t^3) 누적의 공기를 분쇄하는 것이 우주 제어 랩의 최대 숙제다.
