3.2.2.1.3 비행체 기준틀(Body frame)에서 내비게이션 좌표계(Nav frame)로의 실시간 방향 코사인 변환망

전투기가 하늘에서 북쪽으로 날아가는지, 서쪽으로 꼬꾸라지는지, 수백 킬로미터 밖의 정밀 타격 목표물로 똑바로 크루즈 순항하는지를 알려면, 조종석에 달린 센서의 숫자를 ’우리가 딛고 서 있는 진짜 지구 지도 좌표’로 미분 맵핑(Mapping)해야 한다.
이를 위해 공학자들은 3차원 우주에 2개의 절대 방(Frame) 을 만들었다.하나는 비행기 조종석에 꽉 붙어서 같이 돌아가는 비행체 좌표계(Body Frame, b-frame), 다른 하나는 지구 표면의 북쪽, 동쪽, 땅바닥(중력)을 가리키며 가만히 서 있는 내비게이션 좌표계(Navigation Frame, n-frame) 이다.
\text{INS(관성 항법 시스템)} 컴퓨터의 가장 고된 노가다는 초당 2,000번씩 비행기 바디 프레임의 센서값을 읽어, 이 내비게이션 프레임 쪽으로 변환시켜 지구 위에서의 ’절대 자세(Attitude) 지도값’을 그려내는 일이다. 이 다리를 잇는 수학의 괴물이 바로 방향 코사인 행렬(Direction Cosine Matrix, DCM, C_b^n) 이다.

1. 두 방(Frame)을 이어주는 3 \times 3 투영기: \mathbf{C_b^n} 의 본질

방향 코사인 행렬 C_b^n 은 기체 좌표계(b)에 찍힌 어떤 화살표 벡터(예: 기체가 받는 앞으로 가속도 A_x)를, 지도 좌표계(n) 의 북, 동, 하 방향으로 얼마나 쪼개져 투영되는지를 계산해 주는 3 \times 3 (9개의 숫자) 회전 행렬이다.

  • \vec{V}^n = \mathbf{C_b^n} \vec{V}^b (바디 벡터에 행렬을 곱하면, 내비 벡터로 환승 뿅!)
  • 이 9개의 숫자 덩어리 C_b^n 행렬 한가운데에는 우리가 예전에 그토록 뜯어보던 오일러 스크류 각도 \mathbf{\phi, \theta, \psi (\text{Roll, Pitch, Yaw})} 의 수많은 사인/코사인 곱셈(\cos(\theta)\sin(\psi) 등) 비선형 덩어리가 삼각함수로 지저분하게 우글거리고 있다.

2. 실시간 자세 행렬의 업데이트 미분 방정식 (DCM Kinematic Equation)

가장 무서운 점은 전투기가 빛의 속도로 텀블링하고 방향을 틀기 때문에, 저 \phi, \theta, \psi 각도가 매초 미친 듯이 변한다는 것이다. 즉 C_b^n 행렬 자체가 고정된 숫자가 아니라, 시간 t 에 대해 숨 쉬고 떨리며 진화하는 ‘살아있는 미분 행렬’ 이다!
자이로스코프 센서가 내뿜는 매초 각속도 바늘(\vec{\omega}_{ib}^b = [p, q, r]^T) 데이터 스트림을 뽑아, 컴퓨터는 행렬 C_b^n 이 어떻게 실시간으로 변질해 가는지 적분(Integration)해 차기 행렬 렌더링을 칠판에 업데이트한다.

\mathbf{\dot{C}_b^n} = \mathbf{C_b^n} \, \mathbf{\Omega_{ib}^b}

  • \mathbf{\Omega_{ib}^b} 는 자이로스코프의 p, q, r 각속도를 3 \times 3 비대칭 교차 행렬(Skew-Symmetric Matrix, 대각선은 0이고 양쪽에 마이너스 곱등이가 붙어있는 십자 외적 행렬)로 찌그려 틀어 만든 형태다.
  • 바로 이 아주 간단해 보이는 미분 방정식! 과거의 행렬 값 C_b^n 에, 방금 센서가 읽고 온 괴물 행렬파 \Omega 를 달고 와서 \Delta t 시간 동안 곱해 더해주는 짓을 항법 컴퓨터가 무한대로(무한 루프 Loop) 죽을 때까지 때려 박는다.

수많은 텍스트와 \mathbf{C_b^n} 의 행렬 증명에도 불구하고, 항해의 정수는 “강체가 어디로 뻗어 나갈 것인가“를 예측하는 좌표 치환 수학(Transformation Math)이다. 비루한 센서는 오직 자기 코앞(Body)의 물리량밖에 못 느끼지만, 이 직교 방향 코사인 행렬 미분방정식이 돌아가는 컴퓨터 CPU 보드 안에서는 그 작은 센서 값이 지구의 북쪽과 중력 선을 그리는 전지전능한 항로(Navigation Track)의 붓(\text{Brush})으로 맵핑(Mapping)되는 미적분 예술의 정점을 찍는다.
