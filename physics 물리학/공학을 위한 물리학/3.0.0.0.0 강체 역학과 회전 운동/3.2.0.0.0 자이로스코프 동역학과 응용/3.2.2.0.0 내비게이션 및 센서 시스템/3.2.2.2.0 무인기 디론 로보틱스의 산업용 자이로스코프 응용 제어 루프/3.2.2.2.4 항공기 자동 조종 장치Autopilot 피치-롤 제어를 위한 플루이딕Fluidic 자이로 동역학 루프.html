3.2.2.2.4 항공기 자동 조종 장치(Autopilot) 피치-롤 제어를 위한 플루이딕(Fluidic) 자이로 동역학 루프

전자 구리와 트랜지스터 컴퓨터 칩이 전투기에 발라지기 전, 1940년대의 2차 세계 대전 폭격기들이나 독일의 V2 로켓 안에는 놀랍게도 전기 배선이 끊어져도 아무 상관이 없이 영원히 작동하는 ‘공기 분사(Pneumatic)’ 자동 조종 장치(Autopilot)가 들어 있었다.
이것이 바로 전기의 힘을 0 만큼도 빌리지 않고 오직 압축 공기(Air) 가스가 뿜어져 나오는 유로 튜브(Tube) 들살과, 기압 차이만으로 비행기의 자세를 똑바로 꺾어버리는 플루이딕(Fluidic) 자이로스코프 제어 루프다. 기압차 역학이 어떻게 전자 \text{CPU} 와 코딩 소스 없이도 스스로 비행기의 날개(Elevator)를 비트는 물리 엔진 컴퓨터를 창조했는지 살펴본다.

1. 차압(Differential Pressure)의 창조자: 에어리얼 자이로(Air-driven Gyro)

플루이딕 오토파일럿의 중심에는 모터가 아니라, 진공 펌프(Vacuum Pump)가 빨아들이는 공기의 힘으로 분당 20,000\text{RPM} 으로 돌아가는 터빈 철팽이(Air Rotor) 가 들어 있다.

이 팽이는 조종석 밑바닥에 있는 짐벌 플랫폼 안에 놓여 있다.
비행기가 돌풍을 맞고 코를 들면(Pitch \, Up), 자이로 팽이는 자기 각도를 지키려고 가만히 버틴다! 그러면 팽이를 감싸고 있는 비행기 껍데기만 위로 위이잉 꺾여 올라가는데, 이때 플루이딕(기압) 차이가 발생한다.

  • 팽이를 감싼 껍데기에는 양옆으로 구멍 2개(A 구멍, B 구멍)가 뚫려 있어 공기를 똑같이 절반씩 빨아들이며 대칭의 기류(\Delta P = 0)를 만들고 있었다.
  • 허나 비행기 코가 들려 껍데기가 약간 위로 딸려 올라가면(\text{Tilt}), A 구멍 앞은 팽이의 쇳덩어리 벽면이 가로막아버려(Block) 막히고, B 구멍 앞은 확 시원하게 열려버린다!(오일러 상대 앵글 밸런스 붕괴)
  • 차압의 폭파: 순간 B 구멍으로만 공기(Suction)가 미친 듯이 미친 듯이 빨려 들어가며 기압차(\Delta Pressure) 의 대혼돈 тен서가 열린다!

2. 뉴매틱 서보(Pneumatic Servo) 다층 증폭과 꼬리날개 타격

센서에서 생긴 이 미세한 솔바람 차압(\Delta P)은 절대로 무거운 꼬리 날개를 들어 올릴 힘이 못 된다. 그래서 이 차압 신호는 릴레이 밸브(Air \, Relay)라 불리는 다단계 기압 증폭 튜브 관을 통과한다.
마치 트랜지스터 전자 증폭기(Amp) 회로와 똑같이, 약간의 기압 쏠림 신호가 밸브 피스톤(Spool)을 살짝 건드리면, 그 뒤에 있던 엔진 급 압축 공기가 “확” 밀려들어가 거대한 피스톤 실린더를 쾅! 밀어낸다(제어 압력 증폭기).

  • 실린더가 밀리면, 그 뒤에 연결된 강철 와이어(케이블)를 확 끄집어 당기고, 강철 와이어는 뒷날개 승강타(Elevator) 플랩을 스윽 꺾어 내린다(기압의 로터스 \text{Force} 치환).
  • 꼬리날개가 꺾이면 비행기 코가 다시 아래로 처박히면서 수평 자리에 복귀한다. 다시 비행기가 평형 각도를 찾으면, 아까 막혀있던 A 구멍이 스르륵 돌아와 열리며 양쪽 A 와 B 의 기압이 50:50 동일 기압으로 안정을 찾고, 모든 밸브 피스톤과 와이어가 우글거리는 제자리(Neutral) 동작을 영구 멈춤한다!
