3.2.2.2.3 이륜 자율 주행 로봇(Segway) 및 외발 자전거의 역진자(Inverted Pendulum) 자이로 스태빌라이징

막대기 위원반에 무거운 추를 달아 손바닥 위에 올려놓고 떨어지지 않게 중심을 잡는 놀이를 기억하는가? 공학에서 이 아슬아슬하게 넘어가려는 거꾸로 선 몽둥이를 역진자(Inverted Pendulum) 시스템이라고 부른다.
바퀴가 앞뒤로 두 개도 아니고 오직 양옆에 평행하게 딱 2개만 달린 이륜 바이크(세그웨이, Segway)나 서빙 로봇은, 가만히 놔두면 중력에 의해 100\% 앞이나 뒤로 콰당 엎어지는 태생적 파멸형 물리 구조를 안고 태어났다. 그런데 사람이 그 위에 올라타 몸을 앞으로 기우뚱 넘겼는데도 바이크는 엎어지지 않고, 마치 마술처럼 사람이 쓰러지려는 방향(앞)으로 바퀴를 “스르륵스르륵 먼저 잽싸게 굴려 나가며” 받침대 위에서 마이클 잭슨의 린(Lean) 동작처럼 중심을 세운다. 이 무한 전진/균형 복원 역학을 푸는 마법의 센서가 바로 자이로스코프(\text{Gyro})의 피치(Pitch) 추적기다.

1. 떨어지는 무게 중심을 밑동 바퀴가 발사되어 떠받친다

이륜 로봇 바디가 바퀴 위에 90^\circ 로 똑바로(정상 밸런스) 서 있을 때를 \theta = 0^\circ 라 하자.
사람이 발판 위에 서서 코를 앞으로 10^\circ 숙이면(\theta = +10^\circ) 발바닥 바로 위, 즉 로봇의 \text{자이로/가속도 센서 칩} 이 “나 지금 찌그러지는 속도 \dot{\theta} 가 엄청나! 넘어지고 있어!” 라며 \text{CPU} 로 경악의 에러 신호를 때린다.

그러면 하단 제어 보드의 방정식은 즉각적으로 넘어지는 쪽(앞쪽, \mathbf{+X}) 방향으로 바퀴 모터의 회전 속도를 미친 듯이 전진(\mathbf{Acceleration}) 가속시킨다.

  • 바퀴가 나보다 훨씬 더 빨리 앞으로 치고 나가면(바퀴 베이스가 축을 앞지름)? 내 몸뚱어리 상체는 관성(F=-ma) 때문에 엉덩이가 뒤쪽(\mathbf{-X} 곡선, 뒤) 으로 확 쏠리게 넘어간다!
  • 어라? 분명 체중을 앞으로 기울여서 기체가 앞으로 넘어지려 했는데, 컴퓨터가 바퀴를 팍 치고 나가게 하니 내 몸이 관성으로 뒤로 쏠려버려 다시 수직 0^\circ 자리로 빳빳하게 기적이 일어난 듯 복원된 것이다.

\mathbf{Motor \, Torque (\tau) = K \cdot \theta + K_d \cdot \dot{\theta} + K_v \cdot v_{\text{wheel}}}

결국 이것은 “사람이 앞으로 넘어지려는 힘” 과 “로봇이 바퀴를 굴려 하부축을 앞지름으로써 만드는 뒤로 쏠리는 관성 구배(\tau 복원력)” 가 무한대로 치열하게 싸우면서 0 의 밸런스(\text{Equilibrium})를 귀신같이 잡는 적분 피드백 동력학이다.

2. 상태 공간 제어(LQR) 와 칼만 필터의 최적 제어기

단순히 비례-미분(PID) 제어기만으로 사람을 싣고 넘치는 역진자를 세우기엔 그 넘어지는 가속도의 지수 함수적 발산(\mathbf{e^{kt}} 쌍곡선 폴호드) 곡률이 너무나 사악하다(\text{Non-minimum Phase} 시스템).
현대의 고급 휠봇(Wheel-bot)이나 호버보드는 수십만 번의 연립 행렬 계산을 때리는 LQR(Linear Quadratic Regulator) 최적 제어기와 칼만 필터에 제어권을 넘겼다. 이 제어기는 단지 현재 각도(\theta)만 집착하지 않고, 바퀴가 굴러간 전체 거리(x), 바퀴의 속도(v), 기둥이 넘어지는 각도(\theta), 넘어지는 각속도(\dot{\theta}) 무려 \text{4차원 상태 변수 Vector} 네 가지를 행렬로 동시에 엮어서(State Space Matrix),
“어떻게 모터에 전기 전압을 부어 넣어야 이 4가지 파멸의 숫자가 최단 시간(Energy Optimizer)에 깔끔하게 0으로 죽을까?” 를 미적분 방정식 해로 찍어내 매초 지령을 내린다.

세그웨이나 자율 주행 배달 로봇같이 무릎 위에 사람이나 수박을 이고 달리는 역진자 시스템은, 로버스트(Robust) 제어 없이는 인간의 불규칙한 허리 꿈틀거림 질량 센터(Center of Gravity)의 비선형 요동에 속아 추락하기 십상이다.
단순히 “넘어지려 하면 넘어지는 쪽으로 바퀴를 발사해서 밑바탕을 지탱해 준다“는 F=ma 의 관성 보정 철학은, 밀리 초(ms) 단위의 융합 센서 반응 속도가 기계 모터의 응답 시간을 압도할 때 기적처럼 인간을 태우는 가장 우아한 균형 자아(Self-balancing) 미적분 방정식으로 꽃핀다.
