3.2.2.2.2 쿼드콥터(Quadcopter) 드론의 호버링(Hovering) 안정을 위한 PID 기반 각가속도 상쇄 제어망

바람이 거세게 부는 들판에 쿼드콥터 프레임을 갖다 놓으면, 드론은 오직 4개의 날개 프로펠러가 일으키는 아래쪽 공기 양력(F_{thrust})으로만 공간에 자기를 박제시켜야 한다(Hovering).
그런데 우주와 달리, 지구에서는 한류나 돌풍 등 미친 듯한 잡음 덩어리(마이크로-버스트)가 드론의 옆구리를 후려친다. 프로펠러 엔진은 회전하는 모터인데 바깥에서 힘이 처들어 오니, 오일러 제어의 심장인 “자이로스코프 센서” 칩(\text{IMU})은 초당 수천 번 비명을 지른다. “바디가 1.5 도 수평(H)에서 뒤틀려 오른쪽으로 엎어지고 있어!” 이 뒤집힌 각도(Error \theta)를 다시 0 수평 도로 복원시키기 위한 컴퓨터 칩셋(FC, Flight Controller) 내의 제어 공식이 바로 전설적인 비례-적분-미분(PID) 피드백 폐루프 제어기(Closed-Loop Controller) 이다.

1. 드론 롤(Roll)/피치(Pitch) 추락의 Error 값과 PID 보상 모터 회전수

드론이 바람을 맞아 오른쪽(Roll \phi)으로 기우뚱 +2^\circ 돌아간 순간, 자이로스코프가 이 회전 각속도 \dot{\phi} 를 즉각 적분해 2^\circ 위치를 도출한다.
나의 목표 타겟(Set-point)은 가만히 매달려 있는 0^\circ 이다.
그럼 오차 \mathbf{Error(e) = 0 - (+2) = -2^\circ} 가 수학적으로 뜨게 된다.

이 에러 값을 드론 칩 보드에 달린 \text{PID} 수식에 때려넣는다.

\mathbf{Correction \, Output_u(t) = K_p \cdot e(t) + K_i \int e(t) dt + K_d \frac{de(t)}{dt}}

  • P 항(Proportional): “아 삐뚤어졌네? 쎄게 치자!”
    현재 +2^\circ 틀어진 만큼, 그 에러(-2)에 비례상수(K_p)를 곱해 오른쪽 모터 모터 스피드는 미친 듯이 올리고, 왼쪽 모터 스피드는 줄여버려서 오른쪽 프로펠러가 더 세게 땅을 차고(Thrust \uparrow) 올라오게 하여 기체를 왼쪽으로 후려치는 단순 무식한 힘이다. 허나 이것만 쓰면 원래 자리로 돌아오다 속도를 못 이겨서 반대쪽으로 넘어가는 흔들림(Oscillation) 폭주를 맞는다.
  • D 항(Derivative): “어어, 그만 너무 돌아간다 브레이크 밟아!”
    에러가 변하는 속도(기울기 도함수 \dot{e})를 보고 “오, 지금 제자리(0^\circ)로 되돌아가는 속도가 너무 빠른데?” 싶으면, D 게인(K_d)이 마이너스 토크를 걸어 모터 펀치력에 미리 브레이크 강제 점성을 거는 충격 완화제(Damper) 다. P 와 D 가 만나야 드론은 깔끔하게 흔들리지 않고 탁! 고정된다(PD 칼 군무).
  • I 항(Integral): “바람이 계속 한쪽으로만 부네? 영점 조절.”
    아무리 PD 가 기체를 잘 세워도, 바람이 계속 우측으로 불면 드론은 억지로 버티느라 항상 미세하게 +0.1^\circ 정도 옆으로 누워있게 잔여 오차(Steady-State Error)가 뜬다. I 보정(K_i)식은 시간이 지날수록 이 0.1 도 에러 먼지들을 계속 적분해 무거운 폭탄으로 누적시킨 뒤 모터에 때려 넣어, 결국 끝까지 그 바람벽을 이기고 기체를 완벽한 0.0000^\circ 정중앙에 수평 영점으로 꽂아버리는 집념의 수학 항이다.

수많은 공학 용어가 모터(Actuator) 텐서 출력을 수놓지만, 초심자들의 드론이 하늘에서 개구리 점프를 타듯 부들부들 떨며(Oscillating) 미친 듯 솟다가 흙바닥에 나뒹구는 이유는 오직 저 P 상수나 D 상수 튜닝(Tuning)이 강체 질량 I 에 비해 너무 악랄하게 세서 오버슈트(Overshoot)를 때렸기 때문이다.
기계는 자기 모멘트 관성에 맞는 K_p, K_d 황금 계수가 존재하며, 자이로스코프(IMU)가 던져주는 각도 편차 Error 를 “얼마나 세게(P), 얼마나 부드럽게 브레이크 잡고(D), 끝까지 뿌리 뽑을 것인가(I)” 의 철학이 시계톱니바퀴처럼 엮여야만 중력을 정복한 채 허공 화판에 점(1\text{m} 텐서 정지 고정 좌표)을 짓는다.
