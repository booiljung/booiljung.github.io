3.2.2.2.5 선박 및 대형 요트의 전복(Capsizing) 방지용 안티 롤(Anti-roll) 액티브 자이로 스태빌라이저

바다 위를 달리는 수백 톤짜리 럭셔리 대형 요트 나 크루즈선에 갑자기 5미터짜리 거대 너울성 파도가 배의 우측구리(Starboard)를 확 치고 올라오면, 배는 견디지 못하고 좌측으로 미친 듯이 \text{Roll} 하우스 되어 승객들에게 죽음의 멀미나 나뒹굼을 선사한다.
이를 막고자 물밑 핀(Fin) 플랩을 달기도 하나, 배가 가만히 정박해(Zero Speed) 닻을 내리고 흔들리는 최악의 멀미 공명장에서는 외부에 달린 물지느러미가 속도가 없어 아무런 유체 양력을 못 낸다. 오직 배 안쪽 엔진 룸 깊숙한 곳에서 자기 스스로 역학적 멀미 소멸 구체를 만들어 내는 자이로 스태빌라이저(Gyroscopic Stabilizer), 즉 공 포장지에 싸서 초고속으로 굴려대는 ‘안티 롤(Anti-Roll) 거대 무쇠 팽이’ 만이 배의 흔들림을 역으로 깨부수는 오일러 선박 동역학 타격의 결정체다.

1. 수직 스핀(Z 축) 팽이와 롤(X 축) 외란의 조우, 피치(Y 축) 모멘트의 외적 분노

선박 안치실에 무게 수 톤(ton)짜리 무지막지한 쇳덩어리 원반 자이로스코프를 달아놓는다. 그 원반은 모터 전기를 먹고 수직 Z축 방아깨비 방향으로 수천 \text{RPM} (10,000\text{RPM}) 이 불타며 거대 각운동량 벡터 \vec{H}_{spin} 기둥을 하늘로 세워놓고 맹렬히 돌고 있다.

  • 갑자기 외해 파도가 쳐서 배가 기우뚱 우측(X축, 롤)으로 넘어간다(각속도 \Omega_x). 배 바닥에 박힌 팽이의 하늘 수직 기둥 축마저도, 오른쪽으로 꺾인다.
  • 팽이는 자기를 오른쪽으로 꺾어버린 파도의 힘에 대해 격노의 짐벌 반작용 외적 오일러 타격을 친다. 수직(Z)으로 돌던 팽이가 롤(X)로 꺾였으니, 그 팽이를 감싸고 있는 브래킷(짐벌) 고정 틀에는 \mathbf{Y} 축(피치, 앞뒤 끄덕임) 방향으로 미친듯한 돌림힘(Precession Torque)이 꽝 터지며 팽이가 엎어지려 요동친다!

2. 모터 유압(Hydraulic) 피드백: “피치를 꺾으면, 롤이 회복된다” 의 진실의 문

하지만 배의 흔들림(X축 파도) 을 막으려는 건데 갑자기 팽이가 앞뒤(Y축 피치) 로 요동을 친다? 이걸 어떻게 써먹는가?
제어 공학자는 바로 이 피치(Y) 방향으로 요동치는 팽이의 짐벌 축에, 강력한 유압 실린더 브레이크 모터 (Active Control Cylinder) 를 달아뒀다!

  • 팽이가 파도 \text{Roll} 에 맞아 피치(Y) 축 앞뒤로 미치광이 세차 운동(Precession)을 시작하며 돌려고 할 때, 유압 실린더 피드백 제어 모터가 그 움직임을 무식하게 딱딱하게 잡고 역으로 더 꺾어버리거나 점성 댐핑(Damping) 압력을 가한다.
  • 오일러의 식 \vec{\tau} = \vec{\omega}_{precess} \times \vec{H} 는 완벽한 양방향 거울이다! 유압 실린더(Y축)로 팽이의 피치 고갯짓 \vec{\omega}_y 를 속도로 꽉 쥐어 짜내면, 팽이는 다시 이 힘을 외적으로 튕겨내 파도가 쳤던 \mathbf{X} 축(롤 방향)의 정반대 힘 으로 배를 밀어붙인다! (Anti-Roll Restoring Moment). 즉 팽이의 머리를 앞뒤로 흔들게 브레이크를 걸면 배가 좌우로 오뚝이처럼 튕겨 일어서 버린다!

\mathbf{Active~Gyro~Stabilizer} 는 단순한 무거운 짐 덩이가 아니다. 파도(\text{Wave Disturbance})가 바다를 스쳐 배를 기우뚱하게 하는 1차 역학(Roll)을 피치(Pitch)라는 위상 공간 축으로 빼돌린 다음 브레이크 압력을 먹여 상쇄 토크를 다시 반사(Reflection)시켜버리는 공간(1D \rightarrow 3D) 변조 마법진이다.
핀(Fin) 플랩이 부러지고 키가 날아가도, 배의 안방 깊숙한 밀폐 진공 탄소 구체 속에서 절대 멈추지 않고(1시간 공회전 속도 유지) 위상을 토해내는 이 자이로 역학은 바다의 신념(Oceanic Cybernetics)이다.
