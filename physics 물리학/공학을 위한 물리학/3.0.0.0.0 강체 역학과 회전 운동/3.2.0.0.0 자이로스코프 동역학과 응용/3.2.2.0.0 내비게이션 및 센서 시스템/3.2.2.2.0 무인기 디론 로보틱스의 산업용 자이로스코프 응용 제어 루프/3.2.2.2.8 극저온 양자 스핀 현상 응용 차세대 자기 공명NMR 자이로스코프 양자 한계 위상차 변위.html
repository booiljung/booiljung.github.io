3.2.2.2.8 극저온 양자 스핀 현상 응용 차세대 자기 공명(NMR) 자이로스코프 양자 한계 위상차 변위

오일러의 우주(강체 역학)를 거쳐, 아인슈타인의 사냑 간섭계(Ring Laser)마저 지나 공학의 항해는 드디어 우주 가장 밑단 미시 세계의 바닥인 양자(Quantum) 로 돌입했다.
눈에 보이지 않는 공기 팽이도, 가느다란 유리 광섬유도 아니다. 차세대 대륙간 탄도 공격과 잠수함의 마지막 무기는 원자핵(Nucleus) 과 전자(Electron) 가 무지막지하게 자기 스스로 돌고 있는 “본질적 양자 스핀(Spin, \hbar)” 자체를 자이로스코프의 거대한 무쇠 팽이(각운동량 \vec{H}) 대신 써먹는 괴물 자기 공명(Nuclear Magnetic Resonance, NMR) 자이로스코프다.
10^{-10} 도(degree)의 오차율 한계에 부딪힌 역사가 양자 얽힘 스핀계로 어떻게 항법을 창조하는지, 무기계의 궁극 비밀을 해부한다.

1. 핵스핀 각운동량과 라모 세차 운동(Larmor Precession)

거대한 기계 팽이가 \text{RPM} 회전을 먹고 우주를 쳐다보며 도는 오일러 각운동량 모델은 원자핵 속에서도 똑같이 일어난다.
크세논(Xenon) 가스나 헬륨 (^3\text{He}) 원자핵 수조에 갇힌 미세 입자 위로 강력한 직류 자기장(B_0)을 Z축을 따라 쏴버리자. 그러면 정신없이 제멋대로 얽혀 춤추던 구슬(원자 스핀) 들이 자기장의 힘을 받고 일제히 \mathbf{Z} 축으로 등뼈를 세운 거대 자석 팽이 구슬로 정렬(Alignment)된다(양자 자화벡터 \vec{M} 생성).

여기서, 옆구리 X축에서 또 다른 약한 교류 자기장(RF 펄스)을 팍 쳐서 이 스핀 원반축을 억지로 90^\circ 비틀어 눕혀 놓으면 어떻게 되는가? 팽이는 쓰러지지 않고 Z축 기둥을 중심으로 위이이잉 돌며 고깔모자 원을 그리는 미친 춤을 춘다! 바로
라모 세차(Larmor Precession, \vec{\omega}_L = \gamma \vec{B}_0) 다!
\gamma 는 입자 고유의 자기 회전 비율상수다. 즉, 자기장만 걸어주면 이 가스 구슬 입자 수십억 개가 마치 한 몸집의 보이지 않는 자이로 팽이가 되어 자기만의 미친 진동 주파수파(\omega_L)를 우주 속에 내뿜으며 뱅뱅 돌기 시작한다.

2. 양자 공명 위상 엇박자 (Phase Shift) 로 읽어내는 절대 각속도 \Omega

자! 이제 이 칩을 단 첩보 위성이 우주에서 오른쪽 방향으로 \mathbf{\vec{\Omega}_{body}} 속도로 회전한다고 치자.
위성이 돌아가면 위성 껍데기에 붙어있는 칩도 돌아간다. 그런데 신기하게도, 자기장 속 허공에 떠서 빙글빙글 돌고 있는 핵스핀들(원자 팽이)은 위성의 껍데기가 어떻게 돌든 관심이 없고 자기만의 라모 속도(\omega_L)로 절대 공간을 바라보며 영원히 돌고 있다(관성 유지).

  • 결국 센서를 장착한 비행선(사람)의 눈에는 핵스핀 팽이의 돌아가는 주파수가 원래 속도 \omega_L 랑 약간 달라져서 보이게 된다. 내가 돌아가는 속도 \vec{\Omega} 가 팽이의 엇박자 위상 주파수 변화량(\Delta \omega) 에 고스란히 묻어 나오는 것이다!

\mathbf{\omega_{\text{observed}} = \omega_L \pm \Omega_{\text{body}}}

스마트폰이나 드론에 들어가는 기계 막대기를 쓰지 않고, 유리 앰풀 병 안에 원자 가스를 몇 방울 떨어트리고 자기장 레이저 광 펌핑(Laser Optical Pumping)을 쏜 뒤, 이 원자들이 돌면서 뿜어내는 가짜 위상 엇박자 파장 주파수(초정밀 Atomic Clock) 차이 \Delta \omega 나 코일 유도 기전력(Faraday Law) 전압 아날로그만 딱 읽어내면 우주선의 현재 텀블링 속도가 그냥 숫자로 툭 튀어나온다.
