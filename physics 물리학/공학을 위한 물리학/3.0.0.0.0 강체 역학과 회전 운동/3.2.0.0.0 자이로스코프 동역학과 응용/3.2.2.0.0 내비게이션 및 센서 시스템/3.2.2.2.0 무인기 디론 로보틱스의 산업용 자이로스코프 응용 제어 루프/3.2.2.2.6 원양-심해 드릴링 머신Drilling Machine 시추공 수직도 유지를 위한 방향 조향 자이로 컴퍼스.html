3.2.2.2.6 원양/심해 드릴링 머신(Drilling Machine) 시추공 수직도 유지를 위한 방향 조향 자이로 컴퍼스

육지에서 북쪽을 찾으려면 만 원짜리 쇠 지남철(자기 나침반) 하나면 끝난다.
하지만 북극해 한가운데서 석유를 캐기 위해 수심 5,000\text{m} 의 바닷속을 파 내려가, 지구 지하 3 킬로미터 깊숙한 마그마 옆 암반 속에서 거대한 강철 다이아몬드 지형 톱날(Drill Bit) 이 땅을 후벼 팔고 있을 때, 기사는 지금 드릴이 동쪽으로 파고 있는지, 남쪽으로 파고 있는지 도저히 알 길이 없다.
지하 수 킬로미터 밑에서는 자기장 나침반도 쓸모없게 되고 전파(GPS)도 막혀버리며 센서 주변은 전부 자성을 띤 두꺼운 쇠 파이프로 둘러싸여 있기 때문이다. 오직 드릴 머리가 “지구의 진짜 자전축 북극(True North)” 을 찾아 방향을 잃지 않고 암흑을 뚫고 나가 수직으로 꽂히도록 인도하는 자이로-컴퍼스(Gyro-compass / Downhole Gyro) MWD (Measurement While Drilling) 네비게이션이 심해 공학 조향 기술의 정점이다.

1. 북향 추적자(North-Seeking): 북극 자전속도 15deg/hr 의 역학적 제압

지구가 하루에 한 바퀴 돈다는 사실은 평범해 보이지만, 우주 공간에서 보면 지구는 매시간마다 15도(15^\circ / hr)씩 뱅글뱅글 돌아가는 빙판(자전 원반) 이다.
드릴 머리에 박힌 아주 정밀한 북향 탐색(North-seeking) 자이로스코프 센서는 캄캄한 지하에서도 멈춰 서서 이 희미한 15도의 지구 자전 각속도를 칩으로 느껴버린다.

  • 원판(지구)의 우측으로 잡아당김 결박:
    드릴 속 팽이가 어떤 임의의 방향으로 수직 기둥 각운동량(\vec{H})을 박아놓고 돌고 있다 치자. 몇 시간이 지나면 어김없이 지구가 자전을 하면서 밑바닥 땅 자체가 이 팽이의 발을 걸고 넘어뜨려버린다.
    “어 땅이 자전해서 내 척추를 밀어내네?”
    이 팽이는 지구가 바닥을 꺾어버리는 아주 미세한 각속도 오메가 타격파에 반응해, 즉각적으로 오일러 외적인(Precession) 반작용으로 고개를 확 돌린다. 그리고 자이로 팽이는 자기의 スピン 축이 “지구 자전축(진짜 우주 북극 방향 화살표)” 과 모조리 평행 나란하게 일치(\text{Align})할 때까지 고개를 흔들다가 딱 멈춘 채 안정을 찾아 잠든다(북극을 쳐다봄).
  • 즉, 무시무시한 고철 지옥 섭씨 200도 달궈진 시추관 한가운데서 자이로는 10분만 기다리면 윙 돌다가 척추를 우주의 절대 북쪽(True North) 방위각 에 결속시키고 아지무스(\text{Azimuth}) 0도 등대 역할을 도맡는다! 이를 통해 기사(Driller)는 드릴 비트를 1도라도 옆쪽으로 파내려가 석유 광맥을 놓칠 뻔한 사태를 막아낸다.

2. 끊임없이 방향을 조정하는 루터 (Rotary Steerable System, RSS)

드릴이 똑바로 땅 밑, 혹은 석유가 흐르는 가로 지층 50킬로미터를 일직선으로 수평 횡단하도록(Directional Drilling) 유지할 때, 지하 자이로 센서가 “앗! 지금 각도가 수직(0도)에서 우측으로 +3도 삐딱하게 암반을 파고 있어!” 라고 외치면, 드릴 앞부분에 달린 첨단 역학기구 로터리 조향 시스템(RSS) 이 움직인다.

이게 바로 관성 센서의 “수직 오차 측정치“를 먹고 작동하는 \text{액티브 3D 오일러 타격 피드백}이다.

  • RSS 에는 드릴 파이프 둘레를 따라 유압으로 불쑥 튀어나오는(Push-the-bit) 밀대(Pad) 쇠발이 3개 숨겨져 있다. 센서가 +3도 삐뚤게 간다는 신호를 쏘자마자, 오른쪽 쇠발이 땅벽 쪽으로 “꽝!” 유압 발차기를 먹인다. 땅벽을 차는 밀어내기 힘으로 인해 드릴 머리채는 다시 왼쪽으로 기우뚱 꺾이며 0^\circ 수직을 맞춘다.
