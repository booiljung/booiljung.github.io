3.2.2.2 무인기, 디론, 로보틱스의 산업용 자이로스코프 응용 제어 루프

자이로스코프는 대륙간 탄도 미사일이나 잠수함 같은 거대 우주 방위 산업의 권좌에서 내려와, 이제 우리가 매일 타는 이동 수단과 스마트폰, 드론의 심장부(Micro-controller) 칩 속으로 파고들었다.
하늘에 뜬 쿼드콥터 드론이 거센 돌풍 속에서도 그림처럼 멈춰 서는(Hovering) 호버링의 마법부터, 넘어지려는 외발자전거를 마이클 잭슨처럼 꼿꼿이 반사적으로 세워 올리는 세그웨이(Segway)의 역진자 제어, 그리고 거친 풍랑 속 요트의 멀미를 잠재우는 안티-롤(Anti-roll) 거대 팽이의 억제력까지. 불안정한 기계 시스템 스스로가 센서를 통해 ’나의 기울어짐’을 인지하고 즉시 모터를 역회전시켜 진동 미적분 방정식을 소멸시켜버리는, 산업용 피드백(Feedback) 액티브 제어 루프의 응용 세계를 해부한다.
