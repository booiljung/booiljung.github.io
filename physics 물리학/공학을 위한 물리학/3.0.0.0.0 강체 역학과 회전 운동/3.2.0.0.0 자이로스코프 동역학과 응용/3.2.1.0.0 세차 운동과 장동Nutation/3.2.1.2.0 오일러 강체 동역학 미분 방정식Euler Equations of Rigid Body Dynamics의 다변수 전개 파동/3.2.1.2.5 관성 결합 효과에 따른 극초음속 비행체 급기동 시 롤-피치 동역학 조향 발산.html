3.2.1.2.5 관성 결합 효과에 따른 극초음속 비행체 급기동 시 롤/피치 동역학 조향 발산

마하 10을 뚫고 우주 가장자리(Sub-orbital)로 튀어 오르는 거대 여객기 모형의 극초음속 비행체(Hypersonic Glider)나 가늘고 긴 미사일(연필 모양 관성체, I_1 \approx I_2 \gg I_3)을 상상해 보라. 이 기괴할 만큼 뾰족한 무기체계에서 날개면에 붙은 수직 꼬리날개 타면들을 꺾어, “우주인아 롤(Roll, 기체 팽이 돌기) 기동을 쳐라!” 라고 명령을 때려 모터를 돌리는 순간, 비행기는 자기 맘대로 갑자기 고개를 하늘로 치켜들거나 팍 찍어누르고, 옆(\text{Yaw}) 으로 질질 새는 끔찍한 오일러 비선형 굴절(관성 커플링 발산, Inertial Coupling Divergence)에 사로잡혀 통제 모니터를 새빨갛게 물들인다.

1. 초음속의 연필 관성 텐서: 길쭉한 강체의 롤 가속도(\dot{p})

항공 공학자들은 각속도 \vec{\omega}_{1,2,3} 을 \text{Roll }(p), \text{Pitch }(q), \text{Yaw }(r) 세 글자 텐서로 쪼개어 부른다.
아주 얇고 긴 바늘 모형 미사일은, 양 옆 날개나 배때지의 뭉툭함보다 길쭉함이 엄청 크기 때문에 주 관성 모멘트가 기형적이다.

  • \mathbf{I_y (Pitch)} 와 \mathbf{I_z (Yaw)} 는 비행기 꼬리와 대가리를 흔들어야 하니 무지막지하게 무거운 쇳덩어리(크다) 다.
  • 반면 \mathbf{I_x (Roll)} 은 그냥 연필 구르듯 바늘을 돌리면 되니 무지막지하게 가벼운 스티로폼(작다) 이다. (I_y \approx I_z \gg I_x)

이 바늘을 엄청난 속도로 회전 롤(Roll, p) 치면 무슨 일이 일어날까?
오일러 미분방정식 2번(피치)과 3번(요) 식 꼬리에 매달린 외적 텐서를 노려봐라.

\dot{q} = \frac{\tau_y}{I_y} + \frac{I_z - I_x}{I_y} \mathbf{r\,p}
\dot{r} = \frac{\tau_z}{I_z} + \frac{I_x - I_y}{I_z} \mathbf{p\,q}

2. 롤 스피드 \mathbf{p} 발진 펌핑: 피치와 요의 공진 발산

  • 수식의 (I_z - I_x) 를 보라. 아까 보았듯 I_z 는 I_x 보다 압도적으로, 무지막지하게 크다. 즉 이 외적 꼬리 상수 텐서는 절대로 0 에 가까운 먼지가 아니다. 어마어마한 배율 증폭기(Multiplier) 계수다!
  • 조종사가 이 미사일을 나사못 박듯 롤(Roll 스피드 \mathbf{p}) 을 분당 5 바퀴씩 미친 듯이 감으며 가속시켰다(p > 0 발동). 그러면 저 \dot{q} (피치 가속도 발진) 식 꼬리에 붙어있던 \mathbf{r} (요잉 속도) 이 아주 미풍(바람)의 영향으로 먼지만하게 찍혔다 치더라도, \mathbf{p}(무지막지한 롤 스피드) 와 곱해지면서 폭발(\times 1000 배 증폭)해 \mathbf{\dot{q}} (피치가 위로 코 꺾이는) 미분 각가속도 칩을 미친 듯이 하늘로 발사해버린다!

이것을 관성 결합(Inertial Coupling) 동역학이라 부른다. 질량을 묶어 날아가는 비대칭(I_{\text{pitch}} \gg I_{\text{roll}}) 바늘 모터는, 자기를 돌아가게 만드는 모터 추진력(p) 자체가, 자신을 위아래로 춤추게 부수는 각가속도(\dot{q}) 미분 에너지 펌프로 스스로 변태(Transformation)되는 텐서를 품고 있다.
전투기가 롤 타겟팅을 윙 윙 돌 때 코가 위아래, 좌우로 오줌발 새듯 출렁거려 타겟 조준점이 미친 듯이 빗나가지 않게 영점을 통제하는 것. 오일러 \vec{\omega} \times (\mathbf{I}\vec{\omega}) 다변수 함수 분해기를 뜯어역으로 소자시켜주는 이 컴퓨터 반력(\vec{\tau}_{\text{comp}})이 제트 항법 제어공학 소프트웨어 피드백 제어의 정수다.
