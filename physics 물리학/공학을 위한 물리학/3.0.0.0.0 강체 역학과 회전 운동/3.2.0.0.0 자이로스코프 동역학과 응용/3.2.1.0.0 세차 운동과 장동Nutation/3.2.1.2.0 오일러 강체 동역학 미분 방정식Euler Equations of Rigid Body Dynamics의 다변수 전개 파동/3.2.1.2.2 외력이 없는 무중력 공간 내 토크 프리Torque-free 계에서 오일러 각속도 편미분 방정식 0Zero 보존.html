3.2.1.2.2 외력이 없는 무중력 공간 내 토크 프리(Torque-free) 계에서 오일러 각속도 편미분 방정식 0(Zero) 보존

우주선이 엔진의 불을 모두 끄고, 태양풍의 압력이나 중력의 미세 구배마저 완벽히 차단된 순수한 심우주(Deep Space) 무중력 상태 한가운데를 날아가는 순간을 가정해보라.
밖에서 건드리는 녀석이 아무도 없으니 토크 편미분 방정식의 촤변은 모조리 \tau_1 = \tau_2 = \tau_3 = 0 (토크-프리) 으로 소멸한다.
하지만 우주인의 손을 떠나기 직전 스핀(\vec{\omega})을 머금고 던져진 렌치(Wrench)나 스마트폰은 밖에서 아무런 힘을 안 주는데도 혼자서 허공에서 위아래로 꿈틀거리고 요동치며 빙빙 돌아간다. 외부 힘 없이 강체의 스핀 텐서만이 뿜어내는 “닫힌 우주계” 안방 에너지 구속 속에서 오일러 방정식은 어떤 자기 보존(Self-conservation) 미분을 거치는가?

1. 토크 영(Zero) 공간에 갇힌 3 연립 미분방정식 궤도

좌변이 0 이 된 오일러 방정식은, 자기 관성(I)과 다른 축의 각속도(\omega)들끼리 처절한 물물 교환을 벌이는 회전 혈투장으로 돌변한다.

  1. \dot{\omega}_1 = \frac{I_2 - I_3}{I_1} \omega_2 \omega_3
  2. \dot{\omega}_2 = \frac{I_3 - I_1}{I_2} \omega_3 \omega_1
  3. \dot{\omega}_3 = \frac{I_1 - I_2}{I_3} \omega_1 \omega_2
  • 상수(Constant) 회전의 성립 조건: 이 미분이 영원히 0 으로 멈춰 아름다운 선형 스핀을 돌려면? 오른쪽에 놓인 두 각속도 변수(\omega_a, \omega_b) 중 하나가 태생부터 0 이어야 한다. 즉, 오직 주 관성축(1, 2, 3) 중 딱 하나에 스택을 100\% 몰빵해서 빙글 던져 돌렸을 때는(예: \omega_1 \ne 0, \omega_2=\omega_3=0), 이 모든 \dot{\omega} 항이 곱하기 0 으로 소멸하며 완벽하고 예쁜 등속 선형 회전을 뿜어낸다.
  • 파동의 타격: 하지만 \omega_1, \omega_2, \omega_3 에 단 1\text{RPM} 의 속도라도 골고루 오차로 분산 입력되면, 저 오른쪽의 외적 곱셈 찌꺼기들이 서로 미분항을 물고 늘어지는 영구 프랙탈 기관이 박동한다. 각 축의 스피드가 커졌다가 작아지면서, 에너지를 끝도 없이 서로 돌려막기(Coupling Oscillator) 하는 것이다!

2. 닫힌계의 영원한 방어벽: 각운동량과 운동에너지의 절대 보존

이 기괴하게 요동치는 미분방정식의 해(Solution) 무리는 절대 허공으로 수학이 튀어 나가지(Diverge) 못하도록 2가지 우주 상수 구배에 멱살이 잡혀 있다.

  1. 각운동량 크기 불변성 (L^2 상수):
    I_1^2 \omega_1^2 + I_2^2 \omega_2^2 + I_3^2 \omega_3^2 = C_1 (반경이 달라지는 동역학 구면체)
  2. 에너지 보존의 벽 (2E 상수):
    I_1 \omega_1^2 + I_2 \omega_2^2 + I_3 \omega_3^2 = C_2 (장축과 단축이 존재하는 타원체 껍질 에너지 벽)

만약 3개 방정식의 비선형 굴착이 이 2개의 둥근 구면 껍질 밖으로 파열해 튀어나가려 하면 물리 법칙에 역행하므로, 오일러 미분의 모든 파동 해(\omega(t) 궤적)는 반드시 저 두 개의 둥근 껍질이 딱 포개어져 만나는 ‘교차 곡선(Intersection Polhode)’ 의 길바닥을 따라서만 영원히 질주해야 한다.
