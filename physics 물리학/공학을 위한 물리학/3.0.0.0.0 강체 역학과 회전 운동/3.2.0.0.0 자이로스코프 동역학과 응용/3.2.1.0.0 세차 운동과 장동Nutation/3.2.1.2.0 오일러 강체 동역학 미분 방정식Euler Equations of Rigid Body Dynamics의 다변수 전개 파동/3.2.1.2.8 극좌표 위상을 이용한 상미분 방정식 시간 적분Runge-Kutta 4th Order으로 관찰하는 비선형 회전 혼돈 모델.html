3.2.1.2.8 극좌표 위상을 이용한 상미분 방정식 시간 적분(Runge-Kutta 4th Order)으로 관찰하는 비선형 회전 혼돈 모델

오일러의 3원 연립 미분방정식(\dot{\omega} = I^{-1}(\vec{\tau} - \vec{\omega} \times (\mathbf{I}\vec{\omega})))은 사람이 손으로 연필을 잡고 풀 수 있는 해석적(Analytic) 적분 공식(사인/코사인 f(t)) 형태가 안 나온다. 외부 토크가 시간에 따라 기괴하게 바뀌거나, 비대칭 관성 덩어리가 3차원에 난무하는 순간 이 함수는 쌍곡선 껍질을 터트리고 혼돈(Chaos) 영역으로 발산한다.
그래서 현대 우주 공학자들은 미적분을 포기하고 이 미친 강체 다이내믹스의 1 초 뒤 미래를 알기 위해, 컴퓨터 \text{CPU} 클록을 무지막지하게 갈아 쪼개 먹는 숫자의 힘, Runge-Kutta 4th Order (RK4) 수치 적분(Numerical Integration) 연동 시뮬레이터 파동 분쇄기에 로켓을 던져 넣는다.

1. 연속적 미분 공간을 칼로 써는(Discretization) 타임 스텝 \Delta t

만약 내가 발으로 찬 벽돌이 지금 1초 시점에 [p, q, r] 로 돌고 있을 때, 딱 1.001 초(\Delta t=0.001) 뒤에는 과연 스피드가 어떻게 변해 있을지(다음 각속도, 다음 자세 \theta) 알 수 있는 유일한 공학적 방법은, 오일러 미분식 \dot{\omega}(\text{기울기}) 에 시간 조각 \Delta t 를 곱해서 과거의 수치에 무식하게 우겨 더해 나가는 체인 더하기다.

허나 단순 1차 오일러 적분(\omega_{next} = \omega_{now} + \dot{\omega} \cdot \Delta t)을 쓰면, 아까 배운 그 악랄한 pr, pq 계열의 \text{자이로 외적 커플링(Gyro Cross-Product)} 곡스퍼가 일으킨 거대한 비선형 휘어짐(Curvature)을 1차선 직진 연산이 도저히 따라가지 못해 수식이 우주로 발산 폭발하고 시뮬레이터 그래픽이 나뒹군다.

2. 룽게-쿠타(RK4) 의 4지점 텐서 곡률 흡수와 기하 수렴

RK4는 \Delta t 하나를 넘어갈 때, 그냥 현재 위치의 기울기만 쓰지 않는다.

  1. 지금 서 있는 곳의 기울기 (k_1)
  2. \Delta t 의 절반(0.5) 지점에 가상의 미분 타격 점을 찍고 예측한 기울기 (k_2)
  3. 그 가상 점 k_2 를 기반으로 한 번 더 가운데 축을 틀어 찍어본 절반 지점 기울기 (k_3)
  4. \Delta t 끝 지점 피니시(1.0) 타겟 예측망의 기울기 (k_4)

이 네 개의 기울기 타격 칩을 \frac{k_1 + 2k_2 + 2k_3 + k_4}{6} 라는 마법의 황금비율로 버무려(\text{Weighted Average}) 단 1 보 전진하는 수식이다. 이 묵직한 \text{CPU} 연산의 희생(4번이나 미분 계산기를 더 돎)이 곁들여져야만, 지독하게 휘어지려는 \mathbf{e^{kt}} 쌍곡선 오일러 텀블링의 파괴적 발진 미분 곡선을, 컴퓨터가 선형 탈선(Linear Off-track Error) 하지 않고 기가 막히게 쥐어 짜내며 0.0001\text{ rad/s} 단위로 보존 위상 공간을 타겟 추적해 잡아낸다.
