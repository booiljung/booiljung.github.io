3.2.1.2.6 단일 모멘텀 휠 및 복합 제어 모멘트 자이로(CMG)를 장착한 우주선 피드백 비례 제어

무중력 우주 공간에는 허블 우주 망원경(HST)이나 첩보 위성을 떠받칠 땅(마찰)도, 날개를 펄럭일 공기(유체)도 없다. 오직 추진 가스(Thruster)를 뿜어 각도를 틀어야 하는데, 안타깝게도 가스는 무게와 부피가 커서 금방 고갈된다.
그래서 인류는, “각운동량 보존 법칙(Conservation of Angular Momentum)” 의 맹점, 즉 태양 전지로 전기를 빨아먹어 인공위성 배 속 안의 기계 팽이(모멘텀 휠) 속도를 올려버리면, 팽이가 먹어치운 각운동량(+H) 만큼 위성체 동체가 반대편으로(-H) 돌아가게 만드는 무동력 영구 역학 제어 치트키를 채택했다.
본 절에서는 1차원의 반력 휠(Reaction Wheel, RW) 각가속도 통제와, 90^\circ 세차 토크 마법으로 수천 톤의 우주 정거장(ISS)까지 뒤집어엎는 제어 모멘트 자이로(Control Moment Gyro, CMG)의 피드백 미분 제어식을 해체한다.

1. 1차원 반작용 휠(Reaction Wheel): 오일러 각운동량 분가 법칙

단순히 모터의 RPM 스피드만 높였다 줄이는 단일축 반력 휠로 위성 본체를 제어하는 오일러 방정식의 구조를 뜯어보자. 우주선 동체(\text{Body}, B)와 동체 안에 박힌 모터 휠(\text{Wheel}, w) 두 강체의 Z축을 딱 일자로 맞춘 채 합쳐놓은, “합성 관성 텐서계” 미분 속도 분할식.

(I_{B,z} + I_{w,z}) \dot{\omega}_{B,z} + I_{w,z} \dot{\omega}_{w,z} = \tau_{\text{ext}}

  • 외부 토크 \tau_{ext} = 0(우주 진공) 일 때:
    만약 우주선 내부 피드백 컴퓨터가 휠 모터를 위이잉 돌려서 휠 각가속도 \dot{\omega}_{w,z} 가 + 플러스 \text{10 rad/s}^2 로 급발진을 시작했다 치자. 우변이 0 이 되어야 하니, 우주선 바디(Body)의 각가속도 \dot{\omega}_{B,z} 는 이를 맞추기 위해 강제로 - 마이너스 방향으로 굴러가야 한다!
  • 즉 내가 오른쪽으로 위성을 돌리고 싶으면? 속에 달린 내장 팽이를 아주 거칠게 왼쪽 방향으로 가속(\text{Acceleration})시켜 스피드를 올리면 된다. 그러면 기체는 스피드를 반작용으로 먹고 아름답게 우측으로 피치-요(Pitch, Yaw) 방향타를 트는 각 텐서 제어의 정수를 이룬다.

2. 모터 한계와 스핀 감가, 단일 CMG의 세차 토크 폭발

반작용 휠(Reaction Wheel)은 끔찍한 단점이 있는데, 계속 한 방향으로만 비행기를 틀려면 팽이는 무한대 RPM 속도로 올라갈 때까지 가속 페달만 밟아야 한다. 휠 베어링은 폭발을 면치 못하고 한계치(Saturation) 에 도달한다.
허나, 90^\circ 로 비틀어버리는 교차 십자 외적 무기 \text{제어 모멘트 자이로(CMG)} 를 쓰면 차원이 다르다! 모터 스피드는 항상 10,000 RPM 으로 똑같이(상수 Constant) 유지한 채, 그 감싸고 있는 뼈대 \text{짐벌(Gimbal) 축} 만 윙윙 좌우로 각도를 살짝씩 흔들어(\dot{\delta} 제어 각속도) 주기만 하면!
자이로 공식 \vec{\tau} = \mathbf{\vec{\Omega}_{gimbal}} \times \vec{H}_{wheel} 망치에 의해, 모터 스피드는 안 올렸는데도 수천 배 증폭된 괴랄한 반력 토크가 90^\circ 우주선 배때지로 튀어 나가 위성(수백 톤짜리 정거장 본체)을 확확 넘겨버리는 자이로 토크 증폭 지렛대 가 발동한다.

복합 모멘트 \mathbf{J} 자코비안 군집 배열(4 개 이상의 피라미드 \text{CMG})은 이 강력한 증폭 우주선의 조향 각속도 커맨드를 실시간으로 배분하면서, 어느 짐벌 축 하나라도 0(\text{Zero}) 가 되어 막히지 않도록 영공간(Null Space) 궤적을 실시간으로 도망 다니며 헤엄치는 4 차원 고차 텐서 수학의 아름다운 탈출 미분 예술이다.
