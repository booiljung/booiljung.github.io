3.2.1.2.3 주 관성 모멘트 행렬을 대입한 I1, I2, I3 간 크기 차이로 인한 세차/장동/스핀 연립 쌍곡선 안정성 분석

자유 우주 공간에 던져진 3차원 강체의 회전 운명은, 그 기계 내부 깊숙이 박힌 납 덩어리들의 기하학적 분포, 즉 주 관성 모멘트 행렬 크기(I_1, I_2, I_3) 비율 차이가 낳은 폭군의 식성이다. 우주선 동체를 피치(Pitch I_1), 롤(Roll I_2), 요(Yaw I_3) 로 던졌을 때, 수식이 어떤 축은 극진히 살려주고 어떤 축은 쌍곡선 루프로 무참히 찢어발기는가?
앞 절의 테니스 라켓 텀블링에서 보았던 안장점(Saddle Point)을 수학 공학의 연립 행렬 고유값(Eigenvalue) 자코비 행렬(Jacobian) 선형화 안정성(Stability) 테스트로 뜯어부수자.

1. 평형점(Equilibrium Point)의 미소 외란(Small Perturbation) 테스트

우주선을 \mathbf{Z} 축(3 번 관성축)을 메인으로 잡아, 수만 \text{RPM} 의 항속 속도 \omega_0 로 예쁘게 스핀 돌리고 있다 치자.
이때 평형 상태 \omega = [0, 0, \omega_0]^T 에 살짝 마이크로 단위의 먼지가 부딪혀 \mathbf{X}, \mathbf{Y} 축 방에 잡음(Noise) 진동 \delta\omega_1, \delta\omega_2 이 조금 묻어 들어갔다. 오일러 방정식에 이 아주 미세한 \delta 치환들을 강제로 밀어 넣고 이차항(너무 작은 제곱)을 잘라낸 1 차 미분 선형화 방정식이 토해낸 텐서 행렬이다.

\dot{\delta\omega_1} = \frac{I_2 - I_3}{I_1} \omega_0 \cdot \delta\omega_2
\dot{\delta\omega_2} = \frac{I_3 - I_1}{I_2} \omega_0 \cdot \delta\omega_1

이 두 방정식을 연립해 두 번 미분(\ddot{\delta})하면, 우주의 심장 박동인 단순 조화 진동자(Simple Harmonic Oscillator) 미분 방정식 포맷이 터져 나온다.

\ddot{\delta\omega_1} + \left[ \mathbf{\frac{(I_3 - I_2)(I_3 - I_1)}{I_1 I_2} \omega_0^2} \right] \delta\omega_1 = 0

  • 괄호 \mathbf{[ \,\, ]} 안의 무지막지한 상수 덩어리 값 부호(Sign)가 바로 나의 위성이 우주 미아가 될지 예쁘게 진동할지를 결정짓는 생사의 파동 판별식이다.

2. 생존 구배(I_3 의 상대적 위치)와 발산(Divergence)의 멸망식

괄호 안 상수의 부호 특성에 따라 2 계 미분 방정식 해 궤적 공간이 요동친다.

  • 안정의 축복(I_3 가 압도적으로 가장 작거나, 가장 클 때): 만약 I_3 가 I_1 과 I_2 보다 모두 작다면((I_3 - I_2) < 0, (I_3 - I_1) < 0 음수 2개 곱셈) \rightarrow 괄호 전체가 플러스(+) 양수 상수가 된다! 양수일 땐 미분해 궤적이 \delta\omega_1 = \sin(A t) 예쁜 삼각함수 파도로 떨어지며, 오류가 커지지 않고 영원히 제자리 근처에서 잔파도만 치다 안정을 유지하는(\text{Stable Center}) 궤도를 탄다 (세차와 장동 진동 융합 복구).
  • 파멸의 음수(I_3 가 I_1 과 I_2 중간 크기로 애매하게 낄 때): 만약 I_1 > I_3 > I_2 라면? 괄호 안 왼쪽은 양수, 오른쪽 반쪽은 음수가 되어 곱하면 곱창 난 마이너스(-) 의 파동 계수상수가 출현한다! 이때 미분식 해(Solution) 는 싸인파가 아니라 지수 함수 \mathbf{e^{+kt}} 괴수를 소환한다. 초기 타격이 0.001 이었던 진폭 찌꺼기가 단 몇 초 만에 무한대로 뻗어 발산(Divergence) 폭발하며, 위성체를 완전히 허공에서 뒤집어 까버리는(\text{Hyperbolic Saddle}) 쌍곡 결계가 붕괴된다.
