3.2.1.2.1 주 관성축(Principal Axes of Inertia) 좌표계와 뉴턴 제2법칙 모멘트 공식 외적 항의 결합 방정식 유도

거꾸로 뒤집힌 자동차 엔진 블록이나 날아가는 벽돌이 공중에서 어떻게 뒤틀리며 텀블링할지 예측하려면, 가장 먼저 뉴턴의 제2법칙 회전 버전인 \vec{\tau} = \frac{d\vec{H}}{dt} 에서 모멘텀 \vec{H} 를 풀기 쉬운 행렬로 해체해야 한다.
그런데 우주인조차 혀를 내두르는 3 차원 관성 텐서(Inertia Tensor) \mathbf{I} 는 가로, 세로, 높이 외에 무조건 비스듬한 축의 비대칭 \text{관성곱(Products of Inertia, } I_{xy}, I_{yz}) 지뢰들을 3 \times 3 행렬 속에 빽빽하게 품고 있어 미적분을 불가능하게(Coupled Chaos) 만든다.
이를 회피하기 위해 오일러는 강체를 해부하여, 오직 대각선 요소만 살아남은 완벽히 독립된 3 가지 뼈대 화살표, 주 관성축(Principal Axes of Inertia) 이라는 신성한 로컬 죄표계를 도입하고, 여기에 로컬-글로벌 미분 포아송(Poisson) 공식을 때려 박아 오일러 방정식을 유도해 냈다.

1. 뼈대의 정렬: 텐서 대각화(Diagonalization)와 관성곱의 학살

돌덩이가 아무리 기괴하게 생겼다 한들, 선형 대수학(고윳값 분해, Eigenvalue Decomposition)에 의하면 어떤 강체든 무조건 관성곱(I_{xy}=I_{yz}=I_{zx}=0)이 수학적으로 완전히 0(Zero)으로 소멸해 증발해 버리는, 기적의 직교 축 3개가 그 돌덩어리 한가운데를 뚫고 지나가고 있다!
이를 ’주 관성축’이라 부르며, 이 주 관성축 공간 좌표 (1, 2, 3) 에 올라탄 비행사(로봇 제어기) 입장에서 우주선의 관성 텐서 \mathbf{I} 행렬은 오직 I_1, I_2, I_3 세 개의 대각선 스칼라 숫자만 남는 깨끗한 상태로 멸균 정화된다.

  • \vec{H}_{local} = [I_1\omega_1, I_2\omega_2, I_3\omega_3]^T (드디어 관성 모멘트와 스핀 각속도의 깔끔한 1:1 결합 타격)

2. 오일러 연립 동역학 텐서의 파동: \mathbf{I}\dot{\vec{\omega}} + \vec{\omega} \times (\mathbf{I}\vec{\omega})

뉴턴의 d\vec{H}/dt 는 기계 밖에 가만히 서 있는 신(Global Inertial)의 눈으로 본 미분이다. 하지만 기계 센서는 팽이 내부 로컬 주 관성축 방(\text{Body Frame})에서 측정하므로, 우리는 저 깔끔한 \vec{H}_{local} 을 강제로 ’회전하는 공간(방) 자체가 통째로 돌아가며 치는 외적 미분 연성’으로 보상해 주어야 한다 (체인 룰 도함수 변환).

\vec{\tau}_{\text{ext}} = \left(\frac{d\vec{H}}{dt}\right)_{\text{local}} + \vec{\omega} \times \vec{H}

이 공식에 저 깔끔해진 대각 관성 \vec{H} = [I_1\omega_1, \dots]^T 를 때려 넣고, \omega = [\omega_1, \omega_2, \omega_3]^T 간의 크로스 프로덕트(\times) 외적을 씹어 돌리면, 3차원 우주 모든 회전 기계의 영혼을 통제하는 3 줄짜리 괴물, 오일러 동역학 방정식(Euler’s Equations of Motion) 이 폭발한다.

  1. \tau_1 = I_1 \dot{\omega}_1 + (I_3 - I_2) \omega_2 \omega_3
  2. \tau_2 = I_2 \dot{\omega}_2 + (I_1 - I_3) \omega_3 \omega_1
  3. \tau_3 = I_3 \dot{\omega}_3 + (I_2 - I_1) \omega_1 \omega_2

수식 끝자락에 붙어있는 (I_3 - I_2) \omega_2 \omega_3 외적 꼬리야말로 3차원 우주가 허락한 가장 극악무도한 ’자이로스코프적 상호 간섭 텐서(Coupling Dynamics)’의 현현이다.
1번 축, 2번 축, 3번 축은 결코 단 하나의 독립된 모터방(Room)처럼 따로 미분 적분되지 않고, 한 축이 회전(\omega)하면 옆 방, 앞방의 각속도와 들러붙어 피와 살을 섞고 끝없는 에너지 진동 셔플을 벌인다. 이것이 수만 톤의 전함이나 거대 우주 정거장이라도 축 제어를 자칫 잘못 건드리면 단 1초 만에 전혀 엉뚱한 방향으로 발작하며 전복되는 역학 방정식의 진실이다.
