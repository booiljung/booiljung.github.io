3.2.1.2 오일러 강체 동역학 미분 방정식(Euler Equations of Rigid Body Dynamics)의 다변수 전개 파동

18세기 레온하르트 오일러(Leonhard Euler)는 뉴턴의 단순한 질점계 F=ma 공식의 우주적 한계를 부수고, 부피와 모양(관성 모멘트)이 존재하는 거대한 3차원 바위 덩어리가 우주에서 어떻게 구르고 비틀리며 날아가는지를 증명하는 3차원 비선형 연립 미분방정식 텐서 네트워크를 창조해 냈다.
이 ’오일러 강체 동역학 방정식’은 우주선 자세 제어, 전투기의 공중제비 텀블링, 그리고 하드디스크 모터 진동에 이르기까지 자전하는 모든 기계 문명을 지배하는 단 하나의 절대 수학이다. 본 장에서는 오일러 방정식의 태동부터, 자이로스코프의 혼돈 피드백 파동, 푸앵카레 구(Poincare Sphere)의 위상 공간적 마법까지 역학 방정식이 그리는 비선형 회전 혼돈의 근원을 낱낱이 파헤친다.
