3.2.1 세차 운동(Precession)과 장동(Nutation)

공중에 뜬 거대한 팽이(자이로스코프)를 손가락으로 살짝 쳐서 쓰러뜨리려고 해보자. 이 팽이는 우리가 민 방향으로 픽 쓰러지지 않고, 기괴하게도 민 방향의 90^\circ 강제 직각 방향으로 비틀려 맴도는 세차 운동(Precession)을 시작한다. 그리고 초기 타격의 조건에 따라 바닥을 찍으며 끄덕거리는 떨림(Nutation)까지 동반하며 복잡한 쌍곡선 궤적을 뿜어낸다.
강체 회전의 무결점 보존력이 어떻게 인간의 직관을 배반하는 3차원 십자가 궤적으로 꺾여 돌아가는지, 그 우주론적 회전 궤도의 진수들을 낱낱이 파헤친다.
