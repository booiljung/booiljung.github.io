3.2.1.1.2 대칭 팽이(Symmetrical Top) 관성 모멘트 조건에서의 정상 강제 세차(Steady Precession) 속도 공식 유도

길거리에 앉아 빙글빙글 도는 무거운 금속 팽이를 보라. 팽이는 땅바닥을 찍고 비스듬히 누워 있는데도 안 쓰러지고 마치 마법처럼 중력을 버티며 둥글게 바닥을 긁고 돌아간다(Precession).
이 팽이가 동그랗고 대칭적인 원반(Symmetrical Top, I_x = I_y) 형태일 때, 외부 중력(mg)이 잡아당겨 눕히려는 힘과, 돌아가려는 자이로 각운동량(\vec{H})이 절묘하게 평형을 이루며 완벽히 일정한 속도로 하늘의 깔때기를 그리는 정상 세차(Steady Precession, \Omega) 파동이 일어난다. 도대체 지구 중력이 찍어 누르는데 그 깔때기가 도는 각속도 \Omega 는 몇 \text{RPM} 인지, 뉴턴-오일러 역학 텐서의 정밀 미적분 톱니바퀴를 돌려보자.

1. 전역 각운동량 렌더 타격: 중력 토크 벡터(\vec{\tau})

비스듬히 꺾인 팽이 축 중심에 중력은 밑바닥으로 직각 화살표(mg)를 그으며 팽이의 척추를 자빠뜨리려고 외적 토크 망치를 친다.

  • 바닥 끝 뾰족한 점(Pivot)부터 중력이 작용하는 무게 중심(CG)까지의 길이를 l, 팽이가 수직축(하늘)에서 기울어진 각도를 \theta (Nutation Angle)라 하자.
  • 피벗 점을 기준으로 팽이를 눕히려는 횡방향 전복 토크의 크기는 \tau = m g l \sin\theta 다.
  • 이 \vec{\tau} 가 앞 1장에서 배운 자이로 강성 체인 원리인 \vec{\tau} = \vec{\Omega} \times \vec{H} 모터 속으로 빨려 들어간다.

즉, 저 거대한 중력 토크가 기계를 눕히는 것이 아니라 오로지 바닥을 쓸며 돌아다니는 세차 공전 스피드(\Omega)의 기름(Energy) 으로 100\% 타들어가 변환된다.

2. 자이로 대수 해부학 방정식: \boldsymbol{\Omega = \frac{mgl}{I_z \omega_{spin}}}

수식 \tau = \Omega \cdot H \cdot \sin\theta (방향이 수직이므로 내적 외적 상쇄 각)를 전개하고, 팽이의 자전축 각운동량을 H = I_z \omega_{spin} (자전축 관성 모멘트 \times 미친 듯이 도는 스핀 속도) 로 폭파 치환해 대입한다.

mgl \sin\theta = \Omega (I_z \omega_{spin}) \sin\theta
\therefore \mathbf{\Omega_{precession} = \frac{mgl}{I_z \omega_{spin}}}

이 단 한 줄의 정상 세차 방정식(Steady Precession Equation) 이 팽이와 지구 자전 세차 운동의 숨통을 쥐고 있는 공학의 무결점 여왕이다. \n 수식의 형태를 차갑게 노려보아라.

  • \boldsymbol{\omega_{spin}} 의 기하 이탈 (강성의 증거): 놀랍게도 팽이가 기울어진 각도(초기 치명상) \theta 파라미터가 등사 양변에서 완전히 약분되어 소멸(Cancel-out) 한다! 팽이가 10^\circ 로 기울든, 죽을락 말락 80^\circ 로 극단으로 꺾여서 누워있든! 세차로 맴도는 공전 속도(\Omega)는 오직 팽이의 무게(mg)와 스핀 회전 능력(I_z \omega) 에만 지배당할 뿐, 얼마나 기울었는지(\theta) 따위는 절대 묻지도 따지지도 않고 똑같은 \text{RPM} 깔때기를 그린다.
  • 반비례의 기적 (반중력 자이로 효과): 분모에 있는 \omega_{spin} 을 보라. 팽이가 초당 100바퀴, 1000바퀴로 자기 자신 스핀(자전)을 더 미친 속도로 뽑아낼수록 분모가 커지면서 세차는 더 느려져 하늘에 조용히 멈춰 선다(\Omega \rightarrow 0, 자이로 강성 극대화). 반대로 마찰 때문에 스핀 속도가 죽어서 분모가 작아지면 세차 속도 분자가 미친 듯이 솟구쳐 오르면서(Wobble 폭주) 우당탕거리며 땅에 꼴아 박게 된다.

이 \Omega = \frac{\tau}{H} 스칼라 분수식 하나를 풀기 위해 스위스의 레온하르트 오일러(\text{Leonhard Euler})는 수년간 3차원 미적분과 싸웠다. 만약 로켓 안에 달린 제어 모멘트 자이로(\text{CMG}) 휠의 속도가 너무 낮아 분모가 죽는다면, 아주 미세한 우주 먼지의 충돌 토크 하나에도 인공위성 동체 전체가 \Omega 통제 불능 스핀을 먹고 고철 덩어리가 된다. 그래서 모든 우주 산업 자이로는 기계 베어링을 배제하고 자기 부상으로 허공에 띄워 진공 속에서 미친 마하의 속도(60,000\text{ RPM} 스핀)로 분모를 폭발시켜 강성 방벽을 세울 수밖에 없다.
