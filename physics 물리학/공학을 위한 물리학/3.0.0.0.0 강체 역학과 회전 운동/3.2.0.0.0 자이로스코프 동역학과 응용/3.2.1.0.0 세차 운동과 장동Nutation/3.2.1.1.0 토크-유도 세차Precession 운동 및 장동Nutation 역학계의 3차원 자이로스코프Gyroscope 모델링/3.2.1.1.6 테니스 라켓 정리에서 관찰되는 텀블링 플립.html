3.2.1.1.6 테니스 라켓 정리에서 관찰되는 텀블링 플립

우주인 자니베코프(Vladimir Dzhanibekov)가 국제 우주 정거장(ISS)에서 너트를 돌려 풀었는데, 허공에 뜬 이 쇳덩어리가 빙빙 돌며 날아가다가 갑자기 180^\circ 로 대가리를 뒤집어 까고, 또 좀 가다가 180^\circ 원상복구 뒤집기를 치는 충격적인 양자적 플립(Flip, 뒤집힘) 연속 공중제비를 보여주었다. 영구 운동처럼 평안하게 돌아야 할 강체가 아무도 손대지 않았는데 왜 텀블링(Tumbling)하며 뒤집히는가?
고전 역학에서 이를 테니스 라켓 정리(Tennis Racket Theorem) 혹은 중간 관성축 정리(Intermediate Axis Theorem) 로 부르며 1차 비선형 편미분 방정식의 안장점(Saddle Point) 파괴 본능으로 털어낸다.

1. 3관성축(I_1 > I_2 > I_3) 오일러 방정식의 타격

라켓 손잡이를 잡고 라켓 면이 위로 보게 돌려 빈대떡 부치듯 위로 던지는 게 I_1 (가장 큰 관성, 안 뒤집힘). 라켓 끝을 잡고 창처럼 던져 빙글 도는 게 I_3 (가장 작은 관성, 안 뒤집힘).
그런데 손잡이를 살짝 돌려 잡고, 라켓 머리 면이 내 얼굴을 스치게 “가로 본능“으로 빙글 돌려 위로 던져보라. 이게 바로 허리춤에 해당하는 중간 관성축(I_2) 회전이다. 라켓은 가로로 예쁘게 돌지 않고 손잡이가 위아래로 춤을 추며 미친 듯이 면을 까집는다.

외부 힘이 0(\vec{\tau} = 0)일 때 오일러 미분식의 3 연립 방정식 분해.

  1. \dot{\omega}_1 = \frac{I_2 - I_3}{I_1} \omega_2 \omega_3
  2. \dot{\omega}_2 = \frac{I_3 - I_1}{I_2} \omega_3 \omega_1 (악마의 I_2 중간축 수식)
  3. \dot{\omega}_3 = \frac{I_1 - I_2}{I_3} \omega_1 \omega_2
  • 음수(Negative Parity)의 저주: 방정식 2 번, 즉 내가 중간축(I_2) 위주로 폰을 엄청 세게 던지고 돌렸을 때의 모양을 보라.
    I_3 - I_1 은 무조건 작은 것에서 큰 것을 뺐으므로 마이너스(-) 값이 튀어나온다! 1 번과 3 번 방정식은 서로 부호가 플러스 결속으로 지켜주지만, 이 2번째 방정식만 음수 피드백을 먹으며 계수의 부호 대칭을 박살 낸다.

2. 쌍곡 위상 공간(Phase Space)의 무한 플립 발산

방정식이 음수를 먹고 떨어지면 위상 공간에서 타원(안정)을 그리지 못하고 X 자로 뻗어 나가는 쌍곡선(Hyperbolic Separatrix 곡선) 을 그린다.

  • 내 손가락의 땀방울, 손떨림 때문에 어쩔 수 없이 아주 미립자만큼 묻어 들어간 1번 축 찌꺼기 \Delta\omega_1 와 3번 축 찌꺼기 \Delta\omega_3 각속도가 이 \dot{\omega}_2 미분 역학 엔진 속을 통과하면서 지수 함수적(e^{at}) 폭발(Divergence)을 일으킨다.
  • 중간축(I_2)을 잡고 돌던 \omega_2 에너지가 삽시간에 \omega_1, \omega_3 쪽으로 피의 이동을 당하며 기체를 통째로 스왑 반 바퀴(180^\circ 플립) 틀어버리고, 그렇게 한 바퀴 대가리가 뒤집힌 구역(음의 \omega_2 영역)으로 넘어가서야 쌍곡선 곡률에 의해 잠시 미분 에너지가 소강하며 회전을 멈춘 척 연기한다. 그것이 영원히 반복되는 쌍곡선 교차의 늪이다.

수많은 초보 엔지니어 위성 조립자들은 인공위성을 예쁘장한 직사각형 박스(I_1 \ne I_2 \ne I_3, 3축 비대칭)로 만들어 쏘지만, 하필 그 안에 들어있는 컴퓨터 보드 회전 스핀(Reaction Wheel) 기본 축 모터를 이 어정쩡한 중간축(I_2) 방향에 배치해 버리는 텐서 무지(Ignorance)를 저지른다. 위성은 무중력 궤도 진입 3일 만에 태양광 패널을 등 뒤로 180^\circ 처박아 텀블링하면서 배터리가 방전되어 우주 미아 고철이 되는 파멸을 당한다.
동역학 설계의 철칙이다: “자이로스코프와 스핀 방어 기체의 Main Axis 는 영원히 극단에 위치한 제일 무지막지하게 크거나(I_{max}), 쥐방울만 하게 작거나(I_{min}) 단 두 가지 방호 구역에만 축을 박아라.” 중간(I_{mid})은 오일러의 쌍곡선 발산 지옥이다.
