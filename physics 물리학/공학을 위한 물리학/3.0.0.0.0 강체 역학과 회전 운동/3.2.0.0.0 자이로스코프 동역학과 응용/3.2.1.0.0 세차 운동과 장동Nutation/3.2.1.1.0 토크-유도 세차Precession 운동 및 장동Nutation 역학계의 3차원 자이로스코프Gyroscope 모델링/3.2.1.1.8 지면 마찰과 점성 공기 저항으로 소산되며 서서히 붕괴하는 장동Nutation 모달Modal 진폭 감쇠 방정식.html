3.2.1.1.8 지면 마찰과 점성 공기 저항으로 소산되며 서서히 붕괴하는 장동(Nutation) 모달(Modal) 진폭 감쇠 방정식

자이로스코프나 팽이가 무중력 진공 우주 공간(\text{Vacuum})에서 돌면, 한 번 맞아서 생겨난 끄덕거림 진동(장동, \text{Nutation})은 열역학적 에너지 보존 법칙에 의해 영원히(십만 년이 지나도) 상하로 진동하며 매끄러운 세차 궤도의 평화를 깨트리는 악몽으로 남는다.
하지만 지구의 현실계 강체 역학에선, 공기가 물체를 휘감는(Drag) 점성 저항(\text{Aerodynamic Viscosity})과 바닥 끝 뾰족 점과 대지 흙바닥이 미세하게 깎여 나가는 피벗 마찰 토크(Friction Torque)가 존재한다.
이 척박한 마찰 공간 안에서 요동치는 장동 진동 궤적은 어떤 미분 감쇠(Damping) 수학을 타며 점차 소멸하고 안정을 찾아 들어가는가(Decay Rate)?

1. 장동 편미분 미적분 방정식의 소산(Dissipation) 댐핑 항 개입

정상 팽이 오일러 진동 역학의 2계 동역학 미분 방정식 (I\ddot{\theta} + \dots = 0) 에, 기계 공학자들은 강제적으로 공기 저항과 바닥 마찰 토크가 갉아먹는 마찰 감쇠 계수(C_{friction}) 항을 속도(\dot{\theta})에 비례해 집어넣는다.

I_x \ddot{\theta} + \mathbf{C_{damp}} \dot{\theta} + (\dots)\theta = \text{Error}

  • 이 댐핑 미분 항이 수식 속으로 파고들면, 무한대 진폭으로 영원히 흔들거려야 했을 \theta (Nutation 각) 해집합이 시간에 따라 기하급수적으로 0 으로 수렴(Limit \rightarrow 0)해 버리는 지수 감쇠 파동(Exponential Decay Function, e^{-\zeta \omega_n t}) 렌더링으로 돌변하며 생명력을 잃는다.
  • \mathbf{t \rightarrow \infty} 일 때 팽이의 위아래 끄덕임(장동)은 오일러의 장미 곡선(사이클로이드) 껍질을 부수고 공기 중으로 열(Q)로 에너지를 다 뺏긴 다음, 가장 안정적이고 매끈한 일직선 고도 유지 세차(\text{Steady Precession}) 원형 파동 으로 흡수 붕괴(안락사)된다.

2. 팽이의 느려짐(Spin Deceleration)과 최후의 대전복 파열

하지만 마찰(Friction)의 저주는 장동 끄덕임을 예쁘게 진압해 주는 착한 역할로만 끝나지 않는다.
공기 점성력과 바닥 축 마찰은 장동만 갉아먹는 게 아니라, 팽이가 버티고 꼿꼿하게 서있는 무적의 방어벽, 자전(Spin \omega_z) 강성 속도 자체 역시 서서히 토크로 긁으며 회전수를 죽여버린다(\ddot{\phi} < 0).

  • 스핀 \omega_z 가 마찰에 의해 점점 느려지며 분모가 작아진다.
  • 이전에 배웠던 정상 세차 각속도 기구학 ( \Omega = \frac{mgl}{I_z \mathbf{\omega}} \uparrow ) 공식에 의해, 분모(\omega)가 죽어가기 시작하면 세차 공전 속도는 반비례로 미친 듯이 이성을 잃고 빙빙빙 빨라지며 발작한다!
  • 그리고 팽이가 버틸 수 있는 최소 한계 마지노선 \mathbf{Spin\ Threshold} 에 도달하는 순간, 세차 원뿔 각도마저 힘을 잃고 눕혀버터 땅바닥으로 꼬라박는 통제 불능 엎어짐(Tumbling Collapse) 모달 항복을 시전한다.
