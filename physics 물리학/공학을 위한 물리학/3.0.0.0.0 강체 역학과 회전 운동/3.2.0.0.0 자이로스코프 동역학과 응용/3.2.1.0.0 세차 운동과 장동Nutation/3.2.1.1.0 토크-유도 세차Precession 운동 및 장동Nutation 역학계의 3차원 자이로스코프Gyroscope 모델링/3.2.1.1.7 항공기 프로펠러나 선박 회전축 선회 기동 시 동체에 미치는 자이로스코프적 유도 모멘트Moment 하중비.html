3.2.1.1.7 항공기 프로펠러나 선박 회전축 선회 기동 시 동체에 미치는 자이로스코프적 유도 모멘트(Moment) 하중비

프로펠러 전투기(P-51 머스탱)나 전함 안에 박힌 수백 톤짜리 엔진 터빈 로터 질량 속으로 들어가자. 비행기 조종사가 적기를 피하려고 갑자기 미친 듯이 조종간을 당겨 코를 쳐들거나(\text{Pitch-up}), 함장이 구축함을 왼쪽으로 급선회(\text{Turn}) 시킨다.
이 찰나, 비행기 코와 함선의 선수는 그냥 조종간을 당긴 예쁜 각도로만 파도와 공기를 가르는 게 아니다. 기체 내부에서 수천 \text{RPM} 으로 돌아가고 있던 무지막지한 강철 프로펠러(터빈축 팽이) 자체가 “내 방향 축(\vec{H})을 네놈 함장이 맘대로 후려쳐서 돌리려 하네?” 라며 극대노의 자이로스코프적 보복 모멘트(\vec{\vec{M}}_{gyro})를 양산해 내서 베어링 엔진룸과 비행기 동체 전체를 우당탕 찌그러지게 뒤틀어버린다!

1. 베어링 피로 파괴 곡률: \vec{M}_{gyro} = I \vec{\Omega}_{\text{turn}} \times \vec{\omega}_{spin}

우리가 계속 봤던 자이로 공식은 “토크를 때리면(\vec{\tau}) \rightarrow 세차 공전(\vec{\Omega})이 발생한다” 였다.
이제 이것을 기계 공학 설계자 동역학 반력 시점으로 완전히 뒤집어서 거꾸로 보아라.
팽이가 돌고 있는데(\vec{\omega}_{spin}), 조종사가 자기 맘대로 억지로 함선 축을 틀어서 강제로 팽이한테 세차 공전(\vec{\Omega}_{\text{turn}})을 강요, 먹여 버리면 어떻게 될까?

역학 반작용 토크 보복(\vec{M}_{\text{gyro}} 모멘트)이 팽이 회전축(프로펠러축)을 지지하는 베어링 지지대를 90^\circ 직각 방향에서 망치로 때리며 박살 내기 시작한다!

  • 전투기 프로펠러 기동(Pitch \rightarrow Yaw 파열):
    \mathbf{P-51} 무스탕 전투기가 기관총을 쏘려고 기수를 하늘로 팍 쳐드는 순간(Pitch-up, Y축 회전 강요), 프로펠러는 앞(X축) 쪽으로 회전 중이었다 치자.
    \vec{M}_{gyro} = \vec{\Omega}_{\text{pitch}} \times \vec{H}_{\text{prop}} 십자 교차 외적의 저주에 의해, 모멘트는 Y 와 X 가 지뢰를 밟은 방향, 즉 수직 $\mathbf{Z (\text{Yaw 축}) 쪽으로 난데없이 비행기 대가리를 왼쪽으로 홱 차버린다!}
    조종사는 위로만 올라가려 했는데 비행기가 알아서 왼쪽으로 코를 미끄러지며(Sideslip) 총구 방향을 어긋나게 허공에 쏘아대고, 조종사는 “어? 왜 기수가 지 혼자 좌측으로 미끄러져?” 라며 급히 우측 페달(Rudder)을 차서 필사적으로 그 자이로 힘을 상쇄 진압해야만 한다.

  • 전함 터빈 코너링 하중(Yaw \rightarrow Pitch 침몰 위협):
    구축함 밑바닥 엔진 스크류가 돌고(X축) 있는데 함장이 적 뇌격을 피한다고 급하게 좌현으로 키를 돌렸다(Yaw 강요).
    \Omega_{\text{yaw}} (Z축) \times \omega_{spin} (X축) 의 타격은 \mathbf{Y (\text{Pitch}) 축 모멘트} 가 되어, 어마어마한 바다 항력이 날카롭게 뱃머리 코를 바다 밑창으로 쑥 처박아 물속으로 잠수시키거나, 반대로 하늘로 뱃머리를 들썩여 물 밖으로 점프시키며(\text{Pitching Over}) 선체 용골(Keel) 허리를 우두둑 찢어버리려는 수천 톤 단위의 굽힘 하중을 토해낸다.

2. 모멘트 하중(Loading Stress) 통제 텐서 행렬 강성

엔지니어는 এই 하중 텐서 \vec{M}_{gyro} = \mathbf{I} \, (\vec{\Omega} \times \vec{\omega}) 를 극복해야 한다.

  • \vec{\Omega} (전함의 최대 선회 속도), \vec{\omega} (로터 모터 최고 \text{RPM}), 그리고 무쇠 덩어리 관성 모멘트 \mathbf{I} 이 세 곱셈 덩어리가 최악으로 맞물려 폭발할 때(Worst Case Loading), 그 기계 엔진룸을 받치는 쇳덩어리 조인트나 베어링 철판이 부러지지 않게 수 십 배 두껍게 볼팅(Bolting)을 쳐서 안전 강도 구배율(\text{Safety Factor})을 세팅한다.

우주의 물체는 비행기, 자동차, 로켓 따위를 막론하고 그 안에 강한 스핀을 품은 톱니바퀴 모터 세트장(\vec{H})을 하나라도 가졌다면, 그것은 더 단순한 질점계 F=ma 의 노예가 아니다.
기체를 돌리려(선회율 \vec{\Omega}) 조종간을 잡아당기는 순간 당신의 팔목을 박살 내고 기체의 뼈를 구부려 꺾는 제3 공간 방향의 유령의 힘 \vec{\vec{M}}_{gyro} \, 텐서 반력을 어떻게 기계 베어링 설계 구조체(강도 S)가 견뎌낼 것인가, 아니면 반대 엔진을 돌려 상쇄(Nullify)시킬 것인가가 회전 기계 모멘트 구속학의 정수다.
