3.2.1.1.3 초기 각속도 부여 차이에 따라 발생하는 고각 진동 동요를 칭하는 장동(Nutation) 주기 및 진폭 응답

팽이나 자이로스코프를 회전시켜 책상에 내려놓을 때, 당신이 엄청난 로봇 기계의 정밀도로 아주 부드럽게 세차 원주 속도(\Omega)를 딱 맞추어(Matching) 놓았다면 팽이는 완벽한 원통형 깔때기를 그리며 부드럽게 돌아갈 것이다(정상 세차, Steady Precession).
하지만 인간의 떨리는 손으로 아무렇게나 팽이를 던져 놨다면? 팽이는 처음 자신이 궤도 파도에 올라탈 때 “어? 내가 돌아야 하는 외부 자기 토크 세차 스피드랑, 지금 내가 맘대로 튀어나간 속도 오차가 안 맞네?“라며 공황에 빠진다.
이 팽이는 스스로의 각운동량 뼈대(수직각 \theta 기울기)를 위아래로 끄덕끄덕 격렬하게 흔들고 진동치면서 어떻게든 수식을 회복시켜 에너지 평형계에 자기를 맞춰가는 잔인한 우주 관성 동역학 오토캐치를 수행한다. 이 위아래로 흔들리는(Nodding) 끄덕임과 잔 파도 진동을 장동(Nutation) 이라 부른다.

1. 장동(Nutation)의 라그랑지안(Lagrangian) 에너지계와 섭동(Wobble) 진폭

장동은 세차 속도 공전면 위에 얹힌 “결함 오프셋 파동“이다.

  • 초기 충격 불일치: 만약 팽이를 수직으로 잡고 있다가 갑자기 탁 놔버렸다 치자(Initial \Omega = 0 드랍). 중력이 팽이를 땅으로 밀면서(토크 증가) \theta 가 팍 쓰러지려다가, 팽이는 뒤늦게 “아차, 세차로 도망쳐야지!” 하고 회전(\vec{\Omega} \times \vec{H})을 발동시킨다. 하지만 이미 늦어서 밑으로 푹 찍혀버린 낙폭이 있기 때문에, 팽이는 스피드가 터져 나오면서 다시 반동으로 위축(회복력)되어 올라오는 궤적의 요동을 반복한다(Cycloid 형 요동 투사).
  • 이 요동치는 장동 진동폭(Amplitude)과 주기파가 미분 방정식 해 공간에서 우당탕 튕긴다.

I_x \ddot{\theta} + (I_z - I_x) \omega_z \Omega \sin\theta - mgl \sin\theta = 0

이 거대 미분방정식이 토해내는 고각(\theta)의 요동 함수는, “얼마나 세차 궤도에 안착하지 못한 초깃값(Initial Condition)을 줬느냐” 에 따라 잔물결을 탄다. 잔물결이 심할수록 우주선은 끔찍한 진동(Wobble) 멀미에 시달리게 된다.

2. 장동의 고주파 주기(Frequency of Nutation) 도출

만약 스핀 속도(\omega_{spin}) 가 엄청나게 크다면(w_{spin} \gg 1), 이 장동의 끄덕이는 떨림 주파수(\omega_n)는 다음과 같이 수축한다.

\omega_{nutation} = \frac{I_z}{I_x} \cdot \omega_{spin}

  • 관성 모멘트 비례: 돌아가는 팽이 원반의 관성 축비(I_z / I_x) 에 의해 떨림 주파수가 완전히 좌우된다! 디스크가 거세게 돌아갈수록 이 끄덕임 진동 횟수는 \text{라디오 고주파(High Frequency)} 로 치솟는다.
  • 반대로 진동폭(\text{Amplitude})은 급격히 감소한다. 즉 팽이가 빨리 돌면, 미세하게 윙~ 하고 떨리긴 해도 그 위아래 차이는 맨눈으로는 거의 안 보일(수렴 파괴) 정도로 \text{Steady} 해진다는 역학적 반비례 결합이다.

장동(\text{Nutation})은 팽이의 “버그(Bug)“가 아니라, 비관성 회전 좌표계가 갑작스런 외부 폭력(초기 상태 불일치)을 당했을 때 각운동량 불변 수식을 지켜내기 위한 최소 동작 구배 보상 메커니즘의 찬란한 산물이다. 자이로스코프는 진동을 발산하여 부서지려 하지 않고, 오히려 이 끄덕임 텐서(\theta, \dot{\theta})의 변위 피드백을 위아래로 뱉어가면서 허공에서 궤도를 보정하고 적응(Adaptive Convergence)하여 어떻게든 정상 회전으로 복선 하려는 불굴의 기계 생명력을 갖는다.
