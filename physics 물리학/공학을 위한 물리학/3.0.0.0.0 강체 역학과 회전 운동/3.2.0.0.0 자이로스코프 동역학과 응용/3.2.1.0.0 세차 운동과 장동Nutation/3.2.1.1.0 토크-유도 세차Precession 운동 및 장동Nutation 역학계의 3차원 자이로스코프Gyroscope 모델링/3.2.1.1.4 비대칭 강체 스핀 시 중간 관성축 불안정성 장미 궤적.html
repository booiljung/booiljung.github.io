3.2.1.1.4 비대칭 강체 스핀 시 중간 관성축 불안정성 장미 궤적

완벽하게 둥글게 가공된 팽이(대칭 팽이)를 던지면, 팽이는 부드럽게 세차 원을 그린다. 허나, 당신이 손에 쥐고 있는 스마트폰이나 각진 벽돌책, 혹은 테니스 라켓처럼 가로, 세로, 두께가 다 길쭉길쭉 다르게 생긴 “비대칭(Asymmetrical) 강체” 삼차원 돌덩어리를 허공에 냅다 스핀(Spin)시켜 집어던지면 어떻게 될까?
이 무질서의 덩어리 안에는 3개의 주 관성 모멘트(I_1, I_2, I_3) 축이 존재하는데, 이 중 크기가 중간인 애매한 관성축(\text{Intermediate Axis})을 붙잡고 던지면, 축이 허공에서 1초 만에 뒤집히고 제멋대로 요동치며 빙글 곤두박질치는 기괴한 미분 파동 궤적을 싼다. 이 파멸적 스핀 불안정성 궤적을 3차원 운동량 타원체 위에 그리면, 기묘하게 피어나는 장미 곡선(Rose Curve) 텐서가 드러난다.

1. 최장축, 최단축의 무사안일(Stability)과 중간축의 저주

오일러의 동역학 미분 방정식 \mathbf{I} \dot{\vec{\omega}} + \vec{\omega} \times (\mathbf{I}\vec{\omega}) = 0 (무중력 상태, 외부 토크 없음 프리 회전)을 풀면 무시무시한 축의 생존 법칙(Stability)이 결정된다.
어떤 책(스마트폰)의 각 관성축 크기를 I_1(\text{가장 김}) > I_2(\text{중간}) > I_3(\text{가장 짧음}) 이라 치자.

  • 안정 영역 (최대 관성축 I_1, 최소 관성축 I_3): 스마트폰의 가장 납작한 바닥(최대 관성축, 팽이 돌리듯)이나 가장 길쭉한 측면선(최소 관성축, 핫도그 굴리듯)을 잡아 돌려 위로 던지면, 핸드폰은 공중에서 회전축을 아주 얌전하게 쭉 유지하며 등속 스핀을 돌고 예쁘게 떨어진다.
  • 불안정성 악몽 영역 (중간 축 I_2): 폰을 가로로 잡고 뒤집기(플립, 중간 관성축)를 시전하여 던져보라. 돌던 폰은 갑자기 허공에서 반 바퀴 180^\circ 휙 뒤틀려 엎어지더니(\text{Tumble}), 다시 원자세로 복구하며 더럽고 끔찍하게 몸을 허우적댄다. (이것이 자니베코프 효과, Dzhanibekov Effect 와 테니스 라켓 정리의 본질이다).

\text{중간축 } \dot{\omega}_2 \text{ 방정식} \Rightarrow \text{쌍곡선 발산(Hyperbolic Divergence) 방정식 오차 폭주}

2. 각운동량 보존 구면 속의 장미 궤적(Rose Curve Locus) 파동

오일러가 꿰뚫어 본 회전 수학 기하학에서, 스핀하며 날아가는 폰(\vec{\omega} 벡터)의 뾰족한 끝(화살표 꼬챙이)이 허공의 3차원 투명한 구슬에 그리는 잉크 궤적(Polhode)을 살펴보라.
무중력 자이로 공간에서는 딱 두 가지가 무조건 보존되어야 한다.

  1. 총 각운동량의 크기 H^2 = (I_1 \omega_1)^2 + (I_2 \omega_2)^2 + (I_3 \omega_3)^2 = \text{일정 (가상 球 Surface)}
  2. 총 회전 운동 에너지 2E = I_1 \omega_1^2 + I_2 \omega_2^2 + I_3 \omega_3^2 = \text{일정 (가상 타원체 Ellipsoid)}

이 구면(Sphere)과 타원체(Ellipsoid)라는 두 개의 투명 껍질이 딱 포개져서 만나는 교차 선(Intersection Curve) 박음질이 곧 스마트폰 회전축이 타고 흘러야 할 영원한 궤도인데,

  • I_1 과 I_3 쪽 교차 궤적은 아주 매끈한 폐곡선 작은 원(Circle)이라서 폰이 떨리지 않고 안정을 유지한다.
  • 허나 I_2 (중간축) 부근의 교차 궤적은 두 타원체가 기괴하게 배를 찌르며 맞붙어 X 자 무늬 쌍곡선(Separatrix) 을 창조하며 분열된다!
