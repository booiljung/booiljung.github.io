3.3.1 회전체 동역학 (Rotor Dynamics)

증기 터빈, 제트 가스 엔진, 거대한 발전소의 발전기가 내뿜는 출력은 초당 수천, 수만 번의 스피드로 맹렬하게 돌아가는 중앙 강철 축(Rotor)의 회전 위상에서 시작된다. 하지만 철봉은 완벽한 강체(Rigid Body) 로 머물러 주지 않는다.
길다란 샤프트(Shaft)에 거대한 디스크 질량(Disk) 이 매달려 회전할 때, 그 스피드가 어느 한계선(임계 속도)을 뚫고 지나가는 순간, 샤프트는 자신의 탄성(Elasticity)과 질량 편심(Eccentricity)에 의해 스스로 휘어지며 줄넘기 줄처럼 공간을 후려치는 치명적인 자려 진동(Self-excited Vibration)을 일으킨다. 회전 기계가 파괴되지 않고 음속의 회전 위상을 뚫어내는 오토-밸런싱(Auto-balancing) 과 캠벨 선도(Campbell Diagram) 도해법의 정수를 다룬다.
