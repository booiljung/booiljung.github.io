3.3.1.1.8 회전축 균열 발생 시 강성 비대칭 개폐 호흡 효과로 인한 비선형 서브 하모닉

육중한 철제 샤프트가 10년, 20년 똑같은 진동 공진과 엄청난 열팽창 피로(Fatigue) 속에 혹사당하면 겉으론 흠집 하나 없어 보여도 축 내부 표면(Surface)에 미세한 실금(Crack) 이 쩍 하고 갈라지기 시작한다.
그런데 이 균열이 파여진 축이 계속 모터 스피드로 팽팽 돌 때 벌어지는 진동 파동은 너무나 해괴하고 끔찍하다. 바로 축이 1바퀴 돌 때마다 갈라진 균열의 입(\text{Crack Opening})이 딱! 벌어졌다가, 반 바퀴를 돌면 균열 입이 자기들끼리 끼리릭 부딪혀 다시 쓰읍! 하고 꽉 닫혀 버리는(Closing) 소름 돋는 “호흡(Breathing) 효과” 가 터지기 때문이다. 이 숨 쉬는 균열이 엔진 진동 FFT 그래프에 내뱉는 정체불명의 심령파동(Sub-harmonic)을 분석한다.

1. 개폐(Breathing)가 찢어놓는 비선형적 짝짝이 강성 박동(K)

만약 금간 데 없이 그냥 튼튼한 축이면, 모터가 뺑뺑 위아래로 한 바퀴 돌 동안 축의 휨 뻣뻣함(k) 은 어느 방향으로 구부리든 늘 일관된 동일한 숫자 k_0 를 유지한다(등방성 선형 응답).
하지만 축 표면에 균열(실금)이 깊게 파여 있다면 얘기가 180도 뒤집힌다!

  • 균열 바닥 처짐(Tension, 입 쩍 벌어짐):
    축이 빙글 돌다가 균열 부위가 아랫배 방향 쪽으로 오면, 축 자체의 중력과 휘어짐 파워 때문에 금이 쫙 벌어져 아가리를 쩍 벌린다(Crack Open). 이 순간 축에는 살이 파인 빈 공간이 생겼으므로 기계의 굽힘 강성 \mathbf{k} 가 후루룩 떨어져 엄청나게 물렁물렁(\Delta k 감소) 약해진다.
  • 균열 머리 올라감(Compression, 입 꽉 다묾):
    축이 다시 반 바퀴 빙글 제자리를 더 돌아서 균열 부위가 머리 꼭대기 쪽으로 솟구치면?
    축이 밑으로 휘어지는 곡률 때문에 위쪽 살점들이 서로 부딪치며 뭉개지는 압축(Compression)을 당한다. 즉, 갈라졌던 아가리가 “탁!” 하고 꽉 닫혀서 쇳덩어리들끼리 밀착해버린다! 이빨이 물려버렸으므로 축의 굽힘 강성 \mathbf{k} 가 다시 원래의 100\% 엄청나게 단단한 강철(초기 k_0) 상태로 딱딱하게 굳어 부활한다.

엔진이 1,000 \text{RPM} 으로 도는 내내 1초에 16번씩 강성이 물렁 \rightarrow 뻣뻣 \rightarrow 단단 \rightarrow 붕괴 를 반복하는 미친 심장 박동 변주곡 비선형(Non-linear) 진동파를 토해낸다!

2. 진단모니터 위의 유령 시그널: 2X 와 서브 하모닉(1/2X, 1/3X)

자, 이런 미친 짓을 겪는 균열 로터(Cracked Rotor) 에 진단 센서를 박아 주파수 그래프(예, FFT)를 띄워보면 멀쩡하던 기계에선 절대 안 보이던 악마의 뿔 파동들이 모니터에 자라기 시작한다.

  • 2X 고조파(Double Harmonic) 급등:
    축이 한 바퀴 도는 동안(1X) 균열이 열렸다 닫혔다(2가지 폼 변신) 하므로, 1회전에 “툭, 탁!” 두 번씩 쇳덩어리 강성 충격망이 생긴다. 그래서 정상 스피드 주파수의 꼭 2배 우측 자리 옆에, 2X (2배수 Harmonic 진폭) 뾰족탑이 미친 듯이 솟구쳐 오른다.
  • 수학적 살인: 서브 하모닉(Sub-harmonic Fractional)
    “그럼 스피드를 임계 속도 1/2 이나 1/3 지점(보통 안전지대) 으로 딱 깔아 놓으면 살지 않겠느냐?”
    호흡하는 미분 방정식 체계가 쏟아내는 가장 지독한 독극물이 바로 이 분수파다. 내 속도가 \text{RPM } 1,500 밖에 안 되어 안전지대 같은데, 균열은 내부적으로 2X(2회전 몫), 즉 자기 혼자 3,000\text{RPM} 짜리 진폭 에너지를 자가 증식시켜 기계 뼈 구조를 때린다! (Weight Dominance) 이 \mathbf{1/2X, 1/3X} 주파수 발작 찌꺼기 피크들이 모니터에 뜨기 시작하면?
    지금 축 어딘가가 소리 없이 부러지기 직전인 것이다. 바로 공장 발전을 중단시키고 크랭크를 버려야 할 대멸망 선고다.

터빈 공학에서 진동 진폭 치수가 커지는 것보다 훨씬 소름 돋는 재앙은, 어제까진 없었던 \mathbf{Sub-harmonic~(1/2X)} 나 2\text{X} 찌꺼기 주파수 피크들이 스펙트럼 바닥에서 좀비처럼 기어올라와 봉우리를 맺는 현상이다.
선형 스프링 공식(F=kx)의 아늑한 우주를 찢고 나와, 매 초마다 몸통 강성(K)의 매트릭스 값이 파열과 압축을 무한 반복하는 ‘호흡하는 진자(Breathing Pendulum)’ 의 세계! 이것이 거대 회전 로터 기계의 노후화와 피로 파괴(\text{Fatigue}) 균열이 던지는 마지막 오일러 미적분 경고장이다.
