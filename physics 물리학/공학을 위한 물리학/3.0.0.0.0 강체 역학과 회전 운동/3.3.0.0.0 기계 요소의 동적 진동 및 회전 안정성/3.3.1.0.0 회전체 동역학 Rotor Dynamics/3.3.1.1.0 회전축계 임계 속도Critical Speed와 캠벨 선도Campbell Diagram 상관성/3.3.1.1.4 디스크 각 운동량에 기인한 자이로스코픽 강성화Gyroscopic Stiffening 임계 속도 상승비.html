3.3.1.1.4 디스크 각 운동량에 기인한 자이로스코픽 강성화(Gyroscopic Stiffening) 임계 속도 상승비

앞선 수식들을 보면, 샤프트 파이프에 매달린 디스크(원판)를 그냥 무거운 쇳덩어리(질량 m) 핫도그 꼬치로만 계산하면 도저히 맞지 않는 기이한 현상이 엔진의 초고속 영역에서 터진다.
왜 축이 점점점 미친 듯이 속도를 울부짖으며 돌 때, 원래 터져야 할 임계 속도 공명 지점이 갑자기 안 터지고 한~~~참 더 높은 \text{RPM} 영역으로 마치 누가 잡아끌기라도 하듯 뒤로 도망쳐버리면서 기계가 튼튼해지는가?
바로 디스크 덩어리가 그냥 고철이 아니라 자기가 뱅뱅 도는 상태기 때문에 뿜어져 나오는 극도의 오일러 방벽, 즉 “나의 이 면 각도(수직)를 죽을 때까지 유지하겠다” 는 자이로스코픽 관성 고집, \mathbf{자이로 강성화(Gyroscopic Stiffening)} 효과 때문이다.

1. 회전 원반의 우주적 저항: 나를 비틀지 마라 (I_p 의 반항)

막대기 정가운데 커다란 판때기(극관성 모멘트 I_p)를 박아놓고 축을 돌리자(\omega).
그런데 축이 덜덜 떨리면서(Whirling \Omega) 가운데 부분이 활처럼 구부러지는 순간, 이 판때기도 평평하게 일자로 돌지 못하고 함께 고개가 꺾인다(\text{Tilt} / 각속도 발생).
자이로스코프의 오일러 법칙 \vec{M} = I_p \vec{\Omega} \times \vec{\omega} 에 의해! 이 각도로 자빠지려던 디스크는 그 자빠짐에 정면으로 맞서서 축을 원래의 빳빳한 일자수평() 모양으로 강제로 패서 되돌려버리려는 저항 텐서(M_{gyro}) 를 어마어마한 돌림힘으로 발동시킨다!

  • 공장이 멈춰 있을 때 샤프트 파이프 자체만의 물리적 쇳물 단단함(Bending Stiffness)이 k 였다면,
  • 기계가 돌아갈 때 디스크가 내뿜는 자이로스코프 저항력이 이 강성에 플러스(+) 보너스(가상 스프링)로 합산돼 버리는 것이다.
  • 즉, 축이 빨리 돌면 돌수록 \mathbf{\omega}, 이 자이로스코픽 굽힘 억제 저항력(Stiffening)이 비례해서 점점 더 미친 듯이 강해진다!

2. 임계 속도(Critical Speed) 의 강제 에스컬레이팅 상승 이동

고유 진동수 \omega_n = \sqrt{k_{\text{total}} / m} 의 분자 k_{\text{total}} (총 강성) 자체가, 기계가 돌면서 기하급수적으로 더 크고 단단하게 부풀어 오르기 시작한다!

\mathbf{k_{total} \approx k_{shaft} + (\frac{I_p \omega \Omega}{L^2})} \quad (\text{자이로 강성항 합산})

  • 원래 계산상으로는 \text{RPM} 이 이쯤 올라오면(\omega = 1000) 축 고유진동수 \omega_n (1000) 랑 쾅 부딪혀서 터졌어야 하는데!
  • 막상 \text{RPM} 이 1000 에 도달해 보니, 축의 총 강성 k 가 이미 자이로 팽이 효과로 너무 단단해져서 고유진동수 \omega_n 도자기 몸값을 불려서 1200 이란 꼭대기 숫자로 저 위로 치사하게 도망가(Shift) 버린 것이다.
  • 쫓아가는 모터 운송 스피드 \omega(빨간 선) 와 도망가는 커트라인 점 \omega_n(파란 공진 선) 의 숨 막히는 추격전. 자이로 효과가 이 파란 선 한계점 \omega_n 을 자꾸 위쪽 우상단으로 들어 올려(Stiffening Uplift Effect), 기계가 웬만해서는 공명 박살 파괴 구역에 잘 걸리지 않고 더 높은 하이퍼 퍼포먼스(안정영역) 를 버틸 수 있게 면죄부를 준다.
