3.3.1.1.2 샤프트(Shaft)의 굽힘 탄성과 질량 중심 편심(Eccentricity)에 의한 자려 진동(Whirling) 궤적

앞선 제프콧(Jeffcott) 모델에서 샤프트의 중심축이 밖으로 휘어지면, 그 중심점(Center of Mass) G 는 더 이상 직선 파이프 한가운데에 머물러 있지 않다. 축이 휘어진 채로 \text{RPM} 세탁기 탈수 모드마냥 10,000 번을 빙빙 돌면 어떻게 될까?
바로 휘어진 활대처럼 축의 배꼽 자체가 허공에 커다란 **“훌라후프 원의 궤적(Orbital Path)”**을 그리며 소용돌이치게 된다. 우리는 이 끔찍한 현상을 단순히 “진동(Vibration)” 이라 부르지 않고, 축의 기하학적 중심이 공중을 나선형으로 빙빙 떠도는 자려 진동(Whirling, 휠링) 이라 칭한다.

1. 1차원 상하 진동파(Vibration) 가 아닌, 2차원 평면 소용돌이(Orbit)

우리가 흔히 아는 자동차 서스펜션(스프링) 진동은 위아래로만 덜덜덜 흔들리는 1차원 직선(Y축) 변위다.
하지만 모터를 통째로 잡고 빙글빙글 고속으로 팽이치는 회전체 기계의 샤프트는, X 방향으로도 흔들리고 부드럽게 이어서 Y 방향으로도 깎여 흔들린다. 두 축의 흔들림이 합쳐지면 가로 변위 행렬 x = X \cos(\omega t), 세로 행렬 y = Y \sin(\omega t) 가 된다.

  • x^2 + y^2 = R^2 즉, 원의 방정식이 되어버린다!
  • 샤프트의 무게 중심(질량점 포인트 G) 덩어리 하나가 허공에 타원(\text{Ellipse})이나 커다란 훌라후프 동그라미(\text{Circle Orbit}) 를 빙글빙글 매 초마다 그리며 통째로 공간을 헤집고 휘젓는다. 이 공간적인 회전 요동을 휠(Whirl)이라고 한다. 샤프트는 자기 자신이 자기 몸통의 스핀 \omega 로 돌면서 동시에, 자기 척추뼈 전체가 하나의 거대한 나팔바지 나선을 수놓는 \text{Whirl} 각속도를 투페이스(2-Face) 로 미분 전개한다.

2. 굽힘 탄성 계수 k 의 저주 (유연 로터 설계)

이 무지막지한 훌라후프 소용돌이(\text{Whirling Orbit})의 크기 반경 R 을 얼마나 가둬죽일 수 있느냐는 바로 철근(샤프트)의 굽힘 탄성 계수(Bending Stiffness, \mathbf{k}) 에 달려 있다. 샤프트가 대나무처럼 낭창낭창 길거나 가늘면 파이프 굽힘 강성계 \frac{3EI}{L^3} 수치가 뚝 떨어져서(아주 작은 k), 휘어지는 진폭의 원 R 이 금방 허공을 때리며 베어링 껍데기(Casing)를 긁어버린다(소음 \uparrow, 파괴 \uparrow).

\mathbf{R = \left| \frac{m e \omega^2}{k - m \omega^2 + j c \omega} \right|}

  • 이 반경 수식에서 원심력 분자 m e \omega^2 이 커지더라도, 분모의 강성 \mathbf{k} (샤프트의 단단함) 가 엄청나게 무식하게 세고 크면(k >> m\omega^2) 휠링 반경 R 은 아주 미세한 1\text{mm} 언저리의 정적 점(Point) 에 묶여 안심 운전을 선사한다(Stiff Rotor, 강성축 전술).
  • 하지만 샤프트의 굽힘 강성 한계상 결국 회전 속도 \omega 가 올라가다 임계 진동(\omega_n) 을 만나는 순간, 식의 강성 반발력은 소멸되고 오직 공기 저항 점성 감쇠(\mathbf{c}) 하나만이 이 반경의 폭발(10\text{m} 나팔꽃 파동) 을 이 악물고 참아내는 동아줄이 된다.

수 세기 동안 공장의 기계공들은 기계가 덜덜 떨어 부서지는(1\text{차원} 진동) 것만 보고 바닥에 못을 세게 박으라 고함쳤지만, 그 시동 모터의 커버 속에서 돌아가던 철제 심장은 사실 미세하게 휘어져(Eccentricity e) 허공에 끊임없이 보이지 않는 원의 면적(\text{Whirl})을 그려내는 2차원 마법진 훌라후프를 돌리고 있었다.
이 휠링(Whirling)의 폭발을 막는 강철 파이프 굽힘의 강성(k= \frac{3EI}{L^3}) 탄성 지배에서부터, 공진을 파괴하고 저너머 무한대 극한의 고속 원점 회복지대로 도피하는 현대 터빈 동력 조형은 결국 \text{Rotor} 휘어짐의 위상각 반전(Phase Inversion 180^\circ)을 오묘하게 길들인 위상 공학이다.
