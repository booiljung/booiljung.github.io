3.3.1.1 회전축계 임계 속도(Critical Speed)와 캠벨 선도(Campbell Diagram) 상관성

회전하는 거대 기계, 즉 터빈이나 모터의 철근 축은 완벽하게 단단한 강체(Rigid Body)처럼 보이지만, 고속 회전이 시작되면 그 축은 하나의 탄성 스프링(Spring)과 진자(Pendulum)가 되어 맹렬하게 휘어지며 요동친다.
샤프트가 특정한 스피드(초당 회전수, \text{RPM}) 영역을 통과할 때 기계 전체가 산산조각 날 듯이 떨리는 끔찍한 공진(Resonance) 파동 구간을 만나게 되는데, 이를 임계 속도(Critical Speed) 라 부른다. 그리고 이 지독한 공진 파괴점이 회전 스피드(\omega)가 오를수록 자이로스코프 효과에 의해 미세하게 빗겨 나가 진화하는 “살아 움직이는 주파수 지도” 가 바로 회전체 동역학의 궁극 도해, 캠벨 선도(Campbell Diagram) 이다.
