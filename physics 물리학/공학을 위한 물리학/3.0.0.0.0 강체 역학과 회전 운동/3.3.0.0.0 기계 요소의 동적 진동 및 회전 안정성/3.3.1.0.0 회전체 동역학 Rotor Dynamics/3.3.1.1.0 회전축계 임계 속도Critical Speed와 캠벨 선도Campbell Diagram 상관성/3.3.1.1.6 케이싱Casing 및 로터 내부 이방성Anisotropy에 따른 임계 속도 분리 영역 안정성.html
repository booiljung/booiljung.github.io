3.3.1.1.6 케이싱(Casing) 및 로터 내부 이방성(Anisotropy)에 따른 임계 속도 분리 영역 안정성

책이나 이론상의 샤프트 모델은 완벽하게 둥글고 원통형(Symmetric)이다. 하지만 실제 공장에 깔린 거대한 압축기 샤프트를 보면 발전기 무게를 줄이려고 옆면을 깎아 평평하게 만들거나(2극 발전기 슬롯), 축이 놓인 지반(Pedestal) 조차도 위아래로 쿵쿵 찧는 단단함(K_y, 세로 기둥) 과 좌우로 흔들리는 물렁함(K_x, 가로 벌어짐) 이 태생적으로 짝짝이다.
이처럼 X방향 강성과 Y방향 강성이 똑같지 않고 다르게 태어난 기형을 진동 동력학에서는 이방성(Anisotropy, 비대칭성) 이라 칭하며, 이 짝짝이 강성이 자이로스코프와 만나면 캠벨 선도 위에서 1개의 평화로운 공진점마저 두 개로 박살 내며 광대한 ’불안정 영역 늪(Instability Zone)’을 찢어 발겨버리는 파괴의 근원을 탐구한다.

1. 2극(2-Pole) 로터의 태생적 단면 이방성: I_{xx} \neq I_{yy}

발전기 로터(축)는 구리 전선을 잔뜩 감아 넣기 위해 축의 양옆 살을 깊게 도려내 파어 낸 직사각형(타원형) 단면을 가진다.

  • 뾰족한 쪽으로 축을 구부리려 할 때의 강성(k_1)은 엄청나게 뻣뻣하게 저항하지만,
  • 살이 파여 넓적한 쪽으로 축을 구부리면(k_2) 너무나 물렁물렁하게 툭 꺾여버린다(k_1 > k_2).
    결국 이 축은 모터가 한 바퀴(1회전) 뺑 돌 동안, 뻣뻣 \rightarrow 물렁 \rightarrow 뻣뻣 \rightarrow 물렁 을 미친 듯이 박동하며 번갈아 겪게 되고, 기계의 고유 진동수 방정식에 시간 \mathbf{t} 가 섞여 들어가 버리는 악마의 마티유 방정식(Mathieu Equation) 을 연성시킨다. 이 시변(Time-varying) 방정식은 어느 한 곳에 묶이지 않으며 미친 듯이 진폭을 나팔꽃처럼 허공으로 끝없이 뿜어 키우는 불치병 안정성(Stability \, Loss) 파탄을 부른다.

2. 임계 속도 스플릿과 무한 발산 지옥의 개장

만약 기계가 대칭(Symmetry) 원형 축이라면, X 방향 진동 공진점이나 Y 방향 공진점이 \omega_n = 3,000 \text{RPM} 딱 한 군데서 똑같이 한 번 터지고 끝난다.
엔지니어는 “눈 질끈 감고 3,000 \text{RPM} 구간만 풀 악셀로 휘리릭 빛의 속도로 뚫고 지나가면 공명을 견뎌냈다!” 라며 안도한다(초임계 Passing 도주).

하지만, 강성이 짝짝이(K_x \neq K_y) 인 기계 부품(케이싱 또는 로터 살파임)이 껴 있다면?

  • X축(가로) 에서는 물렁하니까 약한 공진점(\omega_{nx} = 2,800\text{RPM}) 이 먼저 “쾅!” 터져서 축이 부르르 가로로 떨기 시작한다.
  • 그런데 곧바로 안심하기도 전에 Y축(세로-단단함) 공진점이 뒤이어 “쾅!” (3,200\text{RPM}) 하고 연달아 축을 위아래로 후려 팬다!

그 사이 구간, 즉 \mathbf{2,800\text{RPM} \sim 3,200\text{RPM}} 에 달하는 거대한 영역은 대체 무슨 일이 벌어질까? 그냥 축이 떨면서 1초 만에 지나갈 수 있는 점(Point)이 아니다. 수학적으로 이 영역은 고유 진폭이 음수 감쇠를 품고 무한대(Infinity) 로 거품처럼 부풀어 오르는 지수 파탄 영역 \mathbf{\text{Instability \, Region}} (불안정 늪) 이 되어버린다.

태생적인 비대칭(Anisotropy) 형체는 “임계 속도를 2개의 봉우리로 스플릿(Split) 시키는” 악랄한 주파수 찢기를 거행하며 공돌이들이 뚫고 지나가야 할 지옥의 거리를 넓혔다. 거기에 지구의 중력 처짐(Gravity \, Sag)이 2\text{X}(2배수) 리듬으로 가세하여 \text{RPM} 이 절반밖에 안 되었을 때 기계를 폭파시키는 서브하모닉(\text{Sub-Harmonic}) 암살까지.
결국 이 짝짝이 강성계는, 모터 속도 선(Line)상에서 아예 아무 데서도 운전하지 못하게 파탄 그래프 밴드(Band)를 색칠해버리는, 안정성 설계(Stability Analysis)의 마티유 한계점 철조망이다.
