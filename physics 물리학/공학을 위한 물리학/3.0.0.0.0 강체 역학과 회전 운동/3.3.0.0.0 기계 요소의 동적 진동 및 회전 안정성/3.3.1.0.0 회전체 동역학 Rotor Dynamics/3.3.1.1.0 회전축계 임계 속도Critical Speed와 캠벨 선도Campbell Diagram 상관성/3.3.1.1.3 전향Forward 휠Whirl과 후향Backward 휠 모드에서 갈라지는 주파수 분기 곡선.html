3.3.1.1.3 전향(Forward) 휠(Whirl)과 후향(Backward) 휠 모드에서 갈라지는 주파수 분기 곡선

샤프트가 고속으로 빙글빙글 돌면서 우주 공간에 커다란 훌라후프 궤적(Whirl 휠)을 그린다고 했다. 그런데 가만히 생각해 보면, 모터가 돌고 있는 방향(스핀 \omega)과 축의 기하학적 중심이 허공을 빙빙 도는 나선형 궤도 방향(휠링 각속도 \Omega)이 꼭 똑같아야 할 물리적 강박은 없다!
이 두 개의 회전 위상이 부딪히며 공학자들의 컴퓨터 그래프를 박살 냈던 두 줄기의 갈라진 진동수 분기 곡선, 전향 휠(Forward Whirl) 과 후향 휠(Backward Whirl) 의 역학적 분열(Split) 세계를 해부한다.

1. 전진하는 나선과 역주행하는 나선: 휠 모드의 양면성

샤프트가 시계 방향(\text{CW})으로 \omega = 1,000 \text{RPM} 스피드로 팽이처럼 돌고 있다 치자.
이때, 축의 중심(Center) 파이프 허리가 부웅부웅 훌라후프를 그리며 외벽을 타는 공조 궤적(Orbit) 속도 \Omega 의 방향이 문제다!

  • 전향 휠 (Forward Whirl, + \Omega):
    샤프트가 스핀하는 방향(\text{CW})과 똑같이 훌라후프(Orbit)도 시계 방향으로 휭휭 돌 때다. 우리가 흔히 엔진에서 무게 중심 불량(Unbalance, e)으로 덜덜덜 떠는 모든 진동은 99\% 이 전향 휠이다. 스핀 속도가 높아질수록 원심력과 관성이 한 방향으로 뻗치며 파도 타듯 사이좋게 밖으로 훌라후프 원을 키워나간다.
  • 후향 휠 (Backward Whirl, -\Omega):
    샤프트는 우루루루 시계 방향(\text{CW})으로 도는데! 그 파이프의 중심 궤적은 미친놈처럼 반시계 방향(\text{CCW}) 으로 역주행 훌라후프를 그리며 팽팽 돌고 있을 때다.
    “아니, 축이 우측으로 도는데 궤적은 어떻게 좌측으로 도나?”
    기계에 윤활유 마찰이 깨지거나(Dry Friction Rub), 베어링 케이싱 내부 강성이 상하좌우가 극심하게 짝짝이(x 강성과 y 강성의 이방성, Anisotropy)일 때, 축이 퉁- 하고 벽에 튕기며 물리적 역방향 나팔꽃으로 역진동 소용돌이를 빼기(-) 시작한다.

2. 주파수가 두 가닥으로 찢어진다 (Bifurcation of Frequencies)

이 두 가지 빙글거림(Whirl)의 방향 차이는 진동 주파수 방정식에서 자이로스코프(Gyroscope) 교차 결합 계수 \textbf{G (자이로 행렬)} 내의 부호를 정반대로 뒤집어 놓는다.
축이 제자리에 서 있을 때는 고유 진동수가 하나(\omega_n) 였는데, 모터 속도 \omega 가 쭉쭉 올라가기 시작하면 자이로스코픽 강성 텐서(Gyroscopic Stiffening)가 전향/후향 휠 모드의 분모에 각각 + 와 - 로 짝지어 다르게 붙어버린다!

\mathbf{\Omega_{forward} = \sqrt{\frac{k}{m - I_p/L^2}}} \quad (\text{갈수록 주파수 위로 등반 올라감})
\mathbf{\Omega_{backward} = \sqrt{\frac{k}{m + I_p/L^2}}} \quad (\text{갈수록 주파수 아래로 푹 꺼짐})
(단순화 비례식)

즉, \Omega 축 그래프에서 하나의 점(가만히 있을 때의 진동수) 에서 출발한 선이, 모터 속도 \omega(오른쪽 축)가 증가함에 따라 V 자 모양 혹은 나팔 모양으로 위아래 두 가닥으로 갈기갈기 찢겨(Split) 하나는 치솟고 하나는 가라앉는 분기 선도(Bifurcation Curve) 가 탄생한다.

엔지니어의 눈에 들어온 회전 기계 모니터에는, 속도를 올릴수록 진동수가 하나로 안 뭉치고 두 가닥, 네 가닥의 위아래 뿔뿔이 흩어지는(\text{Splitting}) 미지수의 세계가 그려진다.
오른 나사와 왼 나사가 궤도 기하 공간에서 오일러 각운동량 자이로 법칙을 만나 서로 밀고 당기는 강성 행렬 부호를 정반대로 뒤바꾸어 버린 대가, 즉 \text{Gyroscopic Matrix} [\mathbf{G}] 가 뿜어내는 비대칭(Skew-symmetric) 마력을 선형 렌더링으로 직관화시킨 가장 위대한 분기(Bifurcation)의 서곡이다.
