3.3.1.1.5 고차 진동 모드를 예측하는 캠벨 선도 맵핑 및 운전 속도 단면과의 교차 공진점 도출

비행기 엔진이나 화력 발전소 모터 축은 바이올린 기타 줄(String)과 같다. 줄은 처음 퉁길 땐 그냥 하나로 크게 휘청이는 기본 진동 모드(1st Bending Mode)를 가지지만, 더 세게 튕기면 한 번 안으로 접히는 S자 모양(2차 진동 Mode), \mathbf{N}자 모양 타율(3차 진동 Mode) 의 엄청나게 복잡한 파동 주름치기 모양으로 진폭이 갈라진다.
놀랍게도 기계 축의 회전 스피드(\text{RPM}, \Omega)를 무지막지하게 올려나가다 보면, 이 1차, 2차, 3차 고유 진동수 한계점들이 아까 말한 자이로스코프(Gyro) 효과를 처먹고 가닥가닥 다르게 위아래로 거미줄처럼 찢겨 휘어 도망치기(Splitting) 시작한다!
이 기괴하게 살아 숨 쉬며 여러 가닥으로 도망 다니는 ‘고유 진동수 무리(\omega_n)’ 들의 거미줄 변이 지도축을 칠판에 펼치고, 우리가 실제로 엔진 액셀레이터를 밟아 도달하려는 ‘내 모터 구동 스피드(Excitation Frequency \omega = \Omega 직선)’ 가 도대체 저 거미줄 가닥 중 어디 어디 와 박아치며 폭발 공진(Critical Resonance) 구역을 만드는지 찌르는 한 방의 차트, 그것이 회전 동역학의 신성한 성배 캠벨 다이어그램(Campbell Diagram) 이다.

1. 다중 진동수 곡선의 스플릿 Map (캠벨 도해)

그래프의 가로축(X)은 기계가 뺑뺑 도는 모터 스피드 속도(Rotor Speed, \Omega) 다.
세로축(Y)은 기계의 부위별 고유 공진 진동 파동 주파수(Natural Frequency, \omega) 다.

  • 가로축 X (모터 스피드) 가 0 \text{RPM} 인 세로 Y축 지점에는 1차 벤딩 주파수(\omega_1 = 50\text{Hz}), 2차 주파수(\omega_2 = 120\text{Hz}) 점들이 사이좋게 박혀있다.
  • 자, 엑셀을 밟아 X축 우측으로 모터 RPM \Omega 를 쫙 올려보자!
    그러면, \omega_1(50\text{Hz}) 점 하나에서 갑자기 두 가닥 실선이 \mathbf{<} 자 모양으로 쫙 찢어지며 퍼져 나간다! (위로 뻗는 놈이 자이로 강성을 처먹은 Forward 전향 휠, 밑으로 꺼지는 놈이 강성 빼앗긴 Backward 후향 휠이다).
    \omega_2(120\text{Hz}) 점에서도 두 가닥 선이 위아래로 넓게 퍼지기 시작한다.
  • 즉 모터가 빨라질수록(X축 우측), 세로축 방향(Y축)으로 뻗어나가는 거미줄 주파수 곡선 맵(Map)이 찬란하게 그려지게 된다.

2. 사형 선고: 동기 가진선(Synchronous Excitation Line, \mathbf{\omega = 1X \cdot \Omega}) 의 교차 칼날 결박

기계의 오차 불균형 무게 중심이 쿵쾅! 하고 축을 후려패는 박자는, 무조건 모터가 한 바퀴 돌 때마다 정확히 딱 한 번(1주기)씩 친다(Synchronous, 동기 가진 \mathbf{1X}).
즉, 내가 모터를 초당 100 바퀴 돌리면(X=100), 망치도 초당 100 번(Y=100) 내리친다는 소리다.

  • 이 규칙을 그래프에 그리면, X 와 Y 값이 똑같은 45^\circ 대각선 직선(\omega = 1\cdot\Omega) 이 캠벨 선도 원점에서 대각선 위로 무섭게 쫙 뻗어 올라가게 된다.
  • 이 \mathbf{45^\circ} 로 찌르는 엑스칼리버 대각선 칼날**이, 아까 앞에서 우상단/우하단으로 살살 거미줄 치며 도망가던 복잡한 곡선들(1차 \text{Forward}, 2차 \text{Backward} 곡선 등) 과 어느 속도(X) 상공 지점에서 크로스 교차 딱 맞물려 부딪히는지(Intersection) 가 이 도해의 전부다!
  • 그 교차점(Intersection \, Point) 좌표를 읽어 내리면? “아!! 우리 터빈 엔진은 \text{RPM 3,250} 지점에서 정확히 치명적인 1차 전향 임계 \text{Critical Speed} 가 터져서 폭발 하는구나! 그리고 여길 악으로 버티고 더 돌려 \text{RPM 8,100} 에 오면 이번엔 2차 후향 공진점에 박아서 축이 S 자로 터져 죽는구나!” 를 예측 진단해 낼 수 있다!

캠벨(Campbell)의 엑스칼리버 차트는 멈춰있는 고요한 부품 쪼가리 치수 도안만 봐서는 절대 알 수 없는 쇳덩어리들의 살아있는 파괴 비명을 모니터 \text{2D Plane} 안에 가둬버린 마법도구다. “속도가 오르면 기계의 본질이 변한다“는 각운동량 자이로 철학과 “스피드가 오차의 박자 진동수를 때린다(1X \text{Line})” 는 두 가지 지독한 물리 법칙 곡선방정식이 허공의 한 점(X_c, Y_c) 에서 교차해 키스할 때 로터 샤프트는 비명을 지르며 요절 박쇄된다. 이 교차점을 내 목표 운전 스피드(Operating \, Speed) 영역 좌우 위아래에서 피드백 강성 댐퍼 등으로 철저하게 빗겨내 지우는 행위(Resonance Avoidance)가 수조 원대 발전 터빈 동력계통 시스템의 최후 조율이다.
