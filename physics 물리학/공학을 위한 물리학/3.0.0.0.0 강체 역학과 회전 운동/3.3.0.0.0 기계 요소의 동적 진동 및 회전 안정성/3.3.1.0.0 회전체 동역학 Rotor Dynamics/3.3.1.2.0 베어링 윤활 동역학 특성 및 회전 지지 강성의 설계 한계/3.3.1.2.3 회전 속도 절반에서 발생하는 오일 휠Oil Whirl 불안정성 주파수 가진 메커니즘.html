3.3.1.2.3 회전 속도 절반에서 발생하는 오일 휠(Oil Whirl) 불안정성 주파수 가진 메커니즘

터빈 공장의 오퍼레이터들이 가장 심장이 덜컹거리는 기괴한 순간이 있다.
모터를 아주 부드럽게 시동 걸어 점점 속도를 올리고 있는데, 도대체 공진점(임계속도)에 가지도 않았고 무게 밸런스 점검도 백 점 완벽하게 깎아놨는데, 멀쩡하게 잘 도는 쇳덩어리 축이 갑자기 미친 듯이 텅!텅! 치며 허공을 빙빙 크게 돌기 시작하는 요동(\text{Instability}) 이 일어난다.
그런데 모니터를 보니 축이 덜덜 훌라후프를 떠는 주파수 속도(\Omega)가, 모터가 뺑뺑 도는 속도(\mathbf{\omega})의 “절반” 수준(약 \mathbf{0.4 \sim 0.48 \times \omega} 배) 에 딱 맞춰져서 박동하고 있었다! 부러지는 축도 없고 질량 차이도 없는데 도대체 어디서 이 절반(Half-speed) 이라는 미친 숫자의 유령 진동 파동이 터빈을 후려치는가?
이것이 그 유명한 오일 휠(Oil Whirl) 로터 파괴 진동 역학의 서막이다.

1. 펌핑 오일(Pumping Oil) 평균 속도 반토막의 기적

전 장에 봤듯, 축이 빙글 돌면 틈새 기름이 축 엉덩이 우하단의 아주 좁은 수렴 구간(Wedge) 쪽으로 콸콸콸 밀려 들어가 갇혀버리며 압력을 만든다고 했다.
그런데 그 비좁은 지옥 구멍에 갇혀 돌아가는 이 노란 오일 물방울 덩어리들의 속도를 보자.

  • 한쪽 면은 초스피드로 빙글 도는 축 표면(스피드 \omega) 이 밀어준다.
  • 반대쪽 면은 미동도 하지 않는 단단한 베어링 구멍통 쇳덩어리(스피드 0) 벽면에 찰딱 붙어 버틴다.
  • 오일은 이 사이를 채우고 흐르는 끈적한 유체!
    축에 붙은 오일은 \omega 로 달리고, 벽에 붙은 오일은 0 으로 뻗어 있으니, 이 오일 국물 덩어리 한복판이 평균적으로 밀려가는 속도는 \mathbf{(\omega + 0) / 2 = \omega / 2}, 즉 축 스피드의 “절반” 속도로 베어링 통 안을 둥둥 허공 훌라후프로 빙글 돌게 된다!

2. 유막 쐐기가 오히려 축을 밀며 추격하는 “오일 휠 빙빙 돌이”

이제 압박되어 수천 기압을 분출하는 그 부양력 덩어리 산봉우리(Wedge Pressure) 자체가 \frac{\omega}{2} 속도 로 축을 따라잡으려 빙글빙글 그 주위를 쫓아 돌고 있다.

  • 축의 엉덩이가 살짝 흔들려서 제 방향을 삐끗 벗어나면? 중심이 뒤틀린다. (Eccentricity 궤적 원 그림).
  • 이 뒤틀린 엉덩이 틈새 쪽으로 따라붙은 “반토막 속력의 기름 압력 봉우리” 가, 하필 엇걸림 결합 강성(\mathbf{K_{xy}} 펀치) 으로 축의 옆구리를 뒤에서 후려쳐서, 궤도를 한 칸 더 넓혀 버린다!
  • 밀린 축 엉덩이는 바깥쪽으로 한 발 더 벗어나서 더 큰 훌라후프 원 궤도를 그리며 속도를 낸다. 그래도 오일 봉우리는 또 \omega/2 의 무서운 속도로 쫓아와서 뒤에서 툭 밀어버린다 (Energy Pumping). 축은 도무지 도망칠 수가 없다 (Destabilizing Force).
    결국 축의 미세하게 떨렸던 한 점의 실수가, 오일 압박 봉우리가 영원히 뒤에서 밀어주는 그네 펌프질을 만나 기계 속도의 딱 0.48 배(절반) 주기로 미친 듯 소용돌이치며 진폭 훌라후프가 기하급수적으로 폭발 팽창해 베어링 보호막을 파괴해 버린다. 무한 진폭 발산(Instability), 오일 휠의 파멸이다.

엔진이 3,000 \text{RPM} 으로 도는데, 모니터에 1,450 \text{RPM} 위치에서 뾰족한 산봉우리가 솟구친다? 그것은 외부의 어떤 부러진 기어가 친 소리가 아니다. 엔진이 자기의 등 뒤에서 기생충처럼 살을 불려 나가는 미치광이 액체(유막 압력 쐐기 덩어리) 에게 스스로 몸뚱어리를 구타당하며 자살 파괴(Self-excitation) 로 나아가는 자려 진동(Oil Whirl) 통곡 소리다. 이 \mathbf{1/2 \times \omega} 에 버금가는 \text{Sub-synchronous} 주파수는 회전 동역학이 내딛는 악마성 텐서 행렬의 진면목을 과시한다.
