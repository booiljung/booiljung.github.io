3.3.1.2 베어링 윤활 동역학 특성 및 회전 지지 강성의 설계 한계

초거대 가스 터빈이나 핵 잠수함의 모터 축(Shaft)이 초당 수백 바퀴를 도는 동안, 그 쇳덩어리가 쇠 마찰 없이 영원히 허공에 떠 있을 수 있게 하는 기적의 받침대가 바로 베어링(Bearing) 이다.
과거의 공학자들은 베어링을 그저 “축을 잡아주는 단순한 쇠구슬 벽” 쯤으로 여겼다. 하지만 회전 동역학의 문이 열리자 진실은 달랐다. 베어링 틈새를 흐르는 점성 기름(오일) 막사나, 자성을 띠는 허공의 전자기장 틈새 자체가, 살아 숨 쉬며 거대한 ‘스프링(Stiffness)’ 과 ‘댐퍼(Damping)’ 로 돌변하여 모터의 목숨을 쥐고 흔드는 핵심 동역학 방정식의 주인이었다. 축을 지탱하려는 유체 역학과 축을 찢으려는 파괴적 진동(Whirl)이 피 튀기게 맞붙는 베어링 내부의 비선형(\text{Non-linear}) 세계를 파헤친다.
