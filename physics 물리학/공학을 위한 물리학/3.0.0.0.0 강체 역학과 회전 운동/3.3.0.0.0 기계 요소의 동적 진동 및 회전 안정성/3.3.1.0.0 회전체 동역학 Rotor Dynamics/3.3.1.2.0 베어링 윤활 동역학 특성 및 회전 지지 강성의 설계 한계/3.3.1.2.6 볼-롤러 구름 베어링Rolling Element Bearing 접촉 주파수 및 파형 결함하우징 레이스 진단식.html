3.3.1.2.6 볼/롤러 구름 베어링(Rolling Element Bearing) 접촉 주파수 및 파형 결함(하우징, 레이스) 진단식

오일이 수만 기압으로 떠받치는 유막 형상의 저널(Journal) 베어링 구조가 거대 터빈용이라면, 자동차 바퀴나 로봇 관절, 그리고 수백만 개의 전동 모터에 들어있는 것은 모래알같이 쇠구슬이 촘촘히 박힌 구름 베어링(Rolling Element Bearing) 이다.
매끈한 쇳덩어리 구슬(Ball) 이나 원통(Roller) 조각이 커다란 두 개의 금속 동심원(내륜, 외륜) 틈새에 끼어 무식하게 서로를 으득으득 짓이기며 굴러가는 이 구조는, 유막 부양의 조용함과는 거리가 멀다. 하지만 오히려 이 구슬들이 철컹철컹 돌아가는 그 불협화음(Vibration) 이야말로 쇳덩어리에 실금이 가거나 찌그러졌을 때, 수술 전 청진기처럼 기계 파멸의 부위를 칼같이 찍어내는 수학적 결함 진단(Fault Diagnosis) 파동의 성소가 된다.

1. 쇳덩어리 행성 구동계의 해부 - 내륜, 외륜, 전동체, 케이지

구름 베어링 속에는 마치 태양계 지동설처럼 돌아가는 4개의 우주 행성 기하학이 있다.

  1. 내륜(Inner Race): 모터 축에 딱 맞물려(Shrink Fit) 돌며 뜨거운 모터 속도를 자기가 다 떠안고 돌아가는 안쪽 태양 링.
  2. 외륜(Outer Race): 케이싱 구멍 껍데기에 붙어서 바깥 세계와 고정되어 평생 한 1의 회전도 없이 가만히 멈춰 서 있는(0\text{RPM}) 테두리 반지.
  3. 전동체(Rolling Element, 구슬): 내륜과 외륜 사이에 꽉 끼어 무지막지한 하중을 두 어부바 한 채, 그 안을 미친 듯이 빙글빙글 굴러 돌며 달리는 Z 개의 구슬 행성들.
  4. 케이지(Cage): 구슬들이 서로 부딪히고 깨지지 않도록, 구슬 사이에 일정한 거리를 맞춰 띄워주면서 구슬 무리와 함께 빙글빙글 느릿하게 큰 원을 도는 기차망.

기계의 떨림 측정 FFT 모니터 화면에 어떤 특정한 Hz 숫자의 높은 봉우리가 찍히면, 우리는 “아! 구슬 표면이 파였구나” 혹은 “내륜 링에 실금이 쫙 갔어!” 라고 기계를 뜯어보기도 전에 X레이 판독처럼 알아낼 수 있다.
각각의 쇠구슬 행성이 고장 부위를 밟고(Roll) “탕! 탕!” 치고 지나갈 때마다 만들어내는 그 일정한 타격 주파수(\text{Impact Frequency}) 방정식이 고정 불변의 숫자로 도출되기 때문이다.

2. 4대 결함 엑스레이 (Defect Frequency Equations)

만약 쇠구슬 직경이 d, 피치 원지름이 D, 접촉각이 \alpha, 축이 도는 속도가 f_r (\text{Hz}), 구슬의 총개수가 Z 개라고 치자.

  • [BPFO] 외륜 결함 (Ball Pass Frequency Outer):
    바깥 링(외륜) 에 금이 파여있다 치자. 돌고 있는 구슬 무리 Z 개가 차례차례 그 바깥 금을 “드르륵!” 밟고 지나가며 내뿜는 타격 주기 주파수다.
    \mathbf{BPFO = \frac{Z \cdot f_r}{2} \left[ 1 - \frac{d}{D}\cos\alpha \right]}
  • [BPFI] 내륜 결함 (Ball Pass Frequency Inner):
    축과 같이 돌고 있는 안쪽 링에 금이 갔다 치자. 이 금 밟힘 소리는 외륜보다 더 잦고 매섭게 때린다.
    \mathbf{BPFI = \frac{Z \cdot f_r}{2} \left[ 1 + \frac{d}{D}\cos\alpha \right]}
  • [BSF] 전동체 결함 (Ball Spin Frequency):
    구슬 자기 몸통 표면 자체에 살점이 뜯겨 나간 흉터가 난 상태다. 자전하며 도는 구슬 흉터가 밑 바닥(내륜) 을 치고, 다시 위 천장(외륜) 을 칠 때마다 내는 비명소리.
    \mathbf{BSF = \frac{D \cdot f_r}{2d} \left[ 1 - \left(\frac{d}{D}\cos\alpha\right)^2 \right]}
  • [FTF] 케이지 결함 (Fundamental Train Frequency):
    구슬들을 묶고 빙빙 도는 기차망 자체가 깨져서 덜컹덜컹 씹히는 달리는 주기파.
    \mathbf{FTF = \frac{f_r}{2} \left[ 1 - \frac{d}{D}\cos\alpha \right]}

저널 베어링 오일 휩이 물컹한 유체의 2\times2 강성 매트릭스가 뿜어내는 ’저주파 무한 발산 에너지 불안정’을 견디는 일이라면, 모터 축과 양끝을 조이는 구름 쇠구슬 베어링의 공학적 경계는 모래알과 같은 수많은 전동 쇠구슬 기하체가 \text{Hz} 라는 주파수 타임라인 위로 던져 올리는 아주 명확한 선형 충격 기하 구도(Kinematics) 다.
