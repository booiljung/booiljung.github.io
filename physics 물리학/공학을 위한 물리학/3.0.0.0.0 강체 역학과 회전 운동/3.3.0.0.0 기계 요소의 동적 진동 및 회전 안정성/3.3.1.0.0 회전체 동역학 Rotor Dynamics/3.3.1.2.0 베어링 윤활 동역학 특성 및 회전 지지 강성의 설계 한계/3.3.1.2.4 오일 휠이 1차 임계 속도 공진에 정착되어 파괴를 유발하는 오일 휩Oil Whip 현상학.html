3.3.1.2.4 오일 휠이 1차 임계 속도 공진에 정착되어 파괴를 유발하는 오일 휩(Oil Whip) 현상학

앞선 챕터에서 다룬 단일 오일 휠(Oil Whirl) 불안정 진폭은 축 속도(\omega)가 오르면 오를수록 정직하게 그 “절반(\cong 0.48 \times \omega)” 에 해당하는 주파수 숫자 \Omega_{whirl} 로 징그럽게 터빈을 쫓아다니며 커진다고 했다.
그런데 기계가 돌아가는 \text{RPM} 세팅 스피드(\omega)를 미친 듯이 더 올리면?
축을 때려 부수는 오일 휠 주파수 숫자 \Omega_{whirl} 도 그에 맞춰 비례하여 계속 쫓아 올라가는데! 갑자기 터빈의 태생적 근본, “1차 고유 진동수(1st Critical Speed, \omega_{n1})” 한계선과 딱 교차하여 맞붙는 무시무시한 충돌점이 찾아온다!
이 순간, 오일 휠은 더이상 \omega 의 절반 속도로 쫓아 올라가지 않고, 축 고유 진동수 \omega_{n1} 꼭대기 낭떠러지 바윗돌 자리에 아예 “안착(Lock-on)” 해버린다. 엔진을 아무리 광란의 높은 스피드로 돌려도 영원히 이 공진 주파수 자파괴(Self-excitation)에 고정되어 기계를 가루로 찢어놓는, 베어링 공학 최악의 흑화체, 바로 오일 휩(Oil Whip, 채찍 파괴) 이다.

1. 2배수 초임계 도주(Supercritical) 영역의 덫

보통 기계들의 1차 공진 임계 속도를 \omega_{n1} = 2,000 \text{RPM} 이라고 치자.
엔지니어는 당연히 이 2,000 지점을 살짝 풀 액셀로 피해, 안전하게 저 높은 4,000 \text{RPM} (2배 스피드인 짱짱한 초임계 구역) 에 엔진이 돌아가게 세팅해두고 안심하며 집에 자러 간다.

  • 자, 모터 속도 \omega = 4,000 이다. 베어링 틈새에서는? 절반 스피드의 기생충 오일 휠 \Omega_{whirl} = 1,950 \sim 2,000 \text{RPM} (약 절반 0.48 곱하기) 파동 덩어리가 생성된다.
  • 이 오일 휠이 가진 폭동 주파수(\sim 2,000) 가, 기계의 원래 1차 고유 뼈대(굽힘) 진동수(\omega_{n1} = 2,000) 와 미친 우연처럼 한 치의 오차도 없이 일치해 버린다! (Lock-on Resonance)

이건 단순한 회전체 불균형 덜덜댐이 아니다!
엔진 틈새 노란 국물 기름의 빙글빙글 도는 폭력이, 2\text{m} 짜리 강철 샤프트 뼈대 통째를 \text{2000 RPM} 고유 주파수 나팔꽃 모양으로 찢어버리도록 가진(Excitation) 시켜 기름과 철강 뼈대가 합작 시너지를 일으키는(Resonance Couping) 멸망이다!

2. 락-온(Lock-on) : 엔진 속도를 더 올려도 안 떨어지는 찰거머리 진동

“엑셀을 밟아! \text{RPM}을 5000, 6000으로 올려서 진동을 버리고 도망쳐!”
초보자 오퍼레이터가 속도를 확 올려버린다 치자(\omega = 6,000).

  • 원래 방식이면 오일 휠은 항상 절반을 쫓아가니 \Omega_{whirl} = 3,000 으로 도망가 올라갔어야 했다.
  • 하지만 진동 FFT 스펙트럼 모니터를 보면 피가 싹 식는다. 오일 휩(Whip) 에 걸려버린 주파수 피크는 안 올라가고 영원히 \mathbf{2,000(\omega_{n1})} 자리에 딱 들러붙어(Lock-in) 멈춰 서서 산처럼 증폭되고 있다.
  • 기계 속도 \omega 가 아무리 무한대 만(10,000) 스피드를 찍어도, 오일 휩 진폭은 영원히 고유진동수 \omega_{n1} 지뢰밭 늪에 알박기를 하고 쇳덩이를 채찍질(\text{Whip})하여 완전히 박살 난 폐철로 만들어 버린다! (The Whip Limit).

수식 상에서 무한대 주파수로 오일 휠을 키운다면 우주 밖으로 버릴 수 있겠지만, 로터 동력학계의 강성 매트릭스 \mathbf{K_{xy}} 는 “기계 1차 자아 고유 붕괴점(1st Critical Mode)” 과 만났을 때 그 에너지를 이탈시키지 않고 가장 자가 흡수가 잘 되는 거대한 종(Bell)처럼 징그럽게 부여안고 가둔다.
결국 엔지니어들에게 \mathbf{2\times \text{Critical Speed} (\omega_{n1})} 라는 속도 한계 지표는 절대 뚫고 지나가거나 안착해서 머무르면 안 되는 오일 휩의 \text{Red-Line} 성역이 되었다. 축은 가벼운 날개(훌라후프) 일 때 자려 진동을 맞이하고, 그것이 기둥 진동(Whip) 으로 굳어버리는 파국을 막기 위해 오직 형상 파괴 (Non-circular 베어링 깎기) 수술대에 오를 뿐이다.
