3.3.1.2.1 저널 베어링(Journal Bearing) 내 윤활유 레이놀즈 방정식과 동적 압력 분포 유체막 막두께

무게가 무려 100톤에 달하는 발전기 증기 터빈의 모터 축이 어떻게 구슬(Ball Bearings) 하나 없이 매끈한 쇠 구멍통(Journal) 속에서 허공에 둥둥 뜬 채로 매초 수백 바퀴를 도는 마법을 부릴까?
그 쇠붙이 구멍통 빈틈에 펌프로 콸콸 부어 넣고 있는 “노란색 기름(윤활유 오일)” 몇 방울 속에 기적이 숨어 있다. 기름의 미끌거림이 마찰을 없앤다? 그건 초보 역학이다.
100톤의 쇳덩이를 허공 1밀리미터 구멍 틈새 위로 영원히 띄워 올리는 진정한 무중력 마법은, 오일이 좁은 틈새에 억지로 빨려 들어가며 토해내는 어마어마한 비압축성 유체 폭발 압력, 즉 레이놀즈 방정식(Reynolds Equation) 의 수력학적 부양 점성 에너지(Hydrodynamic Lubrication)다.

1. 쐐기 효과(Wedge Effect): 깔때기에 낀 기름 점성의 분노

멈춰있던 엔진 축(Rotor Journal)은 무게가 무거워서 베어링 구멍통(Bearing Housing) 맨 밑바닥 철판에 찰딱 달라붙어(금속간 접촉 마찰) 주저앉아 있다.
축이 시동을 걸고 미친 듯이 시계 방향으로 돌기 시작하면, 축 표면에 미끈하게 뺨을 맞대고 있던 기름(Lubricant) 파티클 입자들도 마찰력 때문에 축을 따라 시계 방향으로 우수수 딸려 돌아간다.

  • 축 둘레 전체에 붙어 핑그르르 돌기 시작하던 오일 피의 강물은, 아까 쇳덩어리 축이 밑바닥 쪽으로 처져서 생긴 기가 막힌 모양의 비대칭 틈새(수렴 구간, Converging Wedge) 속으로 무지막지하게 빨려 들어가게 된다!
  • 들어갈 땐 틈이 넓었는데 파고들어 갈수록 틈이 좁아지는 깔때기 모양이다. 점성(Viscosity) 때문에 뒤로 도망치지도 못하고 강제로 좁디좁은 틈새로 엄청난 속도로 오일들이 강제 압사(Squeezed) 밀려 들어간다.
  • “나갈 길은 좁은데 기름은 자꾸 뒤에서 미친 듯이 들어온다!” 액체는 부피가 쪼그라들지 않는 비압축성(Incompressible Flow) 물체이므로, 오일의 폭동이 시작된다. 이 좁은 틈새에서 순식간에 수백, 수천 기압(Bar)의 초고압 수력 압력(Hydrodynamic Pressure) 이 쾅 솟구쳐 터진다!

2. 오스본 레이놀즈(Osborne Reynolds) 2D 편미분 방정식

이 좁은 골목길에 끼인 점성 윤활유가 벽면으로 뿜어내는 어마어마한 압력(P) 폭풍의 형태를 3차원 편미분 폭탄 체계로 쪼개 기계 설계 역사상 가장 경이로운 수식으로 남긴 이가 오스본 레이놀즈다.

\mathbf{\frac{\partial}{\partial x} \left( \frac{h^3}{\mu} \frac{\partial P}{\partial x} \right) + \frac{\partial}{\partial z} \left( \frac{h^3}{\mu} \frac{\partial P}{\partial z} \right) = 6U \frac{\partial h}{\partial x} + 12 \frac{\partial h}{\partial t}}

  • 수식의 꽃마차 \mathbf{h} (Film Thickness, 유막 막두께):
    구멍 벽과 회전축 사이 틈새 두께(h) 가 우변에서 x(길이) 에 대해 얼마나 급격히 좁아지느냐(기울기 \frac{\partial h}{\partial x}, 쐐기 효과) 가 이 모든 폭발적인 압력(P) 을 만들어내는 공장 원동력이다!
  • 이 우변에서 솟구치는 거대한 압력이 100톤짜리 축 엉덩이를 땅에서 번쩍 위로 들어 올려, 구멍 벽 허공(막두께 h) 에 자기 부상 열차 마냥 영원토록 무마찰로 둥둥 수중 부양을 시켜 버린다! (Load \, Carrying \, Capacity)
  • 우변 마지막 항(\mathbf{\frac{\partial h}{\partial t}} - 스퀴즈 Squeeze):
    축이 위아래로 쿵쿵! 진동하면서 벽면 틈새(h) 두께가 실시간으로 변할 때, 오일이 튕겨 나갈 틈도 없이 짜부라지며(Squeeze) 뱉어내는 타격 흡수용 유체 압력(초강력 워터 댐퍼 장벽)이다. 이 Squeeze 필름 항이 터빈 축 진동을 우아하게 빨아먹는 꿀항아리 점성이 된다.

엔지니어 정비공들이 아무리 공들여 쇠구슬과 철판으로 매끈한 링 통로를 짜맞춰도 무거운 축이 빙글 돌면 다 박살 난다. 터빈 기계를 지배하는 것은 오로지 한 방울 오일의 끈적임 점성율 \mathbf{\mu} (Viscosity) 이, 비좁은 마이크로미터 막두께 공간 \mathbf{h(x)} 에서 짜부라지며 오일러에게 발악하는 미분 반발 압력 \text{Reynolds Equation} 의 신묘한 유체 동력 파동이다. 허공에 편심을 가지고 비스듬히 뜬 축, 이것이 진동 동력학의 틈새 스프링 강성 텐서의 고향이다.
