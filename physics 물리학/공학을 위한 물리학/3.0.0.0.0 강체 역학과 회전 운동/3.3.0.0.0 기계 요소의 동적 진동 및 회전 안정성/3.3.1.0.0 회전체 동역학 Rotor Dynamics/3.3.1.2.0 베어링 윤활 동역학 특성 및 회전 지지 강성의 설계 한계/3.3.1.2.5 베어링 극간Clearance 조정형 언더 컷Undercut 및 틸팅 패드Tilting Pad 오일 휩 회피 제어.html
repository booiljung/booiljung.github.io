3.3.1.2.5 베어링 극간(Clearance) 조정형 언더 컷(Undercut) 및 틸팅 패드(Tilting Pad) 오일 휩 회피 제어

엔진 축이 미친 듯이 돌 때 부드러운 오일 쐐기막(Oil Wedge) 하나가 축을 죽음의 “오일 휩(Oil Whip)” 나팔꽃 공진으로 몰고 간다는 걸 전 장에서 배웠다. 그 파멸의 뿌리는 “오일 펌핑 압력이 한 방향에서 축을 밀면서 미끄러져 엇걸림 강성(Cross-coupled Stiffness K_{xy})을 낳기 때문“이었다.
그렇다면 터빈 공학자들은 이 수학적 악마를 어떻게 기계 설계 도면으로 때려죽였을까? 답은 놀랍게도 오일이 좁은 틈새를 억지로 타고 돌며 만드는 ’압력 댐(Pressure Dam)’의 기하학적 형태 자체를 드릴로 파괴해 버리는 “베어링 형상 깎기(Geometry Profiling)” 수술이다. 단순한 원통 구멍에 불과했던 저널 베어링이 타원형 언더 컷(Undercut)부터 피아노 건반 같은 틸팅 패드(Tilting Pad)로 쪼개지며 진화해 온 역학적 구배를 살펴본다.

1. 레몬 보어(Lemon Bore) 와 언더 컷(Undercut): 옆구리를 허물다

멀쩡하게 둥근 베어링 파이프 구멍통 안에 축이 돌고 있다면? 오일은 아주 긴 공간을 따라 빙빙 따라 돌며 쐐기 압력을 하나로 뭉쳐 키워나간다.
“아! 오일이 뭉쳐서 엄청난 덩어리로 축을 옆으로 툭 치니까 무한대(Instability) 휠링이 걸리는구나! 그럼 오일이 못 뭉치게 오일이 흐르는 통로(베어링 옆면)를 도끼로 넓게 깎아버리자!”

  • 레몬 보어(Lemon Bore): 베어링의 원형 구멍 좌우 양옆 살을 드릴로 도려내서 마름모꼴 레몬(Lemon) 모양 타원형으로 만들어 버린다.
  • 축이 돌면서 오일을 끌고 가 쐐기 폭압을 터트리려 하는데? 앗, 양옆 틈새가 뻥 뚫려 넓어버리니 압력이 찍 터져서 허공으로 다 새어나가 죽어버린다(Pressure Relief).
  • 오일 압력 지대가 상, 하 두 군데로 짧게 끊어지면서, 그 지독한 \mathbf{K_{xy}} 엇걸림 미는 힘이 급격히 약해진다 (Stability Threshold \text{RPM} 상승). 축이 수직 위아래로만 짱짱해지고 좌우 훌라후프 치려는 저주가 풀려 기계가 더 높은 속도에서도 울부짖지 않고 견딘다!

2. 궁극의 휩 방어선: 틸팅 패드(Tilting Pad) 베어링의 “무적 대칭 평형”

옆구리를 깎는 걸로 속도를 올리는 데 한계가 오자, 터빈 설계자들은 아예 통짜 베어링이라는 쇳덩어리 벽면 철학 자체를 갈가리 찢어버렸다. 원 구멍통 벽면을 피아노 건반처럼 4조각, 5조각의 조각 달(Pad) 쇳덩어리들로 쪼갠 뒤, 그 밑에 시소처럼 까딱까딱 움직일 수 있는 핀(Pivot)을 달아놨다.

  • 축이 고속으로 팽팽 돌 때, 오일은 이 5개의 조각 달 패드 밑으로 융단폭격처럼 깔려 들어간다.
  • 각각의 패드는 축이 밀어붙이는 압력과 마찰력을 느끼며, 자기 스스로 “까딱!(Tilt)” 거리고 핀을 중심으로 고개를 숙이거나 들어 축의 움직임에 맞붙어 저항한다 (Self-adjusting Pressure Wedge).
  • 이 “스스로 기울임” 역학 구조 덕분에, 각 패드마다 조그마하고 독립적인 미니 쐐기 압력 덩어리가 5개로 잘게 부수어져 형성된다! 오일이 축을 타고 한 바퀴 크게 돌며 에너지 펌프질하던 지옥의 순환 고리가 아예 절단(Severed)된 것이다.

수학적으로 틸팅 패드 베어링의 동역학 행렬을 꺼내보면 엔지니어가 환호성을 지르는 기적이 담겨있다.

\mathbf{K_{xy} = K_{yx} = 0}

그 끔찍했던 악마의 씨앗, 어슷걸림 가로지르기 강성(Cross-coupled) 텐서가 완벽하게 0 (\text{Zero}) 으로 수학적 소멸사망 해 버린다! 오직 수직으로 밭쳐 올리는 주 강성(K_{xx}, K_{yy})만이 패드 핀의 조율하에 절대 보존(Conservative Force) 역학을 이루어낸다.
