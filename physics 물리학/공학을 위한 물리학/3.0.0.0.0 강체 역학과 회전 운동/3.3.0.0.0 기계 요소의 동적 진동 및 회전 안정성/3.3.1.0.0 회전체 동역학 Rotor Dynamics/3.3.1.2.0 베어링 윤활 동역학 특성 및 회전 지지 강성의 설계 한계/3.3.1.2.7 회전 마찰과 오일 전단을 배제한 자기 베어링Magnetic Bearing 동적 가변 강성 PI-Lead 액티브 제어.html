3.3.1.2.7 회전 마찰과 오일 전단을 배제한 자기 베어링(Magnetic Bearing) 동적 가변 강성 PI-Lead 액티브 제어

“어떻게 하면 시속 수천 킬로미터로 도는 터빈 축을 단 하나의 쇳조각 마찰 점 하나 없이, 기름 한 방울 묻히지 않고 공중에 띄울 수 있을까?”
이 미친 발상을 이루어낸 것이 구리 코일 전자석의 마법, 능동형 자기 베어링(Active Magnetic Bearing, AMB) 이다.
자기 베어링은 축을 허공에 완전히 공중 부양(Levitation) 시켜 돌린다. 물리적인 고체 링이나 오일 윤활막의 기계적 스프링(K, C) 따위는 처음부터 이 우주에 존재하지 않는다. 대신, 자석이 당기는 전자기력(F_{mag}) 이 축을 붙잡는데, 거대한 문제는 자석이란 놈이 본능적으로 축이 조금이라도 가까워지면 미친 듯이 더 확! 끌어당겨서 쇳덩어리 벽에 쾅!(Negative Stiffness 본성) 하고 박치기시켜 죽여버리는 선천적 살인 스위치를 가졌다는 것이다. 이 악마 같은 강성에 마이크로 칩의 PID 피드백 수학을 주입해 부드러운 가상 스프링 강철 장막으로 연성하는 자기 비행 제어 공학의 코어 텐서를 열어본다.

1. 전자기력의 마이너스 강성(-K_s): 가까워지면 집어삼킨다

벽에 전자석 코일(N극) 이 있고, 공중에 자석 축 철덩어리(S극) 가 떠 있다고 치자.
자석의 끌어당기는 힘 F = \frac{\mu_0 \cdot N^2 \cdot i^2 \cdot A}{4 \cdot (g - x)^2} 는 거리(g-x) 가 가까워질수록 밑바닥 분모가 엄청나게 작아지면서, 당기는 힘 F 가 무한대 폭발로 솟아오른다.

  • 축이 아주 살짝 흔들려서 벽(자석) 쪽으로 다가갔다! (+x 변위)
  • 그럼 자석은 “오지 마!” 하고 밀어내서(복원력) 제자리에 놔둬야 하는데, 반대로 “앗싸!” 하고 아까보다 훨씬 더 세게 축을 자기 쪽으로 확 잡아당겨 버린다!
  • 한 번 궤도를 이탈하면 무조건 벽면으로 스스로 곤두박질 쳐 박아 박살 난다. 즉, 자기 베어링의 순수 강성 행렬은 탄성 스프링 파워가 기계 모가지를 부러뜨리는 마이너스 값(\mathbf{K_s < 0}, 음의 강성, 불안정 발산형) 을 태생적으로 가지고 태어난다. 기름 오일 휩(Whip) 따위와는 비교도 안 되는 순수한 빛의 속도의 즉각적인 파멸이다.

2. 디지털 마법결계: 광속 PID 피드백과 가상 강성(+K_i) 의 덧씌움

이 마이너스 강성(K_s) 을 때려잡지 않으면 터빈은 시동을 켜자마자 벽면 강철에 달라붙어 0.1초 만에 죽는다. 공학자들은 마이컴(컨트롤러) 을 꽂는다.

  • 벽에 센서(Gap Sensor) 를 박아둔다. “앗! 축이 자석 쪽으로 0.1\text{mm} 다가오고 있다!” 센서가 광속으로 외친다.
  • 제어기(Controller, PID/Lead-Lag): 센서의 비명을 듣자마자, “어 그래 다가와? 그럼 자석 구리선 코일에 흐르는 전류 i 를 확! 빼서 약하게 만들어!!” 하고 전류를 낮춘다(-\Delta i).
  • 자석 전류가 죽어버려서 자석의 끌어당기는 힘이 급격히 사라지고, 축은 당기는 힘이 없어진 만큼 다시 반대편으로 멀어지며 떨어져 중앙으로 돌아간다!

이 피드백(Feedback) 행위를 위치 x 마다 시시각각 전류 i 로 때려대는 수식 텐서를, 위치 비례 가상 강성(\mathbf{K_i}) 이라고 부른다. 제어기가 전류 i를 미친 듯이 변환하며 만들어내는 이 디지털 구름 K_i 전류 강성 수치가, 원초적 마이너스 악마 강성 K_s 의 힘을 훨씬 압도하여 찍어눌러버려(|K_i| \gg |K_s|), 결국 우리 눈에 보이는 “가짜 토탈 자기 스프링(K_{total} = K_i - K_s > 0 양수 강성 쉴드)” 으로 기계를 평화롭게 띄워버린다.

베어링의 동특성 파라미터를 기름(Fluid) 으로부터 벗어나 인간이 0과 1의 소프트웨어 알고리즘 코드로 프로그래밍(Programmable \, Stiffness \, \& \, Damping) 할 수 있는 시대가 자기 베어링 액티브(Active) 제어계망이다.
오일 휩(Oil Whip)도 없고 교차 강성(K_{xy}) 도 존재하지 않지만, 대신 센서의 위상 지연(\text{Phase Lag}) 과 고차 모드 진동을 무한대로 펌핑시켜 터뜨려 버리는 센서 스필오버(Spillover, 제어 누락 찌꺼기 폭발) 의 악마성으로 옮겨왔을 뿐이다. 수식으로 빚어낸 빛의 전류 댐퍼 조형물이 바로 \text{Magnetic Bearing PI-Lead} 공중에 떠 있는 멈춰진 자력의 산이다.
