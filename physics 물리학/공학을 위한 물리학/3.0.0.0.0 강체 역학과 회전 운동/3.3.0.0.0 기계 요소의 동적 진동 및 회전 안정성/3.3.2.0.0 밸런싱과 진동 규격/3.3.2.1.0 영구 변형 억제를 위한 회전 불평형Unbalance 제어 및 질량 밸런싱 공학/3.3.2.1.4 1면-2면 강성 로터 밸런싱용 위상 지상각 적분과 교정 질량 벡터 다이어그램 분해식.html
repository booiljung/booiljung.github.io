3.3.2.1.4 1면/2면 강성 로터 밸런싱용 위상 지상각 적분과 교정 질량 벡터 다이어그램 분해식

영향 계수법([a]) 이 아무리 똑똑하다지만, “아니, 컴퓨터가 알아서 C(교정납의 쪼가리 질량) 값을 15g 각도 230^\circ 에 달라고 뱉어냈는데! 내가 진짜 그 미지의 덩어리가 거깄는지 컴퓨터를 믿고 축을 뚫을 수가 있나? 내가 직접 도면에 각도기로 그림을 그려서 납땜 위치를 눈으로 찾아 박겠다!”
라며 아날로그 진동 전문가들이 종이 한 장과 자 하나로 축의 무거운 지점(Heavy Spot) 과 고진폭 지점(High Spot) 사이의 보이지 않는 미궁을 역추적하는 성스러운 의식, 벡터 다이어그램(Vector Diagram) 작도를 펼친다.

1. 헤비 스팟(Heavy) 과 하이 스팟(High): 위상 지연각(\alpha) 의 사기극

축이 돌아가는데 어떤 용접 똥(10g) 이 3시 방향 에 붙어서 축을 괴롭힌다고 치자. 이것이 진짜 무게 덩어리인 \mathbf{Heavy \, Spot} (무거운 놈 위치) 다.

  • 그러면 축 엉덩이가 훌라후프를 쓸리듯 회전할 때, 센서가 가리키는 파장이 가장 높게 봉우리를 솟구쳐 치는 진폭 최고점 \mathbf{High \, Spot} 은 당연히 3시일까?
  • 절대 아니다! 축을 돌리고 센서를 대보면, 무거운 놈(\text{Heavy}) 이 3시를 막 치고 지나간 뒤에 한참 뒤쪽 뒷북 위치(예: 12시 방향) 쯤에서 허공으로 불룩 하고 진동이 제일 크게 튀어나온다(\text{High}).
  • 엔진 축 파이프가 바로 튀어나오지 않고 감쇠(Damper 끈적임) 와 질량(관성 관성) 때문에 늦게 부풀어 오르는 지각 반응, 즉 위상 지상각(Phase \, Angle \alpha) 이 이 둘 사이를 어마어마한 각도 시간차로 찢어놓았기 때문이다.

센서가 잡아내는 “각도“는 단지 “진동 훌라후프 꼭대기(\text{High})” 일뿐, 거기에 무거운 불평형 납덩어리가 매달려있다는 보장이 1\%도 없다. \text{High} 를 보고 \text{Heavy} (진짜 범인) 위치를 도면 안에서 시계 방향, 반시계 방향으로 거꾸로 각도기 역회전시켜 찾아내는 작업이 1면 벡터 분해식이다.

2. 3번의 시운전 벡터선 연결 마법 (Vector Diagramming)

엔지니어는 종이에 십자 동심원을 그린다. (이게 축 전면 단면이다.)

  1. 최초 운전 점(Initial O 벡터): 처음에 젠 진동이 2\text{mm}, 30^\circ 에 터졌다. 중심에서 길이 2cm, 각도 30도 위치(1시) 로 화살표 하나를 그린다. 끝부분을 \mathbf{V_0} 점이라 부른다.
  2. 시험 무게(Trial Mass T) 부착 후 돌림 (O \rightarrow P 벡터): 내가 0도(3시) 부위에 조그만 시험 무게 클립 하나(T) 를 딱 박아두고 다시 기계를 윙~ 돌려본다.
  • 엇! 진동이 3\text{mm}, 110^\circ 로 변했다! (2번 화살표). 그림을 그리고 끝부분을 \mathbf{V_1} 점이라 부른다.
  1. 차분(Diff) 효과 벡터 작도(\mathbf{0 \rightarrow 1} 연결): 두 화살표 끝부분인 점 \mathbf{V_0} 에서 점 \mathbf{V_1} 으로 연결하는 기괴한 대각선 화살표를 긋는다.
  • 이 화살표가 바로! “내가 아까 붙인 시험 클립(\mathbf{T}) 하나만 순수하게 독자적으로 축의 진폭을 저렇게 바꿔놓았다!” 는 기적의 순수 효과 벡터(Effect Vector \mathbf{V_T}) 다! 길이는 4.5, 대각선 각도가 나온다.
