3.3.2.1.1 로터 질량 중심축과 회전 기하 중심축 간의 반경 오프셋 비대칭 정적 불평형(Static Unbalance) 구배

동전 두께처럼 얇은 오토바이 브레이크 디스크나 자동차 선풍기 팬을 들여다보자.
이 얇은 원반 모양 로터(Disk) 들은, 공장에서 주물을 뜰 때 구멍 하나가 더 막히거나 톱니 한쪽 면 철 찌꺼기가 1그램 이라도 더 많이 묻으면, 아주 명백하고 강력한 가장 기초적인 파괴 운동, 정적 불평형(Static Unbalance) 을 발작시킨다.
아주 단순하게 축(Shaft)의 겉 껍데기 기하학적 중심선(Center of Geometry) 과, 내부 질량 밀도가 뽑아내는 진정한 “질량 중심(Center of Mass)” 의 선이 나란히 가지 못하고, 중간에 살짝 \text{e (Eccentricity)} 만큼 옆으로 비끌어진 상태를 말한다. 이 원반이 축을 타고 돌기 시작하면, 축 전체가 바깥으로 팽창하려는 원통 원심력 망치가 되어 베어링 양쪽을 동시에 퉁! 퉁! 내리친다.

1. 정역학적 하방 처짐(추가 달린 타이어)

정적 불평형, 영어로는 Static (가만히 서 있는) 이다.
이름 그대로 기계를 모터로 돌리지 않고, 그냥 베어링 거치대(V블록) 위에 축을 올려놓고 가만히 손을 놓아보기만 해도 당장 육안으로 고장이 확인되는 원초적 불량이다.

  • 저 얇은 원판 한쪽에 눈에 보이지 않는 무거운 10g 짜리 밀도 뭉침(Unbalance Mass m_u) 이 숨어있다고 치자.
  • 축을 가만히 허공 가대에 놓아주면? 그 쇳덩어리 무거운 10g 질량이 있는 부분이 지구 중력(Gravity) 에 이끌려 스르륵~ 하고 바퀴가 자기 혼자 굴러가더니, 맨 밑바닥 6시 방향으로 향한 채 멈춰 덜렁덜렁 서버린다(오뚝이 코어).
  • 자전거 바퀴살에 물 묻은 양말을 걸어두고 바퀴를 들어보면 물 젖은 무거운 쪽이 스르륵 하고 밑으로 항상 같은 곳에서 멈춰서는 것과 똑같은 이치다.
    이처럼 회전 원심력 없이 그 자체의 무게 쏠림(오프셋)으로 지구의 중력에 반응하여 한쪽 무거운 쪽이 바닥을 향해 떨어지는 질량 편심, 이것이 가장 기초적인 한 덩어리 U = m_u \cdot r 스태틱 언발란스(Static Unbalance) 다.

2. 동위상(In-phase) 진폭: 베어링 2쌍의 동시 타격

모터 전원을 올려 \omega 속도로 이 축을 미친 듯이 돌려버리면 어떻게 될까?
그 반경 끝자락에 숨은 질량 폭탄 m_u 는 축이 빙글 돌 때마다 커다란 원심력 \mathbf{F_c = m_u \cdot r \cdot \omega^2} 을 발생시키며 축을 밖으로 강제로 구부러뜨려 훌라후프 나팔 모양으로 돌린다 (Cylindrical Mode, 원통 휠).

  • 이때 축 양쪽을 단단하게 붙잡고 있던 베어링 A(왼쪽) 와 베어링 B(오른쪽) 의 진동 모니터를 살핀다.
  • 축 전체가 마치 고무줄넘기를 팡 돌리듯 가운데가 불룩하게 부풀며 돌기 때문에, 무거운 m_u 가 위(12시)로 갈 땐 왼쪽 베어링 A와 오른쪽 베어링 B 둘 다 “위쪽“으로 당겨지고! 무거운 게 밑(6시)으로 오면 둘 다 동시에 “아래축“으로 내려찍힌다.
  • 즉 왼쪽 모니터 파동과 오른쪽 파동 곡선이 파도타기처럼 한 치의 오차도 없이 똑같은 타이밍(위상차 0^\circ) 으로 봉우리와 골짜기를 함께 쿵! 쾅! 내리찍는다! 이를 진동학 역학 기저로 동위상 타격(In-phase Vibration) 이라고 부르며, 이것을 해결하기 위한 1면 단면 교정(Single Plane Balancing) 역학 해가 펼쳐진다.
