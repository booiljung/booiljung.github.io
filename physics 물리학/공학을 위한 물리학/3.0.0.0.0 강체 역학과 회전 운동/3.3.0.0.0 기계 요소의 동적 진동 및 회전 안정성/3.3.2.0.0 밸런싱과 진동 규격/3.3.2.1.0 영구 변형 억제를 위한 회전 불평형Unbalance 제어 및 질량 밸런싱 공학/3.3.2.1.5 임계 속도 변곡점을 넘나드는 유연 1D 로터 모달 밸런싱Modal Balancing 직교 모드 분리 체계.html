3.3.2.1.5 임계 속도 변곡점을 넘나드는 유연 1D 로터 모달 밸런싱(Modal Balancing) 직교 모드 분리 체계

지금껏 다룬 밸런싱 방식들(1면 납땜, 영향 계수) 은 커다랗고 뻣뻣한 원통이 빙글빙글 돌 때, 그 쇳덩이 자체가 엿가락처럼 휘어지는 공진점(Critical Speed) 은 아직 만나지 않았다는 전제(Rigid Rotor, 강성 로터) 하에 계산되는 방식이다.
하지만 전투기 터빈이나 화력 발전기의 축은 길이가 무려 수 미터(\text{m}) 가 넘어가는 낭창낭창한 대나무 파이프다. 액셀을 밟아 스피드(\omega)가 올라가면 어느 순간 1차 공진을 만나 축 가운데가 훌라후프로 휘고(U자 모양), 스피드를 더 내면 2차 공진을 만나서 축이 S 자 모양(S자 곡선, Node 발현) 으로 갈기갈기 휘어져 버린다(Flexible Rotor, 유연 로터).
축 자체가 돌면서 기괴한 파도 뱀 모양으로 휘어버리는데, 대체 예전 방식처럼 양쪽 끝(2면 평면) 베어링 진동만 읽고 답을 찾을 수 있을까? 축 한가운데 껍데기에 붙은 찌꺼기 용접 똥(Unbalance mass)은 기계를 끄면 아무 일도 안 하다가 축이 S 자로 꺾이는 순간에만 뱀 허리를 미친 듯이 후려친다!
유연하게 휘어지는 \text{Flexible Rotor} 를 길들이기 위해, 각 휘어지는 S 자 타일 곡선(Mode Shape) 들을 하나씩 독립적으로 도려내어 밸런스를 맞추는 궁극의 역학 수술, 모달 밸런싱(Modal Balancing) 의 경이로움을 펼친다.

1. 모드 형상(Mode Shape) 과 진동의 직교성(Orthogonality)

수 미터짜리 축이 천천히 돌면 가운데 배가 불룩 튀어나오는 U 자 모양(1차 모드) 을, 속도를 미친 듯이 올리면 중간 한 점(노드, Node) 은 가만히 있고 양쪽 팔이 역상으로 흔들리는 S 자 뱀 모양(2차 모드) 을 겪으며 초고속 운전 스피드(Operating \, Speed) 에 도달한다.
그런데 자연계의 역학 기하에는 직교성(Orthogonality) 이라는 위대한 독립 선언이 존재한다.

  • “제1차 U자 모양으로 휘어지는 파동” 과 “제2차 S자 모양으로 휘어지는 파동” 은, 축 뼈대 공간 내에서 수학적으로 수직(\mathbf{90^\circ} 직교, 내적 값 = 0) 을 이룬다!
  • 즉, 내가 터빈 1차 공진 스피드에서 축을 U 자로 안 흔들리게 하려고 가운데에 납덩어리를 붙여 밸런싱을 맞췄다고 치자.
  • 스피드를 올려서 S자 모양(2차 모드) 뱀 진동 구역으로 들어갔을 때, 아까 1차 튀는 걸 잡으려고 붙여놨던 납덩이는, 이 S자 진동에 어떤 악영향도, 방해도 주지 않는다! (투명 인간 취급)
  • 서로의 모드(모양 파장) 스피드 영역에서 완전히 다른 나라 사람처럼 독립되어 버리는 기적!

이 미친 자연의 공식을 이용하면 유연 로터 밸런싱은 다음과 같은 “각개 격파(N면 N 모드 밸런싱)” 방정식으로 쪼개져 풀린다.

2. 모드별 각개 격파 (Modal Balancing Procedure)

  1. 1차 공진 탈출 (U 자 허물기): 엔진을 1차 고유 진동 속도(예: 1,500 RPM) 근처까지만 조심히 올려 돌린다. 축 가운데 배꼽이 크게 흔들리면, 그 “가운데(가장 많이 튀어나온 최댓값 부위)” 에다가 첫 번째 납땜을 떡 하니 붙여서(1면 밸런싱), U자 훌라후프가 아예 사라질 때까지 영점을 맞춘다. 이제 1차를 무사히 탈출할 수 있게 되었다!
  2. 2차 공진 탈출 (S 자 뱀 꺾기): 1차가 얌전해졌으므로 악셀을 더 밟아 3,000 RPM 2차 스피드로 과감히 엔진을 올린다! 그랬더니 이번엔 S 자로 춤추며 양쪽 끝단이 지랄발광을 한다 (그런데 아까 1차 잡으려고 붙여논 납은 S자의 정중앙 Node 즉 축이 안 흔들리는 지점(가만히 있는 점)이라서 아무 영향도 안 줌!).
  • S자 폭발을 잡지 위해서는 양 끝 2군데(혹은 Node를 피한 절반 위치 2곳) 에다가 짝꿍 우력(Couple)처럼 두 번째 교정 납(C_2, C_3) 들을 부착하여 S 자만 맞춤형 핀셋 2면 밸런싱으로 지워버린다.

수십 장의 터빈 날개가 각자의 무게 편심(Unbalance) 을 가지고 돌아가는데, 그것이 각 공진 지대를 뚫을 때마다 엇박자 파동을 하나씩 던지며 뱀처럼 똬리를 튼다.
모달 밸런싱(Modal Balancing) 은 이 모든 오차 질량들을 다 지우려 들지 않는다. 오직 “1차 훌라후프 파동을 불러일으키는 나쁜 녀석들의 \mathbf{1차 \, 직교 \, 성분}” 들의 합만 딱 그 스피드에서 묶어서 1번 부숴버리고, 그 위로 더 도망가서 “2차 S 파동을 지랄 나게 하는 \mathbf{2차 \, 직교 \, 성분}” 만 묶어서 한 번 더 부수어 버린다. 모든 것을 알 필요 없이, 공명선(Critical Speed)이 스스로 토해내는 독립 파동(Eigenvector) 만을 족집게로 찍어 파괴하는 유연 기계 선형대수의 아름다움이다.
