3.3.2.1.2 커플(Couple) 베어링 진동이 역상(+180°/-180°)으로 나타나는 모멘트 불평형 우력 발생 역학

앞서 얇은 동전 두께의 팽이에선 무게 쏠림이 1개(m_u) 라서 축이 원통처럼 배가 빵빵하게 부르르 떨리는 정적(Static) 불평형이 생겼다.
그런데 이번엔 동력이 걸리는 기계가 얇은 팽이가 아니라, 엄청나게 크고 기다란 원기둥 소시지 모양의 거대 실린더 로터(예: 종이 공압 롤러나 전동 모터 몸통 코어) 라고 치자.
이 기다란 원기둥의 맨 왼쪽 윗부분(12시 방향) 에 철 찌꺼기 용접 똥(10g)이 엄청 무겁게 묻어버렸다. 그리고 난데없이 맨 오른쪽 끝부분 밑바닥(6시 방향) 에 똑같이 철 찌꺼기(10g) 가 돌연변이처럼 묻어버렸다고 가정하자!
서로 완벽하게 반대 방향(12시와 6시)에 똑같은 무게(10g) 불평형 덩어리 두 개가 축 양 끝을 잡고 돌어간다. 이게 어떤 미친 파괴를 낳을까?

1. 정적(Static)으론 완벽히 균형 잡힌 기묘함

기계를 멈추고 축을 V자 선반 허공 가대에 가만히 올려놔 보자.
놀랍게도 아까 얇은 디스크 팽이처럼 밑으로 빙글 하고 굴러떨어지는 무거운 처짐 사태가 발생하지 않는다!
왜냐고? 왼쪽 끝에 박힌 12시 윗방향 덩어리 10g 이 위에서 버티고, 오른쪽 끝에 박힌 6시 방향 10g 이 아래서 당겨서… 질량 중심이 절묘하게 “정가운데” 0점에 제로(0) 밸런스가 잡혀버리기 때문이다!
정지 상태에서 보면 “아, 이 엔진 축 완벽하게 50:50 밸런스 잘 맞네!” 하고 눈가림으로 엔지니어를 속여 넘긴다. 합력 벡터 \mathbf{\sum F = 0} 가 되어 정적 불평형은 수식적으로 0 처리가 된 것이다.

2. 모터를 켜면 터지는 파멸적 우력(Couple): 180도 엇갈림의 박동

하지만 이 거대한 사기극 로터에 전기코드를 꽂아 수천 \text{RPM} (\omega) 스피드로 돌려버리는 순간 무서운 지옥이 열린다.

  • 돌아가기 시작하면 왼쪽 끝에 있는 12시 덩어리는 어마어마한 원심력 망치(F_1) 가 되어서 축 왼쪽 모가지를 위로 뜯으려 쾅 쾅 패대기친다!
  • 그 찰나! 똑같은 순간에 오른쪽 끝 6시 방향에 숨어있던 똑같은 무게의 덩어리도 미친듯한 원심력 망치(F_2) 로 돌변해 오른쪽 축 엉덩이를 정반대인 밑으로 확 쥐어뜯어 끌어 내린다!

왼쪽은 위로 쳐올리고, 오른쪽은 밑으로 처박는다. 즉, 두 원심력이 합쳐져서 축 전체를 하나의 커다란 프로펠러 헬리콥터처럼 가로로 냅다 “비틀어 꺾어버리는” 거대한 회전 돌림힘(모멘트 토크 \mathbf{Moment \, Torque}, M_c = F \cdot L) 을 발작시킨다!
축 전체가 나비넥타이나 8자 모양(Conical Mode) 으로 허공에 원추형 나팔을 좌우로 빙빙 엇갈리게 털며 돌기 시작한다. 우리는 이것을 두 짝꿍(커플) 의 힘이 낳은 돌림힘 불평형, 즉 우력 불평형(Couple Unbalance) 이라 부른다.
