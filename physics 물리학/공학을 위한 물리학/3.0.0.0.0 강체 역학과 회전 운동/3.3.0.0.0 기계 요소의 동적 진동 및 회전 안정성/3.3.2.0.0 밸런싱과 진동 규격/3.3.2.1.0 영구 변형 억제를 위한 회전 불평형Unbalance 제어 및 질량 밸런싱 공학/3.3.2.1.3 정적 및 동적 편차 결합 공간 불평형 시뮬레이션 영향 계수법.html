3.3.2.1.3 정적 및 동적 편차 결합 공간 불평형 시뮬레이션 영향 계수법

아까 본 정적 불평형(1개 오프셋) 과 우력 불평형(정반대 2개 오프셋) 은 그저 교과서에서나 볼 법한 이분법이다. 현실 산업 현장에 깔린 가스 터빈이나 송풍기 팬들은 이 두 가지가 미친 듯이 뒤섞인 동적 불평형(Dynamic Unbalance) 이라는 끔찍한 우주 짬뽕 상태가 기본이다.
축의 어딘가는 20g 더 무겁고, 어딘가는 5g이 모자라며 방향도 30^\circ, 150^\circ, 250^\circ 사방팔방으로 제멋대로 무겁다!
“그럼 어디에! 얼마나 무거운 납덩어리(교정 질량)를 달아야 저 진동 파동을 상쇄시킬 수 있을까?”
이 문제를 풀기 위해 엔지니어가 수학의 꽃, ‘행렬 방정식’ 을 소환해 축의 어떤 곳에 살짝 납을 매달아 보고(“툭!”), 그게 축 진동에 얼마나 큰 영향을 줬는지 알아내는 마법, 영향 계수법(Influence Coefficient Method) 을 해부한다.

1. 알 수 없는 불평형 덩어리를 2개의 가상 단면(Plane) 에 몰아넣기

긴 축 전체에 떡볶이 국물 튀듯 사방에 흩뿌려진 미세한 철 찌꺼기 덩어리(Unbalance Mass 100개) 를 우리가 칼로 다 긁어낼 순 없다. 동력학의 위대한 1법칙은 “아무리 복잡한 긴 쇳덩어리 불평형(Dynamic) 도, 축 양쪽의 오직 두 개의 평면(Plane) 안에 납덩어리 딱 2개를 계산해 박아 넣으면 세상 모든 원심력 파동을 \mathbf{0}으로 소멸킬 수 있다!” 로 선언된다.

  • 미지수 2개: 그래서 엔지니어는 축 좌측 모서리 평면(\text{Plane 1}) 에 달아야 할 납덩어리의 위상과 무게 \mathbf{W_1}, 축 우측 모서리 평면(\text{Plane 2}) 에 달아야 할 \mathbf{W_2} 라는 \text{Vector(무게, 위상각)} 2개의 해답만 찾으면 끝난다.
  • 측정 2개: 베어링 양 끝단 A, B 에 달아놓은 진동 센서가 지금 쿵쾅! 하고 찍어내는 현재의 개판 오분 전 진동 벡터 \mathbf{V_A}, \mathbf{V_B} 를 컴퓨터로 읽어낸다. (1 \text{ mm/s} 진폭, \angle 30^\circ 방향).

2. 넌 뭘 넣으면 얼마나 튀니? - 영향 계수 행렬 [\mathbf{\alpha}] 찾기 도발

자, 공식 초기 진동 행렬 {V} = [\alpha] \times {W_알 수 없는 불량 찌꺼기 결합 덩어리}
이제 저 \alpha (Alpha) 알파 매트릭스를 알아야 저 복잡한 쓰레기 무게를 알 수 있다. 이 \alpha 매트릭스는 “기계 평면 1번구멍에 내가 임시로 1\text{g}짜리 시험용 구슬을 박아봤을 때, 베어링 모니터의 진동이 얼마나 확 변하는가?” 를 알아내는 수치다! 즉 기계의 개별 자극 반응도(Influence Coefficient)다.

  • 시험 주행(Trial Run):
    엔지니어가 축을 세운 뒤 1번 평면에다 아는 무게(Test 질량 T_1) 하나를 몰래 박아놓고 엔진을 우왕~ 하고 한 번 돌려본다.
  • “오! 1번 구멍에 달았더니 A 센서는 3 만큼 더 떨고, B 센서는 1 만큼 변하네!” 그 비율값(벡터 차분)이 바로 \mathbf{\alpha_{11}, \alpha_{21}} 이 된다.
  • 다시 세우고, 2번 구멍에 T_2 시험 무게를 박고 엔진을 다시 켜본다! “B 센서는 5 만큼 변하네!” 그게 바로 \mathbf{\alpha_{12}, \alpha_{22}} 이다.

이 시험 도발 2번으로, 나의 이 고귀한 모터 기계가 몸통 어딘가에 이물질이 들어왔을 때 각 베어링 센서로 어떻게 지랄발광을 떠는지를 완벽하게 2\times2 [영향 계수 행렬 \alpha] 공식으로 포박(Mapping) 했다!
