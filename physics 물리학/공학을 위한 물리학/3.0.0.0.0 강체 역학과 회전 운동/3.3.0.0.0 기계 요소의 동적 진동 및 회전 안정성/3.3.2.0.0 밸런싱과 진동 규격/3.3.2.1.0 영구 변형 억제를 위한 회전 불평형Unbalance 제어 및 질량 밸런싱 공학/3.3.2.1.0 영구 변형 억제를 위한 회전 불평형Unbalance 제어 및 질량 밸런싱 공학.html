3.3.2.1 영구 변형 억제를 위한 회전 불평형(Unbalance) 제어 및 질량 밸런싱 공학

모든 기계 진동의 80\% 이상을 차지하는 만악의 근원, 불평형(Unbalance).
샤프트 통나무 자체의 밀도 불균일, 톱니바퀴 조립 시 핀이나 나사구멍 하나가 더 파여서 생긴 수십 그램(\text{gram})의 편차. 이 작은 먼지 조각만 한 무게 쏠림이 \text{3,000 RPM} 이라는 제곱 스피드의 원심력 날개를 다는 순간, 수 톤(\text{Ton}) 급의 거대 해머가 되어 축을 강제로 휘어 꺾고(Bending), 영원히 펴지지 않는 소성 변형(Plastic Deformation)과 피로 파괴를 자행한다.
엔지니어가 이 쇳덩어리 속 숨겨진 보이지 않는 편심 덩어리(U = m \cdot e) 의 위치와 양을 레이더처럼 정확히 포착하여, 그 반대편에 납덩이를 박아 귀신처럼 파동을 상쇄 시키는 밸런싱(Balancing) 역학의 1면, 2면 벡터 분해 방정식부터 유연 모달(Modal) 밸런싱 튜닝까지 거대 기하학 수술실을 해부한다.
