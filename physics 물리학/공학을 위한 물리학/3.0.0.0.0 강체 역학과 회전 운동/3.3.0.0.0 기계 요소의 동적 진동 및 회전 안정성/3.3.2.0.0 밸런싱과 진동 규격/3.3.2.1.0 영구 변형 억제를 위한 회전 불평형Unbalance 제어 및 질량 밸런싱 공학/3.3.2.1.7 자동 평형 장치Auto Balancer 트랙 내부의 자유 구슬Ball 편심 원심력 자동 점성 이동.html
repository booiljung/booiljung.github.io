3.3.2.1.7 자동 평형 장치(Auto Balancer) 트랙 내부의 자유 구슬(Ball) 편심 원심력 자동 점성 이동

수만 \text{RPM} 으로 돌아가는 \text{CD/DVD-ROM} 드라이브 장치나, 무거운 빨래 더미가 이쪽저쪽 뭉쳐서 미친 듯이 뺑뺑 돌아가는 세탁기 탈수통을 보자.
빨래가 뭉친 위치가 매번 다르고, \text{CD} 의 오차도 매번 다른데(불평형의 무작위 랜덤 펌핑), 기계 엔지니어가 세탁기 안에 들어앉아서 매번 납덩어리를 붙였다 뗐다 밸런싱을 맞출 수는 없다!
기계가 알아서 스스로 자기 뼈대의 불평형 덩어리 무게 중심을 찾아서! 무거운 쪽 반대편 골짜기로 납덩이(구슬) 들을 스스로 슈루루룩 카오스 점성 굴리기 이동을 시켜 무한의 \mathbf{Zero \,\, 밸런싱} 을 유지해 버리는 마법, 볼형 자동 평형 장치(Ball Auto Balancer) 의 기하 궤도학(Orbit) 을 파헤친다.

1. 맹수(Unbalance) 와 구슬(Ball) 의 초임계 도주 본능

자동 평형 장치의 뼈대는 허무할 정도로 단순하다.
기계(로터)의 가장자리에 동그란 레일 홈통(Race Track) 파이프를 빙 둘러서 파놓고, 그 파이프 속에 조그마한 쇳덩어리 구슬(Ball) 들을 몇 개 집어넣고는 윤활유(기름) 를 살짝 묻혀 채운 뒤 밀봉해 버린 게 다다.

  1. 저속/가속기(Sub-critical Speed):
  • 이제 빨래(무거운 불평형 덩어리 U) 가 한쪽에 뭉친 세탁기 코어를 돌리기 시작하자.
  • 기계가 느리게 돌거나, 고유 공진점(\omega_n) 을 넘기지 않았을 때는 축의 진동 위상 파동(Phase Angle) 이 원래 무거운 똥 덩어리(Heavy \, Spot) 랑 똑같은 위치에 머문다.
  • 이때 파이프 속의 쇠구슬들은 어떻게 될까? 원심력을 받아서 구슬들도 홈통을 타고 가장 바깥쪽으로 빙빙 도는데, 하필 원래 무거운 빨래 덩어리가 뭉친 곳 옆구리에 구슬 덩어리들마저 옹기종기 미친 듯이 다 같이 뭉쳐버린다!! (동위상 구심력 현상)
  • 빨래도 무거운데 구슬까지 그 옆에 뭉쳐서 가세하니 불평형이 2배로 수백 \text{kg} 커져서, 시동 초반엔 탈수기가 미친 듯이 “쾅쾅쾅쾅!!” 세탁실 바닥을 부술 듯이 요동치며 엔진 발작(Aggravating Unbalance)을 한다. (그래서 세탁기 탈수 시작할 때 쿵쾅 소리가 크다.)
  1. 초임계 돌파와 기적의 구슬 도망 (Super-critical Speed):
  • 세탁기 엔진이 “에이씨 다 죽어라!” 하고 무식하게 \text{RPM} 을 확! 올려서 위험한 공진점 스피드를 뚫고 초고속(Super-critical 구역) 에 올라서게 만들면 진정한 오일러 뉴턴의 마법이 터진다.
  • 훌라후프 파동(위험 위상 위상차 각도 \mathbf{\alpha}) 이 아까는 무거운 놈과 같은 0^\circ 에 있다가, 공진점을 넘기는 순간 픽! 뒤집혀서 \mathbf{180^\circ} 정반대쪽 계곡으로 위상이 완전 역전 반전(Phase Inversion) 해 버린다!
  • 쇳덩어리 구슬들은 훌라후프가 흔들리며 바깥으로 가장 크게 밀어내는 궤적(High Spot) 쪽으로 원심력 때문에 도망가서 미끄러지려 하는데, 이 훌라후프(High) 위치가 무거운 빨래 덩어리(Heavy) 와 \mathbf{180^\circ} 정반대편 이 되어버렸다!
  • 구슬들이 홈통 안 기름을 뚫고 촤라라락~ 빨래 반대편 우주 꼭대기 위치로 미끄러져 가서(Self-Aligning) 자기들끼리 예쁘게 주차 정렬해 버린다!! 빨래 덩어리(12시) \Leftrightarrow 구슬들(6시).
  • 구슬의 질량 합(카운터 밸런스 역벡터) 이 정확하게 빨래의 수백 \text{kg} 흔들림과 1:1 싸움으로 맞붙어, 쾅쾅 치던 기계 진동이 거짓말처럼 \text{Zero} 바닥으로 0\text{mm} 고요해지는 소멸. 이것이 오토 밸런서(자연치유) 다.

수많은 찌꺼기와 결함 오차 질량이 \text{Disk} 원반 한 구석에 몰려 기계 척추를 부수려 할 때(Agitating), 거대한 납덩이를 손수로 땜질해주던 엔지니어의 자리를 레일 트랙 하나와 쇠구슬 세 알이 완벽하게 낚아채어 빼앗아 버렸다.
이것은 미분방정식이 증명한 자연 기하 역학계의 \mathbf{Phase(위상)} 가, 임계 속도라는 산봉우리를 기점으로 \text{+부호에서 -부호($180^\circ$ 역상)} 로 뒤바뀌며 스스로 가장 멀리 도망가 정착하는 성질(\text{Minimization of Energy Orbit}) 을 극강으로 이용 해먹은 튜닝 예술이다. 무거운 불량을 자기 부하로 삼아 자신을 스스로 밧줄 포박하는, 가전과 나노 회전체의 오토 밸런싱(Auto Balancing) 시스템 방정식이다.
