3.3.2.1.8 불평형 원심력이 굽힘 응력(Bending Stress) 한계를 넘어설 시 축의 소성 변형 및 피로파괴 S-N 곡선

기계의 밸런싱을 제대로 하지 않고 납 용접도 대충 남겨 두어 모터가 미친 듯이 징그럽게 떨리면서(2\text{mm/s} \dots 5\text{mm/s} \cdots) 구동하는 발전소 현장을 상상해 보자.
“뭐, 소리만 좀 시끄럽고 떨리기만 할 뿐인데 그냥 좀 돌립시다! 터지지만 않으면 되지.”
이 무식한 방관은 불과 수 달 후, 두께가 집채만 한 100톤이 넘는 강철 압축기 샤프트를 두 동강 내며 대폭발을 낳는다. 왜 진동을 방치하면 강철이 플라스틱 엿가락처럼 박살이 나는가?
바로 보이지 않는 오차 찌꺼기가 낳은 원심력이 어마어마한 굽힘 응력(Bending Stress, \sigma) 이 되어 축 파이프의 피부 깊숙한 곳을 “늘렸다 압축시켰다” 무한 펌프질을 해대기 때문이다. 이른바 영구 변형(Plastic Deformation)과 피로 한도(Fatigue Limit) 파탄의 사신, S-N 곡선의 멸망을 알아본다.

1. 보이지 않는 가위질: 번갈아 치는 굽힘 응력(Alternating Stress)

축 어딘가에 용접 똥(Unbalance Mass) 하나가 찍혀 있어서 축이 도는 내내 불룩한 배(High \, Spot) 가 되어 훌라후프를 돈다고 치자.
이 훌라후프 파이프를 자세히 현미경으로 뜯어보면 어떨까?

  • 배가 볼록 튀어나온 껍데기(바깥쪽 표면) 철근은 쭉~~ 길게 잡아 늘어지는 인장 응력(Tension, + \sigma) 을 어금니를 악물며 버티고 찢어지고 있다.
  • 반면에 오목하게 들어간 반대쪽 껍데기(안쪽 표면) 철근 피부는 꾹~~ 쭈그러들며 짓눌리는 압축 응력(Compression, - \sigma) 을 얻어맞고 폐달이 밟혀 무너지고 있다.

문제는 축이 가만히 휘어있는 게 아니라 미친 듯이 \text{3,000 RPM}으로 빙글빙글 돈다는 것이다!
축 겉껍질 철근 세포 하나 입장에서 보자. 1초 전엔 배불뚝이 바깥쪽에 있어서 찢어질 듯 당겨졌는데(+\sigma), 반 바퀴(0.01초) 훅 돌고 나면 자기가 축 밑바닥 오목한 곳에 처박혀 무식하게 눌리는(-\sigma) 발작을 당한다! 1초에 50번씩 + \dots - \dots + \dots - 피부가 뜯겼다가 뭉개지는 끔찍한 고문, 이른바 교번 굽힘 응력(Alternating Bending Stress) 사이클에 강제로 묶여버린다.

2. 볼펜 구부러뜨리기 폭파: S-N 곡선(Wöhler Curve) 과 파멸의 교차점

우리도 볼펜 철사를 끊을 때, 한 번에 세게 힘을 주는 게 아니라 위아래로 계속 깔짝깔짝 굽혔다 폈다 하면 어느 순간 피로(Fatigue)가 누적돼 톡 하고 끊어진다. 터빈 강철 파이프도 완벽하게 똑같이 죽는다.
얼마만큼의 진동(스트레스 \mathbf{S}) 을 가하면, 기계가 도르륵도르륵 총 몇 바퀴 마일리지(\mathbf{N} 번의 사이클) 만에 박살 내는지를 철제별로 지도 그려놓은 사형 선고 그래프가 그 유명한 S-N 곡선(응력-수명 커브) 이다.

  • 파단(Yield \, Limit) 초과 구역: 애초에 불평형 무게 똥 덩어리가 100\text{kg} 짜리라서 원심력 파워가 미쳐버린다면? 모터 전원 올리자마자 쇠 파이프 항복 응력 한계점(S_{yield}) 을 박살 내면서, 5분 만에 축 영구 소성 변형(Plastic Deformation) 이 와서 축이 90도로 꺾이고 모터가 방 안을 뚫고 날아간다.
  • 피로 폭발 한계선 (Endurance Limit S_e): 만약 무게 똥이 좀 작아서(100g) 당장은 안 터진다고 방심했다간 더 비참하게 죽는다. 이 작은 진동이 파이프 겉껍질에 아주아주 미세한 극세사 스크레치 실금(Micro Crack) 을 무수히 낸 채로 위아래 + - 굽힘 고문(볼펜 구부리기) 을 때리면, 기계 수명 곡선 \mathbf{N = 10^7 (천만 번 회전)} 사이클 부근, 대충 3달 뒤에 축이 아무 경고음도 없이 갑자기 단칼에 무 치듯 피로 파괴 댕강 두 동강(Fatigue Failure) 이 나서 수천 억대 발전 공장이 산산조각 날아간다.

수식 속 벡터와 영향 계수 납덩이들은 막연한 이상한 숫자 놀음이 아니라, S-N 스펙트럼 곡선 위에서 초 단위로 생명을 갉아 먹어가는 내 터빈 엔진의 교번 굽힘 응력(\text{Alternating Stress} \,\, \sigma) 폭주량 을 어떻게든 S-N 곡선의 평탄한 낭떠러지 바닥 밑 ‘불멸 저항선(Endurance Limit \mathbf{S_e})’ 웅덩이로 밀어 넣어 생명을 보존하기 위한 구명 튜브다.
그 안전 마지노선의 기준선, 이른바 국제 공학자들의 약속 “ISO 등급(Grade) 한계 진동수” 의 법전장 3.3.2.2 국제 규격(ISO) 산업 회전 기계 모니터링 편 으로 진입하는 완벽한 역학 브릿지다.
