3.3.2.2 국제 규격(ISO) 기반 산업 회전 기계 상태 감시(CM) 진동 허용치

기계가 내지르는 ‘진동’ 이라는 파동의 비명 소리는, 단순한 소음이 아니라 피부 속 철근이 찢어지기 전 엔지니어에게 보내는 기계의 생체 신호(Bio-signal) 며 마지막 탈출 경고다. 우리는 혈압 수치가 140을 넘으면 고혈압 판정을 내리듯, 수만 시간 가동해야 하는 거대 산업 회전 기계가 내뿜는 진동 신호에도 공학적 사형 선고와 안전의 마지노선을 굳건히 쳐놓았다.
이 장에서는 인간의 막연한 “좀 흔들리네?” 라는 핑계를 압살하는 명확한 국제법, 즉 ISO (국제 표준화 기구) 및 API (미국 석유 학회) 가 주창한 상태 감시(Condition Monitoring) 기반 진동 속도(RMS) 와 상대 변위 허용 규격의 엄정한 역학 법전을 해독한다.
