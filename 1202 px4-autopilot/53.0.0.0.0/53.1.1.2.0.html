53.1.1.2 나이퀴스트-섀넌 표본화 정리(Nyquist-Shannon Sampling Theorem) 위배에 따른 이산 제어기(Discrete Controller)의 앨리어싱(Aliasing) 및 발산 모델

디지털 시스템 기반의 비행 제어기(FCS)인 PX4-Autopilot 및 Ardupilot 체계는 본질적으로 이산 제어기(Discrete Controller)로 동작한다. 무인기(UAV)의 물리적 역동성(Dynamics)은 연속 시간(Continuous-Time) 상에 존재하지만, 이를 제어하는 센서 데이터의 취득과 제어 명령(Command)의 산출은 이산 시간(Discrete-Time) 주기마다 이루어진다. 이 과정에서 지상 관제 시스템(GCS) 또는 외부 자율 제어 에이전트(ROS2 컴퓨터 등)와의 통신 링크 상태가 열화되거나 단절될 경우, 제어 이론의 근간이 되는 나이퀴스트-섀넌 표본화 정리(Nyquist-Shannon Sampling Theorem)가 위배되어 치명적인 궤도 발산(Divergence)이 생길 수 있다.

1. 제어 신호 표본화 정리와 데이터 링크의 역할

나이퀴스트-섀넌 표본화 정리에 따르면, 기체의 고주파수 동역학 대역폭(Bandwidth) 내 최고 주파수 성분을 f_{max}라 할 때, 외부 제어기가 왜곡 없이 연속적인 제어 지령을 디지털 시스템에 재현(Reconstruction)하기 위해서는 데이터 링크의 업데이트 주기(Sampling Frequency, f_s)가 최소 다음의 조건을 만족해야 한다.

f_s > 2 f_{max}

이 기준 주파수 2 f_{max}를 나이퀴스트 주파수(Nyquist Frequency)라고 한다.
정상 통신 상태에서는 GCS의 MAVLink 제어 루프나 ROS2 환경의 OffboardControlMode 메시지 송신이 f_s 조건을 충분히 상회(예: 10Hz ~ 50Hz)하여 원활한 궤적 추종이 가능하다.

2. 통신 품질 저하로 인한 표본화 임계점 미달 현상 (Undersampling)

데이터 링크에 심각한 패킷 드롭(Packet Drop)이나 일시적 통신 음영(Data Link Loss)이 발생하면, 외부에서 기체로 들어오는 기준 지령의 실효 갱신 주기(Effective Sampling Rate)는 급격히 하락하게 된다.
만약 초당 50번(f_s = 50\text{Hz}) 도착해야 할 궤적 설정점(Trajectory Setpoint)이 통신 장애로 인해 초당 1~2번 수준으로 띄엄띄엄 도착할 경우, 이는 표본화 주기가 기체의 동역학적 반응 주파수보다 길어지는 과소표본화(Undersampling) 현상을 초래한다.

3. 앨리어싱(Aliasing) 유발 및 허위 궤적(Ghost Trajectory) 생성 메커니즘

과소표본화된 궤적 명령은 이산화 모델에서 앨리어싱(Aliasing) 효과를 발생시킨다.
수학적으로 이산화된 신호 x_s(t)의 스펙트럼은 원신호의 스펙트럼이 f_s 간격으로 반복되는 형태를 띠게 되는데, f_s < 2 f_{max} 인 상태에서는 인접한 스펙트럼 간에 중첩(Fold-over) 현상이 발생한다.

결과적으로, 원래는 기체의 신속한 자세 보정을 위해 요구되었던 고주파수 지령 신호가 PX4 내부의 물리적 응답 필터링 과정에서 전혀 엉뚱한 저주파수 대역의 가짜 신호(Alias) 형태로 해석된다. 이를 제어 공학 측면에서는 제어 명령의 오인(Spurious Command)으로 분류하며, 기체 입장에서는 외부 에이전트가 존재하지 않는 가상의 급기동(Ghost Trajectory)을 명령한 것으로 인지하게 만든다. 이로 인해 기체는 예기치 못한 방향으로 맹렬하게 발산 진동(Oscillatory Divergence)하게 된다.

4. PX4-Autopilot 및 Ardupilot의 안전 장치 철학과 한계 극복

이러한 앨리어싱에 의한 발산은 내부 필터(Low-pass Filter, Anti-aliasing Filter) 만으로는 복구가 불가능하다. 통신을 통해 유입되는 데이터 자체의 갱신 속도가 물리적 한계점 이하로 무너졌기 때문이다.
따라서, GCS 관제 시스템과 오픈소스 오토파일럿은 다음의 수학적 당위성에 기반하여 예외 처리 로직(Exception Handling Logic)을 강제한다.

  1. 최저 샘플링 주파수 강제 (Timeout 설정):
    PX4의 오프보드 모드(Offboard Mode) 제어에서는 타임아웃 파라미터 COM_OBL_LOSS_T를 통해 최소 업데이트 주기를 강제한다. (예: 최소 2Hz 이상).
    수신되는 MAVLink 또는 uXRCE-DDS 메시지의 주파수가 설정된 임계 주파수(f_s \ll 2f_{max}) 구간으로 추락하는 순간, PX4 Commander 모듈은 치명적인 앨리어싱이 물리적 자세 제어 루프를 장악하기 전에 오프보드 제어권을 박탈하고 내부 제어기만의 Hold 혹은 RTL(Return) 로직으로 자동 천이시킨다.
  2. 이산 표본화 주기의 확률론적 추정:
    QGroundControl(GCS) 측면에서도 기체와의 하향 링크(Downlink)를 모니터링하여, 통신 링크의 실효 샘플링 퀄리티가 일정 비율 이하로 떨어지면 “Communication Disturbed” 등의 경고를 통해 오퍼레이터가 과도한 수동 관제 명령을 내리지 못하도록 인터페이스 조작을 제한한다.

결론적으로, 무인기 데이터 링크의 상실은 표본화 정리의 관점에서 바라볼 때, 시스템 전체의 가관측성(Observability) 및 가제어성(Controllability)을 원천적으로 붕괴시키는 주파수 앨리어싱 스펙트럼의 침투 과정이다. 따라서 안전한 무인기 시스템은 반드시 통신 신뢰도를 이산 시간 제어 파라미터의 핵심 지표로 다루어야 한다.
