53.5.1 통신 열화 수준의 퍼지 논리(Fuzzy Logic) 기반 분류 및 휴먼 머신 인터페이스(HMI) 경고 자동 조정

장거리 무인항공기 시스템(UAS; Unmanned Aerial System) 운용 시 지상 관제 시스템(GCS; Ground Control Station)과 기체 사이의 통신 링크는 전파 간섭(RF Interference), 안테나 지향각 오차, 물리적 장애물 등에 의해 아날로그적으로 점진적인 열화(Degradation)를 겪는다. 전통적인 통신 상태 진단은 단순히 “연결됨(Connected)” 또는 “단절됨(Disconnected)” 형태의 이진 논리(Binary Logic)로 평가되었으나, 이는 일시적인 지연이나 간헐적 패킷 손실 상황에서 운영자에게 적절한 상황 인식(Situational Awareness)을 제공하기에 부족하다. 본 절에서는 통신 상태의 모호성을 정량적으로 처리하기 위해 퍼지 논리(Fuzzy Logic)를 도입하여 통신 열화 수준을 분류하고, 이에 따라 휴먼 머신 인터페이스(HMI; Human-Machine Interface)의 경고 강도를 자동 조정하는 알고리즘 및 소프트웨어 구조를 상세히 기술한다. (참고 버전: PX4-Autopilot v1.14, QGroundControl v4.3)

1. 이진 논리 판단의 한계와 퍼지 논리의 도입 배경

MAVLink 프로토콜을 사용하는 QGroundControl(QGC) 등 기존 GCS의 통신 감시는 주로 하트비트(Heartbeat) 메시지의 수신 타임아웃(예: 3초 이상 수신 불량)에 의존하는 이진 형태를 띠어왔다. 하지만 무선 주파수 환경에서는 패킷 드롭률(Packet Drop Rate)의 증가, 지연 시간(Latency)의 급증, 신호 강도(RSSI; Received Signal Strength Indicator)의 하락이 복합적으로 일어나며, 완전히 단절되기 전의 “경고 상태“가 존재한다.

퍼지 논리(Fuzzy Logic)는 0과 1 사이의 진릿값(Truth Value)을 허용하는 다치 논리(Many-valued Logic)의 일종이다. 이를 통신 평가에 적용하면 RSSI, 패킷 드롭률, 그리고 하트비트 지연시간이라는 복수 개의 매개변수(Parameters)를 연속적인 소속 함수(Membership Function)에 대입하여 “양호(Good)”, “경고(Warning)”, “위험(Critical)” 등과 같은 상태로 세밀하게 매핑할 수 있다. (참고: Fuzzy Control Systems in Aerospace)

2. 통신 열화 수준 평가를 위한 퍼지 추론 시스템(FIS) 설계

퍼지 추론 시스템(FIS; Fuzzy Inference System)은 크게 퍼지화(Fuzzification), 퍼지 규칙 기반 추론(Rule-base Inference), 역퍼지화(Defuzzification)의 3단계로 구성된다.

2.1 입력 변수 및 소속 함수 정의 (Fuzzification)

통신 링크의 건전성을 평가하기 위한 시스템의 세 가지 주요 입력 공간(Input Space)은 다음과 같이 정의된다.

  1. 지연 시간 (Latency): 밀리초(ms) 단위. (Low, Medium, High)
  2. 패킷 드롭률 (Packet Drop Rate): 백분율(%) 단위. (Low, Medium, High)
  3. RSSI (Signal Strength): 데시벨 밀리와트(dBm) 단위. (Weak, Moderate, Strong)

이 값들은 각각 사다리꼴(Trapezoidal) 또는 삼각형(Triangular) 형태의 소속 함수를 통해 0.0 ~ 1.0 사이의 퍼지 값으로 변환된다.

2.2 퍼지 규칙(Fuzzy Rules)의 적용

경험적 데이터에 기반한 다중 조건 추론 규칙을 정의한다.

  • IF (RSSI is Weak) AND (Packet Drop Rate is High) THEN (Link Quality is Critical)
  • IF (RSSI is Moderate) AND (Latency is Medium) THEN (Link Quality is Warning)
  • IF (RSSI is Strong) AND (Packet Drop Rate is Low) THEN (Link Quality is Good)

2.3 HMI 경고 강도 도출 (Defuzzification)

이러한 규칙들에 의해 계산된 결과 집합들은 무게 중심법(Center of Gravity Method)과 같은 역퍼지화 과정을 거쳐 0에서 100 사이의 **통신 열화 지수(Communication Degradation Index)**라는 하나의 실수(Real Number)로 귀결된다. GCS의 프론트엔드는 이 지수를 넘겨받아 경고의 수준을 결정한다.

3. HMI 경고 자동 조정 알고리즘 구현 개요

도출된 통신 열화 지수에 따라 QGroundControl과 같은 관제 소프트웨어의 UI(QML 기반)는 운영자의 시각 및 청각적 인지를 자동으로 제어한다. 이는 경고 피로(Alarm Fatigue)를 방지하고 실제 위험에 처했을 때만 강력한 자극을 제공하는 데 의의가 있다.

  • 안전 구간 (Index 0 ~ 30): 정상(Connected) 상태. UI 상 배터리, 텔레메트리 값이 녹색(Green)으로 표시되며, 별도의 소리 알람이 발생하지 않는다.
  • 경고 구간 (Index 31 ~ 70): 주의(Warning) 상태. UI 컴포넌트 내 텔레메트리 숫자가 황색(Yellow)으로 변환되고 2초 주기의 부드러운 박동(Pulsing) 애니메이션이 추가된다. 시스템은 “Check Telemetry Link“라는 약한 톤의 음성 알림을 송출한다.
  • 위험 구간 (Index 71 ~ 100): 위험(Critical) 상태. UI가 적색(Red)으로 급격하게 깜빡이며(Blinking 빈도 증가), 운영자 주의를 집중시키기 위해 피치(Pitch) 빈도가 격상된 오디오 톤 및 경보음이 재생된다. MAVLink 데이터 처리가 스레드에서 차단되며 무효화(Invalidation) 로직이 구동된다.

다음은 통신 열화 지수를 HMI에 반영하는 추상화된 C++ 백엔드 로직 예시이다.

// 퍼지 로직 제어기에서 넘어온 열화 지수를 HMI 이벤트로 변환하는 클래스
void LinkQualityManager::updateHMI(float degradationIndex) {
    HmiAlertLevel newLevel = HmiAlertLevel::NORMAL;
    
    if (degradationIndex > 70.0f) {
        newLevel = HmiAlertLevel::CRITICAL;
        // 적색 점멸 및 고주파 타격음 발생 지시
        emit criticalAlertTriggered();
    } else if (degradationIndex > 30.0f) {
        newLevel = HmiAlertLevel::WARNING;
        // 황색 표시 및 경고음 지시
        emit warningAlertTriggered();
    } else {
        newLevel = HmiAlertLevel::NORMAL;
        // 정상 상태 복귀
        emit normalStateRestored();
    }
    
    if (this->_currentAlertLevel != newLevel) {
        this->_currentAlertLevel = newLevel;
        // UI(QML) 컴포넌트로 상태 변화 시그널 전송
        emit alertLevelChanged(this->_currentAlertLevel);
    }
}

4. 결론 및 고찰

통신 열화 수준에 퍼지 논리를 적용하는 아키텍처는 이분법적인 통신 상태 판단이 가지는 맹점을 극복한다. 이를 통해 GCS는 기체 통신의 점진적 악화 과정을 선제적(Proactive)으로 운영자에게 알려주며 대응 가능 시간을 확보해 준다. MAVLink 기반의 드론 제어 환경에서 휴먼 에러(Human Error)를 감소시키기 위해서는 이와 같이 기계가 인지 공학적 요소를 판단하여 복합적인 HMI 경고 레벨을 스스로 조정하는 지능화 기법의 탑재가 필수적이다.
