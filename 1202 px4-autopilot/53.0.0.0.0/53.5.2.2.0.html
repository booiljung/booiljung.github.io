53.5.2.2 확장 칼만 필터(EKF)의 사전 오차 공분산 행렬(A Priori Error Covariance Matrix) 발산 비율을 시각화한 불확실성 반경(Uncertainty Radius) 오버레이 렌더링

지상 관제 시스템(GCS)에서 무인기(UAV)의 위치를 추측 항법(Dead Reckoning)으로 예측할 때, 예측된 궤적(Estimated Trajectory)이 실제 기체의 위치와 어느 정도 떨어져 있을지(오차의 범위)를 직관적으로 파악하는 것은 매우 중요하다. 특히 PX4-Autopilot의 확장 칼만 필터(EKF2, Extended Kalman Filter 2)는 기체의 동역학 모델과 센서 측정치를 융합하여 상태를 추정하며, 이때 상태 변수의 불확실성(Uncertainty)을 나타내는 **오차 공분산 행렬(Error Covariance Matrix)**을 함께 연산한다. GCS는 통신 단절(Data Link Loss) 직전에 기체로부터 전달받은 최후의 공분산 지표를 활용하여 시간에 따른 오차 발산 비율(Divergence Rate)을 계산하고, 이를 지도(Map) 인터페이스 상에 불확실성 반경(Uncertainty Radius) 오버레이로 렌더링(Rendering)함으로써 관제사의 인지 공학적(Cognitive Engineering) 의사결정을 지원한다.

1. 확장 칼만 필터(EKF2) 오차 공분산 행렬의 이해

확장 칼만 필터는 비선형 시스템의 상태를 추정하기 위해 테일러 급수로 1차 선형화(Linearization)를 수행한다. 필터의 동작은 크게 예측(Prediction)과 갱신(Update) 단계로 나뉘며, 이때 기체의 위치 불확실성을 나타내는 핵심 지표가 공분산 행렬 P이다.

1.1 사전 오차 공분산 행렬(A Priori Error Covariance Matrix)

다음 갱신 단계(Measurement Update)를 통과하기 전, 시스템 모델(Kinematic Model)만을 기반으로 예측된 오차의 크기를 **사전 오차 공분산 행렬(A Priori Error Covariance Matrix, P_{k}^{-})**이라 한다. 이는 전파 수식(Propagation Equation)을 따른다:

P_{k}^{-} = F_{k-1} P_{k-1} F_{k-1}^T + Q_{k-1}

여기서:

  • P_{k}^{-} : 시점 k에서의 사전 오차 공분산 행렬
  • F_{k-1} : 상태 천이 행렬(State Transition Matrix)의 자코비안(Jacobian)
  • P_{k-1} : 이전 시점의 사후(A Posteriori) 오차 공분산 행렬
  • Q_{k-1} : 시스템 모델 노이즈 공분산 행렬(Process Noise)

통신이 단절된 상황에서는 GCS가 GPS와 같은 관측치(Measurement)를 기체로부터 수신하여 갱신할 수 없으므로, GCS 내부의 예측 모듈은 Q항의 누적 효과로 인해 P_{k}^{-}의 위치 관련 행렬 공간(Position Sub-matrix)이 시간에 비례하여 양의 확정(Positive Definite) 행렬 형태로 점진적 확장(Divergence)됨을 연산하게 된다.

2. 불확실성 반경(Uncertainty Radius)의 연산 및 시각화 모델링

QGroundControl(v4.3 기준)과 같은 GCS는 단절 직전의 MAVLink ESTIMATOR_STATUS 및 ODOMETRY 메시지를 통해 위치 분산(Variance) 값을 확보한다. (예: 위치 변수 x, y 축에 대한 분산 \sigma_x^2, \sigma_y^2)

2.1 2차원 불확실성 타원(Error Ellipse) 혹은 반경(Radius) 계산

NED(North-East-Down) 프레임의 2차원 평면(North-East) 지도 투영을 위해, 2D 위치 오차 공분산 행렬의 고유값 분해(Eigenvalue Decomposition)를 통해 오차 타원(Error Ellipse)의 장축과 단축 벡터를 구할 수 있다. GCS 렌더링 자원 최소화를 위해 가장 큰 고유값(최대 분산)을 반경으로 하는 단순화된 불확실성 반경 R(t) 모델을 다음과 같이 적용한다.

R(t) = R_0 + \kappa \cdot (\Delta t)^{\gamma} + \text{Drift}(V_{\text{wind}}, \Delta t)

여기서:

  • R_0 : 단절 직전 수신된 1\sigma(약 68% 신뢰구간) 또는 3\sigma(약 99.7% 신뢰구간) 초기 오차 반경
  • \kappa : 시간에 따른 EKF 공분산 발산 비율 팩터 (센서 노이즈 및 시스템 식별 상수)
  • \gamma : 발산의 비선형 가중치 비율 (일반적으로 시간의 제곱 \Delta t^2에 비례하여 확장)
  • \text{Drift} : 외풍(Wind) 벡터 데이터 추정치 기여분

3. QGroundControl UI 오버레이 렌더링 아키텍처

GCS 지도 컴포넌트에 그려지는 불확실성 반경(Uncertainty Radius)은 기체가 위치할 확률 분포의 경계를 시각적으로 나타낸다.

3.1 QML 기반 렌더링 특징

  • 동적 반경 업데이트: C++ 백엔드(예: TrajectoryPredictor)가 매 10Hz 단위로 갱신된 반경 데이터 R(t)를 QML FlightMap으로 전달한다.
  • 그라데이션 및 투명도(Opacity): 반경 내 기체가 존재할 확률 밀도가 중심부에 가까울수록 높으므로, MapCircle의 채우기 색상(Fill Color)은 점진적 투명도(Gradient Opacity)를 갖도록 방사형 그라데이션(Radial Gradient)이 적용된다.
  • HMI 경고 강조: 반경이 특정 치명적 수치(예: 100m)를 초과하면 색상이 노란색(Yellow)에서 적색(Red)으로 점층적 변화를 주어 관제사의 인지적 주의(Cognitive Attention)를 환기시킨다.

3.2 오버레이 렌더링 프로세스 (Mermaid 다이어그램)

4. ArduPilot (Mission Planner) 시스템과의 관제 설계 차이 비교

PX4-Autopilot 및 QGroundControl의 불확실성 반경 시각화 방식은 ArduPilot의 Mission Planner 방식과 본질적인 설계 철학 차이를 가진다.

  • PX4-Autopilot (QGC): QGroundControl의 디자인은 터치 인터페이스와 실시간 HMI 상호작용을 중시하므로, EKF 공분산 발산 모델을 기반으로 한 부드러운 그래픽 원(Circle/Ellipse)의 확장을 통해 통신 단절 후 경과된 시간의 심각성을 시각적 넓이로 직관적으로 전달한다.
  • ArduPilot (Mission Planner): 전통적인 Mission Planner의 경우 관제 화면에 오차 반경 타원을 실시간으로 그리고 점진적으로 확장하는 동적 HMI 요소를 적극적으로 표현하지 않는 경향이 있다. 대신 정적인 위치 마커에 의존하고, “EKF Variance” 정보를 수치 지표(숫자 게이지 바) 형태로 제공하여, 오퍼레이터가 이 수치를 직접 판독하도록 요구하는 경향이 짙다.

5. 지상 관제 로직 구현 시 고려사항 및 ROS2 연동 기법

GCS 개발자가 QGroundControl 소스 코드를 수정하여 커스텀 렌더링 로직을 추가하거나, ROS2를 통해 자율 에이전트(Autonomous Agent) 관제 시스템을 설계할 경우 다음 사항을 고려해야 한다.

  1. 지리-픽셀 좌표 변환의 부하 방지: 3차원 WGS84 좌표계에서의 미터(meter) 단위 반경을 타일(Tile) 기반 웹 맵의 픽셀 단위(Pixel Radius)로 변환하는 연산은 지도를 확대/축소(Zoom)할 때마다 재계산되어야 한다. 이를 위해 QGC 내부의 C++ QGeoMap 변환 유틸리티를 활용하여 렌더링 부하를 줄여야 한다.
  2. ROS2를 통한 고부하 군집 비행 관제: 군집 비행(Swarm) 관제 시 다수 기체의 불확실성 반경 교집합(Intersection Area) 연산을 ROS2 기반 고성능 엣지 컴퓨팅 노드에서 오프플로딩(Off-loading)하여 계산한 후, 충돌 위험도 지표를 HMI 서버로 전달하는 방식으로 업그레이드할 수 있다. 이는 공중 충돌 방지 시스템 알고리즘 구현의 기초 단계가 된다.

6. 요약

예측 궤적의 단순한 점선 표기를 넘어, 확장 칼만 필터(EKF)의 사전 오차 공분산 행렬 발산 비율을 시각화한 불확실성 반경 렌더링은 최신 PX4-Autopilot 기반의 상황 인지(Situation Awareness)를 고도화하는 핵심 기술이다. GCS는 수치적 수학 모델과 사용자 HMI 인터페이스를 정교하게 연결하여, 통신 단절이라는 최악의 결함(Fault) 상황에서도 관제사에게 기체의 추적 가능 영역을 동적이고 과학적인 투명도 그래픽 오버레이 형태로 제공함으로써 비행체의 탐색 구조 및 위기 대응 능력을 비약적으로 끌어올린다.
