37.1.1.2 마르코프 연쇄(Markov Chain) 기반의 비결정적(Non-deterministic) 패킷 손실 확률 모델링

지상 관제 시스템(GCS)과 PX4-Autopilot이 교환하는 텔레메트리(Telemetry) 패킷은 무선 채널의 시변(Time-varying) 특성에 의해 비결정적(Non-deterministic)인 손실을 경험한다. 단순한 독립 동등 분포(IID, Independent and Identically Distributed) 모델로는 이러한 네트워크의 연쇄적인 패킷 드롭(Burst Packet Loss)을 효과적으로 모사할 수 없다. 따라서 제어 시스템 및 통신 관제 소프트웨어를 설계할 때, 무선 채널의 상태 전이 특성을 반영할 수 있는 마르코프 연쇄(Markov Chain) 기반의 손실 모델링이 주로 활용된다. 본 절에서는 통신 링크의 확률론적 열화를 정량화하는 대표적인 마르코프 채널 모델들을 분석한다.

1. 독립 동등 분포(IID) 모델의 한계와 마르코프 모델의 도입

패킷 손실률(Packet Loss Rate, PLR)이 p일 때, 기존의 IID 모델은 매 패킷 전송 시마다 동일한 확률 p로 손실이 발생한다고 가정한다.

  • IID 모델의 문제점: 무선 채널에서 발생하는 장애물에 의한 음영(Shadowing)이나 전파 간섭은 짧은 시간 동안 지속되는 경향이 있다. 따라서 한 패킷이 손실되었다면 바로 다음 패킷 역시 손실될 확률이 단순 p보다 훨씬 높다. IID 접근법은 이러한 ‘군집 손실(Burst Loss)’ 특성을 반영하지 못하므로, GCS의 타임아웃(Timeout) 윈도우나 재전송(Retransmission) 알고리즘을 과소평가하게 만든다.

이를 극복하기 위해, 현재 채널의 손실 상태가 바로 직전의 상태에만 의존한다는 마르코프 성질(Markov Property)을 가정한 1차 마르코프 연쇄 모델이 도입된다.

2. 길버트-엘리엇(Gilbert-Elliott) 2상태 마르코프 모델

무선 통신망의 연쇄적 패킷 손실을 모사하기 위해 제어 공학에서 가장 널리 쓰이는 마르코프 연쇄는 길버트-엘리엇(Gilbert-Elliott, GE) 모델이다. 이 모델은 무선 채널의 상태를 ’Good(좋음, G)’과 ’Bad(나쁨, B)’라는 두 가지 이산 상태(Discrete State)로 추상화한다.

2.1 모델의 상태와 전이 확률(Transition Probabilities)

  • Good 상태 (G): 패킷이 성공적으로 수신될 확률이 높거나 100%인 상태이다.
  • Bad 상태 (B): 통신 간섭이나 페이딩으로 인해 패킷 손실 확률이 높거나 100%인 상태(Burst 에러 발생 구간)이다.

상태 간의 전이 확률은 다음과 같이 정의된다.

  • P_{GG}: G 상태에서 다음 패킷 전송 시에도 G 상태에 머무를 확률 (일반적으로 1 - p)
  • P_{GB}: G 상태에서 다음 패킷 전송 시 B 상태로 전이할 확률 (p)
  • P_{BB}: B 상태에서 다음 패킷 전송 시에도 B 상태에 머무를 확률 (일반적으로 1 - q)
  • P_{BG}: B 상태에서 다음 패킷 전송 시 G 상태로 복귀할 확률 (q)

2.2 길버트-엘리엇 모델의 상태 천이도

    Good --> Good: 1 - p (유지)

    Bad --> Good: q (채널 복구)

이 모델에서 p는 채널이 갑자기 악화될 확률이며, q는 불량 채널이 다시 정상 복구될 확률이다. 군집 손실(Burst Loss)이 발생하는 실제 무선 텔레메트리 환경에서는 p가 매우 작고(에러 발생 빈도는 낮음), q 역시 상대적으로 작게(한 번 에러가 발생하면 오래 지속됨) 나타난다. 이때, 평균 버스트 에러 길이(Average Burst Length)는 1/q로 계산된다.

3. 마르코프 패킷 손실 모델이 PX4/GCS 관제에 미치는 통찰

마르코프 연쇄 기반의 모델링은 GCS와 PX4 간의 이산 사건 제어 시스템(DECS) 설계에 다음과 같은 중대한 엔지니어링 통찰을 제공한다.

3.1 동적 응답 대기 시간(Dynamic Timeout) 산정

QGroundControl이 PX4로 치명적인 명령(MAV_CMD_DO_RTL 등)을 전송했을 때 응답 패킷(COMMAND_ACK)을 수신하지 못했다면, 채널은 현재 Bad(B) 상태에 진입했을 확률이 높다.
만약 GCS가 고정된 짧은 시간 간격으로 즉시 1차, 2차 재전송을 시도한다면, 채널이 아직 Bad 상태 (P_{BB} 확률에 의해)에 머물러 있을 가능성이 크기 때문에 이 재전송 패킷들 역시 연쇄적으로 증발(Drop)할 것이다. 따라서 재전송 스케줄러(Retransmission Scheduler)는 마르코프 모델의 평균 버스트 길이를 고려하여, 점진적 백오프(Exponential Backoff) 알고리즘을 적용하거나 재전송 딜레이를 조정해야 한다.

3.2 상태 추정기(Estimator)의 오차 누적 방어

PX4의 내부 상태 추정기인 EKF2(Extended Kalman Filter)가 외부 비전 텔레메트리(Visual Odometry, VDOM)나 RTK GPS 보정 데이터(RTCM)를 수신할 때도 마르코프 모델이 응용된다.
센서 융합 관점에서, 측정 업데이트(Measurement Update) 데이터가 연쇄적으로 유실되는 버스트 손실 구간에서는 예측(Prediction) 단계만이 연속적으로 수행되어 추정 공분산(Covariance) 오차가 발산할 위험이 있다. 모델링된 1/q 시간에 해당하는 손실 윈도우를 사전에 수치화해두면, EKF2가 텔레메트리 가용성 상실(Data Link Loss)에 대응하여 Failsafe 위치 제어 모드로 전환하는 임계점(Threshold)을 안전하게 설계할 수 있다.

4. 결론

무선 텔레메트리 채널의 비결정적 패킷 손실을 단순한 평균치 기반의 IID 모델로 단정하는 것은 관제 시스템 설계 시 심각한 결함을 유발할 수 있다. 상태 간의 기억력(Memory) 특성을 내포하는 마르코프 연쇄, 특히 길버트-엘리엇 모델을 통한 확률론적 해석은 QGroundControl이나 ROS2 에이전트가 통신 단절을 단순히 ’우발적 오류’가 아닌 ’채널의 일시적인 상태 변이’로 다루게 끔 유도한다. 이는 곧 무인항공기 신뢰성 관제 시스템에서 가장 핵심적인 예외 처리 및 백오프 알고리즘 설계의 강력한 수학적 근거가 된다.
