49.4.1.1 패킷 손실률(Packet Loss Rate)과 제어 시스템 위상 여유(Phase Margin) 감소의 선형적 관계

1. 제어 공학 측면에서의 위상 여유(Phase Margin) 개요

무인 항공기(UAV)의 자세 및 위치 제어 시스템은 주파수 영역(Frequency Domain)에서 보데 선도(Bode Plot)를 통해 안정성을 평가할 수 있다. 이때 위상 여유(Phase Margin, PM) 는 폐루프 시스템이 불안정성(발산)으로 넘어가기 전까지 견딜 수 있는 추가적인 위상 지연의 크기를 의미한다. 공학적으로 기체의 제어기가 외란(Wind Gust 등)에 대해 오버슈트(Overshoot) 없이 강건하게 반응하려면 최소 45^\circ \sim 60^\circ 수준의 위상 여유가 확보되어야 한다.

PX4-Autopilot의 Offboard 모드와 같이 외부 지상 관제 시스템(GCS)에서 제어 지령을 무선으로 송신하는 경우, 네트워크 상의 패킷 손실은 단순한 정보 누락을 넘어 제어 루프의 치명적인 위상 지연(Phase Lag) 요소로 변환된다.

2. 패킷 손실(Packet Loss)의 등가 시간 지연(Equivalent Time Delay) 변환

네트워크 상에서 MAVLink 또는 ROS2 제어 패킷이 손실될 경우, PX4의 commander 및 하위 제어 루프는 새로운 지령이 도착할 때까지 이전 지령 값을 유지하는 영차 유지기(ZOH, Zero-Order Hold) 거동을 보인다.

정상적인 샘플링 주기를 T_s (예: 50Hz 제어 시 0.02초)라고 할 때, n개의 패킷이 연속으로 손실되면 시스템은 사실상 타의에 의해 n \times T_s 만큼의 추가적인 시간 지연(Time Delay, T_d)을 겪게 된다. 따라서 일정 시간 동안 관측된 평균 패킷 손실률(Packet Loss Rate, PLR)은 제어기가 겪는 평균 시간 지연 E[T_d] 와 비례 관계를 맺는다. 평균적으로 패킷 손실률이 증가할수록 ZOH로 인해 지령이 갱신되지 못하는 총 대기 시간이 길어지기 때문이다.

3. 손실률과 위상 여유 감소의 수학적 선형 관계

순수 시간 지연(Pure Time Delay) T_d가 피드백 제어 시스템에 추가될 경우, 이득(Gain)의 크기에는 영향을 주지 않지만 위상(Phase)에는 주파수에 비례하는 지연을 유발한다. 지연 시스템의 전달 함수는 e^{-s T_d} 로 표현되며, 주파수 응답에서의 위상 지연 \phi_{lag}는 다음과 같다.

\phi_{lag} = -\omega \cdot T_d

여기서 제어 시스템의 이득 교차 주파수(Gain Crossover Frequency)를 \omega_c 라고 할 때, 손실된 패킷으로 인해 유발된 추가 시간 지연 \Delta T_d는 기존 위상 여유를 다음과 같이 깎아내린다.

\Delta PM_{reduction} = \omega_c \cdot \Delta T_d

위 수식에 나타나듯, 감소하는 위상 여유(\Delta PM_{reduction})는 시간 지연(\Delta T_d)과 선형적(Linear)인 관계를 가진다. 그리고 앞절에서 설명했듯 평균 시간 지연은 평균 패킷 손실률(PLR)에 비례하므로, 결론적으로 패킷 손실률의 증가는 제어 시스템의 위상 여유를 선형적으로 감소시킨다는 공학적 결론이 도출된다.

4. PX4-Autopilot 및 Ardupilot 튜닝 실무로의 연계

위상 여유가 선형적으로 깎여나가 0^\circ 에 근접하면, 드론은 지령을 추종하지 못하고 크게 진동(Oscillation)하다가 추락하게 된다.

4.1 PX4-Autopilot 통신 품질 대응 전략

  • 무선 링크의 패킷 손실률이 높아 위상 여유가 부족해진 경우, PX4 내부의 위치 및 속도 비례 제어기(P Gain)인 MC_ROLL_P, MC_PITCH_P 등을 일시적으로 낮추어 이득 교차 주파수 \omega_c 자체를 왼쪽(저주파 대역)으로 이동시켜야 한다.
  • \omega_c 가 낮아지면 수식 \omega_c \cdot \Delta T_d 에서 감소하는 위상 여유 단위가 줄어들게 되므로, 다량의 패킷 손실 하에서도 시스템이 발산하는 것을 막을 수 있다. 이는 통신 상태가 나쁠 때 드론의 반응성(Agility)을 둔화시키는 대신 생존성(Stability)을 확보하는 이론적 근거가 된다.

4.2 Ardupilot과의 비교

  • Ardupilot 또한 내부적으로 동일한 위상 지연 메커니즘을 겪는다. Ardupilot의 ATC_ANG_PIT_P 등의 파라미터를 낮추면 이 현상을 완화할 수 있다.
  • 차이점은 PX4의 경우 Offboard 궤적 스트리밍 튜닝 가이드라인에서 네트워크 통신 주파수와 제어 주파수의 분리를 좀 더 명시적으로 통제하며, GCS 측에서 MAVLink를 통한 실시간 동적 이득 게인 스케줄링(Gain Scheduling) 알고리즘을 구현하기 위한 모듈화가 잘 되어 있다는 점이다.

5. 결론 및 제어학적 통찰

소프트웨어 엔지니어가 무선 드론 제어를 다룰 때 흔히 간과하는 점은, “패킷 몇 개가 빠져도 다음 패킷으로 덮어씌워지므로 문제가 없다“는 오해이다. 위상 여유와 패킷 손실률의 선형적 인과관계를 수학적으로 검토해 보면, 무작위로 누락된 패킷 하나하나가 제어계의 Damping Ratio를 직접적으로 깎아먹는 독약으로 작용함을 알 수 있다. 따라서 GCS 애플리케이션 개발자는 UDP 소켓을 설계할 때 텔레메트리 패킷 드롭률을 필터링하여 드론의 제어 이득(Control Gains)을 동적으로 안전 영역으로 하향시키는 지능화된 관제 로직을 설계해야 한다.
