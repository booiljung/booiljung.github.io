49.6.2.1 최소 저항크(Minimum Jerk) 및 최소 스냅(Minimum Snap) 목적 함수(Objective Function) 도출을 위한 미분 방정식 해법

1. 개요

멀티로터 무인항공기(UAV)의 궤적을 계획함에 있어 단순히 위치의 오차를 줄이는 것을 넘어, 제어 명령의 변화를 부드럽게 만들어 기계적인 스트레스를 최소화해 주는 최적 제어가 필수적이다. 이를 수학적으로 해결하기 위한 방법론이 최소 저항크(Minimum Jerk) 및 최소 스냅(Minimum Snap) 목적 함수 기반 궤적 최적화이다. PX4-Autopilot 및 고등 자율 비행 시스템에서는 이러한 최적 제어 이론의 결과물인 다항식 형태의 궤적을 이용하여 비행한다.

2. 미분 평탄성(Differential Flatness)과 최소스냅/저항크의 논리

멀티로터의 동역학 모델은 미분 평탄성(Differentially Flat) 시스템이라는 독특한 특징이 있다. 시스템의 상태(State)와 제어 입력(Control Input)을 4개의 평탄 출력(Flat Outputs)인 위치 (x, y, z) 및 요각(Yaw angle, \psi)과 그 미분값들로 온전히 표현할 수 있는 성질이다.

  • 위치의 1차 미분은 속도, 2차 미분은 가속도(추력 벡터와 연관), 3차 미분은 저항크(Jerk; 각속도와 연관), 4차 미분은 스냅(Snap; 각가속도 및 모터 토크와 연관)이다.
  • 즉, 제어 입력인 모터의 토크가 급격하게 변하는 것을 막기 위해서는 스냅(Snap)을 최소화해야 하며, 이는 곧 자세의 급격한 변동을 억제하여 에너지 소모와 오버슈트(Overshoot)를 줄여준다.

3. 목적 함수(Objective Function) 정의

최적 궤적을 찾기 위해서는 주어진 시간 T 동안 변화하는 특정 파생물(Derivative)의 에너지를 최소화하는 목적 함수 J를 정의해야 한다. p(t)를 시간 t에서의 위치라고 할 때,

최소 저항크(Minimum Jerk) 목적 함수:
J_{jerk} = \int_{0}^{T} \left\lVert \frac{d^3 p(t)}{dt^3} \right\rVert^2 dt

최소 스냅(Minimum Snap) 목적 함수:
J_{snap} = \int_{0}^{T} \left\lVert \frac{d^4 p(t)}{dt^4} \right\rVert^2 dt

4. 오일러-라그랑주 방정식(Euler-Lagrange Equation)과 미분 방정식 해법

범함수(Functional)인 목적 함수 J를 최소화하는 함수 p(t)를 찾는 것은 변분법(Calculus of Variations)의 영역이다.

J = \int_{0}^{T} \mathcal{L}(t, p, p', \dots, p^{(k)}) dt 에 대하여 오일러-라그랑주 방정식을 적용하면 다음과 같은 최적화 필요 조건 미분 방정식이 도출된다.
\frac{\partial \mathcal{L}}{\partial p} - \frac{d}{dt}\left( \frac{\partial \mathcal{L}}{\partial p'} \right) + \cdots + (-1)^k \frac{d^k}{dt^k}\left( \frac{\partial \mathcal{L}}{\partial p^{(k)}} \right) = 0

최소 스냅(k=4)의 경우, \mathcal{L} = (p^{(4)})^2 이므로 이를 대입하면 다음과 같은 간단한 상미분 방정식이 나온다.
p^{(8)}(t) = 0

이 미분 방정식의 일반해(General Solution)는 시간 t에 대한 7차 다항식(7th order polynomial)이 된다.

p(t) = c_0 + c_1 t + c_2 t^2 + c_3 t^3 + c_4 t^4 + c_5 t^5 + c_6 t^6 + c_7 t^7

마찬가지로 최소 저항크(k=3)의 경우, p^{(6)}(t) = 0이 되어 일반해는 5차 다항식(Quintic Polynomial)으로 유도된다.

p(t) = c_0 + c_1 t + c_2 t^2 + c_3 t^3 + c_4 t^4 + c_5 t^5

5. 경계 조건(Boundary Conditions) 및 제약 행렬 계산

다항식의 미정 계수 구조인 c_i 값을 구하기 위해서는 경계 조건을 활용하여 선형 대수 시스템 차원의 식을 연립 형태로 작성해 역행렬을 도출한다.

예를 들어 시작 지점(t=0)과 목표 지점(t=T)에서의 위치, 속도, 가속도 값이 MAVLink를 통해 주어졌다고 가정하라 (p_0, v_0, a_0, p_T, v_T, a_T).
최소 저항크의 경우 6개의 계수가 필요하며 마침 6개의 경계 조건이 주어졌으므로, 이를 행렬 A x = b 형태로 세워 계수 행렬 x = [c_0, ..., c_5]^T를 단번에 계산해 낼 수 있다.

6. 결론

PX4의 내부 코드나 GCS 시스템에서 궤적을 전송할 때, 이러한 미분 방정식을 통해 검증된 5차 다항식(최소 Jerk) 혹은 7차 다항식(최소 Snap) 프로파일을 실질적으로 활용하게 된다. 이 해법은 단일 구간 궤적뿐만 아니라 수많은 웨이포인트(Waypoint)를 거치는 다중 구간 궤적 문제를 유기적으로 최적화하는 모델에 응용할 수 있는 기반 이론 뼈대가 된다.
