51.6.2.1 방전 곡선(Discharge Curve)의 비선형성 및 순간 전압 강하(Voltage Drop) 현상을 반영한 내부 저항(Internal Resistance) 상태 추정 연산

무인항공기의 동력원인 리튬 폴리머(Li-Po) 또는 리튬 이온(Li-Ion) 배터리는 잔여 용량에 따라 전압이 일정하게 감소하는 선형적 특성을 가지지 않는다. 더욱이 기체가 급가속하거나 강한 바람에 저항할 때 발생하는 대전류 소모는 순간적인 전압 강하(Voltage Drop)를 유발하여, 단순 전압 측정만으로는 배터리의 정확한 잔존 상태(State of Charge, SoC)를 판단하기 어렵게 만든다. PX4-Autopilot은 이러한 방전 곡선(Discharge Curve)의 비선형성과 배터리의 **내부 저항(Internal Resistance)**을 연산 지표로 활용하여 극심한 부하 환경에서도 신뢰성 높은 SoC 추정치를 산출한다.

1. 방전 곡선(Discharge Curve)의 비선형성 이해

일반적인 Li-Po 배터리의 방전 곡선은 크게 세 구간으로 나뉜다.

  • 초기 급락 구간 (100% ~ 90%): 완충 전압(셀당 4.2V)에서 사용 시작 직후 단시간에 전압이 약 4.0V 언저리까지 가파르게 떨어진다.
  • 안정적 평탄 구간 (90% ~ 20%): 전압이 4.0V에서 3.6V로 매우 완만하게 떨어지는(Flat) 영역이다. 단순 선형 수식으로 이 구간의 SoC를 추정하면 치명적인 오차가 발생할 수 있다.
  • 후반 절벽 구간 (20% 이하): 3.6V 이하로 떨어지기 시작하면 남은 용량이 급격히 소진되며 전압이 수직 하강(Voltage Cliff)한다.

PX4 시스템은 이 비선형성을 극복하기 위해 단순히 V_CHARGED와 V_EMPTY를 이은 직선 방정식이 아닌, 실제 배터리의 방전 특성을 반영한 다항식 또는 룩업 테이블(Lookup Table) 방식을 수식 연산에 차용한다.

2. 순간 전압 강하(Voltage Drop) 현상과 상태 추정 원리

모터가 높은 토크를 낼 때 인버터(ESC)는 배터리로부터 높은 부하 전류(Load Current, I)를 뽑아 쓴다. 배터리 셀 내부에 존재하는 고유의 내부 저항(Internal Resistance, R_{in})으로 인해 옴의 법칙에 따른 순간 전압 강하 현상이 발생한다.

V_{drop} = I \cdot R_{in}

이때 센서버스를 통해 측정되는 단자 전압(V_{measured})은 실제 배터리가 낼 수 있는 개방 회로 전압(Resting Voltage, V_{resting})보다 낮게 측정된다.

V_{measured} = V_{resting} - V_{drop}
V_{resting} = V_{measured} + (I \cdot R_{in})

이 등식을 통해 알아낸 V_{resting}을 앞선 ’비선형 방전 곡선 룩업 테이블’의 입력값으로 사용하면, 현재 모터를 얼마나 강하게 돌리고 있는지와 무관하게 일관된 배터리의 실제 액체적 잔류 용량(SoC)을 추정해 낼 수 있다.

3. 알고리즘 아키텍처 및 소스 코드 분석

PX4의 모듈은 전압 및 전류 데이터를 필터링한 후 BAT1_R_INTERNAL 파라미터 값에 기반해 V_{resting} (내부 소스코드로는 voltage_resting 도는 voltage_v)을 구하고 이를 방전 테이블에 대입한다.

기본적으로 src/modules/battery_status/의 루틴에서 다음과 같은 방식의 추정 로직이 가동된다.

// 배터리 잔량(%)을 반환하는 내부 추정 로직의 의사 코드(Pseudo Code) 예시
float Battery::estimateStateOfCharge(const float voltage_resting)
{
    // 1. 셀 당 전압으로 스케일 역산
    float cell_resting_voltage = voltage_resting / _params.n_cells;

    // 2. 비선형 방전 구간 정의 및 룩업 (LiPo 기준)
    // PX4는 BAT1_V_EMPTY와 BAT1_V_CHARGED 구간 안에서
    // 배터리 화학 특성(셀 종류)에 매칭되는 Lookup Table 구간분석을 수행한다.
    
    if (cell_resting_voltage >= _params.v_charged) {
        return 1.0f; // 100%
    } else if (cell_resting_voltage <= _params.v_empty) {
        return 0.0f; // 0%
    }
    
    // 특정 전압 배열에 대한 보간(Interpolation) 함수 적용
    // _discharge_curve_voltage 배열과 _discharge_curve_capacity 배열 매핑
    float capacity_pct = interpolateLookupTable(cell_resting_voltage);
    
    return capacity_pct;
}

이 알고리즘은 극단적인 액셀러레이션으로 V_{measured}가 V_EMPTY 이하로 곤두박질치는 상황이라도, I \times R_{in}을 더한 V_{resting}이 여전히 룩업 테이블 상단에 위치함을 산출해 내므로 섣부른 페일세이프(Failsafe)나 경고 알람(Buzzer)의 오작동을 차단한다.

4. 파라미터(Parameter) 설정 및 구성 가이드

이 비선형 추정 및 저항 보상 알고리즘이 QGroundControl(GCS)과 정확히 연동되려면 다음 파라미터가 명확히 설정되어야 한다.

  • BATx_R_INTERNAL: 배터리의 직렬 내부 저항(Ω). 너무 낮게 설정하면 드론 하강(급상승) 시 배터리 퍼센트가 출렁이고, 너무 높게 설정하면 고전류 소모 시 오히려 SoC가 상승하는 기현상이 발생한다. 스마트 배터리가 배터리 상태(SoH)에 따라 이 값을 동적으로 UAVCAN 버스를 통해 주입할 수 있도록 설정하는 것이 가장 이상적이다.
  • BATx_SOURCE: 배터리 방전 특성 선택. LiPo, Li-Ion 등 케미컬(Chemical) 특성에 따라 방전 곡선의 형태와 평탄 구간의 기울기가 다르므로 배터리 화학 종류를 정확히 지정해야 알맞은 룩업 테이블이 활성화된다.

5. Ardupilot 대비 차이점

  • 배터리 곡선 제어: Ardupilot은 전통적으로 전압과 소모 전력량(mAh)을 선형적으로 추종하거나 임계점을 조정하는 방식을 많이 썼으며, 최근 버전에 와서야 비선형 곡선 룩업 모델 확장 기능(BATT_CURVE) 및 내부 저항 보상을 적극 채용했다.
  • 아키텍처 관점: PX4-Autopilot은 battery_status uORB 데몬에 이러한 배터리 화학적 물성 연산(Chemistry-agnostic/specific Lookup)을 깊숙이 통합하여 기본 동작으로 삼는데 더 주안점을 두었으며, MAVLink 프로토콜을 통해 BATTERY_STATUS 메시지로 GCS에 보정된 SoC를 직접 전송함으로써 GCS의 연산 의존도를 제거하였다.
