51.2.4.1 비선형 공기역학적 지면 효과(Ground Effect)로 인한 양력 계수(Lift Coefficient) 증가 현상을 상쇄하기 위한 적분기 안티 와인드업(Integrator Anti-windup) 기법

무인항공기(UAV)가 이착륙을 위해 지면에 근접할 때, 로터(Rotor) 하강 기류(Downwash)가 지면과 충돌하면서 압력이 상승하고 이로 인해 추가적인 양력 계수(Lift Coefficient)가 발생하는 비선형 공기역학적 지면 효과(Non-linear Aerodynamic Ground Effect)를 겪게 된다. 이러한 현상은 멀티로터 기체가 착륙(Land) 궤적의 최종 단계에 진입했을 때 하강 속도를 의도치 않게 감소시키고, 기체가 바운싱(Bouncing) 현상을 일으키거나 착륙 지점에서 벗어나게 만드는 주된 원인이 된다.

최신 PX4-Autopilot(펌웨어 v1.14.0 이상 기준)에서는 이러한 비선형적 양력 증가를 보상하고, 고도 제어기(Altitude Controller) 내 적분기(Integrator)가 과도하게 누적되어 불안정성을 초래하는 것을 방지하기 위해 정교한 적분기 안티 와인드업(Integrator Anti-windup) 기법을 적용하고 있다. 이 장에서는 지면 효과의 공기역학적 특성을 분석하고, 이를 상쇄하기 위한 PX4의 제어 아키텍처 및 소스 코드를 심층적으로 분석하라.

1. 지면 효과(Ground Effect)의 공기역학적 기초 및 제어역학적 위협

지면 효과는 헬리콥터 및 멀티로터 역학에서 널리 연구되어 온 현상이다. 로터와 지면 사이의 거리가 로터 직경의 약 0.5배에서 1배 이내로 좁혀질 때 두드러지게 발생한다(관련 논문: A Study on Ground Effect in Multirotor UAVs 참고).

  • 유도 속도(Induced Velocity)의 감소: 지면이 공기의 흐름을 막아 로터 평면을 통과하는 유도 속도가 줄어들고, 동일한 모터 추력(Thrust) 명령에도 불구하고 더 큰 유효 양력이 발생한다.
  • 적분기 와인드업(Integrator Windup) 현상 유발: PX4의 다단계 PID 제어기 중 고도 제어(Altitude Control) 루프에서는 목표 하강 속도를 유지하기 위해 오차를 적분(Integrate)한다. 하지만 지면 효과로 인해 예상보다 큰 양력이 발생하면, 제어기는 하강 속도 오차를 없애기 위해 추력을 계속해서 줄이게 되고 적분 오차가 음의 방향으로 크게 축적된다. 착륙 직후 시동이 해제(Disarming)되지 않거나 지면의 요철로 인해 순간적으로 고도를 다시 높이려 할 때, 음으로 누적된 적분기 값 때문에 기체가 즉각적으로 반응하지 못하거나 오버슈트(Overshoot)를 일으킨다.

2. PX4-Autopilot의 적분기 안티 와인드업 메커니즘 및 아키텍처 이해

안티 와인드업(Anti-windup)은 적분항(Integral Term)의 누적을 제어 입력의 물리적 한계점(Saturation) 이전에 멈추게 하거나, 특정한 비행 상태(예: Land Mode 진입, 지면 근접 감지 등)에 맞추어 적분기의 누적을 동적으로 클램핑(Clamping)하는 기법이다.

PX4에서는 PositionControl 아키텍처 내에서 추력 명령이 액추에이터의 한계에 도달했을 때 적분 루프를 동결시키는 방식을 취한다. 착륙 단계 동안에는 레이저 고도계(Laser Altimeter) 및 기압계, IMU가 융합된 EKF2(Extended Kalman Filter 2)의 고도 추정 상태에 따라 지면 감지 모듈(Land Detector)이 동작한다. 이 때 특정 고도(예: MPC_LAND_ALT2 파라미터 이하)에 진입하면 지면 효과 상쇄를 위한 제한 조치가 가동된다.

3. 소스 코드 분석: 안티 와인드업 논리의 구현

PX4의 소스 코드 트리 src/modules/mc_pos_control/PositionControl/ControlMath.cpp 및 PositionControl.cpp 내부를 살펴보면, PID 제어기의 적분항이 와인드업되는 것을 막기 위한 로직을 확인할 수 있다.

// PX4-Autopilot 제어기 내부의 Anti-windup 유사 구현 로직 예시
if (is_hovering_in_ground_effect) {
    // 지면 효과로 인해 상승/하강에 필요한 추력이 비선형적으로 변하는 구간
    // 목표 추력(thrust_sp)이 최소/최대 제한값에 도달했거나,
    // 오차의 부호가 특정 조건을 만족할 경우 적분기 누적 정지
    if ((thrust_sp(2) >= hover_thrust_max && velocity_error(2) > 0.0f) ||
        (thrust_sp(2) <= hover_thrust_min && velocity_error(2) < 0.0f)) {
        _vel_int.zero(); // 적분항 초기화 또는 동결(Clamping)
    }
}

제어 루프는 Z축(고도) 속도 오차를 계산한 후, 오차에 비례 이득(P)과 적분 이득(I)을 곱한다. 이때 Z축 추력 명령이 하한선에 도달하게 되는 지면 효과 발생 시, _vel_int 값을 증가시키지 않고 고정함으로써 바운싱 현상을 미연에 방지한다.

4. Ardupilot 대비 차이점 및 관제 철학

Ardupilot은 전통적으로 파라미터 튜닝을 통한 고정된 마진을 제공하는 방식에 비중을 둔다. 예를 들어 LAND_SPEED_HIGH에서 LAND_SPEED로 감속할 때 지면 효과 구간에 진입하면, 적분기 와인드업을 다루는 I-term 제한 계수값을 독립적으로 두어 스로틀을 제어한다.

반면, PX4는 동적 호버 추력 추정기(Hover Thrust Estimator) 와 연동하여 기체의 질량 변화와 대기 밀도를 실시간으로 추정하고, 이를 통해 적분기 베이스라인을 이동시키는 정교한 수학적 최적화를 수행한다. EKF2에서 산출되는 혁신(Innovation) 기반의 비정상 양력 증가분을 식별하고, Position Controller 내의 적분기 상한/하한을 실시간으로 갱신하는 것이 PX4 시스템의 설계 철학이다.

5. QGroundControl 및 파라미터 설정

지면 효과와 안티 와인드업 관련된 주요 제어 기법을 활성화하기 위해 QGroundControl(QGC v4.3)에서 다음 파라미터들을 조정하라.

  • MPC_Z_VEL_I: 고도축 속도 제어기의 적분 이득치. 너무 크면 지면 근접 시 와인드업에 더욱 취약해진다.
  • MPC_LAND_ALT1 및 MPC_LAND_ALT2: 다단계 착륙 시 고도를 구분하는 파라미터. 지면 효과는 주로 로터 직경 크기 근처의 고도에서 발생하므로, MPC_LAND_ALT2가 이 고도 이하로 설정되도록 구성하라.
  • EKF2_GND_EFF_DZ: 지면 효과 보상을 위해 확장 칼만 필터(EKF2)에서 사용하는 기압계 데드존(Deadzone) 크기이다. 지면 효과로 정압(Static Pressure)이 변하여 고도 추정에 오류가 생기는 것을 막기 위함이다.

지면 효과로 인한 양력 계수 변동을 성공적으로 다루기 위해서는 제어기의 안티 와인드업 로직뿐 아니라 위와 같은 EKF2 센서 퓨전 레벨에서의 능동적인 대처가 복합적으로 구성되어야 한다. 이를 통해 PX4는 착륙 불시착이나 전복을 방지하고 부드러운 터치다운(Touchdown)을 구현한다.
