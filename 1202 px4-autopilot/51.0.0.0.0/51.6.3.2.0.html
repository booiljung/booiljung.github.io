51.6.3.2. 자동-수동 제어 모드 전환 시 발생하는 목표 자세 각속도(Target Angular Rate)의 스텝 응답(Step Response) 불연속성을 완화하기 위한 유한 임펄스 응답(FIR) 필터 적용 기법

자동 비행 모드(Auto Mode)에서 수동 비행 모드(Manual Mode)로 주도권이 넘어갈 때, 제어 시스템이 가장 취약해지는 지점 중 하나는 목표 지령치(Setpoint)의 기구학적 단절 현상이다. 특히 드론의 자세 제어기(Attitude Controller) 하단부인 각속도 제어 루프(Rate Control Loop)에서는 목표 자세 각속도(Target Angular Rate)의 급격한 변동이 치명적이다. 본 절에서는 이러한 스텝 응답(Step Response) 형태의 지령치 불연속성(Discontinuity)이 야기하는 제어 동역학적 충격을 완화하기 위해 PX4-Autopilot에 적용되는 유한 임펄스 응답(Finite Impulse Response, FIR) 기반의 스무딩(Smoothing) 및 필터링 적용 기법에 대해 심층적으로 다룬다.

1. 지령치 불연속성과 스텝 응답(Step Response) 현상 동역학

기체가 복귀(Return) 궤적 상에서 안정된 피치(Pitch) 이동을 수행하며 자동 비행 중이라고 가정해 보자. 이 때 조종사가 개입을 위해 조종기 스틱을 좌측(Roll 방향)으로 강하게 치우친 채 오버라이드(Override)를 트리거했다면, 구동 중이던 제어 루프는 수 밀리초(ms) 단위의 틱 루프에서 궤적 생성기의 목표 각속도 세트포인트를 \vec{\omega}_{auto} (Roll = 0)에서 \vec{\omega}_{manual} (Roll = X rad/s)로 순식간에 변경하게 된다.

\Delta \vec{\omega}_{sp} = \vec{\omega}_{manual} - \vec{\omega}_{auto}

이러한 수직적 스텝파 형태의 오차(\Delta \vec{\omega}_{sp})가 순간적으로 PID 각속도 제어기의 비례 제어기(P-Term)와 미분 제어기(D-Term)에 포워딩(Forwarding)될 경우, 임펄스 형태의 과도한 토크(Torque) 명령으로 환산되어 모터 믹서(Motor Mixer)로 전송된다. 결과적으로 기체는 심한 저크(Jerk, 가속도의 미분)를 일으키며 덜컹거리거나, 최악의 경우 순간적인 모터의 포화(Actuator Saturation) 현상을 초래하여 제어성을 상실할 위험이 존재한다.

2. PX4-Autopilot의 FIR 제어 스무딩 아키텍처

이러한 기계적 임펄스를 억상하고 매끄러운 전이를 유도하기 위해, PX4의 mc_rate_control (멀티콥터 각속도 제어기) 모듈과 전단(Front-end) 지령치 처리부에서는 저역 통과 특성을 띄는 입력 필터링 기법을 적용한다.

2.1 FIR 필터를 통한 지령치 스무딩의 수학적 원리

소프트웨어적으로 설계된 유한 임펄스 응답(FIR) 필터는 이전 시점의 입력 데이터 모음을 가중 합산하여 노이즈나 스텝 파형을 부드러운 형태로 뭉뚱그린다.

FIR 필터의 출력 y[n]은 입력 신호 x[n] (새로운 목표 각속도)에 대해 다음과 같이 정의된다.

y[n] = \sum_{k=0}^{N} h[k] \cdot x[n-k]

여기서 h[k]는 필터 계수이며, 연속된 시간축에서 스텝 형태로 투입된 각속도 지령치는 필터 윈도우(Window) 사이즈 N에 비례하여 부드러운 곡선(S-Curve 또는 지수 감소 형태)으로 출력된다. 이로써 제어기의 D-Term이 충격 발산(D-kick)하는 현상을 원천적으로 차단한다.

2. ArduPilot 대비 관제 및 알고리즘 차이점

  • ArduPilot: 주로 제어기 전단의 감도 파라미터(예: ATC_INPUT_TC)를 사용하여 조종 입력에 대한 시간 상수 기반 1차 저역 통과 필터(First-order Low Pass Filter)를 활용해 “입력 셰이핑(Input Shaping)“을 달성한다. 이는 스틱의 즉각적인 반영률을 느슨하게 만들지만 주파수 응답 측면에서 FIR 대비 응답 형태가 다소 수동적인 지수적 감쇠로 한정된다.
  • PX4-Autopilot: PX4는 시스템 구성상 FIR 또는 IIR 2차(Biquad) 필터를 이용해 제어 입력은 물론 각 센서 계통의 데이터를 전방위적으로 스무딩한다. 최근 릴리즈(v1.14 이후)의 경우, 다항식 형태(Polynomial) 궤적 생성기가 융합되어 수동 오버라이드 시 속도/가속도의 연속성 뿐 아니라 각속도 레벨의 스텝까지 미분기(Differentiator)가 오작동하지 않도록 궤적 유효성을 미리 확보한 뒤 제어기에 넘겨주는 방식으로 진보하였다.

결론적으로, 수동 오버라이드로 인한 제어권 천이(Mode Rollback) 구간에서 목표 자세 각속도에 적용되는 FIR 스무딩 기법은 조종사의 급격한 개입으로 인한 물리적 파손이나 심각한 비행 불안정성을 방지한다. 이는 복귀 등 민감한 비행 절차의 예외(Failsafe) 처리에서 소프트웨어가 기계 공학적 한계를 매끄럽게 보완해주는 대표적인 인텔리전트 제어 방법론이다.
