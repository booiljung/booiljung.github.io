Chapter 14. 거리 측정 센서(LiDAR, Sonar, Radar) 및 옵티컬 플로우 연동

무인항공기(UAV)가 단순히 GPS(전역 측정)에 의존하여 높은 고도의 하늘을 항법(Navigation)하는 단계를 넘어, 지면에 근접한 저고도 비행, 실내 비행(GPS-Denied Environment), 그리고 정밀한 이착륙(Precision Landing) 임무를 완수하기 위해서는 지형지물과의 직접적인 상대 거리를 가늠할 수 있는 ’눈’과 ’촉각’이 필수적이다.

이 장(Chapter)에서는 PX4-Autopilot 생태계의 눈과 촉각을 담당하는 **거리 측정 센서(Distance Sensors: LiDAR, Sonar, Radar)**와 지면의 흐름을 픽셀 단위로 읽어내어 2차원 평면 속도(Velocity)를 추정하는 핵심 비전 센서인 옵티컬 플로우(Optical Flow) 시스템의 전반적인 아키텍처와 통합(Integration) 로직을 심층적으로 해부한다.
단순한 하드웨어의 결선 방법을 넘어서, 저수준 커널 드라이버에서 센서 데이터가 어떻게 샘플링되고 uORB 메시지로 퍼블리싱되는지 파이프라인을 추적한다. 나아가, 이 국소 센서(Local Sensor) 관측치들이 최종적으로 EKF2(Extended Kalman Filter) 추정기 내부에서 전역 센서(GPS/Barometer) 데이터와 어떻게 융합(Fusion)되어 드론의 절대 상태(State)를 강건하게 방어해 내는지 수식과 소스코드 레벨에서 철저히 분석할 것이다.
