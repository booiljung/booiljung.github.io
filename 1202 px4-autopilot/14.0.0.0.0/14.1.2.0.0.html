## 0.1 PX4-Autopilot vs Ardupilot: лЎңм»¬ м„јм„ң мІҳлҰ¬ м•„нӮӨн…ҚмІҳ л№„көҗ

мҳӨн”ҲмҶҢмҠӨ л¬ҙмқён•ӯкіөкё° мғқнғңкі„мқҳ м–‘лҢҖ мӮ°л§Ҙмқё PX4-Autopilotкіј ArdupilotмқҖ лҸҷмқјн•ң н•ҳл“ңмӣЁм–ҙ(Pixhawk мӢңлҰ¬мҰҲ л“ұ)мҷҖ лҸҷмқјн•ң LiDAR л°Ҹ Optical Flow м„јм„ңлҘј м§Җмӣҗн•Ём—җлҸ„ л¶Ҳкө¬н•ҳкі , лӮҙл¶Җм ҒмңјлЎң м„јм„ң лҚ°мқҙн„°лҘј нҡҚл“қн•ҳкі  мІҳлҰ¬н•ҳлҠ” мҶҢн”„нҠёмӣЁм–ҙ м¶”мғҒнҷ”(Software Abstraction) л°©мӢқкіј м•„нӮӨн…ҚмІҳ мІ н•ҷм—җм„ң к·№лӘ…н•ң м°ЁмқҙлҘј ліҙмқёлӢӨ.

Ardupilotмқҙ мҳӨлһң м—ӯмӮ¬ мҶҚм—җм„ң кІҖмҰқлҗң лӢЁмқј лЈЁн”„(Single-loop) м§Җн–Ҙнҳ• C++ к°қмІҙ м§Җн–Ҙ лқјмқҙлёҢлҹ¬лҰ¬ мЎ°н•© л°©мӢқ(Monolithic Architecture)мқ„ кі мҲҳн•ңлӢӨл©ҙ, PX4лҠ” мөңмӢ  н•ӯкіө м „мһҗ мҶҢн”„нҠёмӣЁм–ҙмқҳ нҠёл Ңл“ңмқё uORB кё°л°ҳмқҳ л¶„мӮ°нҳ• л§ҲмқҙнҒ¬лЎңм„ңл№„мҠӨ м•„нӮӨн…ҚмІҳ(Micro-services Architecture)мҷҖ POSIX к·ңкІ©мқҳ лӘЁл“Ҳ кІ°н•©лҸ„лҘј мІ м ҖнһҲ л”°лҘҙкі  мһҲлӢӨ.

мқҙ кІ°м •м Ғмқё мІ н•ҷмқҳ м°ЁмқҙлҠ” мӢңмҠӨн…ң к°ңл°ңмһҗк°Җ мғҲлЎңмҡҙ көӯмҶҢ м„јм„ңлҘј м»ӨмҠӨн…Җ нҺҢмӣЁм–ҙм—җ мқҙмӢқ(Porting)н•ҳкұ°лӮҳ, л№„н–ү мӨ‘ н•ҳл“ңмӣЁм–ҙ кІ°н•Ё(I2C лІ„мҠӨ н–ү л“ұ)мқҙ л°ңмғқн–Ҳмқ„ л•Ң мӢңмҠӨн…ңмқҙ лҢҖмІҳн•ҳлҠ” мғқмЎҙл Ҙ(Survivability) м–‘мғҒм—җ мӨ‘лҢҖн•ң мҳҒн–Ҙмқ„ лҜём№ңлӢӨ. н•ҳмң„ кі„мёөмқҳ л“ңлқјмқҙлІ„ лЎңм§Ғл¶Җн„° мғҒмң„ кі„мёөмқҳ м„јм„ң мңөн•©кё°(Kalman Filter)м—җ мқҙлҘҙкё°к№Ңм§Җ м–‘лҢҖ м§„мҳҒмқҳ м„Өкі„ мӮ¬мғҒмқ„ мӢ¬мёөм ҒмңјлЎң н•ҙл¶Җн•ңлӢӨ.
