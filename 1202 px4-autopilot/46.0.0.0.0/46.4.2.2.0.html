46.4.2.2. 수평면(XY-Plane) 2차원 원통형 수용 영역(Cylindrical Acceptance Region) 투영 및 수직축(Z-Axis) 선형 수용 영역과의 독립적 논리곱(Logical AND) 연산 회로

멀티로터의 도달 판별(Arrival Determination) 알고리즘은 3차원 유클리드 공간 전체를 단일 구(Sphere)로 취급하지 않고, 수평(XY) 기동과 수직(Z) 기동을 분리해 기하학적 교집합(Intersection)으로 판단하는 비선형 로직을 지닌다. PX4-Autopilot은 이 분리된 논리 회로를 통해 위치 제어 과정에서 발생할 수 있는 특이점(Singularity)이나 강건성 저하를 방지한다 (기반 펌웨어 버전: PX4-Autopilot v1.14 기준). 본 절에서는 수평면에서의 2차원 원통형 투영과 수직축 선형 영역 간의 독립적 논리곱(Logical AND) 연산 구조를 집중적으로 살펴본다.

1. 기하학적 수용 영역의 분리 및 투영(Projection)

복잡한 3차원 위치 오차 벡터 \mathbf{E}_{pos} = \begin{bmatrix} e_x \\ e_y \\ e_z \end{bmatrix}는 논리 회로 상에서 두 개의 독립적인 하위 성분으로 영사(Projection)되어 처리된다.

1.1 수평면(XY-Plane) 2차원 원통형 수용 영역

기체의 수평 오차 벡터 \mathbf{E}_{xy} = \begin{bmatrix} e_x \\ e_y \end{bmatrix}는 목표 좌표계를 원점으로 하는 2차원 평면으로 투영된다.
네비게이터(Navigator)는 이 \mathbf{E}_{xy} 벡터의 유클리드 크기(L2-Norm)가 설정 파라미터 NAV_ACC_RAD 보다 작은지를 평가한다.

이 조건만을 3차원 공간으로 환원(Back-projection)하여 생각하면, Z축 방향으로는 무한한 높이를 갖는 원통형(Cylindrical) 수용 영역이 형성됨을 알 수 있다. 즉, 기체가 목표 좌표의 바로 수백 미터 상공에 있더라도 수평면 조건식 자체는 참(True)으로 반환된다.

1.2 수직축(Z-Axis) 1차원 선형 수용 영역

고도 오차 스칼라 e_z = z - z_{sp} (NED 좌표계 기준이므로 보통 Down 방향이 양수)는 목표 고도를 중심으로 하는 상하 1차원의 스칼라 구간으로 취급된다.
수직축 수용 조건은 고도의 절대 오차 |e_z|가 NAV_MC_ALT_RAD보다 작은지 확인하는 것이다.

이 조건만을 3차원 공간으로 환원하면, X축과 Y축 방향으로 무한히 펼쳐진, 두 겹의 평행판(Parallel plates) 수용 영역이 형성된다.

2. 독립적 논리곱(Logical AND) 연산 회로

최종적인 도달 여부 A_{reached}는 앞서 서술한 두 가지 독립적인 기하학적 영역이 겹치는 3차원 교집합(Intersection) 구간으로 기체가 진입했을 때 비로소 선언된다. 이는 프로그래밍 논리 상 두 조건식 간의 논리곱(AND, &&) 회로로 귀결된다.

A_{reached} = (\|\mathbf{E}_{xy}\|_2 < R_{xy}) \land (|e_z| < R_z)
(여기서 R_{xy}는 NAV_ACC_RAD, R_z는 고도 수용 파라미터를 의미한다.)

2.1 독립 연산의 제어공학적 강건성(Robustness)

이렇게 수평과 수직의 조건을 엄격하게 독립시키는 논리 회로 체계는 제어 시스템의 오작동을 강력하게 억제한다.

예를 들어, 기체가 심한 측풍(Crosswind)을 받아 XY 좌표로는 아직 목적지에 도달하지 못했지만, 고도 Z는 일찍이 맞춰진 상황을 가정하자. 3차원 구(Sphere) 방정식 \sqrt{e_x^2 + e_y^2 + e_z^2} < R_{global} 하나만으로 식을 통합해두었다면, e_z = 0이 전체 식의 결괏값을 크게 낮추어, 실제로는 e_{xy} 오차가 제법 큼에도 불구하고 구의 경계선을 통과한 것으로 잘못 인식(False Positive)될 수 있다.

현대 멀티로터의 동역학 응답 속도는 프로펠러의 양향성 제약 때문에 Z축 상승/하강 응답과 XY축 기울어짐 응답에서 현저한 대역폭(Bandwidth) 차이를 보인다. PX4가 채택한 2D 원통 + 1D 선형 영역의 논리곱 방식은 동역학 응답이 가장 늦는 축이 오차 제약에 완벽히 들어올 때까지 다음 상태 전이를 홀딩(Holding)함으로써 각 축의 비대칭성을 효과적으로 보상하고 제어 무결성(Integrity)을 보장하는 설계이다.
