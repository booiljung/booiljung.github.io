28.3.2.3.2. FlightTaskFailsafe: GPS 상실 또는 데이터 링크 단절 시 위치 추정 에러를 고려한 안전 하강(Safe Descent) 모델

드론이 자율 비행 중 GPS 안테나가 파손되거나, 지상 관제소와의 데이터 링크가 완전히 끊긴 절망적인 상황을 상상해 보자. 기체를 안정적으로 호버링(Hovering)시키려 해도, 내 위치를 알 수 없으니 제어기는 폭주하기 십상이다.

이러한 최악의 하드웨어/통신 결함 시나리오가 커맨더(Commander) 데몬에 의해 감지되면, 다른 모든 화려한 비행 모드들은 즉각 강제 종료되고 최후의 생존 프로토콜인 FlightTaskFailsafe 모드가 FMM(Flight Mode Manager)의 제어권을 강제로 탈취한다.

1. 위치 추정(Position Estimate)의 포기와 개방 루프(Open-loop) 선언

FlightTaskFailsafe의 가장 큰 특징은 **“현재 센서가 뱉어내는 모든 2차원 평면 위치 데이터(X, Y)를 쓰레기로 간주하고 무시한다”**는 철학이다.

  • 수평 제어의 포기:
    GPS가 고장 났는데 억지로 제자리 위치를 고수하려(Position Hold) 들면, EKF2 추정기는 미친 듯이 튀는 가짜 위치 데이터를 기반으로 모터에 풀(Full) 파워 역추력을 걸어버릴 것이다. 기체는 순식간에 뒤집히고(Flip) 추락한다.
    따라서 FlightTaskFailsafe는 update() 루틴에서 X, Y 평면의 위치 및 속도 셋포인트(_trajectory_setpoint) 계산을 아예 포기(Bypass)한다.
  • 자세 유지(Attitude Leveling)로의 강등:
    대신 XY 평면에 대해서는 단순히 “기체를 수평(Level)으로만 유지하라“는 원시적인 Roll/Pitch = 0 명령만을 하달한다. 비록 기체는 측풍에 의해 정처 없이 바람에 떠밀려(Drift) 가겠지만, 적어도 뒤집혀서 자유 낙하하는 끔찍한 사태는 막을 수 있다.

2. 안전 하강(Safe Descent) 궤적 모델

XY 평면의 제어를 포기했다면, 남은 것은 기체가 배터리가 다 떨어져 하늘에서 돌덩이처럼 떨어지기 전에 어떻게든 부드럽게 지면에 코를 박게 만드는 것뿐이다(Controlled Crash).

  • 기압계 의존적 Z축 제어:
    GPS 고도조차 믿을 수 없는 상황에서, 이 태스크는 가장 신뢰도가 높은(고장이 잘 안 나는) 내부 통합 기압계(Barometer)의 변화량 데이터만을 바탕으로 Z축 하강 제어 루프를 구성한다.
  • 고정된 맹목적 하강 속도(Blind Descent Rate):
    파라 파라미터 체계에는 비상시 기체가 땅을 향해 꽂히는 속도를 정의하는 튜닝값(예: NAV_DESC_STAT, 보통 1.0 ~ 2.0 m/s 수준의 매우 느린 속도)이 존재한다. FlightTaskFailsafe는 오로지 이 고정된 음의 속도(-Z) 목표치만을 생성하여 하위 제어기에 전달한다.

3. 착륙 판정과 모터 정지(Disarm)

기체가 바람에 떠밀리며 맹목적으로 하강하다가 마침내 나무 위나 땅바닥에 처박혔다면, 모터가 계속 돌며 화재를 일으키는 것을 막아야 한다.

  • 충격 / 바운싱 감지:
    FlightTask 기저 클래스와 FMM은 기체가 하강 명령(명령된 -Z 속도)을 지속적으로 내리고 있음에도 불구하고, 실제 Z축 고도 변화율(기압계 미분치)이 수 초간 0으로 수렴하거나 가속도계에 강한 **스파이크(충격)**가 감지되면, 이를 “수풀이나 지면 등 어딘가에 물리적으로 안착했다(Landed)“고 판정한다.
  • 생존 사이클의 종료:
    착륙 판정이 떨어지는 즉시 FlightTaskFailsafe는 커맨더(Commander) 모듈을 향해 “비상 착륙 완료” 플래그를 던진다. 커맨더는 이 신호를 받는 즉시 즉각적인 모터 정지(Disarm) 명령을 액추에이터 라우팅을 통해 강제 하달하며, 드론의 구슬픈 Failsafe 생애 주기는 끝을 맺게 된다.

이 독립되고 원시적인 FlightTaskFailsafe 클래스 덕분에, 다중 센서 융합 시스템이 완전히 붕괴된 위기 상황 속에서도 PX4 드론은 지상에 있는 인명 피해를 최소화하고 기체 프레임의 완파를 막아내는 기적 같은 복구력(Resilience)을 발휘할 수 있는 것이다.
