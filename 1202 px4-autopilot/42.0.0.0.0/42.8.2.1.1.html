42.8.2.1.1 강건 제어(Robust Control) 이론 기반의 H-무한대(H_\infty) 놈(Norm) 경계치 확장을 통한 외란 수용성 분석

1. 개요 및 분석의 필요성

PX4-Autopilot의 시동 허용 오차(Tolerance) 파라미터를 경험적 직관(Heuristic Intuition)에 의존하여 단순히 증가시키는 행위는 치명적인 오작동 스펙트럼을 기체에 방치하는 결과를 낳는다. 따라서 비행 동역학 모델(Flight Dynamics Model) 불확실성과 외부에서 유입되는 외란(Disturbance) 사이의 관계를 수학적으로 정규화하여, 제어기가 수용할 수 있는 한계치를 정량 평가할 필요가 있다.

이 절에서는 현대 제어 이론인 강건 제어(Robust Control) 기법 중 H-무한대(H_\infty) 놈(Norm) 해석을 도입하여, 시동 조건(Arming Condition) 완화 시 제어 시스템이 감내할 수 있는 외란의 최대 경계치(Boundary)를 확장하고 이의 신뢰성을 수학적으로 분석하는 방법론을 제시한다.

2. 제어 시스템 내의 외란과 불확실성 모델링

일반적인 멀티로터의 선형화된 동작 상태 공간 모델(State-Space Model)은 외란 피드백을 포함하여 다음과 같이 정의된다.

\begin{aligned} \dot{x}(t) &= A x(t) + B u(t) + B_w w(t) \\ z(t) &= C_z x(t) + D_{zu} u(t) + D_{zw} w(t) \\ y(t) &= C_y x(t) + D_{yw} w(t) \end{aligned}

  • w(t) : 외부 유입 외란(External Disturbance) 및 센서 측정 노이즈 벡터 (예: 돌풍, 진동, 센서 바이어스)
  • z(t) : 성능 평가 출력(Performance Output), 즉 제어 품질을 판별하는 지표
  • y(t) : 측정 출력(Measured Output), PX4 EKF2가 계산하는 실제 좌표와 자세 등

여기서 시동 허용 오차(COM_ARM_IMU_ACC 등) 임계값 완화 작업은 수학적으로 외부 유입 외란 w(t)의 초기 수용 크기 폭을 인위적으로 증가시킴을 의미한다.

3. H-무한대(H_\infty) 놈(Norm)을 활용한 수용성 제약

시스템의 폐루프 전송 함수(Closed-loop Transfer Function)를 T_{zw}(s)라 할 때, H-무한대(H_\infty) 이론의 목적은 외란 w가 성능 지표 z에 미치는 최악의 에너지 증폭률(Worst-Case Energy Gain)을 최소화하거나 특정 수준 이하로 제약하는 것이다. H_\infty 놈은 주파수 응답의 최대 특이값(Maximum Singular Value)을 의미하므로 다음과 같이 정의된다.

\| T_{zw} \|_\infty = \sup_{\omega} \bar{\sigma} ( T_{zw}(j\omega) ) < \gamma

(\bar{\sigma}는 행렬의 최대 특이값, \gamma는 성능 경계(Performance Bound) 상수이다.)

3.1 임계값 확장에 따른 \gamma 상승 메커니즘

PX4의 시동 완화 설정은 시스템이 초기 오차(Initial Tolerance)를 내재한 상태에서 이륙하게 만듦으로써, 본질적으로 초기 상태 추정 불확실성을 가속화한다. 이 시나리오에서 H-무한대 정리로 분석 시, 초기 오차 허용 경계의 증가는 폐루프 전송 함수의 \gamma 값을 상향시키는 결과를 초래한다.

즉, 높은 \gamma 환경이 조성됨에 따라 롤오버(Rollover) 등 기체 추락에 임박하는 최악의 조건(Worst-case Scenario)이 발현될 확률이 커진다.

3.2 강건 통제 조건 (Robustness Condition)

소형정칙정리(Small Gain Theorem)에 의해, 플랜트(Plant)의 모델 불확실성 상한을 \Delta(s) (단, \|\Delta\|_\infty \le 1)라 하고 이에 묶인 가중치 함수를 W_m(s)라 할 때, 안정성을 잃지 않으려면 다음을 만족해야만 한다.

\| W_m(s) T(s) \|_\infty < 1

Tolerance 임계값을 넓힌다는 것은 W_m(s) 대역 내에서 수용해야 할 물리적 모델 불확실성을 넓히겠다는 선언이므로, PX4 내부 제어기(PID) 루프 T(s)는 이를 상쇄할 수 있도록 훨씬 보수적(Conservative)이고 강건하게 튜닝되어야 함을 시사한다.

4. 외란 수용성 분석 기반 안정성 평가 (SITL 모의 실험 루틴)

이러한 강건 제어 모델링을 오퍼레이터가 직접 실무에서 검증하기 위한 가상 환경 평가 워크플로우를 제시한다.

  1. 파라미터 변경: COM_ARM_* 그룹의 임계값을 조정한다.
  2. 외란 주입(Disturbance Injection): Gazebo 환경 내에서 돌풍(Wind Gusts)이나 동적 부하(Dynamic Payload Shift) 스크립트를 주입하여 시스템 전송 함수상의 w(t) 에너지를 고갈에 가깝게 포화시킨다.
  3. 노름 경계 측정: 로그 분석(Flight Review)에서 Actuator Outputs의 진동수준(Oscillation Level)과 EKF2의 누적 이노베이션(Accumulated Innovation) 에러 출력 z(t)를 비교한다.
  4. 결과 피드백: 증폭률이 과도하게 상승하여 엑추에이터 포화(Actuator Saturation)에 도달한다면, 해당 기체는 완화된 시동 조건 속에서 통제 불능에 빠질 확률이 100% 임을 뜻한다. 즉, 해당 임계값 확장은 **안전 허용선 초과(Bounds Exceeded)**로 기각해야 한다.

5. 결론

PX4-Autopilot의 시동 오차(Tolerance) 매개변수를 조정할 때, 이를 단순한 ’하드코딩(Hardcoding)된 상수 변경’으로 접근해서는 안 된다. H-무한대(H_\infty) 노름 이론은 시동 전 임계값 확장이 비행 동역학 시스템 폐루프의 최악의 에러 증폭(Worst-case Error Amplification)으로 직결됨을 수학적으로 증명한다.

따라서 시동 제약의 예외 처리가 수반되어야 하는 임무는, 필히 강건 제어 관점의 시뮬레이션(Small Gain Theorem Validation)을 거쳐 이 임계값이 폐루프의 안정성을 붕괴시키지 않음을 사전 수용성 분석표(Acceptance Analysis Matrix) 형태로 입증한 뒤에 수행해야만 한다.
