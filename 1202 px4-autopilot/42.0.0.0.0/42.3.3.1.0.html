42.3.3.1 타원체 피팅(Ellipsoid Fitting) 알고리즘을 통한 연자성 및 경자성(Soft/Hard Iron) 왜곡 계수 추정 검증

자기장 센서(Magnetometer)가 외부의 간섭 없이 순수한 지구 자기장만을 측정한다면, 3차원 공간상에서 기체를 모든 방향으로 회전시킬 때 그려지는 측정 데이터 포인트들의 다발(Point Cloud)은 완벽한 중심 원점(0,0,0)을 갖는 구(Sphere) 형태를 이루어야 한다. 그러나 현실의 무인항공기(UAS) 제어 보드와 기체 프레임에는 모터, 배터리, 나사못 등 다양한 강자성/상자성 물질이 섞여 존재한다. 이로 인한 경자성(Hard Iron) 왜곡과 연자성(Soft Iron) 왜곡은 이 완벽한 구면 궤적을 찌그러진 타원체(Ellipsoid) 형태로 변형시키고 중심 좌표계마저 이동시킨다.

PX4-Autopilot의 나침반 캘리브레이션 루틴은 수집된 원시(Raw) 측정 데이터 다발을 바탕으로 이 타원체의 기하학적 방정식을 역산하는 타원체 피팅(Ellipsoid Fitting) 알고리즘을 가동한다. 본 절에서는 해당 피팅 알고리즘의 수학적 전개와 연/경자성 왜곡 계수를 추정하여 검증하는 PX4의 내부 아키텍처에 대하여 상세히 분석한다.

1. 경자성 및 연자성 왜곡의 공간 수학적 모델링

지자기 데이터를 타원체 피팅으로 교정하는 수학적 행렬 방적식은 3차원 아핀 변환(Affine Transformation)에 기반을 둔다. 교정된 완벽한 자기장 벡터 \mathbf{m}_{calib} 는 미가공 측정 벡터 \mathbf{m}_{raw} 에 대해 다음과 같은 형태로 근사된다.

\mathbf{m}_{calib} = \mathbf{W} (\mathbf{m}_{raw} - \mathbf{V})

앞선 절에서 거론된 식과 위상학적으로 동일하나, 기하학적 의미를 덧붙여 다시 분해하자면 다음과 같다.

  • 오프셋 벡터 \mathbf{V} (경자성 왜곡 추론): 영구적으로 기체 프레임에 부착된 자성체가 뿜어내는 ’편향성(Bias)’이다. 수학적으로는 찌그러진 타원체의 ‘중심점(Center)’ 좌표를 나타낸다.
  • 형태 왜곡 행렬 \mathbf{W} (연자성 왜곡 추론): 기체의 철재 부품에 지구 자기장이 닿아 유도 결합(Inductive Coupling)되면서 타원체의 축을 팽창/수축시키거나 비틀어버리는 교차 축 민감도(Cross-axis Sensitivity)이다. 대칭적인 3 \times 3 행렬 형태를 지니며, 구면을 어떻게 찌그러뜨렸는지 타원체의 ’회전과 스케일 팩터’를 담고 있다.

타원체 피팅의 본질은 N개의 방대한 센싱 데이터 집합 \{\mathbf{m}_{raw, i}\}_{i=1}^N 이 주어졌을 때, 변환된 벡터 \mathbf{m}_{calib} 의 크기가 지구 글로벌 노름(Norm) 1 상수 R에 수렴하도록 모델 비용 함수(Cost Function)
\sum_{i=1}^{N} \left( \vert\vert \mathbf{W} (\mathbf{m}_{raw, i} - \mathbf{V}) \vert\vert^2 - R^2 \right)^2
를 최소화(Minimize)하는 \mathbf{W} 와 \mathbf{V} 파라미터 12개(대칭행렬 가정 시 9개)를 수치해석적으로 확정하는 과정이다.

2. PX4 아키텍처: 타원체 피팅 최소 자승법 루틴

Commander 모듈은 MAVLink의 캘리브레이션 명령을 수신하면 지정된 면(Side)별로 고정된 상태(Static)에서 자력 데이터를 버퍼링한다. 모든 면의 수집이 완료되면 내부 수학 라이브러리(matrix 기반)를 빌려 비선형 최소 자승법(Non-linear Least Squares) 또는 선형화된 고유값 분해(Eigenvalue Decomposition) 기반 피팅 코드를 호출한다.

피팅 완료 후 획득한 중심점 \mathbf{V} 는 PX4 파일럿 파라미터 내 CAL_MAG0_XOFF, YOFF, ZOFF로 매핑되고, 형태 행렬의 주 대각 스케일 팩터 성분은 CAL_MAG0_XSCALE, YSCALE, ZSCALE 파라미터에 직접 반영되어 영구 저장된다.

2.1 캘리브레이션 적합도(Fitness) 및 유효성 한계 검증

단지 수학적 계수가 구해졌다고 끝이 아니다. PX4는 타원체 피팅 데이터가 너무 납작하거나 기괴한 위치에 존재하는 허구의 폐곡선이 아닌지 검열한다.

  1. 오프셋 한계 검증 (Hard Iron Limit): 연산된 타원 중심 \mathbf{V} 벡터의 길이가 터무니없이 클 경우, 즉 기존 지구자기장 대비 70% 이상의 오프셋을 가진다면, 기기 옆에 거대한 자석이 탑재되었다는 의미이므로 캘리브레이션 루틴을 고의로 실패(Fail) 처리한다.
  2. 연역적 투영 반경 검증 (Soft Iron Limit): 스케일 \mathbf{W} 의 팩터가 기본 1.0 체제에서 0.5 미만 혹은 1.5를 초과하는 등 기괴한 값으로 떨어질 경우, 센서 칩셋 자체의 브라켓 뒤틀림(Hardware Defect)으로 판정하고 경고를 내보낸다.

3. Ardupilot의 타원체 피팅 아키텍처(AP_Compass)와의 비교 철학

  • PX4-Autopilot: 면(Side) 중심의 엄격한 “6방향 정방위 정지(Static) 캘리브레이션“을 통한 샘플 포인트 추출을 기초로 타원체 피팅을 전개한다. 정지 상태의 포인트 클라우드 군집화를 통해 모터 진동이나 움직임에서 기인하는 동적 가속도 커플링(Coupling) 노이즈를 철저히 이격시킨 뒤 순수한 지자기 평균점을 추출, 이로써 연/경자성을 깔끔하게 분리(Decoupling)하려는 엄격하고 통제된(Controlled) 학술적 모델링 궤도를 지향한다.
  • Ardupilot: ’Fitness Ratio’를 강조한 완전한 3D 구체(Sphere) 스윕(Sweep) 방식을 사용한다. 비행기를 허공에 들고 무작위로 계속 돌리면서(Dance Calibration) 표면에 수천 개의 막대한 포인트 클라우드를 엉기게 수집한다. 이후 Levenberg-Marquardt 식이나 구면 경계 탐색 알고리즘에 방대한 점을 그대로 때려 넣어 타원체를 피팅해낸다. 연/경자성 분리도는 높으나 정지되지 않은 환경 탓에 노이즈 필터링 최적화 부담은 더 크다. 사용의 편의성은 Ardupilot 쪽이 범용적이지만, PX4는 센서 축이 정방향과 이루는 공간 직교성을 이론적으로 정교하게 잡아내는 데 유리하다.

4. 자율 비행(GCS, ROS2) 제어를 위한 응용 가이드

신형 기체나 무거운 페이로드(Payload)를 장착한 특수 목적 드론을 설계했다면, 캘리브레이션 타원체 피팅 결과 피드백을 통해 기구학적 설계의 옳고 그름을 추론해야 한다.

  • 파라미터 분석을 통한 역설계 추적: 캘리브레이션 성공 후 도출된 오프셋(CAL_MAGx_XOFF 등)의 절대 편차가 특정 축(예: X축)으로 유독 거대하게 쏠려있다면, 이는 해당 축 방향 기판에 고전류 DC 스위칭 레귤레이터나 강철 볼트가 배치되어있기 때문이다. 설계자는 피팅 편향 계수 자체를 일종의 “자기-안테나 노이즈 지향성 디버거“로 해석하여, FC 기판이나 지지대(Mast)의 방향을 180도 돌려보는 하드웨어 물리 수정 조치를 취해야 한다.
  • 복합 센서 융합 관점(ROS2): ROS2의 uORB 메시지 로거(sensor_mag)에서 추출된 3D 데이터가 오목하거나 찌그러진 타원체의 파편처럼 보인다면, 타원체 피팅 파라미터가 비정상적으로 저장된 것이다. ROS2 노드 안에서 VIO 카메라 오도메트리와 EKF의 헤딩(Yaw)이 자꾸 어긋난다면 주저하지 말고 CAL_MAG 소프트 아이언/하드 아이언 캘리브레이션 튜닝을 QGC에서 외부 야지에 나가 재수행(Re-calibration)할 것을 권고한다.

5. 결론

타원체 피팅 알고리즘은 센서를 괴롭히는 철골(Steel) 구조물 및 전류 덩어리로부터 생성된 자장 역학 왜곡(Hard/Soft Iron)이라는 질병을 진단하고 치료(Cure)하는 수술대와 같다. PX4-Autopilot은 이 타원체 피팅 곡선이 통계적으로 최소 기준을 통과하지 못할 시 과감히 MAG INCONSISTENT로 시스템 이륙을 차단한다. 개발자들은 단순히 녹색 바(Bar)가 100% 차오르는 캘리브레이션 ‘완료(Done)’ 상태 창에 만족할 것이 아니라, 그 심연에 반영된 \mathbf{V}와 \mathbf{W}의 기하수치 팩터가 기체의 물리적 강건성을 어떻게 반증하고 있는지 파라미터 창을 직접 열람하여 평가하는 습관을 들여야 한다.
