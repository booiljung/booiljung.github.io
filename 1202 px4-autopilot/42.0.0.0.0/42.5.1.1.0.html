42.5.1.1. 쿨롱 계수(Coulomb Counting) 적분 오차 보정을 위한 개방 회로 전압(Open Circuit Voltage) 비선형 매핑

무인항공기(UAV)의 정밀한 비행 시간 예측을 위해 PX4-Autopilot은 단순히 전압을 측정하는 방식을 넘어, 흐르는 전류를 적분하여 소모된 용량을 추정하는 쿨롱 계수(Coulomb Counting) 기법을 기본적으로 활용한다. 그러나 센서의 미세한 스케일 오차(Scale Error)나 온도 변화에 따른 노이즈 누적 등으로 인해 순수 적분 방식은 비행시간이 길어질수록 실제 잔량과의 오차가 커지는(Drift) 문제를 안고 있다.
이러한 한계를 극복하기 위해 PX4(Firmware v1.14 기준)는 쿨롱 계수의 적분 결과와 개방 회로 전압(Open Circuit Voltage, OCV) 기반의 비선형 방전 곡선(Discharge Curve) 매핑 결과를 융합(Fusion)하는 고도화된 배터리 잔량(State of Charge, SoC) 추정 알고리즘을 사용한다. 본 장에서는 이 보정 메커니즘의 수학적 원리와 소스 코드 수준의 아키텍처를 심도 있게 분석한다.

1. 쿨롱 계수(Coulomb Counting)의 원리와 한계

배터리 모듈에서 측정된 전류 I(t)를 시간대별로 적분하여 소모된 총 전하량(전력량)을 바탕으로 잔량을 산출하는 방식이 쿨롱 계수이다.

SoC_{coulomb}(t) = SoC_0 - \frac{1}{C_{total}} \int_0^t I(\tau) d\tau

  • SoC_0: 초기 배터리 잔량 백분율(일반적으로 연결 직후의 OCV를 통해 추정)
  • C_{total}: 배터리의 총 명목 용량(Nominal Capacity, 파라미터 BAT1_CAPACITY로 설정)
  • I(\tau): 시간 \tau에 측정된 순간 소비 전류

1.1 적분 오차(Drift) 발생 원인

  1. 전류 센서 노이즈 및 오프셋: 파워 모듈이나 전류 센서 칩셋 자체의 선형성 불량(Non-linearity)이나 아날로그 오프셋(Offset)이 존재하면, 오차가 지속적으로 적분(누적)되어 비행 후반부에는 실제 용량과 큰 차이를 보이게 된다.
  2. 배터리 노후화(Aging): 배터리는 충·방전 사이클이 늘어날수록 실제 C_{total}이 명목 용량보다 적어진다. 쿨롱 계수만으로는 이 수명 저하 측면의 용량 감소분을 추정할 수 없다.

2. 개방 회로 전압(OCV) 비선형 매핑을 통한 보정

위와 같은 오차 보정을 위해, PX4는 현재 측정된 단자 전압(V_{meas})과 전류 데이터로부터 내부 저항(R_{in}) 영향을 배제한 ’개방 회로 전압(OCV)’을 추정한다.

2.1 무부하 상태 OCV 역추산

비행 중 기체가 강한 추력을 내면 단자 전압(V_{meas})은 배터리의 등가 직렬 저항(ESR)에 의해 급강하한다. PX4는 다음과 같은 등식으로 부하가 없을 때의 이상적 전압(OCV)을 역추산한다.

OCV = V_{meas} + I \cdot R_{in}

  • R_{in} 파라미터(BAT1_R_INTERNAL): 배터리의 내부 저항. 사용자가 기입하거나, PX4의 자체적 동적 저항 추정 알고리즘에 의해 필터링된 값이 사용된다.

1.1 OCV 기반 SoC 맵핑 (SoC_{voltage})

도출된 OCV 전압을 사전 정의된 배터리의 비선형 방전 다항식 혹은 구간별 선형 보간표에 대입하여, 전압 관점에서의 잔량인 SoC_{voltage}를 산출한다.

  • 맵핑 특성: OCV 전압이 뚝 떨어지는 완전 방전 직전(Knee point)에 다다르면 SoC_{voltage}의 낙폭이 급격히 커진다.

2. PX4의 보정 및 융합 알고리즘

결론적으로 PX4의 battery_status 알고리즘은 단기적으로 높은 정밀도를 띄는 쿨롱 계수(SoC_{coulomb})와 센서의 누적 오차가 없는 장기적 관점의 전압 맵핑 잔량(SoC_{voltage})을 가중 평균(Weighted Average)하거나 보정 인자를 주어 최종 SoC를 결정한다.

2.1 상호 보완 로직 구현 (가중치 적용)

펌웨어 내부적으로는 비행 시간, 전류량, 및 배터리 전압 영역(평탄 구간인지 방전 막바지 구간인지)에 따라 가중치 w를 동적으로 조절하여 융합한다.

// PX4 배터리 융합 로직 개념 (단순화된 예시)
float SoC_final;
if (OCV > V_FLAT_ZONE_LOWER) {
    // 평탄 구간: 전압 맵핑 오차가 크므로 쿨롱 계수의 가중치를 높임
    SoC_final = w * SoC_coulomb + (1 - w) * SoC_voltage; 
} else {
    // 완전 방전 임박: 전압 하강이 뚜렷하므로 OCV 기반 맵핑을 더 신뢰 (w를 작게 조절)
    SoC_final = (w_small) * SoC_coulomb + (1 - w_small) * SoC_voltage;
}

이와 같이 PX4는 비행 극후반부로 접어들 때 SoC_{voltage}의 반영 비율을 늘림으로써, 쿨롱 계수의 적분 오차가 쌓여 “잔량이 남았다고 표기되지만 기체가 추락하는” 치명적인 버그를 원천 봉쇄한다.

4. 결론 및 요약

PX4-Autopilot의 배터리 추정 알고리즘은 쿨롱 계수(Coulomb Counting)의 실시간 정밀 적분 능력과 OCV(Open Circuit Voltage) 비선형 매핑의 절대적 기준력(장기 안정성)을 융합하는 고도화된 아키텍처이다. 적분 과정에서 필연적으로 발생하는 전류 센서 노이즈오프셋에 의한 수치 누적(Drift) 오차를 방지하기 위해, 배터리 내부 저항을 고려하여 OCV를 복원해내는 과정은 극히 과학적이고 산업적인 신뢰성을 자랑한다.
따라서, GCS 운항 담당자는 파라미터(BAT1_CAPACITY 및 BAT1_R_INTERNAL)를 배터리 특성에 맞게 최대한 정확히 기입해야 이 융합 루프가 제대로 동작하고 안정적인 Return-to-Launch 시간을 담보 받을 수 있음을 필수적으로 인지해야 한다.
