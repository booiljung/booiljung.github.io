Chapter 29.2. 하드웨어 드라이버 및 조종기(RC) 입력 전처리 커널 분석 (src/modules/manual_control)

본 절에서는 무인항공기 비행 제어를 위한 물리적 첫 관문인 하드웨어 드라이버와 수동 조종 제어기(RC)의 입력 전처리(Pre-processing) 커널 아키텍처를 심도 있게 해부한다. 특히, src/modules/manual_control 데몬 및 이와 긴밀히 연동되는 하위 디바이스 드라이버 레이어를 분석함으로써, 이기종 통신 프로토콜로 유입되는 원시(Raw) 전기적 신호가 PX4-Autopilot 시스템 내에서 어떻게 정제되고 표준화된 기구학적 데이터로 캐스팅(Casting)되는지 학술적으로 접근한다.

조종자의 의도가 투영된 펄스(Pulse) 및 직렬(Serial) 데이터는 수신 버퍼 처리, 채널 맵핑, 스케일링, 비선형 곡선 적용, 지터(Jitter) 억제 등 다양한 디지털 신호 처리(DSP) 및 소프트웨어 캘리브레이션 루프를 거치게 된다. 본 장에서는 이러한 전파 신호가 비행 궤적 정보로 전환되는 일련의 커널 레벨 파이프라인과 다중 관제 소스(RC 대 GCS) 간의 런타임 제어 권한 다중화(Multiplexing) 시스템을 상세히 다룰 것이다.
