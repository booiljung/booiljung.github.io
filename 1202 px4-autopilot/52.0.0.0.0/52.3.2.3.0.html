52.3.2.3. 배터리 동적 부하 모델 기반 내부 저항(IR) 증가율과 전압 강하 하한선(Cut-off) 교차점 예측 모델 연동

본 절에서는 무인항공기(UAV)의 비상 제어(Emergency Control) 및 비행 종료(Flight Termination) 시스템을 트리거하는 다변수 제어 조건 중, **배터리의 동적 부하(Dynamic Load)**와 내부 저항(Internal Resistance, IR) 증가 추이를 수리 모델링하여 임계 전압 하한선(Cut-off Voltage)과의 교차점을 예측하는 알고리즘을 심혈을 기울여 해석한다. 본 내용은 PX4-Autopilot v1.14 환경을 기준으로 기술된다.

1. 배터리 동적 부하로 인한 전압 강하(Voltage Sag)의 공학적 모델링

리튬 폴리머(LiPo) 배터리와 같은 화학 전지는 전류 부하(I)가 급격히 증가할 때, 옴의 법칙(V = I \times R)에 의해 배터리의 내부 저항(IR)으로 인해 출력 단자 전압이 순간적으로 하강하는 전압 강하(Voltage Sag) 현상을 겪는다.

무인항공기의 경우, 고속 상승 비행이나 강풍 보상(Wind Compensation) 시 구동계 액추에이터(모터)에 막대한 전류가 소모된다. 만약 단순한 임계 전압 필터(Static Voltage Threshold)만을 사용하여 페일세이프(Failsafe)나 비행 종료를 결정한다면, 순간적인 전류 스파이크(Current Spike)로 인한 전압 강하를 배터리 소진(Depletion)으로 오인하는 불필요한 거짓 양성(False Positive) 비상사태가 발생할 수 있다.

1.1 내부 저항(IR) 증가율의 비선형성

배터리의 방전율(Depth of Discharge, DoD)이 증가하고 배터리의 코어 온도가 변화함에 따라 내부 저항(IR) 값은 비선형적으로 증가한다. 따라서 제어 시스템은 정적인 배터리 모델이 아닌, 실시간 관측 가능한 전류와 전압을 소스로 하는 동적 추정 필터(Dynamic Estimation Filter)를 적용해야 한다.

2. PX4-Autopilot의 내부 저항 및 전압 강하 예측 알고리즘 아키텍처

PX4는 battery_status 모듈 내에서 1차 저역 통과 필터(Low-Pass Filter) 및 전류 적산(Coulomb Counting)을 결합하여 동적 부하 모델을 구현한다. 이를 통해 하한선(Cut-off)과의 실질적인 교차점을 예측한다.

2.1 소스 코드 레벨 구조 이해

관련 연산은 주로 src/modules/battery_status의 라이브러리에서 수행되며, uORB 토픽 battery_status를 통해 발행(Publish)된다.

  • 무부하 전압(Resting Voltage) 추정: 현재 측정된 전압(V_{meas})과 전류(I_{meas}), 그리고 기하급수적으로 갱신되는 예측 내부 저항(IR_{est})을 활용하여 실제 소진 상태를 반영하는 무부하 전압(V_{rest} = V_{meas} + I_{meas} \times IR_{est})을 연산한다.
  • 필터링된 잔여 시간 예측: V_{rest} 값이 하한선(BATx_V_EMPTY)과 만나는 지점(교차점)을 현재의 평균 전류 소모량으로 적분하여 남은 비행 시간(Time-to-Empty)을 예측한다.

2.2 파라미터 연동 구조 (Configuration)

시스템 설정 시 관측되는 주요 변수는 다음과 같다.

  • BATx_V_EMPTY: 셀당 방전 종료 임계 전압 (일반적으로 LiPo 기준 3.3V ~ 3.5V).
  • BATx_R_INTERNAL: 사용자가 공칭(Nominal) 값으로 명시하거나, 내부 연산을 통해 추적되는 배터리 기본 내부 저항 설정.
  • BATx_V_LOAD_DROP: 최대 스로틀(Throttle) 개방 시 허용되는 최대 전압 강하의 하한 마진.

3. Ardupilot 대비 비상 제어 로직 차이점

PX4와 Ardupilot 모두 훌륭한 배터리 헬스 모니터링 시스템을 갖추고 있으나, 필터링 관점에서 설계 철학의 세부적 차이가 존재한다.

  • PX4-Autopilot: 상태 추정 모듈 내에서 1차 필터(First-Order Filter)의 시상수(Time Constant) 파라미터를 명시적으로 노출하여, 기체 중량 대비 출력(Thrust-to-Weight Ratio)이 높은 레이싱 드론부터 고정익에 이르는 다양한 기체의 전압 노이즈 특성을 매우 정교하게 튜닝할 수 있도록 아키텍처가 설계되어 다. 즉, 일시적인 스파이크와 지속적인 방전을 수학적으로 강력히 분리하는(Decoupling) 모델 연동 메커니즘을 제공한다.
  • Ardupilot: AP_BattMonitor 라이브러리를 통해 전류 적산과 전압 필터링을 수행한다. 배터리 페일세이프(Battery Failsafe) 액션을 단계별(Low, Critical)로 트리거하는 데 중점을 두며, 내부 저항 추정을 실시간 비상 제어의 하드 임계(Hard Threshold)로 직결하기보다는 주로 사용자 경고나 안전 복귀(RTL)의 척도로 활용하는 경향이 강하다.

4. 지상 관제 시스템(GCS) 연동 및 QGroundControl에서의 시각화

지상 관제 시스템은 통신 단절(Data Link Loss) 및 비행 종료 모니터링 체계를 구성하기 위해 배터리의 상태 데이터를 필수적으로 의존한다. MAVLink 데이터 스트림(BATTERY_STATUS)은 내부 주기 로직에 의해 정기적으로 GCS로 발행되며, QGroundControl은 이 메시지를 파싱하여 다음과 같은 관제 연산을 수행한다.

  1. 상태 시각화: QGC 백엔드(BatteryFactGroup)는 퍼센티지(Percentage) 외에도 남은 비행 시간을 동적으로 환산하여 UI에 표기한다.
  2. 비상 제어 트리거 예측: 관제사(Operator) 또는 자율 에이전트(Autonomous Agent)는 동적 부하 모델이 교차점을 지시하기 직전, MAVLink 통신 대역폭 지연을 고려하여 사전 비행 모드 전환(예: 자동 착륙(Land), 고도 유지(Hold))을 수행하는 방어적 관제 기술을 적용할 수 있다.

5. 요약

고신뢰도를 요구하는 비행 종료 시스템이나 비상 정지 메커니즘에서, 단일 전압 샘플링에 의한 의사 결정은 중대한 시스템적 오류를 범할 수 있다. PX4-Autopilot은 단순 비교가 아닌 **잔여 역량의 수학적 투영(Mathematical Projection)**을 지원하는 내부 저항 모델을 사용하며, 제어 엔지니어는 기체의 구동계 부하 프로파일에 맞춰 배터리 모델 파라미터를 최적화해야 생존성과 시스템 안정성을 도모할 수 있다.
