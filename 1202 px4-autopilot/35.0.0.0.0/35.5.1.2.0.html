35.5.1.2. 절대 위치 추정(Absolute Position Estimation) 모듈의 전역 기준점(Global Origin) 획득 여부 판단 및 로컬 투영(Local Projection) 준비성

PX4-Autopilot이 위치 고정(Position Hold), 출발지 복귀(RTL), 혹은 지정된 경로 이동(Waypoint Mission)과 같은 자율 비행 기동을 수행하기 위해서는 기체가 우주 공간 속 자신의 위치를 수리적으로 확정지어야만 한다. 하지만 둥근 지구 표면의 위경도(WGS84) 좌표계는 매 프레임마다 모터 출력을 제어해야 하는 고속 PID 제어 루프가 연산하기에는 너무 느리고 복잡하다.

본 절에서는 EKF2(Extended Kalman Filter 2)가 어떻게 전 지구적 절대 좌표계를 획득(Global Origin)하고, 이를 제어기가 소화할 수 있는 평면 직교 좌표계로 로컬 투영(Local Projection)하는지, 그리고 비행 전(Pre-flight) FSM이 이 아키텍처의 완성도를 어떻게 검증하는지 분석한다.

1. 전역 기준점(Global Origin) 획득의 엄밀성 검증

전원이 인가된 기체가 야외(Outdoor) 환경에 놓이면, GPS/GNSS 수신기는 위성 신호를 탐색하기 시작한다. FSM의 PreflightCheck 모듈은 이 과정에서 단순히 “수신된 위성의 개수“만을 묻지 않고, 측정의 질적 깊이를 엄격하게 평가한다.

  1. 위성 가시성 및 기하학적 정밀도: 최소 위성 수(기본 6개 이상)는 물론이고, 위성의 배치 상태를 의미하는 HDOP(수평 정밀도 저하율, Horizontal Dilution of Precision)와 VDOP(수직 정밀도 저하율) 파라미터가 EKF2_REQ_HDOP 통과 기준(예: 2.0 미만)을 만족해야 한다.
  2. 혁신 분산(Innovation Variance) 수렴: EKF2는 GPS가 제공하는 위치와 IMU가 적분해 낸 궤적을 융합한다. 두 센서 간의 위치 오차 혁신(Position Innovation) 값이 일정 임계치 이내로 축소되어 융합 필터가 수학적 안정을 찾을 때 비로소 vehicle_global_position uORB 토픽의 is_valid 플래그가 참(True)으로 전환(Assertion)된다.

이 까다로운 조건이 모두 충족되었을 때, EKF2는 현재의 위도, 경도, 그리고 평균 해수면(AMSL) 고도를 **전역 기준점(Global Origin)**으로 선언(Set Home)하고 EEPROM 및 시스템 메모리에 잠금(Lock) 처리한다.

2. 평면 직교 좌표계로의 로컬 투영(Local Projection) 최적화

글로벌 위치가 픽스(Fix)되면, 제어 시스템은 그 즉시 지구 곡률을 무시할 수 있는 3차원 유클리드 평면 공간(NED: North, East, Down)을 기체의 발밑에 펼쳐버린다.

  • 토폴로지 매핑(Topology Mapping): MapProjection 수학 라이브러리에 의해, 방금 획득한 전역 기준점(Global Origin)이 로컬 NED 좌표계의 완전한 영점 (X=0, Y=0, Z=0) 이 된다. 이제부터 제어기는 위성 항법의 도/분/초 단위가 아닌, 미터(Meter) 단위의 X축(북쪽), Y축(동쪽), Z축(지하 방향) 직교 좌표계를 대상으로 빠르고 직관적인 물리 역학 연산을 수행한다.
  • 이를 통해 PX4의 고속 위치 제어기(FlightTask 계열)는 복잡한 구면 삼각법(Spherical Trigonometry)을 우회하고 선형 PID 연산만으로 기체를 밀리미터 단위로 미세 조종할 수 있는 강력한 연산 이득(Computational Gain)을 얻는다.

3. 자율 비행 FSM의 Arming 인터락(Interlock) 메커니즘

비행 전(Pre-flight) 단계에서 GCS(지상 관제소)가 자율 임무(Mission) 모드 이륙을 명령하면, FSM은 다음의 부울 대수(Boolean Logic) 검증을 통과해야만 한다.

if (intended_mode == MISSION || intended_mode == POSITION) {
    if (!global_position_locked || !local_projection_ready) {
        return ARMING_DENIED_MISSING_POSITION;
    }
}

만약 실내에서 억지로 미션 이륙을 시전하거나, GPS 수신 품질이 저하되어 로컬 NED 평면이 채 깔리지 않았다면, GCS에는 “ARMING DENIED: NAV ALIGNMENT” 에러 메시지가 퍼블리싱되며 액추에이터 출력이 하드웨어적으로 거절된다.

다만, 오직 인간 조종사의 수동 스틱 컨트롤에 의지하는 ‘매뉴얼(Manual)’ 혹은 ‘고도 고정(Altitude)’ 모드의 경우, 이 엄격한 전역 기준점 확보 검증을 바이패스(Bypass)할 수 있는 권한 부여(COM_ARM_WO_GPS) 경로가 마련되어 있어 실내 검수 및 수동 제어의 유연성을 발휘한다.

결론적으로 절대 위치 투영 시스템의 개입은, 하늘을 날고자 하는 자율 비행 에이전트가 “나는 현재 우주의 어느 절대 좌표에 존재하며, 내 1미터 앞은 논리직교계 상에서 어디인가?“라는 존재론적 질문에 완벽히 대답할 수 있을 때까지 스스로의 이륙을 인내하게 만드는, 자율 비행의 가장 근원적인 안전 장치이자 핵심 철학이다.
