40.3.1.1.1. 저역 통과 필터(Low-Pass Filter) 관점의 임펄스 응답(Impulse Response) 특성

QGroundControl(설명 기준: v4.3)과 PX4-Autopilot(설명 기준: v1.14) 간의 MAVLink 통신에서, 패킷 왕복 시간(RTT)을 추정하기 위해 사용되는 지수 가중 이동 평균(EWMA) 필터는 신호 처리 관점에서 볼 때 1차 무한 임펄스 응답(IIR: Infinite Impulse Response) 저역 통과 필터(Low-Pass Filter)와 동일한 구조를 가진다. 본 절에서는 통신망에 갑작스러운 지연 스파이크(Spike)가 발생했을 때 이 필터가 어떻게 반응하는지, 임펄스 응답(Impulse Response) 특성을 통해 해석한다.

1. EWMA 필터의 임펄스 응답 모델링

단위 임펄스 신호 \delta[n]은 시간 n=0에서만 1이고 나머지 모든 시간에서는 0인 입력 신호다. 이는 드론의 통신 환경에서 평상시 0ms 지연을 유지하다가, n=0 순간에 정확히 한 번 매우 큰 네트워크 지연(예: 간섭으로 인한 처리 지연)이 발생한 상황을 수학적으로 이상화한 것이다.

EWMA 필터의 차분 방정식은 다음과 같다.
y[n] = (1 - \alpha) y[n-1] + \alpha x[n]

여기에 입력 x[n] = \delta[n]을 인가하고, 초기 조건 y[-1] = 0이라 가정하면 출력(임펄스 응답) h[n]은 다음과 같이 전개된다.

  • n = 0: h[0] = (1-\alpha)(0) + \alpha(1) = \alpha
  • n = 1: h[1] = (1-\alpha)(\alpha) + \alpha(0) = \alpha(1-\alpha)
  • n = 2: h[2] = (1-\alpha)[\alpha(1-\alpha)] + 0 = \alpha(1-\alpha)^2
  • …
  • n = k: h[k] = \alpha(1-\alpha)^k \quad (k \ge 0)

즉, EWMA 필터의 임펄스 응답 h[n]은 \alpha(1-\alpha)^n u[n] (여기서 u[n]은 단위 계단 함수)의 형태를 지닌 매끄러운 지수 붕괴(Exponential Decay) 곡선이다.

2. 저역 통과 필터(Low-Pass Filter)로서의 특성 해석

네트워크 지연 시간 데이터 모음(RTT 트레이스)에는 두 가지 주파수 성분이 혼재되어 있다.

  1. 고주파(High-Frequency) 성분: 무선 채널의 잡음(Noise)이나 일시적인 처리 지연으로 인해 발생하는 짧고 급격한 지연 스파이크.
  2. 저주파(Low-Frequency) 성분: 기체와의 거리 증가나 통신 채널 대역폭 저하로 인해 서서히 증가하는 유효한 핑(Ping) 상승 추세.

임펄스 응답이 지수적으로 천천히 붕괴한다는 것은 시스템이 과거 입력들을 누적하여 평균화(Averaging)한다는 수학적 의미다. 따라서 짧은 순간에 발생하는 고주파 핑 스파이크는 필터 출력에 곱해지는 가중치 \alpha (예: 0.125)에 의해 즉각적으로 억제(Attenuation)된다. 반대로, 지연이 장기적으로 지속되는 저주파 성분은 누적합 연산을 거쳐 원형을 그대로 통과시킨다(Pass).

3. PX4 및 QGroundControl에서의 MAVLink 타임아웃 적용 사례

만일 QGroundControl이 MAVLink 트랜잭션을 처리할 때 매번 들어오는 순수한 RTT 샘플 x[n]만을 기준으로 타임아웃 타이머를 1:1로 적용한다면(즉, \alpha=1인 All-Pass Filter 적용), 단 한 번의 통신 간섭 임펄스에도 타임아웃이 3~4초로 급격히 길어지는 스래싱(Thrashing)이 발생할 것이다.

이를 방지하기 위해 필터 계수 \alpha를 0.125 수준으로 설계함으로써, 단일 임펄스(Spike)가 RTO(Retransmission Timeout) 추정치에 미치는 최대 한도를 전체 지연 변동의 12.5\%로 엄격히 제한하고 있다. 이후 수신되는 정상적인 패킷들에 의해 임펄스의 잔영은 (1-0.125)^n의 비율로 신속히 감쇠하여 시스템이 원래의 베이스라인(Baseline)으로 부드럽게 복귀한다.

4. 임펄스 반응 파형의 실무적 직관

엔지니어는 필터의 임펄스 응답을 보고 시스템의 ’기억력’을 가늠할 수 있다.

  • \alpha가 1에 가까울수록 임펄스 응답은 델타 함수에 가까워져(메모리 없음), 입력값을 그대로 추종한다.
  • \alpha가 0에 가까울수록 임펄스 응답은 길고 넓게 퍼져(긴 메모리), 과거의 영향이 시스템에 매우 오래 잔존한다.

결론적으로 저역 통과 관점의 임펄스 응답 폭(Width)을 분석하는 것은, GCS 관제 환경에서 순간적인 통신 단절을 노이즈로 보고 무시할 것인지, 아니면 페일세이프(Failsafe) 발동의 전조로 판단하여 즉각 타임아웃 한계선을 갱신할 것인지를 결정하는 가장 중요한 제어 공학적 지표가 된다.
