40.3.1.1. 지수 가중 이동 평균(EWMA) 평활화 필터의 극점-영점(Pole-Zero) 해석

무인항공기(UAV)와 지상 관제 시스템(GCS) 사이의 MAVLink 통신 타임아웃을 동적으로 추정하기 위해 가장 범용적으로 쓰이는 필터가 지수 가중 이동 평균(EWMA: Exponentially Weighted Moving Average) 상수 이득 필터이다. 본 절에서는 시계열 제어 이론의 관점에서 EWMA 필터를 Z-변환(Z-Transform)하여 극점(Pole)과 영점(Zero)을 추출하고, 필터의 동적 특성과 안정성 마진(Stability Margin)을 명확히 해석한다.

1. EWMA 필터의 이산 시간 시스템 지배 방정식

EWMA 기반 왕복 지연 시간(RTT) 추정 필터는 다음과 같은 1차 차분 방정식(Difference Equation)으로 정의된다.

y[n] = (1 - \alpha) y[n-1] + \alpha x[n]

  • x[n]: n 번째로 측정된 새로운 MAVLink 패킷의 RTT 샘플
  • y[n]: n 번째 시점에서 갱신된 평활화 지연 시간 (Smoothed RTT)
  • \alpha: 평활화 인자(Smoothing Factor), 0 < \alpha < 1

QGroundControl 및 PX4-Autopilot의 내부 네트워크 모니터링 모듈에서 \alpha 값은 일반적으로 1/8 (0.125) 등으로 고정되어 적용된다.

2. Z-변환(Z-Transform)과 전달 함수(Transfer Function) 유도

필터의 동적 응답을 주파수 영역(s-domain의 이산형 변환)에서 해석하기 위해 양변에 Z-변환을 취한다.
초기 조건이 0이라고 가정하면 Z-변환의 이동 성질(Time Shifting Property)에 의해 Y(z)z^{-1}이 적용된다.

Y(z) = (1 - \alpha) z^{-1} Y(z) + \alpha X(z)

위 식을 전달 함수 H(z) = \frac{Y(z)}{X(z)}로 정리하면 다음과 같다.

H(z) = \frac{\alpha}{1 - (1 - \alpha) z^{-1}} = \frac{\alpha z}{z - (1 - \alpha)}

3. 극점-영점(Pole-Zero) 위치 해석

전달 함수 H(z)의 분모와 분자를 0으로 만드는 z 값을 각각 극점(Pole)과 영점(Zero)이라고 한다.

  • 영점(Zero): z = 0
  • 극점(Pole): z = 1 - \alpha

안정성(Stability) 판별:
이산 시간 제어 시스템이 점근적으로 안정(Asymptotically Stable)하기 위해서는 모든 극점이 Z-평면의 단위원(Unit Circle, \vert z \vert < 1) 내부에 위치해야 한다.
EWMA 필터에서 0 < \alpha < 1 이므로, 극점은 항상 0 < 1 - \alpha < 1 범위를 가지며 단위원 내부에 존재한다. 따라서 이 시스템은 항상 안정성을 보장(BIBO Stable)한다.

4. 제어 파라미터 \alpha의 실무적 튜닝

통신 지연의 튀는 현상(Spike)을 제거하고, 의미 있는 지연 상승 추세를 감지하기 위해서는 모델의 극점을 조절하는 일(Tuning \alpha)이 결정적이다.

  1. 지수 붕괴(Exponential Decay) 관점:
    과거 데이터의 가중치는 (1-\alpha)^k의 비율로 감소한다. 시정수(Time Constant) \tau는 대략 \tau \approx 1/\alpha 로 근사할 수 있다. \alpha=0.125인 경우, 약 8회의 MAVLink 트랜잭션이 지나면 최신 RTT 모델에 완전히 반영됨을 의미한다.
  2. 노이즈 크기에 따른 적응적 극점 배치:
    기체가 가시권(VLOS: Visual Line of Sight) 밖으로 멀어져 텔레메트리 연결 신호 강도(RSSI)가 떨어지면, \alpha 값을 강제로 줄여 극점(Pole)을 z=1에 가깝게 이동시켜 주파수 차단 대역(Cut-off Frequency)을 낮춰야 한다. 반면 LTE망과 같이 대역폭이 충분하나 불규칙한 라우팅 지연이 발생하는 통신 환경에서는 \alpha를 0.2 \sim 0.3 수준으로 높여 시스템 반응성을 확보하는 방안을 고려하라.

이와 같은 극점-영점 제어 이론 분석은 GCS가 통신 상태 변화에 따라 MAVLink 타임아웃 파라미터를 실시간으로 자가 튜닝(Self-Tuning)할 수 있는 학술적 기반을 제공한다.
