40.1.3. 은닉 마르코프 모델(Hidden Markov Model) 기반 무선 링크 에러 상태 추론

무인항공기(UAV)의 가시거리(LOS) 및 비가시거리(NLOS) 통신 환경에서는 다중 경로 페이딩과 간섭으로 인해 MAVLink 패킷 손실이 무작위적(Random)이거나 군집적(Bursty)으로 발생한다. 이러한 통신 링크의 에러 상태는 지상 관제 시스템(GCS)에서 실시간으로 직접 관측하기 어렵다. 본 절에서는 은닉 마르코프 모델(Hidden Markov Model, HMM)을 활용하여 관측 가능한 MAVLink 데이터 패킷의 정상 수신 여부로부터 기저에 존재하는 통신 채널의 보이지 않는 에러 상태(Hidden State)를 수학적으로 추론하고 대응하는 기법을 분석한다.

1. 통신 링크의 은닉 마르코프 모델(HMM) 정식화

통신 채널의 상태는 시간에 따라 변화하며, 가장 단순하고 널리 쓰이는 시변 에러 모델은 길버트-엘리엇 모델(Gilbert-Elliott Model)이다. 이는 2-상태 HMM의 일종으로, 채널의 상태를 ’좋음(Good)’과 ‘나쁨(Bad)’ 두 가지 은닉 변수로 정의한다.

  • 은닉 상태(Hidden States) S = \{G, B\}
  • G (Good) : 비트 에러율(BER)이 매우 낮아 MAVLink 패킷이 정상적으로 수신되는 상태
  • B (Bad) : 페이딩이나 교란으로 BER이 급증하여 패킷 유실(Drop)이 집중적으로 발생하는 상태
  • 관측 기호(Observations) O = \{0, 1\}
  • 0 : MAVLink 패킷 유실 (ACK 미수신 또는 CRC 오류)
  • 1 : MAVLink 패킷 정상 수신

1.1 길버트-엘리엇 2-상태 전이 모델

채널이 G 상태에서 B 상태로 전이할 확률을 P_{gb}, 반대로 B에서 G로 전이할 확률을 P_{bg}라고 할 때, 상태 전이 행렬 파라미터는 드론의 비행 속도와 환경(도심지, 해상 등)에 의해 결정된다.

GCS(예: QGroundControl)는 송신한 COMMAND_LONG 메시지에 대한 COMMAND_ACK 수신 여부 시퀀스를 관측(O)함으로써, 비터비 알고리즘(Viterbi Algorithm)이나 전방-후방 알고리즘(Forward-Backward Algorithm)을 통해 현재 링크가 일시적인 B 상태인지 아니면 영구적인 절단 상태인지를 실시간으로 추론할 수 있다.

2. PX4-Autopilot 및 GCS 관점의 에러 추론 적용

PX4-Autopilot 환경에서는 텔레메트리 모듈(SiK Radio, Microhard 등)에서 하드웨어적으로 에러 정정(FEC) 및 재전송 메커니즘을 1차 수행하지만, Application Layer인 MAVLink 수준에서는 여전히 잔류 에러가 관측된다. HMM 기반의 에러 추론은 GCS 알고리즘 개발 시 다음과 같이 활용된다.

2.1 동적 타임아웃 및 재전송 제어

단순한 타이머 기반 타임아웃 시스템에서는 고정된 시간(예: 0.5초) 내에 ACK가 오지 않으면 무조건 패킷을 재전송한다. 그러나 HMM 관측 결과 현재 채널이 깊은 페이딩 장기구간(B 상태의 체류 확률이 높음)에 진입했다고 추론될 경우, 즉시 재전송하는 것은 대역폭 낭비일 뿐만 아니라 채널 혼잡(Congestion) 상태를 가중시킨다. GCS 에이전트는 채널이 다시 G 상태로 전이할 확률적 기댓값을 계산한 뒤, 타임아웃을 적응적으로 연장표준(Adaptive Backoff)하여 패킷을 지연 발송해야 한다.

3. Ardupilot과 PX4의 통신 품질 모니터링 비교

  • Ardupilot (Mission Planner): 링크 품질(Link Quality)을 단순히 최근 N개 패킷 중 손실된 패킷의 이동 평균(Moving Average) 비율로 계산하여 UI에 노출시킨다. 이는 직관적이지만 군집 에러(Bursty Error) 상황에서 미래의 패킷 성공 확률을 예측하기에는 한계가 있다.
  • PX4-Autopilot (동적 관제 설계 시): 고도화된 자율 에이전트 비행 관제 시스템(GCS)을 구축할 때, MAVSDK를 활용하여 패킷 드랍 시퀀스를 수집하고 백그라운드에서 HMM 전이 확률 비터비 디코딩(Viterbi Decoding)을 수행하라. 추론된 현재 네트워크 상태 공간(State Space)이 B에 고착되었다면, 기체에 RTL 명령을 내리거나 경로 계획(Path Planning) 알고리즘을 즉각 수정하여 통신 음영 지역을 탈출하도록 제어 로직을 구현할 수 있다.

4. 결론

무인기 텔레메트리 링크의 패킷 유실을 이산적인 이벤트로만 치부하지 않고, 은닉 마르코프 모델(HMM)을 통해 시계열적인 상태 전이 과정으로 정식화하는 것은 고급 관제 시스템 설계의 핵심이다. 관찰된 패킷 수신 성공/실패 시퀀스로부터 채널의 숨겨진 상태를 선제적으로 추론함으로써, 불필요한 재전송을 막고 Data Link Loss 상황을 지능적으로 예방하는 강건한 제어망을 구축하라.
