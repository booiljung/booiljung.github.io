40.1.1. 대기권 RF 전파의 다중 경로 페이딩 확률 밀도 함수(PDF) 분석

무인기(UAV) 운영 환경에서 지상 관제 시스템(GCS)과 PX4-Autopilot이 탑재된 기체 간의 무선 주파수(RF) 통신은 열악한 대기권 전파 특성의 지배를 받는다. 텔레메트리 안테나에서 송신된 신호는 지표면, 건물, 나무 또는 대기권의 굴절 등으로 인해 여러 갈래로 나뉘어 수신 안테나에 도달하며, 이를 다중 경로 전파(Multipath Propagation)라고 일컫는다. 다중 경로 페이딩 현상은 수신 신호의 진폭과 위상에 확률적인 변화를 일으켜 MAVLink 패킷 유실률을 급격히 상승시키는 주된 원인이 된다. 본 절에서는 GCS 명령 타임아웃 및 재전송 설계의 근간이 되는 다중 경로 페이딩 현상의 확률 밀도 함수(PDF, Probability Density Function)를 해석적으로 분석한다.

1. 다중 경로 페이딩(Multipath Fading)의 공학적 모델링

송신단(GCS)에서 전파된 MAVLink COMMAND_LONG 메시지는 여러 반사체와 산란체에 부딪혀 각각 다른 경로장(Path Length)과 도달 시간(Time of Arrival)을 가지고 기체의 수신단 안테나에 도착한다. 이 때 여러 시그널은 수신 안테나에서 중첩(Superposition)되며 보강 간섭(Constructive Interference)과 상쇄 간섭(Destructive Interference)을 무작위로 발생시킨다.

특히 위상 차이가 180도(\pi)에 근접한 신호들이 상쇄 간섭을 일으키는 딥 페이딩(Deep Fading) 상태에서는 수신 신호 전력(Receive Signal Power)이 민감도(Sensitivity) 이하로 떨어져 패킷이 복구 불능 상태가 된다. QGroundControl은 이러한 페이딩이 유발하는 일시적인 패킷 로스를 극복하기 위해 Timeout 파라미터를 갖는 큐(Queue)를 설계하게 된다.

2. 레일리 페이딩(Rayleigh Fading) 모델과 PDF 분석

도심지나 울창한 산림과 같이 GCS 방향의 직접 가시거리(LOS, Line-of-Sight)가 완전히 차단된 비가시선(NLOS, Non-Line-of-Sight) 환경에서, 수신 신호의 포락선(Envelope) r은 보통 레일리(Rayleigh) 분포를 따른다.

2.1 레일리 확률 밀도 함수 (Rayleigh PDF)

수신되는 다중 경로 신호들의 직교 성분(In-phase 및 Quadrature 성분)이 각각 평균이 0인 독립적인 가우스 분포(Gaussian Distribution)를 갖는다고 가정할 때, 포락선 확률 변수 R의 확률 밀도 함수 f(r)은 다음과 같이 정의된다.

f(r) = \begin{cases} \frac{r}{\sigma^2} e^{-\frac{r^2}{2\sigma^2}} & \text{for } r \geq 0 \\ 0 & \text{for } r < 0 \end{cases}

여기서:

  • r: 수신 신호 포락선의 진폭
  • \sigma^2: 송신 신호의 단일 직교 성분의 평균 전력 (Time-average power of the received signal before envelope detection)

이 모델은 PX4가 파라미터를 업로드/다운로드하거나 미션 웨이포인트를 교환하는 도중에 직접파가 단절될 경우의 극한 페이딩 환경을 수학적으로 설명한다. 이 PDF 누적 분포(CDF)를 적분하게 되면, 수신 신호가 특정 스레시홀드 R_{th} 미만으로 떨어져 페이딩 아웃티지(Outage)가 발생할 확률을 산출할 수 있다.

3. 라이시안 페이딩(Rician Fading) 모델과 PDF 분석

레일리 페이딩과 달리, 드론이 상공을 비행하며 GCS 간의 직접 가시거리(LOS) 경로가 확보되는 경우를 고려해야 한다. LOS 성분이 다중 경로 성분보다 우세한 경우의 포락선 분포는 라이시안(Rician) 분포를 따른다.

3.1 라이시안 확률 밀도 함수 (Rician PDF)

우세한 고정 성분(Specular component)과 약한 산란 성분들을 함께 모델링하기 위해, 라이시안 PDF는 0차 제1종 변형 베셀 함수(Modified Bessel function of the first kind of order zero) I_0(\cdot)를 포함하여 다음과 같이 전개된다.

f(r) = \frac{r}{\sigma^2} e^{-\frac{r^2 + A^2}{2\sigma^2}} I_0\left( \frac{rA}{\sigma^2} \right), \quad r \geq 0

여기서 A는 LOS 경로를 통해 수신된 결정론적(Deterministic) 우세 신호의 최대 진폭을 의미한다.

3.1.1 라이스 팩터 (K-factor)

안테나 엔지니어링에서는 보통 LOS 전력과 다중 경로 전력의 비(Ratio)를 나타내는 K-factor를 사용한다. K가 0에 수렴하면 순수한 레일리 페이딩 모델이 되고, K가 무한대에 근접할수록 페이딩 없는 깨끗한 AWGN 채널 구조를 띄게 된다.

K = \frac{A^2}{2\sigma^2}

일반적인 고도에서의 PX4 멀티로터 비행(예: 30~50m) 시, K-factor는 대략 5~15dB 범위 내에서 변동한다. 이 때 패킷 유실률은 레일리 모델보다 훨씬 개선되므로, GCS 개발 시 타임아웃 주기를 재설정하는 데 중요한 힌트가 된다.

4. GCS의 MAVLink 재전송(Retry) 파라미터 튜닝 시사점

PX4 개발자와 GCS 운용자는 비행 환경이 NLOS(Rayleigh PDF 지배)인지, 강한 LOS(Rician PDF 지배)인지에 따라 타임아웃과 재전송을 다르게 설계해야 한다.

  1. NLOS 환경(건물 밀집 지역 등): 레일리 분포에서 딥 페이딩 확률이 기하급수적으로 높아진다. 이때는 타임아웃 허용 시간을 늘리거나(예: 1초 이상), 재전송 횟수(Max Retries) 제한을 높여(예: 5회~7회) 기체가 잠시 통신 음영지에 들어갔다 빠져나오는 기간을 버틸 수 있도록 해야 한다.
  2. LOS 환경(개활지 비행): 라이시안 분포의 특성상 딥 페이딩은 잘 발생하지 않는다. 따라서 타임아웃을 상당히 짧게(예: 0.3초) 유지하여, GCS가 제어 커맨드 대기열을 신속하게 비워 QGroundControl 렌더링 스레드의 프리징(Freezing) 가능성을 줄이는 것이 유리하다.

결론적으로, 다중 경로 페이딩의 PDF 함수는 일견 이론적인 수학 함수로 보이지만, 실질적인 MAVLink 트랜잭션의 생존성을 평가하고 백엔드 C++ 타임아웃(QTimer) 주기를 검토하는 정량적인 지표(Metrics)를 제공한다.
