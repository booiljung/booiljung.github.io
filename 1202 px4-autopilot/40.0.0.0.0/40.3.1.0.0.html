40.3.1. 자기회귀 누적 이동평균(ARIMA) 모델 기반 네트워크 지연 예측

지상 관제 시스템(GCS)과 PX4-Autopilot를 탑재한 무인항공기(UAV) 간의 통신은 RF 통신망(예: 433MHz/915MHz 텔레메트리, LTE/5G 연동망)의 물리적 변화나 대역폭 제약으로 인해 지연 시간(Latency)의 변동성(Jitter)이 심각하게 나타난다. 단순한 지수 가중 이동 평균(EWMA) 모델만으로는 복잡한 비선형적 지연 변동 패턴이나 장기적인 추세를 예측하기 어렵다. 본 절에서는 시계열 분석의 대표적 기법인 자기회귀 누적 이동평균(ARIMA: Autoregressive Integrated Moving Average) 모델을 MAVLink 제어 타임아웃의 척도로 활용하는 방안 및 수학적 백그라운드를 조명한다.

1. ARIMA 모델의 개요와 MAVLink 트랜잭션 지연

ARIMA 모델은 시계열 데이터가 가지는 과거의 관측값과 오차항을 기반으로 미래의 값을 예측하는 확률론적 모델이다. MAVLink 패킷 왕복 시간(RTT) 데이터를 분석할 때, ARIMA 모델은 ARIMA(p, d, q)의 형태로 정의된다.

  • p (AR: Autoregressive): 자기회귀 차수. 과거 p개의 RTT 관측값이 현재 지연 시간에 미치는 영향(가중치)을 모델링한다.
  • d (I: Integrated): 차분(Difference) 차수. 통신망 지연의 비정상성(Non-stationarity)을 제거하여 정상성(Stationarity) 있는 데이터로 변환하기 위해 RTT 차이값을 몇 번 구하는지를 나타낸다. 보통 d=1을 통해 장기적인 핑(Ping) 증가 추세(Trend)를 억제한다.
  • q (MA: Moving Average): 이동평균 차수. 과거 q개의 예측 오차(Error)가 현재 RTT에 미치는 영향을 모델링한다.

2. 수식 모델링

시점 t에서의 패킷 지연 시간을 Y_t라 하고, 차분 연산자 B(Backshift operator, B Y_t = Y_{t-1})를 적용한 후의 모델은 다음과 같이 전개된다.

\left( 1 - \sum_{i=1}^p \phi_i B^i \right)(1-B)^d Y_t = \left( 1 + \sum_{j=1}^q \theta_j B^j \right) \epsilon_t

  • \phi_i: 시차 i에 대한 자기회귀 계수
  • \theta_j: 시차 j에 대한 이동평균 계수
  • \epsilon_t: 백색 잡음(White Noise) 오차항

이처럼 복잡한 수리 모델을 QGroundControl(설명 기준: v4.3)이나 PX4(설명 기준: v1.14) 내부에서 직접 부동소수점 행렬 연산으로 실시간 처리하는 것은 리소스 측면에서 비효율적일 수 있다. 따라서 실무 환경에서는 미리 학습된 p, d, q 차수의 단순화된 계수를 적용하여 근사치 예측기(Approximated Predictor)를 구현하는 전략이 권장된다.

3.1. 버스트 트래픽(Burst Traffic) 대비와 모델 한계

UAV에서 긴급히 많은 MAVLink PARAM_VALUE 또는 FTP 데이터 패킷이 쏟아져 들어올 때(Burst Traffic), RTT는 선형적 변화에서 벗어나 난대수적 펄스 형태로 급등할 수 있다. ARIMA 모델은 선형 결합(Linear Combination)을 전제로 하므로 갑작스런 트래픽 충격(Impulse)을 즉각 반영하는 데는 어느 정도 시간 지연이 발생한다. 이를 보완하기 위해:

  1. 잔차 모니터링: 현재 지연값 Y_t와 예측 지연값 간의 차이 오차 \epsilon_t가 일정 임계값(Threshold)을 초과하면 (Outlier 발생), ARIMA 예측치를 우회하고 즉각적으로 보수적인 최대 타임아웃을 강제 적용하라.
  2. 윈도우 크기(Window Size) 한정: 실시간 연산량 통제를 위해 참조하는 과거 p와 q의 뎁스를 5 이하의 매우 짧은 시프트 레지스터 구조로 제한하라.

4. 제어 공학적 결론

자기회귀 누적 이동평균(ARIMA) 통신 지연 예측 기법은 GCS에서 기체로 보내는 명령(COMMAND_LONG)에 대한 타임아웃 허용 오차를 지능화하는 데 큰 도움을 준다. 단순 반응형 알고리즘을 넘어 과거의 네트워크 성능 트렌드를 학습(Learning)함으로써, 미세한 조종석 지연을 초래하는 불필요한 송신 재시도 루프를 차단하고 관제 신뢰도를 획기적으로 개선할 수 있다.
