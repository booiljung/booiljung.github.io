40.1.1.1. 레일리 페이딩(Rayleigh Fading) 환경의 비결정론적 패킷 유실률(PLR) 추정

1. 개요

지상 관제 시스템(GCS)과 PX4-Autopilot이 탑재된 무인항공기(UAV) 간의 통신에서, 가시거리(Line of Sight, LOS)가 완전히 차단된 도심지(Urban) 환경이나 빽빽한 산악 지대 비행 시에는 직접파(Direct Wave) 수신이 극도로 제한된다. 이때 수신기에 도달하는 전파는 오직 무수한 장애물에 반사되고 산란된 다중 경로(Multipath) 신호들의 합 벡터일 뿐이다. 이처럼 뚜렷한 주 신호가 없는 심각한 통신 환경의 진폭 변동을 통계적으로 나타낸 것이 레일리 페이딩(Rayleigh Fading) 모델이다.
본 장에서는 텔레메트리 라디오 모듈이 작동하는 비결정론적(Non-Deterministic) 무선 환경에서, 섀넌 용량과 확률 밀도 함수(PDF)를 결합하여 실제 예상되는 패킷 유실률(Packet Loss Rate, PLR)을 수학적으로 추정하는 기법을 소개한다.

2. 레일리 페이딩 확률 밀도 함수(PDF)의 수학적 유도

산란체가 무한히 많은 환경에서 각각의 반사파는 중심 극한 정리(Central Limit Theorem)에 의해 실수부와 허수부가 각각 독립이고 동일한 분산 \sigma^2을 가지는 영평균 가우스 분포(Zero-Mean Gaussian Distribution)를 따른다. 이 독립적인 두 직교(Orthogonal) 가우시안 변수의 크기(Magnitude) 성분 r은 곧 수신 신호의 진폭이 되며, 이는 **레일리 분포(Rayleigh Distribution)**를 따른다.

수신 신호 진폭 r의 확률 밀도 함수(PDF)는 다음과 같이 정의된다.

p(r) = \frac{r}{\sigma^2} e^{-\frac{r^2}{2\sigma^2}} \quad (r \ge 0)

2.1. 신호 대 잡음비(SNR) 중심의 지수 분포 변환

수신 진폭 r 자체보다 통신 품질을 결정짓는 핵심은 수신 전력(Power)에 기인한 신호 대 잡음비(SNR), \gamma이다. \gamma는 r^2에 비례하므로, 레일리 페이딩 환경에서 순시 신호 대 잡음비 \gamma의 분포는 다음의 지수 분포(Exponential Distribution)로 변환된다.

p(\gamma) = \frac{1}{\bar{\gamma}} e^{-\frac{\gamma}{\bar{\gamma}}} \quad (\gamma \ge 0)

  • \gamma: 측정 시점의 순시 신호 대 잡음비(Instantaneous SNR)
  • \bar{\gamma}: 장기적인 평균 신호 대 잡음비(Average SNR)

3. 비결정론적 패킷 유실률(PLR) 추정 메커니즘

GCS가 송신하는 제어 명령이나 PX4-Autopilot이 보고하는 HEARTBEAT, ATTITUDE 메시지는 변조(Modulation) 방식에 따라 일정 수준 이상의 SNR(\gamma_{th})이 확보되어야만 정상 파싱(Parsing)이 가능하다. 순시 SNR \gamma가 임계값 \gamma_{th} 이하로 떨어지는 현상을 심층 페이딩(Deep Fade)이라 한다.

3.1 페이드 아웃 확률 (Outage Probability)

심층 페이딩이 발생할 확률, 곧 해당 시점에 패킷이 유실될 비결정론적 패킷 유실률(Outage Probability, P_{out})은 p(\gamma)를 0부터 \gamma_{th}까지 적분하여 구한다.

P_{out} = P(\gamma < \gamma_{th}) = \int_0^{\gamma_{th}} \frac{1}{\bar{\gamma}} e^{-\frac{\gamma}{\bar{\gamma}}} d\gamma = 1 - e^{-\frac{\gamma_{th}}{\bar{\gamma}}}

이 수식에 따르면, MAVLink 네트워크의 평균 수신 감도 \bar{\gamma}가 높더라도 \gamma_{th}와의 비율에 따라 일정한 확률로 반드시 통신 단절(Packet Drop)이 발생한다. 이것이 레일리 페이딩 환경이 갖는 가장 치명적인 제약이다.

4. QGroundControl 및 MAVLink 관제 실무 적용

QGroundControl(QGC) 등 지상국 소프트웨어를 설계할 때, 레일리 페이딩 하의 비결정론적 PLR을 극복하기 위해 다음의 조치를 취하라.

  1. 동적 재전송(Dynamic Retransmission) 타임아웃 산정: P_{out}이 10\%를 초과하는 도심지 비행 구역에서는 MAVLink COMMAND_LONG 전송 시 첫 번째 타임아웃 대기 시간을 짧게, 두 번째부터는 지수 백오프(Exponential Backoff)를 적용토록 타이머를 튜닝하라.
  2. FEC(Forward Error Correction) 마진 확장: 단순 체크섬(Checksum) 무결성 검사뿐만 아니라 통신 계층의 인터리빙(Interleaving)을 보강하여, 버스트 에러에 대비한 여유 마진(Margin) 구조를 채택하라.
  3. UI 기반 확률론적 경고: RADIO_STATUS 메시지의 노이즈 레벨 변동폭이 클 경우(즉, \bar{\gamma}가 요동칠 경우), 운영자에게 “비가시거리 환경에 진입 중이며 패킷 드롭이 예상됨“을 QML 기반 인터페이스에 알림(Alert) 팝업으로 시각화하라.
