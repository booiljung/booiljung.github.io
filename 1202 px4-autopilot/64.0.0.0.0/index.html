  • 64.3.3.1 브레즌햄 선 알고리즘(Bresenham’s Line Algorithm)의 3차원 확장을 통한 광선 추적(Raycasting) 기반 자유 공간(Free Space) 조명 연산
  • 64.3.3.2 유클리드 부호 거리장(Euclidean Signed Distance Field, ESDF) 고속 스위핑(Fast Sweeping) 방법론의 에이코날 방정식(Eikonal Equation) 점근적 해법 계산

  • 64.4.1.2 허용적(Admissible) 및 일관적(Consistent) 휴리스틱(Heuristic) 함수 조건의 수학적 증명과 체비쇼프(Chebyshev) 거리 메트릭 성능 비교

  • 64.5.2.1 추진체 추력 벡터 변동 최소화를 위한 오일러-라그랑주 방정식(Euler-Lagrange Equation) 기반 최소 저크(Minimum Jerk) 대수 최적화

  • 64.5.2.3 비행 복도(Safe Flight Corridor) 부등식 제약과 경유점 등식 제약을 통합하는 카루시-쿤-터커(KKT) 조건 기반 이차 계획법(Quadratic Programming) 솔버 설계
