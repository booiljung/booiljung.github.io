5.8. Failsafe(안전 장치) 및 예외 처리(Exception Handling) 철학

하늘을 나는 고가의 로봇에게 컴포넌트 고장, 통신 단절, 배터리 방전은 ’만약(If)’의 문제가 아니라 ‘언제(When)’ 터질지의 문제입니다.

오토파일럿의 진정한 가치는 평화로운 순항(Cruise) 상태가 아니라, 예상치 못한 재난 상황에서 기체가 어떻게 **우아하게 성능을 저하시킬 것인가(Graceful Degradation)**를 결정하는 페일세이프(Failsafe) 아키텍처에서 드러납니다. PX4와 Ardupilot은 동일한 센서 고장에 직면하더라도, 이를 대응하는 소프트웨어 아키텍처의 설계 철학에서 큰 차이를 보입니다.

0.1 우아한 성능 저하(Graceful Degradation)의 개념

양대 펌웨어 모두 **“센서 하나가 죽었다고 하늘에서 돌멩이처럼 떨어지거나 통제 불능으로 날아가서는 안 된다”**는 대원칙을 공유합니다.

  • 만약 임무(Mission) 비행 중 GPS 모듈에 번개가 쳐서 타버렸다고 가정해 보겠습니다.
  • 잘못 설계된 시스템은 좌표를 잃고 추락하겠지만, 훌륭한 페일세이프 시스템은 즉각 GPS 신뢰도를 0으로 깎아내리고, 상태 추정기(EKF)의 의존성을 IMU와 기압계로 넘긴 뒤 자동 비행 모드에서 고도 유지(Altitude Hold) 기능만 살아있는 수동 비행 모드로 기체의 상태를 ’우아하게 강등(Degrade)’시킵니다.
  • 조종자는 비록 자동 복귀(RTL)는 불가능해졌지만, 수동으로 스틱을 조작하여 기체를 안전하게 불시착시킬 기회를 얻게 됩니다.

0.2 PX4: 엄격한 중앙 상태 머신(State Machine)의 통제

PX4는 페일세이프를 아키텍처적으로 매우 엄격하고 학술적으로 접근합니다.

  • PX4의 펌웨어 심장부에는 **커맨더(Commander)**라는 거대한 중앙 통제 모듈이 자리 잡고 있습니다.
  • 모든 센서 모듈 시스템은 독자적으로 페일세이프를 판단하지 못하며, 오직 자신의 상태(Health)를 커맨더에게 보고(Report)만 할 수 있습니다.
  • 결정권자인 커맨더 모듈은 깐깐한 상태 머신(State Machine) 구조를 통해, 특정 모드에서 특정 센서가 고장 날 경우 사전에 정의된 단 하나의 강제 상태 천이(예: Position 모드에서 GPS 유실 \rightarrow 즉시 Altitude 모드로 하락 또는 Hold 모드로 전환)만을 기계적으로 집행합니다. 이러한 중앙 집중식 단일 장애점(SPOF) 관리는 시스템의 예측 가능성을 극한으로 끌어올립니다.

0.3 Ardupilot: 실용주의 기반의 다중 안전망(Redundancy)

반면, Ardupilot은 진흙탕 같은 거친 실제 필드 테스트 경험이 집약되어 있어 훨씬 유연하고 실용적인 예외 처리 철학을 가집니다.

  • Ardupilot은 소프트웨어적인 페일세이프 상태 진입을 최대한 뒤로 미루고, 하드웨어 **센서 다중화(Redundancy)**를 통한 은폐를 선호합니다.
  • 예를 들어 EKF3 코어는 3개의 IMU와 2개의 GPS 데이터를 동시에 융합하며 실시간으로 점수를 매깁니다. 1번 GPS가 튀면 조도나 페일세이프 트리를 태우지 않고, 조용히 2번 GPS로 메인 가중치를 변경해 버립니다. 조종자는 기체가 고장 났었는지조차 모르게 비행을 마칩니다.
  • 또한, 배터리 부족 시 ’1차 경고(Warn) \rightarrow 2차 RTL \rightarrow 3차 즉시 착륙(Land)’과 같이 사용자가 파라미터 계층별로 액션을 세밀하게 프로그래밍할 수 있는 유연성을 제공합니다.

0.4 철학의 결론: 예측 가능성 대 생존력

  • PX4는 인증된 상업용 드론(Certified Drone) 시장을 겨냥하여, 특정 오류가 발생하면 미리 수리적으로 증명된 안전한 상태 트리로 즉각 떨어지는 예측 가능한(Predictable) 기계적 통제를 지향합니다.
  • Ardupilot은 “어떻게든 집으로 살려서 돌아오는 것“을 목표로, 이중 삼중의 센서 블렌딩과 각 모듈에 분산된 유연한 페일세이프 트리거를 통해 **진흙탕 속의 생존력(Survivability)**을 극대화하는 방향으로 진화해 왔습니다.
