36.3.2. 프리스 전송 방정식(Friis Transmission Equation) 확장: 자유 공간 경로 손실(FSPL)과 안테나 이득(Antenna Gain)의 적분 모델

지상 관제 시스템(GCS)과 PX4-Autopilot 기반 무인항공기 간의 안정적인 원격 전송(Telemetry) 통신망을 설계하기 위해서는 무선 전파의 감쇠량을 수학적으로 예측하는 과정이 필수적이다. 이를 위한 가장 고전적이고 핵심적인 이론이 프리스 전송 방정식(Friis Transmission Equation)이다. 본 절에서는 기본 프리스 방정식을 3차원 공간에서 동적으로 자세(Attitude)를 변경하는 기체의 안테나 지향성(Directivity)을 고려한 적분 모델로 확장하여 고찰한다.

1. 프리스 전송 방정식의 기초와 유도

가장 이상적인 자유 공간(Free Space) 환경에서 송신 안테나와 수신 안테나가 서로 최적의 방향으로 정렬되어 있을 때, 수신 전력 P_r은 프리스 방정식에 의해 다음과 같이 정의된다.

P_r = P_t \cdot G_t \cdot G_r \cdot \left( \frac{\lambda}{4 \pi d} \right)^2

  • P_r: 수신 전력 (Received Power, 단위: Watt)
  • P_t: 송신 전력 (Transmitted Power, 단위: Watt)
  • G_t, G_r: 송신 및 수신 안테나의 선형 이득 (Antenna Gain)
  • \lambda: 전파의 파장 (Wavelength, 단위: m, \lambda = c / f)
  • d: 송수신 안테나 간의 거리 (Distance, 단위: m)

이 방정식에서 거리 d의 역제곱에 비례하여 전력이 감소하는 항을 분리해 낸 것이 자유 공간 경로 손실(Free-Space Path Loss, FSPL)이다.

2. 자유 공간 경로 손실(FSPL)의 로그 스케일 표현

통신 공학에서는 전력을 로그 스케일(Decibel, dB)로 다루는 것이 계산에 용이하다. 프리스 방정식을 dBm 및 dB 스케일 변환하면, 수신 전력 모델을 링크 버짓(Link Budget) 계산식 형태로 표현할 수 있다.

P_r(dBm) = P_t(dBm) + G_t(dBi) + G_r(dBi) - FSPL(dB)

여기서 FSPL을 주파수 f (단위: MHz)와 거리 d (단위: km)의 함수로 나타내면 다음과 같다.

FSPL(dB) = 20 \log_{10}(d) + 20 \log_{10}(f) + 32.44

이는 기체의 고도나 거리가 증가할수록 텔레메트리 모듈(예: 915MHz SiK Radio)의 수신 감도가 기하급수적으로 이득을 상실함을 직관적으로 보여준다.

3. UAV 기동성을 고려한 안테나 이득의 적분 모델 확장

점원(Point Source) 기반의 이상적인 등방성(Isotropic) 안테나와 달리, 실제 PX4 드론에 장착되는 다이폴 안테나(Dipole Antenna)는 도넛 형태의 방사 패턴(Radiation Pattern)을 갖는다. 즉, 특정 앙각(Elevation)과 방위각(Azimuth)에서는 전파 방사가 거의 없는 영점(Null)이 존재한다.

무인항공기는 위치 제어(Position Control) 시 목표 속도에 도달하기 위해 기체의 피치(Pitch)와 롤(Roll) 각도를 동적으로 변화시킨다. 기체가 기울어지면 안테나의 지향 각도 역시 변화하므로, 특정 시점 t에서의 실효적인 안테나 이득은 상수 G가 아닌 기체의 자세 오일러 각(Euler Angles: \phi, \theta, \psi)에 종속되는 함수 G(\theta_{rel}, \phi_{rel})가 된다.

이를 수학적 적분 모델로 확장하면, 기동 중인 드론에서 방사되는 유효 등방성 복사 전력(EIRP: Effective Isotropic Radiated Power)의 기대값 E[EIRP]은 기체의 확률적 자세 분포 표면적에 대한 적분으로 산출된다.

E[EIRP] = P_t \iint_{\Omega} G(\theta, \phi) \cdot p(\theta, \phi) \, d\Omega

  • G(\theta, \phi): 3차원 공간상의 안테나 방사 패턴 함수
  • p(\theta, \phi): 특정 비행 모드(예: Acro, Mission)에서 드론이 해당 공간각(Solid Angle) \Omega 상에 관제탑(GCS)을 둘 확률 밀도 함수(PDF)

1.1 PX4 통신 신뢰성への 시사점

만약 드론이 고속으로 선회(Bank Turn)하여 롤(Roll) 각도가 크게 기울거나 수직으로 하강하여 안테나 패턴의 Null 지점이 QGroundControl 관제 안테나를 향하게 되면, 프리스 방정식 상의 G_t 값이 순간적으로 -\infty dB 근처로 수렴하게 된다. 거리가 아무리 가깝더라도 물리적인 MAVLink 데이터 패킷 드롭이 발생하는 이유가 바로 이 지향성 이탈에 있다.

2. 결론

무인기 시스템 설계 전문가는 통신 거리에만 의존하는 단순 FSPL 모델에서 벗어나, 안테나 방사 패턴의 비등방성(Anisotropy)과 PX4-Autopilot의 역동적 기동(Kinematics) 간의 상관관계를 통신 설계에 반영해야 한다. 안테나 이득의 공간 적분 모델은 듀얼 안테나 다이버시티(Antenna Diversity) 채택, 혹은 기체 형상에 따른 최적화된 안테나 부착 위치(예: 기체 하단 수직 배치)를 결정하는 최상위 공학적 지침 역할을 수행한다.
