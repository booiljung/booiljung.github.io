36.1.2. 통신 지연(Latency) 및 패킷 드롭아웃(Packet Dropout)의 상태 공간(State Space) 모델링과 리아푸노프 안정성(Lyapunov Stability) 분석

1. 개요 (Introduction)
비행 역학을 원격 체계에서 통제하는 지상 관제 시스템(GCS) 및 네트워크 제어 시스템(NCS) 통합 환경において, 무선 라디오 링크를 가로지르는 MAVLink 통신 스트림은 본질적으로 통신 지연(Communication Latency)과 예측 불가능한 패킷 드롭아웃(Packet Dropout)이라는 추계학적(Stochastic) 물리 결함을 가혹하게 동반한다. PX4-Autopilot과 QGroundControl(QGC, 또는 커스텀 MAVSDK 클라이언트) 간의 폐루프 제어(Closed-loop Control) 연쇄를 엄정히 해석함에 있어, 이러한 네트워크 불완전성을 명시적으로 포함한 상태 공간(State Space) 확장 모델링과 그에 따른 시스템의 리아푸노프 안정성(Lyapunov Stability) 이론 분석은 매우 중요하다. 이는 특히 통신망을 통한 오프보드(Offboard) 군집 제어나 외부 연산기(Companion Computer) 개입 제어의 군사적, 산업적 수학적 신뢰성을 증명하는 핵심 공학적 토대이다.

2. 지연 및 패킷 드롭아웃을 포함한 이산 상태 공간 모델 (Discrete-Time State Space Model formulation)
무인기 기체의 비선형 연속 시간 동역학을 비행 제어기(FCU)의 이산 시간(Discrete-time) 샘플링 주기 T_s에 따라 단순 선형 변환할 때, GCS와 기체 간의 불안정한 네트워크 효과 변수를 이 수식 트리에 명확히 포함시켜야 한다.

  • 기본 이산 동역학 모델 (Nominal Discrete Dynamics):
    이상적인 환경에서의 기체 상태 갱신 식은 아래와 같다.
    x_{k+1} = A_d x_k + B_d u_k
  • 네트워크 결함이 반영된 통제 제약 모델 (Network-induced Constraints Model):
    GCS(또는 원격 제어망)에서 정교하게 계산된 오프보드 제어 위치 표적 입력 u_k가 실제 기체 모듈로 전달되는 라우팅 과정에서, 가변적인 시간 지연(Time Delay) \tau_k와 패킷 드롭 유실 여부를 나타내는 베르누이 확률 변수(Bernoulli Random Variable) \alpha_k \in \{0, 1\} (여기서 1은 수신 성공, 0은 드롭아웃을 의미함)를 도입한다. 이때 기체의 내부 제어기가 실제로 채택하여 인가하는 제어 입력 갱신식 \tilde{u}_k는 다음과 같이 정의된다.
    \tilde{u}_k = \alpha_k u_{k-\tau_k} + (1 - \alpha_k) \tilde{u}_{k-1}
    수학적으로 해석하자면, 통신상의 이유로 패킷이 정상 도착하지 않았을 때(\alpha_k = 0) 보편적으로 PX4 오프보드 수신단은 가장 널리 쓰이는 Zero-Order Hold(ZOH) 파형 유지 방식에 따라 직전 틱의 제어 입력 \tilde{u}_{k-1}을 그대로 유지하거나(Hold), 내부 MAVLink 타임아웃 펄스 임계치가 만료되기 전 관성대로 이전 명령을 추종한다. 이 수식을 전체 기체 상태 공간 역학 모델 행렬식에 증강(Augment)하면, 네트워크 확률적 결함이 단단히 엮인 복합 상태 방정식(Augmented State Equation)을 구성하여 이산 시뮬레이션에 대입할 수 있다.

3. 리아푸노프 안정성 이론 적용 및 수학적 판별 (Lyapunov Stability Analysis)
상기 증강된 지연-드롭아웃 확률 모델이 시간의 흐름 k \to \infty에 따라 외란으로 발산하지 않고 원점 혹은 의도한 목표 궤적 상태 공간으로 수렴함을 강건하게 보장하기 위해서는 결정론적으로 리아푸노프 안정성 기준(Lyapunov Stability Criterion)을 만족해야만 한다.

  • 리아푸노프-크라소프스키 범함수 (Lyapunov-Krasovskii Functional):
    입출력 지연 \tau_k의 범위 시차존재 시스템의 점근적 안정성(Asymptotic Stability)을 증명하기 위해, 단순 상태 변수뿐만 아니라 과거 지연 상태 공간 메모리의 가상 에너지를 통계적으로 포괄하여 적분하는 리아푸노프 범함수 V(x_k, k)를 다음과 같이 정의한다.
    V(x_k) = x_k^T P x_k + \sum_{i=1}^{d} x_{k-i}^T Q x_{k-i}
    (단, P \succ 0, Q \succ 0 인 양의 정부호 대칭 행렬이며, d는 시스템 구조 상 허용되는 최대 상한 지연 차수 스텝 횟수 제약값이다.)
  • 안정성 조건 (Stability Condition)과 LMI (Linear Matrix Inequality) 최적화:
    전체 시스템 루프가 점근적으로 안정하기 위한 기본 궤환 조건은 \mathbb{E}[\Delta V] < 0 으로 요약된다. 현실의 통신 환경에서 이는 제어 이론상 복잡한 선형 행렬 부등식(LMI) 문제로 변환 및 치환되어, GCS 통신망에 주어진 평균 최대 패킷 손실 확률 \bar{\alpha}와 한계 최대 지연 \tau_{max} 레이턴시 내에서 시스템을 잡아줄 제어기 강건 이득(Robust Gain) K가 유효하게 존재할 수 있는지 그 수학적 해방정식 통과 해존재성을 사전에 판별하는 잣대가 된다.

4. PX4 시스템 내의 타임아웃 및 안정성 보호 기법 방어 체계 (Safety Mechanisms in PX4)
전술한 기계적 수학적 리아푸노프 불안정성(Lyapunov Instability) 영역 한계선, 즉 외부 제어기와 통신망이 한계 지연폭 \tau_{max} 나 임계 연속 패킷 드롭아웃 확률 한계치를 파단 초과하는 순간을 시스템이 역학적으로 맞이하여 발산 붕괴하기 이전에, PX4-Autopilot은 실용주의적으로 파라미터 기반의 강력하고 절대적인 안전장치(Failsafe) 타이머 펄스를 펌웨어 우선순위로 강제 동작시킨다.

  • COM_OF_LOSS_T 파라미터 방어막: PX4 체계에서 오프보드 제어 명령(Setpoints)의 수신 타임아웃 붕괴 한계치를 정하는 절대 파라미터. 주로 0.5초 정도로 타이트하게 설정되는 이 시간 내재 변수를 초과하여 \alpha_k = 0이 유지될 때, 기체는 GCS 제어 루프 에러 누적에 의한 제어 에너지 발산을 막고 모터 포화를 방지하기 위해 즉각적으로 동작 상태 모드를 자체 Position(내부 위치 추정 단위 제어) 궤환 락 모드나 회귀 모드인 RTL(Return to Launch) 등으로 즉할 전환 선언한다. 이로써 외부 지연에 이끌려가던 외루프(Outer Loop)를 단절시키고 기체의 내부 루프(Inner Loop) 리아푸노프 독립 안정성을 가장 최우선적으로 단독 되찾는다.
  • Ardupilot과의 철학적 반응성 통제 차이: 생태계 측면에서 상대적으로 Ardupilot은 GCS(Mission Planner)와의 경로점 지령(Waypoint Command) 및 제어기 갱신 지연 시 레거시 무선 통신 우선순위 강제(Override)나 채널 유지 정책 등에 따라 시스템이 통신 단절에 다소 유연하지만 한 박자 늦게 반응하는 전통적 관성이 있었다. 반면 모듈형 마이크로서비스 지향의 최신 PX4 커맨더(Commander) 스레드 머신은 오프보드 제어 수신단 타임아웃 상태 전이의 강제 폴백(Fallback) 모델이 매우 경직되고 체계적으로 수학적인 문법상 닫혀있어, 예측 불가능한 네트워크의 확률적 불안정성 도래에 비교가 불가능할 정도로 빠르고 보수적으로 회피 대응 반응을 보인다.

5. 결론 (Conclusion)
초연결 사이버-물리 시스템에서 불가피한 통신 지연 \tau_k와 패킷 드롭 불확실성 \alpha_k 변수를 적극적으로 내포한 GCS-무인기 통신망의 확률적 상태 공간 모델링은, 현대 무인 비행 시스템의 신뢰성과 안전성을 근원적으로 선제 평가하는 최고의 분석 도구이다. 이처럼 노이즈와 확률적 네트워크 통제 장애 요소를 모두 떠안은 통합 동역학이 종국에는 리아푸노프 안정성(Lyapunov Stability) 조건을 충족하도록 GCS의 전송 주파수 한계를 제약하고, 기체 MAVLink의 수신단 타임아웃 펄스(Timeout Pulse)를 보수적으로 설정 방어하는 것이야말로 악조건 극한 환경에서의 지연 내결함성 다중 무인기 군집 비행(Swarm Flight) 성공을 이끄는 확고한 수학적 보증 수표이다.
