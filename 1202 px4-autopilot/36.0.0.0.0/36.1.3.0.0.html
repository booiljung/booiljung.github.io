36.1.3. 트래픽 클래스(C2, 텔레메트리) 분류를 위한 큐잉 이론(Queuing Theory): M/M/1 및 M/D/1 대기행렬 모델 적용

1. 개요 (Introduction)
지상 관제 시스템(GCS)과 PX4-Autopilot 생태계 간의 MAVLink 무선 통신망은 대역폭 자원이 극히 제한적인 전형적인 병목(Bottleneck) 구간이다. 이 무선 데이터 링크를 통과하는 트래픽은 크게 인간 조종자 및 자율 비행 알고리즘의 즉방향 하달 명령인 ‘지휘-통제(C2, Command and Control)’ 트래픽과, 비행체의 실시간 관측 데이터인 ‘텔레메트리(Telemetry)’ 트래픽으로 대별된다. 고신뢰성 제어망(NCS) 구축을 위해서는 이 이종(Heterogeneous) 트래픽들이 공유 라디오 채널 병목 구간을 지날 때 발생하는 전송 대기 지연(Queuing Delay)을 엄밀하게 예측하고 제어해야 한다. 본 장에서는 큐잉 이론(Queuing Theory)의 수학적 대기행렬 모델인 M/M/1과 M/D/1 시스템을 적용하여, 트래픽 클래스별 우선순위 분배와 버퍼 지연(Buffer Latency)의 이론적 하한을 분석하고 보장하는 통신 설계 프레임워크를 수립한다.

2. PX4-MAVLink 패킷 도착 및 서비스 과정의 추계학적 모델링 (Stochastic Modeling of Traffic)
큐잉 이론의 범주 안에서 무인기와 GCS 시스템 통신망을 수치적으로 해석하기 위해서는 패킷의 도착 분포(Arrival Process)와 서비스 분포(Service Process)를 확률적으로 명확히 규정해야 한다.

  • 푸아송 도착 과정 (Poisson Arrival Process):
    일반적으로 GCS에서 발생하는 C2 명령(예: 조이스틱 매뉴얼 제어 입력 조향, 오프보드 속도 Setpoint 등)의 이벤트 인입은 무작위적이고 독립적으로 일어난다고 가정할 수 있다. 단위 시간당 패킷 도착률(Arrival Rate)을 \lambda라 할 때, MAVLink 패킷 도착 시간 간격은 지수 분포(Exponential Distribution)를 따르는 ‘M(Markovian)’ 분포 모델로 훌륭하게 모사된다.
  • 서비스 시간 분포 (Service Time Distribution):
    무선 라디오 모듈(Telemetry Radio)이 GCS로부터 수신된 단일 MAVLink 패킷을 물리적으로 변조 및 전송 완료하는 데 걸리는 서비스 처리율(Service Rate)을 \mu라 칭한다.
    통신 망의 특성에 따라 다음과 같이 두 개의 모델로 나뉜다.
  1. M/M/1 대기행렬 (Markovian Service): 직렬화(Serialization)된 패킷의 길이가 가변적이고, 공중의 무선 채널 간섭(Interference)으로 인해 전송 시간이 지수 분포 변동성을 가진다고 가정하는 보수적인 모델이다.
  2. M/D/1 대기행렬 (Deterministic Service): 모든 MAVLink 패킷이 프레이밍 규격(MAVLink v2의 경우 고정 헤더와 체크섬 포함)에 따라 전송 길이가 거의 일정하고 비트레이트 처리가 지연 없이 일정(Deterministic, D)할 때 성립하는 모델이다.

3. 클래스별 기반 M/M/1 및 M/D/1 대기행렬 지연 수학적 전개 (Mathematical Analysis of Queueing Latency)
시스템의 전체 트래픽 부하율(Utilization Factor)을 \rho = \lambda / \mu (\rho < 1)라 할 때, GCS 무선 인터페이스 송신 버퍼(Buffer)에 적체되어 패킷이 대기하는 평균 시간(Mean Waiting Time, W_q)은 큐잉 이론의 폴라첵-킨친 공식(Pollaczek-Khinchine Formula)에 의해 지배된다.

  • M/M/1 모델의 평균 대기 시간:
    패킷 전송 시간의 변동성이 매우 클 경우(시스템의 분산이 큰 환경), 패킷이 송신 버퍼에 갇혀 소비하는 이론상 대기 시간은 다음과 같다.
    W_{q(M/M/1)} = \frac{\rho}{\mu (1 - \rho)}
  • M/D/1 모델의 평균 대기 시간:
    현대 PX4 환경과 같이 MAVLink 패킷 크기가 최적화되어 있고 DMA 기반 하드웨어 처리로 전송 속도가 고정된 결정론적(Deterministic) 환경에서는 평균 대기 시간이 M/M/1 모델 대비 정확히 절반으로 감소한다.
    W_{q(M/D/1)} = \frac{\rho}{2 \mu (1 - \rho)}

위 수식들이 직접적으로 시사하는 바는 수학적으로 자명하다. 트래픽 부하율 \rho가 1(100% 한계 용량)에 임계적으로 근접할 때 대기 지연 시간 W_q는 점근적으로 무한대(Infinity)로 발산 폭증한다. 통신 매체의 고유 전송률 \mu가 제한된 상황에서, QGroundControl이나 무인기가 필요 이상으로 과도한 텔레메트리 레이트(Telemetry Rate) \lambda를 남발하게 될 경우, 시스템 백엔드 버퍼에 심각한 ‘버퍼블로트(Bufferbloat)’ 현상이 유발되어 치명적인 제어 지연 탈락을 가져오게 됨을 수학적으로 엄중히 경고한다.

4. 우선순위 큐잉(Priority Queuing)을 도입한 C2 와 텔레메트리 트래픽의 비선형적 분리 (Non-Preemptive Priority Modeling)
무작위로 뒤섞인 네트워크 패킷망 구조에서 GCS는 필시 C2 패킷(비상 긴급 명령, 제어 입력)을 센서 텔레메트리 패킷보다 시스템적으로 우선 처리해야 한다. 이를 해결하고자 MAVLink 라우팅 모델에 비선점형 우선순위 큐잉(Non-Preemptive Priority Queuing) 이론 모델학을 이식할 수 있다.

  • 클래스 1 (C2 트래픽): 고우선순위(\lambda_1, \mu_1)
  • 클래스 2 (텔레메트리 트래픽): 저우선순위(\lambda_2, \mu_2)

우선순위 큐의 대기 지연 법칙(Cobham’s Formula)에 따르면, 1순위 C2 명령 제어 클래스의 평균 대기 시간 W_{q1}은 저순위 데이터의 부하 압박으로부터 매우 강력하게 면역 보호를 받는다.
W_{q1} = \frac{R}{1 - \rho_1} \quad \text{(단, } R \text{은 서비스 중인 패킷의 평균 잔여 전송 시간)}
반면 2순위 텔레메트리 클래스의 대기 시간 W_{q2}는 다음과 같이 제1클래스 부하의 잔재 패널티를 고스란히 짊어진 채 지연된다.
W_{q2} = \frac{R}{(1 - \rho_1)(1 - \rho_1 - \rho_2)}

  • PX4 및 Ardupilot 생태계에서의 우선순위 제어 실무 구현:
    실제 시스템 차원에서 이 M/D/1 기반 우선순위 수학적 큐잉 철학은 소프트웨어 엔지니어링 코드와 QGroundControl 관제 세션단에 적용된다. 가령 PX4의 MAVLink 전송 모듈(MAVLink Streaming API)은 MAVLINK_MSG_ID_COMMAND_LONG 등의 제어 패킷을 최상위 스케줄링 큐에서 최우선 할당하여 인터럽트성(Interrupt-driven)으로 선처리한다. 반대로 MAVLINK_MSG_ID_SYS_STATUS 계열 등의 대용량 상태 패킷은 대기행렬(Queue)의 후순위 버퍼 링(Ring Buffer)으로 밀려나 전송률 강제 제한(Rate Limiting) 조치를 수시로 당하게 하여, 전체 네트워크 \rho가 한계치 1을 넘지 않게 통제함으로써 W_{q1} 레이턴시 발산을 근원적으로 방위한다.

5. 결론 (Conclusion)
큐잉 이론의 M/M/1 및 M/D/1 대기행렬 모델은 GCS 데이터 링크 내부에서 발생하는 본질적인 전송 적체 현상을 예측 및 통제할 해석적 수학 도구로서 기능한다. 제약된 무선 통신 대역폭 인프라 속에서 C2 지휘-통제 명령과 대용량 텔레메트리 정보를 한 채널에 병합 운용할 때 나타날 수 있는 지연 붕괴 발산 현상을 방지하기 위해서는, 트래픽 유입률 \lambda를 감시하여 부하율 \rho를 제어함과 동시에 절대적 우선순위 큐 분할 통제를 GCS 소켓 프로토콜 계층에 이식해야 한다. 이러한 수학적 큐잉(Queuing) 기저 원리를 철저히 수용함으로써, PX4 생태계는 극한의 트래픽 병목 조건하에서도 흔들림 없는 빠른 통신 제어망과 안전한 무인기 사이버-물리 제어 반응성을 확고히 이룩한다.
