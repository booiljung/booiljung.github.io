59.7.5. 조종용 아날로그 스틱(HOTAS/Gamepad) 입력 신호의 칼만 필터(Kalman Filter) 기반 노이즈 평활화 및 지수 함수(Exponential Function) 비선형 조향 매핑

1. 개요 및 스틱 조작의 공학적 의의

무인항공기(UAV)의 수동 비행(Manual/Stabilized/Acro Mode)이나 반자동 비행 모드(Position/Altitude Mode)에서 조종자는 HOTAS(Hands On Throttle-And-Stick) 또는 표준 게임패드(Gamepad)를 통해 기체를 직접 조종한다. 이 과정에서 아날로그 스틱의 물리적인 변위는 ADC(Analog-to-Digital Converter)를 거쳐 이산화된 디지털 값(예: -1.0 ~ 1.0)으로 변환된 후, MAVLink의 MANUAL_CONTROL 메시지 형태로 QGroundControl(이하 QGC) 등 지상 관제 시스템(GCS)에서 PX4-Autopilot으로 전송된다.

그러나 저가형 포텐셔미터(Potentiometer) 기반 스틱이나 장기간 사용된 마그네틱 홀 센서(Hall Sensor) 기반 조종기는 전기적, 기계적 마모로 인해 입력 신호에 고주파 노이즈(Jitter)가 발생한다. 이러한 노이즈가 기체 제어기(PX4 Flight Controller)에 그대로 전달되면 액추에이터(Actuator)의 미세한 떨림을 유발하여 전력 소모를 증가시키고 비행 안정성을 저해한다. 또한, 인간의 손가락 근육은 스틱을 완전히 꺾었을 때보다 중앙 부분(Center)에서 훨씬 정밀한 조작을 요구한다. 따라서 본 절에서는 칼만 필터(Kalman Filter)를 통한 입력 노이즈 평활화(Smoothing) 기법과 지수 함수(Exponential Function)를 이용한 비선형 조향 매핑(Non-linear Steering Mapping) 알고리즘의 수학적 설계 원리를 탐구한다.

2. 입력 신호의 칼만 필터(Kalman Filter) 기반 노이즈 평활화

단순 이동 평균(Moving Average) 필터나 저주파 통과 필터(Low-Pass Filter)는 노이즈를 줄여주지만 조종 입력의 응답성(Latency)을 악화시켜 조종자에게 위상 지연(Phase Lag)을 유발한다. 반면, **칼만 필터(Kalman Filter)**는 시스템 모델을 기반으로 예측(Prediction)과 새로운 측정값의 보정(Update)을 수학적으로 수행하여, 오차 공분산을 최소화하면서도 지연이 없는 최적의 상태 추정(Optimal State Estimation)을 달성한다.

조종 수동 입력(예: 롤, 피치 축)에 1차원 이산 칼만 필터(1D Discrete Kalman Filter)를 적용하는 과정은 다음과 같다.

  1. 상태 예측(State Prediction):
    물리적인 스틱의 움직임은 관성(Inertia)에 의해 급격히 변하지 않는다고 가정할 수 단 있다. 이전 측정치 x_{t-1} 과 시스템 노이즈 불확실성 P_{t-1} 로부터 현재 상태 x_t^- 와 불확실성 P_t^- 를 예측한다.
  • x_t^- = x_{t-1}
  • P_t^- = P_{t-1} + Q (단, Q는 프로세스 노이즈 공분산)
  1. 측정 업데이트(Measurement Update):
    ADC를 통해 샘플링된 실제 입력값 z_t를 받아들이고, 센서 정밀도에 기반한 측정 노이즈 공분산 R을 반영하여 칼만 이득(Kalman Gain, K_t)을 산출한다.
  • K_t = \frac{P_t^-}{P_t^- + R}
  • x_t = x_t^- + K_t (z_t - x_t^-)
  • P_t = (1 - K_t) P_t^-

조종기 캘리브레이션 시 측정된 노이즈 분산에 따라 Q와 R 파라미터를 동적으로 튜닝하면, 급격한 스틱 조작(예: 회피 기동 전략)에는 즉각 반응하면서도 미세한 손떨림 노이즈(Jitter)는 완벽히 상쇄하는 이상적인 평활화 결과를 얻을 수 있다.

3. 지수 함수(Exponential Function) 비선형 조향 매핑

평활화된 선형 입력 스펙트럼(예: x \in [-1, 1])을 기체의 최종 각속도(Angular Rate) 지령 값에 직접 1:1로 매핑하면, 중앙부에서의 조작이 너무 민감하여 기체가 과도하게 요동치는 문제(Over-correction)가 발생한다.

이를 해결하기 위해 조향 민감도를 재설정하는 곡선, 일명 ’Expo(Exponential Rate)’가 RC 조종자들 사이에서 필수적으로 활용된다. 이는 수학적으로 비선형 함수(Non-linear Function)를 적용하는 것과 동일하며, 대표적인 다항식 매핑 또는 지수 함수 매핑 식은 아래와 같이 유도된다.

  • 3차 다항식 기반 비선형 매핑 (Cubic Mapping):
    y = k \cdot x^3 + (1 - k) \cdot x
    여기서 x는 평활화된 스틱 입력값이며, y는 결과 조향 명령 값이다. 상수 k (0 \le k \le 1)는 ’Expo 계수’로 불리며, 중앙부의 둔화율과 가장자리의 가속율을 결정한다.

이 방식을 적용하면 중앙 부근에서 미분값(기울기)이 현저히 낮아져 조종자가 극도로 정밀한 미세 비행(마이크로 조정)을 할 수 있도록 지원하며, 동시에 스틱을 끝단으로 밀었을 때는 민첩하고 과감하게 최대 각속도(Max Rate)를 이끌어낼 수 있다.

4. GCS 및 데이터 처리 파이프라인(Data Pipeline) 다이어그램

5. QGroundControl 아키텍처 구현 고찰

QGroundControl(v4.x)의 백엔드 C++ 소스 코드인 Joystick.cc 클래스를 분석해 보면, 축(Axis) 캘리브레이션 이후 데드존(Deadband) 처리와 지수 커브(Expo Curve) 적용 로직이 내장되어 있다.

사용자가 GCS 조이스틱 셋업(Joystick Setup) 화면에서 “Expo” 비율을 설정하면, 시스템은 입력된 원시 값을 백분율 기반의 지수 곡선을 따라 역매핑하여 전송한다. 독자적인 GCS 애플리케이션을 구축할 경우, QGC의 기초적인 저주파 필터를 넘어선 칼만 필터의 도입은 정밀 근접 점검 비행이나 인명 구조 드론과 같은 극한의 세밀한 제어가 필요한 환경에서 조종 피로도를 기하급수적으로 낮추는 결정적인 체감 성능 차이를 발생시킨다.
