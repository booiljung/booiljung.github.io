63.7.2. 다중 로터 물리 역학 모델 및 자이로스코프 세차 운동(Gyroscopic Precession) 보상기를 거치지 않는 개별 모터 각속도 및 서보 모터 위상각 개루프(Open-loop) 직접 제어의 시스템 불안정성 극점 이동 현상 분석

무인항공기(UAV)의 비행 제어 시스템, 특히 다중 로터(Multi-rotor) 형상을 지닌 기체에서 각 로터가 발생시키는 추력과 토크는 기체의 고도 및 3차원 자세 제어를 결정짓는 가장 핵심적인 거시적 물리량이다. 범용적인 제어 아키텍처에서는 자세 제어기(Attitude Controller) 및 각속도 제어기(Rate Controller)가 연산한 3축 토크 및 추력 명령을 제어 할당 모듈(Control Allocation Module, 또는 Mixer)이 받아 각 개별 구동기에 필요한 회전 속도 명령으로 변환하는 과정을 거친다. 이때 제어 할당 모듈은 단순히 선형적인 행렬 연산만을 수행하는 것이 아니라, 기체의 공기역학적 비선형성과 구동기의 포화(Saturation) 상태, 그리고 자이로스코프 세차 운동(Gyroscopic Precession)과 같은 동역학적 결합(Dynamic Coupling) 요소를 실시간으로 보상하기 위한 닫힌 루프(Closed-loop) 또는 고도화된 선행 보상(Feed-forward) 체계를 내포하고 있다.

그러나 연구 목적, 특히 강화 학습(Reinforcement Learning) 기반의 종단간(End-to-End) 비행 제어기 설계, 혹은 새로운 추진체 계통의 시스템 식별(System Identification) 과정에서는 Autopilot의 내부 믹서를 거치지 않고 외부 Companion Computer나 ROS2 노드에서 개루프(Open-loop) 형태로 직접 모터의 각속도 또는 서보 모터의 위상각(Phase Angle) 명령을 인가하는 경우가 빈번하다. 이러한 직접 제어(Direct Control) 방식은 설계자에게 완전한 자유도를 부여하지만, Autopilot이 제공하던 안전 장치와 물리 역학적 보상기를 완전히 무력화시키므로 시스템의 근본적인 기하학적, 동역학적 불안정성을 노출시킨다. 본 절에서는 이러한 개루프 직접 제어 체계에서 자이로스코프 세차 운동 보상이 누락되었을 때 발생하는 비선형 결합 현상과, 이로 인해 유발되는 시스템 특성 방정식 극점(Pole)의 우반평면(Right-Half Plane, RHP) 이동 현상을 수학적, 제어 공학적 관점에서 심층적으로 분석한다.

1. 다중 로터의 자이로스코프 세차 운동 및 동역학 결합 모델

다중 로터의 동역학 방정식을 모델링할 때 흔히 강체 역학식과 공기역학적 추력, 항력만을 고려하는 단순화된 뉴턴-오일러 방정식(Newton-Euler Equation)을 사용하지만, 고기동 비행이나 쿼드콥터 기준 피치(Pitch) 및 롤(Roll) 축의 급격한 회전 인가 시에는 로터 자체의 고속 회전 관성이 기체 전체의 자세 회전 각속도와 결합하여 생성되는 자이로스코프 모멘트(Gyroscopic Moment)를 반드시 고려해야 한다.

로터 i의 회전 속도를 \Omega_i, 로터 및 프로펠러 조립체의 관성 모멘트를 J_r이라고 할 때, 단일 로터가 생성하는 자이로스코프 모멘트 \tau_{g,i}는 기체의 각속도 벡터 \omega = [p, q, r]^T와 로터 각속도 벡터 \Omega_i = [0, 0, (-1)^{i+1}\Omega_i]^T의 외적(Cross Product)으로 정의된다.

\tau_{g,i} = J_r (\omega \times \Omega_i)

다중 로터 전체에 작용하는 총 자이로스코프 모멘트 \tau_g는 각 로터에서 발생하는 모멘트의 합으로 표현되며, 수학적으로 다음과 같이 정리된다.

\tau_g = \sum_{i=1}^{n} \tau_{g,i} = J_r \omega \times \begin{bmatrix} 0 \\ 0 \\ \Omega_R \end{bmatrix} = \begin{bmatrix} J_r q \Omega_R \\ -J_r p \Omega_R \\ 0 \end{bmatrix}

여기서 \Omega_R = \sum_{i=1}^{n} (-1)^{i+1} \Omega_i는 모든 로터 각속도의 대수적 합을 의미하며, 잔류 로터 회전 속도(Residual Rotor Speed)라 불린다. 이 수식이 보여주는 핵심적인 의미는, 기체가 롤 축으로 회전(p \neq 0)하려 할 때 이는 피치 축 방향의 모멘트(-J_r p \Omega_R)를 발생시키고, 기체가 피치 축으로 회전(q \neq 0)하려 할 때는 롤 축 방향의 모멘트(J_r q \Omega_R)를 발생시킨다는 점이다.

이러한 세차 운동은 시스템의 각 채널(롤, 피치) 시스템 간 강한 교차 결합(Cross-coupling) 상태를 유도한다. PX4-Autopilot의 기본 제어기 아키텍처에서는 이 동토크(Dynamic Torque)를 피드포워드 성분으로 미리 계산하여 제어 할당 단계에서 보상하지만, ActuatorMotors 메시지를 활용한 ROS2 기반의 개루프 직접 제어에서는 이러한 보상기가 전혀 개입하지 않으므로 이 결합 텀이 그대로 외란(Disturbance)으로 작용하게 된다.

2. 개루프 직접 제어 환경에서의 플랜트 행렬 구성

개루프 시스템 특성을 평가하기 위해 선형화된 근사 동역학(Linearized Approximate Dynamics) 모델에서의 상태 공간(State Space) 모델 기저를 구축한다. 상태 벡터를 x = [\phi, \theta, \psi, p, q, r]^T로 하고 제어 입력 벡터를 u = [\Omega_1, \Omega_2, ..., \Omega_n]^T로 두었을 때, 시스템 방정식은 전형적인 선형 시불변(LTI, Linear Time-Invariant) 모델인 \dot{x} = Ax + Bu의 형태로 나타낼 수 있다.

피드백 루프가 없는 순수 개루프 시스템 행렬 A에는 앞서 언급한 자이로스코프 결합 항이 행렬의 비대각(Off-diagonal) 요소로 병합된다. 롤과 피치 동역학만 분리하여 기체 중심 관성 텐서를 대각 행렬 I = \text{diag}(I_{xx}, I_{yy}, I_{zz})라 가정할 때, 각가속도 방정식은 다음과 같이 근사화할 수 있다.

\dot{p} = \frac{J_r \Omega_{R,0}}{I_{xx}} q + \frac{1}{I_{xx}} \tau_{\phi}(u)
\dot{q} = -\frac{J_r \Omega_{R,0}}{I_{yy}} p + \frac{1}{I_{yy}} \tau_{\theta}(u)

호버링(Hovering) 평형점 근처에서 상태 행렬 A의 롤-피치 각속도 서브시스템(Sub-system) 행렬 A_{pq}는 다음과 같다.

A_{pq} = \begin{bmatrix} 0 & \frac{J_r \Omega_{R,0}}{I_{xx}} \\ -\frac{J_r \Omega_{R,0}}{I_{yy}} & 0 \end{bmatrix}

2.1 시스템 내 극점 이동(Pole Shifting) 및 불안정성 유발 현상

위 A_{pq} 행렬의 고유값(Eigenvalue)을 구하기 위해 특성 방정식(Characteristic Equation) \det(sI - A_{pq}) = 0을 계산하면 다음과 같다.

s^2 + \frac{(J_r \Omega_{R,0})^2}{I_{xx} I_{yy}} = 0

이 시스템의 극점 s는 순수 허수(Purely Imaginary) 쌍으로 도출된다 (s = \pm j \sqrt{(J_r \Omega_{R,0})^2 / (I_{xx} I_{yy})}). 이는 선형화된 시스템 자체가 마진 안정(Marginally Stable) 영역 내지는 언덤프드 진동(Undamped Oscillation)의 경계에 놓여 있음을 나타낸다. 여기서 공기역학적 항력 모델의 부정확성이나 로터 추력 곡선의 비선형 특성, 또는 베어링 및 구조적 비대칭성에 의한 마찰 손실 모델링 누락이 합쳐지면, 복소 평면(Complex Plane) 상의 극점은 우선 우반평면(Right-Half Plane, RHP)으로 즉각적으로 이탈(Shift)할 가능성을 갖는다.

만약 극점 중 하나라도 실역(Real Part) 양의 수치(Positive Value)를 지니게 되면 계단 응답(Step Response) 상에서 오버슈트가 끝없이 증폭되는 발산(Divergence) 현상이 일어난다. 즉, 자이로스코프 세차 운동 보상기가 소거된 상태에서 고주파 위상각 제어(예: 모터 회전수 차전 또는 서보 위상 변경) 명령을 무리하게 인가할 시, 제어기는 기체 동역학의 내부 진동 에너지를 억제하지 못하고 오히려 결합 공진(Coupled Resonance)을 촉발하여 자세의 완전 상실을 일으킨다.

3. PX4 아키텍처 및 설정상 유의사항 분석

현대적인 PX4 펌웨어 버전(v1.14 이상)에서는 ROS2 uXRCE-DDS 브리지를 통해 ActuatorMotors 혹은 ActuatorServos 메시지를 고주기로 퍼블리시함으로써 위와 같은 개루프 제어를 직접 구현할 수 있다. 이러한 토픽의 메시지 페이로드를 인가하기 위해서는 기체 파라미터 COM_OBS_CONTROL을 설정하거나 Offboard 비행 모드에 진입한 뒤, 동기화 스케줄을 액추에이터 루프의 주기에 맞춰(대략 400Hz 이상 권장) 전송해야 한다.

이 접근 방식은 ArduPilot의 Motor Pass-through 메커니즘과 대비된다. ArduPilot의 경우 RC 입력 채널을 통해 하드웨어 PWM을 강제로 포워딩(동기화)하는 레거시 구조에 기인하지만, PX4는 uORB와 ROS2 브리지를 기반으로 소프트웨어 스택에서의 투명한 메시징 파이프라인 우회를 권장하여 높은 이식성을 지닌다. 그러나 두 방식 공통으로 치명적인 결점은 외란 관측기(Disturbance Observer)나 PID 루프가 작동하지 않는다는 점이다.

따라서 ROS2에서 서보 모터의 위상각 및 BLDC 모터 각속도를 직접 제어하는 아키텍처를 설계할 때, 연구자는 아래와 같은 제어 체계를 외부 루프(ROS2 노드 측)에 별도로 필수 구성해야 한다.

  1. 상태 관측 및 강건 제어 루프: PX4의 VehicleOdometry 또는 VehicleLocalPosition 메시지 피드백을 기반으로 ROS2 노드 내부에서 고주파 자세 제어기를 독립적으로 스케줄링해야 한다.
  2. 동적 결합 보상 행렬 계산: 선형 보상용 자이로스코픽 디커플링 보상기(Decoupling Compensator) 행렬 K_{gyro}를 별도로 설계하여, ROS2 노드에서 도출한 모터 제어 명령에 차감 적용해야 한다.
  3. 페일세이프(Failsafe) 가드: 통신 주기가 지터링(Jitter)으로 인해 20ms 이상 누락될 경우 즉각적으로 PX4 Commander의 수동형 비행 모드(Stabilized 또는 Position 모드)로 권한을 자율 피드백할 수 있도록 통신 시간 초과(Timeout) 인터럽트 핸들러를 구축하라.

4. 결론 및 학술적 시사점

결론적으로 다중 로터 드론에 대한 개루프(Open-loop) 엑추에이터 직접 제어 방식은 자이로스코프 세차 운동과 기구학적-동역학적 커플링(Coupling)을 방치함으로써 특성 극점의 RHP 이동을 촉발시켜 시스템 발산을 유발한다. 종단간 기계 학습이나 특정 목적으로 PX4의 내부 제어 루프를 우회하는 경우, 단순히 제어 할당 행렬을 생략하는 수준을 넘어 로터 동역학 기반의 상태 피드백 부재(Absence of State-Feedback)가 미치는 극단적 불안정성을 명확히 인지하고, 외부 컴퓨팅 측에서 이를 보완하는 수학적, 제어공학적 안전장치를 엄격하게 수립하라. 이는 논문 “End-to-End Control of Quadrotors with Deep Reinforcement Learning” 부류의 선행 연구에서 지적하는 물리적 불확정성 관리와 잇닿아 있으며, 비행 시스템의 신뢰성을 담보하기 위한 전제 조건이다.
