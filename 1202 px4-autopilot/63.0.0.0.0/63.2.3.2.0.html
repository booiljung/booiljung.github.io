63.2.3.2. 베이즈 추론(Bayesian Inference) 기반 다중 센서 융합 출력의 정합성 검증 및 불확실성(Uncertainty) 전파 해석

오프보드(Offboard) 관제 시스템이 비행 제어기(FC: Flight Controller)로부터 수신하는 모든 상향 링크(Uplink) 동역학 상태 피드백은 근본적으로 단일 센서의 맹목적 계측값이 아닌, 베이즈 추론(Bayesian Inference)에 입각한 다중 센서 융합(Multi-sensor Fusion)의 수학적 결과물이다. 최신 PX4-Autopilot(v1.14 이상)의 상태 관측기인 EKF2(Extended Kalman Filter)는 이질적인 주파수와 노이즈 특성을 지닌 센서(IMU, GNSS, 기압계, 비전 센서 등)들의 데이터를 결합하여 정합성(Consistency)을 검증하고, 측정치에 내재된 불확실성(Uncertainty)이 최종 추정 상태 벡터로 어떻게 전파(Propagation)되는지 모델링한다. 본 절에서는 이러한 베이지안 확률 모델이 오프보드 제어기의 예외 처리(Failsafe) 로직에 미치는 영향을 제어 공학의 입장에서 심층적으로 조명한다.

1. 베이지안 필터링과 다중 센서 융합(Sensor Fusion) 아키텍처

Ardupilot의 EKF3가 각 센서별 상태 추정 코어를 병렬로 운용하여 중간값을 채택하는 투표 기법(Voting Scheme)을 부분적으로 지원하는 데 반해, PX4의 EKF2는 파라미터를 통해 지연된 측정치(Delayed Measurements)의 시계열 동기화 버퍼(Observation Buffer)를 단일 상태 추정기로 엄격히 통제하는 중앙 집중식 베이지안 업데이트를 따른다.

베이즈 정리(Bayes’ Theorem)에 의해 기체의 현재 상태 \mathbf{x} 공간에서의 사후 확률(Posterior Probability) P(\mathbf{x}|z)는 사전 확률(Prior)과 우도(Likelihood)의 곱으로 모델링되며, 이는 EKF의 예측(Prediction)과 갱신(Update) 주기로 그대로 치환된다.

P(\mathbf{x}_{k} | \mathbf{z}_{1:k}) \propto P(\mathbf{z}_{k} | \mathbf{x}_{k}) P(\mathbf{x}_{k} | \mathbf{z}_{1:k-1})

  1. 사전 확률 P(\mathbf{x}_{k} | \mathbf{z}_{1:k-1}): IMU 관성 센서의 고주파(예: 250Hz 이상) 적분 연산을 통해 도출된 기체의 운동학적 예측 상태.
  2. 우도 함수 P(\mathbf{z}_{k} | \mathbf{x}_{k}): 저주파 관측 센서(위성 항법 등, 예: 5~10Hz)가 제공하는 새로운 측정치가 현재 예측 상태에서 나타날 확률적 일관성.

1.1 정합성 검증(Consistency Validation)과 이노베이션(Innovation)

서로 다른 센서 시스템 간의 관측치가 충돌할 경우(예: GPS 멀티패스에 의한 위치 튀어 오름과 IMU 연속성 상실), EKF2는 ‘이노베이션(Innovation, 측정치 잔차)’ \tilde{\mathbf{y}}_{k} = \mathbf{z}_{k} - \mathbf{H}\hat{\mathbf{x}}_{k}^-의 마할라노비스 거리(Mahalanobis Distance)를 산출하여 칼만 게인(Kalman Gain) \mathbf{K}의 산출 전 단계에서 데이터의 정합성을 검증한다.

만일 \tilde{\mathbf{y}}_{k}가 잔차 공분산 \mathbf{S}_{k} 튜브를 크게 이탈하여 베이지안 관점에서 발생 확률이 극히 낮은 아웃라이어(Outlier)로 판명될 경우, 필터는 해당 센서 측정을 기각(Reject)하고 타 센서(예: 옵티컬 플로우 또는 VIO)로 추정 가중치를 넘겨주는 동적 가중치 재분배(Dynamic Re-weighting)를 수행한다.

2. 불확실성(Uncertainty) 전파와 오프보드 연계 해석

상향 링크 토픽(VehicleLocalPosition 등)에 실려오는 속도/위치 추정치에는 EKF2 내부에서 산출된 오차 공분산 성분(eph, epv 등)이 필연적으로 동반된다. 이는 단순히 센서들의 백색 잡음(White Noise) 합성이 아니라 선형화된 야코비안(Jacobian) 행렬 \mathbf{F}와 \mathbf{H}를 타고 비선형 시스템 역학을 통해 전파(Propagate)된 통계적 매개변수이다.

2.1 ROS2 및 오프보드 제어 계층에서의 불확실성 활용

자율 에이전트는 기체의 제어 목표를 설정함에 있어 상태 변수 추정치 \hat{\mathbf{x}}의 스칼라 값 자체보다, 전파된 공분산 \mathbf{P} 행렬에 기반한 시스템의 ‘자기 확신도 향상(Self-confidence Boosting)’ 여부를 모니터링해야 한다.
예를 들어 실내 환경에서 VIO 기반 오프보드 호버링(Hovering) 제어 시, 비전 피처(Feature) 소실로 인해 베이지안 우도 검증을 통과하는 측정치가 결핍되면 오차 공분산 \mathbf{P}가 단조 증가(Monotonic Increase)하며 불확실성이 팽창(Bloating)하게 된다.

QGroundControl(v4.3 기준)이나 동반 컴퓨터 상의 감시 노드(Watchdog Node)는 VehicleLocalPosition의 분산 성분이 임계치를 초과할 시, 제어 모델을 즉각적으로 하위 계층(결정론적 속도 유지 모드 혹은 착륙)으로 퇴행(Failback)시키는 트리거 함수로 활용한다.

관련 학술 문헌인 “Information-Theoretic Sensor Fusion in Micro Aerial Vehicles“에 서술된 바와 같이, 다중 센서 융합 체계에서 베이즈 추론의 오차 공분산 지표는 비행 제어기가 물리적 외부 요동에 얼마나 강건(Robust)하게 대응할 수 있는지를 오프보드 에이전트 측에 투명하게 보고하는 가장 고도화된 메타 논리 구조라 평가할 수 있다.
