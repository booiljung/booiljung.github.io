43.3.3. 확장 칼만 필터(EKF2/ECL) 상태 공간 모델의 동적 수렴성 및 야코비 행렬(Jacobian Matrix) 선형화 오차 평가

PX4-Autopilot(기준 펌웨어: v1.14.x)의 핵심 논리인 비행 상태 추정(State Estimation)은 확장 칼만 필터(EKF, Extended Kalman Filter) 기반의 ECL(Estimation and Control Library) 모듈, 즉 EKF2에 의해 수행된다. EKF2는 24차원 이상의 상태 벡터(위치, 속도, 자세, 바람, 센서 바이어스 등)를 실시간으로 추정하는 비선형(Non-linear) 동적 시스템이다. 시동(Arming)을 위한 전제 조건 검사 시, 단순한 센서의 가용성 검증을 넘어 EKF2 내부 상태 공간 모델(State Space Model)의 동적 수렴성(Dynamic Convergence)과 비선형 예측 모델을 선형화(Linearization)할 때 발생하는 야코비 행렬(Jacobian Matrix) 오차의 범위 이탈 여부를 엄밀하게 검증해야 한다.

1. EKF2 비선형 상태 공간 모델과 야코비 행렬(Jacobian Matrix)의 역할

무인항공기의 6자유도(6-DoF) 강체 동역학 및 센서 측정 모델은 본질적으로 고도의 비선형 지배 방정식을 따른다. 이를 선형 최적 상태 추정기(Optimal State Estimator)인 칼만 필터 체계에 적용하기 위해서는 현 상태를 기준으로 모델을 1차 테일러 급수전개(Taylor Series Expansion)하여 선형 구역(Linear Regime) 방정식을 구하는 과정이 필수적이다. 이때 등장하는 편미분 행렬이 바로 야코비 행렬(Jacobian Matrix)이다.

1.1 예측(Prediction)과 업데이트(Update) 단계에서의 선형화 오차

EKF2는 IMU(가속도계, 자이로스코프) 데이터를 통해 기체의 상태를 예측(Prediction)하고, GPS나 기압계 같은 외부 측정 센서로 상태를 업데이트(Update)한다. 여기서 상태 천이 모델(State Transition Model) f와 관측 모델(Observation Model) h의 선형화 과정은 다음과 같이 야코비 행렬 \mathbf{F}와 \mathbf{H}로 도출된다.

\mathbf{F} = \left. \frac{\partial f}{\partial \mathbf{x}} \right|_{\hat{\mathbf{x}}} \quad , \quad \mathbf{H} = \left. \frac{\partial h}{\partial \mathbf{x}} \right|_{\hat{\mathbf{x}}}

시동 대기 상태(Standby Mode)에서 기체에 예기치 못한 물리적 진동이나 충격이 가해지거나, 센서 바이어스의 초기 변동성이 크면 비선형 예측치 \hat{\mathbf{x}}의 변화율이 급격해진다. 이 시점에서 테일러 급수의 2차 이상 고차항(Higher-order Terms) 절삭 오차(Truncation Error)가 막대해지며, 이는 선형화 구역의 한계를 넘어서는 ’선형화 오차(Linearization Error)’로 발현되어 추정 알고리즘 자체를 발산(Divergence)시킨다.

2. 동적 수렴성(Dynamic Convergence) 평가 메커니즘

커맨더(Commander) 스레드는 시동을 인가하기 전에 EKF2의 상태가 안정한(Stable) 매니폴드(Manifold) 상에 수렴했는지를 확률통계적으로 확인한다. EKF2는 자신의 오차 공분산(Error Covariance)과 이노베이션(Innovation) 상태를 통해 동적 수렴성을 스스로 평가하고, 그 지표를 estimator_status uORB 토픽으로 송출한다.

2.1 이노베이션 테스트 비율(Innovation Test Ratio) 모니터링

이노베이션(Innovation)은 해당 시점의 실제 센서 측정치와 EKF2가 수학적으로 예측한 측정치 간의 오차 편차를 뜻한다. 특정 센서 채널 데이터 측정 편차가 해당 채널의 이노베이션 공분산 임계값 내에 포함되면 ’통과(Pass)’로 취급되며, 수학적으로 단위가 없는 ’테스트 비율(Test Ratio)’로 변환된다.
PX4는 속도 수렴도(vel_test_ratio), 위치 수렴도(pos_test_ratio), 자기장 수렴도(mag_test_ratio) 등의 값이 1.0(가용 오차 임계치)을 넘지 않고 0 미만에 안정적으로 안착했는지를 확인함으로써 수렴도를 검증한다. 하나라도 장시간 1.0을 초과하면 선형화 범주를 크게 벗어났다는 의미이므로 시동을 불허한다.

3. 평가 메커니즘 아키텍처 (Mermaid Flowchart)

EKF2 선형화 오차 검증 및 시동 전 수렴도 평가 로직은 다음과 같은 데이터 퍼블리싱 흐름을 갖는다.

flowchart TD
    IMU[IMU / 외부 센서] -->|비선형 측정 데이터| EKF2_Predict[EKF2 상태 예측 단계<br/>(Jacobian F 도출)]
    EKF2_Predict --> EKF2_Update[EKF2 상태 업데이트 단계<br/>(Jacobian H 도출)]
    EKF2_Update --> EKF_Eval{선형화 오차 및<br/>Innovation 공분산 산출}
    EKF_Eval --> |"Test Ratio 계산"| Est_Status[(estimator_status<br/>uORB Topic)]
    Est_Status --> Commander[Commander Module<br/>(Preflight Check)]
    Commander --> Cmd_Check{test_ratio < 1.0 &&<br/>상태 안정(Stable)?}
    Cmd_Check -- 예 --> Arm_Pass[수렴성 유효 판정<br/>(Arming Allowed)]
    Cmd_Check -- 아니오 --> Arm_Fail[시동 거부 및 GCS 경고<br/>"Preflight Fail: EKF2 Error"]

4. 설정(Configuration) 및 튜닝 전략

QGroundControl(v4.3.x 이상)을 통한 파라미터 튜닝 시 선형화 오차 민감도와 관련된 제어 파라미터를 숙지해야 한다. 선형화 오차가 일시적으로 발산할 경우 시스템이 얼마나 관대하게 허용할지를 정의하는 인자들이다.

  • EKF2_REQ_VDRIFT, EKF2_REQ_EPH, EKF2_REQ_EPV: 시동을 위한 최대 수직 이동 드리프트, 수평 및 수직 위치 오차 한계 요구치.
  • 관측 노이즈 파라미터 (예: EKF2_GPS_V_NOISE): 센서 노이즈가 클 때 야코비 선형화 오차를 상쇄하기 위해 측정치 편차의 허용폭인 칼만 게인(Kalman Gain)을 조절하는 데에 기여한다. 노이즈 파라미터를 크게 잡으면 측정치의 변동성을 많이 포용하게 되어 test_ratio 상승폭이 낮아진다.

5. Ardupilot 대비 기술적 접근 차이

  • PX4-Autopilot (ECL 기반 EKF2): 확장 칼만 필터의 선형화 오차와 관련된 수렴성 평가를 단일 24 상태(24-state) 벡터 내에서 엄격한 확률론적 수학적 프레임워크로 해석한다. 이노베이션과 공분산 테스트의 결과를 estimator_status 형태로 집약하여 각 센서 신호원의 동적 신뢰도를 시스템 전파의 기준으로 삼는다.
  • Ardupilot (AP_NavEKF3): 마찬가지로 24 상태 기반의 EKF3를 사용하지만, 선형화 오차 누적으로 인한 발산 위험 시 이중/삼중의 EKF 코어 간 분기(Lane Switch)나 DCM(Direction Cosine Matrix) 보완 센서 융합기작(Fallback AHRS)으로 전환하는 유연성을 강조한다. 즉 역학적 선형화 실패 가능성 상황에서 대체 산술 메커니즘으로 이탈하는 제어 구조가 PX4보다 공격적이다.

6. 심화 학술 자료 및 논문 참고

이 주제와 관련된 무인기 항법 체계의 학술적 근거는 “Observability Analysis and State Estimation for UAVs utilizing Extended Kalman Filter” 또는 “Linearization Errors in EKF-Based Navigation Systems“와 같은 문헌을 참조하라. 로보틱스의 칼만 필터 적용의 가장 대표적인 문헌인 “Probabilistic Robotics(Thrun et al.)“에서도 야코비 행렬 선형화 한계 및 비선형 오차 축적 현상을 심도 있게 다룬다.
