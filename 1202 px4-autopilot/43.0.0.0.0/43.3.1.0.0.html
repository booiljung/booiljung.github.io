43.3.1. 미세 기계 전자 시스템(MEMS) 관성 측정 장치(IMU) 및 지자기 센서의 확률론적 잡음 밀도(Noise Density) 모델링

무인항공기(UAV)의 항법 시스템에서 가장 필수적인 센서는 미세 기계 전자 시스템(MEMS; Micro-Electromechanical Systems) 기반의 관성 측정 장치(IMU)와 지자기 센서(Magnetometer)이다. 크기가 작고 가벼우며 저렴하다는 장점이 있지만, 본질적으로 열(Thermal) 특성과 전기적 요인에 기인하는 측정 오차가 수반된다. PX4-Autopilot의 확장 칼만 필터(EKF2)는 이러한 센서들의 확률론적 잡음 밀도(Stochastic Noise Density)를 수학적으로 모델링하여 최적의 상태 추정(State Estimation)을 수행한다. 본 절에서는 이러한 잡음 특성의 분석 방법론과 시스템 적용 메커니즘을 상세히 다룬다.

1. 개요: 결정론적 오차와 확률론적 잡음

MEMS 센서의 오차는 일관되게 발생하는 결정론적(Deterministic) 오차와 시간에 따라 무작위로 변하는 확률론적(Stochastic) 잡음으로 나눌 수 있다.

  • 결정론적 오차(Deterministic Errors): 팩토리 캘리브레이션 및 GCS에서의 온도/가속도/나침반 캘리브레이션을 통해 파라미터화 및 보상이 가능한 고정된 스케일 팩터(Scale Factor), 비직교성(Non-orthogonality), 그리고 초기 바이어스(Bias) 등을 의미한다.
  • 확률론적 잡음(Stochastic Noise): 열화음(Thermal Noise)이나 반도체 소자의 근본적 한계로 유발되는 무작위 신호 요동이다. 이는 캘리브레이션으로 완전히 상쇄할 수 없고, 통계적 모델링을 통해서만 상태 추정기(EKF2)에서 그 영향을 저감할 수 있다.

2. 잡음 밀도(Noise Density) 모델링의 요소 분석

확률론적 잡음을 분석하기 위해 학계 및 산업계에서는 알란 분산(Allan Variance) 차트를 주로 활용한다. PX4 EKF2는 파라미터를 통해 다음의 핵심 잡음 요소를 모델링한다.

2.1 백색 잡음 및 랜덤 워크(White Noise & Random Walk)

고주파 대역에서 넓게 퍼져 나타나는 평탄한 잡음 특성이다. IMU의 자이로스코프(Gyroscope) 데이터 적분 시 이 백색 잡음이 누적되면 시간이 지남에 따라 각도 오차가 증가하게 되며, 이를 **각도 랜덤 워크(ARW; Angle Random Walk)**라 부른다. 가속도계의 경우는 속도 랜덤 워크(VRW; Velocity Random Walk)로 표현된다.

  • 이를 PX4 파라미터 환경에서는 EKF2_GYR_NOISE(자이로 잡음 밀도) 및 EKF2_ACC_NOISE(가속도 잡음 밀도)라는 값으로 모델에 반영한다.

2.2 바이어스 불안정성(Bias Instability)

저주파 영역에서 발생하는 플리커 잡음(Flicker Noise, 1/f 잡음)에 의해, 센서의 초기 오프셋(Bias)이 시간이 지남에 따라 점진적으로 떠내려가듯(Drifting) 변하는 현상을 말한다.

  • PX4 파라미터 환경에서는 이를 EKF2_GYR_B_NOISE(자이로 바이어스 랜덤 워크) 및 EKF2_ACC_B_NOISE(가속도 바이어스 랜덤 워크) 파라미터로 처리하여, 필터 스스로가 주행 중 바이어스의 변화율을 추적하고 보정할 수 있게 한다.

2.3 지자기 센서의 잡음 모델

지구 자기장은 주변의 전선, 모터 케이블에서 발생하는 전자기장 등 환경적 요인으로 인해 동적인 노이즈를 겪는다. 지자기 센서의 연속적인 측정 불확실성을 모델링하기 위해 EKF2_MAG_NOISE(자기장 잡음 밀도) 값을 적용하여 기수(Heading) 변화에 대한 신뢰 가중치를 결정한다.

3. PX4 EKF2 아키텍처 내 적용 메커니즘

PX4의 ECL(Estimation and Control Library) 라이브러리 내부에서, 모델링된 잡음 밀도는 EKF2의 상태 공분산 행렬(State Covariance Matrix)을 시간-업데이트(Time-Update) 단계마다 확장(예측 오차 증가)시키는 프로세스 노이즈(Process Noise) Q 행렬 연산과, 센서 측정 시 측정 오차(Measurement Noise) R 행렬 연산에 각각 주입된다.

void Ekf::predictStateCovariance() 
{
    // ... (중략)
    // EKF2_GYR_NOISE 파라미터에서 기인한 자이로스코프 잡음 분산 반영
    const float dt_sq = _dt * _dt;
    const float gyro_var = _params.gyro_noise * _params.gyro_noise;
    
    // 상태 예측 시 오차 증폭 모델에 프로세스 노이즈 주입

    // ...
}

Ardupilot의 EKF3 아키텍처 역시 비슷한 형태의 잡음 최적화를 지원하나, PX4의 ECL EKF2 시스템은 각 센서의 독립적인 노이즈 파라미터 튜닝을 QGroundControl을 통해 보다 모듈화된 인터페이스로 제공함으로써, 커스텀 진동 스펙트럼(Vibration Spectrum)을 지닌 다양한 기체 폼팩터에 더 유연하게 대응할 수 있도록 설계되었다.

5. 결론

MEMS 기반 IMU와 지자기 센서는 소형화와 경량화라는 이점을 누리는 대신 아날로그 도메인의 확률론적 잡음이라는 원초적인 한계를 지닌다. PX4-Autopilot은 백색 잡음 및 주파수 종속적인 바이어스 드리프트를 비롯한 각 센서들의 노이즈 밀도(Noise Density)를 EKF2 확장 칼만 필터의 공분산 행렬 내에 최적 가중치로 융합함으로써, 급격한 온도 변화나 고주파 진동 조건에서도 예측 가능한 수렴성(Convergence)과 고정밀 위치 추정을 실현한다.
