38.4.2.1 1차 지수 평활화(First-order Exponential Smoothing) 기반 무한 임펄스 응답(IIR) 필터를 활용한 동적 지연 시간(Dynamic Latency) 추정

앞서 언급한 적응형 타임아웃을 구현하기 위해, 시스템은 가장 먼저 ’현재 통신망의 평균적인 패킷 도달 간격(Dynamic Latency)’을 추정해야 한다.
문제는 HEARTBEAT 패킷의 순시적 도달 간격(I_n) 데이터 자체가 노이즈(Noise) 덩어리라는 점이다.

전파 간섭으로 패킷 하나가 딜레이 되었다가 다음 패킷과 뭉쳐서 수신 버퍼로 쏟아져 들어오면, I_n 측정값은 1.0\text{s} \rightarrow 3.5\text{s} \rightarrow 0.2\text{s} 와 같이 미친 듯이 요동치게(Spike) 된다. 이 날것(Raw)의 데이터를 그대로 타임아웃 임계치로 사용했다가는 통신 감시 데몬이 극심한 발진(Oscillation)을 틱을 일으킬 것이다.

1. 차 지수 평활화(Exponential Smoothing) 필터

이러한 고주파 잡음(랜덤 지터)을 걸러내고 통신망의 진짜 기저 상태(Macro-trend)만을 매끄럽게 추출하기 위해, PX4 커맨더 로직이나 GCS 통신망 데몬은 디지털 저대역 통과 필터(Low-Pass Filter)의 일종인 1차 지수 평활화(First-order Exponential Smoothing) 기법을 동원한다.

신호 처리 관점에서 이는 무한 임펄스 응답(IIR, Infinite Impulse Response) 필터 구조를 띠며, 수식은 다음과 같이 전개된다.

\mu_n = \alpha \cdot I_n + (1 - \alpha) \cdot \mu_{n-1}

  • \mu_n: n번째 시점에서 체결된 새롭고 부드러운 ‘평균 도달 간격 추정치 (추정된 동적 지연 시간)’
  • I_n: 방금 도착한 패킷과 직전 패킷 사이의 순시적 ‘실제 도달 간격’
  • \mu_{n-1}: 과거부터 누적되어 온 이전 시점의 평균 도달 간격 추정치
  • \alpha: 평활화 계수 (Smoothing Factor), 0 < \alpha < 1

평활화 계수(\alpha)의 철학과 트레이드오프

이 수식의 핵심은 가중치를 분배하는 계수 \alpha에 있다.

  • \alpha가 1에 가까울 때 (예: 0.8): 필터는 방금 들어온 I_n (최신 측정값)을 80%나 신뢰하고, 과거의 역사 \mu_{n-1}을 20%만 반영한다. 통신망 상태가 나빠진 것을 매우 기민하게(Responsive) 알아채지만, 한 번 튀는 스파이크성 노이즈에도 크게 반응해버리는 신경질적인(Noisy) 필터가 된다.
  • \alpha가 0에 가까울 때 (예: 0.1): 필터는 과거의 역사 관성을 90%나 의지한다. 어지간한 간섭 스파이크에는 꿈쩍도 않는 매우 묵직하고 안정적인(Stable) 모습이지만, 진짜 통신망이 끊어지기 시작하는 치명적인 열화 상황을 늦게 눈치채는 굼뜬(Sluggish) 방어선이 된다.

비행 제어 시스템 아키텍처는 보통 이 \alpha 값을 0.1 ~ 0.3 내외로 보수적으로 세팅하여, 미세한 지터에는 오탐지(False Positive)를 내지 않으면서도, 2~3회 연속으로 패킷이 늦게 오면 서서히 타임아웃 임계치를 늘려가는 유연한 적응형 밴드(Adaptive Band)를 생성해 낸다.

임베디드(NuttX) 환경에서의 무자비한 컴퓨팅 효율성

왜 단순 이동 평균(SMA, Simple Moving Average)을 쓰지 않고 지수 평활화(IIR)를 쓸까?
SMA(유한 임펄스 응답, FIR 방식)를 구현하려면 최근 10개의 I_n 데이터를 계속 큐(Queue) 배열에 저장(RAM 할당)하고 매 틱마다 루프를 돌며 합산 연산(CPU 사이클 스파이크)을 반복해야 한다.

반면, 1차 지수 평활화 알고리즘은 오직 단 하나의 부동소수점 변수(\mu_{n-1})만을 RAM 메모리에 저장하면 된다!
Pixhawk 내부에 탑재된 제한된 리소스의 NuttX RTOS 환경에서, 수백 가지의 감시 스레드가 동시에 돌아가야 할 때 이 극우주적인 O(1)의 메모리/컴퓨팅 효율성은 PX4 아키텍처가 결코 포기할 수 없는 제어 공학적 우수성이다.
