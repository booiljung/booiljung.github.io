38.6.1 PX4 펌웨어(Firmware) 소스 코드 심층 해부 (commander.cpp 및 mavlink_messages.cpp)

PX4 자율비행 소프트웨어 스택의 거대한 구조 속에서, 단일 진실 공급원(Single Source of Truth)의 역할을 수행하는 핵심 뇌관은 src/modules/commander/ 디렉터리에 위치한 커맨더(Commander) 모듈이다.

1. commander.cpp: 진실의 창출자 (The Source of Truth)

NuttX RTOS 위에서 독립적인 백그라운드 태스크(Task)로 동작하는 commander.cpp의 주된 임무는 기체를 감싸고 있는 수백 개의 센서 상태, 배터리 전압, GCS나 조종기로부터 날아온 제어 명령들을 1초에도 수십 번씩 비동기적으로 취합(Aggregation)하는 것이다.

커맨더 루프는 이 방대한 난장판 속에서 “우리는 현재 이륙할 수 있는가(Arming)?”, “우리는 지금 어떤 모드로 날아야 하는가(Nav State)?“에 대한 최종적이고 절대적인 판결을 내린다. 그리고 그 판결 결과를 고도로 압축된 vehicle_status_s 라는 C++ 구조체(uORB 토픽)에 담아낸다.

// msg/VehicleStatus.msg (uORB 정의의 일부)
uint8 nav_state        # Navigation state (예: NAVIGATION_STATE_AUTO_MISSION)
uint8 arming_state     # Arming state (예: ARMING_STATE_ARMED)
uint8 failsafe         # Failsafe 상태 비트맵

커맨더는 자신의 임무를 마치면 이 vehicle_status 토픽을 uORB(Micro Object Request Broker) 통신 버스라는 허공을 향해 퍼블리싱(Publishing)한다. 여기서 돋보이는 PX4 아키텍처의 우아함은, 커맨더는 자신이 외친 진실을 도대체 누가 거두어 가는지, UART 통신 속도가 몇 보드(Baud)인지 전혀 알지도, 신경 쓰지도 않는다는 점이다.

2. mavlink_messages.cpp: 번역가이자 전달자 (The Translator)

모든 상태 연산과 하드웨어 I/O를 메인 루프 하나에 통째로 우겨넣어 타이밍 이슈가 빈번했던 초기 Ardupilot(APM) 시절의 낡은 C 아키텍처와 달리, PX4는 네트워크 직렬화의 책임을 완전히 별개의 스레드인 MAVLink 모듈 (src/modules/mavlink/)로 위임했다.

이 모듈 깊숙한 곳(mavlink_messages.cpp 주변)에는 오직 HEARTBEAT 메시지 생성만을 위해 평생을 바치는 특수한 스트림 핸들러(예: MavlinkStreamHeartbeat 클래스)가 존재한다.

  1. 구독(Subscribe): 이 핸들러는 앞서 커맨더가 퍼블리싱한 vehicle_status_s uORB 토픽을 조용히 구독(Subscribe)하고 있다.
  2. 변환(Translation): PX4 내부에서만 통용되는 상태 열거형(Enum) 변수들(예: NAVIGATION_STATE_AUTO_RTL)을, 글로벌 관제 표준인 MAVLink 프로토콜의 엄격한 비트마스크(예: MAV_MODE_FLAG_CUSTOM_MODE_ENABLED, PX4_CUSTOM_MAIN_MODE_AUTO, PX4_CUSTOM_SUB_MODE_AUTO_RTL)로 비트 단위 스위칭 변환을 수행한다.
  3. 직렬화(Serialization): 구조체를 mavlink_msg_heartbeat_pack(...) 함수를 통해 전파 위를 날아갈 수 있는 바이트 스트림(Byte Stream) 배열로 직렬화한다.
  4. 전송(Transmission): 직렬화된 바이트 배열을 1Hz 주기의 엄격한 스케줄러 타이머에 맞춰 UART DMA 전송 큐에 밀어 넣는다.

이러한 uORB 기반의 물리적/논리적 결합 분리(Decoupling) 설계는, 드론에 Wi-Fi 동글을 추가하든, 5G LTE 모뎀을 3개 더 달든, 기존 제어 코드를 단 한 줄도 건드리지 않고 개별 통신 포트마다 전담 MAVLink 스트림 객체를 손쉽게 찍어내어 병렬로 심장 박동(Heartbeat)을 퍼뜨릴 수 있는 강력한 확장성(Scalability)의 근간이 된다.
