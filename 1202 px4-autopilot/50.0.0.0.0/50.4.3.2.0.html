50.4.3.2 제어기 포화(Saturation) 조건: 수직 위치 제어기의 비례-적분-미분(PID) 연산 출력값의 하한 포화 도달 및 다중 로터 혼합기(Mixer) 추력 지령의 시스템 최솟값 유지 유효성 검사

지면 감지(Land Detection) 시스템 설계 시, 기체의 수직 속도가 영에 수렴한 사실 단일로는 장애물 얹힘이나 강풍에 의한 단순 호버링 정체와 구별하기 어렵다. 이 물리적 한계를 타개하기 위해 PX4 시스템은 제어 관점의 비행 동역학적 스트레스를 측정하는 두 번째 필수 논리 게이트, 즉 제어기 하한 포화(Controller Lower Bound Saturation) 증명을 요구한다.

1. 하강 궤적 추종 시스템의 모순과 오차 누적

자동 착륙(Auto Land) 모드나 수동 조종의 극저 스로틀 하강 명령이 시달된 상황을 가정해 보자. 위치 제어기(Position Controller)는 캐스케이드(Cascade) P-PID 루프를 통해 초당 일정한 수직 속도(예: 0.5 m/s)로 하강하도록 모터에 추력 감소를 지령한다.

그러나 기체가 지면에 닿은 직후, 제어기는 하강 궤적 지령 위치와 실제 고도(EKF 측정치) 사이의 오차(Error)가 줄어들지 않고 오히려 벌어지는 모순된 상황에 직면하게 된다. 기체의 단단한 착륙 장치가 더 이상의 공간적 위치 변화를 구조적으로 막고 있기 때문이다.

2. PID 제어 루프의 하한 포화 도달 (PID Output Saturation)

  1. 비례(P)항 반응: 고도가 지령치를 따라가지 못함에 따라 기체를 강제로 내리기 위해 P항은 Z축 추력 지령(Thrust Setpoint, T_z)을 급격히 감소시킨다.
  2. 적분(I)항 누적: 오차가 지속되는 시간 동안 I항(Integral) 윈드업 현상이 역으로 작용하여, 시스템 내부에 축적된 하강 요구 에너지가 최저 수준으로 파고든다.

결론적으로 PX4의 수직 위치 제어기 출력값은 시스템이 구조적으로 허용하는 “하락 마지노선“에 도달하게 된다. 소프트웨어적으로 land_detector 모듈은 내부 제어 지표 인스턴스를 통해 추력 지령 세트포인트가 파라미터 최저 한계치(MPC_THR_MIN 등)와 근사한 상태로 억류되어 있는지(thrust_sp_z <= threshold) 확인한다. 제어기가 목표를 달성할 수 없어 수학적으로 포기하고 하한 포화(Lower Bound Saturation)에 걸려 있는가를 진단하는 것이다.

3. 다중 로터 혼합기(Mixer / Control Allocator) 출력 유효성 검증

수직 제어기가 최하단 추력을 요구한다고 해서 위상 조건이 모두 통과되는 것은 아니다. 기체가 불규칙한 돌이나 비스듬한 경사면에 걸쳐졌을 경우, 수평을 유지하려는 자세 제어기(Attitude Controller)가 롤(Roll) 또는 피치(Pitch) 방향의 수평 복원 토크(Torque)를 요구할 수 있다.

이때 다중 로터 혼합기(또는 신규 아키텍처의 Control Allocator)는 전체 시스템 추력을 줄이면서도 각 모터(ESC)로 전달되는 펄스 폭 제어량의 분배를 이끌어낸다. 착륙 검증 로직은 이 믹서의 최종 할당 결과값(Actuator Outputs / Motors)들을 지속해서 스캐닝한다.

  • 특정 축의 모터가 한쪽만 강하게 회전하는 비대칭 공회전이 발생하고 있지는 않은가?
  • 전체 평균 추력 지령 값이 시스템의 공회전 유휴 한도(MPC_THR_IDLE 근방 구간) 내에서 튀어오름(Spike) 현상 없이 평활화(Smoothed) 되어 유지되는가?

PX4는 비정상적인 언밸런스(Unbalance) 제어 출력까지도 제어 포화 로직에 포함시킴으로써 시동 해제 전 기체가 착륙 불가능한 비스듬한 장소에서 전복(Flip-over) 되는 것을 논리적으로 억제한다. 즉, 제어 루프 전반의 최저 출력 포화(Minimum Output Saturation) 상태의 지속은 기체가 물리적으로 바닥에 온전히 지지되어 항법 역학에서 이탈했음을 역설적으로 방증하는 핵심 모델이다.
