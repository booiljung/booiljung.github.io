## 0.1 기체 동역학 및 공기역학 모델 기반의 하강 궤적 생성(Trajectory Generation) 제어

착륙(Land) 명령이 시스템에서 최종적으로 인가되면, 기체는 단순히 모터의 추력(Thrust)을 줄이는 원시적인 제어를 넘어, 기체의 동역학(Rigid-body Dynamics)과 로터 시스템의 공기역학적(Aerodynamic) 한계를 고려한 정교한 하강 궤적(Descending Trajectory)을 생성해야 한다. 현대의 자율 비행 시스템인 PX4-Autopilot(펌웨어 버전 1.14 기준)은 부드러우면서도 정확한 목표 지점 도달을 위해 저크(Jerk) 제한 기법과 다단 속도 제어 프로파일을 결합한 궤적 생성기(Trajectory Generator)를 내부적으로 운용한다.

0.1.1 하강 단계에서의 비행 동역학적 특이성

멀티로터(Multicopter) 유형의 기체가 고도를 낮출 때 겪는 가장 큰 공기역학적 난제는 와류 고리 상태(Vortex Ring State, VRS) 현상이다. 기체가 자신의 로터에서 밀어낸 하향풍(Downwash) 속으로 다시 빠르게 하강하게 되면, 로터 팁 주변에서 거대한 와류(Vortex) 재순환 고리가 형성되면서 양력(Lift)이 급격히 상실된다. 이를 방지하기 위해 PX4의 궤적 생성 알고리즘은 하강 최대 속도 파라미터(예: MPC_Z_VEL_MAX_DN)를 와류 고리 상태가 발생하는 임계 속도 하프-슬립(Half-slip) 영역 이전으로 억제하도록 수학적으로 설계되어 있다.

0.1.2 PX4의 하강 궤적 생성(Trajectory Generation) 모델링

PX4 위치 제어기(mc_pos_control) 내부에 위치한 위치 궤적 생성기(Position Trajectory Generator)는 다음과 같은 2계 또는 3계 미분 방정식 모델을 통해 궤적을 연산한다. 일반적인 하강 제어는 Z축 이동(Earth-fixed NED 프레임 기준 Positive-Z 방향)에 집중된다.

기체의 수직 방향 위치(z), 수직 속도(v_z), 수직 가속도(a_z)의 운동학적 구속 조건은 아래와 같이 정의된다.

v_z(t) = \frac{d z(t)}{dt}, \quad a_z(t) = \frac{d v_z(t)}{dt}

PX4 내부의 궤적 생성 함수(FlightTaskAuto 또는 FlightTaskManualPosition)는 사용자가 지정한 목표 고도(Setpoint)에 도달하기 위해 실시간으로 부드러운 가속과 감속 궤적을 산출한다. 이때 하강 궤적은 스텝 응답(Step Response)으로 발생하는 모터 제어의 오버슈트(Overshoot)를 피하기 위해 저크 제한 필터(Jerk-limited Filter) 및 S-커브(S-curve) 프로파일을 적용하여 생성된다.

0.1.4 동역학 모델 기반의 제어 이점

Ardupilot의 하강 궤적 제어 방식이 전통적으로 고도 오차에 대한 극점 배치(Pole Placement) 또는 1차 필터링(First-order Filtering)에 의존했던 반면, PX4-Autopilot은 독립된 궤적 생성 계층을 통해 시스템의 물리적 한계치(최대 틸트 각도, 모터의 최대/최소 추력 한계, 최대 수직 가속도 등)를 사전 평가하여 수학적으로 도달 가능한(Feasible) 유효 목표 상태량(State)만을 하위 제어기로 넘기는 것이 가장 큰 특징이다.

이러한 기법은 바람과 같은 환경 외란(Environmental Disturbance)이 심한 조건에서도 착륙 지점까지의 궤적 오차(Tracking Error)를 최소화한다. 학술 논문 “Real-time Trajectory Generation for Quadrotors” 등에서 논의된 최적 제어(Optimal Control) 기반의 궤적 평활화 이론과 궤를 같이 하는 이 방식은, 단순히 PID 제어기의 비례 이득(Proportional Gain) 조율에만 의존하던 기존의 관행에서 탈피하여, 드론의 승차감(Ride Quality)과 착륙 안정성을 획기적으로 끌어올리는 자율 비행 시스템의 핵심 기반 기술이다.
