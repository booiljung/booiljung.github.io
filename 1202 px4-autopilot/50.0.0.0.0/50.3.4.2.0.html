50.3.4.2. 고도 하강에 따른 확장 칼만 필터(EKF2) 내부의 기압계 주 고도 추정 모델에서 지상 근접 센서 추정 모델로의 관측 행렬(Observation Matrix) 가중치 전이

1. 고도 센서 융합 및 주 고도 소스 전환의 필요성

무인 항공기가 높은 고도에서 임무 비행을 수행할 때는 주로 기압계(Barometer)나 전 지구 위성 항법 시스템(GNSS)의 고도 데이터를 확장 칼만 필터(EKF2, Extended Kalman Filter)의 주 고도 소스(Primary Height Source)로 활용한다. 그러나 착륙을 위해 고도를 하강하여 지면에 근접하게 되면 기체 로터의 하향풍(Downwash)이 지면에 부딪혀 반사되는 지면 효과(Ground Effect)로 인해 국지적인 기압 변화가 발생하고, 이는 기압계 기반 고도 추정에 심각한 오류(Barometer Ground Effect Error)를 야기한다.

이를 극복하기 위해 최신 PX4-Autopilot(버전 1.14 및 최신 펌웨어 기준) EKF2 모듈은 지상 근접 센서(Range Finder, LiDAR/Radar 등)의 유효 측정 거리 내에 진입하면 센서 데이터 융합 가중치를 동적으로 전이(Transition)한다. 단순히 스위치 형태의 이산적 분기(Discrete Branching)를 사용할 경우, 칼만 필터의 상태 벡터(State Vector)에 급격한 단차(Step Change)가 발생하여 기체의 수직 제어 루프(Vertical Control Loop)에 심각한 물리적 요동을 유발할 수 있다. 따라서 EKF2는 관측 행렬(Observation Matrix) 및 노이즈 공분산(Noise Covariance) 관리를 통해 부드러운 가중치 전이를 수행한다.

2. 관측 행렬(H) 및 측정 노이즈 공분산(R)의 수학적 모델

칼만 필터에서 센서의 관측 업데이트(Observation Update) 과정은 예측된 상태 벡터(\hat{x}_{k|k-1})와 실제 센서 관측치(z_k) 간의 이노베이션(Innovation, \tilde{y}_k)을 통해 이루어진다.

\tilde{y}_k = z_k - h(\hat{x}_{k|k-1})

이를 선형화한 관측 행렬(Observation Matrix) H_k는 다음과 같이 정의된다.

H_k = \left. \frac{\partial h}{\partial x} \right|_{\hat{x}_{k|k-1}}

그리고 이노베이션 공분산(Innovation Covariance) S_k와 칼만 이득(Kalman Gain) K_k는 다음과 같이 계산된다.

S_k = H_k P_{k|k-1} H_k^T + R_k
K_k = P_{k|k-1} H_k^T S_k^{-1}

고도 하강 시, PX4의 EKF2 로직은 기압계와 거리 측정 센서 두 가지 관측치를 독립된 순차적 업데이트(Sequential Update) 방식으로 처리하거나, 측정 노이즈 공분산 R_k를 물리적 상황에 맞게 조정하여 가중치를 이동시킨다. 지상 근접 센서가 유효한 구간(Valid Range)에 들어오면, EKF2는 거리 측정기의 측정 노이즈 공분산 파라미터에 부여되는 실효 신뢰도를 높이고(R_{rng} \downarrow) 칼만 이득(K_{rng})을 비례적으로 확장함으로써 거리 측정기 관측치가 상태 업데이트에 미치는 영향을 지배적으로 만든다.

동시에, 주 고도 소스가 기압계에서 거리 측정기로 변경될 때 추정 상태의 물리적 불연속성을 막기 위해 상태 분산 값(State Variance)을 초기화(Reset)하고 오프셋(Offset)을 모델에 보상하는 상태 재설정(State Reset) 과정이 수반된다.

3. 기압계에서 지상 근접 센서로의 가중치 전이 메커니즘 아키텍처

PX4의 EKF2 내부 소스 코드(주로 src/modules/ekf2/EKF/ekf.cpp 및 control.cpp 영역)를 살펴보면, 주 고도 소스를 결정하는 제어 플래그와 파라미터(예: EKF2_HGT_REF, EKF2_RNG_CTRL)를 기반으로 유한 상태 머신(Finite State Machine)이 동작함을 확인할 수 있다.

거리 측정기를 주 고도로 사용하기 위해서는 EKF2_RNG_CTRL 파라미터가 1(Range finder as primary height source)로 설정되어야 하며, 측정 조건이 충족되어야 한다. 전이 통계 검증은 다음과 같은 절차적 로직으로 구성된다.

  1. 지면 근접 감지 (Ground Proximity Detection): 거리 측정기의 데이터가 센서 고유의 유효 임계치(EKF2_RNG_A_VMAX 등) 윈도우 내에 지속해서 존재하는지 판단한다.
  2. 이노베이션 일관성 검증 (Innovation Consistency Check): 기압계와 거리 근접 센서의 이노베이션 값 차이를 정규화하여 분석하고 이상치(Outlier)를 통계적으로 배제한다.
  3. 가중치 전이 (Weight Fade / State Variance Blending): 상태 벡터와 공분산 행렬을 스무딩(Smoothing)하기 위한 페이드 인/페이드 아웃(Fade-In/Out) 함수가 적용되어 스텝 응답 타격 모델을 우회한다.
  4. 상태 분산 초기화 및 스위칭 (State Variance Reset & Switching): 안정화가 확인되면 EKF2의 Z 상태 기준계(Z-axis Reference System)를 거리 측정기 기준으로 완전히 치환한다.

4. EKF2 control.cpp 내의 제어 로직 분석

PX4 소스 코드 수준에서 데이터 소스가 완전히 대체될 때 발생하는 수학적 상태 도약(State Jump) 현상을 제어하기 위해 사용되는 핵심 함수 중 하나는 resetHeight() 함수이다. 주 고도 센서가 기압계에서 거리 측정기로 변경될 때, 기존 EKF2 추정 고도와 새로운 지상 근접 센서 기반 예측 고도 간의 오프셋을 역산하여 관성 항법 로컬 포지션(Local Position NED) Z축 좌표계에 일괄적으로 감산 반영한다.

가중치 전이 시간 구간에서는 관측 행렬 H 차원 축소에 반영되는 각 센서의 신뢰 계수인 이노베이션 게이트(Innovation Gate) 시스템 파라미터(EKF2_BARO_GATE, EKF2_RNG_GATE)가 엄격한 상호작용 피드백을 형성한다. 거리 측정기가 검증 로직을 통과하면, 거리 센서의 이노베이션이 내부 게이트 한계치를 통과하게 되어 능동적 필터 내 관측치로 지배 권한을 인수받는다.

이후 EKF 모드 컨트롤러(Mode Controller)는 내부 논리 플래그(_control_status.flags.rng_hgt)를 참(True)으로 활성화하며, 기존에 사용되던 기압계 플래그(_control_status.flags.baro_hgt) 파라미터를 비활성화한다. 이 시점에서 상태 예측 공분산 행렬 P_{k|k-1}의 고도 분산 성분 및 수직 속도(Velocity Z) 성분은 기압계 노이즈 특성에서 거리 센서 노이즈 특성으로 신속하게 치환되며, 수치 연산에서의 양의 정부호성(Positive Definiteness)을 보호하기 위해 숄레스키 분해(Cholesky Factorization) 혹은 조셉 형태(Joseph Form)의 수학적 공분산 업데이트가 강제된다.

5. ArduPilot (EKF3) 대비 아키텍처 설계 철학의 차이

ArduPilot의 상태 추정 알고리즘인 EKF3도 역시 지상 근접 시의 기압계 센서 왜곡 문제를 인지하고 거리 측정기 기반 모델로 가중치를 동적으로 전이하지만, 그 세부적인 구현 논리와 코드의 철학적 파생에서 핵심적인 차이점을 보유한다.

  • 분산 혼합(Variance Blending) 철학 차이: PX4의 EKF2는 소스 전환 시 이산적 플래그 스위칭과 함께 예측 상태 오프셋 동기 리셋(Synchronous Offset Reset) 기법을 취하여, 물리적 스텝 펄스가 수직 제어기(Z Position Controller)에 인가되는 것을 즉각 차단하는 데 주안점을 둔다. 반면, ArduPilot의 EKF3 알고리즘은 측정 노이즈 공분산 R 자체를 연속적으로 그리고 지수적(Exponentially)으로 혼합(Blending)하는 가중치 지연 체계를 적극적으로 구현한다. (실무 설정에서는 EK3_RNG_USE_HGT 파라미터 활용)
  • 지면 효과 사전 억제(Ground Effect Compensation): PX4는 거리 측정기 장비가 물리적으로 연결되지 않거나 일시적 펄스 장해로 인해 사용 불가인 상태에서도 기압계 데이터에 내재하는 지면 효과를 역추산하여 보상하기 위한 전용 수학적 보상 컴포넌트(Ground Effect Compensation Mechanism)를 자체 내장하고 있다. 이는 가중치 전이 지점이 붕괴하였을 때 발생하는 치명적인 고강도 외란 발산을 근원적으로 무력화하는 안전성을 띠고 있다.

6. 결론 및 공학적 시사점

고도 하강 트래젝토리에 동조화된 관측 행렬 단위의 주 고도 추정 가중치 전이는 무인기 상태 추정 체계에서 정밀 자동 궤도 착륙(Auto Landing System)과 이종 수직 이착륙(VTOL) 기체의 형상 천이 후 물리적 터치다운을 관장하는 가장 핵심 안전 추론 알고리즘이다.

시스템 통합 관점에서 센서에 매몰된 단일 지향성 맹신으로 인한 심각한 추정 오차를 확장 칼만 필터의 확률론적 지석을 통해 능동적으로 블렌딩함으로써, 외부 거스트(Gust), 난기류 물리적 외란이나 단기적 기압계 맥동 현상(Barometer Surge)을 제어 주기 내에서 신속하게 억제한다. 비행 제어기 통합 엔지니어 및 R&D 개발자는 시스템 튜닝 실무 과정에서 반드시 이노베이션 유효 게이트 파라미터(EKF2_RNG_GATE, EKF2_BARO_GATE)와 측정 환경 노이즈 공분산(EKF2_RNG_NOISE, EKF2_BARO_NOISE) 파라미터 간의 상관 스케일 교정을 통해, 본 관측 행렬 가중치 전이 논리가 정상적으로 성립되도록 시뮬레이션 데이터나 실비행 로그(ULog)를 바탕으로 엄격히 감리해야 한다.
