## 0.1 다중 로터(Multicopter) 시스템의 3차원 운동학적 종방향(Longitudinal) 하강 제어 최적화

다중 로터(Multicopter) 기반의 무인항공기에서 자동 착륙(Auto Land) 또는 복귀(Return) 시퀀스의 하강 단계는 단순히 고도를 낮추는 행위가 아니라, 기체의 3차원 공간상의 운동학적(Kinematic) 상태를 Z축(종방향, Longitudinal)으로 안전하게 수렴시키는 고도화된 최적 제어(Optimal Control) 과정이다. PX4-Autopilot(펌웨어 버전 1.14 이상 기준) 시스템 내에서 이러한 수직 하강 제어는 위치 제어기(Position Controller, mc_pos_control)와 비행 작업(Flight Task) 아키텍처에 의해 정밀하게 조율된다.

0.1.1 종방향(Longitudinal) 하강 제어 최적화의 수학적 정의

기체가 3차원 공간에서 횡방향(Lateral, X-Y 평면) 이동을 멈추고 종방향(Z축)으로만 하강하는 경우, 다중 로터의 동역학적 특성상 메인 추진력(Thrust) 방향과 하강 방향이 정확히 평행(Parallel) 상태에 놓이게 된다. 이러한 비행 체제(Flight Regime)에서는 프로펠러의 하향풍(Downwash)이 기체의 항력(Drag) 및 주변 기류와 심각하게 간섭하여 공기역학적 불안정성을 초래하며, 심할 경우 와류 고리 상태(Vortex Ring State, VRS)에 빠지게 된다.

따라서 PX4 내부의 하강 궤적 최적화 문제는 다음과 같은 운동학적 구속 조건(Kinematic Constraints)을 충족하면서 기체를 목표 고도 Z_{target} (통상 지표면)에 도달시키는 최소 시간(Minimum-time) 또는 최소 에너지(Minimum-energy) 제어 문제로 정형화된다.

  • 고도 스텝 구속(Position Constraint): Z_{current} \rightarrow Z_{target}
  • 하강 최대 속도 구속(Velocity Constraint): V_z \le V_{max\_down} (제어 파라미터: MPC_Z_VEL_MAX_DN)
  • 가속도/저크 구속(Acceleration/Jerk Constraint): 부드러운 승차감과 모터의 출력 포화(Saturation) 방지를 위해 수직 가속도 A_z와 가속도 변화율 J_z를 엄격하게 제한

이러한 문제 해결을 위해 PX4의 위치 궤적 생성기(Position Trajectory Generator)는 최적 제어 이론의 형태 형성(Shaping) 알고리즘을 도입하여 종방향 하강 속도의 불연속성을 제거하고 최적화한다.

0.1.3 3차원 운동학적 최적화 구현 (C++ 소스 코드 분석)

PX4 내부의 mc_pos_control 모듈에서 종방향 하강을 위한 코어 로직은 PositionControl::control_position() 및 PositionControl::control_velocity() 함수를 통해 이루어진다.

특히, 하강 속도가 한계치(MPC_Z_VEL_MAX_DN)에 도달했을 때, 과도한 적분기 누적(Integrator Windup)으로 인한 기체의 튀어오름 현상(Bouncing)을 방지하기 위해 안티-와인드업(Anti-windup) 메커니즘을 포함한 속도 오차 최적화 코드가 아래와 유사한 수학적 기반 위에서 구현되어 있다.

// 수직 속도(Z-axis) 제어 최적화 로직의 직관적 예시 차용
void PositionControl::control_velocity(const float dt) {
    // 1. Z축 종방향 속도 오차 계산 (목표 하강 속도 - 현재 추정 하강 속도)
    float vel_error_z = _vel_sp(2) - _vel(2);

    // 2. 비례(Proportional) 제어 이득 적용을 통한 목표 수직 가속도 산출
    float accel_sp_z = vel_error_z * _gain_pos_z;

    // 3. 적분 제어 및 Anti-windup 로직 적용을 통한 정상 상태 오차(Steady-State Error) 최적화
    if (!thr_sp_z_saturated) {
        _vel_int(2) += vel_error_z * _gain_vel_i_z * dt;
    }

    // 4. 관성(Inertia) 피드포워드와 중력 보상(Gravity Compensation)을 포함한 최종 추력 연산
    float thrust_sp_z = accel_to_thrust_z(accel_sp_z + _vel_int(2) - CONSTANTS_ONE_G);

    // 5. 공기역학적 한계치를 넘지 않도록 Clamp 적용 (수학적 최적화의 동역학적 한계 설정)
    _thr_sp(2) = math::constrain(thrust_sp_z, -_param_mpc_thr_max.get(), -_param_mpc_thr_min.get());
}

이러한 캐스케이드(Cascaded) 제어 구조는 단순한 정면 호버링(Hovering) 상태에서의 수직 하강뿐만 아니라, 전진 비행(Forward Flight) 중 사선 방향으로 하강할 때(예: 비행기나 고정익 드론의 활공 경로와 유사한 복합 궤적) X-Y 평면의 위치 제어 오차와 Z축 종방향 위치 오차를 직렬적・직교적(Orthogonal)으로 완벽하게 분리(Decoupling)한다.

결론적으로, PX4의 다중 로터 시스템에 적용된 3차원 운동학적 종방향 제어 최적화 기법은, 로봇공학계의 주요 논문 *“Minimum Snap Trajectory Generation and Control for Quadrotors”*에서 제시된 최적 차수 미분 방정식 최소화 기법의 설계 철학을 반영하여, 극단적인 공력 역류 및 돌풍 조건 속에서도 드론이 완전히 통제된 안정적인 접지면 접근 궤적을 유지할 수 있도록 하는 강력한 공학적 기반을 구현해냈다.
