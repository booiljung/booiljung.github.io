4.1. 개발 생태계(IDE/CLI) 원격 제어 및 버전 관리 통합

단순히 브라우저에서 대화하는 AI 챗봇을 넘어, 에이전트가 소프트웨어 엔지니어의 핵심 도구인 통합 개발 환경(IDE)과 버전 관리 시스템(Version Control)에 어떻게 직접 침투하여 제어권을 행사하는지 다룬다.

MCP는 AI가 코드 에디터의 커서를 움직이고, 터미널을 열고, 깃허브에서 풀 리퀘스트(PR)를 작성하도록 만드는 거대한 파이프라인이 된다. 본 장에서는 코드가 생산되고 저장되는 최전선의 하위 레이어 아키텍처를 해부한다.
