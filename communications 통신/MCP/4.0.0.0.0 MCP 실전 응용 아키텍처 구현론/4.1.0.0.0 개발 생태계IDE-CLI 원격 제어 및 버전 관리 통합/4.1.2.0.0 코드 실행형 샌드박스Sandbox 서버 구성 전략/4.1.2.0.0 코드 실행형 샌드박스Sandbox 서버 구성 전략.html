4.1.2. 코드 실행형 샌드박스(Sandbox) 서버 구성 전략

LLM 에이전트가 코드를 작성하는 것을 넘어, “이 파이썬 스크립트를 지금 당장 실행해서 결과를 확인해 봐“라고 요구받을 때 보안과 시스템 안정성에는 거대한 적신호가 켜진다.

외부 서버, 혹은 심지어 사용자의 로컬 환경에서 알 수 없는 코드가 실행(Eval)된다는 것은 RCE(원격 코드 실행) 해킹과 동일한 파괴력을 지니기 때문이다. 그렇기 때문에 에이전트의 코드 파이프라인에는 반드시 호스트 OS와 완벽하게 격리된 ’샌드박스(Sandbox)’가 필수적이다. 본 장에서는 안전한 런타임 제어망과 통제된 실행 환경을 설계하는 아키텍처를 논한다.
