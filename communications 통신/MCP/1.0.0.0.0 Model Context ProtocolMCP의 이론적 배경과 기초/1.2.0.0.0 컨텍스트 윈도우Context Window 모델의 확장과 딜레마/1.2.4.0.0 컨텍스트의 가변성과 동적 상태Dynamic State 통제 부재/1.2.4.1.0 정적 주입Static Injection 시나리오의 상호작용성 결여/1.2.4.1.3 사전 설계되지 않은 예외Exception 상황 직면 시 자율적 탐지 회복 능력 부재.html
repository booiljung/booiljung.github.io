1.2.4.1.3 사전 설계되지 않은 예외(Exception) 상황 직면 시 자율적 탐지 회복 능력 부재

대규모 언어 모델(LLM)을 연동한 정적 주입(Static Injection) 기반의 파이프라인이 가진 가장 절망적인 약점은 이른바 ’행복한 경로(Happy Path)’에서 벗어났을 때 드러나는 시스템의 총체적 마비 현상이다. 실전 비즈니스 환경에서는 데이터베이스의 스키마 변경, API의 일시적 404 에러, 권한 부족 등 설계자가 사전에 코딩하지 않은 수많은 예외(Exception) 상황이 발생한다. 이때 정적 시스템은 오류를 스스로 탐지(Autonomous Detection)하고 대체 경로를 찾아내는 회복(Recovery) 능력이 전무하다.

0.1 맹목적 실행과 오류 반환의 치명적 결함

일반적인 RAG 프로그래밍(예: 하드코딩된 프롬프트 체인)에서는 시스템이 에이전트의 자율성을 제한하고 정해진 순서대로 데이터를 가져와 밀어 넣는다.

만약 시스템이 “주요 고객의 2분기 계약 문서를 읽어와서 요약해“라는 지시를 받았다고 가정하자. 그런데 해당 경로의 문서가 전날 밤에 삭제되어 존재하지 않거나 액세스 권한(Access Denied) 거부 응답이 돌아왔다면, 정적 시스템은 에러 코드(예: HTTP 404 또는 403) 문장 자체를 문맥(Context)으로 취급해 LLM에게 던져버리는 어리석은 동작을 취한다.
결과적으로 LLM은 “고객의 2분기 계약 내용은 404 Not Found에 의해 성공적으로 체결되었습니다“라는 충격적인 환각(Hallucination)을 생성하거나, 단순히 “오류가 발생했습니다“라는 문장과 함께 파이프라인 전체를 정지(Crash)시켜버린다. 이는 어떠한 우발적 상황에서도 다른 데이터베이스를 뒤져보거나, 우회하는 검색어(Search Term)를 자율적으로 생성해 문제를 돌파해 내는 지능형 자기 회복(Self-healing) 기능이 근본적으로 단절되어 있음을 의미한다.

0.2 정적 예외 처리(Static Exception Handling)의 공학적 한계

개발자들이 이러한 문제를 해결하기 위해 도입하는 일반적인 방식은 try-catch 구문이나 If-else 분기문을 코드 에 덕지덕지 바르는 것이다. “만약 API 1이 죽으면, API 2를 호출한다“와 같은 예외 처리 트리(Exception Tree)를 사람이 수동으로 하드코딩하는 방식이다.

하지만 무한대에 가까운 경우의 수를 가지는 자연어 질의응답 환경에서, 인간이 모든 예외 트리를 사전에 매핑하여 코드로 예측하는 것은 불가능하다. 도메인의 범위가 넓어질수록 하드코딩된 예외 처리망은 필연적으로 구멍이 뚫리게 되며, 결국 비즈니스 로직과 예외 처리 로직이 통제할 수 없을 정도로 얽혀버리는 ‘스파게티 아키텍처(Spaghetti Architecture)’ 붕괴로 나아가게 된다.

0.3 Model Context Protocol(MCP) 기반의 동적 자체 회복(Self-Healing) 라우팅

예외 상황을 맞닥뜨렸을 때 LLM 그 자신이 스스로 “어, 이 문서가 없네? 그럼 다른 폴더를 찾아봐야겠다“라고 자율적으로 판단하여 도구를 다시 쥐게 만드는 진정한 수준의 회복 탄력성(Resilience)은 Model Context Protocol(MCP)의 동적 예외 처리 규약에서 완성된다.

MCP 표준은 외부 시스템(Server)이 도구의 실행 실패를 경험했을 때, 이를 모호한 텍스트가 아닌 명확하게 규격화된 에러 코드(JSON-RPC Error Object)로 반환하도록 강제한다. LLM 클라이언트는 이 규격화된 에러를 받아들고, 인간의 개입 없이도 자체적인 재귀 루트(Self-Correction Loop)를 가동한다.
에이전트는 실패의 원인을 스스로 추론(Reasoning)하고, 보유하고 있는 다른 MCP 도구들의 명세서(Manifest)를 훑어본 뒤 “A 문서고가 닫혔으니, B 웹 서치 도구를 꺼내어 대체 검색을 실시하라“는 지시를 즉각 실행한다. 이것이 바로 정적인 스크립트가 절대 흉내낼 수 없는, Model Context Protocol만이 빚어낼 수 있는 인공지능의 주도적 디버깅(Proactive Debugging)이자 시스템 자율 회복망 네트워크의 진수다.
