1.1.1.3.2 파인튜닝(Fine-Tuning)을 통한 지식 주입 시 발생하는 치명적 망각(Catastrophic Forgetting) 현상

대규모 언어 모델(LLM)의 정적 지식 한계를 극복하고 도메인 특화 지식을 내재화하기 위해 산업계에서 가장 흔하게 시도되는 방법론은 미세 조정, 즉 ’파인튜닝(Fine-Tuning)’이다. 파인튜닝은 사전 학습(Pre-training)이 완료된 거대한 기초 가중치(Base Weights)를 특정 기업이나 태스크의 작은 데이터셋으로 추가 학습시켜, 파라미터(Parameter) 네트워크의 국소적인 연결 강도를 미세하게 재조정하는 기법이다. 그러나 이 접근법을 지식 주입(Knowledge Injection)의 주력 수단으로 사용할 경우, 신경망 아키텍처의 근본적인 취약점인 ‘치명적 망각(Catastrophic Forgetting)’ 현상을 필연적으로 마주하게 된다.

치명적 망각이란, 인공 신경망이 순차적으로 새로운 정보나 에피소드를 학습할 때, 이전에 학습하여 굳어진 기존의 범용적 가중치를 급격하게 덮어쓰거나 파괴함으로써 과거의 지식과 추론 능력을 극심하게 상실하는 현상을 지칭한다. 언어 모델의 가중치 행렬은 상호 독립적인 파일 폴더 구조를 지니지 않고, 수천억 개의 파라미터가 비선형적으로 복잡하게 얽혀 있는 전역적(Global) 상태 공간이다. 따라서 이 공간의 한쪽 모서리에 기업의 맞춤형 매뉴얼 정보를 주입하기 위해 경사 하강법(Gradient Descent)을 가동하면, 그 파급 효과(Ripple Effect)는 신경망 전체로 번져 나가 애써 구축해둔 보편적 자연어 추론 능력과 상식(Common Sense) 영역의 파라미터들마저 왜곡시켜 버린다.

이러한 망각 현상은 파인튜닝 프로세스를 고도로 불안정하고 예측 불가능한 엔지니어링 도박으로 변질시킨다. 데이터 엔지니어가 최신 법률 판례 1,000건을 파인튜닝으로 모델에 주입했다고 가정해 보자. 테스트 결과, 모델이 새로운 판례에 대해서는 정답을 도출할지 모르지만, 부작용으로 인해 갑자기 기본적인 민법 상식을 답변하지 못하거나, 문장의 어법이 망가지고, 심지어 허위 정보 생성(Hallucination) 비율이 폭발적으로 증폭되는 치명적인 성능 퇴행(Regression)을 경험할 수 있다.

이를 방지하기 위해 탄성 가중치 통합(Elastic Weight Consolidation, EWC)이나 파라미터 효율적 미세 조정(PEFT, Parameter-Efficient Fine-Tuning) 기법 중 하나인 LoRA(Low-Rank Adaptation) 등의 방법론이 등장하였으나, 이들 역시 근본적인 한계를 지닌다. 이들은 지식의 구조적 파괴를 ’지연’시키거나 ’완화’시킬 뿐, 신경망 내부 메모리에 팩트(Fact)를 영구적이고 무결하게 결합시키는 해결책은 아니다.

결과적으로, 파인튜닝 메커니즘은 모델의 어조(Tone), 말투 형식(Format), 혹은 지시 수행 능력(Instruction Following)을 교정하는 ’행동 튜닝(Behavior Tuning)’의 목적으로는 매우 훌륭하지만, 변화하는 세상의 수많은 ’사실 지식(Fact Knowledge)’을 업데이트하고 쑤셔 넣기 위한 주입 파이프라인으로서는 최악의 아키텍처적 선택지임을 증명한다. 변화하는 팩트는 모델 외벽의 데이터스토어에 안전하게 격리 보관하고, Model Context Protocol (MCP)와 같은 통합 프로토콜을 통해 추론 시점(Inference Time)에 동적으로 결합시키는 것이 치명적 망각을 영구적으로 우회하는 유일한 시스템 공학적 정답이다.
