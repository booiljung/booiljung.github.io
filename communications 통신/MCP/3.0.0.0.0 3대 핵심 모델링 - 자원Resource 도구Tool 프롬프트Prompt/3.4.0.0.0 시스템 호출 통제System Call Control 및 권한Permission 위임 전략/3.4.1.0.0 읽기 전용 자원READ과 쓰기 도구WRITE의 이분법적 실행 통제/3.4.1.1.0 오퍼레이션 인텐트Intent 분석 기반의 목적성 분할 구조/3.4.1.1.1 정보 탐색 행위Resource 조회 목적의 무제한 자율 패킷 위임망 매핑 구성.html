3.4.1.1.1. 정보 탐색 행위(Resource 조회) 목적의 무제한 자율 패킷 위임망 매핑 구성

에이전트가 resources/read 혹은 정해진 prompts/get 메서드를 호출한다면, 인텐트 분석기는 이를 ‘단순 정보 탐색(Safe-Read)’ 성격으로 규정한다.

  1. 지름길(Fast-Path) 라우팅: 읽기 전용으로 확정된 요청은 무거운 트랜잭션 락(Lock)이나 인간 작업자의 승인 창을 완전히 바이패스(Bypass)하여 즉각 파일 시스템 인터페이스로 진입한다.
  2. 무제한 자율 탐색 보장: LLM은 스스로 코드를 분석하기 위해 초당 수십 개의 파일을 열어볼(Read) 수 있다. 이러한 고속의 지식 축적을 방해하지 않기 위해 읽기 행위에는 과도한 쓰로틀링(Throttling)을 걸지 않으며 에이전트의 완전 자율 탐색망을 뒷받침해 준다.
  3. OS 레벨의 읽기 강제화: 혹여 이 모드에서 에이전트가 간접적으로 파일을 쓰려고 시도하더라도, 이 파이프라인이 할당받은 스레드는 운영체제 단에서 O_RDONLY 모드로만 디스크립터를 열잡기 때문에 원본 훼손(Corruption)이 발생할 여지가 원천 봉쇄된다.
