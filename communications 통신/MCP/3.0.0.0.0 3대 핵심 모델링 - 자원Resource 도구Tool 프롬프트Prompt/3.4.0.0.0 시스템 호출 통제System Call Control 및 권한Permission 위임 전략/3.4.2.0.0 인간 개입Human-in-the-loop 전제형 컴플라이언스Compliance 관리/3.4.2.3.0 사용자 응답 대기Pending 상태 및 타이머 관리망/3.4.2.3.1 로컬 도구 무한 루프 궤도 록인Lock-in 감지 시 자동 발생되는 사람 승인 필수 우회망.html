3.4.2.3.1. 로컬 도구 무한 루프 궤도 록인(Lock-in) 감지 시 자동 발생되는 사람 승인 필수 우회망

사용자가 자리를 비운 사이, 자유를 부여받은(Auto-approve) 에이전트가 “파일 읽기 -> 분석 -> 쓰기“를 반복하다 논리적 오류에 빠져 동일한 행위를 무한 루프(Infinite Loop)치게 될 수 있다. 이는 디스크 풀(Full)을 유도하는 재앙이다.

  1. 에코 체임버(Echo Chamber) 추적기: 감시 데몬은 최근 20회의 턴(Turn) 내에서 동일한 도구를 동일한 파라미터(또는 미세한 편차)로 3회 이상(예: 록인 임계치) 반복 호출하는 이상 징후를 감지한다.
  2. 자율 권한의 오토-드롭(Auto-Drop): 즉시 이 에이전트의 “무조건 승인(Auto-approve)” 캐시 토큰 무결성을 박탈시키고, 워크플로우를 강제로 멈춰 세워(Suspend) 인간 승인 루프(Human-in-the-loop)로 궤도를 우회(Fallback)시킨다.
  3. 명시적 루프 브레이킹: 사용자가 돌아왔을 때, “에이전트가 동일한 get_stock_price('MSFT') 도구를 15회 반복 호출하고 있어 강제 정지시켰습니다. 계속하시겠습니까?” 라는 다이얼로그를 띄워 인간의 판단에 따른 명시적인 루프 파괴(Loop Breaking)를 유도한다.
