3.1.1.4. 접근 권한 제어 및 가시성(Visibility) 필터 검사

VFS(가상 파일 시스템) 모델을 통해 파일 경로를 추상화했더라도, 그 안에 담긴 모든 파일을 에이전트가 자유롭게 열람할 권리가 주어지는 것은 아니다. 에이전트에게는 프로젝트의 구조를 파악할 자유가 필요하지만, 호스트 운영체제의 .ssh/id_rsa나 프로젝트 내의 .env 파일과 같은 치명적인 보안 자산은 그 존재조차 숨겨야 한다.

이 절에서는 에이전트가 resources/list나 read를 시도할 때, 라우터 계층보다 앞서 개입하여 응답 패킷에서 특정 자원을 아예 도려내는 가시성(Visibility) 필터 및 접근 권한 제어망의 작동 원리를 파헤친다.
