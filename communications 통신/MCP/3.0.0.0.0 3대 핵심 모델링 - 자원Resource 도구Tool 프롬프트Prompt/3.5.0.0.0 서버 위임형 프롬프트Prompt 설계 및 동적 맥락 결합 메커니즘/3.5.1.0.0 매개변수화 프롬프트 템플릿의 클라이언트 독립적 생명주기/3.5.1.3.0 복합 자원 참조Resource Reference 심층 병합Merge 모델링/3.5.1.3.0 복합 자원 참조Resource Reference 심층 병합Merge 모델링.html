3.5.1.3. 복합 자원 참조(Resource Reference) 심층 병합(Merge) 모델링

프롬프트는 단순한 텍스트 쪼가리가 아니다. 때로는 수 메가바이트의 로그 파일, 위키 문서, 라이브 데이터베이스 쿼리 결과값이 하나의 문맥(Context) 안에 통째로 용해되어 들어가야 한다.

서버 사이드 프롬프트 생태계에서는 이 무거운 파일(자원) 지시문들을 일일이 문자열로 들고 있는 대신, **포인터(URI)**만을 들고 있다가 렌더링 직전에 치환시키는 고도의 자원 병합(Resource Merge) 엔진이 돌아간다. 본 절은 이 과정을 다룬다.
