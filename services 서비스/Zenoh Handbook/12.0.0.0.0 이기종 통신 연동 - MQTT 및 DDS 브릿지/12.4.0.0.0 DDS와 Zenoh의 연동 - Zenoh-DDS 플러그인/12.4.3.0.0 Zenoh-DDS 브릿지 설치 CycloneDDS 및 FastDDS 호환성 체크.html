12.4.3 Zenoh-DDS 브릿지 설치 (CycloneDDS 및 FastDDS 호환성 체크)

Zenoh 는 오만하지 않다. 세상의 메이저 DDS 벤더 두 곳(Eclipse CycloneDDS, eProsima FastDDS) 의 통신 규격(RTPS/CDR) 과 완벽한 호환을 이뤄냈다.

0.1 [Runbook] 브릿징 엔진 설치 및 구동 확인 전술

1. 브릿지 단독 설치
11장에서 보았던 것과 동일한 데비안 패키지(혹은 Cargo 빌드)를 이용해 브릿지를 꽂는다.

sudo apt install zenoh-bridge-dds

2. DDS 벤더 중독성(Vendor-Locking) 탈피
당신이 속한 회사가 CycloneDDS(ROS2 Galactic 이후 기본)를 쓰든, FastDDS(ROS2 Foxy 기본)를 쓰든 상관없다.
zenoh-bridge-dds 는 기저의 네트워크 와이어 프로토콜(DDSI-RTPS) 을 완벽히 스니핑하여 두 진영 모두에 호환된다.

# 그냥 켜기만 하면 현재 망에 떠다니는 모든 RTPS 프레임을 낚아챈다.
zenoh-bridge-dds -l tcp/0.0.0.0:7447

켜는 순간 화면에 [INFO] Discovered Writer... 라는 로그가 미친 듯이 쇄도한다면 성공적으로 DDS 코어에 빨대를 꽂은 것이다.
