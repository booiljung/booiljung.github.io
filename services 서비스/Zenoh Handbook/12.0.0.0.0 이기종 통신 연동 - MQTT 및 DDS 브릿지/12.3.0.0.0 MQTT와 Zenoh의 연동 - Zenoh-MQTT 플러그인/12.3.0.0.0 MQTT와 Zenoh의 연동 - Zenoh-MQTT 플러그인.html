12.3 MQTT와 Zenoh의 연동: Zenoh-MQTT 플러그인

세상 모든 IoT 강의가 MQTT 로 시작하고 끝난다. 수백만 개의 값싼 ESP32 센서들과 스마트홈 기기들이 이미 거대한 모스퀴토(Mosquitto) 브로커에 묶여 있다.

이 거대한 레거시 자산을 휴지통에 버리고 모두 Zenoh Pico 로 재코딩할 것인가?
아니다. 철저히 MQTT 브로커에 빨대를 꽂고 모든 IoT 센서 데이터를 Zenoh 라우팅 테이블 안으로 단숨에 블랙홀처럼 집어삼키는 zenoh-plugin-mqtt 의 약탈적 브릿징 전술을 시연한다.
