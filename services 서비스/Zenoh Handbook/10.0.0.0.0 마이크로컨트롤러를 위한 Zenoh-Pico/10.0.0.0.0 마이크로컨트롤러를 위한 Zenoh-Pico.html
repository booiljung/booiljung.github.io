10. 마이크로컨트롤러를 위한 Zenoh-Pico

“10원의 기적: 램(RAM) 10 킬로바이트(KB) 위에서 돌아가는 글로벌 통신망.”

지금까지 다루었던 Rust, Node.js, Go 언어의 Zenoh 스택이 “기가바이트(GB)” 단위의 넉넉한 서버나 최소한 라즈베리 파이를 전제했다면, 이 챕터는 차원이 전혀 다른 지옥도(Inferno)를 다룬다.

가로세로 1cm 도 안 되는 검은색 플라스틱 칩.
운영체제(OS)라곤 존재하지 않거나, 기껏해야 FreeRTOS 가 돌아가는 극단적인 환경.
이곳에서는 메모리를 동적 할당(malloc) 하는 행위조차 터부(Taboo)시 되며, 네트워크 스택마저 메인 CPU 코어가 한 땀 한 땀 계산해서 밀어내야 한다.

Zenoh-Pico는 바로 이 마이크로컨트롤러(MCU) 생태계를 완벽하게 지배하기 위해 모든 뼈와 살을 깎아내고 척추 하나만 남겨놓은 “C 원시 세포(Primitive Cell)” 다.
이 챕터는 단순한 코딩 매뉴얼이 아니라, 아두이노(Arduino)나 ESP32, STM32 와 같은 싸구려 천 원짜리 칩들을 전 지구적 클라우드망에 0.1ms 의 지연 시간으로 직결시켜버리는 임베디드 흑마법(Dark Arts) 런북이다.
