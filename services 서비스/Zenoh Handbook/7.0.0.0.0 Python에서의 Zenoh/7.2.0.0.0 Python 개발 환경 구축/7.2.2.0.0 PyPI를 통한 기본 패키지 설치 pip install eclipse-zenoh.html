7.2.2 PyPI를 통한 기본 패키지 설치 (pip install eclipse-zenoh)

환경 격리가 끝났다면 이 한 줄로 끝난다. 가장 안전하고 권장되는 방법이다.

0.1 [Runbook] 표준 인스톨 절차

# 최신 안정화 버전(Stable API) 설치
pip install eclipse-zenoh

만약 터미널에서 Building wheel for eclipse-zenoh (setup.py) ... \ 같은 메시지가 뜨면서 5분 이상 멈춰 있다면 위험 신호다.
이는 당신의 OS나 Python 버전에 맞는 미리 구워진 바이너리(whl) 릴리즈가 없어서, Pip가 소스코드를 처음부터 Rust로 컴파일하려고 시도하는 중압감 상태 임을 뜻한다. 리눅스 보드에 Rust/Cargo 가 설치되어 있지 않다면 곧 빨간 글씨와 함께 폭발하게 될 것이다.
(이 현상이 발생하면 OS 커널 64비트 여부와 Python 버전이 3.8+ 인지 재점검하라.)

설치가 0.5초 만에 완료되었다면 성공이다.

# 파이썬 콘솔에서 타임 스탬프 및 버전 출력 스모크 테스트
import zenoh
print(zenoh.__version__)

버전 문자열(예: 1.0.0)이 깨끗하게 떨어지면, 하부 Rust 코어가 성공적으로 파이썬 C-API 에 안착한 것이다.
