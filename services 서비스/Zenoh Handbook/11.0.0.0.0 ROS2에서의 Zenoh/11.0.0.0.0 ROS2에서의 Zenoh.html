Chapter 11. ROS2에서의 Zenoh

로봇 운영체제(Robot Operating System 2, ROS2)는 지구상의 수많은 로봇 엔지니어들을 분산 컴퓨팅의 공포로부터 해방시켜 주었다.
그러나 ROS2가 미들웨어 표준으로 삼았던 DDS(Data Distribution Service)는 세월이 흐르고 로봇의 무대가 넓어짐에 따라, 그 육중한 덩치와 까다로운 네트워크 제약조건 때문에 점차 로봇 엔지니어들의 숨통을 조이는 족쇄가 되어버리고 말았다.

Wi-Fi가 끊기는 공장 바닥, 여러 대의 모바일 로봇이 클라우드와 연결되어야 하는 드넓은 농장, 마이크로컨트롤러부터 거대한 관제 서버까지 이어지는 이 복잡하고 잔혹한 네트워크 환경에서 살아남기 위해, ROS2 생태계는 마침내 **“차세대 미들웨어”**로서 Zenoh를 지목했다!

이 11장에서는 기존 DDS 엔진이 왜 현대 로보틱스 환경에서 붕괴했는지를 적나라하게 파헤치고, Zenoh가 어떻게 ROS2의 새로운 심장(rmw_zenoh)으로 이식되고 브릿지(zenoh-bridge-dds)로 활용되며 인터넷망과 엣지(Edge) 기기들을 씹어 삼키는지 그 경이로운 통합 아키텍처를 전방위로 해부한다!
