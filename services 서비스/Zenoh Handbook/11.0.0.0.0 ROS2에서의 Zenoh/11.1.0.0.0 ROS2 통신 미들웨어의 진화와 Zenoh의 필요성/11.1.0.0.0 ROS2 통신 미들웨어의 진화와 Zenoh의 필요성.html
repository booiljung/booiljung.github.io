11.1 ROS2 통신 미들웨어의 진화와 Zenoh의 필요성

ROS2(Robot Operating System 2)의 실시간 통신 근간이었던 DDS(Data Distribution Service)는 견고하지만 너무도 크고 무거웠다. 이 11장은 방구석에서 굴러가는 대학생의 장난감 로봇이 아니라, 수십 대의 산업용 자율주행 로봇(AMR)이 5G와 스타링크(Starlink) 망을 오가며 통신해야 하는 상용 로보틱스 시장에서 왜 전 세계가 “Zenoh 로의 대이동” 을 결심했는지에 대한 딥 다이브(Deep-Dive) 아키텍처 분석이다.
