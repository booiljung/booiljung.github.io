18.8.4 모바일 및 에지 디바이스를 위한 경량화된 UI/UX 설계

관제탑은 8코어 i7 PC 지만, 현장에 있는 공장 직원은 덜떨어진 안드로이드 태블릿이나 구형 스마트폰으로 이 대시보드를 켜야 한다. PC 와 똑같은 데이터를 밀어 넣으면 브라우저 앱 자체가 강제 종료(Crash) 된다.

0.1 [Runbook] 타겟 디바이스 맞춤형(Adaptive) 다이어트

기기 스펙에 따라 렌더링의 질량 자체를 바꿔버려라.

1. 해상도 감지식 스트리밍 제어 (Progressive Degradation)
클라이언트 브라우저가 접속할 때 User-Agent 와 해상도가 모바일(Mobile) 라면, Zenoh 에서 쏘는 토픽부터 달라야 한다.

  • PC: subscribe("robot/video/4k")
  • 모바일: subscribe("robot/video/360p") 혹은 차트 내의 점의 개수를 제한하는 다운샘플링 쿼리로 강제 스위칭한다.

2. 에지용 CSS/SVG 오버헤드 박살 내기
모바일 브라우저의 적은 “그림자 폭탄(Box-shadow)” 과 “투명도 연산(Alpha-blending)” 이다.
모바일 기기에 띄운 관제 다이어그램에서는 box-shadow 와 backdrop-filter (Blur) 효과를 강제로 display: none 으로 끄거나 제거해야 하며, 토폴로지를 그릴 때 SVG 노드에 입혀놓은 그라데이션 필터들을 전부 원색(Solid pattern) 으로 갈아 끼워버려라. 이 단순한 CSS 제거만으로 모바일 배터리 광탈과 브라우저 버벅거림을 50% 이상 치료할 수 있다.
