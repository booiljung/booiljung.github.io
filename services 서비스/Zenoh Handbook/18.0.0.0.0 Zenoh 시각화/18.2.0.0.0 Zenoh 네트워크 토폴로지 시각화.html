18.2 Zenoh 네트워크 토폴로지 시각화

분산 시스템의 첫 번째 질문은 항상 “누가, 어디에, 누구와 묶여 있는가?” 이다.
Zenoh 는 스스로 라우팅 테이블(Routing Table) 을 그려가며 우주처럼 팽창하는 유기적인 프로토콜이다. 개발자가 칠판에 그려둔 정적인 아키텍처 다이어그램은 시스템이 켜지는 순간 이미 쓰레기가 된다.

이 장에서는 Zenoh Admin Space(@/router/**) 에 숨겨진 살아있는 라우팅 세포 배열을 뜯어내어, 라우터(Router), 피어(Peer), 클라이언트(Client) 의 실시간 노드 종속성을 살아 숨 쉬는 웹 그래프(D3.js / Cytoscape) 로 렌더링하는 토폴로지 추출 런북을 설파한다.

1. 네트워크 아키텍처 매핑: 라우터, 피어, 클라이언트 식별

그래프를 그리려면 화면에 찍을 놈이 동그라미(라우터) 인지, 네모(클라이언트) 인지, 세모(피어) 인지 노드의 신분증(Type) 을 발급해주어야 한다.

1.0.1 [인스펙션] 노드 신분 추출 전술

라우터의 Admin Space(17.2장) 에서 가져온 sessions 정보를 파싱하여 생물학적 분류를 수행한다.

1. 세션의 WHAT AM I 속성 파싱

// GET /@/router/local/sessions 의 응답 예시 일부
[
  { "id": "A1B2", "whatami": "Client", "locators": "tcp/10.0.0.5:7447" },
  { "id": "C3D4", "whatami": "Peer", "locators": "udp/10.0.0.8:7447" }
]

이 JSON 을 파싱하여 자바스크립트 객체의 type 속성에 우겨 넣는다. 라우터는 크고 굵은 원형 노드로 치환하고, 클라이언트는 라우터 밑에 매달린 작은 나뭇잎(Leaf) 모양으로 계층을 물리적으로 분리하는 디자인이 필요하다.

2. 식별 불가능성(Unidentifiable) 배제
간혹 Zenoh 버전이 낮거나 외부 브릿지(MQTT Bridge) 인 경우 whatami 속성이 모호할 때가 있다. 이런 경우 토픽의 패턴(*/bridge/**) 을 훔쳐보아서라도 프론트엔드 엔진에게 강제로 “얘는 외부 시스템이다” 라는 회색(Gray) 또는 파선(Dashed) 노드 스타일을 부여해야 한다.

2. 노드 간 물리적 및 논리적 연결 상태 시각화

노드를 그렸으면 이제 노드 사이에 선(Edge) 을 그을 차례다. 선이 없으면 망(Network) 이 아니다.

2.0.1 [Runbook] 와이어(Wire) 텐션 렌더링 런북

1. 물리적 연결망 (Physical Topology)
누가 누구에게 소켓(TCP/UDP) 을 꽂았는가에 대한 선이다.

  • 선(Edge)의 굵기: 세션 간의 대역폭 지표(rx_bytes + tx_bytes) 를 비례식으로 매핑한다. 굵은 선은 트래픽이 몰리는 아우토반이고, 실처럼 가는 선은 가끔 핑(Ping) 만 날리는 유휴 연결이다.
  • 선(Edge)의 색깔: 패킷 드롭(dropped messages) 이 잡히는 순간 선의 색깔을 파란색에서 시뻘건 붉은색(Red) 으로 즉각 토글(Toggle) 시켜 관제탑 전체에 피가 터진듯한 시각적 알람(Visual Alerting) 을 줘야 한다.

2. 논리적 구독망 (Logical Pub/Sub Graph)
선으로 이어져 있다고 다 대화하는 건 아니다.
클라이언트 A 가 라우터를 거쳐 클라이언트 B 에게 “비디오” 를 보낸다고 치자.
물리 역학 다이어그램 (A-Router-B) 위에, “Robot_Video” 라는 형광색 논리 실선(Data Flow) 을 오버레이(Overlay) 해주어야 한다. Admin Space 의 Routing Table (구독 매칭 맵) 을 긁어와서, 특정 토픽이 지금 어느 노드들을 관통하고 있는지 하이라이팅하는 기능이 있어야 진정한 라우팅 시각화다.

3. 동적 네트워크 변화(노드 참여 및 이탈)의 실시간 렌더링

Zenoh 망은 생물이다. 드론이 날아오르면 노드(Node) 가 하나 뿅 하고 나타나 피어(Peer) 에 붙고, 와이파이가 끊어지면 사라진다. 시각화 화면이 이 변화를 10초 늦게 보여주면 이미 그 관제 화면은 죽은 데이터다.

3.0.1 [인스펙션] 델타(Delta) 렌더링 전술

화면을 지웠다(Clear) 다시 그리는(Redraw) 짓은 브라우저 CPU 를 찢어버린다.

1. 이벤트 드리븐(Event-Driven) 노드 갱신
REST API 로 주기적인 폴링(Polling) 을 하는 대신, Zenoh 웹소켓 플러그인에 subscribe 를 걸어 “새로운 세션이 붙었다(Join)” 또는 “세션이 터졌다(Leave)” 라는 델타 이벤트(Delta Event) 메시지만 받아야 한다.

2. 브라우저 생동감 부여 (Animation)

  • 참여 (Join): 드론이 켜지면 노드가 투명도(Opacity) 0에서 1로 페이드인(Fade-in) 되면서 주변 라우터의 인력(Gravity) 에 의해 화면 밖에서 빨려 들어오듯이 렌더링해야 관제사가 새로운 기기의 출현을 즉각 눈치챈다.
  • 이탈 (Leave): 드론이 터졌다고 노드를 화면에서 즉시 빡! 하고 지워버리면 인지 부조화가 온다. 노드를 붉은색 해골(또는 회색) 로 바꾼 뒤, 10초에 걸쳐 서서히 페이드아웃(Fade-out) 시키며 “방금 이곳에서 누군가 죽었다“는 여운(Shadow) 을 남겨 시각적 인지력을 보존하라.

4. 토폴로지 시각화를 위한 오픈소스 그래프 도구 연동 예 (D3.js, Cytoscape.js)

모두가 처음부터 WebGL 코드를 짤 수는 없다. 거인들의 어깨, D3.js 와 Cytoscape.js 의 물리 엔진을 훔쳐 써라.

4.0.1 [Runbook] 엔진 선택의 십자군 전술

당신의 네트워크 사이즈에 맞춰 엔진을 택해야 브라우저가 안 뻗는다.

1. 100 노드 미만: D3.js (예술적 자유도)

  • 장점: 커스터마이징의 끝판왕. 노드 모양을 로봇(SVG) 아이콘으로 만들고, 선(Edge) 위를 패킷 알갱이들이 날아가는 애니메이션까지 땀구멍까지 제어할 수 있다.
  • 단점: 100개가 넘어가는 순간 중력(Force-directed) 알고리즘이 꼬이면서 화면의 프레임이 10fps 이하로 곤두박질친다.

2. 1,000 노드 이상: Cytoscape.js (엔터프라이즈의 무식함)

  • 장점: 바이오(Bio) 유전자 맵핑, 엔터프라이즈 망 관제에 특화된 학술적/엔터프라이즈 엔진이다. 수백 개의 노드가 얽혀도 가장 안정적으로 가지런히 트리/그물망 구조를 펴주는 막강한 레이아웃(Layout) 연산 알고리즘을 보유하고 있다.
  • Cola 나 fCose 레이아웃 플러그인을 끼워서 Zenoh 의 JSON 맵을 때려 부으면, 자기가 알아서 라우터들은 한가운데 뭉쳐주고(Core), 클라이언트들은 바깥 테두리(Edge) 로 튕겨내 주는 기가 막힌 자동 정렬을 수행한다.

5. 멀티 리전 및 멀티 클라우드 환경의 글로벌 네트워크 관제

노드가 1만 개, 공장이 10개다. 이걸 노트북 화면 하나에 다 그리면 그냥 새까만 공(Blackball) 만 보인다. 지구 스케일의 시각화 아키텍처는 고도(Altitude) 와 추상화(Abstraction) 에 목숨을 걸어야 한다.

5.0.1 [인스펙션] 줌-인/줌-아웃(ZUI) 매크로 전술

비행기 고도에 따라 지도에 보이는 글씨가 달라져야 한다.

1. 줌-아웃 상태 (대륙 레벨 뷰)
관제탑 스크린에는 미국 리전(AWS), 한국 공장(On-Premise), 유럽 리전 세 개의 거대한 노드 덩어리(Cluster Entity) 만 원형으로 표시한다.
이 거대한 한국 공장 원형 노드 주변에 붉은색 오라(Aura) 가 피어나면, “한국 공장 내부의 라우터 누군가가 터졌다” 는 것을 집계(Aggregation) 하여 최상위 뷰에 뿌려준다.

2. 줌-인 타겟팅 (세포 레벨 뷰)
마우스 휠을 돌려 한국 공장 노드를 줌인(Drill-down) 하는 순간, 커다란 원이 펑! 하고 깨지며 그 안에 숨겨져 있던 1,000 대의 로봇과 브릿지, 라우터들의 디테일한 D3.js 토폴로지가 뼈대를 드러내며 펼쳐져야 한다.
이러한 “계층형 시각적 압축(Semantic Zooming)” 기술이 없으면 어떠한 대시보드도 대규모 클라우드-에지 Zenoh 망을 감당하다가 크롬 브라우저 탭(OOM) 을 찢어버리며 침몰할 것이다.
