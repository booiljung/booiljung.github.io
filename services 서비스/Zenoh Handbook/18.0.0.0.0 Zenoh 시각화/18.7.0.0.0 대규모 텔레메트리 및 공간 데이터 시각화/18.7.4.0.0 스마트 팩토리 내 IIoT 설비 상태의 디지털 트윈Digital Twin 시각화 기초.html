18.7.4 스마트 팩토리 내 IIoT 설비 상태의 디지털 트윈(Digital Twin) 시각화 기초

도요타나 테슬라의 기가 팩토리 관제탑 뒷면에는 팩토리 라인 전체를 본뜬 3D 비디오 게임 같은 화면이 돌아간다. 센서값(숫자) 과 3D 모델(물체) 의 완벽한 동기화, 이것이 디지털 트윈이다.

0.1 [Runbook] 공장 메타버스(Metaverse) 바인딩 런북

Three.js 화면 안에 얹어진 로봇팔 3D 모델(GLTF) 관절에 Zenoh 의 데이터 혈관을 꽂아 넣는다.

1. 메타 데이터-3D 뼈대(Bone) 바인딩
웹 디자이너가 공장 설비를 3D 로 렌더링해 뒀다 치자.
컨베이어 벨트에 있는 로봇 팔 3D 객체의 이름을 코드 상에서 model.getObjectByName('Joint_A') 로 찾는다.
그리고 Zenoh 구독 콜백 함수 안에 이 뼈대를 통째로 물려버린다.

// Zenoh에서 관절 각도가 넘어올 때마다 브라우저 속 3D 로봇팔도 꺾인다.
zenohSession.subscribe('factory/robot1/joint_a', (val) => {
    const angle = parseFloat(val.toString());
    robotArmBone.rotation.z = THREE.MathUtils.degToRad(angle);
});

2. IIoT 색상(Color) 지시등 투영
공장 용광로의 온도(Temperature) 센서 값이 1,000도를 돌파했다면?
단순히 숫자만 띄우지 마라. 쓰리디(3D) 모델 내 용광로 매트리얼(Material) 의 발광(Emissive) 강도를 센서 온도에 비례하게 팍팍 올려주어, 용광로 3D 모델 자체가 화면에서 붉은 용암처럼 스스로 불타오르게 시각적 압박감을 주어야 진짜 디지털 트윈이다.
