18.7 대규모 텔레메트리 및 공간 데이터 시각화

배달 로봇 100대가 강남역 사거리를 횡단하고, 농업용 드론 50대가 거대한 농장을 편대 비행한다.
이 거대한 야전 시스템을 관제하려면, 앞 장의 노드 차원(Topology) 을 넘어서서, 우리가 살고 있는 “물리적인 지구의 좌표계(GPS, GIS)” 위에 Zenoh 의 데이터를 핀셋처럼 꽂아 넣어야만 한다.

이 장에서는 Zenoh 프로토콜을 공간 정보 시스템(GIS) 에 직접 결합하여, 단순한 점의 이동을 넘어선 스마트 팩토리와 디지털 트윈, 드론 군집 비행의 지리적 동태(Geospatial Dynamics) 를 스크린에 융합하는 런북을 설파한다.
