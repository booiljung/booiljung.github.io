이 챕터는 지금까지 6장에서 배운 모든 C++ Zenoh 아키텍처—스마트 포인터 래핑, 멀티스레드 락 프리 큐, 비동기 송수신, 그리고 프로파일링—를 단 하나의 무결점 프로젝트로 통합하는 최종 관문이다.
