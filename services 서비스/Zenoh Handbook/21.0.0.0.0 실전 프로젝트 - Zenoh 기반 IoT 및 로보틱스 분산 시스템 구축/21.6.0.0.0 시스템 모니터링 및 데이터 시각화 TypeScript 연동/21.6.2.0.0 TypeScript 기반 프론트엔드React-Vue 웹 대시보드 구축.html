21.6.2 TypeScript 기반 프론트엔드(React/Vue) 웹 대시보드 구축

관제 모니터에는 프론트엔드 프레임워크인 React(또는 Vue)를 사용한다.

[UI 컴포넌트 아키텍처]

  1. Map Component: 로봇의 현재 x, y 좌표를 찍어줄 2D 캔버스.
  2. Chart Component: 센서의 진동/온도 값을 보여줄 Chart.js 또는 Recharts 모듈.
  3. Control Panel: 관리자가 특정 로봇에게 긴급 정지(E-Stop)나 목적지 하달을 버튼 클릭으로 지시하는 패널.

[상태 관리 포인트]
프론트엔드 앱이 마운트(Mount) 될 때 WebSocket을 통해 Node.js 서버와 직결 채널을 열고, 초당 10번씩 들어오는 수십 개의 로봇 좌표 상태를 React의 Redux 나 Zustand 같은 전역 상태 저장소에 쉼 없이 밀어 넣는다 (Push 방식).
