21.6.3 센서 데이터의 실시간 시계열 그래프 시각화

AMR의 현재 배터리 소모량과 컨베이어 벨트의 진동 센서 데이터를 실시간 선 그래프로 렌더링한다.

[WebSocket 기반 실시간 스트리밍]
Node.js 백엔드가 Zenoh로부터 telemetry/zone_A/sensor/vib 를 수신할 때마다, 즉시 WebSocket 통로로 직렬화하여 뿌린다.

// React 프론트엔드 측 수신 코드 (Chart.js 연동 예시)
useEffect(() => {
  const ws = new WebSocket('ws://cloud-server:3000/stream');
  
  ws.onmessage = (event) => {
    const data = JSON.parse(event.data);
    if (data.topic.includes('vib')) {
      // 차트의 데이터 배열 끝에 새 값을 밀어넣고 랜더링 트리거
      setChartData(prev => [...prev.slice(-50), data.value]); 
    }
  };
  return () => ws.close();
}, []);

배열(Array)의 크기를 항상 최근 50개(또는 100개)로 제한(slice(-50)) 하는 윈도우잉(Windowing) 기법을 적용해야, 며칠 동안 켜진 브라우저의 메모리가 터지는 OOM 장애를 예방할 수 있다.
