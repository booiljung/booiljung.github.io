21.6.1 Zenoh 시스템 모니터링 플러그인 활성화 및 트래픽 메트릭 수집

네트워크 자체가 얼마나 건강한지 파악하는 것이 우선이다. Zenoh는 라우터 헬스 체크를 위한 내장 REST API를 제공한다.

[Admin Space 활용]
클라우드 라우터의 설정에서 rest 플러그인을 활성화("http_port": 8000) 한 뒤, 브라우저에서 http://localhost:8000/@/router/local/** 로 접속해 보라.
현재 활성화된 모든 세션 식별자(Session ID), 연결된 엣지 라우터의 개수, 초당 처리 바이트 수가 JSON 형태로 표출된다.

이 데이터를 Prometheus가 주기적으로 긁어갈 수 있도록 Node.js 서버에 가벼운 “Metrics Exporter” 엔드포인트를 노출해 두면 인프라 계층의 대시보드(Grafana) 준비가 마무리된다.
