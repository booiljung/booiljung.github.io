모든 네트워크 장애는 필연적으로 기형적인 증상(Symptom)을 동반한다. 장애 발생 시 패닉 없이 매뉴얼대로 대응할 수 있도록, 가장 빈번하게 마주치는 Zenoh 연결 문제와 통신 에러를 증상별 테이블 구조로 정리하였다. 증상과 맞닥뜨렸을 때 순차적 검증 액션을 수행하라.
