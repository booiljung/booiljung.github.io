15.4.1 Zenoh 키 표현식(Key Expression) 기반의 정밀한 권한 제어

기존의 방화벽이 IP 주소를 필터링했다면, 데이터 중심(Data-Centric) 네트워크인 Zenoh 에서는 “그 이름표(URL) 에는 접근할 수 없다” 고 명시한다.

0.1 [인스펙션] 문자열 검열주의 기반 ACL

1. 와일드카드 블랙리스트 전술
가장 위험한 패턴은 ** (모든 데이터) 와일드카드 남용이다.
라우터 설정에서 특정 유저(권한 그룹) 에게 하위 자식 노드의 데이터를 뽑아내는 범위 쿼리 자체를 부분적으로 절단시킬 수 있다.

  • 허용: factory/sensor/temp
  • 금지: factory/sensor/**
    이렇게 설정하면 악의적인 유저가 센서 데이터를 일일이 뽑아가는 것(temp, humidity) 은 통과시키되, 단 한 번의 쿼리로 전체 망을 싹쓸이하는 스니핑(Sniffing) 시도는 라우팅 자체에서 블로킹된다.

2. 논리적 망분리와 교차점 차단
admin 계정은 모든 것을 볼 수 있지만, robot_1 계정은 오직 "robot/1/**" 영역에만 데이터를 쓸 수 있게 묶어라.
이를 통해 로봇 하드웨어가 탈취당하더라도, 범죄자가 다른 로봇(robot_2) 의 제어 채널로 스파이 패킷을 전송하는(Lateral Movement) 시도를 라우터 단에서 완벽하게 격리해 낸다.
