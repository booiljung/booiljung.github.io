13.5 다국어 노드(Node) 개발 및 구현 가이드

하나의 거대한 파이프라인 안에서 카메라는 C 로 제어하고, 추론은 Python 으로 돌리며, 결과 전송은 Rust 로 한다?
이런 미친(Polyglot) 짓이 Zenoh-Flow 에서는 가장 권장되는 아키텍처 패턴이다.

개발자는 오직 각 노드의 언어별 SDK 가 요구하는 Trait(Rust) 나 Class(Python) 의 껍데기만 씌워주면 그만이다. 서로 다른 프로그래밍 언어가 메모리 풀을 공유하고 직렬화를 주고받는 기적의 코딩 런북을 언어별로 전개한다.
