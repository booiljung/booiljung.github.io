3.3 Rust에서 Zenoh 개발을 위한 환경 구축

Zenoh 엔진의 밑바닥 100%는 Rust 언어로 직조되어 있다. 따라서 다른 언어(Python, JS)의 클라이언트를 쓸 때 거르쳐야 하는 FFI(Foreign Function Interface) 브릿지 계층의 오버헤드가 Rust 생태계 내에서는 ’제로(Zero)’다.
“네트워크 대역폭의 한계뿐 아니라, 에지 장비의 CPU 사이클 하나까지 극한으로 갈취하겠다“는 결심이 섰다면, 당신의 팀은 메인 어플리케이션을 Rust로 작성해야만 한다.

본 절에서는 메모리 안전성과 비동기(Async) 런타임의 정점인 Tokio 생태계 위에 Zenoh 원시 크레이트(Crate)를 완벽하게 결합시키는 빌드 환경 파이프라인(Build Pipeline)을 구축하는 과정을 다룬다.
