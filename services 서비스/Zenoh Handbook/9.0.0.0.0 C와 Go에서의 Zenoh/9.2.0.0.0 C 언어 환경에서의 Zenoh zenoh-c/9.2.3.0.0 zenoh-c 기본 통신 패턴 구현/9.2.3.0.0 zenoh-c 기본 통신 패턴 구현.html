9.2.3 zenoh-c 기본 통신 패턴 구현

자, 이제 C 언어로 작성된 펌웨어 안에서 다른 대륙에 있는 중앙 서버와 통신 파이프를 열어보자.
zenoh-c 의 코딩 스타일은 일반적인 소켓(Socket) 프로그래밍과 완전히 궤를 달리한다. AF_INET 이나 SOCK_STREAM 같은 고전적인 옵션을 만질 필요가 없다.

대신, “설정(Config)을 깎고”, “세션(Session)을 소환하고”, “객체를 빌려와(Loan)” 함수에 꽂아 넣는 Rust 특유의 선언적 통신 방식을 C의 포인터 문법으로 흉내 내는 과정이다. 이 챕터는 가장 기본적인 Pub/Sub과 Query/Reply 를 C 환경에서 어떻게 버그 없이 구현하는지 보여주는 타격 지점(Strike point)이다.
