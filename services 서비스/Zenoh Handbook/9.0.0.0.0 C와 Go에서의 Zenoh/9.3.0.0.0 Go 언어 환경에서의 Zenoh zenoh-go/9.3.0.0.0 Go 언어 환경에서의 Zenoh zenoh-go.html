9.3 Go 언어 환경에서의 Zenoh (zenoh-go)

C 언어가 “포인터의 칼춤” 이라면, Go 언어는 “동시성의 공장” 이다.
수만 대의 커넥티드 카(Connected Car)들이 0.1초마다 쏘아대는 위치 데이터를 받아줄 클라우드 백엔드를 구축할 때, C 언어로 소켓 1만 개를 열고 Pthread와 Epoll 코드를 손으로 짜는 것은 지옥으로 가는 지름길이다.

하지만 Go 언어를 잡는 순간, go handleRobot(session) 단 한 줄만으로 수만 개의 마이크로 스레드(Goroutine)가 생성되며 각기의 로봇들과 전담 연결을 맺는다.
이 챕터에서는 Zenoh의 미친 멀티캐스트 성능을 Go의 Goroutine 과 멀티코어 채널(Chan)에 찰떡같이 결속시켜, 초당 수십만 개의 데이터를 파싱하고 DB에 꽂아 넣는 클라우드 네이티브(Cloud Native) 백엔드 런북을 전개한다.
수십 기가의 램을 쥐고 있는 서버 노드에서, zenoh-go 의 폭발적인 대역폭을 200% 활용하라.
