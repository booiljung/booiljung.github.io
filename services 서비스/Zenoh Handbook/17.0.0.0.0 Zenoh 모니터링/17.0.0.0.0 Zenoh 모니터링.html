17.0 Zenoh 모니터링

눈먼 조종사는 비행기를 몰 수 없다. 수백 대의 로봇과 센서가 클라우드 라우터로 데이터를 쏟아붓는 분산 시스템 환경에서, 모니터링 시스템의 부재는 곧 대형 사고(Disaster) 를 의미한다.

이 장에서는 “서버가 죽었나 살았나?” 를 넘어서, Zenoh 네트워크의 맥박수(Metrics), 패킷들의 궤적(Tracing), 라우터의 비명(Logging) 을 3차원적으로 수집, 분석, 통제하는 데이터 인프라스트럭처의 완벽한 가시성(Observability) 구축 런북을 전개한다.
