17.2 Zenoh 내장 도구를 활용한 기본 상태 점검

복잡한 모니터링 시스템(Prometheus, Elasticsearch) 을 구축하기 전, 우리가 가진 가장 빠르고 날카로운 메스는 Zenoh 가 자체적으로 내장하고 있는 관리망(Admin Space) 과 CLI 도구다.

이 도구들은 추가적인 설치 없이, 터미널 명령 한 줄과 REST API 만으로 시스템의 핏줄(Topology) 과 맥박(Session) 을 실시간으로 투시할 수 있게 해 준다. 외부의 무거운 감시 장비를 들이기 전, 네이티브 도구를 이용해 Zenoh 의 뼈대를 살펴보는 런북을 전개한다.
