1.8 Zenoh 도입을 통한 시스템 아키텍처의 혁신

새로운 기술을 실무 인프라에 도입한다는 것은 언제나 거대한 도전과 모험을 수반한다. 그럼에도 불구하고 수많은 글로벌 대기업과 차세대 로보틱스 플레이어들이 기존 시스템을 들어내는 출혈을 감수하면서까지 앞다투어 시스템의 뿌리에 Zenoh를 이식하고 있다. 이는 Zenoh가 구시대 패러다임이 남긴 통신의 병목과 구조적 기형성(Deformity)을 철저히 치료하고 아키텍처를 원초적으로 재정의해 주는 독보적인 잠재력을 보여주었기 때문이다.

앞서 짚어보았듯 Zenoh는 단순히 속도가 조금 빠른 프로토콜이 결코 아니다. 이기종의 분열된 기존 시스템 통신망(DDS, MQTT, REST, 데이터베이스) 위로 흩어진 통제권을 단일한 플러그인 프레임워크 아래로 수렴시키는 웅장한 아키텍처 혁명의 조력자다.

본 장의 대미를 장식할 이 절에서는, 시스템 관점으로 이 차세대 프로토콜을 인프라망에 접목했을 때 개발자와 아키텍트가 얻게 될 극적인 아키텍처 복잡성(Complexity) 해소 방안에 대해 평가한다. 또한 그로 인해 창출되는 총소유비용(TCO) 절감이라는 기업의 궁극적 비즈니스 가치를 조명하고, 앞으로 이어질 Zenoh 마스터링 로드맵(Handbook)을 통한 독자 여러분의 여정을 제시하겠다.
