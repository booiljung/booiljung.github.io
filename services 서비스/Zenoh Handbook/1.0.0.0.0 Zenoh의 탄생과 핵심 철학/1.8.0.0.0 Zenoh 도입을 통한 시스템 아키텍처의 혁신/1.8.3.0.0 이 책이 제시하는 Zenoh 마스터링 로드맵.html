1.8.3 이 책이 제시하는 Zenoh 마스터링 로드맵

거대한 철학과 혁신의 물결 속에서, 우리는 왜 Zenoh가 단순한 툴이 아닌 시대의 미들웨어 패러다임인지 심도 있게 확인했다. 그렇다면 이 압도적인 강력함을 기술자로서 어떻게 다루고 쟁취해낼 것인가?

이 “Zenoh Handbook“은 처음 마주하는 실무 개발자부터 엔터프라이즈 레벨의 아키텍트까지, Zenoh라는 거대한 코어를 능숙하게 조율하고 분해하며 마스터(Mastering)하기 위한 가장 견고한 튜토리얼 설계도이자, 실습의 교과서가 되어줄 것이다.

본 서적의 로드맵은 다음과 같이 펼쳐진다:
우선 Chapter 2에서 Zenoh만의 와이어 프로토콜 구조와 네트워크 토폴로지 교본을 통해 시스템 밑바닥 기저의 구조를 세밀히 분석하고 해부한다. 이어지는 Chapter 3부터 Chapter 4까지는 시스템에 엔진을 시동 거는 설치(Installation) 및 핵심 라우터 구성의 첫 실전 세팅법을 다룬다.

중반부로 접어들어 Chapter 5부터 Chapter 9까지, 챕터별로 시스템의 주력 엔진들인 Rust, C++, Python, TypeScript, Go 그리고 C 언어를 활용하여 여러분의 비즈니스 코드를 Zenoh 세계관에 다이렉트로 결합시키는 클라이언트 API 바인딩(Binding) 과정이 실무 코드와 함께 해부된다. 나아가 마이크로컨트롤러(MCU)를 위한 Zenoh-pico, 그리고 거대한 ROS2 로보틱스 생태계를 Zenoh와 병합시키는 실증적 아키텍처 수립 과정을 거치게 될 것이다.

마지막 후반부에서는 이 막대한 분산 파이프라인망을 시각화(Visualization)하고, 성능을 튜닝(Performance Tuning)하며 모니터링 체계를 확립해 실존하는 프로덕션(Production) 아키텍처로써 우뚝 서게 만드는 에지 클라우드 연동까지의 시스템 아키텍트 여정을 담았다.

책의 마지막 페이지를 넘길 즈음, 독자 여러분은 데이터를 다루는 패러다임에서 전례 없는 깨달음과 통제력을 쥐고 역사상 가장 빠른 분산 인프라망을 구축할 만반의 준비를 마칠 것이다. 이제, 그 혁명의 첫 퍼즐인 아키텍처 프로토콜의 중심으로 발걸음을 내디뎌보자.
