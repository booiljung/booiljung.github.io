1.7.2 글로벌 커뮤니티와 주요 기업들의 기여 및 도입 사례

Zenoh의 오픈소스라는 비옥한 토양 위에서, 이는 단순한 기술 데모 수준을 벗어나 글로벌 최고봉 엔지니어들과 다국적 거대 기업들의 핵심 통신 엔진 설계표준으로 채택되기 이르렀다. 로보틱스와 오토모티브(Automotive), 그리고 드론과 우주항공 분야의 굵직한 조직들이 앞다투어 이 우산으로 집결하였다.

도입을 결정하는 대표적인 레퍼런스는 자율주행(Autonomous Vehicle)과 커넥티드카 텔레매틱스(Telematics) 데이터 시스템이다. 오늘날 거대한 센서 대역폭(초당 기가바이트씩 쏟아지는 Lidar, 카메라 스트림)을 끊김 없이 인프라에 올려 태울 역량을 가진 미들웨어는 전무하기에, 다루기 무명하던 수십 개 브랜드사들의 자율주행 엔진이 본 코어를 테스트하고 도입하고 있다.

특히, 차세대 로봇 운영 체제라 불리는 ROS2(Robot Operating System 2)의 실시간 통신 엔진인 DDS가 인터넷 망의 대규모 네트워크 및 Wi-Fi에 쥐약인 치명적 한계를 드러내자, 이를 해결할 ROS2의 표준 공식 미들웨어 대안(Alternative Middleware)으로서 rmw_zenoh가 당당히 공식 생태계 탑티어로 자리 잡았다.

결과적으로, Zenoh는 거대 자동차 회사의 공장 물류 통제 시스템망, 대학 실험실의 학계용 다수 드론 교차 비행 군집 테스트망, 그리고 초저지연성을 무기로 화성 같은 곳에 파견된 우주 항공 원격 제어망의 프로토타이핑(Prototyping)에 이르기까지 실증적 신뢰가 필요한 모든 인더스트리를 향해 뻗어가고 있다.
