14.3.2 FS(File System) 백엔드: 로컬 파일 시스템과의 데이터 연동

디스크에 남겨야 하지만, 데이터베이스를 설치할 권한이 없거나 오버스펙일 때 쓴다.

0.1 [인스펙션] OS 파일 트리를 DB로 승격시키기

운영체제의 mkdir, echo, 파일 쓰기 시스템 콜 자체를 훌륭한 Key-Value 저장소로 우려먹는 런북이다.

1. 파일시스템 백엔드 마운트

storages: {
  local_robot_storage: {
    key_expr: "robot/config/**",
    backend: "fs",
    // 컴퓨터의 실제 하드디스크 폴더 경로를 지정한다!
    dir: "/var/lib/zenoh/my_storage" 
  }
}

2. 자동화된 파일 로깅
이 마운트가 활성화된 이후부터, 누군가가 z_put "robot/config/speed" "100" 을 때리면?

  • FS 플러그인은 리눅스 명령어로 번역한다: mkdir -p /var/lib/zenoh/my_storage/robot/config && echo "100" > speed
  • 파일 브라우저로 열어보면 데이터가 진짜 텍스트 파일과 폴더들 무리로 예쁘게 정리되어 저장되는 미친 직관성을 자랑한다.
