포인트클라우드 데이터의 시각화는 단순히 3차원 공간에서 포인트들을 표시하는 것 이상으로, 데이터의 이해를 돕고 중요한 정보를 강조하기 위해 다양한 보조 요소를 추가하는 것이 중요하다. 이 절에서는 PCL(Point Cloud Library)을 활용하여 포인트클라우드 시각화에 텍스트와 마커를 추가하는 방법에 대해 상세히 다룬다.

텍스트 추가¶

포인트클라우드 시각화에서 텍스트는 데이터의 특정 부분을 설명하거나, 측정값을 표시하는 등 다양한 용도로 사용된다. PCL의 시각화 모듈은 텍스트를 추가하기 위한 여러 기능을 제공한다.

텍스트 추가 기본 방법¶

PCL에서 텍스트를 추가하기 위해서는 pcl::visualization::PCLVisualizer 클래스의 addText 함수를 사용한다. 이 함수는 텍스트의 내용을 지정할 수 있으며, 텍스트의 위치와 크기 등을 조정할 수 있다.

#include <pcl/visualization/pcl_visualizer.h>

int main()
{
    pcl::visualization::PCLVisualizer::Ptr viewer(new pcl::visualization::PCLVisualizer("3D Viewer"));
    viewer->setBackgroundColor(0, 0, 0);

    // 포인트클라우드 추가
    pcl::PointCloud<pcl::PointXYZ>::Ptr cloud(new pcl::PointCloud<pcl::PointXYZ>());
    // ... 포인트클라우드 데이터 로드 ...

    viewer->addPointCloud<pcl::PointXYZ>(cloud, "sample cloud");

    // 텍스트 추가
    viewer->addText("Sample Point Cloud", 10, 10, "text1", 0);

    while (!viewer->wasStopped())
    {
        viewer->spinOnce(100);
    }

    return 0;
}

위 예제에서는 addText 함수를 사용하여 시각화 창의 좌측 상단에 "Sample Point Cloud"라는 텍스트를 추가하였다. addText 함수의 인자는 다음과 같다:

  • 텍스트 내용: "Sample Point Cloud"
  • x 좌표: 10
  • y 좌표: 10
  • 텍스트 ID: "text1"
  • 폰트 ID: 0

텍스트 위치와 스타일 조정¶

텍스트의 위치는 시각화 창 내의 픽셀 단위로 지정할 수 있으며, 폰트 크기와 색상도 조정이 가능한다. 이를 통해 텍스트가 시각적으로 더 명확하게 나타나도록 할 수 있다.

// 텍스트 추가 with 색상과 크기
viewer->addText("Height: 1.75m", 10, 30, "height_text", 0);
viewer->setShapeRenderingProperties(pcl::visualization::PCL_VISUALIZER_TEXT_COLOR, 1.0, 0.0, 0.0, "height_text");
viewer->setShapeRenderingProperties(pcl::visualization::PCL_VISUALIZER_FONT_SIZE, 20, "height_text");

위 코드에서는 "Height: 1.75m"라는 텍스트를 추가하고, 텍스트의 색상을 빨간색으로, 폰트 크기를 20으로 설정하였다.

마커 추가¶

마커는 포인트클라우드 내의 특정 지점을 강조하거나, 방향을 표시하는 등 다양한 용도로 사용된다. PCL은 다양한 형태의 마커를 지원하여 시각화의 효과를 극대화할 수 있다.

기본 마커 추가¶

PCL에서 제공하는 여러 마커 중 하나를 사용하여 포인트클라우드에 마커를 추가할 수 있다. 대표적인 마커로는 구, 화살표, 선 등이 있다.

// 구 마커 추가
pcl::PointXYZ marker_point(1.0, 2.0, 3.0);
viewer->addSphere(marker_point, 0.1, 1.0, 0.0, 0.0, "sphere1");

// 화살표 마커 추가
pcl::PointXYZ arrow_start(0.0, 0.0, 0.0);
pcl::PointXYZ arrow_end(1.0, 1.0, 1.0);
viewer->addArrow(arrow_end, arrow_start, 0.0, 1.0, 0.0, false, "arrow1");

위 예제에서는 빨간색 구 마커와 녹색 화살표 마커를 추가하였다. addSphere와 addArrow 함수는 각각 구와 화살표를 시각화 창에 추가하는 역할을 한다.

마커의 속성 설정¶

마커의 크기, 색상, 투명도 등 다양한 속성을 설정하여 시각적 효과를 조정할 수 있다.

// 마커 색상 및 투명도 설정
viewer->setShapeRenderingProperties(pcl::visualization::PCL_VISUALIZER_COLOR, 0.0, 0.0, 1.0, "sphere1");
viewer->setShapeRenderingProperties(pcl::visualization::PCL_VISUALIZER_OPACITY, 0.5, "sphere1");

// 화살표 두께 설정
viewer->setShapeRenderingProperties(pcl::visualization::PCL_VISUALIZER_LINE_WIDTH, 3, "arrow1");

위 코드에서는 구 마커의 색상을 파란색으로, 투명도를 0.5로 설정하였으며, 화살표의 선 두께를 3으로 설정하였다.

PCL 시각화 API 사용¶

PCL의 시각화 API는 텍스트와 마커를 추가하는 다양한 기능을 제공한다. 이를 효과적으로 활용하기 위해 주요 함수와 그 사용법을 이해하는 것이 중요하다.

텍스트 관련 함수¶

  • addText: 텍스트를 시각화 창에 추가한다.
  • setShapeRenderingProperties: 텍스트의 색상, 폰트 크기 등 속성을 설정한다.
  • removeShape: 특정 텍스트를 제거한다.

마커 관련 함수¶

  • addSphere: 구 형태의 마커를 추가한다.
  • addArrow: 화살표 형태의 마커를 추가한다.
  • addLine: 선 형태의 마커를 추가한다.
  • addPolygon: 다각형 형태의 마커를 추가한다.
  • setShapeRenderingProperties: 마커의 색상, 크기, 투명도 등을 설정한다.
  • removeShape: 특정 마커를 제거한다.

마커의 식별자 관리¶

각 마커와 텍스트는 고유한 식별자(ID)를 가져야 한다. 이를 통해 특정 마커나 텍스트를 수정하거나 제거할 수 있다.

// 마커 추가 시 고유 ID 지정
viewer->addSphere(marker_point, 0.1, 0.0, 1.0, 0.0, "unique_sphere_id");

// 특정 마커 제거
viewer->removeShape("unique_sphere_id");

텍스트 및 마커의 상호작용¶

텍스트와 마커는 단독으로 사용되기도 하지만, 상호보완적으로 사용되어 데이터의 해석을 용이하게 한다. 예를 들어, 특정 포인트에 마커를 추가하고, 그 포인트의 정보를 텍스트로 표시함으로써 데이터의 특정 부분을 강조할 수 있다.

// 특정 포인트에 마커와 텍스트 추가
pcl::PointXYZ target_point(2.0, 3.0, 4.0);
viewer->addSphere(target_point, 0.2, 1.0, 1.0, 0.0, "target_sphere");
viewer->addText("Target Point", 15, 15, "target_text", 0);
viewer->setShapeRenderingProperties(pcl::visualization::PCL_VISUALIZER_TEXT_COLOR, 1.0, 1.0, 1.0, "target_text");
viewer->setShapeRenderingProperties(pcl::visualization::PCL_VISUALIZER_FONT_SIZE, 18, "target_text");

위 코드에서는 특정 포인트에 노란색 구 마커를 추가하고, 시각화 창의 특정 위치에 "Target Point"라는 텍스트를 추가하여 해당 포인트의 정보를 명확하게 표시하고 있다.

동적 텍스트 업데이트¶

포인트클라우드 시각화에서 텍스트는 정적인 정보 제공뿐만 아니라, 실시간으로 변하는 데이터를 표시하는 데에도 유용하게 사용될 수 있다. 예를 들어, 센서 데이터의 실시간 변화를 모니터링하거나, 포인트클라우드 처리 과정에서의 진행 상황을 표시할 때 동적 텍스트 업데이트가 필요하다.

// 동적 텍스트 업데이트 예제
#include <pcl/visualization/pcl_visualizer.h>
#include <thread>
#include <chrono>
#include <sstream>

int main()
{
    pcl::visualization::PCLVisualizer::Ptr viewer(new pcl::visualization::PCLVisualizer("Dynamic Text Example"));
    viewer->setBackgroundColor(0, 0, 0);

    // 포인트클라우드 추가
    pcl::PointCloud<pcl::PointXYZ>::Ptr cloud(new pcl::PointCloud<pcl::PointXYZ>());
    // ... 포인트클라우드 데이터 로드 ...

    viewer->addPointCloud<pcl::PointXYZ>(cloud, "sample cloud");

    // 초기 텍스트 추가
    viewer->addText("Frame: 0", 10, 10, "frame_text", 0);

    int frame = 0;
    while (!viewer->wasStopped())
    {
        // 텍스트 업데이트
        std::stringstream ss;
        ss << "Frame: " << frame++;
        viewer->updateText(ss.str(), "frame_text");

        viewer->spinOnce(100);
        std::this_thread::sleep_for(std::chrono::milliseconds(100));
    }

    return 0;
}

위 예제에서는 updateText 함수를 사용하여 "Frame: " 뒤에 이어지는 숫자를 실시간으로 업데이트하고 있다. 이를 통해 포인트클라우드 처리의 진행 상황을 사용자에게 실시간으로 전달할 수 있다.

사용자 상호작용을 통한 텍스트 및 마커 제어¶

PCL 시각화 도구는 사용자와의 상호작용을 통해 텍스트와 마커를 동적으로 추가, 수정, 제거할 수 있는 기능을 제공한다. 이를 통해 사용자 맞춤형 시각화 환경을 구축할 수 있다.

// 사용자 상호작용을 통한 마커 추가 예제
#include <pcl/visualization/pcl_visualizer.h>
#include <pcl/io/pcd_io.h>

int main()
{
    pcl::visualization::PCLVisualizer::Ptr viewer(new pcl::visualization::PCLVisualizer("Interactive Marker Example"));
    viewer->setBackgroundColor(0.1, 0.1, 0.1);

    // 포인트클라우드 로드
    pcl::PointCloud<pcl::PointXYZ>::Ptr cloud(new pcl::PointCloud<pcl::PointXYZ>());
    pcl::io::loadPCDFile<pcl::PointXYZ>("sample.pcd", *cloud);
    viewer->addPointCloud<pcl::PointXYZ>(cloud, "sample cloud");

    // 콜백 함수 정의
    viewer->registerKeyboardCallback(
        [&viewer](const pcl::visualization::KeyboardEvent& event, void*) {
            if (event.keyDown() && event.getKeySym() == "m")
            {
                pcl::PointXYZ marker_point(0.0, 0.0, 0.0);
                viewer->getCameraParameters(marker_point.x, marker_point.y, marker_point.z);
                viewer->addSphere(marker_point, 0.2, 0.0, 1.0, 0.0, "interactive_sphere");
                viewer->addText("Interactive Marker Added", 10, 50, "interactive_text", 0);
            }
        }
    );

    while (!viewer->wasStopped())
    {
        viewer->spinOnce(100);
    }

    return 0;
}

이 예제에서는 사용자가 키보드에서 'm' 키를 누를 때마다 현재 카메라 위치에 구 마커를 추가하고, 해당 동작을 설명하는 텍스트를 시각화 창에 표시한다. registerKeyboardCallback 함수를 사용하여 키보드 이벤트를 등록하고, 특정 키 입력에 반응하여 텍스트와 마커를 동적으로 제어한다.

마커 그룹화 및 계층 구조 관리¶

복잡한 포인트클라우드 시각화에서는 여러 개의 마커를 그룹화하거나 계층 구조를 통해 관리하는 것이 효율적일 수 있다. PCL은 이러한 목적을 위해 마커 그룹화 기능을 제공한다.

// 마커 그룹화 예제
#include <pcl/visualization/pcl_visualizer.h>

int main()
{
    pcl::visualization::PCLVisualizer::Ptr viewer(new pcl::visualization::PCLVisualizer("Marker Grouping Example"));
    viewer->setBackgroundColor(0.05, 0.05, 0.05);

    // 포인트클라우드 추가
    pcl::PointCloud<pcl::PointXYZ>::Ptr cloud(new pcl::PointCloud<pcl::PointXYZ>());
    // ... 포인트클라우드 데이터 로드 ...
    viewer->addPointCloud<pcl::PointXYZ>(cloud, "sample cloud");

    // 마커 그룹 생성
    std::string group_id = "marker_group";

    // 여러 개의 마커 추가
    for (int i = 0; i < 5; ++i)
    {
        pcl::PointXYZ point(i, i, i);
        std::stringstream ss;
        ss << "sphere_" << i;
        viewer->addSphere(point, 0.1, 0.0, 0.0, 1.0, ss.str(), group_id);
    }

    // 그룹에 속한 모든 마커의 속성 변경
    viewer->setShapeRenderingProperties(pcl::visualization::PCL_VISUALIZER_OPACITY, 0.7, group_id);

    while (!viewer->wasStopped())
    {
        viewer->spinOnce(100);
    }

    return 0;
}

이 코드에서는 여러 개의 구 마커를 동일한 그룹 ID인 marker_group으로 추가하고, 그룹 전체의 투명도를 한 번에 변경하고 있다. 이를 통해 관련된 마커들을 효율적으로 관리하고, 일괄적으로 속성을 조정할 수 있다.

커스텀 마커 생성¶

PCL은 기본적으로 제공하는 마커 외에도 사용자가 직접 정의한 커스텀 마커를 추가할 수 있는 기능을 제공한다. 이를 통해 특정한 요구사항에 맞는 시각화 요소를 구현할 수 있다.

// 커스텀 마커 추가 예제
#include <pcl/visualization/pcl_visualizer.h>

int main()
{
    pcl::visualization::PCLVisualizer::Ptr viewer(new pcl::visualization::PCLVisualizer("Custom Marker Example"));
    viewer->setBackgroundColor(0.2, 0.2, 0.2);

    // 포인트클라우드 추가
    pcl::PointCloud<pcl::PointXYZ>::Ptr cloud(new pcl::PointCloud<pcl::PointXYZ>());
    // ... 포인트클라우드 데이터 로드 ...
    viewer->addPointCloud<pcl::PointXYZ>(cloud, "sample cloud");

    // 다각형 마커 정의
    pcl::PointXYZ p1(1.0, 1.0, 1.0);
    pcl::PointXYZ p2(2.0, 1.0, 1.0);
    pcl::PointXYZ p3(2.0, 2.0, 1.0);
    pcl::PointXYZ p4(1.0, 2.0, 1.0);

    pcl::PolygonMesh mesh;
    // ... 다각형 메시 생성 ...

    // 다각형 마커 추가
    viewer->addPolygon<pcl::PointXYZ>(p1, p2, p3, "custom_polygon");
    viewer->addPolygon<pcl::PointXYZ>(p1, p3, p4, "custom_polygon_2");

    // 마커 색상 및 스타일 설정
    viewer->setShapeRenderingProperties(pcl::visualization::PCL_VISUALIZER_COLOR, 0.0, 1.0, 0.0, "custom_polygon");
    viewer->setShapeRenderingProperties(pcl::visualization::PCL_VISUALIZER_COLOR, 0.0, 1.0, 0.0, "custom_polygon_2");
    viewer->setShapeRenderingProperties(pcl::visualization::PCL_VISUALIZER_OPACITY, 0.5, "custom_polygon");
    viewer->setShapeRenderingProperties(pcl::visualization::PCL_VISUALIZER_OPACITY, 0.5, "custom_polygon_2");

    while (!viewer->wasStopped())
    {
        viewer->spinOnce(100);
    }

    return 0;
}

이 예제에서는 두 개의 삼각형을 결합하여 사각형 다각형 마커를 생성하고, 이를 시각화 창에 추가하고 있다. addPolygon 함수를 사용하여 다각형을 정의하고, 색상과 투명도를 설정함으로써 커스텀 마커를 구현하였다.

마커 애니메이션¶

마커에 애니메이션을 적용하여 시각화의 동적인 요소를 추가할 수 있다. 예를 들어, 마커가 이동하거나 크기가 변하는 등의 효과를 통해 데이터의 변화를 직관적으로 표현할 수 있다.

// 마커 애니메이션 예제
#include <pcl/visualization/pcl_visualizer.h>
#include <thread>
#include <chrono>
#include <cmath>

int main()
{
    pcl::visualization::PCLVisualizer::Ptr viewer(new pcl::visualization::PCLVisualizer("Animated Marker Example"));
    viewer->setBackgroundColor(0.0, 0.0, 0.0);

    // 포인트클라우드 추가
    pcl::PointCloud<pcl::PointXYZ>::Ptr cloud(new pcl::PointCloud<pcl::PointXYZ>());
    // ... 포인트클라우드 데이터 로드 ...
    viewer->addPointCloud<pcl::PointXYZ>(cloud, "sample cloud");

    // 초기 마커 위치
    pcl::PointXYZ initial_point(0.0, 0.0, 0.0);
    viewer->addSphere(initial_point, 0.2, 1.0, 0.0, 0.0, "animated_sphere");

    float angle = 0.0;
    while (!viewer->wasStopped())
    {
        // 마커 위치 업데이트 (원형 궤도)
        pcl::PointXYZ new_position(std::cos(angle), std::sin(angle), 0.0);
        viewer->updateSphere(new_position, 0.2, 1.0, 0.0, 0.0, "animated_sphere");

        angle += 0.05;
        if (angle > 2 * M_PI)
            angle -= 2 * M_PI;

        viewer->spinOnce(100);
        std::this_thread::sleep_for(std::chrono::milliseconds(100));
    }

    return 0;
}

이 코드에서는 구 마커가 원형 궤도를 따라 움직이는 애니메이션을 구현하고 있다. updateSphere 함수를 사용하여 마커의 위치를 지속적으로 업데이트하며, 이를 통해 동적인 시각화 효과를 생성한다.

마커와 텍스트의 상호 연동¶

마커와 텍스트를 연동하여, 특정 마커에 대한 정보를 실시간으로 표시할 수 있다. 이를 통해 사용자는 시각화된 데이터의 각 요소에 대한 자세한 정보를 쉽게 파악할 수 있다.

// 마커와 텍스트 연동 예제
#include <pcl/visualization/pcl_visualizer.h>
#include <sstream>

int main()
{
    pcl::visualization::PCLVisualizer::Ptr viewer(new pcl::visualization::PCLVisualizer("Marker-Text Link Example"));
    viewer->setBackgroundColor(0.3, 0.3, 0.3);

    // 포인트클라우드 추가
    pcl::PointCloud<pcl::PointXYZ>::Ptr cloud(new pcl::PointCloud<pcl::PointXYZ>());
    // ... 포인트클라우드 데이터 로드 ...
    viewer->addPointCloud<pcl::PointXYZ>(cloud, "sample cloud");

    // 마커 추가
    pcl::PointXYZ marker_point(1.5, 1.5, 1.5);
    viewer->addSphere(marker_point, 0.15, 0.0, 0.0, 1.0, "linked_sphere");

    // 텍스트 추가
    std::stringstream ss;
    ss << "Coordinates: (" << marker_point.x << ", " << marker_point.y << ", " << marker_point.z << ")";
    viewer->addText(ss.str(), 10, 70, "linked_text", 0);

    // 마커와 텍스트의 색상 및 위치 설정
    viewer->setShapeRenderingProperties(pcl::visualization::PCL_VISUALIZER_TEXT_COLOR, 1.0, 1.0, 1.0, "linked_text");
    viewer->setShapeRenderingProperties(pcl::visualization::PCL_VISUALIZER_FONT_SIZE, 14, "linked_text");

    while (!viewer->wasStopped())
    {
        viewer->spinOnce(100);
    }

    return 0;
}

이 예제에서는 특정 마커에 대한 좌표 정보를 텍스트로 표시하고 있다. 마커의 위치를 기반으로 텍스트 내용을 동적으로 생성하여, 시각화 창에 추가하였다. 이를 통해 사용자는 마커의 정확한 위치 정보를 직관적으로 확인할 수 있다.

마커 필터링 및 선택¶

대량의 마커가 존재할 경우, 특정 조건에 맞는 마커만을 필터링하여 표시하거나 선택적으로 강조할 필요가 있다. PCL은 마커 필터링 및 선택을 위한 다양한 기능을 제공한다.

// 마커 필터링 예제
#include <pcl/visualization/pcl_visualizer.h>
#include <vector>

int main()
{
    pcl::visualization::PCLVisualizer::Ptr viewer(new pcl::visualization::PCLVisualizer("Marker Filtering Example"));
    viewer->setBackgroundColor(0.0, 0.0, 0.0);

    // 포인트클라우드 추가
    pcl::PointCloud<pcl::PointXYZ>::Ptr cloud(new pcl::PointCloud<pcl::PointXYZ>());
    // ... 포인트클라우드 데이터 로드 ...
    viewer->addPointCloud<pcl::PointXYZ>(cloud, "sample cloud");

    // 여러 마커 추가
    std::vector<pcl::PointXYZ> points = { {1,1,1}, {2,2,2}, {3,3,3}, {4,4,4}, {5,5,5} };
    for (size_t i = 0; i < points.size(); ++i)
    {
        std::stringstream ss;
        ss << "sphere_" << i;
        viewer->addSphere(points[i], 0.1, 0.0, 1.0, 0.0, ss.str());
    }

    // 특정 조건에 맞는 마커 필터링 (예: x > 3)
    for (size_t i = 0; i < points.size(); ++i)
    {
        if (points[i].x > 3.0)
        {
            std::stringstream ss;
            ss << "sphere_" << i;
            viewer->setShapeRenderingProperties(pcl::visualization::PCL_VISUALIZER_COLOR, 1.0, 0.0, 0.0, ss.str());
            viewer->setShapeRenderingProperties(pcl::visualization::PCL_VISUALIZER_OPACITY, 1.0, ss.str());
        }
    }

    while (!viewer->wasStopped())
    {
        viewer->spinOnce(100);
    }

    return 0;
}

이 코드에서는 여러 개의 구 마커를 추가한 후, x 좌표가 3보다 큰 마커의 색상을 빨간색으로 변경하여 강조하고 있다. 조건에 맞는 마커를 필터링하여 시각적으로 구분함으로써, 특정 데이터를 효과적으로 강조할 수 있다.

마커 클러스터링과 레이블링¶

마커를 클러스터링하여 그룹별로 레이블을 지정하면, 대규모 데이터에서도 특정 그룹을 쉽게 식별할 수 있다. PCL은 클러스터링된 마커에 대한 레이블링 기능을 제공한다.

// 마커 클러스터링 및 레이블링 예제
#include <pcl/visualization/pcl_visualizer.h>
#include <pcl/point_types.h>
#include <pcl/filters/extract_indices.h>
#include <pcl/segmentation/extract_clusters.h>

int main()
{
    pcl::visualization::PCLVisualizer::Ptr viewer(new pcl::visualization::PCLVisualizer("Clustering and Labeling Example"));
    viewer->setBackgroundColor(0.1, 0.1, 0.1);

    // 포인트클라우드 로드
    pcl::PointCloud<pcl::PointXYZ>::Ptr cloud(new pcl::PointCloud<pcl::PointXYZ>());
    pcl::io::loadPCDFile<pcl::PointXYZ>("cluster_sample.pcd", *cloud);
    viewer->addPointCloud<pcl::PointXYZ>(cloud, "sample cloud");

    // 클러스터링 설정
    pcl::EuclideanClusterExtraction<pcl::PointXYZ> ec;
    pcl::search::KdTree<pcl::PointXYZ>::Ptr tree(new pcl::search::KdTree<pcl::PointXYZ>());
    tree->setInputCloud(cloud);
    ec.setClusterTolerance(0.5);
    ec.setMinClusterSize(10);
    ec.setMaxClusterSize(1000);
    ec.setSearchMethod(tree);

    std::vector<pcl::PointIndices> cluster_indices;
    ec.extract(cluster_indices);

    int cluster_id = 0;
    for (const auto& indices : cluster_indices)
    {
        pcl::PointCloud<pcl::PointXYZ>::Ptr cluster(new pcl::PointCloud<pcl::PointXYZ>());
        for (const auto& idx : indices.indices)
            cluster->push_back((*cloud)[idx]);

        std::stringstream ss;
        ss << "cluster_" << cluster_id;
        pcl::visualization::PointCloudColorHandlerRandom<pcl::PointXYZ> color_handler(cluster);
        viewer->addPointCloud<pcl::PointXYZ>(cluster, color_handler, ss.str());

        // 클러스터 중심 계산
        Eigen::Vector4f centroid;
        pcl::compute3DCentroid(*cluster, centroid);
        pcl::PointXYZ centroid_point(centroid[0], centroid[1], centroid[2]);

        // 중심에 텍스트 레이블 추가
        std::stringstream label_ss;
        label_ss << "Cluster " << cluster_id;
        viewer->addText3D(label_ss.str(), centroid_point, 0.5, 1.0, 1.0, 1.0, "label_" + std::to_string(cluster_id));

        ++cluster_id;
    }

    while (!viewer->wasStopped())
    {
        viewer->spinOnce(100);
    }

    return 0;
}

이 예제에서는 포인트클라우드를 클러스터링한 후, 각 클러스터의 중심에 해당 클러스터를 식별할 수 있는 텍스트 레이블을 추가하고 있다. EuclideanClusterExtraction을 사용하여 클러스터를 추출하고, 각 클러스터에 고유한 색상을 부여한 후, 중심 위치에 3D 텍스트를 추가함으로써 클러스터를 시각적으로 구분하고 식별할 수 있다.

마커와 텍스트의 성능 최적화¶

포인트클라우드 시각화에서 다수의 텍스트와 마커를 사용하는 경우, 성능 저하가 발생할 수 있다. 이를 방지하기 위해 최적화 기법을 적용할 수 있다.

  • 텍스트 및 마커의 최소화: 필요한 정보만을 표시하여 불필요한 텍스트와 마커의 생성을 줄이다.
  • 효율적인 업데이트: 동적 요소의 업데이트 빈도를 조절하여 CPU 부하를 줄이다.
  • 그룹화 활용: 마커를 그룹화하여 일괄적으로 관리하고, 불필요한 개별 업데이트를 피한다.
  • 렌더링 속성 최적화: 투명도나 복잡한 색상 설정을 최소화하여 렌더링 속도를 향상시킨다.
// 성능 최적화를 고려한 텍스트 및 마커 관리 예제
#include <pcl/visualization/pcl_visualizer.h>
#include <vector>

int main()
{
    pcl::visualization::PCLVisualizer::Ptr viewer(new pcl::visualization::PCLVisualizer("Performance Optimization Example"));
    viewer->setBackgroundColor(0.05, 0.05, 0.05);

    // 포인트클라우드 추가
    pcl::PointCloud<pcl::PointXYZ>::Ptr cloud(new pcl::PointCloud<pcl::PointXYZ>());
    // ... 포인트클라우드 데이터 로드 ...
    viewer->addPointCloud<pcl::PointXYZ>(cloud, "sample cloud");

    // 마커 그룹 추가
    std::string group_id = "optimized_markers";

    // 다수의 마커를 그룹으로 추가
    for (int i = 0; i < 100; ++i)
    {
        pcl::PointXYZ point(i * 0.1, i * 0.1, i * 0.1);
        std::stringstream ss;
        ss << "optimized_sphere_" << i;
        viewer->addSphere(point, 0.05, 0.0, 1.0, 0.0, ss.str(), group_id);
    }

    // 그룹 전체의 속성 설정 (예: 색상과 투명도)
    viewer->setShapeRenderingProperties(pcl::visualization::PCL_VISUALIZER_COLOR, 0.0, 1.0, 0.0, group_id);
    viewer->setShapeRenderingProperties(pcl::visualization::PCL_VISUALIZER_OPACITY, 0.3, group_id);

    // 필요한 경우에만 텍스트 업데이트
    bool update_text = true;
    if (update_text)
    {
        viewer->addText("Optimized Marker Group", 10, 90, "optimized_text", 0);
        viewer->setShapeRenderingProperties(pcl::visualization::PCL_VISUALIZER_TEXT_COLOR, 1.0, 1.0, 1.0, "optimized_text");
        viewer->setShapeRenderingProperties(pcl::visualization::PCL_VISUALIZER_FONT_SIZE, 12, "optimized_text");
    }

    while (!viewer->wasStopped())
    {
        viewer->spinOnce(100);
    }

    return 0;
}

이 예제에서는 다수의 마커를 그룹화하여 관리하고, 그룹 전체의 색상과 투명도를 한 번에 설정함으로써 개별 마커에 대한 반복적인 업데이트를 방지하여 성능을 최적화하고 있다. 또한, 텍스트 업데이트는 필요한 경우에만 수행하여 불필요한 연산을 최소화하고 있다.

마커와 텍스트의 저장 및 로드¶

시각화에 추가된 마커와 텍스트를 저장하고, 필요할 때 다시 로드하여 일관된 시각화 환경을 유지할 수 있다. 이를 위해 설정 정보를 파일로 저장하거나, 코드 내에서 재사용할 수 있는 구조로 관리할 수 있다.

// 마커 및 텍스트 저장과 로드 예제
#include <pcl/visualization/pcl_visualizer.h>
#include <fstream>
#include <nlohmann/json.hpp> // JSON 라이브러리 필요

using json = nlohmann::json;

int main()
{
    pcl::visualization::PCLVisualizer::Ptr viewer(new pcl::visualization::PCLVisualizer("Save and Load Markers Example"));
    viewer->setBackgroundColor(0.2, 0.2, 0.2);

    // 포인트클라우드 추가
    pcl::PointCloud<pcl::PointXYZ>::Ptr cloud(new pcl::PointCloud<pcl::PointXYZ>());
    // ... 포인트클라우드 데이터 로드 ...
    viewer->addPointCloud<pcl::PointXYZ>(cloud, "sample cloud");

    // 마커 및 텍스트 정보 저장
    json marker_data;
    for (int i = 0; i < 10; ++i)
    {
        std::stringstream ss;
        ss << "sphere_" << i;
        pcl::PointXYZ point(i, i, i);
        viewer->addSphere(point, 0.1, 0.0, 0.0, 1.0, ss.str());

        // JSON에 마커 정보 저장
        marker_data["markers"].push_back({
            {"id", ss.str()},
            {"x", point.x},
            {"y", point.y},
            {"z", point.z},
            {"radius", 0.1},
            {"color", {0.0, 0.0, 1.0}}
        });
    }

    // 텍스트 정보 저장
    viewer->addText("Saved Markers Example", 10, 10, "save_text");
    marker_data["text"] = {
        {"id", "save_text"},
        {"content", "Saved Markers Example"},
        {"x", 10},
        {"y", 10},
        {"font_size", 12},
        {"color", {1.0, 1.0, 1.0}}
    };

    // JSON 파일로 저장
    std::ofstream file("markers.json");
    file << marker_data.dump(4);
    file.close();

    // 시각화 창 유지
    while (!viewer->wasStopped())
    {
        viewer->spinOnce(100);
    }

    // JSON 파일 로드 및 마커 재생성
    std::ifstream infile("markers.json");
    json loaded_data;
    infile >> loaded_data;
    infile.close();

    pcl::visualization::PCLVisualizer::Ptr new_viewer(new pcl::visualization::PCLVisualizer("Loaded Markers Example"));
    new_viewer->setBackgroundColor(0.2, 0.2, 0.2);
    new_viewer->addPointCloud<pcl::PointXYZ>(cloud, "sample cloud");

    // 마커 재생성
    for (const auto& marker : loaded_data["markers"])
    {
        pcl::PointXYZ point(marker["x"], marker["y"], marker["z"]);
        new_viewer->addSphere(point, marker["radius"], marker["color"][0], marker["color"][1], marker["color"][2], marker["id"]);
    }

    // 텍스트 재생성
    auto text = loaded_data["text"];
    new_viewer->addText(text["content"], text["x"], text["y"], text["id"]);
    new_viewer->setShapeRenderingProperties(pcl::visualization::PCL_VISUALIZER_TEXT_COLOR, text["color"][0], text["color"][1], text["color"][2], text["id"]);
    new_viewer->setShapeRenderingProperties(pcl::visualization::PCL_VISUALIZER_FONT_SIZE, text["font_size"], text["id"]);

    while (!new_viewer->wasStopped())
    {
        new_viewer->spinOnce(100);
    }

    return 0;
}

이 예제에서는 마커와 텍스트의 정보를 JSON 형식으로 저장하고, 이후에 이를 다시 로드하여 시각화 창에 재생성하는 과정을 보여준다. 이를 통해 복잡한 시각화 설정을 파일로 관리하고, 필요할 때 쉽게 복원할 수 있다. JSON 라이브러리인 nlohmann/json을 사용하여 데이터의 직렬화와 역직렬화를 수행하였다.
