1. 스프린트 리뷰에서의 스크럼 마스터 역할

스프린트 리뷰(Sprint Review)에서 스크럼 마스터는 이벤트가 건설적이고 협력적인 분위기에서 진행되도록 촉진하는 역할을 수행한다. 스프린트 리뷰는 프러덕트 오너가 주도하는 이벤트이나, 스크럼 마스터는 프로세스의 효과성을 보장하는 촉진자 역할을 담당한다.

스크럼 마스터의 촉진 활동은 이해관계자 참석의 촉진이다. 관련 이해관계자가 스프린트 리뷰에 참석하도록 조율하고, 참석의 중요성을 조직에 소통한다. 건설적 피드백 환경의 조성이다. 이해관계자가 비판이 아닌 건설적 피드백을 제공하도록 분위기를 조성하고, 특정 이해관계자의 의견이 지배적이 되지 않도록 균형을 유지한다. 피드백의 체계적 수집이다. 구두로 전달된 피드백이 누락되지 않도록 기록을 관리하고, 프러덕트 오너가 이를 제품 백로그에 반영할 수 있도록 지원한다. 시간 관리이다. 스프린트 리뷰가 시간 제한 내에 효과적으로 완료되도록 시간을 관리한다.

2. 피드백 관리의 체계

스프린트 리뷰에서 수집된 이해관계자 피드백은 즉각적 반영 항목(긴급하고 중요한 피드백으로 다음 스프린트에 반영), 백로그 추가 항목(중요하나 즉각적이지 않은 피드백으로 제품 백로그에 추가), 조사 필요 항목(추가 분석이 필요한 피드백으로 탐색(Spike)이 필요), 그리고 거절 항목(제품 비전과 부합하지 않거나 기술적으로 부적절한 피드백으로 거절 근거를 명시)으로 분류하여 관리된다.

스크럼 마스터는 이러한 피드백 분류와 추적 체계의 효과적 운영을 지원하고, 피드백이 체계적으로 처리되고 있는지를 모니터링한다.
