16.26 스크럼 마스터의 역량 성숙도 모델과 전문 인증 체계

1. 역량 성숙도 단계

스크럼 마스터의 역량 성숙도는 다음 단계로 구분된다. 초급(Junior SM)은 스크럼의 기본 프레임워크를 이해하고, 단일 팀에서 기본적 촉진 활동을 수행하는 수준이다. 중급(Experienced SM)은 팀의 자기 관리 역량을 효과적으로 코칭하고, 조직적 장애물을 식별·제거하며, 다양한 촉진 기법을 구사하는 수준이다. 상급(Advanced SM/Agile Coach)은 복수 팀의 애자일 성숙도를 향상시키고, 조직 수준의 변화를 촉진하며, 경영진에 대한 코칭을 수행하는 수준이다. 전략적(Enterprise Agile Coach)은 기업 전체의 애자일 전환 전략을 수립하고, 조직 구조와 문화의 변혁을 주도하는 수준이다.

2. 전문 인증 체계

스크럼 마스터의 전문성을 인증하는 주요 인증 체계는 다음과 같다. Scrum Alliance의 인증 체계는 Certified ScrumMaster(CSM), Advanced Certified ScrumMaster(A-CSM), 그리고 Certified Scrum Professional-ScrumMaster(CSP-SM)의 3단계이다. Scrum.org의 인증 체계는 Professional Scrum Master I(PSM I), PSM II, 그리고 PSM III의 3단계이다. SAFe의 인증 체계는 SAFe Scrum Master(SSM)과 SAFe Advanced Scrum Master이다. ICAgile의 인증은 ICAgile Certified Professional-Agile Coaching(ICP-ACC)이다.

스크럼 마스터의 역량 개발은 공식 인증 취득에 더하여 실무 경험의 축적, 멘토링, 애자일 커뮤니티 참여, 그리고 관련 분야(촉진, 코칭, 조직 개발)의 학습을 포괄하는 지속적 과정이다.
