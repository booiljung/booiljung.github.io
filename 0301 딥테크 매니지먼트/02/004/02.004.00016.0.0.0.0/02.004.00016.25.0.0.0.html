16.25 애자일 성숙도 평가 모델과 조직 수준 진단

1. 애자일 성숙도의 개념

애자일 성숙도(Agile Maturity)는 조직이 애자일 원칙과 실천을 내재화하고 효과적으로 실행하는 수준을 의미한다. 애자일 성숙도 평가는 스크럼 마스터가 조직의 현재 상태를 객관적으로 진단하고 개선 방향을 설정하기 위한 체계적 도구이다.

2. 성숙도 평가 모델의 유형

Sidky Agile Measurement Index(SAMI)는 Sidky, Arthur and Bohner(2007)가 개발한 모형으로, 협력적(Collaborative), 진화적(Evolutionary), 효과적(Effective), 적응적(Adaptive), 그리고 둘러싸는(Encompassing)의 5단계 성숙도를 제시한다.

Agile Fluency Model은 Shore and Larsen(2018)이 개발한 모형으로, 집중(Focusing), 전달(Delivering), 최적화(Optimizing), 그리고 강화(Strengthening)의 4개 구역(Zone)을 제시한다. 각 구역은 팀이 특정 수준의 애자일 가치를 전달하는 데 유창한(Fluent) 상태를 나타낸다.

3. 조직 수준 진단의 영역

프로세스 차원은 스크럼 이벤트의 정기적 개최, 산출물의 올바른 관리, 그리고 경험적 프로세스 제어의 실천 수준을 평가한다. 팀 차원은 자기 관리 역량, 교차 기능성, 지속적 개선 실천, 그리고 기술적 실천(TDD, CI/CD 등)의 수준을 평가한다. 조직 차원은 경영진의 애자일 이해와 지지, 조직 구조의 애자일 친화성, 인센티브 체계의 정합성, 그리고 조직 문화의 애자일 지향성을 평가한다.

스크럼 마스터는 이러한 진단 결과에 기반하여 팀과 조직의 현재 성숙도 수준과 목표 수준 간의 격차를 식별하고, 우선순위에 따른 개선 행동 계획을 수립한다. 성숙도 진단은 연 1~2회 정기적으로 수행하여 추세적 변화를 추적한다.
