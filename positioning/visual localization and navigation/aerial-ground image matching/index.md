# 항공-지상 이미지 매칭 (Aerial-Ground Image Matching)



이 기술은 항공기나 위성에서 촬영한 이미지와 지상에서 촬영한 이미지를 매칭하여 위치를 특정하는 데 중점을 둔다. 서로 다른 시점(Cross-view)에서 얻은 정보를 활용하는 것이 핵심이다.
