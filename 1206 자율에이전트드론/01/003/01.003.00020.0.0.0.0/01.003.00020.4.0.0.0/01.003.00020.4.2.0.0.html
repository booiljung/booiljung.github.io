## 0.1 상태 추정 필터와 전역 경로 플래너 간 기능 분산 최적화

자율 비행 드론의 항법 제어 소프트웨어 스택을 구성하는 서브시스템들 중에서, 기체의 현재 3차원 공간 위치 및 자세를 연산해 내는 ’상태 추정 필터(State Estimation Filter, 예: EKF 또는 비선형 관찰자)’와 목표 목적지까지의 장기적인 무충돌 궤적을 연산하는 ’전역 경로 플래너(Global Path Planner)’는 연산의 시간 복잡도(Computational Complexity)와 데드라인(Deadline) 민감도 측면에서 시스템 아키텍처 적으로 완벽하게 양 극단의 대척점에 서 있는 컴포넌트들이다.

상태 추정 필터는 IMU, 비전 오도메트리(Visual Odometry), GPS 등에서 폭포수처럼 쏟아지는 불확실성 다중 센서 데이터를 즉각적으로 융합 연산하여 기하학적 비행 제어기(Geometric Controller) 측에 400\text{Hz} 이상의 초고속으로 상태 피드백을 락 없이 주입해 주어야 하는 극단적인 하드-실시간(Hard Real-time) 타이밍 특성을 지닌다. 반면 전역 경로 플래너(예: A*, RRT* 혹은 VLA 모델)는 방대한 대역폭의 3차원 옥트리(Octree) 클라우드 맵 상에서 기체의 동역학 제약이 반영된 최적 궤적 공간을 탐색하기 위해, 수백 밀리초(ms)에서 길게는 수 초 단위의 긴 노드 실행 시간(WCET)을 태생적으로 감내해야만 하는 소프트-실시간(Soft Real-time) 위주의 CPU 바운드(CPU-Bound) 집중 탐색 연산 특성을 띤다.

이 두 가지 극단적인 이질적 타임라인 컴포넌트를 단일 노드의 동일 이그제큐터(Executor) 풀이나 동일 콜백 그룹 인터페이스 공간에 혼재시키는 안일한 설계는, 전역 경로 플래너의 긴 연산 스레드 체류 시간(Blocking Time)이 상태 추정 필터의 라운드 로빈 스케줄링을 무자비하게 지연시킴으로써 기체 추락에 직결되는 필연적인 시스템 제어 지터(Jitter)를 양산한다. 따라서 현대 ROS 2 기반 자율 에이전트 드론 아키텍처는 이를 프레임워크 층위에서 논리적 콜백 위상 분리와 멀티 이그제큐터 단위 토폴로지(Topology) 통합 전략 모델로 극복해 내는 구조적 기능 분산 최적화를 핵심으로 채택한다.

본 비대칭 연산 격리 최적화 아키텍처의 설계론은 다음과 같이 전개된다.
첫째, 상태 추정 필터의 칼만 행렬 연산을 캡슐화한 잦은 서브스크립션과 제어 타이머 집합을 드론 노드 내 최상위 우선순위를 지닌 단폭의 상호 배타적 콜백 그룹(Mutually Exclusive Callback Group)으로 칼같이 분리 명세한다. 더 나아가 이 핵심 그룹은 PREEMPT_RT 커널 등 실시간 운영체제(RTOS) 환경 위에서 RT(Real-Time) 최상위 선점 우선순위를 하드웨어적으로 할당받은 전용의 단일 스레드 이그제큐터(Stand-alone Single-Threaded Executor) 공간에만 단독으로 물리적 바인딩(Binding)된다. 이를 통해 필터 루프는 센서 데이터가 트리거되는 매 순간마다 어떠한 외부 플래닝 연산 하중에도 스레딩 간섭을 받지 않는 절대적 무결성(Integrity) 실행 공간을 보장받는다.

둘째, 전역 경로 플래너를 구성하는 무거운 로컬 맵 데이터 수신 및 헤비 그래프 탐색 계층의 플래닝 콜백 구문들을 복수의 재진입 가능 콜백 그룹(Reentrant Callback Group)으로 응집 군집화시킨다. 이 복합 콜백 그룹들은 타이머 제약을 받지 않는 범용 멀티스레드 이그제큐터(Multi-Threaded Executor) 공간에 하위 할당되어, 코어의 잔여 유휴 스레드 자원 풀을 최대한 활용하며 백그라운드 단에서 궤적 병렬 연산을 가속 수행토록 설계한다.

셋째, 메모리 접근에 있어 이 분리된 두 이그제큐터 간의 단방향 데이터 토폴로지 동기화(Topology Synchronization) 브리지 지점이다. 백그라운드 스핀 프로세싱을 마치고 도출된 전역 경로 플래너는 신규로 산출된 목표 궤적(Trajectory) 추종 행렬을 원자적(Atomic) 스마트 포인터 조작이나 락-프리 큐(Lock-Free Buffer Queue)를 매개로 하여, 전혀 다른 고주파 위상으로 초고속 스핀 중인 상태 추정 필터 혹은 LQR 제어 루프 영역에 비간섭적으로 밀어 넣는다. 이후 상태 추정 필터 스레드는 경로 플래너와 스레드 이행적 공유 의존 충돌(Transitive Dependency Lock)을 빚지 않고도 기존 OS 컨텍스트 스위칭 락 메커니즘을 영리하게 피해, 안전무결하게 매 스텝 갱신된 전역 위치 추종 벡터를 즉시 가져가 국지 제어 비행 연산에 결합시킨다.
결론적으로 이러한 타이밍 양극화된 위상 탑재 기능의 스케줄링 분산과 느슨한 메모리 버퍼 토폴로지 브리지 통합 전략의 건축은, 방대한 연산의 자율 비행 시스템이 엣지 보드의 제한된 MPU 프로세서 위에서 하드 리얼타임 비행 생존성(Survivability)과 AI 탐색 지능(Intelligence)이라는 상반된 두 역학을 충돌 파괴 없이 동시에 무사 수용할 수 있게 뒷받침해 주는 아키텍처적 마일스톤(Milestone) 기술이다.
