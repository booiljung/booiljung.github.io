21.3.2 다중 콜백 인입에 따른 이그제큐터(Executor) 계층 내 자원 경합과 타이머 콜백 지연의 상관관계

다자유도 역학 제어를 수행하는 자율 에이전트 드론의 ROS2 노드는 단일 컴포넌트 내에서도 다수의 데이터 토픽(Topic) 구독, 비동기 서비스(Service) 응답, 그리고 정주기 타이머(Timer)를 복합적으로 병행 운용한다. ROS2의 코어 아키텍처에서 이러한 다양한 비동기 이벤트들은 이그제큐터(Executor) 계층으로 일원화되어 콜백(Callback) 함수의 실행을 승인받는다. 그러나 이그제큐터에 할당된 제한적인 스레드 자원 내에서 복수의 콜백 이벤트가 동시다발적으로 인입될 경우, 치열한 자원 경합(Resource Contention)이 유발되며 이는 필연적으로 고정밀 고주파수 타이머 콜백의 스케줄링 지연(Latency)으로 직결된다. 본 절에서는 다중 콜백 인입 구조 하에서 파생되는 이그제큐터 내부 스레드 경합 모델과 타이머 실행 지연 간의 비선형적 초래 상관관계를 학술적으로 규명한다.

1. 이그제큐터 계층의 큐잉 및 디스패치 구조 병목

ROS2 rclcpp::Executor 노스탤지어는 하위 미들웨어(RMW) 프로토콜 스택으로부터 전달받은 이벤트를 대기 집합(WaitSet) 구조 내에서 블로킹(Blocking) 방식으로 폴링하고, 검출된 이벤트를 적절한 사용자 콜백 함수로 순차 변환(Dispatch)하는 통합 관문 역할을 담당한다. 기본적으로 채택되는 단일 스레드 이그제큐터(Single-Threaded Executor)는 이 과정에서 철저한 실행 직렬화(Execution Serialization)를 메모리 상에서 강제한다. 즉, 큐(Queue)에 선착하여 적재된 대용량 메시지의 렌더링 또는 서브스크립션(Subscription) 콜백 콜이 CPU를 선점(Preempt)하여 점유 중이라면, 그 뒤 큐에서 만료 대기 중인 아무리 중요도가 높은 제어 타이머 콜백일지라도 선행 콜백이 완전한 제어권을 반환(Return)할 때까지 대기행렬(Waiting Line)에서 무한정 블로킹 상태에 고착된다. 다중 스레드 이그제큐터(Multi-Threaded Executor)를 도입하여 병렬성을 꾀하더라도, 스레드 간 뮤텍스(Mutex) 락 경합 및 막대한 문맥 교환(Context Switching) 연산 오버헤드의 장벽에 부딪혀 근본적인 큐잉 지연(Queuing Delay) 계류 모델을 산술적으로 극복해낼 수 없다.

2. 자원 경합에 의한 타이머 우선순위 역전(Priority Inversion) 현상

무작위 다중 콜백 인입 환경에서 가장 우려되는 역학적 변형 인자는 타이머 콜백의 우선순위 역전(Priority Inversion) 현상이다. 로보틱스 고주파수 제어 루프에서 자세 산출용 타이머 콜백은 시스템 전체에서 가장 높은 스케줄링 우선순위(Highest Precedence) 계층을 점유해야 마땅함에도 불구하고, ROS2 기본 이그제큐터는 큐잉된 순서 보존과 순수 라운드 로빈(Round-Robin) 방식 또는 선형 탐색 알고리즘에 기반한 경직된 자원 분배 로직을 수용한다. 비전 데이터나 대용량 라이다(LiDAR) 3차원 포인트 클라우드 처리와 같은 극단적인 연산 집약형(Compute-Intensive) 인입 콜백 프로세스가 이그제큐터의 계산 스레드를 기습적으로 과다 점유할 경우, 짧고 엄격한 시정수를 요하는 제어 프로세스의 타이머 콜백은 꼼짝없이 기아(Starvation) 상태에 빠져들게 된다. 이는 하드웨어 모터의 반응 주파수를 치명적으로 붕괴시키는 촉매제로 발현된다.

3. 콜백 도착률(Arrival Rate)과 지연율(Latency Rate) 상관 방정식

다수의 로보틱스 성능 평가 문헌에서 제시된 학술적 스케줄링 이론에 따르면, 비동기 이벤트 콜백의 무작위 인입 도착률(Arrival Rate, \lambda) 빈도가 급증할수록 최우선 제어 타이머 콜백이 감내해야 하는 평균 체류 지연 시간(Average Latency Time, W_q)은 폴링 이론 및 대기 행렬 선형 근사치에 따라 지수 함수를 그리며 악화되는 강한 양의 상관관계를 노출한다. 즉, 드론 시스템의 동적 이벤트 처리 요구 부하(Event Processing Load)가 허용된 스레드 연산 처리 용량(Processing Capacity, \mu)의 임계 턱밑에 도달하게 될 무렵, 이그제큐터 내부 인터페이스의 콜백 대기 시간은 기하급수적으로 폭증함을 실증한다. 드론 에이전트가 복잡한 장애물 궤적 또는 전파 간섭이 범람하는 센서 데이터 과잉 구역에 진입 시 발생하는 네트워크 트래픽 급증 사태는, 곧바로 제어 타이머의 응답 지터(Jitter) 변형치를 정상 궤도 밖 통제 불능 상태(Uncontrollable Threshold)로 밀어내는 치명적 원인으로 작동한다.

4. 시스템 통제 결정론을 위협하는 구조적 결함 검토

종합적으로 판단할 때, 이그제큐터 계층 내에 형성된 단일화된 자원 경합 풀 환경은 고속 정밀 제어의 영순위 전제 조건인 응답 결정론적 타이밍(Deterministic Timing Behavior) 보장에 대한 강력한 메커니즘 위배 소지를 양산한다. 하나의 프로세스 공간 내에서 예측이 불가한 비정형 외부 이벤트 콜백 군간과, 절대적인 지연 없는 실행을 담보 받아야 할 정형적 제어 주파수 타이머 콜백 트래픽을 동일 선상의 이그제큐터 파이프에 밀어 넣고 경쟁시키는 구조적 관행은, 자율 동작 에이전트 드론 내비게이션의 물리적 안정성 여유(Stability Margin)를 심각히 위협하는 배임으로 판단된다. 이에 대한 하드웨어 지향적인 탈결합(Decoupling), 즉 물리적 스레드 레벨 격리를 전제로 한 콜백 그룹 관리 체계(Callback Group Isolation)와 실시간 커널에 의존하는 선점형 스케줄링(Preemptive Scheduling) 우회 설계 방식의 적용은 더 이상 미룰 수 없는 학술적 필요충분 교정 과제임이 입증된다.
