## 0.1 총합 콜백 지연 및 락 블로킹 유발 인자 정량 분석

ROS 2의 분산 컴퓨팅 환경에서 자율 비행 제어 루프가 요구하는 실시간 데드라인(Deadline)을 온전하게 보장하려면, 콜백 발현 인지가 이루어지는 시점부터 실제 프로세서 점유에 이르는 총합 콜백 지연(Total Callback Latency)을 구성 계층별로 분해하고 정량적으로 분석하는 절차가 필수적이다. 이 총합 지연 모델은 크게 하위 네트워크 스택 및 미들웨어 계층(예: DDS/RTPS)의 메시지 역직렬화 및 수신 지연, RCL(ROS Client Library) 계층의 대기열(Wait Set) 등록 지연, 그리고 이그제큐터(Executor) 계층의 스레드 디스패칭(Dispatching) 및 동기화 지연으로 구조화된다.

전체 지연 시간의 정확한 연산을 위해서는, 퍼블리셔(Publisher)가 메시지 구조체를 전송하는 물리적 순간에서부터 서브스크라이버(Subscriber) 혹은 타이머의 콜백 함수 내부 로직이 메모리로 호출(Invoke)되어 실행을 시작하는 순간(t_{end} - t_{start})까지의 모든 파이프라인 전이를 타임스탬프 기준으로 포집해야 한다. 수집된 계측 지표에서 시스템 성능 병목의 핵심 인자로 작용하는 파라미터는 단연 이그제큐터 계층 내부의 락 블로킹(Lock Blocking) 현상이다. 특히 ROS 2(Humble/Iron 버전 기준)의 기본 멀티스레드 이그제큐터(Multi-Threaded Executor)는 독립적인 비동기 이벤트를 폴링(Polling)하고 스레드 풀(Thread Pool)에 작업을 인가하는 스케줄링 과정에서 전역 뮤텍스 락(Global Mutex Lock)을 반복적으로 획득하고 해제하는 아키텍처적 특성을 지닌다.

락 블로킹을 유발하는 주요 인자를 정량 분석하기 위해서는 상태 머신의 전이 타이밍을 측정해야 한다. 계측 데이터 상에서 콜백 대기열이 준비 상태(Ready)로 전환된 시점(Time-to-Ready)과, 유휴 스레드가 스케줄러 보드에 접근하여 콜백 구문을 추출하고 실행하는 시점(Time-to-Execute) 사이의 델타(\Delta t)를 산출한다. 이 간극이 유의미하게 임계치 이상으로 확장되는 현상은 대체로 대량의 계측용 저주파수 통신 노드와 비행 제어용 고주파수 노드가 동일한 이그제큐터 객체 혹은 상호 배타적 콜백 그룹(Mutually Exclusive Callback Group)에 중첩 배정되었을 때 발생한다. 이는 시스템 내부의 우선순위 역전(Priority Inversion)과 유효 스레드의 기아 상태(Thread Starvation)를 유발하는 직접적 요인으로 기능한다.

데이터 추세 분석 도구(예: Trace Compass 오픈소스 툴킷 기법)를 활용하여 포집된 프로파일링 데이터를 통계적으로 분석할 때, 연구자는 반드시 다음의 양적 지표를 추출해야 한다.
첫째, 단일 제어 루프 컴포넌트 내 콜백 호출 간격의 지터(Jitter) 표준 편차이다. 지터 분포의 편차가 크게 측정되는 것은 락 경합 빈도가 비결정론적(Non-deterministic)으로 발생함을 수치화한 것이며, 운영체제 단위에서 RTOS 기반 실시간 스케줄러(예: SCHED_FIFO)를 강제 적용하더라도 애플리케이션 프레임워크 내부의 뮤텍스 설계 한계로 인해 지연 현상이 완전히 소거되지 않음을 입증한다.
둘째, 최악 실행 시간(WCET, Worst-Case Execution Time) 관점에서의 최대 지연(Maximum Latency) 관측치이다. 초월적 역학(Flight Dynamics) 제어기는 통계적 평균 실행 시간(ACET)보다 단일 사이클의 최악 지연 파동에 기체 유지 제어능력을 급격히 상실할 수 있으므로, 락 블로킹에 의해 파생된 최대 지연 폭이 시스템의 요구 주파수 역수(예: 400Hz의 경우 2.5ms)를 초과하는 침범 빈도 구조를 명확히 도출하여야 한다.

이러한 정량적 인과관계 분석 결과는 기존의 무결성 부족 결함을 지닌 공유 콜백 구조를 다중 커스텀 이그제큐터 레이어로 분리하거나, 메모리 장벽(Memory Barrier) 형태의 락-프리(Lock-Free) 메시지 링 버퍼를 차용하도록 시스템을 재설계하는 확고한 양적 및 학술적 근거로서 사용된다.
