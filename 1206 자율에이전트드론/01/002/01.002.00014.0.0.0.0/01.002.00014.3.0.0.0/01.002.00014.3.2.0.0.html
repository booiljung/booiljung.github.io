14.3.2 브릿지 스레드의 CPU 선호도 및 실시간 우선순위 고정 튜닝

자율 드론 시스템의 컴패니언 컴퓨터(Companion Computer)에는 고정밀 영상 스티칭, 딥러닝 객체 인식 프로파일링, 비전 모달 센서 융합 등 컴퓨팅 파워를 극도로 소모하는 대규모 연산 컴포넌트가 동시에 구동된다. 이 과정에서 ROS 2 생태계의 텔레메트리 패킷 브리지(Micro XRCE-DDS 등) 프로세스와 저수준 비행 제어기(Flight Controller) 간의 필수 통신 동기화 스레드가 시스템 백그라운드 태스크에 의해 스케줄링 후순위로 밀리게 되면 제어 타이밍 지연이나 마감 시간(Deadline) 이탈이 즉각 발생한다. 이를 시스템적으로 차단하기 위해 실시간 우선순위의 구조적 개편과 CPU 친화도(Affinity) 매핑 튜닝을 통신 스레드 계층에 강제 적용한다.

1. POSIX 기반 실시간 스케줄링 정책 단독 할당

리눅스 커널의 메인 프로세스 스케줄러인 CFS(Completely Fair Scheduler)는 운영체제 내 존재하는 모든 프로세스에게 가상의 실행 시간을 공평하게 분배하며 전반적인 응답성을 보장하는 것을 핵심 목표로 채택한다. 그러나 100Hz에서 500Hz에 이르는 고속 루프가 요구되는 드론 제어 생태계 통신 브리지 노드에게 스케줄링의 공평성이란 오히려 치명적인 통신 대기 시간(Latency) 급증을 야기하는 원인이 된다.

이를 극복하기 위해 관리자 권한의 chrt 터미널 명령어 도구를 이용하거나 런타임 ROS 2 노드 이니셜라이징 단계에서의 POSIX API를 활용하여, 핵심 브리지 스레드의 스케줄링 클래스(Scheduling Class)를 특정지어 SCHED_FIFO(First In, First Out) 또는 SCHED_RR(Round-Robin)과 같은 실시간 독점 정책(Real-Time Policy)으로 승격 변경한다. 이러한 정책들이 부여된 특수 프로세스는 시스템 내 최하위 일반 우선순위의 0번(SCHED_OTHER) 부류 태스크 프레임들을 구조적으로 완전히 무시하고 자신들이 요구하는 만큼 즉각적이고 영구적으로 프로세서를 선점한다. 일반적인 실시간 우선순위 값은 1에서 99 사이의 범주로 지정되며, 자율 비행 역학 모델의 필수적 제어 갱신 주기를 수호하기 90 이상의 초상위 우선순위를 통신 브리지 스레드에 고정 할당함으로써 운영 단위의 통신 지터(Jitter) 유발 가능성을 원천 억제한다.

2. CPU 하드웨어 코어 격리(CPU Isolation)와 선호도(Affinity) 매핑 체계

실시간 스케줄링 정책을 절대적으로 지정하더라도 고선점 스레드가 다수의 다중 하드웨어 코어를 무분별하게 옮겨 다니며 비효율적으로 순환 실행되는 코어 이주(Core Migration) 현상이 발생하면, 마이크로프로세서 내부의 L1/L2 하드웨어 레지스트리 캐시 빗나감(Cache Miss)이 누적 발생하여 문맥 교환(Context Switching) 지연 비용 비용이 예상치 못하게 급증한다.

이러한 캐시 오염을 방지하기 위한 초기 기법으로 커널의 부트로더 레벨 또는 시스템 빌드 환경 매개변수 설정에서 isolcpus 명령어 지시자를 선언하여 멀티코어 환경 상의 특정 하위 하드웨어 컴퓨팅 스레드(Core)를 스케줄러의 범용 작업 할당 목록에서 영구적으로 격리시킨다. 해당되는 특정 코어에는 스케줄러가 일반 스레드 프로세스를 결코 자체 분배하지 않는다.

위 단계를 거친 뒤 격리되어 완전히 비워진 전용 코어 영역에 브릿지 에이전트 프로세스만을 한정적이고 독점적으로 매핑시키는 CPU 친화도(Affinity) 튜닝을 실행한다. 리눅스 taskset 도구의 비트마스크 또는 POSIX 계열 sched_setaffinity() 구문을 명시적으로 선언 적용하여, 드론과 상호작용하는 통신 브리지 노드가 오직 격리 지명된 코어(예: 4코어 컴퓨팅 환경에서의 Core 3 단독 지정) 상에서만 구속 구동되게 강제한다. 이러한 설계는 고속 물리적 경로 제어와 기하학적 SLAM 등 극단적인 여타 고부하 연산 컨테이너 작업 프로세스 환경 속에서도, 물리적인 컴퓨팅 경합 충돌 현상과 시스템 메모리 캐시 오염을 물리적으로 단절시키는 확정성을 보장한다.
