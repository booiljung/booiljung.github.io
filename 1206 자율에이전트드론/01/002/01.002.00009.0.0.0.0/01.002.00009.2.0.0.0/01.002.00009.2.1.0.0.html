9.2.1 PX4 비행 스택의 내부 구조적 특징

1. 다중 계층 제어(Multi-Layer Control) 아키텍처의 분해

PX4 비행 스택(Flight Stack)의 내부 구조를 학술적으로 해부해보면, 이는 철저히 독립된 물리적/논리적 도메 영을 지닌 다중 계층 제어(Multi-Layer Control) 파이프라인으로 구성되어 있음을 확인할 수 있다. 가장 최상단에는 임무(Mission) 파싱(Parsing)과 복귀(RTL) 루틴을 계산하는 논리적 ’내비게이터(Navigator)’가 위치하고, 그 하단에 경로의 동역학적 스무딩(Smoothing)을 담당하는 ‘포지션 컨트롤러(Position Controller)’, 마지막 최하단에는 각속도 기반의 고주파 루프인 ’애티튜드(Attitude) 및 레이트(Rate) 컨트롤러’가 직렬로 연결된다. 이러한 계층 구조는 각 층위마다 엄격히 정의된 입출력 텐서(Tensor) 스펙을 지닌다. 즉, 포지션 컨트롤러가 산출한 3D 추력 벡터(Thrust Vector)만이 애티튜드 컨트롤러의 유일한 입력으로 인가되며, 역방향의 간섭은 엄격히 차단된다. 이러한 철저한 직렬화(Serialization) 통제는 특정 계층의 알고리즘을 새로운 논문 기반의 SOTA(State-of-the-Art) 모델(예: L1 Adaptive Control)로 치환할 때, 시스템 전반의 부작용(Side Effect)을 0으로 억제하는 완벽한 디커플링(Decoupling)을 시현한다.

2. EKF2 (Extended Kalman Filter 2) 중심의 상태 추정(State Estimation) 독점

제어 파이프라인을 이끄는 가장 중요한 나침반은 상태 추정(State Estimation) 모듈이며, PX4 아키텍처는 이를 ’EKF2’라는 단일 코어 모듈에 강력하게 독점(Monopolize)시킨다. 펌웨어 내부로 흘러 들어오는 관성 측정 장치(IMU), 기압계(Baro), 지자기 센서, RTK GPS, 심지어 외부 VIO(Visual Inertial Odometry)의 모든 파편화된 원시 신호(Raw Signal)들은 오직 EKF2 코어의 공분산 행렬(Covariance Matrix) 연산 속으로만 융합된다. EKF2는 이 무질서한 텔레메트리 파편들을 24개의 튜닝 블록 계수를 이용해 필터링한 뒤, 단일하고 오차 없는 ‘전역 및 지역 에스티메이트(Global/Local Estimate)’ 상태 행렬을 uORB 버스로 토해낸다. 이 추정 코어의 독점화 설계는, 아무리 노이즈가 심한 값싼 센서나 지연이 생기는 외부 인공지능 비전 신호가 물밀듯 밀려와도, 애티튜드 컨트롤러가 바라보는 현재 기체의 상태 벡터(State Vector)만큼은 절대 붕괴되지 않도록 지켜내는 가장 강력한 수학적 방어벽으로 작동한다.

3. 커맨더(Commander) 노드에 의한 전역 상태 기계(Global State Machine) 통치

PX4 아키텍처에서 모든 모듈들의 생사여탈권과 모드 전환을 지배하는 절대 군주의 역할은 commander 비동기 노드(Node)가 일임한다. 조종사의 RC 송신기 입력이나, 지상 관제소(GCS)의 암(Arm) 명령, 배터리 임계치 경고 등 시스템의 운명을 결정짓는 수많은 비정형 이벤트 트리거(Event Trigger)들은 모두 이 커맨더의 전역 상태 기계(Global State Machine)로 수렴된다. 커맨더 노드는 사전 정의된 수백 가지의 전이 플로우 차트(Transition Flowchart) 행렬을 O(1)의 속도로 검증하여, 현재 순간에 모터의 서보 락(Servo Lock)을 해제할지, 혹은 센서 캘리브레이션 불량으로 이륙을 영구히 거부할지 결정론적으로 판별한다. 이처럼 분산된 모듈형 아키텍처 내에서 단 하나의 옹골찬 중앙집권적 유한 상태 기계(FSM: Finite State Machine)를 배치한 PX4의 설계는 분산 체계가 빠지기 쉬운 논리적 불일치(Inconsistency) 오류를 원천 차단하는 공학적 묘수이다.

4. 결론

PX4 비행 스택의 내부 구조적 특징은 고도로 세분화된 계층 제어 파이프라인과, 그것을 지탱하는 EKF2라는 강력한 추정 철학, 그리고 전체의 질서를 교정하는 커맨더라는 지휘 체계의 정교한 삼각 편대에 있다. 이 삼위일체의 설계는 하드웨어의 생채기를 무시하고 오직 수학적인 궤적 최적화의 진실만을 추구할 수 있는 무균의 실험실을 창조해 내었으며, 차세대 자율 이동체 연구의 글로벌 스탠다드 펌웨어로 군림하게 만든 가장 본질적인 소프트웨어 헌장이다.
