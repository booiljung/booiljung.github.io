53.3 무향 칼만 필터(UKF) 기반 시그마 포인트(Sigma Point) 샘플링 제어 모델의 추정 성능 및 연산 비용

로봇 시스템의 상태 추정 과정에서 발생하는 비선형(Nonlinear) 동역학 및 관측 모델의 근사화 문제를 해결하기 위해 도입된 무향 칼만 필터(Unscented Kalman Filter, UKF)는 측정 모델 확률 분포의 직접적인 매핑을 수행한다. 확장 칼만 필터(Extended Kalman Filter, EKF)가 자코비안(Jacobian) 행렬을 기반으로 1차 테일러 급수(Taylor Series) 전개를 통해 선형화를 수행하는 것과 달리, UKF는 무향 변환(Unscented Transform, UT)을 활용하여 확정적(Deterministic)으로 추출된 시그마 포인트(Sigma Point) 집합을 비선형 함수에 직접 통과시킨다.

1. 무향 변환(UT) 및 시그마 포인트 샘플링 기법의 수학적 전개

무향 변환은 비선형 변환을 거친 확률 변수의 평균(Mean)과 공분산(Covariance)을 계산하기 위한 기법이다. 차원 n을 갖는 상태 벡터 x의 평균이 \mu이고 공분산 행렬이 P일 때, 시그마 포인트 집합 X_i는 다음과 같이 2n+1개로 구성된다.

X_0 = \mu
X_i = \mu + (\sqrt{(n+\lambda)P})_i, \quad i = 1, \dots, n
X_{i+n} = \mu - (\sqrt{(n+\lambda)P})_i, \quad i = 1, \dots, n

여기서 \lambda = \alpha^2(n+\kappa)-n은 시그마 포인트의 분포 거리를 결정하는 스케일링 파라미터이며, \alpha는 포인트들의 퍼짐 정도를 제어하고, \kappa는 보조 스케일링 파라미터로 작용한다. 선정된 시그마 포인트들은 시스템의 비선형 변환 부분 y = f(x)를 통과하여 예측된 시그마 포인트 Y_i = f(X_i)로 매핑되며, 최종적으로 예측된 평균 변수 \mu_y와 공분산 행렬 P_y는 가중치 W_i의 결합을 통해 도출된다.

\mu_y = \sum_{i=0}^{2n} W_i^{(m)} Y_i
P_y = \sum_{i=0}^{2n} W_i^{(c)} (Y_i - \mu_y)(Y_i - \mu_y)^T

이와 같은 샘플링 방식은 확률 분포 전체를 대략화하지 않으므로, 테일러 전개 등 미분 가능성을 요구하는 연산 및 복잡한 자코비안 행렬 정의 단계를 생략할 수 있다는 장점을 지닌다.

2. UKF 기반 상태 추정 모델의 추정 성능 분석

시그마 포인트 매핑을 기반으로 하는 UKF의 상태 추정 성능은 비선형 함수가 강한 곡률(Curvature)을 가질 때 EKF 대비 두드러지는 비교 우위를 나타낸다.

  1. 상위 차수 근사 정밀도: EKF의 경우 1차 미분(자코비안) 결과만을 반영하므로 2차 이상의 비선형성에서 심각한 잘림 오차(Truncation Error)가 누적된다. 반면 UKF의 무향 변환은 확률 분포의 확률 변수 모멘트를 반영하여 가우시안 시스템에서는 테일러 급수 3차 항(3rd-order term) 수준과 동등한 일치성을 제공하며, 비가우시안 시스템에 대해서도 평균적으로 2차 항 수준까지 일치시킨다.
  2. 다중 센서 융합의 상태 복원 강건성: 관성 측정 장치(IMU), 위성 항법계 시스템(GNSS) 등 센서 노이즈가 강하게 편향된 환경에서, UKF 알고리즘은 비선형성 투영의 왜곡을 축소하여 강건한 경로 궤적 복원(Trajectory Restoration) 성능을 발휘한다. 특이점(Singularity) 문제를 근원적으로 회피함으로써 불연속적 동역학, 혹은 쿼터니언(Quaternion) 방위 회전 행렬 처리에서 높은 품질의 추정 오차 관리성을 갖는다.

3. 시그마 포인트 제어 모델의 연산 비용(Computational Cost) 평가

UKF 구조는 예측 및 갱신 단계마다 비선형 투영과 확률론적 가중합 반복 계산을 요구하므로 추정 성능 향상의 이면에서는 선형 행렬 연산 비용의 급증이라는 상충 관계가 발생한다.

  1. O(n^3) 수준의 행렬 분해 처리량: 시그마 포인트를 샘플링하기 위해 매 필터 갱신마다 공분산 행렬 P에 대한 콜레스키 분해(Cholesky Decomposition) 등 제곱근 행렬 (\sqrt{(n+\lambda)P})을 추출해야 한다. 정사각 행렬을 기준으로 콜레스키 분해의 연산 복잡도는 행렬 차원 n에 대하여 O(n^3)의 계산 복잡도를 유발하게 된다.
  2. 샘플링 평가 오버헤드: 2n+1개의 시그마 포인트 전수 수량에 대하여 비선형 모델을 반복 투영하는 과정에서 대규모로 연산 자원을 소모한다. EKF는 비선형 함수 자체를 1회만 계산하지만, 예측 단계마다 2n+1배의 평가를 수행해야 하므로 자원 제약 환경(Edge Computing) 측면에서의 병목이 될 수 있다.
  3. 에지 인퍼런스 최적화 제약: 초당 수백 헤르츠 단위의 실시간성(Real-time Constraints)을 요구하는 마이크로프로세서 환경에서는 UKF 적용 시 상태 차원이 높아지면 런타임 지연(Latency)이 중대하게 발생할 위험성을 내포한다. 모델의 실시간 제약성을 유지하기 위해서는 상태 변수 부분 필터링, 축소 차원 시그마 샘플링 무향 칼만 필터(Square-Root UKF, SR-UKF) 등의 수치해석학적 최적 기법이 수반되어야 한다.

결론적으로 UKF 기반 시그마 포인트 모델은 강력한 수학적 비선형 투영 맵핑에 의거하여 EKF를 상회하는 높은 상태 추정 무결성을 증명한다. 그러나 상태 벡터 차수에 비례하여 행렬 스분해 추론 오버헤드가 급증하기 때문에 로보틱스 시스템에 탑재되는 임베디드 코어와 연산 자원의 트레이드오프(Trade-off)를 면밀하게 분석한 설계를 적용해야만 한다.
