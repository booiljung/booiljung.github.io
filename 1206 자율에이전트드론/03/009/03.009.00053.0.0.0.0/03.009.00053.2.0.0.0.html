53.2 확장 칼만 필터(EKF)의 비선형(Nonlinear) 동역학 자코비안(Jacobian) 선형화(Linearization) 한계 분석

1. 무인 항공기 다이내믹스의 본질적 비선형성(Non-linearity) 대두

선형 칼만 필터(Linear Kalman Filter, LKF)는 동역학 시스템 모델(Kinematic Model)과 측정 모델(Measurement Model)이 순수한 1차 선형 방정식 행렬 연산으로 구성될 때, 수학적으로 최적(Optimal)의 상태 추정 기댓값을 보장한다. 그러나 3차원 공간을 기동하는 무인 항공기(UAV)의 오일러 역학(Euler Dynamics), 즉 쿼터니언(Quaternion)의 회전 미분과 코리올리 힘(Coriolis Force), 원심력(Centrifugal Force)이 결합된 관성 좌표계 추방정식에는 필연적으로 사인(Sine)과 코사인(Cosine) 같은 근원적인 비선형 항(Non-linear Terms)이 파생된다.

이러한 비선형 기하학 공간에서는 가우시안 텐서(Gaussian Tensor) 형태를 띤 불확실성의 확률 밀도 함수(PDF)가 비선형 함수 블록을 통과하는 순간, 그 형태가 일그러지며 더 이상 가우시안 형태를 유지하지 못하는 위상 붕괴(Phase Collapse)가 일어난다. 따라서 LKF 코어를 자율 드론에 그대로 포팅(Porting)할 경우 예측 공분산(Prediction Covariance)이 극도로 왜곡되어 단 몇 초의 비행만으로도 필터망 자체가 파괴(Divergence)된다.

2. 테일러 급수(Taylor Series) 기반 자코비안(Jacobian) 1차 선형화 메커니즘

robot_localization 패키지의 ekf_node는 이 비선형 함수의 장벽을 타파하기 위해 확장 칼만 필터(Extended Kalman Filter, EKF) 알고리즘을 구동한다. EKF의 핵심 수학적 트릭은 임의의 현재 상태 지점(X_t)을 중심으로 비선형 동역학 함수 f(x)를 테일러 급수 전개(Taylor Series Expansion)하는 데 있다.

연산 성능 확보를 위해 2차 미분(Hessian) 이상의 고차항 벡터는 모조리 절단(Truncation) 시켜버리고, 오직 1차 미분 행렬인 자코비안(Jacobian Matrix) F_t = \frac{\partial f}{\partial x} 와 관측 자코비안 행렬 H_t = \frac{\partial h}{\partial x} 만을 추출한다. 이 찰나의 미세 시공간(Micro-interval) 안에서 비선형 곡선을 거대한 평면으로 근사(Approximation)해 버린 뒤, 여기서 도출된 선형 계수 행렬들을 고전 LKF 행렬 공식에 대입하여 상태 업데이트 증분(Delta)을 획득하는 것이 EKF의 본질적 투영(Projection) 원리이다.

3. 고기동(High-Dynamic) 환경 하 절단 오차(Truncation Error) 누적과 필터 발산

그러나 자코비안 기반 1차 궤적 선형화 아키텍처는 에이전트의 제어 곡률반경이 완만할 때만 제한적으로 유효하다. 드론이 전력 대쉬를 하거나 회피 기동을 위해 롤(Roll)과 피치(Pitch)를 급격하게 뒤틀 경우(High-Dynamic Maneuvering), 해당 시동 공간의 비선형성은 극한으로 치솟는다.

이러한 고각 회전 환경에서는 테일러 급수 전개 시 무시해버렸던 2차 이상의 고차항 누락분, 즉 절단 오차(Truncation Error)의 영향력이 기하급수적으로 폭발하게 된다. 예측된 가우시안 평균(Mean)은 비선형 함수 궤적 밖으로 튕겨져 나가고, 공분산 타원체(Covariance Ellipsoid)는 실제 상태 확률을 감싸지 못하는 치명적 과소평가(Under-estimation)를 초래한다. 결국 EKF의 칼만 이득(Kalman Gain) 연산 루프는 GPS나 IMU 관측치 센서들의 오차 분산과 실제 동역학을 서로 부정확하게 교차시키며, 궤적 추정치가 공중분해(Diverge) 되어 드론이 자신의 위치를 영구적으로 상실하는 수학적 특이점(Singularity)에 도달한다.

4. 공분산 팽창(Covariance Inflation) 튜닝과 EKF 응용의 한계론

대부분의 로봇 엔지니어들은 이 절단 오차 발산을 회피하고자, ekf_node의 프로세스 노이즈 공분산(Process Noise Covariance, Q) 파라미터 행렬을 의도적으로 부풀리는 ‘공분산 팽창(Covariance Inflation)’ 튜닝을 상습적으로 자행한다. 수학적 누락분을 인위적인 불확실성 노이즈로 둔갑시켜, 필터가 자체 동역학 추정치를 불신하고 센서 관측 데이터(Z_t)에 과도하게 의존하게 만드는 임시 처방(Hack)이다.

만약 이 처방이 도입되면, robot_localization 엔진은 진동성 노이즈가 강한 원시 센서값의 파동을 그대로 추정 궤적에 투영(Reflection)시키며, 부드러워야 할 (Smoothed) 경로 상쇄 필터로서의 근본 목적을 상실해버린다. 따라서 \mathcal{O}(d^3) 수준의 훌륭한 연산 효율(Computational Cost)을 자랑하는 EKF 임에도 불구하고, 드론처럼 동역학적 비선형 대역폭(Bandwidth)이 넓은 비행 에이전트에는 근본적이고 수학적인 구조적 한계(Structural Limitation)가 내포되어 있음을 설계 이론 단에서 엄밀히 규정해야 한다.
