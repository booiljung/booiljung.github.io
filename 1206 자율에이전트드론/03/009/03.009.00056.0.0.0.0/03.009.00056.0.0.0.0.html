Chapter 56. nav2_controller와 nav2_planner 기반 복합 로스(Loss) 최적 회피 경로 생성

1. 개요

ROS 2(Robot Operating System 2)의 자율 주행 아키텍처인 Nav2(Navigation 2) 스택에서 경로 생성 및 주행 추종 기능은 주로 nav2_planner 노드와 nav2_controller 노드를 통해 수행된다. nav2_planner는 전역 비용 맵(Global Costmap)을 바탕으로 로봇의 현재 위치에서 목표 지점까지의 최적 경로를 탐색하는 전역 경로 계획(Global Path Planning) 알고리즘을 수행한다. 반면 nav2_controller는 nav2_planner가 생성한 전역 경로를 추종하기 위해 로컬 비용 맵(Local Costmap)을 참조하여 로봇의 운동학적 제약 및 동적 환경 변화에 대응하는 국지적 제어 명령(Local Path Control Command)을 도출한다. 복합 로스(Loss) 최적화는 이 두 노드가 산출하는 궤적 산포에서 위치 오차 및 방향 오차를 최소화하고, 정적 및 동적 장애물과의 충돌 확률을 제어 논리에 결합하여 가장 안전하고 효율적인 속도 명령(Twist)을 생성하는 메커니즘을 의미한다.

2. nav2_planner: 전역 경로 탐색 모델링

nav2_planner는 주로 SmacPlanner 프레임워크나 NavFn을 기본 플러그인으로 활용하여 목표 지점까지의 최적의 위상학적 궤적을 연산한다. 검색 알고리즘으로는 2D A*, Hybrid-A*, State Lattice 알고리즘 등이 적용 가능하며, 목적 함수(Objective Function)의 평가를 통해 탐색 정점의 누적 비용을 구한다.

전역 경로의 최적해 도출은 그래프 탐색 문제로 정의된다. 정점 집합을 \mathcal{V}, 간선 집합을 \mathcal{E}라 할 때, 임의의 경로 P = \{v_0, v_1, \dots, v_N\}에 대한 누적 경로 비용 C(P)는 다음과 같다.

C(P) = \sum_{i=0}^{N-1} c(v_i, v_{i+1}) + \sum_{i=1}^{N} \mathcal{K}(v_i)

여기서 c(v_i, v_{i+1})은 인접 노드 간의 이동 거리 비용을 의미하며, \mathcal{K}(v_i)는 전역 비용 맵의 그리드 배열에서 추출된 장애물 및 위협 지수의 페널티 함수이다. 알고리즘은 휴리스틱 함수 h(v)를 도입하여, 예상 잔여 비용을 \hat{f}(v) = g(v) + h(v)로 추정함으로써 전체 탐색 공간의 분기 계수(Branching Factor)를 억제하고 연산 효율을 증대시킨다. 이에 따라 nav2_planner는 \operatorname{argmin}_{P} C(P)를 만족하는 일련의 형태소 곡선 점(Curve Points) 배열 표상을 반환한다.

3. nav2_controller: 국지 궤적 제어 및 동작 최적화 플러그인

nav2_controller 행동 노드는 수신된 전역 궤적 추이를 실시간 단위로 분절시켜 로컬 궤적 평면에 투영한다. 이 과정에서 대표적으로 사용되는 플러그인은 DWB(Dynamic Window Approach Local Planner)와 MPPI(Model Predictive Path Integral)가 있다. DWB 알고리즘의 경우, 동적 윈도우(Dynamic Window) 기법을 사용해 다음 제어 주기 내 도달 가능한 로봇의 속도 공간(Velocity Space)을 정의한다.

국지 경로 후보군 중 최적 제어 명령은 다음과 같은 비용 함수(Cost Function) \mathcal{J}의 선형 결합을 통해 평가된다.

\mathcal{J}(v, \omega) = \alpha \cdot \text{path\_distance}(v, \omega) + \beta \cdot \text{goal\_distance}(v, \omega) + \gamma \cdot \text{obstacle\_cost}(v, \omega)

계수 \alpha, \beta, \gamma는 가중치 매개변수이며, 비용이 최소화되는 제어 벡터 집합 (v^*, \omega^*) = \operatorname{argmin}_{(v,\omega)} \mathcal{J}(v,\omega)가 cmd_vel 토픽을 통해 하부 하드웨어 계층에 하달된다.

MPPI 플러그인의 적용 시에는 제어 입력을 확률 미분 방정식(Stochastic Differential Equation)의 샘플링 난수로 가정하여 대규모 병렬 GPU 계산 기반 몬테카를로 채점(Monte Carlo Scoring) 모델을 차용한다. MPPI 구조에서는 운동학 체계 내 모든 시간 보폭(Time Step)에서 예측되는 분포 오차를 정보 이론적 손실 함수(Information Theoretic Loss Function)로 변환해 기댓값을 조율한다.

3. 복합 로스(Loss) 최적 회피 경로 생성 기준

복합 로스 기반 최적 회피 전략은 단순히 경로를 이탈하고 복귀하는 과정을 넘어, 두 경로 탐색 노드가 발생하는 계층적 지연(Hierarchical Latency) 및 공간 불치성(Spatial Discrepancy) 오차를 융합하는 데 학술적 목적을 둔다. 관측된 국지적 센서 데이터에 의한 돌발 장애물 인식 시, 국지 궤적 제어 루프의 로스 함수 \mathcal{L}_{total}은 다음 항들로 재구성되어야 한다.

\mathcal{L}_{total} = \mathcal{L}_{track} + \lambda \mathcal{L}_{static\_obs} + \mu \mathcal{L}_{dyn\_pred} + \nu \mathcal{L}_{smooth}

항 (Terms)물리적 의미와 영향 (Physical Meaning and Impact)계산 특성 모델링 (Computational Characteristics Modeling)
\mathcal{L}_{track}전역 경로로부터의 위상적, 기하학적 이탈 반경 페널티 제한 함수이다.목표 벡터 투영 잔차 연산 : \sqrt{\Delta x^2 + \Delta y^2}
\mathcal{L}_{static\_obs}정적 환경의 물체와 유효 로봇 폴리곤 외경 간 상호 참조 충돌 지수이다.비용 맵의 인플레이션 반경 내 지수적 증가 모멘텀 평가값이다.
\mathcal{L}_{dyn\_pred}필터 구조로 추정된 동적 객체의 미래 궤적과 로봇의 예측 투영 위치 중첩 밀도이다.공분산 타원체(Covariance Ellipsoid) 교차 면적 비례 : \vert \Sigma_{robot} \cap \Sigma_{obj} \vert
\mathcal{L}_{smooth}이동 객체의 속도, 가속도 및 곡률 기저의 불연속성을 나타내는 저크(Jerk) 제한 항이다.시스템 위상 공간의 기하학적 미분 평면 도함수 패널티 집합이다.

동적 객체 충돌 예측 비용 연산 부기 \mathcal{L}_{dyn\_pred} 내의 교차 영역 면적 치환항 \vert \Sigma_{robot} \cap \Sigma_{obj} \vert는 실존 유체 저항 혹은 기하 역학적 관측 벡터에 따라 비선형 차수를 지닌 확장을 지원한다. nav2_controller는 위 \mathcal{L}_{total}이 최소화 대역(Minimization Band)에 진입하도록 각속도의 회전 중심 스티어링 강도 및 선가속도의 배분 비율을 매력 구조 기반으로 최적화한다. 다변량 매개 변수를 포용하는 이 최적 회피 경로 생성 기법은 예측 불가능한 잡음 요인과 돌발 객체 진입이 빈번하게 교차되는 주행 환경 하에서, 에이전트 시스템(Agent System)의 실시간 주행 무결성(Self-Integrity)을 보존하는 데 핵심적인 기저 알고리즘 단위로 작용한다.
