50.5 다중 분포(Multi-Layer) 포인트의 2차원 평면 압축(Compression) 간 공간 정보 손실 모델 및 오차 범위 추정

1. 3D-2D 투영에 의한 기하학적 정보 손실의 본질

다채널 3D LiDAR는 수직 각도(Elevation Angle) 단위로 정렬된 다수의 레이저 발진기(Multi-Layer Emitters)를 통해 3차원 공간의 깊이 정보(Depth Information)를 계층적으로 획득한다. 이러한 다중 분포(Multi-Layer) 포인트 클라우드 데이터를 pointcloud_to_laserscan 노드를 통해 2차원 극좌표계 스페이스로 투영(Projection)하는 과정은 정보 이론적(Information Theoretic) 관점에서 본질적인 비가역적 데이터 압축(Irreversible Data Compression) 연산에 해당한다. 하나의 2D 각도 인덱스 타겟에 복수의 3D 레이어가 할당되면서 발생하는 Z축 역학 공간의 소거 현상은 제어 플래너 입장에서 장애물의 실제 체적(Volume) 및 수직적 형상 정보의 완전한 상실을 초래한다.

2. 복수 레이어 매핑 시 발생하는 심도 붕괴(Depth Collapse) 역학 모델

min_height와 max_height 파라미터로 구획된 관측 창(Viewing Frustum) 내에 존재하는 임의의 3D 객체를 가정할 때, 이 객체의 표면에서 반사된 다중 링(Ring)의 포인트 집합 P_{obj} = \{p_1, p_2, \dots, p_k\}가 동일한 수평 투영 각도 \theta_j를 공유한다고 모델링하자. 이때 압축 노드의 병합 알고리즘은 안전 충돌 마진 극대화를 위해 집합 P_{obj} 요소들 중 센서 원점으로부터 가장 짧은 유클리드 거리 표본 r_{min} = \min(\|p_i\|) 만을 스캔 격자의 대표 반경으로 취한다.

이러한 최단 거리 선택 함수 연산 직후, r_{min}을 제외한 나머지 k-1개의 공간 데이터는 즉시 메모리상에서 폐기된다. 이를 심도 붕괴(Depth Collapse) 현상이라 칭한다. 예를 들어 전방에 기울어진 벽이 존재할 경우, 드론 센서에 물리적으로 가장 가까운 벽의 최하단 꼭짓점 거리만이 탐지되며, 벽의 경사도나 상단 부분까지의 거리에 대한 증감률 분포 데이터는 플래너에 전혀 전달되지 않는다. 이는 장애물 표면 곡률 인식을 원천적으로 차단한다.

3. 평면 압축에 따른 투영 오차(Projection Error) 추정 방정식

3D 원점으로부터 반경 r_{3D} 에 위치하며 수직 각도 \phi (수평면 기준 앙각/내각) 인 지점에 위치한 포인트 A(x, y, z) 를 가정하자. 이 포인트를 2차원 x-y 평면에 정사영(Orthogonal Projection) 할 경우 투영된 가상의 2D 반경 거리 r_{2D} 는 다음과 같은 코사인(Cosine) 비례 모델에 의해 실제 반사 거리보다 축소 기록되는 기하학적 왜곡을 갖는다.

r_{2D} = \sqrt{x^2 + y^2} = r_{3D} \cos(\phi)

여기서 측정 오차 범위 \epsilon = r_{3D} - r_{2D} 는 \epsilon = r_{3D} (1 - \cos(\phi)) 로 수렴한다. 다채널 센서의 위/아래 가장 외곽 레이어(최대 절대값 \phi)에서 추출된 포인트가 장애물 최단 거리 r_{min} 으로 대표될 확률이 발생할 경우, 센서는 실제 반사된 유클리드 스펙트럼 거리(r_{3D})보다 더 짧은 가상의 직교 투영 거리(r_{2D})를 제어기에 보고하게 된다. 이는 로봇 입장에서는 장애물이 예상보다 근소하게 더 가까이 있는 것으로 오판하게 만드는 음의 바이어스(Negative Bias) 오차를 생산하며 충돌 위험을 과대평가(Overestimation) 하는 결과를 가져온다.

4. 제어 한계(Control Limits) 예측 및 공간 정보 모델 패러다임

상술한 심도 붕괴로 인한 형상 결손 및 정사영 기하 오차 \epsilon 의 누적은, 드론의 Nav2 기반 국지 경로 스택(Local Trajectory Stack)이 장애물 사이의 3차원적 틈새(3D Free-space Passages, 예: 터널 구조, 아치형 테이블 아래)를 안전하게 통과할 수 없도록 보수화시키는 주원인이 된다.

결과적으로 다중 분포 포인트 클라우드의 2차원 이산 평면 압축 파이프라인은 시스템의 통신 대역폭과 제어 지연 시간(Latency)의 수 백 배 이상의 극단적 이득을 보장하는 대역폭 최적화(Bandwidth Minimization) 장치이지만, 반대 급부로 이동체 로봇의 3D 항법의 완전성(Navigation Completeness)을 달성할 수 없는 근원적 한계치를 동반한다. 따라서 이를 극복하기 위해 sensor_msgs/LaserScan 을 주력 회피 센서로 운용하되, 복잡한 통로와 수직 제약 구역에서는 VoxelGrid 등으로 별도 병렬 압축된 원시 3D 점군 데이터를 보조 코스트맵(Costmap) 계층에 선택적으로 투입하여 결속하는 이중 센서 퓨전(Dual Sensor Fusion) 모델링이 요구된다.
