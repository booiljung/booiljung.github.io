50.4 극좌표계(Polar Coordinate) 평면 투영 후 반사파 각해상도(Angle Resolution) 보간(Interpolation) 및 해상도 최적화 이론

1. 평면 투영에 따른 극좌표계 위상 왜곡 및 해상도 불균형성

다채널 3D LiDAR는 수평(Azimuth) 및 수직(Elevation) 방향으로 고유한 각해상도(Angular Resolution)를 가지며 나선형 또는 병렬 링 형태로 물리 공간을 스캐닝한다. 이러한 3차원 점군 데이터가 Z축 임계값 필터를 거쳐 2D 유클리드 평면 공간으로 사상(Mapping)된 후 \theta = \arctan2(y, x) 공식을 통해 2D 극좌표계(Polar Coordinate)로 재해석될 때, 원시 모델이 지녔던 수평 및 수직 해상도의 위상 기하학적 연속성은 파괴된다.

구체적으로 거리 r이 센서 원점으로부터 멀어질수록 동일한 수평 해상도 \Delta\theta_{3D} 가 커버하는 선형적인 아크(Arc) 길이는 비례하여 증가하며, 3D 공간 상에서 층별(Ring)로 미세하게 어긋난 레이(Ray)들이 단일 평면으로 붕괴(Collapse)될 경우 평면 스위프 상의 반사파 분포밀도는 센서와의 거리에 따라 비선형적이고 고도로 불균형한 밀집점(Dense Spots)과 공동(Empty Voids) 현상을 동시에 발생시킨다.

2. LaserScan의 정적 각해상도 격자(Angular Grid) 모델링

ROS 2의 sensor_msgs/msg/LaserScan 인터페이스는 가변 밀도를 허용하는 포인트 클라우드와 달리, \text{angle\_min}에서 \text{angle\_max}까지의 전체 부채꼴 관측 영역을 \text{angle\_increment}라는 일정한 델타(Delta) 크기의 정적 이산 격자망(Static Discretized Grid) 구조로 분할한다.

목표 출력 격자의 총 버퍼(Bin) 개수를 M이라 정의하면, 임의의 투영된 공간 지점 p(r_i, \theta_i)가 매핑되어야 할 배열 내 인덱스(Index) j는 다음과 같은 라운딩(Rounding) 편의 기법으로 산출된다.
j = \lfloor \frac{\theta_i - \text{angle\_min}}{\text{angle\_increment}} \rfloor

이 정적 격자 모델의 가장 큰 맹점은, 3D 투영밀도가 낮거나 객체가 존재하지 않아 투영된 레이가 할당되지 못한 공동 인덱스(Empty Bin)의 존재이다. ROS 2 스펙에서는 이러한 빈 공간을 명시적으로 무효화된 상태(일반적으로 NaN 또는 +Inf)로 처리하여 맵핑 알고리즘이 빈 구역을 프리 스페이스(Free Space)로 인지하도록 유도한다.

3. 격자 보간(Interpolation)의 부재 및 최단 거리 병합(Merge) 규칙

일반적인 컴퓨터 비전의 이미지 다운샘플링(Down-sampling) 파이프라인에서는 픽셀이 결측될 경우 인접 픽셀의 값을 활용하는 쌍선형 보간(Bilinear Interpolation)이나 쌍입방 보간(Bicubic Interpolation)이 널리 쓰인다. 그러나 자율 주행의 pointcloud_to_laserscan 차원 축소 이론에서는 원시 LiDAR 데이터에 대한 공간 보간법을 엄격히 배제한다.

센서 관측 지점 사이의 빈 공간에 대해 임의로 반사파 거리를 추정(Interpolate)할 경우 존재하지 않는 가상의 유령 장애물(Phantom Obstacle)이 생성되어 로컬 경로 계획의 제어 발산(Control Divergence)을 초래하기 때문이다. 보간이 배제된 대신 다수의 포인트가 단일 인덱스 j에 매치될 경우 충돌 회피라는 제어론적 보수성에 입각하여 오직 가장 0에 가까운 거리 변수 r_{min} 만을 대표값으로 취점하는 최단 거리 병합 함수(Minimum Range Merge Function)만이 작동한다.

4. 해상도 튜닝(Resolution Tuning) 및 알고리즘 최적화 역학

angle_increment 파라미터 \Delta\theta_{2D} 설정은 시스템 연산 부하와 환경 인식 정밀도 사이의 직접적인 비용 함수(Cost Function)로 환산된다. \Delta\theta_{2D}를 3D 원시 센서의 방위각 해상도 \Delta\theta_{3D}보다 세밀(Super-resolution)하게 설정할 경우, LaserScan 메시지의 배열 크기 M 은 증가하나 실제 투영할 광선은 절대적으로 부족하여 버퍼 데이터의 대다수가 NaN으로 채워지는 결측의 낭비 역학이 발생한다.

반대로 \Delta\theta_{2D}를 지나치게 성기게(Coarse) 설정할 경우(예: 1^\circ 이상), 멀리 존재하는 작은 너비의 극기둥(Pole)이나 와이어 형태 장애물이 투영 소실(Projection Loss)에 의해 누락될 수 있다. 최적화 이론적 관점에서 볼 때 angle_increment \Delta\theta_{2D}는 대상 LiDAR 센서의 수평 수광부 해상도 분해능 스펙과 수학적으로 완전히 동조화(Synchronization)되거나, Nyquist-Shannon 샘플링 정리 논리에 따라 최소 타겟 장애물 지름이 센서 최대 인식 범위(Max Range)에 위치할 때 요구되는 이론적 각도의 0.5배율 내외로 설정하는 것이 해상도 모델의 최적점(Optimum)이다.
