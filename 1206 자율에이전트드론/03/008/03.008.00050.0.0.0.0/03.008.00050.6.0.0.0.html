50.6 동적 피사체(Dynamic Object)의 스캔 누락(Miss Rate) 현상 억제를 위한 적응적 표본 추출(Adaptive Point Sampling) 규칙

1. 2D 투영 기반 동적 피사체의 제한적 인식 및 스캔 누락 현상

pointcloud_to_laserscan 패키지를 경유한 2차원 투영 시스템은 벽, 기둥, 가구와 같은 정적 피사체(Static Object)를 인식하는 데 최적화되어 있으나, 자율 에이전트 드론 주변을 빠르게 이동하는 동적 피사체(Dynamic Object, 예: 보행자, 다른 비행체) 탐지에서는 스캔 누락(Miss Rate) 현상을 유발할 가능성이 높다. 동적 피사체는 드론의 3D LiDAR 스캔 주기(예: 10Hz, 100ms)와 차원 축소를 위한 2D 투영 및 내부 발행 파이프라인 지연 시간(Latency)이 중첩되는 짧은 순간(Transient Moment)에도 물리적 공간 점유 위치를 급격히 상실한다.

결과적으로 피사체의 체적이 작고 각속도가 빠를수록, 이산화된 2D 각해상도(Angular Resolution Grid) 사이에 피사체가 위치하거나, 투영을 위해 설정된 엄격한 3차원 슬라이스 윈도우(Z_{min} \sim Z_{max})를 위아래로 일시 이탈함으로써 로컬 통제기의 레이저 스캔 배열에서 관측되지 않는 심각한 인지 불연속성 현상을 야기한다.

2. 시공간 주파수(Spatio-Temporal Frequency)와 동역학 이탈 모델

동적 객체의 스캔 누락을 정량화하기 위해 대상의 횡방향 상대 속도(Relative Lateral Velocity)를 v_{lat}, 센서 관측 반경을 r_{obj}, 레이저 스캐너의 스위프 주파수(프레임 레이트)를 f_s 라고 정의하자. 한 스윕(Sweep)에서 다음 스윕 사이 주기 \Delta t = 1/f_s 동안 동적 객체가 이동하는 각도 변위 \Delta \theta_{obj} 는 대략 \Delta \theta_{obj} \approx (v_{lat} \cdot \Delta t) / r_{obj} 가 된다.

만약 이 각도 변위가 레이저 스캔의 최소 이산 각도 구역인 angle_increment(\Delta\theta_{2D}) 보다 크게 발생하면서(\Delta \theta_{obj} \gg \Delta\theta_{2D}), 해당 피사체 표면의 반사 지점이 적거나 희소 밀도(Sparse Density)를 가질 경우, 다음 시점 프레임에서 해당 물체의 데이터 배열 상 인덱스가 기존 연속성을 완전히 상실하고 점핑(Jumping)하게 된다. 이러한 고주파 위상 변화는 시간적 프레임 간 맵핑을 시도하는 TEB 플래너(Timed Elastic Band Planner) 등에서 객체를 “사라진 개체” 처리 후 “새로운 개체“로 등록하도록 강제하며 회피 궤적 계산의 붕괴(Thrashing)를 유발한다.

3. 누락 억제를 위한 적응적 표본 추출(Adaptive Point Sampling) 규칙

정적 격자망의 스캔 누락 한계를 극복하기 위해 pointcloud_to_laserscan 시스템 아키텍처 내측 또는 직전의 전처리 단계에서 적응적 표본 추출(Adaptive Point Sampling) 규칙이 요구된다. 이는 무조건적인 최소 거리(Minimum Range) 단순 매핑 규칙을 조건부로 확장하는 것이다.

  1. 동적 밀집도 샘플링(Density-based Sampling): i번째 2D 분할 격자(Bin)에 매핑되는 3D 포인트의 개수 밀도 함수(Density Function)를 계산하고 밀도가 특정 편차 이하로 급감하는 클러스터(Sparse Cluster)에 가중치를 부여한다. 이는 급격히 이동하며 형태가 분산되는 동적 객체의 파편적 포인트라도 2D 스캔 상의 유효 거리로 강제 배정함으로써 데이터 누락을 줄인다.
  2. 형상 보존형 모폴로지 팽창(Morphological Dilation): 한 번 투영된 1차원 LaserScan 배열 구조 내에서 인접한 비어있는 인덱스(NaN 또는 매우 먼 거리)에 대하여, 감지된 객체의 외곽을 인위적으로 가상 팽창(Dilation)시켜 \Delta \theta 측면의 객체 두께를 확장한다. 이로써 단일 픽셀 수준의 미세한 동적 객체의 횡방향 움직임이 다음 프레임에서 연속된 인덱스 영역 안으로 포섭되도록 마진을 부여한다.
  3. 높이 여유분(Height Margin)의 비대칭 개방: 동적 환경에서는 보행자의 팔 뻗음, 새의 날갯짓 등 찰나의 높이 변화가 빈번하다. Z축 임계값을 완전히 타이트하게 고정하는 대신 동적 장애물이 주로 발생할 수 있는 특정 높이 스펙트럼(예: 드론과 동일한 수평선 구역)에 대해 상하 여유 마진을 추가 부여하여 부분적인 형태 잃음(Shape Truncation)으로 인한 통째 누락을 방지한다.

4. 로컬 플래너(Local Planner)의 회피 성능 강화 역학

적응적 표본 추출 파이프라인을 거친 2D 레이저 스캔 배열은 원시 3D 센서가 제공하는 시공간적 역학을 보수적인 연속성 모델로 포장(Wrapping)한 결과이다. 이러한 렌더링 최적화를 통해 차원 축소 모듈의 결과물은 일시적 노이즈(Flickering)와 누락율(Miss Rate)이 감소된 형태의 안정적인 1차원 프로파일을 제공하게 된다.

견고해진 sensor_msgs/LaserScan 인터페이스는 동적 궤적 플래닝 엔진(Dynamic Trajectory Planning Engine)의 오토마타(Automata) 상태 변환기에서 상태 전이 실패율을 대폭 하향시키며, 엣지 컴퓨팅 기반의 군집 로봇이나 동선 교차가 빈번한 복합 환경 내에서 자율 드론의 능동적 충돌 회피(Active Collision Avoidance) 성공 확률을 보장학적으로 증폭시킨다.
