60.2 OMPL(Open Motion Planning Library) 기반 상태 공간 샘플링(RRT, PRM) 알고리즘의 수렴성(Convergence) 보장

1. 개요 및 학술적 배경

OMPL(Open Motion Planning Library)은 고차원 상태 공간(State Space)에서의 모션 플래닝을 위한 오픈 소스 라이브러리다. MoveIt 2 프레임워크 내에서 기본 플래너(Default Planner)로 채택되어 다관절 매니퓰레이터(Manipulator) 및 자율 이동 에이전트의 충돌 회피 궤적을 생성하는 데 핵심적인 역할을 수행한다. 샘플링 기반 알고리즘(Sampling-based Algorithm)은 고차원 형상 공간(Configuration Space, C-Space)을 명시적으로 구성하는 대신, 무작위 샘플링을 통해 유효한 경로를 탐색하는 확률적 접근법을 취한다. 이러한 방식은 RRT(Rapidly-exploring Random Tree) 및 PRM(Probabilistic Roadmap)과 같은 알고리즘으로 대표되며, 점근적 최적성(Asymptotic Optimality)과 해의 완성도(Probabilistic Completeness)라는 중요한 수학적 성질을 갖는다. 본 절에서는 OMPL 아키텍처 내에서 RRT와 PRM 알고리즘의 이론적 배경과 수렴성(Convergence) 보장 메커니즘을 분석한다.

2. 확률적 완전성(Probabilistic Completeness)과 점근적 최적성(Asymptotic Optimality)

샘플링 기반 모션 플래닝 알고리즘의 핵심적인 수학적 특성은 확률적 완전성과 점근적 최적성으로 정의된다.

2.1 확률적 완전성 (Probabilistic Completeness)

확률적 완전성은 시작 지점과 목표 지점 사이에 유효한 경로가 존재할 경우, 샘플링 횟수 N이 무한대로 수렴함에 따라 그 경로를 발견할 확률이 1에 수렴함을 의미한다.
유효한 경로가 없는 경우에는 알고리즘이 스스로 종료되지 못하고 무한 루프에 빠질 수 있는 단점이 있으므로, 실제 구현 시에는 최대 샘플링 횟수나 최대 연산 시간(Timeout)을 매개변수로 설정하여 알고리즘을 강제로 종료한다. 표준 RRT와 PRM은 확률적 완전성을 보장하는 대표적인 알고리즘이다.

2.2 점근적 최적성 (Asymptotic Optimality)

초기 RRT와 PRM은 주어진 시간 내에 경로를 찾는 데 초점을 맞추었으며, 경로의 길이(Cost) 측면에서 최적화된 결과를 도출하지는 않는다. Karaman과 Frazzoli의 연구 “Sampling-based Algorithms for Optimal Motion Planning” (2011)에 의해 제안된 RRT와 PRM 알고리즘은 무한한 시간이 주어질 경우 샘플링 트리 또는 로드맵이 상태 공간 내의 최적 경로(Optimal Path)로 수렴함을 수학적으로 증명하였다. 이를 점근적 최적성이라 통칭한다. OMPL 내의 최적화 플래너(Optimizing Planner)들은 이러한 성질을 기반으로 동작한다.

3. RRT (Rapidly-exploring Random Tree) 및 RRT*의 수렴성 구조

RRT 알고리즘 패밀리는 고차원 연속 공간에서 단일 트리(Single-tree) 형태의 탐색 구조를 형성한다.

3.1 표준 RRT의 탐색 기법

표준 RRT는 시작 노드에서 출발하여 상태 공간 내 무작위로 샘플링된 상태(Random State)를 향해 트리를 확장한다.

  1. 빈 공간에서 무작위 상태 q_{rand}를 샘플링한다.
  2. 현재 트리에 속한 노드 중 q_{rand}와 가장 가까운 q_{near}를 찾는다.
  3. q_{near}에서 q_{rand} 방향으로 일정 스텝 크기(Step Size)만큼 이동한 지점 q_{new}를 생성한다.
  4. q_{new}가 충돌 없는 자유 공간(Free Space) 영역에 속할 경우, 트리에 노드로 추가한다.

이 구조는 공간을 빠르게 탐색(Voronoi Bias)하는 강력한 이점이 있으나, 생성된 트리의 경로가 최적해로 수렴하지는 않는다.

3.2 RRT* 알고리즘의 최적화 수렴 메커니즘

RRT*는 표준 RRT에 재배선(Rewiring) 과정과 이웃 탐색(Neighbor Search) 과정을 추가하여 점근적 최적성을 확보한다.

  1. 새로운 노드 q_{new}가 생성될 때, 해당 노드 주변의 반경 r 내에 있는 이웃 노드 집합 Q_{near}를 탐색한다.
  2. 트리의 루트에서 Q_{near} 집합 내의 노드 중 하나를 거쳐 q_{new}에 도달하는 비용(Cost)을 계산하고, 최소 비용을 가지는 노드를 q_{new}의 부모(Parent) 노드로 선정한다.
  3. 반대로 q_{new}를 거쳐 Q_{near} 집합 내의 다른 노드에 도달하는 비용이 기존 트리 구조에서의 비용보다 작아질 경우, 해당 노드의 부모를 q_{new}로 변경하는 재배선 작업을 수행한다.

탐색 반경 r은 샘플의 수 N이 증가함에 따라 \gamma (\frac{\log N}{N})^{1/d} (여기서 d는 상태 공간의 차원) 비율로 점차 감소시켜야 수렴성을 수학적으로 보장할 수 있다. OMPL에서는 이 과정을 효율적인 근접 탐색(Nearest Neighbor Search) 자료구조를 사용하여 가속한다.

4. PRM (Probabilistic Roadmap) 및 PRM* 알고리즘 구조

PRM은 오프라인 단계에서 학습(Learning Phase)과 온라인 단계에서 탐색(Query Phase)을 분리하여 수행하는 다중 쿼리(Multi-query) 프레임워크를 기반으로 한다.

4.1 PRM의 로드맵 구축

  1. 자유 공간 내에서 무작위 노드를 N개 샘플링한다.
  2. 각 노드 간의 거리를 기준 반경 이내이거나 K-최근접 이웃(K-Nearest Neighbor) 조건 하에서 로컬 경로 탐색기(Local Planner)로 연결을 시도한다. 연결이 충돌 없이 유효하면 간선(Edge)을 생성한다.
  3. 시작점과 목표점이 주어지면 두 점을 로드맵 상의 인접 노드에 연결하고 그래프 탐색 알고리즘(A* 또는 Dijkstra 등)을 통해 최단 경로를 탐색한다.

PRM 역시 확률적 완전성을 보장하지만, 연결망이 성기게 구성되었을 때에는 좁은 통로(Narrow Passage) 문제를 해결하는 데 어려움이 발생하며, 일정 수준 이상 최적해로 점근하지는 않는다.

4.2 PRM* 알고리즘의 점근적 수렴

RRT와 유사한 원리로, PRM은 노드를 연결할 때 탐색 반경 r의 크기를 샘플 개수 N에 무관한 상수가 아니라 수학적 함수로 설정한다. 반경의 임계값은 r = \gamma_{PRM}(\frac{\log N}{N})^{1/d}를 따른다.
반경을 점차 줄이되 기하급수적 비율로 축소 연산시키면, 로드맵이 무한히 밀집될 때 PRM* 그래프에서 추출된 최단 경로가 연속 공간에서의 이론적 최적 경로 길이와 확률 1로 수렴 보장성을 가진다. OMPL은 PRM을 활용할 때 다양한 지연 평가(Lazy Evaluation) 기법을 지원하여 충돌 검사(Collision Checking) 파이프라인의 병목을 회피하고 수렴 효율을 높인다.

5. OMPL 내 상태 공간 및 플래너 설정 파라미터 규격

로봇 시스템 통합 환경인 MoveIt 2 (ROS 2 Humble / Iron / Jazzy 버전 호환)에 탑재된 OMPL 플러그인(일반적으로 라이브러리 OMPL 1.5.0 이상 버전에 의존)을 기준으로 RRT* 및 PRM* 수렴성을 통제하는 파라미터 구성은 다음과 같다.

5.1 최적화 목적 함수 (Optimization Objective)

OMPL에서 수렴성의 판단 기준이 되는 ’최적(Optimal)’의 정의는 최적화 목적 함수 인터페이스를 통해 파라미터로 결합된다. 대표적으로 경로 길이 최소화 목적 함수와 장애물 간격 최대화 목적 함수가 설정될 수 있다.
RRT* 계열 수렴 플래너를 사용할 때는 플래닝이 목표 영역에 도달하더라도 최적해를 향한 수렴 연산 메커니즘을 지속 구동시키기 위해 전체 제한 시간(Planner Timeout)을 적절히 길게 허용하거나, 지정된 허용 오차 비용 한계점 이하로 경로가 도출될 때까지 연산을 반복 수행시켜야 한다.

5.2 RRT* 주요 파라미터 모델

  • range: 한 번의 델타 스텝 연산에서 트리가 확장되는 최대 간격을 정의한다. 과도하게 크게 설정할 경우 확장은 빠르지만 좁은 형상 공간 내의 정밀한 탐색 수렴이 제한된다.
  • goal_bias: 무작위 샘플링 대신 목표 상태(Goal State)를 강제 지정 샘플링할 확률값이다. 수렴 가속 속도를 높이기 위해 일반적으로 0.05 대역으로 상수를 배치한다.
  • rewire_factor: RRT* 점근 수렴성의 핵심 파라미터로, 이웃 노드를 탐색하는 반경을 계산하기 위한 상수 비례 도출 팩터 계수다.

5.3 PRM 주요 파라미터 모델

  • max_nearest_neighbors: 오프라인 로드맵 무방향 그래프 생성 시, 단일 노드 간 최대 연결 간선의 이웃 탐색 숫자를 명시적으로 제한하여 행렬 메모리 누수와 결합 폭발을 방지한다.

6. 결론

OMPL 프레임워크 내 RRT* 및 PRM* 알고리즘 패밀리는 고차원 복잡계 로봇 매니퓰레이터의 비선형 형상 모션 플래닝 해 도출과 관련하여 확률론적 기하학(Stochastic Geometry) 구조를 내재화한다. 최적해를 향한 점근적 수렴성은 이웃 노드의 재배선 및 최적화 반경의 동적 축소 기법에 대한 수학적 증명을 통해 엄밀하게 제어되며 보장된다.
