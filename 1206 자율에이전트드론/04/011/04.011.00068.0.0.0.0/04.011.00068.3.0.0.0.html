68.3 국소 좌표계(Local Frame) 기반 질량 중심(Center of Mass) 기하학적 정렬 및 관성 모멘트 텐서(Moment Tensor) 오차 보정

자율 에이전트 드론의 동역학 모델링 시뮬레이션에서 질량 중심(Center of Mass, CoM)과 관성 모멘트 텐서(Moment of Inertia Tensor)의 위치 정합성은 비행 제어 알고리즘의 안정성을 직접적으로 결정하는 가장 중요한 변인이다. CAD에서 임포트된 메시(Mesh) 파일의 기하학적 원점(Origin)이 실제 기체의 질량 중심과 일치하지 않는 경우, 물리 엔진 상에서 심각한 회전 오차가 누적되어 발산 현상을 초래한다. 본 절에서는 국소 좌표계(Local Frame) 기준으로 질량 중심을 재정렬(Re-alignment)하고, 평행축 정리(Parallel Axis Theorem)를 적용하여 이탈된 관성 텐서의 수리적 오차를 보정하는 논리를 확립한다.

1. 기하 원점(Geometric Origin)과 질량 중심(CoM)의 편차(Offset) 원리

통상적으로 설계된 프레임 메시의 원점(O_{mesh})은 조립의 편의를 위해 프레임의 바닥면, 혹은 특정 장착 홀(Mounting Hole)의 중심 등으로 임의 지정되는 경우가 많다. 이로 인해 메시 원점과 실제 질량 중심점 간에는 유의미한 변위 편차 벡터 \mathbf{d} = [d_x, d_y, d_z]^T가 존재하게 된다.

SDF(Simulation Description Format) 시뮬레이션 환경 내의 링크(Link) 구조에서 이 편차를 해결하는 방식은 메시 데이터의 원점 자체를 물리적으로 이동시키는 것이 아니라, <inertial> 태그 하위의 <pose> 요소를 활용하여 질량 중심의 새로운 로컬 프레임(Local Frame) 극좌표를 부여하는 것이다.

O_{com} = O_{mesh} + \mathbf{d}

SDF 내에서 이는 기저 프레임(Base Frame) 내 위치한 질량 중심의 오프셋 구조 매핑으로 직결되며, 물리 엔진은 병진 및 회전 역학계산을 이 O_{com} 극좌표를 기준으로 전개한다.

2. 평행축 정리(Parallel Axis Theorem)를 통한 텐서(Tensor) 오차 보정

관성 모멘트 텐서는 회전 중심축이 질량 중심을 지향할 때 가장 최소의 에너지를 지니는 주관성(Principal Inertia)을 나타낸다. 만약 CAD 툴에서 산출된 시스템의 관성 텐서 \mathbf{I}_{mesh}가 질량 중심이 아닌 기하 원점 O_{mesh}를 기준으로 도출되었다면, 물리 엔진에 입력하기 전 이를 질량 중심 기준의 관성 텐서 \mathbf{I}_{com}으로 역산하여 보정해야 한다.

질량이 m이고 변위 벡터가 \mathbf{d}인 강체의 텐서 이동에는 3차원 평행축 정리 행렬식이 적용된다.

\mathbf{I}_{mesh} = \mathbf{I}_{com} + m [(\mathbf{d}^T \mathbf{d}) \mathbf{I}_{3\times3} - \mathbf{d} \mathbf{d}^T]

여기서 \mathbf{I}_{3\times3}은 3 \times 3 단위 행렬(Identity Matrix)이다. 위 식을 전개하여 요소별 오차 보정 식을 정리하면 다음과 같다.

주관성 모멘트 오차 보정:
I_{xx, com} = I_{xx, mesh} - m (d_y^2 + d_z^2)
I_{yy, com} = I_{yy, mesh} - m (d_x^2 + d_z^2)
I_{zz, com} = I_{zz, mesh} - m (d_x^2 + d_y^2)

관성곱 모멘트 오차 보정:
I_{xy, com} = I_{xy, mesh} + m (d_x d_y)
I_{xz, com} = I_{xz, mesh} + m (d_x d_z)
I_{yz, com} = I_{yz, mesh} + m (d_y d_z)

물리 엔진은 위 식을 통하여 보정된 \mathbf{I}_{com} 행렬의 수치만을 정당한 관성 텐서 파라미터로 인정한다. 보정되지 않은 텐서를 주입할 경우 드론은 롤(Roll)과 피치(Pitch) 제어 시 예기치 않은 코리올리 힘(Coriolis Force) 및 비대칭적인 자이로스코픽 표류(Gyroscopic Drift)를 발생시킨다.

3. 국소 좌표계(Local Frame) 재정렬의 파이프라인 무결성 검증

보정된 질량 중심 및 텐서 데이터를 SDF 파이프라인에 이식한 후에는 시뮬레이션 로거(Logger) 도구를 통하여 관성 행렬 기반 검증을 거쳐야 한다. 관성 텐서 성분들이 물리적으로 무결하기 위해서는 다음과 같은 삼각 부등식(Triangle Inequality)을 만족해야 한다.

I_{xx} + I_{yy} \geq I_{zz}
I_{yy} + I_{zz} \geq I_{xx}
I_{xx} + I_{zz} \geq I_{yy}

상기 부등식 조건 중 어느 하나라도 위배된다면, 이는 해당 텐서를 지닌 강체가 유클리드 기하학 공간 내에 실제 질량체로서 존재할 수 없는 비물리적 모델(Non-physical Model)임을 의미한다. 따라서 시뮬레이터 로딩 직전 전처리 파이프라인 상에서 국소 좌표 기반 텐서 오차 보정값을 역추적하여 오입력 여부를 검증하는 루틴이 반드시 선행되어야 한다.
