라플라스 변환은 시간영역에서 정의된 함수를 복소영역으로 사상하는 강력한 도구다. 제어공학에서 선형 미분방정식을 풀이하거나 시스템의 거동을 해석할 때 매우 중요한 역할을 한다. 특히 여러 기본함수들의 라플라스 변환을 미리 숙지해 두면, 복잡한 상황에서도 효율적으로 문제를 해결할 수 있다.

아래에서는 대표적인 기본함수들에 대한 라플라스 변환을 살펴본다. 각 변환은 시간영역 함수가 주어졌을 때, 정의에 따라

\[ \begin{align} \mathcal{L}\{f(t)\} = \int_{0}^{\infty} f(t)\, e^{-st}\, dt \end{align} \]

를 계산하여 얻는다. 여기서 \(s\)는 일반적으로 복소수이며, 변환이 수렴하기 위한 조건으로 \(\Re(s)\)가 특정 값보다 커야 한다는 제약이 따른다.

상수 함수¶

상수 함수는 가장 단순한 형태다. 예를 들어 \(f(t) = 1\)의 라플라스 변환은 다음과 같이 계산된다.

\[ \begin{align} \mathcal{L}\{1\} &= \int_{0}^{\infty} 1 \cdot e^{-st}\, dt \\ &= \left[ \frac{-1}{s} e^{-st} \right]_{0}^{\infty} \\ &= \frac{1}{s} \end{align} \]

이 적분은 \(\Re(s) > 0\)에서 수렴하며, 결과는 \(\dfrac{1}{s}\)가 된다. 상수함수를 \(f(t) = C\)로 확장하면, 선형성에 따라

\[ \begin{align} \mathcal{L}\{C\} = \frac{C}{s}, \quad \Re(s) > 0 \end{align} \]

로 얻는다.

지수 함수¶

제어공학에서 가장 자주 등장하는 함수 중 하나가 지수함수다. \(f(t) = e^{at}\)에 대해 라플라스 변환을 정의에 따라 계산해 보면,

\[ \begin{align} \mathcal{L}\{e^{at}\} &= \int_{0}^{\infty} e^{at} e^{-st}\, dt \\ &= \int_{0}^{\infty} e^{(a-s)t}\, dt. \end{align} \]

이 적분이 수렴하기 위해서는 \(\Re(s) > a\)여야 하며, 실제 적분 계산은 다음과 같이 전개된다.

\[ \begin{align} \int_{0}^{\infty} e^{(a-s)t}\, dt &= \left[\frac{e^{(a-s)t}}{a-s}\right]_{0}^{\infty} = \frac{1}{s-a}. \end{align} \]

결과적으로

\[ \begin{align} \mathcal{L}\{e^{at}\} = \frac{1}{s - a}, \quad \Re(s) > a \end{align} \]

로 요약된다.

사인 함수¶

삼각함수 중 사인 함수는 진동 시스템을 해석할 때 핵심적인 예다. \(f(t) = \sin(\omega t)\)에 대해 다음을 계산한다.

\[ \begin{align} \mathcal{L}\{\sin(\omega t)\} &= \int_{0}^{\infty} \sin(\omega t)\, e^{-st}\, dt. \end{align} \]

이 적분은 부분적분이나 복소지수 치환 등을 통해,

\[ \begin{align} \mathcal{L}\{\sin(\omega t)\} = \frac{\omega}{s^2 + \omega^2}, \quad \Re(s) > 0 \end{align} \]

이라는 결과를 얻는다.

코사인 함수¶

코사인 함수 \(f(t) = \cos(\omega t)\) 역시 유사한 방식으로 계산할 수 있다.

\[ \begin{align} \mathcal{L}\{\cos(\omega t)\} &= \int_{0}^{\infty} \cos(\omega t)\, e^{-st}\, dt \\ &= \frac{s}{s^2 + \omega^2}, \quad \Re(s) > 0 \end{align} \]

사인과 코사인의 라플라스 변환 공식은 주기 신호가 포함된 미분방정식을 해결할 때 매우 유용하다.

단위 계단 함수¶

단위 계단 함수는 \(u(t)\)로 표기하며, \(t < 0\)에서는 0, \(t \ge 0\)에서는 1의 값을 갖는다. 일반적으로 라플라스 변환에서 \(t \ge 0\) 구간만을 고려하므로, \(u(t)\) 자체의 변환은 상수함수 1과 동일한

\[ \begin{align} \mathcal{L}\{u(t)\} = \frac{1}{s}, \quad \Re(s) > 0 \end{align} \]

이 된다. 하지만 실제로 더 많이 쓰이는 것은 지연된 형태 \(u(t-a)\)의 변환이다. 이는

\[ \begin{align} \mathcal{L}\{u(t-a)\} = \frac{e^{-as}}{s}, \quad \Re(s) > 0 \end{align} \]

로 표현되며, 시스템 해석에서 지연 요소를 다룰 때 매우 중요한 역할을 한다.

디랙 델타 함수¶

디랙 델타 함수 \(\delta(t)\)는 짧은 시간 동안 무한대의 값을 갖되 적분값이 1이 되도록 정의된 '일반화 함수'다. 라플라스 변환은 적분 정의를 통해 간단히 구할 수 있다.

\[ \begin{align} \mathcal{L}\{\delta(t)\} &= \int_{0}^{\infty} \delta(t)\, e^{-st}\, dt = 1. \end{align} \]

이러한 충격 함수를 이용하면 시스템의 임펄스 응답을 직접적으로 해석할 수 있다.

램프 함수¶

램프 함수는 시간에 대해 선형 증가하는 형태다. \(f(t) = t\)에 대한 라플라스 변환을 계산해 보자.

\[ \begin{align} \mathcal{L}\{t\} &= \int_{0}^{\infty} t\, e^{-st}\, dt. \end{align} \]

이 적분을 부분적분으로 구하면,

\[ \begin{align} \mathcal{L}\{t\} = \frac{1}{s^2}, \quad \Re(s) > 0. \end{align} \]

더 일반화하면 \(t^n\)에 대한 라플라스 변환은 \(\dfrac{n!}{s^{n+1}}\)가 된다.

예제 코드 활용¶

아래는 Python의 Sympy 라이브러리를 이용하여 위에서 언급한 기본함수의 라플라스 변환을 실제로 계산해 보는 예시다. 각 결과가 이론적으로 확인한 식과 일치함을 직접 살펴볼 수 있다.

import sympy as sp

t = sp.Symbol('t', real=True, nonnegative=True)
s = sp.Symbol('s', real=True, positive=True)

# 상수 함수
f_const = 1
F_const = sp.laplace_transform(f_const, t, s)
print("Laplace of 1 =", F_const)

# 지수 함수 e^(2t)
a = 2
f_exp = sp.exp(a*t)
F_exp = sp.laplace_transform(f_exp, t, s)
print("Laplace of e^(2t) =", F_exp)

# 사인 함수 sin(3t)
omega = 3
f_sin = sp.sin(omega*t)
F_sin = sp.laplace_transform(f_sin, t, s)
print("Laplace of sin(3t) =", F_sin)

# 코사인 함수 cos(3t)
f_cos = sp.cos(omega*t)
F_cos = sp.laplace_transform(f_cos, t, s)
print("Laplace of cos(3t) =", F_cos)

# 램프 함수 t
f_ramp = t
F_ramp = sp.laplace_transform(f_ramp, t, s)
print("Laplace of t =", F_ramp)

이 코드를 실행하면, 각각

  • \(\mathcal{L}{1} = \dfrac{1}{s}\)
  • \(\mathcal{L}{e^{2t}} = \dfrac{1}{s - 2}\)
  • \(\mathcal{L}{\sin(3t)} = \dfrac{3}{s^2 + 9}\)
  • \(\mathcal{L}{\cos(3t)} = \dfrac{s}{s^2 + 9}\)
  • \(\mathcal{L}{t} = \dfrac{1}{s^2}\)

의 결과를 얻을 수 있다.

이처럼 기본함수들의 라플라스 변환을 정확히 이해해 두면, 복잡한 공학 문제에서도 신속하고 정확하게 해를 구할 수 있다. 특히 제어공학에서는 전달함수 기반의 해석, 응답 분석, 시스템 안정성 평가 등에 꾸준히 적용되므로 자주 활용해 보는 것이 중요하다.
