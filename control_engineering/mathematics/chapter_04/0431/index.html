O 라플라스 변환과 Z 변환은 모두 신호 처리 및 제어 시스템 분석에서 중요한 역할을 한다. 이 두 변환은 시스템의 시간 응답을 주파수 도메인에서 분석할 수 있게 해주는 도구로, 각각 연속시간 시스템과 이산시간 시스템에 적용된다. 두 변환의 주요 차이점은 주어진 시스템이 연속시간인지 이산시간인지에 따라 다르며, 변환 방식과 사용되는 수학적 배경 또한 서로 다르다.

라플라스 변환은 연속시간 시스템의 분석에 사용되며, Z 변환은 이산시간 시스템에서 사용된다. 라플라스 변환은 복소수 영역에서 시스템의 동작을 분석하는 반면, Z 변환은 이산 신호의 분석을 위해 주로 복소평면에서 정의된다. 이 두 변환을 비교하는 것은 각 변환의 특성과 사용 방법을 명확히 이해하는 데 도움이 된다.

라플라스 변환¶

라플라스 변환은 시간 영역에서 정의된 연속 신호나 시스템을 복소평면의 \(s\)-영역으로 변환하는 기법이다. 이 변환은 주로 다음과 같은 형태로 정의된다.

\[ X(s) = \mathcal{L}\{x(t)\} = \int_0^\infty x(t) e^{-st} \, dt \]

여기서 \(x(t)\)는 시간 영역에서의 신호, \(X(s)\)는 복소수 영역에서의 변환된 신호이며, \(s = \sigma + j\omega\)는 복소평면 상의 변수이다. 라플라스 변환은 시스템이 어떻게 동작하는지 분석할 때 주파수 응답, 안정성 분석 등을 가능하게 한다. 특히, 시스템의 전달 함수는 라플라스 변환을 사용하여 시스템의 입력과 출력을 연결하는 중요한 도구로 사용된다.

라플라스 변환의 핵심적인 특징 중 하나는 복소수 \(s\)-평면에서의 극점과 영점(zeros and poles)이다. 시스템의 전달 함수는 다음과 같은 형태로 표현된다.

\[ H(s) = \frac{N(s)}{D(s)} \]

여기서 \(N(s)\)는 분자 다항식(영점), \(D(s)\)는 분모 다항식(극점)을 나타낸다. 극점은 시스템의 안정성에 중요한 영향을 미치며, 영점은 시스템의 출력에 미치는 영향을 나타낸다.

Z 변환¶

Z 변환은 이산 신호의 분석에 사용되는 변환으로, 라플라스 변환과 유사하지만 이산 시간 신호에 적합하도록 정의된다. Z 변환은 신호 \(x[n]\)을 복소수 영역의 \(z\)-변수로 변환하는 방법을 제공한다.

\[ X(z) = \mathcal{Z}\{x[n]\} = \sum_{n=0}^{\infty} x[n] z^{-n} \]

여기서 \(x[n]\)은 이산 신호, \(X(z)\)는 복소수 영역에서의 변환된 신호이며, \(z\)는 복소평면의 변수이다. Z 변환은 이산 시간 시스템의 주파수 응답, 안정성 분석 등을 가능하게 한다. Z 변환의 전달 함수 역시 라플라스 변환처럼 극점과 영점을 가지며, 이들을 통해 시스템의 안정성 및 동작을 분석할 수 있다.

Z 변환의 중요한 특징 중 하나는 \(z\)-평면에서의 극점과 영점이다. 시스템의 전달 함수는 다음과 같이 표현된다.

\[ H(z) = \frac{N(z)}{D(z)} \]

여기서 \(N(z)\)는 분자 다항식(영점), \(D(z)\)는 분모 다항식(극점)을 나타낸다. Z 변환을 사용하면 시스템이 이산적인 시간에서 어떻게 반응하는지 분석할 수 있으며, 특히 샘플링 이론, 디지털 필터 설계 등에 중요한 역할을 한다.

라플라스 변환과 Z 변환의 주요 차이점¶

라플라스 변환과 Z 변환은 공통적으로 신호를 주파수 영역으로 변환하여 시스템 분석을 가능하게 하지만, 적용되는 시스템의 시간 도메인이 다르다. 라플라스 변환은 연속 시간 시스템을 다루는 반면, Z 변환은 이산 시간 시스템을 다룬다. 이 두 변환의 주요 차이점을 정리하면 다음과 같다.

  1. 시간 도메인
  2. 라플라스 변환은 연속시간 신호와 시스템에 적용된다.
  3. Z 변환은 이산시간 신호와 시스템에 적용된다.
  4. 변환 변수
  5. 라플라스 변환에서는 변환 변수가 \(s = \sigma + j\omega\)인 복소평면을 사용한다.
  6. Z 변환에서는 변환 변수가 \(z\)인 복소평면을 사용한다.
  7. 적용 대상
  8. 라플라스 변환은 연속 신호의 해석에 주로 사용되며, 시스템의 전달 함수 및 안정성 분석에 유용하다.
  9. Z 변환은 이산 신호의 해석에 사용되며, 디지털 필터 설계와 샘플링 이론에 중요한 역할을 한다.
  10. 해석적 결과
  11. 라플라스 변환을 통해 얻은 결과는 일반적으로 연속시간 시스템의 동작을 설명한다.
  12. Z 변환을 통해 얻은 결과는 이산시간 시스템의 동작을 설명한다.

예제 코드¶

다음은 Python을 사용하여 간단한 Z 변환과 라플라스 변환을 비교하는 예제 코드이다. 이 예제에서는 scipy 패키지를 이용하여 라플라스 변환과 Z 변환을 계산한다.

import numpy as np
import matplotlib.pyplot as plt
from scipy.signal import TransferFunction, lti, dlti, dstep

# 연속시간 시스템 (라플라스 변환)
num = [1]  # 분자
den = [1, 2, 1]  # 분모
system = TransferFunction(num, den)

# 이산시간 시스템 (Z 변환)
dt = 0.1  # 샘플링 시간
num_z = [1]
den_z = [1, -0.5]
system_z = dlti(num_z, den_z, dt=dt)

# 연속시간 시스템의 시간 응답
t = np.linspace(0, 10, 1000)
t, y = lti(num, den).step(t)

# 이산시간 시스템의 시간 응답
t_z, y_z = dstep(system_z)

# 플로팅
plt.figure(figsize=(10, 5))

# 연속시간 시스템
plt.subplot(1, 2, 1)
plt.plot(t, y)
plt.title("Continuous-Time System (Laplace Transform)")
plt.xlabel('Time [s]')
plt.ylabel('Amplitude')

# 이산시간 시스템
plt.subplot(1, 2, 2)
plt.step(t_z, y_z[0])
plt.title("Discrete-Time System (Z Transform)")
plt.xlabel('Time [s]')
plt.ylabel('Amplitude')

plt.tight_layout()
plt.show()

이 코드는 연속시간 시스템과 이산시간 시스템을 각각 라플라스 변환과 Z 변환을 사용하여 시간 응답을 구하고 이를 플로팅하는 예제이다. 연속시간 시스템은 라플라스 변환을 통해 해석되고, 이산시간 시스템은 Z 변환을 통해 해석된다.
