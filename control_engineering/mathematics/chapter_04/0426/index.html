라플라스 변환을 활용한 안정도 판별은 시스템의 미분 방정식을 주파수 영역에서 해석하는 관점에서 출발한다. 시간 영역에서 복잡하게 보이는 응답을 라플라스 영역으로 변환하면 극점 분석과 전이함수(전달함수) 표현이 용이해지고, 그 결과 시스템이 가진 동특성을 보다 명확하게 이해할 수 있다. 시스템 안정도를 확인하기 위해서는 라플라스 변환에서 얻어지는 특성방정식의 근(극점)이 실수부가 음의 값을 갖는지, 혹은 적절한 영역 내에 위치해 있는지를 확인한다. 이러한 근거는 제어이론 전반에서 널리 활용되며, 고전 제어 이론에서부터 근대 및 디지털 제어 이론까지 동일한 원리로 귀결된다.

라플라스 변환을 통해 안정도를 판별하는 개념은 크게 네 가지 흐름으로 살펴볼 수 있다. 먼저 시간 영역의 미분 방정식을 라플라스 변환해 시스템의 전달함수를 구한다. 이어서 전달함수를 통해 특성방정식을 얻고, 특성방정식의 근(극점)이 어떤 영역에 존재하는지 파악한다. 그다음 이 극점들의 분포가 시스템 안정도에 어떠한 영향을 미치는지 살펴보고, 마지막으로 부분 분수 전개나 유한 시간 응답 해석 등을 통해 안정도의 구체적 형태를 분석한다. 여기서는 이 네 가지 흐름을 순차적으로 구체적으로 다루되, 숫자 리스트나 불릿 리스트 없이 단락으로 구분하여 정리한다.

라플라스 변환과 시스템 모델¶

연속시간 영역에서 선형 시불변(Linear Time-Invariant) 시스템은 주로 입력과 출력 사이의 거동을 미분 방정식 형태로 표현한다. 예를 들어 입력을 u(t)u(t), 출력(응답)을 y(t)y(t)라 할 때, 다음과 같은 선형 미분 방정식이 자주 등장한다.

\[ \begin{align} a_n \dfrac{d^n y(t)}{dt^n} + a_{n-1} \dfrac{d^{n-1} y(t)}{dt^{n-1}} + \cdots + a_1 \dfrac{d y(t)}{dt} + a_0 y(t) = b_m \dfrac{d^m u(t)}{dt^m} + b_{m-1} \dfrac{d^{m-1} u(t)}{dt^{m-1}} + \cdots + b_1 \dfrac{d u(t)}{dt} + b_0 u(t) \end{align} \]

계수 aia_i, bjb_j는 모두 실수이며, 보통 시스템의 물리적 특성이나 설계 파라미터에 의해 결정된다. 시스템 차수가 nn이고, 입력의 최고 미분 차수가 mm이라 할 때, 흔히 n≥mn \ge m인 상황을 가정한다. 이런 식으로 시간 영역에서 주어진 시스템을 직접 해석하기는 종종 복잡하지만, 라플라스 변환을 사용하면 문제를 주파수 영역으로 전환할 수 있어 해석이 크게 단순해진다.

시간 영역에서의 미분 방정식에 라플라스 변환을 적용할 때는 특정 초기 조건을 고려해야 한다. 흔히 제어 이론에서는 초기 조건이 0이라고 가정하거나, 초기 조건이 무시될 수 있는 상황을 다루는 경우가 많다. 초기 조건이 0이면, dky(t)dtk\dfrac{d^k y(t)}{dt^k}에 대한 라플라스 변환은 skY(s)s^k Y(s)로 단순화된다. 여기서 Y(s)Y(s)는 출력 신호 y(t)y(t)의 라플라스 변환에 해당하며, U(s)U(s)는 입력 신호 u(t)u(t)의 라플라스 변환이다.

시간 영역 미분 방정식을 라플라스 영역으로 변환하면 다음과 같은 형태가 된다.

\[ \begin{align} a_n s^n Y(s) + a_{n-1} s^{n-1} Y(s) + \cdots + a_1 s Y(s) + a_0 Y(s) = b_m s^m U(s) + b_{m-1} s^{m-1} U(s) + \cdots + b_1 s U(s) + b_0 U(s) \end{align} \]

좌측 항은 출력 Y(s)Y(s)를 묶어 표현할 수 있고, 우측 항은 입력 U(s)U(s)를 묶어 표현할 수 있어 다음과 같은 형태로 정리된다.

\[ \begin{align} \Bigl(a_n s^n + a_{n-1} s^{n-1} + \cdots + a_1 s + a_0\Bigr) Y(s) = \Bigl(b_m s^m + b_{m-1} s^{m-1} + \cdots + b_1 s + b_0\Bigr) U(s) \end{align} \]

이때 시간 영역에서 미분 방정식으로 표현되었던 시스템은 라플라스 영역에서 단순한 대수 방정식 꼴이 된다. 이를 통해 정의되는 전달함수 G(s)G(s)는 출력 Y(s)Y(s)와 입력 U(s)U(s)의 비로 나타나며, 다음과 같은 식으로 표현된다.

\[ \begin{align} G(s) = \dfrac{Y(s)}{U(s)} = \dfrac{b_m s^m + b_{m-1} s^{m-1} + \cdots + b_1 s + b_0} {a_n s^n + a_{n-1} s^{n-1} + \cdots + a_1 s + a_0} \end{align} \]

분모에 해당하는 다항식을 특성다항식이라고 부르고, 이 특성다항식을 0으로 만드는 ss의 해들은 극점(pole)이라 부른다. 제어 이론에서 안정도나 동특성을 논할 때는 주로 이 극점들이 복소평면 상에서 어디에 위치하는지 분석하게 된다.

단순히 시간 영역에서 미분 방정식을 푸는 대신, 라플라스 영역에서 전달함수와 극점을 살피면 시스템의 감쇠 특성, 공진 특성, 모드별 응답이 보다 명확하게 파악된다. 극점 위치가 시스템의 감쇠나 발산을 결정하는 가장 중요한 요소이므로, 복잡한 고차 방정식도 라플라스 변환을 거쳐 특성방정식을 얻게 되면 해석 절차가 간단해진다. 이러한 이유로 라플라스 변환은 선형 제어 이론에서 필수적으로 활용된다.

라플라스 변환은 초과근이나 말단 근을 찾는 과정에서도 유용하다. 초기 조건이 0이 아닐 경우에는 그 영향이 Y(s)Y(s)에 초기 조건 항으로 반영되지만, 제어 공학에서 정상상태 해석이나 안정도 판별이 주목적일 때는 대부분 영 초기 조건 상태로 가정해 단순화한다. 그러나 실제 설계나 시뮬레이션 단계에서는 초기 조건을 포함해 시스템 거동을 전체적으로 살펴보기도 하며, 이때도 라플라스 변환은 여전히 방정식 전개에 유리한 표현을 제공한다.

전달함수가 구해지면, 이를 바탕으로 여러 가지 해석 방법이 가능해진다. 예를 들어 시간영역 응답 y(t)y(t)를 직접 구하고 싶다면 부분 분수 전개(partial fraction expansion)를 통해 각 지수 항 형태로 나누어 해석할 수 있다. 모든 항이 감쇠 항인지, 혹은 발산 항이나 순환 항(진동 항)을 포함하는지를 확인하면, 시스템의 전체 동특성을 빠르게 파악할 수 있다.

라플라스 변환과 시스템 모델의 연관성을 실제 계산 예시로 살펴보면 이해가 더 빠르다. 아래는 Python에서 sympy 라이브러리를 활용해 간단한 2차 미분 방정식을 라플라스 영역으로 가져가 전달함수를 표현하고, 특성방정식을 확인하는 과정의 예시다.

from sympy import symbols, Function, laplace_transform, simplify

t = symbols('t', positive=True)
s = symbols('s', complex=True)
y = Function('y')(t)
u = Function('u')(t)

# 예시 미분 방정식: y''(t) + 3y'(t) + 2y(t) = u(t)
# a2 = 1, a1 = 3, a0 = 2
# b0 = 1 (나머지 계수는 0)
# 초기 조건을 0이라 가정

# y'(t), y''(t)의 라플라스 변환을 sympy로 표현
Y = laplace_transform(y, t, s)
U = laplace_transform(u, t, s)

# y'(t)의 라플라스 변환
Yp = s*Y  # 초기 조건 0 가정
# y''(t)의 라플라스 변환
Ypp = s**2*Y

# 방정식의 라플라스 변환
lap_eq = Ypp + 3*Yp + 2*Y - U

# 식을 간단히 정리
lap_eq_simplified = simplify(lap_eq)
print(lap_eq_simplified)

# 일반적으로 출력: (s^2 + 3s + 2)*Y(s) - U(s) = 0 형태
# 따라서 전달함수 G(s) = Y(s)/U(s) = 1 / (s^2 + 3s + 2)

이 예시는 간단한 2차 계를 예로 들었지만, 차수가 커지면 미분 방정식의 전개나 전달함수의 분자가 복잡해질 수 있다. 그럼에도 불구하고, 라플라스 변환을 통해 모든 항을 ss에 관한 다항식 형태로 만들 수 있으므로, 시스템 해석은 시간을 기준으로 한 복잡한 적분이나 미분 연산보다 훨씬 더 간결하게 수행된다.

라플라스 변환을 적용해 얻은 전달함수는 고전 제어 이론에서 대단히 중요한 역할을 한다. 닫힌루프(closed-loop) 해석, 안정도 및 응답 특성 분석, 주파수영역 해석(보데선도, 나이퀴스트 다이어그램 등) 모두 전달함수를 기준으로 이루어진다. 결국 시간영역에서 시작하는 모델링과, 이를 라플라스 영역으로 옮겨 단순 대수 방정식으로 해석하는 과정이 제어 시스템 해석의 강력한 도구가 되는 셈이다.

이처럼 라플라스 변환과 시스템 모델이 밀접하게 연관되어 있으며, 미분 방정식을 전달함수로 표현하는 기술은 안정도 및 동특성 해석, 그리고 실제 제어기 설계에서 항상 우선적으로 고려된다. 시스템이 여러 개의 입력을 가지거나, 벡터 상태 x(t)\mathbf{x}(t)로 표현되어 행렬 방정식 형태로 확장되더라도, 궁극적으로 라플라스 변환이라는 개념적 도구는 동일하게 적용 가능하다. 이는 멀티입출력(MIMO) 시스템까지 확장될 수 있으나, 그 경우 행렬 라플라스 변환이나 상태방정식 형태로 표현해야 한다. 그럼에도 기본적인 아이디어는 동일하며, 여기에서 출발해 라플라스 변환이 제공하는 다양한 분석 기법이 전개된다.

특성방정식과 극점의 안정 조건¶

특성방정식은 시간영역에서 주어진 시스템의 선형 미분 방정식을 라플라스 변환한 뒤, 그 전달함수를 구했을 때 분모에 해당하는 다항식을 말한다. 예를 들어 시스템의 시간영역 미분 방정식이 고차라면, 라플라스 변환에서 \(s\)를 변수로 하는 다항식이 분모에 등장하게 되는데 이 부분이 바로 특성다항식이다. 이 특성다항식을 0으로 만드는 해들이 극점(pole)이며, 시스템의 안정도는 이 극점들의 위치에 의해 결정된다.

특성방정식은 보통 아래와 같은 형태로 나타난다.

\[ \begin{align} a_n s^n + a_{n-1} s^{n-1} + \cdots + a_1 s + a_0 = 0 \end{align} \]

여기서 \(a_n, a_{n-1}, \ldots, a_0\)는 실수 계수이며, 시스템의 물리적 특성이나 파라미터에 의해 결정된다. 이 식을 0으로 만드는 \(s\)의 값을 \(s_i\)라고 하면, 이 \(s_i\)들은 모두 복소수 영역에서 존재할 수 있다. 그런데 연속시간 시스템에서 안정도가 성립하기 위해서는 이 \(s_i\)의 실수부가 반드시 음수여야 한다. 이 조건을 수학적으로 적어 쓰면, 모든 극점 \(s_i\)에 대해 \(\Re(s_i) < 0\)임을 의미한다.

이런 이유로 인해 라플라스 변환에서 얻어진 극점이 복소평면의 어느 영역에 위치하는지가 중요해진다. 복소변수 \(s\)를 \(s = \sigma + j \omega\)로 표현했을 때, 극점이 좌반평면(left half-plane)에 존재한다는 것은 \(\sigma < 0\)를 만족한다는 뜻이고, 이는 시간영역으로 역변환했을 때 \(e^{s_i t}\)가 지수 감쇠(decay) 항을 이루며 \(t \to \infty\)에서 0으로 수렴함을 의미한다. 따라서 모든 극점이 좌반평면에 있으면, 시스템은 결국 어느 입력이 들어오든(물론 특정 제한이 있을 수 있으나 일반적인 BIBO 안정성 조건 하에서) 출력이 발산하지 않고 유한한 값으로 수렴하게 된다.

극점 중 하나라도 우반평면(right half-plane)에 위치하면, 해당 극점은 실수부가 양수이므로 지수 발산(exponential growth) 항을 만들어낸다. 이 경우 시스템은 발산 모드를 가지므로 BIBO 안정성을 만족하지 못하고, 시스템은 불안정하다고 본다. 만약 극점이 허수축(\(\sigma=0\)) 위에 위치한다면, 이 극점이 단일근(simpler root)이냐, 아니면 다중근(multiple root)이냐에 따라 경계적 안정(marginal stability)의 여부가 달라진다. 일반적인 고전 제어 이론에서는 \(\sigma=0\)에 위치한 단일근이 순수 허수부만 가지는 경우, 지속 진동(sustained oscillation)이 나타난다고 보고, 다중근이면 시스템이 발산할 수도 있다고 해석한다.

특성방정식과 극점의 관계를 더 구체적으로 살펴보기 위해서는, 라플라스 변환을 통한 전달함수 \(G(s)\)를 분수 형태로 나타냈을 때 분모 다항식이 되는 특성다항식을 해석하는 과정이 필요하다. 예를 들어

\[ \begin{align} G(s) = \dfrac{b_m s^m + b_{m-1} s^{m-1} + \cdots + b_1 s + b_0}{a_n s^n + a_{n-1} s^{n-1} + \cdots + a_1 s + a_0} \end{align} \]

에서 분모가 \(a_n s^n + a_{n-1} s^{n-1} + \cdots + a_0\)이며, 이를 0으로 만드는 해가 극점 \(s_i\)다. 이 극점들의 분포만 정확히 알아도, 해당 시스템이 안정한지 불안정한지 빠르게 판별할 수 있다. 나아가 각 극점이 가진 실수부와 허수부를 통해 시스템의 감쇠 특성과 진동 주파수를 동시에 예측할 수도 있다.

시스템에서 안정도를 정의하는 방법은 여러 가지가 있을 수 있지만, 일반적인 선형 시불변(Linear Time Invariant) 연속시간 시스템에서의 BIBO(Bounded Input Bounded Output) 안정도는 “모든 입력이 유한한 범위에서 주어졌을 때 출력이 유한한 값으로 제한된다”라는 조건으로 이해한다. 이 BIBO 안정도를 라플라스 변환으로 연결하면, 결과적으로 극점이 좌반평면에 모두 위치해야 한다는 결론이 자연스럽게 도출된다. 왜냐하면 초기 조건이나 특수 입력이 아니라도, 우반평면에 극점이 존재하면 해당 극점의 모드가 필연적으로 발산 항을 만들어내기 때문이다.

극점의 실수부가 음수라는 조건이 시스템 전반의 감쇠를 의미하므로, 라플라스 변환에서 특성방정식을 살펴볼 때는 단순히 근의 존재 여부뿐 아니라 그 실수부가 음수인지 양수인지, 또는 0인지까지도 정확히 판별해야 한다. 특히 2차 시스템을 예로 들면, 특성방정식을

\[ \begin{align} s^2 + 2 \zeta \omega_n s + \omega_n^2 = 0 \end{align} \]

같은 표준형으로 두었을 때, 감쇠비 \(\zeta\)와 고유진동수 \(\omega_n\)로부터 근의 실수부와 허수부가 어떻게 배치되는지를 빠르게 알 수 있다. \(\zeta>0\)이면 극점들은 모두 좌반평면에 놓이고, \(\zeta=0\)이면 허수축 위에 극점이 놓이므로 영진동이 발생한다. 이처럼 단순 2차 사례만 보아도 실수부가 음수냐 0이냐 양수냐에 따라 완전히 다른 안정도 특성이 나타남을 알 수 있다.

라플라스 변환을 이용해 특성방정식을 실제로 풀어 본 뒤 극점을 계산하고, 그 결과를 통해 안정도를 판별하는 과정은 소프트웨어를 통해 쉽게 자동화할 수 있다. 예시로 Python의 sympy 패키지를 활용하는 방법을 간단히 살펴본다.

from sympy import symbols, solve

s = symbols('s', complex=True)
a2, a1, a0 = symbols('a2 a1 a0', real=True)

# 예시 특성방정식: a2*s^2 + a1*s + a0 = 0
char_eq = a2*s**2 + a1*s + a0

# 계수를 특정 값으로 설정하여 극점 구하기
char_eq_sub = char_eq.subs({a2: 1, a1: 2, a0: 3})
poles = solve(char_eq_sub, s)

print("특성방정식의 극점:", poles)

# 극점의 실수부 확인
is_stable = all([pole.as_real_imag()[0] < 0 for pole in poles])
print("안정 여부:", is_stable)

이 예제에서 \(a_2=1\), \(a_1=2\), \(a_0=3\)와 같이 설정하면, 특성방정식은 \(s^2 + 2s + 3 = 0\)이 된다. 이때 결과로 나오는 극점들은 좌반평면에 위치하거나, 허수축 또는 우반평면에 위치할 수 있다. 각 극점의 실수부가 모두 음수이면 안정한 시스템이 된다. 예시와 같은 \(2, 3\)의 조합으로는 복소수가 나타나며 실수부가 음수이므로 안정 조건을 만족한다.

정리하자면, 특성방정식은 라플라스 영역에서 시스템 동특성을 결정짓는 핵심 요소다. 이 특성방정식을 만족하는 해들이 곧 시스템의 극점이고, 극점이 좌반평면에 존재할 때 연속시간 선형 시스템은 안정하다는 것이 가장 중요한 결론이다. 극점이 허수축이나 우반평면에 존재하면 다른 양상(영진동이나 발산)이 나타나므로 불안정 혹은 경계적 안정이 된다. 이처럼 특성방정식과 극점의 안정 조건은 라플라스 변환으로 시스템 거동을 해석할 때 결코 빼놓을 수 없는 주제다.

라플라스 변환을 통한 안정도 해석의 주요 관점¶

라플라스 변환에서 특성방정식의 근(극점)이 시스템 동특성의 핵심을 결정한다는 사실은 안정도 해석에 매우 중요한 단서를 제공한다. 시간영역에서 미분방정식 형태로 표현된 시스템을 \(s\)-영역으로 옮기면, 전달함수의 분모(특성다항식)에 해당하는 극점을 찾고, 그 극점의 위치(복소평면에서의 분포)로부터 안정도를 바로 확인할 수 있다. 모든 극점의 실수부가 음수이면 시스템은 감쇠응답을 보이며 안정하다고 결론짓고, 하나라도 양수 실수부를 가지면 발산 모드가 존재하므로 불안정하다. 실수부가 0인 극점이 존재할 경우에는 경계적 안정 여부를 추가로 분석해야 한다.

라플라스 변환 관점에서의 안정도 해석은 크게 두 가지 측면에서 중요성이 두드러진다. 하나는 특성방정식의 근을 직접 분석해 응답 형태를 예측하는 것이고, 다른 하나는 시간영역 응답을 부분 분수 전개나 역라플라스 변환으로 복원해 극점의 영향을 구체적으로 해석하는 것이다. 이 두 측면을 종합하면, 단순히 “안정/불안정”을 구분하는 데 그치지 않고, 실제 시스템 응답에 어떤 모드가 나타나는지, 그리고 각 모드가 얼마나 빠르게 감쇠하는지(또는 발산하는지)를 알 수 있다.

시간영역에서 보기엔 복잡하더라도 라플라스 변환으로 옮기면, \(\exp(s t)\) 형태의 지수항이 합성되어 나타난다고 해석할 수 있다. 극점이 모두 좌반평면에 있으면 \(\exp(s_i t)\)에서 모든 \(s_i\)의 실수부 \(\Re(s_i)\)가 음수이므로 \(t \to \infty\)에서 응답이 0으로 수렴한다. 이를 제어 이론에서는 내부 안정성, 혹은 BIBO(Bounded Input Bounded Output) 안정성이라고 부른다. 반면 우반평면에 극점이 하나라도 있으면 그 항이 지수적으로 발산하여 전체 응답이 한없이 커지는 결과를 낳는다.

라플라스 변환에서 전달함수 \(G(s)\)는 분모와 분자 두 부분으로 나누어 볼 수 있다. 분모는 특성다항식이며, 이를 0으로 만드는 근들이 바로 극점이다. 영점이 분자에서 결정되지만, 안정도 자체는 극점이 지닌 시간상수(실수부)의 부호에 의해 크게 좌우된다. 예를 들어 영점이 우반평면에 있더라도 분모의 극점이 모두 좌반평면에 놓이면 여전히 BIBO 안정으로 본다. 물론 영점이 극점에 상당히 근접하거나 일치하는 경우 응답 모드가 상쇄되거나 강화되는 현상이 나타날 수 있지만, 시스템 전체가 발산하지 않는 한 기본적인 안정 여부에는 직접적인 변화가 없다.

라플라스 변환을 통한 안정도 해석은 부분 분수 전개로부터 명확히 드러난다. 전달함수 \(G(s)\)를 부분 분수 형태로 표현하면, 각 극점에 대응되는 항이 지수 응답을 결정한다. 예를 들어 간단히 시스템의 전이함수를

\[ \begin{align} G(s) = \dfrac{\alpha_1}{(s - p_1)} + \dfrac{\alpha_2}{(s - p_2)} \end{align} \]

처럼 가정하고, 이를 역라플라스 변환하면

\[ \begin{align} g(t) = \alpha_1 e^{p_1 t} + \alpha_2 e^{p_2 t} \end{align} \]

가 된다. 여기서 \(p_1, p_2\)는 극점이며, 그 실수부가 음수이면 지수 감쇠가 일어나고, 양수이면 지수 발산이 일어난다. 따라서 각 항이 나타내는 동특성 모드가 안정/불안정을 결정한다고 볼 수 있다.

특성방정식의 근이 실제로 어디에 위치하는지를 찾는 과정에서는 고차 시스템의 경우 직접 해를 구하기가 까다로울 수 있다. 그렇기 때문에 라플라스 변환만으로 단번에 극점을 구하지 못하는 상황에서는 루스-후르비츠(Routh-Hurwitz) 판별법이나, 극점의 이동 경로를 시각화하는 루트 궤적(root locus) 등 다른 보조 해석 수단도 활용한다. 그럼에도 불구하고, 그 근본 원리는 “라플라스 변환에서 분모가 되는 특성다항식을 0으로 만드는 근이 복소평면에서 어느 위치에 있는가”를 파악하는 것에 있으며, 이를 통해 시간영역에서의 지수 항을 재구성해 안정도를 결정짓는 과정을 쉽게 이해할 수 있다.

이러한 분석 과정에서 라플라스 변환의 추가적인 성질도 유용하게 쓰인다. 예를 들어 최종값 정리(final value theorem)를 통해 시간 \(t \to \infty\)에서의 응답 값을 직관적으로 추론할 수 있다. 특성방정식이 모두 좌반평면에 놓여 있지 않으면, 최종값 정리는 성립하지 않거나, 설령 형식적으로 적용해도 무한대 발산 혹은 미정 형태가 되어 시간영역 응답이 안정적으로 수렴하지 않는다. 이는 곧 극점의 배치와 최종값의 존재 여부가 밀접한 관련이 있음을 뜻한다.

라플라스 변환을 통한 안정도 해석이 실질적으로 어떻게 쓰이는지를 Python 예시 코드로 간단히 살펴볼 수 있다. 아래 예시는 2차 시스템의 특성방정식을 정의하고, 그 극점을 구한 뒤 안정 여부를 판단하는 과정을 보여준다.

from sympy import symbols, solve

s = symbols('s', complex=True)
a2, a1, a0 = symbols('a2 a1 a0', real=True)

# 특성방정식 예시: a2*s^2 + a1*s + a0 = 0
# 실제 계수를 대입해 해를 구함
char_eq = a2*s**2 + a1*s + a0

# 가령 a2=1, a1=2, a0=3을 대입
char_eq_sub = char_eq.subs({a2: 1, a1: 2, a0: 3})

poles = solve(char_eq_sub, s)

print("극점:", poles)

# 각 극점의 실수부를 확인
is_stable = True
for p in poles:
    re_part = p.as_real_imag()[0]
    if re_part >= 0:
        is_stable = False
        break

if is_stable:
    print("시스템은 안정이다.")
else:
    print("시스템은 불안정하다.")

여기서 각각의 극점 \(p\)에 대해 실수부가 0 이상이면 안정도를 해치는 요인이 된다고 보고, 해당 항이 존재하면 즉시 불안정하다고 결론짓는다. 보다 복잡한 시스템에서도 동일한 원리로 해석할 수 있으며, 이를 여러 차원의 상태방정식 형태로 확장하면 상태공간 분석과도 연결된다.

라플라스 변환이 제공하는 또 다른 장점은 전달함수 표현을 단순히 분모, 분자로 나누는 것을 넘어 복잡한 계를 여러 부분 모드(극점)로 분해하여 그 역할을 명료하게 구분하는 것이다. 실제 물리 시스템에서는 여러 폴이 상호작용하기 때문에 단순 지수 하나로는 설명할 수 없는 복합 응답이 나타난다. 그러나 라플라스 영역에서 각 폴에 대응되는 지수 모드의 합으로 전체 응답을 표현할 수 있으므로, 안정도뿐 아니라 응답 특성을 정확히 파악하고 조절하기가 훨씬 용이해진다.

실무 관점에서는 다음과 같은 흐름이 일반적이다. 먼저 시스템이나 제어기를 설계한 후 전달함수를 구한다. 이어서 특성방정식을 찾고, 라플라스 변환 관점에서 극점이 모두 좌반평면에 위치하는지 확인한다. 문제가 있으면, 즉 우반평면에 극점이 있거나 허수축 경계에 근이 자리한다면, 보상기(보상 네트워크)나 피드백 이득 등을 조정해 극점을 원하는 위치로 이동시키도록 설계를 수정한다. 이 모든 과정이 라플라스 변환으로 대표되는 주파수 영역 해석과 맞물려 돌아가며, 안정도를 달성하기 위한 근본적이고도 직관적인 판단 기준이 된다.

안정도 판별을 위한 라플라스 변환의 핵심 성질¶

라플라스 변환에서 얻어지는 전달함수와 특성방정식은 시간영역의 복잡한 미분 방정식을 주파수영역의 단순한 대수 방정식으로 바꿔 준다. 이로써 시스템의 극점 위치만 확인해도 안정도를 쉽고 정확하게 판별할 수 있게 된다. 특히 시스템의 응답이 시간이 지남에 따라 어느 값으로 수렴하는지를 빠르게 예측하거나, 발산 혹은 무한 진동을 일으키는지를 판단하는 데 도움이 되는 여러 정리가 존재한다. 그중에서도 최종값 정리는 라플라스 변환을 통한 안정도 판별에서 매우 중요한 위치를 차지한다.

시간영역에서 어떤 출력 신호 \(y(t)\)가 주어졌을 때, \(t \to \infty\)에서의 값을 알고자 하는 상황을 가정한다. 일반적으로 연속시간 제어 시스템에서 \(t \to \infty\)에서의 응답이 유한한 상수로 수렴한다면, 시스템이 안정 동작한다는 점을 의미한다. 이를 수식으로 간단히 나타낼 수 있는 것이 최종값 정리이며, 올바른 조건하에서 다음과 같이 표현한다.

\[ \begin{align} \lim_{t \to \infty} y(t) = \lim_{s \to 0} s Y(s) \end{align} \]

위 식에서 \(Y(s)\)는 출력 \(y(t)\)에 대한 라플라스 변환이다. 이 정리가 의미하는 바는 라플라스 영역에서 \(s \to 0\) 근방에서 \(sY(s)\)가 어떤 값을 갖는지를 보면, 시간영역에서 무한 시간 후의 응답을 알 수 있다는 것이다. 이는 매우 강력한 도구이지만, 아무 때나 적용할 수 있는 것은 아니다. 올바르게 적용하기 위해서는 몇 가지 중요한 전제가 필요하다.

첫째, 시스템의 모든 극점은 반드시 좌반평면에 존재해야 한다. 즉 특성방정식의 근 \(s_i\)에 대해 \(\Re(s_i) < 0\)를 만족해야 하며, 순수하게 허수축 상에 근이 있어도 안 된다. 만일 우반평면에 극점이 존재하거나, 허수축에 중근(multiple pole)이 있으면 최종값 정리가 성립하지 않는다. 이는 \(t \to \infty\)에서 응답이 발산하거나, 영진동이 지속됨으로써 응답이 특정 상수로 수렴하지 않기 때문이다.

둘째, \(y(t)\)가 적어도 유계(bound) 신호가 되어야 하며, 진동이나 발산이 없는 상태에서 최종값에 접근해야 한다. 만약 시스템이 경계적 안정(marginally stable) 상태여서 지속 진동 성분을 포함하면, \(t \to \infty\)에서 응답이 고정된 실수값으로 떨어지지 않는다. 그 결과 라플라스 변환에서 \(sY(s)\)를 \(s \to 0\)로 보냈을 때, 실제 시간영역 응답과는 다른 결과가 나오거나 수렴하지 않는 경우가 발생한다.

최종값 정리를 활용해 안정도를 판별하는 가장 간단한 방법은 \(s \to 0\)에서 \(sY(s)\)가 유한 값으로 수렴하는지를 보는 것이다. 예를 들어 1차 또는 2차 계에서, 전이함수 \(G(s) = Y(s)/U(s)\)가 단순 극점들로 구성되어 있고 모두 \(\Re(s_i) < 0\)이면, 임의의 유계 입력에 대해서도 \(t \to \infty\)에서 \(y(t)\)가 발산하지 않는다. 실제 응답의 최종값을 구하고 싶을 때, \(Y(s) = G(s) U(s)\)를 사용하여 \(sY(s)\)를 구한 뒤 \(s \to 0\) 극한을 취하면 된다.

이 정리는 단순히 최종응답값이 존재하는지 여부뿐 아니라, 극점의 배치가 잘못되어 발산 요소가 포함되어 있는지 빠르게 식별하는 지표가 될 수 있다. 예컨대 우반평면에 극점이 있다면, 어떤 형태의 입력이든 해당 모드가 지수적으로 커지므로 \(sY(s)\) 항이 발산하거나 수렴하지 않게 된다. 이때 시스템은 명백히 불안정하다. 또한 허수축에 단근(simple pole)이 놓여 있는 경우에도 무한 시간 동안 정현파 응답이 이어지므로 최종값 정리가 적용되지 않는다.

최종값 정리가 이렇게 유용함에도 불구하고, 이를 적용하는 데 있어서 흔히 간과하는 점이 있다. 극점이 좌반평면에 몰려 있다 해도, 특정 형태의 입력이나 외란, 혹은 시스템 내의 비선형성이 추가로 포함되어 있다면 단순히 \(s \to 0\)에서의 극한만으로는 실제 동작을 완전히 예측하기 어려울 수 있다. 그러나 선형 시불변(LTI) 계에서의 BIBO 안정성 판단이라는 범위 안에서는 최종값 정리를 기반으로 한 극점 분석이 매우 간단하고 직관적이다.

라플라스 변환의 이러한 성질은 시스템이 안정 동작을 하는지를 확인하는 과정뿐 아니라, 응답이 장기적으로 어떤 값을 갖는지 추정하거나, 제어기 설계 시에 특정 극점 배치를 목표로 튜닝하는 방법을 결정할 때도 큰 도움이 된다. 시스템 설계자가 폐루프 전달함수를 구성한 뒤, 명시적으로 구한 \(Y(s)\)에 대해 \(s \to 0\) 극한값을 살펴보고, 그것이 원하는 목표값으로 수렴하는지 검증하는 식으로 폐루프 안정성을 판단할 수도 있기 때문이다.

안정도 판별에서 라플라스 변환과 결부된 또 다른 중요한 개념은 영역(region of convergence)이다. 라플라스 변환 \(Y(s)\)가 수렴하기 위해서는 일정 범위의 \(s\)에서 적분이 유한해야 하는데, 이 영역을 영역이라고 부른다. 일반적으로 인과성(causality)을 만족하는 LTI 시스템의 경우 영역이 \(s\)평면의 우측 부분에 존재하며, 이 영역이 극점을 모두 포함할 수 있도록 설정된다. 물리적으로는 시간이 0부터 시작하는 신호에 대해, 모든 반응이 유한 시간을 기준으로 일어나므로 \(s \to +\infty\) 방향에서 변환이 수렴해야 한다. 안정도 판별과 직결되는 것은 이 영역과 극점의 상대적 위치이다. 극점이 영역 바깥으로 벗어나 우반평면을 침범하거나, 영역의 경계선인 허수축 상에 배치되어 있으면 수렴이 보장되지 않아 시스템이 불안정하다는 결론을 얻게 된다.

결국 안정도를 판별하기 위해서는 특성방정식의 해인 극점의 위치가 좌반평면에 존재한다는 사실과, 그에 따른 최종값 정리의 적용 가능 여부를 함께 확인하는 것이 핵심이다. 이 과정을 거쳐 \(t \to \infty\)에서의 응답이 상수값으로 수렴함을 알면 시스템은 안정이라고 선언할 수 있다. 반대로 극점이 하나라도 우반평면에 존재하거나, 허수축에 배치된 근이 계속 진동을 만들어 최종값이 정의되지 않게 된다면 시스템은 불안정하거나 경계적으로 안정하다고 해석한다.

이처럼 라플라스 변환을 통한 안정도 분석에서는 최종값 정리를 비롯한 여러 성질이 유기적으로 맞물려 동작한다. 각각의 성질이 지향하는 결론은 동일하다. 극점이 우반평면에 있으면 지수 발산, 좌반평면이면 지수 감쇠, 허수축이면 영진동이 발생한다. 그리고 최종값 정리를 통해 \(t \to \infty\)에서의 응답을 보다 직접적으로 추론할 수 있다. 이는 고전 제어, 근대 제어, 디지털 제어 등 다양한 제어 체계에서도 동일하게 적용되며, 라플라스 변환이 제어공학에서 중요한 기반 이론으로 자리매김하는 이유이기도 하다.

전달함수 극점과 영점의 역할¶

전달함수가 갖는 분모 다항식의 근을 극점(pole), 분자 다항식의 근을 영점(zero)이라 한다. 극점은 안정도를 결정하는 핵심 요소로 작용하고, 영점은 시스템의 응답 형태를 세부적으로 바꾸는 역할을 담당한다. 분모 다항식의 근이 시스템에 지수 항 \(e^{s_i t}\)를 형성하여 감쇠 또는 발산을 주도하기 때문에, 극점의 위치가 곧 안정도의 성패를 가르는 주요 요인이 된다. 반면, 영점은 시스템 응답의 특정 모드를 상쇄하거나 위상 특성을 변화시키며, 과도응답의 변화나 일부 특수 주파수에서의 진폭 왜곡 등에 깊은 영향을 미친다. 하지만 BIBO 안정성을 판별할 때는 극점이 복소평면 좌반평면에 위치하느냐 여부가 훨씬 더 중요하다. 영점이 아무리 우반평면에 존재하더라도 극점이 모두 좌반평면에 있으면 시스템은 안정하다는 점이 고전 제어이론에서 널리 알려진 사실이다.

영점이 극점과 일치하면 시간 응답에서 해당 모드가 상쇄되는 흥미로운 현상이 나타난다. 예를 들어 전달함수가 \(G(s) = \dfrac{(s + 1)}{(s + 1)(s + 2)}\)와 같이 분자와 분모에 공통된 인자가 있을 경우, 실제 물리 시스템에서는 그 인자가 완전 상쇄되지 않고 작은 차이로 남을 수 있지만, 이상화된 수학 모델에서는 부분 분수 전개 시 특정 항이 사라져 좀 더 단순화된 형태의 응답을 얻는다. 이러한 상쇄 효과는 실험적으로는 오차나 잡음, 진동 모드 간섭으로 인해 완전한 소멸이 일어나지 않기도 하지만, 이론적으로 전달함수를 해석할 때 매우 중요한 함의를 제공한다.

영점이 존재하면, 특히 우반평면 영점이 생기는 경우에는 폐루프 제어 설계 시 주파수응답 특성이나 안정 여유(margin)가 달라질 수 있다. 영점의 위치가 지나치게 우반평면에 치우치면, 급격한 위상 변화와 함께 특정 입력에 과도한 진폭 증가가 발생할 위험이 있다. 이를 방지하기 위해 보상기 설계(위상 보상, PID 등의 기법)에서 영점의 효과를 고려한다. 그렇다고 해서 영점이 극점처럼 직접 안정도를 결정하지는 않는다. 분자 다항식의 근은 지수적으로 발산하거나 감쇠하는 항을 만들어내지 않기 때문에, 안정도를 결정짓는 지배적 요소는 어디까지나 극점이다.

영점은 실제 응답에서 어떤 모드가 얼마나 크게 나타나는지, 또는 어떤 모드가 사라지는지를 결정함으로써, 최종 응답 파형의 형태와 특정 주파수 영역에서의 이득(gain)을 조절한다. 이를 수학적으로 명확하게 해석하기 위해 부분 분수 전개를 사용하면 편리하다. 예를 들어 단순화를 위해 극점이 세 개, 영점이 두 개인 전형적인 고차 전달함수를 가정할 수 있다. 이때 전달함수 \(G(s)\)를 여러 항으로 분리하여 각각 역라플라스 변환하면, 시간영역 해석에서 각 항이 얼마만큼 기여하는지 확인할 수 있다. 그 과정에서 영점이 극점과 결합해 특정 항을 소멸시키거나, 혹은 위상 반전을 일으키는 등의 현상을 명확히 살펴볼 수 있다.

전달함수의 극점과 영점이 폐루프에서 어떻게 재배치되는지는 루트 궤적(root locus) 같은 방법으로도 분석할 수 있지만, 궁극적으로는 \(s\)-평면 상에서 분모 다항식이 형성하는 극점 분포가 안정도 판별의 기준선이 된다. 영점은 루트 궤적의 기하학적 형상과 다양한 주파수응답 특성을 좌우하며, 경우에 따라 폐루프 극점이 허수축에 접근하거나 우반평면으로 넘어가는 경로를 만들어내기도 한다. 그러므로 고전 제어든 근대 제어든, 폐루프 전달함수의 극점과 영점 분포를 면밀히 분석해야만 요구하는 성능과 안정도를 동시에 만족하는 설계를 할 수 있다.

영점이 부르는 주파수영역에서의 특성 변화를 살펴보기 위해 보드 선도(Bode plot)를 참조하는 경우가 많다. 영점이 좌반평면에 위치하면 위상과 진폭 곡선이 특정 각속도 구간에서 특징적인 상승이나 하강 패턴을 보이고, 우반평면 영점이면 더 급격한 위상 진폭 변화를 일으켜 과도응답에서 빠르고 큰 진동이나 오버슈트(overshoot)를 유발하기도 한다. 따라서 엔지니어가 제어기를 설계할 때는 영점의 영향을 과소평가해서는 안 되며, 극점을 안정적으로 배치해도 영점 때문에 원하는 성능에 도달하지 못할 수 있다는 사실을 유념해야 한다.

아래 Python 예제 코드는 단순한 전형적 전달함수에서 극점뿐 아니라 영점도 함께 계산하는 방법을 보여준다. Sympy 라이브러리를 사용하여 분자 다항식과 분모 다항식을 각각 정의한 뒤, solve 함수를 통해 근들을 찾고, 극점과 영점이 어떤 위치에 존재하는지 확인한다. 영점은 안정도 자체를 깨뜨리지 않지만, 응답 특성(오버슈트, 과도시간, 주파수 대역 내 이득 등)에 결정적인 영향을 끼칠 수 있으므로 극점과 함께 살펴보아야 한다.

from sympy import symbols, Function, solve, Poly

s = symbols('s', complex=True)
# 예: G(s) = (s + 4) / (s^2 + 3*s + 2)
# 분자 다항식: s + 4
# 분모 다항식: s^2 + 3s + 2

num = s + 4
den = s**2 + 3*s + 2

# 영점과 극점을 각각 solve로 계산
zeros = solve(Poly(num, s), s)
poles = solve(Poly(den, s), s)

print("Zeros of G(s):", zeros)
print("Poles of G(s):", poles)

# 극점 실수부를 확인하여 안정도 평가
stable = all([pole.as_real_imag()[0] < 0 for pole in poles])
print("Is the system stable?", stable)

이 예시에서 분모 다항식의 근은 \(s^2 + 3s + 2 = 0\)의 해이고, 이는 \(s = -1\)과 \(s = -2\)가 된다. 두 극점 모두 음수 실수부를 갖기 때문에 이 시스템은 BIBO 안정이다. 분자 다항식의 근은 \(s + 4 = 0\)이므로 영점은 \(s = -4\)에 놓여 있으며, 이는 안정도에 직접적인 영향을 주지 않지만 주파수응답 특성에서 주파수 \(\omega\)가 커졌을 때의 위상 변화를 이끈다. 이렇게 극점과 영점을 동시해석하면 시스템 성능을 종합적으로 평가할 수 있다.

아래는 극점과 영점의 상대적 위치를 단순하게 나타내는 mermaid 블록이다. 좌반평면과 우반평면을 선으로 구분하고, 극점과 영점이 어디쯤 배치되는지 개념적으로 표현한다. 수직선 왼쪽이 좌반평면, 오른쪽이 우반평면이다.

flowchart LR A((좌반평면)) ---|"극점 (-1, -2), 영점 (-4)"| B((우반평면))

이 다이어그램은 단순 개념도이며, 실제로는 복소 평면에서 영점과 극점이 허수축을 기준으로 어떻게 퍼져 있는지 확인해야 한다. 그러나 최소한 영점과 극점이 서로 다른 위치에 존재하면서, 안정성을 결정하는 기준은 극점이 좌반평면에 있는가라는 사실을 상기시키는 목적으로 참고할 수 있다.

결론적으로 영점은 전달함수의 분자에 의해 결정되며, 시스템 출력 응답의 크기나 위상 특성, 그리고 특정 주파수 대역에서의 동특성 변화를 주도한다. 극점은 전달함수의 분모에 의해 결정되며, 시스템의 지배적 모드를 직접 형성하여 안정도 판별에 가장 중요한 역할을 한다. 영점은 안정 여부를 바꿀 수 없지만, 실제 동작 상황에서 과도응답과 주파수응답 특성을 예측하고, 설계자가 원하는 목표 성능(오버슈트, 상승시간, 위상 여유 등)을 달성하는 데 큰 영향을 미친다. 이처럼 전달함수에서 영점과 극점은 제어 시스템을 더욱 정밀하고 의도한 대로 동작하게 만드는 핵심 요소들이며, 서로의 분포 관계를 종합적으로 살펴야 한다는 점이 이론과 실무에서 모두 강조된다.

안정 영역과 폴의 배치 변화¶

연속시간 LTI(선형 시불변) 시스템에서 라플라스 변환으로 표현된 전달함수의 극점이 모두 좌반평면(left half-plane)에 존재하면 시스템은 안정하다고 한다. 이는 복소변수 \(s = \sigma + j\omega\)에서 \(\sigma < 0\)인 영역을 말하며, 모든 극점이 여기에 놓일 경우 시간이 무한대로 갈 때 각 모드의 응답이 지수적으로 감쇠하기 때문이다. 극점이 우반평면(right half-plane) 즉 \(\sigma > 0\)인 영역에 하나라도 존재하면 시스템은 그 극점에 대응하는 모드에서 지수 발산이 일어나므로 불안정해진다. 경계선인 허수축 \(\sigma = 0\)에 극점이 위치하면 추가적인 해석이 필요하다. 단일 극점이 단순근의 형태로 허수축에 놓인 경우는 경계적 안정(marginal stability)을 나타낼 수도 있지만, 실계수 다항식에서 그와 같은 근이 중근 이상으로 발생하면 대부분 불안정 시스템으로 분류한다.

라플라스 변환 관점에서 극점의 좌반평면 배치 여부는 전적으로 특성방정식의 근분포에 달려 있다. 예를 들어 차수가 \(n\)인 특성다항식

\[ \begin{align} a_n s^n + a_{n-1} s^{n-1} + \cdots + a_1 s + a_0 = 0 \end{align} \]

을 만족하는 근들이 모두 \(\Re(s_i) < 0\)이면 안정 영역 내부에 극점이 존재한다고 해석된다. 이때 시스템 설계 관점에서는 계수 \(a_i\)가 변할 때 극점들이 어떤 경로를 따라 이동하는지 살펴보는 것이 중요하다. 고전 제어 이론에서는 이를 루트 궤적(root locus) 기법으로 시각화한다. 매개변수 \(K\)가 연속적으로 변화함에 따라 특성다항식의 형태가 달라지고, 결과적으로 \(s\)-평면 상의 근들이 이동한다. 안정 영역인 좌반평면을 벗어나면 불안정해지므로, 루트 궤적은 매개변수 조정에 따른 시스템의 안정도 변화 경향을 직관적으로 알 수 있는 대표적인 도구다.

루트 궤적은 특별한 형식의 특성다항식에서 더욱 명확히 드러난다. 예를 들어 폐루프 특성방정식이

\[ \begin{align} 1 + K G(s) = 0 \end{align} \]

의 형태를 띨 때, \(K\)가 0에서 \(\infty\)로 변화하면 극점들이 어떤 경로로 이동하는지가 루트 궤적 기법을 통해 도출된다. 이를 해석할 때에는 극점이 허수축을 지나는 지점을 찾고, 그 시점에서 안정성이 바뀌는지를 확인하는 방식으로 접근한다. 우반평면으로 극점이 넘어가는 순간이 곧 안정성을 잃게 되는 시점이며, 그 임계값을 통해 시스템 설계의 안전 여유를 산정할 수 있다.

안정 영역은 단순히 좌반평면이라는 지리적 설명 외에도 시간영역 응답의 형태와 직접적으로 연결된다. 모든 극점 \(\sigma_i\)가 음수 실수부를 갖고 있으면 해당 모드는

\[ \begin{align} e^{\sigma_i t} \end{align} \]

처럼 시간이 지날수록 빠르게 감쇠하거나 느리게 감쇠할 수 있지만, 결국은 0으로 수렴한다. 다만, 그 극점들이 복소수 형태(\(\sigma_i + j\omega_i\))라면 단순히 감쇠하는 것에 진동 성분이 얹혀 \(e^{\sigma_i t} \cos(\omega_i t)\) 또는 \(e^{\sigma_i t} \sin(\omega_i t)\) 형태의 진동 감쇠가 일어난다. 이 모든 경우에서 \(\sigma_i < 0\)라는 조건이 충족되어야 안정도를 확보한다.

매개변수가 변경될 때 극점은 일반적으로 연속적으로 이동한다. 적은 변화로는 우반평면과 좌반평면 사이의 경계인 허수축을 획기적으로 넘어가지 않지만, 특정 조건이 성립하면 극점이 허수축을 가로질러 우반평면으로 옮겨가게 되고 그 지점에서 시스템은 불안정해진다. 이처럼 매개변수 변화에 따른 극점 이동 과정을 관찰하면 실제 제어 시스템에서 요구하는 안정 여유와 감쇠특성을 종합적으로 설계할 수 있다.

이 과정을 좀 더 구체적으로 살펴보기 위해 매개변수 한 개짜리 예시를 들어보면, 폐루프 특성방정식이

\[ \begin{align} a_2 s^2 + a_1 s + a_0 + K (b_2 s^2 + b_1 s + b_0) = 0 \end{align} \]

의 형태를 가질 때, \(K\)가 0에서 시작하여 점차 증가한다고 가정한다. \(K=0\)일 때 극점은

\[ \begin{align} a_2 s^2 + a_1 s + a_0 = 0 \end{align} \]

의 근으로 결정된다. \(K\)가 커짐에 따라 \(a_2\)와 \(b_2\) 등 계수가 복합적으로 합쳐져 새롭게 형성되는 2차 또는 고차 방정식의 근들이 부드럽게 이동한다. 이 이동 경로가 허수축을 건너는 지점에서,

\[ \begin{align} s = j\omega \end{align} \]

를 대입해 살펴보면 크기 조건이나 위상 조건에 의해 경계 안정 해를 찾을 수 있다. 그 지점에서 실제로 극점이 우반평면으로 넘어가는지 여부를 확인하여 안정도 변화를 분석한다.

아래는 Python을 이용해 특정 2차 시스템에서 매개변수 \(K\) 변화에 따라 극점이 어떻게 이동하는지 대략적인 궤적을 살피는 간단한 예시 코드다. 여기서는 symbolic 루트 궤적을 자동으로 그리는 것보다는, 일정 간격으로 \(K\)를 변화시키면서 실제로 극점을 수치적으로 계산해 표시하는 방식을 택한다.

import numpy as np
import matplotlib.pyplot as plt
import cmath

def poles_of_system(K):
    # 예시: 특성방정식 = s^2 + (2 + K)*s + (5 + 2*K)
    # 이때 a2=1, a1=(2+K), a0=(5+2*K) 형태
    # 근의 공식으로 극점을 구한다
    a2 = 1.0
    a1 = 2.0 + K
    a0 = 5.0 + 2.0*K

    # 판별식 D
    D = a1**2 - 4*a2*a0

    # 근 s1, s2
    s1 = (-a1 + cmath.sqrt(D)) / (2*a2)
    s2 = (-a1 - cmath.sqrt(D)) / (2*a2)

    return s1, s2

K_values = np.linspace(0, 10, 50)  # K를 0부터 10까지 50개 구간으로
pole1_real = []
pole1_imag = []
pole2_real = []
pole2_imag = []

for K in K_values:
    s1, s2 = poles_of_system(K)
    pole1_real.append(s1.real)
    pole1_imag.append(s1.imag)
    pole2_real.append(s2.real)
    pole2_imag.append(s2.imag)

plt.axvline(x=0, color='k', linestyle='--')  # 허수축
plt.scatter(pole1_real, pole1_imag, color='blue', label='Pole1')
plt.scatter(pole2_real, pole2_imag, color='red', label='Pole2')
plt.xlabel('Real axis')
plt.ylabel('Imag axis')
plt.title('Pole movement as K changes')
plt.legend()
plt.grid(True)
plt.show()

이 코드는 매우 간단히 작성된 것으로, \(K\)가 0에서 10까지 바뀔 때 극점들의 실수부와 허수부가 어떻게 달라지는지 확인할 수 있다. 루트 궤적 기법의 핵심 아이디어는 바로 이렇게 매개변수에 따라 특성방정식의 근이 연속적으로 이동한다는 점이며, 해당 이동 경로가 허수축을 넘어가거나 그 가까이 접근할 때 시스템 안정도가 달라진다는 사실을 시각적으로 파악하게 해준다.

좌반평면에 극점이 존재한다는 것은 주파수 영역 해석상으로도 중요한 의미가 있다. 극점이 좌반평면 깊숙이 위치할수록 응답이 빠르게 감쇠하고, 이는 시간영역에서 과도응답의 처리 속도가 빨라지는 결과로 이어진다. 반대로 극점이 허수축 근처, 즉 실수부가 0에 가까워질수록 응답이 느리게 감쇠하거나 크게 진동할 수 있다. 이 때문에 실제 제어 시스템 설계자는 극점이 최소한 원하는 감쇠비 이상으로 좌반평면 안쪽에 위치하도록 매개변수들을 조정한다.

결국 안정 영역과 폴의 배치 변화는 라플라스 변환을 통한 제어 이론 전반에서 핵심이다. 극점이 어떤 매개변수 변화로 인해 우반평면으로 넘어가면 불안정한 모드가 생기는 것이고, 허수축 위에 있으면 경계적 상태를 의미한다. 이런 이유로 모든 전통적 안정도 판별법(Routh-Hurwitz, Nyquist, Bode, Root locus 등)은 복소평면의 좌반평면에 극점이 존재하는지 여부를 결국에는 확인하거나 암묵적으로 보장하도록 되어 있다. 이와 같은 분포 변화 과정을 면밀히 추적하는 것이 제어 시스템의 안전성과 동특성을 확보하는 가장 기초적이고도 중요한 절차다.

라플라스 변환 기반 안정도 분석의 예시 코드 (Python)¶

라플라스 변환을 활용하여 안정도를 판별할 때, 특정 계수로 정의된 특성방정식의 극점(폴)을 직접 찾아서 그 실수부를 확인하는 방식이 매우 자주 활용된다. 이 과정을 실제로 계산하기 위해 파이썬의 상징 연산(sympy 라이브러리)을 이용하면, 대수 방정식의 해를 바로 구하고 복소수 해의 실수부나 허수부를 분리할 수 있어 편리하다. 여기서는 2차 특성방정식을 간단한 형태로 설정한 뒤, 이를 통해 극점을 구하고 시스템 안정도를 확인하는 과정을 예시 코드로 보여준다.

여기서 가정하는 전달함수는 다음과 같이 표현된다. 분모 다항식(특성다항식)을 \(a_2 s^2 + a_1 s + a_0\)라 하면, 2차 시스템의 전달함수는 G(s) = dfrac{1}{a_2 s^2 + a_1 s + a_0} 라는 형태로 둘 수 있다. 이때 특성방정식은 아래와 같이 나타난다.

\[ \begin{align} a_2 s^2 + a_1 s + a_0 = 0 \end{align} \]

이 특성방정식의 근(극점)인 \(s\)들을 구하고, 각각에 대해 실수부가 음수인지 검사하면 안정도를 판단할 수 있다.

아래 파이썬 예시 코드는 이러한 2차 특성방정식을 실제로 풀고, 극점의 실수부를 통해 시스템 안정성을 판별한다. 코드의 각 부분을 순차적으로 살펴보면 다음과 같은 의도가 담겨 있다. 코드 중간에는 각 행이 수행하는 역할을 주석으로 기입하였다.

from sympy import symbols, Function, solve, Poly

# 복소수형 기호 s, 그리고 실수형 기호 a0, a1, a2 정의
s = symbols('s', complex=True)
a0, a1, a2 = symbols('a0 a1 a2', real=True, positive=True)

# 예시 시스템: G(s) = 1 / (a2*s^2 + a1*s + a0)
# 특성방정식: a2*s^2 + a1*s + a0 = 0

# 특성방정식을 심볼로 표현
char_eq = a2*s**2 + a1*s + a0

# 미리 a2, a1, a0에 특정 값을 대입해 실험
char_eq_sub = char_eq.subs({a2: 1, a1: 2, a0: 3})

# 특성방정식을 직접 풀어 극점(근) 찾기
poles = solve(char_eq_sub, s)
print("System poles:", poles)

# 극점별 실수부가 전부 음수인지 검사하여 안정도 판별
is_stable = all([pole.as_real_imag()[0] < 0 for pole in poles])
print("Is the system stable?", is_stable)

첫 번째 단락에서 from sympy import symbols, Function, solve, Poly 구문은 심볼릭(symbolic) 수식을 처리하기 위한 라이브러리를 불러온다. 특히 symbols 함수를 이용해 \(s\), \(a_0\), \(a_1\), \(a_2\) 등의 기호(symbol)를 정의하면, 실제 수 대신 대수 기호로 연산이 가능해진다. 이렇게 정의해두면 \(s\)에 대해 2차 방정식의 해를 손쉽게 구할 수 있고, 그때 구해진 근이 복소수일 경우 실수부와 허수부를 분리하기도 용이하다.

두 번째 단락에서 s = symbols('s', complex=True)는 \(s\)가 복소변수임을 의미한다. 그러므로 방정식을 풀었을 때 복소근이 나와도 그 해를 정확하게 표현한다. a0, a1, a2 = symbols('a0 a1 a2', real=True, positive=True)는 계수를 양의 실수로 제한한다. 이것은 예시 상황에서 2차 방정식의 계수가 모두 양수라는 가정 하에 해를 구하고자 할 때 흔히 쓰는 설정이며, 안정도를 직관적으로 확인하기 좋다.

그다음 char_eq = a2*s**2 + a1*s + a0 구문을 보면, 특성방정식을 하나의 기호식으로 정의한다. 이때 char_eq 자체는 미분 방정식에서 나온 분모 다항식이자, 시스템 안정도를 결정하는 핵심이다. 실제 응용에서는 이 식을 보다 복잡한 형태로 일반화할 수도 있으나, 여기서는 2차 예시로 제한한다.

이후 char_eq_sub = char_eq.subs({a2: 1, a1: 2, a0: 3})에서 subs 메서드를 통해 \(a_2 = 1, a_1 = 2, a_0 = 3\)을 대입한다. 코드 상에서 예시값을 골라 넣어주는 방식이다. 실제 설계 상황에서는 사용자가 다양한 매개변수를 넣어보고, 극점의 변화를 관찰할 수 있다.

다음으로 poles = solve(char_eq_sub, s) 구문을 보면, 이 부분이 실질적으로 \(a_2 s^2 + a_1 s + a_0 = 0\)을 푸는 핵심이다. solve 함수는 주어진 심볼릭 방정식을 입력받아서 해를 리스트 형태로 반환한다. 따라서 poles에는 이차 방정식의 두 근이 복소수 형태로 들어가게 되며, 각 근이 실수일 수도 있고 복소수일 수도 있다.

print("System poles:", poles)로 찍어보면 실제 극점 두 개가 출력된다. 만약 대입한 계수에 따라 근이 실수가 되면 \((s_1, s_2)\) 형태로 단순한 실수 해를 볼 수 있고, 복소수라면 (re + imI, re - imI) 형태로 짝을 이룬 해가 나올 것이다. 이렇게 구해진 근을 가지고 곧바로 실수부와 허수부를 분리하거나, 여러 부가 정보를 뽑아낼 수 있다.

마지막으로 is_stable = all([pole.as_real_imag()[0] < 0 for pole in poles])는 극점 리스트에 대해서 각각의 실수부가 음수인지 확인하고, 모두 음수이면 안정(true)을, 그렇지 않으면 불안정(false)을 반환한다. pole.as_real_imag()는 해당 극점의 실수부와 허수부를 튜플로 돌려주는데, [0] 인덱스가 실수부, [1] 인덱스가 허수부다. 2차 시스템의 경우, 양쪽 극점이 모두 좌반평면에 위치하면 지수 감쇠 형태로 응답이 수렴하여 BIBO 안정이다. 그러므로 is_stable이 True이면 시스템이 안정임을 의미한다.

출력으로 확인할 수 있는 것은 다음과 같은 구조를 가진다. 예를 들어 실제로 이 코드를 실행했을 때,

System poles: [-1 +/- sqrt(1 - 3)]
Is the system stable? True (or False)

와 비슷한 결과가 나올 수 있다. \(1 - 3 = -2\)이므로, 근은 복소수 형태가 된다. 실제 값을 대입하면 대략 \(-1 \pm j\sqrt{2}\)가 될 것이고, 실수부는 \(-1\)이므로 음수다. 따라서 모든 극점이 음수 실수부를 갖고, 시스템이 안정임을 알 수 있다.

이 예시 코드는 라플라스 변환을 직접 계산하지는 않지만, 라플라스 변환의 분모 다항식(특성방정식)만 제대로 표현하면 그 해를 쉽게 구할 수 있다는 사실을 보여준다. 즉, 시간영역 미분 방정식을 라플라스 영역으로 대응시킨 뒤, 분모가 되는 특성다항식을 풀어 해를 구한다는 원리를 그대로 코드로 옮긴 형태라고 볼 수 있다. 실제 고차 시스템, 예를 들어 3차 이상의 특성방정식도 이와 같은 방식으로 해결할 수 있으며, 각각의 근을 실시간으로 모니터링하여 어떠한 파라미터 변화가 극점을 우반평면 쪽으로 옮기는지 확인할 수도 있다. 이것은 제어 시스템 설계에서 매우 중요한 과정이며, 궁극적으로는 전이함수 \(G(s)\)를 통한 안정도 판별 절차가 자동화될 수 있다는 장점을 제공한다.

이상과 같이 파이썬 심볼릭 연산을 활용하면, 라플라스 변환 기반의 안정도 판별을 계산적으로도 쉽게 수행할 수 있다. 시스템 계수를 바꾸어가며 극점의 움직임을 추적할 수 있기 때문에, 고전 제어 이론에서 루트 궤적(root locus) 또는 주파수 응답을 시뮬레이션하는 기초 자료로도 활용할 수 있다. 실제 산업 현장에서는 더욱 복잡한 모델에도 적용할 수 있으며, 간단한 수치 해석과는 달리 대수적인 처리 결과를 직접 확인할 수 있다는 이점이 있다.

결국, 라플라스 변환에서 특성방정식을 정립하고 극점을 구한 뒤, 그 실수부가 음수인지 양수인지를 확인하는 과정이야말로 안정도 판별의 가장 기본적인 접근 방식이다. 위 예시 코드는 이를 실무적으로도 간단히 응용할 수 있다는 사실을 입증한다.

---과 요약

라플라스 변환을 이용하여 안정도를 판별할 때 가장 핵심적인 사실은 특성방정식의 근, 즉 극점이 복소평면에서 좌반평면에 존재해야 한다는 점이다. 극점이 모두 좌반평면에 존재하면 시스템의 시간영역 응답은 지수적으로 감쇠하며 수렴한다. 이는 지수 응답 항이 일반적으로 다음과 같이 표현되기 때문이다.

\[ \begin{align} e^{s t} \end{align} \]

여기서 복소수 \(s\)를 \(s = \sigma + j \omega\)로 놓았을 때, 실수부 \(\sigma < 0\)이면 \(t\)가 무한대로 갈 때 \(e^{\sigma t}\)가 0으로 수렴한다. 이 과정을 통해 \(\sigma < 0\) 조건이 곧 안정도를 만족하는 핵심이라고 할 수 있다.

시간영역에서의 응답 형태가 \(e^{\sigma t}\)로 구성된다는 점은 라플라스 변환의 역변환에서 비롯된다. 특성방정식을 0으로 만드는 근들이 모두 좌반평면에 있으면, 이들 각각에 대응하는 지수 항이 음의 실수부를 포함하므로 지수적 감쇠가 일어난다. 하나라도 우반평면(\(\sigma > 0\))에 근이 존재하면, 그 지수 항은 발산 형태를 보이므로 시스템은 불안정해진다. 경계축인 \(\sigma = 0\)에 극점이 있는 경우에는 엄밀한 해석이 필요하다. 단순 근으로만 존재하면 계속 진동하거나 일정 값을 유지하는 경계적 안정이 될 수 있지만, 경계축 근이 중근 이상이면 시스템은 실제 동작에서 발산 형태(불안정)로 이어질 수 있다.

라플라스 변환은 이러한 시스템 해석을 간단한 대수 방정식 풀이로 바꾸어 준다. 시간영역에서 미분 연산자로 구성된 복잡한 방정식이 라플라스 영역에서 \(s\)라는 복소변수의 다항식 형태로 나타나므로, 각 항의 근을 찾는 문제가 된다. 이 단순화 덕분에 제어공학에서는 안정도를 비롯한 여러 동특성 분석에 라플라스 변환을 광범위하게 활용한다. 예컨대 고차 미분 방정식을 직접 풀지 않고도 시스템의 극점만 확인하여 안정도를 빠르게 판별할 수 있다.

시스템이 가진 극점의 배치를 실제로 계산하거나 시각적으로 표현할 때는 다양한 기법이 동원된다. 루트 궤적 기법은 제어 이득 등의 파라미터 변화에 따라 극점이 복소평면에서 어떻게 이동하는지 보여주며, 이 과정을 통해 어느 순간 극점이 좌반평면에서 우반평면으로 넘어가는 임계점을 확인할 수 있다. 주파수 해석 기법인 보드 선도나 나이퀴스트 선도를 사용할 때도, 본질적으로는 라플라스 변환으로부터 도출된 전달함수를 주파수 영역으로 해석한다는 점이 동일하게 적용된다.

실제 설계 측면에서 라플라스 변환과 극점 분석은 단순 기계·전기계뿐 아니라 화학, 바이오, 로보틱스 등 복잡한 동적 시스템까지 다양한 분야에 걸쳐 활용된다. 모든 연속시간 선형 시스템은 본질적으로 라플라스 변환을 통해 해석이 가능하며, 극점이 갖는 의미가 동특성 전반을 결정한다. 고차 시스템의 경우 부분 분수 전개를 통해 각각의 모드를 따로 떼어 분석하기도 하고, 응답의 각 지수 항이 어떻게 감쇠하거나 발산하는지 추적한다.

라플라스 변환을 활용해 안정도를 판별하기 위해서는 특성방정식과 극점 위치만 정확히 파악하면 된다. 그러나 실제 동작 환경에서는 비선형 요소, 시스템 파라미터의 오차, 외란 등이 존재할 수 있으므로, 이론적으로 안정이더라도 안전 마진이 충분한지를 추가로 확인한다. 그래도 제어 이론의 가장 기초가 되는 안정도 판별은 라플라스 변환에서 출발하며, 이는 고전 제어부터 근대 제어, 디지털 제어 전반에 걸쳐 공통적으로 적용되는 불변의 원칙이다.

아래는 C++를 사용하여 2차 시스템의 특성방정식을 해석하고, 근의 실수부를 통해 시스템 안정 여부를 판별하는 간단한 예시 코드다. 두 개의 계수만 간단히 다뤄 보지만, 확장하면 고차 시스템에도 동일한 원리를 적용할 수 있다.

#include <iostream>
#include <complex>
#include <cmath>
#include <vector>

// 단순 2차 방정식 ax^2 + bx + c = 0의 근을 구한 뒤 안정도 판별
int main() {
    double a = 1.0; 
    double b = 3.0;
    double c = 2.0;

    double D = b*b - 4*a*c;  // 판별식
    if (D >= 0) {
        // 실근 2개
        double s1 = (-b + std::sqrt(D)) / (2*a);
        double s2 = (-b - std::sqrt(D)) / (2*a);

        std::cout << "Pole 1: " << s1 << std::endl;
        std::cout << "Pole 2: " << s2 << std::endl;

        bool stable = (s1 < 0) && (s2 < 0);
        if (stable) {
            std::cout << "System is stable." << std::endl;
        } else {
            std::cout << "System is unstable." << std::endl;
        }
    } else {
        // 복소근 2개
        double realPart = -b / (2*a);
        double imagPart = std::sqrt(-D) / (2*a);

        std::cout << "Pole 1: " << realPart << " + j" << imagPart << std::endl;
        std::cout << "Pole 2: " << realPart << " - j" << imagPart << std::endl;

        bool stable = (realPart < 0);
        if (stable) {
            std::cout << "System is stable." << std::endl;
        } else {
            std::cout << "System is unstable." << std::endl;
        }
    }
    return 0;
}

코드에서 \(a\), \(b\), \(c\)는 2차 특성방정식의 계수이며, 컴퓨터로 극점을 직접 구해 실수부가 음수인지 검사한다. 이는 라플라스 변환의 이론을 단순화한 형태이지만, 근본적인 접근 방식은 동일하다. 실제로는 전달함수를 통해 얻은 차수가 더 높은 특성방정식에서도 이 방법을 확장하여 극점을 모두 계산하고, 모든 극점의 실수부를 살펴볼 수 있다.

정리하자면 라플라스 변환은 미분 연산을 \(s\) 도메인에서 단순화함으로써 시스템 안정도 해석을 용이하게 만든다. 이렇게 얻은 특성방정식의 극점이 좌반평면에 위치해야 시간영역에서 지수적으로 감쇠하고, 우반평면에 존재하면 지수적으로 발산한다는 사실이 연속시간 선형 시스템 안정성의 본질이다. 이는 제어공학 전반에서 불변적으로 적용되는 원칙으로, 다양한 제어 기법과 접목되어 복잡한 동적 시스템을 설계·분석하는 데 활용된다.
