1계 선형 미분방정식 중에서 다음과 같은 형태를 고려한다.

\[ \begin{align} \frac{dy}{dx} + P(x)\,y = Q(x). \end{align} \]

이 식에서 \(P(x)\)와 \(Q(x)\)는 독립변수 \(x\)에 대한 알려진 함수라고 가정한다. \(y\)는 우리가 구하고자 하는 종속변수다. 이 방정식은 제어공학에서 상태방정식을 간단화하거나 시간영역에서 1차 계통의 동작을 해석할 때 자주 등장한다. 적분인자 기법은 이 방정식을 체계적이고 간단한 방식으로 풀 수 있게 해주는 중요한 방법이다.

적분인자 기법을 사용하려면 먼저 적분인자 \(\mu(x)\)를 정의한다. 적분인자는 다음과 같이 정의된다.

\[ \begin{align} \mu(x) = \exp\biggl(\int P(x),dx\biggr). \end{align} \]

\(\exp(\cdot)\)는 지수함수다. 적분인자를 곱하면 미분방정식의 좌변이 특정한 형태, 즉 어떤 곱의 도함수 형태로 변형되도록 만들어준다. 이것이 적분인자 기법의 핵심 아이디어다.

적분인자 \(\mu(x)\)를 원래 방정식에 곱하면 다음과 같은 식을 얻는다.

\[ \begin{align} \mu(x)\frac{dy}{dx} + \mu(x)P(x) \,y = \mu(x)Q(x). \end{align} \]

왼쪽 항을 정리하면 곱의 미분으로 표현될 수 있음을 알 수 있다. 즉,

\[ \begin{align} \mu(x)\frac{dy}{dx} + \mu(x)P(x) \,y = \frac{d}{dx}\bigl(\mu(x),y\bigr). \end{align} \]

여기에서 \(\mu(x)\)를 정의할 때 \(\mu(x)P(x) = \frac{d \mu(x)}{dx}\)라는 관계를 만족하게 만들었기 때문에 위와 같은 형태가 성립한다. 따라서 적분인자를 곱한 후의 방정식은

\[ \begin{align} \frac{d}{dx}\bigl(\mu(x) \,y\bigr) = \mu(x)Q(x) \end{align} \]

의 형태가 된다. 이 식을 \(x\)에 대해 적분하면 다음과 같은 해가 나온다.

\[ \begin{align} \mu(x) \,y = \int \mu(x)Q(x),dx + C \end{align} \]

여기서 \(C\)는 적분상수다. 마지막으로 적분인자 \(\mu(x)\)로 나누어주면 다음과 같이 \(y\)에 대한 해를 얻는다.

\[ \begin{align} y = \frac{1}{\mu(x)}\biggl[\int \mu(x)Q(x) \,dx + C\biggr]. \end{align} \]

\(\mu(x)\)가 지수함수 형태이므로 \(1/\mu(x)\) 역시 지수함수의 형태가 되어, 결과적으로 \(y\)를 명시적으로 구할 수 있다. 적분식에 따라 특정한 적분 결과가 나올 것이며, 초기 조건이나 경계 조건이 부여된다면 \(C\)를 정할 수 있다. 제어공학에서 주어진 입력 함수 \(Q(x)\)에 따라 시스템의 출력을 구하거나 동특성을 해석할 때 이 방식이 적용된다.

이 과정을 더 명확히 정리해보면 먼저 미분방정식

\[ \frac{dy}{dx} + P(x) \,y = Q(x) \]

를 본다. \(P(x)\)가 \(x\)에 대한 적분 가능 함수라고 가정할 때, 적분인자를

\[ \mu(x) = e^{\int P(x)\,dx} \]

로 정의한다. 그런 다음 식 전체에 \(\mu(x)\)를 곱하여

\[ \mu(x)\frac{dy}{dx} + \mu(x)P(x)\,y = \mu(x)Q(x) \]

를 얻는다. 왼편이 \(\frac{d}{dx}[\mu(x) y]\) 꼴이 되므로 이를 \(x\)에 대해 적분하면

\[ \mu(x),y = \int \mu(x)Q(x)\,dx + C \]

가 된다. 마지막으로

\[ y = \mu(x)^{-1}\bigl[\int \mu(x)Q(x)\,dx + C\bigr] \]

로 \(y\)를 해결할 수 있다.

이 과정을 간단한 흐름도로 나타내면 다음과 같다.

flowchart TB A((1계 미분방정식)) --> B{적분인자 정의} B --> C(양변에 적분인자 곱하기) C --> D(좌변을 합성 미분 형태로 재해석) D --> E(적분을 통해 해 구하기) E --> F(해를 명시적으로 표현)

위 흐름은 제어공학적 관점에서도 중요하다. 1차 동적 시스템에서 시스템의 미분방정식을 \(x\)에 관한 비제어 입력 \(Q(x)\)와 시스템 파라미터가 내재된 항 \(P(x)\)로 표현할 수 있기 때문이다. 적분인자 기법을 이용하면, 임의의 외부 입력 \(Q(x)\)에 대해서도 닫힌형 해를 비교적 쉽게 구할 수 있다.

적분인자 기법은 다음과 같은 특성을 가진다. 먼저, \(P(x)\)가 적분 가능한 함수여야 하므로 \(P(x)\)가 구간에서 연속인 경우가 많다. 제어공학 문제에서 물리 시스템의 계수 함수는 일반적으로 연속적이므로 이러한 가정은 실용적으로 충분하다. 적분인자가 지수 형태로 정의되기 때문에 방정식이 선형이 아닌 경우에는 직접 적용하기가 쉽지 않다. 하지만 1계 선형 미분방정식에 한정해서는 가장 단순하고도 직관적인 해법이 된다.

다음은 Python을 이용한 간단한 예시다. 예를 들어

\[ \begin{align} \frac{dy}{dx} + 2\,y = 4x \end{align} \]

라는 1계 미분방정식을 생각해보자. 여기서 \(P(x) = 2\)와 \(Q(x) = 4x\)다. 이때 적분인자는

\[ \begin{align} \mu(x) = e^{\int 2\,dx} = e^{2x} \end{align} \]

가 된다. 적분인자를 곱하면

\[ \begin{align} e^{2x}\frac{dy}{dx} + 2e^{2x}\,y = 4x,e^{2x} \end{align} \]

이 되고, 이는 왼편이

\[ \begin{align} \frac{d}{dx}\bigl(e^{2x}y\bigr) \end{align} \]

의 형태다. 따라서

\[ \begin{align} \frac{d}{dx}\bigl(e^{2x}y\bigr) = 4x\,e^{2x} \end{align} \]

를 적분하여

\[ \begin{align} e^{2x}y = \int 4x\,e^{2x}\,dx + C \end{align} \]

를 얻는다. 오른쪽 항은 적분기법(부분적분 등)을 통해 계산 가능하다. 실제로 적분을 수행하면

\[ \begin{align} \int 4x\,e^{2x}\,dx \end{align} \]

는 적분 결과로 \(2xe^{2x} - e^{2x} + K\) 꼴이 될 것이다. 적분 상수는 모두 \(C\)로 모아 쓸 수 있으므로

\[ \begin{align} e^{2x}y = 2xe^{2x} - e^{2x} + C. \end{align} \]

최종적으로 \(y\)를 구하면

\[ \begin{align} y = 2x - 1 + Ce^{-2x}. \end{align} \]

여기서 \(C\)는 어떤 초기 조건을 만족하도록 정해진다.

Python 코드로 적분을 간단히 확인할 수 있다.

import sympy as sp

x = sp.Symbol('x', real=True)
y = sp.Function('y')(x)
P = 2
Q = 4*x

mu = sp.exp(sp.integrate(P, (x)))
expr = Q*mu
solution = sp.integrate(expr, (x))
y_solution = solution/mu + sp.Symbol('C')/mu
print(y_solution.simplify())

코드를 실행하면 유사한 형태의 일반해가 도출된다. 이 방법은 선형 계수 \(P(x)\)가 주어지고, 외부 입력 \(Q(x)\)가 정해졌을 때 직접 해를 구할 수 있음을 보여준다. 제어공학에서 라플라스 변환을 사용하지 않고 직접 시간영역에서 해석할 필요가 있을 때, 이 적분인자 기법이 유용하다.

제어공학적 해석을 위해서는 주로 초기 조건 \(y(0) = y_0\) 등이 주어질 것이며, 이 조건으로부터 적분상수 \(C\)를 결정한다. 예를 들어, 초기 조건이 주어지면 \(x=0\) 대입으로 \(C\)를 구하여 완전한 해를 얻는다. 이렇게 구한 해는 시간영역에서 시스템의 과도응답과 정상상태 응답 등을 구체적으로 살펴보는 데 활용된다.

적분인자 기법은 다음과 같은 상황에서 매우 유용하다. 제어 시스템에서 1차계의 출력 미분방정식이 특정한 외부 입력 함수 \(Q(x)\)에 의해 구동될 때, 시간영역 해석을 직접 수행할 경우가 많다. 이때 \(P(x)\)가 단순 상수이면 적분인자는 \(e^{Px}\) 형태가 되며, 입력이 다항식, 지수함수, 사인파 등 친숙한 형태라면 적분 결과도 상대적으로 간단하게 얻을 수 있다. 따라서 이 방법을 통해 물리적인 해석과 함께 시스템의 응답을 해석하는 과정이 간소화된다.

1계 미분방정식을 라플라스 변환으로도 풀 수 있지만, 적분인자 기법은 미분방정식 해법 자체를 바로 이용하여 시간영역 해를 명시적으로 파악하는 장점이 있다. 제어공학에서 모델링한 시스템이 1차 성분만 포함하거나, 상수 계수 1차 선형 미분방정식으로 단순화된 경우에는 가장 직관적인 해석 방법 중 하나다. 또한 비선형 항이 추가되면 적용하기 어려워지지만, 1차 비선형 문제를 국소적으로 선형화(linearization)했을 때 해당 해석을 대략적으로 참조하는 경우도 있다.

이상으로 1계 미분방정식에서 적분인자 기법을 사용하는 방법과 유도 과정을 살펴보았다. 적분인자 기법은 1차 선형 미분방정식을 빠르고 확실하게 해석해 주는 중요한 도구다. 제어공학에서 종종 접하게 되는 단순 1차 동특성이나 입력응답 해석에 직접 적용 가능하므로, 시간을 다루는 문제에서 라플라스 변환을 사용하기 전에 이 기법으로도 충분히 문제를 해결할 수 있다. 특히 시간영역에서 직접적인 식을 얻고자 할 때 유용하다. 제어계 해석 과정에서 초기 조건과 함께 시스템 응답을 손쉽게 계산할 수 있으므로, 다양한 제어 실습 및 시뮬레이션 환경에서 널리 쓰인다.
