고계 상수계수 선형미분방정식은 제어공학에서 매우 자주 등장한다. 선형시스템을 해석할 때 고계 미분항을 갖는 시스템이 흔하며, 이를 해결하기 위해 고계 상수계수 선형미분방정식을 체계적으로 다룰 필요가 있다. 기본적으로 계수들이 모두 상수이며, 미분연산이 선형적이라는 점이 중요한 특징이다. 이 장에서는 고계 상수계수 선형미분방정식의 일반적 형태와 해의 구조를 체계적으로 살펴본다.

내용을 구성하기 위해 우선 고계 상수계수 선형미분방정식의 일반형을 정의하고, 특성방정식을 유도한다. 이후 특성방정식의 해에 따라 일반해를 구성하는 방법을 반복근, 실수근, 복소근 등으로 나누어 자세히 다룬다.

제어공학에서 고계 미분방정식은 흔히 입력과 출력 관계를 묘사하기 위해 사용된다. 예를 들어 \(n\)차 미분항까지 포함된 시스템을 고려하면, 출력과 그 미분항 및 입력 신호가 선형 결합 형태로 표현된다. 여기서는 우선 입력이 0인 경우(즉, 동특성만을 묘사하는 경우)를 중심으로 해석한다. 이후 초기조건이나 외부 입력이 있는 경우에도 해석이 가능해지지만, 이 장에서는 동특성을 이해하기 위한 기본 해석을 위주로 한다.

아래에서 고계 상수계수 선형미분방정식의 일반형과 그 해를 구하는 과정을 순차적으로 다룬다.

고계 상수계수 선형미분방정식의 일반형¶

일반적으로 \(n\)차 상수계수 선형미분방정식은 다음과 같은 형태를 가진다.

\[ \begin{align} \frac{d^n y(t)}{dt^n} + a_{n-1} \frac{d^{n-1} y(t)}{dt^{n-1}} + a_{n-2} \frac{d^{n-2} y(t)}{dt^{n-2}} + \dots + a_{1} \frac{d y(t)}{dt} + a_{0} \, y(t) = 0 \end{align} \]

여기서 \(y(t)\)는 시간 \(t\)에 대한 미지함수이고, \(a_{0}, a_{1}, \dots, a_{n-1}\)은 상수이다. \(n\)은 미분방정식의 계수를 의미한다. 계수가 상수라는 점 때문에, 시간에 따른 계수 변화가 없으며, 해를 구하기 위해 특성방정식을 활용할 수 있다.

\(n\)이 클수록 시스템의 동특성은 복잡해질 수 있다. 제어공학에서 실제 시스템은 높은 차수로 표현되는 경우가 흔하지만, 이론적으로는 어떤 \(n\)차라도 특성방정식을 이용해 동일한 절차로 해를 구할 수 있다.

특성방정식과 해의 기본 구조¶

위 미분방정식을 풀기 위한 전형적인 방법은 지수함수 \(e^{\lambda t}\) 꼴로 해를 가정하는 것이다. 즉 \(y(t) = e^{\lambda t}\)라고 가정하고, 이를 방정식에 대입하여 \(\lambda\)를 구한다. \(y(t) = e^{\lambda t}\)라고 하면 미분 연산은 다음과 같이 표현된다.

\[ \begin{align} \frac{d^k y(t)}{dt^k} = \lambda^k , e^{\lambda t} \end{align} \]

이 식을 원래 미분방정식에 대입하면 모든 항에 공통因子 \(e^{\lambda t}\)가 등장한다. \(e^{\lambda t}\)는 \(t\)에 대해 0이 되지 않으므로 소거할 수 있고, 결과적으로 다음과 같은 특성방정식을 얻게 된다.

\[ \begin{align} \lambda^n + a_{n-1}\,\lambda^{n-1} + a_{n-2}\,\lambda^{n-2} + \dots + a_{1}\,\lambda + a_{0} = 0 \end{align} \]

이를 \(n\)차 특성방정식이라고 하며, 모든 근(실근 또는 복소근)을 \(\lambda_1, \lambda_2, \dots, \lambda_n\)이라 할 때, 서로 다른 실근이 \(n\)개 존재한다고 가정하면 해는

\[ \begin{align} y(t) = C_1 e^{\lambda_1 t} + C_2 e^{\lambda_2 t} + \dots + C_n e^{\lambda_n t} \end{align} \]

꼴로 표현된다. 여기서 \(C_1, C_2, \dots, C_n\)은 적분상수이며, 초기조건 등을 통해 결정된다.

복소근이 존재하는 경우¶

특성방정식의 계수가 실수인 경우, 복소근은 반드시 켤레쌍 형태로 나타난다. 예를 들어 \(\lambda = \alpha \pm j \beta\)인 켤레쌍 근이 존재한다고 하자. 이때 \(j = \sqrt{-1}\)이며, \(\alpha, \beta\)는 실수이다. 복소수를 직접 사용할 수도 있지만, 실제 물리 문제에서 \(y(t)\)가 실수값을 가지려면 오일러 공식 \(e^{j \theta} = \cos(\theta) + j \sin(\theta)\)를 사용해 실수부로 표현해야 한다.

\(\alpha + j\beta\)와 \(\alpha - j\beta\) 두 근에서 유도되는 해를 합하면, 아래처럼 실수해 형태가 얻어진다.

\[ \begin{align} y(t) = e^{\alpha t}\Big(D_1 \cos(\beta t) + D_2 \sin(\beta t)\Big) \end{align} \]

여기서 \(D_1, D_2\)는 새로운 적분상수이다. \(\alpha\)가 음수면 시스템 응답이 점차 감쇠하고, 양수면 시간에 따라 발산한다. \(\alpha = 0\)이면 순수한 사인파와 코사인파 형태로 주기 운동이 나타난다.

반복근이 존재하는 경우¶

특성방정식에서 동일한 근이 여러 번 반복될 수도 있다. 예를 들어 \(\lambda_0\)가 중복도로 \(m\)만큼 반복되는 경우를 생각해보자. 이때 \((\lambda - \lambda_0)^m = 0\)이 특성방정식의 한 인수로 존재한다. 반복근이 있을 경우에는 \(e^{\lambda_0 t}\)만으로는 충분한 독립해를 만들 수 없다.

중복근 \(\lambda_0\)에 대해서는 다음과 같은 \(m\)개의 독립해가 존재한다.

\[ \begin{align} e^{\lambda_0 t},\quad t,e^{\lambda_0 t},\quad t^2,e^{\lambda_0 t}, \quad \dots,\quad t^{m-1} e^{\lambda_0 t} \end{align} \]

즉, \(m=2\)이면 \(C_1 e^{\lambda_0 t} + C_2 t e^{\lambda_0 t}\)가, \(m=3\)이면 \(C_1 e^{\lambda_0 t} + C_2 t e^{\lambda_0 t} + C_3 t^2 e^{\lambda_0 t}\)가 일반해에 포함된다.

초기조건 설정과 계수 결정¶

물리 시스템이나 제어 시스템에서 \(n\)차 미분방정식을 사용할 때는 초기조건이 주어진다. 예를 들어 \(y(0)\), \(y'(0)\), \dots, \(y^{(n-1)}(0)\) 등이 주어지면, 일반해에 포함된 \(C_1, C_2, \dots, C_n\)을 결정할 수 있다.

해가 실근, 복소근, 반복근 등 어떻게 구성되어 있든 간에, 궁극적으로 \(n\)개의 초기조건을 대입하여 \(n\)개의 상수를 구하게 된다. 이를 통해 실제 물리계나 제어계에서의 시간응답을 완전히 정의할 수 있다.

제어시스템에서의 해석적 중요성¶

고계 상수계수 선형미분방정식은 제어시스템 해석의 핵심 도구 중 하나다. \(n\)차 시스템이라면 특성방정식은 \(n\)차 다항식이 되며, 그 근(극, pole)이 시스템의 응답 특성을 결정한다. 실제 제어이론에서는 라플라스 변환을 사용해 전달함수로 표현하지만, 전달함수의 분모 다항식이 바로 이 특성방정식과 같다.

특성방정식의 근이 갖는 실수부, 허수부는 시스템 안정성, 감쇠 혹은 발산, 진동 등을 결정한다. 또한 이런 해석은 이산시스템에도 적절히 변환되어 적용될 수 있으며, 상태방정식의 해석에서도 동일한 수학적 뿌리를 갖는다.

아래는 Python(sympy)을 이용한 4차 상수계수 미분방정식 예시 풀이 코드다. 여기서는 임의의 계수 \(a_0, a_1, a_2, a_3\)를 사용한다.

예제 코드 (Python)¶

import sympy as sp

t = sp.Symbol('t', real=True)
a0, a1, a2, a3 = sp.symbols('a0 a1 a2 a3', real=True)
y = sp.Function('y')(t)

# 예시: 4차 상수계수 선형미분방정식
# d^4 y/dt^4 + a3 * d^3 y/dt^3 + a2 * d^2 y/dt^2 + a1 * dy/dt + a0*y = 0

ode = sp.Eq(y.diff(t,4) + a3*y.diff(t,3) + a2*y.diff(t,2) + a1*y.diff(t,1) + a0*y, 0)

# 심볼릭으로 해를 구한다
sol = sp.dsolve(ode)
print(sol)

이 코드를 실행하면 심볼릭 방식으로 일반해가 계산된다. 복소근이 존재하면 \(\mathrm{exp}(\alpha t)\cos(\beta t)\), \(\mathrm{exp}(\alpha t)\sin(\beta t)\) 꼴의 해가 포함되고, 반복근이 존재하면 \(t^k e^{\lambda_0 t}\) 꼴의 항이 추가되는 식으로 표현된다.

이러한 자유응답(free response)은 초기조건과 결합되어 실제 제어시스템의 시간응답을 결정한다. 외부 입력이 있는 경우에는 미분방정식의 오른쪽 항에 해당 입력이 포함되며, 이를 통해 강제응답(forced response)까지 구할 수 있다.

고계 방정식과 상태방정식¶

제어공학에서 \(n\)차 미분방정식을 직접 다루는 대신, 이를 상태방정식 형태로 변환해 해석하기도 한다. 예를 들어 \(n\)차 미분방정식을 다음과 같은 상태공간 표현으로 바꿀 수 있다.

\[ \begin{align} \frac{d,\mathbf{x}(t)}{dt} = \mathbf{A},\mathbf{x}(t) + \mathbf{B},u(t)\ y(t) = \mathbf{C},\mathbf{x}(t) + D,u(t) \end{align} \]

여기서 \(\mathbf{x}(t)\)는 차원이 \(n\)인 상태벡터이고, \(\mathbf{A}\)는 \(n \times n\) 행렬이다. 즉, \(n\)차 미분방정식을 1차 미분방정식 \(n\)개로 구성한 것과 동일하다. 행렬 \(\mathbf{A}\)의 고유다항식은 원래 미분방정식의 특성방정식과 일치하므로, 해석 방법은 결국 동일한 원리에 기반한다.

아래는 2차 계 예시를 블록선도로 단순화한 mermaid 표현이다.

flowchart LR A(("입력<br>u(t)")) --> B["2차 시스템 <br> d^2y/dt^2 + a1 dy/dt + a0 y = u(t)"] B --> C(("출력 <br>y(t)"))

이는 \(n\)차 계로 확장할 수도 있으며, 블록 내부에 상수계수 미분방정식이 존재한다고 이해하면 제어시스템 전체 구조를 파악하는 데 도움이 된다.

고계 상수계수 선형미분방정식은 제어공학에서 가장 기본적인 수학적 모델이며, 자유응답과 강제응답을 모두 설명할 수 있다. 해를 구하기 위해 특성방정식을 도출하고, 그 근에 해당하는 지수함수 해를 조합한다. 복소근은 사인·코사인 형태로 표현되며, 반복근이 있을 경우에는 \(t^k e^{\lambda t}\) 꼴의 해가 추가된다. 초기조건을 통해 적분상수를 결정하면 완전한 시간응답이 정해진다.

이런 해석은 라플라스 변환을 거쳐 전달함수로 확장되고, 상태방정식으로도 동일하게 설명된다. 결국 고계 상수계수 선형미분방정식의 특성해법을 충분히 이해하는 것이 제어공학 전반을 해석하는 핵심 역량이 된다.
