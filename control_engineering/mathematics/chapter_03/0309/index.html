특성방정식은 선형 상미분방정식을 풀 때 핵심이 되는 방법 중 하나다. 주어진 선형 미분방정식의 계수들이 상수(constant)라면, 즉 시간이나 독립변수에 의존하지 않는 계수들을 가지는 경우, 지수 함수를 해로 가정하여 미분방정식의 해를 구하게 된다. 이때 지수 해에서 나타나는 지수의 지수부(보통 \(r\)로 표기)를 결정하기 위해 풀어야 하는 방정식을 특성방정식이라 부른다. 특성방정식을 통해 구한 해의 형태는 미분방정식의 일반해를 표현하게 된다. 그리고 이러한 일반해는 여러 초기조건이나 경계조건을 통해 구체적인 해가 결정된다.

선형 상미분방정식에서 특성방정식이 왜 중요한지 이해하려면, 먼저 상수계수 선형 미분방정식의 표준형을 살펴볼 필요가 있다. 또한 해의 형태가 실수근, 복소근, 중근의 존재 여부에 따라 달라지며, 제어공학에서 시스템의 거동을 분석할 때 바로 이 특성근들이 시스템의 응답 특성을 좌우한다는 사실을 인지하는 것이 중요하다.

선형 상미분방정식의 기본 형태¶

가장 일반적인 \(n\)차 상미분방정식은 다음과 같은 형태로 쓸 수 있다.

\[ \frac{d^n y}{dt^n} + a_{n-1}\,\frac{d^{n-1} y}{dt^{n-1}} + \dots + a_{1}\,\frac{dy}{dt} + a_{0}\,y = 0 \]

위 식에서 \(a_{0}, a_{1}, \dots, a_{n-1}\)는 상수 계수이며, 독립변수 \(t\)에 의존하지 않는다. 이 방정식이 상미분방정식의 표준형이라고 할 수 있다. 예를 들어 2차 미분방정식의 경우에는

\[ \frac{d^2 y}{dt^2} + a_{1}\,\frac{dy}{dt} + a_{0}\,y = 0 \]

와 같은 형태가 된다. 이러한 선형 미분방정식을 풀 때는, 지수 형태의 해를 가정함으로써 문제를 단순화할 수 있다. 즉, \(y = e^{rt}\)와 같은 꼴을 대입하여 미분방정식을 풀면, 미분을 수행할 때마다 지수함수의 형태가 바뀌지 않고 상수 배만 생기기 때문에, 해를 찾기가 훨씬 쉬워진다. 그런데 이때 등장하는 \(r\)에 대한 방정식이 바로 특성방정식이다.

특성방정식의 정의¶

상기의 \(n\)차 상미분방정식에 \(y = e^{rt}\)를 대입해보면 다음과 같은 과정을 거친다. 우선 \(\frac{d^k}{dt^k} (e^{rt}) = r^k e^{rt}\)라는 사실을 이용한다. 이를 \(n\)차 방정식에 대입하면,

\[ \frac{d^n}{dt^n}(e^{rt}) + a_{n-1}\,\frac{d^{n-1}}{dt^{n-1}}(e^{rt}) + \dots + a_{1}\,\frac{d}{dt}(e^{rt}) + a_{0}\,e^{rt} = 0 \]

가 된다. 여기서 각 항은 \(r^n e^{rt}, r^{n-1} e^{rt}, \dots, r e^{rt}, e^{rt}\)의 형태를 띠게 되므로, 공통인수 \(e^{rt}\)를 묶어낼 수 있다. 그러면 다음과 같은 식을 얻는다.

\[ e^{rt} \,\bigl(r^n + a_{n-1}\,r^{n-1} + \dots + a_{1}\,r + a_{0}\bigr) = 0 \]

위 식에서 \(e^{rt} \neq 0\)이므로, 미분방정식을 만족시키려면 괄호 안의 다항식이 \(0\)이 되어야 한다. 즉,

\[ r^n + a_{n-1}\,r^{n-1} + \dots + a_{1}\,r + a_{0} = 0 \]

이 되며, 이 다항식을 바로 미분방정식의 특성방정식이라 한다. 따라서 상수계수 선형 상미분방정식을 풀 때 핵심은 이 특성방정식을 풀어 그 해인 rr을 구하는 것이고, 그 해의 종류(실근, 복소근, 중근 등)에 따라 최종해의 형태가 달라진다.

특성근의 해석¶

특성방정식 \(r^n + a_{n-1}r^{n-1} + \dots + a_{1}r + a_{0} = 0\)을 풀어서 얻은 해들을 특성근이라고 부른다. 예를 들어 2차 미분방정식의 특성방정식이 \(r^2 + a_{1}r + a_{0} = 0\)일 때, 이 이차방정식을 풀어 구해지는 두 근 \(r_1, r_2\)가 곧 특성근이 된다.

제어공학에서 이 특성근들은 폐루프 시스템의 극점(pole)과 밀접한 관련이 있다. 예를 들어 상태방정식을 통해 유도한 특성방정식의 근은 시스템의 고유값(eigenvalue)이고, 이들이 시스템 응답의 안정성과 동특성을 결정한다. 실수부가 음수인지, 영인지, 양수인지에 따라 감쇠, 영감쇠, 발산 거동이 달라진다. 복소근이 존재한다면 시스템은 진동 응답을 보이게 된다.

경우별 해의 형태¶

특성방정식으로부터 구한 근이 모두 서로 다른 실수근인 경우, 해는 각 근에 대응되는 지수함수들의 선형결합으로 표현된다. 예를 들어,

\[ r_1, r_2, \dots, r_n \]

이 서로 다른 실수라면, 미분방정식의 일반해는

\[ y(t) = C_1 \, e^{r_1 t} + C_2 \, e^{r_2 t} + \dots + C_n \, e^{r_n t} \]

와 같이 쓸 수 있다. 여기서 \(C_1, C_2, \dots, C_n\)은 적절한 초기조건이나 경계조건에 의해 결정되는 상수이다.

특성근 중에서 어떤 근이 서로 중복된다면, 즉 중근(이중근, 삼중근 등)이 존재한다면, 단순히 지수함수의 선형결합만으로는 모든 해를 나타낼 수 없다. 예를 들어 2차 방정식의 특성근이 \(r_1 = r_2 = r\)인 경우, 일반해는

\[ y(t) = C_1 \, e^{r t} + C_2 \, t e^{r t} \]

처럼 지수함수에 \(t\)가 곱해진 항이 추가로 포함된다. 일반적으로 중복도가 \(m\)인 근 \(r\)에 대해서는

\[ e^{r t},\quad t\,e^{r t},\quad t^2\,e^{r t},\dots,\quad t^{m-1}\,e^{r t} \]

가 모두 해의 부분해가 되고, 이들을 선형결합한 형태가 미분방정식의 해가 된다.

복소근이 존재할 경우, 보통 제어공학에서 실제 시스템에선 계수가 모두 실수이므로 특성근은 켤레쌍(conjugate pair)으로 등장한다. 예를 들어 \(\alpha \pm j\beta\)와 같은 형태의 켤레 복소근이 있다면, 지수함수와 사인·코사인 항이 결합된 형태로 해가 나타난다. 즉, 2차 미분방정식에서

\[ r_{1,2} = \alpha \pm j\beta \]

라면, 일반해는

\[ y(t) = e^{\alpha t}\Bigl(C_1 \cos(\beta t) + C_2 \sin(\beta t)\Bigr) \]

로 나타난다. 이 해를 다시 복소지수함수를 이용해 표현할 수도 있지만, 제어공학에서는 주로 감쇠(또는 증폭)되는 진동 형태로 해석하곤 한다. 이때 \(\alpha\)는 실수부로서 시스템의 감쇠(또는 발산)을 결정하고, \(\beta\)는 진동 주파수를 결정한다.

고차 미분방정식에서의 일반해 확장¶

보다 높은 차수의 미분방정식에서도 원리는 동일하다. 예를 들어 3차 미분방정식을 예로 들면, 특성방정식은 3차 다항식이 된다. 그 해가 모두 구별되는 실근 3개인 경우 해는 각 지수함수의 선형결합이다. 실근 1개와 복소 켤레근 1쌍인 경우에는 지수함수 해 하나와 감쇠(또는 증폭) 진동 해가 함께 나타나는 형태가 된다. 또는 세 근이 모두 중첩될 수도 있으며, 이 경우엔 t, t^2와 같은 항이 추가로 발생하는 식으로 확장된다.

이처럼 특성방정식을 통해 얻은 근들의 조합에 따라 최종해의 형태는 달라지지만, 결과적으로 해는 모든 근에 대응되는 보조함수(지수함수, 지수함수와 사인·코사인, t배 지수함수 등)의 선형결합으로 정리된다. 이는 선형대수학적 해석에서 “기저 함수를 이루는 기본 해들의 선형결합이 곧 미분방정식의 일반해”라는 사실과도 일맥상통한다.

상태방정식과의 관련성¶

제어공학에서 흔히 쓰이는 상태방정식을

\[ \frac{d\mathbf{x}}{dt} = \mathbf{A}\,\mathbf{x} \]

와 같은 꼴로 두었을 때, 이 계(system)의 특성방정식은

\[ \det(\mathbf{A} - \lambda \mathbf{I}) = 0 \]

이 된다. 이는 결국 행렬 \(\mathbf{A}\)의 고유값 \(\lambda\)를 찾는 문제와 동일하며, 고유값 \(\lambda\)들이 바로 위에서 언급한 특성근 \(r\)의 역할을 한다. 따라서 행렬 \(\mathbf{A}\)의 고유값에 따라 해의 형태, 즉 시스템 응답이 결정된다. 예를 들어 실수 고유값이 음수면 감쇠 지수 응답, 복소 켤레근이라면 진동 응답이 나타난다. 그리고 고유값의 중복 여부에 따라 해에 다항식 항이 추가되는 형태가 발생하기도 한다.

이는 전이행렬 \(\mathbf{\Phi}(t)\)로부터 해를 구하는 과정에서 명확히 나타난다. 전이행렬 \(\mathbf{\Phi}(t)\)는 \(\mathbf{A}\)의 대각화 혹은 조르당 표준형을 통해 유도 가능하며, 결국 \(\mathbf{A}\)의 고유값(특성근)과 고유벡터(또는 일반화 고유벡터)가 해의 형태를 결정한다.

예시와 적용¶

실제 예시로 2차 미분방정식

\[ \frac{d^2 y}{dt^2} + 3\,\frac{dy}{dt} + 2\,y = 0 \]

를 풀어보자. 특성방정식은

\[ r^2 + 3r + 2 = 0 \]

이다. 이 이차방정식을 인수분해하면,

\[ (r + 1)(r + 2) = 0 \]

이므로, 특성근 \(r_1 = -1, r_2 = -2\)를 얻게 된다. 이는 모두 서로 다른 실수근이므로, 해의 일반형은

\[ y(t) = C_1\,e^{-t} + C_2\,e^{-2t} \]

가 된다. 적절한 초기조건이 주어지면 \(C_1, C_2\) 값을 구할 수 있다. 이 해는 제어공학 관점에서 보면 감쇠 지수 응답으로 해석된다. 특성근이 음수이므로 시스템은 안정적으로 원점으로 수렴하는 형태를 보인다.

아래는 Python을 활용하여 이 미분방정식을 상징적으로 풀어보는 간단한 예시 코드다.

import sympy

t = sympy.Symbol('t', real=True)
y = sympy.Function('y')(t)
ode = sympy.Eq(sympy.diff(y, (t, 2)) + 3*sympy.diff(y, t) + 2*y, 0)
sol = sympy.dsolve(ode)
print(sol)

이 코드를 실행하면 Sympy가 미분방정식을 풀어 일반해를 출력해준다. 그 결과는 수작업으로 구한 \(y(t) = C_1\,e^{-t} + C_2\,e^{-2t}\)와 동일하다.

graph LR A((시작)) --> B[미분방정식 수립] B --> C{특성방정식 해} C --> D[실근] C --> E[복소근] C --> F[중근] D --> G[지수함수 선형결합] E --> H[진동 & 지수 결합] F --> I[다항식 & 지수 결합]

이 도식은 특성근의 종류(실근, 복소근, 중근)에 따라 해 형태가 달라짐을 직관적으로 나타낸다. 결국 결론은 특성방정식에서 구한 모든 근에 대응되는 독립해를 모두 선형결합한 형태가 최종 일반해가 된다는 점이다.

특성방정식은 상수계수 선형 상미분방정식을 풀이하는 데 있어 핵심적인 역할을 한다. 이를 통해 얻은 특성근들은 시스템 응답의 지배적인 형태를 결정하며, 제어공학에서도 시스템의 극점, 고유값 해석 등에 바로 적용된다. 실근이냐 복소근이냐, 또 근이 중복되느냐에 따라 해의 구조가 달라지지만, 근본적으로 특성방정식의 뿌리를 해석하는 것이 미분방정식을 푸는 출발점이라 할 수 있다. 이는 고차 미분방정식이나 행렬 미분방정식으로 확장되었을 때도 마찬가지며, 실제 시스템의 안정성과 동특성을 평가하는 핵심으로서 제어공학 전반에 걸쳐 활용된다.
