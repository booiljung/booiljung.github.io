초등함수의 미적분 이론은 미분방정식을 다루는 데 있어 필수적인 도구를 제공한다. 다항함수, 지수함수, 로그함수, 삼각함수 등으로 대표되는 초등함수는 대부분의 공학적 문제에서 광범위하게 활용된다. 초등함수의 미분과 적분을 정확히 이해하면 미분방정식을 해석하고 해를 구하는 과정이 훨씬 수월해진다. 미분과 적분의 기초는 연속성, 극한, 미분의 정의, 적분의 정의를 바탕으로 한다. 여기서는 초등함수에 대해 미분과 적분을 체계적으로 다룬다.

미분법과 적분법을 배우기 전, 초등함수가 무엇인지 간단히 살펴보고 그 분류를 시각적으로 표현한다.

flowchart LR A[초등함수] --> B[다항함수] A --> C[유리함수] A --> D[지수함수] A --> E[로그함수] A --> F[삼각함수] A --> G[쌍곡함수]

초등함수는 합성, 곱, 몫 등 유한한 대수적 연산과 합성 과정으로 만들어진 함수들을 의미한다. 예를 들어 다항함수, 유리함수, 지수함수, 로그함수, 삼각함수, 쌍곡함수 등이 대표적인 초등함수에 속한다.

다항함수의 미분¶

다항함수는 가장 간단한 형태의 초등함수다. 다항함수 \(p(x)\)는 유한 차수의 항들을 합한 형태로 표현된다. 예를 들어

\[ p(x) = a_n x^n + a_{n-1} x^{n-1} + \dots + a_1 x + a_0 \]

처럼 나타난다. 미분은 한 변수가 변화할 때 함수가 어떻게 변하는지를 측정하는 도구이다. 다항함수는 미분법 중에서도 기본 규칙만으로 손쉽게 다룰 수 있다.

먼저 다항함수의 일반항 \(x^n\)에 대한 미분을 생각한다. \(n\)이 자연수일 때

\[ \frac{d}{dx} x^n = n x^{n-1} \]

이 된다. 다항함수 전체에 적용하면 선형성(linearity)에 의해 항별로 미분을 수행할 수 있다.

다음 예시로 다항함수 \(f(x) = 3x^4 - 2x^2 + 5\)를 미분한다. 전개 과정을 단계별로 살펴본다.

\[ f(x) = 3x^4 - 2x^2 + 5 \]
\[ \frac{d}{dx} f(x) = \frac{d}{dx}(3x^4) - \frac{d}{dx}(2x^2) + \frac{d}{dx}(5) \]
\[ \frac{d}{dx} f(x) = 3 \cdot 4 x^{4-1} - 2 \cdot 2 x^{2-1} + 0 \]
\[ \frac{d}{dx} f(x) = 12x^3 - 4x \]

이로부터 다항함수의 미분 법칙은 항별로 쉽고 간단하게 적용된다는 사실을 확인할 수 있다.

유리함수의 미분¶

유리함수는 두 다항함수의 비로 표현되는 함수다. 예를 들어

\[ r(x) = \frac{p(x)}{q(x)} \]

와 같은 형태다. 유리함수의 미분은 몫의 미분(Quotient Rule)을 통해 구할 수 있다. 몫의 미분은

\[ \left(\frac{u}{v}\right)' = \frac{u'v - uv'}{v^2} \]

로 주어진다. 여기서 \(u = p(x), v = q(x)\)라 하면, \(u' = p'(x), v' = q'(x)\)로 표현된다.

예시로

\[ r(x) = \frac{x^2 + 1}{x - 1} \]

을 미분해본다.

\[ u(x) = x^2 + 1, \quad v(x) = x - 1 \]
\[ u'(x) = 2x, \quad v'(x) = 1 \]
\[ r'(x) = \frac{ (2x)(x - 1) - (x^2 + 1)(1) }{(x - 1)^2 } \]
\[ r'(x) = \frac{ 2x^2 - 2x - x^2 - 1}{(x - 1)^2 } \]
\[ r'(x) = \frac{ x^2 - 2x - 1}{(x - 1)^2} \]

이와 같이 유리함수의 미분은 몫의 미분법을 적용해 간단한 대수적 변형으로 구할 수 있다.

지수함수의 미분¶

지수함수는 \(a^x\) (주로 \(a > 0\)) 형태로 나타난다. 가장 많이 사용되는 특별한 경우로 자연상수 \(e\)를 밑으로 한 지수함수 \(e^x\)가 있다. 지수함수 \(e^x\)의 미분은 자기 자신이 된다는 중요한 성질이 있다.

\[ \frac{d}{dx} e^x = e^x \]

밑이 다른 지수함수 \(a^x\)는

\[ a^x = e^{x \ln(a)} \]

로 나타낼 수 있으므로

\[ \frac{d}{dx} a^x = \ln(a) \cdot a^x \]

가 된다.

로그함수의 미분¶

로그함수는 지수함수의 역함수다. 밑이 \(a\)인 로그함수는 \(\log_a (x)\)로 표현되며 \(a^{\log_a(x)} = x\) 관계를 갖는다. 공학에서는 주로 자연로그 \(\ln(x) = \log_e(x)\)가 사용된다. 자연로그의 미분은

\[ \frac{d}{dx} \ln(x) = \frac{1}{x} \]

이며 밑이 \(a\)인 로그함수의 미분은 밑변환 공식을 통해

\[ \frac{d}{dx} \log_a(x) = \frac{1}{x \ln(a)} \]

로 정리된다.

삼각함수의 미분¶

삼각함수 \(\sin(x), \cos(x), \tan(x)\) 등은 주기성을 가지며 제어공학에서 신호의 주기를 표현할 때 자주 등장한다. 기본 미분 공식은 다음과 같다.

\[ \frac{d}{dx}\sin(x) = \cos(x) \]
\[ \frac{d}{dx}\cos(x) = -\sin(x) \]
\[ \frac{d}{dx}\tan(x) = \sec^2(x) \]

그 외의 삼각함수 \(\csc(x), \sec(x), \cot(x)\)에 대해서도 유도 과정을 거쳐 각각

\[ \begin{align} \frac{d}{dx}\csc(x) &= -\csc(x)\cot(x),\\ \frac{d}{dx}\sec(x) &= \sec(x)\tan(x),\\ \frac{d}{dx}\cot(x) &= -\csc^2(x) \end{align} \]

가 된다.

합성함수의 미분¶

합성함수는 여러 초등함수를 합성하여 얻어지며 미분에는 연쇄법칙(Chain Rule)을 적용한다. 합성함수 \(f(g(x))\)에 대해 연쇄법칙은 다음과 같이 표현된다.

\[ \frac{d}{dx}f(g(x)) = f'(g(x)) \cdot g'(x) \]

예를 들어 \(f(x) = \sin(x)\)와 \(g(x) = x^2 + 1\)을 합성한 함수 \(h(x) = f(g(x)) = \sin(x^2 + 1)\)를 미분해보면

\[ h'(x) = \cos(x^2 + 1) \cdot 2x \]

이 된다.

초등함수의 적분¶

적분은 미분의 역연산이며, 넓이 혹은 누적변화량을 나타내는 중요한 개념이다. 초등함수의 적분에서도 기본 공식을 암기하고 적절한 기법을 적용하면 된다. 적분은 크게 부정적분과 정적분으로 구분된다.

부정적분은 미분연산의 역연산이므로 미분했을 때 원래 함수가 나오는 모든 함수를 구하는 과정이다. 적분 기호 \(\int\) 뒤에 적분하고자 하는 함수를 쓴 뒤 적분상수 \(+C\)를 붙인다. 예를 들어 다항함수 \(x^n\)의 부정적분은

\[ \int x^n \, dx = \frac{x^{n+1}}{n+1} + C \quad (n \neq -1) \]

정적분은 구간 \([a, b]\)에서 적분값을 구하는 것이며, 미적분학의 기본정리에 의해 다음과 같이 표현된다.

\[ \int_a^b f(x)\, dx = F(b) - F(a) \]

여기서 \(F(x)\)는 \(f(x)\)의 임의의 부정적분(원시함수)을 의미한다.

다항함수의 적분 예시¶

다항함수 \(p(x) = x^3 - 2x + 1\)의 부정적분은 항별로 적분하여 합하는 방식으로 쉽게 구한다.

\[ \begin{align} \int \bigl(x^3 - 2x + 1\bigr)\, dx &= \int x^3\, dx - \int 2x\, dx + \int 1\, dx \\ &= \frac{x^4}{4} - x^2 + x + C \end{align} \]

여기서 \(C\)는 적분상수다.

지수함수와 로그함수의 적분¶

지수함수 \(e^x\)의 적분은 미분과 마찬가지로 자기 자신이 된다.

\[ \int e^x \, dx = e^x + C \]

밑이 \(a\)인 지수함수의 적분은

\[ \int a^x \, dx = \frac{a^x}{\ln(a)} + C \]

로 주어진다.

로그함수 \(\ln(x)\)의 적분은 부분적분법으로 구할 수 있다.

\[ \int \ln(x)\, dx = x \ln(x) - x + C \]

이 식을 유도하기 위해서는 \(u = \ln(x), dv = dx\)로 두어 부분적분을 적용한다. 전개 과정을 살펴보면

\[ u = \ln(x), \quad du = \frac{1}{x} dx \]
\[ dv = dx, \quad v = x \]
\[ \begin{align} \int \ln(x)\, dx &= x \ln(x) - \int x \cdot \frac{1}{x}\, dx \\ &= x \ln(x) - \int 1\, dx \\ &= x \ln(x) - x + C \end{align} \]

삼각함수의 적분¶

삼각함수의 적분에서도 일부 공식을 기본적으로 기억하거나 유도해낼 수 있어야 한다.

\[ \int \sin(x)\, dx = -\cos(x) + C \]
\[ \int \cos(x)\, dx = \sin(x) + C \]
\[ \int \tan(x)\, dx = -\ln|\cos(x)| + C \]
\[ \int \sec^2(x)\, dx = \tan(x) + C \]

이외에도 복잡한 삼각함수 적분은 다양한 삼각함수 항등식을 사용하거나 치환적분을 통해 해결한다.

정적분과 미적분학의 기본정리¶

정적분은 넓이나 누적되는 값을 나타내며, 미분과 정적분 사이의 관계는 미적분학의 기본정리에 의해 다음과 같이 요약된다.

\[ \frac{d}{dx} \left( \int_a^x f(t)\, dt \right) = f(x) \]

또한

\[ \int_a^b f(x)\, dx = F(b) - F(a) \]

에서 \(F'(x) = f(x)\)를 만족하는 \(F(x)\)를 원시함수라고 한다. 이 공식은 적분 계산을 실제로 수행할 때 핵심이 된다. 적분의 결과는 원시함수를 구한 뒤, 구간의 양 끝점에서 값을 평가한 뒤 빼주는 형태로 구해진다.

간단한 예제 코드 (Python)¶

초등함수의 미분과 적분을 빠르게 계산하기 위해 컴퓨터를 활용하는 예시를 든다. Python에서 sympy 라이브러리를 사용하면 상징적(symbolic) 미분과 적분을 손쉽게 구할 수 있다.

아래는 다항함수 \(f(x) = x^3 - 2x + 1\)에 대한 미분과 적분을 구하는 예시다.

import sympy as sp

x = sp.Symbol('x', real=True)
f = x**3 - 2*x + 1

diff_f = sp.diff(f, x)
int_f = sp.integrate(f, (x))

print("f(x) =", f)
print("f'(x) =", diff_f)
print("∫f(x)dx =", int_f)

코드를 실행하면 다음과 같은 결과가 나온다.

\[ f(x) = x^3 - 2*x + 1 \]
\[ f'(x) = 3 \times x^2 - 2 \]
\[ ∫f(x)dx = \frac{x^4}{4} - x^2 + x \]

초등함수의 미적분은 미분방정식을 포함한 제어공학 전반에서 사용되는 핵심 이론이다. 다항함수, 유리함수, 지수함수, 로그함수, 삼각함수 등은 대부분의 물리적, 공학적 상황에서 자주 등장하는 기본적인 표현 도구다. 이 함수들의 미분과 적분 공식, 그리고 필요한 기법들을 정확히 숙지하면 다양한 공학 문제를 효율적으로 해결할 수 있다.

다항함수는 항별 미분과 적분이 용이하며, 유리함수는 곱과 몫의 미분법을 적용한다. 지수함수와 로그함수는 서로 역함수 관계를 통해 미분과 적분이 간결하게 정리된다. 삼각함수는 주기적 성질을 가지므로 그 미분 적분 공식을 암기하거나 도출해두면 다양한 응용 문제에 유용하다. 모든 초등함수에 대한 적분과 미분은 치환적분, 부분적분, 연쇄법칙 등 기초 기법으로 해결할 수 있으며, 이는 더 복잡한 미분방정식을 풀거나 해를 해석하는 데 있어 필수적인 기반을 제공한다.
