복소수는 실수로만 설명하기 힘든 다양한 물리계나 제어계에서 유용하게 사용된다. 예를 들어 선형 시스템 해석, 전달함수의 특성 분석, 고유값을 통한 시스템 안정성 평가 등에서 복소수의 개념이 핵심적인 역할을 한다. 복소수는 \(j\)를 허수 단위로 하며, \(j^2 = -1\)이라는 특별한 성질을 가진다. 이를 이용하면 2차원 실좌표계를 한 번에 다루는 연산이 가능해진다. 복소수는 실수부와 허수부로 이루어지며, 실수부를 \(a\), 허수부를 \(b\)라고 하면 일반적으로 \(a + jb\)와 같은 형태로 표현한다. 제어공학을 포함한 여러 분야에서 복소수를 자유자재로 다룰 수 있게 되면, 다양한 시스템 해석과 설계 과정이 수월해진다.

복소수의 정의와 기본 성질¶

복소수는 실수와 허수로 이루어진다는 점에서, 실직선 하나로 표현되는 실수보다 한 차원 높은 2차원 평면에서 해석된다. 이 평면을 복소평면이라 부르며, 가로축(수평축)은 실수축, 세로축(수직축)은 허수축으로 놓고, 점 \((a,b)\)를 복소수 \(a + jb\)로 대응시킨다.

실수부를 \(\Re(z)\), 허수부를 \(\Im(z)\)라고 표기할 때, 복소수 \(z\)가 다음과 같이 주어진다고 하자. 인라인으로 표현하면 \(z = a + jb\)이며, 여기서 \(a\)와 \(b\)는 모두 실수다. 이를 복소평면에서 좌표 \((a,b)\)로 해석하는 순간, 복소수를 벡터처럼 취급할 수 있게 된다. 벡터 표기로 나타내면 $$ \begin{align} \mathbf{z} = \begin{pmatrix} a \ b \end{pmatrix} \end{align} $$ 처럼 볼 수도 있는데, 단순히 좌표평면상의 한 점과 동일하다. 다만 복소수만의 특징은 \(j^2 = -1\)이라는 점이다. 이로 인해 실수범위 안에서는 불가능했던 ‘회전’을 단일 곱셈 연산으로 표현하는 등, 복소수를 이용한 간결한 계산이 가능해진다.

복소수의 복소켤레는 허수부의 부호를 바꾼 것으로 정의된다. 복소수 \(z = a + jb\)에 대해 \(z\)의 복소켤레를 \(\overline{z}\)로 표현하면 인라인으로 \(\overline{z} = a - jb\)가 된다. 복소켤레는 제어공학에서 전달함수 해석이나 상태방정식의 고유값 분석에서 자주 등장한다. 복소켤레의 핵심 성질 중 하나는 어떤 복소수 \(z\)를 실수화할 때 자주 쓰인다는 점이다. 예를 들어 \(z \cdot \overline{z}\)를 계산하면, 결과가 실수인 \(a^2 + b^2\)가 된다.

복소수의 크기는 \(|z|\) 혹은 \(r\)로 표현하며, 이는 피타고라스 정리를 이용해 구한다. 복소수 \(z = a + jb\)에 대해 크기 \(r = \sqrt{a^2 + b^2}\)이다. 복소수의 위상각(Argument)은 \(\theta\)로 부르며, 복소평면에서 실수축과 벡터 \(\mathbf{z}\)가 이루는 각도다. 이를 \(\arg(z)\)로 표기하면 $$ \begin{align} \theta = \tan^{-1}!\biggl(\dfrac{b}{a}\biggr) \end{align} $$ 로 정의된다. \(\theta\)는 복소수의 회전 성분을 나타내므로, 주파수 영역 해석이나 위상 보상 설계 등에서 중요한 역할을 담당한다.

복소수의 사칙연산¶

복소수의 덧셈과 뺄셈은 실수부끼리, 허수부끼리 각각 연산한다. 두 복소수 \(z_1 = a_1 + jb_1\), \(z_2 = a_2 + jb_2\)가 주어졌을 때, 덧셈은 $$ \begin{align} z_1 + z_2 &= (a_1 + jb_1) + (a_2 + jb_2) \[1mm] &= \boxed{(a_1 + a_2) + j(b_1 + b_2)} \end{align} $$ 와 같이 단순히 실수끼리, 허수끼리 더하면 된다. 뺄셈도 마찬가지 방식이다.

곱셈은 \(j^2=-1\)이라는 성질을 반영하여 전개한다. 예를 들어 \(z_1 = a_1 + jb_1\), \(z_2 = a_2 + jb_2\)의 곱셈은 $$ \begin{align} z_1 \cdot z_2 &= (a_1 + jb_1)(a_2 + jb_2) \ &= a_1 a_2 + a_1 (jb_2) + (jb_1) a_2 + (jb_1)(jb_2) \ &= a_1 a_2 + j a_1 b_2 + j b_1 a_2 + j^2 b_1 b_2 \ &= a_1 a_2 + j(a_1 b_2 + b_1 a_2) + (j^2)(b_1 b_2) \ &= a_1 a_2 + j(a_1 b_2 + b_1 a_2) - b_1 b_2 \ &= \boxed{(a_1 a_2 - b_1 b_2) + j(a_1 b_2 + b_1 a_2)} \end{align} $$ 이 과정을 통해 복소수의 곱에서 실수부가 \((a_1 a_2 - b_1 b_2)\), 허수부가 \((a_1 b_2 + b_1 a_2)\)로 구해짐을 알 수 있다.

나눗셈은 복소수를 실수 형태로 만들기 위해 켤레를 활용한다. \(z_1 / z_2\)를 계산하기 위해서는 분모에 \(z_2\)의 켤레를 곱하여 실수로 만든 뒤, 기약화한다. 예를 들어 $$ \begin{align} \dfrac{z_1}{z_2} &= \dfrac{a_1 + jb_1}{a_2 + jb_2} \ &= \dfrac{(a_1 + jb_1)(a_2 - jb_2)}{(a_2 + jb_2)(a_2 - jb_2)}\[1mm] &= \boxed{\dfrac{(a_1 + jb_1)(a_2 - jb_2)}{a_2^2 + b_2^2}} \end{align} $$ 가 된다. 분모는 실수 \(a_2^2 + b_2^2\)로 나타나므로, 결국 결과가 \(x + jy\) 형태로 표현될 수 있다. 이처럼 켤레를 적절히 이용하면 분모를 실수화하여 연산을 수월하게 진행할 수 있다.

복소수의 극형식과 지수형 표현¶

복소수는 사각좌표(직교좌표) 형태인 \(a + jb\) 외에도 극좌표 형태로 표현이 가능하다. 크기 \(r\)와 위상각 \(\theta\)를 이용해 $$ \begin{align} z = r\bigl(\cos \theta + j \sin \theta \bigr) \end{align} $$ 로 나타내는 방식을 극형식이라 한다. 여기서 \(r = \sqrt{a^2 + b^2}\), \(\theta = \tan^{-1}(b/a)\)로서 복소수의 기본 성질을 그대로 반영한다. 극형식은 곱셈 연산 시 매우 편리하다. 두 복소수 \(z_1 = r_1(\cos \theta_1 + j\sin \theta_1)\), \(z_2 = r_2(\cos \theta_2 + j\sin \theta_2)\)의 곱을 해보면 $$ \begin{align} z_1 \cdot z_2 &= r_1 r_2 \Bigl[\bigl(\cos \theta_1 + j\sin \theta_1 \bigr)\bigl(\cos \theta_2 + j\sin \theta_2 \bigr)\Bigr] \ &= r_1 r_2 \Bigl[\cos(\theta_1 + \theta_2) + j \sin(\theta_1 + \theta_2)\Bigr] \end{align} $$ 로 나타난다. 복소수 크기는 곱해지고, 위상각은 더해지는 형태다. 위상 요소가 각의 합으로 단순히 연산되는 것은 신호 해석이나 주파수 해석에서 큰 장점을 제공한다.

지수함수를 이용한 표현도 자주 쓰인다. 오일러 공식은 $$ \begin{align} e^{j\theta} = \cos \theta + j\sin \theta \end{align} $$ 라는 아름다운 관계를 제시한다. 이를 활용하면 복소수를 지수형으로 $$ \begin{align} z = r e^{j\theta} \end{align} $$ 로 표현할 수 있다. 이 지수형 표현은 해석 과정이나 미분방정식 풀이 등에서 기하학적 의미를 간단하게 처리할 수 있게 해 준다. 특히 라플라스 변환이나 푸리에 변환에서 \(e^{j \omega t}\) 형태로 나타나는 진동 신호 해석에 유리하며, 동특성 해석 시 복소수가 필수적으로 등장한다.

복소수 연산 예제 (Python)¶

복소수를 다루는 여러 언어 중 Python을 예로 들어 복소수 연산을 간단히 확인해볼 수 있다. Python의 내장 complex 타입은 \(j\) 대신 \(j\)와 동일한 의미로 \(1j\) 혹은 \(1J\) 등을 사용할 수 있다. 다음은 복소수 덧셈, 곱셈, 크기와 위상각을 구하는 간단한 코드 예다.

import cmath

z1 = complex(1, 2)  # 1 + j2
z2 = complex(3, -1) # 3 - j1

z_sum = z1 + z2
z_mul = z1 * z2

mod_z1 = abs(z1)
arg_z1 = cmath.phase(z1)

print("z1 + z2 =", z_sum)
print("z1 * z2 =", z_mul)
print("|z1| =", mod_z1)
print("arg(z1) =", arg_z1)

이 코드를 실행하면 z1과 z2의 합, 곱, 그리고 z1의 크기와 위상각을 구해준다. 복소수 연산이 얼마나 편리하게 이루어지는지를 확인할 수 있다.

복소수를 사각좌표 형태로 입력한 뒤 극좌표 정보를 구할 수도 있고, 반대로 극좌표 형태에서 원하는 사각좌표 형태로 바꿀 수도 있다. 이런 변환은 제어공학에서 전달함수의 영점과 극점을 복소평면 위에서 시각화할 때, 혹은 안정도와 주파수 응답특성을 살펴볼 때 활용된다.

복소수는 결국 2차원 벡터를 편리하게 표현하는 수학적 도구다. 제어 문제에서 자주 등장하는 고유값, 전달함수의 극점과 영점, 주파수응답 등의 개념은 복소수 형태로 표현되는 경우가 흔하다. 복소수를 자유롭게 다룰 수 있어야, 시스템 해석과 설계 과정을 더욱 직관적이고 체계적으로 진행할 수 있다. 복소수의 기초와 연산법을 충실히 이해하고 있으면, 다양한 고급 계수 연산과 복소행렬 연산 역시 한층 수월하게 접근할 수 있다.
