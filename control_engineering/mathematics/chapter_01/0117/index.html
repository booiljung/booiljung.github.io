양의정부호 행렬은 벡터 미분적분학에서 매우 중요한 개념이다. 예를 들어 다변수 함수를 다룰 때, 특정 점이 극소점인지 판별하려면 해시안 행렬이 양의정부호인지 여부를 살펴봐야 한다. 최적화 문제를 풀 때도 목적함수의 곡률을 결정하는 지표로 활용된다. 공분산 행렬처럼 분포의 안정성을 검토하는 작업에서도 양의정부호 성질이 필수적이다. 구체적으로는 2차 형식(quadratic form)을 통해 정의되는데, 어떤 행렬이 양의정부호인지를 이해하고 이를 판별하는 여러 가지 방법은 벡터 미분적분학에서 핵심이 된다.

양의정부호 행렬의 정의¶

실수 대칭 행렬을 \(\mathbf{A}\)라 하고, \(\mathbf{x}\)를 영벡터가 아닌 임의의 벡터라고 하자. 행렬 \(\mathbf{A}\)가 양의정부호라는 것은 다음 조건으로 정의된다.

\[ \begin{align} \mathbf{x}^\top \mathbf{A}\,\mathbf{x} > 0 \quad\text{for all}\;\mathbf{x} \neq \mathbf{0}. \end{align} \]

위 식에서 \(\mathbf{x}^\top \mathbf{A} \mathbf{x}\)는 2차 형식(quadratic form)이라 부른다. 양의정부호 행렬에서는 어떤 방향의 벡터를 취해도 이 2차 형식의 값이 음수가 되지 않을 뿐 아니라 0이 되지 않는다. 만약 \(\mathbf{x}^\top \mathbf{A} \mathbf{x} = 0\)가 오직 \(\mathbf{x} = \mathbf{0}\)에서만 성립한다면, 그 행렬을 양의정부호라고 부른다.

양의정부호 행렬은 반드시 실수 대칭 행렬이어야 하며, 복소수에 대해서는 에르미트(Hermitian) 행렬을 통해 유사한 개념을 정의한다. 여기서는 실수 범위를 전제로 다룬다.

고유값을 통한 해석¶

양의정부호는 행렬의 고유값이 모두 양수임을 의미하기도 한다. 실수 대칭 행렬 \(\mathbf{A}\)는 직교행렬 \(\mathbf{Q}\)와 대각행렬 \(\mathbf{D}\)를 통해 다음과 같이 대각화할 수 있다.

\[ \begin{align} \mathbf{A} = \mathbf{Q} \,\mathbf{D}\,\mathbf{Q}^\top. \end{align} \]

대각행렬 \(\mathbf{D}\)에는 \(\mathbf{A}\)의 고유값들이 대각 원소로 나열된다. 모든 고유값이 양수이면, 아래와 같은 2차 형식 \(\mathbf{x}^\top \mathbf{A} \mathbf{x}\)가 항상 양수를 갖는다.

\[ \begin{align} \mathbf{x}^\top \mathbf{A} \mathbf{x} &= \mathbf{x}^\top \mathbf{Q} \,\mathbf{D}\,\mathbf{Q}^\top \mathbf{x}\\ &= \mathbf{y}^\top \mathbf{D} \mathbf{y}, \end{align} \]

여기서 \(\mathbf{y} = \mathbf{Q}^\top \mathbf{x}\)라고 놓았다. \(\mathbf{D}\)를 \(\mathrm{diag}(\lambda_1,\lambda_2,\dots,\lambda_n)\)이라 하면

\[ \begin{align} \mathbf{y}^\top \mathbf{D} \mathbf{y} = \sum_{i=1}^n \lambda_i\,y_i^2. \end{align} \]

고유값 \(\lambda_i\)가 모두 양수이므로 벡터 \(\mathbf{y}\)가 영벡터가 아닌 이상, 그 합도 양수가 된다. 따라서 \(\mathbf{A}\)가 양의정부호라면 모든 고유값이 양의 실수이며, 반대로 모든 고유값이 양의 실수이면 \(\mathbf{A}\)는 양의정부호가 된다.

판별법과 성질¶

양의정부호 행렬은 판별하는 방법이 다양하지만, 대표적으로 다음 두 가지가 자주 사용된다.

  • 하나는 행렬의 고유값을 직접 구해 모두 양수인지 확인하는 것이고,
  • 다른 하나는 Sylvester의 판정법(Sylvester's criterion)을 이용하는 것이다. Sylvester의 판정법은 모든 주각소행렬(principal minor)의 행렬식이 양수인지 확인하는 방식으로, 고유값 계산을 대신하여 더 효율적일 수 있다.

양의정부호 행렬은 정칙 행렬이기도 하다. 모든 고유값이 양수이므로 행렬식 역시 0이 아니다. 최적화 문제에서 이차 형식이 나타나는 경우, 헤시안 행렬이 양의정부호라는 사실은 해당 점이 지역적 볼록성을 보인다는 것을 의미한다.

양의정부호가 아닌 경우,

  • 행렬의 고유값에 따라 음수가 포함되면 음의정부호,
  • 0이 포함되면 반정부호(semidefinite) 등으로 분류할 수 있다.

이들은 모두 2차 형식에서 제시되는 곡률 또는 차원축 방향으로의 함숫값 변화를 나타낸다.

다변수 미적분학에서의 응용¶

벡터 미분적분학에서 양의정부호 행렬은 주로 Hessian 행렬의 성질을 설명할 때 등장한다. 스칼라 함수 \(f(\mathbf{x})\)의 Hessian \(\mathbf{H}\)는 각 변수에 대한 2차 편도함수로 구성되며, 특정 점 \(\mathbf{x}_0\)에서 \(\mathbf{H}\)가 양의정부호라면 그 점에서 \(f\)는 지역적 최소값을 갖는다. 이는 1변수 함수에서 2차 미분이 양수이면 해당 점이 극소점인 것과 정확히 일치하는 고차원 버전이다.

확률론 관점에서도 공분산 행렬이 양의정부호라는 점은 매우 중요하다. 다변량 정규분포는 공분산 행렬이 양의정부호여야 제대로 정의될 수 있다. 음의 고유값이 존재한다면 분산이 음수가 되는 기괴한 상황이 생겨서, 실제 분포로서의 의미를 상실한다.

예시 코드 (Python)¶

다음은 Python에서 행렬이 양의정부호인지 판별하는 간단한 예시 코드다. 고유값을 구해 모두 양수인지 확인하는 방식이다.

import numpy as np

def is_positive_definite(A):
    # A는 실수 대칭 행렬이라고 가정
    eigenvalues = np.linalg.eigvals(A)
    return np.all(eigenvalues > 0)

# 예시 행렬
A = np.array([
    [2.0, 1.0],
    [1.0, 2.0]
])

print(is_positive_definite(A))  # 모든 고유값이 양수이면 True

수치 해석 상황에서는 매우 작은 양의 고유값이 부동소수점 연산의 불안정성 때문에 음수로 인식될 수도 있으므로, 적절한 임계값을 두어 양의정부호를 판별하기도 한다. Cholesky 분해를 시도해보는 방법도 효율적이다. Cholesky 분해가 정상적으로 이루어지면 해당 행렬은 양의정부호라는 사실을 빠르게 알 수 있다.

양의정부호 행렬의 이러한 성질과 판별법을 이해하면, 다변수 함수의 해석뿐 아니라 확률 모델이나 머신 러닝에서 공분산 구조를 설계할 때도 유용하다. 이는 단순히 수학적 흥미에 그치지 않고, 실제 알고리즘 최적화와 안정성 분석 전반에서 핵심적인 역할을 수행한다.
