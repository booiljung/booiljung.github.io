사영의 개념¶

사영은 어떤 벡터 공간에서 특정 부분공간 위로 벡터를 투영하는 연산이다. 일반적으로 \(\mathbb R^n\) 내에서 어떤 부분공간 \(\mathbf V\)가 주어졌을 때, \(\mathbf x\)라는 임의의 벡터를 \(\mathbf V\) 위로 사영(projection)하는 것을 생각할 수 있다. 사영 연산을 통해, 원래 벡터 \(\mathbf x\)가 아닌 부분공간 \(\mathbf V\)에 존재하는 어떤 벡터 \(\mathbf x_p\)를 찾게 되는데, 이 벡터는 \(\mathbf V\)에서 \(\mathbf x\)와 가장 가까운 벡터라고 해석된다. 사영은 선형대수학과 제어공학에서 매우 중요하게 활용되는데, 예를 들어 시스템 파라미터 추정이나 오차 최소화 문제에서 중요한 역할을 한다.

사영의 일반적 정의는 다음과 같은 선형 변환 \(\mathbf P\)에 의해 주어진다. 벡터 \(\mathbf x\)에 대해 \(\mathbf V\)로 사영한 결과를 \(\mathbf P(\mathbf x)\)라고 하면 \(\mathbf V\)에 속하며, 원래 \(\mathbf x-\mathbf P(\mathbf x)\)는 \(\mathbf V\)에 대해 직교한다는 특성이 있다. 즉 \(\mathbf P\)는 멱영산(idempotent) 성질을 지니며, 이는 \(\mathbf P(\mathbf P(\mathbf x)) = \mathbf P(\mathbf x)\)로 나타난다.

정사영¶

가장 흔히 다루는 사영 연산은 직교사영(orthogonal projection)이다. 어떤 부분공간 \(\mathbf V\)가 주어졌다고 할 때, \(\mathbf x\)를 \(\mathbf V\)로 투영한 벡터를 \(\mathbf x_p\)라고 하면 다음 조건을 만족해야 한다.

모든 \(\mathbf v \in \mathbf V\)에 대해 \(\mathbf x - \mathbf x_p\)가 \(\mathbf V\)에 직교한다. 즉 \(\big(\mathbf x - \mathbf x_p\big)^\top \mathbf v = 0\)가 성립한다.

만약 \(\mathbf V\)가 \(\mathbf A\)의 열공간(column space)이라고 가정해 보자. 열공간을 스팬하는 열 벡터들이 선형 독립이라면, \(\mathbf A^\top \mathbf A\)는 역행렬을 갖게 된다. 이때 직교사영 행렬 \(\mathbf P\)는

\[ \begin{align} \mathbf P &= \mathbf A(\mathbf A^\top \mathbf A)^{-1}\mathbf A^\top \ \end{align} \]

로 주어진다. 이 행렬은 임의의 벡터 \(\mathbf y\)에 대해 \(\mathbf P \mathbf y\)를 \(\mathbf A\)의 열공간으로 사영한 결과를 제공한다.

기하학적 해석¶

직교사영을 기하학적으로 해석하면, 원점에서 벡터 \(\mathbf x\)가 끝나는 점과 부분공간 \(\mathbf V\)가 이루는 최소 거리를 갖는 점을 찾는 과정이라고 볼 수 있다. 벡터 \(\mathbf x\)를 해당 부분공간 위로 직교하게 끌어내린 점이 바로 사영 결과 \(\mathbf x_p\)가 된다. 이때 \(\mathbf x-\mathbf x_p\)는 부분공간 \(\mathbf V\)의 모든 벡터에 대해 수직이 된다. 이 사실은 오차 \(\mathbf x-\mathbf x_p\)가 부분공간에 존재하는 어떤 벡터와도 내적이 0이 된다는 점으로부터 얻어진다.

선형대수학적인 측면에서 사영 행렬 \(\mathbf P\)는 대칭(symmetric)이고 멱영산(idempotent)인 행렬이다. 즉

\[ \begin{align} \mathbf P^\top = \mathbf P \\ \mathbf P^2 = \mathbf P \ \end{align} \]

이 두 성질은 사영을 특징짓는 핵심적인 성질이다.

최소자승법(LS)¶

최소자승법(Least Squares Method)은 관측된 데이터나 측정된 값을 통해, 실제로 측정 불가능하거나 알 수 없는 어떤 파라미터를 추정하는 대표적인 기법이다. 제어공학에서는 시스템 식별, 추정, 필터링 등 다양한 분야에서 최소자승법이 활발히 쓰인다. 예를 들어 선형회귀(linear regression) 문제에서, 주어진 입력과 출력 사이를 선형 모델로 근사할 때 계수를 구하는 과정에 바로 최소자승법이 적용된다.

최소자승법의 기본 아이디어는 우리가 측정한 데이터의 잔차(residual) 제곱합을 최소화하는 방향으로 파라미터를 추정하는 것이다. 예를 들어 모델이

\[ \begin{align} \mathbf y = \mathbf A \mathbf x + \mathbf \epsilon \ \end{align} \]

와 같은 선형 모델로 표현될 때, \(\mathbf A\)는 설계행렬(design matrix), \(\mathbf x\)는 찾고자 하는 미지의 파라미터 벡터, \(\mathbf y\)는 측정된 출력 벡터, \(\mathbf \epsilon\)은 오차항 벡터다. 이때 최소화하고자 하는 목적함수(오차 제곱합)는

\[ \begin{align} J(\mathbf x) = |\mathbf y - \mathbf A \mathbf x |^2 \ \end{align} \]

와 같이 정의된다.

최소자승해의 유도¶

목적함수 \(J(\mathbf x) = (\mathbf y - \mathbf A \mathbf x)^\top (\mathbf y - \mathbf A \mathbf x)\)가 최소가 되는 \(\mathbf x\)를 찾기 위해서는 보통 미분을 통해 조건을 구한다. \(\mathbf x\)에 대해 편미분한 뒤 0으로 놓으면 다음이 성립한다.

\[ \begin{align} -\mathbf A^\top (\mathbf y - \mathbf A \mathbf x) = \mathbf 0 \\ \mathbf A^\top \mathbf y = \mathbf A^\top \mathbf A \mathbf x \ \end{align} \]

이를 보통 정규방정식(Normal Equation)이라고 한다. 만약 \(\mathbf A^\top \mathbf A\)가 역행렬을 갖는다면

\[ \begin{align} \mathbf x = (\mathbf A^\top \mathbf A)^{-1} \mathbf A^\top \mathbf y \ \end{align} \]

이 해가 최소자승해(LS solution)가 된다. 해석적으로 살펴보면, \(\mathbf x\)는 벡터 \(\mathbf y\)를 \(\mathbf A\)의 열공간으로 사영시킨 좌표계수 역할을 한다고 볼 수 있다.

사영과 최소자승법의 연결¶

최소자승법에서 얻어지는 예측값(또는 근사값)은

\[ \begin{align} \mathbf{\hat y} = \mathbf A \mathbf x = \mathbf A(\mathbf A^\top \mathbf A)^{-1}\mathbf A^\top \mathbf y \ \end{align} \]

로 표현된다. 이때의 행렬

\[ \begin{align} \mathbf P = \mathbf A (\mathbf A^\top \mathbf A)^{-1}\mathbf A^\top \ \end{align} \]

는 앞서 언급한 직교사영 행렬과 동일하다. 즉 최소자승법에서의 예측값 \(\mathbf{\hat y}\)는 \(\mathbf y\)를 \(\mathbf A\)의 열공간으로 투영한 결과와 일치한다. 이는 최소자승해가 \(\mathbf y\)와 가장 가까운 열공간의 벡터라는 사실과도 부합한다.

최소자승법의 기하학적 의미¶

최소자승법은 본질적으로 고차원 공간에서 얻은 데이터 점들을 어떤 부분공간(또는 선형 모델) 위로 직교사영시키는 과정과 동일하다. 오차 벡터 \(\mathbf y - \mathbf A \mathbf x\)가 최소가 되도록 하는 \(\mathbf x\)를 찾는 것은, 결국 데이터가 놓여 있는 공간에서 \(\mathbf A\)의 열공간에 대한 사영 거리를 최소화하는 문제와 같다.

데이터가 많거나 오차가 존재할 때, 완벽한 해 \(\mathbf x\)가 존재하지 않아도 사영에 의해 가장 근접한 벡터를 찾을 수 있다. 이것이 바로 최소자승법이 다양한 분야에서 노이즈를 포함한 실제 측정값을 근사하는 강력한 방법임을 보여준다.

예시 코드 (Python)¶

필요한 경우, 실제로 파라미터 벡터를 구하는 과정을 Python을 이용해 간단히 예시로 살펴볼 수 있다. 아래의 코드는 \(\mathbf A\)와 \(\mathbf y\)가 주어졌을 때, 최소자승해를 구하는 간단한 과정이다.

import numpy as np

# 예시 행렬 A와 측정값 벡터 y
A = np.array([[1, 1],
              [1, 2],
              [1, 3],
              [1, 4]], dtype=float)
y = np.array([2, 2.8, 3.6, 4.5], dtype=float)

# A^T A가 역행렬을 가정할 수 있는지 확인 (열 벡터가 선형 독립이어야 함)
# 최소자승해 x = (A^T A)^(-1) A^T y
ATA = A.T @ A
inv_ATA = np.linalg.inv(ATA)
x = inv_ATA @ A.T @ y

# 예측값 hat{y} = A x
y_hat = A @ x

print("최소자승해 x =", x)
print("예측값 y_hat =", y_hat)

행렬 \(\mathbf A\)와 \(\mathbf y\)를 정의하고, 정규방정식을 이용해 \((\mathbf A^\top \mathbf A)^{-1} \mathbf A^\top \mathbf y\)를 계산함으로써 최소자승해를 구한다. 이는 \(\mathbf y\)를 \(\mathbf A\)의 열공간으로 사영하는 벡터의 좌표가 된다. 이후 예측값 \(\mathbf{\hat y}\)를 확인하면, 실제 데이터 \(\mathbf y\)와의 잔차가 최소가 됨을 알 수 있다.

---과 요약

사영(Projection)은 어떤 벡터를 특정 부분공간으로 투영하여 가장 가까운 점을 찾는 과정이며, 직교사영의 경우 사영된 벡터와 원래 벡터 간에 형성되는 오차 벡터가 부분공간에 대해 직교한다. 최소자승법(LS)은 관측된 데이터를 바탕으로 모델의 파라미터를 추정할 때, 오차 제곱합이 최소가 되도록 하는 파라미터를 찾는 기법이다. 이때 얻어지는 해가 데이터 벡터를 설계행렬의 열공간 위로 직교사영한 결과와 동일함을 확인할 수 있다. 이러한 사영과 최소자승법의 밀접한 연결 고리는, 실제 제어공학과 통계학을 포함한 다양한 분야에서 관측값의 잡음이 클 때도 안정적으로 해를 구할 수 있는 강력한 방법론임을 보여준다.
