확률적·스토캐스틱 제어와 머신러닝의 융합¶

확률적·스토캐스틱 제어는 물리적 시스템에 내재된 불확실성과 잡음을 확률 분포로 모델링하고, 이를 토대로 최적 제어 전략을 도출하는 분야다. 전통적 제어에서는 잡음을 억제하거나 단순 모델로 근사하는 방법이 주를 이루었지만, 확률적·스토캐스틱 제어에서는 불확실성을 제어 알고리즘의 일부로 자연스럽게 받아들인다. 이렇게 확률적 성질을 전제로 한 제어 기법은 예측 가능성을 높일 뿐 아니라, 시스템이 의도치 않은 상황에 직면했을 때 유연하게 대응할 수 있도록 해 준다.

머신러닝은 대규모 데이터로부터 규칙을 학습해 내는 기법으로, 전통적 제어 이론만으로는 해석하기 어려운 복잡한 현상을 다룰 수 있다. 최근에는 불확실한 환경에서 동적으로 변화하는 정보를 빠르게 학습하고, 이를 토대로 적절한 제어 입력을 산출하는 알고리즘들이 적극 연구되고 있다. 이는 확률적·스토캐스틱 제어의 문제 구조와 머신러닝, 특히 강화학습(Reinforcement Learning) 기법이 잘 결합될 수 있음을 시사한다.

두 분야를 결합하면 확률적 모델링을 통해 환경의 불확실성을 탐색하고, 머신러닝 기술을 활용해 대규모 상태-행위 공간에서 최적의 정책을 찾아낼 수 있다. 이 결합은 복잡한 로봇 제어, 자율 주행, 에너지 시스템 관리 등 다채로운 응용 분야에서 각광받으며, 향후 더욱 폭넓은 연구가 이루어질 것으로 전망된다.

확률 분포와 확률적 동역학¶

확률적·스토캐스틱 제어의 핵심 아이디어는 시스템의 동역학을 확률 변수로 표현한다는 점에 있다. 예를 들어 시스템의 상태 \(\mathbf{x}_k\)가 시간 \(k\)에서 다음 시간 \(k+1\)로 변화할 때, 잡음이나 불확실성을 고려하면 다음과 같은 형태로 모델링한다. $$ \begin{align} \mathbf{x}_{k+1} &= f\bigl(\mathbf{x}_k, \mathbf{u}_k, \mathbf{w}_k\bigr) \end{align} $$ 여기서 \(\mathbf{x}_k\)는 상태 벡터, \(\mathbf{u}_k\)는 제어 입력, 그리고 \(\mathbf{w}_k\)는 확률적 잡음 항을 나타낸다. \(\mathbf{w}_k\)는 특정 확률 분포(예: 가우시안 분포)로부터 샘플링된다고 가정할 수도 있고, 관측치와 불확실한 파라미터를 종합하여 추정된 분포를 적용할 수도 있다.

이때 원하는 것은 어떤 성능 지표를 최소화(또는 최대화)하는 제어 입력 \(\mathbf{u}_k\)를 구하는 것이다. 일반적으로 무한 시간 구간을 고려하는 경우 다음과 같은 기대 비용 함수를 생각한다. $$ \begin{align} J(\mathbf{x}0) &= \mathbb{E}\Bigl[ \sum{k=0}^{\infty} \gamma^k \, c(\mathbf{x}_k, \mathbf{u}_k) \Bigr] \end{align} $$ 이 식에서 \(c(\mathbf{x}_k, \mathbf{u}_k)\)는 단위 시간에서의 비용 함수를 의미하고, \(0 < \gamma < 1\)는 할인 계수다. \(\mathbb{E}[\cdot]\)는 확률적 잡음 항 \(\mathbf{w}_k\)에 대한 기대값을 의미한다. 목표는 이 기대 비용을 최소화하는 제어 입력(혹은 정책)을 찾아내는 것이다.

확률적·스토캐스틱 제어 문제는 잡음 분포에 대한 사전 정보가 충분하지 않을 수도 있어, 실제 상황에서는 잡음 분포를 온라인으로 추정하거나, 외부 환경 변화를 적응적으로 반영해야 한다. 이렇게 변화하는 분포를 실시간 또는 준실시간으로 업데이트하고, 적절한 제어 입력을 도출하기 위해서는 대규모 데이터에서 확률적 구조를 빠르게 파악할 수 있는 학습 기법이 필요하다. 머신러닝은 이러한 요구에 부합하는 강력한 도구로서 작용한다.

강화학습과 정책 최적화¶

머신러닝에서 확률적·스토캐스틱 제어 문제에 가장 직접적으로 대응되는 분야가 강화학습이다. 강화학습에서는 에이전트가 환경과 상호 작용하며 누적 보상(또는 비용)을 최대화(또는 최소화)하도록 정책을 개선한다. 확률적·스토캐스틱 제어와 유사하게, 강화학습에서도 환경이 확률적 동역학을 가질 수 있고, 에이전트는 장기적 보상에 대한 기대값을 높이기 위해 정책을 반복적으로 평가하고 개선한다.

전형적인 강화학습은 크게 상태(state), 행동(action), 보상(reward)으로 구성된다. 확률적 시스템에서는 상태가 잡음으로 인해 불확실하게 변화하기 때문에, 에이전트는 이러한 불확실성을 고려한 확률적 정책을 채택할 수 있다. 정책 \(\pi(\mathbf{u}_k \mid \mathbf{x}_k)\)가 확률 분포 형식으로 표현될 수도 있으며, 이는 서로 다른 환경 상황이나 잡음 조건에서 다양한 선택지를 허용하는 유연한 접근이다.

에이전트가 누적 보상을 최적화하려면 Bellman 방정식을 만족하도록 가치 함수나 행동-가치 함수를 개선해 나가야 한다. 예컨대 상태 가치 함수를 \(V^\pi(\mathbf{x}_k)\)라 하면, 이 함수는 아래와 같이 정의할 수 있다. $$ \begin{align} V^\pi(\mathbf{x}k) &= \mathbb{E}\Bigl[\,c(\mathbf{x}_k, \mathbf{u}_k) + \gamma \, V^\pi(\mathbf{x}{k+1}) \,\Bigr] \end{align} $$ 여기서 기대값 기호 \(\mathbb{E}[\cdot]\)는 정책 \(\pi\)에 따른 확률적 행동 선택과 시스템 잡음에 대한 평균을 의미한다. 정책이 최적 정책 \(\pi^*\)가 되면, 다음을 만족한다. $$ \begin{align} V^(\mathbf{x}k) &= \min{\mathbf{u}_k} \, \mathbb{E}\Bigl[\,c(\mathbf{x}_k, \mathbf{u}_k) + \gamma \, V^(\mathbf{x}_{k+1}) \,\Bigr] \end{align} $$ 확률적·스토캐스틱 제어의 맥락에서 이러한 최적화 과정을 수행하기 위해, 보강된 방정식을 해결하거나, 경험에 기반한 근사 해법을 도입하는 등이 가능하다. 여기서 머신러닝은 대규모 상태 공간에서 근사 함수(예: 신경망)를 사용하여 정책이나 가치 함수를 표현하고, 무작위 탐색 과정에서 얻은 데이터를 이용해 파라미터를 갱신하는 방식으로 효율적인 학습을 구현한다.

실제로 정책을 학습하는 과정을 간단히 살펴보기 위해, Python으로 매우 기본적인 Q러닝 형태를 예시로 들면 다음과 같은 구조를 가질 수 있다.

import numpy as np

alpha = 0.1  
gamma = 0.99  

# Q 테이블 초기화
Q = np.zeros((num_states, num_actions))

def choose_action(state, epsilon):
    if np.random.rand() < epsilon:
        return np.random.randint(num_actions)
    else:
        return np.argmax(Q[state, :])

for episode in range(max_episodes):
    state = reset_environment()
    done = False
    while not done:
        action = choose_action(state, epsilon)
        next_state, reward, done = step_environment(state, action)

        # 확률적·스토캐스틱 환경에서는 next_state가 잡음에 의해 불확실하게 변화
        # 하지만 Q 학습에서는 경험으로부터 얻은 실제 상태 전이를 그대로 이용
        Q[state, action] += alpha * (reward + gamma * np.max(Q[next_state, :]) - Q[state, action])

        state = next_state

이 코드는 강화학습의 가장 기본적인 형태를 보인 것으로, 실제 확률적·스토캐스틱 환경에서는 상태 전이 과정에 잡음이 섞여 있다. 그러나 Q 테이블 기반 접근법은 고차원 상태 공간이나 복잡한 연속 공간 문제에 직접 적용하기 어렵다는 단점이 있어, 신경망 등 머신러닝 기반 근사 함수를 사용하는 딥 강화학습으로 확장된다.

이와 같은 순환 구조에서 에이전트는 확률적으로 변화하는 환경을 경험하며 점차 최적 정책으로 수렴해 간다. 확률 분포에 대한 사전 지식이 없어도 데이터를 통해 기댓값을 추정하면서 정책을 개선할 수 있다는 점이 스토캐스틱 제어와 맞물려, 다양한 복잡계 문제에 대한 해법을 제시하게 된다.

베이지안 관점과 불확실성 정량화¶

확률적·스토캐스틱 제어에서 잡음이나 모델 파라미터에 대한 불확실성을 더 깊이 다루려면 베이지안 접근법이 유용하다. 베이지안 관점에서는 모델 파라미터를 확률 변수로 바라보며, 주어진 데이터(혹은 관측치)에 따라 사후 확률 분포를 갱신해 나간다.

베이지안 업데이트는 다음과 같은 일반 형태를 따른다. $$ \begin{align} p(\boldsymbol{\theta} \mid D) &= \frac{p(D \mid \boldsymbol{\theta}) \, p(\boldsymbol{\theta})} {p(D)} \end{align} $$ 여기서 \(\boldsymbol{\theta}\)는 모델의 파라미터, \(D\)는 관측 데이터, \(p(\boldsymbol{\theta})\)는 사전 확률(prior), \(p(D \mid \boldsymbol{\theta})\)는 데이터 우도(likelihood)를 의미한다. \(p(D)\)는 정규화 상수로, \(\boldsymbol{\theta}\)에 대해 적분하여 1이 되도록 만든다. 이를 스토캐스틱 제어 문제에 연결하면, 시스템이 어떤 환경 파라미터를 가진다고 가정했을 때, 실제 환경 데이터가 들어올수록 해당 파라미터의 사후 확률을 갱신함으로써 더 정확한 모델을 확보하게 된다.

불확실성을 정량화할 수 있다면, 정책 결정 시에도 이러한 정보를 활용하여 더욱 보수적이거나, 혹은 탐색적인 결정을 내릴 수 있다. 예를 들어 베이지안 강인 제어(Bayesian robust control) 기법은 파라미터에 대한 불확실성 범위를 고정하지 않고, 사후 분포 추정을 통해 동적으로 범위를 조절한다. 이는 실제 시스템 운용 과정에서 변화가 일어날 때, 단순한 고정 모수 모델보다 훨씬 유연하게 대처할 수 있게 해 준다.

머신러닝에서도 베이지안 학습 기법을 적용하면 딥 뉴럴 네트워크의 파라미터 분포를 추정함으로써, 예측의 불확실성을 정량화할 수 있다. 이를 통해 제어 입력을 결정할 때 모델이 얼마나 신뢰할 만한 상태인지, 혹은 추가적인 탐색이나 안전 장치가 필요한지를 평가할 수 있다. 결국 베이지안 접근법은 확률 분포 자체를 동적으로 학습하고, 이 분포를 활용하여 최적 정책을 찾는 효과적인 프레임워크를 제공한다.

미래 연구 방향 및 결론¶

확률적·스토캐스틱 제어와 머신러닝의 융합은 복잡하고 불확실한 환경에서의 의사결정 문제를 해결하는 데 매우 유망하다. 강화학습을 비롯한 다양한 머신러닝 기법은 정확한 수학적 모델을 구하기 어려운 경우에도 경험적으로 시스템 특성을 학습하고, 확률적 잡음이 결합된 상황에서 적절한 정책을 찾게 해 준다. 특히 대규모 상태-행위 공간을 다룰 때는 신경망 등을 활용한 함수 근사로 높은 차원의 문제를 효율적으로 풀 수 있다.

다만 이러한 융합 기법에도 여러 과제가 남아 있다. 매우 높은 차원의 상태 공간에서 표본 효율(sample efficiency)을 높이는 문제, 안정성과 안전성 보장을 위한 수학적 이론의 정교화 문제, 그리고 실시간 학습 과정에서 연산 비용이나 탐색 비용을 줄이는 문제 등이 대표적이다. 또한 베이지안 접근법이나 분포적 강화학습(distributional reinforcement learning)과 같은 방법론은 불확실성을 정량화하고 제어 성능을 보장하는 데 큰 잠재력을 지닌다.

앞으로는 확률 분포 추정 능력을 고도화하고, 이를 토대로 동적 환경에서 최적 정책을 스스로 학습하도록 만드는 이론적·실용적 연구가 활발해질 것으로 예상된다. 확률적·스토캐스틱 제어의 정교한 이론과 머신러닝의 강력한 함수 근사 및 데이터 학습 기법이 접목되면서, 기존 제어 이론이 해결하기 어려웠던 복잡한 문제들을 새롭게 돌파하는 길이 열리고 있다. 이는 자율주행, 로보틱스, 에너지 관리, 금융, 의료 등 다양한 산업 분야에서 더욱 안전하고 효율적인 시스템을 구현하는 기반이 될 것이다.
