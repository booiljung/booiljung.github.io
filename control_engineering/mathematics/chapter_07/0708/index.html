확률변수의 분포를 나타내는 대표적인 방법에는 확률질량함수(PMF)와 확률밀도함수(PDF)가 있다. 확률질량함수는 이산(discrete) 확률변수에, 확률밀도함수는 연속(continuous) 확률변수에 적용된다. 두 함수는 각각 확률변수의 값을 어떤 방식으로 표현하고, 이를 통해 분포가 어떻게 구성되는지 알려 준다.

확률변수가 어떤 실험이나 현상을 수학적으로 표현하는 도구라는 점을 떠올리면, PMF와 PDF는 그 확률변수가 구체적으로 어떤 값들을 얼마나 자주 혹은 어느 정도의 '가능성'으로 취하는지를 나타낸다고 볼 수 있다.

확률질량함수는 확률변수가 특정 값을 갖는 사건의 확률을 직접 표로 나열하거나, 함수 형태로 명시적으로 정의한다. 예를 들어, 동전을 여러 번 던져 나오는 앞면의 개수를 이산 확률변수라고 할 때, 이산적인 값들에 대응하는 확률을 각각 부여함으로써 확률질량함수를 정의할 수 있다.

확률밀도함수는 확률변수가 가질 수 있는 값이 연속적으로 변화할 때 사용한다. 이 경우 특정 한 점에서의 확률은 0이 되고, 어떤 구간에 걸친 확률의 총합을 적분을 통해 구한다. 예를 들어, 측정 장비의 오차가 연속적인 분포를 갖는 상황에서, 오차가 x 근방에서 얼마나 자주 발생하는지를 함수 형태로 나타내는 것이 확률밀도함수다.

확률질량함수(PMF)¶

이산 확률변수 \(X\)가 가질 수 있는 서로 다른 값들을 \(x \in {x_1, x_2, \dots}\)라고 할 때, 확률질량함수는 \(p_X(x)\)로 표현한다. 이때 \(p_X(x)\)는 \(X\)가 정확히 \(x\) 값을 가질 확률이다. 수학적으로는 \(p_X(x) = P(X = x)\)로 정의한다.

확률질량함수는 다음과 같은 성질을 만족한다. 먼저 모든 \(x\)에 대해 \(p_X(x) \ge 0\)이어야 한다. 또한 \(X\)가 가질 수 있는 모든 값을 합했을 때 총합이 1이 되어야 한다. 즉,

\[ \begin{align} \sum_{x} p_X(x) = 1 \end{align} \]

이산 확률변수에서 확률이 정의되는 방식은 값을 하나하나 셀 수 있다는 점에 기반한다. 예를 들어, \(X\)가 주사위를 한 번 던졌을 때 나오는 눈금이라고 하면, \(X\)의 가능한 값은 1, 2, 3, 4, 5, 6이다. 각 눈금이 나올 확률이 모두 같으면

\[ \begin{align} p_X(1) = p_X(2) = \dots = p_X(6) = \dfrac{1}{6} \end{align} \]

과 같이 동일한 확률질량함수를 갖는다.

보다 복잡한 예로 이항분포(Binomial Distribution)가 있다. 이는 \(n\)번의 독립 시행에서 특정 사건이 발생하는 횟수를 나타내는 확률변수에 적용한다. 성공 확률을 \(p\), 실패 확률을 \(1-p\)라 하고, 성공 횟수를 \(k\)라 할 때, 이 확률질량함수는

\[ \begin{align} p_X(k) &= P(X = k) \\ &= \dfrac{n!}{k!(n-k)!} \, p^k \, (1-p)^{n-k} \end{align} \]

와 같이 정의된다. 위 식에서 \(n!\)는 \(n\)의 팩토리얼을, \(k!(n-k)!\)는 \(k\)와 \(n-k\)의 팩토리얼을 의미하며, 이는 조합 \({n \choose k}\)와 동등하게 표현된다. 이렇게 구한 \(p_X(k)\) 값들을 \(k=0\)부터 \(k=n\)까지 모두 합하면 1이 된다.

확률질량함수를 실제 계산 예제로 살펴보면, 다음과 같은 Python 코드를 간단히 작성해볼 수 있다. 여기서는 이항분포의 PMF를 계산하고 시각화한다.

import numpy as np
import matplotlib.pyplot as plt
from math import comb

n = 10
p = 0.3
k_values = np.arange(0, n+1)
pmf = [comb(n, k) * (p**k) * ((1-p)**(n-k)) for k in k_values]

plt.stem(k_values, pmf, basefmt=" ")
plt.xlabel("k")
plt.ylabel("PMF")
plt.title(f"Binomial Distribution (n={n}, p={p})")
plt.show()

이 코드는 Python에서 이항분포를 정의하는 간단한 예이며, 실제로 \(k=0,1,\dots,n\)에 대응하는 확률질량함수를 막대 형태로 시각화한다. 그래프를 통해 어떤 \(k\) 값 근처에서 확률이 크게 나타나는지 직관적으로 파악할 수 있다.

확률밀도함수(PDF)¶

연속 확률변수 \(X\)에 대해서는, \(X\)가 특정 점 \(x\)에 대해 정확히 그 값을 가질 확률이 0이므로, 더 이상 "질량"이라는 개념으로 확률을 정의하기 어렵다. 대신 구간을 활용한다. 예를 들어 \(X\)가 구간 \([a, b]\) 안에 속할 확률을 \(P(a \le X \le b)\)라고 할 때, 이를 적분 형태로 표현한다.

확률밀도함수는 \(f_X(x)\)로 표기하며, \(X\)가 구간 \([x, x + dx]\) 안에서 관측될 '가능성의 정도'를 나타낸다. 구체적으로

\[ \begin{align} P(a \le X \le b) = \int_a^b f_X(x) \, dx \end{align} \]

와 같이 표현되고, \(f_X(x)\)는 다음과 같은 성질을 만족한다. 모든 \(x\)에 대해 \(f_X(x) \ge 0\)이며, 전체 실수 구간에서의 적분값이 1이 되어야 한다.

\[ \begin{align} \int_{-\infty}^{\infty} f_X(x) \, dx = 1 \end{align} \]

예를 들어, 정규분포(Normal Distribution)는 매우 자주 등장하는 연속 확률변수의 대표적인 사례다. 평균 \(\mu\)와 분산 \(\sigma^2\)인 정규분포의 확률밀도함수는

\[ \begin{align} f_X(x) &= \dfrac{1}{\sqrt{2\pi}\,\sigma} \exp\!\Bigl(-\dfrac{(x-\mu)^2}{2\,\sigma^2}\Bigr) \end{align} \]

로 정의된다. 중간 과정을 간단히 살펴보면, 임의의 실수 \(x\) 근방에서의 확률은 \(f_X(x),dx\)로 근사된다. 이를 전체 구간에 대해 적분하면 1이 되어야 하므로, 정규분포에서는 위 식의 형태로 밀도함수가 결정된다. 중요한 점은 \(X\)가 특정 값 \(x_0\)를 가질 확률 \(P(X = x_0)\)는 엄밀히 0이지만, 구간 \([x_0, x_0 + dx]\)에서 \(dx\)가 충분히 작은 값을 가질 때, 그 근방에서의 가능성을 \(f_X(x_0),dx\)라고 볼 수 있다는 점이다.

확률밀도함수는 누적분포함수(CDF)의 미분으로도 정의할 수 있다. 누적분포함수를 \(F_X(x) = P(X \le x)\)라 할 때, 만약 \(F_X(x)\)가 미분 가능하다면

\[ \begin{align} f_X(x) = \dfrac{d}{dx} F_X(x) \end{align} \]

로 정의된다. 이러한 방식으로 누적분포함수에서 기인한 확률밀도함수를 구하면, 연속 확률변수의 분포를 적분과 미분의 관점으로도 쉽게 이해할 수 있다.

연속 확률변수의 예를 들어, 측정된 온도나 거리, 또는 무작위 신호에 잡음이 포함되었을 때, 그 잡음의 분포 등이 있다. 이들은 모두 연속적으로 변하는 값이므로 확률밀도함수를 통해 적분으로 확률을 계산한다.

이처럼 확률질량함수는 이산 확률변수를 다루고, 확률밀도함수는 연속 확률변수를 다룬다. 두 함수 모두 전체 분포의 '모양'을 나타내며, 분포를 이해하고 해석하는 핵심적인 역할을 담당한다. 제어공학의 여러 문제에서 측정 오차나 잡음은 보통 연속 분포로 모델링되고, 시스템의 상태가 특정 값(예: 이벤트 발생 횟수)으로 세어지는 상황은 이산 확률분포로 모델링된다. 이러한 맥락에서 PMF와 PDF는 제어 알고리즘을 설계하고 분석할 때 매우 중요한 기초 도구가 된다.
