확률적 현상을 여러 번 반복했을 때, 그 결과들의 평균값이 일정한 값에 가까워진다는 사실을 다루는 것이 대수의 법칙이다. 일반적으로 확률 변수가 가질 수 있는 불규칙성에도 불구하고, 충분히 많은 시행이 반복되면 평균값이 하나의 특정 값(기댓값)에 수렴한다는 내용을 담고 있다. 이는 통계학, 신호처리, 제어공학 등 다양한 영역에서 측정치의 평균을 통해 실제 시스템의 상태나 모수를 추정할 때 핵심적인 이론적 근거가 된다.

무작위 변수를 다룰 때 자주 \(X_1, X_2, \dots, X_n\)으로 표기한다. 이들을 서로 독립이고 동일한 분포를 갖는 확률 변수라고 하자. 각 \(X_i\)의 기댓값을 \(E[X_i]\)라 하고, 동일 분포를 따른다면 모든 \(X_i\)에 대해 동일한 기댓값이 존재한다고 할 수 있다. 그 값을 \(\mu\)라 하면 \(E[X_i] = \mu\)이다. 확률 변수가 복소수 형태(\(a + b j\) 꼴)여도, 실수부와 허수부 각각의 기댓값이 유한하다면 유사한 논의를 전개할 수 있다.

표본 평균을 다음과 같이 정의한다.

\[ \begin{align} \overline{X}_n = \dfrac{1}{n}\sum_{i=1}^n X_i \end{align} \]

대수의 법칙은 \(n\)이 매우 커질 때 이 표본 평균이 이론적 기댓값 \(\mu\)에 수렴한다는 사실을 말한다. 수렴의 방식에 따라 약한 대수의 법칙과 강한 대수의 법칙으로 구분한다.

표본 평균이 큰 \(n\)에서 기댓값과 가까워진다는 사실은 직관적으로도 이해하기 쉽다. 예를 들어 동전 던지기를 반복하면, 전체 던진 횟수가 증가함에 따라 앞면이 나오는 비율은 점차 동전의 앞면이 나올 확률에 가까워진다. 마찬가지로 어떤 센서에서 잡음이 포함된 신호를 측정할 때, 여러 번 측정해서 평균을 내면 잡음이 점차 상쇄되어 센서 신호의 진정한 평균값에 가까워진다.

약한 대수의 법칙¶

약한 대수의 법칙(Weak Law of Large Numbers)은 확률론적 수렴(또는 수렴 확률 convergence in probability)을 다룬다. 이는 표본 평균 \(\overline{X}_n\)이 확률적으로 기댓값 \(\mu\)에 근접한다는 의미다. 보다 정확하게 말하면 다음과 같은 형태로 표현된다.

\[ \begin{align} \overline{X}_n \xrightarrow{\;\;p\;\;} \mu \end{align} \]

수렴 기호 \(\xrightarrow{p}\)는 확률 수렴을 의미한다. 즉, 임의의 \(\epsilon > 0\)에 대해 \(P\big(|\overline{X}_n - \mu| \geq \epsilon\big)\)이 \(n\)이 커짐에 따라 0에 가까워진다.

약한 대수의 법칙은 주로 체비쇼프 부등식(Chebyshev’s inequality)을 통해 간단히 증명할 수 있다. 예를 들어 확률 변수 \(X_i\)들이 분산 \(\sigma^2\)를 가지고 있다고 하자. 그러면 \(X_i\)는 서로 독립이므로 분산의 합은 더해진다. 즉,

\[ \begin{align} Var\Bigl(\sum_{i=1}^n X_i\Bigr) = \sum_{i=1}^n Var(X_i) = n\sigma^2 \end{align} \]

표본 평균의 분산은

\[ \begin{align} Var(\overline{X}_n) = Var\Bigl(\dfrac{1}{n}\sum_{i=1}^n X_i\Bigr) = \dfrac{1}{n^2} \cdot n\sigma^2 = \dfrac{\sigma^2}{n} \end{align} \]

체비쇼프 부등식에 따르면 확률 변수 \(Y\)에 대해 \(P(|Y - E[Y]| \geq \epsilon) \leq \dfrac{Var(Y)}{\epsilon^2}\)가 성립한다. 따라서 \(\overline{X}_n\)에 대해 적용하면,

\[ \begin{align} P\bigl(|\overline{X}_n - \mu| \geq \epsilon\bigr) \leq \dfrac{Var(\overline{X}_n)}{\epsilon^2} = \dfrac{\sigma^2/n}{\epsilon^2} = \dfrac{\sigma^2}{n \,\epsilon^2} \end{align} \]

이는 \(n \to \infty\)로 갈 때 0이 된다. 결국 \(\overline{X}_n\)이 \(\mu\)에 확률적으로 수렴한다는 결론이 나온다. 요약하면 \(n\)이 충분히 클 때, 표본 평균이 기댓값과 크게 벗어날 확률이 매우 작아진다.

강한 대수의 법칙¶

강한 대수의 법칙(Strong Law of Large Numbers)은 거의 확실한 수렴(almost sure convergence)을 다룬다. 이는 확률 1로 표본 평균이 기댓값 \(\mu\)에 수렴한다는 의미다. 수렴 표기는 다음과 같다.

\begin{align}
\overline{X}_n 
\xrightarrow{\;\;a.s.\;\;} 
\mu
\end{align}

이 때 \(\xrightarrow{a.s.}\)는 거의 확실한 수렴을 뜻한다. 즉, 표본 공간에서 극히 예외적인 경우(확률이 0인 경우)를 제외하고 모든 경우에 \(\overline{X}_n\)이 \(\mu\)에 수렴한다.

강한 대수의 법칙을 증명하는 방법 중 하나는 콜모고로프(Kolmogorov)가 제시한 증명을 많이 사용한다. 여기서는 확률 공리와 급수 수렴 등을 통해 \(n \to \infty\)로 갈 때 \(\overline{X}_n - \mu\)가 0에 수렴함을 보인다. 약한 대수의 법칙과 증명 방식은 상당히 다르지만, 결과적으로는 “매우 강한 형태의 수렴”을 보장한다는 점에서 중요하다.

실제로 약한 대수의 법칙만으로도 대규모 샘플에서 평균이 기댓값 근처에 있을 것이라는 통계적 신뢰를 제공한다. 그러나 강한 대수의 법칙은 확률 공간 전체에서 거의 모든 표본 경로(path)에 대해 실제로 그 평균이 기댓값에 수렴한다는 더 강력한 결론을 제시한다.

두 대수의 법칙을 수식으로 요약하면 다음과 같은 결론에 도달한다.

\[ \begin{align} \boxed{ \lim_{n \to \infty} \dfrac{1}{n} \sum_{i=1}^n X_i = \mu } \end{align} \]

이때 약한 대수의 법칙은 확률적 개념, 강한 대수의 법칙은 거의 확실한 개념으로 차이가 있지만, \(n \to \infty\)로 갈 때 표본 평균이 기댓값에 접근한다는 점은 동일하다.

예시와 시뮬레이션¶

대수의 법칙을 직접 확인해 보는 방법 중 하나는 컴퓨터 시뮬레이션이다. 여러 확률 변수를 생성하고 그 평균을 구해보면, 시뮬레이션 횟수가 늘어남에 따라 평균이 기댓값에 가까워진다는 사실을 수치적으로 볼 수 있다. 다음은 Python 예시 코드다. 동전 던지기를 가정하고, 앞면이 나오면 1, 뒷면이 나오면 0으로 둔 뒤 평균을 계산하는 과정을 살펴보자.

import random
import statistics

num_trials = 10_000
results = []
heads_probability = 0.3  # 동전이 편향되어 앞면이 0.3 확률로 나올 때

current_average = 0.0
for n in range(1, num_trials + 1):
    toss = 1 if random.random() < heads_probability else 0
    results.append(toss)
    current_average = statistics.mean(results)
    # n이 증가할수록 current_average가 heads_probability(0.3)에 가까워진다

print(f"시행 횟수: {num_trials}")
print(f"표본 평균: {current_average:.4f}")
print(f"이론적 기댓값(앞면 확률): {heads_probability}")

위 코드를 실행하면 뒤로 갈수록 표본 평균이 heads_probability에 매우 근접하게 수렴한다는 것을 확인할 수 있다. 이 예제는 비교적 단순한 이산 확률변수를 이용했지만, 임의의 연속 확률 변수에서도 원리는 동일하게 적용된다.

대수의 법칙은 이처럼 샘플 평균이 기본 분포의 기댓값을 추정하는 데 왜 유효한지를 수학적으로 보증해 준다. 무작위 잡음이 있는 측정값이나 센서 신호, 또는 통신 채널에서의 에러 비율 등을 추정할 때에도 동일하게 활용할 수 있다. 복소수 신호(예: \(a + b j\) 형태)라면 실수부와 허수부 각각의 평균이 해당 부분의 기댓값으로 수렴하므로, 통신이나 신호처리 분야에서도 대수의 법칙은 중요한 역할을 한다.

결론적으로 대수의 법칙은 실험적 혹은 계산적 접근뿐만 아니라 이론적 접근 면에서도 확률론과 통계학의 근간을 이루며, 제어공학을 포함한 많은 공학 분야에서 활용된다. 시행 횟수가 늘어날수록 평균값이 안정적으로 기댓값을 향해 수렴한다는 원리를 항상 염두에 두면, 불완전한 정보나 노이즈가 많은 상황에서 유의미한 추정이나 결론을 도출할 수 있다.
