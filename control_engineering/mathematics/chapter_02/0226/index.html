그린 정리는 평면 위에서 정의된 벡터장의 선적분과 면적분 사이의 관계를 밝히는 핵심 정리다. 미분적분학에서 출발하여, 평면 상의 닫힌 곡선을 따라 적분을 수행하는 선적분이 영역 내부에서의 이중적분으로 전환될 수 있음을 보여준다. 이는 실제 계산의 간편화를 위해 자주 활용되며, 물리학과 공학에서도 매우 중요한 역할을 한다.

이 정리는 벡터장 \(\mathbf{F}\)의 성분을 적절히 설정하면 다양한 물리량을 해석할 수 있게 해주기 때문에, 유체 흐름이나 전자기학, 열역학 등에서 경계 조건을 이용해 내부 분포를 추론하는 데에도 활용된다. 제어공학의 맥락에서 2차원 영역에 대한 선 적분을 다룰 때, 그린 정리를 통해 적분 계산을 손쉽게 정리할 수 있고, 경계 조건이나 영역 내부의 특성을 파악하는 데 도움을 준다.

여기서는 먼저 평면 상의 선적분 개념과 그린 정리의 기본 형태를 소개하고, 이어서 정리의 증명과 구체적인 예제를 살펴본다. 이후 이 정리가 제어공학적 해석에 어떻게 응용될 수 있는지 짧게 언급한다.

평면 상의 선적분¶

평면 상에서 정의된 벡터장 \(\mathbf{F}\)가 있을 때, 이를 성분으로 나누어 표기하면 다음과 같은 형태로 쓸 수 있다.

\[ \mathbf{F}(x, y) = \begin{pmatrix} L(x, y) \\ M(x, y) \end{pmatrix} \]

벡터장의 선적분은 경로 \(C\)를 따라 적분하는 연산으로 정의된다. 경로 \(C\)가 매개변수 \(t\)에 의해 \(a \le t \le b\) 구간에서 다음과 같이 표현된다고 하자.

\[ \mathbf{r}(t) = \begin{pmatrix} x(t) \\ y(t) \end{pmatrix} ,\quad t \in [a,b] \]

이때 \(\mathbf{F}(\mathbf{r}(t))\)와 경로 미소 \(\mathbf{r}'(t)\)의 내적을 적분하면 선적분을 얻을 수 있다. 즉,

\[ \int_C \mathbf{F}\cdot d\mathbf{r} = \int_a^b \mathbf{F}(\mathbf{r}(t)) \cdot \mathbf{r}'(t)\, dt \]

성분으로 나타내면 다음과 같은 형태가 된다.

\[ \int_C L\,dx + M\,dy = \int_a^b \Bigl( L(x(t), y(t))\,x'(t) + M(x(t), y(t))\,y'(t) \Bigr)\, dt \]

이는 그린 정리를 이해하기 위한 기본적인 선적분의 정의다.

그린 정리의 정식¶

그린 정리는 단순 폐곡선(닫힌 곡선) \(C\)로 둘러싸인 평면 영역 \(D\) 위에서, 벡터장 \(\mathbf{F}(x, y) = (L(x, y), M(x, y))\)에 대해 다음과 같은 관계를 제시한다.

\[ \oint_C L\,dx + M\,dy = \iint_D \Bigl(\frac{\partial M}{\partial x} - \frac{\partial L}{\partial y}\Bigr)\, dA \]

경계 곡선 \(C\)는 일반적으로 양의 방향(반시계 방향)으로 돌아야 하며, 영역 \(D\)는 \(C\)로 둘러싸인 내부 영역이다. 이 식은 평면 벡터장의 선적분이 영역 내부에서의 미분 연산(편도함수의 차)에 의해 결정된다는 것을 의미한다.

그린 정리는 고차원 공간에서의 스토크스 정리(또는 켈빈-스토크스 정리)로 일반화될 수 있으며, 그 자체가 미적분학의 기본 정리 중 하나로 간주된다.

그린 정리의 기하학적 해석¶

벡터장 \((L, M)\)이 주어졌을 때, \(\frac{\partial M}{\partial x} - \frac{\partial L}{\partial y}\)는 영역 \(D\) 내부에서 벡터장의 회전(curl)과 관련된 양을 나타낸다. 이는 3차원으로 확장했을 때의 \(\nabla \times \mathbf{F}\)(회전 연산)의 \(z\)성분에 해당한다.

평면 상의 특정 영역 \(D\)에서 벡터장 \(\mathbf{F}\)가 어떻게 회전하는지, 그리고 그 회전이 경계 \(\mathbf{r}(t)\)를 따라 적분할 때 어떤 기여를 하는지 설명해 주는 것이 바로 그린 정리다. 즉, 경계에서 측정한 양(선적분)은 영역 내부의 누적된 회전의 총합(이중적분)과 같다는 뜻이다.

영역 \(D\)의 내부를 채우는 미소적 요소 각각에서 기여하는 회전값을 적분하면, 경계 \(C\)를 따라 구한 선적분과 동일해진다.

그린 정리의 증명 스케치¶

그린 정리의 증명은 개별적으로 \(D\)를 작은 직사각형 소자로 분할하는 방식 혹은 직접 미분 계산을 통해 접근할 수 있다. 증명 과정을 간략히 요약하면 다음과 같다.

먼저 \(D\)를 충분히 많은 작은 직사각형들로 쪼갠다고 가정한다. 각 작은 직사각형의 경계를 따라 선적분을 수행하면, 내부에 공통으로 위치한 선분들에서 선적분 항이 서로 상쇄된다. 그 결과로 남는 것은 영역 전체의 바깥 경계만을 따라 적분한 값이 된다.

다른 한편, 편미분 \(\frac{\partial M}{\partial x} - \frac{\partial L}{\partial y}\)를 각 작은 직사각형에서 적분하면, 쪼개진 소자들을 모두 합쳤을 때 \(\iint_D \Bigl(\frac{\partial M}{\partial x} - \frac{\partial L}{\partial y}\Bigr)\, dA\)가 얻어진다.

결국 소자들의 경계를 따라 적분한 누적 값이 영역 전체의 경계를 따라 적분한 값과 같아지므로, 그린 정리가 성립함을 알 수 있다.

보다 엄밀한 증명에서는 \(x\) 방향으로의 적분과 \(y\) 방향으로의 적분을 각각 따로 수행한 뒤, 이를 결합하여 최종 식을 얻는다. 하나의 예로 \(M,dy\)에 대해서 적분을 나눈 뒤, \(dx\)에 대한 적분에서의 편미분 항을 대응시킬 수 있다. 다음과 같은 계산 과정을 살펴볼 수 있다.

\[ \oint_C L\,dx + M\,dy \]

경계를 두 부분(수평 방향 경계, 수직 방향 경계)으로 나누어 적분을 수행하면,

\[ \int_C L\,dx + M\,dy = \int_C L\,dx + \int_C M\,dy \]

수평 방향 경계에서 \(dy = 0\)이거나 \(y\)가 일정하므로, \(L,dx\)에 해당하는 적분에 집중할 수 있다. 수직 방향 경계에서 \(dx = 0\)이거나 \(x\)가 일정하므로, \(M,dy\)에 해당하는 적분에 집중할 수 있다. 이들을 모두 합치고 정리하면 편미분 항과의 대응이 드러난다.

결과적으로, 이 모든 과정을 통해 다음을 얻는다.

\[ \oint_C L\,dx + M\,dy = \iint_D \Bigl(\frac{\partial M}{\partial x} - \frac{\partial L}{\partial y}\Bigr)\, dA \]

이는 그린 정리의 핵심 결론이다.

적분 계산 시의 유의 사항¶

그린 정리를 활용할 때는 다음과 같은 점들을 주의해야 한다.

영역 \(D\)가 단순 연결 영역이어야 하며, 경계 \(C\)는 양의 방향(반시계 방향)으로 둘러싸여 있어야 한다.

\((L, M)\)이 \(D\)에서 연속 편미분 가능해야 한다.

만약 영역 내부에 특이점이 있거나 다중 연결 영역일 경우, 내부에 구멍이 뚫린 형태가 되어 경계가 여러 개 있을 수 있다. 이 경우에는 각 경계를 적절한 방향(바깥 경계는 양의 방향, 내부 구멍 경계는 음의 방향)으로 설정하여 정리의 확장된 형태를 적용해야 한다.

예제: 사각형 영역에서의 그린 정리 검증¶

간단한 예제로, 영역 \(D\)를 좌표축에 평행한 사각형으로 두고, 벡터장을 다음과 같이 설정한다.

\[ \mathbf{F}(x, y) = \begin{pmatrix} L(x, y) \\ M(x, y) \end{pmatrix} = \begin{pmatrix} x \\ y \end{pmatrix} \]

경계 \(C\)는 \(0 \le x \le a\), \(0 \le y \le b\)인 사각형의 테두리를 양의 방향(반시계)으로 한 바퀴 도는 곡선이다.

먼저, 선적분으로 계산한다. 사각형의 경계를 네 구간으로 나누어 적분을 수행한다.

  • 첫째 구간: \((x, y) = (0, 0)\)에서 \((a, 0)\)로 진행(이동 방향은 \(+x\), \(y=0\))
  • 둘째 구간: \((x, y) = (a, 0)\)에서 \((a, b)\)로 진행(이동 방향은 \(+y\), \(x=a\))
  • 셋째 구간: \((x, y) = (a, b)\)에서 \((0, b)\)로 진행(이동 방향은 \(-x\), \(y=b\))
  • 넷째 구간: \((x, y) = (0, b)\)에서 \((0, 0)\)로 진행(이동 방향은 \(-y\), \(x=0\))

각 구간에서 \(L,dx + M,dy = x,dx + y,dy\)를 적분하고 합산한다.

다음으로, 그린 정리를 적용하여 바로 이중적분 형태로 계산한다.

\[ \iint_D \Bigl(\frac{\partial M}{\partial x} - \frac{\partial L}{\partial y}\Bigr)\, dA \]

이때 \(M = y\), \(L = x\)이므로,

\[ \frac{\partial M}{\partial x} = 0, \quad \frac{\partial L}{\partial y} = 0 \]

결과적으로 integrand가 0이 되므로,

\[ \iint_D 0 \, dA = 0 \]

실제로 경계를 따라 직접 선적분을 해 보면, 각 변에서의 적분값이 상쇄되어 최종적으로 0임을 확인할 수 있다. 이것은 간단하지만 그린 정리를 정확히 검증하는 예다.

아래에는 Python으로 사각형 경계를 각 구간별로 나누어 수치적 선적분을 수행해 그린 정리가 성립함을 확인하는 예시 코드를 제시한다. 필요 최소한의 예시로 작성했다.

import numpy as np

def L(x, y):
    return x

def M(x, y):
    return y

def line_integral_segment(x_start, y_start, x_end, y_end, steps=1000):
    # 직선 경로를 steps 개 구간으로 나누어 적분
    t_values = np.linspace(0, 1, steps)
    integral_val = 0.0
    for i in range(steps - 1):
        t1 = t_values[i]
        t2 = t_values[i+1]

        # 현재 구간 시작점
        x1 = x_start + (x_end - x_start)*t1
        y1 = y_start + (y_end - y_start)*t1

        # 현재 구간 끝점
        x2 = x_start + (x_end - x_start)*t2
        y2 = y_start + (y_end - y_start)*t2

        # 미분(= 이동 벡터)
        dx = x2 - x1
        dy = y2 - y1

        # 중점 근사를 사용(Trapezoid 등 다른 방법도 가능)
        xm = (x1 + x2)/2.0
        ym = (y1 + y2)/2.0

        # L dx + M dy
        integrand = L(xm, ym)*dx + M(xm, ym)*dy
        integral_val += integrand
    return integral_val

a = 2.0
b = 3.0

# 사각형 경계 구간 네 개
val1 = line_integral_segment(0, 0, a, 0)  # 아래 변
val2 = line_integral_segment(a, 0, a, b)  # 오른쪽 변
val3 = line_integral_segment(a, b, 0, b)  # 위 변
val4 = line_integral_segment(0, b, 0, 0)  # 왼쪽 변

total_line_integral = val1 + val2 + val3 + val4
print("Numerical line integral =", total_line_integral)

# 그린 정리에 따른 면적분 값
# ∂M/∂x - ∂L/∂y = 0
# 따라서 면적분은 0
print("According to Green's Theorem, double integral =", 0.0)

이 코드를 실행해 보면, \(L = x\), \(M = y\)에 대해 사각형 경계를 따라 선적분을 계산한 결과가 매우 작은 수치 오차를 제외하고 0에 가깝게 나타난다. 이는 그린 정리가 제대로 성립함을 확인하는 구체적인 수치 예시다.

그린 정리의 활용과 제어공학적 의의¶

제어공학에서도 2차원 평면에서의 물리적 혹은 추상적 공간에 대한 해석이 필요한 경우가 많다. 시스템이 2차원으로 모델링될 때, 외곽 경계에서의 측정된 정보나 경계 조건만으로 영역 내부 특성을 추론해야 할 때가 있는데, 그린 정리를 활용하면 내부의 분포나 누적 효과를 알 수 있다.

전달함수 해석이나 상태 방정식을 해석할 때, 2차원 공간에 대한 적분 연산이 나타나는 상황에서 그린 정리를 사용하면 복잡한 적분을 단순화할 수 있다. 예를 들어, 연속체 구조 제어나 미세 유체 제어에서도 경계 정보를 이용해 내부 유동 상태를 추정하거나 제어 입력을 계산하는 이론적 배경으로 그린 정리를 응용할 수 있다.

적분 계산 시, 복잡한 경계 형태라면 직접 선적분을 계산하기보다 그린 정리를 통해 영역 내부 적분으로 전환하는 편이 더 간단할 수 있다. 반대로, 어떤 경우에는 영역 적분이 어렵다면 경계를 따라 진행하는 선적분으로 바꾸는 방식이 유리할 수도 있다.

이상으로 그린 정리의 기본 정의와 해석, 증명 스케치 및 예제를 살펴봤다. 평면 상의 선적분을 다룰 때, 특히 제어공학적 상황에서 경계와 내부 간의 상호 관계를 이해하는 데 핵심 역할을 한다는 점에 주목해야 한다.
