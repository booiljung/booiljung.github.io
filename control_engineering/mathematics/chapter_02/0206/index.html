곡선 매개화는 하나의 변수(일반적으로 \(t\)라 부르는 실수 변수)를 통해 공간상의 곡선 위 점들을 표현하는 방법이다. 예를 들어 2차원 공간에서 곡선을 생각하면, \(t\)를 특정 실수 구간에서 정의하고, 해당 \(t\)에 대응하는 2차원 좌표를 \(\mathbf{r}(t)\)라는 벡터로 쓴다. 이 과정을 매개변수화라고 부른다.

매개변수화된 곡선 \(\mathbf{r}(t)\)를 사용하면, 곡선을 따라 움직이는 점의 속도나 가속도를 미분으로 손쉽게 구할 수 있고, 곡선 위에 정의된 물리량을 적분하거나 곡선 자체의 길이를 계산하는 문제도 간단해진다. 고차원 공간에서도 마찬가지 방식으로 곡선을 다룰 수 있다. 매개변수를 한 개 이상으로 확장하면 곡면에 대한 매개변수화도 가능하다.

매개변수화에서 가장 많이 쓰이는 변수는 \(t\)지만, 상황에 따라 \(s\), \(\tau\) 등의 문자나 실제 물리 시간 같은 다른 변수를 이용하기도 한다. 특정 곡선을 다시 다른 방식으로 표현하는 것은 재매개변수화(reparametrization)라고 한다. 예컨대 \(u = f(t)\)라는 변환을 통해, 곡선을 지나는 “속도”를 달리하거나 특정 구간을 강조하는 등의 조작이 가능하다. 이러한 유연성이 제어공학에서 실제 시스템을 다룰 때에도 큰 이점을 제공한다.

하나의 예로 2차원에서 원점을 중심으로 반지름 1인 원을 생각해보면, 이를 다음과 같이 매개변수화할 수 있다. \(t\)를 \(0 \le t < 2\pi\)로 정의하고,

\[ \mathbf{r}(t) = \begin{pmatrix} \cos t \\ \sin t \end{pmatrix} \]

로 잡으면, \(t\)가 커짐에 따라 원을 시계 반대방향으로 한 바퀴 돈다. 매개변수 \(t\)의 구간을 조절해 곡선의 일부만 표현할 수도 있고, 여러 바퀴를 표현할 수도 있다.

매개변수화의 예시와 시각적 이해 2차원¶

곡선 매개변수화는 단순히 좌표를 기술하는 것을 넘어, 특정 구간을 어떻게 훑을지 결정하고 곡선 위의 운동을 시간에 대응해 해석하는 등 다양한 활용이 가능하다.

예를 들어 2차원에서 두 점을 잇는 선분을 매개변수화해보자. 점 \(\mathbf{A} = (x_A, y_A)\)와 \(\mathbf{B} = (x_B, y_B)\)를 지나는 선분을 매개변수 \(t \in [0,1]\)로 표현하려면 다음과 같은 벡터 함수를 정의할 수 있다.

\[ \begin{align} \mathbf{r}(t) &= (1 - t)\begin{pmatrix} x_A \\ y_A \end{pmatrix} + t\begin{pmatrix} x_B \\ y_B \end{pmatrix} \\ &= \begin{pmatrix} (1 - t)x_A + tx_B \\ (1 - t)y_A + ty_B \end{pmatrix}. \end{align} \]

\(t\)가 0부터 1까지 균일한 간격으로 변화한다고 간주하면, \(t=0\)일 때 점 \(\mathbf{A}=(x_A, y_A)\)에서 시작해서, \(t=1\)일 때 점 \(\mathbf{B}=(x_B, y_B)\)에 도달한다. 이 과정을 실제로 확인하기 위해 아래 예시 코드에서는 \(\mathbf{A}=(0,0)\), \(\mathbf{B}=(2,3)\)로 잡고 \(t\)를 0부터 1까지 0.1 간격으로 변화시키며 선분을 그린다.

phpCopyEditimport numpy as np
import matplotlib.pyplot as plt

# A, B 정의
A = np.array([0, 0])
B = np.array([2, 3])

# t를 0부터 1까지 0.1 간격으로 생성
t_values = np.linspace(0, 1, 11)

# 매개변수화된 점들 계산
points = []
for t in t_values:
    x_t = (1 - t)*A[0] + t*B[0]
    y_t = (1 - t)*A[1] + t*B[1]
    points.append([x_t, y_t])

points = np.array(points)

# 그림 그리기
plt.figure(figsize=(5, 5))
plt.plot(points[:,0], points[:,1], 'ro--', label="Parametrized Line")
plt.plot([A[0], B[0]], [A[1], B[1]], 'b-', label="Direct Line Segment")
plt.axis('equal')
plt.legend()
plt.title("Line from A to B by Parametrization (t interval = 1)")
plt.show()

코드 실행 결과, \(t=0\)일 때 \(\mathbf{A}=(0,0)\)에서 시작해, \(t=1\)일 때 \(\mathbf{B}=(2,3)\)에 이르는 선분이 점선(빨간색)으로 표시된다. 이렇게 매개변수화로 얻어진 점들을 직접 연결한 결과(파란색 선분)와 정확히 일치함을 확인할 수 있다.

이 예시는 \(t\)를 0부터 1까지 일정하게 증가시키며 \(\mathbf{r}(t)\)를 구해 각 점들을 그리는 방식으로, 정의된 선분을 시각적으로 이해하는 한 방법이다.

매개변수화의 예시와 시각적 이해 3차원¶

3차원 공간에서의 예시로 나선을 들어보자. \(z\)축을 기준으로 빙글빙글 도는 나선을 매개변수화하려면, \(t\)를 0부터 원하는 상한까지 정의하고,

\[ \begin{align} \mathbf{r}(t) &= \begin{pmatrix} R \cos t \\ R \sin t \\ h t \end{pmatrix} \end{align} \]

와 같이 둘 수 있다. 여기서 \(R\)은 회전 반지름, \(h\)는 \(z\)축 방향으로 올라가는 비율에 해당한다. \(t\)가 증가할수록 원을 회전하면서 동시에 \(z\)축을 따라 일정 간격으로 올라가는 곡선이 형성된다.

아래는 Python 코드로 2차원 원과 3차원 나선을 매개변수화하고 시각화하는 간단한 예시다. 이 코드를 통해 매개변수화 개념이 실제 계산과 그래프에 어떻게 적용되는지 살펴볼 수 있다.

import numpy as np
import matplotlib.pyplot as plt
from mpl_toolkits.mplot3d import Axes3D

# 2D 원 매개변수화
t_circle = np.linspace(0, 2*np.pi, 200)
x_circle = np.cos(t_circle)
y_circle = np.sin(t_circle)

# 3D 나선 매개변수화
t_helix = np.linspace(0, 6*np.pi, 500)
R = 1.0
h = 0.1
x_helix = R * np.cos(t_helix)
y_helix = R * np.sin(t_helix)
z_helix = h * t_helix

fig = plt.figure(figsize=(12, 5))

# 2D 원 그리기
ax1 = fig.add_subplot(1, 2, 1)
ax1.plot(x_circle, y_circle, 'b-')
ax1.set_aspect('equal', 'box')
ax1.set_title("2D Circle Parametrization")

# 3D 나선 그리기
ax2 = fig.add_subplot(1, 2, 2, projection='3d')
ax2.plot(x_helix, y_helix, z_helix, 'r-')
ax2.set_title("3D Helix Parametrization")

plt.show()

이처럼 매개변수화는 스칼라 변수 \(t\)를 변화시키며 곡선을 표현하는 방식이다. \(t\)의 범위를 적절히 설정하면, 곡선의 일정 구간이나 전체를 쉽게 표현할 수 있다. 또, \(t\)가 시간과 밀접한 의미를 가진다면, 곡선을 시간에 따라 움직이는 운동학적 해석도 가능하다.

매개변수 \(t\)를 다른 표현으로 바꿔 같은 곡선을 기술하는 재매개변수화를 통해, 곡선 위를 지나는 속도나 시점 등을 마음대로 조절할 수도 있다. 이것은 제어공학에서 어떤 로봇의 조인트 궤적을 바꾸거나 위성의 궤도를 수정하는 문제를 다룰 때 유용하게 쓸 수 있다.

결국, 곡선 매개화는 공간 내의 곡선을 다루는 근본적인 방법이며, 이후 벡터 미분·적분을 적용할 때 반드시 필요한 기반이 된다. 매개변수화가 있어야 곡선 위에서 속도, 가속도, 곡률 등을 체계적으로 계산할 수 있고, 곡선을 따라 적분을 수행할 수 있는 명확한 틀이 마련된다. 이렇게 확보된 매개변수화 개념은 제어공학을 비롯한 여러 분야에서 곡선이나 경로 문제를 다룰 때 없어서는 안 될 중요한 도구이다.
