Previous 보존 벡터장(Conservative Vector Field)과 퍼텐셜 함수

공간에서 정의된 스칼라 함수로서 벡터장과 밀접하게 연결되는 퍼텐셜을 스칼라 퍼텐셜이라고 한다. 공간의 각 점에 대해 하나의 실수 값을 갖는 함수를 퍼텐셜이라고 할 때, 어떤 벡터장이 그 퍼텐셜의 그라디언트와 같으면 그 벡터장은 보존력장이라 불린다. 이러한 스칼라 퍼텐셜 개념은 에너지, 전위, 위치에 의한 영향 등을 해석할 때 널리 활용된다.

보존력장의 개념과 함께 스칼라 퍼텐셜을 이해하면, 물리 시스템에서 에너지를 계산하거나 경로에 무관한 일을 정량화할 수 있다. 이는 3차원 공간뿐 아니라 그 이상 차원으로 확장되며, 편미분 연산과 선적분 이론에서 중요한 역할을 한다.

정의와 개념¶

스칼라 퍼텐셜은 공간상의 각 점에서 하나의 실수 값을 가지는 함수다. 만약 어떤 벡터장 \(\mathbf{F}\)가 주어지고, 그것이 \(\mathbf{F} = \nabla \phi\) 꼴로 표현 가능하다면, 여기서 \(\phi\)가 \(\mathbf{F}\)의 스칼라 퍼텐셜이 된다. 기호로 나타내면 다음과 같다. 인라인수식으로 표현하면 \(\mathbf{F}(\mathbf{r}) = \nabla \phi(\mathbf{r})\)다.

벡터장 \(\mathbf{F} = \nabla \phi\)는 \(\nabla \times \mathbf{F} = \mathbf{0}\)을 만족한다. 즉 컬(curl)이 0이라는 사실이 스칼라 퍼텐셜의 존재를 암시한다. 반대로 특정 구역에서 연속적인 벡터장 \(\mathbf{F}\)가 \(\nabla \times \mathbf{F} = \mathbf{0}\)을 만족하면, 일반적으로 \(\mathbf{F}\)는 어떤 스칼라 퍼텐셜의 그라디언트로 표현될 수 있다.

스칼라 퍼텐셜이 존재하는 벡터장, 즉 보존력장은 경로에 무관하게 선적분 값이 동일하다는 특징을 갖는다. 이는 보존력장 \(\mathbf{F} = \nabla \phi\)에 대해, 임의의 경로 \(C\)를 따라 점 \(\mathbf{r}_1\)에서 \(\mathbf{r}_2\)로 이동한다고 할 때 다음과 같은 사실로부터 확인할 수 있다.

\[ \begin{align} \int_{C} \mathbf{F} \cdot d\mathbf{r} &= \int_{C} \nabla \phi \cdot d\mathbf{r}. \end{align} \]

경로 \(C\)를 매개변수 \(t\)로 표현하고, \(d\mathbf{r} = \mathbf{r}'(t),dt\)로 쓰면 다음 단계로 전개된다.

\[ \begin{align} \int_{C} \nabla \phi \cdot d\mathbf{r} &= \int_{a}^{b} \nabla \phi(\mathbf{r}(t)) \cdot \mathbf{r}'(t)\,dt = \int_{a}^{b} \frac{d}{dt} \phi(\mathbf{r}(t)) \,dt. \end{align} \]

이 적분은 \(\phi(\mathbf{r}_2) - \phi(\mathbf{r}_1)\)이 되어, 경로에 무관하게 시작점과 끝점의 퍼텐셜 차이만으로 결정된다.

보존력장과 스칼라 퍼텐셜¶

보존력장이라는 표현은 물리학과 공학에서 에너지를 효율적으로 다룰 때 중요한 의미를 갖는다. 예를 들어, 어떤 힘 벡터장 \(\mathbf{F}\)가 스칼라 퍼텐셜로부터 유도된다면 경로에 따른 일(work)이 일정해지고, 보존되는 물리량을 명확히 정의할 수 있다.

선적분을 통해 일이나 에너지를 구할 때 \(\mathbf{F} = \nabla \phi\) 형태라면 퍼텐셜 함수 \(\phi\)가 중심에 놓이게 된다. 이 때문에 보존력장은 여러 분야에서 실용적으로 쓰이며, 다변수 미적분학에서는 스칼라 퍼텐셜이 존재하는지(즉 컬이 0인지) 여부가 해석의 출발점이 된다.

수학적 전개¶

3차원에서 \(\mathbf{F} = (F_x, F_y, F_z)\)가 스칼라 퍼텐셜 \(\phi\)를 가진다고 하면, 다음 관계를 만족한다.

\[ \dfrac{\partial \phi}{\partial x} = F_x, \quad \dfrac{\partial \phi}{\partial y} = F_y, \quad \dfrac{\partial \phi}{\partial z} = F_z. \]

이를 적분을 통해 구하려면 \(F_x\)를 \(x\)에 대해 적분한 뒤, 아직 고려되지 않은 변수에 대한 함수를 추가로 더하는 식으로 구성한다. 그 뒤 \(\dfrac{\partial \phi}{\partial y} = F_y\)와 \(\dfrac{\partial \phi}{\partial z} = F_z\) 조건을 만족하도록 맞춰 나가면 유일(상수 차이 정도의 의미)한 스칼라 퍼텐셜 \(\phi\)를 구할 수 있다.

보존력장을 이루는 벡터장의 필수 조건은 \(\nabla \times \mathbf{F} = \mathbf{0}\)이므로, 실제로 \(\mathbf{F}\)가 보존력장인지 판별할 때 컬이 0인지 살펴보면 된다. 이 성질은 다차원 물리계에서 매우 중요하게 쓰이는데, 제어공학 문제에서도 경로 독립적인 특성을 확인하거나, 자율주행 등에서 위치 에너지 함수를 정의할 때 스칼라 퍼텐셜이 직접적인 해석 도구로 사용된다.

벡터 퍼텐셜(Vector Potential)¶

벡터 퍼텐셜은 3차원 공간에서 정의되는 벡터 함수로서, 또 다른 벡터장의 컬로 나타난다. 벡터장 \(\mathbf{B}\)가 주어졌을 때, 만약 \(\mathbf{B} = \nabla \times \mathbf{A}\)를 만족하는 \(\mathbf{A}\)가 존재한다면, \(\mathbf{A}\)를 \(\mathbf{B}\)의 벡터 퍼텐셜이라 한다.

전자기학에서 자기장 \(\mathbf{B}\)는 발산이 0이라는 성질, 즉 \(\nabla \cdot \mathbf{B} = 0\)을 가진다. 이러한 성질은 벡터 퍼텐셜의 존재를 뒷받침한다. 실제로 임의의 벡터장 \(\mathbf{B}\)에 대해 \(\nabla \cdot \mathbf{B} = 0\)이면, 적절한 연결성(단일 연결 영역)이 보장되는 공간에서 \(\mathbf{B} = \nabla \times \mathbf{A}\) 꼴이 가능하다는 사실이 알려져 있다.

정의와 개념¶

공간상의 점 \(\mathbf{r} = (x, y, z)\)에서 벡터장 \(\mathbf{B}(\mathbf{r})\)를 생각하자. 만약 \(\mathbf{B}(\mathbf{r}) = \nabla \times \mathbf{A}(\mathbf{r})\)를 만족하는 \(\mathbf{A}(\mathbf{r})\)가 존재하면, \(\mathbf{A}\)는 \(\mathbf{B}\)의 벡터 퍼텐셜이라 불린다.

이때 \(\nabla \cdot (\nabla \times \mathbf{A}) = 0\)이 항상 성립하므로, \(\mathbf{B} = \nabla \times \mathbf{A}\)인 경우는 \(\nabla \cdot \mathbf{B} = 0\)을 자동으로 만족한다. 이 필요충분조건 때문에, 발산이 0인 벡터장이라면 벡터 퍼텐셜이 존재할 가능성이 크다.

벡터 퍼텐셜은 유일하지 않다. 임의의 스칼라 함수 \(\Lambda(\mathbf{r})\)에 대해 \(\mathbf{A}' = \mathbf{A} + \nabla \Lambda\)를 정의하면, 컬 연산에서 스칼라 함수의 그라디언트는 0이 되므로 여전히 \(\mathbf{B}\)를 얻는다. 이를 게이지 변환이라고 하고, 실제 물리량인 \(\mathbf{B}\)에 영향을 주지 않기 때문에 여러 형태의 \(\mathbf{A}\)가 가능하다.

성질과 예시¶

벡터 퍼텐셜의 대표적인 예는 전자기학에서 자기장을 나타낼 때 사용된다. 자기장 \(\mathbf{B}\)가 일정하거나, 발산이 0인 국소적 상황에서 \(\mathbf{A}\)가 존재함을 확인하는 예시가 많다.

예를 들어, \(\mathbf{B} = \begin{pmatrix} 0 \ 0 \ C \end{pmatrix}\)와 같은 일정한 z방향의 자기장이 있다고 하자. 적당한 \(\mathbf{A}\)를 찾으려면, \(\mathbf{A} = \nabla \times \mathbf{A} = \mathbf{B}\) 조건을 만족해야 한다. 하나의 간단한 선택으로 \(\mathbf{A} = \begin{pmatrix} -\dfrac{C}{2}y \ \dfrac{C}{2}x \ 0 \end{pmatrix}\) 를 쓰면,

\[ \begin{align} \nabla \times \mathbf{A} &= \begin{pmatrix} \dfrac{\partial}{\partial y} (0) - \dfrac{\partial}{\partial z} \left(\dfrac{C}{2}x\right) \ \]

6pt] \dfrac{\partial}{\partial z} \left(-\dfrac{C}{2}y\right) - \dfrac{\partial}{\partial x} (0) \

\[ 6pt] \dfrac{\partial}{\partial x} \left(\dfrac{C}{2}x\right) - \dfrac{\partial}{\partial y} \left(-\dfrac{C}{2}y\right) \end{pmatrix} = \begin{pmatrix} 0 \\ 0 \\ C \end{pmatrix}. \end{align} \]

따라서 이것은 한 가지 형태의 벡터 퍼텐셜 예시가 된다. 물론 임의의 스칼라 함수를 더해도 컬 결과는 동일하므로 다양한 형태의 \(\mathbf{A}\)가 가능하다.

응용과 해석¶

벡터 퍼텐셜 개념은 전자기학을 비롯해, 유체역학이나 위상수학, 심지어 제어공학의 특정 문제에서도 활용된다. 전자기학에서는 전기장 \(\mathbf{E}\)와 자기장 \(\mathbf{B}\)를 퍼텐셜 관점에서 통합적으로 다룬다. 대표적으로 시간에 따라 변하는 상황에서 다음 식이 쓰인다. 인라인수식으로 표현하면 \(\mathbf{E} = -\nabla \phi - \dfrac{\partial \mathbf{A}}{\partial t}\)이고, \(\mathbf{B} = \nabla \times \mathbf{A}\)다.

실제 제어공학적으로 복잡한 물리계를 모델링할 때 벡터 퍼텐셜을 사용하면, 문제를 더 단순화하거나 수학적으로 구조화하여 접근할 수 있다. 예컨대 자석이나 전자기 코일과 관련된 시스템에서 자기장을 다룰 때, 직접 \(\mathbf{B}\)를 다루기보다 \(\mathbf{A}\)로부터 \(\mathbf{B}\)를 구해 해석하는 편이 편리한 경우가 많다.

아래 예시는 Python을 이용해 2차원 격자에서 스칼라 퍼텐셜과 벡터 퍼텐셜을 각각 설정하고, 유한 차분으로 그라디언트와 컬을 계산한 뒤 그 결과를 확인하는 단순 시뮬레이션 예시다.

import numpy as np

# 격자 설정
nx, ny = 21, 21
x = np.linspace(-1, 1, nx)
y = np.linspace(-1, 1, ny)
dx = x[1] - x[0]
dy = y[1] - y[0]

# 예시 스칼라 퍼텐셜과 벡터 퍼텐셜
def phi_func(xx, yy):
    return xx**2 + yy**2  # 단순 방사형 형태

def A_func_x(xx, yy):
    return -0.5 * yy      # 간단한 예시
def A_func_y(xx, yy):
    return 0.5 * xx       # 간단한 예시

phi = np.zeros((nx, ny))
Ax = np.zeros((nx, ny))
Ay = np.zeros((nx, ny))

# 격자 위에서 함수 값 계산
for i in range(nx):
    for j in range(ny):
        phi[i,j] = phi_func(x[i], y[j])
        Ax[i,j] = A_func_x(x[i], y[j])
        Ay[i,j] = A_func_y(x[i], y[j])

# 스칼라 퍼텐셜의 그라디언트 계산 (유한 차분)
grad_phi_x = (np.roll(phi, -1, axis=0) - phi) / dx
grad_phi_y = (np.roll(phi, -1, axis=1) - phi) / dy

# 벡터 퍼텐셜의 컬 계산: z성분 = dAy/dx - dAx/dy
curl_A_z = (np.roll(Ay, -1, axis=0) - Ay) / dx - (np.roll(Ax, -1, axis=1) - Ax) / dy

print("중앙 지점에서 스칼라 퍼텐셜의 그라디언트:")
print(grad_phi_x[nx//2, ny//2], grad_phi_y[nx//2, ny//2])

print("중앙 지점에서 벡터 퍼텐셜의 컬 (z성분):")
print(curl_A_z[nx//2, ny//2])

이처럼 실제 계산에서 스칼라 퍼텐셜 \(\phi\)의 그라디언트를 구하면 경로에 무관한 보존력장을 확인할 수 있고, 벡터 퍼텐셜 \(\mathbf{A}\)의 컬을 구하면 발산이 0인 장을 쉽게 표현할 수 있다. 이런 계산과정은 제어공학에서 모델링이나 시뮬레이션, 해석에도 큰 도움을 주며, 복잡한 물리적 구조를 단순화해볼 수 있는 수단이 된다.
