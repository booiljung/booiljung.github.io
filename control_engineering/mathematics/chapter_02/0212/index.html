부분도함수와 편도함수는 여러 변수를 포함하는 함수에서 특정 변수에 대한 변화량을 해석하고 계산하기 위한 핵심 도구다. 이러한 도구는 벡터 미분적분학의 기본을 이루며, 제어공학 전반에 걸쳐 매우 유용하게 쓰인다. 여러 변수로 이루어진 함수가 있을 때, 한 변수를 변화시키면서 나머지 변수들은 고정한 상태로 생각하면, 그 한 변수에 대한 변화율을 부분도함수 혹은 편도함수라고 한다. 이를 통해 다변수 함수의 성질을 세부적으로 파악할 수 있다.

다음에서는 부분도함수와 편도함수의 개념, 계산 방법, 그리고 활용 시 유의점을 순차적으로 살펴본다.

부분도함수의 개념¶

하나 이상의 변수를 갖는 함수에서 특정 변수에 대한 미분을 정의하기 위해서는, 관심을 갖지 않는 나머지 변수들을 모두 상수로 취급해야 한다. 예를 들어, 함수가 \(f(x, y)\) 형태로 정의되어 있을 때, \(x\)만 변화시키고 \(y\)는 고정해 둔다고 하자. 이때 정의되는 \(x\)에 대한 미분이 부분도함수다.

두 변수를 갖는 함수 \(f(x, y)\)에 대해 \(x\)에 대한 부분도함수는 다음과 같이 정의한다. \(y\)는 상수로 여기고, \(x\)가 작은 양 \(\Delta x\)만큼 변화할 때 \(f\)가 얼마나 변화하는지를 본다. 따라서 \(x\)에 대한 부분도함수는 극한의 형태로 표현할 수 있다. 변수가 두 개지만, 확장해서 \(n\)개의 변수를 갖는 일반적인 경우도 마찬가지 방식으로 생각한다.

부분도함수 개념은 다차원 공간에서 어떤 방향으로 진행될 때, 그 함수가 어떠한 변화량을 갖는지 미리 살펴보는 역할을 한다. 예를 들어, 제어공학에서 시스템의 상태변수가 여러 개일 때, 시스템을 모델링한 함수가 특정 상태변수에 따라 어떻게 변하는지를 구할 때 부분도함수가 쓰인다. 하나의 변수만 주목해 다른 모든 변수를 상수로 취급한다면, 여러 변수 중 특정 변수 하나에 집중적인 해석이 가능해진다.

편도함수의 정의와 기호¶

부분도함수를 편도함수라고도 부른다. 일반적으로는 \(\dfrac{\partial}{\partial x}\)라는 기호를 써서 \(x\)에 대한 편도함수를 나타낸다. 만약 함수가 \(f(x, y)\)라면, \(x\)에 대한 편도함수는 보통 다음처럼 쓴다.

\[ \begin{align} \dfrac{\partial f}{\partial x}(x, y) = \lim_{\Delta x \to 0} \dfrac{f(x + \Delta x, y) - f(x, y)}{\Delta x} \end{align} \]

\(y\)에 대한 편도함수 역시 같은 원리로 정의한다.

\[ \begin{align} \dfrac{\partial f}{\partial y}(x, y) = \lim_{\Delta y \to 0} \dfrac{f(x, y + \Delta y) - f(x, y)}{\Delta y} \end{align} \]

다변수 함수에서 편도함수를 모두 구하면, 이들을 하나의 벡터로 모아 그레디언트(Gradient)라는 개념을 정의하기도 한다. 예를 들어, \(f : \mathbb{R}^n \to \mathbb{R}\)일 때, 편도함수들을 각각 모아 다음과 같이 표현할 수 있다.

\[ \begin{align} \nabla f(\mathbf{x}) = \begin{pmatrix} \dfrac{\partial f}{\partial x_1}(\mathbf{x}) \\ \dfrac{\partial f}{\partial x_2}(\mathbf{x}) \\ \vdots \\ \dfrac{\partial f}{\partial x_n}(\mathbf{x}) \end{pmatrix} \end{align} \]

이때 \(\mathbf{x} = (x_1, x_2, \dots, x_n)\)이다. 이러한 편도함수의 집합은 벡터 형태로 함수의 증가 방향과 크기를 나타내어, 제어 최적화 문제나 추정 알고리즘 등에서 유용하게 활용된다.

편도함수 기호는 다음과 같은 중요한 성질을 가지고 있다. \(f(x, y)\)에 대해 \(x\)에 대한 편도함수를 표기할 때 \(\dfrac{\partial f}{\partial x}\)라고 쓰며, 이는 \(f_x\)로 표기하기도 한다. 유사하게 \(y\)에 대한 편도함수는 \(\dfrac{\partial f}{\partial y}\) 혹은 \(f_y\)로 표기한다. 이와 같은 기호를 상황에 따라 적절히 선택해 사용하면 된다.

부분도함수 계산 방법¶

편도함수를 구하는 절차는 본질적으로 일변수 미분과 같다. 다만, 여러 변수를 갖는 함수에서 하나의 변수만을 움직이고, 나머지는 고정한다는 점이 다르다. 간단한 예시로 \(f(x, y) = x^2 + 3xy + y^2\)라는 함수가 있다고 하자. 이때 \(x\)에 대한 편도함수 \(\dfrac{\partial f}{\partial x}\)를 구하고, 이어서 \(y\)에 대한 편도함수 \(\dfrac{\partial f}{\partial y}\)를 구한다.

먼저 \(x\)에 대한 편도함수를 구하기 위해 \(y\)는 상수로 간주하고, \(x\)에 대한 일반적인 미분 규칙을 적용한다.

\[ \begin{align} f(x, y) &= x^2 + 3xy + y^2 \\ \dfrac{\partial f}{\partial x} &= 2x + 3y + 0 \end{align} \]

여기서 \(y^2\) 항은 \(x\)와 무관하므로 상수 취급되어 미분값이 0이다. 따라서 \(x\)에 대한 편도함수는 \(2x + 3y\)가 된다.

다음으로 \(y\)에 대한 편도함수를 구하기 위해 \(x\)를 상수로 본다.

\[ \begin{align} f(x, y) &= x^2 + 3xy + y^2 \\ \dfrac{\partial f}{\partial y} &= 0 + 3x + 2y \end{align} \]

\(x^2\) 항은 \(y\)와 무관하므로 소멸하고, 나머지 항에서 \(3x\)와 \(2y\)가 남아 \(y\)에 대한 편도함수는 \(3x + 2y\)가 된다.

이 과정을 통해 확인할 수 있듯, 편도함수를 구할 때에는 특정 변수만을 독립변수로 취급해 일변수 미분을 수행한다. 함수에 포함된 다른 변수들은 상수 항이 되므로, 미분 과정에서 소멸하거나 그대로 남게 된다.

편도함수를 실제 계산해야 하는 상황이 많을 때는 심볼릭 계산 기능을 지원하는 소프트웨어나 프로그래밍 언어를 활용하기도 한다. 아래 예시는 Python을 이용해 간단한 편도함수를 근사적으로 계산하는 방법을 보여준다.

def f(x, y):
    return x**2 + 3*x*y + y**2

def partial_derivative_x(f, x, y, h=1e-5):
    return (f(x + h, y) - f(x - h, y)) / (2*h)

def partial_derivative_y(f, x, y, h=1e-5):
    return (f(x, y + h) - f(x, y - h)) / (2*h)

# 예시 계산
x_val = 2.0
y_val = 1.0
print("∂f/∂x at (2, 1):", partial_derivative_x(f, x_val, y_val))
print("∂f/∂y at (2, 1):", partial_derivative_y(f, x_val, y_val))

이 코드는 전방차분과 후방차분을 결합한 중앙차분 방법을 이용해 편도함수를 근사 계산한다. \(h\)를 충분히 작게 잡으면, 실제 해석적 미분 결과와 거의 일치하게 만든다. 편도함수의 해석적 표현을 직접 구하기 어려운 복잡한 함수를 다룰 때는 이러한 수치적 방식이 유용하게 쓰일 수 있다.

각 편도함수를 구한 뒤, 이를 조합해 그레디언트를 구하면 다양한 해석이 가능하다. 예를 들어, 해당 지점에서 함수의 국소적인 증가 방향이나 최적화 문제에서의 경사 하강 등을 해석할 수 있다.

편도함수가 확장되는 개념으로, 임의 방향으로의 변화량을 표현하는 방향도함수(Direction Derivative)가 있다. 방향도함수는 특정 벡터 방향으로 함수를 변화시킬 때의 변화율을 뜻한다. 그러나 그레디언트와 방향 벡터의 내적을 이용해 구할 수 있으므로, 편도함수 개념을 잘 이해하고 있으면 쉽게 접근이 가능하다.

편도함수는 후속적으로 다변수 체인 룰, 야코비 행렬, 헤시안 행렬을 비롯한 고차 미분 연산의 기반을 이룬다. 제어 이론에서 여러 입력 신호와 상태 변수에 대한 민감도를 해석할 때도 이 편도함수 개념이 핵심을 이룬다. 때문에, 편도함수 계산에 능숙해지는 것은 매우 중요한 단계다.

이상으로 부분도함수와 편도함수의 기본 개념과 계산법을 살펴보았다. 이는 다차원 공간에서 함수를 다룰 때 반드시 필요한 도구이며, 제어공학에서의 미적분학적 해석 전반에 폭넓게 쓰이게 된다.
