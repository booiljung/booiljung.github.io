동적 시스템을 해석하거나 설계할 때는 시간에 따라 변하는 여러 물리량을 벡터나 행렬로 표현하고, 이를 미분적분 연산을 통해 다룬다. 예를 들어 물체의 위치와 속도를 동시에 추적하고자 할 때 위치와 속도를 하나의 상태 벡터로 묶고, 그 변화를 미분방정식 형태로 기술한다. 이 과정에서 벡터 미분적분학은 시스템 상태 변화를 이해하고 해를 구하는 핵심 도구가 된다.

아래에서는 동적 시스템을 대표적으로 나타내는 상태공간 표현을 예로 들어, 벡터 미분과 적분이 어떻게 적용되는지를 자세히 살펴본다.

동적 시스템의 일반적 표현과 벡터 연산¶

동적 시스템은 시간에 따라 변화하는 상태를 다룬다. 이를 상태 벡터 \(\mathbf{x}(t)\)로 표현할 수 있다. 상태 벡터는 다음과 같은 꼴을 가진다.

\[ \begin{align} \mathbf{x}(t) = \begin{pmatrix} x_1(t)\\ x_2(t)\\ \vdots\\ x_n(t) \end{pmatrix} \end{align} \]

시간 \(t\)는 연속변수이므로 \(\mathbf{x}(t)\)는 연속 시간에 따라 달라지는 벡터 함수가 된다. 미분 연산은 상태 벡터가 얼마나 빠르게 변하는지를 알려준다. \(\mathbf{x}(t)\)에 대해 시간 미분 연산 \(\dfrac{d}{dt}\)을 적용하면

\[ \begin{align} \dfrac{d\mathbf{x}(t)}{dt} = \begin{pmatrix} \dfrac{dx_1(t)}{dt}\\ \dfrac{dx_2(t)}{dt}\\ \vdots\\ \dfrac{dx_n(t)}{dt} \end{pmatrix} \end{align} \]

와 같은 결과가 되어, 각 성분별 시간 변화율을 벡터 형태로 묶은 결과를 얻는다.

적분 연산은 시스템의 동작을 누적적으로 살펴볼 때 유용하다. 예를 들어 시간 구간 \([t_0, t_1]\)에서 \(\mathbf{x}(t)\)를 적분하면

\[ \begin{align} \int_{t_0}^{t_1} \mathbf{x}(t)\,dt = \begin{pmatrix} \int_{t_0}^{t_1} x_1(t)\,dt\\ \int_{t_0}^{t_1} x_2(t)\,dt\\ \vdots\\ \int_{t_0}^{t_1} x_n(t)\,dt \end{pmatrix} \end{align} \]

과 같이 적분도 각 성분별로 계산되어 벡터로 표현된다.

아주 흔히 쓰이는 선형 시불변 시스템의 상태방정식은 다음과 같은 형태로 제시된다.

\[ \begin{align} \dfrac{d\mathbf{x}(t)}{dt} &= \mathbf{A}\,\mathbf{x}(t) + \mathbf{B}\,\mathbf{u}(t) \end{align} \]

여기서 \(\mathbf{x}(t)\)는 상태 벡터, \(\mathbf{A}\)는 \(n \times n\) 행렬, \(\mathbf{B}\)는 \(n \times m\) 행렬, \(\mathbf{u}(t)\)는 \(m\)차원 입력 벡터다. 미분 연산이 시스템의 상태변화를 직접적으로 표현한다. 이 방정식을 풀이하면, 시스템이 시간에 따라 어떻게 진화하는지 해석하고 제어 전략을 세울 수 있다.

상태방정식에서의 벡터 미분¶

\(\mathbf{A}\)가 상수 행렬(시간에 대해 변화하지 않음)이라고 가정할 때 입력이 없는 경우( \(\mathbf{u}(t) = \mathbf{0}\))의 상태방정식은

\[ \begin{align} \dfrac{d\mathbf{x}(t)}{dt} &= \mathbf{A}\,\mathbf{x}(t) \end{align} \]

과 같은 꼴이 된다. 이를 해석하려면 지수 행렬 \(\exp(\mathbf{A} t)\) 개념을 적용한다. 스칼라 지수함수 \(e^{\lambda t}\)를 행렬에 확장한 지수 행렬은 거듭제곱 급수를 통해 정의된다.

\[ \begin{align} \exp(\mathbf{A} t) = \mathbf{I} + \mathbf{A}t + \dfrac{(\mathbf{A}t)^2}{2!} + \dfrac{(\mathbf{A}t)^3}{3!} + \cdots \end{align} \]

이 개념을 이용해 동적 시스템의 해를 구하면, 다음과 같은 결과를 얻는다.

\[ \begin{align} \dfrac{d\mathbf{x}(t)}{dt} &= \mathbf{A}\,\mathbf{x}(t)\\ \mathbf{x}(t) &= \exp(\mathbf{A} t)\,\mathbf{x}(0) \end{align} \]

중간 전개 과정을 살펴보면 \(\mathbf{A}\)가 대각화 가능하다고 가정할 때 \(\mathbf{A} = \mathbf{P}\mathbf{\Lambda}\mathbf{P}^{-1}\)로 나타낼 수 있다. 이때,

\[ \begin{align} \exp(\mathbf{A} t) = \mathbf{P}\,\exp(\mathbf{\Lambda} t)\,\mathbf{P}^{-1} \end{align} \]

가 되고, \(\mathbf{\Lambda}\)는 \(\mathbf{A}\)의 고유값들을 대각선에 갖는 대각 행렬이다. 따라서

\[ \begin{align} \exp(\mathbf{\Lambda} t) = \begin{pmatrix} e^{\lambda_1 t} & 0 & \cdots & 0\\ 0 & e^{\lambda_2 t} & \cdots & 0\\ \vdots & \vdots & \ddots & \vdots\\ 0 & 0 & \cdots & e^{\lambda_n t} \end{pmatrix} \end{align} \]

이 되어, 각 고유값 \(\lambda_i\)마다 스칼라 지수함수를 적용한 형태로 표현된다. 이처럼 벡터 미분연산을 통해 얻은 1차 선형 미분방정식을 지수 행렬을 이용해 해석할 수 있다.

선형 시스템 해석에서의 적분 (상태천이행렬)¶

입력이 존재하는 경우에는 적분을 통해 누적 효과를 고려해야 한다. 일반적으로

\[ \begin{align} \dfrac{d\mathbf{x}(t)}{dt} &= \mathbf{A}\,\mathbf{x}(t) + \mathbf{B}\,\mathbf{u}(t) \end{align} \]

를 적분 연산으로 풀면 상태천이행렬(state transition matrix) 해가 나타난다.

\[ \begin{align} \mathbf{x}(t) &= \exp(\mathbf{A}(t - t_0))\,\mathbf{x}(t_0) + \int_{t_0}^{t} \exp\bigl(\mathbf{A}(t - \tau)\bigr)\,\mathbf{B}\,\mathbf{u}(\tau)\,d\tau \end{align} \]

여기서 \(\exp(\mathbf{A}(t - t_0)),\mathbf{x}(t_0)\)는 초기 상태가 시간 \(t\)까지 변화하는 과정을 의미한다. 두 번째 적분항은 입력 \(\mathbf{u}(t)\)에 의해 유도된 응답을 시간에 걸쳐 적분해 누적 효과를 반영한 것이다. \(\exp\bigl(\mathbf{A}(t - \tau)\bigr),\mathbf{B},\mathbf{u}(\tau)\)는 시각 \(\tau\)에서의 입력이 이후 시각 \(t\)에 어떻게 전달되는지를 나타내며, 이 과정이 \(\tau\)에 대해 적분되어 최종 상태에 영향을 미친다.

예시: Python을 통한 간단한 시뮬레이션¶

아래는 2차원 선형 시스템을 Python으로 시뮬레이션하는 간단한 예시다. 지수 행렬을 직접 계산하기보다는 과학용 라이브러리를 사용해 미분방정식을 수치적으로 푸는 방법을 보인다.

import numpy as np
import matplotlib.pyplot as plt
from scipy.integrate import odeint

# 상태방정식을 정의
def state_equation(x, t, A, B, u_func):
    u = u_func(t)  # 시각 t에서의 입력
    dxdt = A @ x + B @ u
    return dxdt

# 시스템 파라미터 설정
A = np.array([[0, 1],
              [-2, -3]])
B = np.array([[0],
              [1]])

def u_func(t):
    return np.array([1.0])  # 상수 입력

x0 = np.array([1.0, 0.0])  # 초기 상태
t = np.linspace(0, 10, 200)

sol = odeint(state_equation, x0, t, args=(A, B, u_func))

plt.plot(t, sol[:, 0], label='x1(t)')
plt.plot(t, sol[:, 1], label='x2(t)')
plt.xlabel('Time')
plt.ylabel('States')
plt.legend()
plt.title('2차원 선형 시스템 상태변화')
plt.show()

이 코드에서 \(\mathbf{A}\)와 \(\mathbf{B}\)는 시스템 행렬이고, \(u(t)\)는 입력이다. 초기 상태 \(\mathbf{x}(0)\)에서 출발해 시간이 증가함에 따라 벡터 미분방정식을 적분(odeint 함수 사용)하며 해를 구한다. 벡터 미분은 순간 변화율을, 적분은 시간에 따른 누적 변화를 각각 반영하므로, 이 두 연산은 동적 시스템 해석에 필수적이다.

벡터 미분적분학은 상태변수 변화율과 외부 입력의 누적 영향을 동시에 고려해 동적 시스템 거동을 예측하고 설계할 때 중심적인 역할을 한다. 여기서 제시된 상태공간 표현과 기본 예시는 실제 제어 시스템 해석, 신호 처리, 물리 시스템 모델링 등 다양한 분야에서 널리 응용된다.
