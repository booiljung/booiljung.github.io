동적 시스템은 여러 상태 변수가 시간에 따라 변화하는 현상을 수학적으로 해석하고 제어하는 데에 초점을 맞춘다. 이러한 시스템은 보통 여러 개의 상호 연관된 미분방정식으로 표현되며, 각 미분방정식은 벡터 형태의 입력, 상태, 출력 등을 포함한다. 벡터 미분적분학은 이러한 시스템의 동적 거동을 정확하고 체계적으로 다루는 핵심적인 도구가 된다.

벡터 미분은 각 상태 변수에 대한 민감도를 표현하거나, 선형화(linearization)를 수행할 때 Jacobian 행렬을 구하는 과정에 적용된다. 벡터 적분은 시간에 따른 상태 변수의 누적 효과를 살피거나, 에너지 및 성능 지표 등을 분석할 때 유용하다. 이 챕터에서는 동적 시스템에 벡터 미분적분을 적용하는 이론적 배경과 핵심 원리를 엄밀히 다룬다.

동적 시스템을 단순화하여 벡터 방정식으로 표현하면

\[ \mathbf{x}(t) \in \mathbb{R}^n \]

과 같은 상태 벡터를 고려하게 된다. 이때 시간에 대한 상태 벡터의 변화율은

\[ \dot{\mathbf{x}}(t) = \mathbf{f}(\mathbf{x}(t), t) \]

와 같이 표현할 수 있으며, 이는 제어공학에서 흔히 볼 수 있는 벡터 미분방정식이다.

동적 시스템의 다양한 분석에서는 위 형태의 방정식에 벡터 미분 연산과 적분 연산을 주로 사용한다. 예를 들어, 선형화(linearization)를 통해 비선형 시스템 근처에서의 국소 해석을 수행할 수 있고, 상태방정식(State-Space Equation) 형태에서 해(解)를 구하기 위해 적분 연산이 필수적으로 동원된다.

여기서는 먼저 동적 시스템의 일반적 표현 형태와 의미를 살펴보고, 벡터 미분과 적분이 각각 어떠한 방식으로 적용되는지 다룬다. 그리고 단순한 예시에서부터 점진적으로 확장하여 선형 시스템의 해석과 수치해석적 접근 방법까지 살펴본다.

동적 시스템의 일반적 표현¶

동적 시스템은 시간에 따라 변화하는 입력, 상태, 출력 변수로 구성된다. 이를 일반화하면

\[ \mathbf{x}(t) \in \mathbb{R}^n, \quad \mathbf{u}(t) \in \mathbb{R}^m, \quad \mathbf{y}(t) \in \mathbb{R}^p \]

와 같은 세 벡터로 요약할 수 있다. 여기서 \(\mathbf{x}(t)\)는 상태(state), \(\mathbf{u}(t)\)는 입력(input), \(\mathbf{y}(t)\)는 출력(output)에 해당한다.

이 세 벡터 간의 관계를 연속시간으로 표현하면

\[ \dot{\mathbf{x}}(t) = \mathbf{f}(\mathbf{x}(t), \mathbf{u}(t), t), \quad \mathbf{y}(t) = \mathbf{g}(\mathbf{x}(t), \mathbf{u}(t), t) \]

로 나타낼 수 있다. 이때 \(\mathbf{f}\)와 \(\mathbf{g}\)는 일반적으로 비선형 함수를 가정하며, 동적 시스템의 거동을 결정하는 핵심적인 요소가 된다.

벡터 미분적분학은 위 식에서 각각의 부분(예: \(\dot{\mathbf{x}}(t)\))에 대해 미분 연산 및 적분 연산을 수행할 때, 모든 상태 변수가 동시에 어떻게 변화하고 어떻게 누적되는지를 정확하게 파악하는 수단을 제공한다.

벡터 미분의 역할¶

동적 시스템에서 벡터 미분은 다음과 같은 상황에서 핵심적 역할을 담당한다.

동적 식 \(\dot{\mathbf{x}}(t) = \mathbf{f}(\mathbf{x}(t), t)\)에서의 \(\mathbf{f}\)가 비선형일 때, 특정 평형점(equilibrium point) 근방에서의 선형화를 수행한다. 선형화를 위해서는 벡터 \(\mathbf{f}\)에 대한 Jacobian 행렬을 계산해야 한다.

\[ \mathbf{A}(\mathbf{x}_0) = \frac{\partial \mathbf{f}}{\partial \mathbf{x}}\Big|_{\mathbf{x} = \mathbf{x}_0} \]

은 다음과 같이 구성된다.

\[ \begin{align} \mathbf{A}(\mathbf{x}_0) = \begin{pmatrix} \frac{\partial f_1}{\partial x_1} & \frac{\partial f_1}{\partial x_2} & \cdots & \frac{\partial f_1}{\partial x_n} \\ \frac{\partial f_2}{\partial x_1} & \frac{\partial f_2}{\partial x_2} & \cdots & \frac{\partial f_2}{\partial x_n} \\ \vdots & \vdots & \ddots & \vdots \\ \frac{\partial f_n}{\partial x_1} & \frac{\partial f_n}{\partial x_2} & \cdots & \frac{\partial f_n}{\partial x_n} \end{pmatrix} \end{align} \]

이 행렬을 이용하면 시스템을 평형점 근방에서

\[ (x(t)−x0)\dot{\mathbf{x}}(t) \approx \mathbf{A}(\mathbf{x}_0)\, (\mathbf{x}(t) - \mathbf{x}_0) \]

와 같은 선형근사 형태로 볼 수 있다.

벡터 미분 연산은 상태 천이 행렬(State Transition Matrix)을 구하거나, 제어 이득(matrix) 계산 시 필요한 민감도 해석 등에 사용된다. 특히 고급 제어 이론에서는 Hessian 행렬이나 고차 미분 항까지 고려하는 상황도 발생하며, 이는 시스템의 안정성 분석이나 성능 최적화에서 중요하다.

벡터장 \(\mathbf{F}(\mathbf{x})\)에 대한 발산(divergence), 회전(curl)과 같은 미분 연산이 동적 시스템 해석에서 직접적으로 등장하는 경우는 유체역학적 시스템이나 전자기학적 시스템 등 물리적 해석이 필요한 분야에서 의미가 있다. 예를 들어 발산을 통해 시스템 내부로부터의 출력 또는 입력 흐름을 해석할 수 있고, 회전을 통해 순환 성분을 분석할 수 있다. 제어공학에서는 이를 일반화한 해석이 필요한 경우가 많지는 않으나, 물리 기반 모델링 단계에서 중요한 지표를 제시할 수 있다.

벡터 적분의 역할¶

동적 시스템에서 벡터 적분은 다음과 같은 상황에서 응용된다.

시스템의 해(解)를 직접 구하거나, 시간응답(time response)을 계산할 때, 예를 들어

\[ \mathbf{x}(t) = \mathbf{x}(t_0) + \int_{t_0}^t \mathbf{f}(\mathbf{x}(\tau), \tau)\, d\tau \]

와 같은 형태로 적분을 통해 현재 상태를 계산할 수 있다. 이는 선형 시스템과 비선형 시스템 모두에서 해를 구하는 기본식으로 볼 수 있다.

에너지나 비용함수(cost function)를 정의하여 시스템의 성능을 분석할 때, 상태 혹은 출력 벡터를 적분함으로써 누적 비용, 혹은 누적 에너지를 평가할 수 있다. 예를 들어, 선형-이차 최적제어(Linear Quadratic Regulator, LQR)에서 목표는

\[ J = \int_{0}^{\infty} \left( \mathbf{x}^\top(t) \mathbf{Q}\, \mathbf{x}(t) + \mathbf{u}^\top(t) \mathbf{R}\, \mathbf{u}(t) \right) dt \]

와 같은 적분 형태의 비용함수를 최소화하는 것이다.

물리적인 입출력 에너지를 해석하거나 전달 경로를 계산할 때, 벡터 적분은 실제로 계 내부에서 주고받는 에너지를 엄밀하게 추적하는 수단이 된다. 시스템이 여러 개의 상호작용 요소로 구성되어 있을 경우, 각 요소 간의 에너지, 운동량 등 물리량의 전달(흐름)을 적분 형태로 정량화할 수 있다.

예시: 선형 상태방정식에 대한 해석¶

동적 시스템 해석에서 가장 기초적인 형태는 선형 시스템이다. 선형 상태방정식은

\[ \dot{\mathbf{x}}(t) = \mathbf{A}\, \mathbf{x}(t) + \mathbf{B}\, \mathbf{u}(t) \]

로 표현할 수 있다. 여기서 \(\mathbf{A}\)는 \(n \times n\) 차원의 행렬, \(\mathbf{B}\)는 \(n \times m\) 차원의 행렬이다.

입력을 0이라 가정한 자율 시스템(autonomous system)의 경우

\[ x(t)\dot{\mathbf{x}}(t) = \mathbf{A}\, \mathbf{x}(t) \]

라는 단순화된 형태를 얻게 된다. 이 때 시스템의 해는 초기값 \(\mathbf{x}(0)\)에 대해 지수행렬(exponential matrix)을 이용하여

\[ \mathbf{x}(t) = e^{\mathbf{A}t}\, \mathbf{x}(0) \]

와 같이 구할 수 있다.

이 해를 적분 형태로 전개하면

\[ \mathbf{x}(t) = \mathbf{x}(0) + \int_{0}^{t} \mathbf{A}\, \mathbf{x}(\tau)\, d\tau \]

가 된다. 적분 연산을 반복적으로 적용해 나가면, 테일러 급수 전개에 따라

\[ e^{\mathbf{A}t} = \mathbf{I} + \mathbf{A}t + \frac{\mathbf{A}^2 t^2}{2!} + \frac{\mathbf{A}^3 t^3}{3!} + \cdots \]

와 같은 결과가 유도된다. 이는 벡터 적분 및 미분 연산이 상호 보완적으로 작동함을 잘 보여준다.

비자율 시스템, 즉 \(\mathbf{A}\)가 시간에 따라 변하는 경우에는

\[ \dot{\mathbf{x}}(t) = \mathbf{A}(t)\, \mathbf{x}(t) \]

가 되므로, 다음 형태의 해를 얻게 된다.

\[ \mathbf{x}(t) = \mathbf{\Phi}(t, 0) \, \mathbf{x}(0) \]

여기서 \(\mathbf{\Phi}(t, 0)\)는 상태 천이 행렬(state transition matrix)로,

\[ \mathbf{\Phi}(t, 0) = \mathbf{I} + \int_{0}^{t} \mathbf{A}(\tau_1)\, d\tau_1 + \int_{0}^{t} \mathbf{A}(\tau_1) \int_{0}^{\tau_1} \mathbf{A}(\tau_2)\, d\tau_2 \, d\tau_1 + \cdots \]

와 같이 Dyson 시리즈 형태로 적분 전개가 가능하다. 이는 벡터 적분의 누적 과정이 순차적으로 쌓여서 최종적인 해를 구성함을 보여준다.

수치해석적 접근¶

다소 복잡한 동적 시스템의 해석에서는 벡터 미분적분의 해석해(닫힌 해형)를 구하기 어려운 경우가 많다. 이때는 수치해석적 방법을 이용해 벡터 미분방정식을 근사적으로 푼다. 대표적으로 오일러(Euler) 방법, 룽게-쿠타(Runge-Kutta) 계열의 방법 등이 사용된다.

예시로 Python을 이용하여 단순한 2차원 동적 시스템을 수치 적분해보자. 다음은 \(\dot{\mathbf{x}}(t) = \mathbf{f}(\mathbf{x}(t))\) 형태를 4차 룽게-쿠타 방법으로 푸는 간단한 코드 예시이다. 여기서는 2차원 벡터 \(\mathbf{x}(t) = (x_1(t), x_2(t))^\top\)를 가정한다.

import numpy as np
import matplotlib.pyplot as plt

def f(x):
    # x는 [x1, x2] 형태의 numpy 배열
    # 예: 비선형 함수 f(x1, x2) = ( x2, -x1 ) 라고 가정
    return np.array([x[1], -x[0]])

def rk4_step(x, dt):
    k1 = f(x)
    k2 = f(x + 0.5 * dt * k1)
    k3 = f(x + 0.5 * dt * k2)
    k4 = f(x + dt * k3)
    return x + (dt / 6.0) * (k1 + 2*k2 + 2*k3 + k4)

t0 = 0.0
tf = 10.0
dt = 0.01
N = int((tf - t0)/dt)

x_vals = []
t_vals = []

x = np.array([1.0, 0.0])  # 초기 상태
for i in range(N+1):
    t = t0 + i * dt
    x_vals.append(x)
    t_vals.append(t)
    x = rk4_step(x, dt)

x_vals = np.array(x_vals)

plt.plot(t_vals, x_vals[:,0], label='x1(t)')
plt.plot(t_vals, x_vals[:,1], label='x2(t)')
plt.xlabel('Time')
plt.ylabel('States')
plt.legend()
plt.grid(True)
plt.show()

위 코드는 2차원 벡터 미분방정식 \(\dot{\mathbf{x}}(t)=\mathbf{f}(\mathbf{x}(t))\)을 벡터 형태 그대로 다루고 있다. 각 단계마다 \(\mathbf{x}(t)\)의 상태는 누적 적분(룽게-쿠타 방법의 가중합 형태)으로 업데이트된다. 즉, 벡터 적분이 수치적으로 구현된 예라고 볼 수 있다.

이 수치해석 방법에서 벡터 미분 연산은 각 단계에서 기울기(gradient, 여기서는 \(\dot{\mathbf{x}}(t))\)를 평가함으로써 적용되며, 적분 연산은 누적 과정(적분)을 통해 해석적으로 접근하기 어려운 시스템을 근사한다.

mermaid를 이용한 간단한 구조 표현¶

동적 시스템에서 벡터 미분적분의 흐름을 mermaid로 단순화하여 나타내면 다음과 같은 흐름도가 있다.

flowchart LR A["초기 상태 x(0)"] --> B["벡터 미분방정식 dot{x}(t) = f(x(t))"] B --> C["적분 연산 x(t) = x(0) + int f(x(tau)) d(tau)"] C --> D[최종 해석 결과 또는 시뮬레이션]

여기서 초기 상태 \(\mathbf{x}(0)\)에서 출발하여 벡터 미분방정식을 통해 \(\dot{\mathbf{x}}(t)\)를 계산한 뒤, 시간 적분을 누적 수행해 \(\mathbf{x}(t)\)를 구하는 과정을 단순히 흐름으로 표시한 것이다.

--- 및 요약적 고찰

동적 시스템에서 벡터 미분적분은 필수적이다. 벡터 미분은 시스템이 여러 차원에서 어떻게 변화하는지, 각 상태가 서로에게 어떠한 영향을 미치는지를 해석하는 도구를 제공한다. 벡터 적분은 시간에 따른 누적 결과를 통해 실제 시스템의 상태 변화를 추적하거나, 성능 지표(에너지, 비용 등)를 산출하는 역할을 한다.

복잡한 비선형 시스템 분석에서는 Jacobian, Hessian 등 고차 미분 연산이 빈번히 등장하며, 시스템 안정을 평가하거나 비선형 제어기 설계 시 기본이 된다. 또한 적분 연산은 시간영역에서의 상태 예측, 출력을 계산하거나, 최적화 과정에서 비용함수를 구성하는 데 핵심적인 역할을 한다.

이처럼 벡터 미분적분학은 제어공학에서 다루는 거의 모든 문제의 근간이 되며, 다양한 수치해석 방법을 통해 실제 산업 환경에도 적용할 수 있다. 이를 기반으로 동적 시스템을 더 깊이 이해하고, 효과적인 제어 전략을 수립할 수 있다.
