상반평면은 복소평면에서 특정 영역을 다루기 위한 개념이다. 복소수 \(z = a + j b\)에서 \(b > 0\)인 부분을 상반평면이라고 부른다. 해석학에서 이 영역을 다루면 적분 경로나 영상 원리, 편미분 방정식의 해석 등에 유용하다. 제어공학에서도 선형시스템의 안정도 분석이나 주파수응답 해석에서 상반평면을 효과적으로 활용할 수 있다. 특히 \(s = \sigma + j \omega\) 형태의 복소 변수에서 \(\omega > 0\)에 해당하는 상반평면을 다룰 때가 많다. 이는 시간영역 해석에서 거동을 예측하거나 라플라스 변환을 활용할 때 중요한 역할을 한다.

주파수응답 해석에서는 전달함수를 \(s\)-영역에서 \(s = j \omega\)로 대체해 진폭과 위상을 분석한다. 이때 상반평면은 여러 해석적 성질을 담고 있어서, 적분경로를 설정하거나 영점과 극점의 분포를 이해하는 데 중요한 단서를 제공한다. 복소 해석의 주요 정리인 코시 적분 정리(Cauchy’s integral theorem)나 레지듀 정리(Residue theorem)도 상반평면 경로로 설정된 폐곡선을 자주 다룬다. 이러한 관점에서 상반평면이 제어이론과 주파수응답 해석에 어떤 역할을 하는지 살펴본다.

상반평면의 정의와 복소함수의 성질¶

복소수 \(z = a + j b\)에서 \(b > 0\)인 부분을 상반평면이라 한다. 상반평면은 해석적(analytic)인 함수나 특수 적분을 다룰 때 자주 등장한다. 예를 들어, 다음과 같은 복소함수를 생각할 수 있다.

\[ \begin{align} f(z) = \dfrac{1}{z^2 + 1}. \end{align} \]

이 함수를 상반평면에 대한 적분 경로로 둘러싸면, 극점이 어디에 위치하는지에 따라 적분값을 구별할 수 있다. 이때 극점이 상반평면 내부에 있으면 레지듀 정리를 사용하여 적분값을 구하고, 내부에 극점이 없으면 적분값이 0이 된다. 이러한 복소 적분 기법은 제어공학에서 전달함수의 특성식을 해석하거나 필터 설계를 할 때 큰 도움을 준다.

상반평면에서 해석적인(analytic) 함수를 다룰 때는 코시-리만 방정식을 만족하는지를 살핀다. 복소함수 \(f(z) = u(a, b) + j v(a, b)\)가 상반평면에서 해석적이라면, 다음과 같은 코시-리만 방정식을 만족한다.

\[ \begin{align} \dfrac{\partial u}{\partial a} = \dfrac{\partial v}{\partial b}, \quad \dfrac{\partial u}{\partial b} = -\dfrac{\partial v}{\partial a}. \end{align} \]

상반평면에서 해석적이라면, 이러한 성질을 이용해 임의의 경로 적분에서 동일한 결과를 얻을 수 있으며, 경로 변형을 통한 문제 해결이 가능해진다. 이는 주파수응답 해석에서도 중요한데, 나중에 볼 전달함수의 극과 영점이 어떤 영역에 위치하는지에 따라 시스템의 성능과 응답을 분류하기 때문이다.

상반평면의 중요한 특징 중 하나는 실축(\(b = 0\))을 경계로 아래쪽에 아랫반평면(\(b < 0\)), 그리고 위쪽에 상반평면(\(b > 0\))이 존재한다는 점이다. 시스템의 주파수응답은 일반적으로 실축을 따라 \(s = j\omega\) (\(\omega \in \mathbb{R}\))로 설정해 해석한다. 이 축이 상반평면과 아랫반평면을 나누는 경계선이 되므로, 해석의 편의를 위해 상반평면에서의 성질이나 적분을 고려하는 경우가 많다.

주파수응답과 상반평면의 관계¶

주파수응답 해석은 선형 시불변 시스템에서 매우 중요하다. 전달함수 \(G(s)\)가 주어졌을 때, \(s = j\omega\)로 치환하여 \(G(j\omega)\)를 구하고, 이를 다시 데카르트 좌표평면에서 실수부와 허수부로 나누어 해석한다. 예를 들어, 1차 전달함수

\[ \begin{align} G(s) = \dfrac{1}{s + 1} \end{align} \]

에 대해 주파수응답은

\[ \begin{align} G(j\omega) &= \dfrac{1}{j\omega + 1} = \dfrac{1}{1 + j\omega}. \end{align} \]

이 식의 실수부와 허수부를 각각 \(R(\omega)\)와 \(I(\omega)\)라고 하면,

\[ \begin{align} G(j\omega) &= \dfrac{1}{1 + j\omega} = \dfrac{1 - j\omega}{(1 + j\omega)(1 - j\omega)} \\ &= \dfrac{1 - j\omega}{1 + \omega^2}. \end{align} \]

따라서

\[ \begin{align} R(\omega) = \dfrac{1}{1 + \omega^2}, \quad I(\omega) = -\dfrac{\omega}{1 + \omega^2}. \end{align} \]

이 주파수응답을 복소평면에서 해석할 때, 실축 위에 \(\omega\)가 0에서 \(\infty\)까지 움직이면 \(j\omega\)는 상반평면과 아랫반평면을 가르는 축을 따라 이동한다. 극점이 상반평면에 있는지, 혹은 아랫반평면에 있는지, 또는 경계선(실축)에 있는지에 따라 응답이 달라진다.

실제 물리계에서 상반평면 극점은 일반적으로 지수적으로 감쇠하지 않는 해석을 유발하기 때문에, 안정도 분석에서 중요한 의미를 갖는다. 특정 해석에서는 상반평면에 극점이 존재하면 시스템이 발산하는 거동을 보일 수도 있다. 그렇지만 한편으로는 적분 경로를 상반평면으로 설정하면, 특정 적분이 쉽게 해석되거나 상반평면의 해석적 성질(analyticity)을 이용하여 결과를 단순화하는 경우도 많다.

주파수응답 해석 시, 상반평면과 아랫반평면이 동시에 중요한 이유는 복소함수가 상반평면에서만 해석적인 경우도 있고, 아랫반평면에서만 해석적인 경우도 있기 때문이다. 경우에 따라 시스템의 인과성(causality)이나 에너지 유한성 조건이 아랫반평면 해석성과 관련 있을 수 있다. 결국 \(s\)-평면에서 실축(\(\omega\)축)은 주파수응답을 직접적으로 나타내고, 그 축을 기준으로 위쪽(상반평면)과 아래쪽(아랫반평면)이 시스템 해석에 서로 다른 역할을 하게 된다.

상반평면으로 적분 경로를 잡는 예는 매우 많은데, 대표적으로 역 라플라스 변환의 브로멜(Bromwich) 적분이 있다. 이는 실축을 기준으로 위쪽 혹은 아래쪽에 평행선을 긋고 적분을 수행한다. 그러면 적분 경로가 상반평면을 포함하여 폐곡선을 이룰 경우 레지듀 정리에 의해 손쉽게 특정 계수나 함수값을 구하게 된다. 이때 적절한 경로 설정이 용이한 이유는 상반평면(또는 아랫반평면)에서 해석적(analytic)인 성질에 기인한다.

주파수응답 해석에서 나이퀴스트(Nyquist) 궤적을 그릴 때도 상반평면 경로를 설정하거나, 실축을 기준으로 위쪽 경로를 확장해 극점과 영점이 어떻게 배치되어 있는지 확인하기도 한다. 요약하자면, 상반평면은 복소해석에서 매우 중요한 영역이며, 제어공학의 주파수응답 해석을 이해할 때에도 극, 영점 및 폐루프 궤적을 해석하는 기본 요소가 된다.

아래는 간단한 예로, Python을 사용하여 간단한 전달함수 \(G(s) = \dfrac{1}{s+1}\)의 보드선도(Bode plot)에 해당하는 주파수응답을 계산해 그래프로 그려보는 코드이다. 이 코드를 통해 실축(\(s = j\omega\))을 따라 움직이는 \(\omega\)에 대한 상반평면과의 관계를 간접적으로 체험할 수 있다.

import numpy as np
import matplotlib.pyplot as plt

omega = np.linspace(0, 10, 1000)
G_mag = []
G_phase = []

for w in omega:
    # G(jw) = 1 / (1 + j*w)
    # 분자와 분모를 직접 계산
    numerator = 1 + 0j
    denominator = 1 + 1j * w

    G_val = numerator / denominator
    G_mag.append(abs(G_val))
    G_phase.append(np.angle(G_val))

plt.figure(figsize=(10,4))

plt.subplot(1,2,1)
plt.plot(omega, G_mag)
plt.title("Magnitude")
plt.xlabel("omega")
plt.ylabel("|G(jw)|")
plt.grid(True)

plt.subplot(1,2,2)
plt.plot(omega, np.degrees(G_phase))
plt.title("Phase")
plt.xlabel("omega")
plt.ylabel("Phase (degrees)")
plt.grid(True)

plt.tight_layout()
plt.show()

이 코드는 \(j\omega\)를 따라 시스템을 자극했을 때의 응답을 살펴보게 해준다. 여기서 \(j\omega\)는 실축을 기준으로 바라보면 상반평면과 아랫반평면을 갈라놓는 축이 된다. 즉, 주파수응답은 상반평면과 직접적인 관계보다는 축을 기준으로 계산되지만, 상반평면상의 복소 적분 이론이나 극점-영점의 분포 해석 등을 통해 추후 다양한 문제를 해결하는 데 필요한 이론적 기반을 마련한다.

상반평면의 중요한 특징을 다시 강조하자면, \(b > 0\) 영역에서 해석함수를 다룰 때 코시 적분 공식이나 레지듀 정리가 간단히 적용된다. 이는 주파수응답뿐 아니라 라플라스 변환의 역변환, 편미분 방정식의 그린 함수(Green’s function) 해석 등에 다방면으로 쓰인다. 따라서 제어공학을 위해 복소해석을 학습할 때, 상반평면과 주파수응답의 관계를 이해하는 것은 매우 필수적이며, 이는 나이퀴스트, 보드, 근궤적 등의 해석에서 직접 응용된다.
