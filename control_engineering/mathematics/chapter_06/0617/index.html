상Jω축은 복소평면 상에서 실수부가 0인 축을 의미하며, 제어공학에서 시스템의 안정도를 논할 때 매우 중요한 경계를 제공한다. 일반적으로 시스템의 특성방정식에서 해가 상Jω축에 위치하거나 이를 넘어가면 안정도가 바뀌는 경향이 있다. 이를 이해하기 위해서는 복소해석 관점에서 상Jω축의 의미를 살펴보고, 시스템의 안정도 조건을 어떻게 점검하는지 상세히 살펴볼 필요가 있다.

여기서는 상Jω축에 대입되는 변수 \(s\)에 대해 \(s = jω\)를 적용한다. 이때 \(ω\)는 실수이며, \(s\)가 상Jω축을 따라 움직일 때 시스템의 특성방정식에서 어떤 현상이 발생하는지 단계별로 추적한다.

상Jω축의 기하학적 해석¶

복소평면에서 임의의 점 \(s\)는 실수부와 허수부로 표현된다. 만약 \(s\)의 실수부를 0이라 두면 \(s\)는 jω 꼴이 된다. 여기서 ω는 실수이며, ω > 0이면 상부 허수축, ω < 0이면 하부 허수축을 나타낸다. 상Jω축을 기준으로 왼쪽 반평면은 실수부가 음수인 영역, 오른쪽 반평면은 실수부가 양수인 영역이다.

시스템 해석 관점에서 s-평면은 전달함수 또는 상태방정식으로 기술되는 선형 시불변(LTI) 시스템의 동특성 해를 시각화하는 중요한 수단이다. s = jω는 다음과 같은 중요한 의미를 갖는다.

첫째, s의 실수부가 0이므로, 여기서 관찰되는 진동은 감쇠(damping)가 없거나, 매우 특수한 형태로만 존재한다. 둘째, 시스템의 특성방정식이 s = jω에서 만족된다는 것은 실제 제어시스템에서 맥동(oscillation)이나 경계 안정(한계주파수) 상태를 나타낸다.

즉, 상Jω축에 극점이 존재한다면, 시스템이 특정 주파수 ω에서 그 해가 발산도 수렴도 하지 않고 영구진동을 일으키는 한계 상태가 될 수 있다. 따라서 안정도를 논의할 때는 상Jω축의 극점 존재 여부가 매우 중요하다.

안정도 경계의 기본 원리¶

시스템 특성방정식이

\[ \begin{align} \mathbf{\Delta}(s) = 0 \end{align} \]

로 표현된다고 하자. 예를 들어 n차 시스템일 경우, 일반적으로

\[ \begin{align} \mathbf{\Delta}(s) = s^n + a_{n-1}s^{n-1} + \dots + a_1 s + a_0 \end{align} \]

형태가 될 수 있다. 이때 특성방정식의 근 s가 모두 왼쪽 반평면(실수부 < 0)에 존재하면 시스템은 안정하다고 말한다. 만약 실수부가 양수인 근이 하나라도 있으면 시스템은 불안정하고, 상Jω축에 근이 존재하면 경계 안정 상태에 해당한다.

상Jω축을 안정도 경계로 두고 해석하기 위해 s = jω를 대입해보자. 그러면

\[ \begin{align} \mathbf{\Delta}(j\omega) = (j\omega)^n + a_{n-1}(j\omega)^{n-1} + \dots + a_1 (j\omega) + a_0 \end{align} \]

가 된다. 복소수 지수법칙에 의해 \((j\omega)^m = j^m \omega^m\)이며, \(j^2 = -1\), \(j^3 = -j\), \(j^4 = 1\)인 순환성을 갖는다. 따라서 위 식은 실수부와 허수부로 분해 가능하다.

간단한 2차 예시로 살펴보면,

\[ \begin{align} \mathbf{\Delta}(j\omega) = (j\omega)^2 + a_1 (j\omega) + a_0 = -\omega^2 + j(a_1 \omega) + a_0. \end{align} \]

여기서 실수부는 \(a_0 - \omega^2\), 허수부는 \(a_1 \omega\)가 된다. 경계 조건에서 \(\mathbf{\Delta}(j\omega) = 0\)을 만족한다면 실수부와 허수부가 동시에 0이어야 하므로,

\[ \begin{align} \begin{cases} a_0 - \omega^2 = 0 \\ a_1 \omega = 0 \end{cases} \end{align} \]

가 성립한다. 만약 \(a_1 \neq 0\)라면 \(\omega = 0\)이 되어 실수부 식에서 \(a_0 = 0\)이 된다. 혹은 \(a_1 = 0\)이면 다른 형태의 조건으로 해가 결정된다. 이런 식으로 주파수 \(\omega\)를 찾아낼 수 있고, 그때 해당 주파수에서 해 s = jω가 경계 조건을 형성하게 된다.

이를 보다 일반적으로 확장하면 n차 방정식에서도 \(s = j\omega\)를 대입한 후, 실수부와 허수부를 모두 0으로 놓아서 안정도를 바꾸는 주파수 \(\omega\)가 존재하는지 살펴볼 수 있다. 만약 특정 양의 \(\omega\)에서 \(\mathbf{\Delta}(j\omega) = 0\)인 해가 존재한다면, 그 시스템은 상Jω축을 극점으로 갖는다. 즉, 시스템이 발진(oscillation)에 가까운 상태가 되며 안정도 경계를 형성한다.

상Jω축 근방에서의 동작과 해석¶

시스템이 상Jω축 근방에서 어떻게 동작하는지를 분석하면, 시스템이 안정한지, 불안정으로 넘어가는지, 혹은 경계 안정 상태로 유지되는지 판단할 수 있다.

상Jω축 근방이라는 것은 Re(s)가 0에 매우 가까운 영역을 의미한다. 만약 s가 소량의 양의 실수부를 갖고 있다면(오른쪽 반평면), 지수함수 \(e^{\sigma t}\)에서 \(\sigma>0\)인 경우로서 시스템 응답이 발산하게 된다. 반대로 s가 소량의 음의 실수부를 갖고 있다면(왼쪽 반평면), \(\sigma<0\)가 되어 시스템 응답이 수렴하게 된다. 이 작은 차이가 시스템 안정도를 구분짓는 매우 중요한 역할을 한다.

예를 들어,

\[ \begin{align} s = \sigma + j\omega \end{align} \]

로 두고, \(\sigma\)가 0에서 조금이라도 양수가 되면 지수항 \(e^{\sigma t}\)가 발산한다. 따라서 상Jω축을 초과해 오른쪽으로 가는 순간, 시스템은 불안정해진다. 상Jω축보다 왼쪽에 위치하면 안정적으로 감쇠되는 지수항 \(e^{\sigma t}\)를 갖는다.

이처럼 상Jω축 근방에서 극점의 움직임을 추적하면, 제어기의 이득이나 시스템 파라미터 변화에 따라 안정도 경계가 어떻게 이동하는지 알 수 있다. 흔히 Nyquist나 루트 궤적(root locus) 기법 등을 활용하여, 특성방정식의 극점들이 상Jω축을 통과하는지를 관찰해 안정도 변화를 파악한다.

상Jω축에서의 극점 조건과 예시¶

특성방정식이 \(n\)차 다항식일 때, \(s = j\omega\)를 대입했을 때의 실수부와 허수부 조건은 매우 복잡해질 수 있다. 하지만 본질은 다음과 같이 요약된다:

\[ \begin{align} \mathbf{\Delta}(j\omega) = 0 \\ \implies \Re[\mathbf{\Delta}(j\omega)] = 0 \quad \text{및} \quad \Im[\mathbf{\Delta}(j\omega)] = 0. \end{align} \]

이를 만족하는 ω를 찾으면, 그때의 s = jω가 특성방정식을 만족하므로 상Jω축에 극점이 존재함을 뜻한다. 이 값들을 나열해보면, 시스템이 정확히 어느 주파수에서 발진 상태가 되는지(또는 경계 안정이 되는지) 확인할 수 있다.

여기서는 단순화된 예시를 하나 들어본다. 예를 들어, 전달함수가 다음과 같은 2차 시스템을 가정해보자.

\[ \begin{align} G(s) = \dfrac{\omega_0^2}{s^2 + 2 \zeta \omega_0 s + \omega_0^2}. \end{align} \]

이 경우 특성방정식은

\[ \begin{align} s^2 + 2 \zeta \omega_0 s + \omega_0^2 = 0 \end{align} \]

가 된다. 상Jω축에서 근을 찾으려면 s = jω를 대입한다.

\[ \begin{align} (j\omega)^2 + 2 \zeta \omega_0 (j\omega) + \omega_0^2 = 0 \\ -\omega^2 + j(2 \zeta \omega_0 \omega) + \omega_0^2 = 0. \end{align} \]

실수부와 허수부를 각각 0으로 두면,

\[ \begin{align} -\omega^2 + \omega_0^2 = 0 \\ 2 \zeta \omega_0 \omega = 0. \end{align} \]

두 번째 식에서 \(\zeta \neq 0\)이고 \(\omega_0 \neq 0\)라면, 이는 \(\omega = 0\)을 의미하게 된다. 첫 번째 식에 \(\omega = 0\)을 대입하면 \(\omega_0^2 = 0\)이 되어 모순이 일어나므로, 일반적인 경우 상Jω축에 실제 극점이 생기지 않는다. 대신 만약 \(\zeta = 0\)이라면(무감쇠 시스템) 첫 번째 식에서 \(\omega = \omega_0\)를 얻어 \(s = j \omega_0\)가 된다. 이때 진동(발진)이 발생한다.

이처럼 상Jω축에서 극점이 나타나기 위한 조건은 시스템 파라미터에 따라 달라지며, 각 파라미터의 값이 어떻게 변하면 극점이 상Jω축으로 이동하는지 살펴보는 것이 안정도 해석의 핵심이다.

상Jω축 해석을 위한 간단한 Python 예시¶

다음 예시는 Sympy 라이브러리를 사용하여, 단순화된 특성방정식에 s = jω를 대입하고 실수부와 허수부를 분리해보는 과정을 보여준다. 여기서는 구체적인 시스템 설계보다는, \(s = j\omega\) 대입 시 식을 어떻게 분해하고 ω를 찾을 수 있는지 확인하는 목적으로 제시한다.

import sympy as sp

s = sp.Symbol('s', complex=True)
omega = sp.Symbol('omega', real=True, nonnegative=True)
j = sp.I  # 파이썬에서는 i가 아닌 I로 허수를 표현
a0 = sp.Symbol('a0', real=True)
a1 = sp.Symbol('a1', real=True)
a2 = sp.Symbol('a2', real=True)

# 2차 특성방정식 a2*s^2 + a1*s + a0
Delta = a2*s**2 + a1*s + a0

# s = j*omega 대입
Delta_jw = Delta.subs(s, j*omega)

# 실수부와 허수부 분리
real_part = sp.simplify(sp.re(Delta_jw))
imag_part = sp.simplify(sp.im(Delta_jw))

# 결과 출력
print("실수부 =", real_part)
print("허수부 =", imag_part)

# 특정 파라미터 값으로 예를 들어 a2=1, a1=0, a0=-1인 경우
res = [real_part.subs({a2:1, a1:0, a0:-1}),
       imag_part.subs({a2:1, a1:0, a0:-1})]
print("대입 결과: ", res)

이 코드를 실행하면, 주어진 특성방정식에 대해 상Jω축 해석을 어떻게 수행할 수 있는지 알 수 있다. 실제로는 원하는 파라미터들에 대해 \(\omega\)를 찾고, 그 값에서 실수부와 허수부가 모두 0이 되는지 확인해야 한다. 만약 그러한 양의 ω가 존재한다면, 그 시스템은 해당 주파수에서 상Jω축 극점을 갖게 된다.

요약 및 확장¶

상Jω축 해석은 시스템 안정도를 판별하는 중요한 수단이다. 특성방정식에서 s = jω를 대입하여 실수부와 허수부를 0으로 두는 과정은, 시스템이 임계 안정 상태에 도달하는 주파수를 찾는 핵심 절차다. 만약 어떤 파라미터 변화나 이득 조정 등으로 인해 극점이 상Jω축을 넘어 오른쪽 반평면으로 이동한다면, 시스템은 불안정해진다.

실제 제어계 설계에서는 이런 상Jω축 해석을 Nyquist 판별 혹은 근궤적(root locus), Bode 선도 등을 통해 더 직관적으로 다룬다. 그러나 그 핵심에는 복소해석에서 말하는 상Jω축으로의 대입, 그리고 실수부·허수부의 동시 만족 조건이 자리한다.

추가로, 상태방정식으로 기술된 시스템에서 행렬 고유값이 상Jω축에 위치한다는 것은, 연립 1차 미분방정식의 해가 사인·코사인 형태의 영구진동을 일으키는 해로 나타날 수 있음을 의미한다. 예를 들어 상태벡터 \(\mathbf{x}\)가 시간에 따라 어떻게 변화하는지 분석할 때, 해의 지수부가 음수인지, 0인지, 양수인지를 판별하는 것이 곧 안정도 해석의 본질이라고 할 수 있다. 이를 명확히 분석하기 위해 상Jω축 해석과 안정도 경계 조건을 정확히 이해하는 것이 중요하다.
