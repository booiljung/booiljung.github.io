복소함수를 다룰 때는 일반적으로 \(z = a + jb\) 형태로 표현한다. 여기서 \(a\)와 \(b\)는 실수이며, \(j\)는 허수 단위다. 이 복소수 \(z\)에 대해 정의된 복소함수 \(f(z)\)가 있을 때, 함수를 실수부와 허수부로 나누어 \(f(z) = u(a, b) + jv(a, b)\)라고 쓴다. 이때 \(u(a, b)\)와 \(v(a, b)\)는 각각 \(\mathbb{R}^2 \to \mathbb{R}\)인 실함수다.

복소 해석학에서 \(f(z)\)가 복소 미분 가능하다는 것은 실수 2변수 함수로 볼 수 있는 \(u(a,b)\)와 \(v(a,b)\)의 특정 조건(Cauchy-Riemann 방정식)을 만족해야 한다는 의미다.

복소함수의 실수부와 허수부¶

복소함수 \(f(z)\)를 \(f(z) = u(a, b) + jv(a, b)\)로 표현한다. 여기서 \(a = \Re(z)\), \(b = \Im(z)\)다. 복소 미분 가능성을 판별하려면 \(u\)와 \(v\)가 어떤 연속성과 편미분 성질을 만족해야 하는지 살펴야 한다.

\(a\)와 \(b\)는 서로 독립적인 실수 변수이므로, \(u(a,b)\)와 \(v(a,b)\)에 대해 \(\partial/\partial a\)와 \(\partial/\partial b\)를 모두 생각할 수 있다. 복소 미분 가능성이란, 복소수 증분 \(\Delta z\)를 다양한 방향으로 0에 접근시켰을 때 미분의 극한값이 일관되게 존재하는가를 보는 것이다.

복소 미분의 정의¶

실수 1변수 함수 \(g(x)\)에서의 미분은 \(g'(x) = \lim_{\Delta x \to 0}\dfrac{g(x + \Delta x) - g(x)}{\Delta x}\)로 정의된다. 복소함수 \(f(z)\)의 미분은 이와 비슷하게

\[ \lim_{\Delta z \to 0} \dfrac{f(z + \Delta z) - f(z)}{\Delta z} \]

의 형태로 정의된다. 복소평면에서 \(\Delta z\)를 모든 방향으로 0에 접근시켜도 이 극한값이 동일해야만 복소 미분 가능하다고 말한다. 만약 접근 경로마다 극한값이 달라진다면 복소 미분은 존재하지 않는다.

이 정의를 실수부와 허수부 관점으로 옮겨서 이해하는 데에 핵심적인 역할을 하는 것이 Cauchy-Riemann 방정식이다.

Cauchy-Riemann 방정식의 유도¶

복소함수 \(f(z) = u(a,b) + jv(a,b)\)가 있다고 하자. 복소변수 \(z = a + jb\)에서의 증분을 \(\Delta z = \Delta a + j\Delta b\)라고 둘 때, 복소 미분 가능하려면

\[ \dfrac{f(a + \Delta a,\, b + \Delta b) - f(a, b)}{\Delta a + j\,\Delta b} \]

가 \(\Delta z\)가 어떤 방향으로 0에 접근하더라도 같은 값을 가져야 한다.

이 극한이 유일하기 위해, \(\Delta b = 0\)인 경우(즉 \(b\) 고정, \(a\)만 변화)와 \(\Delta a = 0\)인 경우(즉 \(a\) 고정, \(b\)만 변화)에서의 미분값이 일치해야 한다. 이를 통해 \(u(a,b)\)와 \(v(a,b)\)가 만족해야 하는 두 개의 편미분 방정식이 도출된다.

\(\Delta b = 0\)일 때,

\[ \dfrac{f(a + \Delta a,\, b) - f(a, b)}{\Delta a} \]

가 \(f'(z)\)가 되어야 한다. \(f(a + \Delta a, b) = u(a + \Delta a, b) + jv(a + \Delta a, b)\)이므로, 이 식은 \(\Delta a \to 0\) 극한에서

\[ u_a(a, b) + j\,v_a(a, b) \]

가 된다. 여기서 \(u_a = \dfrac{\partial u}{\partial a}\), \(v_a = \dfrac{\partial v}{\partial a}\)다.

\(\Delta a = 0\)일 때,

\[ \dfrac{f(a,\, b + \Delta b) - f(a, b)}{j\,\Delta b} \]

가 동일한 \(f'(z)\)가 되어야 한다. 분모 \(j,\Delta b\)는 \(j\)로 나누는 것과 같으므로 \(\dfrac{1}{j} = -j\)를 이용하면, 이 식은 \(\Delta b \to 0\) 극한에서

\[ -\,j\,u_b(a, b) + v_b(a, b) \]

가 된다. 여기서 \(u_b = \dfrac{\partial u}{\partial b}\), \(v_b = \dfrac{\partial v}{\partial b}\)다.

따라서

\[ u_a(a, b) + j\,v_a(a, b) = -\,j\,u_b(a, b) + v_b(a, b) \]

가 되어야 하고, 이를 실수부와 허수부로 분리하면 Cauchy-Riemann 방정식이 완성된다.

\[ \boxed{ \begin{align} u_a(a, b) &= v_b(a, b), \\ u_b(a, b) &= -\,v_a(a, b). \end{align} } \]

이 두 조건이 동시에 만족되어야 비로소 \(f(z)\)가 복소 미분 가능하다.

Cauchy-Riemann 방정식 예시¶

간단한 예로 \(f(z) = z^2\)를 들 수 있다. 복소수 \(z = a + j,b\)에 대해

\[ z^2 = (a + j\,b)^2 = a^2 + 2\,a\,b\,j + (j\,b)^2 = a^2 - b^2 + j\,(2\,a\,b) \]

이므로, 실수부는 \(u(a, b) = a^2 - b^2\), 허수부는 \(v(a, b) = 2,a,b\)다. 부분 미분을 계산하면

\[ \begin{align} u_a(a, b) &= \dfrac{\partial}{\partial a}\,(a^2 - b^2) = 2\,a, \\ u_b(a, b) &= \dfrac{\partial}{\partial b}\,(a^2 - b^2) = -2\,b, \\ v_a(a, b) &= \dfrac{\partial}{\partial a}\,(2\,a\,b) = 2\,b, \\ v_b(a, b) &= \dfrac{\partial}{\partial b}\,(2\,a\,b) = 2\,a. \end{align} \]

Cauchy-Riemann 방정식 \(u_a = v_b\)와 \(u_b = -v_a\)에 대입하면 \(2a = 2a\)와 \(-2b = -2b\)가 되어 모든 \((a, b)\)에서 성립한다. 따라서 \(f(z) = z^2\)는 복소평면 전체에서 복소 미분 가능함을 알 수 있다.

아래 Python 코드는 \(f(z)=z^2\)에서 \(u\)와 \(v\) 및 그 편미분을 확인하는 예시다.

import sympy

a, b = sympy.symbols('a b', real=True)
j = sympy.Symbol('j', imaginary=True)  # 여기서는 허수 기호를 j로 표기
z = a + j*b
f = z**2

# 실수부, 허수부 분리
u = sympy.simplify(sympy.re(f))
v = sympy.simplify(sympy.im(f))

# 부분 미분 계산
u_a = u.diff(a)
u_b = u.diff(b)
v_a = v.diff(a)
v_b = v.diff(b)

print("u(a,b) =", u)
print("v(a,b) =", v)
print("u_a =", u_a, ", u_b =", u_b)
print("v_a =", v_a, ", v_b =", v_b)

이 코드를 실행하면 직접 계산한 결과와 동일한 식을 얻을 수 있다.

반면 \(f(z) = \overline{z}\)인 경우에는 \(z = a + jb\)일 때 \(\overline{z} = a - jb\)가 되므로 \(u(a,b) = a\), \(v(a,b) = -b\)다. 편미분을 구해보면 \(u_a = 1\), \(u_b = 0\), \(v_a = 0\), \(v_b = -1\)가 되어 \(u_a = v_b\)를 만족하지 않는다. 따라서 이 함수는 복소 미분이 존재하지 않는다.

Cauchy-Riemann 방정식은 복소함수가 복소 해석학적 성질을 가지는지 판별하기 위해 반드시 살펴야 하는 핵심 조건이다. 이를 만족하면 복소 미분이 가능해지고, 해석학적으로 풍부한 성질(예를 들어, 해석적 연속, 적분 정리 등)을 응용할 수 있다. Cauchy-Riemann 방정식을 적용하면 복소함수가 어디에서 복소 미분 가능하고, 어디에서 불가능한지를 빠르고 정확하게 알 수 있다.
