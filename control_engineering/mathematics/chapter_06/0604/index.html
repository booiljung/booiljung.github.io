복소수는 \(a + j b\) 형태로 실수부 \(a\)와 허수부 \(b\)를 갖는다. 이를 시각화하기 위해 복소평면(Argand Diagram)을 사용한다. 복소평면에서는 가로축을 실수축, 세로축을 허수축으로 놓는다. 이렇게 복소수를 평면 위의 점이나 벡터로 나타내면, 여러 연산과 성질을 직관적으로 이해할 수 있다.

복소평면의 도입은 대수적 처리에 머무르던 복소수를 기하학적으로도 탐구할 수 있게 한다. 복소수의 모듈러스와 위상각을 벡터의 길이와 각도에 대응시켜 벡터로 해석할 수 있고, 복소수 덧셈과 곱셈의 효과를 평면상의 이동이나 회전, 크기 변환으로 해석할 수도 있다.

복소평면에서 복소수 \(z = a + j b\)를 표현할 때, 점 \((a, b)\)와 벡터 \(\mathbf{v}\)를 동일시한다. 따라서 복소수의 대수적 연산은 평면상에서의 기하학적 조작과 대응한다. 이 과정을 구체적으로 살펴보면 다음과 같은 내용을 이해할 수 있다.

복소평면의 정의¶

복소평면은 가로축을 실수축으로, 세로축을 허수축으로 설정해 복소수를 2차원 점으로 나타내는 평면이다. 복소수 \(z = a + j b\)에서 실수부 \(a\)는 실수축 좌표, 허수부 \(b\)는 허수축 좌표가 된다. 이 평면을 Argand Diagram이라고 부르기도 한다.

복소평면에서는 모든 복소수가 평면 위의 한 점과 일대일 대응을 갖는다. 예를 들어, \(z = 3 + j 2\)라면 평면에서 좌표 \((3, 2)\)에 해당한다. 실수부 \(3\)는 실수축에서의 위치, 허수부 \(2\)는 허수축에서의 위치가 되어, 결과적으로 점 \((3, 2)\)로 나타낼 수 있다.

복소평면을 이용하면 복소수의 다양한 성질을 시각적으로 파악하기 쉬워진다. 예를 들어, 모듈러스(복소수의 크기)는 평면에서 원점으로부터 해당 점까지의 거리로 나타난다. 그리고 복소수의 위상각(Argument)은 실수축과 복소수를 나타내는 벡터가 이루는 각도 \(\theta\)로 표현된다. 이러한 방식으로 복소수에 대한 이해가 한층 명확해진다.

복소수의 기하학적 해석¶

복소수 \(z = a + j b\)는 평면상에서 벡터로 해석할 수 있다. 벡터 \(\mathbf{z}\)를 \((a, b)\)로 나타내면, 벡터의 길이는 복소수의 모듈러스에 해당하고, 벡터가 실수축과 이루는 각도는 복소수의 위상각에 해당한다.

길이, 즉 모듈러스 \(|z|\)는 다음과 같이 정의된다.

\[ \begin{align} |z| &= \sqrt{a^2 + b^2} \end{align} \]

이를 복소수의 내적 개념이나 피타고라스 정리를 통해 쉽게 이해할 수 있다. 중간 과정을 전개하면 다음과 같은 식으로 볼 수 있다.

\[ \begin{align} z &= a + j b \\ \overline{z} &= a - j b \\ z \, \overline{z} &= (a + j b)(a - j b) = a^2 + b^2 \\ |z|^2 &= a^2 + b^2 \\ |z| &= \sqrt{a^2 + b^2} \end{align} \]

실수축과 벡터 \(\mathbf{z}\)가 이루는 각도는 \(\arg(z)\) 또는 \(\theta\)로 표현하며, 일반적으로 다음과 같이 정의할 수 있다.

\[ \begin{align} \theta = \arg(z) = \tan^{-1}\left(\dfrac{b}{a}\right) \end{align} \]

각도 \(\theta\)는 복소수의 방향 정보를 나타내므로, 이를 이용해 복소수를 극좌표 형태로 표현할 수 있다.

복소수의 극좌표¶

복소수를 극좌표로 나타내면, 복소수의 기하학적 성질을 더욱 명확히 파악할 수 있다. 복소수의 극좌표 표현은 모듈러스와 위상각을 사용해 다음과 같이 표현한다.

\[ \begin{align} z &= |z|\bigl(\cos \theta + j \sin \theta\bigr) \end{align} \]

여기서 \(|z| = r = \sqrt{a^2 + b^2}\), \(\theta = \arg(z) = \tan^{-1}\left(\dfrac{b}{a}\right)\)라고 할 수 있다. 이를 종합하면,

\[ \begin{align} z &= a + j b \\ &= r \bigl(\cos \theta + j \sin \theta\bigr). \end{align} \]

극좌표로 표현하면, 곱셈과 나눗셈 같은 복소수 연산이 각도와 길이의 연산으로 간단해진다. 예를 들어, 복소수 \(z_1\)과 \(z_2\)가 각각 \(|z_1|(\cos \theta_1 + j\sin \theta_1)\), \(|z_2|(\cos \theta_2 + j\sin \theta_2)\)로 주어졌을 때, 곱셈은 모듈러스를 곱하고 위상각을 더하는 형태로 이해할 수 있다.

복소평면에서의 덧셈과 곱셈¶

복소수 덧셈은 평면 벡터의 덧셈과 동일하게 해석할 수 있다. 두 복소수 \(z_1 = a_1 + j b_1\)과 \(z_2 = a_2 + j b_2\)의 합 \(z_1 + z_2\)는 단순히 실수부끼리, 허수부끼리 더하는 것이므로 다음과 같은 형태가 된다.

\[ \begin{align} z_1 + z_2 &= (a_1 + j b_1) + (a_2 + j b_2) \\ &= (a_1 + a_2) + j(b_1 + b_2). \end{align} \]

이를 평면상 벡터 \((a_1, b_1)\)와 \((a_2, b_2)\)를 머리끝 이어 붙이기(parallelogram rule)로 더하는 것과 동일하게 볼 수 있다. 따라서 복소수의 덧셈은 평면에서 벡터를 이동시키는 기하학적 연산이다.

복소수 곱셈은 모듈러스가 곱해지고 위상각이 더해지는 연산이다. 복소수를 직교좌표계로 표기했을 때에는 직접 전개가 번거로울 수 있지만, 극좌표로 표기하면 아주 간단해진다. 예를 들어, \(z_1 = r_1(\cos \theta_1 + j \sin \theta_1)\), \(z_2 = r_2(\cos \theta_2 + j \sin \theta_2)\)라면

\[ \begin{align} z_1 z_2 &= r_1 r_2 \bigl[\cos (\theta_1 + \theta_2) + j \sin (\theta_1 + \theta_2)\bigr]. \end{align} \]

이는 기하학적으로 "길이 \(r_1\)인 벡터를 길이 \(r_2\)만큼 늘리고, 회전각 \(\theta_1\)에 \(\theta_2\)를 더해 최종각도를 결정한다"는 의미로 해석할 수 있다. 즉 복소수 곱셈은 평면에서의 스케일링과 회전을 결합한 변환으로 볼 수 있다.

예시 코드(Python)를 통한 시각화¶

복소수와 복소평면을 직접 시각화하면 이해가 더 쉬워진다. 예를 들어, Python의 matplotlib를 이용해 복소수를 점으로 표시할 수 있다. 아래는 간단한 예시 코드다.

import matplotlib.pyplot as plt

# 예시 복소수들
z1 = complex(2, 1)   # 2 + j1
z2 = complex(-1, 2)  # -1 + j2

# 실수부와 허수부 분리
x_vals = [z1.real, z2.real]
y_vals = [z1.imag, z2.imag]

plt.axhline(0, color='black', linewidth=1)
plt.axvline(0, color='black', linewidth=1)

plt.scatter(x_vals, y_vals, color='red')
plt.text(z1.real, z1.imag, 'z1')
plt.text(z2.real, z2.imag, 'z2')

plt.xlabel('Real axis')
plt.ylabel('Imag axis')
plt.title('Argand Diagram Example')
plt.grid(True)
plt.axis('equal')
plt.show()

이 코드는 \(z_1 = 2 + j 1\), \(z_2 = -1 + j 2\)를 복소평면에 점으로 표시한다. 복소수의 실수부를 x좌표, 허수부를 y좌표로 하여 직교좌표계에 평면 그래프를 그린다. 이렇게 코드를 통해 직접 확인하면 복소수가 평면의 점으로 자연스럽게 대응함을 확인할 수 있다.

기하학적 해석의 의의¶

복소평면에서 복소수를 기하학적으로 해석하면 다양한 수학적·공학적 문제를 직관적으로 다룰 수 있다. 예를 들어, 제어계 설계에서 전달함수의 근을 복소평면에 놓고 고유진동, 감쇠 비율 등을 해석하면, 시스템의 안정성과 응답 특성이 시각적으로 드러난다. 비선형 좌표 변환이 필요한 복잡한 문제도 복소평면을 통해 회전이나 스케일링으로 단순화할 수 있다.

복소평면이라는 2차원 공간을 통하여, 복소수가 그저 대수적인 ‘허수’가 아니라 실수축에 하나의 축이 더해진 확장된 수체임을 체감하게 된다. 덧셈과 곱셈이 평면 상의 벡터 이동과 회전·스케일링으로 해석된다는 점은 복소해석의 가장 큰 매력 중 하나다. 이런 기하학적 해석은 추상적으로만 느껴지던 복소수의 연산과 성질을 직관적인 그림으로 이해하게 만들며, 제어공학을 비롯한 공학적 응용에서 필수적인 도구로 활용된다.
