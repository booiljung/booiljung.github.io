복소해석에서 해석함수와 정칙함수는 매우 핵심적인 개념이다. 복소수 \(z = a + j b\)에서 \(a\)는 실수부, \(b\)는 허수부를 의미하고, \(j\)는 허수 단위이다. 해석함수와 정칙함수는 복소함수가 복소수 영역에서 어떻게 미분 가능하고, 어떤 성질을 만족하는지에 대한 깊은 이해를 제공한다.

아래에서는 해석함수와 정칙함수의 정의, 성질, 그리고 두 개념이 어떻게 이어지는지를 살펴본다.

해석함수의 개념¶

실변수 함수에서 미분 가능성을 논하듯, 복소함수 \(f(z)\)에 대해서도 복소수의 영역에서 미분 가능성을 정의한다. 복소수 \(z = a + j b\)에 대해 \(f(z) = u(a, b) + jv(a, b)\)라고 표현하면, \(u(a, b)\)와 \(v(a, b)\)는 각각 실수값 함수이다.

복소함수가 \(z_0\) 근방에서 거듭제곱급수(파워 시리즈)로 전개될 수 있다면, 그 함수를 해석함수라고 한다. 즉 어떤 점 \(z_0\) 주변의 작은 영역에서

\[ \begin{align} f(z) = \sum_{n=0}^{\infty} c_n (z - z_0)^n \end{align} \]

형식으로 나타낼 수 있으면 해석함수라고 부른다. 이 거듭제곱급수가 수렴하는 영역 안에서는 복소함수가 무한 번 미분 가능함을 보장한다.

해석함수의 핵심은 국소적으로 거듭제곱급수로 표현된다는 데에 있다. 실수해석에서 테일러 전개를 통해 무한 번 미분 가능 함수가 특정 점에서의 정보를 모두 포함하듯이, 복소해석에서는 해석함수가 그 점 근방 전체 영역에 걸쳐 매우 강력한 성질들을 갖게 된다.

정칙함수의 개념¶

복소함수가 어떤 영역에서 복소 미분 가능하면 그 함수를 정칙함수라고 한다. 미분 가능하다는 것은

\[ \begin{align} \lim_{\Delta z \to 0} \dfrac{f(z + \Delta z) - f(z)}{\Delta z} \end{align} \]

이 존재한다는 의미이다. 실수해석에서 정의되는 도함수와 달리 복소수에서는 방향에 관계없이 동일한 미분계수가 존재해야 한다. 따라서 복소함수가 정칙함수가 되려면 더욱 엄격한 조건을 만족해야 한다.

특정 영역에서 모든 점에 대해 복소 미분 가능하면, 그 영역에서 해석함수와 정칙함수는 사실상 같은 의미로 쓰인다. 다만 “해석적”이라는 표현은 거듭제곱급수로 표현 가능함을 강조하고, “정칙”이라는 표현은 복소 미분 가능성을 강조한다. 일반적인 문헌에서는 두 용어를 거의 구분 없이 사용한다.

해석함수와 정칙함수의 관계¶

해석함수는 거듭제곱급수 전개가 가능하다는 점에서 이론적으로 매우 강력한 확장성을 갖는다. 한편 정칙함수는 해당 영역에서의 복소 미분 가능성을 강조한다. 복소 해석에서 중요한 정리는 다음과 같은 사실을 말해 준다. 어떤 영역이 단일 연결 영역일 경우(즉 경로를 통해 해당 영역에서 모든 점을 서로 연결할 수 있는 경우), 그 영역에서 복소 미분 가능한 함수는 그 영역에서 자동으로 거듭제곱급수 전개가 가능하다. 결국 그 영역에서는 “정칙함수”와 “해석함수”가 동일한 개념이 된다.

복소함수가 해석적이거나 정칙이라는 것은 결과적으로 같은 성질을 반영한다. 이러한 복소함수는 무한 번 미분 가능하고, 실수해석에서 볼 수 없는 독특하고 풍부한 구조를 갖는다. 예를 들어, 실수해석에서는 무한 번 미분 가능해도 여러 형태로 함수가 구성될 수 있지만, 복소해석에서는 해석함수의 형태가 훨씬 엄격하게 제약된다.

Cauchy-Riemann 방정식¶

복소함수가 정칙함수가 되기 위한 필요충분조건 중 하나는 Cauchy-Riemann 방정식을 만족하는 것이다. 복소함수 \(f(z) = u(a, b) + j , v(a, b)\)에 대하여, 각 변수 \(a, b\)에 대한 편도함수를 생각할 수 있다. 복소수 \(z = a + j b\)에서 \(u(a, b), v(a, b)\)가 충분히 매끄러운(연속 미분 가능) 함수라면, 다음의 Cauchy-Riemann 방정식을 만족해야 한다.

\[ \begin{align} \dfrac{\partial u}{\partial a} = \dfrac{\partial v}{\partial b}, \quad \dfrac{\partial u}{\partial b} = -\dfrac{\partial v}{\partial a} \end{align} \]

이 조건이 성립해야만 함수를 복소수에 대해 미분해도, 그 값이 방향에 의존하지 않게 된다. 이는 복소 미분의 정의가 모든 방향에서 동일해야 한다는 요구 사항과 대응한다.

Cauchy-Riemann 방정식의 또 다른 표현을 위해, 편의상 \(u_a, u_b, v_a, v_b\)로 각각 부분 편미분을 쓰면

\[ \begin{align} u_a = v_b, \quad u_b = -v_a \end{align} \]

이 된다. 이 두 방정식이 모두 만족되고, \(u\)와 \(v\)가 연속적으로 미분 가능하면, \(f(z) = u + j v\)는 정의된 영역에서 정칙함수가 된다. 동시에, 그 영역에서 거듭제곱급수로 전개 가능하므로 해석함수가 된다.

예시: \(f(z) = z^2\)의 해석성과 정칙성¶

간단한 예로 \(f(z) = z^2\)를 살펴보면, \(z = a + j b\)라고 할 때

\[ \begin{align} z^2 = (a + j b)^2 = a^2 - b^2 + j(2ab). \end{align} \]

여기서 실수부와 허수부를 구분하면

\[ \begin{align} u(a, b) = a^2 - b^2, \quad v(a, b) = 2ab. \end{align} \]

이제 편미분을 계산해 보면

\[ \begin{align} u_a = 2a, \quad u_b = -2b, \quad v_a = 2b, \quad v_b = 2a. \end{align} \]

Cauchy-Riemann 방정식 \(u_a = v_b,, u_b = -v_a\)를 대입해 보면

\[ \begin{align} u_a = 2a, \quad v_b = 2a \quad \Rightarrow \quad u_a = v_b \\ u_b = -2b, \quad -v_a = -2b \quad \Rightarrow \quad u_b = -v_a \end{align} \]

이 방정식을 만족한다. 따라서 \(f(z) = z^2\)는 모든 영역에서 복소 미분 가능하므로 정칙함수이자 해석함수다.

간단한 Python 예시¶

Cauchy-Riemann 방정식을 직접 확인하는 과정을 Python의 기호 연산(sympy 등)을 활용해 살펴볼 수 있다. 간단한 예로 \(f(z) = z^2\)에 대해서 다음과 같이 확인할 수 있다. 이는 실제 계산과정을 반복적으로 검증할 때 유용하다.

# Python 예시 (Sympy 사용)
import sympy as sp

a, b = sp.symbols('a b', real=True)
j = sp.I  # Python에서 허수 단위는 I로 표현됨

# f(z) = z^2
z = a + j*b
f = z**2

# 분리된 실수부와 허수부
u = sp.expand(sp.re(f))
v = sp.expand(sp.im(f))

# 편미분 계산
u_a = sp.diff(u, a)
u_b = sp.diff(u, b)
v_a = sp.diff(v, a)
v_b = sp.diff(v, b)

print("u(a, b) =", u)
print("v(a, b) =", v)
print("u_a =", u_a, ", v_b =", v_b)
print("u_b =", u_b, ", v_a =", v_a)

위 코드를 실행하면, \(u_a = v_b\), \(u_b = -v_a\) 형태가 성립함을 간단히 확인할 수 있다.

결론¶

해석함수는 복소함수를 특정 영역에서 거듭제곱급수로 전개할 수 있는 함수를 말하고, 정칙함수는 특정 영역에서 복소 미분이 가능한 함수를 말한다. 영역이 단순 연결이라는 조건 하에서는 두 개념이 사실상 같은 의미가 된다. 복소함수가 해석적이거나 정칙이라는 것은 그 함수가 매우 엄격한 구조적 제한을 받음을 의미한다.

Cauchy-Riemann 방정식은 정칙함수를 판별하는 중요한 도구이며, 이 방정식을 만족함과 동시에 연속 편미분 가능 조건이 충족되면 해석함수이자 정칙함수가 된다. 이러한 특성은 복소함수의 극적인 성질을 가능케 하며, 해석학적 기법을 제어공학을 포함한 다양한 분야에 적용할 수 있게 해 준다.
