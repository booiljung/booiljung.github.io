복소수는 실수축과 허수축을 동시에 다루는 수 체계다. 복소수를 다룰 때는 실수부와 허수부의 결합이 어떻게 이루어지는지 꼼꼼히 살펴야 한다. 여기서는 복소수를 \(a + j b\)의 형태로 표기하고, \(j^2 = -1\)임을 전제로 논의를 진행한다.

복소수 \(a + j b\)에서 \(a\)는 실수부, \(b\)는 허수부다. 이때 연산은 실수부와 허수부가 각각 어떻게 결합되는지, 그리고 \(j^2 = -1\)이 어떤 역할을 하는지 이해하는 것이 중요하다. 복소평면에서 복소수를 바라보면 기하학적으로 벡터와 유사하게 해석할 수도 있지만, 여기서는 대수적 전개 과정을 통해 각각의 연산을 살펴본다.

복소수의 덧셈¶

복소수 덧셈은 실수부끼리 더하고, 허수부끼리 더하는 방식으로 정의된다. 두 복소수 \(a + j b\)와 \(c + j d\)를 더했을 때, 결과를 \(z_\mathrm{sum}\)이라 하면 다음과 같이 표현한다.

\[ \begin{align} z_\mathrm{sum} &= (a + j b) + (c + j d) \end{align} \]

동류항을 묶어서 정리하면

\[ \begin{align} z_\mathrm{sum} &= (a + c) + j(b + d) \end{align} \]

즉, 실수부는 \(a + c\), 허수부는 \(b + d\)가 된다. 이는 복소평면에서 점 \((a, b)\)와 \((c, d)\)를 각각의 좌표축 방향으로 단순 병렬 이동하는 것과 동일한 기하학적 해석을 가진다.

복소수의 곱셈¶

복소수 곱셈은 분배법칙을 적용해 항을 전개하고, \(j^2 = -1\)을 적절히 사용해야 한다. 두 복소수 \(a + j b\)와 \(c + j d\)를 곱하여 \(z_\mathrm{mul}\)이라 하자.

\[ \begin{align} z_\mathrm{mul} &= (a + j b)(c + j d) \end{align} \]

분배법칙을 적용해 전개하면

\[ \begin{align} z_\mathrm{mul} &= a(c + j d) + j b(c + j d) \\ &= a c + a (j d) + j b c + (j b)(j d), \end{align} \]

여기서 \(j^2 = -1\)을 대입한다.

\[ \begin{align} z_\mathrm{mul} &= a c + j a d + j b c + j^2 b d \\ &= a c + j a d + j b c - b d \quad (\text{왜냐하면 } j^2 = -1) \\ &= (a c - b d) + j (a d + b c). \end{align} \]

결국 실수부는 \(a c - b d\), 허수부는 \(a d + b c\)가 된다. 기하학적으로는 복소평면에서 각도와 크기가 결합되는 해석이 가능하지만, 대수적 전개 과정을 통해서도 간단히 확인할 수 있다.

복소수의 나눗셈¶

복소수 나눗셈은 분모에 허수부가 있으면 직접 나누기 불편하므로, 분모가 되는 복소수의 켤레를 곱하여 분모를 실수로 만드는 과정을 거친다. 복소수 \(a + j b\)를 \(c + j d\)로 나눈 결과를 \(z_\mathrm{div}\)라고 하자.

\[ \begin{align} z_\mathrm{div} &= \dfrac{a + j b}{c + j d} \end{align} \]

분모를 실수화하기 위해 분자와 분모에 켤레 복소수 \(c - j d\)를 곱한다.

\[ \begin{align} z_\mathrm{div} &= \dfrac{a + j b}{c + j d} \times \dfrac{c - j d}{c - j d} \\ &= \dfrac{(a + j b)(c - j d)}{(c + j d)(c - j d)}. \end{align} \]

분자를 먼저 전개한다.

\[ \begin{align} (a + j b)(c - j d) &= a c - a (j d) + (j b) c - (j b)(j d) \\ &= a c - j a d + j b c - j^2 b d \\ &= a c - j a d + j b c + b d \quad (\text{왜냐하면 } j^2 = -1 \rightarrow -j^2 = +1) \\ &= (a c + b d) + j(b c - a d). \end{align} \]

분모는 다음과 같이 정리된다.

\[ \begin{align} (c + j d)(c - j d) &= c^2 - (j d)^2 \\ &= c^2 - j^2 d^2 \\ &= c^2 - (-1) d^2 \\ &= c^2 + d^2. \end{align} \]

그 결과,

\[ \begin{align} z_\mathrm{div} &= \dfrac{(a c + b d) + j(b c - a d)}{c^2 + d^2} \end{align} \]

이를 실수부와 허수부로 분리하면

\[ \begin{align} z_\mathrm{div} &= \dfrac{a c + b d}{c^2 + d^2} + j \dfrac{b c - a d}{c^2 + d^2}. \end{align} \]

결국 복소수 나눗셈은 분모 복소수를 켤레로 곱해 실수 분모를 만든 뒤, 실수부와 허수부를 각각 나누어 가지는 형태로 나타난다.

아래는 Python 코드를 사용해 복소수 연산을 직접 확인해 보는 간단한 예다. 코드를 실행하면 우리가 대수적으로 전개한 결과와 동일함을 알 수 있다.

# 복소수 정의
z1 = 2 + 3j  # 이는 2 + j3 을 의미
z2 = 1 - 4j  # 이는 1 - j4 를 의미

# 덧셈
add_result = z1 + z2

# 곱셈
mul_result = z1 * z2

# 나눗셈
div_result = z1 / z2

print("z1 + z2 =", add_result)
print("z1 * z2 =", mul_result)
print("z1 / z2 =", div_result)

복소수는 전기·전자, 제어공학, 신호처리, 물리 등 다양한 분야에서 필수적으로 사용된다. 덧셈, 곱셈, 나눗셈 연산 과정을 정확히 익혀 두면 더 복잡한 연산이나 응용을 다룰 때에도 큰 도움이 된다.
