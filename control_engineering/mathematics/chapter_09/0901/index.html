고차원 벡터공간은 기존의 2차원이나 3차원 벡터공간을 넘어, 4차원 이상으로 확장된 공간을 다룬다. 이 공간은 선형대수의 핵심 개념인 기저, 차원, 내적 등을 높은 차원으로 일반화한다. 고차원 벡터공간을 이해하면 차수에 상관없이 선형적 구조를 파악할 수 있고, 이는 제어·신호처리·최적화 같은 다양한 분야에서 중요한 역할을 한다.

벡터공간의 정의와 일반화¶

벡터공간은 스칼라 집합(일반적으로 실수 $ \mathbb{R} $ 또는 복소수 $ \mathbb{C} $)과 벡터들의 집합, 그리고 벡터에 대한 덧셈과 스칼라 곱 연산으로 구성된다. 2차원이나 3차원에서는 벡터가 좌표로 표현되어 직관적이지만, 차원이 커지면 좌표로 벡터를 모두 적는 일이 복잡해진다. 그럼에도 불구하고, 고차원에서도 벡터를 하나의 객체로 간주하고, 덧셈과 스칼라 곱 연산 규칙을 똑같이 적용한다는 점은 달라지지 않는다.

예를 들어 고차원 벡터 $ \mathbf{x} $를 다음과 같이 나타낼 수 있다. $$ \begin{align} \mathbf{x} = \begin{pmatrix} x_1 \ x_2 \ \vdots \ x_n \end{pmatrix}. \end{align} $$ 여기서 \(n\)이 큰 값일 뿐, 두 벡터를 더할 때나 스칼라를 곱할 때는 각각의 성분별 연산으로 정의된다. 실수 벡터공간의 경우 스칼라가 실수 \(a\)이고, 복소 벡터공간의 경우 스칼라가 복소수 \(a = a_\mathrm{real} + j , a_\mathrm{imag}\)가 된다. 예를 들어 복소 벡터공간에서 두 벡터 $ \mathbf{x}, \mathbf{y} $와 복소 스칼라 $ a $에 대해 $$ \begin{align} \mathbf{x} + \mathbf{y} &= \begin{pmatrix} x_1 + y_1 \ x_2 + y_2 \ \vdots \ x_n + y_n \end{pmatrix}, \[6pt] a \mathbf{x} &= \begin{pmatrix} (a_\mathrm{real} + j \, a_\mathrm{imag}) x_1 \ (a_\mathrm{real} + j \, a_\mathrm{imag}) x_2 \ \vdots \ (a_\mathrm{real} + j \, a_\mathrm{imag}) x_n \end{pmatrix}. \end{align} $$ 이와 같이 복소수라도 벡터공간의 공리(덧셈 및 곱셈 연산에 대한 교환법칙, 결합법칙, 항등원·역원의 존재 등)는 그대로 성립한다.

차원과 기저¶

벡터공간의 차원은 그 공간의 기저를 이루는 벡터의 개수로 정의된다. 기저(basis)는 그 공간에 속한 임의의 벡터를 선형 결합으로 표현할 수 있게 하며, 동시에 선형 독립성을 만족하는 벡터들의 집합이다. 예를 들어 $ \mathbb{R}^3 $에서 다음과 같은 벡터들은 표준기저가 된다. $$ \begin{align} \mathbf{e}_1 = \begin{pmatrix} 1 \ 0 \ 0 \end{pmatrix}, \quad \mathbf{e}_2 = \begin{pmatrix} 0 \ 1 \ 0 \end{pmatrix}, \quad \mathbf{e}_3 = \begin{pmatrix} 0 \ 0 \ 1 \end{pmatrix}. \end{align} $$ 어떤 벡터 $ \mathbf{x} \in \mathbb{R}^3 $도 $$ \begin{align} \mathbf{x} = x_1 \, \mathbf{e}_1 + x_2 \, \mathbf{e}_2 + x_3 \, \mathbf{e}_3 \end{align} $$ 형태로 표현할 수 있다. 이를 4차원이나 100차원으로 일반화해도 구조는 동일하다. 단지 벡터의 성분 수와 기저 벡터의 개수가 커질 뿐이다.

선형 독립성을 정의할 때는 기저 후보 벡터 $ \mathbf{v}_1, \mathbf{v}_2, \dots, \mathbf{v}_k $에 대해 $$ \begin{align} c_1 \, \mathbf{v}_1 + c_2 \, \mathbf{v}_2 + \cdots + c_k \, \mathbf{v}_k = \mathbf{0} \end{align} $$ 이 되는 유일한 해가 $$ \begin{align} c_1 = 0, \quad c_2 = 0, \quad \dots, \quad c_k = 0 \end{align} $$ 인 경우를 말한다. 이때 선형 독립성과 모든 벡터를 생성할 수 있는 스팬(span)을 고려하여, 해당 벡터들이 기저가 되기에 충분한지를 확인한다.

내적과 노름¶

고차원 벡터공간에서 거리나 각도를 정의하기 위해서는 내적(inner product)이라는 연산이 필요하다. 내적이 주어지면 벡터의 노름(norm)과 각도를 정의할 수 있으며, 이를 통해 직교(orthogonality) 개념도 도입할 수 있다.

실수 벡터공간 $ \mathbb{R}^n $에서 일반적인 유클리드 내적은 $$ \begin{align} \mathbf{x} \cdot \mathbf{y} = x_1 y_1 + x_2 y_2 + \cdots + x_n y_n \end{align} $$ 으로 정의된다. 복소 벡터공간 $ \mathbb{C}^n $에서는 켤레 복소수를 사용해 $$ \begin{align} \langle \mathbf{x}, \mathbf{y} \rangle = \overline{x_1} y_1 + \overline{x_2} y_2 + \cdots + \overline{x_n} y_n \end{align} $$ 형태가 된다. 예를 들어 $ x_i = a + j b $라면 $ \overline{x_i} = a - j b $가 된다.

내적을 사용하면 벡터의 노름은 $$ \begin{align} |\mathbf{x}| = \sqrt{\langle \mathbf{x}, \mathbf{x} \rangle} \end{align} $$ 으로 정의된다. 이 노름을 통해 고차원 공간에서도 벡터의 길이를 측정할 수 있으며, 내적이 존재한다면 서로 직교하는 벡터들로 기저를 구성할 수도 있다. 직교기저(orthonormal basis)는 계산과 해석이 간단해지는 장점이 있다.

복소 벡터공간에서의 고차원성¶

복소 벡터공간은 스칼라로 복소수 $ \mathbb{C} $를 사용하는 공간이다. 예를 들어 $ \mathbb{C}^n $은 복소수를 스칼라로 할 때 \(n\)차원 벡터공간이 된다. 그러나 이것을 실수 벡터공간으로 보면 \(2n\)차원이 된다. 복소 벡터공간을 다룰 때는 이러한 차원의 차이를 유의해야 한다.

복소 벡터공간에서 벡터 $ \mathbf{z} \in \mathbb{C}^n $와 복소 스칼라 $ a = a_\mathrm{real} + j , a_\mathrm{imag} $에 대하여 $$ \begin{align} a \mathbf{z} = \begin{pmatrix} (a_\mathrm{real} + j \, a_\mathrm{imag}) z_1 \ (a_\mathrm{real} + j \, a_\mathrm{imag}) z_2 \ \vdots \ (a_\mathrm{real} + j \, a_\mathrm{imag}) z_n \end{pmatrix}. \end{align} $$ 형태가 되며, 실수 벡터공간과 비슷한 규칙을 따른다. 다만 내적이나 직교성을 정의할 때는 켤레 복소수 연산이 필수적으로 포함된다는 점이 가장 큰 차이다.

복소 벡터공간에서도 기저, 차원, 내적, 노름 등의 개념은 실수 벡터공간과 같은 방식으로 적용할 수 있다. 스칼라가 복소수인 만큼 연산 과정에서 허수 단위 $ j $와 켤레 연산이 들어간다는 점만 주의하면 된다.

고차원 벡터들의 예시와 선형 독립성 검증 (Python)¶

아래 예시는 4차원 벡터 셋의 선형독립성을 Python으로 간단히 확인해 보는 코드다. 벡터들을 열로 쌓아 만든 행렬의 랭크를 구해, 벡터들이 선형독립인지 판별한다.

import numpy as np

# 4차원 벡터 3개 예시
v1 = np.array([1, 2, 3, 4], dtype=float)
v2 = np.array([2, 5, 7, 3], dtype=float)
v3 = np.array([1, 1, 1, 1], dtype=float)

# 이 벡터들을 열로 가지는 행렬 생성
M = np.column_stack((v1, v2, v3))

# 랭크 확인
rank_M = np.linalg.matrix_rank(M)

print("행렬의 랭크:", rank_M)

이때 $ \mathbf{M} $의 랭크가 3이면, 그 3개의 벡터는 4차원 공간에서 선형독립이다. 4차원 공간의 기저가 되려면 4개의 선형독립 벡터가 필요하므로, 여기서는 선형독립성만 확인한 셈이다. 복소 벡터공간을 다룰 때도 NumPy 같은 라이브러리를 이용하면, 벡터나 행렬을 복소수로 정의하고 동일한 방식으로 랭크 등을 계산할 수 있다.

고차원 벡터공간은 나아가 텐서(Tensor) 개념의 기반이 된다. 텐서는 벡터나 행렬을 넘어서는 고차원 배열로 볼 수 있는데, 차원과 기저, 내적 등을 이해하는 것이 텐서 해석에 매우 중요한 기초가 된다. 이러한 개념적 토대를 단단히 이해하면, 보다 복잡한 고차원 구조의 문제도 유연하게 접근할 수 있다.
