머신러닝과 딥러닝에서 텐서는 다차원 데이터를 효율적으로 표현하고 연산하기 위한 핵심 도구로 자리 잡았다. 텐서의 차수가 증가할수록 데이터와 모델이 복잡해지지만, 이를 효과적으로 처리할 수 있는 텐서 연산 기술이 발달함에 따라 다양한 심층 신경망 구조가 가능해졌다.

고차원 텐서는 대규모 데이터의 구조적 특징을 보존하고, 다양한 차원 간의 연산과 변환을 단순화한다. 예를 들어 이미지 데이터를 다룰 때는 합성곱 신경망(Convolutional Neural Network, CNN)에서 (배치, 채널, 높이, 너비)처럼 4차원 텐서를 사용하며, 시계열이나 자연어 처리에서 순환 신경망(Recurrent Neural Network, RNN)은 (배치, 시간, 특성) 같은 3차원 텐서를 사용한다.

이 장에서는 머신러닝과 딥러닝 모델에서 텐서가 어떻게 활용되는지 살펴보고, 모델 학습 과정에서 수행되는 텐서 기반의 연산 구조와 역전파 방법을 다룬다.

텐서 기반 머신러닝 모델의 전반적 이해¶

머신러닝 알고리즘 전반에서 데이터는 일반적으로 벡터나 행렬 형태로 표현되지만, 최근에는 텍스트, 영상, 음성 등 복잡한 데이터가 증가함에 따라 자연스레 차원이 높아진 텐서가 필요해졌다.

예를 들어 이미지는 2차원 픽셀 격자 구조를 가지며, 컬러 이미지는 채널이 추가되어 3차원 텐서가 된다. 여러 장의 이미지를 한 번에 처리하기 위해 배치(batch) 차원을 도입하면 4차원 텐서가 된다.

텐서를 사용하면 이런 고차원 데이터를 직관적으로 다룰 수 있고, 다양한 차원에서의 통계나 연산을 쉽게 확장할 수 있다. 머신러닝 모델을 학습시킬 때 데이터와 가중치 파라미터 역시 텐서로 구성되며, 연산 과정에서 텐서가 서로 곱해지거나 합성곱, 텐서 접힘(folding) 같은 다양한 형태로 변형된다.

텐서를 활용한 데이터 표현과 전처리¶

다차원 텐서를 다룰 때 가장 중요한 것은 각 차원이 무엇을 의미하는지 명확히 정의하고 적절히 정렬(reshape)하거나 전치(transpose)해준다는 점이다.

이미지 데이터를 예로 들면, (배치 크기, 채널 수, 높이, 너비) 구조와 (배치 크기, 높이, 너비, 채널 수) 구조 중 어떤 것을 사용할지 미리 설정해야 모델 구성과 연산 수행이 원활해진다.

자연어 처리를 위해 단어 임베딩(embedding)을 구성할 때는 (배치 크기, 문장 내 단어 개수, 임베딩 차원)으로 텐서를 구성하는 것이 일반적이다. 이렇게 데이터 구성을 텐서로 확립해 두면 전처리 과정에서 불필요한 순환문을 최소화하고 텐서 연산 한두 번으로 대규모 변환을 수행할 수 있다.

고차원 선형 변환과 신경망¶

신경망은 기본적으로 선형 변환과 비선형 활성 함수를 반복적으로 적용하여 입력 데이터를 차원 축소 또는 확장한다. 고차원 선형 변환은 텐서 차원을 유지하며 가중치 파라미터 역시 텐서 형태로 존재한다.

간단한 완전연결 신경망(fully-connected layer)을 벡터로 표시하면 입력 벡터 \(\mathbf{x}\)와 가중치 행렬 \(\mathbf{W}\)의 곱, 그리고 편향 벡터 \(\mathbf{b}\)의 덧셈으로 표현된다.

\[ \begin{align} \mathbf{z} &= \mathbf{W}\mathbf{x} + \mathbf{b} \end{align} \]

이때 \(\mathbf{x}\)는 \(n \times 1\) 형태의 벡터, \(\mathbf{W}\)는 \(m \times n\) 형태의 행렬, \(\mathbf{b}\)는 \(m \times 1\) 형태의 벡터다. 만약 \(\mathbf{x}\)가 2차원 이상의 텐서라면, \(\mathbf{W}\)와의 곱에서도 텐서 연산이 동일하게 확장된다.

예를 들어 \(\mathbf{x}\)가 3차원 텐서라면, 이를 2차원으로 reshape한 뒤 \(\mathbf{W}\)를 곱할 수도 있고, 차원을 늘려서 브로드캐스팅(broadcasting)을 이용할 수도 있다. 이런 텐서 연산 방식을 설계할 때는 차원의 순서를 명확히 이해하는 것이 중요하다.

역전파 알고리즘과 텐서 미분¶

신경망 학습에서는 역전파(backpropagation) 알고리즘을 통해 오차 함수를 가중치에 대해 미분한 값을 계산하고, 그 기울기를 이용하여 가중치를 갱신한다.

역전파에서 중요한 것은 다차원 미분을 올바르게 수행하는 것이며, 이를 텐서 계산 그래프에서 구현한다. 예를 들어 신경망의 출력 \(\mathbf{y}\)와 손실 함수 \(L(\mathbf{y}, \mathbf{t})\)가 있을 때, 가중치 \(\mathbf{W}\)에 대한 미분 \(\dfrac{\partial L}{\partial \mathbf{W}}\)는 텐서 형태가 될 수 있다. 역전파 과정은 사슬 법칙(chain rule)을 다차원으로 확장한 것으로, 레이어가 깊어지면 중간 계산이 여러 텐서 연산으로 구성된다. 각 레이어를 통과할 때마다 출력 텐서와 가중치 텐서에 대한 편미분을 구해 상위 레이어로 전달한다.

\[ \begin{align} \dfrac{\partial L}{\partial \mathbf{W}} &= \dfrac{\partial L}{\partial \mathbf{z}} \dfrac{\partial \mathbf{z}}{\partial \mathbf{W}} \\ &= \left( \dfrac{\partial L}{\partial \mathbf{z}} \right) \left( \dfrac{\partial (\mathbf{W}\mathbf{x} + \mathbf{b})}{\partial \mathbf{W}} \right) \end{align} \]

\(\mathbf{z}\)가 특정 레이어의 출력 텐서라면, \(\dfrac{\partial L}{\partial \mathbf{z}}\)도 같은 차원의 텐서가 되고, 그에 대응하여 가중치 텐서 \(\mathbf{W}\) 역시 다차원의 형태를 갖게 된다. 실제로 프레임워크 내부에서는 텐서 형태에 맞춰 연산이 자동화되어 계산된다.

합성곱 신경망(CNN) 구조에서의 텐서¶

합성곱 신경망은 이미지, 영상 같은 2차원 또는 3차원 구조 데이터 처리에 최적화된 방식이다. 텐서를 활용하면 합성곱 연산도 간단히 표현할 수 있다.

예를 들어 입력 텐서를 \(\mathbf{X}\), 합성곱 필터(커널) 텐서를 \(\mathbf{K}\)라 할 때, 출력 텐서 \(\mathbf{Z}\)는 보통 아래와 같은 형태로 표현된다.

\[ \begin{align} \mathbf{Z}_{(b, k, i, j)} &= \sum_{c} \sum_{p} \sum_{q} \mathbf{X}_{(b, c, i+p, j+q)} \, \mathbf{K}_{(k, c, p, q)} \end{align} \]

여기서

  • \(b\): 배치 차원
  • \(k\): 출력 채널 차원
  • \(c\): 입력 채널 차원
  • \(i\), \(j\): 출력 텐서의 위치
  • \(p\), \(q\): 필터의 공간적 크기

같은 식으로 차원을 정의한다.

학습 과정에서는 이 합성곱 연산의 편미분을 역전파하여 필터 텐서 \(\mathbf{K}\)와 입력 텐서 \(\mathbf{X}\)에 대한 기울기를 구한다. 이런 연산은 텐서 연산 라이브러리(NumPy, PyTorch, TensorFlow 등)를 활용하면 내부적으로 효율적으로 처리된다.

순환 신경망(RNN)과 고차원 텐서¶

순환 신경망은 시계열, 언어 모델링, 음성 처리 등을 위해 시간 축을 추가로 고려한다. 입력 시퀀스가 길어지면 (배치 크기, 시간, 특성)처럼 3차원 텐서로 데이터를 표현하게 된다.

단일 RNN 레이어를 식으로 나타내면, 현재 시점 입력 \(\mathbf{x}_t\)와 이전 시점 은닉 상태 \(\mathbf{h}_{t-1}\)을 받아 새로운 은닉 상태 \(\mathbf{h}_t\)를 계산한다.

\[ \begin{align} \mathbf{h}_t &= \tanh(\mathbf{W}_x \mathbf{x}_t + \mathbf{W}_h \mathbf{h}_{t-1} + \mathbf{b}) \end{align} \]

시간 차원이 추가됨에 따라 \(\mathbf{x}_t\)는 (배치 크기, 특성)의 2차원 텐서가 되고, 전체 시퀀스를 모두 포함하면 (배치 크기, 시간, 특성)의 3차원 텐서가 된다.

이 과정을 여러 시점에 걸쳐 벡터나 행렬이 아닌 텐서 연산으로 표현할 수 있으며, 역전파할 때 역시 시간 차원을 따라가며 BPTT(backpropagation through time)를 수행한다. 이때 은닉 상태 텐서가 매 시점마다 달라지므로, 텐서 연산 그래프를 적절히 구성해 다차원 역전파를 가능하게 한다.

텐서와 고차원 임베딩¶

자연어 처리나 추천 시스템에서 임베딩 레이어는 문장, 아이템 등을 밀집 벡터로 변환한다. 여러 문장을 동시에 처리하거나, 많은 개수의 아이템을 대상으로 임베딩을 구할 때는 (배치 크기, 시퀀스 길이, 임베딩 차원) 형태 같은 고차원 텐서를 사용한다.

이런 임베딩 텐서는 이후 다른 레이어와 연산하면서 차원이 더 늘어날 수 있다. 예를 들어 어텐션(attention) 메커니즘을 사용할 때는 (배치, 헤드 수, 시퀀스 길이, 임베딩 차원)처럼 4차원 텐서가 되기도 한다.

이를 통해 모델이 학습할 때 필요한 모든 데이터가 텐서 형태로 정리되며, 임베딩 차원이 증가하더라도 텐서 연산을 이용해 확장성을 유지한다.

간단한 신경망 텐서 연산 예시 (Python)¶

다차원 텐서를 직접 다뤄보고자 할 때, 다음 예시는 Python의 NumPy를 사용한 간단한 텐서 곱셈 예시다. 필수적인 부분만 담았다.

import numpy as np

# (배치 크기, 높이, 너비, 채널) 형태의 입력 텐서
X = np.random.randn(10, 28, 28, 3)

# (필터 개수, 필터 높이, 필터 너비, 채널) 형태의 필터 텐서
K = np.random.randn(5, 3, 3, 3)

# 합성곱 연산을 단순화하여 텐서 곱 예시만 확인
# 실제 합성곱은 슬라이딩 및 합 연산이 필요
# 여기서는 텐서의 축을 맞춰 단순 곱 연산을 해본다
Z = X * K[0]  # 첫 번째 필터만 임의로 브로드캐스팅 곱

print(Z.shape)  # (10, 28, 28, 3)

예시에서는 합성곱 연산을 온전히 재현하지는 않고, 텐서를 다루는 방식의 단편을 보여준다. 실제 합성곱이나 역전파 구현은 텐서 슬라이싱(slicing)과 채널 병합, 그리고 텐서 축 변환이 필요하다.

텐서는 이렇게 머신러닝과 딥러닝의 거의 모든 구조에서 필수적인 역할을 담당한다. 고차원의 복잡한 데이터도 텐서로 일관성 있게 다룰 수 있으며, 미분과 최적화 과정도 텐서 연산 그래프에서 자동으로 처리된다.
