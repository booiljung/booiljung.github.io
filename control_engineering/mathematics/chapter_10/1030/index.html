군행렬은 미분방정식을 한 바퀴 순회하는 닫힌 경로를 따라 해를 살펴볼 때, 그 해가 어떻게 변화하는지를 선형 연산자로 나타낸 것이다. 일반적으로 비선형 동적 시스템에서 시간이나 공간이 주기성을 가질 때, 한 주기를 돌고 난 후의 해의 변화량을 행렬 형태로 나타낸 군행렬을 모노드로미(matrix)라고 부른다. 여기서 “군”이라는 명칭은 이러한 행렬 연산이 군(group)의 성질, 즉 닫힘성과 역원, 결합 법칙 등을 충족한다는 것에서 유래한다.

가장 간단한 예로 \(t\)에 대해 주기 \(T\)를 갖는 선형 시변계

\[ \begin{align} \dot{\mathbf{x}}(t) = \mathbf{A}(t)\,\mathbf{x}(t) \end{align} \]

를 생각한다. 이 계의 해를 나타내는 근본해 행렬(fundamental matrix solution) \(\mathbf{\Phi}(t, 0)\)를 구하고, \(t = T\)에서의 값을 살펴보면

\[ \begin{align} \mathbf{\Phi}(T, 0) \end{align} \]

가 된다. 이것이 곧 군행렬(모노드로미 행렬)이다. 이를 \(t = nT\)에서 일반화하면

\[ \begin{align} \mathbf{\Phi}(nT, 0) = \bigl(\mathbf{\Phi}(T, 0)\bigr)^n \end{align} \]

이므로 주기 \(T\)마다 해가 어떤 방식으로 누적되는지 정량적으로 파악할 수 있다.

이러한 군행렬은 플로케(Floquet) 이론과 관련이 깊다. 플로케 이론에서 시간적으로 주기적인 계를 해석할 때, 모노드로미 행렬의 고유값을 통해 주기 운동이 안정적인지, 불안정한지를 평가한다. 예를 들어, 모노드로미 행렬의 고유값이 모두 단위원 안에 존재하면 주기 해 주위에 있는 해들이 수렴한다. 반대로 단위원 밖에 존재하는 고유값이 있다면, 주기 해로부터 점차 멀어진다.

좀 더 일반화된 미분기하학적 관점에서는, 다양체 상의 벡터장 \(\mathbf{f}\)가 주기적 궤적이나 닫힌 경로를 만들 때, 그 경로를 한 바퀴 돌아온 후의 변화량을 미분기하학적으로 표현하는 방법이 곧 군행렬(모노드로미)이다. 경로 상의 각 지점에서 접공간(tangent space)을 추적하여 한 주기가 지난 후 접공간이 처음과 어떻게 달라졌는지를 측정하는 선형 사상이 군행렬이며, 이 사상이 행렬 형태로 표현되는 이유는 접공간이 유한 차원을 갖는 선형공간이기 때문이다.

결과적으로 군행렬은 닫힌 경로를 한 바퀴 돌아온 뒤 벡터가 받는 선형 사상을 나타내며, 이를 통해 곡률, 홀로노미, 위상적 구조 등이 주기 운동에 미치는 영향을 해석할 수 있다.

선형근사(Linearization)¶

비선형 동적 시스템을 직접적으로 다루기가 복잡할 때 자주 쓰이는 방법이 선형근사다. 미분기하학에서 선형근사는 다양체 위에서 정의된 지도나 벡터장을 접공간에서의 선형변환으로 근사하여 해석하는 방식이다.

다음과 같은 비선형계

\[ \begin{align} \dot{\mathbf{x}}(t) = \mathbf{f}\bigl(t,\mathbf{x}(t)\bigr) \end{align} \]

가 주어졌다고 하자. 어떤 특정 궤적을 \(\mathbf{x}^*(t)\)라고 할 때, 실제 해 \(\mathbf{x}(t)\)가 \(\mathbf{x}^*(t)\)에서 조금 벗어나 있다면, \(\delta \mathbf{x}(t) = \mathbf{x}(t) - \mathbf{x}^*(t)\)로 놓을 수 있다. 만약 \(\delta \mathbf{x}(t)\)가 충분히 작다면 접공간에서의 1차 근사를 통해

\[ \begin{align} \dot{\delta \mathbf{x}}(t) \approx \frac{\partial \mathbf{f}}{\partial \mathbf{x}}\bigl(t, \mathbf{x}^*(t)\bigr)\,\delta \mathbf{x}(t) \end{align} \]

로 표현된다. 여기서 야코비(자코비 행렬) \(\dfrac{\partial \mathbf{f}}{\partial \mathbf{x}}\)이 \(\mathbf{x}^*(t)\)를 기준으로 하는 국소 선형화(Linearization)를 제공한다.

이 선형근사는 군행렬(모노드로미)와 밀접하게 연결된다. 예를 들어 \(\mathbf{x}^*(t)\)가 주기 해라면, 한 주기 \(T\) 후의 작은 교란 \(\delta \mathbf{x}(T)\)는 군행렬 역할을 하는 야코비 기반의 선형 연산자를 통해 \(\delta \mathbf{x}(0)\)에서 도출된다. 즉, 비선형계에서 실제 운동이 주기를 가진다 하더라도, 주위에서 국소적으로 선형근사를 적용하면 모노드로미 행렬을 효과적으로 계산할 수 있다.

구체적으로는 다음과 같은 과정을 거친다. 먼저 주기 운동 \(\mathbf{x}^*(t)\)을 찾는다. 그 다음 \(\mathbf{x}^*(t)\)에서 야코비 행렬 \(\mathbf{A}(t) = \dfrac{\partial \mathbf{f}}{\partial \mathbf{x}}\bigl(t, \mathbf{x}^*(t)\bigr)\)을 구한다. 이 시변선형계

\[ \begin{align} \dot{\delta \mathbf{x}}(t) = \mathbf{A}(t)\,\delta \mathbf{x}(t) \end{align} \]

의 근본해 행렬 \(\mathbf{\Phi}(t,0)\)을 적분으로 구하면, \(\mathbf{\Phi}(T,0)\)이 바로 모노드로미 행렬이 된다.

아래는 매우 간단한 예시로, Python을 사용하여 시간에 대해 주기 계수를 갖는 2차원 선형 시변계를 풀고, 모노드로미 행렬을 구하는 코드다. 이는 시변행렬 \(\mathbf{A}(t)\)가 \(T\)-주기라고 가정하고, 한 주기만큼 적분하여 근본해 행렬을 계산한다. 복잡한 실제 상황에서는 수치적 안정성과 적절한 적분법을 주의 깊게 선택해야 한다.

import numpy as np
from scipy.integrate import solve_ivp

T = 2*np.pi  # 가정하는 주기
def A(t):
    # 예: 2x2 주기 함수 형태
    return np.array([[0, 1],
                     [-1, 0.1*np.sin(t)]])

def system(t, x):
    # x는 4차원 벡터: 근본해 행렬의 열벡터들을 이어 붙인 형태
    X = x.reshape((2,2))
    dXdt = A(t).dot(X)
    return dXdt.flatten()

# 초기 근본해 행렬: 단위행렬
X0 = np.eye(2).flatten()
sol = solve_ivp(system, [0, T], X0, t_eval=[T])
Phi_T = sol.y[:,-1].reshape((2,2))

print("Monodromy matrix =")
print(Phi_T)

위 코드를 통해 \(\mathbf{A}(t)\)에 대한 근본해 행렬 \(\mathbf{\Phi}(t,0)\)의 \(t = T\) 시점 값을 구하면, 그 값이 곧 모노드로미 행렬이다. 이 행렬의 고유값을 조사함으로써 주기 운동 주위에 대한 안정성을 평가할 수 있다. 예를 들어 고유값의 크기가 모두 1보다 작으면 수렴, 1보다 크면 발산한다.

결국 군행렬(모노드로미)과 선형근사는 미분기하학과 제어 이론을 연결하는 핵심 수단이다. 닫힌 궤적이나 주기 운동이 있을 때, 그 주변에서 일어나는 작은 변동을 체계적으로 분석하기 위해 접공간을 통한 선형화와 모노드로미의 해석이 큰 역할을 한다. 이는 고차원 시스템에서도 국소적으로 계를 단순화하여 의미 있는 결론을 얻을 수 있도록 하며, 시스템의 안정성과 제어 가능성을 폭넓게 이해하는 기반이 된다.
