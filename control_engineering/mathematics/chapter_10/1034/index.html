노벡터장과 내부 공액의 개념은 미분기하학에서 나타나는 구조를 파악하는 중요한 출발점이다. 이는 제어이론에서도 계(系)가 놓인 다양체의 기하학적 속성을 분석할 때 유용하게 쓰인다. 특히, 제어 가능한 시스템의 특성이나 그 궤적의 성질을 이해하기 위해 다양체 위에서 정의되는 벡터장의 거동을 살펴볼 필요가 있다.

노벡터장은 다양체 위의 모든 점에서 해당 벡터장이 계량으로 측정했을 때 길이가 0이 되는 경우를 말한다. 내부 공액은 이러한 벡터장, 혹은 벡터장들이 일으키는 흐름(flow)을 내재적 성질만으로 ‘같은 것’으로 간주할 수 있는지 분석하는 개념이다.

아래에서는 먼저 노벡터장이 가지는 성질을 살펴보고, 뒤이어 내부 공액이 어떤 맥락에서 정의되며 노벡터장 분석과 어떻게 연결되는지 설명한다.

노벡터장의 기초 개념¶

노벡터장(null vector field)은 다양체의 각 점에서 정의된 벡터가 계량(리만 계량 또는 준리만 계량 등)에 의해 길이가 0으로 측정되는 벡터로 이루어져 있다. 만약 계량이 양의 정부호(positive definite)라면 \(\mathbf{g}(\mathbf{v}, \mathbf{v}) = 0\) 이 성립하는 벡터 \(\mathbf{v}\)는 진짜 영벡터(0)일 수밖에 없다. 그러나 로런츠 계량 등 부호수가 섞인 계량(pseudo-Riemannian metric)에서는 \(\mathbf{v} \neq \mathbf{0}\)인데도 \(\mathbf{g}(\mathbf{v}, \mathbf{v}) = 0\) 인 비자명한 상황이 발생한다. 이를 ‘노(null) 벡터’라고 하며, 그러한 벡터로 구성된 벡터장을 ‘노벡터장(null vector field)’이라 한다.

노벡터장은 일반상대론과 같은 물리학적 맥락에서 자주 등장하지만, 제어이론에서도 계량이 양의 정부호로만 설정되는 것은 아니기에 특정 상황에서 의미가 있다. 예를 들어, 준리만 구조를 갖춘 다양체 상의 비선형 제어 시스템을 다룰 때, 시간에 따른 궤적이 노(light-like) 곡선과 비슷한 성질을 가지면 별도의 분류 또는 최적화 조건을 고려할 수 있다.

노벡터장을 간단히 수식으로 표현하면, 다양체 \(\mathcal{M}\) 위에 계량 \(\mathbf{g}\)이 주어졌다고 할 때, 점 \(p \in \mathcal{M}\)에서 노벡터 \(\mathbf{v}_p\)는

\[ \begin{align} \mathbf{g}_p(\mathbf{v}_p, \mathbf{v}_p) = 0 \end{align} \]

를 만족하며, \(\mathbf{v}_p \neq \mathbf{0}\) 이다. 이러한 벡터가 모든 점 \(p\)에서 정의되어 있으면 그것이 노벡터장 \(\mathbf{V}\)가 된다.

노벡터장이 존재하면, 그 다양체에서의 기하학적 성질은 전형적인 양의 정부호 계량과는 다른 양상을 보인다. 예를 들어, 노벡터장을 따라가며 진화하는 곡선은 광자선(light-like geodesic)에 해당할 수 있고, 이는 제어이론에서 시간 최적화나 에너지 최적화 문제를 해석할 때 계량 구조를 통한 새로운 시각을 제공하기도 한다.

내부 공액(Intrinsic Conjugacy)의 개념¶

내부 공액은 외부의 좌표 선택이나 추가 구조 없이, 다양체 자체가 갖는 내재적 성질만으로 두 벡터장 또는 벡터장들이 일으키는 흐름이 동등하다고 볼 수 있는지 묻는 개념이다. 예컨대, 두 벡터장 \(\mathbf{X}\)와 \(\mathbf{Y}\)가 미분동형사상(diffeomorphism)이나 등각사상(conformal map) 같은 적절한 사상 \(\Phi\)에 의해 서로 대응(conjugate)할 수 있을 때, 그 사상이 다양체의 내재적 구조(계량, 접속 등)를 보존한다면 이를 ‘내부 공액’이라고 한다.

특히 노벡터장과 관련해서, 사상 \(\Phi\)가 노벡터장 \(\mathbf{V}\)를 보존한다면

\[ \begin{align} \Phi_* (\mathbf{V}) = \mathbf{V} \end{align} \]

가 성립한다고 할 수 있다( \(\Phi_*\)는 밀어올림(pushforward) 연산을 뜻한다). 이때 \(\Phi\)가 노벡터장이라는 구조적 특성을 훼손하지 않고 유지한다면, 이를 노벡터장의 내부 공액이라 부른다. 이는 단순히 좌표 변환으로 인한 우연적 일치가 아니라, 다양체의 본질적(내재적) 구조를 보존하는 사상으로 이해할 수 있다.

제어적 해석에서 벡터장은 시스템의 순간 변화를 나타내므로, 노벡터장은 특수한 상황의 변화를 표현한다. 그 변화를 ‘내재적’으로 보존하는 사상이 존재한다면, 두 시스템은 본질적으로 같은 성질을 공유한다고 볼 수 있다. 따라서 내부 공액은 여러 표현이 서로 다른 제어 시스템을 본질적으로 동등하게 만드는 변환을 연구하는 장(field)으로서 의미가 있다.

노벡터장과 내부 공액의 상호작용¶

노벡터장은 계량 구조가 부호수를 가질 때(예: 로런츠 계량) 나타나는 특별한 벡터장이다. 내부 공액은 이 노벡터장 구조가 변환을 통해서도 유지되는가를 판단하며, 제어기하학에서 중요한 함의를 지닌다.

예를 들어, 노벡터장 방향으로 진화하는 곡선이 등광곡선(null geodesic)이 되는 경우가 많다. 이는 시간적·에너지적 의미가 특별한 곡선이 될 수 있으며, 제어문제에서 다양체 위 최적 궤적을 찾는 과정에서 이런 등광곡선 특성이 핵심 제약으로 작용할 수 있다. 이런 상황에서 계량 구조가 다른 시스템과 내부 공액인지 판별하는 것은, 두 시스템이 동일한 해석을 가질 수 있음을 보여주는 수단이 된다.

예를 들어, 어떤 준리만 다양체에서 정의된 제어 시스템이 로런츠 계량 구조를 가진 또 다른 시스템과 사상 \(\Phi\)로 연결되어 있다고 하자. \(\Phi_* (\mathbf{V}) = \mathbf{V}'\) 형태로 노벡터장을 대응시키고, 계량 구조의 ‘노(null)’ 성질 역시 대응된다면 두 시스템은 내부 공액적이라고 볼 수 있다. 이는 복잡한 시스템을 단순화된 다른 형태의 시스템으로 대치하더라도, 본질적인 거동이나 최적화 특성 등이 유지됨을 의미한다.

예시를 통한 간단한 해석¶

2차원 시공간 모형을 가정하고, 로런츠 계량 \(\mathbf{g}\)가 주어졌다고 하자. 어떤 점 \(p\)에서 노벡터 \(\mathbf{v}_p\)가 존재하고, 다양체 전체에 걸쳐 노벡터장 \(\mathbf{V}\)가 있다고 가정한다. 만약 \(\Phi\)라는 사상이 \(\mathbf{g}\)를 등각적으로 변환한다면, 즉 \(\Phi^*\mathbf{g} = \Omega^2 \mathbf{g}\) 형태로 바뀐다면, 노벡터장 구조는 사상 전후에도 유지된다. 아래는 Python을 이용하여 등각 사상에서 계량이 변환되는 모습을 대략적으로 살펴보는 예시 코드다.

import sympy as sp

t, x = sp.symbols('t x', real=True)
Omega = sp.Function('Omega')(t, x)  # 등각변환 척도 함수

# 로런츠 계량 예: ds^2 = -dt^2 + dx^2
g = sp.Matrix([[-1, 0],
               [ 0, 1]])

# (t, x)를 (u, v)로 등각변환한다 가정
u = t
v = x * Omega  # x축만 스케일 변화 예시

# 변환 야코비 J = d(u, v) / d(t, x)
J = sp.Matrix([[sp.diff(u, t), sp.diff(u, x)],
               [sp.diff(v, t), sp.diff(v, x)]])

# 새 계량 g_new = (J^T) g J (등각인자 등 추가 작업은 생략)
g_new = J.T * g * J
print(sp.simplify(g_new))

등각 사상에서 노벡터 \(\mathbf{v}_p\)가 가진 \(\mathbf{g}(\mathbf{v}_p, \mathbf{v}_p) = 0\) 성질은

\[ \begin{align} (\Phi^* \mathbf{g})(\Phi^*\mathbf{v}_p, \Phi^*\mathbf{v}_p) = \Omega^2 \, \mathbf{g}(\mathbf{v}_p, \mathbf{v}_p) = 0 \end{align} \]

와 같이 스칼라 배( \(\Omega^2\) )만 달라질 뿐 0인 값은 유지된다. 즉, 노벡터 구조가 내부적으로 보존되므로, \(\Phi\)가 노벡터장을 노벡터장으로 대응시키는 내부 공액 사상의 예시가 될 수 있다.

마무리: 제어기하학에서의 의의¶

노벡터장(null vector field)은 계량에 부호수가 섞여 있을 때 나타날 수 있는 특수한 벡터장 구조다. 제어이론에서 이를 연구하는 이유 중 하나는 다양한 물리학적·시공간적 해석을 하나의 모델에서 다룰 수 있기 때문이다. 준리만 계량이나 로런츠 계량을 통한 제어 모형화는 최적화와 안정성 문제에서 새로운 관점을 제공한다.

내부 공액(intrinsic conjugacy)은 이러한 노벡터장 구조가 내재적 사상에 의해 보존되거나 등가로 간주될 수 있음을 보여준다. 이는 물리·공학 모형에서 여러 표현이 실질적으로 같은 해석을 지닐 수 있음을 드러내는 핵심 개념이다. 노벡터장의 존재와 내부 공액에 대한 판별은 복잡한 시스템을 보다 단순화된 등가 시스템으로 치환하여 본질적 거동을 파악하고, 효율적 해석과 설계를 도울 수 있다.

결국, 노벡터장과 내부 공액의 개념은 미분기하학적인 틀 안에서 제어 시스템을 심도 있게 다룰 때 필수적인 도구 중 하나로 자리매김한다. 특히, 비(非)양의 정부호 계량이 필요한 제어 문제에서 노벡터장의 특성과 내부 공액의 판정은 시스템 간의 내재적 동등성을 밝히고, 더욱 확장된 해석과 효율적인 해법을 모색하는 기반이 된다.
