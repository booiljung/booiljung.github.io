부분다양체(Submanifold)와 제어 입력 공간의 관계¶

미분기하학에서 부분다양체는 더 큰 차원의 다양체 속에 매끄럽게 매입된 구조를 의미한다. 제어 이론에서는 상태공간에 대한 제약이나 특정 거동을 고려할 때, 이러한 부분다양체가 동역학계가 머무를 수 있는 공간을 나타내기도 한다. 제어 입력 공간은 일반적으로 시스템에 주어지는 외부 입력을 벡터로 표현한 집합이며, 이 입력이 어떻게 부분다양체의 기하 구조와 맞물리는지 분석하는 과정은 기하학적 제어 이론에서 매우 중요하다.

부분다양체의 정의와 성질¶

부분다양체는 매끄러운 다양체의 부분 집합 중에서, 국소적으로 원래 다양체의 차원보다 낮은 차원을 갖고, 그 주변에서 매끄러운 천접사(immersions)나 매장(embedding)에 의해 기술될 수 있는 집합을 의미한다. 예를 들어 $ \mathbb{R}^3 $ 내에 존재하는 2차원 구면은 전형적인 부분다양체로 볼 수 있다.

부분다양체를 좀 더 체계적으로 다루려면 국소좌표계와 접다발(tangent bundle)에 대한 이해가 필요하다. 부분다양체 $ M $이 매장된 다양체를 $ N $이라 하고, 두 다양체의 차원을 각각 $ \dim(M) $과 $ \dim(N) $이라 하자. $ M $이 $ N $의 부분다양체라면, $ p \in M $에서의 접공간 $ T_p M $은 $ T_p N $의 부분공간이 된다. 이렇게 접공간이 자연스럽게 포함관계를 유지한다는 점이 부분다양체의 핵심 성질 중 하나다.

부분다양체를 통해 상태공간의 특정 영역을 정의하거나, 동역학계가 만족해야 하는 기하학적 제약(예: 기계적 구속 조건)을 수학적으로 명확히 표현할 수 있다. 이때, 제어 입력에 의해 동역학계가 부분다양체 위에서 움직이도록 만들 수도 있고, 반대로 부분다양체 밖으로 벗어나지 않도록 제어를 구성할 수도 있다.

제어 입력 공간과 부분다양체의 상호작용¶

제어 이론에서 상태공간 $ \mathbb{R}^n $ 또는 일반적인 매끄러운 다양체 $ N $ 위에 정의된 제어계는 다음과 같은 꼴로 표현될 때가 많다. $$ \begin{align} \dot{\mathbf{x}} = \mathbf{f}(\mathbf{x}) + \sum_{k=1}^m u_k \,\mathbf{g}_k(\mathbf{x}) \end{align} $$ 여기서 $ \mathbf{x} \in N,, u_k \in \mathbb{R},, \mathbf{f}(\mathbf{x}),, \mathbf{g}_k(\mathbf{x}) \in TN $이다. $ \mathbf{f} $는 드리프트(drift) 항을, $ \mathbf{g}_k $들은 제어 벡터장(control vector field)을 나타내며, $ u_k $는 제어 입력이다.

입력 공간은 $ \mathbb{R}^m $과 같이 유한 차원의 실벡터 공간인 경우가 많다. 미분기하학적으로는 제어 입력에 의해 생성되는 분포(distribution)를 살펴보는데, 이를 통해 가능한 순간적인 상태 변화 방향(접벡터)을 해석한다. 제어 벡터장 $ \mathbf{g}_k(\mathbf{x}) $들이 스팬(span)하는 분포를 $ \Delta(\mathbf{x}) = \text{span}{\mathbf{g}_1(\mathbf{x}),\cdots,\mathbf{g}_m(\mathbf{x})} $ 로 정의할 수 있다.

부분다양체 $ M \subset N $가 주어지고, 상태가 항상 $ M $ 위에 머무르도록 제어를 설계하려면, $ \Delta(\mathbf{x}) $가 $ M $의 접공간 $ T_{\mathbf{x}} M $과 잘 부합해야 한다. 즉, 모든 $ \mathbf{x} \in M $에 대하여 $ \Delta(\mathbf{x}) \subseteq T_{\mathbf{x}} M $ 이 성립해야 하며, 필요하다면 $ \mathbf{f}(\mathbf{x}) $ 역시 $ T_{\mathbf{x}} M $에 속해야 한다. 이를 만족하지 못하면, 제어 입력을 잘 주더라도 시스템은 $ M $ 밖으로 벗어날 수 있다.

분포가 부분다양체를 정의하거나 반대로 부분다양체가 분포의 적분곡면(integral manifold)이 될 때, 이들은 기하학적 제어 이론에서 중요한 역할을 한다. 예를 들어, 프로베니우스 정리(Frobenius theorem)를 사용하면 주어진 분포가 적분가능(involutive)한 경우에만 부분다양체로 적분(즉, 적분다양체)을 구성할 수 있다는 사실을 알 수 있다.

부분다양체 위에서의 제어 설계¶

부분다양체 $ M $ 위에서 제어 문제를 공식화할 때, 우선 부분다양체가 동역학계의 해 궤적이 존재할 수 있도록 만드는 것이 필수적이다. 이를 위해 제어 입력 공간에서 분포가 만들어내는 방향들이 $ M $의 접공간에 정렬될 것을 요구한다. 추가적으로, 부분다양체 내부에서 원하는 궤적을 구성하거나 안정화시키는 문제를 고민하게 된다.

부분다양체가 제어 계의 해 궤적 전체를 표현하는 경우도 있다. 예를 들어 구속이 주어진 기계 시스템에서는 해당 구속식을 만족하는 모든 점들의 집합이 부분다양체를 이룬다. 이때 제어 입력은 이 부분다양체 위에 머무르면서도 원하는 동작을 실현해야 한다. 간단히 말해, 동역학이 부분다양체를 이탈하지 않도록 입력을 구성함과 동시에, 부분다양체 상에서의 목표 거동을 이끌어내는 제어 방정식을 세우게 된다.

부분다양체 위에서의 제어 문제를 정식화하기 위해, 국소좌표계를 도입해 부분다양체 $ M $의 국소 파라미터를 $ \mathbf{q} \in \mathbb{R}^d \((\) d = \dim(M) $)로 잡는다. 원래 상태공간 $ N $이 $ \dim(N) = n $ 차원이라면, $ n - d $에 해당하는 축방향은 부분다양체의 국소좌표계에서 배제된다. 이때 동역학계는 로컬하게 다음과 같은 형태로 볼 수 있다. $$ \begin{align} \dot{\mathbf{q}} = \mathbf{f}M(\mathbf{q}) + \sum{k=1}^m u_k \,\mathbf{g}_{M,k}(\mathbf{q}) \end{align} $$ 여기서 $ \mathbf{f}M $과 $ \mathbf{g}{M,k} $는 부분다양체의 국소좌표계로 사영(projection)한 벡터장이다. 이 벡터장들이 $ M $ 위를 적절히 덮으면서 유효한 제어 입력 범위를 형성하면, 실제 제어 입력 $ {u_k} $로 인해 $ \mathbf{q} $가 원하는 대로 움직인다. 반대로, $ \mathbf{f} $나 $ \mathbf{g}_k $가 부분다양체의 접공간과 합치되지 않으면, 부분다양체 위에 머무르기 어렵다.

예제: 이중 진자에서의 부분다양체 접근¶

이중 진자(double pendulum) 시스템을 간단한 제약과 함께 생각해보자. 어떤 방식으로든 보조 링크 사이의 각도를 일정하게 유지하도록 설계하고 싶다면, 두 관절 각도 \((\theta_1,\theta_2)\)가 특정 관계를 만족하는 부분다양체를 생각해볼 수 있다. 가령, $ \theta_2 = 2\theta_1 $이라는 구속을 준다면, 모든 상태는 이 관계식이 정의하는 곡면 위에 놓이게 된다.

이 구속을 만족하는 집합은 2차원 공간에서 제약 1개가 들어가므로, $ \dim(M) = 1 $이 된다. 여기에 외부에서 모터 토크를 제어 입력으로 주어 \((u_1,u_2)\)를 가정하면, 실제로 가능한 움직임은 $ \theta_1 $과 $ \theta_2 $의 구속을 만족하는 선 위에서만 이뤄진다. 이러한 맥락에서 제어 입력을 통해 부분다양체를 유지하면서 원하는 동작을 실현할 수 있는지를 고찰하게 된다.

아래는 간단한 Python 코드를 통해, 각도 구속이 있는 이중 진자를 수치적으로 시뮬레이션해볼 수 있는 예시다. 여기서는 보통의 에너지를 포함한 진자 방정식을 단순화해, $ \theta_2 = 2\theta_1 $ 조건이 유지되도록 하면서 제어 입력 $ u_1,u_2 $를 설정한다. 다만 실제로 제약에 따라 동역학 방정식을 재구성해야 한다는 점을 참고해야 한다.

import numpy as np
import matplotlib.pyplot as plt

def double_pendulum_constrained(t, state, u1, u2):
    theta1, dtheta1, theta2, dtheta2 = state

    # 가령, 구속: theta2 = 2 * theta1 (단순 예시)
    # 여기서는 단순화된 계수와 중력, 질량, 길이 등을 설정
    g = 9.81
    l = 1.0
    m = 1.0

    # 실제로는 제약 방정식을 만족하도록 동역학을 재구성해야 함
    # 여기서는 단순 비교를 위한 예시
    f1 = -(g/l)*np.sin(theta1) + u1/m
    f2 = -(g/l)*np.sin(theta2) + u2/m

    return [dtheta1, f1, dtheta2, f2]

# 초기값
theta1_0 = 0.1
theta2_0 = 2*theta1_0
dtheta1_0 = 0.0
dtheta2_0 = 0.0
state = [theta1_0, dtheta1_0, theta2_0, dtheta2_0]

# 시간 설정
t = 0.0
dt = 0.01
t_end = 10.0
time = []
theta1_log = []
theta2_log = []

while t < t_end:
    # 임의의 제어 입력 (예: 0으로 가정)
    u1 = 0.0
    u2 = 0.0

    # 오일러 전진법으로 간단히 적분
    derivatives = double_pendulum_constrained(t, state, u1, u2)
    for i in range(len(state)):
        state[i] += derivatives[i]*dt

    time.append(t)
    theta1_log.append(state[0])
    theta2_log.append(state[2])

    t += dt

plt.plot(time, theta1_log, label="theta1")
plt.plot(time, theta2_log, label="theta2")
plt.legend()
plt.show()

위 코드는 실제 물리적으로 완벽한 제약 시뮬레이션이 아니라, 부분다양체 형태의 구속 조건을 단순히 대입해 본 예시다. 제약 조건에 따라 동역학을 해석하려면 보조 구속력이나 라그랑주 승수 등을 고려해야 한다. 그러나 이처럼 부분다양체의 개념과 제어 입력 사이의 관계를 시뮬레이션 관점에서 간단히 살펴볼 수 있다.

결국 부분다양체와 제어 입력 공간 사이의 관계를 주의 깊게 살펴보면, 구속 조건이나 특정 거동을 유지하면서도 원하는 제어 목표를 달성할 수 있는 방안을 찾을 수 있다. 특히, 분포가 어떻게 부분다양체의 접공간을 구성하는지, 그리고 동역학계가 그 부분다양체 위에서 닫힌 해 궤적(closed orbit) 혹은 원하는 궤적을 가질 수 있는지 검토하는 일이 중요하다. 이를 통해 복잡한 기계 시스템이나 다차원 로봇의 기하학적 제어 문제를 체계적으로 해결할 수 있게 된다.
