감사의 글

ROS2 Humble을 기반으로 로봇 시스템을 연구하고 개발하면서, 실제 필드에서 피드백을 보내 주신 여러 기관 및 연구실에도 감사의 마음을 전한다. 로봇 공학은 실험실과 현장의 괴리가 큰 분야이기 때문에, 실제 환경에서의 데이터와 시나리오에 기반한 경험적 제언은 커다란 가치를 갖는다. 그런 점에서 실제 로봇에게 ROS2를 적용해 주시고, 예상치 못했던 트러블슈팅 과정을 공유해 주신 현장 엔지니어, 연구원, 개발자, 학생들의 참여가 매우 소중하였다.

아울러 지식을 나누는 과정에서 발생한 여러 질문과 이슈에 대해 친절하게 가이드를 제공해 주신 오픈소스 커뮤니티 멤버들도 잊을 수 없다. ROS Answers 포럼, GitHub 이슈 및 PR( Pull Request ) 등 다양한 채널을 통해 논의와 협업이 이루어졌으며, 여기에서 나온 여러 제안을 통해 본문 설명이 더욱 탄탄해지고 예제 코드가 개선될 수 있었다.

특히, DDS(Domain Data Service) 계열 통신 미들웨어에 대한 ROS2 Humble의 호환성과 성능을 직접 테스트해 주신 기업들과 소규모 연구팀의 기여 덕분에, 각종 DDS 벤더 간 호환성과 버그 픽스에 관한 실제 데이터가 집약될 수 있었다. 이 모든 노하우가 축적되어 이번 책이 단순 지식 전달서에 그치지 않고, 실무 기법과 경험이 녹아 있는 ‘실천 가이드’로 기능하게 된 것 같다.

나아가 ROS2가 단순히 로봇에 국한되지 않고, 임베디드 시스템, 자율 주행, 드론, 물류 자동화 등 다양한 영역에 확장될 수 있도록 애쓰신 업계 전문가들의 노력에도 감사의 뜻을 전한다. 로봇 생태계와 인접 분야의 접점을 넓혀 주신 덕분에, 이 책 역시 한정된 분야의 독자에게만 유용한 자료로 머무르지 않고, 다양한 응용 가능성을 보여 줄 수 있었다.
