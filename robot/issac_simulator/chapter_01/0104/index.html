설치 가이드¶

시스템 요구 사항¶

아이작 시뮬레이터를 설치하기 전에, 시스템 요구 사항을 충족해야 합니다. 아래는 최소 및 권장 사양입니다.

최소 사양¶

  • 운영 체제: Ubuntu 20.04 또는 Windows 10
  • CPU: 4코어 이상
  • RAM: 16GB
  • GPU: NVIDIA RTX 2060 이상
  • 디스크 공간: 10GB 이상
  • 드라이버: NVIDIA GPU 드라이버 510 이상

권장 사양¶

  • 운영 체제: Ubuntu 22.04
  • CPU: 8코어 이상
  • RAM: 32GB
  • GPU: NVIDIA RTX 3090 이상
  • 디스크 공간: 20GB 이상
  • 드라이버: 최신 NVIDIA GPU 드라이버

필수 소프트웨어 설치¶

아이작 시뮬레이터를 실행하려면 다음 소프트웨어가 설치되어 있어야 합니다.

1. NVIDIA 드라이버¶

NVIDIA GPU 드라이버를 설치하려면 다음 명령을 사용합니다.

sudo apt update
sudo apt install -y nvidia-driver-510

설치 후, 드라이버가 올바르게 작동하는지 확인하려면 다음 명령을 실행합니다.

nvidia-smi

2. CUDA 및 cuDNN¶

아이작 시뮬레이터는 GPU 가속을 활용하기 위해 CUDA와 cuDNN이 필요합니다.

  • CUDA 설치:
sudo apt install -y cuda
  • cuDNN 설치: NVIDIA 공식 웹사이트에서 cuDNN을 다운로드하고 설치합니다. 설치 경로를 $LD_LIBRARY_PATH에 추가해야 합니다.
export LD_LIBRARY_PATH=/usr/local/cuda/lib64:$LD_LIBRARY_PATH

아이작 시뮬레이터 다운로드 및 설치¶

1. NVIDIA 공식 웹사이트에서 다운로드¶

NVIDIA Isaac Sim 페이지를 방문하여 최신 버전의 설치 파일을 다운로드합니다.

2. 설치 파일 실행¶

다운로드한 설치 파일을 실행하여 설치를 진행합니다. Linux의 경우 다음 명령을 사용할 수 있습니다.

chmod +x ./isaac-sim.run
./isaac-sim.run

Windows에서는 .exe 파일을 실행하여 설치합니다.

3. 설치 디렉터리 설정¶

설치 중, 원하는 디렉터리를 선택합니다. 기본적으로 /opt/NVIDIA/Isaac 디렉터리에 설치됩니다.

환경 변수 설정¶

설치가 완료되면 환경 변수를 설정해야 합니다. Linux의 경우, 다음 명령을 사용합니다.

export ISAAC_SIM_PATH=/opt/NVIDIA/Isaac
export PATH=$ISAAC_SIM_PATH/bin:$PATH

Windows에서는 시스템 환경 변수에 ISAAC_SIM_PATH를 추가하고 설치 경로를 지정합니다.

추가 구성 요소 설치¶

아이작 시뮬레이터의 기능을 완전히 활용하기 위해 몇 가지 추가 구성 요소를 설치해야 합니다.

1. Python 및 pip 설정¶

아이작 시뮬레이터는 Python 기반 스크립트를 지원하므로, Python 3.8 이상이 필요합니다. 설치 여부를 확인하려면 다음 명령을 실행합니다.

python3 --version

만약 Python이 설치되지 않았다면, 아래 명령을 사용해 설치합니다.

sudo apt update
sudo apt install -y python3 python3-pip

2. Python 종속성 설치¶

아이작 시뮬레이터는 몇 가지 필수 Python 라이브러리를 필요로 합니다. 아래 명령을 사용하여 종속성을 설치합니다.

pip3 install -r $ISAAC_SIM_PATH/requirements.txt

3. ROS2 통합 설정¶

아이작 시뮬레이터는 ROS2 Humble과의 통합을 지원합니다. ROS2를 설치하려면 다음 명령을 사용합니다.

sudo apt update
sudo apt install -y ros-humble-desktop

ROS2와 아이작 시뮬레이터를 연결하려면 ROS2 브리지 패키지를 설치해야 합니다.

pip3 install isaac-ros-bridge

환경 변수를 설정하여 ROS2 워크스페이스를 초기화합니다.

source /opt/ros/humble/setup.bash

Docker 기반 설치¶

아이작 시뮬레이터는 Docker 환경에서 실행할 수도 있습니다. Docker를 사용하는 경우 아래 과정을 따라야 합니다.

1. Docker 및 NVIDIA Container Toolkit 설치¶

먼저 Docker를 설치합니다.

sudo apt update
sudo apt install -y docker.io
sudo systemctl start docker
sudo systemctl enable docker

그 후, NVIDIA Container Toolkit을 설치합니다.

distribution=$(. /etc/os-release;echo $ID$VERSION_ID)
curl -s -L https://nvidia.github.io/nvidia-docker/gpgkey | sudo apt-key add -
curl -s -L https://nvidia.github.io/nvidia-docker/$distribution/nvidia-docker.list | sudo tee /etc/apt/sources.list.d/nvidia-docker.list
sudo apt update
sudo apt install -y nvidia-container-toolkit
sudo systemctl restart docker

2. Docker 이미지 다운로드 및 실행¶

아이작 시뮬레이터의 Docker 이미지를 다운로드합니다.

docker pull nvcr.io/nvidia/isaac-sim:latest

Docker 컨테이너를 실행합니다.

docker run --gpus all -it --rm -e DISPLAY=$DISPLAY -v /tmp/.X11-unix:/tmp/.X11-unix nvcr.io/nvidia/isaac-sim:latest

환경 변수를 컨테이너 내부에서 설정하여 사용하면 됩니다.

실행 확인¶

1. GUI 실행¶

아이작 시뮬레이터가 제대로 설치되었다면, 다음 명령으로 실행할 수 있습니다.

./isaac-sim.sh

GUI가 정상적으로 열리고 시뮬레이션 환경이 로드되면 설치가 완료된 것입니다.

2. 명령줄 실행¶

GUI 없이 명령줄에서 특정 스크립트를 실행하려면 다음과 같이 명령을 입력합니다.

./isaac-sim.sh --no-gui --scenario example_scenario.usd

문제 해결 및 디버깅¶

아이작 시뮬레이터 설치 및 실행 중 발생할 수 있는 문제를 해결하기 위해, 다음과 같은 주요 디버깅 방법을 활용할 수 있습니다.

1. GPU 드라이버 문제 확인¶

nvidia-smi 명령으로 GPU 드라이버가 올바르게 작동하는지 확인합니다. 출력이 없거나 오류가 발생하면, GPU 드라이버를 다시 설치합니다.

sudo apt purge -y nvidia*
sudo apt install -y nvidia-driver-510

2. Docker 환경 문제¶

Docker 컨테이너 실행 시, GPU가 제대로 연결되지 않을 경우 NVIDIA Container Toolkit이 올바르게 설치되었는지 확인합니다.

sudo docker info | grep -i nvidia

만약 출력이 없거나 문제가 있다면, NVIDIA Container Toolkit을 다시 설치하고 Docker를 재시작합니다.

sudo apt install -y nvidia-container-toolkit
sudo systemctl restart docker

3. 환경 변수 문제¶

환경 변수가 제대로 설정되지 않았다면, 실행 오류가 발생할 수 있습니다. 아래 명령으로 환경 변수를 확인합니다.

echo $ISAAC_SIM_PATH
echo $LD_LIBRARY_PATH

만약 값이 비어있다면, 환경 변수를 다시 설정합니다.

export ISAAC_SIM_PATH=/opt/NVIDIA/Isaac
export LD_LIBRARY_PATH=/usr/local/cuda/lib64:$LD_LIBRARY_PATH

4. 종속성 문제¶

Python 라이브러리가 누락된 경우, 다음 명령으로 종속성을 다시 설치합니다.

pip3 install -r $ISAAC_SIM_PATH/requirements.txt

5. 실행 로그 확인¶

아이작 시뮬레이터 실행 중 문제가 발생하면, 로그 파일을 확인하여 원인을 파악합니다. 로그 파일은 기본적으로 다음 경로에 저장됩니다.

$ISAAC_SIM_PATH/logs/latest.log

설치 테스트¶

설치가 제대로 완료되었는지 확인하기 위해, 제공된 예제 시뮬레이션 파일을 실행합니다.

1. 기본 예제 실행¶

설치 경로에 포함된 예제 시뮬레이션 파일을 실행합니다.

./isaac-sim.sh --scenario example_scenario.usd

2. ROS2 통합 테스트¶

ROS2와의 통합을 테스트하기 위해, ROS2 노드와 연결되는 샘플 코드를 실행합니다.

ros2 launch isaac_ros_bridge example.launch.py

이후, rostopic list 명령을 실행하여 ROS2 토픽이 정상적으로 노출되는지 확인합니다.

설치 최적화¶

아이작 시뮬레이터의 성능을 최적화하려면 다음 설정을 적용할 수 있습니다.

1. 그래픽 설정 최적화¶

settings.json 파일에서 그래픽 옵션을 수정하여 GPU 성능을 최대화할 수 있습니다. 예를 들어, 해상도와 품질을 조정합니다.

{
  "resolution": "1920x1080",
  "quality": "high"
}

2. 다중 GPU 설정¶

여러 GPU를 사용하는 경우, 아이작 시뮬레이터가 모든 GPU를 활용하도록 설정합니다. NVIDIA Control Panel에서 다중 GPU 모드를 활성화하거나 Docker 컨테이너 실행 시 --gpus all 옵션을 사용합니다.

docker run --gpus all -it --rm nvcr.io/nvidia/isaac-sim:latest
