2.4 Level 1: 운전자 보조(Driver Assistance)

1. 정의

SAE J3016 표준에서 Level 1(Driver Assistance)은 운전 자동화 시스템이 DDT의 횡방향 제어 또는 종방향 제어 중 하나를 지속적으로 수행하는 수준을 의미한다(SAE International, 2021). 시스템은 두 제어 기능 중 하나만을 담당하며, 나머지 제어 기능과 OEDR은 인간 운전자가 수행한다.

2. 기술적 특성

Level 1의 핵심적 특성은 시스템이 DDT의 하위 과업 중 하나의 운동 제어 모드(횡방향 또는 종방향)만을 지속적으로 수행한다는 점이다. 두 가지 운동 제어를 동시에 수행하는 경우에는 Level 2로 분류된다.

2.1 종방향 제어 수행 사례

시스템이 종방향 제어(가감속)를 수행하고 인간 운전자가 횡방향 제어(조향)와 OEDR을 수행하는 유형이다.

  • 적응형 순항 제어(Adaptive Cruise Control, ACC): 전방 차량과의 거리를 레이더 또는 카메라로 측정하여 설정된 차간 거리를 유지하도록 차량의 속도를 자동으로 조절한다. 전방 차량이 감속하면 자동으로 감속하고, 전방이 개방되면 설정 속도까지 가속한다. 운전자는 조향을 직접 수행하여야 하며, 주행 환경 감시의 의무가 유지된다.

2.2 횡방향 제어 수행 사례

시스템이 횡방향 제어(조향)를 수행하고 인간 운전자가 종방향 제어(가감속)와 OEDR을 수행하는 유형이다.

  • 차선 유지 보조(Lane Keeping Assist, LKA): 카메라로 차선 표시를 인식하여 차량이 차선 중앙을 유지하도록 조향을 보조한다. 운전자는 가감속을 직접 수행하여야 하며, 주행 환경 감시의 의무가 유지된다.
  • 차선 중앙 유지(Lane Centering Assist, LCA): LKA의 발전된 형태로, 차선 이탈 시 보정하는 수준을 넘어 차선 중앙을 지속적으로 유지하도록 조향을 수행한다.

3. Level 1에서의 인간 운전자 역할

Level 1에서 인간 운전자의 역할은 Level 0과 본질적으로 유사하다. 시스템이 하나의 운동 제어 모드를 수행하더라도, 인간 운전자는 다음의 의무를 지속적으로 이행하여야 한다.

역할 항목수행 주체비고
횡방향 제어인간 또는 시스템시스템이 하나만 수행
종방향 제어인간 또는 시스템시스템이 하나만 수행
OEDR인간항시 수행 의무
DDT Fallback인간항시 대응 가능 상태 유지

인간 운전자는 시스템이 수행하지 않는 운동 제어 모드를 직접 수행하면서 동시에 OEDR을 지속적으로 수행하여야 한다. 시스템의 보조 기능이 작동 중이더라도 운전자의 주의 의무(Duty of Attention)는 감소하지 않는다.

4. Level 0과의 구분

Level 0와 Level 1을 구분하는 핵심 기준은 시스템의 DDT 수행이 지속적인가의 여부이다.

  • Level 0: 시스템은 DDT를 지속적으로 수행하지 않는다. AEB, ESC 등의 순간적 개입 기능은 허용되나, 이는 위험 상황에 대한 반응적 개입이다.
  • Level 1: 시스템은 DDT의 하나의 운동 제어 모드를 지속적으로 수행한다. ACC는 전방 차량이 존재하는 동안 지속적으로 종방향 제어를 수행하며, LKA는 차선이 인식되는 동안 지속적으로 횡방향 제어를 보조한다.

5. Level 1의 기술적 한계

Level 1 시스템은 하나의 운동 제어 모드만을 수행하므로, 다른 제어 모드와의 조율(Coordination)이 이루어지지 않는다. 예를 들어, ACC는 전방 차량의 속도에 따라 종방향 제어를 수행하나, 곡선 도로에서 속도를 조절하기 위한 횡방향 곡률 정보를 고려하지 않는다. 마찬가지로, LKA는 차선 중앙을 유지하도록 조향하나, 전방 차량과의 간격을 고려한 속도 조절을 수행하지 않는다.

이러한 한계는 두 가지 운동 제어 모드를 동시에 수행하는 Level 2로의 발전 동기가 된다.
