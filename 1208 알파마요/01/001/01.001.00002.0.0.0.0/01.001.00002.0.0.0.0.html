Chapter 2. SAE 자율주행 등급 체계 (Level 0 ~ Level 5)

자율주행 기술의 수준을 명확하고 일관되게 기술하기 위해서는 국제적으로 합의된 분류 체계가 필수적이다. SAE International이 제정한 J3016 표준은 운전 자동화 시스템의 수준을 6단계(Level 0~Level 5)로 분류하는 체계로, 학계, 산업계, 규제 기관에서 자율주행 기술을 논의할 때 사용되는 표준적인 프레임워크이다(SAE International, 2021).

1. 개요

SAE J3016 표준은 운전 자동화 시스템이 동적 운전 과업(Dynamic Driving Task, DDT)을 수행하는 범위, 인간 운전자의 역할, 그리고 대체 수행(Fallback) 주체에 따라 자동화 등급을 구분한다. 이 분류 체계의 핵심적인 개념 요소는 다음과 같다.

  • 동적 운전 과업(DDT): 실시간 운전에 필요한 조작적, 전술적 기능의 총체로서 횡방향 제어, 종방향 제어, 주행 환경 감시(OEDR)를 포함한다.
  • DDT 대체 수행(DDT Fallback): 시스템이 DDT를 더 이상 수행할 수 없는 상황에서 최소 위험 상태(Minimal Risk Condition, MRC)에 도달하기 위한 대응 행동이다.
  • 운용 설계 영역(ODD): 자율주행 시스템이 설계 의도대로 작동하도록 설계된 운용 조건의 범위이다.

Level 0부터 Level 2까지는 인간 운전자가 DDT 수행의 주체이며 시스템은 보조적 역할을 수행한다. Level 3부터 Level 5까지는 시스템이 DDT의 전체를 수행하며, 인간의 역할은 단계적으로 축소된다. Level 2와 Level 3 사이에는 OEDR의 수행 주체가 인간에서 시스템으로 전환되는 질적 경계가 존재하며, 이 경계는 자율주행의 법적, 기술적 책임 소재를 결정하는 핵심 기준이다.
