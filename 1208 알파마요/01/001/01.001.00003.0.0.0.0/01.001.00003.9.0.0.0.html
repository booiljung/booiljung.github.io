3.9 차량 간 통신(V2X) 하위 시스템

1. 개요

V2X(Vehicle-to-Everything) 통신은 차량이 주변의 다른 차량, 도로 인프라, 보행자, 네트워크 등과 무선으로 정보를 교환하는 기술이다. V2X는 차량 탑재 센서의 인지 범위를 초과하는 영역의 정보를 획득할 수 있게 하며, 자율주행 시스템의 상황 인식 능력을 확장한다.

2. V2X 통신의 유형

유형명칭통신 대상주요 응용
V2VVehicle-to-Vehicle다른 차량충돌 경고, 협조적 주행, 군집 주행
V2IVehicle-to-Infrastructure도로 인프라교통 신호 정보, 공사 구간 정보
V2PVehicle-to-Pedestrian보행자 단말보행자 충돌 경고
V2NVehicle-to-Network클라우드 서버교통 정보, 지도 갱신, 원격 관제

3. 통신 기술 표준

3.1 DSRC (Dedicated Short-Range Communications)

DSRC는 5.9 GHz 대역에서 동작하는 차량 전용 단거리 통신 기술이다. IEEE 802.11p(현재 IEEE 802.11bd로 진화) 기반으로, 저지연(수 밀리초) 통신이 가능하다. V2V 및 V2I 통신에 사용되며, 미국에서는 SAE J2735 메시지 표준과 결합하여 사용된다.

3.2 C-V2X (Cellular V2X)

C-V2X는 셀룰러 네트워크 기술 기반의 V2X 통신이다. 3GPP에서 표준화되었으며, LTE-V2X(Release 14)와 NR-V2X(5G, Release 16)로 구분된다. 직접 통신(PC5 사이드링크)과 네트워크 기반 통신(Uu 인터페이스)을 모두 지원한다. 기존 셀룰러 인프라를 활용할 수 있으며, 5G NR-V2X는 높은 신뢰성과 저지연 통신을 제공한다.

4. V2X의 자율주행 활용

V2X는 자율주행 시스템의 인지 능력을 보강하는 역할을 한다. 주요 활용 사례는 다음과 같다.

  • SPaT(Signal Phase and Timing): 교통 신호등의 현재 상태와 변경 예정 시점을 V2I로 수신하여 교차로 통과 판단에 활용한다.
  • 협조적 인지(Cooperative Perception): 다른 차량이나 도로변 센서가 인지한 정보를 V2V/V2I로 공유하여 자차 센서의 사각지대를 보완한다.
  • 긴급 차량 접근 경고: 긴급 차량의 접근 방향과 거리를 V2V로 사전에 인지하여 양보 경로를 계획한다.
  • 공사 구간 정보: 도로 공사 구간의 위치, 임시 차선 배치, 속도 제한 등의 정보를 V2I로 수신한다.

5. V2X의 기술적 과제

  • 보안: V2X 메시지의 위조, 변조, 재전송 공격에 대한 방어가 필요하다. PKI(Public Key Infrastructure) 기반의 인증 체계가 적용된다.
  • 프라이버시: 차량의 위치와 이동 궤적이 지속적으로 전송되므로, 개인 정보 보호를 위한 가명(Pseudonym) 기법이 사용된다.
  • 인프라 보급: V2I 통신의 효용을 실현하기 위해서는 도로 인프라에 통신 장비의 설치가 필요하며, 이는 대규모 투자를 요구한다.
  • 상호 운용성: DSRC와 C-V2X 간의 기술 표준 분열이 존재하며, 양 기술 간의 상호 운용성 확보가 과제이다.
