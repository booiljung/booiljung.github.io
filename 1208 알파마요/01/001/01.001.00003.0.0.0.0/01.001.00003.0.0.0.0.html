Chapter 3. 자율주행 시스템의 구성 요소

자율주행 시스템은 센서, 연산 장치, 소프트웨어, 통신 인프라, 차량 인터페이스 등 다수의 하위 시스템이 유기적으로 결합된 복합 시스템이다. 본 Chapter에서는 자율주행 시스템을 구성하는 각 하위 시스템의 기능, 구조, 그리고 상호 관계를 체계적으로 기술한다.

1. 개요

자율주행 시스템의 전체 구조는 크게 하드웨어 계층과 소프트웨어 계층으로 구분된다. 하드웨어 계층에는 카메라, 라이다, 레이더 등의 센서, 온보드 컴퓨팅 플랫폼, 차량 액추에이터, V2X(Vehicle-to-Everything) 통신 장치 등이 포함된다. 소프트웨어 계층에는 인지, 위치 추정, 판단, 제어 등의 기능 모듈과 이를 지원하는 미들웨어 및 운영 체제가 포함된다.

이들 구성 요소 사이의 정보 흐름은 다음과 같이 요약된다. 센서가 주행 환경의 원시 데이터를 수집하면, 인지 모듈이 이를 처리하여 환경의 구조화된 표현을 생성한다. 위치 추정 모듈은 차량의 현재 위치와 자세를 결정한다. 판단 모듈은 인지 결과와 위치 정보를 바탕으로 주행 경로와 행동 전략을 수립하며, 제어 모듈은 이를 차량 액추에이터 명령으로 변환한다. 고정밀 지도는 인지, 위치 추정, 판단 모듈에 사전 구축된 환경 정보를 제공하고, V2X 통신은 차량 외부의 교통 인프라 및 다른 교통 참여자와의 정보 교환을 가능하게 한다.

본 Chapter에서는 자율주행 시스템의 전체 아키텍처 개요에서 출발하여, 센서 하위 시스템, 인지, 위치 추정, 판단, 제어 하위 시스템, 차량 인터페이스, 고정밀 지도, V2X 통신, 온보드 컴퓨팅 플랫폼, 소프트웨어 미들웨어, 그리고 구성 요소 간 데이터 흐름과 시스템 통합에 이르기까지 순차적으로 학습한다.
