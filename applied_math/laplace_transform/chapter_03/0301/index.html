단위 계단 함수 정의¶

단위 계단 함수 \(u(t)\)는 Heaviside 함수라고도 불리며, 신호 처리 및 제어 시스템에서 중요한 역할을 한다. 이 함수는 시간 \(t\)가 0 이상일 때 1의 값을 가지며, 그 이전에는 0의 값을 갖는다. 수식으로는 다음과 같이 정의된다.

\[ u(t) = \begin{cases} 0 & \text{if } t < 0, \\ 1 & \text{if } t \geq 0. \end{cases} \]

단위 계단 함수는 시스템의 입력이나 상태가 특정 시간 이후에만 활성화되는 경우를 모델링하는 데 사용된다.

단위 계단 함수의 라플라스 변환¶

단위 계단 함수의 라플라스 변환을 구하기 위해, 라플라스 변환의 정의를 사용한다. 라플라스 변환은 다음과 같이 정의된다.

\[ \mathcal{L}\{f(t)\} = \int_0^{\infty} f(t) e^{-st} dt \]

여기서 \(f(t) = u(t)\)를 대입하면, 단위 계단 함수의 라플라스 변환은 다음과 같다.

\[ \mathcal{L}\{u(t)\} = \int_0^{\infty} 1 \cdot e^{-st} dt = \int_0^{\infty} e^{-st} dt \]

위 적분을 계산하면,

\[ \mathcal{L}\{u(t)\} = \left[ \frac{-1}{s} e^{-st} \right]_0^{\infty} \]

무한대에서 \(e^{-st}\)는 0으로 수렴하므로, 결과는 다음과 같다.

\[ \mathcal{L}\{u(t)\} = \frac{1}{s} \quad \text{for} \quad \text{Re}(s) > 0 \]

따라서, 단위 계단 함수의 라플라스 변환은 \(\frac{1}{s}\)이다.

시간 지연이 포함된 단위 계단 함수¶

시간 지연이 포함된 단위 계단 함수 \(u(t-a)\)는 시간 \(a\) 후에 1로 변하는 함수이다. 이 함수는 다음과 같이 정의된다.

\[ u(t-a) = \begin{cases} 0 & \text{if } t < a, \\ 1 & \text{if } t \geq a. \end{cases} \]

이러한 시간 지연이 있는 단위 계단 함수의 라플라스 변환을 구하면 다음과 같다.

\[ \mathcal{L}\{u(t-a)\} = \int_a^{\infty} e^{-st} dt \]

위 적분을 계산하면,

\[ \mathcal{L}\{u(t-a)\} = \left[ \frac{-1}{s} e^{-st} \right]_a^{\infty} \]

그러면, 시간 지연이 포함된 단위 계단 함수 \(u(t-a)\)의 라플라스 변환을 계속해서 계산해 보자. 무한대에서 \(e^{-st}\)는 0으로 수렴하므로, 다음과 같이 계산된다.

\[ \mathcal{L}\{u(t-a)\} = \frac{-1}{s} \left( e^{-s(\infty)} - e^{-sa} \right) = \frac{-1}{s} \left( 0 - e^{-sa} \right) \]

결과적으로,

\[ \mathcal{L}\{u(t-a)\} = \frac{e^{-sa}}{s} \quad \text{for} \quad \text{Re}(s) > 0 \]

따라서, 시간 지연이 포함된 단위 계단 함수 \(u(t-a)\)의 라플라스 변환은 \(\frac{e^{-sa}}{s}\)로 나타난다. 여기서 \(a\)는 시간 지연을 나타내며, 함수가 시간 \(t = a\)에서 1로 변하는 시점을 의미한다.

단위 계단 함수와 시스템 응답¶

단위 계단 함수는 제어 시스템과 신호 처리에서 중요한 역할을 한다. 특히, 시스템의 단위 계단 응답 (step response)을 분석하는 데 사용된다. 시스템에 단위 계단 입력이 주어졌을 때, 시스템의 시간에 따른 출력은 단위 계단 응답으로 불린다. 이를 통해 시스템의 안정성, 시간 응답 특성, 과도 응답을 분석할 수 있다.

라플라스 변환을 통해 복잡한 미분 방정식으로 표현된 시스템을 쉽게 분석할 수 있으며, 시스템의 전달 함수 \(H(s)\)를 이용하여 단위 계단 응답을 구할 수 있다.

\[ Y(s) = H(s) \cdot \frac{1}{s} \]

여기서 \(Y(s)\)는 시스템의 출력의 라플라스 변환이고, \(H(s)\)는 시스템의 전달 함수이다. 단위 계단 함수의 라플라스 변환을 이용하여, 시스템의 응답을 직접 계산할 수 있다.

라플라스 변환을 통한 시간 영역 해석¶

단위 계단 함수와 라플라스 변환을 이용하면, 시스템의 시간 응답을 주파수 영역에서 분석한 후 시간 영역으로 변환할 수 있다. 이를 통해 시스템이 특정 입력에 대해 어떻게 응답하는지 예측할 수 있다. 시스템의 안정성이나 과도 상태 등을 분석할 때도 이러한 라플라스 변환과 단위 계단 함수가 유용하다.

단위 계단 입력은 흔히 실제 물리적 시스템에서 초기 조건을 설정하거나 일정한 입력을 가하는 상황을 모델링하는 데 쓰이기 때문에, 다양한 분야에서 폭넓게 사용된다.
