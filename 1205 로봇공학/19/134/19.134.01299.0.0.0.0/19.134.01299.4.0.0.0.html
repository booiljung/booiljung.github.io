BehaviorTree.CPP의 라이선스와 배포 (License and Distribution of BehaviorTree.CPP)

1. 개요

BehaviorTree.CPP는 MIT 라이선스로 배포되는 오픈 소스 소프트웨어이다. MIT 라이선스는 상업적 사용, 수정, 배포를 자유롭게 허용하며, 원본 라이선스 고지만을 요구하는 관대한(permissive) 라이선스이다.

4. Groot2의 라이선스

Groot2 시각화 도구는 BehaviorTree.CPP와 별도의 라이선스를 가진다. 기본 기능은 무료이며, 고급 기능은 상용 라이선스를 요구한다.
