Timeoutмқҳ мӢңк°„ мҙҲкіј мӢң мһҗмӢқ Halt (TimeoutвҖҷs Child Halt on Time Expiration)

1. к°ңмҡ”

Timeout лҚ°мҪ”л Ҳмқҙн„°к°Җ мӢңк°„ мҙҲкіјлҘј к°җм§Җн•ҳл©ҙ, мһҗмӢқ л…ёл“ңмқҳ halt() л©”м„ңл“ңлҘј нҳём¶ңн•ҳм—¬ мӢӨн–ү мӨ‘мқё н–үлҸҷмқ„ к°•м ңлЎң мӨ‘лӢЁмӢңнӮЁлӢӨ. мқҙ halt л©”м»ӨлӢҲмҰҳмқҖ мһҗмӢқмқҙ ліҙмң н•ң лҰ¬мҶҢмҠӨлҘј м•Ҳм „н•ҳкІҢ н•ҙм ңн•ҳкі , л№„лҸҷкё° мһ‘м—…мқ„ м·ЁмҶҢн•ҳл©°, л¬јлҰ¬м Ғ лҸҷмһ‘(лӘЁн„°, к·ёлҰ¬нҚј л“ұ)мқ„ м•Ҳм „н•ң мғҒнғңлЎң м „нҷҳн•ҳлҠ” лҚ° н•„мҲҳм ҒмқҙлӢӨ.

2.2 мҪ”л“ң мҲҳмӨҖмқҳ лҸҷмһ‘

BT::NodeStatus tick() override
{
    if (status() == BT::NodeStatus::IDLE)
    {
        start_time_ = clock_.now();
        setStatus(BT::NodeStatus::RUNNING);
    }

    // мӢңк°„ мҙҲкіј нҷ•мқё
    auto elapsed = clock_.now() - start_time_;
    if (elapsed >= timeout_duration_)
    {
        // мһҗмӢқмқҙ RUNNING мғҒнғңмқҙл©ҙ halt нҳём¶ң
        haltChild();
        return BT::NodeStatus::FAILURE;
    }

    // мӢңк°„ лӮҙ: мһҗмӢқ tick
    auto child_status = child_node_->executeTick();

    if (child_status != BT::NodeStatus::RUNNING)
    {
        // мһҗмӢқмқҙ мҷ„лЈҢлҗЁ: нғҖмқҙлЁё лҰ¬м…Ӣ
        resetTimer();
    }

    return child_status;
}

3. мһҗмӢқ halt()мқҳ мұ…мһ„

3.1 л№„лҸҷкё° мһ‘м—… м·ЁмҶҢ

ROS2 м•Ўм…ҳ м„ңлІ„лҘј мӮ¬мҡ©н•ҳлҠ” мһҗмӢқмқҳ кІҪмҡ°, halt()м—җм„ң м§„н–ү мӨ‘мқё лӘ©н‘ң(goal)лҘј м·ЁмҶҢн•ҳм—¬м•ј н•ңлӢӨ.

void NavigateAction::halt()
{
    if (action_client_ && goal_handle_)
    {
        action_client_->async_cancel_goal(goal_handle_);
    }
    CoroActionNode::halt();
}

лӘЁн„° м ңм–ҙ н–үлҸҷмқҳ кІҪмҡ°, halt()м—җм„ң мҶҚлҸ„ лӘ…л №мқ„ 0мңјлЎң м„Өм •н•ҳм—¬ лЎңлҙҮмқ„ м •м§ҖмӢңмјңм•ј н•ңлӢӨ.

л§ӨлӢҲн“°л Ҳмқҙм…ҳ н–үлҸҷм—җм„ңлҠ” halt мӢң к·ёлҰ¬нҚјмқҳ нҳ„мһ¬ мғҒнғңлҘј мң м§Җн•ҳм—¬м•ј н•ңлӢӨ. л¬јмІҙлҘј мһЎкі  мһҲлҠ” лҸ„мӨ‘ haltк°Җ нҳём¶ңлҗҳл©ҙ, к·ёлҰ¬нҚјлҘј м—ҙм§Җ м•Ҡкі  нҳ„мһ¬ нҢҢм§Җ мғҒнғңлҘј мң м§Җн•ҳлҠ” кІғмқҙ м•Ҳм „н•ҳлӢӨ.

4. haltChild()мҷҖ halt()мқҳ кҙҖкі„

haltChild()лҠ” BehaviorTree.CPPмқҳ DecoratorNodeк°Җ м ңкіөн•ҳлҠ” л©”м„ңл“ңлЎң, мһҗмӢқмқҙ RUNNING мғҒнғңмқё кІҪмҡ°м—җл§Ң мһҗмӢқмқҳ halt()лҘј нҳём¶ңн•ңлӢӨ.

void DecoratorNode::haltChild()
{
    if (child_node_->status() == NodeStatus::RUNNING)
    {
        child_node_->halt();
    }
    child_node_->setStatus(NodeStatus::IDLE);
}

мһҗмӢқмқҙ мқҙлҜё SUCCESS лҳҗлҠ” FAILURE мғҒнғңмқҙл©ҙ haltлҘј нҳём¶ңн•ҳм§Җ м•ҠлҠ”лӢӨ.

5. haltмқҳ л№„м°ЁлӢЁ мӢӨн–ү

halt() л©”м„ңл“ңлҠ” к°ҖлҠҘн•ң н•ң мӢ мҶҚн•ҳкІҢ мҷ„лЈҢлҗҳм–ҙм•ј н•ңлӢӨ. halt() лӮҙл¶Җм—җм„ң мһҘмӢңк°„ мҶҢмҡ”лҗҳлҠ” лҸҷкё° нҳём¶ң(мҳҲ: м„ңл№„мҠӨ мқ‘лӢө лҢҖкё°)мқ„ мҲҳн–үн•ҳл©ҙ м „мІҙ tickмқҙ м°ЁлӢЁлҗңлӢӨ.

6. м„Өкі„ мӢң кі л Ө мӮ¬н•ӯ

6.1 halt лҜёкө¬нҳ„мқҳ мң„н—ҳ

мһҗмӢқ л…ёл“ңк°Җ halt()лҘј мҳ¬л°”лҘҙкІҢ кө¬нҳ„н•ҳм§Җ м•Ҡмңјл©ҙ, Timeoutм—җ мқҳн•ң к°•м ң мӨ‘лӢЁ нӣ„м—җлҸ„ мһҗмӢқмқҳ л№„лҸҷкё° мһ‘м—…мқҙ кі„мҶҚ мӢӨн–үлҗ  мҲҳ мһҲлӢӨ. лӘЁл“  л№„лҸҷкё° м•Ўм…ҳ л…ёл“ңлҠ” halt()м—җм„ң м§„н–ү мӨ‘мқё мһ‘м—…мқ„ м·ЁмҶҢн•ҳлҠ” лЎңм§Ғмқ„ л°ҳл“ңмӢң нҸ¬н•Ён•ҳм—¬м•ј н•ңлӢӨ.

6.2 мӨ‘мІ© Timeoutм—җм„ңмқҳ halt м „нҢҢ

Timeoutмқҙ мӨ‘мІ©лҗң кІҪмҡ°(Timeout лӮҙл¶Җм—җ лӢӨлҘё Timeoutмқҙ мһҲлҠ” кІҪмҡ°), мҷёл¶Җ Timeoutмқҳ haltк°Җ лӮҙл¶Җ TimeoutмңјлЎң м „нҢҢлҗҳкі , лӮҙл¶Җ TimeoutмқҖ лӢӨмӢң мһҗмӢқмңјлЎң haltлҘј м „нҢҢн•ңлӢӨ. мқҙ м—°мҮ„м Ғ halt м „нҢҢк°Җ мҳ¬л°”лҘҙкІҢ мқҙлЈЁм–ҙм§ҖлҠ”м§ҖлҘј кІҖмҰқн•ҳм—¬м•ј н•ңлӢӨ.

6.3 haltмҷҖ мғҒнғң ліҙмЎҙ

halt нӣ„ лҸҷмқјн•ң н–үлҸҷмқ„ мһ¬мӢңмһ‘н•ҳлҠ” кІҪмҡ°, мқҙм „ мӢӨн–үмқҳ мғҒнғңк°Җ м •лҰ¬лҗҳм–ҙ к№ЁлҒ—н•ң мғҒнғңм—җм„ң мӢңмһ‘н•ҳм—¬м•ј н•ңлӢӨ. мһҗмӢқмқҳ halt()м—җм„ң лӮҙл¶Җ мғҒнғңлҘј мҷ„м „нһҲ мҙҲкё°нҷ”н•ҳм—¬м•ј н•ңлӢӨ.
