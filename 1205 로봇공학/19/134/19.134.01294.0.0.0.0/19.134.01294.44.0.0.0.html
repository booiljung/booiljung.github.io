BehaviorTree.CPP v4м—җм„ңлҠ” SKIPPED мғҒнғң мІҳлҰ¬к°Җ м¶”к°ҖлҗңлӢӨ(Faconti, 2022).

function FallbackNode.tick():
    if status() == IDLE:
        skipped_count вҶҗ 0

    setStatus(RUNNING)

    while current_child_idx < N:
        child вҶҗ children[current_child_idx]
        child_status вҶҗ child.executeTick()

        switch child_status:
            case RUNNING:
                return RUNNING

            case SUCCESS:
                resetChildren()
                current_child_idx вҶҗ 0
                return SUCCESS

            case FAILURE:
                current_child_idx вҶҗ current_child_idx + 1

            case SKIPPED:
                current_child_idx вҶҗ current_child_idx + 1
                skipped_count вҶҗ skipped_count + 1

            case IDLE:
                throw LogicError("Child returned IDLE")

    // лЈЁн”„ мў…лЈҢ: лӘЁл“  мһҗмӢқ нҸүк°Җ мҷ„лЈҢ
    current_child_idx вҶҗ 0
    resetChildren()

    if skipped_count == N:
        return SKIPPED
    return FAILURE
