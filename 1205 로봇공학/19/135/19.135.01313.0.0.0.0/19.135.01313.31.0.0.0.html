문제 이름과 도메인 참조 (Problem Name and Domain Reference)

1. 개요

PDDL 문제 파일의 첫 두 요소는 문제 이름 선언과 참조할 도메인의 지정이다. 문제 이름은 문제 인스턴스를 고유하게 식별하며, 도메인 참조는 사용할 행동과 술어를 정의한 도메인 파일을 지정한다.

2. 구문

(define (problem delivery-mission-1)
  (:domain robot-delivery)
  ...)

3. 도메인 참조의 일치 규칙

문제 파일의 (:domain 이름)은 도메인 파일의 (domain 이름)과 정확히 일치하여야 한다. 불일치 시 계획기가 오류를 보고한다.
