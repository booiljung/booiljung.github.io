서비스 로봇에서의 패러다임 선택 기준 (Paradigm Selection Criteria for Service Robots)

1. 개요

서비스 로봇은 가정, 병원, 호텔, 레스토랑 등 비구조적 환경에서 다양한 서비스를 제공하므로, 동적 임무 변경과 인간-로봇 상호작용에 대한 유연한 대응이 요구된다. 자율 계획의 도입이 특히 효과적인 분야이다.

2. 선택 기준

기준하드코딩 적합자율 계획 적합
고정 서비스 (안내)적합불필요
동적 서비스 요청비효율적매우 적합
다양한 물품 배달비효율적적합
환경 변화 빈번규칙 수정 필요재계획으로 대응
사용자 요청 다양모든 패턴 사전 정의 어려움목표로 변환하여 자동 계획

서비스 로봇은 자율 계획이 가장 큰 가치를 제공하는 분야 중 하나이다.
