산업용 로봇에서의 패러다임 선택 기준 (Paradigm Selection Criteria for Industrial Robots)

1. 개요

산업용 로봇은 높은 정밀도와 안전 인증이 요구되므로, 하드코딩이 주류 패러다임이다. 그러나 유연 제조와 혼류 생산의 확산으로 자율 계획의 도입이 점진적으로 확대되고 있다.

2. 선택 기준

기준하드코딩 적합자율 계획 적합
대량 생산, 고정 공정매우 적합불필요
다품종 소량 생산공정 변경 비용 높음적합
안전 인증 필수용이어려움
사이클 타임 정밀보장 가능어려움
빈번한 공정 변경비용 높음적합
