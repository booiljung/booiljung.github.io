진단 및 문제 해결을 위한 패러다임 비교 (Paradigm Comparison for Diagnostics and Troubleshooting)

1. 개요

자율 계획은 로봇 시스템의 진단(diagnostics)과 문제 해결(troubleshooting)에서 체계적인 접근을 제공한다. 하드코딩 기반 진단은 알려진 결함 패턴에 한정되나, 자율 계획 기반 진단은 목표(“시스템을 정상 상태로 복원”)에서 출발하여 필요한 진단 및 복구 행동을 자동으로 도출한다.

2. 비교

알려진 결함사전 정의된 점검 절차자동 점검 계획
미지 결함대응 불가탐색적 진단 계획
복구 순서 최적화수동비용 기반 자동 최적화
