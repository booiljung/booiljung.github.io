인증 및 검증 관점에서의 패러다임 비교 (Paradigm Comparison from Certification and Verification Perspectives)

1. 개요

로봇 시스템의 인증(certification)과 검증(verification)은 안전 규제 준수의 핵심이다. 하드코딩 기반 시스템은 정적 분석과 형식 검증에 적합하나, 자율 계획 시스템은 동적 행동 생성으로 인해 사전 검증이 어렵다.

3. 자율 계획의 인증 접근

  1. 계획 검증: 생성된 계획이 안전 제약을 충족하는지 실행 전 검증
  2. 도메인 모델 검증: PDDL 도메인의 일관성과 안전 속성 검증
  3. 런타임 감시: 행동 트리의 안전 가드로 실행 중 안전 보장
  4. 계획기 인증: 계획기 알고리즘의 soundness 증명
