IPC 벤치마크 도메인 분석 (Analysis of IPC Benchmark Domains)

1. 개요

IPC 벤치마크 도메인은 계획기의 성능을 평가하기 위해 사용되는 표준화된 PDDL 문제 집합이다. 물류, 블록 월드, 엘리베이터, 로버 등 다양한 도메인이 포함되며, 각 도메인은 특정 계획 특성을 테스트한다.
