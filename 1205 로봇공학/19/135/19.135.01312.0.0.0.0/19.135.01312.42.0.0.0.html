플래너 성능 평가와 국제 플래닝 대회 (Planner Performance Evaluation and International Planning Competition)

1. 개요

국제 플래닝 대회(International Planning Competition, IPC)는 1998년부터 격년으로 개최되는 AI 플래닝 분야의 표준 벤치마크 대회로, 계획기 간의 공정한 비교와 알고리즘 발전을 촉진한다. PDDL을 표준 입력 형식으로 사용한다.

3. 평가 트랙

트랙목표
만족 (Satisficing)유효한 계획을 빠르게 생성
최적 (Optimal)최소 비용의 계획 생성
임의 시간 (Anytime)시간에 따라 계획 품질 개선
학습 (Learning)경험에 기반한 성능 향상

4. 평가 지표

  • 해결률 (Coverage): 주어진 시간/메모리 내에 해결한 문제의 비율
  • 계획 비용 (Plan Cost): 생성된 계획의 총 비용
  • IPC 점수: 해결률과 비용의 가중 조합
