협력적 플래닝과 경쟁적 플래닝 (Cooperative and Competitive Planning)

1. 개요

다중 에이전트 플래닝에서 에이전트 간의 관계에 따라 협력적(cooperative) 플래닝과 경쟁적(competitive) 플래닝으로 구분된다.

2. 비교

3. 로봇 공학에서의 협력적 플래닝

물류 로봇, 탐사 로봇, 구조 로봇 등에서 복수의 로봇이 공동 목표를 위해 협력한다. 태스크 할당과 태스크 플래닝의 결합이 핵심이다.
