다중 에이전트 플래닝의 기초 (Foundations of Multi-Agent Planning)

1. 개요

다중 에이전트 플래닝은 복수의 에이전트(로봇)가 공유된 환경에서 각자 또는 협력적으로 목표를 달성하기 위한 행동을 계획하는 확장된 플래닝 프레임워크이다.

2. 접근 방식

접근설명
중앙 집중단일 계획기가 모든 에이전트의 행동을 동시에 계획
분산각 에이전트가 독립적으로 계획하고 조율
합동 계획에이전트 간 통신을 통한 협력적 계획

3. 핵심 도전

  • 계산 복잡도: 에이전트 수에 따라 상태 공간이 지수적으로 증가
  • 충돌 해소: 공유 자원과 물리적 공간에서의 충돌 방지
  • 통신 제약: 에이전트 간 제한된 통신 대역
