시간 플래닝의 개요와 도전 과제 (Overview and Challenges of Temporal Planning)

1. 개요

시간 플래닝(temporal planning)은 행동의 지속 시간, 병렬 실행, 시간 제약(마감, 창, 동시성)을 모델링하는 확장된 계획 프레임워크이다. PDDL 2.1의 듀레이티브 액션(durative action)에 의해 형식화되며, POPF, OPTIC 등의 시간적 계획기가 이를 지원한다.

2. 핵심 특성

  • 행동에 지속 시간(duration)이 부여됨
  • 행동의 시작(at start), 진행(over all), 종료(at end) 시점에서의 조건과 효과
  • 병렬 실행 가능한 행동의 동시 실행

3. 도전 과제

도전설명
계산 복잡도연속 시간에서의 탐색은 이산 시간보다 어려움
동시성 관리병렬 행동 간의 상호 작용 처리
자원 제약시간에 따른 자원 소모 모델링
