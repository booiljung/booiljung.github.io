39.13 병렬 로봇의 동역학 모델링

병렬 로봇(parallel robot)은 여러 다리가 플랫폼을 동시에 지지하는 닫힌 운동 사슬 구조이다. 직렬 로봇과 근본적으로 다른 동역학 특성을 가지며, 이의 체계적 모델링이 학술적 과제이다. 본 절에서는 병렬 로봇의 동역학 모델링을 학술적으로 다룬다.
