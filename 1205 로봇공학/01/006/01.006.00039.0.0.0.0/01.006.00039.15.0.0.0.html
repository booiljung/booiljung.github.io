39.15 인간형 로봇의 전신 동역학 모델링

인간형 로봇(humanoid robot)의 전신 동역학 모델링은 많은 자유도와 복잡한 운동 사슬 구조를 포함하는 학술적 과제이다. 부유 기저, 다중 접촉 등의 특수성을 체계적으로 다루는 방법이 학술적으로 확립되어 있다. 본 절에서는 인간형 로봇의 전신 동역학 모델링을 학술적으로 다룬다.
