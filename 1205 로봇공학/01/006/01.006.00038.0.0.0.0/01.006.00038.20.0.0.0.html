38.20 작업 공간 동역학 기반 힘 제어

힘 제어(force control)는 로봇이 환경과의 상호 작용 힘을 원하는 값으로 유지하는 제어 방식이다. 작업 공간 동역학 모델을 활용한 힘 제어는 학술적으로 엄밀한 프레임워크를 제공한다. 본 절에서는 작업 공간 동역학 기반 힘 제어를 학술적으로 다룬다.
