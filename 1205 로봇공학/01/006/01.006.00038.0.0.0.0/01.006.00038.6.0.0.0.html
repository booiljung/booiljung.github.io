38.6 작업 공간 코리올리 항과 원심력 항의 계산

2.2 자코비안 미분

자코비안 미분 항 \dot{\mathbf{J}}\dot{\vec{q}}이 개입한다.

2.3 최종 표현

작업 공간 비선형 항은 다음과 같이 표현된다.

\mathbf{\mu}(\vec{x}, \dot{\vec{x}}) = \mathbf{J}^{-\top} \mathbf{C}\dot{\vec{q}} - \mathbf{\Lambda} \dot{\mathbf{J}} \dot{\vec{q}}
