32.23 영공간 투영과 여유 자유도 활용

영공간 투영(null space projection)은 여유 자유도 로봇에서 엔드 이펙터 작업에 영향을 주지 않으면서 보조 목적을 달성하는 수학적 기법이다. 여유 자유도 활용의 핵심 도구이며, 현대 로봇 공학의 중요한 학술적 주제이다. 본 절에서는 영공간 투영과 여유 자유도 활용을 다룬다.

1. 영공간 투영의 수학적 기초

1.1 영공간 투영 행렬

영공간 투영 행렬 \mathbf{N}은 다음과 같이 정의된다.

\mathbf{N} = \mathbf{I} - \mathbf{J}^+ \mathbf{J}

여기서 \mathbf{J}^+는 자코비안의 의사 역행렬이다.

32.23.1.2 투영의 특성

\mathbf{N}은 임의의 벡터를 자코비안의 영공간으로 투영한다. \mathbf{J} \mathbf{N} = \mathbf{0}을 만족한다.

32.23.1.3 이상 투영

\mathbf{N}은 이상(idempotent) 행렬이다. \mathbf{N}^2 = \mathbf{N}이 성립한다.

32.23.2 관절 속도의 일반 해

32.23.2.1 일반 해

원하는 엔드 이펙터 속도 \dot{\vec{x}}_d에 대한 관절 속도의 일반 해는 다음과 같다.

\dot{\vec{q}} = \mathbf{J}^+ \dot{\vec{x}}_d + \mathbf{N} \dot{\vec{q}}_0

1.2 최소 노름 해

첫 번째 항은 최소 노름 해로, 엔드 이펙터 속도를 달성하는 가장 작은 노름의 관절 속도이다.

1.3 영공간 성분

두 번째 항은 영공간 성분으로, 엔드 이펙터 속도에 영향을 주지 않는 추가 관절 속도이다.

2. 임의 벡터의 선택

2.1 \dot{\vec{q}}_0의 역할

\dot{\vec{q}}_0은 임의의 벡터로, 보조 목적에 따라 설계된다.

2.2 보조 목적 함수

일반적으로 보조 목적은 스칼라 함수 H(\vec{q})로 표현된다. \dot{\vec{q}}_0은 H의 경사도 방향으로 설정된다.

\dot{\vec{q}}_0 = k \nabla H(\vec{q})

32.23.3.3 게인

k는 보조 작업의 가중치이다.

32.23.4 여유 자유도 활용의 목적

32.23.4.1 관절 한계 회피

관절 한계로부터 멀어지는 방향으로 영공간 운동을 활용한다.

H(\vec{q}) = -\sum_{i=1}^{n} \left(\frac{q_i - q_{i,\text{mid}}}{q_{i,\max} - q_{i,\min}}\right)^2

2.3 특이점 회피

매니퓰러빌리티 지수를 최대화하는 방향으로 영공간 운동을 활용한다.

H(\vec{q}) = w(\vec{q}) = \sqrt{\det(\mathbf{J} \mathbf{J}^\top)}

32.23.4.3 장애물 회피

장애물까지의 거리를 최대화하는 영공간 운동이 가능하다.

32.23.4.4 에너지 최소화

관절 속도의 노름을 최소화하여 에너지를 최소화한다.

32.23.5 Liegeois의 방법

32.23.5.1 Liegeois의 원래 제안

Liegeois는 1977년에 여유 자유도 로봇의 보조 작업 활용을 위한 영공간 투영 방법을 제안했다.

32.23.5.2 수학적 형식

\dot{\vec{q}} = \mathbf{J}^+ \dot{\vec{x}}_d + (\mathbf{I} - \mathbf{J}^+ \mathbf{J}) \nabla H(\vec{q})

2.4 학술적 영향

이 방법은 여유 자유도 로봇 연구의 기반이 되었으며, 이후 다양한 확장과 개선이 이루어졌다.

3. 계층적 제어

3.1 우선 순위 제어

여러 작업이 있는 경우, 우선 순위에 따라 계층적으로 제어한다.

3.2 수학적 표현

n개의 계층 작업이 있는 경우 관절 속도는 다음과 같이 재귀적으로 계산된다.

\dot{\vec{q}}_i = \dot{\vec{q}}_{i-1} + (\mathbf{J}_i \mathbf{N}_{i-1})^+ (\dot{\vec{x}}_{d,i} - \mathbf{J}_i \dot{\vec{q}}_{i-1})

32.23.6.3 우선 순위 유지

저수준 작업이 고수준 작업에 영향을 주지 않는다.

32.23.7 동적 일관성

32.23.7.1 동적 영공간 투영

동역학을 고려한 영공간 투영은 관성 행렬을 활용한다.

\mathbf{N}_{\text{dyn}} = \mathbf{I} - \mathbf{J}^\top (\mathbf{J} \mathbf{M}^{-1} \mathbf{J}^\top)^{-1} \mathbf{J} \mathbf{M}^{-1}

여기서 \mathbf{M}은 관성 행렬이다.

3.3 동적 일관성의 의미

동적 영공간 운동은 동적 효과 측면에서도 주 작업에 영향을 주지 않는다.

3.4 학술적 의의

Khatib의 학술적 연구가 동적 영공간 투영의 기반이 되었다.

의료 로봇은 장애물 회피와 미세 제어를 위해 여유 자유도를 활용한다.

5.1 조건수

영공간 투영 행렬의 조건수가 수치적 안정성에 영향을 미친다.

5.2 감쇠

감쇠 최소 제곱법과 결합하여 수치적 안정성을 확보한다.
