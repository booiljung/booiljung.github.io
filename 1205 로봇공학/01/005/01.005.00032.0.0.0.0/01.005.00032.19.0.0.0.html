32.19 공간 자코비안(Space Jacobian)과 물체 자코비안(Body Jacobian)

공간 자코비안(space Jacobian)과 물체 자코비안(body Jacobian)은 엔드 이펙터 속도를 각각 관성 좌표계와 본체 좌표계에서 표현하는 두 종류의 자코비안이다. 두 자코비안은 Adjoint 변환으로 연결되며, 응용에 따라 선택적으로 활용된다. 본 절에서는 공간 자코비안과 물체 자코비안의 정의와 관계를 다룬다.

1. 공간 자코비안의 정의

1.1 정의

공간 자코비안 \mathbf{J}_s(\vec{q})는 관절 속도를 공간 트위스트(관성 좌표계에서의 엔드 이펙터 트위스트)로 매핑한다.

\vec{V}_s = \mathbf{J}_s(\vec{q}) \dot{\vec{q}}

32.19.1.2 크기

공간 자코비안은 6 \times n 행렬이다.

32.19.1.3 열 구조

각 열은 해당 관절의 스크류를 현재 구성에서 공간 좌표계로 변환한 것이다.

32.19.2 공간 자코비안의 유도

32.19.2.1 POE 순기구학

POE 순기구학은 다음과 같이 표현된다.

\mathbf{T}(\vec{q}) = e^{[\vec{\xi}_1] q_1} e^{[\vec{\xi}_2] q_2} \cdots e^{[\vec{\xi}_n] q_n} \mathbf{M}

1.2 공간 자코비안의 열

공간 자코비안의 i번째 열은 다음과 같다.

\vec{J}_{s,i} = \text{Ad}_{e^{[\vec{\xi}_1] q_1} \cdots e^{[\vec{\xi}_{i-1}] q_{i-1}}} \vec{\xi}_i

32.19.2.3 첫 번째 열

\vec{J}_{s,1} = \vec{\xi}_1이다. 변환이 없으므로 원래 스크류가 그대로이다.

32.19.3 물체 자코비안의 정의

32.19.3.1 정의

물체 자코비안 \mathbf{J}_b(\vec{q})는 관절 속도를 본체 트위스트(엔드 이펙터 좌표계에서의 트위스트)로 매핑한다.

\vec{V}_b = \mathbf{J}_b(\vec{q}) \dot{\vec{q}}

1.3 크기

물체 자코비안도 6 \times n 행렬이다.

1.4 열 구조

각 열은 해당 관절의 스크류를 본체 좌표계로 변환한 것이다.

2. 물체 자코비안의 유도

2.1 역방향 구성

물체 자코비안은 엔드 이펙터로부터 역방향으로 유도된다.

2.2 물체 자코비안의 열

i번째 열은 다음과 같다.

\vec{J}_{b,i} = \text{Ad}_{e^{-[\vec{\xi}_n] q_n} \cdots e^{-[\vec{\xi}_{i+1}] q_{i+1}} \mathbf{M}^{-1}} \vec{\xi}_i

32.19.4.3 마지막 열

\vec{J}_{b,n}은 마지막 관절의 스크류를 본체 좌표계로 직접 변환한 것이다.

32.19.5 두 자코비안의 관계

32.19.5.1 Adjoint 변환

공간 자코비안과 물체 자코비안은 Adjoint 변환으로 연결된다.

\mathbf{J}_s = \text{Ad}_{\mathbf{T}_{sb}} \mathbf{J}_b

2.3 역관계

역관계는 다음과 같다.

\mathbf{J}_b = \text{Ad}_{\mathbf{T}_{sb}^{-1}} \mathbf{J}_s

32.19.5.3 일관성

두 자코비안이 동일한 물리적 상황을 서로 다른 좌표계에서 표현하므로, 일관된 결과를 제공한다.

32.19.6 트위스트의 좌표계 변환

32.19.6.1 공간 트위스트에서 본체 트위스트로

\vec{V}_b = \text{Ad}_{\mathbf{T}_{sb}^{-1}} \vec{V}_s

2.4 본체 트위스트에서 공간 트위스트로

\vec{V}_s = \text{Ad}_{\mathbf{T}_{sb}} \vec{V}_b

32.19.6.3 계산의 간결성

Adjoint 변환은 효율적으로 구현 가능하다.

32.19.7 응용에 따른 선택

32.19.7.1 공간 자코비안의 활용

외부 환경과의 상호 작용이 관성 좌표계에서 기술되는 경우 공간 자코비안이 자연스럽다. 예를 들어, 세계 좌표계에서의 장애물 회피이다.

32.19.7.2 물체 자코비안의 활용

엔드 이펙터의 관점에서 기술되는 작업(예: 도구 회전, 촉각 센서의 힘 측정)은 물체 자코비안이 편리하다.

32.19.7.3 제어 설계

임피던스 제어, 힘 제어 등의 설계에서 좌표계 선택이 중요하다.

32.19.8 DH 기반 자코비안과의 관계

32.19.8.1 동등성

DH 기반 자코비안과 스크류 기반 공간/물체 자코비안은 수학적으로 동등하다.

32.19.8.2 관점의 차이

DH는 링크 중심, 스크류는 관절 중심의 관점이다.

32.19.8.3 실무적 변환

실무적으로 DH 기반 자코비안을 계산한 후 Adjoint 변환으로 공간/물체 자코비안을 얻을 수 있다.

32.19.9 특이점 분석

32.19.9.1 공간 자코비안의 특이점

공간 자코비안의 계수 결핍이 관성 좌표계에서의 운동 능력 상실을 의미한다.

32.19.9.2 물체 자코비안의 특이점

물체 자코비안의 계수 결핍이 본체 좌표계에서의 운동 능력 상실을 의미한다.

32.19.9.3 동등성

두 자코비안의 특이점 구성은 동일하다. 단, 기하학적 해석이 다를 뿐이다.

32.19.10 학술적 활용

본 절에서 다룬 공간 자코비안과 물체 자코비안은 현대 로봇 공학의 중요한 학술적 도구이다. 두 자코비안의 유연한 활용이 복잡한 로봇 작업의 효과적 분석과 제어를 가능하게 한다.
