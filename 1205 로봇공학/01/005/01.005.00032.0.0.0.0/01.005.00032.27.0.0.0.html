32.27 매니퓰러빌리티 지수(Manipulability Index)의 정의

매니퓰러빌리티 지수(manipulability index)는 Yoshikawa가 1985년에 제안한 스칼라 지표로, 로봇의 국소 운동 능력을 정량화한다. 이 지수는 매니퓰러빌리티 타원체의 부피와 비례하며, 로봇 분석과 제어의 학술적 기준이 된다. 본 절에서는 매니퓰러빌리티 지수의 정의를 다룬다.

1. Yoshikawa의 매니퓰러빌리티 지수

1.1 정의

Yoshikawa의 매니퓰러빌리티 지수는 다음과 같이 정의된다.

w(\vec{q}) = \sqrt{\det(\mathbf{J}(\vec{q}) \mathbf{J}^\top(\vec{q}))}

32.27.1.2 기하학적 의미

이 지수는 매니퓰러빌리티 타원체의 부피와 비례한다. 부피가 큰 구성일수록 매니퓰러빌리티가 크다.

32.27.1.3 특이값 곱

특이값 분해를 활용하면 다음과 같이 표현된다.

w(\vec{q}) = \prod_{i=1}^{m} \sigma_i

여기서 \sigma_i는 자코비안의 특이값이다.

2. 지수의 특성

2.1 비음수성

w(\vec{q}) \geq 0이다.

2.2 특이점에서의 값

w = 0인 구성이 특이점이다.

2.3 최대값

w가 최대가 되는 구성이 매니퓰러빌리티가 가장 큰 구성이다.

3. 정사각 자코비안의 경우

3.1 절대값

정사각 자코비안(m = n)의 경우 지수는 행렬식의 절대값이다.

w(\vec{q}) = |\det(\mathbf{J}(\vec{q}))|

32.27.3.2 해석

행렬식은 자코비안이 단위 볼륨을 얼마나 변환하는지의 척도이다.

32.27.3.3 수치적 계산

정사각 행렬의 행렬식 계산은 일반적인 SVD보다 효율적이다.

32.27.4 여유 자유도 로봇의 경우

32.27.4.1 비정사각 자코비안

여유 자유도 로봇(n > m)의 자코비안은 비정사각이다.

32.27.4.2 Gram 행렬식

이 경우 \det(\mathbf{J} \mathbf{J}^\top)은 Gram 행렬식이다.

32.27.4.3 일반화

Yoshikawa 지수는 정사각과 비정사각 모두에 적용 가능한 일반적 정의이다.

32.27.5 단위와 차원

32.27.5.1 차원

매니퓰러빌리티 지수의 차원은 로봇 구조에 따라 다르다. 6자유도 로봇의 경우 [m]^3 [rad]^3의 차원을 가진다.

32.27.5.2 혼합 단위

선속도(m/s)와 각속도(rad/s)의 혼합으로 인한 단위 문제가 있다.

32.27.5.3 정규화

비교를 위해 종종 무차원화된 매니퓰러빌리티 지수가 활용된다.

32.27.6 분리된 매니퓰러빌리티 지수

32.27.6.1 선속도 매니퓰러빌리티

선속도 자코비안 \mathbf{J}_v의 매니퓰러빌리티 지수는 다음과 같다.

w_v = \sqrt{\det(\mathbf{J}_v \mathbf{J}_v^\top)}

3.2 각속도 매니퓰러빌리티

각속도 자코비안 \mathbf{J}_\omega의 매니퓰러빌리티 지수는 다음과 같다.

w_\omega = \sqrt{\det(\mathbf{J}_\omega \mathbf{J}_\omega^\top)}

32.27.6.3 분리의 장점

단위 문제가 해소되며, 각 작업의 능력을 개별적으로 평가할 수 있다.

32.27.7 대안 지수

32.27.7.1 조건수 역수

조건수의 역수도 매니퓰러빌리티 지표로 활용된다.

w_\kappa = \frac{\sigma_{\min}}{\sigma_{\max}} = \frac{1}{\kappa}

3.3 최소 특이값

최소 특이값 \sigma_{\min}은 가장 약한 방향의 능력을 정량화한다.

3.4 기타 지표

다양한 지표(equivalent dexterous workspace, isotropic index 등)가 제안되어 있다.

4. 응용

4.1 로봇 설계

매니퓰러빌리티 지수는 로봇 설계의 성능 기준이 된다.

4.2 경로 계획

매니퓰러빌리티가 큰 구성을 따라 경로를 계획한다.

4.3 여유 자유도 활용

영공간 운동을 통해 매니퓰러빌리티를 최대화한다.

5. 경사도

5.1 지수의 경사도

매니퓰러빌리티 지수의 경사도 \nabla w(\vec{q})는 여유 자유도 활용에서 중요하다.

5.2 계산

수치 미분 또는 해석적 미분으로 경사도를 계산한다.

5.3 최적화

경사도를 활용하여 매니퓰러빌리티를 최대화하는 영공간 운동을 생성한다.

6. 학술적 활용

본 절에서 다룬 매니퓰러빌리티 지수의 정의는 로봇의 국소 운동 능력을 정량화하는 학술적 표준이다. 로봇 설계, 제어, 경로 계획의 다양한 학술적·실무적 분야에서 광범위하게 활용된다.
