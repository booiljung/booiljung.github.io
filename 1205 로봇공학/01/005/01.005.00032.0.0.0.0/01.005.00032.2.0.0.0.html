32.2 관절 공간 속도와 작업 공간 속도의 관계

관절 공간 속도와 작업 공간 속도의 관계는 속도 기구학의 핵심 주제로, 로봇의 두 기본적 상태 공간 사이의 수학적 매핑을 정립한다. 이 관계는 자코비안 행렬을 통해 선형적으로 표현되며, 로봇 제어와 분석의 기반이 된다. 본 절에서는 두 공간의 속도 관계를 학술적으로 다룬다.

1. 관절 공간과 작업 공간

1.1 관절 공간

관절 공간(joint space)은 로봇의 모든 관절 변수로 구성된 공간이다. 자유도 n의 로봇에 대해 관절 공간은 \mathbb{R}^n이다.

\vec{q} = [q_1, q_2, \ldots, q_n]^\top \in \mathbb{R}^n

32.2.1.2 작업 공간

작업 공간(task space, operational space)은 엔드 이펙터의 자세가 존재하는 공간이다. 6자유도 작업의 경우 작업 공간은 SE(3) 또는 \mathbb{R}^6(매개변수화 후)이다.

\vec{x} = \begin{bmatrix} \vec{p} \\ \vec{\phi} \end{bmatrix} \in \mathbb{R}^6

여기서 \vec{p}는 위치, \vec{\phi}는 자세(오일러 각 등)이다.

1.2 차원 차이

관절 공간과 작업 공간의 차원은 일반적으로 같지 않다. 6자유도 매니퓰레이터는 n = 6, 7자유도 여유 자유도 로봇은 n = 7이다.

2. 순기구학 관계

2.1 순기구학 함수

순기구학은 관절 공간으로부터 작업 공간으로의 비선형 매핑이다.

\vec{x} = \vec{f}(\vec{q}), \quad \vec{f}: \mathbb{R}^n \to SE(3)

32.2.2.2 비선형성

순기구학 함수 \vec{f}는 일반적으로 비선형이다. 회전 변환의 비선형 특성에 기인한다.

32.2.2.3 시간 미분

관절 변수가 시간의 함수일 때 \vec{q} = \vec{q}(t), 작업 공간 변수도 시간의 함수 \vec{x} = \vec{x}(t)가 된다.

32.2.3 속도 관계의 유도

32.2.3.1 연쇄 법칙

작업 공간 변수의 시간 미분은 연쇄 법칙에 의해 다음과 같다.

\dot{\vec{x}} = \frac{d\vec{x}}{dt} = \frac{\partial \vec{f}}{\partial \vec{q}} \frac{d\vec{q}}{dt} = \mathbf{J}(\vec{q}) \dot{\vec{q}}

2.2 자코비안 행렬

\mathbf{J}(\vec{q}) = \partial \vec{f}/\partial \vec{q}는 자코비안 행렬이며, 관절 공간 속도를 작업 공간 속도로 매핑하는 선형 변환이다.

2.3 선형성

관절 속도와 작업 공간 속도의 관계는 특정 구성 \vec{q}에서 자코비안을 통한 선형 매핑이다. 단, 자코비안 자체는 \vec{q}의 비선형 함수이다.

3. 자코비안의 차원

3.1 일반적 크기

자코비안 \mathbf{J}(\vec{q})는 m \times n 행렬이다. 여기서 m은 작업 공간의 차원, n은 관절 공간의 차원이다.

3.2 6자유도 매니퓰레이터

6자유도 매니퓰레이터의 경우 m = n = 6이며, 자코비안은 6 \times 6 정사각 행렬이다.

3.3 여유 자유도 로봇

7자유도 여유 자유도 로봇의 경우 m = 6, n = 7이며, 자코비안은 6 \times 7 비정사각 행렬이다.

4. 속도 매핑의 기하학적 해석

4.1 접공간 매핑

자코비안은 관절 공간의 접공간(tangent space)으로부터 작업 공간의 접공간으로의 선형 매핑이다.

4.2 국소 선형화

자코비안은 특정 구성 \vec{q}_0에서의 순기구학 함수의 국소 선형화이다.

\vec{x} \approx \vec{f}(\vec{q}_0) + \mathbf{J}(\vec{q}_0)(\vec{q} - \vec{q}_0)

32.2.5.3 자코비안의 종속성

자코비안은 현재 관절 구성에 의존한다. 따라서 로봇이 움직이면 자코비안도 변화한다.

32.2.6 역속도 매핑

32.2.6.1 역자코비안

작업 공간 속도로부터 관절 공간 속도를 계산하는 역매핑은 자코비안의 역행렬(정사각 자코비안) 또는 의사 역행렬(비정사각 자코비안)로 표현된다.

\dot{\vec{q}} = \mathbf{J}^{-1}(\vec{q}) \dot{\vec{x}}

4.3 여유 자유도의 경우

여유 자유도 로봇의 경우 무한한 해가 존재하며, 의사 역행렬이 최소 노름 해를 제공한다.

\dot{\vec{q}} = \mathbf{J}^+(\vec{q}) \dot{\vec{x}} + (\mathbf{I} - \mathbf{J}^+ \mathbf{J}) \dot{\vec{q}}_0

32.2.6.3 자유도 부족의 경우

자유도 부족 로봇(m > n)의 경우 해가 존재하지 않을 수 있으며, 최소 제곱 해가 활용된다.

32.2.7 관절 공간과 작업 공간 제어

32.2.7.1 관절 공간 제어

관절 공간 제어는 관절 변수를 직접 제어하는 방식이다. 각 관절의 위치와 속도 제어기가 활용된다.

32.2.7.2 작업 공간 제어

작업 공간 제어는 엔드 이펙터의 자세를 직접 제어하는 방식이다. 자코비안 기반 매핑을 통해 관절 명령으로 변환한다.

32.2.7.3 하이브리드 제어

실무적으로 두 제어 방식이 결합되어 활용된다. 엔드 이펙터 경로는 작업 공간에서 계획되고, 관절 명령은 관절 공간에서 실행된다.

32.2.8 속도 매핑의 수치적 계산

32.2.8.1 자코비안의 구성

자코비안은 각 관절에 대한 순기구학의 편미분으로 구성된다. 효율적 알고리즘으로 실시간 계산이 가능하다.

32.2.8.2 선형 연립 방정식

역속도 기구학은 선형 연립 방정식을 푸는 문제로 변환된다. LU 분해, SVD 등의 수치적 방법이 활용된다.

32.2.8.3 실시간 제어

실시간 제어에서 자코비안과 그 역은 수 kHz의 주기로 계산된다.

32.2.9 속도 매핑의 특이점

32.2.9.1 특이 구성

자코비안의 계수가 감소하는 구성이 특이점이다. 이 경우 역속도 기구학이 제대로 작동하지 않는다.

32.2.9.2 기하학적 해석

특이점에서 엔드 이펙터가 특정 방향으로 운동할 수 없게 된다.

32.2.9.3 실무적 처리

특이점 근방에서는 감쇠 최소 제곱법 등의 수치적 안정화 기법이 활용된다.

32.2.10 학술적 활용

본 절에서 다룬 관절 공간 속도와 작업 공간 속도의 관계는 자코비안 이론과 속도 기구학의 핵심 학술적 내용이다. 이 관계는 로봇 제어, 경로 계획, 동역학 등의 후속 학술적 주제의 기반이 된다.
