32.14 선속도 자코비안과 각속도 자코비안의 분리

기하학적 자코비안은 상단 3행과 하단 3행으로 분리하여 선속도 자코비안과 각속도 자코비안으로 구별하여 다룰 수 있다. 이 분리는 위치 제어와 자세 제어를 독립적으로 처리하거나, 일부 작업만을 제어하는 경우에 유용하다. 본 절에서는 두 자코비안의 분리와 활용을 다룬다.

1. 블록 분할

1.1 자코비안의 블록 표현

기하학적 자코비안은 다음과 같이 두 블록으로 분할된다.

\mathbf{J} = \begin{bmatrix} \mathbf{J}_v \\ \mathbf{J}_\omega \end{bmatrix}

32.14.1.2 선속도 자코비안

상단 3 \times n 블록 \mathbf{J}_v는 선속도 자코비안이다. 관절 속도를 엔드 이펙터 선속도로 매핑한다.

\vec{v} = \mathbf{J}_v(\vec{q}) \dot{\vec{q}}

1.2 각속도 자코비안

하단 3 \times n 블록 \mathbf{J}_\omega는 각속도 자코비안이다. 관절 속도를 엔드 이펙터 각속도로 매핑한다.

\vec{\omega} = \mathbf{J}_\omega(\vec{q}) \dot{\vec{q}}

32.14.2 관절 유형별 분리

32.14.2.1 회전 관절

회전 관절 i의 선속도 자코비안 열은 \hat{z}_{i-1} \times (\vec{p}_n - \vec{p}_{i-1})이고, 각속도 자코비안 열은 \hat{z}_{i-1}이다.

32.14.2.2 직동 관절

직동 관절 i의 선속도 자코비안 열은 \hat{z}_{i-1}이고, 각속도 자코비안 열은 영 벡터이다.

32.14.2.3 통합 표현

두 자코비안 열은 별도로 계산되어 각각의 블록에 배치된다.

32.14.3 분리의 기하학적 의미

32.14.3.1 선속도 자코비안

선속도 자코비안은 관절 속도로부터 엔드 이펙터 원점의 선속도로의 매핑을 나타낸다. 이는 위치 제어의 핵심이다.

32.14.3.2 각속도 자코비안

각속도 자코비안은 관절 속도로부터 엔드 이펙터의 각속도로의 매핑을 나타낸다. 이는 자세 제어의 핵심이다.

32.14.3.3 독립성

두 자코비안은 수학적으로 독립적으로 다룰 수 있다. 단, 실제 로봇의 관절 속도는 두 속도에 동시에 기여한다.

32.14.4 위치 제어에의 활용

32.14.4.1 위치 제어 전용

위치 제어만이 필요한 경우(예: 점 접촉 작업), 선속도 자코비안만 활용한다.

\dot{\vec{q}}_v = \mathbf{J}_v^+ \vec{v}_d

여기서 \vec{v}_d는 원하는 선속도, \mathbf{J}_v^+는 선속도 자코비안의 의사 역행렬이다.

1.3 3자유도 작업

위치만 제어되는 3자유도 작업에서 선속도 자코비안은 3 \times n 행렬이다.

1.4 여유 자유도

n > 3이면 여유 자유도가 있으며, 선속도 자코비안의 영공간이 비자명하다.

2. 자세 제어에의 활용

2.1 자세 제어 전용

자세 제어만이 필요한 경우(예: 카메라 지향, 안테나 지향), 각속도 자코비안만 활용한다.

\dot{\vec{q}}_\omega = \mathbf{J}_\omega^+ \vec{\omega}_d

32.14.5.2 3자유도 자세

자세만 제어되는 3자유도 작업에서 각속도 자코비안은 3 \times n 행렬이다.

32.14.5.3 결합된 제어

위치와 자세가 모두 필요한 경우 두 자코비안을 통합하여 전체 자코비안으로 다룬다.

32.14.6 분리된 특이점 분석

32.14.6.1 선속도 특이점

\mathbf{J}_v의 계수 결핍은 위치 제어의 특이점을 야기한다. 특정 방향으로의 이동이 불가능해진다.

32.14.6.2 각속도 특이점

\mathbf{J}_\omega의 계수 결핍은 자세 제어의 특이점을 야기한다. 특정 축 주위의 회전이 불가능해진다.

32.14.6.3 결합 특이점

전체 자코비안의 특이점은 두 서브 자코비안의 특이점 또는 두 행 공간의 결합에서 발생한다.

32.14.7 분리된 매니퓰러빌리티

32.14.7.1 선속도 매니퓰러빌리티

선속도 자코비안의 매니퓰러빌리티는 위치 제어 능력을 정량화한다.

w_v = \sqrt{\det(\mathbf{J}_v \mathbf{J}_v^\top)}

2.2 각속도 매니퓰러빌리티

각속도 자코비안의 매니퓰러빌리티는 자세 제어 능력을 정량화한다.

w_\omega = \sqrt{\det(\mathbf{J}_\omega \mathbf{J}_\omega^\top)}

32.14.7.3 단위 차이

두 매니퓰러빌리티는 차원이 다르므로(길이 vs 무차원) 직접 비교가 어렵다. 이는 실무적 고려 사항이다.

32.14.8 부분 작업 제어

32.14.8.1 작업 우선 순위

일부 작업(예: 엔드 이펙터 위치)을 우선하고 나머지를 보조 작업으로 처리하는 계층적 제어가 가능하다.

32.14.8.2 투영 행렬

주 작업의 영공간으로 보조 작업을 투영하여, 주 작업에 영향을 주지 않으면서 보조 작업을 수행한다.

32.14.8.3 여유 자유도 활용

여유 자유도 로봇에서 이러한 계층적 제어가 특히 유용하다.

32.14.9 실무적 구현

32.14.9.1 계산 효율

분리된 자코비안은 각각의 작업에 맞추어 효율적으로 계산할 수 있다.

32.14.9.2 제어 알고리즘

각종 제어 알고리즘에서 분리된 자코비안의 활용이 실무적이다.

32.14.9.3 모듈화

선속도 제어 모듈과 각속도 제어 모듈을 분리하여 시스템의 유지 보수성을 향상시킬 수 있다.

32.14.10 학술적 활용

본 절에서 다룬 선속도 자코비안과 각속도 자코비안의 분리는 로봇 제어의 다양한 응용에서 활용된다. 특히 특정 작업만을 제어하거나 계층적 제어를 설계하는 경우에 학술적·실무적 도구로 활용된다.
