32.8 직동 관절의 기하학적 자코비안 열 벡터

직동 관절의 기하학적 자코비안 열 벡터는 직동 관절이 엔드 이펙터의 선속도에만 기여하고 각속도에는 기여하지 않는 특성을 반영한다. 회전 관절에 비해 구조가 단순하며, 직교 좌표 매니퓰레이터와 SCARA 로봇 등에서 활용된다. 본 절에서는 직동 관절의 자코비안 열 벡터의 유도와 해석을 다룬다.

1. 직동 관절의 운동학적 기여

1.1 순수 이동

직동 관절 i는 관절 축 \hat{z}_{i-1} 방향의 순수 이동만을 허용한다. 관절 변수는 이동 변위 d_i이다.

1.2 선속도 기여

관절 i가 단위 선속도로 움직일 때 엔드 이펙터의 선속도는 \hat{z}_{i-1} 방향이다.

\vec{v}|_{\dot{d}_i = 1} = \hat{z}_{i-1}

32.8.1.3 각속도 기여

직동 관절은 회전을 발생시키지 않으므로 엔드 이펙터의 각속도 기여가 0이다.

\vec{\omega}|_{\dot{d}_i = 1} = \vec{0}

2. 열 벡터의 최종 형식

2.1 6-벡터 형식

직동 관절 i의 기하학적 자코비안 열 벡터는 다음과 같다.

\vec{J}_i = \begin{bmatrix} \vec{J}_{v,i} \\ \vec{J}_{\omega,i} \end{bmatrix} = \begin{bmatrix} \hat{z}_{i-1} \\ \vec{0} \end{bmatrix}

32.8.2.2 상단 3행

상단 3행 \vec{J}_{v,i} = \hat{z}_{i-1}은 관절 축 방향의 단위 벡터이다.

32.8.2.3 하단 3행

하단 3행 \vec{J}_{\omega,i} = \vec{0}은 영 벡터이다.

32.8.3 회전 관절과의 비교

32.8.3.1 구조적 차이

회전 관절은 각속도와 선속도에 모두 기여하는 반면, 직동 관절은 선속도에만 기여한다.

32.8.3.2 기하학적 정보

직동 관절의 열 벡터는 관절 축 방향만 알면 계산되며, 관절 위치나 엔드 이펙터 위치가 필요하지 않다.

32.8.3.3 계산의 단순성

직동 관절의 열 벡터 계산은 회전 관절보다 단순하다.

32.8.4 필요한 기하학적 정보

32.8.4.1 관절 축 방향

\hat{z}_{i-1}은 관절 i의 이동 축 방향 단위 벡터이다. 순기구학의 동차 변환 행렬의 z열로부터 추출된다.

32.8.4.2 외적의 불필요

회전 관절과 달리 외적 계산이 필요하지 않다.

32.8.4.3 관절 위치의 불필요

관절 i의 위치 정보는 자코비안 열 벡터 계산에 필요하지 않다.

32.8.5 좌표계 선택

32.8.5.1 기저 좌표계 표현

자코비안 열 벡터는 일반적으로 기저 좌표계에서 표현된다. 이 경우 \hat{z}_{i-1}은 기저 좌표계에서의 관절 축 방향이다.

32.8.5.2 본체 좌표계 표현

본체 좌표계에서의 표현도 가능하다.

32.8.5.3 변환

기저 좌표계와 본체 좌표계 사이의 변환은 회전 행렬로 이루어진다.

32.8.6 순기구학과의 연결

32.8.6.1 축 방향의 추출

\hat{z}_{i-1}은 순기구학 계산 과정에서 각 좌표계의 동차 변환 행렬의 z열로 자동 계산된다.

32.8.6.2 재귀적 계산

재귀적 순기구학 계산 중에 자코비안 열 벡터가 동시에 계산된다.

32.8.6.3 효율성

계산이 단순하므로 실시간 제어에 적합하다.

32.8.7 기하학적 해석

32.8.7.1 방향 벡터

직동 관절의 자코비안 열 벡터는 관절 축의 방향 벡터이다.

32.8.7.2 선형 속도

이 방향 벡터는 엔드 이펙터가 해당 관절에 의해 발생하는 선형 속도의 방향이다.

32.8.7.3 회전 영향 없음

직동 관절은 엔드 이펙터의 자세를 변경하지 않으므로 각속도 기여가 0이다.

32.8.8 특수 상황

32.8.8.1 여러 직동 관절

여러 직동 관절이 있는 경우, 각각의 자코비안 열 벡터는 해당 관절 축 방향이다.

32.8.8.2 평행 관절

두 직동 관절의 축이 평행한 경우, 두 열 벡터가 선형 종속이 된다. 이는 특이점을 야기할 수 있다.

32.8.8.3 직교 관절

세 직동 관절의 축이 서로 직교하는 경우(직교 좌표 매니퓰레이터), 세 열 벡터가 완전 선형 독립이며 최적의 작업 공간 커버를 제공한다.

32.8.9 예시

32.8.9.1 직교 좌표 매니퓰레이터

3자유도 직교 좌표 매니퓰레이터의 기하학적 자코비안의 선속도 부분은 단위 행렬이다.

\mathbf{J}_v = \mathbf{I}_{3 \times 3}

각속도 부분은 영 행렬이다.

2.2 SCARA 로봇

SCARA 로봇의 직동 관절(수직 축)의 자코비안 열 벡터는 [0, 0, 1, 0, 0, 0]^\top(기저 좌표계에서)이다.

2.3 혼합 구조

회전 관절과 직동 관절이 혼합된 로봇의 자코비안은 각 관절 유형별 열 벡터를 조합하여 구성된다.

3. 학술적 활용

본 절에서 다룬 직동 관절의 기하학적 자코비안 열 벡터는 직교 좌표 매니퓰레이터, SCARA 로봇, 혼합 구조 매니퓰레이터의 기구학 분석에 직접 활용된다. 단순한 구조로 인해 교육적 가치와 실무적 유용성을 모두 제공한다.
