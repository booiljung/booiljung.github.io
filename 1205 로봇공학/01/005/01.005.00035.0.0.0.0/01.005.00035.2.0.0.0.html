35.2 병렬 기구와 직렬 기구의 비교

병렬 기구와 직렬 기구의 비교는 두 로봇 구조의 학술적·실무적 특성을 대비시키는 주요 주제이다. 각각의 장단점을 이해하면 응용에 맞는 로봇 선택이 가능하다. 본 절에서는 병렬 기구와 직렬 기구를 체계적으로 비교한다.

1. 구조적 차이

1.1 개방 vs 폐쇄

직렬 기구는 개방 체인, 병렬 기구는 폐쇄 루프이다.

1.2 체인 수

직렬은 하나의 체인, 병렬은 여러 체인이다.

1.3 구동기 배치

직렬은 각 관절에, 병렬은 기저 근방에 배치된다.

2. 작업 공간

2.1 크기

직렬이 일반적으로 더 크다.

2.2 형태

직렬은 단순한 형태, 병렬은 복잡한 형태이다.

2.3 부피 비율

병렬 기구의 부피 비율이 낮다.

3. 강성

3.1 직렬

직렬은 링크 끝으로 갈수록 강성이 감소한다.

3.2 병렬

병렬은 구조적으로 높은 강성을 가진다.

3.3 정밀 작업

정밀 작업에 병렬이 유리하다.

4. 정밀도

4.1 오차 누적

직렬은 오차가 누적된다.

4.2 오차 평균화

병렬은 오차 평균화로 높은 정밀도이다.

4.3 절대 정밀도

병렬의 절대 정밀도가 우수하다.

5. 속도와 가속도

5.1 병렬의 장점

구동기가 기저에 고정되어 고속 운동이 가능하다.

5.2 직렬의 한계

말단 관절 구동기의 부담이 크다.

5.3 픽앤플레이스

고속 픽앤플레이스에 병렬이 유리하다.

6. 하중 지지

6.1 분산 부하

병렬은 하중을 여러 다리에 분산한다.

6.2 자체 중량 비율

병렬의 자체 중량 대비 하중 비율이 높다.

6.3 중량물 운반

중량물 운반에 병렬이 유리하다.

7. 기구학

7.1 순기구학

직렬은 쉽고 병렬은 어렵다.

7.2 역기구학

병렬은 쉽고 직렬은 복잡할 수 있다.

7.3 대칭적

이러한 대칭적 특성이 흥미롭다.

8. 특이점

8.1 직렬의 특이점

세 가지 주요 유형(어깨, 팔꿈치, 손목)이다.

8.2 병렬의 특이점

Type I, II, III의 세 가지 유형이다.

8.3 복잡성

병렬의 특이점 분석이 더 복잡하다.

9. 응용 분야

9.1 직렬 적합

도달 범위가 중요한 작업이다.

9.2 병렬 적합

정밀 위치 결정, 고속 작업, 비행 시뮬레이터이다.

9.3 상호 보완

두 구조가 상호 보완적이다.

10. 학술적 활용

본 절에서 다룬 병렬 기구와 직렬 기구의 비교는 로봇 선택과 설계의 학술적·실무적 기준이다. 두 구조의 이해가 효과적 로봇 활용의 기반이다.
