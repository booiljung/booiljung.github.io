34.20 병렬 기구의 작업 공간 해석

병렬 기구의 작업 공간 해석은 직렬 매니퓰레이터와 구조적으로 다른 학술적 접근을 요구한다. 폐쇄 루프 구조와 특이점의 복잡성이 작업 공간 해석의 어려움을 가중시킨다. 본 절에서는 병렬 기구의 작업 공간 해석을 다룬다.

1.3 특이점의 영향
