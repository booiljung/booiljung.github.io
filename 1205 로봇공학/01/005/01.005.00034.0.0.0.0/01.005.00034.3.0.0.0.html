34.3 능숙 작업 공간 (Dexterous Workspace)

능숙 작업 공간(dexterous workspace)은 엔드 이펙터가 임의의 자세를 취할 수 있는 위치의 집합으로, 도달 가능 작업 공간의 부분 집합이다. 자세의 자유도까지 포함한 작업 수행 능력을 정량화하며, 복잡한 조작 작업의 수행 가능성을 결정한다. 본 절에서는 능숙 작업 공간을 다룬다.
