34.29 장애물 환경에서의 유효 작업 공간

장애물 환경에서의 유효 작업 공간(effective workspace in obstacle environments)은 자유 공간의 이상적 작업 공간이 장애물에 의해 축소된 실제 운용 영역이다. 실무적 로봇 운용의 현실적 작업 공간 평가에 필수적이다. 본 절에서는 장애물 환경에서의 유효 작업 공간을 다룬다.
