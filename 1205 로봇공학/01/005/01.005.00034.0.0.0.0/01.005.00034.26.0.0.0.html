34.26 협동 로봇의 공유 작업 공간 설계

협동 로봇(collaborative robot, cobot)의 공유 작업 공간 설계는 인간과 로봇이 안전하게 협력하는 환경을 구축하는 학술적·실무적 과제이다. 안전 요구사항과 작업 효율성을 균형 있게 고려해야 한다. 본 절에서는 협동 로봇의 공유 작업 공간 설계를 다룬다.

1.1 안전 우선

협동 로봇은 인간 안전이 최우선이다.

1.2 저속·저력

일반적으로 속도와 힘이 제한된다.

1.3 다양한 센서

접촉 센서, 비전 센서 등이 장착된다.

2. 공유 작업 공간의 개념

2.1 인간-로봇 공유

인간과 로봇이 같은 공간에서 작업한다.

2.2 상호작용

동적 상호작용이 가능하다.

2.3 안전 영역

안전 영역과 작업 영역이 설계된다.
