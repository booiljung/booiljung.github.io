19.7 베르누이 방정식과 에너지 보존

1. 개요

베르누이 방정식은 정상 상태의 비점성 비압축성 유동에서 유선을 따라 에너지의 특정 조합이 보존됨을 기술하는 관계식이며, 유체역학의 가장 널리 사용되는 공학적 해석 도구 중 하나이다. 이 방정식은 다니엘 베르누이가 1738년 저작 Hydrodynamica에서 제시하였으며, 유체의 압력, 속도, 위치 에너지 사이의 교환 관계를 간결하게 기술한다. 본 절은 베르누이 방정식의 유도, 가정과 적용 범위, 에너지 해석, 일반화된 에너지 방정식, 펌프와 터빈이 포함된 확장, 손실을 고려한 확장, 로봇 공학적 응용을 학술적으로 정리한다.

2. 베르누이 방정식의 유도

베르누이 방정식은 정상 상태, 비점성, 비압축성, 유선을 따라의 조건 하에서 오일러 방정식을 유선 방향으로 적분함으로써 유도된다. 유선 좌표에서 오일러 방정식의 운동량 성분은 다음과 같다.

\rho\,u\,\frac{du}{ds} + \frac{dp}{ds} + \rho g\,\frac{dz}{ds} = 0

이를 유선을 따라 적분하면 다음의 베르누이 방정식이 유도된다.

\frac{1}{2}\rho u^2 + p + \rho g z = \text{const}

이 방정식은 유선 상의 임의의 두 점 사이에서 다음과 같이 표현된다.

\frac{1}{2}\rho u_1^2 + p_1 + \rho g z_1 = \frac{1}{2}\rho u_2^2 + p_2 + \rho g z_2

각 항은 운동 에너지, 유동 일(압력), 위치 에너지의 단위 부피당 값을 나타낸다.

가정과 적용 범위

베르누이 방정식의 유도에는 여러 가정이 사용되며, 이를 벗어난 상황에서는 그대로 적용되지 않는다. 가정은 다음과 같다. 첫째, 유동은 정상 상태여야 한다. 둘째, 유체는 비점성으로 취급되거나 점성 손실이 무시 가능해야 한다. 셋째, 유동은 비압축성으로 간주된다. 넷째, 방정식은 동일한 유선 상의 두 점에 적용된다. 다섯째, 유동의 체적력은 중력만이 고려된다. 이러한 가정이 만족되는 범위에서 베르누이 방정식은 매우 유용한 해석 도구가 된다.

에너지 해석 관점

베르누이 방정식은 유체의 총 기계 에너지가 유선을 따라 보존됨을 의미한다. 운동 에너지, 유동 일, 위치 에너지의 세 항은 서로 교환될 수 있으며, 한 항의 증가는 다른 항의 감소로 보상된다. 예를 들어 파이프의 좁아지는 부분에서는 속도가 증가하고 압력이 감소하며, 높이가 증가하는 부분에서는 압력이나 속도가 감소한다. 이러한 에너지 교환의 관점은 유체 현상의 직관적 이해를 제공한다.

유선과 비회전 유동에서의 확장

위에서 유도된 베르누이 방정식은 유선을 따라서만 성립하며, 서로 다른 유선 사이에서는 일반적으로 상수가 달라진다. 그러나 유동이 비회전(irrotational)인 경우, 즉 \nabla\times\mathbf{u} = 0인 경우에는 베르누이 방정식이 유동 전체에서 하나의 상수를 가진다. 이 경우 유동은 속도 포텐셜 \phi로 기술될 수 있으며(\mathbf{u} = \nabla\phi), 베르누이 방정식은 다음과 같이 확장된다.

\rho\frac{\partial\phi}{\partial t} + \frac{1}{2}\rho|\nabla\phi|^2 + p + \rho g z = f(t)

이는 비정상 비회전 유동에서도 유효한 일반화된 베르누이 방정식이다.

3. 정체점과 동압력

유선을 따라 유동이 감속되어 속도가 영이 되는 점을 정체점(stagnation point)이라 한다. 이 점에서의 압력은 정체 압력(stagnation pressure)이라 불리며, 베르누이 방정식으로부터 다음과 같이 표현된다.

p_0 = p + \frac{1}{2}\rho u^2

여기서 p_0는 정체 압력, p는 정적 압력이다. 차이 \frac{1}{2}\rho u^2는 동압력(dynamic pressure)이라 불리며, 유동 속도의 측정에 이용된다. 피토관(Pitot tube)은 정체 압력과 정적 압력의 차이를 측정하여 동압력을 구하고, 이로부터 유속을 계산한다. 이 원리는 항공기의 속도 측정과 드론의 공기 속도 센서에 널리 사용된다.

일반화된 에너지 방정식

베르누이 방정식은 비점성 가정의 특수한 경우이며, 보다 일반적인 에너지 방정식은 열역학 제1법칙을 유체 시스템에 적용하여 유도된다. 제어 체적에 대한 에너지 방정식은 다음과 같은 형태를 가진다.

\frac{dE_{CV}}{dt} = \dot Q - \dot W + \sum_{\text{in}}\dot m\left(h + \frac{u^2}{2} + gz\right) - \sum_{\text{out}}\dot m\left(h + \frac{u^2}{2} + gz\right)

여기서 \dot Q는 열 전달률, \dot W는 외부 일의 시간 변화율, h는 비엔탈피이다. 이 식은 압축성 유동, 열 전달, 외부 일이 포함된 경우에 적용되며, 베르누이 방정식은 이 일반 방정식의 특수한 경우로 간주된다.

4. 펌프와 터빈이 포함된 확장

파이프 시스템에 펌프나 터빈이 있는 경우, 베르누이 방정식은 외부 일의 항을 포함하도록 확장된다.

\frac{p_1}{\rho g} + \frac{u_1^2}{2g} + z_1 + h_p = \frac{p_2}{\rho g} + \frac{u_2^2}{2g} + z_2 + h_t

여기서 h_p는 펌프가 유체에 공급하는 수두, h_t는 터빈이 유체로부터 추출하는 수두이다. 이 확장된 형태는 펌프와 터빈이 포함된 유체 시스템의 해석에 사용되며, 산업용 로봇의 유압 시스템과 수중 로봇의 수력 펌프 해석에 적용된다.

손실 항을 고려한 확장

실제 유동에서는 점성, 난류, 유로의 곡률, 부품의 저항 등으로 인해 에너지 손실이 발생한다. 손실을 고려한 베르누이 방정식은 다음과 같이 확장된다.

\frac{p_1}{\rho g} + \frac{u_1^2}{2g} + z_1 = \frac{p_2}{\rho g} + \frac{u_2^2}{2g} + z_2 + h_L

여기서 h_L은 1과 2 사이의 총 수두 손실이다. 주요 손실은 파이프 벽의 마찰로 인해 발생하며 Darcy-Weisbach 방정식으로 기술된다. 부가 손실은 엘보, 밸브, 확장과 축소와 같은 국소 부품에서 발생하며 경험적 계수로 표현된다. 이러한 확장은 실무적 유체 시스템 해석의 표준 도구이다.

5. 압축성 유동에서의 베르누이 방정식

이상 기체의 등엔트로피 유동에 대해서도 베르누이형 방정식이 유도될 수 있다. 이 경우 비압축성 베르누이 방정식의 압력 항이 엔탈피 항으로 대체된다.

\frac{u^2}{2} + h + gz = \text{const}

또는 이상 기체의 경우 다음과 같이 표현된다.

\frac{u^2}{2} + \frac{\gamma}{\gamma - 1}\frac{p}{\rho} + gz = \text{const}

이 형태는 노즐, 확산기, 고속 유동의 해석에 사용된다. 저속 유동의 경우 이 형태는 비압축성 베르누이 방정식으로 수렴한다.

6. 베르누이 방정식의 적용 예

베르누이 방정식은 여러 실용적 문제에 직접 적용된다. 첫째, 벤투리관은 단면적의 변화에 따른 압력 차이로부터 유량을 측정한다. 둘째, 탱크 하부의 구멍에서 유출되는 유체의 속도는 u = \sqrt{2gh}로 예측되며, 이를 토리첼리의 법칙이라 한다. 셋째, 날개 주위의 양력은 상면과 하면의 속도 차이로 인한 압력 차이로 부분적으로 설명된다. 넷째, 피토관은 항공기와 드론의 공기 속도 측정의 표준 장치이다. 이러한 예제들은 베르누이 방정식의 실용적 가치를 잘 보여준다.

7. 로봇 공학적 응용

베르누이 방정식은 드론의 공기 속도 측정, 수중 로봇의 유속 추정, 공압 소프트 로봇의 압력-유량 관계 해석, 유압 시스템의 에너지 손실 평가, 노즐과 분사기의 설계 등 다양한 응용에 사용된다. 특히 드론의 피토관과 정적 포트는 비행 제어의 기초 데이터를 제공하며, 이들의 원리는 베르누이 방정식에 직접 기반한다. 소프트 로봇의 유체 분배 회로 설계에서는 손실 항을 고려한 확장된 베르누이 방정식이 필수적이다.

8. 본 절의 의의

본 절은 베르누이 방정식과 에너지 보존 법칙의 유도, 가정, 확장, 실무적 응용을 체계적으로 정리한다. 베르누이 방정식은 유체역학의 가장 널리 사용되는 해석 도구이며, 제한된 가정 하에서도 매우 유용한 결과를 제공한다. 본 절의 내용은 이후 절에서 다룰 오일러 방정식과 나비에-스토크스 방정식의 해석과 결합되어 유동의 완전한 이해를 지원한다.

9. 학습 권장사항

독자는 벤투리관의 압력 차이로부터 유량을 계산하는 문제, 탱크 출구 유출 속도의 계산, 피토관 측정으로부터 유속의 도출과 같은 고전적 문제를 직접 풀어 볼 것을 권장한다. 또한 손실 수두를 고려한 파이프 시스템의 해석을 Darcy-Weisbach 방정식과 결합하여 수행하는 것도 유익하다. 드론의 피토관 데이터를 분석하여 공기 속도를 추정하는 실습은 실무 감각을 제공한다.
