10.50 MRPмқҳ м„ұм§Ҳкіј нҡҢм „ н‘ңнҳ„ лІ”мң„

1. MRPмқҳ мЈјмҡ” м„ұм§Ҳ

мҲҳм • лЎңл“ңлҰ¬кІҢмҠӨ л§Өк°ңліҖмҲҳ(MRP)лҠ” кі м „ лЎңл“ңлҰ¬кІҢмҠӨ л§Өк°ңліҖмҲҳмқҳ н•ңкі„лҘј к·№ліөн•ҳл©ҙм„ң лҸ…нҠ№н•ң мҲҳн•ҷм Ғ м„ұм§Ҳмқ„ к°Җм§„лӢӨ. ліё м Ҳм—җм„ңлҠ” MRPмқҳ мЈјмҡ” м„ұм§Ҳкіј нҡҢм „ н‘ңнҳ„ лІ”мң„лҘј мІҙкі„м ҒмңјлЎң л¶„м„қн•ңлӢӨ.

2. кё°ліё м„ұм§Ҳ мҡ”м•Ҫ

2.1 м •мқҳ

\boldsymbol{\sigma} = \tan(\phi/4)\hat{\mathbf{u}}

2.2 л§Өк°ңліҖмҲҳмқҳ мҲҳ

3 л§Өк°ңліҖмҲҳ (мһҗмң  л§Өк°ңліҖмҲҳ, м ңм•Ҫ м—ҶмқҢ)

2.3 нҠ№мқҙм җ мң„м№ҳ

\phi = 2\pi (мӢӨм ңлЎңлҠ” нҡҢм „ кіөк°„мқҳ вҖңл°ҳлҢҖнҺёвҖқ)

2.4 лҢҖмІҙ MRP

\boldsymbol{\sigma}^S = -\boldsymbol{\sigma}/\lVert\boldsymbol{\sigma}\rVert^2

3. нҡҢм „ к°ҒлҸ„м—җ л”°лҘё л§Өк°ңліҖмҲҳ нҒ¬кё°

MRPмқҳ нҒ¬кё° \lVert\boldsymbol{\sigma}\rVert = \tan(\phi/4)мқҳ нҡҢм „ к°ҒлҸ„ мқҳмЎҙм„ұмқ„ м •лҹүм ҒмңјлЎң л¶„м„қн•ңлӢӨ.

3.1 н•өмӢ¬ кҙҖм°°

  • \phi = 180В°м—җм„ң \lVert\boldsymbol{\sigma}\rVert = 1 (м •нҷ•нһҲ 1)
  • \phi < 180В°м—җм„ң \lVert\boldsymbol{\sigma}\rVert < 1
  • \phi > 180В°м—җм„ң \lVert\boldsymbol{\sigma}\rVert > 1
  • \phi \to 360В°м—җм„ң \lVert\boldsymbol{\sigma}\rVert \to \infty

4. лӢЁмң„ кіөкіј нҡҢм „ лІ”мң„

4.1 MRP лӢЁмң„ кіө

MRPмқҳ нҒ¬кё°к°Җ 1мқё кіө \lVert\boldsymbol{\sigma}\rVert \leq 1мқҙ нҡҢм „ к°Ғ \phi \in [0, \pi]м—җ н•ҙлӢ№н•ңлӢӨ. мқҙлҠ” нҡҢм „ лІ”мң„мқҳ м Ҳл°ҳмқҙлӢӨ.

4.2 лӢЁмң„ кіө л°”к№Ҙ

\lVert\boldsymbol{\sigma}\rVert > 1мқҙл©ҙ \phi > \piм—җ н•ҙлӢ№н•ңлӢӨ. лҢҖмІҙ MRPлЎң м „нҷҳн•ҳм—¬ лӢӨмӢң лӢЁмң„ кіө лӮҙл¶ҖлЎң л§Өн•‘н•  мҲҳ мһҲлӢӨ.

4.3 лҢҖмІҙ л©”м»ӨлӢҲмҰҳ

лҢҖмІҙ MRP

\boldsymbol{\sigma}^S = -\frac{\boldsymbol{\sigma}}{\lVert\boldsymbol{\sigma}\rVert^2}

лҠ” лӢӨмқҢмқҳ кҙҖкі„лҘј л§ҢмЎұн•ңлӢӨ.

\lVert\boldsymbol{\sigma}\rVert\cdot\lVert\boldsymbol{\sigma}^S\rVert = 1

мҰү, мӣҗлһҳ MRPмқҳ нҒ¬кё°к°Җ 2мқҙл©ҙ лҢҖмІҙ MRPмқҳ нҒ¬кё°лҠ” 0.5мқҙлӢӨ.

5. нҡҢм „ лІ”мң„ л¶„м„қ

MRPк°Җ лҢҖмІҙ л©”м»ӨлӢҲмҰҳмқ„ мӮ¬мҡ©н•  л•Ң лӢӨмқҢмқҳ нҡҢм „ лІ”мң„лҘј н‘ңнҳ„н•  мҲҳ мһҲлӢӨ.

5.1 мӣҗлһҳ MRPл§Ң мӮ¬мҡ© (лҢҖмІҙ м—ҶмқҢ)

\phi \in [0, 2\pi)лҘј л§Өк°ңнҷ”н•ҳм§Җл§Ң \phi = \pi к·јмІҳм—җм„ң л§Өк°ңліҖмҲҳк°Җ нҒ¬кё° 1мқ„ л„ҳм–ҙк°„лӢӨ.

5.2 лҢҖмІҙ м „нҷҳ мӮ¬мҡ©

  • \phi \in [0, \pi]: \lVert\boldsymbol{\sigma}\rVert \in [0, 1]
  • \phi \in [\pi, 2\pi): лҢҖмІҙ MRPлЎң н‘ңнҳ„, \lVert\boldsymbol{\sigma}^S\rVert \in [0, 1]

мқҙлЎңмҚЁ л§Өк°ңліҖмҲҳ нҒ¬кё°к°Җ н•ӯмғҒ [0, 1] лІ”мң„мқҙлӢӨ.

5.3 н‘ңнҳ„ л¶Ҳк°ҖлҠҘн•ң нҡҢм „

\phi = 2\pi (м •нҷ•нһҲ н•ң л°”нҖҙ)лҠ” л§Өк°ңліҖмҲҳмқҳ нҠ№мқҙм җмқҙм§Җл§Ң, мӢӨм ңлЎңлҠ” \phi = 0кіј к°ҷмқҖ нҡҢм „мқҙлӢӨ. л”°лқјм„ң мӢӨмҡ©м ҒмңјлЎң н‘ңнҳ„ л¶Ҳк°ҖлҠҘн•ң нҡҢм „мқҖ м—ҶлӢӨ.

6. MRPмқҳ л¶Җнҳё мқҙмӨ‘м„ұ

MRPлҸ„ мҝјн„°лӢҲм–ёмІҳлҹј л¶Җнҳё мқҙмӨ‘м„ұмқҙ мһҲлӢӨ. мӣҗлһҳ MRP \boldsymbol{\sigma}мҷҖ лҢҖмІҙ MRP \boldsymbol{\sigma}^Sк°Җ к°ҷмқҖ нҡҢм „мқ„ лӮҳнғҖлӮёлӢӨ.

6.1 л‘җ к°Җм§Җ н‘ңнҳ„

мһ„мқҳмқҳ нҡҢм „м—җ лҢҖн•ҙ MRPлҠ” л‘җ к°Җм§Җ н‘ңнҳ„мқ„ к°Җм§„лӢӨ.

  • лӮҙл¶Җ н‘ңнҳ„: \lVert\boldsymbol{\sigma}\rVert < 1 (лҢҖмқ‘н•ҳлҠ” м§§мқҖ нҡҢм „ кІҪлЎң)
  • мҷёл¶Җ н‘ңнҳ„: \lVert\boldsymbol{\sigma}\rVert > 1 (лҢҖмқ‘н•ҳлҠ” кёҙ нҡҢм „ кІҪлЎң)

мқҙлҠ” мҝјн„°лӢҲм–ёмқҳ \mathbf{q}мҷҖ -\mathbf{q}мқҳ кҙҖкі„мҷҖ мң мӮ¬н•ҳлӢӨ.

6.2 кҙҖлЎҖ

мқјл°ҳм ҒмңјлЎң лӮҙл¶Җ н‘ңнҳ„(\lVert\boldsymbol{\sigma}\rVert < 1)мқ„ мӮ¬мҡ©н•ңлӢӨ. мқҙлҠ” л§Өк°ңліҖмҲҳ нҒ¬кё°к°Җ м ңн•ңлҗҳм–ҙ мҲҳм№ҳм ҒмңјлЎң м•Ҳм •м ҒмқҙлӢӨ.

7. лҢҖмІҙ м „нҷҳ м •мұ…

лҢҖмІҙ MRPлЎңмқҳ м „нҷҳ мӢңм җмқҖ м•Ңкі лҰ¬мҰҳм—җ л”°лқј лӢӨлҘҙлӢӨ.

7.1 м •мұ… 1: нҒ¬кё° кё°л°ҳ

\lVert\boldsymbol{\sigma}\rVertк°Җ 1мқ„ мҙҲкіјн•ҳл©ҙ мҰүмӢң м „нҷҳн•ңлӢӨ.

if norm(sigma) > 1:
    sigma = -sigma / (norm(sigma) ** 2)

7.2 м •мұ… 2: мһ„кі„к°’ кё°л°ҳ

лҚ” нҒ° мһ„кі„к°’(мҳҲ: 1.5)мқ„ мӮ¬мҡ©н•ҳм—¬ л№ҲлІҲн•ң м „нҷҳмқ„ н”јн•ңлӢӨ.

if norm(sigma) > threshold:
    sigma = -sigma / (norm(sigma) ** 2)

мқҙлҠ” м „нҷҳ мӢң л°ңмғқн•  мҲҳ мһҲлҠ” мҲҳм№ҳ мҳӨм°ЁлҘј к°җмҶҢмӢңнӮЁлӢӨ.

7.3 м •мұ… 3: м—°мҶҚм„ұ кё°л°ҳ

мӢңк°„ мӢңнҖҖмҠӨм—җм„ң л§Өк°ңліҖмҲҳмқҳ м—°мҶҚм„ұмқ„ мң м§Җн•ҳкё° мң„н•ҙ м „нҷҳмқ„ н”јн•ҳкұ°лӮҳ мЎ°м •н•ңлӢӨ.

8. MRPмқҳ мҲҳн•ҷм Ғ м„ұм§Ҳ

8.1 к°Җнҷҳм„ұ

MRP лҚ§м…ҲмқҖ к°Җнҷҳм Ғмқҙм§Җл§Ң, нҡҢм „ н•©м„ұмқҖ м•„лӢҲлӢӨ. MRPмқҳ нҡҢм „ н•©м„ұ кіөмӢқмқҖ ліөмһЎн•ң л№„к°Җнҷҳ кіөмӢқмқҙлӢӨ.

8.2 мһҗмң  л§Өк°ңліҖмҲҳ

MRPлҠ” лӢЁмң„ л…ёлҰ„ к°ҷмқҖ м ңм•Ҫмқҙ м—ҶмңјлҜҖлЎң, мөңм Ғнҷ” ліҖмҲҳлЎң м§Ғм ‘ мӮ¬мҡ©лҗ  мҲҳ мһҲлӢӨ.

8.3 л§ӨлҒ„лҹ¬мӣҖ

MRPлҠ” S^3м—җм„ң \mathbb{R}^3лЎңмқҳ мһ…мІҙ мӮ¬мҳҒмқҙлҜҖлЎң, лҢҖл¶Җл¶„мқҳ мҳҒм—ӯм—җм„ң л§ӨлҒ„лҹҪлӢӨ.

8.4 мЎ°л°Җм„ұ

MRPмқҳ лҢҖмІҙ л©”м»ӨлӢҲмҰҳмқҖ мҷ„м „н•ң SO(3) л§Өк°ңнҷ”лҘј м ңкіөн•ңлӢӨ. лӢЁмқј л§Өк°ңліҖмҲҳ н‘ңнҳ„мқҙ л¶Ҳк°ҖлҠҘн•ң мҳҒм—ӯмқ„ лҢҖмІҙлЎң мІҳлҰ¬н•ңлӢӨ.

9. MRPмқҳ мҡҙлҸҷн•ҷм Ғ м„ұм§Ҳ

MRPмқҳ мӢңк°„ лҜёл¶„кіј к°ҒмҶҚлҸ„мқҳ кҙҖкі„лҠ” лӢӨмқҢкіј к°ҷлӢӨ.

\dot{\boldsymbol{\sigma}} = \frac{1}{4}[\mathbf{B}(\boldsymbol{\sigma})]\boldsymbol{\omega}

м—¬кё°м„ң \mathbf{B}(\boldsymbol{\sigma})лҠ”

\mathbf{B}(\boldsymbol{\sigma}) = (1 - \lVert\boldsymbol{\sigma}\rVert^2)\mathbf{I} + 2[\boldsymbol{\sigma}]_\times + 2\boldsymbol{\sigma}\boldsymbol{\sigma}^T

мқҙ н–үл ¬мқҖ \lVert\boldsymbol{\sigma}\rVert \neq 0мқё лӘЁл“  мҳҒм—ӯм—җм„ң мһҳ м •мқҳлҗҳм–ҙ мһҲлӢӨ.

9.1 нҠ№мқҙм җмқҳ л¶Җмһ¬

мҡҙлҸҷн•ҷ л°©м •мӢқм—җ нҠ№мқҙм җмқҙ м—ҶлӢӨ(м Ғм–ҙлҸ„ \lVert\boldsymbol{\sigma}\rVert < 1 мҳҒм—ӯм—җм„ң). мқҙлҠ” MRPк°Җ мһҗм„ё м Ғл¶„кіј м ңм–ҙм—җ м Ғн•©н•ң мқҙмң мқҙлӢӨ.

10. MRPмқҳ нҡҢм „ н–үл ¬ ліҖнҷҳ

10.1 MRPм—җм„ң нҡҢм „ н–үл ¬лЎң

\mathbf{R}(\boldsymbol{\sigma}) = \mathbf{I} + \frac{8[\boldsymbol{\sigma}]_\times^2 - 4(1 - \lVert\boldsymbol{\sigma}\rVert^2)[\boldsymbol{\sigma}]_\times}{(1 + \lVert\boldsymbol{\sigma}\rVert^2)^2}

мқҙлҠ” лӢ«нһҢ нҳ•нғң кіөмӢқмқҙлӢӨ.

10.2 нҡҢм „ н–үл ¬м—җм„ң MRPлЎң

нҡҢм „ н–үл ¬ \mathbf{R}лЎңл¶Җн„° MRPлҠ” мҝјн„°лӢҲм–ёмқ„ кІҪмң н•ҳм—¬ кі„мӮ°лҗңлӢӨ.

\mathbf{q} = \mathrm{matrix\_to\_quaternion}(\mathbf{R})

\boldsymbol{\sigma} = \frac{\mathbf{q}_v}{1 + q_w}

11. MRPмқҳ нҠ№мқҙм җкіј нҡҢн”ј

11.1 мң мқјн•ң нҠ№мқҙм җ: \phi = 360В°

мқҙлЎ м ҒмңјлЎң \phi = 2\pi мҰү 360лҸ„ нҡҢм „мқҙ нҠ№мқҙм җмқҙм§Җл§Ң, мқҙлҠ” \phi = 0кіј к°ҷмқҖ нҡҢм „мқҙлӢӨ. мӢӨм ңлЎңлҠ” мӢӨмҡ©м Ғ л¬ём ңк°Җ м—ҶлӢӨ.

11.2 нҡҢн”ј л©”м»ӨлӢҲмҰҳ

лҢҖмІҙ MRP м „нҷҳмқ„ нҶөн•ҙ л§Өк°ңліҖмҲҳ нҒ¬кё°лҘј м ңн•ңн•ҳл©ҙ нҠ№мқҙм җ к·јмІҳлЎң м ‘к·јн•ҳм§Җ м•ҠлҠ”лӢӨ. мқҙлҠ” MRPмқҳ мЈјмҡ” мқҙм җмқҙлӢӨ.

12. MRPмқҳ мҡ©лҸ„

12.1 мһҗм„ё м ңм–ҙ

MRPлҠ” мҡ°мЈјм„ кіј лЎңлҙҮмқҳ мһҗм„ё м ңм–ҙм—җм„ң мӮ¬мҡ©лҗңлӢӨ. мһҗмң  л§Өк°ңліҖмҲҳмқҙлҜҖлЎң м ңм–ҙ лІ•м№ҷ м„Өкі„м—җ нҺёлҰ¬н•ҳлӢӨ.

12.2 мһҗм„ё м¶”м •

мқјл¶Җ мһҗм„ё м¶”м • м•Ңкі лҰ¬мҰҳмқҙ MRPлҘј мӮ¬мҡ©н•ңлӢӨ. нҠ№нһҲ м№јл§Ң н•„н„°мқҳ мғҒнғң лІЎн„°м—җ MRPк°Җ нҸ¬н•Ёлҗ  мҲҳ мһҲлӢӨ.

12.3 мҡ°мЈј н•ӯкіө

MRPлҠ” мҡ°мЈј н•ӯкіөм—җм„ң мһҗмЈј мӮ¬мҡ©лҗңлӢӨ. NASAмҷҖ ESAмқҳ мқјл¶Җ мҡ°мЈјм„ мқҙ MRP кё°л°ҳ мһҗм„ё мӢңмҠӨн…ңмқ„ мӮ¬мҡ©н•ңлӢӨ.

12.4 лЎңлҙҮ кІҪлЎң кі„нҡҚ

л§ӨлӢҲн“°л Ҳмқҙн„°мқҳ мһҗм„ё кІҪлЎң кі„нҡҚм—җм„ң MRPк°Җ мӮ¬мҡ©лҗ  мҲҳ мһҲлӢӨ. мһҗмң  л§Өк°ңліҖмҲҳк°Җ мөңм Ғнҷ”м—җ мң мҡ©н•ҳлӢӨ.

13. MRPмқҳ мһҘлӢЁм җ м •лҰ¬

13.1 мһҘм җ

  1. 3 л§Өк°ңліҖмҲҳ: мөңмҶҢ л§Өк°ңліҖмҲҳ
  2. мһҗмң  л§Өк°ңліҖмҲҳ: м ңм•Ҫ м—ҶмқҢ
  3. нҠ№мқҙм җ нҡҢн”ј: лҢҖмІҙ л©”м»ӨлӢҲмҰҳмңјлЎң кұ°мқҳ мҷ„м „н•ң нҡҢм „ лІ”мң„
  4. л§ӨлҒ„лҹ¬мӣҖ: лҢҖл¶Җл¶„мқҳ мҳҒм—ӯм—җм„ң л§ӨлҒ„лҹ¬мӣҖ
  5. мһҗм„ё м ңм–ҙ м Ғн•©: м ңм–ҙ лІ•м№ҷ м„Өкі„м—җ нҺёлҰ¬

13.2 лӢЁм җ

  1. ліөмһЎн•ң кіөмӢқ: мҡҙлҸҷн•ҷкіј н•©м„ұ кіөмӢқмқҙ мҝјн„°лӢҲм–ёліҙлӢӨ ліөмһЎ
  2. лҢҖмІҙ м „нҷҳ: кө¬нҳ„ ліөмһЎм„ұ мҰқк°Җ
  3. SLERP м—ҶмқҢ: мҝјн„°лӢҲм–ёмІҳлҹј мһҗм—°мҠӨлҹ¬мҡҙ ліҙк°„ м—ҶмқҢ
  4. мқјл°ҳм„ұ л¶ҖмЎұ: мҝјн„°лӢҲм–ёліҙлӢӨ лҚң мқјл°ҳм Ғ

14. MRPмҷҖ мҝјн„°лӢҲм–ё м„ нғқ к°Җмқҙл“ң

14.1 MRPлҘј м„ нғқн•ҳлҠ” кІҪмҡ°

  • мһҗмң  л§Өк°ңліҖмҲҳк°Җ н•„мҡ”н•ң мөңм Ғнҷ”
  • мҡ°мЈј н•ӯкіө мқ‘мҡ© (м „нҶөм Ғ мқҙмң )
  • 3 л§Өк°ңліҖмҲҳ м„ нҳё

14.2 мҝјн„°лӢҲм–ёмқ„ м„ нғқн•ҳлҠ” кІҪмҡ°
