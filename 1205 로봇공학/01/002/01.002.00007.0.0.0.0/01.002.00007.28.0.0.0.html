7.28 미적분학의 기본 정리

1. 개요

미적분학의 기본 정리(Fundamental Theorem of Calculus, FTC)는 미분과 적분이 서로 역연산임을 엄밀하게 확립하는 정리이다. 이 정리는 크게 두 부분으로 구성되며, 제1기본 정리는 적분으로 정의된 함수의 미분 가능성을, 제2기본 정리는 정적분의 계산을 역도함수로 환원하는 방법을 서술한다.

2. 제1기본 정리 (미분의 적분)

2.1 정리의 진술

f가 닫힌 구간 [a, b]에서 연속인 함수라 하자. 다음과 같이 정의된 함수 F는 열린 구간 (a, b)에서 미분가능하며, 그 도함수는 f와 같다.

F(x) = \int_a^x f(t) \, dt, \quad x \in [a, b]

F'(x) = \frac{d}{dx} \int_a^x f(t) \, dt = f(x), \quad \forall x \in (a, b)

2.2 증명의 핵심

미분의 정의에 따라 다음을 보인다.

F'(x) = \lim_{h \to 0} \frac{F(x + h) - F(x)}{h} = \lim_{h \to 0} \frac{1}{h} \int_x^{x+h} f(t) \, dt

적분의 평균값 정리에 의해, x와 x + h 사이에 c_h가 존재하여 다음이 성립한다.

\frac{1}{h} \int_x^{x+h} f(t) \, dt = f(c_h)

h \to 0이면 c_h \to x이고, f의 연속성에 의해 f(c_h) \to f(x)이므로 다음을 얻는다.

F'(x) = f(x)

2.3 일반화된 형태

적분의 상한과 하한이 모두 x의 함수인 경우, 라이프니츠 적분 규칙(Leibniz integral rule)을 적용한다.

\frac{d}{dx} \int_{\alpha(x)}^{\beta(x)} f(t) \, dt = f(\beta(x)) \cdot \beta'(x) - f(\alpha(x)) \cdot \alpha'(x)

예를 들어, 다음과 같은 도함수를 구할 수 있다.

\frac{d}{dx} \int_0^{x^2} e^{-t^2} \, dt = e^{-x^4} \cdot 2x

3. 제2기본 정리 (적분의 미분)

3.1 정리의 진술

f가 닫힌 구간 [a, b]에서 연속이고, F가 f의 역도함수(즉, F'(x) = f(x))이면 다음이 성립한다.

\int_a^b f(x) \, dx = F(b) - F(a)

이를 뉴턴-라이프니츠 공식(Newton-Leibniz formula)이라고도 부르며, F(b) - F(a)를 간략히 \bigl[F(x)\bigr]_a^b 또는 F(x) \Big\vert_a^b로 표기한다.

3.2 증명의 핵심

G(x) = \int_a^x f(t) \, dt로 정의하면, 제1기본 정리에 의해 G'(x) = f(x)이다. F도 f의 역도함수이므로, (F - G)'(x) = 0이 되어 F(x) - G(x) = C(상수)이다.

x = a를 대입하면 F(a) - G(a) = C이고, G(a) = 0이므로 C = F(a)이다. 따라서

G(x) = F(x) - F(a)

x = b를 대입하면 다음을 얻는다.

\int_a^b f(x) \, dx = G(b) = F(b) - F(a)

4. 두 기본 정리의 관계

제1기본 정리와 제2기본 정리는 서로 역관계에 있다. 이를 요약하면 다음과 같다.

미분과 적분의 역관계:

\frac{d}{dx} \int_a^x f(t) \, dt = f(x) \quad \text{(제1기본 정리: 적분 후 미분 = 원래 함수)}

\int_a^b F'(x) \, dx = F(b) - F(a) \quad \text{(제2기본 정리: 미분 후 적분 = 순변화)}

이 역관계는 미분 연산자 \dfrac{d}{dx}와 적분 연산자 \displaystyle\int_a^x (\cdot) \, dt가 서로 역연산임을 나타낸다.

5. 벡터값 함수로의 확장

벡터값 함수 \mathbf{f}: [a, b] \to \mathbb{R}^n에 대해, 각 성분이 연속이면 기본 정리가 성분별로 적용된다.

\frac{d}{dt} \int_a^t \mathbf{f}(\tau) \, d\tau = \mathbf{f}(t)

\int_a^b \mathbf{F}'(t) \, dt = \mathbf{F}(b) - \mathbf{F}(a)

여기서 \mathbf{F}(t) = [F_1(t), F_2(t), \ldots, F_n(t)]^\top이고 F_i'(t) = f_i(t)이다.

6. 로봇공학에서의 응용

6.1 관절 변위의 계산

로봇 관절의 각속도 \dot{q}(t)가 시간 구간 [t_0, t_f]에서 주어졌을 때, 관절 변위(angular displacement)는 제2기본 정리에 의해 다음과 같이 구한다.

q(t_f) - q(t_0) = \int_{t_0}^{t_f} \dot{q}(t) \, dt

예를 들어, 등속도 운동 \dot{q}(t) = \omega_0의 경우 다음과 같다.

q(t_f) - q(t_0) = \omega_0 (t_f - t_0)

사인파 형태의 각속도 \dot{q}(t) = A \sin(\omega t)인 경우 다음과 같다.

q(t_f) - q(t_0) = \int_{t_0}^{t_f} A \sin(\omega t) \, dt = \frac{A}{\omega} \bigl[-\cos(\omega t)\bigr]_{t_0}^{t_f} = \frac{A}{\omega} \bigl[\cos(\omega t_0) - \cos(\omega t_f)\bigr]

6.2 일과 에너지 정리

로봇 매니퓰레이터에서 일반화 힘 \boldsymbol{\tau}(t)가 한 일은 다음과 같다.

W = \int_{t_0}^{t_f} \boldsymbol{\tau}(t)^\top \dot{\mathbf{q}}(t) \, dt

운동 에너지 T = \frac{1}{2} \dot{\mathbf{q}}^\top \mathbf{M}(\mathbf{q}) \dot{\mathbf{q}}의 시간 변화율과 관련지으면, 일-에너지 정리(work-energy theorem)를 적분 형태로 표현할 수 있다.

W = T(t_f) - T(t_0) + \int_{t_0}^{t_f} \frac{\partial V}{\partial \mathbf{q}} \dot{\mathbf{q}} \, dt

여기서 V(\mathbf{q})는 위치 에너지이다. 이 관계는 제2기본 정리의 직접적인 응용이다.

6.3 충격량-운동량 관계

외부 힘 \mathbf{F}(t)가 시간 구간 [t_1, t_2] 동안 작용할 때, 충격량(impulse) \mathbf{J}는 다음과 같다.

\mathbf{J} = \int_{t_1}^{t_2} \mathbf{F}(t) \, dt = \mathbf{p}(t_2) - \mathbf{p}(t_1)

여기서 \mathbf{p}(t)는 선운동량으로, \mathbf{F}(t) = \dot{\mathbf{p}}(t)이다. 이는 뉴턴의 제2법칙을 적분한 결과로, 제2기본 정리의 응용이다.

6.4 궤적 계획에서의 경계 조건

다항식 궤적 q(t) = \sum_{k=0}^{m} a_k t^k에서, 초기 시각과 최종 시각 사이의 속도 변화는 가속도의 적분으로 주어진다.

\dot{q}(t_f) - \dot{q}(t_0) = \int_{t_0}^{t_f} \ddot{q}(t) \, dt

이 관계를 통해 궤적의 경계 조건(초기/최종 위치, 속도, 가속도)을 만족하는 다항식 계수를 결정할 수 있다. 5차 다항식 궤적의 경우 6개의 경계 조건을 설정하면 다음 시스템을 풀어야 한다.

q(t_0) = q_0, \quad \dot{q}(t_0) = v_0, \quad \ddot{q}(t_0) = a_0
q(t_f) = q_f, \quad \dot{q}(t_f) = v_f, \quad \ddot{q}(t_f) = a_f

6.5 센서 데이터의 적분

관성 측정 장치(IMU)에서 측정된 각속도 \boldsymbol{\omega}(t)로부터 자세(orientation)를 복원할 때, 제1기본 정리에 의해 적분 함수의 미분이 원래 측정값을 복원해야 한다는 조건을 활용한다. 수치적으로는 다음과 같은 이산 적분을 수행한다.

\boldsymbol{\theta}(t_{k+1}) = \boldsymbol{\theta}(t_k) + \int_{t_k}^{t_{k+1}} \boldsymbol{\omega}(\tau) \, d\tau \approx \boldsymbol{\theta}(t_k) + \boldsymbol{\omega}(t_k) \Delta t

이 과정에서 적분 오차가 누적되는 드리프트(drift) 현상이 발생하며, 이를 보정하기 위해 상보 필터(complementary filter)나 칼만 필터(Kalman filter)를 사용한다.
