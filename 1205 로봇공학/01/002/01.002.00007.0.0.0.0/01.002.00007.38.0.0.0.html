7.38 로봇 링크의 동역학 파라미터 산출

로봇 매니퓰레이터의 동역학 방정식을 수립하기 위해서는 각 링크의 동역학 파라미터(dynamic parameters)를 정확하게 알아야 한다. 이 파라미터에는 질량, 무게 중심 위치, 관성 텐서가 포함되며, 이들은 링크의 기하 형상과 밀도 분포로부터 적분을 통해 산출된다. 이 절에서는 실제 로봇 링크에 대한 동역학 파라미터의 체계적 산출 방법을 다룬다.

1. 동역학 파라미터의 구성

n자유도 로봇 매니퓰레이터에서 i번째 링크의 동역학 파라미터는 다음으로 구성된다.

  • 질량: m_i
  • 무게 중심 위치: \mathbf{r}_{c,i} = (r_{cx,i}, r_{cy,i}, r_{cz,i})^T (링크 좌표계 기준)
  • 관성 텐서: \mathbf{I}_i (무게 중심 기준, 3 \times 3 대칭 행렬)

관성 텐서는 대칭 행렬이므로 독립적인 성분은 6개이다. 따라서 각 링크는 1 + 3 + 6 = 10개의 독립 동역학 파라미터를 가진다.

\boldsymbol{\pi}_i = \begin{pmatrix} m_i \\ m_i r_{cx,i} \\ m_i r_{cy,i} \\ m_i r_{cz,i} \\ I_{xx,i} \\ I_{xy,i} \\ I_{xz,i} \\ I_{yy,i} \\ I_{yz,i} \\ I_{zz,i} \end{pmatrix} \in \mathbb{R}^{10}

여기서 1차 모멘트 m_i r_{cx,i} 등을 직접 사용하면 파라미터가 질량에 대해 선형이 되어 식별(identification)에 유리하다.

2. CAD 기반 파라미터 산출

2.1 기하 모델링

현대의 로봇 설계에서는 CAD(Computer-Aided Design) 소프트웨어를 통해 각 링크의 3차원 형상을 정밀하게 모델링한다. 이때 링크 형상은 수학적으로 유계 영역 E_i \subset \mathbb{R}^3로 표현된다.

2.2 적분에 의한 파라미터 계산

각 링크의 재질과 밀도 분포 \rho_i(\mathbf{r})가 알려져 있으면, 동역학 파라미터는 다음의 적분으로 계산된다.

질량:

m_i = \iiint_{E_i} \rho_i(\mathbf{r}) \, dV

1차 모멘트 (무게 중심):

m_i \bar{x}_i = \iiint_{E_i} x \rho_i(\mathbf{r}) \, dV, \quad m_i \bar{y}_i = \iiint_{E_i} y \rho_i(\mathbf{r}) \, dV, \quad m_i \bar{z}_i = \iiint_{E_i} z \rho_i(\mathbf{r}) \, dV

관성 텐서 (원점 기준):

\bar{I}_{xx,i} = \iiint_{E_i} (y^2 + z^2) \rho_i \, dV, \quad \bar{I}_{xy,i} = -\iiint_{E_i} xy \rho_i \, dV

나머지 성분도 동일한 패턴으로 구한다.

무게 중심 기준 관성 텐서:

원점 기준 관성 텐서 \bar{\mathbf{I}}_i로부터 평행축 정리를 적용하여 무게 중심 기준 관성 텐서를 구한다.

\mathbf{I}_i = \bar{\mathbf{I}}_i - m_i \left[(\mathbf{r}_{c,i}^T \mathbf{r}_{c,i})\mathbf{E}_3 - \mathbf{r}_{c,i}\mathbf{r}_{c,i}^T\right]

3. 복합 링크의 파라미터 합성

실제 로봇 링크는 본체, 모터, 감속기, 센서 등 여러 부품으로 구성된다. 총 k개의 부품에 대해 각 부품의 파라미터가 알려져 있을 때, 링크 전체의 파라미터는 다음과 같이 합성된다.

3.1 전체 질량

m = \sum_{j=1}^{k} m_j

3.2 전체 무게 중심

\mathbf{r}_c = \frac{1}{m}\sum_{j=1}^{k} m_j \mathbf{r}_{c,j}

3.3 전체 관성 텐서

각 부품의 관성 텐서를 공통 원점(예: 링크 좌표계 원점)으로 평행 이동시킨 후 합산한다.

\mathbf{I}_{\text{total}} = \sum_{j=1}^{k} \left[\mathbf{I}_{c,j} + m_j\left((\mathbf{d}_j^T\mathbf{d}_j)\mathbf{E}_3 - \mathbf{d}_j\mathbf{d}_j^T\right)\right]

여기서 \mathbf{d}_j = \mathbf{r}_{c,j} - \mathbf{r}_c는 j번째 부품의 무게 중심에서 전체 무게 중심까지의 변위이다.

4. 표준 형상 분해법

복잡한 링크 형상을 표준 기본 형상(직육면체, 원통, 구, 원뿔 등)의 결합으로 근사한다. 빈 공간(구멍)은 음의 질량을 가진 형상으로 모델링한다.

4.1 예제: 중공 원통 + 플랜지

중공 원통 본체(외경 R_2, 내경 R_1, 높이 h_1)와 원판형 플랜지(반지름 R_3, 높이 h_2)로 구성된 링크를 고려한다.

본체 질량과 관성:

m_1 = \rho \pi (R_2^2 - R_1^2) h_1

I_{zz,1} = \frac{m_1}{2}(R_1^2 + R_2^2)

I_{xx,1} = I_{yy,1} = \frac{m_1}{12}(3R_1^2 + 3R_2^2 + h_1^2)

플랜지 질량과 관성 (자체 무게 중심 기준):

m_2 = \rho \pi R_3^2 h_2

I_{zz,2} = \frac{m_2 R_3^2}{2}

I_{xx,2} = I_{yy,2} = \frac{m_2}{12}(3R_3^2 + h_2^2)

플랜지가 본체의 끝에 부착되어 있으면, 평행축 정리를 사용하여 공통 좌표계로 환산한 후 합산한다.

5. 관성 텐서의 좌표 변환

링크 i의 관성 텐서가 프레임 A에서 {}^A\mathbf{I}_i로 표현되어 있을 때, 회전 행렬 {}^B\mathbf{R}_A를 사용하여 프레임 B에서의 관성 텐서는 다음과 같이 변환된다.

{}^B\mathbf{I}_i = {}^B\mathbf{R}_A \cdot {}^A\mathbf{I}_i \cdot ({}^B\mathbf{R}_A)^T

이 변환은 관성 텐서가 2차 텐서(second-order tensor)임을 반영한다.

6. 기저 파라미터 (Base Parameters)

n자유도 로봇의 동역학 파라미터 총 수는 10n이지만, 이 중 일부는 동역학 방정식에 독립적으로 기여하지 않는다. 동역학 방정식에 독립적으로 영향을 미치는 최소 파라미터 집합을 기저 파라미터(base parameters)라 한다.

로봇의 운동 방정식을 다음과 같이 파라미터에 대해 선형인 형태로 쓸 수 있다.

\boldsymbol{\tau} = \mathbf{Y}(\mathbf{q}, \dot{\mathbf{q}}, \ddot{\mathbf{q}}) \boldsymbol{\pi}

여기서 \mathbf{Y}는 리그레서 행렬(regressor matrix)이고, \boldsymbol{\pi}는 10n차원 파라미터 벡터이다. 기저 파라미터는 \mathbf{Y}의 열공간(column space)에서 선형 독립인 열에 대응하는 파라미터 또는 그 선형 결합이다.

기저 파라미터의 수 n_b는 다음을 만족한다.

n_b = \text{rank}(\mathbf{Y}) \leq 10n

일반적으로 n_b < 10n이며, 특히 다음과 같은 경우에 파라미터 수가 줄어든다.

  • 회전 관절의 경우 z축 방향 1차 모멘트와 I_{zz}가 이전 링크의 파라미터와 결합된다.
  • 첫 번째 링크의 일부 파라미터는 고정 베이스로 인해 동역학에 영향을 미치지 않는다.

7. 마찰 파라미터

실제 로봇에서는 관절 마찰도 동역학 파라미터에 포함된다. 가장 일반적인 모델은 쿨롱 마찰과 점성 마찰의 결합이다.

\tau_{f,i} = f_{c,i} \text{sgn}(\dot{q}_i) + f_{v,i} \dot{q}_i

여기서 f_{c,i}는 쿨롱 마찰 계수, f_{v,i}는 점성 마찰 계수이다. 이를 포함하면 각 관절에 2개의 파라미터가 추가되어 전체 파라미터 수는 10n + 2n = 12n이 된다.

8. 파라미터 산출의 실제적 고려사항

8.1 측정 기반 산출

CAD 데이터가 불완전하거나 실제 부품의 밀도가 불균일한 경우, 다음의 실험적 방법을 병행한다.

  • 질량: 저울을 사용한 직접 측정
  • 무게 중심: 다점 현수법(suspension method) 또는 저울 위 회전법
  • 관성 모멘트: 복합 진자법(compound pendulum method) 또는 비틀림 진자법(torsional pendulum method)

비틀림 진자법에서 진동 주기 T와 비틀림 강성 k_t를 측정하면, 관성 모멘트는 다음과 같이 구해진다.

I = \frac{k_t T^2}{4\pi^2}

8.2 시스템 식별 기반 산출

로봇을 실제로 구동하며 관절 토크와 운동 데이터를 측정하여 기저 파라미터를 추정하는 방법이다. 최소자승법(least squares)을 사용한다.

\hat{\boldsymbol{\pi}}_b = (\mathbf{Y}_b^T \mathbf{Y}_b)^{-1} \mathbf{Y}_b^T \boldsymbol{\tau}_{\text{meas}}

여기서 \mathbf{Y}_b는 기저 파라미터에 대응하는 리그레서 행렬이고, \boldsymbol{\tau}_{\text{meas}}는 측정된 토크 벡터이다.

9. 파라미터 정확도의 영향

동역학 파라미터의 오차는 다음에 직접적으로 영향을 미친다.

응용 분야파라미터 오차의 영향
피드포워드 제어토크 보상 오차, 추적 성능 저하
중력 보상정적 위치 오차
충돌 감지오검출(false positive) 또는 미검출(false negative)
에너지 최적 궤적비최적 에너지 소비

따라서 고성능 로봇 제어를 위해서는 동역학 파라미터의 정확한 산출이 필수적이며, CAD 기반 계산과 실험적 식별을 상호 보완적으로 활용하는 것이 바람직하다.
